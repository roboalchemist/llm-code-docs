KEY FINDINGS:
=============

DISTRIBUTED GRAPH PROCESSING:
- Pregel (Google): Foundation model, web-scale
- Apache Giraph: 50B+ edges, 2.8-8.6x faster than GraphX
- Spark GraphX: 10B edges max, best for Spark users
- Plato (Tencent): 10x faster, 16-116x less memory (best performance)
- GraphScope (Alibaba): Unified analytics+queries+learning platform (2024 Flex)
- PowerGraph: Legacy vertex-cut pioneer

ANALYTICS LIBRARIES:
- NetworkX: Best for learning/prototyping, pure Python, slow on large graphs
- igraph: 27x faster than NetworkX, C core, excellent for production
- graph-tool: C++ with Python bindings, statistical focus
- SNAP: Stanford-based, widely used (brief reference)

GRAPH NEURAL NETWORKS:
- PyTorch Geometric: Pythonic API, extensive models, flexibility
- DGL: 2.6x faster training than PyG, superior sampling, memory efficient

GRAPH DATABASES:
- Neo4j: Market leader, mature ecosystem, Cypher query language
- TigerGraph: Enterprise analytics, terabyte-scale, hybrid memory-disk
- Memgraph: In-memory, lightning-fast, excellent for real-time (fraud detection)
- JanusGraph: Distributed, 100B+ edges, open-source, big data integration
- ArangoDB: Multi-model (documents+graphs+search), flexible

VISUALIZATION:
- Gephi: Large networks (100k+ nodes), excellent scalability
- Cytoscape: Biological networks, 200+ specialized plugins
- Graphviz: Automatic layout, DOT language, DAG specialization

MAJOR INSIGHTS:
================

Performance Hierarchy (by speed):
1. Plato (10x fastest)
2. Giraph (3-9x faster)
3. GraphX (baseline 1.0)
4. igraph (27x faster than NetworkX)
5. NetworkX (slowest for large graphs)

Scale Rankings:
1. Pregel/TigerGraph/JanusGraph: Terabyte/petabyte scale
2. Giraph/Plato: 50B+ edges
3. GraphX: 10B edges
4. Neo4j/Memgraph: Billions of relationships
5. igraph/graph-tool: Millions of nodes
6. NetworkX: Thousands of nodes

Real-time Performance Leaders:
1. Memgraph (in-memory)
2. Neo4j (optimized traversal)
3. Plato (distributed clusters)

ML/GNN Leaders:
1. DGL (2.6x faster training, memory efficient)
2. PyG (flexibility, Pythonic)
3. Plato (distributed GNN training)
4. GraphScope (integrated learning)

Enterprise-Grade:
1. TigerGraph (terabyte scale)
2. Neo4j (mature ecosystem)
3. Memgraph (speed + open-source)
