COMPREHENSIVE LIST: 13 MAJOR MANAGED GRAPH DATABASE SERVICES
================================================================================

PRIMARY CLOUD PROVIDER OFFERINGS
────────────────────────────────────────────────────────────────────────────

1. AMAZON NEPTUNE (AWS)
   - Status: Mature, fully managed
   - Query Languages: Gremlin, SPARQL, openCypher
   - Scale: Billions of relationships, 15 read replicas
   - Strengths: AWS integration, semantic reasoning, knowledge graphs
   - Pricing: Instance + I/O + Storage
   - Entry Cost: ~$150-300/month
   - Best For: Knowledge graphs, metadata, fraud detection

2. AZURE COSMOS DB (Gremlin API)
   - Status: Mature, globally distributed
   - Query Languages: Gremlin + multi-model (SQL, MongoDB)
   - Scale: Global regions, serverless options
   - Strengths: Multi-model, global distribution, SLAs
   - Pricing: RU-based (request units) + storage
   - Entry Cost: ~$100-200/month
   - Best For: Global applications, multi-model workloads

3. GOOGLE CLOUD SPANNER GRAPH
   - Status: Newly released, strong consistency focus
   - Query Languages: SQL + graph extensions
   - Scale: Petabyte-scale with horizontal sharding
   - Strengths: ACID across graphs, SQL compatibility, BigQuery integration
   - Pricing: vCPU + storage + committed discounts
   - Entry Cost: ~$330+/month
   - Best For: Large relational graphs, strong consistency needs

================================================================================
INDEPENDENT MANAGED SERVICES (MULTI-CLOUD)
────────────────────────────────────────────────────────────────────────────

4. NEO4J AURADB
   - Status: Market leader, mature ecosystem
   - Query Language: Cypher (SQL-inspired)
   - Scale: Petabyte-scale, native graph storage
   - Strengths: Developer experience, 50+ graph algorithms, large community
   - Pricing: Monthly subscription + overages
   - Entry Cost: $65-150/month
   - Best For: Relationship-heavy queries, recommendations, developer productivity
   - Availability: AWS, Azure, GCP

5. TIGERGRAPH CLOUD
   - Status: Analytics specialist, MPP architecture
   - Query Language: GSQL (proprietary, procedural)
   - Scale: Trillion-edge graphs, billions nodes/hour ingestion
   - Strengths: Compute-intensive analytics, pattern matching, high throughput
   - Pricing: Usage-based (compute hours + storage)
   - Entry Cost: Free tier available, paid ~$100+/month
   - Best For: Fraud detection, personalization, deep graph analytics
   - Availability: AWS, Azure, Kubernetes

6. NEBULAGRAPH CLOUD
   - Status: Rapidly growing, transparent pricing
   - Query Language: nGQL (SQL-like)
   - Scale: Storage/compute separation, billions-trillions edges
   - Strengths: Distributed architecture, transparent pricing, vector search
   - Pricing: Clear hourly rates ($0.049-$3.76/hour)
   - Entry Cost: $36-50/month (Standard plan)
   - Best For: Large distributed graphs, transparent pricing preference
   - Availability: AWS Marketplace, Azure Marketplace

7. ARANGODB OASIS
   - Status: Mature multi-model platform
   - Query Language: AQL (unified across all models)
   - Scale: Cluster-based horizontal scaling
   - Strengths: Multi-model (graph + documents), unified language
   - Pricing: Per-node + storage + bandwidth
   - Entry Cost: $35-50/month
   - Best For: Mixed documents + graphs, unified query needs
   - Availability: AWS, Azure, GCP

8. DGRAPH CLOUD
   - Status: GraphQL-first, native support
   - Query Languages: GraphQL (native) + DQL (alternative)
   - Scale: Auto-sharding, distributed execution
   - Strengths: Native GraphQL, real-time subscriptions, rapid API development
   - Pricing: Usage-based + storage
   - Entry Cost: Free tier available, paid ~$100+/month
   - Best For: GraphQL APIs, real-time subscriptions
   - Availability: Multi-cloud

================================================================================
SPECIALIZED GRAPH PLATFORMS
────────────────────────────────────────────────────────────────────────────

9. REDIS ENTERPRISE CLOUD (Graphs)
   - Status: In-memory, ultra-low latency
   - Query Language: Cypher (on top of Redis)
   - Scale: RAM-limited (in-memory)
   - Strengths: Microsecond latency, real-time scoring, caching layer
   - Pricing: Per-GB RAM hour
   - Entry Cost: $7-15/month
   - Best For: Real-time rankings, session stores, caching layer

10. ONTOTEXT GRAPHDB
    - Status: Enterprise RDF platform
    - Query Language: SPARQL (semantic/RDF)
    - Scale: Enterprise clustering
    - Strengths: Semantic reasoning, inference, OWL ontologies
    - Pricing: Custom (GraphDB Cloud managed service)
    - Entry Cost: Not listed (requires consultation)
    - Best For: Knowledge graphs, semantic reasoning, linked data

11. RDFOX
    - Status: High-performance semantic engine
    - Query Language: SPARQL
    - Scale: Distributed in-memory reasoning
    - Strengths: Parallel reasoning, semantic performance
    - Pricing: Custom (managed cloud available)
    - Entry Cost: Not listed (requires consultation)
    - Best For: Real-time semantic reasoning at scale

12. FAUNADB
    - Status: Serverless multi-model
    - Query Languages: GraphQL + relational
    - Scale: Global serverless, automatic scaling
    - Strengths: Serverless, global distribution, strong consistency
    - Pricing: Per-request (pay-what-you-use)
    - Entry Cost: $100-200/month typical
    - Best For: Serverless applications, GraphQL APIs

13. GOOGLE CLOUD BIGQUERY (Graph Analytics)
    - Status: Data warehouse with graph capabilities
    - Query Language: SQL + graph extensions (AGE planned)
    - Scale: Petabyte-scale serverless
    - Strengths: Massive analytical scale, Vertex AI integration
    - Pricing: Per-GB scanned + analysis
    - Entry Cost: Usage-based
    - Best For: Analytical workloads on massive graphs
