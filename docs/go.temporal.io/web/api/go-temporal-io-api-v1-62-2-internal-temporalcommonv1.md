### temporalcommonv1

This package contains [payload_json.go](https://github.com/temporalio/api-go/blob/v1.62.2/internal/temporalcommonv1/payload_json.go) which is copied to [../../common/v1](https://github.com/temporalio/api-go/blob/v1.62.2/common/v1) after proto generation. The code is kept here separately to keep the delete-entire-directory cleanup before code generation simple.

This code adds "shorthand" formatting support to payloads by implementing `MaybeMarshalProtoJSON` and `MaybeUnmarshalProtoJSON` on both `Payloads` and `Payload`. Tests for this code are in [../../proxy/marshal_test.go](https://github.com/temporalio/api-go/blob/v1.62.2/proxy/marshal_test.go).
