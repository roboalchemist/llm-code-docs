#### type [Buffer](https://github.com/temporalio/temporal/blob/v1.30.0/common/effect/buffer.go#L9)[¶](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer "Go to Buffer")

type Buffer struct {
	// contains filtered or unexported fields
}

Buffer holds a set of effect and rollback functions that can be invoked as a batch with a defined order. Once either Apply or Cancel is called, all buffered effects are cleared. This type is not threadsafe. The zero-value of a Buffer is a valid state.

#### func (*Buffer) [Apply](https://github.com/temporalio/temporal/blob/v1.30.0/common/effect/buffer.go#L29)[¶](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer.Apply "Go to Buffer.Apply")

func (b *[Buffer](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer)) Apply(ctx [context](https://pkg.go.dev/context).[Context](https://pkg.go.dev/context#Context)) [bool](https://pkg.go.dev/builtin#bool)

Apply invokes the buffered effect functions in the order that they were added to this Buffer. It returns true if any effects were applied.

#### func (*Buffer) [Cancel](https://github.com/temporalio/temporal/blob/v1.30.0/common/effect/buffer.go#L43)[¶](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer.Cancel "Go to Buffer.Cancel")

func (b *[Buffer](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer)) Cancel(ctx [context](https://pkg.go.dev/context).[Context](https://pkg.go.dev/context#Context)) [bool](https://pkg.go.dev/builtin#bool)

Cancel invokes the buffered rollback functions in the reverse of the order that they were added to this Buffer. It returns true if any effects were canceled.

#### func (*Buffer) [OnAfterCommit](https://github.com/temporalio/temporal/blob/v1.30.0/common/effect/buffer.go#L16)[¶](https://pkg.go.dev/go.temporal.io/server@v1.30.0/common/effect#Buffer.OnAfterCommit "Go to Buffer.OnAfterCommit")
