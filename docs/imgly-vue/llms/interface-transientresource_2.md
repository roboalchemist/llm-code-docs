# Interface: TransientResource

Represents a transient resource.

The `TransientResource` interface provides a set of properties that describe a transient resource, including a URL and the size of the resource.

## Properties[#](#properties)

| Property | Type |
| --- | --- |
| `URL` | `string` |
| `size` | `number` |
