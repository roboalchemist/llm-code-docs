# Deactivating and reactivating users

Admins (and managers) have the option to deactivate a user. Once an Admin deactivates a user in the team, the user will be signed out from all devices and won't be able to sign in again.

Only the Admin can deactivate or reactivate other Admins.

### Deactivate a user

1. Sign in to the Tabnine console as an Admin.
2. Go to the **Users** page.
3. Select the user you want to deactivate and click on the three-dot menu on the right.
4. Click **Deactivate.**

### Reactivate a user

1. Sign in to the Tabnine console as an Admin.
2. Go to the **Users** page.
3. Select the deactivated user you want to reactivate and click on the three-dot menu on the right.
4. Click **Activate.**
