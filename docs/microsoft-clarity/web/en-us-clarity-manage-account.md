Manage account
--------------

[](https://learn.microsoft.com/en-us/clarity/manage-account#step-1)
### Step 1

Go to the user icon in the top right corner and select the drop-down menu.

![Image 13: Select the dropdown icon in the top right corner.](https://learn.microsoft.com/en-us/media/manage-account-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#step-2)
### Step 2

Select **Manage account** to change your role and preferences.

![Image 14: Select manage account.](https://learn.microsoft.com/en-us/media/manage-account-step-2.png)

Note

Name and Email cannot be updated.

[](https://learn.microsoft.com/en-us/clarity/manage-account#change-your-role)
Change your role
----------------

In the Account tab, select **Your role** from **Basics** to change/modify your role from the dropdown.

![Image 15: Select role.](https://learn.microsoft.com/en-us/media/change-your-role.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#download-android-app)
Download Android app
--------------------

You can now download the Clarity Mobile App for Android by scanning the QR code. The iOS version isn't currently supported.

![Image 16: Scan QR code to download android app.](https://learn.microsoft.com/en-us/media/download-android-app.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#change-your-preferences)
Change your preferences
-----------------------

[](https://learn.microsoft.com/en-us/clarity/manage-account#change-your-language)
### Change your language

Change the language by selecting **Language** from Preferences. By default, the language is set to **English(US)**.

![Image 17: Select language.](https://learn.microsoft.com/en-us/media/change-your-language.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#supported-languages)

### Change Country/Region

Select a Country/Region from the drop-down.

![Image 18: Change country/region.](https://learn.microsoft.com/en-us/media/change-country.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#change-date-format)
### Change date format

Select a date format from the drop-down.

![Image 19: Change date format.](https://learn.microsoft.com/en-us/media/change-date-format.png)

[](https://learn.microsoft.com/en-us/clarity/manage-account#select-start-of-the-week)
### Select start of the week

Choose the start of the week for your Clarity project data.
