Sign up
-------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#why-am-i-having-trouble-signing-up-to-clarity)
### Why am I having trouble signing up to Clarity?

Try the following ways to finish signing up to Clarity:

*   Verify the accuracy of entered email ID and password.

*   Look for typos and try the sign-up process again.

*   Consider clearing your browser cache, as it might be hindering the sign-up. Clear the browser cache and try again.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#installation)
Installation
------------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#why-am-i-unable-to-install-the-tracking-code)
### Why am I unable to install the tracking code?

Follow the troubleshooting steps:

*   Ensure that you adhere to the instructions outlined in [Setup](https://learn.microsoft.com/en-us/clarity/clarity-setup).

*   Directly copy the tracking code onto your page. Avoid using text editors before pasting it on your page.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#why-am-i-unable-to-post-any-requests-to-httpswwwclaritymscollect)
### Why am I unable to POST any requests to [https://www.clarity.ms/collect](https://www.clarity.ms/collect)?

Try the following troubleshooting steps:

*   Ensure proper installation of the tracking code by revisiting the [Setup](https://learn.microsoft.com/en-us/clarity/clarity-setup).

*   Refresh the page after a few minutes and follow the [Verification](https://learn.microsoft.com/en-us/clarity/clarity-setup#verify-your-installation) steps to locate and inspect the `collect` file.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#why-do-i-see-a-different-pages-data-in-my-project)
### Why do I see a different page's data in my project?

Ensure that you add the right tacking code for the associated project. Follow the steps to validate the tracking code:

1.   Go to **Projects** and open the relevant project.

2.   Select **Setup** ->**Installation methods** ->**Install manually**. Copy the Clarity code.

3.   Compare the script to the script on your website's `<head>` section. Make sure to check the `src`.

4.   If the script differs, remove the existing script, and paste the correct script on your page.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-installation#why-is-clarity-not-tracking-the-subdomains)
### Why is Clarity not tracking the subdomains?

If Clarity isn't tracking certain web pages, ensure that you installed the script on all the pages you wish to be monitored. For more information, see [Setup](https://learn.microsoft.com/en-us/clarity/clarity-setup).

Note

[Contact us](mailto:clarity-onboarding@microsoft.com) if the issue persists after following this troubleshooting guide.
