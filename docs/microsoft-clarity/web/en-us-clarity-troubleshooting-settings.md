Project management
------------------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#why-am-i-unable-to-add-a-project)
### Why am I unable to add a project?

You can add a project only if you sign up to Clarity. Check [Getting Started](https://learn.microsoft.com/en-us/clarity/getting-started) to know more.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#why-am-i-unable-to-access-my-project)
### Why am I unable to access my project?

Assuming you installed the Clarity tracking code correctly, follow these steps:

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#check-your-browser-cache)
#### Check your browser cache.

*   Check your browser cache that might be preventing you from signing in and accessing a project.

*   Open Clarity in a private window, clear cache, and try signing in again.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#check-your-project-permissions)
#### Check your project permissions.

Check with the Admin of the project if you have access to the project. Have your Admin follow these steps:

*   Open the project and navigate to **Settings** ->**Teams**.

*   Ensure your email ID is in the list of team members.

*   If you aren't part of the team, check how to [add a team member](https://learn.microsoft.com/en-us/clarity/team-management#add-team-members).

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#check-your-sign-in-authenticator)
#### Check your sign-in authenticator.

*   Ensure you're using the right authenticator to sign in. It should be the same one you used to sign up for Clarity. To know the supported authenticators, refer to Step 2 in ['How to sign up for Clarity?'](https://learn.microsoft.com/en-us/clarity/getting-started#sign-up-for-clarity).

*   Even with the right email, using a different authenticator doesn't allow you to sign in successfully.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#why-is-my-project-dashboard-empty)
### Why is my project dashboard empty?

Ensure that you added the correct tacking code for the relevant project. Try the following steps to troubleshoot:

*   Verify that you're signing in with the same authentication provider for your project.

*   Confirm that the project's tracking code is within the `<head>` section of your page.

*   If your page doesn't receive any visitors within the past few hours, you can't see any data on your dashboard. Try visiting your page and wait for 30 minutes.

*   Disable **Don't track** on your page. Refresh the page after a few minutes and follow the [Verification](https://learn.microsoft.com/en-us/clarity/clarity-setup#verify-your-installation) steps.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#project-access)
Project access
--------------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#how-do-i-gain-access-to-a-project-if-im-no-longer-a-member)
### How do I gain access to a project if I'm no longer a member?

Access to project details is restricted for security and privacy reasons until website ownership is verified.

Ownership verification involves:

*   If your emailID is listed as a member or an admin of the project.
*   Clarity tracking code is installed on your website.

If your emailID isn't registered or Clarity tracking code isn't installed, follow the instructions to verify your ownership:

1.   Add `<meta name="clarity-site-verification" content=" bed880bc-73d3-40c7-8c29-18f4eb483514" />` to the `<head>` section of your site that this project is setup for.
2.   [Contact us](mailto:clarityMS@microsoft.com) with project ID and authentication provider, once this change is deployed.
3.   If this matches, we can manually add you as the project admin.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#team-management)
Team management
---------------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#what-if-a-team-member-didnt-receive-the-invite)
### What if a team member didn't receive the invite?

Follow these steps:

*   Verify that you entered a valid email address in the invite.

*   If the recipient's email is of an official account, confirm that the organization's IT team allows email messages from **@microsoft.com**.

*   Instruct the team member to check spam or bulk email folders.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-settings#why-am-i-unable-to-remove-a-team-member)
### Why am I unable to remove a team member?

You can remove a team member from your project only if you're an Admin of the project. For more details, refer how to [remove team members](https://learn.microsoft.com/en-us/clarity/team-management#remove-team-members).

[Contact us](mailto:clarity-onboarding@microsoft.com) if the issue persists after following this troubleshooting guide.
