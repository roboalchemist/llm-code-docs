Install Clarity on WordPress platform using Clarity Plugin
----------------------------------------------------------

Note

These steps apply to WordPress sites and to WooCommerce sites on WordPress.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-1)
### Step 1

Sing in to your WordPress dashboard. Navigate to **Plugins**. Select **Add new plugin**.

![Image 14: Add new plugin.](https://learn.microsoft.com/en-us/clarity/media/wordpress-plugins.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-2)
### Step 2

Search for **Microsoft Clarity** and select **Install and Activate**.

![Image 15: Search for Clarity.](https://learn.microsoft.com/en-us/clarity/media/wordpress-search-1.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-3)
### Step 3

From the nav menu, open the Clarity plugin. Select **Sign in** to start connecting existing projects or create a new project.

![Image 16: Sign in to clarity.](https://learn.microsoft.com/en-us/clarity/media/wordpress-clarity-sign-in.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-4)
### Step 4

Choose a project from the list of existing projects and select **Continue**.

![Image 17: Select a project from the list.](https://learn.microsoft.com/en-us/clarity/media/select-project-wordpress.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#or)
#### Or

Select **New project** to create a new project and enter the details.

![Image 18: Add new project for wordpress.](https://learn.microsoft.com/en-us/clarity/media/wordpress-clarity-new-project1.png)

![Image 19: Enter new project details for wordpress.](https://learn.microsoft.com/en-us/clarity/media/wordpress-clarity-new-project2.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-5)
### Step 5

Once the project is successfully integrated, you can view Clarity's Dashboard, Recordings, Heatmaps, and Insights.

![Image 20: Select go to project.](https://learn.microsoft.com/en-us/clarity/media/wordpress-go-to-project.png)

Your WordPress site is now configured to track user behavior on your site.

[Verify your installation](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup#verify-your-installation)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#configure-content-insights-metrics-for-your-wordpress-blog)
Configure Content Insights Metrics for your WordPress Blog
----------------------------------------------------------

If your website is a blog, you can follow these steps to view the [Content Insights Metrics](https://learn.microsoft.com/en-us/clarity/insights/content-insights) through Clarity embedded experience inside WordPress.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-1-accessing-the-wordpress-theme-file-editor)
### Step 1: Accessing the WordPress Theme File Editor

Navigate to **Menu** ->**Appearance** and select **Theme File Editor**.

![Image 21: Select theme file editor from the left menu.](https://learn.microsoft.com/en-us/clarity/media/wordpress-content-insights-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-2-locating-the-appropriate-theme-file)
### Step 2: Locating the Appropriate Theme File

*   Identify the theme file responsible for displaying individual blog post content. The file name might vary based on your WordPress theme.
*   **Example**: For the theme **Twenty Twenty**, the file could be found under **Template-parts** with the name **content.php**.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-3-editing-the-theme-file-and-adding-data-attribute)
### Step 3: Editing the Theme File and Adding Data Attribute

*   In the **Theme File Editor**, locate and open the file corresponding to the blog post content.
*   Within the content area section of the file, find the opening tag of the `<article>` element.
*   Insert the data attribute `data-clarity-region="article"` into the article tag.
*   Review the modifications to ensure accuracy and select **Update File** to save the changes.

![Image 22: Select update file to save changes.](https://learn.microsoft.com/en-us/clarity/media/wordpress-content-insights-step-4.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wordpress#step-4-verifying-content-insights-integration)
### Step 4: Verifying Content Insights Integration

*   Visit one of your blog posts on the live website.
*   Right-click on the blog post content and select **Inspect** or **Inspect Element** from the context menu (depending on your browser).
*   In the developer tools, navigate to the HTML section and locate the `<article>` tag of your blog post content.
*   Verify that the data attribute `data-clarity-region="article"` is present in the `<article>` tag.

Note

Direct modification of theme files can be risky, especially if you are unfamiliar with coding and WordPress development. If you run into any problems, it's recommended to consult a developer or consider using a child theme to make modifications securely.
