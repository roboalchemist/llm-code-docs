# Getint Version 1.83

### Release date: June 04, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

We’ve rolled out key updates to enhance security, optimize integrations, and improve overall functionality across platforms and apps.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* **Stronger Security**: Updated third-party libraries to ensure enhanced protection and compliance.
* **Expanded Link Mapping**: Added support for one-to-many link mappings, providing greater flexibility in data connections.

#### **App Changes and Bug Fixes**

* **HubSpot**: Resolved an issue causing duplicated attachments, ensuring cleaner data handling.
* **Freshservice**: Improved ticket fetching with better pagination for more efficient case management.
* **Salesforce**: Boosted performance and introduced the ability to define custom search queries for tailored data retrieval.
