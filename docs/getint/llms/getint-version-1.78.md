# Getint Version 1.78

### Release date: March 27, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release brings a variety of new features, important performance improvements, and crucial bug fixes, all designed to provide a better and more reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Security enhancements for improved protection.
* The merge fields feature has been improved based on customer feedback.
* More options in the context menu on the integration details page.

#### **App Changes and Bug Fixes**

* **Azure**: Added a fix for syncing user fields in Azure DevOps Server.
* **ClickUp**: Fixed custom fields on ticket creation.
* **Git**: GitHub and GitLab no longer need separate GIT connector licenses.
* **HubSpot**: Pipeline names are now fetched correctly.
* **Salesforce**: Improved OAuth authentication + added support for date, datetime, and checkbox fields.
