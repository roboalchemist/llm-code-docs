# Getint Version 1.90

### Release date: September 30, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release boosts performance and reliability across the board, with faster bulk operations, improved log access, and key fixes for Airtable, Jira, ServiceNow, Shotgrid, and Zendesk integrations.

#### New Features and Improvements

* **Bulk operations**: Improved performance and stability for bulk import and unsync actions, enabling faster processing of large datasets.

#### **App Changes and Bug Fixes**

* **Log access**: Fixed issues with viewing logs for specific runs, making execution details more reliable and easier to track.
* **Airtable**: Fixed syncing of comments and improved reliability of status fields.
* **Jira**: Improved handling of Number fields, including support for null values.
* **ServiceNow**: Added support for business service and glide\_date\_time fields.
* **Shotgrid**: Fixed status mapping and improved overall sync behavior.
* **Zendesk**: Enhanced comment syncing to reduce duplication and improve consistency.
