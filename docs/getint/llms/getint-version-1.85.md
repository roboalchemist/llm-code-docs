# Getint Version 1.85

### Release date: July 07, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This update introduces behind-the-scenes improvements and useful enhancements to ensure your integrations run smoothly.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Introduced a sorting option for integrations by ID, allowing users to view them in chronological order.
* **Performance and Security Improvements**: Backend optimizations have been implemented to improve system responsiveness and reinforce data protection protocols.

#### **App Changes and Bug Fixes**

* **Salesforce**: The logic for fetching tickets has been enhanced to improve synchronization accuracy and reduce latency, particularly in environments with high data volumes.
