# Getint Version 1.89

### Release date: September 09, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release includes platform security upgrades, smarter status mapping, Jira OAuth for On-Premise, and key fixes for Jira, ServiceNow, and Shotgrid integrations.

#### New Features and Improvements

* **Auto-builder improvements**: Enhanced the auto-builder for status mapping, making it easier and faster to configure sync rules. This reduces manual setup and improves accuracy across integrations.
* **Jira OAuth for On-Premise**: Introduced OAuth support for Jira in Getint On-Premise, offering a modern and secure authentication method for self-hosted environments.
* **Security enhancements**: Resolved several UI vulnerabilities and strengthened overall platform security to ensure safer usage across all integrations and better protection of sensitive customer data.

#### **App Changes and Bug Fixes**

* **Jira**: Adjusted syncing of public comments and attachments, fixed issues with projects using the same key across two instances, and improved support for Number fields with “none” values.
* **ServiceNow**: Added support for glide\_date\_time fields.
* **Shotgrid**: Fixed status mapping and synchronization issues.
