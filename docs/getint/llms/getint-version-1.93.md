# Getint Version 1.93

### Release date: November 05, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release improves field handling, expands database compatibility, and introduces a faster way to recover failed syncs. App updates include more precise user data in GitHub, expanded field support in ServiceNow, and better mapping options in Airtable.

#### New Features and Improvements

* **Merge Fields Enhancements**: Improved the Merge Fields feature with bug fixes to ensure more reliable field concatenation across integrations.
* **On-Premise Support**: Getint now works with Oracle 19 and newer, allowing broader deployment options.
* **Sync Recovery from UI**: Syncs that failed during the CREATE phase can now be resynced directly from the UI, eliminating the need for manual intervention or support requests.

#### **App Changes and Bug Fixes**

* **GitHub**: Usernames are now fetched instead of user IDs. This change improves integration reliability and ensures clearer user identification across synced platforms.
* **ServiceNow**: Added `business_service` and `problem_id` fields for incident mapping.
* **Airtable**: The item URL is now exposed as a mappable field on the counterpart side. This allows users to reference Airtable records directly from the counterpart side.
