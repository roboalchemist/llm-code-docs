# Snowflake

We’ve partnered with Snowflake to <a href="https://venturebeat.com/ai/why-snowflake-is-backing-embedding-startup-voyage-ai-to-improve-enterprise-rag/" target="_blank">integrate</a> our embedding models as part of the <a href="https://docs.snowflake.com/en/release-notes/2024/other/2024-09-12-voyage-embed-model" target="_blank">Cortex AI</a> service within the <a href="https://www.snowflake.com/en/data-cloud/platform/" target="_blank">Snowflake AI Data Cloud</a>. The product integration is part of the broader Snowflake <a href="https://www.snowflake.com/en/blog/snowflake-invests-in-voyage-ai-optimizing-multilingual-rag/" target="_blank">investment</a> in Voyage AI.

`voyage-multilingual-2` is available in the text embedding function (<a href="https://docs.snowflake.com/en/sql-reference/functions/embed_text_1024-snowflake-cortex" target="_blank">EMBED\_TEXT\_1024</a>) of <a href="https://docs.snowflake.com/en/release-notes/2024/other/2024-09-12-voyage-embed-model" target="_blank">Snowflake Cortex AI</a>.
