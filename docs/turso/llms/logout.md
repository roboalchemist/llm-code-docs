# auth logout

To logout of the Turso CLI, run the following command:

```bash  theme={null}
turso auth logout
```

## Flags

| Flag        | Description                                   |
| ----------- | --------------------------------------------- |
| `-a, --all` | Invalidate all sessions for the current user. |
