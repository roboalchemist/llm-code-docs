# org members invite

To invite someone who isn't already a Turso user to the current active organization, run the following command:

```bash  theme={null}
turso org members invite <email> [flags]
```

## Flags

| Flag            | Description               |
| --------------- | ------------------------- |
| `-a`, `--admin` | Add the user as an admin. |
