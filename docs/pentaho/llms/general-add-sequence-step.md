# General

Enter the following information in the **Step name** and **Name of value** options:

| Option            | Description                                                                                                            |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Step name**     | Specify the unique name of the step on the canvas. **Step name** is set to Step sequence by default.                   |
| **Name of value** | Specify the name of the new sequence value to add to the PDI stream. **Name of value** is set to valuename by default. |
