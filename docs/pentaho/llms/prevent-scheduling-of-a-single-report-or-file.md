# Prevent Scheduling of a Single Report or File

In the Pentaho User Console, all new reports can be scheduled by default, but as an administrator, you can control which files can be scheduled to run.

1. From the User Console **Home** menu, click **Browse Files**.

   The Browse Files page appears.
2. Browse to the location of a report from the **Browse** pane on the left of the Browse Files page.

   The **Files** pane in the center populates with a list of reports.
3. Click to select the report for which you want to deny scheduling, then click **Properties** in the **Folder Actions** pane.

   The Properties window for that report appears.
4. Click the **General** tab in the Properties window, then unselect the **Allow Scheduling** check box.
5. Click **OK**.

The **Schedule** option next to this report is no longer displayed and the report cannot be scheduled.
