# Apply a summary function

You can assign a function that performs summary calculations on numeric values in columns, groups, or in the entire report. Perform the following steps to apply a summary function:

1. Click the down arrow next to a report column containing numeric values.
2. Select **Summary** from the menu, then choose the summary type. These types are described in the following table:

   | Summary Type   | Description                                                                                          |
   | -------------- | ---------------------------------------------------------------------------------------------------- |
   | None           | No summary function assigned.                                                                        |
   | Average        | Calculates the average value in a given column.                                                      |
   | Count          | Counts the items in a group or report, but does not require a numeric value.                         |
   | Count Distinct | Counts the distinct occurrences of a certain value in a column; does not require a numeric value.    |
   | Maximum        | Identifies the highest or largest value in a column.                                                 |
   | Minimum        | Identifies the lowest or smallest value in a column.                                                 |
   | Sum            | Calculates a total sum of the group or report (group level, and running total in the report footer). |
3. Save the report.
