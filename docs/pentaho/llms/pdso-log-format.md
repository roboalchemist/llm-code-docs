# Log format

The Data Optimizer log format is as follows:

`<date> <LOG_LEVEL> ldo.<component>:<line>: [p:<pid>][t:<thread>] <message>`

**Note:** The date format is `YYYY-mm-dd HH:MM:ss,ms`.
