# Import emails from data sources

You can import emails from different data sources and manage them in the User Console to share reports with recipients via an email server.

You can initially import email addresses from different data sources through the `settings.xml` file used by the scheduling plugin, then manage the email addresses in the User Console to share reports with recipients via an email server. If you want your JDBC or LDAP data sources to manage and maintain updates to the email addresses used for scheduling, see the **Importing and updating email addresses used for scheduling from data sources** section in the **Administer Pentaho Data Integration and Analytics** document for instructions.
