# Configure fields tab

![Configure fields tab](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-655dd5a619e1306c21da913dc74290dcacfad906%2FPDI_HbaseRowDecoder_DialogBox.png?alt=media)

This tab includes the following input fields to the step:

| Field                  | Description                                                           |
| ---------------------- | --------------------------------------------------------------------- |
| **Key field**          | Select the incoming PDI field containing the input key.               |
| **HBase result field** | Select the incoming PDI field containing the serialized HBase result. |
