# Options tab

![Bulk load into Azure SQL DB Options tab](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-d28f0e161d7fcb88638ac46d40c001cb645cd9b7%2FBulk%20load%20into%20Azure%20SQL%20DB%20Options%20tab.png?alt=media)

Enter the following information on the **Options** tab to define how the data is loaded into the destination table:

| Option             | Description                                                                                                                             |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| **Truncate table** | Select to remove all the data in the table before bulk loading the current data. Clear to append data during a new data load (default). |
