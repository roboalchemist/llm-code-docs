# Importing a data source

If your role allows it, you can import a data source into Data Optimizer.

Perform the following steps to import a data source:

1. On the **Menu**, click **Management**.

   The Manage Your Environment page opens.
2. On the **Data Sources** card, click **Import Data Source**.

   The Import Data Sources page appears with a list of discovered data sources from Data Catalog.
3. Select the data source(s) that you want to import.
4. Click **Import**.

   A message confirming that the data source was mounted appears at the top of the page.

The data source is mounted in Data Optimizer.
