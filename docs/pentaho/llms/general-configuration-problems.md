# General configuration problems

The issues in this section explain how to resolve common configuration problems.

When updating to Pentaho 9.0, you are required to perform a one-time operation to update your cluster configurations to use multiple cluster features. For more information, see [Set up Pentaho to connect to a Hadoop cluster](https://docs.pentaho.com/install/9.3-install/pentaho-configuration/tasks-to-be-performed-by-an-it-administrator/set-up-the-pentaho-server-to-connect-to-a-hadoop-cluster). If you do not update your cluster configurations, then the legacy configuration from the Big Data plugin is used.
