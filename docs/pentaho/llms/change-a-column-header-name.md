# Change a column header name

You can customize your column headers to make them more descriptive or simple.

![Change Column Header Names](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-4716676d165b677af6dcd7890256b05c26e6c185%2FCustomize_PIR_Change_Column_Header_Names.png?alt=media)

Perform the following steps to change a column header name:

1. Double-click the column header.
2. Type the new header name in the dialog box and press Enter.
