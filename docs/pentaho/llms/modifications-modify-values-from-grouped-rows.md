# Modifications

| Field                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Hierarchical path** | Specifies the path that has to be modified or created inside the target hierarchical field. See [Hierarchical data path specifications](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/hierarchical-data/hdt-path-specification)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Action**            | <p>Specify the action to perform on the <strong>Target field name</strong>. The modification actions you can perform with this step are:- <strong>SET</strong> – Sets the <strong>Target hierarchical path</strong> in the <strong>Target field name</strong> to the value specified in the <strong>Value field name</strong> field.</p><ul><li><strong>SET\_IGNORE\_NULL\_VALUE</strong> - Performs the <strong>SET Modify</strong> action if the value field is not null.</li><li><strong>SET\_IF\_NULL</strong> - Sets the <strong>Target hierarchical path</strong> to the value specified in the <strong>Value field name</strong> field if the target hierarchical path of the HDT is null.</li><li><strong>UNSET</strong> - Unsets the <strong>Target hierarchical path</strong> in the Target HDT <code>(before: { hello: null, hello2: 10 }, after: {} )</code></li><li><strong>UNSET\_IF\_NULL</strong> - Unsets the <strong>Target hierarchical path</strong> if the value stored at the path is null <code>(before: { hello: null, hello2: 10 }, after: { hello2: 10} )</code></li></ul> |
| **Value field name**  | Specifies the field for performing the previous action.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
