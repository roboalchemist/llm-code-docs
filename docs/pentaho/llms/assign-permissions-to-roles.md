# Assign Permissions to Roles

Perform the following steps to assign permissions to roles in the Pentaho User Console.

1. Click the **Manage Roles** tab.
2. Highlight a role in the **Roles** list.
3. Select the applicable check boxes in the **Operation Permissions** list.

You have assigned permissions to the highlighted role, and users associated with that role now have those permissions.
