# Validating installation of Pentaho Data Mastering

Validate that the installation of Pentaho Data Mastering was successful by logging into the Pentaho Data Mastering application.

Complete the following steps to log into the Pentaho Data Mastering application and validate the installation:

1. In a browser, navigate to the Pentaho Data Mastering application by using the FQDN of the Pentaho Data Mastering application server. For example: `http://*Application FQDN*`.

   The Pentaho Data Mastering login page appears.
2. Enter a valid **Username** and **Password** and click **Sign In**.

   The Pentaho Data Mastering application landing page opens. For example, the following image shows the Pentaho Data Mastering application landing page that you see after you log into the application:
