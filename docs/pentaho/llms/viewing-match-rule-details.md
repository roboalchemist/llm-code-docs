# Viewing match rule details

View match rule details to review how you configured Pentaho Data Mastering to identify matches in the data from different data sources.

Perform the following steps to view match rule details:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Match Rules** card, click **Match Rules**.

   The Match Rules page opens with the list of match rules shown in a table.
3. At the end of the row for the match rule that you want to view, click the more actions icon and select **View Details**.
