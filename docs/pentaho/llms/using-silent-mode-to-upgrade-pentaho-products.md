# Using silent mode to upgrade Pentaho products

You can use silent mode to upgrade an archive or manual installation of Pentaho products in an environment that does not support the display of a graphical interface.

Silent mode installation is a non-graphical method for running the Pentaho Upgrade Installer. In silent mode, no prompts are issued throughout the upgrade process.

For instructions on using silent mode to upgrade Pentaho products, see the procedure for the type of installation that you have.

* [Use silent mode to upgrade archive installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer-in-silent-mode-cp/using-silent-mode-to-upgrade-pentaho-products/use-silent-mode-to-upgrade-archive-installation-pentaho-upgrade-installer-silent-mode)To upgrade an archive installation without using a graphical interface, use the Pentaho Upgrade Installer in silent mode from a command line.
* [Use silent mode to upgrade manual installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer-in-silent-mode-cp/using-silent-mode-to-upgrade-pentaho-products/use-silent-mode-to-upgrade-manual-installation-pentaho-upgrade-installer-silent-mode)To upgrade a manual installation without using a graphical interface, use silent mode to install upgrade files into an empty folder, copy those upgrade files into your existing installation, and then update any customizations that you made to your installation.
