# Adding match rules in bulk

You can add multiple match rules at one time to control how Pentaho Data Mastering identifies matches in the data from different sources.

To create multiple match rules at one time, import a JSON file that contains all the information for the match rules.

Perform the following steps to create multiple match rules at one time:

1. On the left navigation menu, click **Management**. The Manage Your Environment page opens.
2. In the **Match Rules** card, click Match Rules.

   The Match Rules page opens.
3. Click **Add Match Rule** > **Import Match Rule(s)**.

   The Import Match Rule(s) dialog box opens.
4. Browse to and select the JSON file that contains the match rules information to import.

   **Tip:** You can drag and drop the JSON file into the Import Match Rule(s) dialog box.
5. Click **Continue**.

   A confirmation message is shown in the top-right corner of the page and the match rule records are added to the match rule table.
