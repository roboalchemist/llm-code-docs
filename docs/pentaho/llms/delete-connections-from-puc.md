# Delete connections from PUC

Perform the following steps to delete a connection in PUC:

**CAUTION:**

Deleting a connection affects all reports, charts, dashboards, and other content that are associated with the connection.

1. Click **File** > **Manage Data Sources**.

   The Manage Data Sources dialog box appears.
2. Highlight the name of the data source you want to delete and select **Delete** in the **More actions and options** menu.

   The data source no longer appears in the list of data sources.
