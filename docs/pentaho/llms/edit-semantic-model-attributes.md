# Edit semantic model attributes

You can edit the default role, description, and caption for a semantic model.&#x20;

Complete the following steps to edit model attributes in a semantic model:&#x20;

1. Log into the **Pentaho User Console** (PUC). &#x20;
2. Open the Semantic Model Editor by taking one of the following actions:

   * If you are using the Modern Design of PUC, in the menu on the left side of the page, click **Semantic Model Editor**.
   * If you are using the Classic Design of PUC, click **File** > **Semantic Model Editor**.&#x20;

   The **Semantic Model Editor** opens.&#x20;
3. Open the semantic model that you want to edit by completing the following sub steps:&#x20;
   1. In the **Semantic Models** list, navigate to the model you want to open by searching or scrolling through the list.&#x20;
   2. Click **Open**. The model opens in the canvas.&#x20;
4. Click the **More Actions** icon and select **Edit Model Attributes**. The **Edit Model Attributes** window opens.&#x20;
5. Edit one or more of the following options: &#x20;

   <table><thead><tr><th width="162.6666259765625">Option </th><th>Description </th></tr></thead><tbody><tr><td>Default Role </td><td>The name of the default role for connections to this schema. </td></tr><tr><td>Description </td><td>Description of the semantic model to help users understand the purpose or contents of the semantic model. </td></tr><tr><td>Measures Caption </td><td>String of text that is displayed instead of the measure’s name. You can use captions to provide a user-friendly label for measures or for localization so that the measure's name appears in the local language. </td></tr></tbody></table>
6. Click **Apply**. The model attributes are applied.&#x20;
7. In the **Semantic Model Editor**, click **Save** to save changes to the model.&#x20;
