# Validating installation of the OpenObserve tool

Validate that the installation of the OpenObserve APM tool components was successful by logging into the OpenObserve application.

Complete the following steps to log into the OpenObserve application and validate the installation:

1. In a browser, navigate to the Pentaho Data Mastering application by using the FQDN of the Pentaho Data Mastering application server. For example: `http://*Application FQDN*`.

   The Pentaho Data Mastering login page appears.
2. Enter a valid **Username** and **Password** and click **Sign In**.

   The Pentaho Data Mastering application landing page opens.
3. Click **Open Observability**.

   The OpenObserve application login page appears.
4. Enter a valid **Username** and **Password** and click **Sign In**.

   The OpenObserve application landing page opens. The following image shows the OpenObserve application:
