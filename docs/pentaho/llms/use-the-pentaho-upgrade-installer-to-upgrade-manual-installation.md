# Use the Pentaho Upgrade Installer to upgrade manual installation

To upgrade a manual installation of Pentaho products, use the Pentaho Upgrade Installer to install upgrade files into an empty folder, and then copy the upgrade files into your manual installation.

**Note:** If you are upgrading in an environment that does not support the display of a graphical interface, then you might want to upgrade in silent mode from a command line. For more information, see [Using the Pentaho Upgrade Installer in console or silent mode](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer-in-silent-mode-cp).

You must activate your new licenses as described in the **Administer Pentaho Data Integration and Analytics** document before restoring a repository using the import-export utility.

To upgrade a manual installation using the {Pentaho Upgrade Installer, complete the following tasks in order:

1. [Install upgrade files into empty folder](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-manual-installation/install-upgrade-files-into-empty-folder)
2. [Update files in manual installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-manual-installation/update-files-in-manual-installation)
