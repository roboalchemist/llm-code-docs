# Log lines in Kettle

To differentiate log lines from a mapping, edit the `kettle.properties` file and set the **KETTLE\_LOG\_MARK\_MAPPINGS** variable to `Y`.

Set this variable to `Y` to precede log lines with the mapping step name and the mapping itself.
