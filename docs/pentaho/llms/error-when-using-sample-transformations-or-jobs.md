# Error when using sample transformations or jobs

If you are running a shipped sample transformation file (`.ktr`) or sample job file (`.kjb`), you may experience one or more errors. These errors are caused by using the JAR files from previous versions of Pentaho.

To fix this problem, you must use the JAR files included in the latest release with the shipped sample transformation and job files.
