# Row font

| Option                   | Description                              |
| ------------------------ | ---------------------------------------- |
| **Row font name**        | Specify the font for rows.               |
| **Row font size**        | Specify the font size.                   |
| **Row font color**       | Specify the font color.                  |
| **Row background color** | Specify a background color for the rows. |
