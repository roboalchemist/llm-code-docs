# Bulk load into Azure SQL DB

You can execute a job to bulk load into Azure SQL DB.

Use the **Bulk load into Azure SQL DB** job entry to load data into an Azure SQL database from Azure Data Lake Storage. The **Bulk load into Azure SQL DB** only supports CSV format data. See [Azure SQL documentation](https://docs.microsoft.com/en-us/azure/azure-sql/).
