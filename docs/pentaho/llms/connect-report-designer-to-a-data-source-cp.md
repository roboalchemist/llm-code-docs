# Connect to a data source

This article explains each data source connection used with Report Designer. You must have a report file open in order to proceed, and your data source must be accessible before you can connect to it in Report Designer. If you are using Report Designer on Linux, after you add a data source it will not appear in the **Data** tab until you double-click the **Data Sets** item.

For instructions on connecting to different types of data sources, see the following topics:

* [JDBC](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/connect-report-designer-to-a-data-source-cp/jdbc-connecting-to-a-data-source-in-prd)

* [OLAP (advanced)](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/connect-report-designer-to-a-data-source-cp/olap-advanced-connecting-to-a-data-source-in-prd)
