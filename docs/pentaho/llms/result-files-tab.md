# Result files tab

![Result files tab](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-145270ee5fda36e7c07fef0bbdbe0dd4fd976d88%2FJobExecutor_step_ResultFiles_tab.png?alt=media)

In this tab, specify the destination of the result files.

| Option                                       | Description                                                                       |
| -------------------------------------------- | --------------------------------------------------------------------------------- |
| **Target step for result files information** | Use the drop-down menu to select a step in the transformation as the target step. |
| **Result file name field**                   | Specify the name of the field for the result files.                               |
