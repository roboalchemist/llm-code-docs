# ORC types

The ORC to PDI data type values are shown in the table below:

| ORC Type  | PDI Type  |
| --------- | --------- |
| String    | String    |
| TimeStamp | TimeStamp |
| Binary    | Binary    |
| Decimal   | BigNumber |
| Boolean   | Boolean   |
| Date      | Date      |
| Integer   | Integer   |
| Double    | Number    |
