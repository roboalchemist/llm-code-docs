# Print a dashboard

You can print a dashboard or save it in PDF format from Dashboard Designer.

1. From User Console **Home**, click **Create New**, then **Dashboard**.
2. Navigate to the dashboard that you want to print or create a new dashboard.
3. Click on the print icon in the tool bar.

   ![Print dashboard icon](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-c29e5c9fef8219a7f58878fabdaeebc4f85b4256%2FPrint%20dashboard%20option.png?alt=media)
4. In the Print Preview Settings window, select paper size and page orientation.

   ![Print Preview Settings window](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-5e3dc9ac37534cebb7ef795f980c8ad18a65b7e0%2FPrint%20dashboard%20preview%20settings.png?alt=media)

   **Note:** You can make changes to the reports in the dashboard after it opens in the new tab.

   ![Print dashboard preview with sample graphs](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-49879919f99640713b45ed3a9e8d602bf8abd8f5%2FPrint%20dashboard%20preview%20sample%20graphs.png?alt=media)
5. After making the needed selections, click the **Print** button to open the browser's print dialog.
6. Select the same settings in the browser’s print dialog as you made in the Print Preview Settings window.
7. Use the browser’s print dialog to either save the report as PDF or print the dashboard.

   ![Print dashboard browser's print dialog box](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-c9eaa26796b57631778d1833a0b79f63d7847a6d%2FPrint%20dashboard%20preview%20sample%20print%20dialog.png?alt=media)

   The dashboard is printed.
