# Use the Pentaho Upgrade Installer to upgrade archive installation

To upgrade an archive installation, use the Pentaho Upgrade Installer to guide you through the upgrade process step-by-step.

**Note:** If you are upgrading in an environment without a graphical interface, you can upgrade in silent mode from a command line. For more information, see [Using the Pentaho Upgrade Installer in console or silent mode](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer-in-silent-mode-cp).

**Note:** You must activate your new licenses as described in the **Administer Pentaho Data Integration and Analytics** document before restoring a repository using the import-export utility.

To upgrade an archive installation using the Pentaho Upgrade Installer, complete the following tasks in order:

1. [Get started by checking your environment](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-archive-installation/get-started-by-checking-your-environment-pentaho-upgrade)
2. [Specify customized items to address after upgrading](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-archive-installation/specify-items-to-include-or-exclude-pentaho-upgrade)
3. [Back up your existing Pentaho products and install Pentaho 10.2](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-archive-installation/back-up-your-pentaho-products-and-install-pentaho-9.1-pentaho-upgrade)
