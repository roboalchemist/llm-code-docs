# Adding merge rules in bulk

You can add multiple merge rules at one time to control how matched data is merged to create Master Records. To create multiple merge rules at one time, import a JSON file that contains all the information for the merge rules.

Perform the following steps to create multiple merge rules at one time:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Merge Rules** card, click **Merge Rules**.

   The Merge Rules page opens.
3. Click **Add Merge Rule** > **Import Merge Rule(s)**.

   The Import Merge Rule(s) dialog box opens.
4. Browse to and select the JSON file that contains the merge rules information to import.

   **Tip:** You can drag and drop the JSON file into the Import Merge Rule(s) dialog box.
5. Click **Continue**.

   A confirmation message is shown in the top-right corner of the page and the merge rules are added to the merge rule table.
