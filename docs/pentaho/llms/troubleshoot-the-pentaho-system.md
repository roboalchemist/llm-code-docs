# Troubleshoot the Pentaho system

Use these common categories to find fixes:

{% hint style="info" %}
If you installed Pentaho Server on your own Tomcat, `pentaho.log` is under `pentaho-server/tomcat/logs`.
{% endhint %}

You can monitor transformations, jobs, and database connections through [PDI logging](https://docs.pentaho.com/pdia-admin/administer/legacy-optimization-pages/pdi-logging). For the Pentaho Server, review `pentaho.log` in `pentaho-server/logs`.

Use logging first when you troubleshoot Pentaho components. Logging helps you validate behavior and narrow root causes fast.
