# Use a different report template

You can select a different template for your report using the **General**tab.

![Report Template General tab](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-d05a3afc0a4247401b538d39ab240b605ddc4eca%2FPIR%20Report%20Template%20General%20tab.png?alt=media)

Perform the following steps to select a different template:

1. Click the **General**tab, then use the left and right arrows to scroll through available templates.
2. Chose a template by double-clicking on it.
3. Click **Yes** or **No** when prompted whether to keep that formatting display.

The report display is updated to reflect the new template.
