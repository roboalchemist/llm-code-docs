# kedro.pipeline

## kedro.pipeline

`kedro.pipeline` provides functionality to define and execute data-driven pipelines.

| Name                                                                                                              | Type   | Description                                         |
| ----------------------------------------------------------------------------------------------------------------- | ------ | --------------------------------------------------- |
| [`kedro.pipeline.Node`](https://docs.kedro.org/en/stable/api/pipeline/kedro.pipeline.node/index.md)               | Class  | A class to define a node in a Kedro pipeline.       |
| [`kedro.pipeline.Pipeline`](https://docs.kedro.org/en/stable/api/pipeline/kedro.pipeline.Pipeline/index.md)       | Class  | Represents a Kedro pipeline.                        |
| [`kedro.pipeline.llm_context`](https://docs.kedro.org/en/stable/api/pipeline/kedro.pipeline.llm_context/index.md) | Module | Provides functionality to define LLM context nodes. |
