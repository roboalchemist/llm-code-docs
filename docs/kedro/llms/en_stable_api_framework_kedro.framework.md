## kedro.framework

`kedro.framework` provides Kedro's framework components

| Module                                                                                                       | Description                                                              |
| ------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------ |
| [`kedro.framework.cli`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.cli/index.md)         | Implements commands available from Kedro's CLI.                          |
| [`kedro.framework.context`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.context/index.md) | Provides functionality for loading Kedro project context.                |
| [`kedro.framework.hooks`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.hooks/index.md)     | Provides primitives to use hooks to extend KedroContext's behaviour.     |
| [`kedro.framework.project`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.project/index.md) | Provides utilities to configure a Kedro project and access its settings. |
| [`kedro.framework.session`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.session/index.md) | Provides access to `KedroSession` responsible for project lifecycle.     |
| [`kedro.framework.startup`](https://docs.kedro.org/en/stable/api/framework/kedro.framework.startup/index.md) | Provides metadata for a Kedro project.                                   |
