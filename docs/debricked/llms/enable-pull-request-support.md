# Enable Pull Request support

For OpenText Core SCA to be able to generate Pull Requests on your behalf, you should do some configuration based on the platform:

* GitHub: To enable pull requests, you are required to [integrate with GitHub](https://docs.debricked.com/tools-and-integrations/integrations/github) using the GitHub app.
* GitLab: if you have set up an[ integration to GitLab](https://docs.debricked.com/tools-and-integrations/integrations/gitlab), to enable merge requests, you should provide additional credentials (as explained in the article).
* Azure DevOps: if you have set up the [Azure DevOps integration](https://docs.debricked.com/tools-and-integrations/integrations/azure-devops), to enable pull requests, you must provide additional credentials (as explained in the article).<br>
