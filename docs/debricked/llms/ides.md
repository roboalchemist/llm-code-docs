# Integrated Development Environments (IDEs)

### OpenText Core SCA visual studio extension

{% hint style="info" %}
Note that this is a community-contributed project not supported by OpenText Core SCA.
{% endhint %}

[Here](https://github.com/Debricked-Community/DebrickedVSE) you can find a community-contributed extension for Visual Studio. To learn how to set it up and use follow [the guidelines in the GitHub project](https://github.com/Debricked-Community/DebrickedVSE?tab=readme-ov-file).

### OpenText Core SCA visual studio code extension (Alpha)

{% hint style="warning" %}
OpenText Core SCA is developing an extension for Visual Studio Code, currently in alpha. Head over to <https://github.com/debricked/VS-Code-extension> if you want to try it out.
{% endhint %}
