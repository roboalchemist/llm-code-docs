Pricing and Billing
Overview

Understanding Desearch's pricing and billing system is essential for effectively managing your API usage and optimizing costs. This guide covers everything you need to know about payment methods, billing cycles, transaction tracking, and pricing tiers for all Desearch API services.

Billing

Managing your payments and balance is simple through the Billing menu in your Desearch Console. The billing system is designed to be transparent, flexible, and developer-friendly, giving you complete control over your API spending.

Payment Methods

You can add funds to your Desearch API account through two convenient payment methods:

Card Payment

Securely add a credit or debit card to your account for instant top-ups. We support all major card networks including Visa, Mastercard, and American Express. Card payments are processed securely through our payment provider with industry-standard encryption.

Cryptocurrency Payments

For users who prefer decentralized payment options, we currently support TAO (the native token of the Bittensor network). Cryptocurrency payments offer:

Lower transaction fees for larger amounts
Direct integration with the Bittensor ecosystem
No currency conversion for TAO holders
Auto-Recharge

For seamless, uninterrupted API access, you can enable auto-recharge by linking a card to your account. This powerful feature allows:

Automatic top-ups: When your balance reaches a customizable threshold, funds are automatically added
Customizable amounts: Set how much to recharge each time
Threshold configuration: Define at what balance level the recharge should trigger
Email notifications: Receive alerts when auto-recharge occurs

This ensures your applications never experience downtime due to insufficient balance.

Transaction History and Cost Tracking

The Billing menu provides comprehensive financial visibility:

Detailed transaction history: View all processed payments, including date, amount, and payment method
API usage breakdown: See costs for each API request made using your API key
Daily and monthly summaries: Track spending patterns over time
Export capabilities: Download transaction data for accounting and analysis

Pricing

Desearch offers flexible, usage-based pricing designed to scale with your needs. Our pricing is based on the AI-Model search capabilities provided through Subnet 22 (Desearch) on the Bittensor network.

Pricing Tiers

Our pricing structure includes services across different capability levels:

Basic Search Services
Basic Web Search: Standard web search queries with fast response times
Basic X Search: Simple Twitter/X content search
Web Links Search: URL-specific content retrieval
AI-Powered Search Services
AI Search: Advanced semantic search with AI understanding
X Posts Search: Deep Twitter/X analysis with AI processing
Deep Research: Comprehensive multi-source research queries
Additional Services
Date-filtered searches: Historical data access
High-volume endpoints: Optimized for large-scale applications

📘 Pricing Rate

The pricing table is subject to change based on individual agreements and usage patterns. Please keep a tap on Pricing for latest updates

Cost Optimization Tips

To optimize your API costs:

Use caching: Cache frequently requested data to reduce API calls
Batch requests: Combine multiple queries when possible
Choose appropriate endpoints: Use basic search when AI features aren't needed
Monitor usage: Regularly review your usage patterns in the console
Set budget alerts: Configure notifications for spending thresholds
Enterprise and Custom Pricing

For high-volume users or enterprises with specific requirements, we offer custom pricing arrangements:

Volume discounts for large-scale usage
Dedicated support and SLA guarantees
Custom rate limits and priority access
Invoiced billing options

Contact our sales team to discuss custom pricing for your organization.
