prose-h2:font-medium prose-h3:mt-8 prose-h3:mb-4 prose-h3:text-xl prose-pre:bg-gray-100"><h1 id="custom-base-images-for-github-actions">Custom Base Images for GitHub Actions<a href="#custom-base-images-for-github-actions" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h1>
<p>Accelerate your workflows by pre-installing dependencies directly into your runner&#x27;s base image.
Instead of installing packages at runtime during every job, custom base images let you create optimized runners with your tools already available, eliminating repetitive setup steps and reducing job execution time.</p>
<h2 id="preinstalled-packages">Preinstalled packages<a href="#preinstalled-packages" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h2>
<div class="docs-steps ml-4 mb-12 border-l border-gray-200 pl-6 [counter-reset:step]"><h4 id="enable-custom-base-image">Enable custom base image<a href="#enable-custom-base-image" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>Go to the desired <a href="https://cloud.namespace.so/workspace/actions/profiles">Runner Profile</a> and select a customizable base image.</p><h4 id="select-apt-packages">Select APT packages<a href="#select-apt-packages" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>Type the Ubuntu packages to install.</p><figure class="relative flex flex-col items-center justify-center"><img alt="Select APT packages" loading="lazy" width="600" height="271" decoding="async" data-nimg="1" class="rounded-md shadow-lg border border-gray-100 overflow-hidden center" style="color:transparent" srcSet="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackages.0255795a.png&amp;w=640&amp;q=75 1x, /_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackages.0255795a.png&amp;w=1200&amp;q=75 2x" src="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackages.0255795a.png&amp;w=1200&amp;q=75"/></figure><p>You can find the list of available packages from the <a href="https://packages.ubuntu.com/jammy/allpackages?format=txt.gz">Ubuntu website</a>.</p><h4 id="submit-the-profile">Submit the profile<a href="#submit-the-profile" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>After updating your profile definition the custom base image will be built.
Once the image is ready to use, you will see a confirmation in the profile editor.</p><figure class="relative flex flex-col items-center justify-center"><img alt="APT packages ready" loading="lazy" width="600" height="288" decoding="async" data-nimg="1" class="rounded-md shadow-lg border border-gray-100 overflow-hidden center" style="color:transparent" srcSet="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackagesready.e5df5a17.png&amp;w=640&amp;q=75 1x, /_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackagesready.e5df5a17.png&amp;w=1200&amp;q=75 2x" src="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Faptpackagesready.e5df5a17.png&amp;w=1200&amp;q=75"/></figure></div>
<p>You can modify the list of pre-installed packages at any time. Any update to the selection will automatically update the base image.</p>
<h2 id="custom-dockerfile-base">Custom Dockerfile base<a href="#custom-dockerfile-base" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h2>
<p>Using a custom Dockerfile, you can add any dependencies on top of the Namespace base image.
Dockerfile syntax is fully supported, allowing customization beyond pre-installing APT packages.</p>
<p>Important to know:</p>
<ul>
<li>The final layer <strong>must</strong> be based on <var>NAMESPACE_BASE_IMAGE_REF</var>.
The base image contains the software required to integrate with GitHub, without it your workflows won&#x27;t start.</li>
<li>The user in the final image <strong>must</strong> be <var>runner</var>.
GitHub&#x27;s runner software expects to run as this user.</li>
<li>After an image is built, it needs to be distributed and optimized.
Workflows using this profile may take longer to start than normal while this is still in progress.</li>
</ul>
<p>To get started with a Dockerfile-based custom image:</p>
<div class="docs-steps ml-4 mb-12 border-l border-gray-200 pl-6 [counter-reset:step]"><h4 id="enable-custom-base-image-1">Enable custom base image<a href="#enable-custom-base-image-1" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>Go to the desired <a href="https://cloud.namespace.so/workspace/actions/profiles">Runner Profile</a> and select a customizable base image.</p><h4 id="add-customization-steps">Add customization steps<a href="#add-customization-steps" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>Add your custom Dockerfile steps to the input. Make sure to build on NAMESPACE_BASE_IMAGE_REF in your final layer:</p><figure data-rehype-pretty-code-figure=""><pre tabindex="0" data-language="shell" data-theme="github-light"><code data-language="shell" data-theme="github-light" style="display:grid"><span data-line=""><span style="color:#6F42C1">ARG</span><span style="color:#032F62"> NAMESPACE_BASE_IMAGE_REF=&quot;&quot;</span></span>
<span data-line=""><span style="color:#6F42C1">FROM</span><span style="color:#24292E"> ${NAMESPACE_BASE_IMAGE_REF} </span><span style="color:#032F62">AS</span><span style="color:#032F62"> base</span></span></code></pre></figure><figure class="relative flex flex-col items-center justify-center"><img alt="Custom Dockerfile" loading="lazy" width="600" height="479" decoding="async" data-nimg="1" class="rounded-md shadow-lg border border-gray-100 overflow-hidden center" style="color:transparent" srcSet="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image.38f2b805.png&amp;w=640&amp;q=75 1x, /_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image.38f2b805.png&amp;w=1200&amp;q=75 2x" src="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image.38f2b805.png&amp;w=1200&amp;q=75"/></figure><h4 id="submit-the-profile-1">Submit the profile<a href="#submit-the-profile-1" class="autolink" aria-hidden="true" tabindex="-1"><span class="subheading-anchor"></span></a></h4><p>After updating your profile definition the custom base image will be built.
Once the image is ready to use, you will see a confirmation in the profile editor.</p><figure class="relative flex flex-col items-center justify-center"><img alt="Custom Dockerfile ready" loading="lazy" width="600" height="474" decoding="async" data-nimg="1" class="rounded-md shadow-lg border border-gray-100 overflow-hidden center" style="color:transparent" srcSet="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image-ready.0a7167ab.png&amp;w=640&amp;q=75 1x, /_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image-ready.0a7167ab.png&amp;w=1200&amp;q=75 2x" src="/_next/image?url=%2F_next%2Fstatic%2Fmedia%2Fdockerfile-based-custom-image-ready.0a7167ab.png&amp;w=1200&amp;q=75"/></figure></div>
