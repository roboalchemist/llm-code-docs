# Networking

Our core networking provides secure, isolated access to your compute instances at no performance compromise.

## Ingress

Namespace allows instances to expose [ingresses](/docs/architecture/networking/ingress) publicly, to allow secure access of your applications without manual pain points.
Ingresses can be exposed on custom domains with automatic TLS management and encrypted ingress access.

## Performance

Learn how Namespace transparently optimizes your [network performance](/docs/architecture/networking/performance) through intelligent caching, direct peering, and dynamic traffic shaping.

## Security

Namespace networking is designed with [security](/docs/architecture/networking/security) at the core. Learn about our multifaceted approach to keep your data secure and isolated.
