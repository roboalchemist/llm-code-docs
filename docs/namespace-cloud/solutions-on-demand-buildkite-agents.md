# On-Demand Buildkite Agents

Namespace is deeply integrated with Buildkite allowing you to run jobs quickly
that scale with you without requiring any infrastructure maintenance.

Namespace is available natively in Buildkite.
Buildkite Hosted Agents combine the best of Buildkite and Namespace. Head over to [buildkite.com](https://buildkite.com) and create a Hosted Agent queue to get started.

Buildkite Hosted Agents are a fully managed platform offering best performance without the burden to manage your own agents.

[Learn more →](https://buildkite.com/docs/pipelines/hosted-agents/overview)
