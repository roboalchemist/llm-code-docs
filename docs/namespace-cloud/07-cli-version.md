# nsc version

Display the current version of the Namespace CLI tool.

## Basic Usage

```bash
nsc version
```

## Output Example

Running this command produces information including:

- The version number (e.g., v0.0.484)
- The commit hash
- The commit date with timestamp
- The system architecture (e.g., linux/amd64)

## Options

**--build_info**: Output comprehensive build information beyond the standard version details.

## Purpose

This utility helps developers verify which version of the `nsc` CLI is currently installed on their system, making it useful for troubleshooting compatibility issues or confirming that updates have been applied.
