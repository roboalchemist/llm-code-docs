# nsc describe

Describe an instance.

`describe` displays detailed information about a Namespace instance.

## Usage

```
nsc describe <instance-id>
```

### Example

```
$ nsc describe h9am86n6gi25m
```

## Options

### --json

Print output in JSON format.
