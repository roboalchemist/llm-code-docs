# Authentication

> API Auth ID and Token

The **Authentication** page provides the **Auth ID** and **Auth Token** needed for secure API access.

  <img src="https://mintcdn.com/plivo/R0bB3pSVhMhsNJEx/aiagent/images/auth.png?fit=max&auto=format&n=R0bB3pSVhMhsNJEx&q=85&s=03f4761a340c9bdeb93c25933175f6b8" width="1530" height="674" data-path="aiagent/images/auth.png" />
</Frame>

These credentials are used in two key scenarios:

1. **Verify Product**: When integrating and using the Verify product, the Auth ID and Token are required to authenticate API requests.
2. **Triggering an AI Agent**: When making an API call to trigger an AI Agent, these credentials are also needed to ensure secure communication.

You can find your **Auth ID** and **Auth Token** under the authentication settings, and they will be automatically applied when you select the option to **Require Authentication** in the configuration of your API request trigger.
