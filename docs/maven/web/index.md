Welcome to Apache Maven
=======================

Apache Maven is a build tool for Java projects. Using a project object model (POM), Maven manages a project's compilation, testing, and documentation.

You can find more information in the "About Maven" section of the sidebar. This includes an in-depth description of [what Maven is](https://maven.apache.org/what-is-maven.html) and a list of some of its main features.

This site is separated into the following sections:

| Use | [Download](https://maven.apache.org/download.html), [Install](https://maven.apache.org/install.html), [Configure](https://maven.apache.org/configure.html), [Run](https://maven.apache.org/run.html) Maven | [Maven Plugins](https://maven.apache.org/plugins/index.html) and [Maven Extensions](https://maven.apache.org/extensions/index.html) |
| --- | --- | --- |
| Information for those needing to build a project that uses Maven | Lists of plugins and extensions to help with your builds. |
| **Maven 4**: Learn [what's new](https://maven.apache.org/whatsnewinmaven4.html) and [how to migrate a project](https://maven.apache.org/guides/mini/guide-migration-to-mvn4.html) to Maven 4. The [Maven Upgrade Tool](https://maven.apache.org/tools/mvnup.html) provides upgrade support. | Upcoming: How to migrate plugins to the new Maven 4 API? |
| Extend | [Write Maven Plugins](https://maven.apache.org/plugin-developers/index.html) | [Improve the Maven Central Repository](https://maven.apache.org/repository/index.html) |
| Information for developers writing Maven plugins. | Information for those who may or may not use Maven, but are interested in getting project metadata into the [central repository](https://maven.apache.org/repository/index.html). |
| Contribute | [Help Maven](https://maven.apache.org/guides/development/guide-helping.html) | [Develop Maven](https://maven.apache.org/developers/index.html) |
| Information if you'd like to get involved. Maven is an open source community and welcomes contributions. | Information for those who are currently Maven developers, or who are interested in contributing to the Maven project itself. |

Each guide is divided into a number of trails to get you started on a particular topic, and includes a reference area and a "cookbook" of common examples.

You can access the guides at any time from the left navigation. If you are looking for a quick reference, you can use the [documentation index](https://maven.apache.org/guides/index.html).

### How to Get Support

Support for Maven is available in a variety of different forms.

To get started, search the documentation, [issue management system](https://maven.apache.org/issue-management.html), the [wiki](https://cwiki.apache.org/confluence/display/MAVEN), or the [mailing list archives](https://maven.apache.org/mailing-lists.html) to see if the problem has been solved or reported before.

If the problem has not been reported before, the recommended way to get help is to subscribe to the [Maven Users Mailing list](https://maven.apache.org/mailing-lists.html). Many other users and Maven developers will answer your questions there, and the answer will be archived for others in the future.

You can also reach the Maven developers on [Slack](https://maven.apache.org/community.html).
