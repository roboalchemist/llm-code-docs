The following table lists skins that are available for you to use in your Maven generated site.

**Note: You always have to define a skin, if you are using a [site descriptor](https://maven.apache.org/plugins/maven-site-plugin/examples/sitedescriptor.html). There is no default skin applied automatically.**

To apply one of these skins to your project's site, you use the `skin` element of the [site descriptor](https://maven.apache.org/plugins/maven-site-plugin/examples/sitedescriptor.html). This is a regular artifact or dependency-like element. For example, to use the Maven Fluido Skin, you would include this in your `site.xml` file:

1. `<project>`
2. `...`
3. `<skin>`
4. `<groupId>org.apache.maven.skins</groupId>`
5. `<artifactId>maven-fluido-skin</artifactId>`
6. `<version>2.1.0</version>`
7. `</skin>`
8. `...`
9. `</project>`
