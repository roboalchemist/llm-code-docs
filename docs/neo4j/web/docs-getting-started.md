Get started with Neo4j
======================

[](https://neo4j.com/docs/getting-started/#_explore_the_capabilities_of_graph_databases)Explore the capabilities of graph databases
-----------------------------------------------------------------------------------------------------------------------------------

With a property graph database at its core, Neo4j offers an ecosystem of tools, applications, and libraries which aim to help you get started with the technology.

[### What is Neo4j ![Image 2: icon neo4j](https://neo4j.com/docs/getting-started/_images/icon-neo4j.svg) Learn what a graph database is and how to use it in your projects. [What is Neo4j?](https://neo4j.com/docs/getting-started/whats-neo4j/)](https://neo4j.com/docs/getting-started/whats-neo4j/)

[### What is a graph database ![Image 3: icon graph](https://neo4j.com/docs/getting-started/_images/icon-graph.svg) Learn the principles of a graph database. [What is a graph database](https://neo4j.com/docs/getting-started/graph-database/)](https://neo4j.com/docs/getting-started/graph-database/)

[### What is Cypher ![Image 4: icon developer](https://neo4j.com/docs/getting-started/_images/icon-developer.svg) Learn Cypher®, Neo4j’s graph query language, and start thinking about graphs and patterns. [What is Cypher](https://neo4j.com/docs/getting-started/cypher/)](https://neo4j.com/docs/getting-started/cypher/)

[](https://neo4j.com/docs/getting-started/#_work_with_data)Work with data
-------------------------------------------------------------------------

[### Model your data for Neo4j ![Image 5: icon csvtodb](https://neo4j.com/docs/getting-started/_images/icon-csvtodb.svg) Discover strategies to model your data and to improve your workflow. [/docs/model](https://neo4j.com/docs/model)](https://neo4j.com/docs/model)

[### Import data to Neo4j ![Image 6: icon import](https://neo4j.com/docs/getting-started/_images/icon-import.svg) Read more about the different ways of importing data to your Neo4j graph database. [/docs/import/](https://neo4j.com/docs/import/)](https://neo4j.com/docs/import/)

[### Create applications with Neo4j ![Image 7: icon code](https://neo4j.com/docs/getting-started/_images/icon-code.svg) Run through detailed examples of how to integrate Neo4j with your preferred programming language. [/docs/create-applications](https://neo4j.com/docs/create-applications)](https://neo4j.com/docs/create-applications)

[### Data science with Neo4j ![Image 8: icon gds](https://neo4j.com/docs/getting-started/_images/icon-gds.svg) Learn how to use Neo4j Graph Data Science, a library of graph algorithms for analysts and data scientists. [/docs/gds](https://neo4j.com/docs/gds)](https://neo4j.com/docs/gds)

[### Visualize data in Neo4j ![Image 9: icon dataviz](https://neo4j.com/docs/getting-started/_images/icon-dataviz.svg) Learn how to export your graph data in Neo4j for display as a visualization. [/docs/visualize/](https://neo4j.com/docs/visualize/)](https://neo4j.com/docs/visualize/)
