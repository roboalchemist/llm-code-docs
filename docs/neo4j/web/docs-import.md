Import your data
================

[](https://neo4j.com/docs/import/#_how_to_use_your_own_data_on_a_graph_database)How to use your own data on a graph database
----------------------------------------------------------------------------------------------------------------------------

Learn how to import various types of data into Neo4j. From JSON to APIs to another database, retrieve data from nearly any source, and use it to populate your graph.

[### `LOAD CSV` ![Image 2: icon import](https://neo4j.com/docs/_images/icon-import.svg) Use Cypher® for importing bulk data into Neo4j. Suited for non-admin users. [https://neo4j.com/docs/cypher-manual/current/clauses/load-csv/](https://neo4j.com/docs/cypher-manual/current/clauses/load-csv/)](https://neo4j.com/docs/cypher-manual/current/clauses/load-csv/)

[### From relational into graph ![Image 3: icon csvtodb](https://neo4j.com/docs/_images/icon-csvtodb.svg) Learn how to translate and migrate tabular data and relational datasets into a graph database. [https://neo4j.com/docs/getting-started/data-import/relational-to-graph-import/](https://neo4j.com/docs/getting-started/data-import/relational-to-graph-import/)](https://neo4j.com/docs/getting-started/data-import/relational-to-graph-import/)

[### Neo4j Connectors ![Image 4: icon connectors](https://neo4j.com/docs/_images/icon-connectors.svg) Learn how to use connectors to connect data from various sources into Neo4j. [Connect data sources](https://neo4j.com/docs/connectors/)](https://neo4j.com/docs/connectors/)

[### Neo4j Data Importer ![Image 5: icon neo4j](https://neo4j.com/docs/_images/icon-neo4j.svg) Start using Neo4j’s tool for importing data and get started quickly with testing and prototyping. [https://neo4j.com/docs/data-importer/current/](https://neo4j.com/docs/data-importer/current/)](https://neo4j.com/docs/data-importer/current/)

[### Neo4j Admin Import ![Image 6: icon admin](https://neo4j.com/docs/_images/icon-admin.svg) Use Neo4j admin import for initial loading of large amounts of data. [https://neo4j.com/docs/operations-manual/current/tools/neo4j-admin/neo4j-admin-import/#import-tool-full](https://neo4j.com/docs/operations-manual/current/tools/neo4j-admin/neo4j-admin-import/#import-tool-full)](https://neo4j.com/docs/operations-manual/current/tools/neo4j-admin/neo4j-admin-import/#import-tool-full)

[### APOC ![Image 7: icon library](https://neo4j.com/docs/_images/icon-library.svg) Learn how to import data from various data formats, including JSON, XML, and XLS using the APOC library. [**Import data using APOC Core**](https://neo4j.com/docs/apoc/current/import/)](https://neo4j.com/docs/apoc/current/import/)

[](https://neo4j.com/docs/import/#_keep_exploring)Keep exploring
----------------------------------------------------------------

Here are some recommended resources start importing your data:

1.   [**GraphAcademy: Importing CSV data into Neo4j**](https://graphacademy.neo4j.com/courses/importing-cypher/)

Learn how to import CSV data into Neo4j using Cypher. 
2.   [**Import unstructured data**](https://neo4j.com/developer/genai-ecosystem/importing-graph-from-unstructured-data/)

Learn how to create a knowledge graph from unstructured data. 
3.   [**Importing JSON data from a REST API into Neo4j**](https://neo4j.com/docs/getting-started/data-import/json-rest-api-import/)

See techniques for loading data from JSON-based REST APIs into Neo4j.
