# Loading Screen

## Create

<figure><img src="https://3352617162-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F90tV7pYflEQdiAr2VfWu%2Fuploads%2Fm4A8HpTtzOzF91gaHnE9%2Fimage.png?alt=media&#x26;token=df55e62d-038f-4b0d-8707-c2fe68b543a8" alt=""><figcaption></figcaption></figure>

With our editor, you can create your unique loading screen. This screen will appear to the user after your website starts loading (right after the launch screen)&#x20;

You can control this screen with SDK or Plugins

## Upload custom image

* The size should be 2048x2048, and the image should have a PNG format.
* Check out the [template](https://www.figma.com/file/Vvxp9bVA6PjrxJ0MziPmwA/Natively-Icon-%26-Logo-Template?node-id=2571%3A21358) below for more details 👇🏻
