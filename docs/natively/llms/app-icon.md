## Upload custom

* The size should be 1024x1024, and the image should have a PNG format.
* Without transparent background. (It is very important because Apple doesn't allow icons with a transparent background)
* Check out the [template](https://www.figma.com/file/Vvxp9bVA6PjrxJ0MziPmwA/Natively-Icon-%26-Logo-Template?node-id=2571%3A21350) below for more details 👇🏻
