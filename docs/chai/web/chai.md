Chai Assertion Library
----------------------

The Chai API is broken down by style or task.

##### Assertion Styles

*   The [Expect / Should](https://devdocs.io/chai/api/bdd/index) API covers the BDD assertion styles.
*   The [Assert](https://devdocs.io/chai/api/assert/index) API covers the TDD assertion style.

##### Plugins

*   The [Plugin API](https://devdocs.io/chai/api/plugins/index) will be of use to anyone interested in building plugins as helpers to DRY up your tests, or for release to the community.

##### Tools

*   The [Online Test Suite](https://www.chaijs.com/api/test/) is a live run of Chai’s test suite. Use it to ensure browser compatibility.
