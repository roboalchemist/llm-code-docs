# wandb job list

List jobs in a project

## Usage

```bash  theme={null}
wandb job list [OPTIONS]
```

## Options

| Option            | Description                                     |
| :---------------- | :---------------------------------------------- |
| `--project`, `-p` | The project you want to list jobs from.         |
| `--entity`, `-e`  | The entity the jobs belong to (default: models) |
