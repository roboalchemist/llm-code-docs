Note

The Active Template Library (ATL) continues to be supported. However, we're no longer adding features or updating the documentation.

[](https://learn.microsoft.com/en-us/cpp/atl/reference/atl-global-variables?view=msvc-170#_patlmodule)
_pAtlModule
-----------

A global variable storing a pointer to the current module.

 C++

Copy

```cpp
__declspec(selectany) CAtlModule * _pAtlModule
```

[](https://learn.microsoft.com/en-us/cpp/atl/reference/atl-global-variables?view=msvc-170#remarks)

### Remarks

Methods on this global variable can be used to provide the functionality that the (now obsolete) class CComModule provided in Visual C++ 6.0.

[](https://learn.microsoft.com/en-us/cpp/atl/reference/atl-global-variables?view=msvc-170#example)

### Example

 C++

Copy

```cpp
LONG lLocks = _pAtlModule->GetLockCount();
```

[](https://learn.microsoft.com/en-us/cpp/atl/reference/atl-global-variables?view=msvc-170#requirements)

### Requirements

**Header:** atlbase.h
