# Overview

> Create buyable items for your members to spend tokens on!

### Creating a Store Item

To create a Store Item, use the `/create` command and choose `📦 Store Item`.

<Frame caption="Use the /create command">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_e3b6f6.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=8cf68daf6e0d9217253150064e658fde" width="568" height="154" data-path="images/drip-features/img_e3b6f6.png" />
</Frame>

After typing the command and clicking enter, the following embed will be sent to the channel

<Frame caption="Edit store embed">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_85bb17.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=efbaa8de8a03d040037c439d61a64e27" width="992" height="660" data-path="images/drip-features/img_85bb17.png" />
</Frame>

Choose the option you would like to edit by clicking one of the buttons or selecting from the dropdown menu
