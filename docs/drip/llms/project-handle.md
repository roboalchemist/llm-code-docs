# Project Handle

> Set the project handle(s) for quests

After selecting `🪪 Project Handle` from the dropdown menu in [Quest Settings](/admin-settings/admin-dashboard/admin-settings/quest-settings), a new embed will appear:

<Frame caption="Project Handle Menu">
  <img src="https://mintcdn.com/driprewards/9R8wvri2DbuZkKko/images/admin-settings/admin-dashboard/admin-settings/quest-settings/img_2d6a7b.jpg?fit=max&auto=format&n=9R8wvri2DbuZkKko&q=85&s=018bd1133652746dc18aecc0b31bedc2" width="988" height="468" data-path="images/admin-settings/admin-dashboard/admin-settings/quest-settings/img_2d6a7b.jpg" />
</Frame>

To add a project handle, click on Twitter from the dropdown menu:

<Frame caption="Click on Twitter from the dropdown menu">
  <img src="https://mintcdn.com/driprewards/9R8wvri2DbuZkKko/images/admin-settings/admin-dashboard/admin-settings/quest-settings/img_3d5cc2.png?fit=max&auto=format&n=9R8wvri2DbuZkKko&q=85&s=20021edbb02b70384a7e12f5cb3a3447" width="988" height="474" data-path="images/admin-settings/admin-dashboard/admin-settings/quest-settings/img_3d5cc2.png" />
</Frame>

A form will appear for you to enter the project's Twitter handle
