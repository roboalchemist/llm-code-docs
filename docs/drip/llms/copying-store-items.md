# Copying Store Items

> You can copy a store item to send in a different channel

To copy an already existing Store Item, right-click the store item you want to duplicate and copy the message ID.

<Frame caption="Click on Copy Message ID">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/store/img_7688b0.jpg?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=a50816590cb18d3cd5b3befc59999f1c" width="1118" height="960" data-path="images/drip-features/store/img_7688b0.jpg" />
</Frame>

<Info>
  If you don't see the option to copy the message ID, enable Developer Mode
  under Settings -> Advanced -> Developer Mode
</Info>

Now use the `/create type: 📦 Store Item` command. Click the `💾Copy Existing` button and paste the ID from the message you copied earlier

<Frame caption="Click on Copy Exisiting">
  <img src="https://docs.drip.re/~gitbook/image?url=https%3A%2F%2F2728597434-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FxUaE8LDXTeS85PkKxJJa%252Fuploads%252FsFkx98W1flIxPFOM0CX7%252F%2523%25F0%259F%258F%25AA%25E2%2594%2586store%2520Drip%2520Documentation%2520-%2520Discord%25202024-01-26%2520at%252011.26.51%2520AM.jpg%3Falt%3Dmedia%26token%3Dbad65aac-47c3-46ff-9eb9-e1869d20077c&width=768&dpr=4&quality=100&sign=7478b9fd&sv=1" />
</Frame>

Now you will see that the entire configuration of the Store item is copied to that embed
