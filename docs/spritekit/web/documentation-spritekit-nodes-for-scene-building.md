Nodes for Scene Building
========================

Define the appearance or layout of scene content.

[Topics](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#topics)
---------------------------------------------------------------------------------------------

### [Base Nodes](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#Base-Nodes)

Provide a reference, point of view, or foundation to all things displayed in a scene.

[Using Base Nodes to Lay Out Sprite Kit Content](https://developer.apple.com/documentation/spritekit/using-base-nodes-to-lay-out-spritekit-content)

Use nonvisual nodes to define the layout of a scene.

[`class SKNode`](https://developer.apple.com/documentation/spritekit/sknode)

The base class of all SpriteKit nodes.

[`class SKCameraNode`](https://developer.apple.com/documentation/spritekit/skcameranode)

A node that determines which parts of the scene are visible within a view.

[`class SKReferenceNode`](https://developer.apple.com/documentation/spritekit/skreferencenode)

A node that’s defined in an archived `.sks` file.

### [Nodes that Draw](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#Nodes-that-Draw)

Display images, shapes, particles, text, video, tiles, or even 3D content.

[Maximizing Node Drawing Performance](https://developer.apple.com/documentation/spritekit/maximizing-node-drawing-performance)

Structure your nodes for maximum performance.

[`class SKSpriteNode`](https://developer.apple.com/documentation/spritekit/skspritenode)

An image or solid color.

[`class SKShapeNode`](https://developer.apple.com/documentation/spritekit/skshapenode)

A mathematical shape that can be stroked or filled.

[`class SKEmitterNode`](https://developer.apple.com/documentation/spritekit/skemitternode)

A source of various particle effects.

[`class SKLabelNode`](https://developer.apple.com/documentation/spritekit/sklabelnode)

A graphical element that draws text.

[`class SKVideoNode`](https://developer.apple.com/documentation/spritekit/skvideonode)

A graphical element that plays video content.

[`class SKTileMapNode`](https://developer.apple.com/documentation/spritekit/sktilemapnode)

A two-dimensional array of images.

[`class SK3DNode`](https://developer.apple.com/documentation/spritekit/sk3dnode)

3D SceneKit content drawn as a flattened sprite.

### [Nodes for Environmental Effects](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#Nodes-for-Environmental-Effects)

Provide environmental effects to your scene, such as audio, lighting, or areas with specific physics characteristics.

[`class SKAudioNode`](https://developer.apple.com/documentation/spritekit/skaudionode)

A node that plays audio.

[`class SKLightNode`](https://developer.apple.com/documentation/spritekit/sklightnode)

A node that lights surrounding nodes.

[`class SKFieldNode`](https://developer.apple.com/documentation/spritekit/skfieldnode)

A node that applies physics effects to nearby nodes.

### [Nodes that Modify Drawing](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#Nodes-that-Modify-Drawing)

Modify the rendering of child nodes by cropping, applying Core Image filters, or performing 3D transformations.

[`class SKEffectNode`](https://developer.apple.com/documentation/spritekit/skeffectnode)

A node that renders its children into a separate buffer, optionally applying an effect, before drawing the final result.

[`class SKCropNode`](https://developer.apple.com/documentation/spritekit/skcropnode)

A node that masks pixels drawn by its children so that only some pixels are seen.

[`class SKTransformNode`](https://developer.apple.com/documentation/spritekit/sktransformnode)

A node that allows its children to rotate in 3D.

[See Also](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#see-also)
-------------------------------------------------------------------------------------------------

### [Essentials](https://developer.apple.com/documentation/spritekit/nodes-for-scene-building#Essentials)

[Drawing Sprite Kit Content in a View](https://developer.apple.com/documentation/spritekit/drawing-spritekit-content-in-a-view)

Display visual content using SpriteKit.

[`class SKScene`](https://developer.apple.com/documentation/spritekit/skscene)

An object that organizes all of the active SpriteKit content.
