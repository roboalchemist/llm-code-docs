**Azure DevOps Services | Azure DevOps Server | Azure DevOps Server 2022**

Set user preferences on the user profile page in Azure DevOps. You can change the user picture, display name, preferred email address, and UI theme. These settings only apply to Azure DevOps.

Update your picture, display name, email address, language, date and time format, time zone, and other interface preferences on your Azure DevOps user profile page.

[](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops#set-preferences)
Set preferences
---------------

*   [Current page](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops#tabpanel_1_current-page)
*   [Microsoft Entra profile preview](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops#tabpanel_1_preview-page)

To set your preferences, complete the following steps:

1.   From the home page, select ![Image 11](https://learn.microsoft.com/en-us/azure/devops/media/icons/user-settings-gear.png?view=azure-devops)**User settings**>**Profile**.

![Image 12: Screenshot shows buttons to get to Azure DevOps profile.](https://learn.microsoft.com/en-us/azure/devops/media/open-user-settings-profile-preview.png?view=azure-devops)

2.   From the **Profile** page, change the profile picture, display name, contact email address, and region. Select **Save**.

![Image 13: Screenshot to choose and edit the Profile page.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/edit-about-page-preview.png?view=azure-devops)

1.   Open the user profile menu, and then select **My profile**.

![Image 14: Screenshot of select the user profile menu, and then My profile.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/user-preferences/open-profile-menu-2020.png?view=azure-devops)

2.   From the **General** tab, change the following information:

    *   Profile picture
    *   Display name
    *   Preferred email address
    *   Whether borders appear for fields on work item forms.

![Image 15: Screenshot of User Profile dialog, General tab.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/user-preferences/user-profile-dialog-general-tab.png?view=azure-devops)

3.   From the **Locale** tab, change the preferred language, date and time pattern, and time zone.

![Image 16: Screenshot of User Profile dialog, Locale tab.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/user-preferences/user-profile-dialog-locale-tab.png?view=azure-devops)

4.   To change the UI theme, go back to the profile menu and select **Theme**. Choose between **Dark** and **Light**.

5.   Select **Save**.

The system updates your user profile settings.

[](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops)

We're previewing the option to use your profile picture, display name, contact email address, and geography from Microsoft Entra in Azure DevOps.

To enable this feature, go to the Azure DevOps home page, select ![Image 17](https://learn.microsoft.com/en-us/azure/devops/media/icons/user-settings-gear.png?view=azure-devops)**User settings**, and then choose **Profile**.

![Image 18: Screenshot shows Microsoft Entra profile information in the Profile page.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/entra-profile-preview.png?view=azure-devops)

If you experience problems with Microsoft Entra profile information, you can [turn it off](https://learn.microsoft.com/en-us/azure/devops/project/navigation/preview-features?view=azure-devops) and restore your original profile details. Provide detailed feedback if you do; the product team reviews all feedback to address concerns.

![Image 19: Screenshot shows window for entering feature improvements and feedback for Microsoft Entra profile information.](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/media/turn-off-entra-profile-information.png?view=azure-devops)

To receive Azure DevOps updates and resources, check the box and select **Save**.

Your user profile settings are updated.

FAQs
----

See the following frequently asked questions and answers.

[](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops#q-how-can-i-change-my-profile-picture-in-microsoft-entra)
#### Q: How can I change my profile picture in Microsoft Entra?

A:

1.   Sign in to your [Microsoft 365 account](https://www.microsoft365.com/).
2.   Select the camera icon or **Change Photo**, next to your current profile picture.
3.   Upload a new photo from your device and adjust it as needed.
4.   **Save** your changes.

[](https://learn.microsoft.com/en-us/azure/devops/organizations/settings/set-your-preferences?view=azure-devops#q-why-did-i-stop-getting-email-notifications-from-azure-devops)
#### Q: Why did I stop getting email notifications from Azure DevOps?

A: Azure DevOps sends notification emails to the email address configured for your account in Microsoft Entra. If you aren't getting notifications, make sure you configured an email address in Microsoft Entra.

The New Teams user interface isn't supported for Azure DevOps Server 2020 and earlier versions.
