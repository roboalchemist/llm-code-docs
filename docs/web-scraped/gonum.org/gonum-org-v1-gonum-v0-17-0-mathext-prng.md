### Overview [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/mathext/prng#pkg-overview "Go to Overview")

Package prng provides random source PRNG implementations.

PRNG implementations provided in package prng may be used directly as rand.Source values for the golang.org/x/exp/rand package, and for the math rand package via a wrapper type.

The xoshiro PRNGs are described in [http://vigna.di.unimi.it/ftp/papers/ScrambledLinear.pdf](http://vigna.di.unimi.it/ftp/papers/ScrambledLinear.pdf) and [http://prng.di.unimi.it/](http://prng.di.unimi.it/). A Xoshiro256starstar value is only valid if returned by NewXoshiro256starstar.
