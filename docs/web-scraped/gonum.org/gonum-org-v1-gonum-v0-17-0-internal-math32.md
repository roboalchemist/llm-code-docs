func Abs(x [float32](https://pkg.go.dev/builtin#float32)) [float32](https://pkg.go.dev/builtin#float32)

Abs returns the absolute value of x.

Special cases are:

Abs(±Inf) = +Inf
Abs(NaN) = NaN

#### func [Copysign](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L42)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#Copysign "Go to Copysign")

func Copysign(x, y [float32](https://pkg.go.dev/builtin#float32)) [float32](https://pkg.go.dev/builtin#float32)

Copysign returns a value with the magnitude of x and the sign of y.

#### func [Hypot](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L56)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#Hypot "Go to Hypot")

func Hypot(p, q [float32](https://pkg.go.dev/builtin#float32)) [float32](https://pkg.go.dev/builtin#float32)

Hypot returns Sqrt(p*p + q*q), taking care to avoid unnecessary overflow and underflow.

Special cases are:

Hypot(±Inf, q) = +Inf
Hypot(p, ±Inf) = +Inf
Hypot(NaN, q) = NaN
Hypot(p, NaN) = NaN

#### func [Inf](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L81)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#Inf "Go to Inf")

func Inf(sign [int](https://pkg.go.dev/builtin#int)) [float32](https://pkg.go.dev/builtin#float32)

Inf returns positive infinity if sign >= 0, negative infinity if sign < 0.

#### func [IsInf](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L95)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#IsInf "Go to IsInf")

func IsInf(f [float32](https://pkg.go.dev/builtin#float32), sign [int](https://pkg.go.dev/builtin#int)) [bool](https://pkg.go.dev/builtin#bool)

IsInf reports whether f is an infinity, according to sign. If sign > 0, IsInf reports whether f is positive infinity. If sign < 0, IsInf reports whether f is negative infinity. If sign == 0, IsInf reports whether f is either infinity.

#### func [IsNaN](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L104)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#IsNaN "Go to IsNaN")

func IsNaN(f [float32](https://pkg.go.dev/builtin#float32)) (is [bool](https://pkg.go.dev/builtin#bool))

IsNaN reports whether f is an IEEE 754 “not-a-number” value.

#### func [Max](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L120)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#Max "Go to Max")added in v0.11.0

func Max(x, y [float32](https://pkg.go.dev/builtin#float32)) [float32](https://pkg.go.dev/builtin#float32)

Max returns the larger of x or y.

Special cases are:

Max(x, +Inf) = Max(+Inf, x) = +Inf
Max(x, NaN) = Max(NaN, x) = NaN
Max(+0, ±0) = Max(±0, +0) = +0
Max(-0, -0) = -0

#### func [Min](https://github.com/gonum/gonum/blob/v0.17.0/internal/math32/math.go#L146)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/internal/math32#Min "Go to Min")added in v0.11.0

func Min(x, y [float32](https://pkg.go.dev/builtin#float32)) [float32](https://pkg.go.dev/builtin#float32)

Min returns the smaller of x or y.

Special cases are:

Min(x, -Inf) = Min(-Inf, x) = -Inf
Min(x, NaN) = Min(NaN, x) = NaN
Min(-0, ±0) = Min(±0, -0) = -0
