func Zacai(ZR, ZI, FNU [float64](https://pkg.go.dev/builtin#float64), KODE, MR, N [int](https://pkg.go.dev/builtin#int), YR, YI [][float64](https://pkg.go.dev/builtin#float64), RL, TOL, ELIM, ALIM [float64](https://pkg.go.dev/builtin#float64)) (
	ZRout, ZIout, FNUout [float64](https://pkg.go.dev/builtin#float64), KODEout, MRout, Nout [int](https://pkg.go.dev/builtin#int), YRout, YIout [][float64](https://pkg.go.dev/builtin#float64), NZ [int](https://pkg.go.dev/builtin#int), RLout, TOLout, ELIMout, ALIMout [float64](https://pkg.go.dev/builtin#float64))

ZACAI APPLIES THE ANALYTIC CONTINUATION FORMULA

K(FNU,ZN*EXP(MP))=K(FNU,ZN)*EXP(-MP*FNU) - MP*I(FNU,ZN)
      MP=PI*MR*CMPLX(0.0,1.0)

TO CONTINUE THE K FUNCTION FROM THE RIGHT HALF TO THE LEFT HALF Z PLANE FOR USE WITH ZAIRY WHERE FNU=1/3 OR 2/3 AND N=1. ZACAI IS THE SAME AS ZACON WITH THE PARTS FOR LARGER ORDERS AND RECURRENCE REMOVED. A RECURSIVE CALL TO ZACON CAN RESULT if ZACON IS CALLED FROM ZAIRY.

#### func [Zmlri](https://github.com/gonum/gonum/blob/v0.17.0/mathext/internal/amos/amos.go#L1721)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/mathext/internal/amos#Zmlri "Go to Zmlri")

#### func [Zs1s2](https://github.com/gonum/gonum/blob/v0.17.0/mathext/internal/amos/amos.go#L2112)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/mathext/internal/amos#Zs1s2 "Go to Zs1s2")

func Zs1s2(zr, s1, s2 [complex128](https://pkg.go.dev/builtin#complex128), scale, lim [float64](https://pkg.go.dev/builtin#float64), iuf [int](https://pkg.go.dev/builtin#int)) (s1o, s2o [complex128](https://pkg.go.dev/builtin#complex128), nz, iufo [int](https://pkg.go.dev/builtin#int))

Zs1s2 tests for a possible underflow resulting from the addition of the I and K functions in the analytic continuation formula where s1 == K function and s2 == I function.

When kode == 1, the I and K functions are different orders of magnitude.

When kode == 2, they may both be of the same order of magnitude, but the maximum must be at least one precision above the underflow limit.

#### func [Zseri](https://github.com/gonum/gonum/blob/v0.17.0/mathext/internal/amos/amos.go#L1940)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/mathext/internal/amos#Zseri "Go to Zseri")

func Zseri(z [complex128](https://pkg.go.dev/builtin#complex128), fnu [float64](https://pkg.go.dev/builtin#float64), kode, n [int](https://pkg.go.dev/builtin#int), y [][complex128](https://pkg.go.dev/builtin#complex128), tol, elim, alim [float64](https://pkg.go.dev/builtin#float64)) (nz [int](https://pkg.go.dev/builtin#int))

Zseri computes the I bessel function for real(z) >= 0 by means of the power series for large |z| in the region |z| <= 2*sqrt(fnu+1).

nz = 0 is a normal return. nz > 0 means that the last nz components were set to zero due to underflow. nz < 0 means that underflow occurred, but the condition |z| <= 2*sqrt(fnu+1) was violated and the computation must be completed in another routine with n -= abs(nz).

#### func [Zuchk](https://github.com/gonum/gonum/blob/v0.17.0/mathext/internal/amos/amos.go#L1376)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/mathext/internal/amos#Zuchk "Go to Zuchk")

func Zuchk(y [complex128](https://pkg.go.dev/builtin#complex128), scale, tol [float64](https://pkg.go.dev/builtin#float64)) [int](https://pkg.go.dev/builtin#int)

Zuchk tests whether the magnitude of the real or imaginary part would underflow when y is scaled by tol.

y enters as a scaled quantity whose magnitude is greater than

1e3 + 3*dmach(1)/tol

y is accepted if the underflow is at least one precision below the magnitude of the largest component. Otherwise an underflow is assumed as the phase angle does not have sufficient accuracy.
