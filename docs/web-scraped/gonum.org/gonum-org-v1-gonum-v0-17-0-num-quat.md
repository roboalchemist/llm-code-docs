### Overview [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/num/quat#pkg-overview "Go to Overview")

Package quat provides the quaternion numeric type and functions.

For a good treatment of uses and behaviors of quaternions, see the interactive videos by Ben Eater and Grant Sanderson here [https://eater.net/quaternions](https://eater.net/quaternions).

Example (Rotate) [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/num/quat#example-package-Rotate "Go to Example (Rotate)")

Rotate a cube 120° around the diagonal vector [1, 1, 1].

Output:

 0 {x:0 y:0 z:0} -> {x:0 y:0 z:0} 1 {x:0 y:0 z:1} -> {x:1 y:0 z:0} 2 {x:0 y:1 z:0} -> {x:0 y:0 z:1} 3 {x:0 y:1 z:1} -> {x:1 y:0 z:1} 4 {x:1 y:0 z:0} -> {x:0 y:1 z:0} 5 {x:1 y:0 z:1} -> {x:1 y:1 z:0} 6 {x:1 y:1 z:0} -> {x:0 y:1 z:1} 7 {x:1 y:1 z:1} -> {x:1 y:1 z:1}
