#### func [IntersectionOfNodes](https://github.com/gonum/gonum/blob/v0.17.0/graph/internal/set/set.go#L163)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#IntersectionOfNodes "Go to IntersectionOfNodes")

func IntersectionOfNodes(a, b [Nodes](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#Nodes)) [Nodes](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#Nodes)

IntersectionOfNodes returns the intersection of a and b.

The intersection of two sets, a and b, is the set containing all the elements shared between the two sets, for instance:

{a,b,c} INTERSECT {b,c,d} = {b,c}

The intersection between a set and itself is itself, and thus effectively a copy operation:

{a,b,c} INTERSECT {a,b,c} = {a,b,c}

The intersection between two sets that share no elements is the empty set:

{a,b,c} INTERSECT {d,e,f} = {}

#### func [UnionOfNodes](https://github.com/gonum/gonum/blob/v0.17.0/graph/internal/set/set.go#L131)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#UnionOfNodes "Go to UnionOfNodes")

func UnionOfNodes(a, b [Nodes](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#Nodes)) [Nodes](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/graph/internal/set#Nodes)

UnionOfNodes returns the union of a and b.

The union of two sets, a and b, is the set containing all the elements of each, for instance:

{a,b,c} UNION {d,e,f} = {a,b,c,d,e,f}

Since sets may not have repetition, unions of two sets that overlap do not contain repeat elements, that is:

{a,b,c} UNION {b,c,d} = {a,b,c,d}
