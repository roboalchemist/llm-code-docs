EqualWithinRel returns true when the difference between a and b is not greater than tol times the greater absolute value of a and b,

#### func [NaNWith](https://github.com/gonum/gonum/blob/v0.17.0/floats/scalar/scalar.go#L78)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/floats/scalar#NaNWith "Go to NaNWith")

func NaNWith(payload [uint64](https://pkg.go.dev/builtin#uint64)) [float64](https://pkg.go.dev/builtin#float64)

NaNWith returns an IEEE 754 "quiet not-a-number" value with the payload specified in the low 51 bits of payload. The NaN returned by math.NaN has a bit pattern equal to NaNWith(1).

#### func [ParseWithNA](https://github.com/gonum/gonum/blob/v0.17.0/floats/scalar/scalar.go#L95)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/floats/scalar#ParseWithNA "Go to ParseWithNA")

func ParseWithNA(s, missing [string](https://pkg.go.dev/builtin#string)) (value, weight [float64](https://pkg.go.dev/builtin#float64), err [error](https://pkg.go.dev/builtin#error))

ParseWithNA converts the string s to a float64 in value. If s equals missing, weight is returned as 0, otherwise 1.

Example [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/floats/scalar#example-ParseWithNA "Go to Example")

package main

import (
	"bufio"
	"fmt"
	"log"
	"strings"

	"gonum.org/v1/gonum/floats/scalar"
	"gonum.org/v1/gonum/stat"
)

func main() {
	// Calculate the mean of a list of numbers
	// ignoring missing values.
	const data = `6
missing
4
`

	var vals, weights []float64
	sc := bufio.NewScanner(strings.NewReader(data))
	for sc.Scan() {
		v, w, err := scalar.ParseWithNA(sc.Text(), "missing")
		if err != nil {
			log.Fatal(err)
		}
		vals = append(vals, v)
		weights = append(weights, w)
	}
	err := sc.Err()
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(stat.Mean(vals, weights))

}
