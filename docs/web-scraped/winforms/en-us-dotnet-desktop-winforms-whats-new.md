.NET 9
------

WinForms in .NET 9 introduces several modern improvements. The asynchronous forms functionality provides APIs to help run UI-related operations asynchronously, making it easier to integrate with contemporary asynchronous programming patterns. The removal of BinaryFormatter eliminates a long-standing security risk by preventing unsafe deserialization practices. Additionally, experimental dark mode support has been added, allowing applications to adapt their color schemes to better suit dark environments.

* [Async forms](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#async-forms)
* [BinaryFormatter no longer supported](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#binaryformatter-no-longer-supported)
* [Dark mode](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#dark-mode)
* [FolderBrowserDialog enhancements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#folderbrowserdialog-enhancements)
* [System.Drawing new features and enhancements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#systemdrawing-new-features-and-enhancements)
* [ToolStrip](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net90#toolstrip)

[](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/#net-8)
.NET 8
------

In .NET 8, Windows Forms has again enhanced DPI support, notably through Visual Studio DPI improvements. This enhancement allows the Windows Designer to run in a DPI-unaware mode independently from Visual Studio, ensuring that your app’s design remains sharp while Visual Studio itself stays at its native DPI setting. Another key focus area was data binding improvements and button commands.

* [Data binding improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net80#data-binding-improvements)
* [Button commands](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net80#button-commands)
* [Visual Studio DPI improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net80#visual-studio-dpi-improvements)
* [High DPI improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net80#high-dpi-improvements)
* [Miscellaneous improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net80#miscellaneous-improvements)

[](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/#net-7)
.NET 7
------

In .NET 7, significant improvements have been made to High DPI rendering. These enhancements ensure that nested controls, such as buttons within panels on tab pages, scale correctly according to the current monitor's DPI settings. This feature, which is opt-in for .NET 7, will be enabled by default in .NET 8.

* [Overview of WinForms on .NET 7](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net70)
* [High DPI improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net70#high-dpi-improvements)

.NET 6
------

The focus of .NET 5 for WinForms included updated templates for C# that use global directives, file-scoped namespaces, and nullable reference types. A new application bootstrap was introduced, which simplifies the configuration of Windows Forms applications by using the `ApplicationConfiguration.Initialize` method.

* [Overview of WinForms on .NET 6](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60)
* [Updated templates for C#](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#updated-templates-for-c)
* [New application bootstrap](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#new-application-bootstrap)
* [Change the default font](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#change-the-default-font)
* [Visual Studio designer improvements](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#visual-studio-designer-improvements)
* [High DPI improvements for PerMonitorV2](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#high-dpi-improvements-for-permonitorv2)
* [New APIs](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#new-apis)
* [Updated APIs](https://learn.microsoft.com/en-us/dotnet/desktop/winforms/whats-new/net60#updated-apis)

.NET 5
------
