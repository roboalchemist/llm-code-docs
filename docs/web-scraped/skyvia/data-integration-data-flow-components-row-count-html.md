Row Count
=========

**Row Count** is a component that counts the input rows and writes the result to a data flow variable. It passes its input records to its output without any modification.

To configure Row Count, select a variable to put the number of the rows to. The variable must have a numeric type. You can also give **Name** to the component displayed on the diagram.
