_This page provides the OpenAPI yaml file and the associated details of the KDB.AI's REST API._

Before you start, ensure you have an active KDB.AI [Server](https://kx.com/kdb-ai-server-download/) license. Recommended tool: [cURL](https://curl.se/download.html)

Servers

Computed URL:`https://cloud.kdb.ai`

#### Server variables

protocol
host

### [Database](https://code.kx.com/kdbai/latest/reference/rest-api.html)
Create, retrieve and delete databases.

### [Table](https://code.kx.com/kdbai/latest/reference/rest-api.html)
Create, update, retrieve and delete tables.

### [Index](https://code.kx.com/kdbai/latest/reference/rest-api.html)
Retrieve and list indexes.

### [Data](https://code.kx.com/kdbai/latest/reference/rest-api.html)
Insert, query, and search data.

### [Info](https://code.kx.com/kdbai/latest/reference/rest-api.html)
Get information on tables, databases, sessions, processes, or system.
