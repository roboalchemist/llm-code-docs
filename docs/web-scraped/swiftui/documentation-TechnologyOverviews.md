Technology Overviews
====================

Learn about the wide range of technologies you use to develop software for Apple platforms.

[Overview](https://developer.apple.com/documentation/TechnologyOverviews/#overview)
-----------------------------------------------------------------------------------

Whether you’re new to Apple platforms or you’ve been working with them for years, finding the technology you need is an important first step. With so many technologies available to you, it’s sometimes difficult to know where to start. The following topics offer a high-level view of the technologies available to you, and guidance about which technologies you might choose to solve particular problems.

[Get started](https://developer.apple.com/documentation/TechnologyOverviews/#Get-started)
-----------------------------------------------------------------------------------------

Start your exploration with the foundational technologies you use to build your app or game, and make your interface shine by adopting Liquid Glass.

[![Image 1](https://docs-assets.developer.apple.com/published/2db40995841673a8617698db9d1eb6b8/app-design-and-ui.png) App design and UI](https://developer.apple.com/documentation/technologyoverviews/app-design-and-ui)

[![Image 2](https://docs-assets.developer.apple.com/published/1c8090b00efa18bfa9e9875cf6a94f5a/games-PageImage-card.png) Games](https://developer.apple.com/documentation/technologyoverviews/games)

[Discover Apple technologies](https://developer.apple.com/documentation/TechnologyOverviews/#Discover-Apple-technologies)
-------------------------------------------------------------------------------------------------------------------------

Explore the technologies you use to build your app’s unique experience. Whether you’re buiding core features like your app’s data model, accessing device-specific hardware, making your app more intelligent, or adopting features unique to Apple devices, Apple frameworks help you do so easily and efficiently.

[![Image 3](https://docs-assets.developer.apple.com/published/2cfb1380b7cf0c783a627bceb5a0972f/data-management-PageImage-card%402x.png) Data management](https://developer.apple.com/documentation/technologyoverviews/data-management)

[![Image 4](https://docs-assets.developer.apple.com/published/dea5cae4199f64efbbf8f844825bbfeb/core-experiences.png) Core experiences](https://developer.apple.com/documentation/technologyoverviews/core-experiences)

[![Image 5: An illustration of the Apple Intelligence logo hovering above a grid of colorful squares.](https://docs-assets.developer.apple.com/published/ead88b7bbfa12b802a7f6cc8a81333a7/Apple-Intelligence-Machine_Learning_overview-PageImage-card%402x.png) Apple Intelligence and machine learning](https://developer.apple.com/documentation/technologyoverviews/ai-machine-learning)

[![Image 6: An illustration of multicolor audio waves.](https://docs-assets.developer.apple.com/published/f84ac97d4fb2bc50643882815696ed48/audio-and-video-PageImage-card.png) Audio and video](https://developer.apple.com/documentation/technologyoverviews/audio-and-video)

[![Image 7](https://docs-assets.developer.apple.com/published/7cc36e385ec24db444f5c9afd0559128/sensors-networking-and-hardware-PageImage-card%402x.png) Hardware, networking, and sensors](https://developer.apple.com/documentation/technologyoverviews/hardware-networking-sensors)

[Explore the Apple Developer Documentation](https://developer.apple.com/documentation/TechnologyOverviews/#Explore-the-Apple-Developer-Documentation)
-----------------------------------------------------------------------------------------------------------------------------------------------------

Consult the [Apple Developer Documentation](https://developer.apple.com/documentation/) for in-depth information about individual technologies. The developer documentation includes API reference, articles, sample code, and tutorials to help you learn how to use a given technology. It also offers guidance about the best way to solve specific challenges.
