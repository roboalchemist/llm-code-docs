Sign up for New Relic
=====================

At New Relic, our mission is to help you create more perfect software. So we give you **free access** to our observability platform, **forever**. For [free](https://docs.newrelic.com/docs/accounts/accounts-billing/new-relic-one-pricing-billing/new-relic-one-pricing-billing/#free), you can ingest up to 100 GBs per month, and have one [full platform user](https://docs.newrelic.com/docs/accounts/accounts-billing/new-relic-one-user-management/user-type/) and **unlimited** basic users.

Ready to get started?

[Sign up for New Relic](https://newrelic.com/signup)

### Next steps [](https://docs.newrelic.com/docs/accounts/accounts-billing/account-setup/create-your-new-relic-account/#next-steps)

Here are some next steps after signing up:

*   Start reporting data using our [**Add your data** UI](https://one.newrelic.com/launcher/nr1-core.settings)
*   [Get ideas on how to bring all your data together](https://docs.newrelic.com/docs/using-new-relic/welcome-new-relic/get-started/introduction-new-relic/#bring-your-data)
*   [Watch New Relic University videos](https://learn.newrelic.com/)
*   If you unintentionally signed up, see [Delete account](https://docs.newrelic.com/docs/accounts/accounts-billing/account-setup/downgradecancel-account/#cancel-unused-org)

Add accounts to an existing New Relic organization [](https://docs.newrelic.com/docs/accounts/accounts-billing/account-setup/create-your-new-relic-account/#existing)
---------------------------------------------------------------------------------------------------------------------------------------------------------------------

If you already have an existing New Relic organization and want to add accounts to it, see [Add accounts](https://docs.newrelic.com/docs/accounts/accounts-billing/account-structure/add-accounts/).
