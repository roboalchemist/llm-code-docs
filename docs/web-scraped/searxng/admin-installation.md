*You're spoilt for choice*, choose your preferred method of installation.

-   [[Installation container]](installation-docker.html#installation-container)

-   [[Installation Script]](installation-scripts.html#installation-scripts)

-   [[Step by step installation]](installation-searxng.html#installation-basic)

The [[Step by step installation]](installation-searxng.html#installation-basic) is an excellent illustration of *how a SearXNG instance is build up* (see [[uWSGI Setup]](architecture.html#architecture-uwsgi)). If you do not have any special preferences, it's recommended to use the [[Installation container]](installation-docker.html#installation-container) or the [[Installation Script]](installation-scripts.html#installation-scripts).

Attention

SearXNG is growing rapidly, you should regularly read our [[Migrate and stay tuned!]](update-searxng.html#migrate-and-stay-tuned) section. If you want to upgrade an existing instance, you should read this section first!
