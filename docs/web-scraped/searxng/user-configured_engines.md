SearXNG supports 247 search engines of which 85 are enabled by default.

Engines can be assigned to multiple [[categories]](../admin/settings/settings_engines.html#engine-categories). The UI displays the tabs that are configured in [[categories_as_tabs]](../admin/settings/settings_categories_as_tabs.html#settings-categories-as-tabs). In addition to these UI categories (also called *tabs*), engines can be queried by their name or the categories they belong to, by using a [[!bing syntax]](search-syntax.html#search-syntax).
