### [macOS](#id9)[¬∂](#macos)

Let‚Äôs install beartype with [Homebrew](https://brew.sh) on [macOS](https://en.wikipedia.org/wiki/MacOS) courtesy our third-party

### [Arch Linux](#id10)[¬∂](#arch-linux)

Let‚Äôs install beartype with `pacman` on [Arch Linux](https://archlinux.org) ‚Äì where beartype is now

cd python-beartype
makepkg -si

Truly, Arch Linux has now seen the face of quality assurance. It looks like a
