.NET Framework system requirements
==================================

The tables in this article provide the hardware, operating system, and software requirements for the following .NET Framework versions:

* .NET Framework 4.8 and its point release (4.8.1).
* .NET Framework 4.7 and its point releases (4.7.1 and 4.7.2).
* .NET Framework 4.6.2.

For information on earlier .NET Framework versions, see [.NET Framework versions and dependencies](https://learn.microsoft.com/en-us/dotnet/framework/install/versions-and-dependencies).

Development environments that enable you to develop apps for .NET Framework have a separate set of requirements.

Important

All .NET Framework versions since .NET Framework 4 are in-place updates, so only a single 4.x version can be present on a system. In addition, particular versions of .NET Framework are preinstalled on some versions of the Windows operating system. This means that:

* If there's a later 4.x version installed on the machine already, you can't install a previous 4.x version.
* If the OS comes preinstalled with a particular .NET Framework version, you can't install a previous 4.x version on the same machine.
* If you install a later version, you don't have to first uninstall the previous version.

For download information and links, see [Install .NET Framework for developers](https://learn.microsoft.com/en-us/dotnet/framework/install/guide-for-developers).

For information on the support lifecycle of .NET Framework versions, see [Microsoft support lifecycle](https://support.microsoft.com/lifecycle/search?sort=PN&alpha=Microsoft%20.NET%20Framework&Filter=FilterNO).

---------------------

|  | Requirement |

| Minimum disk space (64-bit) | 4.5 GB |

[](https://learn.microsoft.com/en-us/dotnet/framework/get-started/system-requirements#installation-requirements)
Installation requirements
-------------------------

.NET Framework requires administrator privileges for installation. If you don't have administrator rights to the computer where you'd like to install .NET Framework, contact your network administrator.

[](https://learn.microsoft.com/en-us/dotnet/framework/get-started/system-requirements#compatible-operating-systems)
Compatible operating systems
----------------------------

* [Client operating systems](https://learn.microsoft.com/en-us/dotnet/framework/get-started/system-requirements#client-operating-systems)
* [Server operating systems](https://learn.microsoft.com/en-us/dotnet/framework/get-started/system-requirements#server-operating-systems)

For all platforms, to ensure the best compatibility and security, we recommend that you install critical updates available from [Windows Update](https://support.microsoft.com/help/12373/windows-update-faq).

On 64-bit operating systems, .NET Framework supports both WOW64 (32-bit processing on a 64-bit machine) and native 64-bit processing.

[](https://learn.microsoft.com/en-us/dotnet/framework/get-started/system-requirements#client-operating-systems)
