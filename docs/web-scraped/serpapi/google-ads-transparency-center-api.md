### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-search-query) Search Query

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-search-query-advertiser-id)

advertiser_id

Optional

Parameter defines the Google Advertiser ID. It can be found in the Ads Transparency Center advertiser URL.

For example, the advertiser ID for the URL `https://adstransparency.google.com/advertiser/AR17828074650563772417` is `AR17828074650563772417`(Tesla Inc).

It accepts either a single ID or multiple IDs separated by commas (`,`). Example,

Single ID: `AR17828074650563772417`

Multiple IDs: `AR17828074650563772417,AR03911981557522366465,...`

Alternatively, text can be used for free-text search.

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-ads-transparency-parameters) Advanced Ads Transparency Parameters

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-ads-transparency-parameters-text)

text

Optional

Parameter defines the text you want to search, typically related to a domain search within the context of Google Ads. You can use anything that you would normally use in a standard Google Ads Transparency Center search.

It can be used as an alternative to advertiser_id. The advertiser_id can be optional when text is present.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-ads-transparency-parameters-platform)

platform

Optional

Parameter defines the target platform where ads are displayed. When not set, it will return results from all platforms (default).

Available options:

`PLAY` - Google Play

`MAPS` - Google Maps

`SEARCH` - Google Search

`SHOPPING` - Google Shopping

`YOUTUBE` - YouTube.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-ads-transparency-parameters-political-ads)

political_ads

Optional

The parameter determine if the advertiser promotes political advertisement. Political advertisement is a different category than regular advertisements, and will not appear on the results unless this parameter is set to `true`. If this parameter is set to `true`, only political advertisements will appear on the results. By default it is `false`.

This parameter can only be used alongside with the `region` parameter.

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-geographic-location) Geographic Location

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-geographic-location-region)

region

Optional

Parameter defines from where you want the search to originate. By default it search results from anywhere. Head to the [Google Ads Transparency Center Regions](https://serpapi.com/google-ads-transparency-center-regions) for a full list of supported regions.

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-date-range) Date Range

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-date-range-start-date)

start_date

Optional

Parameter defines the start date for which you want the search results to begin. The format for this field is `YYYYMMDD`.

Example: `20221201`

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-date-range-end-date)

end_date

Optional

Parameter defines the end date for which you want the search results to conclude. The format for this field is `YYYYMMDD`.

Example: `20221231`

To set the date for **Today / Single Day**, you should set end_date as start_date + `1 day`.

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-filters) Advanced Filters

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-advanced-filters-creative-format)

creative_format

Optional

Parameter defines the format you want to filter by. Only search results with a similar format will be returned.

Available options:

`text` - Text

`image` - Image

`video` - Video

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-pagination) Pagination

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-pagination-num)

num

Optional

Parameter defines the maximum number of results to return. (e.g., `40` (default) returns `40` results and `100` returns `100` results).

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-pagination-next-page-token)

next_page_token

Optional

Parameter defines the next page token. It is used for retrieving the next page results.

### [](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters)![Image 32](https://serpapi.com/assets/square-logo-v3-small.png) Serpapi Parameters

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-engine)

engine

Required

Set parameter to `google_ads_transparency_center` to use the Google Ads Transparency Center API engine.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-no-cache)

no_cache

Optional

Parameter will force SerpApi to fetch the Google Ads Transparency Center results even if a cached version is already present. A cache is served only if the query and all parameters are exactly the same. Cache expires after 1h. Cached searches are free, and are not counted towards your searches per month. It can be set to `false` (default) to allow results from the cache, or `true` to disallow results from the cache. no_cache and async parameters should not be used together.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-async)

async

Optional

Parameter defines the way you want to submit your search to SerpApi. It can be set to `false` (default) to open an HTTP connection and keep it open until you got your search results, or `true` to just submit your search to SerpApi and retrieve them later. In this case, you'll need to use our [Searches Archive API](https://serpapi.com/search-archive-api) to retrieve your results. async and no_cache parameters should not be used together. async should not be used on accounts with [Ludicrous Speed](https://serpapi.com/plan) enabled.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-zero-trace)

zero_trace

Optional

Enterprise only. Parameter enables [ZeroTrace mode](https://serpapi.com/zero-trace-mode). It can be set to `false` (default) or `true`. Enable this mode to skip storing search parameters, search files, and search metadata on our servers. This may make debugging more difficult.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-api-key)

api_key

Required

Parameter defines the SerpApi private key to use.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-output)

output

Optional

Parameter defines the final output you want. It can be set to json (default) to get a structured `JSON` of the results, or `html` to get the raw html retrieved.

[](https://serpapi.com/google-ads-transparency-center-api#api-parameters-serpapi-parameters-json-restrictor)

json_restrictor

Optional

Parameter defines the fields you want to restrict in the outputs for smaller, faster responses. See [JSON Restrictor](https://serpapi.com/json-restrictor) for more details.

[](https://serpapi.com/google-ads-transparency-center-api#api-results) API Results
----------------------------------------------------------------------------------

### [](https://serpapi.com/google-ads-transparency-center-api#api-results-json-results) JSON Results

JSON output includes structured data for suggested searches.

A search status is accessible through `search_metadata.status`. It flows this way: `Processing` ->`Success` || `Error`. If a search has failed, `error` will contain an error message. `search_metadata.id` is the search ID inside SerpApi.

### [](https://serpapi.com/google-ads-transparency-center-api#api-results-html-results) HTML Results

HTML output is useful to debug JSON results or support features not supported yet by SerpApi. HTML output gives you the raw HTML results from Google Ads Transparency Center.

This API does not have html response, just a text. `search_metadata.prettify_html_file` contains prettified version of the result. It is displayed in the playground.
