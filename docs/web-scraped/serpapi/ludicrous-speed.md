FAQs
----

Is Ludicrous Speed available for all the subscription plans?

Ludicrous Speed can be switched on or off for all paid plans, including self-served, high-volume, and enterprise plans.

How does Ludicrous Speed work?

With Ludicrous Speed enabled, we use **2x the server resources** to automatically create numerous parallel requests compared to our already aggressive Best Effort SerpApi. Our advanced validators reject bad HTMLs, CAPTCHA and error pages, and other abnormalities, ensuring you receive only successful results quickly and efficiently.

Can Ludicrous Speed be used with all of SerpApi's search APIs?

Yes, Ludicrous Speed is fully supported for all SerpApi's search APIs.

How much does Ludicrous Speed cost?

Ludicrous Speed is priced at 2x the price of Best Effort.

How to enable Ludicrous Speed?

To enable Ludicrous Speed, first, create an account on our platform. Once logged in, go to your Dashboard >[Change Plan](https://serpapi.com/change-plan), choose the plan that suits your needs, and enable the “Ludicrous Speed” option. If you already have a subscription, you can upgrade your plan on the same page by enabling the “Ludicrous Speed” option.

Can Ludicrous Speed be used for live use cases?

Yes, Ludicrous Speed is a strong option for live use cases where fast response times are important. If response times are crucial, we recommend our [Ludicrous Speed Max](https://serpapi.com/ludicrous-speed-max) option.
