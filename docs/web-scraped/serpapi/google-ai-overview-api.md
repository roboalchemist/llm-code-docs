Google AI Overview API
======================

[API uptime 99.999%](https://serpapi.com/status/google_ai_overview)

Our Google AI Overview API allows you to scrape results from the AI Overview block from Google search results.

The API endpoint is `https://serpapi.com/search?engine=google_ai_overview`

This engine is used only when Google requires a separate request to fetch AI Overview results. See [AI Overview Results](https://serpapi.com/ai-overview) for results embedded in the main search results.

[](https://serpapi.com/google-ai-overview-api#new-ai-mode-api) New Google AI Mode API available
-----------------------------------------------------------------------------------------------

We have introduced new search engine called [Google AI Mode API](https://serpapi.com/google-ai-mode-api) that allows you to scrape results from the Google AI Mode page.

[](https://serpapi.com/google-ai-overview-api#api-parameters) API Parameters
----------------------------------------------------------------------------

### [](https://serpapi.com/google-ai-overview-api#api-parameters-search-query) Search Query

[](https://serpapi.com/google-ai-overview-api#api-parameters-search-query-page-token)

page_token

Required

Parameter defines the token to fetch the AI Overview from. Use `ai_overview.page_token` from the [AI Overview Results](https://serpapi.com/ai-overview) section of our [Google Search API](https://serpapi.com/search-api) — see the [extra request example](https://serpapi.com/ai-overview#api-examples-example-with-an-extra-request-required). 

`ai_overview.page_token`**expires within 1 minute** of the search and should be used immediately.

### [](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters)![Image 32](https://serpapi.com/assets/square-logo-v3-small.png) Serpapi Parameters

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-engine)

engine

Required

Set parameter to `google_ai_overview` to use the Google AI Overview API engine.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-no-cache)

no_cache

Optional

Parameter will force SerpApi to fetch the Google AI Overview results even if a cached version is already present. A cache is served only if the query and all parameters are exactly the same. Cache expires after 1h. Cached searches are free, and are not counted towards your searches per month. It can be set to `false` (default) to allow results from the cache, or `true` to disallow results from the cache. no_cache and async parameters should not be used together.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-async)

async

Optional

Parameter defines the way you want to submit your search to SerpApi. It can be set to `false` (default) to open an HTTP connection and keep it open until you got your search results, or `true` to just submit your search to SerpApi and retrieve them later. In this case, you'll need to use our [Searches Archive API](https://serpapi.com/search-archive-api) to retrieve your results. async and no_cache parameters should not be used together. async should not be used on accounts with [Ludicrous Speed](https://serpapi.com/plan) enabled.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-zero-trace)

zero_trace

Optional

Enterprise only. Parameter enables [ZeroTrace mode](https://serpapi.com/zero-trace-mode). It can be set to `false` (default) or `true`. Enable this mode to skip storing search parameters, search files, and search metadata on our servers. This may make debugging more difficult.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-api-key)

api_key

Required

Parameter defines the SerpApi private key to use.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-output)

output

Optional

Parameter defines the final output you want. It can be set to json (default) to get a structured `JSON` of the results, or `html` to get the raw html retrieved.

[](https://serpapi.com/google-ai-overview-api#api-parameters-serpapi-parameters-json-restrictor)

json_restrictor

Optional

Parameter defines the fields you want to restrict in the outputs for smaller, faster responses. See [JSON Restrictor](https://serpapi.com/json-restrictor) for more details.

[](https://serpapi.com/google-ai-overview-api#api-results) API Results
----------------------------------------------------------------------

### [](https://serpapi.com/google-ai-overview-api#api-results-json-results) JSON Results

JSON output includes structured data for `text_blocks`, `thumbnail` and `references`.

A search status is accessible through `search_metadata.status`. It flows this way: `Processing` ->`Success` || `Error`. If a search has failed, `error` will contain an error message. `search_metadata.id` is the search ID inside SerpApi.

### [](https://serpapi.com/google-ai-overview-api#api-results-html-results) HTML Results

HTML output is useful to debug JSON results or support features not supported yet by SerpApi. HTML output gives you the raw HTML results from Google AI Overview.
