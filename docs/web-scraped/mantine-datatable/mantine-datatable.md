![Image 2: Mantine DataTable](https://icflorescu.github.io/mantine-datatable/_next/static/media/hero.55e347d5.png)

Mantine DataTable is a[lightweight](https://bundlephobia.com/package/mantine-datatable), dependency-free component that brings datagrid-like functionality to your data-rich user interfaces, with dark-mode support and intuitive Gmail-style additive batch rows selection out of the box.

Lightweight yet customizable

Supports [asynchronous data loading](https://icflorescu.github.io/mantine-datatable/examples/asynchronous-data-loading/),[pagination](https://icflorescu.github.io/mantine-datatable/examples/pagination/), intuitive[rows selection](https://icflorescu.github.io/mantine-datatable/examples/records-selection/), column[sorting](https://icflorescu.github.io/mantine-datatable/examples/sorting/), custom[cell data rendering](https://icflorescu.github.io/mantine-datatable/examples/column-properties-and-styling/),[context menus](https://icflorescu.github.io/mantine-datatable/examples/using-with-mantine-contextmenu/),[row expansion](https://icflorescu.github.io/mantine-datatable/examples/expanding-rows/),[nesting](https://icflorescu.github.io/mantine-datatable/examples/nested-tables/),[drag-and-drop reordering support](https://icflorescu.github.io/mantine-datatable/examples/row-dragging/) and[more](https://icflorescu.github.io/mantine-datatable/examples/complex-usage-scenario/)

Typescript based

The entire codebase is [written in TypeScript](https://github.com/icflorescu/mantine-datatable), options are[well typed](https://icflorescu.github.io/mantine-datatable/type-definitions/) and documented with JSDoc, so you can build type safe applications with confidence

Free and open-source

This package is released under the [MIT license](https://github.com/icflorescu/mantine-datatable/blob/main/LICENSE), same as[Mantine](https://mantine.dev/), so you can freely build fantastic data-rich applications with it

Use anywhere

You can use it in any modern React framework supported by[Mantine](https://mantine.dev/), such as[Next.js](https://nextjs.org/), [Vite](https://vitejs.dev/),[Create React App](https://create-react-app.dev/),[Remix](https://remix.run/) or [Gatsby](https://www.gatsbyjs.com/)
