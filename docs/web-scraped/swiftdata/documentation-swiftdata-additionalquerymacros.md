Additional query macros
=======================

Supplementary macros that enable you to narrow query results and tell SwiftData how to sort and order those results.

[Topics](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#topics)
------------------------------------------------------------------------------------------

### [Basic queries](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#Basic-queries)

[`macro Query(animation: Animation)`](https://developer.apple.com/documentation/swiftdata/query(animation:))

Fetches all instances of the attached model type, using the specified animation to animate any subsequent changes.

[`macro Query(transaction: Transaction)`](https://developer.apple.com/documentation/swiftdata/query(transaction:))

Fetches all instances of the attached model type, using the specified transaction to animate any subsequent changes.

### [Predicate-based queries](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#Predicate-based-queries)

[`macro Query<Element>(filter: Predicate<Element>?, sort: [SortDescriptor<Element>], animation: Animation)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:animation:))

Fetches a sorted subset of the attached model type that satisfy the specified predicate.

[`macro Query<Value, Element>(filter: Predicate<Element>?, sort: KeyPath<Element, Value>, order: SortOrder, animation: Animation)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:order:animation:)-80h6f)

Fetches a subset of the attached model type, in a specific order, by sorting on a nonoptional attribute.

[`macro Query<Value, Element>(filter: Predicate<Element>?, sort: KeyPath<Element, Value?>, order: SortOrder, animation: Animation)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:order:animation:)-pb15)

Fetches a subset of the attached model type, in a specific order, by sorting on an optional attribute.

[`macro Query<Element>(filter: Predicate<Element>?, sort: [SortDescriptor<Element>], transaction: Transaction?)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:transaction:))

Fetches and sorts the subset of the attached model type that satisfy the specified predicate.

[`macro Query<Value, Element>(filter: Predicate<Element>?, sort: KeyPath<Element, Value>, order: SortOrder, transaction: Transaction?)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:order:transaction:)-6kkiu)

Fetches a subset of the attached model type, in a specific order, by sorting on a nonoptional attribute.

[`macro Query<Value, Element>(filter: Predicate<Element>?, sort: KeyPath<Element, Value?>, order: SortOrder, transaction: Transaction?)`](https://developer.apple.com/documentation/swiftdata/query(filter:sort:order:transaction:)-8tk8u)

Fetches a subset of the attached model type, in a specific order, by sorting on an optional attribute.

### [Descriptor-based queries](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#Descriptor-based-queries)

[`macro Query<Element>(FetchDescriptor<Element>, animation: Animation)`](https://developer.apple.com/documentation/swiftdata/query(_:animation:))

Fetches only the subset of the attached model type that satisfy the provided fetch descriptor’s criteria.

[`macro Query<Element>(FetchDescriptor<Element>, transaction: Transaction?)`](https://developer.apple.com/documentation/swiftdata/query(_:transaction:))

Fetches only the subset of the attached model type that satisfy the provided fetch descriptor’s criteria.

[See Also](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#see-also)
----------------------------------------------------------------------------------------------

### [Model fetch](https://developer.apple.com/documentation/swiftdata/additionalquerymacros#Model-fetch)

[Filtering and sorting persistent data](https://developer.apple.com/documentation/swiftdata/filtering-and-sorting-persistent-data)

Manage data store presentation using predicates and dynamic queries.

[`macro Query()`](https://developer.apple.com/documentation/swiftdata/query())

Fetches all instances of the attached model type.

[`struct Query`](https://developer.apple.com/documentation/swiftdata/query)

A type that fetches models using the specified criteria, and manages those models so they remain in sync with the underlying data.

[`struct FetchDescriptor`](https://developer.apple.com/documentation/swiftdata/fetchdescriptor)

A type that describes the criteria, sort order, and any additional configuration to use when performing a fetch.
