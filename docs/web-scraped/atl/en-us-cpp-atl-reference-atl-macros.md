ATL Macros
==========

In this article
---------------

Note

The Active Template Library (ATL) continues to be supported. However, we're no longer adding features or updating the documentation.

To find an ATL macro by category, see the following topics.

[Aggregation and Class Factory Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/aggregation-and-class-factory-macros?view=msvc-170)

 Provide ways of controlling aggregation and of declaring class factories.

[Category Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/category-macros?view=msvc-170)

 Define category maps.

[COM Map Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/com-map-macros?view=msvc-170)

 Define COM interface maps.

[Compiler Options Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/compiler-options-macros?view=msvc-170)

 Control specific compiler features.

[Composite Control Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/composite-control-macros?view=msvc-170)

 Define event sink maps and entries.

[Connection Point Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/connection-point-macros?view=msvc-170)

 Define connection point maps and entries.

[Debugging and Error Reporting Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/debugging-and-error-reporting-macros?view=msvc-170)

 Provide useful debugging and trace facilities.

[Exception Handling Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/exception-handling-macros?view=msvc-170)

 Provide support for exception handling.

[Message Map Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/message-map-macros-atl?view=msvc-170)

 Define message maps and entries.

[Object Map Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/object-map-macros?view=msvc-170)

 Define object maps and entries.

[Object Status Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/object-status-macros?view=msvc-170)

 Sets flags belonging to ActiveX controls.

[Property Map Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/property-map-macros?view=msvc-170)

 Define property maps and entries.

[Registry Data Exchange Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/registry-data-exchange-macros?view=msvc-170)

 Perform Registry Data Exchange operations.

[Registry Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/registry-macros?view=msvc-170)

 Define useful type library and registry facilities.

[Service Map Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/service-map-macros?view=msvc-170)

 Define service maps and entries.

[Snap-In Object Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/snap-in-object-macros?view=msvc-170)

 Provide support for snap-in extensions.

[String Conversion Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/string-conversion-macros?view=msvc-170)

 Provide string conversion features.

[Window Class Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/window-class-macros?view=msvc-170)

 Define window class utilities.

[Windows Messages Macros](https://learn.microsoft.com/en-us/cpp/atl/reference/windows-messages-macros?view=msvc-170)

 Forward window messages.
