Troubleshooting
---------------

If you encounter issues, follow the guides below for help. For topics not covered here, you can [file an issue](https://github.com/projectcontour/contour/issues), or talk to us on the [#contour channel](https://kubernetes.slack.com/messages/contour) on Kubernetes Slack.

### [Troubleshooting Common Proxy Errors](https://projectcontour.io/docs/main/troubleshooting/common-proxy-errors/)

A guide on how to investigate common errors with Contour and Envoy.

### [Envoy Administration Access](https://projectcontour.io/docs/main/troubleshooting/envoy-admin-interface/)

Review the linked steps to learn how to access the administration interface for your Envoy instance.

### [Contour Debug Logging](https://projectcontour.io/docs/main/troubleshooting/contour-debug-log/)

Learn how to enable debug logging to diagnose issues between Contour and the Kubernetes API.

### [Envoy Debug Logging](https://projectcontour.io/docs/main/troubleshooting/envoy-debug-log/)

Learn how to enable debug logging to diagnose TLS connection issues.

### [Visualize the Contour Graph](https://projectcontour.io/docs/main/troubleshooting/contour-graph/)

Learn how to visualize Contour’s internal object graph in [DOT](https://en.wikipedia.org/wiki/Dot) format, or as a png file.

### [Show Contour xDS Resources](https://projectcontour.io/docs/main/troubleshooting/contour-xds-resources/)

Review the linked steps to view the [xDS](https://www.envoyproxy.io/docs/envoy/latest/api-docs/xds_protocol) resource data exchanged by Contour and Envoy.

### [Profiling Contour](https://projectcontour.io/docs/main/troubleshooting/profiling-contour/)

Learn how to profile Contour by using [net/http/pprof](https://golang.org/pkg/net/http/pprof/) handlers.

### [Envoy container stuck in unready/draining state](https://projectcontour.io/docs/main/troubleshooting/envoy-container-draining/)

Read the linked document if you have Envoy containers stuck in an unready/draining state.
