The Telerik UI CheckBox TagHelper and HtmlHelper for ASP.NET Core are server-side wrappers for the Kendo UI CheckBox widget.

The Telerik UI CheckBox for ASP.NET Core is based on the conventional HTML checkbox element. It allows you to add more customizable checkboxes to your application.

[Initializing the CheckBox](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/overview#initializing-the-checkbox)
--------------------------------------------------------------------------------------------------------------------------------------------------

The following example demonstrates how to define the CheckBox.

HtmlHelper TagHelper

Razor

```Razor
@(Html.Kendo().CheckBox()
        .Name("eq1")    
        .Checked(true)
        .Label("My Telerik Checkbox")
    )
```

Razor

```Razor
<kendo-checkbox name="eq1"
                checked="true"
                label="My Telerik Checkbox">
    </kendo-checkbox>
```

[Functionality and Features](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/overview#functionality-and-features)
----------------------------------------------------------------------------------------------------------------------------------------------------

* [Appearance](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/appearance)—The CheckBox enables you to customize its appearance based on your requirements.
* [Events](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/events)—The CheckBox allows you to handle its events and implement custom functionality.
* [Accessibility](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/accessibility/overview)—The CheckBox is accessible for screen readers, supports WAI-ARIA attributes, and delivers [keyboard shortcuts](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/accessibility/keyboard-navigation) for faster navigation.

> To learn more about the appearance, anatomy, and accessibility of the CheckBox, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/checkbox/)—an information portal offering rich component usage guidelines, descriptions of the available style variables, and globalization support details.

[Two-way Binding](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/checkbox/overview#two-way-binding)
------------------------------------------------------------------------------------------------------------------------------

To use the Telerik UI Checkbox as an editor for a field in your C# Model, declare by using the CheckboxFor helper:

HtmlHelper TagHelper

Razor

```Razor
@(Html.Kendo().CheckBoxFor(m=>m.BooleanFieldName)
        .Label("Editable Checkbox"))
    )
```

Razor

```Razor
<kendo-checkbox name="BooleanFieldName"
                label="Editable Checkbox">
    </kendo-checkbox>
```
