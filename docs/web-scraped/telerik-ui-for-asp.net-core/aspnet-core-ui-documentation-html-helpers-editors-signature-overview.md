The Telerik UI Signature for ASP.NET Core is a server-side wrapper for the Kendo UI Signature widget.

The Telerik UI Signature for ASP.NET Core enables the user to create handwritten signatures.

* [Demo page for the Signature HtmlHelper](https://demos.telerik.com/aspnet-core/signature/index)

[Basic Configuration](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/signature/overview#basic-configuration)
---------------------------------------------------------------------------------------------------------------------------------------

The following example demonstrates the basic configuration for the Signature.

HtmlHelper TagHelper

Razor

```Razor
@(Html.Kendo().Signature()
                .Name("signature")
                .Maximizable(false)
                .HideLine(true))
```

Razor

```Razor
<kendo-signature name="signature"
                         maximizable="false"
                         hide-line="true">
        </kendo-signature>
```

[Functionality and Features](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/signature/overview#functionality-and-features)
-----------------------------------------------------------------------------------------------------------------------------------------------------

* [Image Export](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/signature/image-export)—You can export the Signature to a PNG file.
* [Form Integration](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/signature/form-integration)—The Signature allows you to integrate it inside a Telerik UI Form component.
