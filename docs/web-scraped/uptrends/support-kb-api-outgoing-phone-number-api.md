Overview
--------

[Uptrends integrations](https://www.uptrends.com/support/kb/alerting/integrations/what-are-integrations/) send alerts when your monitors detect any errors or issues. With the [phone alert integration](https://www.uptrends.com/support/kb/alerting/phone-alerts/), you’ll get a call with an automated message about the alert, so you’re instantly informed about the situation.

Use case
--------

The OutgoingPhoneNumber API lists all outgoing phone numbers used as official caller IDs for Uptrends alerts.

Prerequisites
-------------

Before using the OutgoingPhoneNumber API, make sure you have:

*   [Uptrends API credentials](https://www.uptrends.com/support/kb/api/authentication-v4/)
*   [A Premium level account or higher](https://www.uptrends.com/support/kb/account/payments-and-subscriptions/subscriptions-overview/)
*   A [Phone alert integration setup](https://www.uptrends.com/support/kb/alerting/phone-alerts/#set-up-phone-alert-integration)

OutgoingPhoneNumber API fields
------------------------------

The OutgoingPhoneNumber API contains various API fields:

| Field name | Description |
| --- | --- |
| `Id` | The unique identifier of the outgoing phone number. |
| `PhoneNumber` | The international phone number used by Uptrends: * `+4921195987963` — German phone number * `+441584322044`— United Kingdom (UK) phone number * `+14243224154` — United States of America (USA) phone number * `+31858887978` — Netherlands phone number * `+33970730750` — French phone number |
| `Description` | The country associated with the phone number: * German * United Kingdom * United States of America (USA) * Netherlands * French |

For more information about the list of supported countries for phone alert integration, refer to [Phone alerts troubleshooting](https://www.uptrends.com/support/kb/alerting/phone-alerts-troubleshooting/#list-of-supported-countries).

OutgoingPhoneNumber API endpoints
---------------------------------

The OutgoingPhoneNumber API provides the following endpoints:

| Method | Endpoint | Description |
| --- | --- | --- |
| `GET` | `/OutgoingPhoneNumber` | Retrieve a list of all the official Uptrends outgoing phone numbers. |

For detailed request and response formats and interactive API testing, refer to the [Uptrends OutgoingPhoneNumber API documentation](https://api.uptrends.com/v4/swagger/index.html?url=/v4/swagger/v1/swagger.json#/OutgoingPhoneNumber).
