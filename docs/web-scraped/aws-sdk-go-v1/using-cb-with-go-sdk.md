# AWS CodeBuild Examples Using the AWS SDK for Go

CodeBuild is a fully managed build service that compiles source code, runs tests, and produces software packages that are ready to deploy. The AWS SDK for Go examples can integrate AWS CodeBuild into your applications. The examples assume you have already set up and configured the SDK (that is, youâ€™ve imported all required packages and set your credentials and region).
