Markdown Content:
Exim is available as a native package for a number of software distributions. The sources are available as [tarballs](https://downloads.exim.org/exim4/) and as a [Git Repo (see below).](https://www.exim.org/download.html#git)

Further information on the binary and OS distributions can be found in the [Exim Wiki.](https://github.com/Exim/exim.wiki/ObtainingExim)

If we published _maintenance_ releases, you can find the tarballs in the [fixes area,](https://downloads.exim.org/exim4/fixes/) and in individual [git branches](https://www.exim.org/download.html#git) suffixed with _+fixes._ Please note, that we normally do not publish maintenance releases as tarballs, unless they contain critical changes.

### Verification of Downloads

All published tarballs are cryptographically signed with an OpenPGP implementation (such as GnuPG). The signatures are distributed alongside the tarballs. The signatures are created with keys belonging to the developers. The keys can be found in our [maintainers keyring](https://downloads.exim.org/Exim-Maintainers-Keyring.asc). (Please crosscheck these keys with keys you can find at other sources.)

The exim.org domain supports [the WKD mechanism](https://wiki.gnupg.org/WKD) for OpenPGP key retrieval.

### Exim Git Repositories

If you prefer tracking our commits to the source tree directly via Git:

*   Main Git Repository: `https://code.exim.org/exim/exim.git`
*   Web interface: [https://code.exim.org/exim/exim](https://code.exim.org/exim/exim)

All releases get a signed Git tag.
