Markdown Content:
Exim has a set of documentation released with it. A text file of the main documentation is released as part of the Exim tar archive. Additionally, postscript and texinfo forms of the documentation are available in separate tar archives on the ftp sites. There is also a [book](https://www.uit.co.uk/all-books/the-exim-smtp-mail-server.html).

User supported documentation, FAQs and hints can be found in the [wiki](https://wiki.exim.org/).

There are 3 main sets of documentation for Exim, all of which are also available below in html form.

The main user and configuration manual is the specification document

[The Exim Specification - Version 4.99.1 (HTML)](https://www.exim.org/exim-html-current/doc/html/spec_html/index.html)[(PDF)](https://www.exim.org/exim-pdf-current/doc/spec.pdf) - The master documentation for Exim containing all required detail to install, configure and use Exim. Changes to the documentation (normally reflecting changes to the functionality of Exim) are shown on a green background like this segment.

[The Exim Filter Specification - Version 4.99.1 (HTML)](https://www.exim.org/exim-html-current/doc/html/spec_html/filter.html)[(PDF)](https://www.exim.org/exim-pdf-current/doc/filter.pdf) - Additional information on the Exim filter language.
