## Translate & Escape functions


Strings that require translation and is used in attributes of html tags must be escaped.


- esc_html__()
- esc_html_e()
- esc_html_x()
- esc_attr__()
- esc_attr_e()
- esc_attr_x()

## Functions also available in javascript


- __()
- _x()

- _nx()
- sprintf()


Note: To be able to use these functions available in your javascript, you have toset up your plugin/theme javascript localization.
