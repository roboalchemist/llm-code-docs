WordPress provides two core APIs to make the administrative interfaces easy to build, secure, and consistent with the design of WordPress Administration.

TheSettings APIfocuses on providing a way for developers to create forms and manage form data.

TheOptions APIfocuses on managing data using a simple key/value system.
