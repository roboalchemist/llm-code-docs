Casdoor authenticator app
=========================

**Casdoor Authenticator** ([app.casdoor.org](https://app.casdoor.org/)) is an open-source TOTP app ([GitHub](https://github.com/casdoor/casdoor-authenticator)) similar to Google Authenticator or Microsoft Authenticator. It provides MFA with time-based one-time passwords (TOTP) on iOS and Android and can sync accounts with Casdoor.

### Features[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#features "Direct link to Features")

*   **MFA** — Generate TOTP codes for 2FA.
*   **Offline** — Codes work without internet.
*   **Sync** — Accounts sync across devices via Casdoor.
*   **Privacy** — Data encrypted and stored securely.
*   **UI** — Simple, intuitive interface.

What is TOTP?[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#what-is-totp "Direct link to What is TOTP?")
----------------------------------------------------------------------------------------------------------------------------------

**TOTP** (Time-based One-Time Password) is a standard 2FA method. Codes are generated from a shared secret and the current time ([RFC 6238](https://tools.ietf.org/html/rfc6238)): they change every 30 seconds, work offline, and are widely supported.

Using the app[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#using-the-app "Direct link to Using the app")
-----------------------------------------------------------------------------------------------------------------------------------

### Step 0: Install[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#step-0-install "Direct link to Step 0: Install")

*   **Android**: [Releases](https://github.com/casdoor/casdoor-authenticator/releases) or [app.casdoor.org](https://app.casdoor.org/).
*   **iOS**: See [app.casdoor.org](https://app.casdoor.org/) and the [repo](https://github.com/casdoor/casdoor-authenticator).
*   **Build from source**: [Casdoor Authenticator – Building from source](https://github.com/casdoor/casdoor-authenticator#building-from-source).

### Step 1: Enable MFA account storage (optional)[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#step-1-enable-mfa-account-storage-optional "Direct link to Step 1: Enable MFA account storage (optional)")

To store TOTP accounts in Casdoor, enable the **MFA accounts** setting on the Casdoor server.

![Image 11: Mfa account setting](https://casdoor.github.io/assets/images/mfa-account-setting-08f077e48bd79881635a03175db66d2c.png)

### Step 2: Connect to Casdoor[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#step-2-connect-to-casdoor "Direct link to Step 2: Connect to Casdoor")

After installing the app (and enabling MFA accounts if you use sync), connect in one of these ways:

| Method | Steps |
| --- | --- |
| **Manual** | Tap **Enter Server Manually**, enter server URL, client ID, and organization name, then sign in. |
| **QR code** | Tap **Scan QR Code**, scan the QR from **My Account** → **MFA accounts** on the Casdoor server. |
| **Demo** | Tap **Try Demo Server** to use the preconfigured demo instance. |

![Image 12: Login](https://casdoor.github.io/assets/images/login-d2525382e7dacf1477ee5423abe4427e.png)

You can then view TOTP codes and manage 2FA accounts in the app.

Migration from other authenticators[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#migration-from-other-authenticators "Direct link to Migration from other authenticators")
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### From Google Authenticator[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#from-google-authenticator "Direct link to From Google Authenticator")

In Google Authenticator: **Menu** → **Transfer accounts** → select accounts → **Export** (QR code). In Casdoor Authenticator, scan that QR to import.

![Image 13: Export TOTP data](https://casdoor.github.io/assets/images/google-export-3806b66cfeb2eaa4a99900044f356db9.png)![Image 14: Import TOTP data from Google Authenticator](https://casdoor.github.io/assets/images/import-totp-google-23a1610fcf2745727dbd99e53fa68dfe.gif)

### From Microsoft Authenticator (Android, root required)[​](https://casdoor.github.io/docs/how-to-connect/totp-authenticator-app#from-microsoft-authenticator-android-root-required "Direct link to From Microsoft Authenticator (Android, root required)")

1.   On the device with Microsoft Authenticator, the app data is under `/data/data/com.azure.authenticator/databases/`. Root access is required to read it.
2.   Copy the `PhoneFactor` database file.
3.   In Casdoor Authenticator: import → **Import from Microsoft Authenticator** → select the `PhoneFactor` file.
4.   The app imports the TOTP accounts.

![Image 15: Import TOTP data from Microsoft Authenticator](https://casdoor.github.io/assets/images/import-totp-microsoft-e9de43b1dbcc803c7d0f0dea2600336d.gif)
