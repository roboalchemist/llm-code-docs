### Trapezoid 

#### Properties 

- ::: 
  [[x](#x)][Number]

  The x-coordinate of top left point of the trapezoid.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[y](#y)][Number]

  The y-coordinate of top left point of the trapezoid.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[upperWidth](#upperWidth)][Number]

  The width of the upper horizontal side of the trapezoid.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[lowerWidth](#lowerWidth)][Number]

  The width of the lower horizontal side of the trapezoid.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[height](#height)][Number]

  The height of trapezoid.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[isUpdateAnimationActive](#isUpdateAnimationActive)][Boolean]

  If set to true, trapezoid will update and render with a gradual fade-in animation from left to right.

  [DEFAULT: ]`false`
  :::
