### Rectangle 

#### Properties 

- ::: 
  [[x](#x)][Number]

  The x-coordinate of top left point of the rectangle.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[y](#y)][Number]

  The y-coordinate of top left point of the rectangle.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[width](#width)][Number]

  The width of rectangle.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[height](#height)][Number]

  The height of rectangle.

  [DEFAULT: ]`0`
  :::

- ::: 
  [[radius](#radius)][Number]

  If set a value, the option is the radius of all the rounderd corners. If set a array, the option are in turn the radiuses of top-left corner, top-right corner, bottom-right corner, bottom-left corner.

  [DEFAULT: ]`0`
  :::
