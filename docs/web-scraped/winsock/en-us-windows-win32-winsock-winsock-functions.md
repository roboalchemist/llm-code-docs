[**accept**](https://learn.microsoft.com/en-us/windows/win32/api/Winsock2/nf-winsock2-accept)Permits an incoming connection attempt on a socket.
[**AcceptEx**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/nf-mswsock-acceptex)Accepts a new connection, returns the local and remote address, and receives the first block of data sent by the client application.
[**bind**](https://learn.microsoft.com/en-us/windows/win32/api/winsock/nf-winsock-bind)Associates a local address with a socket.

[**connect**](https://learn.microsoft.com/en-us/windows/win32/api/Winsock2/nf-winsock2-connect)Establishes a connection to a specified socket.
[**ConnectEx**](https://learn.microsoft.com/en-us/windows/win32/api/Mswsock/nc-mswsock-lpfn_connectex)Establishes a connection to a specified socket, and optionally sends data once the connection is established. Only supported on connection-oriented sockets.
[**DisconnectEx**](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/legacy/ms737757(v=vs.85))Closes a connection on a socket, and allows the socket handle to be reused.

[**GetAddressByName**](https://learn.microsoft.com/en-us/windows/win32/api/Nspapi/nf-nspapi-getaddressbynamea)Queries a namespace, or a set of default namespaces, to retrieve network address information for a specified network service. This process is known as service name resolution. A network service can also use the function to obtain local address information that it can use with the [**bind**](https://learn.microsoft.com/en-us/windows/win32/api/winsock/nf-winsock-bind) function.
