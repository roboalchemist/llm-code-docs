Constructor [#](https://docs.kuzzle.io/sdk/jvm/1#constructor)
=============================================================

Use this constructor to create a new instance of the SDK. Each instance represents a different connection to a Kuzzle server with specific options.

:::: tabs ::: tab Java

Arguments [#](https://docs.kuzzle.io/sdk/jvm/1#arguments)
---------------------------------------------------------

```
public Kuzzle(
  AbstractProtocol networkProtocol,
  bool subscribeToSelf
) throws IllegalArgumentException

public Kuzzle(
  AbstractProtocol networkProtocol
) throws IllegalArgumentException
```

| Argument | Type | Description |
| --- | --- | --- |
| `networkProtocol` | AbstractProtocol | Protocol used by the SDK instance |
| `autoResubscribe` | boolean (`true`) | Class which contains options |

networkProtocol [#](https://docs.kuzzle.io/sdk/jvm/1#networkprotocol)
---------------------------------------------------------------------

The protocol used to connect to the Kuzzle instance. It can be one of the following available protocols:

*   [WebSocket](https://docs.kuzzle.io/sdk/java/3/protocols/websocket)

Return [#](https://docs.kuzzle.io/sdk/jvm/1#return)
---------------------------------------------------

The `Kuzzle` SDK instance.

Throws [#](https://docs.kuzzle.io/sdk/jvm/1#throws)
---------------------------------------------------

Can throw an [IllegalArgumentException](https://docs.kuzzle.io/sdk/java/3/exceptions/illegal-argument-exception)

Usage [#](https://docs.kuzzle.io/sdk/jvm/1#usage)
-------------------------------------------------

`Kuzzle sdk = new Kuzzle(new WebSocket("kuzzle"));`

`val sdk: Kuzzle = Kuzzle(WebSocket("kuzzle"))`
