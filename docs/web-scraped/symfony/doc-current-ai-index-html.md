Symfony AI Documentation
========================

[Edit this page](https://github.com/symfony/ai/edit/main//docs/index.rst)

Symfony AI is a set of components that integrate AI capabilities into PHP applications, providing a unified interface to work with various AI platforms like OpenAI, Anthropic, Google Gemini, Azure, and more.

[Getting Started](https://symfony.com/doc/current/ai/index.html#getting-started "Permalink to this headline")
-------------------------------------------------------------------------------------------------------------

Symfony AI consists of several components and bundles that work together to bring AI capabilities to your application:

* [Platform Component](https://symfony.com/doc/current/ai/components/platform.html): Unified interface to various AI models and providers
* [Agent Component](https://symfony.com/doc/current/ai/components/agent.html): Framework for building AI agents with tools and workflows
* [Chat Component](https://symfony.com/doc/current/ai/components/chat.html): API to interact with agents and store conversation history
* [Store Component](https://symfony.com/doc/current/ai/components/store.html): Data storage abstraction for vector databases and RAG applications
* [Mate Component](https://symfony.com/doc/current/ai/components/mate.html): MCP server for AI assistant integration with your application
* [AI Bundle](https://symfony.com/doc/current/ai/bundles/ai-bundle.html): Symfony integration bringing all components together
* [MCP Bundle](https://symfony.com/doc/current/ai/bundles/mcp-bundle.html): Integration for the Model Context Protocol SDK

[Quick Start](https://symfony.com/doc/current/ai/index.html#quick-start "Permalink to this headline")
-----------------------------------------------------------------------------------------------------

```
namespace App\Agent;

use Symfony\AI\Agent\AgentInterface;
use Symfony\AI\Platform\Message\Message;
use Symfony\AI\Platform\Message\MessageBag;

final readonly class ChatService
{
    public function __construct(
        private AgentInterface $agent,
    ) {
    }

    public function chat(string $userMessage): string
    {
        $messages = new MessageBag(
            Message::forSystem('You are a helpful assistant.'),
            Message::ofUser($userMessage),
        );

        $result = $this->agent->call($messages);

        return $result->getContent();
    }
}
```

[Key Features](https://symfony.com/doc/current/ai/index.html#key-features "Permalink to this headline")
-------------------------------------------------------------------------------------------------------

**Multi-Platform Support** Connect to OpenAI, Anthropic Claude, Google Gemini, Azure OpenAI, AWS Bedrock, Mistral, Ollama, and many more platforms with a unified interface. **Tool Calling** Extend your agents with custom tools to interact with external APIs, databases, and services. Built-in tools available for common tasks. **Retrieval Augmented Generation (RAG)** Build context-aware applications using vector stores with support for ChromaDB, Pinecone, Weaviate, MongoDB Atlas, and more. **Structured Output** Extract structured data from unstructured text using PHP classes or array schemas. **Multi-Modal Support** Process text, images, audio, and other content types with compatible models. **Streaming Support** Stream responses from AI models for real-time user experiences. **Memory Management** Add contextual memory to agents for personalized conversations. **Testing Tools** Mock agents and platforms for reliable testing without external API calls.
