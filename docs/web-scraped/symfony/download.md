Download Symfony
================

[Step 1. Install Symfony CLI](https://symfony.com/download#step-1-install-symfony-cli "Permanent link to this headline")
------------------------------------------------------------------------------------------------------------------------

The Symfony CLI is a developer tool to help you build, run, and manage your Symfony applications directly from your terminal. It's Open-Source, works on macOS, Windows, and Linux, and you only have to install it once in your system. [Learn more about Symfony CLI](https://symfony.com/doc/current/setup/symfony_server.html).

* Docker
* Linux
* macOS
* Windows
* [Source Code](https://github.com/symfony-cli/symfony-cli)[Changelog](https://github.com/symfony-cli/symfony-cli/releases)

To add the Symfony CLI binary to existing projects that already use Docker, update your `Dockerfile` with the following instruction:

```
COPY --link \
    --from=ghcr.io/symfony-cli/symfony-cli:latest \
    /usr/local/bin/symfony /usr/local/bin/symfony
```

Choose one of the following installation methods:

Installer — via `wget` or `curl`

 Copy Copy

`$ curl -sS https://get.symfony.com/cli/installer | bash`

 Copy Copy

[Step 2. Create New Symfony Applications](https://symfony.com/download#step-2-create-new-symfony-applications "Permanent link to this headline")
------------------------------------------------------------------------------------------------------------------------------------------------

Use the `symfony` binary provided by the Symfony CLI to create new applications:

If you are building a **traditional web application**:

 Copy Copy

`$ symfony new --webapp my_project`

If you are building a **microservice**, **console application** or **API**:

 Copy Copy

`$ symfony new my_project`

You can also read Symfony Docs to learn about [installing Symfony with Composer](https://symfony.com/doc/current/setup.html).
