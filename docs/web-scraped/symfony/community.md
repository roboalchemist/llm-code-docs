Symfony is committed to foster an open and welcoming environment for everyone.

* Read the [Symfony Code of Conduct](https://symfony.com/doc/current/contributing/code_of_conduct/code_of_conduct.html) which governs all the official Symfony code repositories, activities and events.

* Take part in the [Symfony Diversity Initiative](https://github.com/symfony/diversity), a repository of ideas to improve diversity.
* Follow the [diversity news and updates](https://symfony.com/blog/category/diversity) from the official Symfony blog.
