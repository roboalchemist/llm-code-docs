**How are these stats obtained?**

First, the total and monthly downloads are obtained for each package through the [Packagist.org API](https://packagist.org/apidoc). Then, the monthly downloads are used to estimate the average daily downloads and the downloads per second. Lastly, these calculated averages are used to simulate the highly precise, but not perfect, real-time updates.
