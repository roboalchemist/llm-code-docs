Symfony Packages
================

Symfony Packages are decoupled libraries for PHP applications. Battle-tested in hundreds of thousands of projects and downloaded [billions of times](https://symfony.com/stats/downloads), they're the foundation of [the most important PHP projects](https://symfony.com/projects).

[Installation](https://symfony.com/packages#installation "Permanent link to this headline")
-------------------------------------------------------------------------------------------

Use [Composer](https://getcomposer.org/) to install any of the Symfony Packages in your PHP project:

$ composer require
