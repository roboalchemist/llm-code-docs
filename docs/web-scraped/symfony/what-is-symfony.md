What is Symfony
===============

Symfony is a set of PHP Packages, a Web Application framework, a Philosophy, and a Community — all working together in harmony.

### [Symfony Framework](https://symfony.com/what-is-symfony#symfony-framework "Permanent link to this headline")

The leading PHP framework to create websites and web applications. Built on top of the Symfony Packages.

### [Symfony Packages](https://symfony.com/packages)

A set of decoupled and reusable packages on which the best PHP applications are built, such as Drupal, Prestashop, and Laravel.

### [Symfony Community](https://symfony.com/community)

A passionate group of over 600,000 developers from more than 120 countries, all committed to helping PHP surpass the impossible.

### [Symfony Philosophy](https://symfony.com/what-is-symfony#symfony-philosophy "Permanent link to this headline")

Embracing and promoting professionalism, best practices, standardization and interoperability of applications.
