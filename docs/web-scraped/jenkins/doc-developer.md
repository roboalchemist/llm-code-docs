Extend Jenkins
==============

Jenkins is an extensible automation server with more than 2000 plugins providing integrations for hundreds of tools and services. The resources below will teach you how to _extend_ Jenkins with new features.

If you want to _use_ Jenkins's existing functionality and plugin features only, please refer to the [Jenkins User Documentation](https://www.jenkins.io/doc) instead.
