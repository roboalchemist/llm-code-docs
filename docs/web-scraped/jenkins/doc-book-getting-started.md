This page provides an overview of the documentation in the Jenkins User Handbook.

If you want to get up and running with Jenkins, see [Installing Jenkins](https://www.jenkins.io/doc/book/installing) for procedures on how to install Jenkins on your supported platform of choice.

If you are a typical Jenkins user (of any skill level) who wants to know more about Jenkins usage, see [Using Jenkins](https://www.jenkins.io/doc/book/using). Also refer to the separate [Pipeline](https://www.jenkins.io/doc/book/pipeline) and [Blue Ocean](https://www.jenkins.io/doc/book/blueocean) chapters for more information about these core Jenkins features.

If you are a Jenkins administrator and want to know more about managing Jenkins nodes and instances, see [Managing Jenkins](https://www.jenkins.io/doc/book/managing).

If you are a system administrator and want to learn how to back-up, restore, maintain as Jenkins servers and nodes, see [Jenkins System Administration](https://www.jenkins.io/doc/book/system-administration).
