This chapter contains topics for typical Jenkins users (of all skill levels) about Jenkins usage which is outside the scope of the core Jenkins features: Pipeline and Blue Ocean.

If you want to create and configure a Pipeline project through a `Jenkinsfile` or through Blue Ocean, or you wish to find out more about these core Jenkins features, refer to the relevant topics within the respective [Pipeline](https://www.jenkins.io/doc/book/pipeline) and [Blue Ocean](https://www.jenkins.io/doc/book/blueocean) chapters.

If you are a Jenkins administrator and want to know more about managing Jenkins nodes and instances, see [Managing Jenkins](https://www.jenkins.io/doc/book/managing).

If you are a system administrator and want to learn how to back-up, restore, maintain as Jenkins servers and nodes, see [Jenkins System Administration](https://www.jenkins.io/doc/book/system-administration).

If you are a Jenkins user looking for some troubleshooting tips, see [Troubleshooting Jenkins](https://www.jenkins.io/doc/book/troubleshooting)

For an overview of content in the Jenkins User Handbook, see [User Handbook Overview](https://www.jenkins.io/doc/book/getting-started).
