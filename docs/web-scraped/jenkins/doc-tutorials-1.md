Tutorials overview
==================

This part of the Jenkins User Documentation contains a series of introductory tutorials to help you begin building your applications in an automated fashion with Jenkins.

If you’re a developer who wants to improve your understanding of Continuous Integration (CI) / Continuous Delivery (CD) concepts, or you might already be familiar with these concepts but don’t yet know how to implement them in Jenkins, then these tutorials are a great place to start.

[](https://www.jenkins.io/doc/tutorials/#getting-started)Getting started with Jenkins
-------------------------------------------------------------------------------------

*   [Guided Tour to Jenkins](https://www.jenkins.io/doc/pipeline/tour/getting-started)

[](https://www.jenkins.io/doc/tutorials/#pipeline)Pipeline
----------------------------------------------------------

The following tutorials show how to use key features of Jenkins to facilitate implementing CI/CD processes to build your applications:

*   Sending Notifications in Pipeline ([Declarative Pipeline](https://www.jenkins.io/blog/2017/02/15/declarative-notifications/), [Scripted Pipeline](https://www.jenkins.io/blog/2016/07/18/pipeline-notifications/))

*   [End-to-End Multibranch Pipeline Project Creation](https://www.jenkins.io/doc/tutorials/build-a-multibranch-pipeline-project)

*   [Creating a shared library](https://www.jenkins.io/blog/2017/02/15/declarative-notifications/#moving-notifications-to-shared-library)

*   [Faster Pipelines with the Parallel Test Executor Plugin](https://www.jenkins.io/blog/2016/06/16/parallel-test-executor-plugin/)

*   [Pipeline Development Tools](https://www.jenkins.io/blog/2017/05/18/pipeline-dev-tools/)
