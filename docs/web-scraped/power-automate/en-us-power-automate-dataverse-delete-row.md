Use a flow to delete a row from Dataverse
=========================================

Feedback

 Summarize this article for me 

In this article
---------------

Follow these steps to delete a row from Dataverse.

1.   Create a flow with the **When a new email arrives (V3)** trigger to your flow.

2.   Select **New step** to add an action to your flow.

3.   Enter **delete row** into the **Search connectors and actions** search box on the **Choose an operation** card.

4.   Select **Microsoft Dataverse**.

![Image 5: Filter to delete row actions.](https://learn.microsoft.com/en-us/power-automate/media/dataverse-how-tos/delete-row-action.png)

5.   Select **Delete a new row** action.

![Image 6: Delete a row card.](https://learn.microsoft.com/en-us/power-automate/media/dataverse-how-tos/delete-row-card.png)

6.   Select the table name, and then enter an ID in **Row ID**.

![Image 7: Set the table name and row ID.](https://learn.microsoft.com/en-us/power-automate/media/dataverse-how-tos/delete-row-card-complete.png)

The **Row ID** column is the unique ID of the row that you are deleting.

Tip

You can retrieve the **Row ID** by using the dynamic content that's generated from earlier steps in your flow.
