Run a cloud flow on a schedule
==============================

Create a cloud flow that performs one or more tasks such as sending a report in email.

*   Once a day, an hour, or a minute.
*   On a date that you specify.
*   After many days, hours, or minutes that you specify.

[](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#create-a-scheduled-cloud-flow)
Create a scheduled cloud flow
-----------------------------

You can create a scheduled cloud flow using natural language in Copilot, or create it from scratch. If you have access to Copilot, select the **Using copilot** tab to create your flow. If you don't have access to Copilot, select the **Without copilot** tab.

*   [Using copilot](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#tabpanel_1_using-copilot)
*   [Without copilot](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#tabpanel_1_without-copilot)

You can ask Copilot to create a scheduled cloud flow.

1.   Sign in to [Power Automate](https://make.powerautomate.com/).

2.   On the navigation pane to the left, make sure that **Home** is selected.

3.   In the **Create your automation with Copilot** field, type the following prompt:

 Copilot prompt Copy  
```copilot-prompt
Create a flow that runs Monday every week starting 04/14/2025 which sends an email to contoso@gmail.com that their MPR doc is due.**
```
4.   Select **Generate**.

![Image 10: Screenshot of a prompt to create a scheduled flow.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/copilot-generate.png)

5.   Copilot generates a flow based on your prompt. You can review the generated flow and make any necessary adjustments. If you're satisfied with the suggested flow, select **Keep it and continue**.

![Image 11: Screenshot of a generated flow.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/copilot-generate-keep-it.png)

6.   Review the connected apps and services. A green checkmark means the connection is ready to go.

If you don't have a green checkmark, select the connection to set it up.

7.   Select **Create flow**.

[](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#use-copilot-to-configure-actions)
### Use Copilot to configure actions

1.   In the designer, select **Copilot**.

2.   In the Copilot panel, ask Copilot to make changes to your scheduled flow. For example, you can enter the following prompt:

 Copilot prompt Copy  
```copilot-prompt
Change the interval from every week to 2 weeks.
```

After Copilot generates a response, it confirms that it made the update successfully. If you change your mind, you can select **Undo** to revert the changes.

![Image 12: Screenshot of a prompt to create a scheduled flow in the Copilot pane.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/copilot-designer-prompt.png)

3.   If you're not using Copilot to configure your actions, go to [Configure cloud flow triggers and actions](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#configure-cloud-flow-triggers-and-actions).

1.   Sign in to [Power Automate](https://make.powerautomate.com/).

2.   Select **My flows**>**New flow**>**Scheduled cloud flow**.

3.   In the fields next to **Starting**, specify the date and time when your flow should start.

4.   In the fields next to **Repeat every**, specify the flow's recurrence.

5.   Select **Create**.

![Image 13: Screenshot of the options to set up a scheduled flow.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/select-recurrence-aa.png)

6.   Go to [Configure cloud flow triggers and actions](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#configure-cloud-flow-triggers-and-actions).

[](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#configure-cloud-flow-triggers-and-actions)
Configure cloud flow triggers and actions
-----------------------------------------

Power Automate allows you to use either the [new designer](https://learn.microsoft.com/en-us/power-automate/flows-designer) or the [classic designer](https://learn.microsoft.com/en-us/power-automate/flows-designer) to configure your cloud flow. The steps are similar in both designers. Learn more (with examples) in [Identify differences between the new designer and the classic designer](https://learn.microsoft.com/en-us/power-automate/flows-designer#identify-differences-between-the-new-designer-and-the-classic-designer).

1.   Make sure you [Created a scheduled cloud flow](https://learn.microsoft.com/en-us/power-automate/run-scheduled-tasks#create-a-scheduled-cloud-flow).

2.   Select the trigger or action to configure.

3.   Configure the parameters.

    *   New designer: In the designer, select the trigger or action. The configuration pane opens on the left side of the screen.
    *   Classic designer: In the designer, select **Recurrence**>**Show advanced options**. When you select **Show advanced options**, the dropdown name changes to **Hide advanced options**.

Expand table

| New designer | Classic designer |
| --- | --- |
| ![Image 14: Screenshot of a prompt to create a scheduled flow in the action configuration pane.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/select-recurrence-new.png) | ![Image 15: Screenshot of show/hide advanced options.](https://learn.microsoft.com/en-us/power-automate/media/run-scheduled-tasks/select-recurrence1.png) |

4.   In the **Time zone** field, select a time zone from the dropdown list to specify whether the **Start time** reflects a local time zone, Coordinated Universal Time (UTC), or other time zone.

5.   In the **Start time** field, enter a start time in this format: YYYY-MM-DDTHH:MM:SSZ.

    *   YYYY = four-digit year
    *   MM = two-digit month (01-12)
    *   DD = two-digit day of the month (01-31)
    *   T = literal character that separates the date and time
    *   HH = two-digit hour in 24-hour format (00-23)
    *   MM = two-digit minute (00-59)
    *   SS = two-digit second (00-59)
    *   Z = literal character that indicates UTC time

For example, enter **2025-03-24T22:00:00Z** to specify 3:00 PM Pacific time on March 24, 2025.

6.   If you specified **Day** under **Frequency**, select the time of day when the flow should run in the **At these hours** and **At these minutes** dropdown lists.

7.   If you specified **Week** under **Frequency**, do the following.

    *   In the **On these days** and **At these hours** dropdown menus, select the day or days of the week on which the flow should run and the time or times of day when the flow should run.
    *   In the **At these minutes** field, enter the minute values (from 0 to 59) separated by a comma.

If you run a flow on the month frequency, the flow runs on the same date each month.

8.   Add the action or actions that you want the flow to take, as described in [Create a cloud flow from scratch](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow).
