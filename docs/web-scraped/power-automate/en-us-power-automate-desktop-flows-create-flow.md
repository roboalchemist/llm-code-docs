Using Power Automate, you can develop desktop flows that automate tasks on your desktop and the Web. This section presents all the available ways to create desktop flows and start designing your automations.

[](https://learn.microsoft.com/en-us/power-automate/desktop-flows/create-flow#start-creating-desktop-flows-using-examples)
Start creating desktop flows using examples
-------------------------------------------

Power Automate provides an extensive collection of examples to introduce users to RPA. You can find all the available examples under the **Examples** tab in the console.

![Image 11: Screenshot of the Examples tab in the console.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/console-examples.png)

To copy an example to your flows, right-click the example and select **Create a copy**. The copied flow will be stored under the **My flows** tab.

![Image 12: Screenshot of the Create a copy option of an example.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/create-copy-example.png)

If you've opened and edited an example in the flow designer, select **Save as** to copy the edited example to your flows. The changes can't be saved to the original example flow.

![Image 13: Screenshot of the Save as button in an open example.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/save-as-example.png)

[](https://learn.microsoft.com/en-us/power-automate/desktop-flows/create-flow#create-desktop-flows-through-the-console)
Create desktop flows through the console
----------------------------------------

To create a desktop flow using Power Automate for desktop:

1.   Launch the application and select **New flow** in the console.

![Image 14: Screenshot of the New flow button in the console.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/console.png)

2.   Enter a name for the desktop flow, and select **Create**.

![Image 15: Screenshot of the Build a flow dialog.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/build-flow-dialog.png)

3.   Design your flow in the flow designer and select **Save**. Close the flow designer and the flow will appear in the **My flows** tab of the console.

![Image 16: Screenshot of a desktop flow in the console.](https://learn.microsoft.com/en-us/power-automate/desktop-flows/media/create-flow/console-flow.png)

[](https://learn.microsoft.com/en-us/power-automate/desktop-flows/create-flow#create-desktop-flows-through-cloud-flows)
Create desktop flows through cloud flows
----------------------------------------

Power Automate enables you to create desktop flows through cloud flows and trigger them remotely. To find more information about this functionality, go to [Trigger desktop flows from cloud flows](https://learn.microsoft.com/en-us/power-automate/desktop-flows/trigger-desktop-flows).
