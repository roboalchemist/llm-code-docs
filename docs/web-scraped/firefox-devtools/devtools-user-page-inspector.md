Title: Page Inspector — Firefox Source Docs documentation

URL Source: https://firefox-source-docs.mozilla.org/devtools-user/page_inspector

Markdown Content:
Use the Page Inspector to examine and modify the HTML and CSS of a page.

You can examine pages loaded in the local copy of Firefox or in a remote target such as Firefox for Android. See [about debugging](https://firefox-source-docs.mozilla.org/devtools-user/about_colon_debugging/index.html) to learn how to connect the developer tools to a remote target.

User Interface Tour[](https://firefox-source-docs.mozilla.org/devtools-user/page_inspector#user-interface-tour "Link to this heading")
---------------------------------------------------------------------------------------------------------------------------------------

To find your way around the Inspector, here’s a [quick tour of the UI](https://firefox-source-docs.mozilla.org/devtools-user/page_inspector/ui_tour/index.html).

You can split the Rules view out into its own pane, separate from the other tabs on the CSS pane — this is called [3-pane mode](https://firefox-source-docs.mozilla.org/devtools-user/page_inspector/3-pane_mode/index.html).
