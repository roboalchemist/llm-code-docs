DOM Property Viewer[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#dom-property-viewer "Link to this heading")
=======================================================================================================================================================

The DOM Property Viewer is disabled by default. Enable it in the [Developer Tools Settings](https://firefox-source-docs.mozilla.org/devtools-user/settings/index.html#tool-toolbox-settings)

The DOM Property Viewer lets you inspect the properties of the [DOM](https://developer.mozilla.org/en-US/docs/Glossary/DOM) as an expandable tree structure, starting from the [window](https://developer.mozilla.org/en-US/docs/Web/API/Window) object of the current page or the [selected iframe](https://firefox-source-docs.mozilla.org/devtools-user/working_with_iframes/index.html).

![Image 5: ../../_images/dom_inspector.png](https://firefox-source-docs.mozilla.org/_images/dom_inspector.png)
Enabling the DOM Property Viewer[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#enabling-the-dom-property-viewer "Link to this heading")
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The DOM Property Viewer is not enabled by default. To enable it, open the [developer tools settings](https://firefox-source-docs.mozilla.org/devtools-user/settings/index.html#tool-toolbox-settings) and check the “DOM” box under “Default Firefox Developer Tools”.

Opening the DOM Property Viewer[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#opening-the-dom-property-viewer "Link to this heading")
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Once enabled, you can select the _DOM_ panel in the Web Developer Tools, accessible from the Browser Tools submenu

The [Toolbox](https://firefox-source-docs.mozilla.org/devtools-user/tools_toolbox/index.html) will appear at the bottom of the browser window, with the DOM Property Viewer activated. It’s just called “DOM” in the Toolbox.

DOM Property Viewer user interface[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#dom-property-viewer-user-interface "Link to this heading")
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### DOM tree[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#dom-tree "Link to this heading")

The different properties of the DOM are displayed as an expandable tree. The left-hand side shows the property’s name, and the right-hand side shows its value. Up to three properties of an object and items of an array are displayed. If a property has more elements than this, you’ll see a “more…” annotation, and will need to click the property to see all elements. A lock icon indicates that a property is not writable.

### Refreshing the display[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#refreshing-the-display "Link to this heading")

If the DOM changes you can hit the _Refresh_ button to update the display:

![Image 6: Button to update the DOM Inspector display](https://firefox-source-docs.mozilla.org/_images/dom_inspector_refresh_button.png)
### Filtering[](https://firefox-source-docs.mozilla.org/devtools-user/dom_property_viewer/index.html#filtering "Link to this heading")

There is a search box within the toolbar:

![Image 7: ../../_images/dom_inspector_search_box.png](https://firefox-source-docs.mozilla.org/_images/dom_inspector_search_box.png)
This filters the list to show only items which match the search term. Items match the search term if their name contains the search term. Matching is case-sensitive.
