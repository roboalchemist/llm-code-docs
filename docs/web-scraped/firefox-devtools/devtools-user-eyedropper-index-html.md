Eyedropper[](https://firefox-source-docs.mozilla.org/devtools-user/eyedropper/index.html#eyedropper "Link to this heading")
============================================================================================================================

The Eyedropper tool enables you to select colors in the current page. It works like a magnifying glass over the page, enabling you to select with pixel precision. Underneath the magnifying glass it shows the color value for the current pixel using whichever scheme you’ve selected in [Settings > Inspector](https://firefox-source-docs.mozilla.org/devtools-user/settings/index.html#settings-inspector)> Default color unit:

![Image 3: ../../_images/eyedropper.png](https://firefox-source-docs.mozilla.org/_images/eyedropper.png)
You can use it in one of two ways:

*   to select a color from the page and copy it to the clipboard

*   to change a color value in the Inspector’s Rules view to a color you’ve selected from the page

Copying a color to the clipboard[](https://firefox-source-docs.mozilla.org/devtools-user/eyedropper/index.html#copying-a-color-to-the-clipboard "Link to this heading")
------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Open the Eyedropper in one of these two ways:

*   select “Eyedropper” under the “Browser Tools” menu

*   open the [Page Inspector](https://firefox-source-docs.mozilla.org/devtools-user/page_inspector/index.html) tab and click the eyedropper button in its toolbar

As you move the mouse around the page you’ll see the current color value in the Eyedropper change. Clicking copies the current color value to the clipboard.

Changing a color value in the Rules view[](https://firefox-source-docs.mozilla.org/devtools-user/eyedropper/index.html#changing-a-color-value-in-the-rules-view "Link to this heading")
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Color values appearing in the Inspector’s Rules view have color samples next to them: clicking the sample shows a [color picker popup](https://firefox-source-docs.mozilla.org/devtools-user/page_inspector/how_to/inspect_and_select_colors/index.html). The popup contains an eyedropper icon: click this icon to activate the Eyedropper.

Now, when you click the Eyedropper, the color in the Rules view is set to the color you selected.

Keyboard shortcuts[](https://firefox-source-docs.mozilla.org/devtools-user/eyedropper/index.html#keyboard-shortcuts "Link to this heading")
--------------------------------------------------------------------------------------------------------------------------------------------

See [All keyboard shortcuts > Eyedropper](https://firefox-source-docs.mozilla.org/devtools-user/keyboard_shortcuts/index.html#keyboard-shortcuts-eyedropper).
