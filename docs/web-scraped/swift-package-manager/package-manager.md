Swift Package Manager
=====================

Organize, manage, and edit Swift packages.

[Overview](https://www.swift.org/swiftpm/documentation/packagemanagerdocs/#Overview)
------------------------------------------------------------------------------------

The Swift Package Manager lets you share your code as a package, depend on and use other shared packages, as well as build, test, document, and run your code.

Note

Swift Package Manager is previewing the Swift Build build system. Learn more [here](https://www.swift.org/swiftpm/documentation/packagemanagerdocs/swiftbuildpreview).
