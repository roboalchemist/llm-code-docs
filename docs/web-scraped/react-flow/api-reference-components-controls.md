The `<Controls />` component renders a small panel that contains convenient buttons to zoom in, zoom out, fit the view, and lock the viewport.

``` 
import  from '@xyflow/react'
 
export default function Flow()  edges=>
      <Controls />
    </ReactFlow>
  )
}
```
