appstruct[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-appstruct "Link to this term")
A raw application data structure (a structure of complex Python objects), passed to the [`colander.SchemaNode.serialize()`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.SchemaNode.serialize "colander.SchemaNode.serialize") method for serialization. The [`colander.SchemaNode.deserialize()`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.SchemaNode.deserialize "colander.SchemaNode.deserialize") method accepts a [cstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-cstruct) and returns an appstruct.

cstruct[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-cstruct "Link to this term")
A data structure generated by the [`colander.SchemaNode.serialize()`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.SchemaNode.serialize "colander.SchemaNode.serialize") method, capable of being consumed by the [`colander.SchemaNode.deserialize()`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.SchemaNode.deserialize "colander.SchemaNode.deserialize") method.

schema[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-schema "Link to this term")
A nested collection of [schema node](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-schema-node) objects representing an arrangement of data.

schema node[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-schema-node "Link to this term")
A schema node is an object which can serialize an [appstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-appstruct) to a [cstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-cstruct) and deserialize a [appstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-appstruct) from a [cstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-cstruct) an (object derived from [`colander.SchemaNode`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.SchemaNode "colander.SchemaNode") or one of the colander Schema classes).

type[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-type "Link to this term")
An object representing a particular type of data (mapping, boolean, string, etc) capable of serializing an [appstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-appstruct) and of deserializing a [cstruct](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-cstruct). Colander has various built-in types ([`colander.String`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.String "colander.String"), [`colander.Mapping`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.Mapping "colander.Mapping"), etc) and may be extended with additional types (see [Defining a New Type](https://docs.pylonsproject.org/projects/colander/en/latest/extending.html#defining-a-new-type)).

validator[¶](https://docs.pylonsproject.org/projects/colander/en/latest/glossary.html#term-validator "Link to this term")
A Colander validator callable. Accepts a `node` object and a `value` and either raises a [`colander.Invalid`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.Invalid "colander.Invalid") exception or returns `None`. Used as the `validator=` argument to a schema node, ensuring that the input meets the requirements of the schema. Built-in validators exist in Colander (e.g. [`colander.OneOf`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.OneOf "colander.OneOf"), [`colander.Range`](https://docs.pylonsproject.org/projects/colander/en/latest/api.html#colander.Range "colander.Range"), etc), and new validators can be defined to extend Colander (see [Defining a New Validator](https://docs.pylonsproject.org/projects/colander/en/latest/extending.html#defining-a-new-validator)).
