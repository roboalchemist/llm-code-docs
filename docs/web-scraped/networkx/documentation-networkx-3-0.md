NetworkX is a Python package for the creation, manipulation, and study of the structure, dynamics, and functions of complex networks. It provides:

*   tools for the study of the structure and dynamics of social, biological, and infrastructure networks;

*   a standard programming interface and graph implementation that is suitable for many applications;

*   a rapid development environment for collaborative, multidisciplinary projects;

*   an interface to existing numerical algorithms and code written in C, C++, and FORTRAN; and

*   the ability to painlessly work with large nonstandard data sets.

With NetworkX you can load and store networks in standard and nonstandard data formats, generate many types of random and classic networks, analyze network structure, build network models, design new network algorithms, draw networks, and much more.

The audience for NetworkX includes mathematicians, physicists, biologists, computer scientists, and social scientists. Good reviews of the science of complex networks are presented in Albert and Barabási [[BA02]](https://networkx.github.io/documentation/networkx-3.0#ba02), Newman [[Newman03]](https://networkx.github.io/documentation/networkx-3.0#newman03), and Dorogovtsev and Mendes [[DM03]](https://networkx.github.io/documentation/networkx-3.0#dm03). See also the classic texts [[Bollobas01]](https://networkx.github.io/documentation/networkx-3.0#bollobas01), [[Diestel97]](https://networkx.github.io/documentation/networkx-3.0#diestel97) and [[West01]](https://networkx.github.io/documentation/networkx-3.0#west01) for graph theoretic results and terminology. For basic graph algorithms, we recommend the texts of Sedgewick (e.g., [[Sedgewick01]](https://networkx.github.io/documentation/networkx-3.0#sedgewick01) and [[Sedgewick02]](https://networkx.github.io/documentation/networkx-3.0#sedgewick02)) and the survey of Brandes and Erlebach [[BE05]](https://networkx.github.io/documentation/networkx-3.0#be05).

Python[#](https://networkx.github.io/documentation/networkx-3.0#python "Permalink to this heading")
---------------------------------------------------------------------------------------------------

Python is a powerful programming language that allows simple and flexible representations of networks as well as clear and concise expressions of network algorithms. Python has a vibrant and growing ecosystem of packages that NetworkX uses to provide more features such as numerical linear algebra and drawing. In order to make the most out of NetworkX you will want to know how to write basic programs in Python. Among the many guides to Python, we recommend the [Python documentation](https://docs.python.org/3/) and the text by Alex Martelli [[Martelli03]](https://networkx.github.io/documentation/networkx-3.0#martelli03).
