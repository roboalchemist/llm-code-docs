SObjectWidget
=============

The [SObjectWidget](API\Runtime\UMG\SObjectWidget) allows UMG to insert an [SWidget](API\Runtime\SlateCore\SWidget) into the hierarchy that manages the lifetime of the UMG [UWidget](API\Runtime\UMG\UWidget) that created it.

 On this page

Navigation
----------

[API](https://dev.epicgames.com/documentation/en-us/unreal-engine/API)>[API/Runtime](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime)>[API/Runtime/UMG](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG)

The [SObjectWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG/SObjectWidget) allows UMG to insert an [SWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/SlateCore/SWidget) into the hierarchy that manages the lifetime of the UMG [UWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG/UWidget) that created it. Once the [SObjectWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG/SObjectWidget) is destroyed it frees the reference it holds to The [UWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG/UWidget) allowing it to be garbage collected. It also forwards the slate events to the [UUserWidget](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/UMG/UUserWidget) so that it can forward them to listeners.
