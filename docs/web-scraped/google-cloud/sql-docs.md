# Cloud SQL documentation

[Read product documentation](https://docs.cloud.google.com/sql/docs/introduction)

Cloud SQL is a fully-managed database service that helps you set up, maintain, manage, and administer your relational databases on Google Cloud Platform.

You can use Cloud SQL with [MySQL](/sql/docs/mysql), [PostgreSQL](/sql/docs/postgres), or [SQL Server](/sql/docs/sqlserver).

Not sure what database option is right for you? Learn more about our [database services](https://cloud.google.com/products/databases).
