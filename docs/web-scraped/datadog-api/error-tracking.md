### OverviewSearch issues endpoint allows you to programmatically search for issues within your organization. This endpoint returns a list of issues that match a given search query, following the event search syntax. The search results are limited to a maximum of 100 issues per request.OAuth apps require the `error_tracking_read` authorization scope to access this endpoint.
