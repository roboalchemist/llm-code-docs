Class
SKRange
=======

A definition of a range of floating-point values.

iOS 8.0+iPadOS 8.0+Mac Catalyst 13.1+macOS 10.10+tvOS visionOS 1.0+watchOS 2.0+

`class SKRange`

[Overview](https://developer.apple.com/documentation/spritekit/skrange#overview)
--------------------------------------------------------------------------------

You typically use a [`SKRange`](https://developer.apple.com/documentation/spritekit/skrange) to clamp a value so that it is within the specified range.

[Topics](https://developer.apple.com/documentation/spritekit/skrange#topics)
----------------------------------------------------------------------------

### [Creating a Range Object](https://developer.apple.com/documentation/spritekit/skrange#Creating-a-Range-Object)

[`convenience init(value: CGFloat, variance: CGFloat)`](https://developer.apple.com/documentation/spritekit/skrange/init(value:variance:))

Creates and initializes a new range object using a value and a maximum distance from that value.

[`class func withNoLimits() -> Self`](https://developer.apple.com/documentation/spritekit/skrange/withnolimits())

Creates and initializes a new range object that encompasses all possible values.

[`convenience init(lowerLimit: CGFloat)`](https://developer.apple.com/documentation/spritekit/skrange/init(lowerlimit:))

Creates and initializes a new range object that specifies only a minimum value.

[`convenience init(upperLimit: CGFloat)`](https://developer.apple.com/documentation/spritekit/skrange/init(upperlimit:))

Creates and initializes a new range object that specifies only a maximum value.

[`convenience init(constantValue: CGFloat)`](https://developer.apple.com/documentation/spritekit/skrange/init(constantvalue:))

Creates and initializes a new range object that specifies a constant value.

[`init(lowerLimit: CGFloat, upperLimit: CGFloat)`](https://developer.apple.com/documentation/spritekit/skrange/init(lowerlimit:upperlimit:))

Initializes a new range object.

### [Inspecting a Range Object’s Limits](https://developer.apple.com/documentation/spritekit/skrange#Inspecting-a-Range-Objects-Limits)

[`var lowerLimit: CGFloat`](https://developer.apple.com/documentation/spritekit/skrange/lowerlimit)

The minimum possible value.

[`var upperLimit: CGFloat`](https://developer.apple.com/documentation/spritekit/skrange/upperlimit)

The maximum possible value.
