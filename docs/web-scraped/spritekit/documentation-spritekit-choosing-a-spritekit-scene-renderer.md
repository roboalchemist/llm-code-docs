Choosing a SpriteKit Scene Renderer
===================================

Compare the different ways to display a SpriteKit scene.

[Overview](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#overview)
------------------------------------------------------------------------------------------------------------

To display SpriteKit content in a watch app, you use [`WKInterfaceSKScene`](https://developer.apple.com/documentation/WatchKit/WKInterfaceSKScene), whereas on iOS, macOS, and tvOS, you use [`SKView`](https://developer.apple.com/documentation/spritekit/skview) to display a SpriteKit scene. If you need to mix SpriteKit with any custom Metal content, use [`SKRenderer`](https://developer.apple.com/documentation/spritekit/skrenderer).

Important

SKRenderer is not available on watchOS.

### [On iOS, tvOS, and macOS, Draw Using a View or a Renderer](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#On-iOS-tvOS-and-macOS-Draw-Using-a-View-or-a-Renderer)

Choose from the options below to display SpriteKit content on iOS, tvOS, or macOS.

#### [Draw Using a View](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#Draw-Using-a-View)

To draw SpriteKit content the most common way, set up an [`SKView`](https://developer.apple.com/documentation/spritekit/skview) and set its [`scene`](https://developer.apple.com/documentation/spritekit/skview/scene) property. See [Drawing SpriteKit Content in a View](https://developer.apple.com/documentation/spritekit/drawing-spritekit-content-in-a-view) for an example of using `SKView`.

#### [Draw Using a Renderer](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#Draw-Using-a-Renderer)

When you want to mix SpriteKit content with your own Metal content, use [`SKRenderer`](https://developer.apple.com/documentation/spritekit/skrenderer) to draw your [`SKScene`](https://developer.apple.com/documentation/spritekit/skscene). You might use `SKRenderer` to replace specific portions of a SpriteKit app with custom Metal code, or vice versa. `SKRenderer` combines the full control of Metal with the ease of use of SpriteKit and its library of useful objects.

For example, to take full control of just your game’s tiled background while using SpriteKit to draw the rest, use an `SKRenderer` and `MTLView` combination instead of `SKView`, and then write the tiled background code yourself in Metal.When you’re ready to mix the SpriteKit content with the Metal content, you call `SKRenderer`‘s [`render(withViewport:commandBuffer:renderPassDescriptor:)`](https://developer.apple.com/documentation/spritekit/skrenderer/render(withviewport:commandbuffer:renderpassdescriptor:)). Finally, call [`present(_:)`](https://developer.apple.com/documentation/Metal/MTLCommandBuffer/present(_:)) on a Metal view when you’re ready to present a frame.

### [On watchOS, Draw Using an Interface Scene](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#On-watchOS-Draw-Using-an-Interface-Scene)

To display SpriteKit content on Apple Watch, you create a [`WKInterfaceSKScene`](https://developer.apple.com/documentation/WatchKit/WKInterfaceSKScene) and then set its [`scene`](https://developer.apple.com/documentation/WatchKit/WKInterfaceSKScene/scene) property. [`WKInterfaceSKScene`](https://developer.apple.com/documentation/WatchKit/WKInterfaceSKScene) functions much like `SKView`, but without the debugging visualization options.

[See Also](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#see-also)
------------------------------------------------------------------------------------------------------------

### [Scene Renderers](https://developer.apple.com/documentation/spritekit/choosing-a-spritekit-scene-renderer#Scene-Renderers)

[`class SKView`](https://developer.apple.com/documentation/spritekit/skview)

A view subclass that renders a SpriteKit scene.

[`class SKRenderer`](https://developer.apple.com/documentation/spritekit/skrenderer)

An object that renders a scene into a custom Metal rendering pipeline and drives the scene update cycle.

[`class WKInterfaceSKScene`](https://developer.apple.com/documentation/WatchKit/WKInterfaceSKScene)

A visual WatchKit element that displays a SpriteKit scene.
