Class
Padding
=======

The values that define content padding within the map view frame.

MapKit JS 5.0+

`class Padding`

[Overview](https://developer.apple.com/documentation/mapkitjs/padding#overview)
-------------------------------------------------------------------------------

Use padding to define edge insets on the map. MapKit JS then uses these insets when it positions items on the map, such as the map controls or annotations. For example, if you want to add your own control on top of the map and ensure that the map doesn’t position annotations underneath it, you can add padding to the map and then position your control within the padded area.

You can use a [`Padding`](https://developer.apple.com/documentation/mapkitjs/padding) object when setting the map’s [`padding`](https://developer.apple.com/documentation/mapkitjs/map/padding) property or as an option of [`showItems(items, options)`](https://developer.apple.com/documentation/mapkitjs/map/showitems). Positive values add padding to the inside edges of the map. MapKit JS clips negative values to `0`.

Creates a padding object and initializes its values with the provided object literal.

[`new Padding(top)`](https://developer.apple.com/documentation/mapkitjs/padding/paddingconstructor2)

Creates a padding object and initializes its top inset margin with the provided value.

[`new Padding(top, right)`](https://developer.apple.com/documentation/mapkitjs/padding/paddingconstructor3)

Creates a padding object and initializes it with the provided top and right side values.

[`new Padding(top, right, bottom)`](https://developer.apple.com/documentation/mapkitjs/padding/paddingconstructor4)

Creates a padding object and initializes it with the provided top, right, and bottom values.

[`new Padding(top, right, bottom, left)`](https://developer.apple.com/documentation/mapkitjs/padding/paddingconstructor5)

[`interface PaddingConstructorOptions`](https://developer.apple.com/documentation/mapkitjs/paddingconstructoroptions)

Initial values of the edge insets for padding.
