Class
Map
===

An embeddable interactive map that you add to a webpage.

MapKit JS 5.0+

`class Map extends MapKitEventTarget`

[Mentioned in](https://developer.apple.com/documentation/mapkitjs/map#mentions)
-------------------------------------------------------------------------------

[Loading the latest version of MapKit JS](https://developer.apple.com/documentation/mapkitjs/loading-the-latest-version-of-mapkit-js)

[Adding interactivity to overlays](https://developer.apple.com/documentation/mapkitjs/adding-interactivity-to-overlays)

[Overview](https://developer.apple.com/documentation/mapkitjs/map#overview)
---------------------------------------------------------------------------

A map is a self-contained view object that you embed on a webpage. It’s possible to have multiple independent maps on a single webpage, although typically webpages only need one interactive map.

[Topics](https://developer.apple.com/documentation/mapkitjs/map#topics)
