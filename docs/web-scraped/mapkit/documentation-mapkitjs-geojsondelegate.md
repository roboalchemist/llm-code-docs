Interface
Geo JSONDelegate
================

A delegate object that controls a GeoJSON import to override default behavior and provide custom style.

MapKit JS 5.0+

`interface GeoJSONDelegate`

[Overview](https://developer.apple.com/documentation/mapkitjs/geojsondelegate#overview)
---------------------------------------------------------------------------------------

The delegate object provides hooks into the key moments in the GeoJSON import process. When you pass a delegate object to the [`importGeoJSON(data, callback)`](https://developer.apple.com/documentation/mapkitjs/mapkit/importgeojson) method, you can:

*   Modify or replace annotations and overlays before MapKit JS adds them to the [`ItemCollection`](https://developer.apple.com/documentation/mapkitjs/itemcollection) object.

*   Override the default GeoJSON mapping behavior.

*   Provide custom styling by implementing [`styleForOverlay(overlay, geoJSON)`](https://developer.apple.com/documentation/mapkitjs/geojsondelegate/styleforoverlay).

*   Receive feedback on the result from importing GeoJSON data.
