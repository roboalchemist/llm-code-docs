Class
Map Rect
========

A rectangular region, in map units, of a two-dimensional map projection.

MapKit JS 5.0+

`class MapRect`

[Overview](https://developer.apple.com/documentation/mapkitjs/maprect#overview)
-------------------------------------------------------------------------------

Use a `mapkit.MapRect` to represent a rectangular region within a map projection. Map units are a value from `0` to `1` that represent an interpolated location within the height or width of the full map projection.

[Topics](https://developer.apple.com/documentation/mapkitjs/maprect#topics)
---------------------------------------------------------------------------

### [Creating a map rectangle](https://developer.apple.com/documentation/mapkitjs/maprect#Creating-a-map-rectangle)

[`new MapRect(x, y, width, height)`](https://developer.apple.com/documentation/mapkitjs/maprect/maprectconstructor)

Creates an object that represents a rectangular region of the map projection.

### [Defining a map rectangle](https://developer.apple.com/documentation/mapkitjs/maprect#Defining-a-map-rectangle)

[`origin`](https://developer.apple.com/documentation/mapkitjs/maprect/origin)

The origin point of a rectangle.

[`size`](https://developer.apple.com/documentation/mapkitjs/maprect/size)

The width and height of a rectangle, starting from the origin point.

### [Obtaining rectangle metrics](https://developer.apple.com/documentation/mapkitjs/maprect#Obtaining-rectangle-metrics)

[`maxX()`](https://developer.apple.com/documentation/mapkitjs/maprect/maxx)

Returns the maximum x-axis value of a rectangle.

[`maxY()`](https://developer.apple.com/documentation/mapkitjs/maprect/maxy)

Returns the maximum y-axis value of a rectangle.

[`midX()`](https://developer.apple.com/documentation/mapkitjs/maprect/midx)

Returns the midpoint along the x-axis of a rectangle.

[`midY()`](https://developer.apple.com/documentation/mapkitjs/maprect/midy)

Returns the midpoint along the y-axis of a rectangle.

[`minX()`](https://developer.apple.com/documentation/mapkitjs/maprect/minx)

Returns the minimum x-axis value of a rectangle.

[`minY()`](https://developer.apple.com/documentation/mapkitjs/maprect/miny)

Returns the minimum y-axis value of a rectangle.

### [Working with map rectangles](https://developer.apple.com/documentation/mapkitjs/maprect#Working-with-map-rectangles)

[`copy()`](https://developer.apple.com/documentation/mapkitjs/maprect/copy)

Returns a copy of a map rectangle.

[`equals(anotherRect)`](https://developer.apple.com/documentation/mapkitjs/maprect/equals)

Compares whether two map rectangles are equal.

[`scale(scaleFactor, scaleCenter)`](https://developer.apple.com/documentation/mapkitjs/maprect/scale)

Returns a scaled map rectangle for a map location.

[`toCoordinateRegion()`](https://developer.apple.com/documentation/mapkitjs/maprect/tocoordinateregion)

Returns the region that corresponds to a map rectangle.

### [Instance Methods](https://developer.apple.com/documentation/mapkitjs/maprect#Instance-Methods)

[`toString()`](https://developer.apple.com/documentation/mapkitjs/maprect/tostring)

Returns a string representation of the map rectangle object.

[See Also](https://developer.apple.com/documentation/mapkitjs/maprect#see-also)
-------------------------------------------------------------------------------

### [Map units](https://developer.apple.com/documentation/mapkitjs/maprect#Map-units)

[`class MapPoint`](https://developer.apple.com/documentation/mapkitjs/mappoint)

A location, in map units, of a point on the Earth’s surface projected onto a 2D map.

[`class MapSize`](https://developer.apple.com/documentation/mapkitjs/mapsize)

A pair of values, in map units, that define the width and height of a rectangular area of a map projection.

[`class CameraZoomRange`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange)
