Interface
Look Around Options
===================

Options for initializing a LookAround view.

MapKit JS 5.79+

`interface LookAroundOptions extends CommonLookAroundOptions`

[Topics](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#topics)
-------------------------------------------------------------------------------------

### [Instance Properties](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#Instance-Properties)

[`showsCloseControl`](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions/showsclosecontrol)

A Boolean value that indicates whether the Look Around view displays a close control.

[`showsDialogControl`](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions/showsdialogcontrol)

A Boolean value that indicates whether the Look Around view displays shows a control that allow someone to expand the Look Around view.

[Relationships](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#relationships)
---------------------------------------------------------------------------------------------------

### [Inherits From](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#inherits-from)

*   [`CommonLookAroundOptions`](https://developer.apple.com/documentation/mapkitjs/commonlookaroundoptions)

[See Also](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#see-also)
-----------------------------------------------------------------------------------------

### [Exploring at street level](https://developer.apple.com/documentation/mapkitjs/lookaroundoptions#Exploring-at-street-level)

[`class LookAround`](https://developer.apple.com/documentation/mapkitjs/lookaround)

A view that allows someone to see a street level view of a place.

[`class LookAroundPreview`](https://developer.apple.com/documentation/mapkitjs/lookaroundpreview)

A class that renders a preview of a Look Around view.

[`interface LookAroundPreviewOptions`](https://developer.apple.com/documentation/mapkitjs/lookaroundpreviewoptions)

Options for initializing a LookAroundPreview object.

[`class LookAroundScene`](https://developer.apple.com/documentation/mapkitjs/lookaroundscene)

Object that represents the current location of the view.

[`interface CommonLookAroundOptions`](https://developer.apple.com/documentation/mapkitjs/commonlookaroundoptions)

Options that control the behavior of Look Around views.

[`class AbstractLookAround`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround)

An abstract class that provides a common interface for Look Around views.

[`lookAroundViews`](https://developer.apple.com/documentation/mapkitjs/mapkit/lookaroundviews)

A list of all the Look Around objects that are currently active on a page.
