Class
Place
=====

A place object that returns from a geocoder lookup, a reverse lookup, or a fetch request for points of interest.

MapKit JS 5.0+

`class Place`

[Overview](https://developer.apple.com/documentation/mapkitjs/place#overview)
-----------------------------------------------------------------------------

[`GeocoderResponse`](https://developer.apple.com/documentation/mapkitjs/geocoderresponse), [`SearchResponse`](https://developer.apple.com/documentation/mapkitjs/searchresponse), and [`PointsOfInterestSearchResponse`](https://developer.apple.com/documentation/mapkitjs/pointsofinterestsearchresponse) return places.

[Topics](https://developer.apple.com/documentation/mapkitjs/place#topics)
-------------------------------------------------------------------------

### [Place name and category](https://developer.apple.com/documentation/mapkitjs/place#Place-name-and-category)

[`name`](https://developer.apple.com/documentation/mapkitjs/place/name)

The name of the place.

[`formattedAddress`](https://developer.apple.com/documentation/mapkitjs/place/formattedaddress)

The address of the place, formatted using its conventions of its country or region.

[`areasOfInterest`](https://developer.apple.com/documentation/mapkitjs/place/areasofinterest)

Common names of the area in which the place resides.

[`pointOfInterestCategory`](https://developer.apple.com/documentation/mapkitjs/place/pointofinterestcategory)

The category of the place.

### [Street address](https://developer.apple.com/documentation/mapkitjs/place#Street-address)

[`thoroughfare`](https://developer.apple.com/documentation/mapkitjs/place/thoroughfare)

The street name at the place.

[`subThoroughfare`](https://developer.apple.com/documentation/mapkitjs/place/subthoroughfare)

The number on the street at the place.

[`fullThoroughfare`](https://developer.apple.com/documentation/mapkitjs/place/fullthoroughfare)

A combination of thoroughfare and subthoroughfare.

[`postCode`](https://developer.apple.com/documentation/mapkitjs/place/postcode)

The postal code of the place.

### [City and district address](https://developer.apple.com/documentation/mapkitjs/place#City-and-district-address)

[`locality`](https://developer.apple.com/documentation/mapkitjs/place/locality)

The city of the place.

[`subLocality`](https://developer.apple.com/documentation/mapkitjs/place/sublocality)

The name of the area within the locality.

[`dependentLocalities`](https://developer.apple.com/documentation/mapkitjs/place/dependentlocalities)

Common names for the local area or neighborhood of the place.

[`administrativeArea`](https://developer.apple.com/documentation/mapkitjs/place/administrativearea)

The state or province of the place.

[`administrativeAreaCode`](https://developer.apple.com/documentation/mapkitjs/place/administrativeareacode)

The short code for the state or area.

### [Country address](https://developer.apple.com/documentation/mapkitjs/place#Country-address)

[`country`](https://developer.apple.com/documentation/mapkitjs/place/country)

The country or region of the place.

[`countryCode`](https://developer.apple.com/documentation/mapkitjs/place/countrycode)

The country or region associated with the place.

### [Location](https://developer.apple.com/documentation/mapkitjs/place#Location)

[`coordinate`](https://developer.apple.com/documentation/mapkitjs/place/coordinate)

The latitude and longitude for the place.

[`region`](https://developer.apple.com/documentation/mapkitjs/place/region)

The geographic region associated with the place.

### [Place IDs](https://developer.apple.com/documentation/mapkitjs/place#Place-IDs)

[`id`](https://developer.apple.com/documentation/mapkitjs/place/id)

The Place ID referencing a feature.

[`alternateIds`](https://developer.apple.com/documentation/mapkitjs/place/alternateids)

A list of alternate Place IDs referencing a feature.

[See Also](https://developer.apple.com/documentation/mapkitjs/place#see-also)
-----------------------------------------------------------------------------

### [Places](https://developer.apple.com/documentation/mapkitjs/place#Places)

[`placeDetails`](https://developer.apple.com/documentation/mapkitjs/mapkit/placedetails)

[`interface PlaceSelectionAccessoryOptions`](https://developer.apple.com/documentation/mapkitjs/placeselectionaccessoryoptions)

The options for selection accessories.

[`class PlaceAnnotation`](https://developer.apple.com/documentation/mapkitjs/placeannotation)

An annotation for a place.

[`class PlaceDetail`](https://developer.apple.com/documentation/mapkitjs/placedetail)

[`class PlaceSelectionAccessory`](https://developer.apple.com/documentation/mapkitjs/placeselectionaccessory)
