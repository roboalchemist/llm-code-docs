Class
Directions
==========

An object that provides directions and estimated travel time based on the options you provide.

MapKit JS 5.0+

`class Directions extends Service`

[Mentioned in](https://developer.apple.com/documentation/mapkitjs/directions#mentions)
--------------------------------------------------------------------------------------

[MapKit JS 5](https://developer.apple.com/documentation/mapkitjs/mapkit-js-5)

[Topics](https://developer.apple.com/documentation/mapkitjs/directions#topics)
------------------------------------------------------------------------------

### [Creating a directions object](https://developer.apple.com/documentation/mapkitjs/directions#Creating-a-directions-object)

[`new Directions(options)`](https://developer.apple.com/documentation/mapkitjs/directions/directionsconstructor)

Creates a directions object with options you provide.

[`interface DirectionsConstructorOptions`](https://developer.apple.com/documentation/mapkitjs/directionsconstructoroptions)

Options that you may provide when creating a directions object.

### [Getting estimated arrival times](https://developer.apple.com/documentation/mapkitjs/directions#Getting-estimated-arrival-times)

[`eta(request, callback)`](https://developer.apple.com/documentation/mapkitjs/directions/eta)

Retrieves estimated arrival times to up to 10 destinations from a single starting point.

[`interface EtaRequestOptions`](https://developer.apple.com/documentation/mapkitjs/etarequestoptions)

The options you may provide for requesting estimated arrival times.

[`interface EtaResponse`](https://developer.apple.com/documentation/mapkitjs/etaresponse)

The estimated arrival times for a set of destinations.

[`interface EtaResult`](https://developer.apple.com/documentation/mapkitjs/etaresult)

The mode of transportation, distance, and travel time estimates for a single destination.

### [Getting directions](https://developer.apple.com/documentation/mapkitjs/directions#Getting-directions)

[`route(request, callback)`](https://developer.apple.com/documentation/mapkitjs/directions/route)

Retrieves directions and estimated travel time based on the specified start and end points.

[`interface DirectionsRequest`](https://developer.apple.com/documentation/mapkitjs/directionsrequest)

The requested start and end points for a route, as well as the planned mode of transportation.

[`interface DirectionsResponse`](https://developer.apple.com/documentation/mapkitjs/directionsresponse)

The directions and estimated travel time that return for a route.

[`class Route`](https://developer.apple.com/documentation/mapkitjs/route)

Information about a route, including step-by-step instructions, distance, and estimated travel time.

[`class RouteStep`](https://developer.apple.com/documentation/mapkitjs/routestep)

A single step of the route between the requested start and end points.

### [Canceling a directions request](https://developer.apple.com/documentation/mapkitjs/directions#Canceling-a-directions-request)

[`cancel(id)`](https://developer.apple.com/documentation/mapkitjs/service/cancel)

Cancels a request using the provided request ID.

### [Static properties](https://developer.apple.com/documentation/mapkitjs/directions#Static-properties)

[`Transport`](https://developer.apple.com/documentation/mapkitjs/directions/transport)

A static property that refers to an object that describes the available transport type values.
