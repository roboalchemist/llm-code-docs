Class
Camera Zoom Range
=================

A minimum and maximum camera distance, in meters, from the center of the map.

MapKit JS 5.23+

`class CameraZoomRange`

[Topics](https://developer.apple.com/documentation/mapkitjs/camerazoomrange#topics)
-----------------------------------------------------------------------------------

### [Defining a zoom range](https://developer.apple.com/documentation/mapkitjs/camerazoomrange#Defining-a-zoom-range)

[`new CameraZoomRange()`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/camerazoomrangeconstructor)

Constructs an instance of a camera zoom range object with no minimum or maximum camera distance.

[`new CameraZoomRange(rangeParams)`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/camerazoomrangeconstructor1)

Creates an instance of a camera zoom range object with an object literal.

[`new CameraZoomRange(min, max)`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/camerazoomrangeconstructor2)

Creates an instance of a camera zoom range object with the specified numeric arguments that specify minimum and maximum camera distances.

[`interface CameraZoomRangeConstructorOptions`](https://developer.apple.com/documentation/mapkitjs/camerazoomrangeconstructoroptions)

Initialization options for the camera zoom range.

### [Setting minimum and maximum distances](https://developer.apple.com/documentation/mapkitjs/camerazoomrange#Setting-minimum-and-maximum-distances)

[`minCameraDistance`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/mincameradistance)

The minimum allowed distance of the camera from the center of the map in meters.

[`maxCameraDistance`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/maxcameradistance)

The maximum allowed distance of the camera from the center of the map in meters.

### [Instance Methods](https://developer.apple.com/documentation/mapkitjs/camerazoomrange#Instance-Methods)

[`copy()`](https://developer.apple.com/documentation/mapkitjs/camerazoomrange/copy)

Returns a copy of the camera zoom region.

[`class MapRect`](https://developer.apple.com/documentation/mapkitjs/maprect)

A rectangular region, in map units, of a two-dimensional map projection.
