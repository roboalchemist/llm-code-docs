H2O Driverless AI Python Client[¶](https://docs.h2o.ai/driverless-ai/pyclient/docs/html/index.html#h2o-driverless-ai-python-client "Permanent link")
====================================================================================================================================================

The Driverless AI Python Client provides an API to easily interact with a [H2O Driverless AI](https://h2o.ai/products/h2o-driverless-ai/).

Supported Driverless AI Servers[¶](https://docs.h2o.ai/driverless-ai/pyclient/docs/html/index.html#supported-driverless-ai-servers "Permanent link")
----------------------------------------------------------------------------------------------------------------------------------------------------

The client version number indicates the most recent Driverless AI server supported by that specific client version. However, all client versions are backwards compatible with Driverless AI servers down to version 1.10.0.

Requirements[¶](https://docs.h2o.ai/driverless-ai/pyclient/docs/html/index.html#requirements "Permanent link")
--------------------------------------------------------------------------------------------------------------

*   Python >=3.8 (Python 3.8 support is deprecated and will be removed in a future release. Python 3.9+ is recommended.)

### Dependencies[¶](https://docs.h2o.ai/driverless-ai/pyclient/docs/html/index.html#dependencies "Permanent link")

*   `requests`
*   `urllib3>=1.26.0`
*   `tabulate`
*   `toml`

### Optional Dependencies[¶](https://docs.h2o.ai/driverless-ai/pyclient/docs/html/index.html#optional-dependencies "Permanent link")

*   `fsspec`
*   `pandas`
