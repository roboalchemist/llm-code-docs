# Zeabur 

Deploy Nitro apps to Zeabur.

</div>

<div>

**Preset:** `zeabur`

[[]](https://zeabur.com)[][] Read more in [Zeabur].

[]Integration with this provider is possible with [zero configuration](/deploy/#zero-config-providers).

## [[[]]Deploy using git](#deploy-using-git) 

#### Push your code to your git repository (Currently only GitHub supported). 

#### [Import your project](https://zeabur.com/docs/get-started) into Zeabur. 

#### Zeabur will detect that you are using Nitro and will enable the correct settings for your deployment. 

#### Your application is deployed!
