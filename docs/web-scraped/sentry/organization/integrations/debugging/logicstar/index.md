The Logicstar integration enables automatic fixes for your Sentry issues with Logicstar AI. This integration is maintained and supported by Logicstar; for more details reach out to support@logicstar.ai.

## Install and Configure

Sentry owner, manager, or admin permissions are required to install this integration.

1. Navigate to **Settings > Integrations > Logicstar**

2. Follow the full [Logicstar installation instructions](https://logicstar.ai/documentation/sentry-integration).
