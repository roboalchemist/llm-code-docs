Kirim masukan
Menggunakan Google Charts Tetap teratur dengan koleksi  Simpan dan kategorikan konten berdasarkan preferensi Anda
==================================================================================================================

![Image 2: ilustrasi intro visualisasi](https://developers.google.com/static/chart/interactive/images/visualization_intro.png?hl=id)
Google Chart memberikan cara sempurna untuk memvisualisasikan data di situs Anda. Dari diagram garis sederhana hingga peta hierarki hierarki yang kompleks, [galeri diagram](https://developers.google.com/chart/interactive/docs/gallery?hl=id) menyediakan banyak jenis diagram yang siap digunakan.

Cara paling umum untuk menggunakan Google Chart adalah dengan JavaScript sederhana yang Anda sematkan di halaman web. Anda dapat memuat beberapa library Google Chart, mencantumkan data yang akan dibuat diagramnya, memilih opsi untuk menyesuaikan diagram, dan terakhir membuat objek diagram dengan id yang dipilih. Kemudian, di halaman web, Anda membuat <div> dengan id tersebut untuk menampilkan Google Chart.

Hanya itu yang Anda perlukan untuk [memulai](https://developers.google.com/chart/interactive/docs/quick_start?hl=id).

Diagram ditampilkan sebagai class JavaScript, dan Google Chart menyediakan [banyak jenis diagram](https://developers.google.com/chart/interactive/docs/gallery?hl=id) untuk Anda gunakan. Tampilan defaultnya biasanya adalah semua yang Anda perlukan, dan Anda dapat [menyesuaikan diagram](https://developers.google.com/chart/interactive/docs/customizing_charts?hl=id) kapan saja agar sesuai dengan tampilan dan nuansa situs Anda. Diagram sangat interaktif dan menampilkan [peristiwa](https://developers.google.com/chart/interactive/docs/events?hl=id) yang memungkinkan Anda menghubungkannya untuk membuat [dasbor](https://developers.google.com/chart/interactive/docs/gallery/controls?hl=id) yang kompleks atau pengalaman lain [yang terintegrasi dengan halaman web Anda](https://developers.google.com/chart/interactive/docs/examples?hl=id#full_html_page_example). Diagram dirender menggunakan teknologi HTML5/SVG untuk menyediakan kompatibilitas lintas browser (termasuk VML untuk versi IE lama) dan portabilitas lintas platform ke iPhone, iPad, dan Android. Pengguna Anda tidak akan pernah menggunakan plugin atau software apa pun. Jika memiliki browser web, mereka dapat melihat diagram Anda.

Semua jenis diagram diisi dengan data menggunakan class [DataTable](https://developers.google.com/chart/interactive/docs/reference?hl=id#DataTable), sehingga memudahkan untuk beralih di antara jenis diagram saat Anda bereksperimen guna menemukan tampilan yang ideal. DataTable menyediakan metode untuk mengurutkan, memodifikasi, dan memfilter data, dan dapat diisi langsung dari halaman web Anda, database, atau penyedia data apa pun yang mendukung protokol [Sumber Data Alat Diagram](https://developers.google.com/chart/interactive/docs/queries?hl=id). (Protokol tersebut mencakup bahasa kueri yang mirip SQL dan diterapkan oleh Google Spreadsheet, Google Fusion Tables, dan penyedia data pihak ketiga seperti SalesForce. Anda bahkan dapat menerapkan protokol di situs Anda sendiri dan menjadi penyedia data untuk layanan lainnya.)

**Siap membuat diagram pertama Anda? Buka [panduan memulai](https://developers.google.com/chart/interactive/docs/quick_start?hl=id).**

Alat kami terus berkembang untuk memenuhi kebutuhan Anda dengan lebih baik; kami mengandalkan masukan Anda untuk membantu kami memprioritaskan fitur mana yang akan disertakan. Bergabunglah dengan [grup diskusi](http://groups.google.com/group/google-visualization-api?hl=id) kami.
