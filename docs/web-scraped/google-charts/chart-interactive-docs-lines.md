* Lines connecting data points in Google Charts like Area, Line, and Combo Charts can be customized for color, thickness, and dashing.

* Line colors can be changed using the `colors` option for the chart palette or the `series` option for specific series.

* The thickness of lines can be controlled using the `lineWidth` option for all lines or the `series` option for individual series.

* Different styles of dashed lines are possible with the `lineDashStyle` option, which takes an array of numbers representing dash and gap lengths.

[](https://developers.google.com/chart/interactive/docs/lines)Overview
----------------------------------------------------------------------

Some Google Charts, such as the Area, Line, and Combo Charts, have lines connecting data points. You can customize the color, thickness, and dashing of the lines using the techniques on this page.

[](https://developers.google.com/chart/interactive/docs/lines)Changing the Color
--------------------------------------------------------------------------------

You can change the color of the lines that connect data points in Google Charts in two subtly different ways: with the `colors` option to change the chart palette, or with the `series` option to specify the color for particular series.

In the following chart, we set the color of each series explicitly.

Above, the colors are specified by hex value (e.g., `'#e2431e'`, or `'#f00'` for a saturated red) but can also be specified by English name. [](https://developers.google.com/chart/interactive/docs/lines)Dashed Lines
--------------------------------------------------------------------------

Many styles of dashed lines are possible via the `lineDashStyle` option, which takes an array of numbers. The first number indicates the length of a dash, and the second indicates the gap after it. If there is a third number, that's the length of the next dash, and a fourth number, if present, is the length of the next gap.

When the chart is drawn, these lengths are repeated, so

```
[4,
4]
```

 means a succession of 4-length dashes and 4-length gaps. `[5, 1, 3]` means a 5-length dash, a 1-length gap, a 3-length dash, a 5-length gap, and so on. Some examples:

 var options = {
 hAxis: { maxValue: 10 },
 vAxis: { maxValue: 18 },
 chartArea: { width: 380 },
 lineWidth: 4,
 **series: { 0: { lineDashStyle: [1, 1] }, 1: { lineDashStyle: [2, 2] }, 2: { lineDashStyle: [4, 4] }, 3: { lineDashStyle: [5, 1, 3] }, 4: { lineDashStyle: [4, 1] }, 5: { lineDashStyle: [10, 2] }, 6: { lineDashStyle: [14, 2, 7, 2] }, 7: { lineDashStyle: [14, 2, 2, 7] }, 8: { lineDashStyle: [2, 2, 20, 2, 20, 2] } },**
 colors: ['#e2431e', '#f1ca3a', '#6f9654', '#1c91c0',
