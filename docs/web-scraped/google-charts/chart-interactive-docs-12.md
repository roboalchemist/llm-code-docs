שליחת משוב
שימוש ב-Google Graphs קל לארגן דפים בעזרת אוספים  אפשר לשמור ולסווג תוכן על סמך ההעדפות שלך
============================================================================================

![Image 2: איור של סרטון אינטרו](https://developers.google.com/static/chart/interactive/images/visualization_intro.png?hl=he)
התרשימים של Google מספקים דרך מושלמת להציג באופן חזותי נתונים באתר שלך. מתרשימי קו פשוטים ועד למפות עצים היררכיות מורכבות, [גלריית התרשימים](https://developers.google.com/chart/interactive/docs/gallery?hl=he) כוללת מספר רב של סוגי תרשימים מוכנים לשימוש.

הדרך הנפוצה ביותר להשתמש ב-Google charts היא באמצעות JavaScript פשוט שמוטמע בדף האינטרנט. עכשיו אפשר לטעון ספריות מסוימות ב-Google Chart, לציין את הנתונים בתרשים, לבחור אפשרויות להתאמה אישית של התרשים וליצור אובייקט בתרשים עם id לבחירתך. לאחר מכן, בשלב מאוחר יותר בדף האינטרנט, יוצרים <div> עם ה-id הזה כדי להציג את תרשים Google.

זה כל מה שצריך כדי [להתחיל](https://developers.google.com/chart/interactive/docs/quick_start?hl=he).

התרשימים נחשפים כשיעורי JavaScript, ו-Google כן מספק [סוגים רבים של תרשימים](https://developers.google.com/chart/interactive/docs/gallery?hl=he) לשימושך. לרוב, מראה ברירת המחדל הוא כל מה שנחוץ לך, ותמיד אפשר [להתאים אישית תרשים](https://developers.google.com/chart/interactive/docs/customizing_charts?hl=he) כדי להתאים אותו למראה ולחוויה שבאתר. התרשימים הם אינטראקטיביים מאוד וחושפים [אירועים](https://developers.google.com/chart/interactive/docs/events?hl=he) שמאפשרים לקשר ביניהם [למרכזי בקרה](https://developers.google.com/chart/interactive/docs/gallery/controls?hl=he) או לחוויות אחרות [המשולבות עם דף האינטרנט.](https://developers.google.com/chart/interactive/docs/examples?hl=he#full_html_page_example). העיבוד של התרשימים מתבצע באמצעות טכנולוגיית HTML5/SVG כדי לספק תאימות בין דפדפנים (כולל VML לגרסאות ישנות יותר של IE) וניידות בפלטפורמות שונות במכשירי iPhone, iPad ו-Android. המשתמשים אף פעם לא יצטרכו להתעסק עם יישומי פלאגין או תוכנה כלשהי. אם יש לו דפדפן אינטרנט, הוא יכול לראות את התרשימים שלכם.

כל סוגי התרשימים מאוכלסים בנתונים באמצעות המחלקה [DataTable](https://developers.google.com/chart/interactive/docs/reference?hl=he#DataTable), כך שקל לעבור בין סוגי התרשימים ככל שעורכים ניסויים כדי למצוא את המראה האידיאלי. ה-DataTable מספק שיטות למיון, לשינוי ולסינון של נתונים, וניתן לאכלס אותו ישירות מדף האינטרנט, ממסד נתונים או מכל ספק נתונים שתומך בפרוטוקול [Data Source Tools של כלי התרשימים](https://developers.google.com/chart/interactive/docs/queries?hl=he). (פרוטוקול זה כולל שפת שאילתה דמוית SQL ומיושם על ידי Google Sheets, Google Fusion Tables וספקי נתונים של צד שלישי כמו SalesForce. אפשר גם להטמיע את הפרוטוקול באתר שלך ולהפוך לספק נתונים עבור שירותים אחרים.)

**מוכנים ליצור את התרשים הראשון? כדאי להיכנס אל [המדריך למתחילים](https://developers.google.com/chart/interactive/docs/quick_start?hl=he).**

הכלים שלנו מתפתחים כל הזמן כדי לענות בצורה טובה יותר על הצרכים שלך. אנחנו תלויים במשוב שלך כדי לעזור לנו לתעדף את התכונות שייכללו. ב[קבוצת הדיון שלנו](http://groups.google.com/group/google-visualization-api?hl=he).
