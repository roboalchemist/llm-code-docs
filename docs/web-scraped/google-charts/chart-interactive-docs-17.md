ส่งความคิดเห็น
การใช้ Google Charts จัดทุกอย่างให้เป็นระเบียบอยู่เสมอด้วยคอลเล็กชัน  บันทึกและจัดหมวดหมู่เนื้อหาตามค่ากำหนดของคุณ
==================================================================================================================

![Image 2: ภาพแสดงบทนํา](https://developers.google.com/static/chart/interactive/images/visualization_intro.png?hl=th)
Google Charts เป็นวิธีการแสดงภาพข้อมูลบนเว็บไซต์ที่สมบูรณ์แบบ [แผนภูมิแผนภูมิ](https://developers.google.com/chart/interactive/docs/gallery?hl=th)มีแผนภูมิประเภทที่พร้อมใช้งานมากมาย ตั้งแต่แผนภูมิเส้นแบบธรรมดาไปจนถึงแผนผังต้นไม้แบบลําดับชั้นที่ซับซ้อน

วิธีที่ใช้กันโดยทั่วไปสําหรับ Google Charts คือ JavaScript แบบง่ายที่ฝังอยู่ในหน้าเว็บ คุณโหลดไลบรารีของ Google Chart บางรายการ แสดงรายการข้อมูลที่ต้องการสร้างแผนภูมิ เลือกตัวเลือกเพื่อปรับแต่งแผนภูมิ และสุดท้าย สร้างออบเจ็กต์แผนภูมิด้วย รหัส ที่คุณเลือก จากนั้นในหน้าเว็บ ให้สร้าง <div> โดยใช้ id นั้นเพื่อแสดงแผนภูมิของ Google

นี่คือทั้งหมดที่คุณต้องการ[เริ่มต้นใช้งาน](https://developers.google.com/chart/interactive/docs/quick_start?hl=th)

แผนภูมิจะแสดงเป็นคลาส JavaScript และ Google Charts มี[แผนภูมิหลายประเภท](https://developers.google.com/chart/interactive/docs/gallery?hl=th)ให้คุณใช้ ลักษณะที่ปรากฏเริ่มต้นมักจะเป็นสิ่งที่คุณต้องการ และคุณสามารถ[ปรับแต่งแผนภูมิ](https://developers.google.com/chart/interactive/docs/customizing_charts?hl=th)ได้เสมอเพื่อให้เหมาะกับรูปลักษณ์ของเว็บไซต์ แผนภูมิเป็นเนื้อหาที่มี[การโต้ตอบ](https://developers.google.com/chart/interactive/docs/events?hl=th)แบบอินเทอร์แอกทีฟและเปิดเผยสูง ซึ่งช่วยให้คุณเชื่อมต่อเพื่อสร้าง[แดชบอร์ด](https://developers.google.com/chart/interactive/docs/gallery/controls?hl=th)หรือประสบการณ์อื่นๆ ที่[ผสานรวมกับหน้าเว็บ](https://developers.google.com/chart/interactive/docs/examples?hl=th#full_html_page_example)ที่ซับซ้อนได้ แผนภูมิจะแสดงผลโดยใช้เทคโนโลยี HTML5/SVG เพื่อให้ความเข้ากันได้กับหลายเบราว์เซอร์ (รวมถึง VML สําหรับ IE เวอร์ชันเก่า) และความสามารถในการถ่ายโอนระหว่างแพลตฟอร์มไปยัง iPhone, iPad และ Android ผู้ใช้จะไม่ต้องยุ่งยากกับปลั๊กอินหรือซอฟต์แวร์ใดๆ หากมีเว็บเบราว์เซอร์ ก็จะเห็นแผนภูมิของคุณ

ประเภทแผนภูมิทั้งหมดจะมีข้อมูลโดยใช้คลาส [DataTable](https://developers.google.com/chart/interactive/docs/reference?hl=th#DataTable) ซึ่งจะช่วยให้คุณสลับระหว่างแผนภูมิประเภทต่างๆ ได้ง่ายในระหว่างการทดสอบเพื่อค้นหาลักษณะที่เหมาะสม DataTable มีวิธีการจัดเรียง แก้ไข และกรองข้อมูล และสามารถป้อนข้อมูลได้โดยตรงจากหน้าเว็บ ฐานข้อมูล หรือผู้ให้บริการข้อมูลใดๆ ที่รองรับโปรโตคอล[แหล่งข้อมูลเครื่องมือแผนภูมิ](https://developers.google.com/chart/interactive/docs/queries?hl=th) (โปรโตคอลดังกล่าวใช้ภาษาสืบค้นแบบ SQL และมีการใช้งานโดย Google สเปรดชีต, Google ตารางฟิวชันและผู้ให้บริการข้อมูลบุคคลที่สาม เช่น SalesForce นอกจากนี้คุณยังใช้โปรโตคอลในเว็บไซต์ของคุณเองและกลายเป็นผู้ให้บริการข้อมูลสําหรับบริการอื่นๆ ได้ด้วย)

**พร้อมที่จะสร้างแผนภูมิแรกหรือยัง ไปที่[คู่มือเริ่มใช้งานฉบับย่อ](https://developers.google.com/chart/interactive/docs/quick_start?hl=th)**

เครื่องมือของเราเปลี่ยนแปลงอย่างต่อเนื่องเพื่อให้ตรงตามความต้องการของคุณมากยิ่งขึ้น เราอาศัยความคิดเห็นของคุณในการช่วยเราจัดลําดับความสําคัญของฟีเจอร์ที่จะใส่ เข้าร่วม[กลุ่มสนทนา](http://groups.google.com/group/google-visualization-api?hl=th)
