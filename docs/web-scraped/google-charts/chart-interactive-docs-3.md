![Image 2: ilustración de la introducción a la visualización](https://developers.google.com/static/chart/interactive/images/visualization_intro.png?hl=es-419)
Google Charts proporciona una manera perfecta de visualizar datos en tu sitio web. Desde gráficos de líneas simples hasta mapas jerárquicos complejos y complejos, la [galería de gráficos](https://developers.google.com/chart/interactive/docs/gallery?hl=es-419) proporciona una gran cantidad de tipos de gráficos listos para usar.

La forma más común de usar los gráficos de Google es mediante el simple uso de JavaScript que incorporas en la página web. Carga algunas bibliotecas de gráficos de Google, enumera los datos que se representarán, selecciona opciones para personalizar el gráfico y, por último, crea un objeto de gráfico con el id que elijas. Más adelante, en la página web, crearás un elemento <div> con ese id para mostrar el gráfico de Google.

Eso es todo lo que necesitas para [comenzar](https://developers.google.com/chart/interactive/docs/quick_start?hl=es-419).

Los gráficos se exponen como clases de JavaScript y Google Charts proporciona [muchos tipos de gráficos](https://developers.google.com/chart/interactive/docs/gallery?hl=es-419) que puedes usar. Por lo general, la apariencia predeterminada será todo lo que necesitas y siempre puedes [personalizar un gráfico](https://developers.google.com/chart/interactive/docs/customizing_charts?hl=es-419) para que se adapte al aspecto de tu sitio web. Los gráficos son muy interactivos y exponen [eventos](https://developers.google.com/chart/interactive/docs/events?hl=es-419) que te permiten conectarlos para crear [paneles](https://developers.google.com/chart/interactive/docs/gallery/controls?hl=es-419) o experiencias de otros tipos [integrados en tu página web](https://developers.google.com/chart/interactive/docs/examples?hl=es-419#full_html_page_example). Los gráficos se renderizan con la tecnología de HTML5/SVG a fin de proporcionar compatibilidad en varios navegadores (incluida VML para las versiones anteriores de IE) y portabilidad entre plataformas en iPhones, iPads y Android. Los usuarios nunca tendrán que jugar con complementos ni software. Si tienen un navegador web, pueden ver los gráficos.

Todos los tipos de gráficos se propagan con datos mediante la clase [DataTable](https://developers.google.com/chart/interactive/docs/reference?hl=es-419#DataTable), lo que facilita el cambio entre los tipos de gráfico a medida que experimentas para encontrar la apariencia ideal. La tabla de datos proporciona métodos para ordenar, modificar y filtrar datos, y se puede propagar directamente desde tu página web, una base de datos o cualquier proveedor de datos que admita el protocolo de [fuente de datos de herramientas de gráficos](https://developers.google.com/chart/interactive/docs/queries?hl=es-419). (Ese protocolo incluye un lenguaje de consulta similar a SQL y lo implementan las hojas de cálculo de Google, las tablas de fusión de Google y los proveedores de datos de terceros, como SalesForce. Incluso puedes implementar el protocolo en tu propio sitio web y convertirte en un proveedor de datos para otros servicios).

**¿Estás listo para crear tu primer gráfico? Consulta la [guía de inicio rápido](https://developers.google.com/chart/interactive/docs/quick_start?hl=es-419).**

Nuestras herramientas evolucionan constantemente para satisfacer mejor tus necesidades. Dependemos de tus comentarios, ya que nos ayudan a priorizar qué funciones incluir. Únete a nuestro [grupo de discusión](http://groups.google.com/group/google-visualization-api?hl=es-419).
