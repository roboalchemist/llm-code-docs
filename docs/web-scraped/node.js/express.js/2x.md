Preview the website locally
---------------------------

[](https://expressjs.com/2x/#preview-the-website-locally)

1.   Clone this repository `git clone https://github.com/expressjs/expressjs.com.git --single-branch --branch 2x`
2.   Install [Node.js](https://nodejs.org/en/download/).
3.   Run `npm install` to install the dependencies.
4.   Run `npm run preview` to start the server.
5.   Open `http://localhost:3000` in your browser.
