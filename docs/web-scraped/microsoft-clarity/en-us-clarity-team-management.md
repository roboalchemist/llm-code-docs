Once you sign up to Clarity and [add a new project](https://learn.microsoft.com/en-us/clarity/getting-started#add-a-new-project), you're now the project's Admin. You can only modify project settings if you're an Admin of the project.

Select a project. Go to the **Settings** ->**Teams** to start managing your team.

[](https://learn.microsoft.com/en-us/clarity/team-management#team-roles)
Team roles
----------

Clarity supports two roles:

*   **Admin**: An Admin has complete access to the project. There can be multiple Admins for a project.

*   **Team member**: A member added by the Admin has read and write access to the project.

[](https://learn.microsoft.com/en-us/clarity/team-management#project-access)
Project access
--------------

Here are the permission levels for each role.

Expand table

| Actions | Admin | Team member |
| --- | --- | --- |
| Change roles | Yes | No |
| Add a team member | Yes | No |
| Remove team member | Yes | No |
| Delete project | Yes | No |
| Manage settings | Yes | No |
| View Heatmaps | Yes | Yes |
| Share Heatmaps | Yes | Yes |
| View Recordings | Yes | Yes |
| Save segments | Yes | Yes |

[](https://learn.microsoft.com/en-us/clarity/team-management#add-team-members)
Add team members
----------------

Admins can add team members through their email IDs. There's no limit on the number of users you can add to a project. However, only 10 pending invites are allowed at a time.

[](https://learn.microsoft.com/en-us/clarity/team-management#step-1)
### Step 1

Select a project. Go to **Settings** ->**Team** ->**Add team member**.

![Image 16: Select add team member.](https://learn.microsoft.com/en-us/media/add-team-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-2)
### Step 2

Enter an email ID and select the member role. Select **Add**.

![Image 17: Enter email ID and select add.](https://learn.microsoft.com/en-us/media/add-team-step-2.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-3)
### Step 3

The new team member can view the project in **My Projects** once they accept the invitation received through email.

[](https://learn.microsoft.com/en-us/clarity/team-management#remove-team-members)
Remove team members
-------------------

Important

*   You can only remove members in your project if you are an Admin. Removing a user will deny them access to the project. You can readd the user to your project.
*   If you are the only project member and remove yourself, the project will be deleted.
*   Once you see confirmation of removing the user, we strongly recommend to remove the [Data export tokens](https://learn.microsoft.com/en-us/clarity/clarity-data-export-api#best-practices), if any, associated with the removed user. ![Image 18: Remove data export api token.](https://learn.microsoft.com/en-us/media/data-export-replace-token.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-1-1)
### Step 1

Select a project. Go to **Settings** ->**Team**. Choose the member you wish to remove from the list of team members and select **Remove**.

![Image 19: Go to team and remove.](https://learn.microsoft.com/en-us/media/remove-team-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-2-1)
### Step 2

On the pop-up window, select **Yes, remove** to confirm the deletion.

![Image 20: confirm on the popup window.](https://learn.microsoft.com/en-us/media/remove-team-step-2.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#remove-admin)
### Remove Admin

The Admins can delete themselves from the project.

Important

If you are the only Admin, assign a new Admin before removing yourself.

[](https://learn.microsoft.com/en-us/clarity/team-management#step-1-2)
#### Step 1

Select a project. Go to **Settings** ->**Team**.

![Image 21: Go to team.](https://learn.microsoft.com/en-us/media/change-role-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-2-2)
#### Step 2

Select **Leave team** and confirm on the pop-up window.

![Image 22: Leave team.](https://learn.microsoft.com/en-us/media/leave-team.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#change-user-role)
Change user role
----------------

Note

*   A user can be an 'Admin' or 'Member'. Check [project access](https://learn.microsoft.com/en-us/clarity/team-management#project-access) to know more.
*   You can change the user role to Admin only if you are an Admin of the project.

[](https://learn.microsoft.com/en-us/clarity/team-management#step-1-3)
### Step 1

Select a project. Go to **Settings** ->**Team**.

![Image 23: Navigate to team.](https://learn.microsoft.com/en-us/media/change-role-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-2-3)
### Step 2

Choose the member from the list of team members and select the user role from the menu.

![Image 24: Select user role.](https://learn.microsoft.com/en-us/media/change-role-step-2.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#cancel-an-invite)
Cancel an invite
----------------

After adding a team member, you can cancel the invite before they accept it.

[](https://learn.microsoft.com/en-us/clarity/team-management#step-1-4)
### Step 1

Select a project. Go to **Settings** ->**Team**.

![Image 25: Navigate to team in settings.](https://learn.microsoft.com/en-us/media/change-role-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/team-management#step-2-4)
### Step 2

Choose the member from the list of team members and select **Cancel invite**.

![Image 26: Select cancel invite.](https://learn.microsoft.com/en-us/media/cancel-invite.png)
