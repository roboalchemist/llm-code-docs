Important

Clarity shouldn't be used on any websites/apps targeting users under the age of 18.

Onboarding Clarity is free and easy. If you're new to Clarity, start by signing up.

Follow the steps below to sign up and start using Clarity right away.

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#sign-up-for-clarity)
Sign up for Clarity
-------------------

1.   Go to [https://clarity.microsoft.com/](https://clarity.microsoft.com/) and select **Sign up**.

![Image 20: signup step 1.](https://learn.microsoft.com/en-us/clarity/media/signup-step-1.png)

2.   Choose a sign-up method and follow the steps on the screen.

![Image 21: signup step 2.](https://learn.microsoft.com/en-us/clarity/media/signup-step-2.png)

3.   On successful sign-in, you can view the **Projects** page.

![Image 22: signup step 3.](https://learn.microsoft.com/en-us/clarity/media/signup-step-3.png)

[Try Demo](https://clarity.microsoft.com/demo/projects/view/3t0wlogvdz/dashboard?date_d=Last%203%20days)

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#add-a-new-project)
Add a new project
-----------------

You can add an unlimited number of projects for each domain or website.

1.   Sign in to your [Clarity](https://clarity.microsoft.com/) account.

2.   Select **New project** to start adding a project.

![Image 23: Select add a new project.](https://learn.microsoft.com/en-us/clarity/media/add-project-step-1.png)

3.   Enter the Name and Website details. Select **Add new project** once done.

![Image 24: Add details and select create.](https://learn.microsoft.com/en-us/clarity/media/add-project-step-2.png)

4.   You're now redirected to **Getting Started**. Install Clarity by using any of the installation methods. Check [Clarity Setup](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup) to learn more.

![Image 25: Getting started to setup.](https://learn.microsoft.com/en-us/clarity/media/add-project-step-3.png)

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#clarity-tour)
### Clarity tour

Note

Clarity tour is available in **Settings** even after you finish the setup process.

1.   Scroll down to view a checklist of self guided tour, which helps to explore our features while you finish the setup.

![Image 26: Clarity tour in getting started.](https://learn.microsoft.com/en-us/clarity/media/clarity-tour-in-getting-started.png)

2.   You can also view the tour later. Go to **Settings** ->**Clarity tour**.

![Image 27: Clarity tour in settings.](https://learn.microsoft.com/en-us/clarity/media/clarity-tour-in-settings.png)

3.   Once you add the code, you can view [real time](https://learn.microsoft.com/en-us/clarity/session-recordings/clarity-real-time) data with the number of current live users as shown below:

![Image 28: View number of live recordings.](https://learn.microsoft.com/en-us/clarity/media/clarity-live-recordings-installation.png)

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#view-projects)
### View projects

1.   You can view all your projects on the **Projects** page. You can also access the projects through the 'My Projects' dropdown menu.

![Image 29: projects page and dropdown.](https://learn.microsoft.com/en-us/clarity/media/view-projects.png)

2.   Once the project is successfully added, you'll be the Admin with permissions to manage the project settings. Check [Team management](https://learn.microsoft.com/en-us/clarity/setup-and-installation/team-management) to know more.

Important

Each project you add to your Clarity account will have its unique tracking code. To start using Clarity for your project, you must finish the [Setup](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup) process. 

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#project-management)
Project management
------------------

Once you create a project, go to the **Settings** tab. You can manage your project and team and set up the Clarity tracking Code through Settings.

Note

You can only modify project settings if you are an Admin of the project.

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#overview)
### Overview

You can rename the project and change the website industry in the overview section.

![Image 30: Project settings overview for an Admin.](https://learn.microsoft.com/en-us/clarity/media/project-settings-overview.png)

You can also find the Project ID here.

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#website-industry)
### Website industry

The available options under the **Website industry** are:

![Image 31: Project settings industry.](https://learn.microsoft.com/en-us/clarity/media/project-settings-industry.png)

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#delete-a-project)
### Delete a project

Warning

*   You can delete a project only if you are an Admin of the project.
*   Once you delete a project, you will not be able to recover any data associated with that project.
*   Deleting a project will remove it completely for all team members. You will not be able to undo this action.

You can delete a project in two ways:

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#delete-the-project-under-settings)
#### Delete the Project under Settings

1.   Go to the **Settings** tab and select **Overview**.

![Image 32: Go to team.](https://learn.microsoft.com/en-us/clarity/media/delete-project-step-1.png)

2.   Select **Delete this project**. Optionally provide feedback and confirm **Delete** on the pop-up window.

![Image 33: Select delete this project.](https://learn.microsoft.com/en-us/clarity/media/delete-project-step-2.png)

[](https://learn.microsoft.com/en-us/clarity/setup-and-installation/getting-started#delete-project-on-the-projects-page)
#### Delete Project on the Projects Page

You can also delete the project on the **Projects** page. Select the **Delete icon** on the project. Optionally provide feedback and confirm **Delete** on the pop-up window.

![Image 34: Delete project on projects page.](https://learn.microsoft.com/en-us/clarity/media/delete-project-step-3.png)
