Heat maps are a great way to analyze user behavior in an aggregated manner. Once you onboard Clarity and land onto Heatmaps, Clarity generates heat maps for [popular pages](https://learn.microsoft.com/en-us/clarity/heatmaps-overview#heatmaps-zero-state). Select **View Heatmap** to start using the heatmap. They're created instantly within a matter of seconds.

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#heatmaps-for-a-group-of-pages)
Heatmaps for a group of pages
-----------------------------

View heatmaps for multiple pages entering a URL and selecting the **+ Add URL** button. Clarity [supports various operators](https://learn.microsoft.com/en-us/filters/path-filters-operators) to include with the URL. The default operator is **is exactly**.

![Image 8: Enter group of pages.](https://learn.microsoft.com/en-us/media/heatmaps-add-url-group-of-pages.png)

If a visited URL is already selected, you can **+ Add URL** through [filters](https://learn.microsoft.com/en-us/filters/clarity-filters#filtering-by-path). Learn more on how to use [regex tester](https://learn.microsoft.com/en-us/filters/path-filters-operators#choosing-matches-regex-or-excludes-regex-operators).

![Image 9: Heatmaps for group of pages.](https://learn.microsoft.com/en-us/media/heatmaps-for-group-of-pages.png)

Note

For the default operator, **is exactly**, heat maps are displayed on the pages that match specified URLs. For example, `http://www.contoso.com` and `https://www.contoso.com` are different. The applied filters are combined for all the pages that match the conditions.

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#supported-filters)
Supported filters
-----------------

Heat maps filters help you sort and customize your visualizations. For more information, see [Filters overview](https://learn.microsoft.com/en-us/filters/clarity-filters).

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#segments)
Segments
--------

To avoid multiple manual typings, you can save groups of filters into segments. Segments can be applied to filters and custom tags. For more information, see [Segments](https://learn.microsoft.com/en-us/filters/clarity-segments).

Important

You can't use GA segments on Heatmaps.

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#compare-heatmaps)

Color coding in a Heatmap
-------------------------

Clarity uses color scale representation to understand the user interaction with your website better. The color scale allows you to view your user movement throughout the website.

![Image 10: color coding scale in Heatmaps.](https://learn.microsoft.com/en-us/media/hot-cold_scale.png)

*   **Red** indicates hot areas where users interact more that's most popular.

*   **Blue** indicates cold areas where the users interact the least that is least popular.

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#view-recordings)
View Recordings
---------------

You can also view the recordings of clicked data to understand and analyze user clicks. Learn more on [how to view recordings](https://learn.microsoft.com/en-us/clarity/click-maps#view-recordings).

Note

'View recordings' is only available for Click maps.

Heatmaps limitations
--------------------

*   Heat maps are only displayed on the site pages where the Clarity tracking code is installed. Check the [Setup](https://learn.microsoft.com/en-us/setup-and-installation/clarity-setup) process to know how to install it.
*   Heat maps data is retained for up to 13 months.
*   Heat maps are limited to up to 100,000 page views per heat map.
*   Clarity doesn't support capturing screenshots of different dynamic sections, such as a menu that opens as you hover/click. A heat map won't display clicks or scrolls collected over these dynamic elements.
*   Clarity uses CSS element selectors to create a heat map and chooses a random playback to overlay the heat map on the top. The page state depends on the playback. 
*   Clarity scroll maps are built for the `<body>` tag. Clarity can't show scroll map inside the `<div>` container.
*   Clarity can only render scroll maps for pages with a maximum height of 65,535 pixels.
*   You can't use GA segments on Heatmaps.

[](https://learn.microsoft.com/en-us/clarity/heatmaps-features#faq)
