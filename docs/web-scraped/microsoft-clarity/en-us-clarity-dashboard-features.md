Overview
--------

The Clarity dashboard provides aggregate metrics to help you get an overall understanding of the traffic on your site. At a glance, you can see:

*   Users who were clicking on nonexistent links.
*   Users who scrolled up and down a page searching for something they couldn't readily find.
*   Concurrent JavaScript errors that occur across your clients.
*   Time a user spends navigating on your site.

The statistics provided can be sliced using filters and segments, which allow you to drill down on specific user behavior patterns. This can provide the basis to make good decisions on modifying your site.

![Image 16: Clarity dashboard.](https://learn.microsoft.com/en-us/media/dashboard.png)

Note

The dashboard displays data based on the user's local machine timezone.

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#customize-dashboard)
Customize Dashboard
-------------------

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#switch-dashboards)
### Switch dashboards

Clarity now supports multiple dashboards. As you hover through **Dashboards**, you can view the dropdown menu. This allows you to effortlessly navigate between dashboards ensuring continuity in your analysis. Any filters or segments applied are retained across dashboards, providing a consistent view of your data. The supported dashboards are:

*   Clarity
*   [Advertising](https://learn.microsoft.com/en-us/advertising-dashboard/ad-getting-started)
*   [Google Analytics](https://learn.microsoft.com/en-us/ga-integration/ga-dashboard)
*   [Agent Performance](https://learn.microsoft.com/en-us/agents-dashboard/agent-dashboard-overview)

The Clarity dashboard is the default view and provides an overview of user interactions and insights specific to your project. It is automatically available for all projects without any additional setup. The other dashboards, Advertising, Google Analytics and Agent Performance, are accessible only if your project has been integrated with these services.

![Image 17: Clarity dashboards.](https://learn.microsoft.com/en-us/media/new-dashboards.png)

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#rearrange-cards)
### Rearrange cards

The cards on the dashboard are customizable. Choose the **hamburger menu** icon from the left of the card and drag it to rearrange according to your preferences.

![Image 18: Rearrange card.](https://learn.microsoft.com/en-us/media/rearrange-widget.gif)

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#hide-card)
### Hide card

Note

When downloading dashboard, hidden cards are not visible.

You can hide a card by selecting **Menu** ->**Hide card**.

![Image 19: Hide card.](https://learn.microsoft.com/en-us/media/hide-card.png)

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#access-hidden-cards)
#### Access hidden cards

Hidden cards can be accessed at the bottom of the dashboard and easily reinstated by selecting **Unhide card**.

![Image 20: Unhide card.](https://learn.microsoft.com/en-us/media/unhide-card.png)

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#set-a-default-tab)
### Set a default tab

For cards containing multiple tabs, you can designate a default tab by choosing **Menu** ->**Select default tab**. The initial tab is automatically set as the default.

![Image 21: Set default tab in a card.](https://learn.microsoft.com/en-us/media/set-default-tab-card.png)

[](https://learn.microsoft.com/en-us/clarity/dashboard-features#watchlist)
Watchlist
---------

Note

Your watchlist is unique to you and not shared with your team.

The Watchlist card helps you track key user interactions and the data that matters most to you. You can easily monitor specific segments, custom tags, or labels by adding them to your watchlist. Once an item is added, the card displays the percentage of sessions that match your selection. Applying filters updates this percentage.

![Image 22: Watchlist.](https://learn.microsoft.com/en-us/media/watchlist.png)

To get started, select **Add to watchlist**.

![Image 23: Add to watchlist.](https://learn.microsoft.com/en-us/media/add-to-watchlist.png)

Choose the type of data to track from the list of **Segment, Custom tag, or Label**.

![Image 24: Watchlist type.](https://learn.microsoft.com/en-us/media/choose-watchlist-data-type.png)

For example, if you add `Segments with source`, your watchlist displays the percentage of sessions where users visited the page with a specific source within the last three days. To view this, go to **Segment** -> select the corresponding segment from the dropdown ->**Add**.

![Image 25: Segment watchlist.](https://learn.microsoft.com/en-us/media/segment-watchlist.png)

Similarly, you can add a Custom Tag by selecting a tag from the dropdown or a Label by choosing a label from the dropdown, and finally selecting **Add**.

![Image 26: Adding other to watchlist.](https://learn.microsoft.com/en-us/media/label-and-custom-tag-watchlist.gif)
