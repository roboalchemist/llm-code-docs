| Term | Definition |
| --- | --- |
| [ANID](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for Anonymous ID. |
| API | Stands for Application Programming Interface. |
| [CLID](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for Clarity ID. |
| [CCPA](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | California Consumer Privacy Act (CCPA) is a California law governing personal information of California residents. Learn more [here](https://www.microsoft.com/en-ca/privacy/ccpa) and from [Microsoft Privacy statement](https://www.microsoft.com/privacy/privacystatement). |
| Country/Region | In this document, the term "country/region" refers to geographical areas. The terms "country" and "region" are treated equivalently in this context. |
| GDPR | General Data Protection Regulation (GDPR); a set of EU regulations on data protection and privacy. Learn more about [Microsoft Privacy statement](https://www.microsoft.com/privacy/privacystatement). |
| [GUID](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for Globally Unique Identifier. |
| HW | Stands for Hardware. |
| [JWT](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for JSON Web Token. |
| [MUID](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for Microsoft User ID. |
| [PII](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for Personally Identifiable Information. |
| Popular pages | Pages popular by number of page views. |
| Site | Synonymous to website. |
| Traffic | A user movement from another domain, email, app, or any other channel to your website. |
| URL | The complete web address used to find a particular web page. |
| Session | The amount of time a user spent on your site. A single user can open multiple sessions on the same day or over several days, weeks, or months. A session ends if the user is inactive for 30 minutes. |
| SDK | Stands for Software Development Kit. |
| [SEO](https://learn.microsoft.com/en-us/clarity/glossary-of-terms) | Stands for SSearch Engine Optimization. |
| User inactivity | The amount of time a user left your site in a session. For example, if a user visited from [https://www.contoso.com](https://www.contoso.com/) to [https://www.fabrikam.com](https://www.fabrikam.com/) and return to [https://www.contoso.com](https://www.contoso.com/), the time spent on [https://www.fabrikam.com](https://www.fabrikam.com/) would be inactive time for [https://www.contoso.com](https://www.contoso.com/). |
| Vertical Scroll | Scrolling in opposite directions (up and down). |
| Webpage | Synonymous to page. |
| Website | Group of pages. |
| Widget | Synonymous to card. |
