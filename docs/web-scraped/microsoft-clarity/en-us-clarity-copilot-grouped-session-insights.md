Grouped Session Insights with Copilot in Clarity helps you quickly understand the user journeys across your website. Using generative AI, this feature summarizes multiple session recordings at a time. You can generate these insights for any group of session recordings, using Clarity filters like events, device type, UTM campaign and more.

[](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#how-to-view-grouped-session-insights)
How to view Grouped Session Insights?
-------------------------------------

Important

Copilot is only available on the sites where the Clarity tracking code is installed. Check the [Setup](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup) process for how to install it.

You can generate grouped session insights in two ways:

*   [Summarize top 10 recordings](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#summarize-top-10-recordings)
*   [Summarize custom selected recordings](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#summarize-custom-selected-recordings)

[](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#summarize-top-10-recordings)
### Summarize top 10 recordings

This method allows you to quickly generate a summary of the top 10 recordings in the playlist view.

1.   Go to **Projects** ->**Recordings**. Select the **Summarize Recordings** button on the top right.

![Image 16: Summarize recordings button in Recordings.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step1.png)

2.   Select the **Top 10 recordings** option in the drop-down list.

![Image 17: Select top 10 recordings.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step2.png)

3.   Instantly, view the summary of these 10 recordings.

![Image 18: View summary.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step3.png)

4.   To watch the recordings mentioned in each callout, select on the numbers at the end of the callout or open the **Summarized recordings** module.

![Image 19: Interact with the generated Grouped Session Insights.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step4.png)

5.   Share your feedback using the thumbs up/down button or **Share feedback** button.

![Image 20: Share feedback.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step5.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#summarize-custom-selected-recordings)
### Summarize custom selected recordings

This method allows you to custom-select recordings for summarization.

1.   Go to **Projects** ->**Recordings**. Select the **Summarize Recordings** button on the top right.

![Image 21: Summarize recordings button in Recordings.](https://learn.microsoft.com/en-us/clarity/media/gsi-option1-step1.png)

2.   Select the **Custom select** option in the drop-down list.

![Image 22: Custom select recordings.](https://learn.microsoft.com/en-us/clarity/media/gsi-option2-step1.png)

3.   Select the recordings of interest.

Note

The number of recordings you can summarize at once with this option changes based on the recordings selected. Once you reach the upper limit, you can't select more recordings. 
![Image 23: Selection mode for Grouped Session Insights.](https://learn.microsoft.com/en-us/clarity/media/gsi-option2-step2.png)

4.   Select the **Generate** button. Instantly, view the summary of these recordings.

![Image 24: View the summary of recordings.](https://learn.microsoft.com/en-us/clarity/media/gsi-option2-step3.png)

5.   To watch the recordings mentioned in each callout, select on the numbers at the end of the callout or open the **Summarized recordings** module.

![Image 25: Select the numbers at the end of callout.](https://learn.microsoft.com/en-us/clarity/media/gsi-option2-step4.png)

6.   Share your feedback using the thumbs up/down button or **Share feedback** option.

![Image 26: Share feedback option.](https://learn.microsoft.com/en-us/clarity/media/gsi-option2-step5.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/grouped-session-insights#limitations)
Limitations
-----------

Note

Clarity uses Generative AI models to power Copilot features. It is well-known that Generative AI models can sometimes misinterpret or generate incorrect information. Share your feedback using the thumbs up/down button or email us at [clarityMS@microsoft.com](mailto:clarityMS@microsoft.com) to help us learn and improve.

*   Currently, Grouped Session Insights aren't supported for Mobile Apps.
*   Grouped Session Insights can't summarize all your project's recordings at once.
*   Sharing and downloading insights aren't supported.
