Install Brand agents from Shopify
=================================

Note

*   Brand agents is currently in beta. Join the waitlist to be informed when you're eligible to test them.
*   Currently, Brand agents is only available for Shopify sites.
*   Currently, Brand agents is only available for websites and not for mobile apps.
*   Refer to the [Brand agents terms](https://clarity.microsoft.com/terms/agents) for detailed usage guidelines.

This guide walks you through installing Brand agents directly from your Shopify store. This is the recommended approach if you're setting up Clarity and Brand agents for the first time on Shopify, as it streamlines the installation and configuration process within the Shopify interface.

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/shopify-install-guide#step-1-install-the-clarity-shopify-app)
### Step 1: Install the Clarity Shopify app

Sign in to Shopify and go to the [Shopify App Store](https://apps.shopify.com/). Search and select [**Microsoft Clarity: AI Insights**](https://apps.shopify.com/microsoft-clarity) ->**Install**.

![Image 11: Install Clarity from Shopify store.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-1.png)

For a detailed step-by-step guide, refer to the [Clarity Shopify App installation guide](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify).

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/shopify-install-guide#step-2-access-brand-agents-settings)
### Step 2: Access Brand agents settings

To access the Brand agents experience, select **Go to brand agents settings** in your Clarity dashboard.

![Image 12: Access settings from Clarity platform.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-2.png)

Note

After installation, it might take a few hours for data to appear in your dashboard.

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/shopify-install-guide#step-3-join-the-waitlist)
### Step 3: Join the waitlist

To join the waitlist, enable the **Clarity Agents JS** app embed block in your Shopify theme using the following steps:

1.   Select **Go to store settings**. This opens a new tab.

![Image 13: Go to settings.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-3-a.png)

2.   In the new tab, go to your app embed block settings.

3.   Ensure both **Clarity JS** and **Clarity Agents JS** toggles are turned **ON**.

4.   Select **Save**.

![Image 14: Enable toggles.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-3-b.png)

5.   Return to your Clarity App tab. The page refreshes automatically. If it doesn't, select **Refresh**.

![Image 15: Refresh App page.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-3-c.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/shopify-install-guide#step-4-await-confirmation-email)
### Step 4: Await confirmation email

You're now on the Brand agents waitlist. You receive an email when you're approved and ready to get started.

![Image 16: Waitlist confirmation.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-shopify-install-step-4.png)

Tip

If you decide not to participate in the waitlist, select **Take me off the waitlist**. This removes your application and revokes all permissions provided during registration.

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/shopify-install-guide#step-5-configure-and-publish-your-agent)
### Step 5: Configure and publish your agent

When you receive your confirmation email, return to configure final details, test your agent, and publish it. Check how to [configure Brand agents](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents) for more details.

[Configure Brand agents](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents)
