Clarity's custom tags are customizable filters that allow you to analyze recordings and heatmaps in different directions. You can easily create custom tags and share them across your team. You can pass additional information through custom tags.

There are no limits on the number of tags you can add to your project.

The following are some of the scenarios where you can use custom tags:

*   Show recordings for a specific user.
*   Record or analyze and search through custom fields.
*   Spot users by an ID, one that is significant to your application database.
*   Rename userID to a meaningful name that allows you to search through them easily.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#where-do-custom-tags-show-up)
Where do custom tags show up?
-----------------------------

You can create custom tags accessible through the Recordings, Heatmaps, and Dashboard sections of Clarity. You can find custom tags under the **Filters** section.

![Image 10: Custom tags for Heatmaps.](https://learn.microsoft.com/en-us/media/filters-custom.png)

You can also find the custom tag in the [event details](https://learn.microsoft.com/en-us/session-recordings/inline-player#events) of a session recording.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#how-to-setup-custom-tags)
How to setup custom tags?
-------------------------

You can create your own filters by setting tags in your website code.

*   The tags can be any alphanumeric value you like including spaces. For example, **Checkout Page1**.
*   The tag and its value can't be longer than 255 characters.
*   A single page can have no more than 128 tags. Any other tags are ignored.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-1)
#### Step 1

Identify the information you want to send to Clarity. This information should be a key-value pair.

**Key**: The tag name that identifies the kind of information you're sending.

**Value**: The value attached to the tag.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-2)
#### Step 2

Add this code snippet into your website code:

`clarity("set", "key", "value");`

**Example**:

To create custom tags to filter out recordings based on flight, the key can be `flight` and value can be `testflight1`.

So, the code snippet would be - `clarity("set", "flight", "testflight1");`

You can also add multiple tags to a project.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#add-custom-tags-to-your-project)
Add custom tags to your project
-------------------------------

As you add the tag in your website code, it's updated in your project within 30 minutes to 2 hours. Meanwhile, you can view the custom tag in the [event details](https://learn.microsoft.com/en-us/session-recordings/inline-player#events) of a session recording.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-1-1)
#### Step 1

To start using the tag, select a project feature and go to **Filters**. Select the tag under **Custom tags**.

![Image 11: Select custom tags for Heatmaps.](https://learn.microsoft.com/en-us/media/add-custom-tags-1.png)

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-2-1)
#### Step 2

Select the tag value. You can view the applied custom tag badges at the top as shown here.

![Image 12: Add custom tags for Heatmaps.](https://learn.microsoft.com/en-us/media/add-custom-tags-2.png)

Select **Apply**.

[](https://learn.microsoft.com/en-us/clarity/custom-tags#remove-a-custom-tag-filter)
### Remove a custom tag filter

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-1-2)
#### Step 1

Go to **Filters** ->**Custom tags**.

![Image 13: Select custom tags to delete.](https://learn.microsoft.com/en-us/media/delete-custom-tags-1.png)

[](https://learn.microsoft.com/en-us/clarity/custom-tags#step-2-2)
#### Step 2

Select the **Delete icon** next to the tag.

![Image 14: Delete custom tags.](https://learn.microsoft.com/en-us/media/delete-custom-tags-2.png)

[](https://learn.microsoft.com/en-us/clarity/custom-tags#setup-javascript-triggers-through-google-tag-manager)
Setup JavaScript triggers through Google Tag Manager
----------------------------------------------------

If you use Google Tag Manager to load the Clarity script on your site, you can set up JavaScript triggers that fire the Clarity tracking code.

To set it up, review [Google Tag Manager](https://learn.microsoft.com/en-us/third-party-integrations/google-tag-manager).

Check out this guide on[how to set up JavaScript Triggers in Google Tag Manager](https://support.google.com/tagmanager/answer/7679316?hl=en).
