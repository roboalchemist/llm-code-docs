Choose **Exclude selection** box to exclude these filters instead of including them and select **Apply** to apply the changes. Uncheck the box to include the filters instead. Clarity offers exclusion for the following filters:

Expand table

| Filter group | Supported filters | Options |
| --- | --- | --- |
| [User info](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-by-user-info) | Browser Operating system Location | Select multiple Select multiple Select multiple |
| [User actions](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-by-user-actions) | Smart events | Select multiple `Or` `And` |
| [Traffic](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-by-traffic) | Campaign Medium Source | Select multiple Select multiple Select multiple |
| [Page](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-by-page) | JS errors | Select multiple `Or` `And` |
| [Product](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-by-product) | Brand | Select multiple |
| [Custom filters](https://learn.microsoft.com/en-us/clarity/clarity-filters#filtering-with-custom-filters) | Custom labels Custom tags, key, and value Custom ID - user ID, page ID, session ID | Select multiple Select multiple Select multiple |
