Set up Clarity on Shopify - new installation
--------------------------------------------

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-1)
### Step 1

Sign in to Shopify and go to the **[Shopify App Store](https://apps.shopify.com/)**. Search and select **[Microsoft Clarity: AI Insights](https://apps.shopify.com/microsoft-clarity)** listing ->**Install**.

![Image 15: Go to shopify store and install clarity.](https://learn.microsoft.com/en-us/clarity/media/shopify-step1.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-2)
### Step 2

On the pop-up window, review the data access permissions and select **Install**.

![Image 16: Review data access permissions.](https://learn.microsoft.com/en-us/clarity/media/shopify-step2.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-3)
### Step 3

Note

If you already manually pre-installed Clarity, go to the [pre installed set up](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#clarity-setup-for-shopify-with-pre-installed-snippet).

If this is your first time installing Clarity, activate the Clarity JS to get started. Select **Go to app settings** and a new tab opens.

![Image 17: Go to app settings.](https://learn.microsoft.com/en-us/clarity/media/shopify-step3.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-4)
### Step 4

On the new tab, ensure that the **Clarity JS** is toggled on, and select **Save**. Close this tab and return to the installation page.

![Image 18: Toggle the Clarity JS switch.](https://learn.microsoft.com/en-us/clarity/media/shopify-step4.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-5)
### Step 5

The dashboard should refresh automatically. If it doesn't update the view, select **Refresh**.

![Image 19: Refresh to update the settings.](https://learn.microsoft.com/en-us/clarity/media/shopify-step5.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-6)
### Step 6

Clarity is now successfully installed. Allow a few hours for your dashboard to populate with data. For quick access, consider pinning the app to your side panel.

![Image 20: View Clarity dashboard on Shopify.](https://learn.microsoft.com/en-us/clarity/media/shopify-step6.png)

[Verify your installation](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup#verify-your-installation)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#clarity-setup-for-shopify-with-pre-installed-snippet)
Clarity Setup for Shopify with Pre-Installed Snippet
----------------------------------------------------

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-1-1)
### Step 1

Note

If you're installing Microsoft Clarity from the beginning, refer [new installation](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#set-up-clarity-on-shopify---new-installation).

If you previously installed Clarity prior to the app, you need to connect your existing project. Select **Connect Project** and an email is sent to the project admin.

![Image 21: Select send configuration email.](https://learn.microsoft.com/en-us/clarity/media/shopify-step7.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-2-1)
### Step 2

Follow the link on the email and sign-in to Clarity to connect the project.

![Image 22: Follow the link on email.](https://learn.microsoft.com/en-us/clarity/media/shopify-step8.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-3-1)
### Step 3

Go back to the installation page. The dashboard should refresh automatically. If it doesn't update the view, select **Refresh**.

![Image 23: Refresh the installation page.](https://learn.microsoft.com/en-us/clarity/media/shopify-step9.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/shopify#step-4-1)
### Step 4

Clarity is now successfully installed. Allow a few hours for your dashboard to populate with data. For quick access, consider pinning the app to your side panel.

![Image 24: View Clarity dashboard on Shopify.](https://learn.microsoft.com/en-us/clarity/media/shopify-step6.png)

[Verify your installation](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup#verify-your-installation)
