Note

Clarity is free forever. You'll never be asked to upgrade to a paid version. No credit card is required to sign up. Check the [setup instructions](https://learn.microsoft.com/en-us/setup-and-installation/getting-started) to learn more.

Use Clarity's new extension to quickly view instant Heatmaps on your live site and watch recent session recordings on any page. It helps you understand your user views and is available on any website with Clarity tracking code installed.

Understanding your user navigation, interaction, and browsing pattern can provide new insights about your users. Empathize with your users and see where features fail or succeed to improve your product, grow revenue, and improve user retention. Clarity provides you with all these insights by observing content layout and user's interactions with the page.

Setting up Clarity extension is easy and consists of two steps.

1.   [Install the extension](https://learn.microsoft.com/en-us/clarity/clarity-live-extension#install-the-extension)
2.   [Use Clarity Live](https://learn.microsoft.com/en-us/clarity/how-to-use-clarity-live-extension)

Install the extension
---------------------

[](https://learn.microsoft.com/en-us/clarity/clarity-live-extension#on-microsoft-edge)
### On Microsoft Edge

1.   Open the Microsoft Edge browser. Go to **[Extensions](https://microsoftedge.microsoft.com/addons)** and search for **[Clarity Live](https://microsoftedge.microsoft.com/addons/detail/microsoft-clarity-live/bngpiglbalmenaabohcooocpnljgfemj)**.

![Image 13: Select Clarity live extension in Microsoft Edge extensions.](https://learn.microsoft.com/en-us/media/clarity-live-extension-search-edge.png)

1.   Open the extension. Select **Get**.

![Image 14: Select get in the edge.](https://learn.microsoft.com/en-us/media/add-extension-to-edge.png)

2.   Confirm **Add extension** on the pop-up window.

![Image 15: Confirm add extension on pop-up window in edge.](https://learn.microsoft.com/en-us/media/confirm-add-extension-edge.png)

3.   Upon successful addition, you can find Clarity in the extensions panel with a prompt as shown.

![Image 16: Successfully added the extension on Microsoft Edge.](https://learn.microsoft.com/en-us/media/extension-add-successful-edge.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-live-extension#on-google-chrome)
### On Google Chrome

1.   Open the Google Chrome web store. Go to **[Extensions](https://chromewebstore.google.com/category/extensions)** and search for **[Clarity Live](https://chromewebstore.google.com/detail/microsoft-clarity-live/cjfdbemmaeeohgibnhdhlakiahifjjcf)**.

![Image 17: Select Clarity live extension in Google chrome web store extensions.](https://learn.microsoft.com/en-us/media/clarity-live-extension-search.png)

Tip

You can now add extensions from Chrome Web Store to Microsoft Edge. Select 'Add to Chrome'.

1.   Open the extension. Select **Add to Chrome**.

![Image 18: Select add to chrome.](https://learn.microsoft.com/en-us/media/add-extension-to-chrome.png)

2.   Confirm **Add extension** on the pop-up window.

![Image 19: Confirm add extension on pop-up window.](https://learn.microsoft.com/en-us/media/confirm-add-extension.png)

3.   Upon successful addition, you can find Clarity in the extensions panel with a prompt as shown.

![Image 20: Successfully added the extension on Chrome.](https://learn.microsoft.com/en-us/media/extension-add-successful.png)

[Checkout Clarity Live](https://chrome.google.com/webstore/detail/microsoft-clarity-live/cjfdbemmaeeohgibnhdhlakiahifjjcf)
