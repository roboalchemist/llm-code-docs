Dashboard
---------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-dashboard#the-dashboard-displays-recordings-for-a-visited-url-however-when-i-select-the-recordings-icon-there-are-no-recordings-why)
### The dashboard displays recordings for a visited URL. However, when I select the recordings icon, there are no recordings. Why?

Clarity session replay is a reconstructed presentation of how a user experiences a website. These sessions can't be visualized as the data received is insufficient to display a full playback experience. The issue is prevalent when limited sessions are in a filter set. Expand the filters to increase the number of sessions in the results, thereby mitigating the issue.

E-Commerce Insights
-------------------

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-dashboard#how-do-i-start-getting-data-for-the-product-information-filters)
### How do I start getting data for the Product Information filters?

Clarity enables [Product Information filters](https://learn.microsoft.com/en-us/clarity/e-commerce-insights#product-information-filters) by using your site's Product JSON LD instrumentation. If you see no data for these filters, you need to add this instrumentation for your products. See [Product JSON-LD Schema](https://jsonld.com/product/).

To enable the full set of Product Information filters, make sure the following six fields are present in your JSON LD instrumentation:

*   Price
*   Brand
*   Product name
*   Availability
*   Rating
*   Number of ratings

Once you add this instrumentation, your Product Information filters should be available in about an hour.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-dashboard#how-do-i-start-getting-data-for-the-most-viewed-products-card)
### How do I start getting data for the Most Viewed Products card?

Clarity enables the [Most Viewed Products card](https://learn.microsoft.com/en-us/clarity/e-commerce-insights#most-viewed-products-card) by detecting the Product name field of your site's Product JSON LD instrumentation. If you see no data for this card, you need to add this field for your products. See [Product JSON-LD Schema](https://jsonld.com/product/).

Once you add this instrumentation, your Product Information filters should be available in about an hour.

[](https://learn.microsoft.com/en-us/clarity/troubleshooting-dashboard#why-does-my-most-viewed-products-widget-show-incorrect-data)
### Why does my Most Viewed Products widget show incorrect data?

Clarity enables [Most Viewed Products](https://learn.microsoft.com/en-us/clarity/e-commerce-insights#most-viewed-products-card) by detecting the Product name field of your site's Product JSON LD. Any variances can cause corresponding deviations on the widget values. Some tips:

*   Make sure that your LD schema is a valid JSON object. For example, check special characters.
*   Ensure that instrumentation is present on the product pages you would like to see the data.
