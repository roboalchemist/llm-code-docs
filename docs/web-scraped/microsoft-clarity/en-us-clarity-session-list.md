Important

Clarity retains recordings for 30 days from the time of recording. However, **Favorite** recordings and a randomly selected recording are retained for up to 13 months.

The session list showcases Clarity's recordings of your user sessions, ordered by default based on recency. The list is in the left panel in the Recordings section.

![Image 12: Session list.](https://learn.microsoft.com/en-us/media/session-list.png)

The content of the list varies according to the filters you set. Use restrictive criteria to see a targeted and focused list.

[](https://learn.microsoft.com/en-us/clarity/session-list#live-recordings)
Live recordings
---------------

Start viewing live recordings that recreate users interactions with your website, including mouse movements, clicks, scrolls, navigation, and more, through the list of live users. Learn more on [recordings in real time](https://learn.microsoft.com/en-us/clarity/clarity-real-time).

[](https://learn.microsoft.com/en-us/clarity/session-list#view-session-cards)
View session cards
------------------

Each card represents a separate session recording. Play a recording by selecting its card. Each card contains relevant metadata to help you decide which recording to watch.

![Image 13: Session list card.](https://learn.microsoft.com/en-us/media/session-list-card.png)

Let's explore the card in detail.

Expand table

| Display | Definition |
| --- | --- |
| Entry URL | The page URL where the user entered the site. This URL marks the beginning of a session. |
| Exit URL | The page URL where the user left the site. This URL marks the ending of a session. |
| Timestamp | The time of day when the user entered the site. |
| Date | The specific day when the user was on the site. |
| Referrer | The referring site where the URL of a specified page matches the one the user visited before your site. |
| Session duration | The total amount of time spent in a session. |
| Click count | The total number of clicks in a session. |
| Page count | The total number of pages visited in a session. |
| User ID | Clarity user ID to include only results from specific user. You can also view [Visitor profile](https://learn.microsoft.com/en-us/clarity/visitor-profile) by selecting the ID. |
| Country | The specific country/region where the user session is recorded. |
| Operating system | The specific operating system where the user session is recorded. |
| Device | The specific device type where the user session is recorded. |

[](https://learn.microsoft.com/en-us/clarity/session-list#take-action-on-session-cards)
Take action on session cards
----------------------------

The bottom of each card contains icons that allow you to take action on the session.

Expand table

| Display | Action |
| --- | --- |
| Session Insights | Quickly view [Session Insights](https://learn.microsoft.com/en-us/copilot/session-insights) about the recordings created using Generative AI. |
| Favorite | Each card has a favorite icon. Favorite the recordings you want to keep for future analysis. Learn more about [Favorites](https://learn.microsoft.com/en-us/clarity/session-list#favorite-the-sessions). |
| Share | Share the session recording with others on your team. Learn more about [sharing](https://learn.microsoft.com/en-us/setup-and-installation/share-clarity). |
| View visitor profile | View the [Visitor profile](https://learn.microsoft.com/en-us/clarity/visitor-profile) of the specific user. |
| More details | See [more details](https://learn.microsoft.com/en-us/clarity/inline-player#view-details) about the session recording. |

[](https://learn.microsoft.com/en-us/clarity/session-list#favorite-the-sessions)
Favorite the sessions
---------------------

Favorites are recordings you want to keep for analysis or to share with others. You can favorite a recording from the session list, the inline player, or by adding a note (which automatically marks the recording as a favorite).

*   Add session cards to your favorites list by selecting the **star icon** on any card. When selected, the card appears in the Favorites tab. To view a favorite recording, select the Favorites tab.

*   Remove a card from the favorites list by deselecting the star.

Note

You and your team members can view your favorite recordings. Clarity automatically removes sessions from your favorites list after one year.

[](https://learn.microsoft.com/en-us/clarity/session-list#view-favorite-recordings)
### View favorite recordings

Important

Clarity retains **Favorite** recordings for up to 13 months.

![Image 14: Session list favorites.](https://learn.microsoft.com/en-us/media/session-favorites.png)

Each card has a favorite icon. As you select the icon, you can view the recording under the **Favorite recordings** tab. There's no limit on the number of favorited sessions. You can sort these recordings using the [Sort](https://learn.microsoft.com/en-us/clarity/session-list#sort-the-sessions) tab.

This tab contains favorited recordings to applied filters. For example, when you add the '7 days' time filter, you can view favorite recordings from the last seven days.

[](https://learn.microsoft.com/en-us/clarity/session-list#how-can-i-view-all-favorited-recordings)
#### How can I view all favorited recordings?

Try the following, if you don't find all favorited recordings:

*   Clear all the filters.

*   Change the date range to the last 12 months.

[](https://learn.microsoft.com/en-us/clarity/session-list#watched-or-unwatched-state)
Watched or unwatched state
--------------------------

The color of a session card's outline indicates its state.

Expand table

| Outline | Meaning | Image |
| --- | --- | --- |
| Normal | An unwatched recording. | ![Image 15: Unwatched recording.](https://learn.microsoft.com/en-us/media/unwatched-recording.png) |
| Bold | A recording you're currently watching. | ![Image 16: Current recording.](https://learn.microsoft.com/en-us/media/current-recording.png) |
| Dimmed | A recording you watched. | ![Image 17: Watched recording.](https://learn.microsoft.com/en-us/media/watched-recording.png) |

Note

Currently, you can't filter recordings by their watched or unwatched state.

[](https://learn.microsoft.com/en-us/clarity/session-list#sort-the-sessions)
Sort the sessions
-----------------

The list shows the top number of sessions. By default, they're ordered by most recent. Sort them by selecting **Sort**.

![Image 18: Sorting in session list.](https://learn.microsoft.com/en-us/media/session-list-sort.png)

Available sorting options are:

*   Date
*   Duration
*   Clicks
*   Pages
