Session insights help you understand the user journey across your website. Without requiring you to watch time-consuming recordings to discover the key learnings from a session replay, you can now instantly view insights about the recordings. Copilot in Clarity generates critical session takeaways in plain language using Generative AI. This allows you to save time, especially with longer session recordings and thus be more productive.

[](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#how-to-view-session-insights)
How to view Session Insights?
-----------------------------

Note

Copilot is only available on sites where the Clarity tracking code is installed. Check the [Setup](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup) process to know how to install it.

You can view session insights in two ways:

*   [View in Session list](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#view-in-session-list)
*   [View in More details](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#view-in-more-details)

[](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#view-in-session-list)
### View in Session list

1.   Go to a **Project** ->**Recordings**. ![Image 15: Recordings tab in a project.](https://learn.microsoft.com/en-us/clarity/media/recordings-tab.png)

2.   Select the **Generate session insights** icon on any recording. ![Image 16: Select insights icon to toggle off.](https://learn.microsoft.com/en-us/clarity/media/select-insights-icon.png)

3.   Instantly, view a description of what transpired in this session without watching the whole session. ![Image 17: View insights in session list.](https://learn.microsoft.com/en-us/clarity/media/view-insights-in-session-list.png)

4.   Reselect the **Generate session insights** icon to toggle off the insights. ![Image 18: Select insights icon to toggle off.](https://learn.microsoft.com/en-us/clarity/media/select-insights-icon.png)

5.   Select the **Copy** icon to copy the insights. ![Image 19: Select copy insights icon.](https://learn.microsoft.com/en-us/clarity/media/copy-insights-icon.png)

6.   You can share your feedback using the thumbs up/down button. ![Image 20: Select feedback icon.](https://learn.microsoft.com/en-us/clarity/media/insights-feedback-icon.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#view-in-more-details)
### View in More details

1.   Go to a **Project** ->**Recordings**. ![Image 21: Recordings tab in a project.](https://learn.microsoft.com/en-us/clarity/media/recordings-tab.png)

2.   Select the **More details** on any recording. ![Image 22: Select more details icon.](https://learn.microsoft.com/en-us/clarity/media/more-details-icon.png)

3.   Select the **Insights** tab to view the generated insights. ![Image 23: Select insights tab.](https://learn.microsoft.com/en-us/clarity/media/insights-tab.png)

4.   You can share your feedback using the thumbs up/down button. ![Image 24: Select feedback icon in insights tab.](https://learn.microsoft.com/en-us/clarity/media/insights-feedback-icon-insights-tab.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#limitations)
Limitations
-----------

*   Currently, Session Insights aren't supported for Mobile Apps.
*   Session Insights aren't available for sessions less than five seconds. As the session is too short to generate any meaningful sentences, they're grayed out.
*   Sharing and downloading insights aren't supported.

Note

Clarity uses Generative AI models to power Copilot features. It is well-known that Generative AI models can sometimes misinterpret or generate incorrect information. Share your feedback using a thumbs up/down button if you notice any incorrect information.

[](https://learn.microsoft.com/en-us/clarity/copilot/session-insights#faq)
