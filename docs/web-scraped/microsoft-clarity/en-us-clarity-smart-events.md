What are smart events?
----------------------

Smart events capture specific user interactions on your site by combining variety of signals that provide simple and intuitive user actions that occur during a session. With smart events, you can filter to the specific user actions and group different actions together. For example, you can utilize click data, page views, and other session signals to create higher level events such as "Purchase", "Contact Us", or "Add to Cart".

[](https://learn.microsoft.com/en-us/clarity/smart-events#why-are-smart-events-useful)
Why are smart events useful?
----------------------------

Smart events track key user actions on your website, making it easy and fast for you to view the data that matters. Smart events also help you track different clicks across your website as a single action. For instance, Sign-in across pages can be grouped together to view as one sign-in action. Clarity automatically defines smart events for you, and you can customize them or define new events–all code-free.

[](https://learn.microsoft.com/en-us/clarity/smart-events#different-types-of-smart-events)
#### Different types of smart events

*   Button Clicks 
*   [API events](https://learn.microsoft.com/en-us/clarity/clarity-api#add-custom-events)
*   Auto events 
*   Page visits

View smart events
-----------------

[](https://learn.microsoft.com/en-us/clarity/smart-events#dashboard)
### Dashboard

You can find the automatically generated smart events in the dashboard. Select the recordings or heatmaps icon to view the filtered data.

![Image 15: Smart events in dashboard.](https://learn.microsoft.com/en-us/media/smart-events-dashboard.png)

[](https://learn.microsoft.com/en-us/clarity/smart-events#recordings-and-filters)
### Recordings and filters

Smart events are highlighted on playback controls and are available in **Recordings** ->**More details** ->**Events**.

![Image 16: Smart events in recordings.](https://learn.microsoft.com/en-us/media/smart-events-recordings.png)

[](https://learn.microsoft.com/en-us/clarity/smart-events#create-a-new-smart-event)
Create a new smart event
------------------------

Important

Only admins of a project can create smart events. A maximum of 20 custom smart events can be created.

You can create new smart events completely code-free via **Settings** ->**Smart events** ->**New event**. You can either begin with Clarity's automatic events or start from scratch. To create new events using code, refer to [API events](https://learn.microsoft.com/en-us/clarity/clarity-api#add-custom-events).

Once you create a new smart event, it starts showing up on the dashboard after a website user engages in that defined action.

[](https://learn.microsoft.com/en-us/clarity/smart-events#choosing-an-auto-event)
### Choosing an auto event

Clarity automatically surfaces any detected smart events that you can select to start creating with. As you select an auto event, select the **Next**. Choose a category from **Button clicks**, **API events**, or **Page visits** to add more events. Select **Next** to enter a name and **Save** the event.

![Image 17: Choose auto events.](https://learn.microsoft.com/en-us/media/choose-auto-event.gif)

[](https://learn.microsoft.com/en-us/clarity/smart-events#choosing-start-from-scratch)
### Choosing "Start from scratch"

If you choose **Start from scratch**, select **Next** to continue. Choose a category from **Button clicks**, **API events**, **Auto events**, or **Page visits** to add more events. Select **Next** to enter a name and **Save** the event.

![Image 18: Choose start from scratch.](https://learn.microsoft.com/en-us/media/choose-start-from-scratch.gif)

Once you create new smart events, they're shown up in **Settings** ->**Smart events** page as "Defined by the user".

[](https://learn.microsoft.com/en-us/clarity/smart-events#edit-smart-events)
Edit smart events
-----------------

Important

Only admins of a project can edit smart events.

You can edit smart events from the **Settings** vertical, based on how the smart event was defined.

![Image 19: Smart events in settings.](https://learn.microsoft.com/en-us/media/smart-events-settings.png)

Both Clarity-defined smart events and user-defined events can be edited no-code via your Clarity project. [API events](https://learn.microsoft.com/en-us/clarity/clarity-api#add-custom-events) are instrumented by code and must be edited through code.

You can also filter your view of smart events by how they were defined:

![Image 20: List of smart events.](https://learn.microsoft.com/en-us/media/smart-events-list.png)

[](https://learn.microsoft.com/en-us/clarity/smart-events#edit-auto-events)
### Edit auto events

Clarity automatically detects and groups user actions on your website into known events and are listed in the **Smart events** as "Defined by Clarity". You can customize these auto events. Clarity currently supports nine auto event types:

*   Purchase
*   Add to Cart
*   Begin Checkout
*   Contact Us
*   Submit Form
*   Request Quote
*   Sign Up
*   Login
*   Download

Select **Edit** icon or **Edit event** to start editing. Choose from **Button clicks**, **API events**, or **Page visits**. Select **Save** to finish editing. Once auto events are edited, they're categorized as "Clarity defined, edited" in the **Settings** ->**Smart events** page.

![Image 21: Edit automatic events.](https://learn.microsoft.com/en-us/media/edit-automatic-events.gif)

[](https://learn.microsoft.com/en-us/clarity/smart-events#edit-user-defined-events)
### Edit user-defined events

Events that are created by your project admin are surfaced in the Smart Events Settings as **Defined by User**. Select **edit** icon or the event to start editing. Choose from **Button clicks**, **API events**, **Auto events**, or **Page visits**. Select **Save** to finish editing.

![Image 22: Edit user-defined events.](https://learn.microsoft.com/en-us/media/edit-user-defined-events.gif)

[](https://learn.microsoft.com/en-us/clarity/smart-events#hide-or-delete-an-event)
Hide or Delete an event
-----------------------

Important

*   Hide is only available for auto events.
*   API events cannot be hidden or deleted through the settings page. To delete an API event, manually remove it in your website code.

[](https://learn.microsoft.com/en-us/clarity/smart-events#hide-an-event)
### Hide an event

Hide an auto event by selecting **Hide**, which prevents it from appearing on your dashboard and recordings. Re-enable it later by selecting **Unhide**. Hidden events are greyed out in the Smart Events list on the **Settings** page.

Note

A hidden auto event can still be used to create new events.

![Image 23: Select hide an event.](https://learn.microsoft.com/en-us/media/hide-an-event.png)

[](https://learn.microsoft.com/en-us/clarity/smart-events#delete-an-event)
### Delete an event

Important

Delete is only available for user defined events. Deleting an event permanently removes it. This action is irreversible and deleted event is unrecoverable.

Select the **delete** icon and confirm **Delete** on the pop-up window.

![Image 24: Select delete an event.](https://learn.microsoft.com/en-us/media/delete-an-event.png)
