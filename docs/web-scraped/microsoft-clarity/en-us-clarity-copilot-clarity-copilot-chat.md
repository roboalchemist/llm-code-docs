Chat with Copilot in Clarity to better understand your project analytics. Using Generative AI, Copilot provides instant, easy-to-understand answers to your dashboard questions.

[](https://learn.microsoft.com/en-us/clarity/copilot/clarity-copilot-chat#how-to-view-chat)
How to view Chat?
-----------------

Note

Copilot is only available on sites where the Clarity tracking code is installed. Learn [how to install Clarity](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup#install-tracking-code).

1.   Go to **Project** ->**Dashboard**.

2.   Select the **Copilot** icon on the top right.

![Image 9: Select copilot icon in dashboard.](https://learn.microsoft.com/en-us/clarity/media/select-copilot-icon-dashboard.png)

3.   To enter your question, start by selecting the input area to begin typing or select the **Prompt Guide** icon to browse through sample questions. Select the **Send** icon to submit your question. Copilot might take a few seconds to generate a response.

![Image 10: Select input area in copilot.](https://learn.microsoft.com/en-us/clarity/media/select-input-area-copilot.gif)

4.   If your response contains filters, select the **Dashboard** icon to filter your dashboard or the **Recordings** icon to navigate to the Recordings tab with the same filters applied.

![Image 11: Filter copilot summary to dashboard or recordings](https://learn.microsoft.com/en-us/clarity/media/filter-copilot.png)

5.   Share your feedback using the **thumbs up/down** button, or **copy** the response to Clipboard using the **Clipboard** button.

![Image 12: Share feedback or copy copilot summary.](https://learn.microsoft.com/en-us/clarity/media/feedback-or-copy-copilot.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/clarity-copilot-chat#limitations)
Limitations
-----------

Note

Clarity uses Generative AI models to power Copilot features. It is well-known that Generative AI models can sometimes misinterpret or generate incorrect information. Share your feedback using the thumbs up/down button or email us at [clarityms@microsoft.com](mailto:clarityms@microsoft.com) to help us learn and improve.

*   Currently, chatting with Copilot is based solely on your dashboard data. So, it doesn't support sharing links to specific Recordings or Heatmaps nor offering suggestions based on understanding your website design.

*   Sharing and downloading insights aren't supported.

[](https://learn.microsoft.com/en-us/clarity/copilot/clarity-copilot-chat#faq)
