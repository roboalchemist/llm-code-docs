To avoid multiple manual typings, you can save groups of filters into segments. Segments can be applied to filters and custom tags.

You can create, apply, delete, clear, and upgrade a segment from any of the following features:

*   Dashboard
*   Recordings
*   Heatmaps

Tip

There is no limit to the number of filters that you can apply to a segment, so feel free to add as many as you want.

[](https://learn.microsoft.com/en-us/clarity/clarity-segments#create-a-segment)
Create a Segment
----------------

Create new segments from any one of the Dashboard, Recordings, or Heatmaps. Your segments are available to everyone on your Project.

1.   Add a new Filter. To do so, go to **Filters** and select the required filters from the list of available filters. Select **Apply** to save the filters. ![Image 21: Apply a filter.](https://learn.microsoft.com/en-us/media/apply-a-filter.png)

2.   As you apply the filters, you can save them as a segment. To do so, select **Save as segment**. ![Image 22: Save as a segment.](https://learn.microsoft.com/en-us/media/save-as-segment.png)

3.   You can either choose to save the filters as a new segment or update an existing one. Select **Save as new** to create a new segment.

![Image 23: Save as a new segment.](https://learn.microsoft.com/en-us/media/segment-save-as-new.png)

4.   Enter a name for your segment in the **Segment name** field that describes the segment you created and select **Save** to save it. Once it's saved, this segment is applied. ![Image 24: Name a new segment.](https://learn.microsoft.com/en-us/media/name-segment.png)

Note

Each segment must have a unique name. 
5.   You can access the saved segments from the **Segments** drop down list. ![Image 25: List of segments.](https://learn.microsoft.com/en-us/media/segments-list.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-segments#apply-a-segment)
Apply a Segment
---------------

1.   Select a segment from the **Segments** drop down list. ![Image 26: Apply a segment from list of segments.](https://learn.microsoft.com/en-us/media/segments-list.png)
2.   Once applied, you can find the segment name and details as shown. ![Image 27: Applied a segment.](https://learn.microsoft.com/en-us/media/applied-segment.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-segments#clear-a-segment)
Clear a Segment
---------------

To clear the applied segment selection, select **Clear**.

![Image 28: Clear a segment.](https://learn.microsoft.com/en-us/media/clear-segment.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-segments#delete-a-segment)
Delete a Segment
----------------

1.   Select the **Segments** button to list the saved segments. ![Image 29: List of segments to delete.](https://learn.microsoft.com/en-us/media/segments-list.png)

2.   Select the **Delete icon** next to the segment you wish to remove. ![Image 30: Delete a segment.](https://learn.microsoft.com/en-us/media/delete-segment.png)

3.   Confirm the deletion by selecting **Delete**. ![Image 31: Confirm to delete a segment.](https://learn.microsoft.com/en-us/media/delete-segment-confirm.png)

Note

Deleted segments are removed for all users. 

[](https://learn.microsoft.com/en-us/clarity/clarity-segments#update-an-existing-segment)
Update an Existing Segment
--------------------------

1.   Select the required **Filters** from the list of available filters. Select 'Apply' to save the filters. ![Image 32: List of filters to update.](https://learn.microsoft.com/en-us/media/apply-a-filter.png)

2.   Select **Save as segment**. ![Image 33: Save as a segment for updating an existing segment.](https://learn.microsoft.com/en-us/media/save-as-segment.png)

3.   In the pop-up window, select **Update existing**. ![Image 34: Update an existing segment.](https://learn.microsoft.com/en-us/media/update-segment.png)

4.   Edit the segment's name if necessary and select **Save**. ![Image 35: Save an update.](https://learn.microsoft.com/en-us/media/save-updated-segment.png)

Note

Not every filter is supported in all of these views. The unsupported filters in a segment are disabled as shown. ![Image 36: Unsupported filter in a Heatmap.](https://learn.microsoft.com/en-us/media/unsupported-filter.png)
