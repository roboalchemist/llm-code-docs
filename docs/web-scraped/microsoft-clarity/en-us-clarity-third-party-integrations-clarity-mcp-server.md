The Microsoft Clarity MCP (Model Context Protocol) Server provides AI-driven, natural language access to website analytics. It serves as an interface between Clarity analytics data and AI agents, enabling users to query and visualize metrics using structured, conversational prompts.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#what-is-mcp-model-context-protocol)
What is MCP (Model Context Protocol)?
-------------------------------------

MCP is a protocol that standardizes the interaction between AI systems and local or remote services through structured commands. It functions as a universal connector for AI, facilitating seamless access to data and services regardless of the backend. The Clarity MCP Server utilizes this protocol to enable retrieval of analytics data via natural language queries.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#key-features)
Key features
------------

*   Query analytics data using natural language prompts
*   Filter by dimensions such as browser, operating system, country/region, or device
*   Retrieve metrics including scroll depth, engagement time, total traffic, and more
*   Integrates with Claude for Desktop and other MCP-compatible clients
*   Compatible with any AI agent that supports MCP

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#getting-started)
Getting started
---------------

Important

**Prerequisites**:

*   Node.js v16 or later
*   Microsoft Clarity project and [Data export API Token](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-data-export-api#obtaining-access-tokens).
*   For AI-powered querying, use [Claude for Desktop](https://claude.ai/download) or another MCP-compatible client.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#installation)
### Installation

To start the server, run the following command:

 Azure CLI 

Copy

```azurecli
npx @microsoft/clarity-mcp-server--
clarity_api_token=your-token-here
```

The API token can be provided directly via the command line interface or as a tool parameter when invoking commands.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#example-mcp-client-configuration)
### Example MCP client configuration

 JSON 

Copy

```json
{
	"mcpServers": {
		"@microsoft/clarity-mcp-server": {
			"command": "npx",
			"args": [
				"@microsoft/clarity-mcp-server",
				"--clarity_api_token=your-api-token-here"
			]
		}
	}
}
```

For additional configuration details, refer to the [GitHub repository](https://github.com/microsoft/clarity-mcp-server?tab=readme-ov-file#configuring-mcp-clients).

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#obtaining-an-api-token)
### Obtaining an API Token

1.   Go to your Clarity project.
2.   Navigate to **Settings** ->**Data Export**.
3.   Select **Generate new API token**.
4.   Store the token securely.

Note

Each project permits up to 10 API requests per day, with a maximum of 3 days' data and up to 3 dimensions per request.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/clarity-mcp-server#examples)

Planned enhancements
--------------------

The Clarity MCP Server is under active development, with several key improvements planned for upcoming releases:

*   **Increased API limits:** Expansion of daily request limits for the Clarity data export API to better support high-volume usage scenarios.
*   **Predictive heatmaps:** Introduction of predictive engagement heatmaps, enabling users to generate insights based on images or URLs.
*   **Advanced AI integration:** Deeper integration of AI capabilities, including contextual heatmap insights and additional analytics features.
*   **Multi-project support:** Enhanced support for managing and analyzing data across multiple projects, addressing the needs of enterprise analytics teams.
*   **Broader ecosystem compatibility:** Ongoing efforts to support a wider range of AI agents and facilitate collaboration with additional MCP servers.

For feedback, support, or additional questions, contact the Clarity team at [clarityms@microsoft.com](mailto:clarityms@microsoft.com).
