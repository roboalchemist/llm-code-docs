Heatmap Insights with Copilot in Clarity helps you more easily understand the user behaviors and trends of your website pages. Generative AI rapidly analyzes across different heatmaps to summarize the insights for a specific website page, across all three devices. This way, you can uncover key learnings and save time on data analysis.

[](https://learn.microsoft.com/en-us/clarity/copilot/heatmaps-insights#how-to-view-heatmaps-insights)
How to view Heatmaps Insights?
------------------------------

Important

Copilot is only available on the sites where the Clarity tracking code is installed. Check the [Setup](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup) process to know how to install it.

1.   Go to **Projects** ->**Heatmaps**.

2.   Select the page URL you're interested in.

![Image 12: Select a URL from popular pages.](https://learn.microsoft.com/en-us/clarity/media/heatmaps-zero-state.png)

3.   Select the **Summarize heatmaps** icon on the top right.

![Image 13: Select summarize heatmaps icon.](https://learn.microsoft.com/en-us/clarity/media/summarize-heatmaps-icon.png)

4.   Instantly, view a summarization of all your heatmaps for this page URL in the left panel.

![Image 14: View heatmaps summary.](https://learn.microsoft.com/en-us/clarity/media/view-heatmaps-summary.png)

5.   Toggle **Slang on** for casual wording insights.

![Image 15: Summary with simple words.](https://learn.microsoft.com/en-us/clarity/media/summary-slang-on.png)

6.   Reselect the **Summarize heatmaps** icon or select **Back to click data** to toggle off the insights.

![Image 16: Reselect summarize heatmaps icon.](https://learn.microsoft.com/en-us/clarity/media/reselect-summarize-heatmaps-icon.png)

7.   You can share your feedback using the thumbs up/down button.

![Image 17: Select heatmaps feedback icon.](https://learn.microsoft.com/en-us/clarity/media/heatmaps-insights-feedback-icon.png)

8.   Copy the insights using the **Copy** icon.

![Image 18: Select insights copy icon.](https://learn.microsoft.com/en-us/clarity/media/heatmaps-insights-copy-icon.png)

[](https://learn.microsoft.com/en-us/clarity/copilot/heatmaps-insights#limitations)
Limitations
-----------

Note

Clarity uses Generative AI models to power Copilot features. It is well-known that Generative AI models can sometimes misinterpret or generate incorrect information. Share your feedback using the thumbs up/down button or email us at [clarityMS@microsoft.com](mailto:clarityMS@microsoft.com) to help us learn and improve.

*   Currently, Heatmap Insights aren't supported for Mobile Apps.
*   Heatmap Insights currently don't intake data from the Area heatmap or Conversion heatmap.
*   Sharing and downloading insights aren't supported.

[](https://learn.microsoft.com/en-us/clarity/copilot/heatmaps-insights#faq)
