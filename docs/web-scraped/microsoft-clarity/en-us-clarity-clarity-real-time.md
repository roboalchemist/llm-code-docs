After installing the tracking code, you can start viewing live recordings that recreate user interactions with your website. These include mouse movements, clicks, scrolls, navigation, and more. Live recordings can help you understand the functionality of your website, identify user issues, and optimize performance.

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#view-live-recordings)
View live recordings
--------------------

Once you install the code, you can view a list of live users as shown:

![Image 12: View number of live recordings.](https://learn.microsoft.com/en-us/media/clarity-live-recordings-installation.png)

You can also go to **Project** ->**Recordings**. The number of live users is shown in the left panel.

![Image 13: View your live recordings.](https://learn.microsoft.com/en-us/media/live-recordings.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#list-of-live-recordings)
### List of live recordings

When you expand the section, you can view the list of live recordings.

![Image 14: List of live recordings.](https://learn.microsoft.com/en-us/media/list-of-live-recordings.png)

The list also contains watched recordings that aren't currently live and are inactive for 5 minutes.

While watching a recording, you can see the **Live** label, which indicates the current user activity. To stream the most recent user activity, select the **Go Live** button.

![Image 15: Live label and go live button.](https://learn.microsoft.com/en-us/media/go-live-button.gif)

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#no-live-users)
### No live users

You might see **No users online now** if there no live users.

![Image 16: No live recordings.](https://learn.microsoft.com/en-us/media/no-live-recordings.png)

You're notified as soon as there's a live user appears, as shown:

![Image 17: Notifying a live user.](https://learn.microsoft.com/en-us/media/live-user-notification.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#dashboard-data)
Dashboard data
--------------

You can also view the number of live and distinct users in the dashboard. The data is updated automatically. Distinct users refer to the number of different users, as detected by unique [first-party Clarity cookies](https://learn.microsoft.com/en-us/setup-and-installation/clarity-cookies#first-party-and-third-party-cookies) set on the project domain over a selected period.

![Image 18: Live data in dashboard.](https://learn.microsoft.com/en-us/media/live-data-in-dashboard.png)

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#supported-filters)

[](https://learn.microsoft.com/en-us/clarity/clarity-real-time#limitations)
Limitations
-----------

*   Currently, the following features aren't supported:

    *   Adding a label to a live recording.
    *   Creating Custom tags for a live recording.
    *   Date filter with custom dates.
    *   Download a live recording.
    *   Favorite live recording.
    *   Sharing a live recording.
    *   Sort the live recordings.
    *   View heatmaps from recordings.

*   Segments can be used with the [supported filters](https://learn.microsoft.com/en-us/clarity/clarity-real-time#supported-filters).
