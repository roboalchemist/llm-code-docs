Clarity offers a set of metrics and insights to help you better understand the reading and browsing patterns of articles on your site.

![Image 12: Content Insights in Dashboard.](https://learn.microsoft.com/en-us/media/content-insights-in-dashboard.png)

Content Insights has the following two sections:

*   [Content Insights card on Dashboard](https://learn.microsoft.com/en-us/clarity/content-insights#content-insights-card)

    *   [Key metrics](https://learn.microsoft.com/en-us/clarity/content-insights#key-metrics)
    *   [Clarity callouts](https://learn.microsoft.com/en-us/clarity/content-insights#clarity-callouts)

*   [User Actions filters](https://learn.microsoft.com/en-us/clarity/content-insights#user-actions-filters)

Content Insights card
---------------------

Note

Clarity Content Insights and metrics are automatically available for most WordPress blog sites. For others, refer to [Content Insights Setup](https://learn.microsoft.com/en-us/clarity/content-insights#content-insights-setup).

[](https://learn.microsoft.com/en-us/clarity/content-insights#key-metrics)
### Key Metrics

The top half of Clarity's content insights card shows key metrics about how users interact with articles on your site:

![Image 13: Content Insights card.](https://learn.microsoft.com/en-us/media/content-insights-card.png)

[](https://learn.microsoft.com/en-us/clarity/content-insights#reader-type-module)
#### Reader type module

In this widget module, Clarity shows the percentage breakdown of your sessions with pageviews. This percentage is based on the number of articles viewed.

*   **One and done**: Indicates the sessions where users viewed only one article.
*   **Casual**: Indicates the sessions where users viewed 2 to 3 articles.
*   **Serious**: Indicates the sessions where users viewed four or more articles.

[](https://learn.microsoft.com/en-us/clarity/content-insights#reading-behavior-module)
#### Reading behavior module

In this widget module, Clarity shows the count of article pageviews based on how users read the article.

*   **Engaged**: Indicates page views where users finished reading the article.
*   **Abandoned at headline**: Indicates page views where the user only read the article headline.

[](https://learn.microsoft.com/en-us/clarity/content-insights#dig-deeper-on-the-content-insights-modules)
#### Dig deeper on the Content Insights modules

There are three ways to interact with key metrics:

*   Select a metric to apply it as a dashboard filter.

![Image 14: Content insights to dashboard.](https://learn.microsoft.com/en-us/media/content-insights-to-dashboard.gif)

*   Watch recordings for a key metric by selecting the **Recordings** icon.

![Image 15: Content insights to recordings.](https://learn.microsoft.com/en-us/media/content-insights-to-recordings.gif)

*   View a Heatmap for a key metric by selecting the **Heatmap** icon.

![Image 16: Content insights to heatmap.](https://learn.microsoft.com/en-us/media/content-insights-to-heatmap.gif)

*   View reader engagement over time by selecting the **Content Insights trend line** icon.

![Image 17: Content insights trends.](https://learn.microsoft.com/en-us/media/content-insights-trends.png)

[](https://learn.microsoft.com/en-us/clarity/content-insights#clarity-callouts)
### Clarity Callouts

![Image 18: Clarity callouts.](https://learn.microsoft.com/en-us/media/clarity-callouts.png)

The second half of Content insights card shows key callouts related to these metrics. Clarity identifies interesting data points based on users' interaction in different segments with your site articles.

[](https://learn.microsoft.com/en-us/clarity/content-insights#user-actions-filters)
User Actions filters
--------------------

Content insights also include two filters:

*   [Reader type](https://learn.microsoft.com/en-us/clarity/content-insights#reader-type-module)
*   [Reading behavior](https://learn.microsoft.com/en-us/clarity/content-insights#reading-behavior-module)

See [Content Insights filters](https://learn.microsoft.com/en-us/filters/clarity-filters#filtering-by-user-actions).

[](https://learn.microsoft.com/en-us/clarity/content-insights#content-insights-setup)
Content Insights Setup
----------------------

Content insights card is automatically applied for most WordPress blog sites. To view content insights on your Clarity dashboard, add the following attribute to the `article` element or tag on the DOM for article pages:

> data-clarity-region="article"

**Example**:

Below is an example code snippet:

 HTML 

Copy

```html
<html>
   <head>
   <body>
              ...
              <article data-clarity-region="article">
              ...
              </article>
   </body>
</html>
```
