Install Clarity on Wix platform
-------------------------------

Note

Clarity can only be used with registered account domains on Wix.

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wix#step-1)
### Step 1

Sign in to your [Wix](https://www.wix.com/) account. Open the WIX site you want to install Clarity. Navigate to **Apps** ->**App Market**. Search for Microsoft Clarity.

Select **Add to site** which adds Clarity to your selected WIX project.

![Image 11: Select add to site.](https://learn.microsoft.com/en-us/clarity/media/wix-step1.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wix#step-2)
### Step 2

From the opened the Clarity app, select **Sign in** to start connecting existing projects or create a new project.

![Image 12: Select sign in to clarity.](https://learn.microsoft.com/en-us/clarity/media/wix-step2.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wix#step-3)
### Step 3

Choose a project from the list of existing projects and select **Connect**.

![Image 13: Choose a project from the list.](https://learn.microsoft.com/en-us/clarity/media/wix-step3a.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wix#or)
#### OR

Select **Add new project**.

![Image 14: Select add new project.](https://learn.microsoft.com/en-us/clarity/media/wix-step3b.png)

Enter the details and save it.

![Image 15: Add a new project for wix.](https://learn.microsoft.com/en-us/clarity/media/wix-step3c.png)

[](https://learn.microsoft.com/en-us/clarity/third-party-integrations/wix#step-4)
### Step 4

Once the project is successfully integrated, go to **Microsoft Clarity** from the navigation menu on the left to view the Clarity Dashboard, Session Recordings, Heatmaps, & Insights.

![Image 16: View the project dashboard.](https://learn.microsoft.com/en-us/clarity/media/wix-step4.png)

[Verify your installation](https://learn.microsoft.com/en-us/clarity/setup-and-installation/clarity-setup#verify-your-installation)
