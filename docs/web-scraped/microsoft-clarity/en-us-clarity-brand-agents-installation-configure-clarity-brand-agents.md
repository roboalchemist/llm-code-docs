Configure Brand agents
======================

Note

*   Brand agents is currently in beta. Join the waitlist to be informed when you're eligible to test them.
*   Currently, Brand agents is only available for Shopify sites.
*   Currently, Brand agents is only available for websites and not for mobile apps.
*   Refer to the [Brand agents terms](https://clarity.microsoft.com/terms/agents) for detailed usage guidelines.

Once you're approved to join the Brand agents beta, you can configure your agent to match your brand identity and business needs. This guide walks you through customizing your agent's appearance, messaging, and behavior before publishing it live on your website.

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-1-get-started)
### Step 1: Get started

As soon as you're onboarded, you'll receive an email notification. You can return to Clarity (through your Shopify store settings or directly at Clarity) and configure your brand agent.

Select **Configure and publish** to get started.

![Image 16: Configure and publish brand agent.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-1.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-2-customize-your-greeting-messages)
### Step 2: Customize your greeting messages

Use the panel on the left to define the details of your agent greeting message. Changes are reflected in the preview on the right side.

You can modify:

*   Logo
*   Greeting emoji
*   Company name
*   Agent name

Once you're satisfied with the preview, select **Customize appearance** to continue.

![Image 17: Customize greeting messages.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-2.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-3-customize-appearance)
### Step 3: Customize appearance

Use the panel on the left to define the details of your agent appearance. Changes are reflected in the preview on the right side.

You can modify:

*   Theme color
*   Font

Once you're satisfied with the preview, select **Review brand identity** to continue.

![Image 18: Customize appearance.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-3.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-4-review-brand-identity)
### Step 4: Review brand identity

Use the central panel to provide your agent input, so it reflects your brand. This section comes pre-filled with AI-generated content. You can edit it manually to suit your needs. You can always regenerate if you want to return to the original content.

You can modify:

*   **Brand overview**: A short description of your company.
*   **Brand offering**: Describing your products or services.
*   **Tone**: Instructions on how your agent should interact with the audience.

When you're satisfied with the content, select **Review and publish** for a final check before the agent goes live.

![Image 19: Review brand identity.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-4.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-5-review-and-publish)
### Step 5: Review and publish

Your agent is ready to launch. Before publishing, use the **Live preview** to test the experience. You can always return to the previous steps if any changes are needed.

![Image 20: Use live preview.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-5-a.png)

When you're ready, select **Publish agent**. This makes it available on your website for your users to see and interact with.

![Image 21: Select publish agent.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-5-b.png)

Your agent is now live. Select **Go to settings**.

![Image 22: Brand agent is live.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-5-c.png)

[](https://learn.microsoft.com/en-us/clarity/brand-agents/installation/configure-clarity-brand-agents#step-6-edit-your-agent)
### Step 6: Edit your agent

Once your agent is live, you can edit its settings and republish it as many times as you want.

You can toggle your agent on and off. If you turn it off, it's hidden and is not available to your users.

![Image 23: Toggle brand agent on and off.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-6-a.png)

You can also modify all its settings. Expand one or more sections (**Welcome message**, **Appearance**, or **Brand overview, tone, and voice**) and edit as many fields as needed.

![Image 24: Modify agent settings.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-6-b.png)

Changes are only reflected when you select **Publish updates**. This button is disabled if there are no changes to publish.

![Image 25: Publish updates to agent.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-6-c.png)

Finally, you can always visit your site to check and test the behavior of your agent.

![Image 26: Edit agent settings.](https://learn.microsoft.com/en-us/clarity/media/brand-agents-configure-step-6-d.png)
