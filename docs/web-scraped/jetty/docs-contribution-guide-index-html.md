Markdown Content:
The Eclipse Jetty Contribution Guide targets developers and writers who want to make contributions to the Jetty project by contributing code, writing documentation, or engaging in the larger community.

[](https://jetty.org/docs/contribution-guide/index.html#cg-intro-mailing-lists)Mailing Lists
--------------------------------------------------------------------------------------------

One of the easiest ways to get involved is via our mailing lists:

*   **Jetty Users** ([jetty-users](https://accounts.eclipse.org/mailing-list/jetty-users), [archives](https://www.eclipse.org/lists/jetty-users)) is for discussion and questions that are of broad interest to the community of Jetty users.

*   **Jetty Developers** ([jetty-dev](https://accounts.eclipse.org/mailing-list/jetty-dev), [archives](https://www.eclipse.org/lists/jetty-dev)) is more narrowly focused on technical topics and discussion regarding Jetty internals.

*   **Jetty Announcements** ([jetty-announce](https://accounts.eclipse.org/mailing-list/jetty-announce), [archives](https://www.eclipse.org/lists/jetty-announce)) is for announcements about new releases and other updates from the project’s maintainers.

[](https://jetty.org/docs/contribution-guide/index.html#cg-intro-stack-overflow)Stack Overflow
----------------------------------------------------------------------------------------------

Another great resource, both for Jetty novices who need help and Jetty experts who want to contribute, is [StackOverflow](http://stackoverflow.com/). In particular, the [`jetty`](https://stackoverflow.com/questions/tagged/jetty) and [`embedded-jetty`](https://stackoverflow.com/questions/tagged/embedded-jetty) tags see regular traffic.

[](https://jetty.org/docs/contribution-guide/index.html#cg-intro-filing-issues)Filing Issues
--------------------------------------------------------------------------------------------

You can flag potential bugs or suggest new Jetty features on our [issue tracker](https://github.com/eclipse/jetty.project/issues).

Before filing a new issue, [check the tracker](https://github.com/eclipse/jetty.project/issues) to see if it’s already been filed by someone else. If you do file an issue, make sure to label it appropriately, as this will help the development team (and other users) find it more easily.

[](https://jetty.org/docs/contribution-guide/index.html#cg-intro-help-wanted)Help Wanted
----------------------------------------------------------------------------------------

If you want to contribute to Jetty but don’t have a specific task or goal in mind, consider looking through our ["Help Wanted" issue backlog](https://github.com/eclipse/jetty.project/issues?q=is%3Aopen+is%3Aissue+label%3A%22Help+Wanted%22). These tasks range from the simple to the complex, but they’re all ones we’ve identified as being particularly well-suited for new contributors to tackle.

[](https://jetty.org/docs/contribution-guide/index.html#cg-intro-commercial-support)Commercial Support
------------------------------------------------------------------------------------------------------
