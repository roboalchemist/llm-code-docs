### Apache Taglibs Downloads

Welcome to the Apache Taglibs download page. This page provides download links for obtaining the latest version of the [Apache Standard Taglib](https://tomcat.apache.org/taglibs/standard/), as well as links to the archives of older releases.

### Quick Navigation

[KEYS](https://downloads.apache.org/tomcat/taglibs/KEYS) | [Standard Taglib 1.2.5](https://tomcat.apache.org/download-taglibs.cgi#Standard-1.2.5) | [Browse](https://dlcdn.apache.org/tomcat/taglibs/) | [Archives](https://archive.apache.org/dist/tomcat/taglibs/)

### Release Integrity

You **must**[verify](https://www.apache.org/info/verification.html) the integrity of the downloaded files. We provide OpenPGP signatures for every release file. This signature should be matched against the [KEYS](https://downloads.apache.org/tomcat/taglibs/KEYS) file which contains the OpenPGP keys of Taglib's Release Managers. We also provide a `SHA512` checksum for every release file. After you download the file, you should calculate a checksum for your download, and make sure it is the same as ours.

### Mirrors

You are currently using **https://dlcdn.apache.org/**. If you encounter a problem with this mirror, please select another mirror. If all mirrors are failing, there are _backup_ mirrors (at the end of the mirrors list) that should be available.
