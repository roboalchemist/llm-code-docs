Jakarta WebSocket
=================

Primary tabs
------------

*   [Overview](https://projects.eclipse.org/projects/ee4j.websocket)
*   [Downloads](https://projects.eclipse.org/projects/ee4j.websocket/downloads)
*   [Who's Involved](https://projects.eclipse.org/projects/ee4j.websocket/who)
*   [Developer Resources](https://projects.eclipse.org/projects/ee4j.websocket/developer)
*   [Governance](https://projects.eclipse.org/projects/ee4j.websocket/governance)
*   [Contact Us](https://projects.eclipse.org/projects/ee4j.websocket/contact)

Jakarta WebSocket specifies the API that Java developers can use when they want to integrate WebSockets into their applications - both on the server side as well as on the Java client side.

An implementation for the WebSocket API is provided by the [Eclipse Tyrus](https://projects.eclipse.org/projects/ee4j.tyrus) project.
