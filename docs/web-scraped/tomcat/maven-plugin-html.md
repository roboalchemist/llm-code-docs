### Maven Plugin

The Apache Tomcat Maven Plugin provides goals to manipulate WAR projects within the Apache Tomcat® servlet container. You can run your War Apache Maven project through [Apache Maven](http://maven.apache.org/) without deploying your WAR file to an Apache Tomcat instance.

More details are available in the Maven generated site:

*   [Trunk documentation](https://tomcat.apache.org/maven-plugin-trunk/) (under development)
*   [Version 2.2](https://tomcat.apache.org/maven-plugin-2.2/) (released version: 2013-11-11)
*   [Version 2.1](https://tomcat.apache.org/maven-plugin-2.1/) (released version: 2013-02-25)
*   [Version 2.0](https://tomcat.apache.org/maven-plugin-2.0/) (released version: 2012-09-14)
*   [Version 2.0-beta-1](https://tomcat.apache.org/maven-plugin-2.0-beta-1/) (released version: 2012-02-01)
