### 2026-02-11 End of support for Apache Tomcat Native 1.3.x

The Apache Tomcat® team announces that support for Apache Tomcat Native 1.3.x will end on 31 March 2027.

This means that after 31 March 2027:

*   releases from the 1.3.x branch are highly unlikely
*   bugs affecting only the 1.3.x branch will not be addressed
*   security vulnerability reports will not be checked against the 1.3.x branch

Three months later (i.e. some time after 30 June 2027):

*   the 1.3.x download pages will be removed
*   the latest 1.3.x release will be removed from the mirror system
*   the links to the 1.3.x documentation will be removed from tomcat.apache.org

Note that all 1.3.x releases will always be available from the archive.
