Jakarta Expression Language
===========================

Primary tabs
------------

*   [Overview](https://projects.eclipse.org/projects/ee4j.el)
*   [Downloads](https://projects.eclipse.org/projects/ee4j.el/downloads)
*   [Who's Involved](https://projects.eclipse.org/projects/ee4j.el/who)
*   [Developer Resources](https://projects.eclipse.org/projects/ee4j.el/developer)
*   [Governance](https://projects.eclipse.org/projects/ee4j.el/governance)
*   [Contact Us](https://projects.eclipse.org/projects/ee4j.el/contact)

Jakarta Expression Language (also referred to as the EL) provides an important mechanism for enabling the presentation layer (web pages) to communicate with the application logic (managed beans). The Expression Language is used by several Jakarta EE technologies, such as Jakarta Faces technology, Jakarta Pages technology, Jakarta Security and Jakarta Contexts and Dependency Injection (CDI). The Expression Language can also be used in stand-alone environments.
