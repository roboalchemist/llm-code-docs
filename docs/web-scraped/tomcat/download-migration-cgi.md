### Tomcat Migration Tool for Jakarta EE Software Downloads

Welcome to the Apache Tomcat® Migration tool for Jakarta EE software download page. This page provides download links for obtaining the latest version of Tomcat Migration Tool for Jakarta EE software, as well as links to the archives of older releases.

### Quick Navigation

[KEYS](https://downloads.apache.org/tomcat/jakartaee-migration/KEYS) | [1.0.10](https://tomcat.apache.org/download-migration.cgi#1.0.10) | [Browse](https://dlcdn.apache.org/tomcat/jakartaee-migration/v1.0.10) | [Archives](https://archive.apache.org/dist/tomcat/jakartaee-migration)

### Release Integrity

You **must**[verify](https://www.apache.org/info/verification.html) the integrity of the downloaded files. We provide OpenPGP signatures for every release file. This signature should be matched against the [KEYS](https://downloads.apache.org/tomcat/jakartaee-migration/KEYS) file which contains the OpenPGP keys of Tomcat's Release Managers. We also provide `SHA-512` checksums for every release file. After you download the file, you should calculate a checksum for your download, and make sure it is the same as ours.

### Mirrors

You are currently using **https://dlcdn.apache.org/**. If you encounter a problem with this mirror, please select another mirror. If all mirrors are failing, there are _backup_ mirrors (at the end of the mirrors list) that should be available.
