### Product Support

If you have questions or comments about the Apache Tomcat® software or documentation on this site, please subscribe to the appropriate [mailing list](https://tomcat.apache.org/lists.html).

### Security Issues

If you would like to report a security issue with Apache Tomcat, please contact [security@tomcat.apache.org](mailto:security@tomcat.apache.org). **Only** security issues should be sent to this address.

### General Apache Issues

The Tomcat Project is an effort of the [Apache Software Foundation](http://apache.org/foundation/). The address for general ASF correspondence and licensing questions is:

[apache.AT.apache.DOT.org](mailto:apache.AT.apache.DOT.org)
