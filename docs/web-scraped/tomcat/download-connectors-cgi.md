### Tomcat Connectors (mod_jk) Downloads

Use the links below to download the Apache Tomcat® Connectors software from one of our mirrors. You **must**[verify the integrity](https://tomcat.apache.org/download-connectors.cgi#Verify_the_Integrity_of_the_Files) of the downloaded files using signatures downloaded from our main distribution directory.

Only current recommended releases are available on the main distribution site and its mirrors. Older releases and the historical mod_jk2 are available from the [archive download site](https://archive.apache.org/dist/tomcat/tomcat-connectors/).

#### Choose a Mirror

You are currently using **https://dlcdn.apache.org/**. If you encounter a problem with this mirror, please select another mirror. If all mirrors are failing, there are _backup_ mirrors (at the end of the mirrors list) that should be available.

You may also consult the [complete list of mirrors](http://www.apache.org/mirrors/).

#### Verify the Integrity of the Files

You **must**[verify](https://www.apache.org/info/verification.html) the integrity of the downloaded files. We provide OpenPGP signatures for every release file. This signature should be matched against the [KEYS](https://downloads.apache.org/tomcat/tomcat-connectors/KEYS) file which contains the OpenPGP keys of Tomcat's Release Managers. We also provide `SHA512` checksums for every release file. After you download the file, you should calculate a checksum for your download, and make sure it is the same as ours.
