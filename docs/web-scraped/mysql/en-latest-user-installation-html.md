The last stable release is available on PyPI and can be installed with `pip`:

$ python3 -m pip install PyMySQL

To use “sha256_password” or “caching_sha2_password” for authenticate, you need to install additional dependency:

$ python3 -m pip install PyMySQL[rsa]

Requirements[](https://pymysql.readthedocs.io/en/latest/user/installation.html#requirements "Link to this heading")
--------------------------------------------------------------------------------------------------------------------

* Python – one of the following:

  * [CPython](http://www.python.org/)>= 3.7

  * Latest [PyPy](http://pypy.org/) 3

* MySQL Server – one of the following:

  * [MySQL](http://www.mysql.com/)>= 5.7

  * [MariaDB](https://mariadb.org/)>= 10.3
