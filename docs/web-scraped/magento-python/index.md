`pymagento` is a Python library around the Magento 2 API.

Installation[](https://pymagento2.readthedocs.io/#installation "Permalink to this headline")
---------------------------------------------------------------------------------------------

With `pip`:

pip install pymagento

With `poetry`:

poetry add pymagento
