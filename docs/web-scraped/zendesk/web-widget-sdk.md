## Web

Zendesk messaging for websites is supported by the Web Widget. To learn more, see Working with messaging in the Web Widget.

The Web Widget API is a client-side JavaScript API for configuring the Web Widget settings and how it is displayed. This enables you to customize the Web Widget to your requirements on specific website pages.
