Introduction - Reference Manual
===============================

Welcome to the ActiveBatch®  Reference Manual.

This document is intended for users who will be designing, implementing, and deploying the ActiveBatch scheduling and workload automation system. It describes all the ActiveBatch objects and features. AbatConsole, the Windows user interface, is used when describing how to access features and accomplish tasks.
