# PopupPanel
Inherits:Popup<Window<Viewport<Node<Object
A popup with a panel background.

## Description
A popup with a configurable panel background. Any child controls added to this node will be stretched to fit the panel's size (similar to howPanelContainerworks). If you are making windows, seeWindow.
