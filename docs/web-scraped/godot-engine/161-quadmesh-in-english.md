## Description
Class representing a squarePrimitiveMesh. This flat mesh does not have a thickness. By default, this mesh is aligned on the X and Y axes; this rotation is more suited for use with billboarded materials. AQuadMeshis equivalent to aPlaneMeshexcept its defaultPlaneMesh.orientationisPlaneMesh.FACE_Z.
