# XRControllerTracker

Inherits:XRPositionalTracker<XRTracker<RefCounted<Object
A tracked controller.

## Description

An instance of this object represents a controller that is tracked.
As controllers are turned on and theXRInterfacedetects them, instances of this object are automatically added to this list of active tracking objects accessible through theXRServer.
TheXRController3Dconsumes objects of this type and should be used in your project.
