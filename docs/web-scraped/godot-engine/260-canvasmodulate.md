# CanvasModulate
Inherits:Node2D<CanvasItem<Node<Object
A node that applies a color tint to a canvas.

## Description
CanvasModulateapplies a color tint to all nodes on a canvas. Only one can be used to tint a canvas, butCanvasLayers can be used to render things independently.
