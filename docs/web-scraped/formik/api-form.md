# <Form />

Form is a small wrapper around an HTML<form>element that automatically hooks into Formik'shandleSubmitandhandleReset. All other props are passed directly through to the DOM node.

`<form>`
`handleSubmit`
`handleReset`

```
1// so...2<Form/>34// is identical to this...5<formonReset={formikProps.handleReset}onSubmit={formikProps.handleSubmit}{...props}/>
```
