Troubleshooting[¶](https://coveralls-python.readthedocs.io/en/latest/troubleshooting.html#troubleshooting "Link to this heading")
=================================================================================================================================

If you are having difficulties submitting your coverage to coveralls.io, debug mode may help you figure out the problem:

$ coveralls debug

Debug mode doesn’t send anything, it just outputs prepared json and reported files list to stdout.

We also have an [issue tracker](https://github.com/TheKevJames/coveralls-python/issues) on GitHub.
