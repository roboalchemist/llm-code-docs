Can I use Motion+ for commercial projects?

Yes - Motion+ APIs and examples are licensed under MIT. This means once you've added code to your site, it's yours forever.

### Builder's License

If your product effectively resells Motion+ capabilities to your users, you need a **Builder's License**.

This applies if you are building a platform (like Framer, Figma, or a SaaS tool) where:

*   **Your users** can add Motion+ components (like the Carousel or Ticker) to their own projects.

*   **Your users** utilize Motion+ tooling (like the Studio SDK or Bezier Editor) to edit or author content.

In short: If the Motion+ features are for _your_ team to build your product, the Standard license is fine. If the Motion+ features are a selling point for _your customers_ to build _their_ products, you need a Builder's License.

Please contact [matt@motion.dev](mailto:matt@motion.dev) for details.

We do offer student discounts in some countries and PPP for others - you can send proof of residence or student ID to [matt@motion.dev](mailto://matt@motion.dev) for a discount code.
