[Screaming Frog SEO Spider](https://www.screamingfrog.co.uk/seo-spider/) is a leading desktop tool for SEO auditing. It crawls websites to analyze metadata, find broken links, audit redirects, discover duplicate content, and generate XML sitemaps.

By routing Screaming Frog through Scrapfly's [Proxy Mode](https://scrapfly.io/docs/scrape-api/proxy-mode), you get automatic anti-bot bypass, JavaScript rendering, proxy rotation, and geo-targeting — without changing how you use Screaming Frog.

[Why Use Scrapfly with Screaming Frog?](https://scrapfly.io/docs/scrape-api/proxy-mode/screaming-frog#why)
----------------------------------------------------------------------------------------------------------

*   **Bypass anti-bot protections** — Crawl sites protected by Cloudflare, Akamai, PerimeterX, and others.
*   **JavaScript rendering** — See the fully rendered page as Googlebot does, not just raw HTML.
*   **Proxy rotation** — Avoid IP bans during large-scale audits with automatic proxy rotation across datacenter or residential pools.
*   **Geo-targeting** — Audit localized content by routing through specific countries.

[Step-by-Step Setup](https://scrapfly.io/docs/scrape-api/proxy-mode/screaming-frog#setup)
-----------------------------------------------------------------------------------------

1.   Open **Screaming Frog SEO Spider**.
2.   Navigate to **Configuration > System > Proxy** (or **File > Settings > Proxy** on older versions).
3.   Check **Use Proxy Server**.
4.    Enter the proxy server details: 
**Address**`proxy.scrapfly.io`
**Port**`7777`
5.   Check **Use Proxy Credentials**.
6.    Enter your Scrapfly credentials: 
**Username**Scraping options as dash-separated key-value pairs.

 Example: `country-us-asp-true`

 Or use any string like `scrape` for default settings.
**Password**Your Scrapfly API key
7.   Click **OK** to save.
8.   Enter a URL in the address bar and click **Start** to begin crawling through Scrapfly.

[Recommended Proxy Usernames](https://scrapfly.io/docs/scrape-api/proxy-mode/screaming-frog#recommended-options)
----------------------------------------------------------------------------------------------------------------

The proxy username controls which Scrapfly features are enabled. Here are common configurations for SEO auditing:

| Use Case | Username | Description |
| --- | --- | --- |
| **Default** | `scrape` | JS rendering + anti-bot enabled (proxy mode defaults). |
| **US geo-targeting** | `country-us` | Crawl as if located in the United States. |
| **Residential proxies** | `proxyPool-public_residential_pool` | Use residential IPs for heavily protected sites. |
| **Fast static audit** | `renderJs-false-asp-false` | Skip JS rendering and anti-bot for faster crawling of static sites. |
| **Full protection** | `country-us-proxyPool-public_residential_pool` | US residential proxies with JS rendering and anti-bot. |

> **Tip:**In proxy mode, `renderJs`and `asp`both default to `true`. This means Screaming Frog will see the fully rendered page by default, matching how search engines see the site.


In Screaming Frog, check the response tab for any crawled URL. You should see the fully rendered HTML content. If you encounter issues, check:

*   Your API key is correct (used as the proxy password).
*   The proxy address and port match the values above.
*   Your Scrapfly account has sufficient API credits.

[SEO Auditing Tips](https://scrapfly.io/docs/scrape-api/proxy-mode/screaming-frog#tips)
---------------------------------------------------------------------------------------

*   **Crawl budget:** Each page crawled by Screaming Frog consumes Scrapfly API credits. Use [the `cost` option](https://scrapfly.io/docs/scrape-api/proxy-mode#options) to set a per-request budget limit. 
*   **Caching:** Enable `cache-true` in the username to avoid re-scraping pages you've already crawled. This saves API credits when re-running audits. 
*   **forwardHeaders:** Set `forwardHeaders-false` in the username to let Scrapfly manage all headers for optimal anti-bot evasion. 
*   **Speed vs protection:** For sites without anti-bot protection, use `renderJs-false-asp-false` for significantly faster crawl speeds. 

See the full [Proxy Mode options reference](https://scrapfly.io/docs/scrape-api/proxy-mode#options) for all available settings.
