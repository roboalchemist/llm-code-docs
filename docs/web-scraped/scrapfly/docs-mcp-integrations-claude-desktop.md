[Setup Instructions](https://scrapfly.io/docs/mcp/integrations/claude-desktop#setup)
------------------------------------------------------------------------------------

Follow these steps to integrate Scrapfly MCP with Claude Desktop. The entire setup takes less than 5 minutes.

1.   **Add Scrapfly MCP via Custom Connector**Easiest
The fastest way to connect Scrapfly MCP to Claude Desktop - no file editing required:

    1.   In your [Scrapfly Dashboard](https://scrapfly.io/dashboard), click **API Keys** in the top bar and copy the **MCP URL**
    

[![Image 33: Click API Keys in the Scrapfly Dashboard top bar and copy the MCP URL](https://cdn.scrapfly.io/0.1.176/www/public/img/doc/mcp/claude-desktop/step-1a-copy-mcp-url.png?version=0.1.176)](https://scrapfly.io/docs/mcp/integrations/claude-desktop "Expand as fullscreen")

    2.   In Claude Desktop, go to **Settings → Connectors** and click **Add custom connector**
    3.   Paste the MCP URL and save
    

[![Image 34: Add custom connector in Claude Desktop Settings and paste the Scrapfly MCP URL](https://cdn.scrapfly.io/0.1.176/www/public/img/doc/mcp/claude-desktop/step-1bc-add-connector.png?version=0.1.176)](https://scrapfly.io/docs/mcp/integrations/claude-desktop "Expand as fullscreen")

That's it! Scrapfly tools will be available immediately.

[![Image 35: Scrapfly MCP tools available in Claude Desktop after setup](https://cdn.scrapfly.io/0.1.176/www/public/img/doc/mcp/claude-desktop/step-1-tools-available.png?version=0.1.176)](https://scrapfly.io/docs/mcp/integrations/claude-desktop "Expand as fullscreen")

2.   **Alternative: Manual Configuration**
If you prefer to edit the configuration file directly, open it via **Claude Desktop → Settings → Developer → Edit Config**, then choose an authentication method:

[OAuth2 Recommended Secure](https://scrapfly.io/docs/mcp/integrations/claude-desktop)[API Key For automation](https://scrapfly.io/docs/mcp/integrations/claude-desktop) Add the following configuration to your `claude_desktop_config.json` file:

```
{
  "mcpServers": {
    "scrapfly": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://mcp.scrapfly.io/mcp"
      ]
    }
  }
}
``` [](https://scrapfly.io/docs/mcp/integrations/claude-desktop)[](https://scrapfly.io/docs/mcp/integrations/claude-desktop)  **How it works:** On first connection, Claude Desktop will display an OAuth2 URL in the chat. Click it to authorize in your browser - happens once, then managed automatically. [See project-scoped setup](https://scrapfly.io/docs/mcp/authentication#project-scoped) for team collaboration.   Add the following configuration to your `claude_desktop_config.json` file:

```
{
  "mcpServers": {
    "scrapfly": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://mcp.scrapfly.io/mcp?key="
      ]
    }
  }
}
``` [](https://scrapfly.io/docs/mcp/integrations/claude-desktop)[](https://scrapfly.io/docs/mcp/integrations/claude-desktop)  [Sign up for free](https://scrapfly.io/register) to get your API key.    
3.   **Restart Claude Desktop**
After saving the configuration file, completely quit and restart Claude Desktop to apply the changes.

How to Properly Restart
**On macOS:**

    *   Right-click the Claude icon in the Dock and select "Quit"
    *   Or press ⌘ + Q while Claude is active
    *   Reopen Claude from Applications or Spotlight

**On Windows:**

    *   Right-click the Claude icon in the system tray and select "Quit"
    *   Or close all Claude windows and exit from Task Manager if needed
    *   Reopen Claude from Start Menu or Desktop shortcut

4.   **Verify the Integration**
After restarting, check that the MCP tools are available:

    1.   Click the **"+"** button in the chat input, then select **Connectors**
    2.   You should see **Scrapfly MCP** toggled on in the connectors list
    3.   Try a test prompt: _"Can you scrape https://news.ycombinator.com and show me the top 5 posts?"_

[![Image 36: Scrapfly MCP tools visible under the hammer icon in Claude Desktop](https://cdn.scrapfly.io/0.1.176/www/public/img/doc/mcp/claude-desktop/step-4-verify-tools.png?version=0.1.176)](https://scrapfly.io/docs/mcp/integrations/claude-desktop "Expand as fullscreen")

**Pro Tip:** Claude will automatically call the `scraping_instruction_enhanced` tool first to get the required "pow" parameter before making scraping requests. You do not need to worry about this - it happens behind the scenes!

[Troubleshooting](https://scrapfly.io/docs/mcp/integrations/claude-desktop#troubleshooting)
-------------------------------------------------------------------------------------------

##### MCP tools not appearing in Claude Desktop

*   Ensure `npx` is available in your PATH
*   Check config file is valid JSON (use [JSONLint](https://jsonlint.com/))
*   Restart Claude Desktop completely
*   Check Claude Desktop logs for errors

##### Checking Claude Desktop logs

**macOS:**`~/Library/Logs/Claude/`

**Windows:**`%APPDATA%\Claude\logs\`

Look for errors related to MCP server connection or npx command execution.

##### OAuth2 authentication fails

*   Ensure your default browser is set correctly
*   Manually copy OAuth2 URL from Claude Desktop and paste in browser
*   Check that you're logged into Scrapfly in your browser
*   Fallback: use API key authentication
