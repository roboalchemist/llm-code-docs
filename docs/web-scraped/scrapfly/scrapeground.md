ScrapeGround
============

Educational platform for learning about common web scraping patterns and techniques. Each scenario page covers one topic and provides a real-life code example in Python and Javascript:

[![Image 2: Referer Header](https://cdn.scrapfly.io/srapeground/0.0.18/media/referer.svg) Vital HTTP header that can help prevent scraper blocking](https://scrapfly.io/scrapeground/headers/referer)

[![Image 3: CSRF Header](https://cdn.scrapfly.io/srapeground/0.0.18/media/csrf.svg) Popular hidden header value that requires HTML parsing and chain scraping](https://scrapfly.io/scrapeground/headers/csrf)

[![Image 4: Cookies](https://cdn.scrapfly.io/srapeground/0.0.18/media/cookies.svg) What are cookies and how they work in web scraping - explained through a login example](https://scrapfly.io/scrapeground/cookies)

[![Image 5: Static Paging](https://cdn.scrapfly.io/srapeground/0.0.18/media/staticpaging.svg) How to scrape static data pagination in web scraping](https://scrapfly.io/scrapeground/paging/static)

[![Image 6: Endless or Dynamic Paging](https://cdn.scrapfly.io/srapeground/0.0.18/media/dynamicpaging.svg) How to scrape dynamic static data pagination like endless paging in web scraping](https://scrapfly.io/scrapeground/paging/endless)
