Proxy Saver Billing
===================

[View as markdown](https://scrapfly.io/docs/proxy-saver/billing?view=markdown)

[Copy for LLM](https://scrapfly.io/docs/proxy-saver/billing#)

[Copy for LLM](https://scrapfly.io/docs/proxy-saver/billing#)[![Image 29](https://unpkg.com/@lobehub/icons-static-webp@latest/light/openai.webp) Open in ChatGPT](https://chatgpt.com/?hints=search&prompt=Read%20from%20https%3A%2F%2Fscrapfly.io%2Fdocs%2Fproxy-saver%2Fbilling%20so%20I%20can%20ask%20questions%20about%20it.)[![Image 30](https://unpkg.com/@lobehub/icons-static-webp@latest/light/claude-color.webp) Open in Claude](https://claude.ai/new?q=Read%20from%20https%3A%2F%2Fscrapfly.io%2Fdocs%2Fproxy-saver%2Fbilling%20so%20I%20can%20ask%20questions%20about%20it.)[![Image 31](https://unpkg.com/@lobehub/icons-static-webp@latest/light/perplexity-color.webp) Open in Perplexity](https://www.perplexity.ai/search/new?q=Read%20from%20https%3A%2F%2Fscrapfly.io%2Fdocs%2Fproxy-saver%2Fbilling%20so%20I%20can%20ask%20questions%20about%20it.)

We bill based on the transited bandwidth that go through the Proxy Saver service. We bill at the rate of **$0.2 per GB**, this is deducted into API credit based on your account subscription. Each proxy saver instance provide statistics on the amount of data transited and the current credit balance.

Some optional features can incur extra costs:

| Feature | Cost |
| --- | --- |
