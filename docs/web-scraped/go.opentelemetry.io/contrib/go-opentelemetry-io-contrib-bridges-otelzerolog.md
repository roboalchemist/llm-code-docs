### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzerolog#pkg-overview "Go to Overview")

*   [Record Conversion](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzerolog#hdr-Record_Conversion)

Package otelzerolog provides a [Hook](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzerolog#Hook), a [zerolog.Hook](https://pkg.go.dev/github.com/rs/zerolog#Hook) implementation that can be used to bridge between the [zerolog](https://pkg.go.dev/github.com/rs/zerolog) API and [OpenTelemetry](https://opentelemetry.io/docs/concepts/signals/logs/).

#### Record Conversion [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzerolog#hdr-Record_Conversion "Go to Record Conversion")

The [zerolog.Event](https://pkg.go.dev/github.com/rs/zerolog#Event) records are converted to OpenTelemetry [log.Record](https://pkg.go.dev/go.opentelemetry.io/otel/log#Record) in the following way:

*   Message is set as the Body using a [log.StringValue](https://pkg.go.dev/go.opentelemetry.io/otel/log#StringValue).
*   Level is transformed and set as the Severity. The SeverityText is also set.

The Level is transformed to the OpenTelemetry Severity types in the following way.

*   [zerolog.DebugLevel](https://pkg.go.dev/github.com/rs/zerolog#DebugLevel) is transformed to [log.SeverityDebug](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityDebug)
*   [zerolog.InfoLevel](https://pkg.go.dev/github.com/rs/zerolog#InfoLevel) is transformed to [log.SeverityInfo](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityInfo)
*   [zerolog.WarnLevel](https://pkg.go.dev/github.com/rs/zerolog#WarnLevel) is transformed to [log.SeverityWarn](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityWarn)
*   [zerolog.ErrorLevel](https://pkg.go.dev/github.com/rs/zerolog#ErrorLevel) is transformed to [log.SeverityError](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityError)
*   [zerolog.PanicLevel](https://pkg.go.dev/github.com/rs/zerolog#PanicLevel) is transformed to [log.SeverityFatal1](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal1)
*   [zerolog.FatalLevel](https://pkg.go.dev/github.com/rs/zerolog#FatalLevel) is transformed to [log.SeverityFatal2](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal2)

NOTE: Fields are not transformed because of [https://github.com/rs/zerolog/issues/493](https://github.com/rs/zerolog/issues/493).

Example [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzerolog#example-package "Go to Example")

package main

import (
	"os"

	"github.com/rs/zerolog"

	"go.opentelemetry.io/contrib/bridges/otelzerolog"
	"go.opentelemetry.io/otel/log/noop"
)

func main() {
	// Use a working LoggerProvider implementation instead e.g. using go.opentelemetry.io/otel/sdk/log.
	provider := noop.NewLoggerProvider()

	// Create a logger that emits logs to both STDOUT and the OTel Go SDK.
	hook := otelzerolog.NewHook("my/pkg/name", otelzerolog.WithLoggerProvider(provider))
	logger := zerolog.New(os.Stdout).With().Logger()
	logger = logger.Hook(hook)
}
