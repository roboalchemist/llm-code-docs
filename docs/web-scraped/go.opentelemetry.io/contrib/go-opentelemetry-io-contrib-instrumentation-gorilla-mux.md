### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/gorilla/mux#pkg-overview "Go to Overview")

Package mux provides functions to trace the gorilla/mux package ([https://github.com/gorilla/mux](https://github.com/gorilla/mux)).

Currently only the routing of a received message can be instrumented. To do it, use the Middleware function.
