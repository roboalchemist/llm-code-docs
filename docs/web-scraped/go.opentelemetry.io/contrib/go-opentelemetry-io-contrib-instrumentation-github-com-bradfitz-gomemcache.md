### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/github.com/bradfitz/gomemcache#pkg-overview "Go to Overview")

Package gomemcache provides tracing instrumentation for the memcached client ([https://github.com/bradfitz/gomemcache](https://github.com/bradfitz/gomemcache)).

The instrumentation works by wrapping the memcache client by calling `NewClientWithTracing` and tracing it's every operation.

#### func [NewClientWithTracing](https://github.com/open-telemetry/opentelemetry-go-contrib/blob/instrumentation/github.com/bradfitz/gomemcache/v0.11.0/instrumentation/github.com/bradfitz/gomemcache/gomemcache.go#L51)[¶](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/github.com/bradfitz/gomemcache#NewClientWithTracing "Go to NewClientWithTracing")

func NewClientWithTracing(client *[memcache](https://pkg.go.dev/github.com/bradfitz/gomemcache/memcache).[Client](https://pkg.go.dev/github.com/bradfitz/gomemcache/memcache#Client), opts ...[Option](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/github.com/bradfitz/gomemcache#Option)) *[Client](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/github.com/bradfitz/gomemcache#Client)

NewClientWithTracing wraps the provided memcache client to allow tracing of all client operations. Accepts options to set trace provider and service name, otherwise uses registered global trace provider and default value for service name.

Every client operation starts a span with appropriate attributes, executes the operation and ends the span (additionally also sets a status error code and message, if an error occurs). Optionally, client context can be set before an operation with the WithContext method.
