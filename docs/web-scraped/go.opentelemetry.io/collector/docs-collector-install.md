Install the Collector
=====================

You can deploy the OpenTelemetry Collector on a variety of operating systems and architectures. The following instructions show you how to download and install the latest stable version of the Collector for your environment.

Before you begin, make sure you understand Collector fundamentals, including [deployment patterns](https://opentelemetry.io/docs/collector/deploy/), [components](https://opentelemetry.io/docs/collector/components/), and [configuration](https://opentelemetry.io/docs/collector/configuration/).

Build from source[](https://opentelemetry.io/docs/collector/install/#build-from-source)
---------------------------------------------------------------------------------------

You can build the latest version of the Collector based on the local operating system using the following commands:

```sh
git clone https://github.com/open-telemetry/opentelemetry-collector.git
cd opentelemetry-collector
make install-tools
make otelcorecol
```
