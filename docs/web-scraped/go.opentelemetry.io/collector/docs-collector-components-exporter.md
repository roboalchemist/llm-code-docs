Exporters
=========

List of available OpenTelemetry Collector exporters

Exporters send telemetry data to observability backends and destinations. For more information on how to configure exporters, see the [Collector configuration documentation](https://opentelemetry.io/docs/collector/configuration/#exporters).

Note

Components marked with ⚠️ are unmaintained and have no active codeowners. They may not receive regular updates or bug fixes.
