#### type [CPUModeAttr](https://github.com/open-telemetry/opentelemetry-go/blob/v1.40.0/semconv/v1.32.0/processconv/metric.go#L27)[¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.32.0/processconv#CPUModeAttr "Go to CPUModeAttr")

type CPUModeAttr [string](https://pkg.go.dev/builtin#string)

CPUModeAttr is an attribute conforming to the cpu.mode semantic conventions. It represents a process SHOULD be characterized *either* by data points with no `mode` labels, *or only* data points with `mode` labels.
