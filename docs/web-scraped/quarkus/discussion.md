Discussion about Quarkus
========================

### Multiple ways to connect with others via forum, stack, chat, or email groups.

### Quarkus's Collaborative Community Forum

The core foundation of every good open source project is open discussion and collaboration. Quarkus is committed to this belief. We've enabled GitHub Discussions so the community can have open communication. This gives a focal point for the community to ask and answer questions regarding the project. This means all of that great knowledge can be shared and is discoverable. Additionally, other conversations about the Quarkus Community can be had there as well. Come collaborate with us.

[Go to the Quarkus GitHub Discussions](https://github.com/quarkusio/quarkus/discussions)

### Quarkus Chat Options

For usage questions, ask on [Stack Overflow with the `quarkus` tag](https://stackoverflow.com/questions/tagged/quarkus).

For questions related to the development of Quarkus:

*   Use the [quarkus-dev Google Groups](https://groups.google.com/d/forum/quarkus-dev)
*   Chat using [Zulip](https://quarkusio.zulipchat.com/#narrow/stream/187038-dev) (`#dev` stream)
