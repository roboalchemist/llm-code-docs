Quarkus 3.32.2 - Maintenance release
====================================

By ![Image 2](https://www.gravatar.com/avatar/ec96387a1a8295b6fd6dffb68f80c352)[Guillaume Smet](https://quarkus.io/author/gsmet)

Today, we released Quarkus 3.32.2, the first maintenance release for our 3.32 stream (we skipped 3.32.0).

We encourage you to test it, even if you track LTS releases, as 3.32 will be the foundation for 3.33 LTS.

[](https://quarkus.io/blog/quarkus-3-32-2-released/#update)Update
-----------------------------------------------------------------

To update to Quarkus 3.32, we recommend updating to the latest version of the Quarkus CLI and run:

```bash
quarkus update
```

Note that `quarkus update` can update your applications from any version of Quarkus (including 2.x) to Quarkus 3.32.

For more information about the adjustments you need to make to your applications, please refer to the [Quarkus 3.32 migration guide](https://github.com/quarkusio/quarkus/wiki/Migration-Guide-3.32).
