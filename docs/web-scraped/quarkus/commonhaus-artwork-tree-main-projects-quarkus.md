Logo Assets
-----------

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#logo-assets)
There are two alignments of the Quarkus logo (vertical and horizontal), use either depending on space requirements.

> **Background Color** Notice there are multiple versions of logo coloring for use on different background colors (light and dark options). The "default" logo is for use on a light background while the "reverse" logo is for use on dark backgrounds. The one-color versions are designed for use when you have to present it monochromatically.

Logo Do's and Don't
-------------------

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#logo-dos-and-dont)
The key to keeping the Quarkus brand strong is consistency. We do this by maintaining a uniform way of presenting the logo across all mediums over time. This means we need to respect the spacing requirements and supplied color options.

Logo Spacing
------------

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#logo-spacing)
[![Image 13: Use proper spacing around vertical logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_spacing_vert.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_spacing_vert.png)

[![Image 14: Use proper spacing around horizontal logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_spacing_hori.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_spacing_hori.png)

The logo needs space to breath so make sure you leave at least an icon's height and width around it.

What NOT to do
--------------

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#what-not-to-do)
Make sure you use the logo correctly and don't do silly stuff like...

[![Image 15: Don't recolor the logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_recolor.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_recolor.png)

Don't use other colors in the logo.

[![Image 16: Don't resize elements of the logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_resize.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_resize.png)

Don't resize individual elements of the logo.

[![Image 17: Don't skew the logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_skew.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_skew.png)

Don't squish or strech the logo to change it's proportions.

[![Image 18: Don't put a dropshadow on the logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_dropshadow.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_dropshadow.png)

Don't add a dropshadow effect on the logo on a white background.

[![Image 19: Don't add a gradient to the logo](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_gradient.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_gradient.png)

Don't add gradient effects to interior of the logo.

[![Image 20: Don't use the logotype by itself](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_logotype.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/logo_logotype.png)

Don't use the logotype by itself.

Color Palette
-------------

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#color-palette)
### Primary Colors

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#primary-colors)
[![Image 27: Primary color palette](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/colorpalette_primary.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/colorpalette_primary.png)

The Triad of Quarkus brand colors: Quarkus Blue, Dark Blue and Red. These colors are used most broadly in building visual associations with Quarkus.

*   Quarkus Blue (#4695EB) Pantone Process Blue U
*   Quarkus Dark Blue (#0D1C2C) Pantone 3035 U
*   Quarkus Red (#FF004A) Pantone 032 U

### Secondary Colors

[](https://github.com/commonhaus/artwork/tree/main/projects/quarkus#secondary-colors)
[![Image 28: Secondary color palette](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/colorpalette_secondary.png)](https://raw.githubusercontent.com/commonhaus/artwork/refs/heads/main/projects/quarkus/images/colorpalette_secondary.png)

The secondary colors extend the range of colors available for use in graphs, charts and other limited uses.

*   Quarkus Orange (#F18F01)
*   Quarkus Green (#6AB443)
*   Quarkus Purple (#6622CC)
