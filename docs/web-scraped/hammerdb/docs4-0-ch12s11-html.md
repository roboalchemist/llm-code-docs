Title: 11. Calculate the Geometric Mean

URL Source: https://www.hammerdb.com/docs4.0/ch12s11.html

Markdown Content:
For comparison of HammerDB TPROC-H workloads between systems it is recommended to use the geometric mean of the query times. This can be a straightforward comparison between power tests. When comparing throughput tests it is recommended to compare the geomean of the slowest virtual user when comparing throughput tests of an equal number of virtual users across systems.
