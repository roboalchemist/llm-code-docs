Title: 6. Installing and Starting HammerDB on Windows

URL Source: https://www.hammerdb.com/docs4.0/ch01s06.html

Markdown Content:
To install HammerDB on Windows you have the option of using the self-extracting installer or zipfile. The self-extracting installer will create an uninstall executable for you. A zipfile installation can be deleted manually. In both cases the install is entirely self-contained within the installation directory.

### [](https://www.hammerdb.com/docs4.0/ch01s06.html)6.1.Self Extracting Installer

Double click on the Setup file and the language choice is shown.

Click continue to begin the installation.

**Figure 1.1.HammerDB Version**

![Image 1: HammerDB Version](https://www.hammerdb.com/docs4.0/resources/ch1-4.PNG)

Read and Accept the GPL License Agreement.

**Figure 1.2.GPL v3 License**

![Image 2: GPL v3 License](https://www.hammerdb.com/docs4.0/resources/ch1-5.PNG)

Choose the installation directory.

**Figure 1.3.Choose the Installation Directory**

![Image 3: Choose the Installation Directory](https://www.hammerdb.com/docs4.0/resources/ch1-6.PNG)

Press Next to begin the install.

**Figure 1.4.Files copying**

![Image 4: Files copying](https://www.hammerdb.com/docs4.0/resources/ch1-7.PNG)

The installer will extract the files into the chosen directory.

**Figure 1.5.Complete**

![Image 5: Complete](https://www.hammerdb.com/docs4.0/resources/ch1-8.PNG)

Complete the Install by viewing the Readme File and running HammerDB. If both options are chosen HammerDB will run after the Readme is closed.

**Figure 1.6.Complete the Setup Wizard**

![Image 6: Complete the Setup Wizard](https://www.hammerdb.com/docs4.0/resources/ch1-8a.PNG)

HammerDB will start ready for you to use

**Figure 1.7.HammerDB Started**

![Image 7: HammerDB Started](https://www.hammerdb.com/docs4.0/resources/ch1-9.PNG)

### [](https://www.hammerdb.com/docs4.0/ch01s06.html)6.2.Zip File

As an alternative to the self-extracting installer you can download and extract the zipfile into a directory of your choice.

**Figure 1.8.Zip File**

![Image 8: Zip File](https://www.hammerdb.com/docs4.0/resources/ch1-10.PNG)

### [](https://www.hammerdb.com/docs4.0/ch01s06.html)6.3.Starting HammerDB

After installation double-click on the "Windows Batch File" hammerdb to start hammerdb.

**Figure 1.9.hammerdb batch file**

![Image 9: hammerdb batch file](https://www.hammerdb.com/docs4.0/resources/ch1-11.PNG)

### [](https://www.hammerdb.com/docs4.0/ch01s06.html)6.4.Uninstalling HammerDB

For a zipfile installation, delete the hammerDB directory. For an installer based installation double-click on uninstall and follow the on-screen prompts.

**Figure 1.10.Uninstall**

![Image 10: Uninstall](https://www.hammerdb.com/docs4.0/resources/ch1-12.PNG)
