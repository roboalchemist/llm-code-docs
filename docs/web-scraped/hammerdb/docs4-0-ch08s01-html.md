From the tree-view select Mode Options.

**Figure 8.2.Mode Options**

![Image 1: Mode Options](https://www.hammerdb.com/docs4.0/resources/ch11-2.PNG)

This displays the Mode Options as shown in Figure 3 confirming that the current mode is Local.

**Figure 8.3.Mode Options Select**

![Image 2: Mode Options Select](https://www.hammerdb.com/docs4.0/resources/ch11-3.PNG)

Select Primary Mode and click OK.

**Figure 8.4.Primary Mode Select**

![Image 3: Primary Mode Select](https://www.hammerdb.com/docs4.0/resources/ch11-6.PNG)

Confirm the selection.

**Figure 8.5.Mode Confirmation**

![Image 4: Mode Confirmation](https://www.hammerdb.com/docs4.0/resources/ch11-4.PNG)

This will show that Master Mode is now active and the ID and hostname it is running on.

**Figure 8.6.Mode Active**

![Image 5: Mode Active](https://www.hammerdb.com/docs4.0/resources/ch11-5.PNG)

Note that this will also be recorded in the console display and the current Mode displayed in the status bar at the bottom right of the Window.
