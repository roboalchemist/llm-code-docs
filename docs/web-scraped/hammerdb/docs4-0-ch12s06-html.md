Title: 6. Loading the Driver Script

URL Source: https://www.hammerdb.com/docs4.0/ch12s06.html

Markdown Content:
After selected the Driver Script Options the Driver Script is loaded. The configured options can be seen in the Driver Script window and also modified directly there. The Load option can also be used to refresh the script to the configured Options. Pay particular attention to the Scale Factor value shown as "scale_factor" if different from the schema that you have loaded. The "scale factor" value is inherited from the schema build so you may have to change it manually in the script to the same size of the schema that you paln to test.
