Title: 7. Installing and Starting HammerDB on Linux

URL Source: https://www.hammerdb.com/docs4.0/ch01s07.html

Markdown Content:
To install HammerDB on Linux you have the option of using the self-extracting installer or a tar.gz file. The self-extracting installer will create an uninstall executable for you. A tar.gz installation can be deleted manually. In both cases the install is entirely self-contained within the installation directory.

### [](https://www.hammerdb.com/docs4.0/ch01s07.html)7.1.Self Extracting Installer

To install from the self-extracting installer using a graphical environment refer to the previous section on the self-extracting installer for Windows, the installation method is the same.

### [](https://www.hammerdb.com/docs4.0/ch01s07.html)7.2.Tar.gz File

To install from the tar.gz run the command

tar -zxvf HammerDB-4.0.tar.gz 
This will extract HammerDB into a directory named HammerDB-4.0.

### [](https://www.hammerdb.com/docs4.0/ch01s07.html)7.3.Starting HammerDB

To start HammerDB change to the HammerDB directory and run locally as follows.

./hammerdb

### [](https://www.hammerdb.com/docs4.0/ch01s07.html)7.4.Uninstalling HammerDB

To uninstall HammerDB on Linux run the uninstall executable for the self-extracting installer or remove the directory for the tar.gz install.
