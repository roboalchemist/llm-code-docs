Usage by Hour API
=================

This endpoint returns account usage statistics by hour, sorted by date ascending. Usage data is for you entire account and is not broken down by application.

[URI](https://docs.trackjs.com/data-api/usage-by-hour/#uri "Permalink Here")
----------------------------------------------------------------------------

GET https://api.trackjs.com/{CUSTOMER_ID}/v1/usage/hourly![Image 3: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
The following querystring parameters are accepted:

### [endDate](https://docs.trackjs.com/data-api/usage-by-hour/#enddate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return usage data from before this date and time.

* * *

### [startDate](https://docs.trackjs.com/data-api/usage-by-hour/#startdate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return usage data from after this date and time.

[Response](https://docs.trackjs.com/data-api/usage-by-hour/#response "Permalink Here")
