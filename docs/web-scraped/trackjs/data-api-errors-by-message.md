Errors by Message API
=====================

This endpoint returns the count of Error count by Message, sorted by count descending.

[URI](https://docs.trackjs.com/data-api/errors-by-message/#uri "Permalink Here")
--------------------------------------------------------------------------------

GET https://api.trackjs.com/{CUSTOMER_ID}/v1/errors/messages![Image 3: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
The following querystring parameters are accepted:

### [application](https://docs.trackjs.com/data-api/errors-by-message/#application "Permalink Here")

String

Optional

Filter the results to only the Application key provided.

* * *

### [endDate](https://docs.trackjs.com/data-api/errors-by-message/#enddate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return errors before this date. Time precision is within 1 second.

* * *

### [startDate](https://docs.trackjs.com/data-api/errors-by-message/#startdate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return errors after this date. Time precision is within 1 second.

* * *

### [page](https://docs.trackjs.com/data-api/errors-by-message/#page "Permalink Here")

Number

Optional

The page of data you want returned. By default, the first page of data is returned. See [Paging](https://docs.trackjs.com/data-api/#paging).

* * *

### [size](https://docs.trackjs.com/data-api/errors-by-message/#size "Permalink Here")

Number 1-1000

Optional

The size of the page of data you want returned. See [Paging](https://docs.trackjs.com/data-api/#paging).

* * *

### [sort](https://docs.trackjs.com/data-api/errors-by-message/#sort "Permalink Here")

Sort String

Optional

By default the endpoint returns results sorted by date in descending order. You may adjust the sort field and sort direction. Supported fields are 
```plaintext
"count"
```
 and 
```plaintext
"usercount"
```
. Sort directions are specified by appending 
```plaintext
"|asc"
```
 or 
```plaintext
"|desc"
```
. Default value is 
```plaintext
"count|desc"
```
 if not specified.

[Response](https://docs.trackjs.com/data-api/errors-by-message/#response "Permalink Here")
------------------------------------------------------------------------------------------

{ "data": [{ "key": "Cannot read property on of undefined", "count": 54, "userCount": 12, "lastSeen": "2016-09-22T22:42:23.579+00:00", "status": "Investigating", "trackJsUrl": "https://my.trackjs.com/messages/filtered?message=Cannot+read+property+%27on%27+of+undefined&" }], "metadata": { "startDate": "2014-09-11T00:00:00Z", "page": 1, "size": 3, "hasMore": true, "trackJsUrl": "https://my.trackjs.com/messages" }}![Image 4: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
[Example](https://docs.trackjs.com/data-api/errors-by-message/#example "Permalink Here")
----------------------------------------------------------------------------------------

curl -H "Authorization: {API_KEY}" "https://api.trackjs.com/{CUSTOMER_ID}/v1/errors/messages?startDate=2018-11-20&endDate=2018-11-27&sort=usercount|desc"![Image 5: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
