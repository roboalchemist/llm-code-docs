Page Views by Hour
==================

TrackJS records the number of times the Agent loads on your site as Page Views. This endpoint returns the Error count by hour, sorted by date descending.

This endpoint returns the page view counts per hour, sorted by date descending.

[URI](https://docs.trackjs.com/data-api/pageviews-by-hour/#uri "Permalink Here")
--------------------------------------------------------------------------------

GET https://api.trackjs.com/{CUSTOMER_ID}/v1/hits/hourly![Image 3: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
The following querystring parameters are accepted:

### [application](https://docs.trackjs.com/data-api/pageviews-by-hour/#application "Permalink Here")

String

Optional

Filter the results to only the Application key provided.

* * *

### [endDate](https://docs.trackjs.com/data-api/pageviews-by-hour/#enddate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return errors before this date and time. Time precision is within 1 second.

* * *

### [startDate](https://docs.trackjs.com/data-api/pageviews-by-hour/#startdate "Permalink Here")

ISO 8601 String

Optional

Filter the results to only return errors after this datee and time. Time precision is within 1 second.

* * *

### [page](https://docs.trackjs.com/data-api/pageviews-by-hour/#page "Permalink Here")

Number

Optional

The page of data you want returned. By default, the first page of data is returned. See [Paging](https://docs.trackjs.com/data-api/#paging).

* * *

### [size](https://docs.trackjs.com/data-api/pageviews-by-hour/#size "Permalink Here")

Number 1-1000

Optional

The size of the page of data you want returned. See [Paging](https://docs.trackjs.com/data-api/#paging).

[Response](https://docs.trackjs.com/data-api/pageviews-by-hour/#response "Permalink Here")
------------------------------------------------------------------------------------------

{ "data": [{ "key": "2020-05-28T15:00:00Z", "count": 2835 }, { "key": "2020-05-28T14:00:00Z", "count": 3890 }, { "key": "2020-05-28T13:00:00Z", "count": 1990 }], "metadata": { "startDate": "2020-05-20T02:00:00Z", "endDate": "2020-05-28T16:00:00Z", "totalCount": 11, "page": 1, "size": 3, "hasMore": true }}![Image 4: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
[Example](https://docs.trackjs.com/data-api/pageviews-by-hour/#example "Permalink Here")
----------------------------------------------------------------------------------------

curl -H "Authorization: {API_KEY}" "https://api.trackjs.com/{CUSTOMER_ID}/v1/hits/hourly?startDate=2020-05-20T02:00:00Z&endDate=2020-05-28T16:00:00Z"![Image 5: Copy](https://docs.trackjs.com/assets/images/copy.svg)Copy
