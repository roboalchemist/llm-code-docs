CLI
===

How to install and use the Nuxt Charts CLI for chart management.

[](https://nuxtcharts.com/pricing)Please support us by purchasing a **license** to access premium components, blocks, and templates!

[1. Install CLI](https://nuxtcharts.com/docs/getting-started/cli#_1-install-cli)
--------------------------------------------------------------------------------

Install the Nuxt Charts CLI globally using your preferred package manager:

pnpm yarn npm bun

```
pnpm add -g nuxt-charts-cli
```

```
yarn global add nuxt-charts-cli
```

```
npm install -g nuxt-charts-cli
```

```
bun add -g nuxt-charts-cli
```

[2. Create Token](https://nuxtcharts.com/docs/getting-started/cli#_2-create-token)
----------------------------------------------------------------------------------

Create your token via [nuxtcharts.com/payments](https://nuxtcharts.com/payments) (API Tokes > CLI Access Token)

[3. Usage](https://nuxtcharts.com/docs/getting-started/cli#_3-usage)
--------------------------------------------------------------------

### [Set config token](https://nuxtcharts.com/docs/getting-started/cli#set-config-token)

Terminal

```
nuxt-charts config set-token <your-token-here>
```

### [List charts](https://nuxtcharts.com/docs/getting-started/cli#list-charts)

Terminal

```
nuxt-charts list
```

### [Add a chart](https://nuxtcharts.com/docs/getting-started/cli#add-a-chart)

Terminal

```
nuxt-charts add calendar
```

* * *

For more CLI options and advanced usage, see the official documentation or run:

Terminal

```
nuxt-charts --help
```
