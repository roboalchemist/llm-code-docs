Documentation

Drone by [Harness](https://harness.io/)™ is a modern Continuous Integration platform that empowers busy teams to automate their build, test and release workflows using a powerful, cloud native pipeline engine.

Installation Guides [See All](https://docs.drone.io/server/overview/)

[GitHub](https://docs.drone.io/server/provider/github/)[GitHub Enterprise](https://docs.drone.io/server/provider/github-enterprise/)[Gitee](https://docs.drone.io/server/provider/gitee/)[Bitbucket](https://docs.drone.io/server/provider/bitbucket-cloud/)[Bitbucket Server](https://docs.drone.io/server/provider/bitbucket-server/)[Gitea](https://docs.drone.io/server/provider/gitea/)

Drone comes in two distributions: the Enterprise Edition and the Community Edition. If you only want to download open source software, the [Community Edition](https://github.com/harness/drone/blob/drone/BUILDING_OSS) is the best option. The [Community Edition](https://github.com/harness/drone/blob/drone/BUILDING_OSS) does not contain any proprietary code.
