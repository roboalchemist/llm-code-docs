`virtualenv` is a tool to create isolated Python environments. Since Python 3.3, a subset of it has been integrated into the standard library under the `venv` module. For how `virtualenv` compares to the stdlib `venv` module, see [Explanation](https://virtualenv.pypa.io/en/latest/explanation.html).
