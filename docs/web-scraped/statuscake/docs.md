Introduction
============

StatusCake is a website monitoring platform with a focus on providing
observability for the applications you maintain. Here you will find developer
documentation to help integrate StatusCake into your own monitoring suites.

Create an [uptime check](https://app.statuscake.com/AddSite.php) or use our API
to automate your experience
