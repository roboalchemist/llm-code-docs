These are standalone applications that utilize VisPy.

### DIVE[#](https://vispy.org/thirdparty.html#dive "Link to this heading")

[DIVE](https://github.com/Dive576/DIVE) is an extendable Qt GUI designed to simplify the process for plotting pandas DataFrames in VisPy.

Key features include: 2D/3D axes, animated plots, and data filtering/selection.
