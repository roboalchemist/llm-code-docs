[#](https://ajv.js.org/components.html#ajv-classes) Ajv classes
---------------------------------------------------------------

[lib/core.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/core.ts) - core Ajv class without any keywords. All Ajv methods for managing schemas and extensions are defined in this class.

[lib/ajv.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/ajv.ts) - subclass of Ajv core with JSON Schema draft-07 keywords.

[lib/2019.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/2019.ts) - subclass of Ajv core with JSON Schema draft-2019-09 keywords.

[lib/jtd.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/jtd.ts) - subclass of Ajv core with JSON Type Definition support.

[#](https://ajv.js.org/components.html#schema-compilation) Schema compilation
-----------------------------------------------------------------------------

[lib/compile(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile) - code for schema compilation

[lib/compile/index.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/index.ts) - the main recursive function code for schema compilation, functions for reference resolution, the interface for schema compilation context (`SchemaCxt`).

[lib/compile/context.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/context.ts) - the class for keyword code generation `KeywordCxt`. All pre-defined keywords and user-defined keywords that use `code` function are passed an instance of this class.

[lib/compile/rules.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/rules.ts) - data structure to store references to all all keyword definitions that were added to Ajv instance, organised by data type.

[lib/compile/subschema.ts(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/subschema.ts) - creates schema context (`SchemaCxt`) to generate code for subschemas - used by all applicator keywords in [lib/vocabularies/applicator(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/vocabularies/applicator).

[lib/compile/codegen(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/codegen) - the api for [code generation](https://ajv.js.org/codegen.html).

[lib/compile/validate(opens new window)](https://github.com/ajv-validator/ajv/blob/master/lib/compile/validate) - code to iterate the schema to generate code of validation function.
