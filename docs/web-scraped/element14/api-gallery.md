# API Gallery 

## Product Search API in Use

Our Product Search API is designed to give our partners access to the most up to date pricing, stock and product detail available. Our Stock levels and pricing are maintained in real-time giving you up to the minute, dependable data. This data is then supplemented with Product Descriptions, Product Images, Datasheets and other key information.

Since we launched our first API in 2009, the number of users has continued to grow. From well known industry partners, customers and various desktop applications. The following highlights a great example of how integrating product data from Newark, Farnell and element14 can really add value to your end users.\
\
