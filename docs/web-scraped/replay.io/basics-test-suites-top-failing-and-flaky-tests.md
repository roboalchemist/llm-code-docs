Test Suite Dashboard

Top failing and flaky tests
===========================

Replay gives you an overview of your top failures and flaky tests, allowing you to focus on the most critical issues.
---------------------------------------------------------------------------------------------------------------------

![Image 1: Top failing and flaky tests](https://docs.replay.io/_next/image?url=%2Fimages%2Ffailing-tests.png&w=1920&q=75)

Sort by top failures
--------------------

You can sort your tests by those that fail the most, helping you prioritize the most critical issues for fixing.

The percentage shown in the icon indicates how often this test fails across all runs.

![Image 2: Test runs sorted by failure rate](https://docs.replay.io/_next/image?url=%2Fimages%2Fsort-by-failed.png&w=2048&q=75)

Sort by flaky rate
------------------

This view displays your most flaky tests, which are tests that fail initially but pass upon retry.

This behavior is often due to factors connected to the testing environment, (seemingly) random frontend timing issues, or unpredicted race conditions.

These cause inconsistent results across test executions.

![Image 3: Test runs sorted by flakiness rate](https://docs.replay.io/_next/image?url=%2Fimages%2Fsort-by-flaky.png&w=1920&q=75)
