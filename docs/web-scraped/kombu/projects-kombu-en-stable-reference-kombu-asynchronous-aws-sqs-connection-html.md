Override make_request to support different protocols.

botocore has changed the default protocol of communicating with SQS backend from ‘query’ to ‘json’, so we need a special implementation of make_request for SQS. More information on this can be found in: [https://github.com/celery/kombu/pull/1807](https://github.com/celery/kombu/pull/1807).

protocol_params: Optional[dict[str, dict]] of per-protocol additional parameters.
Supported for the SQS query to json protocol transition.
