Samples & Demos
===============

The recommended samples and demos for learning how to use MonoGame.

Official Mono Game Samples[](https://docs.monogame.net/articles/samples.html#official-monogame-samples)
-------------------------------------------------------------------------------------------------------

These are [the official samples](https://github.com/MonoGame/MonoGame.Samples):

2D Samples[](https://docs.monogame.net/articles/samples.html#2d-samples)
------------------------------------------------------------------------

[AutoPong Sample 2D This project shows you how to make the classic game of pong, with basic soundfx, in 300 lines of code. The sample has no content and no content manager use, just code! Check out the AutoPong sample](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/AutoPong)

[Platformer Sample 2D The Platformer Starter Kit is a near-complete, self-contained game solution that includes both game code and game assets. The game is a standard 2D platformer with levels, enemies, and collectable gems. \nThis sample is also available as a full cross-platform project template `mg2dstartkit`. Check out the Platformer sample](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/Platformer2D)

[NeonShooter Sample 2D This project shows you how to make a Geometry Wars-inspired twin-stick shooter, with neon graphics, crazy particle effects, and awesome music. Check out the NeonShooter sample](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/NeonShooter)

3D Samples[](https://docs.monogame.net/articles/samples.html#3d-samples)
------------------------------------------------------------------------

[FuelCell Sample 3D FuelCell is designed around a central concept providing basic gameplay mechanisms that are easily extensible. The FUelCell sample also includes a complete developer diary documenting its construction. Click to view 3D FuelCell sample](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/FuelCell)

[ShipGame Sample 3D Ship Game is a 3D spaceship combat game set inside a complex tunnel system featuring advanced lighting and textures, a full GPU particle system, and physics. Click to view 3D ShipGame sample](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/ShipGame)

[Coming Soon More 3D sample projects coming soon. See what is coming](https://docs.monogame.net/articles/samples.html)

Community delivered full sample projects[](https://docs.monogame.net/articles/samples.html#community-delivered-full-sample-projects)
------------------------------------------------------------------------------------------------------------------------------------

[GreedyKid The repository for "Boo! Greedy Kid" on all platforms (Windows, macOS, and Linux) and its level editor (Windows-only). Click to browse the GreedyKid Source](https://github.com/FlyingOakGames/GreedyKid)

[Old School Adventure The repository for "Old School Adventure" which was originally published using XNA on Xbox (XBLIG) and Windows Phone. Click to browse the Old School Adventure Source](https://github.com/ZenithMoonStudios/OldSchoolAdventure)

[Coming Soon More full sample projects coming soon. See what is coming](https://docs.monogame.net/articles/samples.html)

Official Mono Game tutorial source[](https://docs.monogame.net/articles/samples.html#official-monogame-tutorial-source)
-----------------------------------------------------------------------------------------------------------------------

* [Building 2D Games](https://github.com/MonoGame/MonoGame.Samples/tree/3.8.4/Tutorials/learn-monogame-2d)

XNA Game Studio Archive[](https://docs.monogame.net/articles/samples.html#xna-game-studio-archive)
--------------------------------------------------------------------------------------------------

The [XNA Game Studio archive](https://github.com/SimonDarksideJ/XNAGameStudio) contains all of the original XNA sample content. This contains a vast array of samples demonstrating almost every feature of XNA, which is easily translatable to MonoGame.

There are pre-converted samples available on the repository as well.

Mono Game.Extended Samples[](https://docs.monogame.net/articles/samples.html#monogameextended-samples)
------------------------------------------------------------------------------------------------------

[MonoGame.Extended](https://github.com/craftworkgames/MonoGame.Extended) is an extension for MonoGame that makes it easier to make games. They have [demos](https://github.com/craftworkgames/MonoGame.Extended-samples/tree/master/src/Demos) and [games](https://github.com/craftworkgames/MonoGame.Extended-samples/tree/master/src/Games) that can be tried out.

Cart Blanche's Samples[](https://docs.monogame.net/articles/samples.html#cartblanches-samples)
----------------------------------------------------------------------------------------------

This is a large library of [MonoGame Samples](https://github.com/CartBlanche/MonoGame-Samples?tab=readme-ov-file#monogame-samples) covering a wide range of topics, including:

* 27 [2D samples](https://github.com/CartBlanche/MonoGame-Samples?tab=readme-ov-file#2d-samples)
* 8 [3D samples](https://github.com/CartBlanche/MonoGame-Samples?tab=readme-ov-file#3d-samples)
* [Physics](https://github.com/CartBlanche/MonoGame-Samples/tree/main/FarseerPhysics/FarseerSamples)
* [Networking](https://github.com/CartBlanche/MonoGame-Samples?tab=readme-ov-file#networking-samples) (Not fully functional yet)
* [Miscellaneous](https://github.com/CartBlanche/MonoGame-Samples?tab=readme-ov-file#miscellaneous-samples)
* [Shaders](https://github.com/CartBlanche/MonoGame-Samples/blob/main/2DShaderSample/README.md)

It also includes games such as:

* [Card Game Starter Kit](https://github.com/CartBlanche/MonoGame-Samples/tree/main/CardsStarterKit)
* [Flood Control](https://github.com/CartBlanche/MonoGame-Samples/tree/main/GameDevelopmentbyExample/FloodControl)
* [Role Playing Game Starter Kit](https://github.com/CartBlanche/MonoGame-Samples/tree/main/RolePlayingGame)
* [Pacman](https://github.com/CartBlanche/MonoGame-Samples/tree/main/PacMan)
* [Tetris](https://github.com/CartBlanche/MonoGame-Samples/tree/main/Tetris)

With more on the way.
