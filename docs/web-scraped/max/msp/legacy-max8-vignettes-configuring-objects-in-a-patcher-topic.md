Configuring Objects In A Patcher
--------------------------------

Objects have many options in the form of tag [attributes](https://docs.cycling74.com/max8/vignettes/attributes_intro) and arguments. These attributes and arguments have defaults which can be overridden and set directly in the object box when you make it, or changed afterwards by either editing the object box or using the [Object Inspector](https://docs.cycling74.com/max8/vignettes/inspector). Each object has attributes that control their visual representation as well, such as location in the patcher, font type, colors, etc. Objects can be [encapsulated](https://docs.cycling74.com/max8/vignettes/encapsulate_deencapsulate) into [subpatchers](https://docs.cycling74.com/max8/vignettes/subpatchers), and set to appear in [Presentation Mode](https://docs.cycling74.com/max8/vignettes/presentation_mode).
