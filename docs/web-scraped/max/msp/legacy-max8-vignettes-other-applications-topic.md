Using Max with other applications
---------------------------------

One of the great strengths of Max is its ability to "play nicely with others". Max is often used as the glue between various applications, systems or hardware in projects. The single most prominent example of this is the integration with [Ableton Live.](http://ableton.com/) Other ways to integrate Max with applications include MIDI, and building network functionality with TCP or UDP. The maxurl object gives Max the ability to interact with website APIs, so that Max patchers can talk with the internet.

| Name | Description |
| --- | --- |
| [Max For Live](https://docs.cycling74.com/max8/vignettes/max_for_live_topic) | Max For Live |
| [udpsend](https://docs.cycling74.com/max8/refpages/udpsend) | Send messages over a network |
| [udpreceive](https://docs.cycling74.com/max8/refpages/udpreceive) | Receive messages over a network |
| [mxj](https://docs.cycling74.com/max8/refpages/mxj) | Execute Java in Max |
| [midiout](https://docs.cycling74.com/max8/refpages/midiout) | Transmit raw MIDI data |
| [maxurl](https://docs.cycling74.com/max8/refpages/maxurl) | Make HTTP requests |
