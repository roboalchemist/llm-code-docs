Markdown Content:
Allo! Allo! In this 25-minutes codelab, you will build a fully functional web application from scratch with the help of [Yeoman](https://yeoman.io/) and FountainJS. The sample app will be written in [React](https://facebook.github.io/react/), [Angular2](https://angular.io/) or [Angular1](https://angularjs.org/).

![Image 1](https://yeoman.io/static/yeoman-008.cfa02809bf.png)

Don’t know any React or Angular? That’s okay, we’ll walk you through it. However, we do assume that you have some previous JavaScript experience.

Build this sample app with Yeoman
---------------------------------

The sample web app you’ll build today will be a implementation of [TodoMVC](http://todomvc.com/). You will be able to add todos, delete todos, filter todos, and together we’ll add a feature to save todos offline.

![Image 2: Finished TodoMVC app](https://yeoman.io/static/00_Finished_TodoMVC_app.17f395442f.png)

What's in this codelab?
-----------------------

We will build the above TodoMVC app from scratch. Each step builds on the previous so go through each step one by one.

* **Step 1:**[Set up your dev environment »](https://yeoman.io/codelab/setup)
* **Step 2:**[Install a Yeoman generator »](https://yeoman.io/codelab/install-generators)
* **Step 3:**[Use a generator to scaffold out your app »](https://yeoman.io/codelab/scaffold-app)
* **Step 4:**[Review the Yeoman-generated app directory structure »](https://yeoman.io/codelab/review-generated-files)
* **Step 5:**[Preview your app in the browser »](https://yeoman.io/codelab/preview-inbrowser)
* **Step 6:**[Test with Karma and Jasmine »](https://yeoman.io/codelab/run-unit-tests)
* **Step 7:**[Make todos persistent with local storage »](https://yeoman.io/codelab/local-storage)
* **Step 8:**[Get ready for production »](https://yeoman.io/codelab/prepare-production)
* Like what you see? [Yeoman can do more »](https://yeoman.io/codelab/keep-going)

It will take approximately 25 minutes to complete this codelab. By the end, you’ll have a snazzy TodoMVC app and your computer will be set up to build even more awesome web apps in the future.
