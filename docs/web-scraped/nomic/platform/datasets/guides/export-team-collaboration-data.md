# Nomic Documentation

Source: https://docs.nomic.ai/platform/datasets/guides/export-team-collaboration-data

Atlas is particularly valuable for analyzing team collaboration data, as it helps organizations understand communication patterns and knowledge sharing across their tools and teams. By visualizing conversations, documents, and project discussions in Atlas's interactive maps, teams can uncover emerging topics, identify knowledge gaps, and spot collaboration opportunities that may be hidden in traditional views. This semantic analysis of workplace communication can improve team coordination, surface institutional knowledge, and guide organizational strategy.

## Slack Export​

- Workspace Settings > Import/Export Data
- Start Export
- Download the ZIP file
- Use a tool like slack-export-viewer to convert to CSV/JSON
Slack Export Documentation

With Slack, data export capabilities vary by plan:

- Free plan workspaces can export messages from public channels
- Business+ and Enterprise Grid plans enable full export including private channels and DMs
