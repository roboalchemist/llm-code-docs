# Nomic Documentation

Source: https://docs.nomic.ai/platform/datasets/data-maps/data-stories

Atlas Data Stories are interactive scrollable web pages that let you tell stories with your data maps.

## Example Data Stories​

Here are some examples of Atlas Data Stories that we've published on the Nomic Blog:

### Introducing Nomic Embed: A Truly Open Embedding Model

We show differences in how Nomic Embed qualitatively embeds data differently from OpenAI's Ada

### What Is US Congress Posting About In The Leadup To The Election?

We analyze Twitter data to distinguish what political parties in US Congress are posting about

### See Your Data With Dimensionality Reduction

We compare dimensionality reduction algorithms PCA, t-SNE, UMAP, and Nomic Project on MNIST image data

### Why Are Web Browsers The Best Data Browsers?

We explain why web browsers are the right technology for interacting with your data
