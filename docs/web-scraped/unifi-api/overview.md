## Official API Documentation

Ubiquiti's official UniFi Network API for programmatic control and management of UniFi network infrastructure.

### What You Can Do

The UniFi API enables:
- Device management (Access Points, Switches, Gateways, UDM/UDR)
- Client management (list, block/unblock, get statistics)
- Network and SSID configuration
- Port forwarding and firewall rules
- Traffic insights and event monitoring
- Wireless settings and advanced configurations

### Key Features

- RESTful JSON API
- API key-based authentication via X-API-Key header
- Early Access tier: 100 requests/minute
- Stable v1 tier: 10,000 requests/minute
- Comprehensive device management capabilities
