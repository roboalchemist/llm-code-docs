### [Key Features](https://www.telerik.com/kendo-react-ui/components/dialogs#key-features)

The KendoReact Dialogs provide the following key features. Note that this is not an exhaustive list.

*   Highly customizable and easy to style **appearance**
    *   [Built-in themes](https://www.telerik.com/kendo-react-ui/components/styling)
    *   [Theme customization](https://www.telerik.com/kendo-react-ui/components/styling/customizing)
    *   [Self-created themes](https://www.telerik.com/kendo-react-ui/components/styling/theme-builder)

*   Fully customizable [Action Buttons](https://www.telerik.com/kendo-react-ui/components/dialogs/dialog/action-buttons)
*   Flexible [Title customization](https://www.telerik.com/kendo-react-ui/components/dialogs/dialog/title)
*   Adjustable dimensions via the `width` and `height` properties
*   [Globalization](https://www.telerik.com/kendo-react-ui/components/dialogs/globalization)
*   Out-of-the-box **accessibility** and **keyboard navigation**

[How Do the KendoReact Dialogs Work?](https://www.telerik.com/kendo-react-ui/components/dialogs#how-do-the-kendoreact-dialogs-work)
-----------------------------------------------------------------------------------------------------------------------------------

> The KendoReact Dialogs provide [free React dialogs](https://www.telerik.com/kendo-react-ui/components/dialogs#package-contents) that don't require a commercial license key or an active trial license key to install or use.

The KendoReact Dialogs are built natively for React, with no dependencies. They leverage key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

You have complete control over your React dialogs and you can extend or override their default behaviors and appearance via props and event handlers.

### [React Dialogs with Other KendoReact Components](https://www.telerik.com/kendo-react-ui/components/dialogs#react-dialogs-with-other-kendoreact-components)

You can combine multiple KendoReact components with the React dialogs in the same application:

*   Display confirmation Dialog in a [Grid](https://www.telerik.com/kendo-react-ui/components/grid) for row actions
*   Embed [Form](https://www.telerik.com/kendo-react-ui/components/form) inside a Window

> The Dialogs package is a dependency for multiple other KendoReact components.

### [React Dialogs with Other Telerik Components](https://www.telerik.com/kendo-react-ui/components/dialogs#react-dialogs-with-other-telerik-components)

Based on the frameworks and technologies used in your project, you can combine UI components from different Telerik and Progress suites.

### [Are the KendoReact Dialogs Free to Use?](https://www.telerik.com/kendo-react-ui/components/dialogs#are-the-kendoreact-dialogs-free-to-use)

Yes. All KendoReact Dialogs are free to use without any license or registration with Progress or Telerik.

### [How to Create Dialogs in React?](https://www.telerik.com/kendo-react-ui/components/dialogs#how-to-create-dialogs-in-react)

To create an dialog in React, you need to install the KendoReact Dialogs package and then import the respective dialog components in your application. After that, you can use the various available dialogs in your React application. For more information, refer to the [Getting Started with the KendoReact Dialogs](https://www.telerik.com/kendo-react-ui/components/dialogs/get-started) article.
