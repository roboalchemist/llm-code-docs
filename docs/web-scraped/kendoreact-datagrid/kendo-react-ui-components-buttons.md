The KendoReact Buttons is a collection of customizable, responsive, and interactive UI components that you can use to trigger actions or display content in your React apps.

Use React Buttons for Free You can use the free components from the React Buttons package in production—no sign-up or license required. Buttons are part of KendoReact, an enterprise-grade UI library with 120+ [free](https://www.telerik.com/kendo-react-ui/components/free) and premium components. To test-drive premium components, [start a 30-day trial](https://www.telerik.com/try/kendo-react-ui).

### [Key Features](https://www.telerik.com/kendo-react-ui/components/buttons#key-features)

The KendoReact Buttons provide the following key features. Note that this is not an exhaustive list.

*   Highly customizable and easy to style **appearance**
    *   Built-in appearance options to adjust the form and dimensions of your buttons
    *   [Built-in themes](https://www.telerik.com/kendo-react-ui/components/styling)
    *   [Theme customization](https://www.telerik.com/kendo-react-ui/components/styling/customizing)
    *   [Self-created themes](https://www.telerik.com/kendo-react-ui/components/styling/theme-builder)
    *   [Unstyled mode](https://www.telerik.com/kendo-react-ui/components/styling/unstyled)

*   Built-in **disabled mode** via the `disabled` property
*   Customization with **[icons](https://www.telerik.com/kendo-react-ui/components/buttons/button/icons)**, **[avatars](https://www.telerik.com/kendo-react-ui/components/buttons/chip/customization)**, and **[animations](https://www.telerik.com/kendo-react-ui/components/buttons/dropdownbutton/customization)**
*   [Globalization](https://www.telerik.com/kendo-react-ui/components/buttons/globalization)
*   Out-of-the-box **accessibility** and **keyboard navigation**

> The KendoReact Buttons provide both [free React buttons](https://www.telerik.com/kendo-react-ui/components/buttons#free-react-buttons) and premium buttons which require a commercial license key or an active trial license key. If you need to develop apps using the [premium React buttons](https://www.telerik.com/kendo-react-ui/components/buttons#premium-react-buttons) or other premium KendoReact components and features, follow the instructions on the [My License page](https://www.telerik.com/kendo-react-ui/components/my-license) to activate your license.

The KendoReact Buttons are built natively for React, with no dependencies. They leverage key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

You have complete control over your React buttons and you can extend or override their default behaviors and appearance via props and event handlers.

### [React Buttons with Other KendoReact Components](https://www.telerik.com/kendo-react-ui/components/buttons#react-buttons-with-other-kendoreact-components)

You can combine multiple KendoReact components with the React buttons in the same application:

*   Add buttons to your [Grid](https://www.telerik.com/kendo-react-ui/components/grid) header or rows.
*   Use buttons to submit data with [Inputs](https://www.telerik.com/kendo-react-ui/components/inputs).
*   Build more interactive [Dialogs](https://www.telerik.com/kendo-react-ui/components/dialogs) with buttons.
*   Show [Notifications](https://www.telerik.com/kendo-react-ui/components/notification) when you trigger an action with a button.
*   Toggle between [Chart](https://www.telerik.com/kendo-react-ui/components/charts) types in complex reporting apps.

> The Buttons package is a dependency for multiple other KendoReact components.

### [Are the KendoReact Buttons Free to Use](https://www.telerik.com/kendo-react-ui/components/buttons#are-the-kendoreact-buttons-free-to-use)

The KendoReact Buttons provide both [free React buttons](https://www.telerik.com/kendo-react-ui/components/buttons#free-react-buttons) and [premium buttons](https://www.telerik.com/kendo-react-ui/components/buttons#premium-react-buttons) which require a commercial license key or an active trial license key.

You can start using the free components without any license or registration with Progress or Telerik.

### [How to Create Buttons in React](https://www.telerik.com/kendo-react-ui/components/buttons#how-to-create-buttons-in-react)

To create a button in React, you need to install the KendoReact Buttons package and then import the respective button components in your application. After that, you can use the various available button components in your React application. For more information, refer to the [Getting Started with the KendoReact Buttons](https://www.telerik.com/kendo-react-ui/components/buttons/get-started) article.

### [How to Upgrade to the Premium Buttons](https://www.telerik.com/kendo-react-ui/components/buttons#how-to-upgrade-to-the-premium-buttons)

If you want to use any of the premium [React buttons](https://www.telerik.com/kendo-react-ui/components/buttons#premium-react-buttons), UI components, or component features, you need a valid commercial license or an active trial license. A [free trial](https://www.telerik.com/try/kendo-react-ui) is available, if you want to try out the premium features before purchase.
