[The KendoReact Data Grid](https://www.telerik.com/kendo-react-ui/components/grid) enables you to style its columns, rows, and cells.

Use React Styling for Free You can use the free components from the React Styling package in production—no sign-up or license required. Styling is part of KendoReact, an enterprise-grade UI library with 120+ [free](https://www.telerik.com/kendo-react-ui/components/free) and premium components. To test-drive premium components, [start a 30-day trial](https://www.telerik.com/try/kendo-react-ui).
[Styling the KendoReact Data Grid Columns](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#styling-the-kendoreact-data-grid-columns)
----------------------------------------------------------------------------------------------------------------------------------------------------------

To style the columns of the Grid, either:

*   [Add custom cells for columns](https://www.telerik.com/kendo-react-ui/components/grid/cells), or
*   [Add class names for `td` elements](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#setting-class-names).

### [Setting Class Names](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#setting-class-names)

The [`className`](https://www.telerik.com/kendo-react-ui/components/grid/api/gridcolumnprops#classname) property applies additional class to the `td` element and can be used for styling the background, text color, text alignment, and other styling options of the Grid.

jsx

```jsx
<GridColumn field="ProductName" title="Product Name" className="product-name">
```

css

```css
product-name {
    color: 'white';
    background-color: '#888888';
}
```

[Styling the KendoReact Data Grid Rows](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#styling-the-kendoreact-data-grid-rows)
----------------------------------------------------------------------------------------------------------------------------------------------------

To style the rows of the Grid, utilize the [`rows`](https://www.telerik.com/kendo-react-ui/components/grid/api/gridprops#rows) function which allows you to modify the appearance of the rows based on the provided values:

*   [Implement custom rows in KendoReact Grid](https://www.telerik.com/kendo-react-ui/components/grid/rows/custom-rows)

[Styling the KendoReact Data Grid Individual Elements](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#styling-the-kendoreact-data-grid-individual-elements)
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

You can individually style the elements of the Grid by using CSS.

> The Grid provides multiple elements that can be individually styled. Before you apply the styling options to the desired elements, inspect the element and use selectors of higher priority.

### [Styling the KendoReact Data Grid Page Numbers](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#styling-the-kendoreact-data-grid-page-numbers)

The following example demonstrates how to style the appearance of the page numbers in the Grid.

css

```css
.k-pager-numbers .k-button {
        color: "black";
    }
```

[Applying Conditional Styling](https://www.telerik.com/kendo-react-ui/components/grid/styling/basics#applying-conditional-styling)
----------------------------------------------------------------------------------------------------------------------------------

You can use the custom `cells.data` functionality of the Grid to change the appearance of the cells on condition and based on the provided value.
