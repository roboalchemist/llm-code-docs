The KendoReact Data Grid enables you to display a context menu by using the Context Menu component out-of-the-box. The **onContextMenu** event is fired when grid cells are rendered and the context menu is triggered on a right click.

![Image 4: ninja-icon](https://www.telerik.com/kendo-react-ui/components/static/d2ecd6c1a01f6b1598a481623b8f4389/start-free-trial-icon.inline.svg)The Grid Context Menu feature is part of [KendoReact](https://www.telerik.com/kendo-react-ui) premium, an enterprise-grade UI library with 120+ [free](https://www.telerik.com/kendo-react-ui/components/free) and premium components for building polished, performant apps. The built-in context menu of the Grid allows you to control the rows selection and the column sorting. To enable it you should:

1.   Set the [`contextMenu`](https://www.telerik.com/kendo-react-ui/components/grid/api/gridprops#contextmenu) prop of the Grid to `true` to enable the ContextMenu component.

2.   Enable the [`selectable`](https://www.telerik.com/kendo-react-ui/components/grid/api/gridprops#selectable) prop allow rows selection.

3.   (Optional) Set the [`sortable`](https://www.telerik.com/kendo-react-ui/components/grid/api/gridprops) prop to true to allow column sorting, and take care of the built-in `sort` state.

4.   (Optional) Add a column with `columnType="checkbox"` to render a checkbox selection column.

jsx

```jsx
<Grid
    contextMenu={true}
    sortable={true}
    selectable={{ enabled: true }}
>
    <Column filterable={false} columnType="checkbox" width={50} />
```

--------------------------------------------------------------------------------------------------------------

The KendoReact Data Grid Context menu provides options for exporting its data to Excel.

To enable the Excel export:

1.   Install `kendo-react-excel-export` package.

sh  
```sh
npm i @progress/kendo-react-excel-export
``` 
2.   Import the ExcelExport component in your React Application.

jsx  
```jsx
import { ExcelExport } from '@progress/kendo-react-excel-export';
``` 
3.   Wrap the Grid in the ExcelExport component and use the ExcelExport `save` function to export the Grid and save it to excel file.
