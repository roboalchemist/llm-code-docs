### [Key Features](https://www.telerik.com/kendo-react-ui/components/dropdowns#key-features)

The KendoReact Dropdowns provide the following key features. Note that this is not an exhaustive list.

*   Highly customizable and easy to style **appearance**
    *   Built-in appearance options to adjust the form and dimensions of your dropdowns
    *   [Built-in themes](https://www.telerik.com/kendo-react-ui/components/styling)
    *   [Theme customization](https://www.telerik.com/kendo-react-ui/components/styling/customizing)
    *   [Self-created themes](https://www.telerik.com/kendo-react-ui/components/styling/theme-builder)
    *   [Unstyled mode](https://www.telerik.com/kendo-react-ui/components/styling/unstyled)

*   Built-in **disabling** of items
*   Option to enable adaptive mode for the component via the `adaptive` prop
*   Displaying grouped data
*   Built-in form support
*   [Globalization](https://www.telerik.com/kendo-react-ui/components/dropdowns/globalization)
*   Out-of-the-box **accessibility** and **keyboard navigation**

### [React Dropdowns with Other KendoReact Components](https://www.telerik.com/kendo-react-ui/components/dropdowns#react-dropdowns-with-other-kendoreact-components)

You can combine multiple KendoReact components with the React dropdowns in the same application:

*   Add dropdowns to your [Grid](https://www.telerik.com/kendo-react-ui/components/grid) custom cells.
*   Use dropdowns to submit data with [Forms](https://www.telerik.com/kendo-react-ui/components/form).

> The Dropdowns package is a dependency for multiple other KendoReact components.

### [Are the KendoReact Dropdowns Free to Use](https://www.telerik.com/kendo-react-ui/components/dropdowns#are-the-kendoreact-dropdowns-free-to-use)

The KendoReact Dropdowns provide both [free React dropdowns](https://www.telerik.com/kendo-react-ui/components/dropdowns#free-react-dropdowns) and [premium dropdowns](https://www.telerik.com/kendo-react-ui/components/dropdowns#premium-react-dropdowns) which require a commercial license key or an active trial license key.

You can start using the free components without any license or registration with Progress or Telerik.

### [How to Create Dropdowns in React](https://www.telerik.com/kendo-react-ui/components/dropdowns#how-to-create-dropdowns-in-react)

To create a dropdown in React, you need to install the KendoReact dropdowns package and then import the respective dropdown components in your application. After that, you can use the various available dropdown components in your React application. For more information, refer to the [Getting Started with the KendoReact Dropdowns](https://www.telerik.com/kendo-react-ui/components/dropdowns/get-started) article.

### [How to Upgrade to the Premium Dropdowns](https://www.telerik.com/kendo-react-ui/components/dropdowns#how-to-upgrade-to-the-premium-dropdowns)

If you want to use any of the premium [React dropdowns](https://www.telerik.com/kendo-react-ui/components/dropdowns#premium-react-dropdowns), UI components, or component features, you need a valid commercial license or an active trial license. A [free trial](https://www.telerik.com/try/kendo-react-ui) is available, if you want to try out the premium features before purchase.
