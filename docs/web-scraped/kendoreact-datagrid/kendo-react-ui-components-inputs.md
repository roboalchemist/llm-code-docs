The KendoReact Inputs provide the following key features. Note that this is not an exhaustive list.

*   Easy form integration
*   Out-of-the-box accessibility and keyboard navigation
*   Built-in control of the component's state using the `value` property and the `onChange` event
*   Highly customizable and easy to style **appearance**
    *   [Built-in themes](https://www.telerik.com/kendo-react-ui/components/styling)
    *   [Theme customization](https://www.telerik.com/kendo-react-ui/components/styling/customizing)
    *   [Self-created themes](https://www.telerik.com/kendo-react-ui/components/styling/theme-builder)
    *   [Unstyled mode](https://www.telerik.com/kendo-react-ui/components/styling/unstyled)

*   [Globalization](https://www.telerik.com/kendo-react-ui/components/labels/globalization)

The KendoReact Inputs are built natively for React, with no dependencies. They leverage key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

### [Are the KendoReact Inputs Free to Use](https://www.telerik.com/kendo-react-ui/components/inputs#are-the-kendoreact-inputs-free-to-use)

The KendoReact Inputs provide both [free React inputs](https://www.telerik.com/kendo-react-ui/components/inputs#free-react-inputs) and [premium inputs](https://www.telerik.com/kendo-react-ui/components/inputs#premium-react-inputs) which require a commercial license key or an active trial license key.

You can start using the free components without any license or registration with Progress or Telerik.

### [How to Create Inputs in React](https://www.telerik.com/kendo-react-ui/components/inputs#how-to-create-inputs-in-react)

To create an input in React, you need to install the KendoReact Inputs package and then import the respective input components in your application. After that, you can use the various available input components in your React application. For more information, refer to the [Getting Started with the KendoReact Inputs](https://www.telerik.com/kendo-react-ui/components/inputs/get-started) article.

### [How to Upgrade to the Premium Inputs](https://www.telerik.com/kendo-react-ui/components/inputs#how-to-upgrade-to-the-premium-inputs)

If you want to use any of the premium [React inputs](https://www.telerik.com/kendo-react-ui/components/inputs#premium-react-inputs), UI components, or component features, you need a valid commercial license or an active trial license. A [free trial](https://www.telerik.com/try/kendo-react-ui) is available, if you want to try out the premium features before purchase.
