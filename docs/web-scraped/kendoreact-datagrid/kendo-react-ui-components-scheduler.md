React Scheduler is a powerful component designed for scheduling and managing events, appointments, and tasks within React applications. It provides out-of-the-box support for timezone conversion, resources visualization, grouping, editing, and recurring events.

[Key Features of React Scheduler](https://www.telerik.com/kendo-react-ui/components/scheduler#key-features-of-react-scheduler)
------------------------------------------------------------------------------------------------------------------------------

The React Scheduler component delivers a range of handy developer-friendly features whose number and further development are not limited by the list in this section. The KendoReact team constantly invests efforts to improve the performance, add more value to the existing Scheduler library, and develop new features.

[Resources](https://www.telerik.com/kendo-react-ui/components/scheduler#resources)
----------------------------------------------------------------------------------

The Scheduler allows the developer to visualize an event by coloring it depending on a field value. [Read more about the Scheduler resources...](https://www.telerik.com/kendo-react-ui/components/scheduler/resources)

[Grouping](https://www.telerik.com/kendo-react-ui/components/scheduler#grouping)
--------------------------------------------------------------------------------

You can group the events in the Scheduler based on their resources and field values. [Read more about grouping...](https://www.telerik.com/kendo-react-ui/components/scheduler/grouping)

[Editing in React Scheduler](https://www.telerik.com/kendo-react-ui/components/scheduler#editing-in-react-scheduler)
--------------------------------------------------------------------------------------------------------------------

The React Scheduler component enables the user to edit the events by dragging them to different slots, changing their duration through resizing, and updating any of their fields with the advanced edit form. [Read more about editing...](https://www.telerik.com/kendo-react-ui/components/scheduler/editing)

[Recurring Events](https://www.telerik.com/kendo-react-ui/components/scheduler#recurring-events)
------------------------------------------------------------------------------------------------

The React Scheduler component enables you to display occurrences of recurring events. [Read more about how to display recurring events...](https://www.telerik.com/kendo-react-ui/components/scheduler/recurring)

[Timezones Conversion](https://www.telerik.com/kendo-react-ui/components/scheduler#timezones-conversion)
--------------------------------------------------------------------------------------------------------

The Scheduler can use the timezone of the client machine or a predefined timezone and also convert all events to the desired timezone. [Read more about timezone conversion...](https://www.telerik.com/kendo-react-ui/components/scheduler/timezones)

[Adaptive Slot Height](https://www.telerik.com/kendo-react-ui/components/scheduler#adaptive-slot-height)
--------------------------------------------------------------------------------------------------------

The React Scheduler enables you to dynamically change the height of the slots in order to fit a predefined number of items. [Read more about adaptive slot height...](https://www.telerik.com/kendo-react-ui/components/scheduler/adaptive-slot-height)

[Auto Item Height](https://www.telerik.com/kendo-react-ui/components/scheduler#auto-item-height)
------------------------------------------------------------------------------------------------

You can configure the SchedulerViewItem to get automatically resized depending on the size of its content. Alternatively, you can set a constant `height` value. [Read more about auto item height...](https://www.telerik.com/kendo-react-ui/components/scheduler/auto-item-height)

[Globalization](https://www.telerik.com/kendo-react-ui/components/scheduler#globalization)
------------------------------------------------------------------------------------------

By using the available globalization options in KendoReact, you can translate the Scheduler messages by adapting them to specific culture locales. [Read more about auto globalization..](https://www.telerik.com/kendo-react-ui/components/scheduler/globalization)
