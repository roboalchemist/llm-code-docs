Features
--------

### [Simple Installation](https://spiderd.io/docs/install.html)

Hassle-free installation with just one shell command, up and running in no time.

`# bash install.sh`
### Lightweight

Single self-contained, standalone executable program (~= 30MB) without external dependencies, super easy to install and upgrade.

### Performant

Experience lightning-fast and efficient performance with Spider. Designed to deliver exceptional speed and responsiveness, ensuring a seamless user experience.

### Efficient & User-friendly

With its efficient design, Spider ensures optimal performance while providing a seamless user experience.

### [Secure SMTP and HTTP](https://spiderd.io/docs/settings-ssl-cert.html)

Request free Let's Encrypt SSL cert with one click. Renew automatically.

### [S3 Support](https://spiderd.io/docs/settings-storage.html#s3)

Save archived email messages and attachments on Amazon S3 compatible object storage.

### [Custom Branding](https://spiderd.io/docs/settings-custom-branding.html)

Use your own company logo image, favico icon, product name, and website link.

### Blazing Fast Full-Text Search

Blazing fast searching from anywhere using a web browser.

### Two-Factor Auth (2FA)

Login using One-Time-Password (OTP) based authentication.

### Monitor and Audit

Ensure employees are collaborating effectively.

### One-Click Restore

End users can restore the archived emails to mailbox with one click.

### [Daily Archiving Report](https://spiderd.io/docs/settings-notifications.html)

Administrators can get a brief daily archiving report.

### Lower Storage Costs

The email messages and attachments are de-duplicated, encrypted and compressed to reduce disk space.

### Enhance Email Server Indirectly

Safe to remove old emails from the email server, free the disk space and speed up the performance of IMAP service.
