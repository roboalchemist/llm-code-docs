TensorFlow 2 focuses on simplicity and ease of use, with updates like eager execution, intuitive higher-level APIs, and flexible model building on any platform.

Many guides are written as Jupyter notebooks and run directly in Google Colab—a hosted notebook environment that requires no setup. Click the _Run in Google Colab_ button.

Essential documentation
-----------------------

[### Install Tensor Flow](https://www.tensorflow.org/install) Install the package or build from source. GPU support for CUDA®-enabled cards. 

[### Migrate to Tensor Flow 2](https://www.tensorflow.org/guide/migrate) Learn how to migrate your TF1.x code to TF2. 

[### Keras](https://www.tensorflow.org/guide/keras/overview) Keras is a high-level API that's easier for ML beginners, as well as researchers. 

[### Tensor Flow basics](https://www.tensorflow.org/guide/eager) Learn about the fundamental classes and features that make TensorFlow work. 

[### Data input pipelines](https://www.tensorflow.org/guide/data) The `tf.data` API enables you to build complex input pipelines from simple, reusable pieces. 

[### Tensor Flow 2 best practices](https://www.tensorflow.org/guide/effective_tf2) Learn about the best practices for effective development using TensorFlow 2. 

[### Save a model](https://www.tensorflow.org/guide/checkpoint) Save a TensorFlow model using checkpoints or the SavedModel format. 

[### Accelerators](https://www.tensorflow.org/guide/distributed_training) Distribute training across multiple GPUs, multiple machines or TPUs. 

[### Performance](https://www.tensorflow.org/guide/function) Best practices and optimization techniques for optimal TensorFlow performance. 

Libraries and extensions
------------------------

 Explore [additional resources](https://www.tensorflow.org/resources) to build advanced models or methods using TensorFlow, and access domain-specific application packages that extend TensorFlow. 

*   [chevron_right](https://www.tensorflow.org/decision_forests)[### Tensor Flow Decision Forests](https://www.tensorflow.org/decision_forests) A library to train, run and interpret decision forest models (e.g., Random Forests, Gradient Boosted Trees) in TensorFlow.    
*   [chevron_right](https://www.tensorflow.org/hub)[### Tensor Flow Hub](https://www.tensorflow.org/hub) A library for the publication, discovery, and consumption of reusable parts of machine learning models.    
*   [chevron_right](https://www.tensorflow.org/tfx/guide/serving)[### Serving](https://www.tensorflow.org/tfx/guide/serving) A TFX serving system for ML models, designed for high-performance in production environments.    
*   [chevron_right](https://www.tensorflow.org/federated)[### Tensor Flow Federated](https://www.tensorflow.org/federated) A framework for machine learning and other computations on decentralized data.    
*   [chevron_right](https://www.tensorflow.org/neural_structured_learning)[### Neural Structured Learning](https://www.tensorflow.org/neural_structured_learning) A learning paradigm to train neural networks by leveraging structured signals in addition to feature inputs.    
*   [chevron_right](https://www.tensorflow.org/graphics)[### Tensor Flow Graphics](https://www.tensorflow.org/graphics) A library of computer graphics functionalities ranging from cameras, lights, and materials to renderers.    
*   [chevron_right](https://github.com/tensorflow/addons)[### SIG Addons](https://github.com/tensorflow/addons) Extra functionality for TensorFlow, maintained by SIG Addons.    

*   [chevron_right](https://www.tensorflow.org/tensorboard)[### Tensor Board](https://www.tensorflow.org/tensorboard) A suite of visualization tools to understand, debug, and optimize TensorFlow programs.    
*   [chevron_right](https://www.tensorflow.org/datasets)[### Datasets](https://www.tensorflow.org/datasets) A collection of datasets ready to use with TensorFlow.    
*   [chevron_right](https://www.tensorflow.org/model_optimization)[### Model Optimization](https://www.tensorflow.org/model_optimization) The TensorFlow Model Optimization Toolkit is a suite of tools for optimizing ML models for deployment and execution.    
*   [chevron_right](https://www.tensorflow.org/probability)[### Probability](https://www.tensorflow.org/probability) TensorFlow Probability is a library for probabilistic reasoning and statistical analysis.    
*   [chevron_right](https://www.tensorflow.org/mlir)[### MLIR](https://www.tensorflow.org/mlir) MLIR unifies the infrastructure for high-performance ML models in TensorFlow.    
*   [chevron_right](https://www.tensorflow.org/xla)[### XLA](https://www.tensorflow.org/xla) A domain-specific compiler for linear algebra that accelerates TensorFlow models with potentially no source code changes.    
*   [chevron_right](https://github.com/tensorflow/io)[### SIG IO](https://github.com/tensorflow/io) Dataset, streaming, and file system extensions, maintained by SIG IO.
