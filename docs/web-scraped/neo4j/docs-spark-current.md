Neo4j Connector for Apache Spark
================================

The Neo4j Connector for Apache Spark provides integration between Neo4j and Apache Spark.

You can use the connector to process and transfer data between Neo4j and other platforms such as [Databricks](https://neo4j.com/docs/spark/current/databricks/) and several [data warehouses](https://neo4j.com/docs/spark/current/dwh/). Based on the Spark DataSource API, the connector supports all the programming languages that Spark supports.

[](https://neo4j.com/docs/spark/current/#_graphs_and_dataframes)Graphs and DataFrames
-------------------------------------------------------------------------------------

The connector uses _schema inference_ to convert Neo4j graphs into Spark table-based DataFrames. For example, consider a graph with the following schema:

![Image 3: Example graph](https://neo4j.com/docs/spark/current/_images/example-graph.svg)

The connector creates a DataFrame with `:Customer` and `:Product` nodes connected by the `BOUGHT` relationship, along with any node or relationship properties. The [Schema inference](https://neo4j.com/docs/spark/current/read/schema/) section shows a more detailed example of this process, while the [Data type mapping](https://neo4j.com/docs/spark/current/types/) section shows how data types are mapped between Neo4j and Spark.

The connector supports writing DataFrames to Neo4j as well, and custom Cypher® queries both for [reading](https://neo4j.com/docs/spark/current/read/query/) and for [writing](https://neo4j.com/docs/spark/current/write/query/) data.

[](https://neo4j.com/docs/spark/current/#_compatibility)Compatibility
---------------------------------------------------------------------

### [](https://neo4j.com/docs/spark/current/#_neo4j_compatibility)Neo4j compatibility

The connector supports Neo4j 2025.x, 5.x, and 4.4, whether run as a managed service in [Neo4j Aura](https://neo4j.com/docs/aura/), as a single instance, or as a cluster. It supports both the Community and the Enterprise Edition.

### [](https://neo4j.com/docs/spark/current/#_spark_and_scala_compatibility)Spark and Scala compatibility

The connector currently supports Spark versions 3.3, 3.4 and 3.5 as well as Databricks 12.2, 13.3, 14.3 with Scala 2.12 and Scala 2.13.
