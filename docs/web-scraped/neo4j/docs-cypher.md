Cypher
======

[](https://neo4j.com/docs/cypher/#_neo4js_declarative_query_language)Neo4j’s declarative query language
-------------------------------------------------------------------------------------------------------

Learn how to write queries to your database, have easy and quick access to commands through the Cheat Sheet, and extend your application with the use of the APOC library.

[### Cypher Docs ![Image 2: icon developer](https://neo4j.com/docs/_images/icon-developer.svg) Read an instructive guide on Neo4j’s declarative query language and learn how to use it within Neo4j tools and deployments. [https://neo4j.com/docs/cypher-manual/](https://neo4j.com/docs/cypher-manual/)](https://neo4j.com/docs/cypher-manual/)

[### APOC ![Image 3: icon library](https://neo4j.com/docs/_images/icon-library.svg) Discover how to use the APOC library through the APOC Core guide and see further options in APOC Extended. [https://neo4j.com/docs/apoc/](https://neo4j.com/docs/apoc/)](https://neo4j.com/docs/apoc/)

[### Cypher Cheat Sheet ![Image 4: icon cheatsheet](https://neo4j.com/docs/_images/icon-cheatsheet.svg) Find reference of all available Cypher® features. [https://neo4j.com/docs/cypher-cheat-sheet/](https://neo4j.com/docs/cypher-cheat-sheet/)](https://neo4j.com/docs/cypher-cheat-sheet/)

[](https://neo4j.com/docs/cypher/#_keep_exploring)Keep exploring
----------------------------------------------------------------

Here are some recommended resources to get started with Cypher:

1.   [**Getting Started with Cypher**](https://neo4j.com/docs/getting-started/cypher-intro/)

Learn how to write Cypher statements by applying them to a graph example. 
2.   [**Graph Academy: Cypher Fundamentals**](https://graphacademy.neo4j.com/courses/cypher-fundamentals/)

Learn Cypher in a 1-hour course. 
3.   [**Cypher queries**](https://neo4j.com/docs/cypher-manual/current/queries/)

Continue your exploration of Cypher queries in the Cypher Manual.
