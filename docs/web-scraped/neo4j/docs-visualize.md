Data visualization
==================

[](https://neo4j.com/docs/visualize/#_use_tools_to_explore_and_create_visualizations_of_your_data)Use tools to explore and create visualizations of your data
-------------------------------------------------------------------------------------------------------------------------------------------------------------

Learn how to create visualizations and get new perspectives over your data using tools designed by Neo4j and connectors to popular data visualization platforms.

### Neo4j Bloom/Explore

![Image 2: icon bloom](https://neo4j.com/docs/_images/icon-bloom.svg)

Visualize and explore graph data using natural language search and an intuitive and graphical interface.

[Neo4j Bloom](https://neo4j.com/docs/bloom-user-guide/)

[Explore (Aura)](https://neo4j.com/docs/aura/preview/explore/introduction/)

### Neo4j Browser/Query

![Image 3: icon developer](https://neo4j.com/docs/_images/icon-developer.svg)

Write and execute Cypher® queries and visualize the results in nodes and relationships.

[Neo4j Browser](https://neo4j.com/docs/browser/)

[Query (Aura)](https://neo4j.com/docs/aura/preview/query/introduction/)

[### Tableau ![Image 4: icon tableau](https://neo4j.com/docs/_images/icon-tableau.svg) Use Neo4j Connector for BI to make a connection between Neo4j and Tableau, and visualize data directly. [BI Connector](https://neo4j.com/bi-connector/)](https://neo4j.com/bi-connector/)

[### Neo4j Visualization Library ![Image 5: icon library](https://neo4j.com/docs/_images/icon-library.svg) Explore a collection of libraries that can be used to build custom graph visualizations. [NVL](https://neo4j.com/docs/nvl/current/)](https://neo4j.com/docs/nvl/current/)

[](https://neo4j.com/docs/visualize/#_keep_exploring)Keep exploring
-------------------------------------------------------------------

Here are some recommended resources to get started with data visualization:

1.   [**15 Tools for Visualizing Your Neo4j Graph Database**](https://neo4j.com/developer-blog/15-tools-for-visualizing-your-neo4j-graph-database/)

Read an overview of the graph visualization landscape. 
2.   [**Showing Charts for Neo4j Query Results using amCharts and Structr**](https://medium.com/neo4j/showing-charts-for-neo4j-query-results-using-amcharts-and-structr-efae0b7a04f0)

Learn how to render a bar chart based upon the example movies graph. 
3.   [**Working With Neo4j Date And Spatial Types In A React.js App**](https://medium.com/neo4j/working-with-neo4j-date-and-spatial-types-in-a-react-js-app-5475b5042b50)

Learn how to build a dashboard app with Neo4j, Mapbox, React, and Nivo Charts.
