Neo4j Connector for Kafka
=========================

The Neo4j Connector for Kafka streams data between Neo4j or Aura databases and platforms based on Apache Kafka® using the Kafka Connect framework.

The connector is distributed in two versions:

1.   The [Neo4j Connector for Confluent](https://neo4j.com/docs/kafka/installation/#confluent-dist) is available for Confluent Platform® and Confluent Cloud®.

2.   The [Neo4j Connector for Apache Kafka](https://neo4j.com/docs/kafka/installation/#kafka-dist) is available for platforms built on the open-source software (OSS) Apache Kafka®, including Amazon MSK®.

[](https://neo4j.com/docs/kafka/#_components)Components
-------------------------------------------------------

### [](https://neo4j.com/docs/kafka/#_sink)Sink

The sink component consumes messages from Apache Kafka topics and applies configured changes into a Neo4j or Aura database.

### [](https://neo4j.com/docs/kafka/#_source)Source

The source component listens for changes occurring in a Neo4j or Aura database and publishes messages into Apache Kafka topics. It can be configured to read changes using either the [Change Data Capture](https://neo4j.com/docs/cdc) feature or a provided custom query.

Change Data Capture is a new feature introduced with Neo4j 5.13.0 and Aura 5, and the Source connector with CDC support requires at least these versions of Neo4j and Aura.

[](https://neo4j.com/docs/kafka/#_compatibility)Compatibility
-------------------------------------------------------------

### [](https://neo4j.com/docs/kafka/#_neo4j_compatibility)Neo4j compatibility

The connector supports Neo4j 2025.x and 5.x, whether run as a managed service in [Neo4j Aura](https://neo4j.com/docs/aura/), as a single instance, or as a cluster. It supports both the Community and the Enterprise Edition.

Some features, such as constraints and CDC, are only available with Neo4j Enterprise Edition or AuraDB Enterprise, and will not be available in Neo4j Community Edition, or Free and Professional Tiers of AuraDB.

### [](https://neo4j.com/docs/kafka/#_kafka_connect_compatibility)Kafka Connect compatibility

The connector is designed to be compatible with versions of Apache Kafka Connect 3.4.1 and later. This includes Amazon MSK and Confluent Connect 7.4 and later on the Confluent Platform and Cloud.

### [](https://neo4j.com/docs/kafka/#_java_compatibility)Java compatibility

The connector is built using Java 11, and is compatible to run with Java 11 and LTS versions of Java, i.e. 17 and 21.

[](https://neo4j.com/docs/kafka/#_support)Support
-------------------------------------------------
