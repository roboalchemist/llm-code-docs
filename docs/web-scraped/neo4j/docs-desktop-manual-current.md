Neo4j Desktop
=============

Neo4j Desktop 2.x is a client application to help you work with Neo4j locally, on- or offline. It allows you to create and manage any number of local databases as your machine can support. The databases are hosted by a local database management system (DBMS) instance, and you can have multiple instances and run in multiple windows simultaneously. You can also create a remote connection to an instance that is running on an external server or in [Neo4j Aura](https://neo4j.com/docs/aura).

A Developer edition license of the Neo4j graph database is included with Neo4j Desktop. The Developer edition offers all the capabilities and features of Neo4j Enterprise Edition, for development use by an individual person on a single machine. See [Operations Manual → Neo4j feature details](https://neo4j.com/docs/operations-manual/current/introduction/#edition-details) for more information on the Neo4j Enterprise Edition.

Desktop 2.x can run Neo4j DBMS versions from 5.1.0 to latest calendar version, meaning that you can create one or more DBMS instances to maintain different Neo4j versions, separate your databases by the type of data they contain, or to achieve a particular desired configuration of the DBMS. 2.xalso provides built-in tools that let you back up your databases locally, upgrade instances, and install plugins for added functionality from the Desktop UI. You can import and model data from CSV files with the **Import** service. **Query** is a tool to run Cypher® queries against your graph data and view results. **Explore**, powered by Neo4j Bloom, is used to visualize graph data using search inputs, and can be used with our without a plugin.

Desktop 2.x is the evolution of the original Neo4j Desktop. It offers a new design and more unified capabilities, and as such, mirrors the [Aura](https://neo4j.com/docs/aura) console, the fully-managed graph platform as a cloud service. See [Version comparison](https://neo4j.com/docs/desktop-manual/current/comparison/) for a full breakdown of the differences between the two versions. Desktop 1.5.9 and 1.6.x are still available for the time being, but will be deprecated together with Neo4j 4.1.0.
