# Getting Started with Tailscale

**Source:** https://tailscale.com/kb/1016/start-using

---

Install Tailscale on macOSThe current version of the Tailscale client requires macOS 12.0 (Monterey) or later.
Download and install the Tailscale client using one of the following options:
Standalone variant from Tailscale's package server (recommended).
Mac App Store variant. The Tailscale client is free to download, but downloading any app from the Mac App Store may require an Apple ID with a valid credit card attached.
Our open source variant from our GitHub repository.
Follow the Tailscale onboarding flow, which will guide you to install the Tailscale VPN configuration.
Sign up with your SSO identity provider.
For more information about the different variant options, refer to Three ways to run Tailscale on macOS.
MDM deployments
You can use configuration profiles and user defaults to personalize your setup of Tailscale for macOS programmatically. This can help you deploy Tailscale using solutions like MDM.
Troubleshooting
For common macOS issues and suggestions to help resolve them, refer to the Troubleshooting guide.
