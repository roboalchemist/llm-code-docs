Older versions of ArcGIS Pro help are eventually archived. When that happens, the content is still available online, but the version switch is removed from topics and a note is added at the top to indicate the content is no longer updated.

Below are links to all versions of ArcGIS Pro help that were available when ArcGIS Pro 3.6 was released so you can find the help you need, especially if you are using an older version of the software.

##### Tip:

The URLs follow a pattern that includes the version number (for example, /2.7/). You can switch between versions in your browser by changing the version number in the URL and pressing the Enter key. (The most recent version of the help always redirects from the version number to [/latest/](https://pro.arcgis.com/en/pro-app/latest/get-started/archived-arcgis-pro-help.htm) so users who want the latest content can access it without referencing version numbers.)

Online help for ArcGIS Pro 2.5 and earlier versions is no longer available. [Download and install the offline help](https://pro.arcgis.com/en/pro-app/3.6/get-started/download-arcgis-pro.htm#GUID-198CB511-1ED6-4238-87D1-2D9375A09972) to refer to those versions.
