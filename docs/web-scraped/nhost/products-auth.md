Overview
========

authentication auth login sign in sign up user management identity

Nhost Auth is a ready-to-use authentication service seamlessly integrated with the [GraphQL API](https://docs.nhost.io/products/graphql) and its [Permission System](https://docs.nhost.io/products/graphql/permissions) from Hasura. This allows you to easily add user authentication to your application without having to build and maintain your own authentication system.
