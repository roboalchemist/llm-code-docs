flyctl - The Fly.io CLI
=======================

flyctl, the Fly.io CLI, is one of the primary ways to interact with the Fly.io platform. You’ll use the `fly` command to create and deploy apps, manage Machines and volumes, configure networking, and more.

First, [install flyctl](https://fly.io/docs/flyctl/install/).

Also check out [flyctl and continuous integration](https://fly.io/docs/flyctl/integrating/) which covers environment variables, JSON and other automation related information about flyctl.

The following list includes some important commands to get started.

----------------------------------------------------------------------------------

*   Signing Up: [fly auth signup](https://fly.io/docs/flyctl/auth-signup/)
*   Logging In: [fly auth login](https://fly.io/docs/flyctl/auth-login/)

------------------------------------------------------------------
