Fly Launch helps you configure and orchestrate your app. If you followed one of our [Getting started](https://fly.io/docs/getting-started/) guides, then youâ€™re already using Fly Launch.

![Image 1](https://fly.io/static/images/docs-island.webp)
Fly Launch includes:

*   the `fly launch` command to get started with a new app 
*   the `fly.toml` file for app configuration 
*   the `fly deploy` command to deploy or redeploy your app 
*   the `fly scale` command for quick horizontal and vertical scaling
