Networking
==========

Networking on Fly.io.

*   **[Connect to an App Service](https://fly.io/docs/networking/app-services/):** An overview of how to connect to your app over the private WireGuard network (6PN), and how to make your app reachable from the internet.

*   **[Public networking](https://fly.io/docs/networking/services):** Details about public network services on Fly.io, including allocating IP addresses, finding a Machine’s outbound IP, connection handlers, and redirects.

*   **[Private networking](https://fly.io/docs/networking/private-networking):** Learn about Fly.io’s IPv6 private network (6PN) and DNS on Fly Machines.

*   **[Custom Private Networks](https://fly.io/docs/networking/custom-private-networks):** Isolate users, data, and code on custom private networks.

*   **[Flycast - Private Fly Proxy Services](https://fly.io/docs/networking/flycast):** Route requests to private apps through Fly Proxy to take advantage of features like load balancing and autostop/autostart based on traffic.

*   **[Egress IP Addresses](https://fly.io/docs/networking/egress-ips/):** How to get stable outbound IPs from your Fly apps using machine-scoped assignments or a shared proxy setup.

*   **[Dynamic request routing](https://fly.io/docs/networking/dynamic-request-routing/):** Use Fly.io request and response headers to customize request routing to regions, apps, and even specific Machines.

*   **[Custom domains](https://fly.io/docs/networking/custom-domain/):** Add a custom domain for your app and troubleshoot certificate creation.

*   **[Understanding Cloudflare](https://fly.io/docs/networking/understanding-cloudflare/):** How to use Cloudflare on Fly.io.

*   **[HTTP request headers](https://fly.io/docs/networking/request-headers/):** Fly.io-specific and standard HTTP headers added by the HTTP connection handler.

*   **[Run UDP services](https://fly.io/docs/networking/udp-and-tcp/):** How to set up apps that use UDP.

*   **[TLS support](https://fly.io/docs/networking/tls/):** Supported TLS cipher suites.
