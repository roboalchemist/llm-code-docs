### Downloading CanvasJS

There are two Source files that come in the downloaded Zip folder

    1.   Uncompressed file – canvasjs.js (available only in the Paid Version)
    2.   Compressed file – canvasjs.min.js

Download CanvasJS library from [here](https://canvasjs.com/download-html5-charting-graphing-library/). Extract the downloaded file and put the content in the same folder as the html file.

### Adding CanvasJS to Your Web Pages

Include the file inside the <head> tag of your web page.

 You can Include Any of the above two Javascript files. While its ok to use uncompressed version on the development machine, its recommended to use compressed version in the production environment.

1. <head>
2. <script type="text/javascript" src="canvasjs.min.js"></script>
3. </head>

### Testing Installation

Create an HTML file in the same directory as the library and copy the following code into it.

1. <!DOCTYPE HTML>
2. <html>
3. <head>
4. <script type="text/javascript" src="canvasjs.min.js"></script>
5. <script type="text/javascript">
6. window.onload = function () {
7. var chart = new CanvasJS.Chart("chartContainer", {
8. data: [
9. {
10. type: "column",
11. dataPoints: [
12. { x: 10, y: 10 },
13. { x: 20, y: 15 },
14. { x: 30, y: 25 },
15. { x: 40, y: 30 },
16. { x: 50, y: 28 }
17. ]
18. }
19. ]
20. });

21. chart.render();
22. }
23. </script>
24. </head>

25. <body>
26. <div id="chartContainer" style="height: 500px; width: 50%;"></div>
27. </body>
28. </html>

Save the file, and Open it in a Browser. A Column Chart Should Render on the browser.

### 1. Install CanvasJS Charts

Install [CanvasJS Charts](https://www.npmjs.com/package/@canvasjs/charts) Package from NPM package manager.

1. npm install @canvasjs/charts

### 2. Import Chart

As CanvasJS Charts package is installed, it can be imported using different module formats like AMD, CommonJS, etc.

1. import CanvasJS from '@canvasjs/charts';

### 3. Define container for the chart

Define a div (chart-container) where chart has to be rendered.

1. <div id="chartContainer"></div>

### 4. Create & Render the chart

Create chart by passing the container-id of the div that you created in the previous step and chart-options. Once the chart is created, render the chart by calling chart.render();

1. //Create Chart
2. var chart = new CanvasJS.Chart("container", {
3. //Chart Options - Check https://canvasjs.com/docs/charts/chart-options/
4. title:{
5. text: "Basic Column Chart in JavaScript"
6. },
7. data: [{
8. type: "column",
9. dataPoints: [
10. { label: "apple", y: 10 },
11. { label: "orange", y: 15 },
12. { label: "banana", y: 25 },
13. { label: "mango", y: 30 },
14. { label: "grape", y: 28 }
15. ]
16. }]
17. });
18. //Render Chart
19. chart.render();
