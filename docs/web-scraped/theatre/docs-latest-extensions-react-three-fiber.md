React Three Fiber
=================

*   [Theatre.js Docs](https://www.theatrejs.com/docs/latest)
*   [Extensions](https://www.theatrejs.com/docs/latest/extensions)
*   React Three Fiber

`import extension from '@theatre/r3f/dist/extension'`

The r3f extension helps you easily bind your r3f elements to Theatre.js.

For a tutorial on how to use the extension, see the [Getting started with React Three Fiber](https://www.theatrejs.com/docs/latest/getting-started/with-react-three-fiber) guide.

To learn about the API, check out the [@theatre/r3f API docs](https://www.theatrejs.com/docs/latest/api/r3f).
