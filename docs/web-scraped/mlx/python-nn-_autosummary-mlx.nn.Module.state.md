# mlx.nn.Module.state[\#](#mlx-nn-module-state "Link to this heading")

*[property][ ]*[[Module.]][[state]][\#](#mlx.nn.Module.state "Link to this definition")

:   The module's state dictionary

    The module's state dictionary contains any attribute set on the module including parameters in [[`Module.parameters()`]](mlx.nn.Module.parameters.html#mlx.nn.Module.parameters "mlx.nn.Module.parameters")

    Unlike [[`Module.parameters()`]](mlx.nn.Module.parameters.html#mlx.nn.Module.parameters "mlx.nn.Module.parameters"), the [[`Module.state`]](#mlx.nn.Module.state "mlx.nn.Module.state") property is a reference to the module's state. Updates to it will be reflected in the original module.
