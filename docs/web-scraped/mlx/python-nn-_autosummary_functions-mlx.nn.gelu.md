# mlx.nn.gelu[\#](#mlx-nn-gelu "Link to this heading")

*[class][ ]*[[gelu]][(]*[[x]]*[)][\#](#mlx.nn.gelu "Link to this definition")

:   Applies the Gaussian Error Linear Units function.

    ::: 
    \\\[\\textrm(x) = x \* \\Phi(x)\\\]
    :::

    where [\\(\\Phi(x)\\)] is the Gaussian CDF.

    See also [[`gelu_approx()`]](mlx.nn.gelu_approx.html#mlx.nn.gelu_approx "mlx.nn.gelu_approx") and [[`gelu_fast_approx()`]](mlx.nn.gelu_fast_approx.html#mlx.nn.gelu_fast_approx "mlx.nn.gelu_fast_approx") for faster approximations.
