# mlx.nn.selu[\#](#mlx-nn-selu "Link to this heading")

*[class][ ]*[[selu]][(]*[[x]]*[)][\#](#mlx.nn.selu "Link to this definition")

:   Applies the Scaled Exponential Linear Unit.

    ::: 
    \\\[\\begin\\text(x) = \\begin \\lambda x & \\text x \> 0 \\\\ \\lambda \\alpha (\\exp(x) - 1) & \\text x \\leq 0 \\end\\end\\\]
    :::

    where [\\(\\lambda = 1.0507\\)] and [\\(\\alpha = 1.67326\\)].

    See also [[`elu()`]](mlx.nn.elu.html#mlx.nn.elu "mlx.nn.elu").
