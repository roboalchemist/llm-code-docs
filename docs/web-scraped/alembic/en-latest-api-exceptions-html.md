Indicates the database is not at current head revisions.

Raised by the [`command.current()`](https://alembic.sqlalchemy.org/en/latest/api/commands.html#alembic.command.current "alembic.command.current") command when the [`command.current.check_heads`](https://alembic.sqlalchemy.org/en/latest/api/commands.html#alembic.command.current.params.check_heads "alembic.command.current") parameter is used.
