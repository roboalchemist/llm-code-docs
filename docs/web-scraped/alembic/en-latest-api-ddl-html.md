These are some of the constructs used to generate migration instructions. The APIs here build off of the `sqlalchemy.schema.DDLElement` and [Custom SQL Constructs and Compilation Extension](https://docs.sqlalchemy.org/en/21/core/compiler.html#sqlalchemy-ext-compiler-toplevel "(in SQLAlchemy v2.1)") systems.

For programmatic usage of Alembic’s migration directives, the easiest route is to use the higher level functions given by [Operation Directives](https://alembic.sqlalchemy.org/en/latest/api/operations.html#alembic-operations-toplevel).
