About Maven-Indexer
===================

Apache Maven Indexer (former Sonatype Nexus Indexer) is the defacto standard for producing indexes of Maven repositories. The Indexes are produced and consumed by all major tools in the ecosystem.

[](https://maven.apache.org/maven-indexer/index.html)
Project Modules
---------------

This project has declared the following modules:

| Name | Description |
| --- | --- |
| [Maven :: Indexer Core](https://maven.apache.org/maven-indexer/indexer-core/index.html) | Indexer Core contains the core support for indexing Maven repositories, searching them, publishing and consuming remotely published indexes. |
| [Maven :: Indexer CLI](https://maven.apache.org/maven-indexer/indexer-cli/index.html) | Indexer CLI produces a CLI tool for indexing and publishing Maven repositories. |
| [Maven :: Indexer Reader](https://maven.apache.org/maven-indexer/indexer-reader/index.html) | Indexer Reader is a dependency-less library that is able to read published (remote) index with incremental update support, making usable to integrate published Maven Indexes into any engine without depending on maven-indexer-core and its transitive dependencies. |
| [Maven :: Indexer Examples](https://maven.apache.org/maven-indexer/maven-indexer-examples/index.html) | This module contains Maven Indexer examples. |
| [Maven :: Search API](https://maven.apache.org/maven-indexer/search-api/index.html) | Indexer Search API. |
| [Maven :: Search API Indexer Backend](https://maven.apache.org/maven-indexer/search-backend-indexer/index.html) | Indexer Search Backend implemented by Indexer Core. |
| [Maven :: Search API SMO Backend](https://maven.apache.org/maven-indexer/search-backend-smo/index.html) | Indexer Search Backend implemented by SMO. |
| [Maven :: Search API Remote Repository Backend](https://maven.apache.org/maven-indexer/search-backend-remoterepository/index.html) | Indexer Search Backend implemented by remote repository (limited). |
