Markdown Content:
The following accesskeys are implemented throughout the site in an attempt to help users using screen-readers. If you find a page that does not conform to this, please let us know using the instructions on the [Feedback](https://validator.w3.org/feedback.html) page.

Accesskey Legend
----------------

1.   Home: access key "1" leads back to the service's home page.

2.   Skip Navigation: access key "2"...

    *   ... in results pages, leads you to the beginning of the actual results
    *   ... in documentation pages, leads you to the table of contents
    *   ... in other pages (home page, advanced interface pages), leads you to the first form

3.   Documentation: access key "3" leads to the documentation index for the service.

4.   Feedback: access key "4" leads to the feedback instructions.
