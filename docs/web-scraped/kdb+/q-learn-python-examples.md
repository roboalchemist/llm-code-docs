### [Whether an Armstrong number](https://www.geeksforgeeks.org/python-program-to-check-armstrong-number/)[¶](https://code.kx.com/q/learn/python/examples/#whether-an-armstrong-number "Permanent link")

```python
import numpy as np
def is_armstrong(x):
    s = 0
    t = x
    while t:
        s += (t % 10) ** len(str(x))
        t //= 10
    return s == x
```

```python
>>> [is_armstrong(x) for x in (153, 120, 1253, 1634)]
[True, False, False, True]
```

```q
isArmstrong:{x=sum{x xexp count x}10 vs x}
```

```q
q)isArmstrong each 153 120 1253 1634
1001b
```

The steps of `isArmstrong` explain themselves.

q)10 vs 153                 / decode base-10 integer

1 125 27f
q)sum 1 5 3 xexp 3
153f

### [Nth Fibonacci number](https://www.geeksforgeeks.org/python-program-for-n-th-fibonacci-number/)[¶](https://code.kx.com/q/learn/python/examples/#nth-fibonacci-number "Permanent link")

```python
# next Fibonacci pair
def nfp(x): return [x[1], sum(x)]

# Nth Fibonacci pair
def fibp(n):
    if n<2: return [0, 1]
    return nfp(fibp(n-1))

def fib(n): return fibp(n)[0]
```

```python
>>> fib(9)
21
```

```q
nfp:{(x 1),sum x}
fib:{first(x-1)nfp/0 1}
```

```q
q)fib 9
21
```

Above we see `nfp` applied with the [Do iterator `/`](https://code.kx.com/q/ref/accumulators/#do).

The Python solution recurses down from `n` to reach the initial state of `[0, 1]`, while the q solution iterates `n-1` times on an initial state of `0 1`.

### [Whether a Fibonacci number](https://www.geeksforgeeks.org/python-program-for-how-to-check-if-a-given-number-is-fibonacci-number/)[¶](https://code.kx.com/q/learn/python/examples/#whether-a-fibonacci-number "Permanent link")

```python
import math
def is_fibonacci(n):
    phi = 0.5 + 0.5 * math.sqrt(5.0)
    a = phi * n
    return n == 0 or abs(round(a) - a) < 1.0 / n
```

```python
>>> [is_fibonacci(x) for x in (8, 34, 41)]
[True, True, False]
```

x is a Fibonacci number if either of 5 x 2±4 is a perfect square.

```q
is_ps:{x={x*x}"j"$sqrt x}                       / is perfect square?
is_fibonacci:{.[or]is_ps flip 4 -4+/:5*x*x}
```

```q
q)is_fibonacci 8 34 41
110b
```

The iteration implicit in q’s operators means that `is_fibonacci` also iterates implicitly.
