Installation [#](https://windicss.org/guide/installation#installation)
======================================================================

Windi CSS provides **first-class integrations** for your favorite tools. Select yours and get started.

Build Tools [#](https://windicss.org/guide/installation#build-tools)
--------------------------------------------------------------------

Support for build-tools are framework-agnostic. They work for most frameworks without specific configurations.

[Vite](https://windicss.org/integrations/vite)[Webpack](https://windicss.org/integrations/webpack)[Rollup](https://windicss.org/integrations/rollup)[PostCSS](https://windicss.org/integrations/postcss)[![Image 3](https://windicss.org/assets/logo.svg) CLI](https://windicss.org/integrations/cli)

Frameworks [#](https://windicss.org/guide/installation#frameworks)
------------------------------------------------------------------

In addition to general build-tools support, we also provide integrations for the following frameworks that offer out-of-the-box experience.

[Nuxt](https://windicss.org/integrations/nuxt)[Vue CLI](https://windicss.org/integrations/vue-cli)[Gridsome](https://windicss.org/integrations/gridsome)[Svelte](https://windicss.org/integrations/svelte)
