Title: Block IPs by Reputation

URL Source: https://wafris.org/docs/rule_setting/block_ips_by_reputation/

Markdown Content:
The Block IPs by Reputation feature allows you to block IP addresses based on their reputation score. This score is determined by the IP’s history of malicious activity, such as spamming, hacking attempts, or other nefarious behavior. By blocking IPs with a poor reputation, you can significantly reduce the risk of your site being attacked or compromised.

[](https://wafris.org/docs/rule_setting/block_ips_by_reputation/#how-to-block-ips-by-reputation) How to Block IPs by Reputation
-------------------------------------------------------------------------------------------------------------------------------

1. Navigate to the Block IPs by Reputation section in your dashboard.
2. Enable the Feature

[](https://wafris.org/docs/rule_setting/block_ips_by_reputation/#best-practices) Best Practices
-----------------------------------------------------------------------------------------------

* Combine this feature with other security measures, such as [Block IPs](https://wafris.org/rule_setting/block_ips), [Block CIDR Ranges](https://wafris.org/rule_setting/block_cidrs), and [Block Countries](https://wafris.org/rule_setting/block_countries), to create a robust security strategy.
* If an IP address is blocked that should not be, you can unblock it permanently by adding it to the [Allowed IPs](https://wafris.org/rule_setting/allowed_ips) list.
