# Source: https://wafris.org/docs

Title: Overview

URL Source: https://wafris.org/docs

Markdown Content:
Wafris is an open source, easy to install and manage web application firewall (WAF). Installed into your web server, framework or platform Wafris natively integrates with your system, letting you:

* Monitor inbound web requests for anomalies
* Automatically block requests from suspicious IP addresses
* Filter Geographically to reduce your surface area
* Block requests based on custom rules for paths, user agents and IP addresses
