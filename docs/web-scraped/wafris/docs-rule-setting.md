Title: Rule Setting

URL Source: https://wafris.org/docs/rule_setting/

Markdown Content:
The Rule Setting section in your dashboard allows you to manage various security features to protect your site from malicious traffic and attacks. These features can be customized to suit your specific security needs.

* [Allow CIDRs](https://wafris.org/docs/rule_setting/allow_cidrs/)

 Allow specific IP address ranges to bypass other rules.

* [Allow IPs](https://wafris.org/docs/rule_setting/allow_ips/)

 Allow specific IP addresses to bypass other rules.

* [Block CIDRs](https://wafris.org/docs/rule_setting/block_cidrs/)

 Block specific IP address ranges from accessing your site.

* [Block Countries](https://wafris.org/docs/rule_setting/block_countries/)

 Block access to your site from specific countries.

* [Block Hosts](https://wafris.org/docs/rule_setting/block_hosts/)

 Block specific hosts from accessing your site.

* [Block IPs](https://wafris.org/docs/rule_setting/block_ips/)

 Block specific IP addresses from accessing your site.

* [Block IPs by Reputation](https://wafris.org/docs/rule_setting/block_ips_by_reputation/)

 Block IPs by reputation score.

* [Block Parameters](https://wafris.org/docs/rule_setting/block_parameters/)

 Block specific URL parameters from being passed in requests to your site.

* [Block Paths](https://wafris.org/docs/rule_setting/block_paths/)

 Block specific URL paths from being accessed on your site.

* [Block Rate Limits](https://wafris.org/docs/rule_setting/block_rate_limits/)

 Block rate limits for specific IP addresses.

* [Block User Agents](https://wafris.org/docs/rule_setting/block_user_agents/)

 Block specific user agents from accessing your site.

* [Rule Sets](https://wafris.org/docs/rule_setting/rule_sets/)

 Manage rule sets to apply to your site.
