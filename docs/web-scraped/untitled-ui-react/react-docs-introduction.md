Introduction
============

Welcome to Untitled UI React! This library is a collection of UI components based on the world's largest and most popular Figma UI kit and design system.

[What is Untitled UI React?](https://www.untitledui.com/react/docs/introduction#what-is-untitled-ui-react)
----------------------------------------------------------------------------------------------------------

[Untitled UI React](https://www.untitledui.com/react/) is the world's largest collection of open-source React components built with Tailwind CSS and React Aria. Just copy, paste, and build.

Skip months of design and development with pixel-perfect, production-ready components. Untitled UI React is built with React Aria and TypeScript v5.8, and styled with the latest version of Tailwind CSS v4.1.

It's the perfect starting point for any project—everything you need to design and build modern, beautiful interfaces and websites.

[Tech stack](https://www.untitledui.com/react/docs/introduction#tech-stack)
---------------------------------------------------------------------------

We've kept our tech stack minimal, so you can focus on building. Untitled UI is powered by:

[![Image 1: React Logo](https://www.untitledui.com/react/logos/integrations/react.svg) React v19.1](https://react.dev/blog/2024/12/05/react-19)[Tailwind CSS v4.1](https://tailwindcss.com/blog/tailwindcss-v4)[TypeScript v5.8](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-8.html)[React Aria v1.9](https://react-spectrum.adobe.com/react-aria/index.html)

[Accessibility](https://www.untitledui.com/react/docs/introduction#accessibility)
---------------------------------------------------------------------------------

Accessibility isn’t optional—it’s the baseline for any modern build.

To ensure every component meets accessibility standards and best practices, we built our components based on [React Aria](https://react-spectrum.adobe.com/react-aria/index.html). React Aria is a library of unstyled primitives that make it easy to build high-quality and accessible components.

[How is this different from a library?](https://www.untitledui.com/react/docs/introduction#how-is-this-different-from-a-library)
--------------------------------------------------------------------------------------------------------------------------------

Unlike traditional component libraries that are installed as a dependency, Untitled UI React gives you the source code.

When you select a component, its code is added directly to your project. This means no package dependencies to manage and no vendor lock-in! You have full control to modify, extend, or adapt the components to your specific needs—because you own the code.

[Next steps](https://www.untitledui.com/react/docs/introduction#next-steps)
---------------------------------------------------------------------------

Check out our [installation guide](https://www.untitledui.com/react/docs/installation) to get started.

Want to contribute? Head to our [GitHub repo](https://github.com/untitleduico/react) to open an issue or pull request—we’re always looking to improve and make these components even more useful for you.

Please refer to our[frequently asked questions](https://www.untitledui.com/faqs)page for more.

### What is Untitled UI React?

[Untitled UI React](https://www.untitledui.com/react) is the world's largest collection of open-source React components. Everything you need to design and develop modern, beautiful interfaces—fast.

Skip months of design and development with pixel-perfect, production-ready components. Untitled UI React is built with React Aria and TypeScript, and styled with Tailwind CSS. It's the perfect starting point for any project, no matter the size.

### Is Untitled UI React free and open-source?

We've made a huge portion of [Untitled UI React](https://www.untitledui.com/react) completely free and open-source! open-source components are licensed under [the MIT license](https://opensource.org/license/MIT), which means you can use them for free in unlimited commercial projects.

Untitled UI React [base components](https://www.untitledui.com/react/components) are open-source, along with hundreds of additional [application UI components](https://www.untitledui.com/react/application-ui) and [marketing website sections](https://www.untitledui.com/react/marketing).

The [PRO version](https://www.untitledui.com/buy/react) includes hundreds more advanced UI components and 250+ [dashboards](https://www.untitledui.com/react/components/dashboards), [settings pages](https://www.untitledui.com/react/components/settings-pages), and [marketing page examples](https://www.untitledui.com/react/marketing/landing-pages).

### What tech stack does Untitled UI React use?

Untitled UI React components are built with and powered by:

*   [React v19.1](https://react.dev/blog/2024/12/05/react-19)
*   [Tailwind CSS v4.1](https://tailwindcss.com/blog/tailwindcss-v4)
*   [TypeScript v5.8](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-8.html)
*   [React Aria](https://react-spectrum.adobe.com/react-aria/index.html)

We’ve kept our tech stack minimal, so you can focus on building instead of wrestling with dependencies. You own the code so you can build and extend without relying on third-party maintainers.

We're always working to make Untitled UI better than ever, including implementing new updates into our products as fast as possible. All Untitled UI products come with free updates for life.

### What version of Tailwind CSS does Untitled UI React use?

Untitled UI React components are built using the latest version of [Tailwind CSS (v4.1)](https://tailwindcss.com/blog/tailwindcss-v4), giving you access to the newest utility classes, performance improvements, and best practices from the Tailwind team.

### Do Untitled UI React components work with React v19.1?

Untitled UI React components are built with the latest version of [React (v19.1)](https://react.dev/blog/2024/12/05/react-19), so you get modern features, improved performance, and full compatibility with the newest React updates out of the box.

### What version of TypeScript does Untitled UI React use?

Untitled UI React components are built with the latest version of [TypeScript (v5.8)](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-8.html), so you get modern features, better type safety, and performance improvements out of the box.

### Can I contribute to Untitled UI React?

Yes! We're always working to improve Untitled UI React and make the components even more useful. If you'd like to contribute, head over to our [GitHub repository](https://github.com/untitleduico/react) and open an issue or submit a pull request.

### Does Untitled UI React have a Figma library?

Yes, Untitled UI React is perfectly aligned with [Untitled UI Figma](https://www.untitledui.com/figma), the world’s largest and most popular Figma UI kit and design system.

Keep design and code perfectly in sync, with free updates for life.
