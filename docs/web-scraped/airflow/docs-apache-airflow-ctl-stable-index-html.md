What is airflowctl?[¶](https://airflow.apache.org/docs/apache-airflow-ctl/stable/index.html#what-is-airflowctl "Link to this heading")
======================================================================================================================================

airflowctl is a command line tool that helps you communicate with API and provide similar functionality with Apache Airflow CLI. It is designed to be easy to use and provides a simple interface for managing your Airflow API calls and daily operations.

Please head over to [Quick Start](https://airflow.apache.org/docs/apache-airflow-ctl/stable/start.html) to get started.
