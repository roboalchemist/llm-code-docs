### Overview [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jmoiron/sqlx#pkg-overview "Go to Overview")

Package sqlx provides functions to trace the jmoiron/sqlx package ([https://github.com/jmoiron/sqlx](https://github.com/jmoiron/sqlx)). To enable tracing, first use one of the "Register*" functions to register the sql driver that you will be using, then continue using the package as you normally would.

For more information on registering and why this needs to happen, please check the github.com/DataDog/dd-trace-go/contrib/database/sql package.

func Connect(driverName, dataSourceName [string](https://pkg.go.dev/builtin#string)) (*[sqlx](https://pkg.go.dev/github.com/jmoiron/sqlx).[DB](https://pkg.go.dev/github.com/jmoiron/sqlx#DB), [error](https://pkg.go.dev/builtin#error))

MustConnect connects to a database and panics on error. To get tracing, the driver must be formerly registered using the database/sql integration's Register.

#### func [MustOpen](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/jmoiron/sqlx/sql.go#L34)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jmoiron/sqlx#MustOpen "Go to MustOpen")added in v1.0.0

MustOpen is the same as Open, but panics on error. To get tracing, the driver must be formerly registered using the database/sql integration's Register.

#### func [Open](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/jmoiron/sqlx/sql.go#L23)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jmoiron/sqlx#Open "Go to Open")

Open opens a new (traced) connection to the database using the given driver and source. Note that the driver must formerly be registered using database/sql integration's Register.
