### Overview [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/go-redis/redis#pkg-overview "Go to Overview")

Package redis provides tracing functions for tracing the go-redis/redis package ([https://github.com/go-redis/redis](https://github.com/go-redis/redis)). This package supports versions up to go-redis 6.15.

Example [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/go-redis/redis#example-package "Go to Example")

To start tracing Redis, simply create a new client using the library and continue using as you normally would.

Share Format Run

Example (Pipeliner) [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/go-redis/redis#example-package-Pipeliner "Go to Example (Pipeliner)")

You can also trace Redis Pipelines. Simply use as usual and the traces will be automatically picked up by the underlying implementation.
