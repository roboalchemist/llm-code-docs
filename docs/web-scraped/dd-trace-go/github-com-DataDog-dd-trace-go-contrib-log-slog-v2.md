func IsAlreadyWrapped(h [slog](https://pkg.go.dev/log/slog).[Handler](https://pkg.go.dev/log/slog#Handler)) [bool](https://pkg.go.dev/builtin#bool)

IsAlreadyWrapped checks whether the given handler is already wrapped by this package. This can happen when a lot of packages are using slog and you want to avoid double wrapping. This was originally created to avoid a deadlock in calling slog.SetDefault multiple times. See [https://github.com/golang/go/blob/86c5e2d025b6923424540d653a0c59cafc7571ca/src/log/slog/logger.go#L64-L69](https://github.com/golang/go/blob/86c5e2d025b6923424540d653a0c59cafc7571ca/src/log/slog/logger.go#L64-L69)
