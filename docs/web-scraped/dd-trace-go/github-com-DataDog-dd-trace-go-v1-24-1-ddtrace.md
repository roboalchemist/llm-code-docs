### Overview [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#pkg-overview "Go to Overview")

Package ddtrace contains the interfaces that specify the implementations of Datadog's tracing library, as well as a set of sub-packages containing various implementations: our native implementation ("tracer"), a wrapper that can be used with Opentracing ("opentracer") and a mock tracer to be used for testing ("mocktracer"). Additionally, package "ext" provides a set of tag names and values specific to Datadog's APM product.

To get started, visit the documentation for any of the packages you'd like to begin with by accessing the subdirectories of this package: [https://godoc.org/gopkg.in/DataDog/dd-trace-go.v1/ddtrace#pkg-subdirectories](https://godoc.org/gopkg.in/DataDog/dd-trace-go.v1/ddtrace#pkg-subdirectories).

type FinishConfig struct {
// FinishTime represents the time that should be set as finishing time for the	// span. Implementations should use the current time when FinishTime.IsZero().

	Error [error](https://pkg.go.dev/builtin#error)

#### type [Span](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/ddtrace/ddtrace.go#L41)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#Span "Go to Span")

type Span interface {
// SetTag sets a key/value pair as metadata on the span.	SetTag(key [string](https://pkg.go.dev/builtin#string), value interface{})

// SetOperationName sets the operation name for this span. An operation name should be	// a representative name for a group of spans (e.g. "grpc.server" or "http.request").
	SetOperationName(operationName [string](https://pkg.go.dev/builtin#string))

// BaggageItem returns the baggage item held by the given key.	BaggageItem(key [string](https://pkg.go.dev/builtin#string)) [string](https://pkg.go.dev/builtin#string)

// SetBaggageItem sets a new baggage item at the given key. The baggage	// item should propagate to all descendant spans, both in- and cross-process.
	SetBaggageItem(key, val [string](https://pkg.go.dev/builtin#string))

// Finish finishes the current span with the given options. Finish calls should be idempotent.	Finish(opts ...[FinishOption](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#FinishOption))

// Context returns the SpanContext of this Span.	Context() [SpanContext](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#SpanContext)
}

Span represents a chunk of computation time. Spans have names, durations, timestamps and other metadata. A Tracer is used to create hierarchies of spans in a request, buffer and submit them to the server.

#### type [SpanContext](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/ddtrace/ddtrace.go#L67)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#SpanContext "Go to SpanContext")

type SpanContext interface {
// SpanID returns the span ID that this context is carrying.	SpanID() [uint64](https://pkg.go.dev/builtin#uint64)

// TraceID returns the trace ID that this context is carrying.	TraceID() [uint64](https://pkg.go.dev/builtin#uint64)

// ForeachBaggageItem provides an iterator over the key/value pairs set as	// baggage within this context. Iteration stops when the handler returns
	// false.
	ForeachBaggageItem(handler func(k, v [string](https://pkg.go.dev/builtin#string)) [bool](https://pkg.go.dev/builtin#bool))
}

SpanContext represents a span state that can propagate to descendant spans and across process boundaries. It contains all the information needed to spawn a direct descendant of the span that it belongs to. It can be used to create distributed tracing by propagating it using the provided interfaces.

#### type [Tracer](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/ddtrace/ddtrace.go#L21)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#Tracer "Go to Tracer")

type Tracer interface {
// StartSpan starts a span with the given operation name and options.	StartSpan(operationName [string](https://pkg.go.dev/builtin#string), opts ...[StartSpanOption](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#StartSpanOption)) [Span](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#Span)

// Extract extracts a span context from a given carrier. Note that baggage item	// keys will always be lower-cased to maintain consistency. It is impossible to
	// maintain the original casing due to MIME header canonicalization standards.
	Extract(carrier interface{}) ([SpanContext](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#SpanContext), [error](https://pkg.go.dev/builtin#error))

// Inject injects a span context into the given carrier.	Inject(context [SpanContext](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/ddtrace#SpanContext), carrier interface{}) [error](https://pkg.go.dev/builtin#error)

// Stop stops the active tracer and sets the global tracer to a no-op. Calls to	// Stop should be idempotent.
	Stop()
}

Tracer specifies an implementation of the Datadog tracer which allows starting and propagating spans. The official implementation if exposed as functions within the "tracer" package.
