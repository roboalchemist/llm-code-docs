### Overview [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/redis/rueidis/v2#pkg-overview "Go to Overview")

Package rueidis provides tracing functions for tracing the redis/rueidis package ([https://github.com/redis/rueidis](https://github.com/redis/rueidis)).

Example [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/redis/rueidis/v2#example-package "Go to Example")

To start tracing Redis, simply create a new client using the library and continue using as you normally would.
