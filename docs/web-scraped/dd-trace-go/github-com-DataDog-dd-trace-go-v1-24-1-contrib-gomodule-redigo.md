func Dial(network, address [string](https://pkg.go.dev/builtin#string), options ...interface{}) ([redis](https://pkg.go.dev/github.com/gomodule/redigo/redis).[Conn](https://pkg.go.dev/github.com/gomodule/redigo/redis#Conn), [error](https://pkg.go.dev/builtin#error))

Dial dials into the network address and returns a traced redis.Conn. The set of supported options must be either of type redis.DialOption or this package's DialOption.

#### func [DialURL](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/gomodule/redigo/redigo.go#L91)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/gomodule/redigo#DialURL "Go to DialURL")

func DialURL(rawurl [string](https://pkg.go.dev/builtin#string), options ...interface{}) ([redis](https://pkg.go.dev/github.com/gomodule/redigo/redis).[Conn](https://pkg.go.dev/github.com/gomodule/redigo/redis#Conn), [error](https://pkg.go.dev/builtin#error))

DialURL connects to a Redis server at the given URL using the Redis URI scheme. URLs should follow the draft IANA specification for the scheme ([https://www.iana.org/assignments/uri-schemes/prov/redis](https://www.iana.org/assignments/uri-schemes/prov/redis)). The returned redis.Conn is traced.

#### func (Conn) [Do](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/gomodule/redigo/redigo.go#L178)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/gomodule/redigo#Conn.Do "Go to Conn.Do")

func (tc [Conn](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/gomodule/redigo#Conn)) Do(commandName [string](https://pkg.go.dev/builtin#string), args ...interface{}) (reply interface{}, err [error](https://pkg.go.dev/builtin#error))

Do wraps redis.Conn.Do. It sends a command to the Redis server and returns the received reply. In the process it emits a span containing key information about the command sent. When passed a context.Context as the final argument, Do will ensure that any span created inherits from this context. The rest of the arguments are passed through to the Redis server unchanged.
