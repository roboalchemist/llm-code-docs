#### func [WithCallbacks](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/jinzhu/gorm/gorm.go#L46)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jinzhu/gorm#WithCallbacks "Go to WithCallbacks")added in v1.18.0

func WithCallbacks(db *[gorm](https://pkg.go.dev/github.com/jinzhu/gorm).[DB](https://pkg.go.dev/github.com/jinzhu/gorm#DB), opts ...[Option](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jinzhu/gorm#Option)) *[gorm](https://pkg.go.dev/github.com/jinzhu/gorm).[DB](https://pkg.go.dev/github.com/jinzhu/gorm#DB)

WithCallbacks registers callbacks to the gorm.DB for tracing. It should be called once, after opening the db. The callbacks are triggered by Create, Update, Delete, Query and RowQuery operations.

#### func [WithContext](https://github.com/DataDog/dd-trace-go/blob/v1.24.1/contrib/jinzhu/gorm/gorm.go#L76)[¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go@v1.24.1/contrib/jinzhu/gorm#WithContext "Go to WithContext")added in v1.18.0

func WithContext(ctx [context](https://pkg.go.dev/context).[Context](https://pkg.go.dev/context#Context), db *[gorm](https://pkg.go.dev/github.com/jinzhu/gorm).[DB](https://pkg.go.dev/github.com/jinzhu/gorm#DB)) *[gorm](https://pkg.go.dev/github.com/jinzhu/gorm).[DB](https://pkg.go.dev/github.com/jinzhu/gorm#DB)

WithContext attaches the specified context to the given db. The context will be used as a basis for creating new spans. An example use case is providing a context which contains a span to be used as a parent.
