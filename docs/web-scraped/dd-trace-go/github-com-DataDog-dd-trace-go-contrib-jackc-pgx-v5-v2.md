func WithErrCheck(fn func(err [error](https://pkg.go.dev/builtin#error)) [bool](https://pkg.go.dev/builtin#bool)) [Option](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/jackc/pgx.v5/v2#Option)

WithErrCheck specifies a function fn which determines whether the passed error should be tagged into the span as an error. fn is called whenever a pgx operation finishes with an error

When the function returns true, the span will be tagged with the error. When the function returns false, the span will not be tagged with the error. When the function is nil, the span will be tagged with the error.
