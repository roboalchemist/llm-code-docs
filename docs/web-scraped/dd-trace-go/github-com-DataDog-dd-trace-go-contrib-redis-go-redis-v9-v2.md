### Overview [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/redis/go-redis.v9/v2#pkg-overview "Go to Overview")

Package redis provides functions to trace the redis/go-redis package ([https://github.com/redis/go-redis](https://github.com/redis/go-redis)).

Example [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/redis/go-redis.v9/v2#example-package "Go to Example")

To start tracing Redis, simply create a new client using the library and continue using as you normally would.

Example (Pipeliner) [¶](https://pkg.go.dev/github.com/DataDog/dd-trace-go/contrib/redis/go-redis.v9/v2#example-package-Pipeliner "Go to Example (Pipeliner)")

You can also trace Redis Pipelines. Simply use as usual and the traces will be automatically picked up by the underlying implementation.
