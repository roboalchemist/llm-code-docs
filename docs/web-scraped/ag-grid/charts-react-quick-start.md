React Charts Quick Start
========================

![Image 2: react logo](https://www.ag-grid.com/charts/_astro/react.CtDRhtxt.svg)React

AG Charts is a high-performance, canvas-based React charting library for building React Charts and React Graphs with an intuitive API, a wide-range of series types and a rich feature set. Available in Community and Enterprise editions. Visit [Community vs. Enterprise](https://www.ag-grid.com/charts/react/community-vs-enterprise/) to learn more.

Create a React Chart [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#create-a-react-chart)
--------------------------------------------------------------------------------------------------------

Add React Charts to your application in 60 seconds:

### NPM Install [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#npm-install)

Install the AG Charts React library:

*    NPM 
*    Yarn 

```bash
npm install ag-charts-react
```

```bash
yarn add ag-charts-react
```

### Module Install [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#module-install)

Register the `AllCommunityModule` to access all Community features:

```js
import { AllCommunityModule, ModuleRegistry } from 'ag-charts-community';

// Enable all Community features
ModuleRegistry.registerModules([AllCommunityModule]);
```

To minimize bundle size, only register the modules you want to use. See the [Modules](https://www.ag-grid.com/charts/react/module-registry/) page for more information.

### Import the React Chart [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#import-the-react-chart)

```js
// React Chart Component
import { AgCharts } from 'ag-charts-react';
```

### Define Chart Data and Series [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#define-chart-data-and-series)

All configuration is done via the [`options`](https://www.ag-grid.com/charts/options/) object. At a minimum, you need to provide the `data` and `series` properties:

```tsx
const ChartExample = () => {
    // Chart Options: Control & configure the chart
    const [chartOptions, setChartOptions] = useState({
        // Data: Data to be displayed in the chart
        data: [
            { month: 'Jan', avgTemp: 2.3, iceCreamSales: 162000 },
            { month: 'Mar', avgTemp: 6.3, iceCreamSales: 302000 },
            { month: 'May', avgTemp: 16.2, iceCreamSales: 800000 },
            { month: 'Jul', avgTemp: 22.8, iceCreamSales: 1254000 },
            { month: 'Sep', avgTemp: 14.5, iceCreamSales: 950000 },
            { month: 'Nov', avgTemp: 8.9, iceCreamSales: 200000 },
        ],
        // Series: Defines which chart type and data to use
        series: [{ type: 'bar', xKey: 'month', yKey: 'iceCreamSales' }],
    });

    // ...
};
```

### React Chart Component [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#react-chart-component)

Return the `AgCharts` component with the `chartOptions` as a prop:

```jsx
return (
    // AgCharts component with options passed as prop
    <AgCharts options={chartOptions} />
);
```

### Running the React Chart [Copy Link](https://www.ag-grid.com/charts/react/quick-start/#running-the-react-chart)

Below is a live example of the application running. Click `</> Code` to see the code.

Hide files Viewing: index.jsx

*   index.jsx

Language: 

```jsx
import React, { useState } from "react";
import { createRoot } from "react-dom/client";
import { AgCharts } from "ag-charts-react";
import { AllCommunityModule, ModuleRegistry } from "ag-charts-community";

ModuleRegistry.registerModules([AllCommunityModule]);

const ChartExample = () => {
  const [options, setOptions] = useState({
    // Data: Data to be displayed in the chart
    data: [
      { month: "Jan", avgTemp: 2.3, iceCreamSales: 162000 },
      { month: "Mar", avgTemp: 6.3, iceCreamSales: 302000 },
      { month: "May", avgTemp: 16.2, iceCreamSales: 800000 },
      { month: "Jul", avgTemp: 22.8, iceCreamSales: 1254000 },
      { month: "Sep", avgTemp: 14.5, iceCreamSales: 950000 },
      { month: "Nov", avgTemp: 8.9, iceCreamSales: 200000 },
    ],
    // Series: Defines which chart type and data to use
    series: [{ type: "bar", xKey: "month", yKey: "iceCreamSales" }],
  });

  return <AgCharts options={options} />;
};

const root = createRoot(document.getElementById("root"));
root.render(<ChartExample />);
```
