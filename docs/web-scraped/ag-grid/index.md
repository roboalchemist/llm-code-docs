JavaScript Data Grid FAQs

### Frequently Asked Questions

#### How many rows can AG Grid handle? How fast is AG Grid? Is AG Grid free? Answers to these common questions and more.

Can I build JavaScript Data Grids for free with AG Grid?

Yes, it's free to get started with AG Grid and you'll have access to core features like filtering, sorting, cell editing, pagination and much more.

Is AG Grid compatible with AG Charts?

Yes, AG Grid is designed to be compatible with AG Charts, allowing seamless integration between your JavaScript Data Grids and [JavaScript Charts](https://www.ag-grid.com/charts/react/quick-start/).

Is AG Grid compatible with my Framework?

Yes, AG Grid comes with support for [React](https://ag-grid.com/react-data-grid/getting-started/), [Angular](https://ag-grid.com/angular-data-grid/getting-started/) and [Vue](https://ag-grid.com/vue-data-grid/getting-started/). If you're using another framework, such as Solid, Svelte or Python, you can leverage one of our [community built integrations.](https://ag-grid.com/community/tools-extensions/)

How is AG Grid different from Tanstacks' React Table

We like to say AG Grid 'comes with batteries included'. Tanstack Table is headless, meaning you need to build the UI from scratch, where-as AG Grid can be used out-of-the-box.

What is an AG Grid licence?

An AG Grid licence is required to use [AG Grid Enterprise](https://ag-grid.com/license-pricing/) in production. Try it for free, or request a trial licence at [info@ag-grid.com](mailto:info@ag-grid.com) to remove the watermark & console messages.

How fast is AG Grid?

AG Grid has always been focused on performance and reliability with large data sets; We use virtualisation for both rows and columns to make sure your users have the smoothest experience possible, whatever the size of your data.

Can I customise the style of my Data Grid?

Yes, you can customise AG Grid with our [Theming API](https://ag-grid.com/react-data-grid/themes/) or use our [Theme Builder](https://ag-grid.com/theme-builder/) to build themes with our intuitive UI.

Can I use my own components with AG Grid?

Yes, you can add your own components to [cells](https://ag-grid.com/react-data-grid/component-cell-renderer/) and [headers](https://ag-grid.com/react-data-grid/column-headers-components/#custom-component), and replace things like [filters](https://ag-grid.com/react-data-grid/component-filter/), [overlays](https://ag-grid.com/react-data-grid/overlays-overview/), and [tooltips](https://ag-grid.com/react-data-grid/tooltips/#custom-component) with your own functionality if you want to.

What is the difference between AG Grid Community and Enterprise?

[AG Grid Community](https://ag-grid.com/react-data-grid/getting-started/) is the free version with basic features, while [AG Grid Enterprise](https://ag-grid.com/license-pricing/) offers additional advanced features and support.

How much does AG Grid cost?

[AG Grid Community](https://ag-grid.com/react-data-grid/getting-started/) is free forever. [AG Grid Enterprise](https://ag-grid.com/license-pricing/) costs $999 USD per licence. Visit the [Pricing page](https://ag-grid.com/license-pricing/) for more information.
