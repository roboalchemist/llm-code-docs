[Vite](https://vuepress.vuejs.org/reference/bundler/vite#vite)
==============================================================

[![Image 3: @vuepress/bundler-vite](https://badgen.net/npm/v/@vuepress/bundler-vite/next?label=%40vuepress%2Fbundler-vite%40next)](https://www.npmjs.com/package/@vuepress/bundler-vite "@vuepress/bundler-vite")
Vite bundler is provided by [@vuepress/bundler-vite](https://www.npmjs.com/package/@vuepress/bundler-vite) package.

[Usage](https://vuepress.vuejs.org/reference/bundler/vite#usage)
----------------------------------------------------------------

Install the bundler package:

```bash
npm i -D @vuepress/bundler-vite@next
```

Specify the bundler option in your config file:

.vuepress/config.ts

```ts
import { viteBundler } from '@vuepress/bundler-vite'
import { defineUserConfig } from 'vuepress'

export default defineUserConfig({
  bundler: viteBundler({
    viteOptions: {},
    vuePluginOptions: {},
  }),
})
```

[Options](https://vuepress.vuejs.org/reference/bundler/vite#options)
--------------------------------------------------------------------

### [viteOptions](https://vuepress.vuejs.org/reference/bundler/vite#viteoptions)

*   Details:

Accepts all options of Vite.

*   Also see:

    *   [Vite > Config](https://vite.dev/config/)

### [vuePluginOptions](https://vuepress.vuejs.org/reference/bundler/vite#vuepluginoptions)

*   Details:

Accepts all options of [@vitejs/plugin-vue](https://www.npmjs.com/package/@vitejs/plugin-vue).

*   Also see:

    *   [Vite > Plugins > Official Plugins](https://vite.dev/plugins/#vitejs-plugin-vue)
