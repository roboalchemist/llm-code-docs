Developer's Tag Index
=====================

About this document
-------------------

The goal of this guide is to enable everyone who would like to contribute to metasfresh or to just simply study the code to do so.

Java version
------------

Please make sure you are using **Java 8** while building the code.

Getting started
---------------

*   [Get the code](https://docs.metasfresh.org/developers_collection/en/getting_started_clone_en.html) : How to obtain the source code from Github 
*   [Set up your Maven settings file](https://docs.metasfresh.org/developers_collection/en/getting_started_maven_settings.html) : Maven `settings.xml` file for Eclipse and Command Line 
*   [Get the code into your IDE](https://docs.metasfresh.org/developers_collection/en/getting_started_ide_en.html) : How to import the source code into your IDE (Eclipse) 
*   [Get your local DB up and running](https://docs.metasfresh.org/developers_collection/en/getting_started_db_en.html) : How to import the latest metasfresh DB seed into your PostgreSQL database 
*   [Build metasfresh from command line](https://docs.metasfresh.org/developers_collection/en/getting_started_mvn_cmdline_en.html) : How to create a distributable using Maven from your command line 
*   [Handle multiple repos from command line](https://docs.metasfresh.org/developers_collection/en/getting_started_myrepos_en.html) : How to make life easier without GitHub desktop
