## Quick Facts

- **Type**: Static code analysis tool for JavaScript/TypeScript
- **Focus**: Runtime errors and code quality (not style/linting)
- **Framework Support**: React, Vue.js, Node.js, Astro
- **Key Feature**: Data-flow analysis for semantic understanding
- **Integration**: GitHub, VS Code, IntelliJ, SonarQube, CI/CD pipelines
- **Pricing**: Free for public projects, $9-$19/seat for teams

## Key Differentiators

1. **Data-Flow Analysis**: Traces how data flows through code
2. **Runtime Error Detection**: Finds bugs that could crash at runtime
3. **Framework-Aware**: Specialized rules for React and Vue.js
4. **Reduced False Positives**: Smart impact classification
5. **Team Dashboard**: Centralized quality monitoring
