### Constants [¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/benchmark#pkg-constants "Go to Constants")

[View Source](https://github.com/grpc/grpc-go/blob/v1.79.1/benchmark/benchmark.go#L90)const PreloadMsgSizeHeader = "preload-msg-size"

PreloadMsgSizeHeader indicates that the client is going to ask for a fixed response size and passes this size to the server. The server is expected to preload the response on startup.

[View Source](https://github.com/grpc/grpc-go/blob/v1.79.1/benchmark/benchmark.go#L85)const UnconstrainedStreamingDelayHeader = "unconstrained-streaming-delay"

UnconstrainedStreamingDelayHeader is used to pass the maximum amount of time the server should sleep between consecutive RPC responses.

[View Source](https://github.com/grpc/grpc-go/blob/v1.79.1/benchmark/benchmark.go#L81)const UnconstrainedStreamingHeader = "unconstrained-streaming"

UnconstrainedStreamingHeader indicates to the StreamingCall handler that its behavior should be unconstrained (constant send/receive in parallel) instead of ping-pong.
