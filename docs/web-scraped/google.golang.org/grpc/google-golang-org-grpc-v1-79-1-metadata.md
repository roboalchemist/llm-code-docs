#### func [New](https://github.com/grpc/grpc-go/blob/v1.79.1/metadata/metadata.go#L59)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/metadata#New "Go to New")

func New(m map[[string](https://pkg.go.dev/builtin#string)][string](https://pkg.go.dev/builtin#string)) [MD](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/metadata#MD)

New creates an MD from a given key-value map.

Only the following ASCII characters are allowed in keys:

*   digits: 0-9
*   uppercase letters: A-Z (normalized to lower)
*   lowercase letters: a-z
*   special characters: -_.

Uppercase letters are automatically converted to lowercase.

Keys beginning with "grpc-" are reserved for grpc-internal use only and may result in errors if set in metadata.

#### func (MD) [Append](https://github.com/grpc/grpc-go/blob/v1.79.1/metadata/metadata.go#L130)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/metadata#MD.Append "Go to MD.Append")added in v1.12.0

func (md [MD](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/metadata#MD)) Append(k [string](https://pkg.go.dev/builtin#string), vals ...[string](https://pkg.go.dev/builtin#string))
