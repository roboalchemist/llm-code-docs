### Overview [¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/binarylog#pkg-overview "Go to Overview")

Package binarylog implementation binary logging as defined in [https://github.com/grpc/proposal/blob/master/A16-binary-logging.md](https://github.com/grpc/proposal/blob/master/A16-binary-logging.md).

Notice: All APIs in this package are experimental.

#### type [Sink](https://github.com/grpc/grpc-go/blob/v1.79.1/binarylog/sink.go#L45)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/binarylog#Sink "Go to Sink")

type Sink interface {
// Write marshals the log entry and writes it to the destination. The format	// is not specified, but should have sufficient information to rebuild the
	// entry. Some options are: proto bytes, or proto json.
	//
	// Note this function needs to be thread-safe.
	Write(*[binlogpb](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/binarylog/grpc_binarylog_v1).[GrpcLogEntry](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/binarylog/grpc_binarylog_v1#GrpcLogEntry)) [error](https://pkg.go.dev/builtin#error)
// Close closes this sink and cleans up resources (e.g. the flushing	// goroutine).
	Close() [error](https://pkg.go.dev/builtin#error)
}

Sink represents the destination for the binary log entries.
