#### type [Options](https://github.com/grpc/grpc-go/blob/v1.79.1/credentials/tls/certprovider/pemfile/watcher.go#L55)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/credentials/tls/certprovider/pemfile#Options "Go to Options")

type Options struct {
// CertFile is the file that holds the identity certificate.	// Optional. If this is set, KeyFile must also be set.
	CertFile [string](https://pkg.go.dev/builtin#string)
// KeyFile is the file that holds identity private key.	// Optional. If this is set, CertFile must also be set.
	KeyFile [string](https://pkg.go.dev/builtin#string)
// RootFile is the file that holds trusted root certificate(s).	// Optional.
	RootFile [string](https://pkg.go.dev/builtin#string)
// SPIFFEBundleMapFile is the file that holds the spiffe bundle map.	// If a given provider configures both the RootFile and the
	// SPIFFEBundleMapFile, the SPIFFEBundleMapFile will be preferred.
	// Optional.
	SPIFFEBundleMapFile [string](https://pkg.go.dev/builtin#string)
// RefreshDuration is the amount of time the plugin waits before checking	// for updates in the specified files.
	// Optional. If not set, a default value (1 hour) will be used.
	RefreshDuration [time](https://pkg.go.dev/time).[Duration](https://pkg.go.dev/time#Duration)
}

Options configures a certificate provider plugin that watches a specified set of files that contain certificates and keys in PEM format.
