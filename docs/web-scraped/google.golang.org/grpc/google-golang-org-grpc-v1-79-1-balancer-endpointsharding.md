#### type [ExitIdler](https://github.com/grpc/grpc-go/blob/v1.79.1/balancer/endpointsharding/endpointsharding.go#L54)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/balancer/endpointsharding#ExitIdler "Go to ExitIdler")added in v1.74.0

type ExitIdler interface {
// ExitIdle instructs the LB policy to reconnect to backends / exit the	// IDLE state, if appropriate and possible. Note that SubConns that enter
	// the IDLE state will not reconnect until SubConn.Connect is called.
	ExitIdle()
}

ExitIdler provides access to only the ExitIdle method of the child balancer.

#### type [Options](https://github.com/grpc/grpc-go/blob/v1.79.1/balancer/endpointsharding/endpointsharding.go#L63)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/balancer/endpointsharding#Options "Go to Options")added in v1.71.0

type Options struct {
// DisableAutoReconnect allows the balancer to keep child balancer in the	// IDLE state until they are explicitly triggered to exit using the
	// ChildState obtained from the endpointsharding picker. When set to false,
	// the endpointsharding balancer will automatically call ExitIdle on child
	// connections that report IDLE.
	DisableAutoReconnect [bool](https://pkg.go.dev/builtin#bool)
}

Options are the options to configure the behaviour of the endpointsharding balancer.
