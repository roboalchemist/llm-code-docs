**注意：** Custom Search JSON API 不再開放新客戶使用，[Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=zh-tw) 是搜尋最多 50 個網域的理想替代方案。或者，如果您的用途需要完整網頁搜尋功能，請[與我們聯絡](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=zh-tw)，表達您有意採用完整網頁搜尋解決方案，並取得更多相關資訊。
現有的 Custom Search JSON API 客戶必須在 **2027 年 1 月 1 日**前改用替代解決方案。

您可以透過 Custom Search JSON API 開發網站和應用程式，以程式輔助方式擷取及顯示程式化搜尋引擎的搜尋結果。透過這個 API，您可以發出 RESTful 要求，以 JSON 格式取得**網頁搜尋**或**圖片搜尋**結果。

資料格式
----

Custom Search JSON API 可以 [JSON](https://developers.google.com/custom-search/docs/glossary?hl=zh-tw#json) 資料格式傳回結果。

相關文件
----

Custom Search JSON API 使用 [OpenSearch 1.1 規格](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)。

必要條件
----

### 搜尋引擎 ID

使用 Custom Search JSON API 前，請先建立及設定程式化搜尋引擎。如果您尚未建立程式化搜尋引擎，請先前往[程式化搜尋引擎控制台](https://programmablesearchengine.google.com/controlpanel/all?hl=zh-tw)。

如要進一步瞭解不同的設定選項，請參閱[教學課程](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=zh-tw)。

建立程式化搜尋引擎後，請前往[說明中心](https://support.google.com/programmable-search/answer/2649143?hl=zh-tw)，瞭解如何找出搜尋引擎 ID。

### API 金鑰

使用 Custom Search JSON API 時，必須提供 API 金鑰。

定價
--

下列價格僅適用於現有的 Custom Search JSON API 客戶，直到 2027 年 1 月 1 日服務停用為止。新客戶無法使用這項 API。Custom Search JSON API 每天提供 100 次免費搜尋查詢。如需更多配額，請前往 API 控制台申請[帳單](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=zh-tw)。額外要求費用為每 1, 000 次查詢 $5 美元，每天最多 1 萬次查詢。

監控
--

如要監控 Custom Search JSON API 的基本使用情形，請使用 [Cloud Platform 主控台的 API 資訊主頁](https://console.cloud.google.com/apis/dashboard?hl=zh-tw)。如需更進階的監控功能，請使用 [Google Cloud 作業套件](https://cloud.google.com/products/operations?hl=zh-tw) (原稱 Stackdriver)。

您可以使用 Google Cloud Operations 建立[自訂資訊主頁](https://cloud.google.com/monitoring/charts?hl=zh-tw)、[設定快訊](https://cloud.google.com/monitoring/alerts?hl=zh-tw)，以及[以程式輔助方式存取指標資料](https://cloud.google.com/monitoring/api/v3?hl=zh-tw)。如要在 Google Cloud Operations 中存取 Custom Search JSON API 用量資料，請選取「資源類型：已使用的 API」，並在查詢建構工具中依「service = 'customsearch.googleapis.com'」進行篩選。

如要瞭解 API 資訊主頁和 Google Cloud Operations 套件提供的各種監控和快訊功能，請參閱「[監控 API 用量](https://cloud.google.com/apis/docs/monitoring?hl=zh-tw)」。
