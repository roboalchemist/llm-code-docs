**Remarque**:  L'API JSON Custom Search n'est plus disponible pour les nouveaux clients. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=fr) est une alternative intéressante pour effectuer des recherches sur un maximum de 50 domaines. Si votre cas d'utilisation nécessite une recherche Web complète, [contactez-nous](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=fr) pour nous faire part de votre intérêt et obtenir plus d'informations sur notre solution de recherche Web complète.
Les clients existants de l'API JSON Custom Search ont jusqu'au **1er janvier 2027** pour passer à une autre solution.

L'API JSON Custom Search vous permet de développer des sites Web et des applications pour récupérer et afficher les résultats de recherche de Programmable Search Engine de manière programmatique. Cette API vous permet d'utiliser des requêtes RESTful pour obtenir des résultats de **recherche sur le Web** ou de **recherche d'images** au format JSON.

Format des données
------------------

L'API JSON Custom Search peut renvoyer des résultats au format de données [JSON](https://developers.google.com/custom-search/docs/glossary?hl=fr#json).

Documents associés
------------------

L'API JSON Custom Search utilise la [spécification OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Prérequis
---------

### ID du moteur de recherche

Avant d'utiliser l'API JSON Custom Search, vous devez d'abord créer et configurer votre Programmable Search Engine. Si vous n'avez pas encore créé de Programmable Search Engine, vous pouvez commencer par accéder au [panneau de configuration de Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all?hl=fr).

Suivez le [tutoriel](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=fr) pour en savoir plus sur les différentes options de configuration.

Une fois que vous avez créé un Programmable Search Engine, consultez le [Centre d'aide](https://support.google.com/programmable-search/answer/2649143?hl=fr) pour savoir comment trouver l'ID de votre moteur de recherche.

### Clé API

L'API JSON Custom Search nécessite l'utilisation d'une clé API.

Tarifs
------

Les tarifs suivants ne s'appliquent qu'aux clients existants de l'API JSON Custom Search jusqu'à l'arrêt du service le 1er janvier 2027. Cette API n'est pas disponible pour les nouveaux clients. L'API JSON Custom Search permet d'effectuer sans frais 100 requêtes de recherche par jour. Si vous avez besoin d'un quota plus important, vous pouvez vous inscrire à la [facturation](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=fr) dans la console APIs. 1 000 requêtes supplémentaires coûtent 5$, et vous pouvez utiliser jusqu'à 10 000 requêtes par jour.

Surveillance
------------

La surveillance de base de l'API JSON Custom Search est disponible dans le [tableau de bord des API de la console Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=fr). Pour une surveillance plus avancée, la [suite Google Cloud Operations](https://cloud.google.com/products/operations?hl=fr) (anciennement Stackdriver) est disponible.

Google Cloud Operations vous permet de créer des [tableaux de bord personnalisés](https://cloud.google.com/monitoring/charts?hl=fr), de [configurer des alertes](https://cloud.google.com/monitoring/alerts?hl=fr) et d'[accéder aux données de métriques de manière programmatique](https://cloud.google.com/monitoring/api/v3?hl=fr). Pour accéder aux données d'utilisation de l'API Custom Search JSON dans Google Cloud Operations, sélectionnez "Type de ressource: API consommée" et filtrez sur "service = 'customsearch.googleapis.com'" dans le générateur de requêtes.

Consultez [Surveiller l'utilisation de votre API](https://cloud.google.com/apis/docs/monitoring?hl=fr) pour en savoir plus sur les différentes fonctionnalités de surveillance et d'alerte fournies par le tableau de bord des API et la suite Google Cloud Operations.
