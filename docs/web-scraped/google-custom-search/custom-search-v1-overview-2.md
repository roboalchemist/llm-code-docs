**Nota:** La API de Custom Search JSON está cerrada para los clientes nuevos. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=es) es una alternativa favorable para buscar en hasta 50 dominios. Como alternativa, si tu caso de uso requiere una búsqueda web completa, [comunícate con nosotros](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=es) para expresar tu interés en nuestra solución de búsqueda web completa y obtener más información sobre ella.
Los clientes existentes de la API de Custom Search JSON tienen hasta el **1 de enero de 2027** para migrar a una solución alternativa.

La API de Custom Search JSON te permite desarrollar sitios web y aplicaciones para recuperar y mostrar resultados de la búsqueda del Motor de Búsqueda Programable de forma programática. Con esta API, puedes usar solicitudes de RESTful para obtener resultados de la **búsqueda web** o la **búsqueda de imágenes** en formato JSON.

Formato de los datos
--------------------

La API de Custom Search JSON puede devolver resultados en formato de datos [JSON](https://developers.google.com/custom-search/docs/glossary?hl=es#json).

Documentos relacionados
-----------------------

La API de Custom Search JSON usa la [especificación de OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Requisitos previos
------------------

### ID del motor de búsqueda

Antes de usar la API de Custom Search JSON, primero deberás crear y configurar tu Motor de Búsqueda Programable. Si aún no creaste un Motor de Búsqueda Programable, puedes comenzar por visitar el [panel de control del Motor de Búsqueda Programable](https://programmablesearchengine.google.com/controlpanel/all?hl=es).

Sigue el [instructivo](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=es) para obtener más información sobre las diferentes opciones de configuración.

Después de crear un Motor de Búsqueda Programable, visita el [Centro de ayuda](https://support.google.com/programmable-search/answer/2649143?hl=es) para saber cómo encontrar tu ID del motor de búsqueda.

### Clave de API

La API de Custom Search JSON requiere el uso de una clave de API.

Precios
-------

Los siguientes precios solo se aplican a los clientes existentes de la API de Custom Search JSON hasta la interrupción del servicio el 1 de enero de 2027. Esta API no está disponible para los clientes nuevos. La API de Custom Search JSON proporciona 100 búsquedas por día de forma gratuita. Si necesitas más, puedes registrarte para la [facturación](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=es) en la Consola de APIs. Las solicitudes adicionales cuestan USD 5 por cada 1,000 consultas, hasta 10,000 consultas por día.

Supervisión
-----------

La supervisión básica de la API de Custom Search JSON está disponible a través del [Panel de la API de Cloud Platform Console](https://console.cloud.google.com/apis/dashboard?hl=es). Para una supervisión más avanzada, está disponible [Google Cloud's Operations suite](https://cloud.google.com/products/operations?hl=es) (anteriormente Stackdriver).

Con Google Cloud Operations, puedes crear [paneles personalizados](https://cloud.google.com/monitoring/charts?hl=es), [configurar alertas](https://cloud.google.com/monitoring/alerts?hl=es) y [acceder a los datos de métricas de forma programática](https://cloud.google.com/monitoring/api/v3?hl=es). Para acceder a los datos de uso de la API de Custom Search JSON en Google Cloud Operations, selecciona "Tipo de recurso: API consumida" y filtra por "service = 'customsearch.googleapis.com'" en el Generador de consultas.

Consulta [Supervisa el uso de tu API](https://cloud.google.com/apis/docs/monitoring?hl=es) para obtener información sobre las diferentes capacidades de supervisión y alertas que proporcionan el panel de la API y Google Cloud Operations Suite.
