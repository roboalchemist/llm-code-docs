**หมายเหตุ:** เราไม่รับลูกค้าใหม่สำหรับ Custom Search JSON API [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=th) เป็นทางเลือกที่เหมาะสมสำหรับการค้นหาโดเมนได้สูงสุด 50 โดเมน หรือหากกรณีการใช้งานของคุณจำเป็นต้องค้นหาเว็บแบบเต็ม โปรด[ติดต่อเรา](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=th)เพื่อแสดงความสนใจและรับข้อมูลเพิ่มเติมเกี่ยวกับโซลูชันการค้นหาเว็บแบบเต็มของเรา
ลูกค้า Custom Search JSON API ที่มีอยู่จะมีเวลาจนถึงวันที่ **1 มกราคม 2027** ในการเปลี่ยนไปใช้โซลูชันอื่น

Custom Search JSON API ช่วยให้คุณพัฒนาเว็บไซต์และแอปพลิเคชันเพื่อดึงและแสดงผลการค้นหาจาก Programmable Search Engine โดยอัตโนมัติได้ API นี้ช่วยให้คุณใช้คำขอ RESTful เพื่อรับผลการค้นหาจาก**การค้นหาเว็บ**หรือ**การค้นหารูปภาพ**ในรูปแบบ JSON ได้

รูปแบบข้อมูล
------------

Custom Search JSON API สามารถแสดงผลลัพธ์ในรูปแบบข้อมูล [JSON](https://developers.google.com/custom-search/docs/glossary?hl=th#json)

เอกสารที่เกี่ยวข้อง
-------------------

Custom Search JSON API ใช้[ข้อกำหนด OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)

ข้อกำหนดเบื้องต้น
-----------------

### รหัสเครื่องมือค้นหา

ก่อนใช้ Custom Search JSON API คุณจะต้องสร้างและกำหนดค่า Programmable Search Engine ก่อน หากยังไม่ได้สร้าง Programmable Search Engine ให้เริ่มต้นโดย ไปที่[แผงควบคุม Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all?hl=th)

ดูข้อมูลเพิ่มเติมเกี่ยวกับตัวเลือกการกำหนดค่าต่างๆ ได้จาก[บทแนะนำ](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=th)

หลังจากสร้าง Programmable Search Engine แล้ว ให้ไปที่[ศูนย์ช่วยเหลือ](https://support.google.com/programmable-search/answer/2649143?hl=th)เพื่อดูวิธีค้นหารหัสเครื่องมือค้นหา

### คีย์ API

Custom Search JSON API กำหนดให้ใช้คีย์ API

ราคา
----

การกำหนดราคาต่อไปนี้จะมีผลกับลูกค้า Custom Search JSON API ที่มีอยู่เท่านั้น จนกว่าจะมีการหยุดให้บริการในวันที่ 1 มกราคม 2027 API นี้ไม่มีให้บริการ สำหรับลูกค้าใหม่ Custom Search JSON API ให้คำค้นหา 100 รายการต่อวันโดยไม่มีค่าใช้จ่าย หากต้องการมากกว่านี้ คุณสามารถลงชื่อสมัครใช้[การเรียกเก็บเงิน](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=th)ได้ใน คอนโซล API คำขอเพิ่มเติมมีค่าใช้จ่าย $5 ต่อคำค้นหา 1,000 รายการ โดยจำกัดไว้ที่คำค้นหา 10,000 รายการต่อวัน

การตรวจสอบ
----------

การตรวจสอบพื้นฐานสำหรับ Custom Search JSON API พร้อมใช้งานผ่าน[แดชบอร์ด API ของ Cloud Platform Console](https://console.cloud.google.com/apis/dashboard?hl=th) สำหรับ[ชุดเครื่องมือการดำเนินการของ Google Cloud](https://cloud.google.com/products/operations?hl=th) (เดิมคือ Stackdriver) พร้อมให้บริการสำหรับการตรวจสอบขั้นสูงเพิ่มเติม

Google Cloud Operations ช่วยให้คุณสร้าง[แดชบอร์ด ที่กำหนดเอง](https://cloud.google.com/monitoring/charts?hl=th)[ตั้งค่า การแจ้งเตือน](https://cloud.google.com/monitoring/alerts?hl=th) และ[เข้าถึงข้อมูลเมตริก แบบเป็นโปรแกรม](https://cloud.google.com/monitoring/api/v3?hl=th)ได้ หากต้องการเข้าถึงข้อมูลการใช้งาน Custom Search JSON API ใน Google Cloud Operations ให้เลือก "ประเภททรัพยากร: API ที่ใช้" แล้วกรองใน "service = 'customsearch.googleapis.com'" ใน เครื่องมือสร้างคำค้นหา

ดู[การตรวจสอบการใช้งาน API](https://cloud.google.com/apis/docs/monitoring?hl=th) เพื่อดูการอภิปรายเกี่ยวกับความสามารถในการตรวจสอบและการแจ้งเตือนต่างๆ ที่มีให้ โดยแดชบอร์ด API และ Google Cloud Operations Suite
