**注:** カスタム検索 JSON API は、新規のお客様にはご利用いただけません。[Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=ja) は、最大 50 個のドメインを検索する場合に最適な代替手段です。または、ウェブ全体を検索する必要があるユースケースについては、[こちら](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=ja)からウェブ全体を検索するソリューションについてお問い合わせください。
既存の Custom Search JSON API のお客様は、**2027 年 1 月 1 日**までに代替ソリューションに移行する必要があります。

Custom Search JSON API を使用すると、プログラム可能検索エンジンの検索結果をプログラムで取得して表示するウェブサイトやアプリケーションを開発できます。この API を使用すると、RESTful リクエストを使用して、**ウェブ検索**または**画像検索**の結果を JSON 形式で取得できます。

データ形式
-----

Custom Search JSON API は、結果を [JSON](https://developers.google.com/custom-search/docs/glossary?hl=ja#json) データ形式で返すことができます。

関連ドキュメント
--------

Custom Search JSON API は [OpenSearch 1.1 仕様](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)を使用します。

前提条件
----

### 検索エンジン ID

Custom Search JSON API を使用する前に、プログラム可能検索エンジンを作成して構成する必要があります。プログラム可能検索エンジンをまだ作成していない場合は、まず[プログラム可能検索エンジンのコントロール パネル](https://programmablesearchengine.google.com/controlpanel/all?hl=ja)にアクセスしてください。

さまざまな構成オプションの詳細については、[チュートリアル](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=ja)をご覧ください。

プログラム可能検索エンジンを作成したら、[ヘルプセンター](https://support.google.com/programmable-search/answer/2649143?hl=ja)で検索エンジン ID を確認する方法をご覧ください。

### API キー

Custom Search JSON API では、API キーを使用する必要があります。

料金
--

以下の料金は、2027 年 1 月 1 日のサービス終了まで、既存のカスタム検索 JSON API のお客様にのみ適用されます。この API は新規のお客様はご利用いただけません。Custom Search JSON API では、1 日あたり 100 件の検索クエリを無料で提供しています。さらに必要な場合は、API Console で[お支払い情報](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=ja)を登録できます。追加のリクエストは 1, 000 件あたり $5 で、1 日あたり最大 1 万件のクエリまでです。

モニタリング
------

Custom Search JSON API の基本的なモニタリングは、[Cloud Platform Console の API ダッシュボード](https://console.cloud.google.com/apis/dashboard?hl=ja)で利用できます。より高度なモニタリングには、[Google Cloud のオペレーション スイート](https://cloud.google.com/products/operations?hl=ja)（旧称 Stackdriver）を使用できます。

Google Cloud Operations を使用すると、[カスタム ダッシュボード](https://cloud.google.com/monitoring/charts?hl=ja)を作成し、[アラートを設定](https://cloud.google.com/monitoring/alerts?hl=ja)して、[プログラムから指標データにアクセス](https://cloud.google.com/monitoring/api/v3?hl=ja)できます。Google Cloud Operations でカスタム検索 JSON API の使用状況データにアクセスするには、クエリビルダーで [リソースタイプ: 使用済み API] を選択し、[service = 'customsearch.googleapis.com'] でフィルタします。

API ダッシュボードと Google Cloud オペレーション スイートが提供するさまざまなモニタリング機能とアラート機能については、[API 使用状況のモニタリング](https://cloud.google.com/apis/docs/monitoring?hl=ja)をご覧ください。
