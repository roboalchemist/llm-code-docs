**Observação**: a API JSON Custom Search não está mais disponível para novos clientes. A [Vertex AI para Pesquisa](https://cloud.google.com/use-cases/site-search?hl=pt-br) é uma alternativa interessante para pesquisar até 50 domínios. Se o seu caso de uso exigir a pesquisa completa na Web, [entre em contato](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=pt-br) para manifestar seu interesse e receber mais informações sobre nossa solução de pesquisa completa na Web.
Os clientes atuais da API JSON Custom Search têm até **1º de janeiro de 2027** para fazer a transição para uma solução alternativa.

Com a API JSON Custom Search, é possível desenvolver sites e aplicativos para recuperar e mostrar resultados de pesquisa do Mecanismo de Pesquisa Programável de forma programática. Com essa API, você pode usar solicitações RESTful para receber resultados de **pesquisa na Web** ou **pesquisa de imagens** no formato JSON.

Formato de dados
----------------

A API JSON Custom Search pode retornar resultados no formato de dados [JSON](https://developers.google.com/custom-search/docs/glossary?hl=pt-br#json).

Documentos relacionados
-----------------------

A API Custom Search JSON usa a [especificação OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Pré-requisitos
--------------

### ID do mecanismo de pesquisa

Antes de usar a API JSON Custom Search, você precisa criar e configurar seu Mecanismo de Pesquisa Programável. Se você ainda não criou um Mecanismo de Pesquisa Programável, acesse o [painel de controle do Mecanismo de Pesquisa Programável](https://programmablesearchengine.google.com/controlpanel/all?hl=pt-br).

Siga o [tutorial](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=pt-br) para saber mais sobre diferentes opções de configuração.

Depois de criar um Mecanismo de Pesquisa Programável, acesse a [Central de Ajuda](https://support.google.com/programmable-search/answer/2649143?hl=pt-br) para saber como localizar o ID do mecanismo de pesquisa.

### Chave de API

A API Custom Search JSON exige o uso de uma chave de API.

Preços
------

Os preços a seguir se aplicam apenas aos clientes atuais da API JSON Custom Search até a descontinuação do serviço em 1º de janeiro de 2027. Essa API não está disponível para novos clientes. A API Custom Search JSON oferece 100 consultas de pesquisa por dia sem custo financeiro. Se você precisar de mais, inscreva-se para o [faturamento](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=pt-br) no Console de APIs. Solicitações extras custam US $5 para cada mil consultas, até 10 mil consultas por dia.

Monitoramento
-------------

O monitoramento básico da API Custom Search JSON está disponível no [Painel de APIs do console do Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=pt-br). Para um monitoramento mais avançado, o [pacote de operações do Google Cloud](https://cloud.google.com/products/operations?hl=pt-br) (antigo Stackdriver) está disponível.

Com as operações do Google Cloud, é possível criar [painéis personalizados](https://cloud.google.com/monitoring/charts?hl=pt-br), [configurar alertas](https://cloud.google.com/monitoring/alerts?hl=pt-br) e [acessar dados de métricas de forma programática](https://cloud.google.com/monitoring/api/v3?hl=pt-br). Para acessar os dados de uso da API JSON do Custom Search no Google Cloud Operations, selecione "Tipo de recurso: API consumida" e filtre por "service = 'customsearch.googleapis.com'" no Criador de consultas.

Consulte [Como monitorar o uso da API](https://cloud.google.com/apis/docs/monitoring?hl=pt-br) para uma discussão sobre os diferentes recursos de monitoramento e alerta fornecidos pelo painel de API e pelo pacote de operações do Google Cloud.
