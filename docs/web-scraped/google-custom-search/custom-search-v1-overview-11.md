**Примечание:**API пользовательского поиска в формате JSON закрыт для новых клиентов. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=ru) — это предпочтительная альтернатива для поиска до 50 доменов. В качестве альтернативы, если ваш сценарий использования требует полномасштабного веб-поиска, [свяжитесь с нами](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=ru) , чтобы выразить заинтересованность и получить дополнительную информацию о нашем решении для полномасштабного веб-поиска. 
Существующие клиенты, использующие API пользовательского поиска в формате JSON, должны перейти на альтернативное решение до **1 января 2027 года** .

API Custom Search JSON позволяет разрабатывать веб-сайты и приложения для программного получения и отображения результатов поиска из программируемой поисковой системы. С помощью этого API вы можете использовать RESTful-запросы для получения результатов **веб-поиска** или **поиска изображений** в формате JSON.

Формат данных
-------------

API пользовательского поиска в формате JSON может возвращать результаты в формате данных [JSON](https://developers.google.com/custom-search/docs/glossary?hl=ru#json) .

Сопутствующие документы
-----------------------

API пользовательского поиска в формате JSON использует [спецификацию OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md) .

Предварительные требования
--------------------------

### Идентификатор поисковой системы

Перед использованием API пользовательского поиска в формате JSON вам сначала необходимо создать и настроить свою программируемую поисковую систему. Если вы еще не создали программируемую поисковую систему, вы можете начать с посещения [панели управления программируемой поисковой системой](https://programmablesearchengine.google.com/controlpanel/all?hl=ru) .

Следуйте [инструкциям в руководстве](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=ru) , чтобы узнать больше о различных параметрах конфигурации.

После создания программируемой поисковой системы посетите [справочный центр](https://support.google.com/programmable-search/answer/2649143?hl=ru) , чтобы узнать, как найти идентификатор вашей поисковой системы.

### ключ API

Для использования API пользовательского поиска в формате JSON требуется ключ API.

Цены
----

Указанные цены действуют только для существующих клиентов Custom Search JSON API до прекращения предоставления услуги 1 января 2027 года. Этот API недоступен для новых клиентов. Custom Search JSON API предоставляет 100 поисковых запросов в день бесплатно. Если вам нужно больше, вы можете оформить [платную подписку](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=ru) в консоли API. Дополнительные запросы стоят 5 долларов США за 1000 запросов, до 10 000 запросов в день.

Мониторинг
----------

Базовый мониторинг API пользовательского поиска в формате JSON доступен через [панель управления API в консоли Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=ru) . Для более расширенного мониторинга доступен [пакет инструментов Google Cloud Operations](https://cloud.google.com/products/operations?hl=ru) (ранее Stackdriver).

С помощью Google Cloud Operations вы можете создавать [пользовательские панели мониторинга](https://cloud.google.com/monitoring/charts?hl=ru) , [настраивать оповещения](https://cloud.google.com/monitoring/alerts?hl=ru) и [программно получать доступ к данным метрик](https://cloud.google.com/monitoring/api/v3?hl=ru) . Чтобы получить доступ к данным об использовании API Custom Search JSON в Google Cloud Operations, выберите «Тип ресурса: Используемый API» и отфильтруйте по «service = 'customsearch.googleapis.com'» в построителе запросов.

См. раздел [«Мониторинг использования API»](https://cloud.google.com/apis/docs/monitoring?hl=ru) для ознакомления с различными возможностями мониторинга и оповещения, предоставляемыми панелью управления API и пакетом инструментов Google Cloud Operations.
