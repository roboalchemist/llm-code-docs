**Lưu ý:** Custom Search JSON API không còn được cung cấp cho khách hàng mới. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=vi) là một lựa chọn thay thế phù hợp để tìm kiếm tối đa 50 miền. Ngoài ra, nếu trường hợp sử dụng của bạn cần có tính năng tìm kiếm toàn bộ web, hãy [liên hệ với chúng tôi](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=vi) để bày tỏ sự quan tâm và tìm hiểu thêm thông tin về giải pháp tìm kiếm toàn bộ web của chúng tôi.
Khách hàng hiện tại của Custom Search JSON API có thời hạn đến **ngày 1 tháng 1 năm 2027** để chuyển sang một giải pháp thay thế.

Custom Search JSON API cho phép bạn phát triển các trang web và ứng dụng để truy xuất và hiển thị kết quả tìm kiếm từ Công cụ tìm kiếm có thể lập trình theo cách có lập trình. Với API này, bạn có thể sử dụng các yêu cầu RESTful để nhận kết quả **tìm kiếm trên web** hoặc **tìm kiếm hình ảnh** ở định dạng JSON.

Định dạng dữ liệu
-----------------

Custom Search JSON API có thể trả về kết quả ở định dạng dữ liệu [JSON](https://developers.google.com/custom-search/docs/glossary?hl=vi#json).

Tài liệu có liên quan
---------------------

Custom Search JSON API sử dụng [Quy cách OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Điều kiện tiên quyết
--------------------

### Mã công cụ tìm kiếm

Trước khi sử dụng API Tìm kiếm tuỳ chỉnh JSON, trước tiên bạn cần tạo và định cấu hình Công cụ tìm kiếm có thể lập trình. Nếu chưa tạo Công cụ tìm kiếm có thể lập trình, bạn có thể bắt đầu bằng cách truy cập vào [bảng điều khiển Công cụ tìm kiếm có thể lập trình](https://programmablesearchengine.google.com/controlpanel/all?hl=vi).

Hãy làm theo [hướng dẫn](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=vi) để tìm hiểu thêm về các lựa chọn cấu hình.

Sau khi tạo một Công cụ tìm kiếm có thể lập trình, hãy truy cập vào [trung tâm trợ giúp](https://support.google.com/programmable-search/answer/2649143?hl=vi) để tìm hiểu cách xác định Mã công cụ tìm kiếm.

### Khóa API

Custom Search JSON API yêu cầu bạn sử dụng khoá API.

Giá
---

Mức giá sau đây chỉ áp dụng cho những khách hàng hiện tại của Custom Search JSON API cho đến khi dịch vụ này ngừng hoạt động vào ngày 1 tháng 1 năm 2027. API này không dành cho khách hàng mới. Custom Search JSON API cung cấp miễn phí 100 lượt truy vấn tìm kiếm mỗi ngày. Nếu cần thêm, bạn có thể đăng ký [thanh toán](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=vi) trong Bảng điều khiển API. Các yêu cầu bổ sung có giá 5 USD cho mỗi 1.000 truy vấn, tối đa 10.000 truy vấn mỗi ngày.

Giám sát
--------

Bạn có thể theo dõi cơ bản API Tìm kiếm tuỳ chỉnh JSON thông qua [Trang tổng quan API của Bảng điều khiển Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=vi). Để giám sát nâng cao hơn, bạn có thể sử dụng [bộ công cụ vận hành của Google Cloud](https://cloud.google.com/products/operations?hl=vi) (trước đây là Stackdriver).

Với Google Cloud Operations, bạn có thể tạo [trang tổng quan tuỳ chỉnh](https://cloud.google.com/monitoring/charts?hl=vi), [thiết lập cảnh báo](https://cloud.google.com/monitoring/alerts?hl=vi) và [truy cập vào dữ liệu chỉ số theo phương thức lập trình](https://cloud.google.com/monitoring/api/v3?hl=vi). Để truy cập vào dữ liệu sử dụng Custom Search JSON API trong Google Cloud Operations, hãy chọn "Resource type: Consumed API" (Kiểu tài nguyên: API đã sử dụng) và lọc theo "service = 'customsearch.googleapis.com'" trong Trình tạo truy vấn.

Hãy xem phần [Theo dõi mức sử dụng API](https://cloud.google.com/apis/docs/monitoring?hl=vi) để biết thông tin thảo luận về các chức năng giám sát và cảnh báo do Trang tổng quan API và bộ công cụ Google Cloud Operations cung cấp.
