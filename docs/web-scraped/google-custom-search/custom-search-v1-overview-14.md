**توجه:**API جستجوی سفارشی JSON برای مشتریان جدید بسته است. [جستجوی هوش مصنوعی Vertex](https://cloud.google.com/use-cases/site-search?hl=fa) یک جایگزین مطلوب برای جستجوی حداکثر ۵۰ دامنه است. از طرف دیگر، اگر مورد استفاده شما نیاز به جستجوی کامل وب دارد، برای ابراز علاقه و کسب اطلاعات بیشتر در مورد راهکار جستجوی کامل وب ما [با ما تماس بگیرید](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=fa) . 
مشتریان فعلی API جستجوی سفارشی JSON تا **۱ ژانویه ۲۰۲۷** فرصت دارند تا به یک راهکار جایگزین منتقل شوند.

API جستجوی سفارشی JSON به شما امکان می‌دهد وب‌سایت‌ها و برنامه‌هایی را توسعه دهید که نتایج جستجو را از موتور جستجوی قابل برنامه‌ریزی (Plugable Search Engine) به صورت برنامه‌نویسی‌شده بازیابی و نمایش دهند. با استفاده از این API، می‌توانید از درخواست‌های RESTful برای دریافت نتایج **جستجوی وب** یا **جستجوی تصویر** در قالب JSON استفاده کنید.

قالب داده
---------

API جستجوی سفارشی JSON می‌تواند نتایج را در قالب داده [JSON](https://developers.google.com/custom-search/docs/glossary?hl=fa#json) برگرداند.

اسناد مرتبط
-----------

API جستجوی سفارشی JSON از [مشخصات OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md) استفاده می‌کند.

پیش‌نیازها
----------

### شناسه موتور جستجو

قبل از استفاده از API جستجوی سفارشی JSON، ابتدا باید موتور جستجوی قابل برنامه‌ریزی خود را ایجاد و پیکربندی کنید. اگر قبلاً موتور جستجوی قابل برنامه‌ریزی ایجاد نکرده‌اید، می‌توانید با مراجعه به [کنترل پنل موتور جستجوی قابل برنامه‌ریزی](https://programmablesearchengine.google.com/controlpanel/all?hl=fa) شروع کنید.

برای آشنایی بیشتر با گزینه‌های مختلف پیکربندی، [آموزش](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=fa) را دنبال کنید.

پس از ایجاد یک موتور جستجوی قابل برنامه‌ریزی، برای آشنایی با نحوه یافتن شناسه موتور جستجوی خود، به [مرکز راهنما](https://support.google.com/programmable-search/answer/2649143?hl=fa) مراجعه کنید.

### کلید API

جستجوی سفارشی JSON API نیاز به استفاده از کلید API دارد.

قیمت‌گذاری
----------

قیمت‌گذاری زیر فقط برای مشتریان فعلی Custom Search JSON API تا زمان قطع سرویس در ۱ ژانویه ۲۰۲۷ اعمال می‌شود. این API برای مشتریان جدید در دسترس نیست. Custom Search JSON API روزانه ۱۰۰ پرس‌وجوی جستجو را به صورت رایگان ارائه می‌دهد. اگر به تعداد بیشتری نیاز دارید، می‌توانید در کنسول API برای [دریافت صورتحساب](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=fa) ثبت‌نام کنید. هزینه درخواست‌های اضافی ۵ دلار برای هر ۱۰۰۰ پرس‌وجو و تا سقف ۱۰ هزار پرس‌وجو در روز است.

نظارت
-----

نظارت اولیه برای API جستجوی سفارشی JSON از طریق [داشبورد API کنسول پلتفرم ابری](https://console.cloud.google.com/apis/dashboard?hl=fa) در دسترس است. برای نظارت پیشرفته‌تر، [مجموعه عملیات گوگل کلود](https://cloud.google.com/products/operations?hl=fa) (که قبلاً Stackdriver نام داشت) در دسترس است.

با Google Cloud Operations می‌توانید [داشبوردهای سفارشی](https://cloud.google.com/monitoring/charts?hl=fa) ایجاد کنید، [هشدارها را تنظیم کنید](https://cloud.google.com/monitoring/alerts?hl=fa) و [به داده‌های معیارها از طریق برنامه‌نویسی دسترسی داشته باشید](https://cloud.google.com/monitoring/api/v3?hl=fa) . برای دسترسی به داده‌های استفاده از API جستجوی سفارشی JSON در Google Cloud Operations، "نوع منبع: API مصرفی" را انتخاب کنید و در Query Builder روی "service = 'customsearch.googleapis.com'" فیلتر کنید.

برای بحث در مورد قابلیت‌های مختلف نظارت و هشدار ارائه شده توسط داشبورد API و مجموعه عملیات ابری گوگل، به [بخش نظارت بر میزان استفاده از API خود](https://cloud.google.com/apis/docs/monitoring?hl=fa) مراجعه کنید.
