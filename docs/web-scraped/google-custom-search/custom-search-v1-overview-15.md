**ध्यान दें:** Custom Search JSON API, नए ग्राहकों के लिए उपलब्ध नहीं है. 50 डोमेन तक खोजने के लिए, [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=hi) एक बेहतर विकल्प है. इसके अलावा, अगर आपको पूरे वेब पर खोज करने की सुविधा का इस्तेमाल करना है, तो [हमसे संपर्क करें](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=hi). इससे हमें पता चलेगा कि आपको पूरे वेब पर खोज करने की सुविधा इस्तेमाल करनी है. साथ ही, आपको इस बारे में ज़्यादा जानकारी मिलेगी.
Custom Search JSON API के मौजूदा ग्राहकों के पास, किसी दूसरे समाधान पर माइग्रेट करने के लिए **1 जनवरी, 2027** तक का समय है.

Custom Search JSON API की मदद से, वेबसाइटें और ऐप्लिकेशन बनाए जा सकते हैं. इनकी मदद से, प्रोग्राम के हिसाब से Programmable Search Engine से खोज के नतीजे पाए जा सकते हैं और उन्हें दिखाया जा सकता है. इस एपीआई की मदद से, RESTful अनुरोधों का इस्तेमाल करके JSON फ़ॉर्मैट में **वेब पर खोज** या **इमेज खोज** के नतीजे पाए जा सकते हैं.

डेटा फ़ॉर्मैट
-------------

Custom Search JSON API, नतीजों को [JSON](https://developers.google.com/custom-search/docs/glossary?hl=hi#json) डेटा फ़ॉर्मैट में दिखा सकता है.

मिलते-जुलते दस्तावेज़
---------------------

Custom Search JSON API, [OpenSearch 1.1 स्पेसिफ़िकेशन](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md) का इस्तेमाल करता है.

ज़रूरी शर्तें
-------------

### सर्च इंजन का आईडी

Custom Search JSON API का इस्तेमाल करने से पहले, आपको अपना Programmable Search Engine बनाना और कॉन्फ़िगर करना होगा. अगर आपने अब तक Programmable Search Engine नहीं बनाया है, तो [Programmable Search Engine के कंट्रोल पैनल](https://programmablesearchengine.google.com/controlpanel/all?hl=hi) पर जाकर शुरू करें.

कॉन्फ़िगरेशन के अलग-अलग विकल्पों के बारे में ज़्यादा जानने के लिए, [ट्यूटोरियल](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=hi) देखें.

Programmable Search Engine बनाने के बाद, [सहायता केंद्र](https://support.google.com/programmable-search/answer/2649143?hl=hi) पर जाएं. यहां आपको सर्च इंजन आईडी ढूंढने का तरीका मिलेगा.

### एपीआई पासकोड

Custom Search JSON API का इस्तेमाल करने के लिए, एपीआई पासकोड की ज़रूरत होती है.

कीमत
----

यहां दी गई कीमतें, Custom Search JSON API के मौजूदा ग्राहकों के लिए ही लागू होंगी. ये कीमतें, 1 जनवरी, 2027 को सेवा बंद होने तक लागू रहेंगी. यह एपीआई, नए ग्राहकों के लिए उपलब्ध नहीं है. Custom Search JSON API में, हर दिन 100 खोज क्वेरी मुफ़्त में मिलती हैं. अगर आपको ज़्यादा कोटा चाहिए, तो API Console में [बिलिंग](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=hi) के लिए साइन अप करें. ज़्यादा अनुरोध करने पर, 1,000 क्वेरी के लिए पांच डॉलर का शुल्क लगता है. हालांकि, हर दिन ज़्यादा से ज़्यादा 10,000 क्वेरी की जा सकती हैं.

मॉनिटर करने के लिए
------------------

Custom Search JSON API की बुनियादी मॉनिटरिंग, [Cloud Platform Console के एपीआई डैशबोर्ड](https://console.cloud.google.com/apis/dashboard?hl=hi) के ज़रिए की जा सकती है. ज़्यादा बेहतर मॉनिटरिंग के लिए, [Google Cloud का Operations suite](https://cloud.google.com/products/operations?hl=hi) (पहले इसे Stackdriver कहा जाता था) उपलब्ध है.

Google Cloud Operations की मदद से, [कस्टम डैशबोर्ड](https://cloud.google.com/monitoring/charts?hl=hi) बनाए जा सकते हैं, [सूचनाएं सेट अप](https://cloud.google.com/monitoring/alerts?hl=hi) की जा सकती हैं, और [प्रोग्राम के हिसाब से मेट्रिक डेटा ऐक्सेस](https://cloud.google.com/monitoring/api/v3?hl=hi) किया जा सकता है. Google Cloud Operations में Custom Search JSON API के इस्तेमाल से जुड़ा डेटा ऐक्सेस करने के लिए, "संसाधन का टाइप: इस्तेमाल किया गया एपीआई" चुनें. इसके बाद, क्वेरी बिल्डर में "service = 'customsearch.googleapis.com'" पर फ़िल्टर करें.

एपीआई डैशबोर्ड और Google Cloud Operations Suite की निगरानी और सूचना देने की अलग-अलग सुविधाओं के बारे में जानने के लिए, [एपीआई के इस्तेमाल पर नज़र रखना](https://cloud.google.com/apis/docs/monitoring?hl=hi) लेख पढ़ें.
