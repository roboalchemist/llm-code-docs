**Not:** Custom Search JSON API, yeni müşterilere kapatılmıştır. 50'ye kadar alan adında arama yapmak için [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=tr)'ü kullanabilirsiniz. Alternatif olarak, kullanım alanınız tam web araması gerektiriyorsa tam web arama çözümümüzle ilgilendiğinizi belirtmek ve bu çözüm hakkında daha fazla bilgi edinmek için [bize ulaşın](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=tr).
Mevcut Custom Search JSON API müşterileri, alternatif bir çözüme geçmek için **1 Ocak 2027**'ye kadar süreye sahiptir.

Custom Search JSON API, Programlanabilir Arama Motoru'ndaki arama sonuçlarını programatik olarak almak ve görüntülemek için web siteleri ve uygulamalar geliştirmenize olanak tanır. Bu API ile, JSON biçiminde **web arama** veya **görsel arama** sonuçları almak için RESTful istekleri kullanabilirsiniz.

Veri biçimi
-----------

Custom Search JSON API, sonuçları [JSON](https://developers.google.com/custom-search/docs/glossary?hl=tr#json) veri biçiminde döndürebilir.

İlgili belgeler
---------------

Custom Search JSON API, [OpenSearch 1.1 Spesifikasyonu](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)'nu kullanır.

Ön koşullar
-----------

### Arama motoru kimliği

Custom Search JSON API'yi kullanmadan önce Programlanabilir Arama Motorunuzu oluşturup yapılandırmanız gerekir. Henüz bir Programlanabilir Arama Motoru oluşturmadıysanız [Programlanabilir Arama Motoru denetim masasını](https://programmablesearchengine.google.com/controlpanel/all?hl=tr) ziyaret ederek başlayabilirsiniz.

Farklı yapılandırma seçenekleri hakkında daha fazla bilgi edinmek için [eğitimi](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=tr) inceleyin.

Programlanabilir Arama Motoru oluşturduktan sonra, arama motoru kimliğinizi nasıl bulacağınızı öğrenmek için [Yardım Merkezi](https://support.google.com/programmable-search/answer/2649143?hl=tr)'ni ziyaret edin.

### API anahtarı

Custom Search JSON API'nin kullanılabilmesi için API anahtarı gerekir.

Fiyatlandırma
-------------

Aşağıdaki fiyatlandırma, yalnızca mevcut Custom Search JSON API müşterileri için 1 Ocak 2027'de hizmetin sonlandırılmasına kadar geçerlidir. Bu API yeni müşteriler tarafından kullanılamaz. Custom Search JSON API, günde 100 arama sorgusunu ücretsiz olarak sağlar. Daha fazla kotaya ihtiyacınız varsa API Konsolu'ndan [faturalandırmaya](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=tr) kaydolabilirsiniz. Ek istekler, 1.000 sorgu başına 5 ABD doları tutarındadır ve günlük 10.000 sorgu ile sınırlıdır.

İzleme
------

Custom Search JSON API için temel izleme, [Cloud Platform Console'un API Kontrol Paneli](https://console.cloud.google.com/apis/dashboard?hl=tr) üzerinden kullanılabilir. Daha gelişmiş izleme için [Google Cloud'un işlem paketi](https://cloud.google.com/products/operations?hl=tr) (eski adıyla Stackdriver) kullanılabilir.

Google Cloud Operations ile [özel kontrol panelleri](https://cloud.google.com/monitoring/charts?hl=tr) oluşturabilir, [uyarılar ayarlayabilir](https://cloud.google.com/monitoring/alerts?hl=tr) ve [metrik verilerine programatik olarak erişebilirsiniz](https://cloud.google.com/monitoring/api/v3?hl=tr). Google Cloud Operations'da Custom Search JSON API kullanım verilerine erişmek için Sorgu Oluşturucu'da "Kaynak türü: Tüketilen API"yi seçin ve "service = 'customsearch.googleapis.com'" ifadesine göre filtreleyin.

API kontrol paneli ve Google Cloud işlem paketi tarafından sağlanan farklı izleme ve uyarı özellikleriyle ilgili tartışma için [API Kullanımınızı İzleme](https://cloud.google.com/apis/docs/monitoring?hl=tr) başlıklı makaleye bakın.
