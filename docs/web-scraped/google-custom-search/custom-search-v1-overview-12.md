**הערה:** ‏ Custom Search JSON API סגור ללקוחות חדשים. ‫[Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=he) הוא חלופה מומלצת לחיפוש של עד 50 דומיינים. לחלופין, אם תרחיש השימוש מחייב חיפוש מלא באינטרנט, אפשר [ליצור איתנו קשר](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=he) כדי לקבל מידע נוסף על הפתרון שלנו לחיפוש מלא באינטרנט.
לקוחות קיימים של Custom Search JSON API צריכים לעבור לפתרון חלופי עד **1 בינואר 2027**.

‫Custom Search JSON API מאפשר לכם לפתח אתרים ואפליקציות כדי לאחזר ולהציג תוצאות חיפוש מ-Programmable Search Engine באופן פרוגרמטי. באמצעות ה-API הזה, אפשר להשתמש בבקשות RESTful כדי לקבל תוצאות של **חיפוש באינטרנט** או **חיפוש תמונות** בפורמט JSON.

פורמט נתונים
------------

‫Custom Search JSON API יכול להחזיר תוצאות בפורמט נתונים של [JSON](https://developers.google.com/custom-search/docs/glossary?hl=he#json).

מסמכים קשורים
-------------

‫Custom Search JSON API משתמש ב[מפרט OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

דרישות מוקדמות
--------------

### מזהה מנוע החיפוש

לפני שמשתמשים ב-Custom Search JSON API, צריך קודם ליצור ולהגדיר את Programmable Search Engine. אם עדיין לא יצרתם Programmable Search Engine, תוכלו להתחיל בכניסה ל[לוח הבקרה של Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all?hl=he).

[במדריך](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=he) מוסבר על אפשרויות ההגדרה השונות.

אחרי שיוצרים מנוע חיפוש באמצעות Programmable Search Engine, אפשר להיכנס ל[מרכז העזרה](https://support.google.com/programmable-search/answer/2649143?hl=he) כדי ללמוד איך למצוא את מזהה מנוע החיפוש.

### מפתח API

כדי להשתמש ב-Custom Search JSON API, צריך להשתמש במפתח API.

תמחור
-----

התמחור הבא רלוונטי רק ללקוחות קיימים של Custom Search JSON API עד להפסקת השירות ב-1 בינואר 2027. ה-API הזה לא זמין ללקוחות חדשים. ‫Custom Search JSON API מספק 100 שאילתות חיפוש ביום בחינם. אם אתם צריכים יותר, אתם יכולים להירשם ל[חיוב](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=he) במסוף לממשקי API. עלות בקשות נוספות היא 5 $לכל 1,000 שאילתות, עד 10,000 שאילתות ביום.

מעקב
----

אפשר לבצע ניטור בסיסי של Custom Search JSON API באמצעות [מרכז הבקרה של ממשקי ה-API במסוף Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=he). לצורך מעקב מתקדם יותר, אפשר להשתמש ב-[Google Cloud's Operations Suite](https://cloud.google.com/products/operations?hl=he) (לשעבר Stackdriver).

בעזרת Google Cloud Operations אפשר ליצור [מרכזי בקרה בהתאמה אישית](https://cloud.google.com/monitoring/charts?hl=he), [להגדיר התראות](https://cloud.google.com/monitoring/alerts?hl=he) ו[לגשת לנתוני מדדים באופן פרוגרמטי](https://cloud.google.com/monitoring/api/v3?hl=he). כדי לגשת לנתוני השימוש ב-Custom Search JSON API ב-Google Cloud Operations, בוחרים באפשרות Resource type: Consumed API (סוג המשאב: API שנצרך) ומסננים לפי service = 'customsearch.googleapis.com' (שירות = 'customsearch.googleapis.com') ב-Query Builder (בונה השאילתות).

במאמר [מעקב אחר השימוש ב-API](https://cloud.google.com/apis/docs/monitoring?hl=he) מוסבר על יכולות המעקב וההתראות השונות שזמינות בלוח הבקרה של ה-API ובחבילת Google Cloud Operations.
