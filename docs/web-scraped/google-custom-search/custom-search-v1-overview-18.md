**注意**： Custom Search JSON API 不再向新客户开放。[Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=zh-cn) 是搜索最多 50 个网域的理想替代方案。或者，如果您的应用场景需要进行完整的网页搜索，请[与我们联系](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=zh-cn)，说明您有兴趣使用我们的完整网页搜索解决方案并获取更多相关信息。
现有 Custom Search JSON API 客户必须在 **2027 年 1 月 1 日**之前改用替代方案。

借助 Custom Search JSON API，您可以开发网站和应用，以通过编程方式检索和显示可编程搜索引擎的搜索结果。借助此 API，您可以使用 RESTful 请求以 JSON 格式获取**网页搜索**或**图片搜索**结果。

数据格式
----

Custom Search JSON API 可以返回 [JSON](https://developers.google.com/custom-search/docs/glossary?hl=zh-cn#json) 数据格式的结果。

相关文档
----

Custom Search JSON API 使用 [OpenSearch 1.1 规范](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)。

前提条件
----

### 搜索引擎 ID

在使用 Custom Search JSON API 之前，您需要先创建并配置可编程搜索引擎。如果您尚未创建可编程搜索引擎，可以先访问[可编程搜索引擎控制面板](https://programmablesearchengine.google.com/controlpanel/all?hl=zh-cn)。

请按照[教程](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=zh-cn)详细了解不同的配置选项。

创建可编程搜索引擎后，请访问[帮助中心](https://support.google.com/programmable-search/answer/2649143?hl=zh-cn)，了解如何查找搜索引擎 ID。

### API 密钥

Custom Search JSON API 需要使用 API 密钥。

价格
--

以下价格仅适用于现有的 Custom Search JSON API 客户，直至该服务于 2027 年 1 月 1 日停止。此 API 不适用于新客户。 Custom Search JSON API 每天免费提供 100 次搜索查询。如果您需要更多，可以在 API 控制台中申请[账单](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=zh-cn)。额外请求的费用为每天每 1000 次至 1 万次查询 5 美元。

监控
--

您可以通过 [Cloud Platform Console 的 API 信息中心](https://console.cloud.google.com/apis/dashboard?hl=zh-cn)对 Custom Search JSON API 进行基本监控。如需更高级的监控，可以使用 [Google Cloud 的运维套件](https://cloud.google.com/products/operations?hl=zh-cn)（以前称为 Stackdriver）。

借助 Google Cloud Operations，您可以创建[自定义信息中心](https://cloud.google.com/monitoring/charts?hl=zh-cn)、[设置提醒](https://cloud.google.com/monitoring/alerts?hl=zh-cn)，并[以编程方式访问指标数据](https://cloud.google.com/monitoring/api/v3?hl=zh-cn)。如需在 Google Cloud Operations 中访问 Custom Search JSON API 使用情况数据，请在查询构建器中选择“资源类型：已使用的 API”，并按“service = 'customsearch.googleapis.com'”进行过滤。

如需了解 API 信息中心和 Google Cloud Operations 套件提供的各种监控和提醒功能，请参阅[监控 API 用量](https://cloud.google.com/apis/docs/monitoring?hl=zh-cn)。
