**Hinweis**: Die Custom Search JSON API ist für neue Kunden nicht mehr verfügbar. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=de) ist eine gute Alternative für die Suche in bis zu 50 Domains. Wenn Sie die vollständige Websuche benötigen, [kontaktieren Sie uns](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=de), um Ihr Interesse an unserer Lösung für die vollständige Websuche zu äußern und weitere Informationen zu erhalten.
Bestehende Kunden der Custom Search JSON API haben bis zum **1.Januar 2027** Zeit, auf eine alternative Lösung umzustellen.

Mit der Custom Search JSON API können Sie Websites und Anwendungen entwickeln, um Suchergebnisse aus der Programmable Search Engine programmatisch abzurufen und anzuzeigen. Mit dieser API können Sie RESTful-Anfragen verwenden, um **Websuche**- oder **Bildersuche**-Ergebnisse im JSON-Format abzurufen.

Datenformat
-----------

Die Custom Search JSON API kann Ergebnisse im [JSON](https://developers.google.com/custom-search/docs/glossary?hl=de#json)-Datenformat zurückgeben.

Zugehörige Dokumente
--------------------

Die Custom Search JSON API verwendet die [OpenSearch 1.1-Spezifikation](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Vorbereitung
------------

### Suchmaschinen-ID

Bevor Sie die Custom Search JSON API verwenden können, müssen Sie zuerst Ihre Programmable Search Engine erstellen und konfigurieren. Falls Sie noch keine Programmable Search Engine erstellt haben, können Sie das [Steuerfeld der Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all?hl=de) aufrufen.

[In dieser Anleitung](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=de) finden Sie weitere Informationen zu verschiedenen Konfigurationsoptionen.

Nachdem Sie eine Programmable Search Engine erstellt haben, können Sie in der [Hilfe](https://support.google.com/programmable-search/answer/2649143?hl=de) nachlesen, wie Sie die Suchmaschinen-ID finden.

### API-Schlüssel

Für die Custom Search JSON API ist ein API-Schlüssel erforderlich.

Preise
------

Die folgenden Preise gelten nur für bestehende Custom Search JSON API-Kunden bis zur Einstellung des Dienstes am 1.Januar 2027. Diese API ist für Neukunden nicht verfügbar. Mit der Custom Search JSON API können Sie pro Tag 100 Suchanfragen kostenlos durchführen. Wenn Sie mehr benötigen, können Sie sich in der API Console für die [Abrechnung](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=de) registrieren. Zusätzliche Anfragen kosten 5$pro 1.000 Anfragen. Pro Tag sind bis zu 10.000 Anfragen möglich.

Monitoring
----------

Das [API-Dashboard der Cloud Platform Console](https://console.cloud.google.com/apis/dashboard?hl=de) bietet grundlegende Monitoring-Funktionen für die Custom Search JSON API. Für ein erweitertes Monitoring ist die [Operations-Suite von Google Cloud](https://cloud.google.com/products/operations?hl=de) (ehemals Stackdriver) verfügbar.

Mit Google Cloud Operations können Sie [benutzerdefinierte Dashboards](https://cloud.google.com/monitoring/charts?hl=de) erstellen, [Benachrichtigungen einrichten](https://cloud.google.com/monitoring/alerts?hl=de) und [programmatisch auf Messwertdaten zugreifen](https://cloud.google.com/monitoring/api/v3?hl=de). Wenn Sie in Google Cloud Operations auf Nutzungsdaten der Custom Search JSON API zugreifen möchten, wählen Sie im Query Builder „Ressourcentyp: Verwendete API“ aus und filtern Sie nach „service = 'customsearch.googleapis.com'“.

Unter [API-Nutzung überwachen](https://cloud.google.com/apis/docs/monitoring?hl=de) finden Sie eine Beschreibung der verschiedenen Überwachungs- und Benachrichtigungsfunktionen, die vom API-Dashboard und der Google Cloud Operations Suite bereitgestellt werden.
