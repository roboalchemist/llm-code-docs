**참고:** Custom Search JSON API는 신규 고객에게 제공되지 않습니다. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=ko)는 최대 50개의 도메인을 검색하는 데 적합한 대안입니다. 또는 사용 사례에 전체 웹 검색이 필요한 경우 [Google에 문의](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=ko)하여 전체 웹 검색 솔루션에 대한 관심을 표현하고 자세한 정보를 알아보세요.
기존 Custom Search JSON API 고객은 **2027년 1월 1일**까지 대체 솔루션으로 전환해야 합니다.

Custom Search JSON API를 사용하면 프로그래밍 방식으로 프로그래밍 검색 엔진에서 검색 결과를 가져와 표시하는 웹사이트와 애플리케이션을 개발할 수 있습니다. 이 API를 사용하면 RESTful 요청을 통해 JSON 형식으로 **웹 검색** 또는 **이미지 검색** 결과를 가져올 수 있습니다.

데이터 형식
------

Custom Search JSON API는 [JSON](https://developers.google.com/custom-search/docs/glossary?hl=ko#json) 데이터 형식으로 결과를 반환할 수 있습니다.

관련 문서
-----

Custom Search JSON API는 [OpenSearch 1.1 사양](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md)을 사용합니다.

기본 요건
-----

### 검색엔진 ID

Custom Search JSON API를 사용하려면 먼저 프로그래밍 검색 엔진을 만들고 구성해야 합니다. 아직 프로그래밍 검색 엔진을 만들지 않은 경우 [프로그래밍 검색 엔진 제어판](https://programmablesearchengine.google.com/controlpanel/all?hl=ko)을 방문하여 시작할 수 있습니다.

[튜토리얼](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=ko)을 따라 다양한 구성 옵션에 대해 자세히 알아보세요.

프로그래밍 검색 엔진을 만든 후 [고객센터](https://support.google.com/programmable-search/answer/2649143?hl=ko)를 방문하여 검색엔진 ID를 찾는 방법을 알아보세요.

### API 키

Custom Search JSON API를 사용하려면 API 키를 사용해야 합니다.

가격 책정
-----

다음 가격은 2027년 1월 1일 서비스 중단까지 기존 Custom Search JSON API 고객에게만 적용됩니다. 신규 고객은 이 API를 사용할 수 없습니다. Custom Search JSON API는 일일 100개의 검색어를 무료로 제공합니다. 더 많은 할당량이 필요한 경우 API 콘솔에서 [결제](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=ko)를 신청할 수 있습니다. 추가 요청은 1,000개 질문당 5달러이며, 일일 최대 10,000개 질문까지 가능합니다.

모니터링
----

[Cloud Platform Console의 API 대시보드](https://console.cloud.google.com/apis/dashboard?hl=ko)를 통해 Custom Search JSON API의 기본 모니터링을 사용할 수 있습니다. 고급 모니터링에는 [Google Cloud 운영 제품군](https://cloud.google.com/products/operations?hl=ko) (이전 명칭: Stackdriver)을 사용할 수 있습니다.

Google Cloud Operations를 사용하면 [맞춤 대시보드](https://cloud.google.com/monitoring/charts?hl=ko)를 만들고, [알림을 설정](https://cloud.google.com/monitoring/alerts?hl=ko)하고, [프로그래매틱 방식으로 측정항목 데이터에 액세스](https://cloud.google.com/monitoring/api/v3?hl=ko)할 수 있습니다. Google Cloud Operations에서 Custom Search JSON API 사용 데이터에 액세스하려면 '리소스 유형: 사용된 API'를 선택하고 쿼리 빌더에서 'service = 'customsearch.googleapis.com''을 기준으로 필터링합니다.

API 대시보드와 Google Cloud Operations 제품군에서 제공하는 다양한 모니터링 및 알림 기능에 대한 설명은 [API 사용량 모니터링](https://cloud.google.com/apis/docs/monitoring?hl=ko)을 참고하세요.
