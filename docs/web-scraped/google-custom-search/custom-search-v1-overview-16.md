**দ্রষ্টব্য:**কাস্টম সার্চ JSON API নতুন গ্রাহকদের জন্য বন্ধ। [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=bn) হল 50টি ডোমেন পর্যন্ত অনুসন্ধানের জন্য একটি অনুকূল বিকল্প। বিকল্পভাবে, যদি আপনার ব্যবহারের ক্ষেত্রে সম্পূর্ণ ওয়েব অনুসন্ধানের প্রয়োজন হয়, তাহলে আপনার আগ্রহ প্রকাশ করতে এবং আমাদের সম্পূর্ণ ওয়েব অনুসন্ধান সমাধান সম্পর্কে আরও তথ্য পেতে [আমাদের সাথে যোগাযোগ করুন](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=bn) । 
বিদ্যমান কাস্টম সার্চ JSON API গ্রাহকরা **1 জানুয়ারী, 2027** পর্যন্ত বিকল্প সমাধানে রূপান্তর করতে পারবেন।

কাস্টম সার্চ JSON API আপনাকে প্রোগ্রামেবল সার্চ ইঞ্জিন থেকে প্রোগ্রাম্যাটিকভাবে অনুসন্ধান ফলাফল পুনরুদ্ধার এবং প্রদর্শন করার জন্য ওয়েবসাইট এবং অ্যাপ্লিকেশন তৈরি করতে দেয়। এই API এর সাহায্যে, আপনি JSON ফর্ম্যাটে **ওয়েব অনুসন্ধান** বা **চিত্র অনুসন্ধান** ফলাফল পেতে RESTful অনুরোধগুলি ব্যবহার করতে পারেন।

পূর্বশর্ত
---------

### সার্চ ইঞ্জিন আইডি

কাস্টম সার্চ JSON API ব্যবহার করার আগে আপনাকে প্রথমে আপনার প্রোগ্রামেবল সার্চ ইঞ্জিন তৈরি এবং কনফিগার করতে হবে। যদি আপনি ইতিমধ্যে একটি প্রোগ্রামেবল সার্চ ইঞ্জিন তৈরি না করে থাকেন, তাহলে আপনি [প্রোগ্রামেবল সার্চ ইঞ্জিন কন্ট্রোল প্যানেলে](https://programmablesearchengine.google.com/controlpanel/all?hl=bn) গিয়ে শুরু করতে পারেন।

বিভিন্ন কনফিগারেশন বিকল্প সম্পর্কে আরও জানতে [টিউটোরিয়ালটি](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=bn) অনুসরণ করুন।

একটি প্রোগ্রামেবল সার্চ ইঞ্জিন তৈরি করার পরে, আপনার সার্চ ইঞ্জিন আইডি কীভাবে খুঁজে পাবেন তা জানতে [সহায়তা কেন্দ্রে](https://support.google.com/programmable-search/answer/2649143?hl=bn) যান।

### API কী

কাস্টম সার্চ JSON API-এর জন্য একটি API কী ব্যবহার করা প্রয়োজন।

মূল্য নির্ধারণ
--------------

নিম্নলিখিত মূল্য শুধুমাত্র বিদ্যমান কাস্টম সার্চ JSON API গ্রাহকদের জন্য প্রযোজ্য, যতক্ষণ না ১ জানুয়ারী, ২০২৭ তারিখে পরিষেবা বন্ধ হয়ে যায়। এই API নতুন গ্রাহকদের জন্য উপলব্ধ নয়। কাস্টম সার্চ JSON API প্রতিদিন ১০০টি সার্চ কোয়েরি বিনামূল্যে প্রদান করে। আপনার যদি আরও বেশি প্রয়োজন হয়, তাহলে আপনি API কনসোলে [বিলিংয়ের](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=bn) জন্য সাইন আপ করতে পারেন। অতিরিক্ত অনুরোধের জন্য প্রতি ১০০০ কোয়েরিতে $৫ খরচ হয়, প্রতিদিন ১০,০০০ কোয়েরি পর্যন্ত।

পর্যবেক্ষণ
----------

কাস্টম সার্চ JSON API-এর জন্য মৌলিক পর্যবেক্ষণ [ক্লাউড প্ল্যাটফর্ম কনসোলের API ড্যাশবোর্ডের](https://console.cloud.google.com/apis/dashboard?hl=bn) মাধ্যমে উপলব্ধ। আরও উন্নত পর্যবেক্ষণের জন্য [গুগল ক্লাউডের অপারেশন স্যুট](https://cloud.google.com/products/operations?hl=bn) (পূর্বে স্ট্যাকড্রাইভার) উপলব্ধ।

গুগল ক্লাউড অপারেশনের মাধ্যমে আপনি [কাস্টম ড্যাশবোর্ড](https://cloud.google.com/monitoring/charts?hl=bn) তৈরি করতে, [সতর্কতা সেট আপ করতে](https://cloud.google.com/monitoring/alerts?hl=bn) এবং [প্রোগ্রাম্যাটিকভাবে মেট্রিক্স ডেটা অ্যাক্সেস করতে](https://cloud.google.com/monitoring/api/v3?hl=bn) পারেন। গুগল ক্লাউড অপারেশনে কাস্টম সার্চ JSON API ব্যবহারের ডেটা অ্যাক্সেস করতে, "রিসোর্স টাইপ: কনজুমড API" নির্বাচন করুন এবং কোয়েরি বিল্ডারে "service = 'customsearch.googleapis.com'" ফিল্টার করুন।

API ড্যাশবোর্ড এবং Google ক্লাউড অপারেশনস স্যুট দ্বারা প্রদত্ত বিভিন্ন পর্যবেক্ষণ এবং সতর্কতা ক্ষমতা সম্পর্কে আলোচনার জন্য [আপনার API ব্যবহার পর্যবেক্ষণ](https://cloud.google.com/apis/docs/monitoring?hl=bn) দেখুন।
