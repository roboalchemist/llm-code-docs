**Nota**: l'API Custom Search JSON non è disponibile per i nuovi clienti. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=it) è un'alternativa vantaggiosa per la ricerca in un massimo di 50 domini. In alternativa, se il tuo caso d'uso richiede la ricerca web completa, [contattaci](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=it) per comunicare il tuo interesse e ricevere maggiori informazioni sulla nostra soluzione di ricerca web completa.
I clienti esistenti dell'API Custom Search JSON hanno tempo fino al **1° gennaio 2027** per passare a una soluzione alternativa.

L'API Custom Search JSON ti consente di sviluppare siti web e applicazioni per recuperare e visualizzare i risultati di ricerca del Motore di ricerca programmabile in modo programmatico. Con questa API, puoi utilizzare richieste RESTful per ottenere risultati di **ricerca web** o **ricerca immagini** in formato JSON.

Formato dei dati
----------------

L'API Custom Search JSON può restituire risultati in formato dati [JSON](https://developers.google.com/custom-search/docs/glossary?hl=it#json).

Documenti correlati
-------------------

L'API Custom Search JSON utilizza la [specifica OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Prerequisiti
------------

### ID motore di ricerca

Prima di utilizzare l'API Custom Search JSON, devi creare e configurare il tuo Motore di ricerca programmabile. Se non hai ancora creato un Motore di ricerca programmabile, puoi iniziare visitando il [pannello di controllo del Motore di ricerca programmabile](https://programmablesearchengine.google.com/controlpanel/all?hl=it).

Segui il [tutorial](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=it) per scoprire di più sulle diverse opzioni di configurazione.

Dopo aver creato un motore di ricerca programmabile, visita il [Centro assistenza](https://support.google.com/programmable-search/answer/2649143?hl=it) per scoprire come trovare l'ID motore di ricerca.

### Chiave API

L'API Custom Search JSON richiede l'utilizzo di una chiave API.

Prezzi
------

I seguenti prezzi si applicano solo ai clienti esistenti dell'API Custom Search JSON fino alla chiusura del servizio il 1° gennaio 2027. Questa API non è disponibile per i nuovi clienti. L'API Custom Search JSON fornisce 100 query di ricerca al giorno senza costi. Se hai bisogno di più spazio, puoi registrarti per la [fatturazione](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=it) nella console API. Le richieste aggiuntive costano 5 $per 1000 query, fino a 10.000 query al giorno.

Monitoraggio
------------

Il monitoraggio di base per l'API Custom Search JSON è disponibile tramite la [dashboard API della console di Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=it). Per un monitoraggio più avanzato, è disponibile la [suite operativa di Google Cloud](https://cloud.google.com/products/operations?hl=it) (precedentemente Stackdriver).

Con Google Cloud Operations puoi creare [dashboard personalizzate](https://cloud.google.com/monitoring/charts?hl=it), [configurare avvisi](https://cloud.google.com/monitoring/alerts?hl=it) e [accedere ai dati delle metriche in modo programmatico](https://cloud.google.com/monitoring/api/v3?hl=it). Per accedere ai dati di utilizzo dell'API Custom Search JSON in Google Cloud Operations, seleziona "Tipo di risorsa: API utilizzata" e filtra in base a "service = 'customsearch.googleapis.com'" nel Query Builder.

Consulta [Monitoraggio dell'utilizzo delle API](https://cloud.google.com/apis/docs/monitoring?hl=it) per una discussione sulle diverse funzionalità di monitoraggio e avviso fornite dalla dashboard delle API e dalla suite operativa Google Cloud.
