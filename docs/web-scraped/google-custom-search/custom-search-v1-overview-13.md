**ملاحظة:** تم إغلاق واجهة Custom Search JSON API أمام العملاء الجدد. [‫Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=ar) هو بديل مناسب للبحث في ما يصل إلى 50 نطاقًا. بدلاً من ذلك، إذا كانت حالة الاستخدام تتطلب البحث في الويب بالكامل، يُرجى [التواصل معنا](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=ar) للتعبير عن اهتمامك بحلّ البحث في الويب بالكامل والحصول على مزيد من المعلومات عنه.
أمام عملاء Custom Search JSON API الحاليين مهلة حتى **1 يناير 2027** للانتقال إلى حلّ بديل.

تتيح لك واجهة Custom Search JSON API تطوير مواقع إلكترونية وتطبيقات لاسترداد نتائج البحث وعرضها من برنامج &quot;محرك البحث المبرمَج&quot; آليًا. باستخدام واجهة برمجة التطبيقات هذه، يمكنك استخدام طلبات RESTful للحصول على نتائج **بحث الويب** أو **بحث الصور** بتنسيق JSON.

تنسيق البيانات
--------------

يمكن أن تعرض واجهة Custom Search JSON API النتائج بتنسيق بيانات [JSON](https://developers.google.com/custom-search/docs/glossary?hl=ar#json).

-------------------

المتطلبات الأساسية
------------------

### معرّف محرّك البحث

قبل استخدام واجهة Custom Search JSON API، عليك أولاً إنشاء &quot;محرك البحث المبرمَج&quot; وضبطه. إذا لم يسبق لك إنشاء &quot;محرك بحث مبرمَج&quot;، يمكنك البدء بالانتقال إلى [لوحة التحكّم في &quot;محرك البحث المبرمَج&quot;](https://programmablesearchengine.google.com/controlpanel/all?hl=ar).

اتّبِع [البرنامج التعليمي](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=ar) لمعرفة المزيد عن خيارات الإعداد المختلفة.

بعد إنشاء "محرك بحث مبرمَج"، انتقِل إلى [مركز المساعدة](https://support.google.com/programmable-search/answer/2649143?hl=ar) للتعرّف على كيفية العثور على معرّف محرك البحث.

### مفتاح واجهة برمجة التطبيقات

تتطلّب واجهة Custom Search JSON API استخدام مفتاح واجهة برمجة التطبيقات.

الأسعار
-------

لا ينطبق السعر التالي إلا على عملاء واجهة Custom Search JSON API الحاليين إلى حين إيقاف الخدمة نهائيًا في 1 يناير 2027. لا تتوفّر واجهة برمجة التطبيقات هذه للعملاء الجدد. توفّر واجهة Custom Search JSON API‏ 100 طلب بحث يوميًا مجانًا. إذا كنت بحاجة إلى المزيد، يمكنك الاشتراك في [الفوترة](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=ar) في API Console. تبلغ تكلفة الطلبات الإضافية 5 دولار أمريكي لكل 1,000 طلب بحث، وبحد أقصى 10,000 طلب بحث في اليوم.

التتبّع
-------

تتوفّر إمكانية المراقبة الأساسية لواجهة Custom Search JSON API من خلال [لوحة بيانات واجهة برمجة التطبيقات في Cloud Platform Console](https://console.cloud.google.com/apis/dashboard?hl=ar). للحصول على ميزات مراقبة أكثر تقدّمًا، تتوفّر [حزمة عمليات Google Cloud](https://cloud.google.com/products/operations?hl=ar) (المعروفة سابقًا باسم Stackdriver).

باستخدام Google Cloud Operations، يمكنك إنشاء [لوحات بيانات مخصّصة](https://cloud.google.com/monitoring/charts?hl=ar) و[إعداد تنبيهات](https://cloud.google.com/monitoring/alerts?hl=ar) و[الوصول إلى بيانات المقاييس آليًا](https://cloud.google.com/monitoring/api/v3?hl=ar). للوصول إلى بيانات استخدام Custom Search JSON API في Google Cloud Operations، اختَر "نوع المورد: واجهة برمجة التطبيقات المستخدَمة" (Resource type: Consumed API) وفلتر حسب "الخدمة = 'customsearch.googleapis.com'" (service = 'customsearch.googleapis.com') في "أداة إنشاء طلبات البحث" (Query Builder).

راجِع مقالة [مراقبة استخدام واجهة برمجة التطبيقات](https://cloud.google.com/apis/docs/monitoring?hl=ar) للاطّلاع على مناقشة حول إمكانات المراقبة والتنبيه المختلفة التي توفّرها لوحة بيانات واجهة برمجة التطبيقات ومجموعة أدوات Google Cloud Operations.
