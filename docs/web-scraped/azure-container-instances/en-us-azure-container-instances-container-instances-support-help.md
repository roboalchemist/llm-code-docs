Self help troubleshooting
-------------------------

![Image 11: Self help content](https://learn.microsoft.com/en-us/azure/container-instances/media/logos/doc-logo.png)

See a list of [common issues in Azure Container instances](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-troubleshooting) and how to resolve them.

[](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-support-help#post-a-question-on-microsoft-qa)
Post a question on Microsoft Q&A
--------------------------------

![Image 12: Microsoft Q&A](https://learn.microsoft.com/en-us/azure/container-instances/media/logos/microsoft-logo.png)

For quick and reliable answers on your technical product questions from Microsoft Engineers, Azure Most Valuable Professionals (MVPs), or our expert community, engage with us on [Microsoft Q&A](https://learn.microsoft.com/en-us/answers/products/azure), Azure’s preferred destination for community support.

If you can't find an answer to your problem using search, submit a new question to Microsoft Q&A using the tag [**azure-container-instances**](https://learn.microsoft.com/en-us/answers/topics/azure-container-instances.html).

[](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-support-help#create-an-azure-support-request)
Create an Azure support request
-------------------------------

![Image 13: Azure support](https://learn.microsoft.com/en-us/azure/container-instances/media/logos/azure-logo.png)

Explore the range of [Azure support options and choose the plan](https://azure.microsoft.com/support/plans) that best fits, whether you're a developer just starting your cloud journey or a large organization deploying business-critical, strategic applications. Azure customers can create and manage support requests in the Azure portal.

* If you already have an Azure Support Plan, [open a support request here](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportrequest).

* To sign up for a new Azure Support Plan, [compare support plans](https://azure.microsoft.com/support/plans/) and select the plan that works for you.

[](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-support-help#check-the-stack-overflow-forum)
Check the Stack Overflow forum
------------------------------

![Image 14: Stack Overflow](https://learn.microsoft.com/en-us/azure/container-instances/media/logos/stack-overflow-logo.png)

The [**azure-container-instances**](https://stackoverflow.com/questions/tagged/azure-container-instances) tag on [Stack Overflow](https://stackoverflow.com/) is used for asking general questions about how the platform works and how you may use it to accomplish certain tasks.

[](https://learn.microsoft.com/en-us/azure/container-instances/container-instances-support-help#create-a-github-issue)

---------------------

![Image 15: GitHub-image](https://learn.microsoft.com/en-us/azure/container-instances/media/logos/github-logo.png)

If you need help with the language and tools used to develop and manage Azure Container Instances, open an issue in its repository on GitHub.
