Welcome to OpenTSDB 2.4, the scalable, distributed time series database. We recommend that you start with the [User Guide](https://opentsdb.net/docs/build/html/user_guide/index.html) then test your understanding with an [Installation](https://opentsdb.net/docs/build/html/installation.html) and read on the [HTTP API](https://opentsdb.net/docs/build/html/api_http/index.html) if you need to develop against it.
