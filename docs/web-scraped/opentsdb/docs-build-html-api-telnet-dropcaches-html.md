dropcaches[¶](https://opentsdb.net/docs/build/html/api_telnet/dropcaches.html#dropcaches "Permalink to this heading")
=====================================================================================================================

Purges the metric, tag key and tag value UID to string and string to UID maps.

Request[¶](https://opentsdb.net/docs/build/html/api_telnet/dropcaches.html#request "Permalink to this heading")
---------------------------------------------------------------------------------------------------------------

The command format is:

dropcaches

Response[¶](https://opentsdb.net/docs/build/html/api_telnet/dropcaches.html#response "Permalink to this heading")
-----------------------------------------------------------------------------------------------------------------

An acknowledgement after the caches have been purged.
