### Quick search

help[¶](https://opentsdb.net/docs/build/html/api_telnet/help.html#help "Permalink to this heading")
===================================================================================================

Returns a list of the commands supported via the Telnet style API. This command does not modify TSD in any way.

Request[¶](https://opentsdb.net/docs/build/html/api_telnet/help.html#request "Permalink to this heading")
---------------------------------------------------------------------------------------------------------

The command format is:

help

Response[¶](https://opentsdb.net/docs/build/html/api_telnet/help.html#response "Permalink to this heading")
-----------------------------------------------------------------------------------------------------------

A space separated list of commands supported.

### Example[¶](https://opentsdb.net/docs/build/html/api_telnet/help.html#example "Permalink to this heading")

available commands: put stats dropcaches version exit help diediedie
