Overview of using Outlook and Power Automate
============================================

Connectors represent the service to which you want to connect. For example, you can use the OneDrive, SharePoint, or Twitter (now X) connectors to use those services. Two of the most popular connectors used in cloud flows to send or receive email are the [Outlook.com](https://learn.microsoft.com/en-us/connectors/outlook/) connector and the [Office 365 Outlook](https://learn.microsoft.com/en-us/connectors/office365/#connector-in-depth) connector. Both connectors offer similar operations that you can use to manage your mail, calendars, and contacts. You can perform actions such as send mail, schedule meetings, add contacts, and more with either of these connectors.

[](https://learn.microsoft.com/en-us/power-automate/email-overview#when-to-use-the-outlookcom-or-office-365-outlook-connector)
When to use the Outlook.com or Office 365 Outlook connector
-----------------------------------------------------------

If you're using a work or school email account, use the Office 365 Outlook connector. If you're using a personal account (Microsoft account), use the Outlook.com connector. In this article, we refer only to the Office 365 Outlook [triggers](https://learn.microsoft.com/en-us/connectors/office365/#triggers) and [actions](https://learn.microsoft.com/en-us/connectors/office365/#actions). You can also use the same techniques for the Outlook.com connector.

[](https://learn.microsoft.com/en-us/power-automate/email-overview#find-outlook-triggers)
Find Outlook triggers
---------------------

A _trigger_ is an event that starts a cloud flow. For example, **When an email is flagged (V3)** is a trigger.

Use the **Search** field to find triggers by entering keywords. For example, if you want to find triggers related to Outlook, type **outlook** in the search box.

![Image 8: Screenshot of a list of Office 365 Outlook triggers.](https://learn.microsoft.com/en-us/power-automate/media/email/email-triggers.png)

Tutorials for how to use triggers in your cloud flows are available in [Trigger a cloud flow based on email properties](https://learn.microsoft.com/en-us/power-automate/email-triggers).

Get the full list of triggers you can use in your cloud flows in [Office 365 Outlook triggers](https://learn.microsoft.com/en-us/connectors/office365/#triggers).

[](https://learn.microsoft.com/en-us/power-automate/email-overview#find-outlook-actions)
Find Outlook actions
--------------------

_Actions_ are the events you want the flow to do after the trigger event takes place. An example of an action for the **When an email is flagged (V3)** trigger is **Create contact (V2)**.

![Image 9: Screenshot of a trigger and an action in a cloud flow.](https://learn.microsoft.com/en-us/power-automate/media/email/trigger-action.png)

You can use the **Search** field to find actions by entering keywords. For example, if you want to find actions related to Outlook, type **outlook** in the search box.

![Image 10: Screenshot of a list of Office 365 Outlook actions.](https://learn.microsoft.com/en-us/power-automate/media/email/email-actions.png)

Tutorials for how to use actions in your cloud flows are available in [Create flows for popular email scenarios](https://learn.microsoft.com/en-us/power-automate/email-top-scenarios) and [Customize email in flows](https://learn.microsoft.com/en-us/power-automate/email-customization).

Get the full list of actions you can use in your cloud flows in [Office 365 Outlook actions](https://learn.microsoft.com/en-us/connectors/office365/#actions).

[](https://learn.microsoft.com/en-us/power-automate/email-overview#shortcuts-to-find-triggers-and-actions)
Shortcuts to find triggers and actions
--------------------------------------

To help you find triggers or actions quickly, use the shortcuts in the following table.

Expand table

| Shortcut | How to use |
| --- | --- |
| Display all triggers or actions | When you search for actions or triggers, a condensed list within each category displays. If you want all options to display, select **See more** at the top of the category. |
| Display your most often used triggers or actions at the top | Select the **Favorite** icon (the star) next to a category name or the trigger or action name. Next time you open the pane, your triggers or actions you marked as favorite appear at the top of the pane. |
