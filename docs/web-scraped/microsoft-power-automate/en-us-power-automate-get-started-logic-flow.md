Create a cloud flow that performs one or more tasks automatically after an event triggers it. For example, create a cloud flow that notifies you by email when someone sends a tweet that contains a keyword you specify. In this example, sending a tweet is the event, and sending mail is the action.

[](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#prerequisites)
Prerequisites
-------------

* An account for [Power Automate](https://make.powerautomate.com/)
* An X account
* Office 365 credentials

[](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#create-a-cloud-flow)
Create a cloud flow
-------------------

You can create a cloud flow using natural language in Copilot, or create it from scratch. If you have access to Copilot, select the **Using copilot** tab to create your flow. If you don't have access to Copilot, select the **Without copilot** tab.

* [Using copilot](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#tabpanel_1_using-copilot)
* [Without copilot](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#tabpanel_1_without-copilot)

With Copilot, you can create a cloud flow using natural language. Copilot generates a flow based on your prompt. You can use it 'as is', or customize it further. Learn more in [Configure your cloud flow in the designer](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#configure-your-cloud-flow-in-the-designer).

1. Sign in to [Power Automate](https://make.powerautomate.com/).

2. Copy the following prompt and paste it in the Copilot field:

```copilot-prompt
when a new tweet is posted, send an email to eug@contoso.com with X username
```
1. Select **Generate**.

Based on the description, Copilot begins to create a suggested _trigger_ and _actions_ for your flow. A trigger is an event that starts a cloud flow. Actions are the events you want the flow to do after the trigger event takes place.

In this example, the trigger is **When a new tweet is posted** and the actions are **Get user** and **Send an email**.

![Image 8: Screenshot of a prompt that uses natural language in Copilot.](https://learn.microsoft.com/en-us/power-automate/media/get-started-logic-flow/copilot-natural-language-new-designer.png)

1. Select **Keep it and continue**.

2. Review your connected apps and services to X and Outlook. A green checkmark indicates that the connection is valid.

3. Select **Next**. Your flow appears on the designer.

4. Save the flow and it's ready to use.

Your flow triggers when new tweets mentioning the key phrase _Contoso Company_ are posted. It also sends an email to the specified email address in the **Send an email** action.

1. Go to [Test your flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#test-your-flow).

1. Sign in to [Power Automate](https://make.powerautomate.com/).

1. From the navigation bar on the left, select **My flows**.

1. Select **New flow**, and then select **Automated cloud flow**.

1. In the **Flow name** field, give your flow a name.

If you skip this step, Power Automate generates a name for you.

1. In the **Search all triggers** field, enter **X**.

2. Select **When a new tweet is posted - X**.

![Image 9: Screenshot of name your flow and search for the Twitter trigger.](https://learn.microsoft.com/en-us/power-automate/media/get-started-logic-flow/name-search-trigger-classic-designer.png)

1. At the bottom of the screen, select **Create**.

Tip

Connectors support multiple types of authentication. For example, SQL Server supports Microsoft Entra ID, SQL Server authentication, Windows authentication, and SQL connection string. Choose which type of authentication you want to use when configuring a connector.
8.   Go to [Configure your cloud flow in the designer](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#configure-your-cloud-flow-in-the-designer).

[](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#configure-your-cloud-flow-in-the-designer)
Configure your cloud flow in the designer
-----------------------------------------

Power Automate allows you to use either the [new designer](https://learn.microsoft.com/en-us/power-automate/flows-designer) or the [classic designer](https://learn.microsoft.com/en-us/power-automate/flows-designer) to configure your cloud flow. The steps are similar in both designers. Learn more (with examples) in [Identify differences between the new designer and the classic designer](https://learn.microsoft.com/en-us/power-automate/flows-designer#identify-differences-between-the-new-designer-and-the-classic-designer).

* [New designer](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#tabpanel_2_new-designer)
* [Classic designer](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#tabpanel_2_classic-designer)

1. Make sure you completed the steps in [Create a cloud flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#create-a-cloud-flow).

2. Below the **When a new tweet is posted** trigger, select the plus sign (**+**).

3. In the **Search for an action or connector** field, enter **send email**.

4. Under **Office 365 Outlook**, select **Send an email (V2)**. The configuration pane opens.

5. In the configuration pane, enter your email address in the **To** field.

6. In the **Subject** field, enter **New tweet from:**, and then type a space.

7. Select the lightning bolt to the right and select **Tweeted by** to add a placeholder for it.

If you don't see the **Tweeted by** dynamic content, select **See more** at the top of the list.

![Image 10: Screenshot of adding the 'Tweeted by dynamic content in the new designer.](https://learn.microsoft.com/en-us/power-automate/media/get-started-logic-flow/add-parameter-new-designer.png)

1. Select the **Body** field, select the lightning bolt, and then select **Tweet text** to add a placeholder for it.

Note

Optionally, you can add more dynamic content, other text, or both to the body of the email.
9.   Close the configuration pane by selecting **<<** in the upper-right corner of the pane.

1. Select **Save** and go to [Test your flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#test-your-flow).

1. Make sure you completed the steps in [Create a cloud flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#create-a-cloud-flow).

1. Select **+ New step**.

1. In the **Choose an operation** field, enter **send email**.

1. In the **Actions** list, select **Send an email (V2) - Office 365 Outlook**.

1. In the **To** field, enter your email address.

1. In the **Subject** field, enter **New tweet from:**, and then type a space.

1. In the **Dynamic content** list, select **Tweeted by** to add a placeholder for it.

![Image 11: Screenshot of adding the Tweeted by dynamic content.](https://learn.microsoft.com/en-us/power-automate/media/get-started-logic-flow/add-parameter-classic-designer.png)

1. Select the **Body** field, and then select **Tweet text** in the **Dynamic content** list to add a placeholder for it.

Note

Optionally, you can add more dynamic content, other text, or both to the body of the email.
9.   Select **Save** and go to [Test your flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#test-your-flow).

[](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#test-your-flow)
Test your flow
--------------

After you [create a cloud flow](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#create-a-cloud-flow) and save it, send a tweet with the keyword that you indicated, or wait for someone else to post such a tweet.

In about a minute after the tweet is posted, an email message notifies you of the new tweet.

Tip

Use the **Send email (V2)** action to format email in which you customize the font, use bold, italic, or underline, customize the color and highlight, create lists or links, and more.

[](https://learn.microsoft.com/en-us/power-automate/get-started-logic-flow#manage-a-cloud-flow)
Manage a cloud flow
-------------------

You can have up to 600 flows in your account. If you already have 600 flows, delete one before you create another flow.

1. Sign in to [Power Automate](https://make.powerautomate.com/).

2. On the navigation pane to the left, select **My flows**.

3. Select the vertical ellipsis (**⋮**) next to the flow you want to manage, and then do any of the following:

    *   To pause a cloud flow, select **Turn off**.

![Image 12: Screenshot of pausing a flow.](https://learn.microsoft.com/en-us/power-automate/media/get-started-logic-flow/pause-flow.png)

    *   To resume a cloud flow, select **Turn on**.

    *   To edit a cloud flow, select **Edit** (or the pencil icon) that corresponds to the flow you want to edit.

    *   To delete a cloud flow, **Delete**, and then select **Delete** on the message box that appears.

    *   To view the run history of a cloud flow, select the flow from the **My flows** page. Then, select **Details** to see the history under the **28 day run history** section. Select a cloud flow run from the list to view the inputs and outputs of each step.
