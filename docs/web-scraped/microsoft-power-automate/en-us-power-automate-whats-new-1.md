Weekly releases
---------------

You can find information about the new features, fixes, and improvements that were released in the past few weeks in [Service Updates for Power Automate](https://learn.microsoft.com/en-us/power-platform/released-versions/power-automate).

Note

Releases are rolled out over several days. New or updated functionality might not appear immediately.

[](https://learn.microsoft.com/en-us/power-automate/whats-new#release-plans)
Release plans
