Work with reports
=================

Reports help you monitor your progress towards your business goals by helping you see how you’re doing. You can also track trends - which can give you an advantage over your competitors.

The reporting features need to be enabled for your model-driven app for you to see the reports area on the left navigation pane. For more information, see [Add reporting features to your model-driven app](https://learn.microsoft.com/en-us/power-apps/maker/model-driven-apps/add-reporting-to-app).

[](https://learn.microsoft.com/en-us/power-apps/user/work-with-reports#run-a-report)
Run a report
------------

1.   From the left navigation pane, select the reports area.

2.   Choose the report you want >**Run Report**.

Note

In **Report Viewer** dialog box, you can leave the search criteria as is, or change it as needed. ![Image 11: Run a report.](https://learn.microsoft.com/en-us/power-apps/user/media/report-run.png) 

[](https://learn.microsoft.com/en-us/power-apps/user/work-with-reports#share-a-report-with-other-users-or-teams)
Share a report with other users or teams
----------------------------------------

1.   From the left navigation pane, select the reports area.

2.   In the list of reports, select the report you want to share.

3.   On the command bar, select **Share**.

![Image 12: share a report.](https://learn.microsoft.com/en-us/power-apps/user/media/report-share.png) 
4.   On the **Share Report** dialog box, select **Add User/Team**.

5.   In the **Look Up Rows** dialog box, find the users or team row that you want to share the report with, and select the check box next to the row.

![Image 13: Select user to share a report.](https://learn.microsoft.com/en-us/power-apps/user/media/report-share1.png) 
6.   Choose **Select** to add the user or team row to the **Selected rows** box and then select **Add**.

![Image 14: Add user to share a report.](https://learn.microsoft.com/en-us/power-apps/user/media/report-share2.png) 
7.   In the **Share Report** dialog box, select the type of share access that you want. The available permissions are: Read, Write, Delete, Append, Assign, or Share. This will add the user or team row to the **Selected rows** box.

![Image 15: Choose share access.](https://learn.microsoft.com/en-us/power-apps/user/media/report-share3.png) 

[](https://learn.microsoft.com/en-us/power-apps/user/work-with-reports#share-a-report-with-your-organization-for-admins)
Share a report with your organization (for admins)
--------------------------------------------------

If the report would be useful for all users, make it available to the organization.

1.   From the left navigation pane, select the reports area.

2.   In the list of reports, select the report you want to share.

3.   On the command bar, select **Edit**.

4.   On the **Actions** menu, select **Make Report Available to Organization**.

![Image 16: Share report with org.](https://learn.microsoft.com/en-us/power-apps/user/media/report-share4.png) 

[](https://learn.microsoft.com/en-us/power-apps/user/work-with-reports#download-a-report)
Download a report
-----------------

1.   From the left navigation pane, select the reports area.
2.   In the list of reports, select the report you want to share.
3.   On the command bar, select **Edit**.
4.   On the **Actions** menu, select **Download Report**.

 The RDL file contains the fetchXML that the report is based on.
5.   Open the report once the download is complete.

[](https://learn.microsoft.com/en-us/power-apps/user/work-with-reports#known-issue)
### Known issue

Reports rely on a hybrid experience called out in [Enable hybrid experience](https://learn.microsoft.com/en-us/power-platform/admin/enable-hybrid-experience) which are not supported in mobile or Teams.
