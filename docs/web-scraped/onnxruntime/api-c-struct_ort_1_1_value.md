Creates an OrtValue instance containing SparseTensor. This constructs a sparse tensor that makes use of user allocated buffers. It does not make copies of the user provided data and does not modify it. The lifespan of user provided buffers should eclipse the life span of the resulting OrtValue. This call constructs an instance that only contain a pointer to non-zero values. To fully populate the sparse tensor call Use\<Format\>Indices() API below to supply a sparse format specific indices. This API is not suitable for string data. Use [CreateSparseTensor()](struct_ort_1_1_value.html#a07bd3c4da603a8fc23ea6e1f86482f6b "This is a simple forwarding method to the other overload that helps deducing data type enum value fro...") with allocator specified so strings can be properly copied into the allocated buffer.
