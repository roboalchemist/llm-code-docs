## Faster, stable visual testing with Percy AI Agents
Percy AI Agents deliver instant setup, eliminate pixel-level noise, and speed up reviews by 3× with clear explanations of meaningful UI changes.

- Generates natural-language summaries of visual changes
- Reduces visual noise and highlights meaningful UI changes
- Tailors the agent’s responses to your website or app using prompting
- Sets up visual tests instantly within your existing repositories
