# Error - Missing Percy token
Learn how to fix missing Percy token error in Percy

A Percy token is a unique, secure authentication key used to integrate Percy with your projects for visual testing and to authorize API requests. If you encounter an error likemissing PERCY_TOKEN, it means the Percy command was executed via CLI, SDK, or CI without setting the PERCY_TOKEN environment variable. This project token should be set in the environment. For more information, seeSet environment variables.
