**Date**: Dec 20, 2025, **Version**: 0.26.0

Welcome! [scikit-image](https://scikit-image.org/) is an image processing toolbox which builds on [numpy](https://numpy.org/), [scipy.ndimage](https://docs.scipy.org/doc/scipy/reference/ndimage.html) and other libraries to provide a versatile set of image processing routines in [Python](https://www.python.org/). Our project and community is guided by the [scikit-image Code of Conduct](https://scikit-image.org/docs/stable/about/code_of_conduct.html).
