Title: About Northwoods

Northwoods Software Corporation is a privately-held company, founded in 1995 by a group of engineers with a shared vision for providing superior graphical user interfaces.

We began by creating the Sanscript programming language and IDE, which required such a strong visual layer that the graphing library became its own product. We have grown to become a world-class supplier of interactive diagram libraries.

We have created diagram components for over 25 years, over many platforms, and reap the benefits of redesigning a library from scratch for each new platform while taking into account our accumulated experience.

We are a company created by engineers for engineers. We pride ourselves on quality software design and consistent developer-to-developer technical support.
