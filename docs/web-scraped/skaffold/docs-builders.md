Build
=====

Skaffold supports different [tools](https://skaffold.dev/docs/builders/builder-types/) for building images across different [build environments](https://skaffold.dev/docs/builders/build-environments/).

|  | [Local Build](https://skaffold.dev/docs/builders/build-environments/local/) | [In Cluster Build](https://skaffold.dev/docs/builders/build-environments/in-cluster/) | [Remote on Google Cloud Build](https://skaffold.dev/docs/builders/build-environments/cloud-build/) |
| --- | --- | --- | --- |
| **Dockerfile** | [Yes](https://skaffold.dev/docs/builders/builder-types/docker/#dockerfile-locally) | [Yes](https://skaffold.dev/docs/builders/builder-types/docker/#dockerfile-in-cluster-with-kaniko) | [Yes](https://skaffold.dev/docs/builders/builder-types/docker/#dockerfile-remotely-with-google-cloud-build) |
| **Jib Maven and Gradle** | [Yes](https://skaffold.dev/docs/builders/builder-types/jib/#jib-maven-and-gradle-locally) | - | [Yes](https://skaffold.dev/docs/builders/builder-types/jib/#remotely-with-google-cloud-build) |
| **Cloud Native Buildpacks** | [Yes](https://skaffold.dev/docs/builders/builder-types/buildpacks/) | - | [Yes](https://skaffold.dev/docs/builders/builder-types/buildpacks/) |
| **Bazel** | [Yes](https://skaffold.dev/docs/builders/builder-types/bazel/) | - | - |
| **ko** | [Yes](https://skaffold.dev/docs/builders/builder-types/ko/) | - | [Yes](https://skaffold.dev/docs/builders/builder-types/ko/#remote-builds) |
| **Custom Script** | [Yes](https://skaffold.dev/docs/builders/builder-types/custom/#custom-build-script-locally) | [Yes](https://skaffold.dev/docs/builders/builder-types/custom/#custom-build-script-in-cluster) | - |

Configuration
-------------

The `build` section in the Skaffold configuration file, `skaffold.yaml`, controls how artifacts are built. To use a specific tool for building artifacts, add the value representing the tool and options for using that tool to the `build` section.

For detailed per-builder [Skaffold Configuration](https://skaffold.dev/docs/design/config/) options, see [skaffold.yaml References](https://skaffold.dev/docs/references/yaml/).
