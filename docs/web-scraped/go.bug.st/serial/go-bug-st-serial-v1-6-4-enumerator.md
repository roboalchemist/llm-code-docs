### Overview [¶](https://pkg.go.dev/go.bug.st/serial@v1.6.4/enumerator#pkg-overview "Go to Overview")

Package enumerator is a golang cross-platform library for USB serial port discovery.

This library has been tested on Linux, Windows and Mac and uses specific OS services to enumerate USB PID/VID, in particular on MacOSX the use of cgo is required in order to access the IOKit Framework. This means that the library cannot be easily cross compiled for darwin/* targets.

#### func [GetDetailedPortsList](https://github.com/bugst/go-serial/blob/v1.6.4/enumerator/enumerator.go#L30)[¶](https://pkg.go.dev/go.bug.st/serial@v1.6.4/enumerator#GetDetailedPortsList "Go to GetDetailedPortsList")

func GetDetailedPortsList() ([]*[PortDetails](https://pkg.go.dev/go.bug.st/serial@v1.6.4/enumerator#PortDetails), [error](https://pkg.go.dev/builtin#error))

GetDetailedPortsList retrieve ports details like USB VID/PID. Please note that this function may not be available on all OS: in that case a FunctionNotImplemented error is returned.

Example [¶](https://pkg.go.dev/go.bug.st/serial@v1.6.4/enumerator#example-GetDetailedPortsList "Go to Example")

package main

import (
	"fmt"
	"log"

	"go.bug.st/serial/enumerator"
)

func main() {
	ports, err := enumerator.GetDetailedPortsList()
	if err != nil {
		log.Fatal(err)
	}
	if len(ports) == 0 {
		fmt.Println("No serial ports found!")
		return
	}
	for _, port := range ports {
		fmt.Printf("Found port: %s\n", port.Name)
		if port.IsUSB {
			fmt.Printf("   USB ID     %s:%s\n", port.VID, port.PID)
			fmt.Printf("   USB serial %s\n", port.SerialNumber)
		}
	}
}
