Namespace[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#namespace "Permanent link")
===================================================================================================

Namespaces provides a mechanism for isolating groups of resources within a single cluster.

[API Reference](https://cdk8s.io/docs/latest/reference/cdk8s-plus-31/typescript/#namespace)

Create a `Namespace`[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#create-a-namespace "Permanent link")
-----------------------------------------------------------------------------------------------------------------------

To create a new namespace in the cluster:

```
import * as kplus from 'cdk8s-plus-31';
import * as k from 'cdk8s';

const app = new k.App();
const chart = new k.Chart(app, 'Chart');

const namespace = new kplus.Namespace(chart, 'BackOfficeNamespace');
```

Like any other resource, if you don’t specify a name, cdk8s will auto-generate one, which you can access by `namespace.name`.

Select namespaces[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#select-namespaces "Permanent link")
-------------------------------------------------------------------------------------------------------------------

Namespaces can also be selected by various mechanisms. These selections are often used in other cdk8s+ API’s, such as [pod selection](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/pod/#pod-selection) during scheduling.

### Select namespaces by name[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#select-namespaces-by-name "Permanent link")

Select a namespace called `backoffice`.

```
import * as kplus from 'cdk8s-plus-31';

const backoffice = kplus.Namespaces.select(this, 'Backoffice', { names: ['backoffice'] });
```

### Select namespace by labels[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#select-namespace-by-labels "Permanent link")

Select all namespaces that have the `processing=batch` label.

```
import * as kplus from 'cdk8s-plus-31';

const batch = kplus.Namespaces.select(this, 'Batch', { labels: { processing: 'batch'} });
```

### Select all namespaces[¶](https://cdk8s.io/docs/latest/plus/cdk8s-plus-31/namespace/#select-all-namespaces "Permanent link")

Select all namespaces in the cluster.

```
import * as kplus from 'cdk8s-plus-31';

const all = kplus.Namespaces.all();
```
