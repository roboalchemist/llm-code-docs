This page documents variables that are provided by CMake or have meaning to CMake when set by project code.

For general information on variables, see the [Variables](https://cmake.org/cmake/help/latest/manual/cmake-language.7.html#cmake-language-variables) section in the cmake-language manual.

Note

CMake reserves identifiers that:

*   begin with `CMAKE_` (upper-, lower-, or mixed-case), or

*   begin with `_CMAKE_` (upper-, lower-, or mixed-case), or

*   begin with `_` followed by the name of any [`CMake Command`](https://cmake.org/cmake/help/latest/manual/cmake-commands.7.html#manual:cmake-commands(7) "cmake-commands(7)").
