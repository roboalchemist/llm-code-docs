Documentação do Cloud CDN
=========================

[Leia a documentação do produto](https://docs.cloud.google.com/cdn/docs/overview?hl=pt-br)

O Cloud CDN (rede de fornecimento de conteúdo) usa os principais pontos de presença distribuídos globalmente do Google para armazenar em cache o conteúdo com balanceamento de carga HTTP(S) próximo aos usuários. O armazenamento de conteúdo em cache nas extremidades da rede do Google agiliza a entrega de conteúdo para os usuários e reduz os custos de veiculação.
