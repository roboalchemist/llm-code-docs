Documentazione di Cloud CDN
===========================

[Leggi la documentazione del prodotto](https://docs.cloud.google.com/cdn/docs/overview?hl=it)

Cloud CDN (Content Delivery Network) utilizza i punti di presenza perimetrali di Google, distribuiti in tutto il mondo, per memorizzare nella cache i contenuti con bilanciamento del carico HTTP(S) esterni nelle vicinanze degli utenti. La memorizzazione nella cache dei contenuti a livello perimetrale della rete Google consente di distribuire più rapidamente i contenuti ai tuoi utenti riducendo al contempo i costi associati.

[Per saperne di più, vai alla pagina del prodotto Cloud CDN.](https://cloud.google.com/cdn/?hl=it)

[Inizia gratuitamente](https://console.cloud.google.com/freetrial?hl=it)
