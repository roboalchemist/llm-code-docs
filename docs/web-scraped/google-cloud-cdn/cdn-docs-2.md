Dokumentasi Cloud CDN
=====================

[Baca dokumentasi produk](https://docs.cloud.google.com/cdn/docs/overview?hl=id)

Cloud CDN (jaringan penayangan konten) menggunakan titik kehadiran edge yang terdistribusi secara global milik Google untuk meng-cache konten HTTP(S) eksternal yang di-load balance di lokasi yang dekat dengan pengguna Anda. Meng-cache konten di edge jaringan Google mempercepat penayangan konten kepada pengguna Anda sekaligus mengurangi biaya penayangan.

[Buka halaman produk Cloud CDN untuk mengetahui informasi selengkapnya.](https://cloud.google.com/cdn/?hl=id)
