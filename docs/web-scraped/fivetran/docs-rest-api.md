REST API resources
------------------

Copy sample requests and view sample responses. See the API fields we use, as well as their types and definitions.

[Users→](https://fivetran.com/docs/rest-api/api-reference/users)

List, invite, edit, and delete your users

[Roles→](https://fivetran.com/docs/rest-api/api-reference/roles)

List predefined and custom roles

[Groups→](https://fivetran.com/docs/rest-api/api-reference/groups)

Understand the groups and connections within those groups

[Teams→](https://fivetran.com/docs/rest-api/api-reference/teams)

List, edit, and delete your teams, manage team memberships and permissions

[Destinations→](https://fivetran.com/docs/rest-api/api-reference/destinations)

Create, and edit destination within the group

[Connections→](https://fivetran.com/docs/rest-api/api-reference/connections)

Create, edit, remove, and manage a subset of your connections

[Certificates→](https://fivetran.com/docs/rest-api/certificates)

Approve TLS certificates

[Webhooks→](https://fivetran.com/docs/rest-api/api-reference/webhooks)

List, create, edit, remove, and test webhooks

[Transformations→](https://fivetran.com/docs/rest-api/api-reference/transformation-management)

List, create, edit, run, and remove transformations

[Transformation Project→](https://fivetran.com/docs/rest-api/api-reference/transformation-projects-management)

List, create, edit, remove, and test transformations projects

[Hybrid Deployment Agent Management→](https://fivetran.com/docs/rest-api/api-reference/hybrid-deployment-agent-management)

List, create, re-authenticate, and remove Hybrid Deployment Agents

[Log Services→](https://fivetran.com/docs/rest-api/api-reference/log-service-management)

List, create, edit, remove, and test log services

[Private Links Management→](https://fivetran.com/docs/rest-api/api-reference/private-links)

List, create, edit and remove private links

[Proxy Agent Management→](https://fivetran.com/docs/rest-api/api-reference/proxy-agent)

List, create, remove proxy agents and manage their attachments to connections

[System Key Management→](https://fivetran.com/docs/rest-api/api-reference/system-key)

List, create, edit, remove and rotate system keys

[Connector Metadata→](https://fivetran.com/docs/rest-api/api-reference/connector-metadata)

Retrieve information about connectors supported by Fivetran
