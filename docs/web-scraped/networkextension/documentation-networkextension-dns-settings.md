DNS settings
============

Create and manage a system-wide DNS configuration that uses built-in encrypted DNS protocols.

[Overview](https://developer.apple.com/documentation/networkextension/dns-settings#overview)
--------------------------------------------------------------------------------------------

With the DNS Settings feature in macOS and iOS, your app can create and manage a configuration that uses one of the built-in DNS protocols: DNS-over-TLS or DNS-over-HTTPS. The user must explictly enable your configuration in order to use the server you specify.

[Topics](https://developer.apple.com/documentation/networkextension/dns-settings#topics)
----------------------------------------------------------------------------------------

### [Essentials](https://developer.apple.com/documentation/networkextension/dns-settings#Essentials)

[`Network Extensions Entitlement`](https://developer.apple.com/documentation/BundleResources/Entitlements/com.apple.developer.networking.networkextension)

The APIs an app can use to customize networking features.

### [DNS configuration](https://developer.apple.com/documentation/networkextension/dns-settings#DNS-configuration)

[`class NEDNSSettingsManager`](https://developer.apple.com/documentation/networkextension/nednssettingsmanager)

An object you use to create and manage a DNS settings configuration.

[`class NEDNSOverHTTPSSettings`](https://developer.apple.com/documentation/networkextension/nednsoverhttpssettings)

The DNS resolver settings for a DNS-over-HTTPS server.

[`class NEDNSOverTLSSettings`](https://developer.apple.com/documentation/networkextension/nednsovertlssettings)

The DNS resolver settings for a DNS-over-TLS server.

[See Also](https://developer.apple.com/documentation/networkextension/dns-settings#see-also)
--------------------------------------------------------------------------------------------

### [DNS configurations](https://developer.apple.com/documentation/networkextension/dns-settings#DNS-configurations)

[DNS proxy provider](https://developer.apple.com/documentation/networkextension/dns-proxy-provider)

Create an on-device DNS proxy using a custom protocol.
