About[](https://doc.qt.io/QtForMCUs/#about "Direct link to this headline")
--------------------------------------------------------------------------

Qt for MCUs 2.12.0 offers a lightweight graphics framework and toolkit that enables you to design, develop, and deploy graphical user interfaces (GUI) on microcontrollers (MCU). It lets you run the application either on bare metal or real-time operating systems.
