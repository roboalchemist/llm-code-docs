Qt offers specialized tools needed to design, develop, and deploy applications on embedded platforms.

* ### Qt for MCUs

A complete graphics framework and toolkit for GUIs on MCUs, with applications running either on bare metal or a real-time operating system.

* ### Qt Safe Renderer

Provides a UI rendering component that can be used to render information in safety-critical systems.

* ### Qt Application Manager

A headless application manager daemon designed for embedded Linux systems with complex UIs.

* ### Qt Interface Framework

Provides a core framework and tools to implement middleware APIs, backends, and services.

Qt for Android Automotive
-------------------------

Framework for developing rich user interfaces for in-vehicle infotainment systems using Qt and Android.

* ### Qt for Android Automotive 6.10

Integrates the Qt Framework with Android bindings, the Qt Interface Framework, and the Android Automotive bindings.

* ### Qt for Android Automotive 6.8

Integrates the Qt Framework with Android bindings, the Qt Interface Framework, and the Android Automotive bindings.

* ### Qt for Android Automotive 6.5

Integrates the Qt Framework with Android bindings, the Qt Interface Framework, and the Android Automotive bindings.

Boot to Qt
----------

Qt development framework supporting multiple embedded platforms.

* ### Boot to Qt 6.10

A commercial offering that provides the Qt development framework for embedded Linux.

* ### Boot to Qt 6.8

A commercial offering that provides the Qt development framework for embedded Linux.

* ### Boot to Qt 6.5

A commercial offering that provides the Qt development framework for embedded Linux.

* ### Qt Device Utilities

Functionality for controlling settings in embedded applications.
