**INSTALLATION**
Install either a _commercial_ or _open source_ version of Qt Creator with Qt Online Installer or as a stand-alone tool on Linux, macOS, or Windows.

[Installation](https://doc.qt.io/qtcreator/creator-how-to-install.html)**GETTING STARTED**
Learn about the Qt Creator features and UI, as well as how you can make Qt Creator behave more like your favorite code editor or IDE.

[Getting Started](https://doc.qt.io/qtcreator/creator-getting-started.html)**TUTORIALS**
Follow the tutorials to create basic Qt applications for the desktop and mobile devices by using Qt Widgets and Qt Quick.

[Tutorials](https://doc.qt.io/qtcreator/creator-tutorials.html)

About[](https://doc.qt.io/qtcreator/#about "Direct link to this headline")
--------------------------------------------------------------------------

Use the Qt Creator cross-platform, complete integrated development environment (IDE) to create applications for [desktop](https://doc.qt.io/qtcreator/creator-desktop-platforms.html), [embedded](https://doc.qt.io/qtcreator/creator-overview.html#embedded-platforms), and [mobile device](https://doc.qt.io/qtcreator/creator-overview.html#mobile-platforms) platforms, such as [Android](https://doc.qt.io/qtcreator/creator-how-tos.html#android) and [iOS](https://doc.qt.io/qtcreator/creator-how-tos.html#ios). Also, you can build applications in web format and run them in web browsers with [Qt for WebAssembly](https://doc.qt.io/qtcreator/creator-setup-webassembly.html).
