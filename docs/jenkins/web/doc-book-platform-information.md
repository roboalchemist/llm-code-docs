Markdown Content:
This chapter provides information about the operating systems, web browsers, Java versions, and other software components commonly used to run Jenkins.

The content is suitable for users of all levels, especially those getting started with Jenkins.

[](https://www.jenkins.io/doc/book/platform-information/#support-policies)Support policies
------------------------------------------------------------------------------------------

The Jenkins project defines support levels for platform components such as Java versions, operating systems, web browsers, and servlet containers. Support levels provide guidance on the effort Jenkins project contributors generally apply to issues related to platform components.

Details of the support policies are available in the specific support policy pages:

*   [Java](https://www.jenkins.io/doc/book/platform-information/support-policy-java/)

*   [Linux](https://www.jenkins.io/doc/book/platform-information/support-policy-linux/)

*   [Windows](https://www.jenkins.io/doc/book/platform-information/support-policy-windows/)

*   [Servlet containers](https://www.jenkins.io/doc/book/platform-information/support-policy-servlet-containers/)

*   [Web browsers](https://www.jenkins.io/doc/book/platform-information/support-policy-web-browsers/)

[](https://www.jenkins.io/doc/book/platform-information/#java-support-and-upgrade-information)Java support and upgrade information
----------------------------------------------------------------------------------------------------------------------------------

In addition to support policies, this chapter provides information on upgrading your Java version in Jenkins. Depending on your current Java version, there are instructions for [upgrading to Java 17](https://www.jenkins.io/doc/book/platform-information/upgrade-java-to-17/) and [upgrading to Java 21](https://www.jenkins.io/doc/book/platform-information/upgrade-java-to-21/).

Jenkins does not support Java 11, but if you need to upgrade from Java 8 to 11 before upgrading to Java 17, please follow the [Java 11 upgrade instructions](https://www.jenkins.io/doc/book/platform-information/upgrade-java-to-11/).
