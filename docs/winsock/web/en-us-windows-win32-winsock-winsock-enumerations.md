Winsock Enumerations
====================

Feedback

In this article
---------------

The following list provides concise descriptions of each Winsock enumeration. For additional information on any enumeration, click the enumeration name.

Expand table

| Enumeration | Description |
| --- | --- |
| [**CONTROL_CHANNEL_TRIGGER_STATUS**](https://learn.microsoft.com/en-us/windows/win32/api/mstcpip/ne-mstcpip-control_channel_trigger_status) | Specifies the status from a query for the **REAL_TIME_NOTIFICATION_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](https://learn.microsoft.com/en-us/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger) to receive background network notifications in a Windows Runtime application. |
| [**eWINDOW_ADVANCE_METHOD**](https://learn.microsoft.com/en-us/windows/win32/api/wsrm/ne-wsrm-ewindow_advance_method) | Specifies the window advance mode used for Reliable Multicast. |
| [**GUARANTEE**](https://learn.microsoft.com/en-us/windows/win32/winsock/guarantee-2) | No longer used. |
| [**MULTICAST_MODE_TYPE**](https://learn.microsoft.com/en-us/windows/win32/api/ws2ipdef/ne-ws2ipdef-multicast_mode_type) | Specifies the filter mode for multicast group addresses. |
| [**NAPI_PROVIDER_LEVEL**](https://learn.microsoft.com/en-us/windows/win32/api/nsemail/ne-nsemail-napi_provider_level) | Specifies the provider authority level of a NS_EMAIL namespace provider for a given domain. |
| [**NAPI_PROVIDER_TYPE**](https://learn.microsoft.com/en-us/windows/win32/api/nsemail/ne-nsemail-napi_provider_type) | Specifies the type of hosting expected for a namespace provider. |
| [**RIO_NOTIFICATION_COMPLETION_TYPE**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/ne-mswsock-rio_notification_completion_type) | Specifies the type of completion queue notifications to use with the [**RIONotify**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/nc-mswsock-lpfn_rionotify) function when sending or receiving data using the Winsock registered I/O extensions. |
| [**SOCKET_SECURITY_PROTOCOL**](https://learn.microsoft.com/en-us/windows/win32/api/mstcpip/ne-mstcpip-socket_security_protocol) | Indicates the type of security protocol to be used on a socket. |
| [**SOCKET_USAGE_TYPE**](https://learn.microsoft.com/en-us/windows/win32/api/Mstcpip/ne-mstcpip-socket_usage_type) | Used to specified the usage type for the socket. |
| [**TCPSTATE**](https://learn.microsoft.com/en-us/windows/win32/api/mstcpip/ne-mstcpip-tcpstate) | Indicates the possible states of a Transmission Control Protocol (TCP) connection. |
| [**WSAECOMPARATOR**](https://learn.microsoft.com/en-us/windows/win32/api/Winsock2/ne-winsock2-wsaecomparator) | Used for version-comparison semantics. |
| [**WSC_PROVIDER_INFO_TYPE**](https://learn.microsoft.com/en-us/windows/win32/api/ws2spi/ne-ws2spi-wsc_provider_info_type) | Specifies the information class of a layered service protocol (LSP). |
