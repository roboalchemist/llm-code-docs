[**addrinfo**](https://learn.microsoft.com/en-us/windows/win32/api/ws2def/ns-ws2def-addrinfoa)Used by the [**getaddrinfo**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfo) function to hold host address information.
[**addrinfoW**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfow)Used by the [**GetAddrInfoW**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfow) function to hold host address information.
[**ADDRINFO_DNS_SERVER**](https://learn.microsoft.com/en-us/windows/win32/api/ws2def/ns-ws2def-addrinfo_dns_server)Represents a custom Domain Name System (DNS) server, used in the Winsock APIs.
[**addrinfoex**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoexw)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information.
[**addrinfoex2**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoex2w)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information when both a canonical name and a fully qualified domain name have been requested.
[**addrinfoex3**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoex3)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information when a specific network interface has been requested.
[**addrinfoex4**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoex4)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information when a handle to the fully qualified domain name has been requested.
[**addrinfoex5**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoex5)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information when a handle to the fully qualified domain name has been requested.
[**addrinfoex6**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-addrinfoex6)Used by the [**GetAddrInfoEx**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfoexa) function to hold host address information when a handle to the fully qualified domain name has been requested.
[**AFPROTOCOLS**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-afprotocols)Supplies a list of protocols to which application programmers can constrain queries. Used for query purposes only.
[**BLOB**](https://learn.microsoft.com/en-us/windows/win32/api/wtypesbase/ns-wtypesbase-blob)Contains information about a block of data. Derived from Binary Large Object.
[**CSADDR_INFO**](https://learn.microsoft.com/en-us/windows/win32/api/ws2def/ns-ws2def-csaddr_info)Contains Winsock address information for a network service or namespace provider.
[**fd_set**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock2/ns-winsock2-fd_set)Used by various Winsock functions and service providers, such as [**select**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock2/nf-winsock2-select), to place sockets into a "set" for various purposes, such as testing a given socket for readability.
[**GROUP_FILTER**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-group_filter)Provides multicast filtering parameters for multicast IPv6 or IPv4 addresses.
[**GROUP_REQ**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-group_req)Provides multicast group information for IPv6 or IPv4 addresses.
[**GROUP_SOURCE_REQ**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-group_source_req)Provides multicast group information for IPv6 or IPv4 addresses that includes the source IP address.
[**hostent**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-hostent)Stores information about a given host, such as host name, IP address, and so forth.
[**in_addr**](https://learn.microsoft.com/en-us/windows/win32/api/winsock2/ns-winsock2-in_addr)Represents an IPv4 Internet address.
[**in_pktinfo**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-in_pktinfo)Stores received packet address information, and is used by Windows to return information about received packets.
[**in6_addr**](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/legacy/ms738560(v=vs.85))Represents an IPv6 Internet address.
[**in6_pktinfo**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-in6_pktinfo)Stores received IPv6 packet address information, and is used by Windows to return information about received packets.
[**INET_PORT_RANGE**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-inet_port_range)Provides input data used by the **SIO_ACQUIRE_PORT_RESERVATION** Ioctl to acquire a runtime reservation for a block of TCP or UDP ports.
[**INET_PORT_RESERVATION_INSTANCE**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-inet_port_reservation_instance)Contains a port reservation and a token for a block of TCP or UDP ports.
[**INET_PORT_RESERVATION_TOKEN**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-inet_port_reservation_token)Contains a port reservation token for a block of TCP or UDP ports.
[**ip_mreq**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-ip_mreq)Provides multicast group information for IPv4 addresses.
[**ip_mreq_source**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-ip_mreq_source)Provides multicast group and source information for IPv4 addresses.
[**ip_msfilter**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-ip_msfilter)Provides multicast filtering parameters for IPv4 addresses.
[**ipv6_mreq**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-ipv6_mreq)Provides multicast group information for IPv6 addresses.
[**linger**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-linger)Maintains information about a specific socket that specifies how that socket should behave when data is queued to be sent and the [**closesocket**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/nf-winsock-closesocket) function is called on the socket.
[**NAPI_DOMAIN_DESCRIPTION_BLOB**](https://learn.microsoft.com/en-us/windows/desktop/api/Nsemail/ns-nsemail-napi_domain_description_blob)Describes a domain handled by a namespace provider for the NS_EMAIL namespace.
[**NAPI_PROVIDER_INSTALLATION_BLOB**](https://learn.microsoft.com/en-us/windows/desktop/api/Nsemail/ns-nsemail-napi_provider_installation_blob)Contains the information required to install a namespace provider for the NS_EMAIL namespace.
[**NS_SERVICE_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-ns_service_infoa)Contains information about a network service or a network service type in the context of a specified namespace, or a set of default namespaces.
[**PROTOCOL_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-protocol_infoa)Contains information about a protocol.
[**protoent**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-protoent)Contains the name and protocol numbers that correspond to a given protocol name.
[**REAL_TIME_NOTIFICATION_SETTING_INPUT**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-real_time_notification_setting_input)Contains input settings to apply for the **REAL_TIME_NOTIFICATION_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](https://learn.microsoft.com/en-us/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger) to receive background network notifications in a Windows Store app
[**REAL_TIME_NOTIFICATION_SETTING_OUTPUT**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-real_time_notification_setting_input)Contains the output settings from a query for the **REAL_TIME_NOTIFICATION_CAPABILITY** transport setting for a TCP socket that is used with [**ControlChannelTrigger**](https://learn.microsoft.com/en-us/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger) to receive background network notifications in a Windows Store app.
[**RIO_EXTENSION_FUNCTION_TABLE**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/ns-mswsock-rio_extension_function_table)Contains information on the functions that implement the Winsock registered I/O extensions.
[**RIO_BUF**](https://learn.microsoft.com/en-us/windows/desktop/api/Mswsockdef/ns-mswsockdef-rio_buf)Specifies a portion of a registered buffer used for sending or receiving network data with the Winsock registered I/O extensions.
[**RIO_BUFFERID**](https://learn.microsoft.com/en-us/windows/win32/winsock/rio-bufferid)Specifies a registered buffer descriptor used with the Winsock registered I/O extensions.
[**RIO_CQ**](https://learn.microsoft.com/en-us/windows/win32/winsock/riocqueue)Specifies a completion queue descriptor used for I/O completion notification by send and receive requests with the Winsock registered I/O extensions.
[**RIO_NOTIFICATION_COMPLETION**](https://learn.microsoft.com/en-us/windows/desktop/api/Mswsock/ns-mswsock-rio_notification_completion)Specifies the method for I/O completion to be used with a [**RIONotify**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/nc-mswsock-lpfn_rionotify) function for sending or receiving network data with the Winsock registered I/O extensions.
[**RIO_RQ**](https://learn.microsoft.com/en-us/windows/win32/winsock/riorqueue)Specifies a socket descriptor used by send and receive requests with the Winsock registered I/O extensions.
[**RIORESULT**](https://learn.microsoft.com/en-us/windows/desktop/api/Mswsockdef/ns-mswsockdef-rioresult)Contains data used to indicate request completion results used with the Winsock registered I/O extensions.
[**RM_FEC_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Wsrm/ns-wsrm-rm_fec_info)Specifies settings for using forward error correction (FEC) with Reliable Multicast.
[**RM_RECEIVER_STATS**](https://learn.microsoft.com/en-us/windows/desktop/api/Wsrm/ns-wsrm-rm_receiver_stats)Provides statistical information for a Reliable Multicast receiver session.
[**RM_SEND_WINDOW**](https://learn.microsoft.com/en-us/windows/desktop/api/Wsrm/ns-wsrm-rm_send_window)Specifies the Reliable Multicast send window.
[**RM_SENDER_STATS**](https://learn.microsoft.com/en-us/windows/desktop/api/Wsrm/ns-wsrm-rm_sender_stats)Provides statistical information for a Reliable Multicast sender session.
[**servent**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-servent)Stores or returns the name and service number for a given service name.
[**SERVICE_ADDRESS**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_address)Contains address information for a service.
[**SERVICE_ADDRESSES**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_addresses)Contains an array of [**SERVICE_ADDRESS**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_address) data structures.
[**SERVICE_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_infoa)Contains information about a network service or a network service type.
[**SERVICE_TYPE_INFO_ABS**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_type_info_absa)Contains information about a network service type.
[**SERVICE_TYPE_VALUE_ABS**](https://learn.microsoft.com/en-us/windows/desktop/api/Nspapi/ns-nspapi-service_type_value_absa)Contains information about a network-service type value. The information may be specific to a namespace.
[**sockaddr**](https://learn.microsoft.com/en-us/windows/win32/winsock/sockaddr-2)Contains socket address information. The [**sockaddr**](https://learn.microsoft.com/en-us/windows/win32/winsock/sockaddr-2) structure varies depending on the protocol selected. Limited to IPv4; use [**SOCKADDR_STORAGE**](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/legacy/ms740504(v=vs.85)) instead.
[**SOCKADDR_IRDA**](https://learn.microsoft.com/en-us/windows/desktop/api/Af_irda/ns-af_irda-sockaddr_irda)Used with IrDA socket operations, defined by address family _AF\_IRDA_.
[**SOCKADDR_STORAGE**](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/legacy/ms740504(v=vs.85))Stores socket address information, and is sufficiently large to store IPv4 or IPv6 address information, promoting protocol-family and protocol-version independence. Use this structure in place of the [**sockaddr**](https://learn.microsoft.com/en-us/windows/win32/winsock/sockaddr-2) structure.
[**SOCKET_ADDRESS**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-socket_address)Stores protocol-specific address information.
[**SOCKET_ADDRESS_LIST**](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/legacy/aa385467(v=vs.85))Stores an array of [**SOCKET_ADDRESS**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-socket_address) structures that contain protocol-specific address information.
[**SOCKET_PEER_TARGET_NAME**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-socket_peer_target_name)Contains the IP address and name for a peer target and the type of security protocol to be used on a socket.
[**SOCKET_SECURITY_QUERY_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-socket_security_query_info)Contains security information returned by the [**WSAQuerySocketSecurity**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-wsaquerysocketsecurity) function.
[**SOCKET_SECURITY_QUERY_TEMPLATE**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-socket_security_query_template)Contains the security template used by the [**WSAQuerySocketSecurity**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-wsaquerysocketsecurity) function.
[**SOCKET_SECURITY_SETTINGS**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-socket_security_settings)Specifies generic security requirements for a socket.
[**SOCKET_SECURITY_SETTINGS_IPSEC**](https://learn.microsoft.com/en-us/windows/desktop/api/Mstcpip/ns-mstcpip-socket_security_settings_ipsec)Specifies various security requirements and settings that are specific to IPsec.
[**timeval**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-timeval)Used to specify time values. Associated with the Berkeley Software Distribution (BSD) file Time.h.
[**TRANSMIT_FILE_BUFFERS**](https://learn.microsoft.com/en-us/windows/desktop/api/mswsock/ns-mswsock-transmit_file_buffers)Specifies data to be transmitted before and after file data during a [**TransmitFile**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/nf-mswsock-transmitfile) transfer operation.
[**TRANSMIT_PACKETS_ELEMENT**](https://learn.microsoft.com/en-us/windows/desktop/api/Mswsock/ns-mswsock-transmit_packets_element)Specifies a single data element to be transmitted by the [**TransmitPackets**](https://learn.microsoft.com/en-us/windows/desktop/api/Mswsock/nc-mswsock-lpfn_transmitpackets) function.
[**TRANSPORT_SETTING_ID**](https://learn.microsoft.com/en-us/windows/win32/api/transportsettingcommon/ns-transportsettingcommon-transport_setting_id)Specifies the transport setting ID used by the [**SIO_APPLY_TRANSPORT_SETTING**](https://learn.microsoft.com/en-us/windows/win32/winsock/sio-apply-transport-setting) and [**SIO_QUERY_TRANSPORT_SETTING**](https://learn.microsoft.com/en-us/windows/win32/winsock/sio-query-transport-setting) IOCTLs to apply or query the transport setting for a socket.
[**WSABUF**](https://learn.microsoft.com/en-us/windows/desktop/api/ws2def/ns-ws2def-wsabuf)Enables the creation or manipulation of a data buffer.
[**WSACOMPLETION**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsacompletion)Specifies completion notification settings for I/O control calls made to a registered namespace.
[**WSADATA**](https://learn.microsoft.com/en-us/windows/desktop/api/winsock/ns-winsock-wsadata)Contains information about the Windows Sockets implementation.
[**WSAMSG**](https://learn.microsoft.com/en-us/windows/desktop/api/Ws2def/ns-ws2def-wsamsg)Stores address and optional control information about connected and unconnected sockets. Used with the [**LPFN_WSARECVMSG (WSARecvMsg)**](https://learn.microsoft.com/en-us/windows/win32/api/mswsock/nc-mswsock-lpfn_wsarecvmsg) function.
[**WSANAMESPACE_INFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsanamespace_infow)Contains registration information for a namespace provider.
[**WSANAMESPACE_INFOEX**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsanamespace_infoexw)Contains enhanced registration information for a namespace provider.
[**WSANETWORKEVENTS**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsanetworkevents)Stores a socket's internal information about network events.
[**WSANSCLASSINFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsansclassinfow)Provides individual parameter information for a specific Winsock namespace.
[**WSAOVERLAPPED**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaoverlapped)Provides a communication medium between the initiation of an overlapped I/O operation and its subsequent completion.
[**WSAPOLLFD**](https://learn.microsoft.com/en-us/windows/win32/api/winsock2/ns-winsock2-wsapollfd)Stores socket information used by the [**WSAPoll**](https://learn.microsoft.com/en-us/windows/win32/api/winsock2/nf-winsock2-wsapoll) function.

[**WSAPROTOCOL_INFOW**](https://learn.microsoft.com/en-us/windows/win32/api/winsock2/ns-winsock2-wsaprotocol_infoa)Store or retrieves complete information for a given protocol. The protocol name is represented as an array of Unicode characters.
[**WSAPROTOCOLCHAIN**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaprotocolchain)Contains a counted list of Catalog Entry identifiers that comprise a protocol chain.
[**WSAQUERYSET**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaquerysetw)Provides relevant information about a given service.
[**WSAQUERYSET2**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaqueryset2w)Provides relevant information about a given service.
[**WSASERVICECLASSINFO**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaserviceclassinfow)Contains information about a specified service class.
[**WSAVERSION**](https://learn.microsoft.com/en-us/windows/desktop/api/Winsock2/ns-winsock2-wsaversion)Provides version comparison in Winsock.
