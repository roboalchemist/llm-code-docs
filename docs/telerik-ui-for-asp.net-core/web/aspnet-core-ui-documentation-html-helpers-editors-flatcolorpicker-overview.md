Updated on Dec 10, 2025

The Telerik UI FlatColorPicker TagHelper and HtmlHelper for ASP.NET Core are server-side wrappers for the Kendo UI FlatColorPicker widget.

The FlatColorPicker is the HSV color selector which is used by default in the `kendo.ui.FlatColorPicker` popup when no palette is set.

* [Demo page for the FlatColorPicker](https://demos.telerik.com/aspnet-core/flatcolorpicker)

[Initializing the FlatColorPicker](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#initializing-the-flatcolorpicker)
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

The following example demonstrates how to define the FlatColorPicker.

Razor

```Razor
@(Html.Kendo().FlatColorPicker()
          .Name("flatcolorpicker") // The name of the FlatColorPicker is mandatory. It specifies the "id" attribute of the widget.
          .Value("#ff0000") // Set the value of the FlatColorPicker.
    )
```

> Starting with the 2024 Q3 release, the HtmlHelper version of the component supports [declarative initialization](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/helper-basics/declarative-initialization).

[Events](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#events)
-------------------------------------------------------------------------------------------------------------------

You can subscribe to all FlatColorPicker [events](https://www.telerik.com/aspnet-core-ui/documentation/api/flatcolorpicker).

### [Handling by Handler Name](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#handling-by-handler-name)

The following example demonstrates how to subscribe to events by a handler name.

HtmlHelper TagHelper

Razor

```Razor
@(Html.Kendo().FlatColorPicker()
            .Name("flatcolorpicker")
            .Events(e => e
                  .Change("flatcolorpicker_change")
            )
      )
      <script>
          function flatcolorpicker_change(e) {
              //Handle the change event.
          }
      </script>
```

Razor

```Razor
@addTagHelper *, Kendo.Mvc
@addTagHelper *, Microsoft.AspNetCore.Mvc.TagHelpers

<script>
        function change(e) {
             console.log("Change in picker #" + this.element.attr("id") + " :: " + e.value);
        }
</script>

<kendo-flatcolorpicker name="flatColorPicker" value="#00f" on-change="change">
</kendo-flatcolorpicker>
```

### [Handling by Template Delegate](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#handling-by-template-delegate)

The following example demonstrates how to subscribe to events by a template delegate.

Razor

```Razor
@(Html.Kendo().FlatColorPicker()
            .Name("flatcolorpicker")
            .Events(e => e
                .Change(@<text>
                  function(e) {
                      // Handle the change event inline.
                  }
                  </text>)
            )
      )
```

[Referencing Existing Instances](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#referencing-existing-instances)
-------------------------------------------------------------------------------------------------------------------------------------------------------------------

To reference an existing Telerik UI FlatColorPicker instance, use the [`jQuery.data()`](https://api.jquery.com/jQuery.data/) configuration option. Once a reference is established, use the [FlatColorPicker client-side API](https://docs.telerik.com/kendo-ui/api/javascript/ui/flatcolorpicker#methods) to control its behavior.

Razor

```Razor
// Place the following after your Telerik UI FlatColorPicker for ASP.NET Core declaration.
  <script>
  $(function() {
      // The Name() of the FlatColorPicker is used to get its client-side instance.
      var flatcolorpicker = $("#flatcolorpicker").data("kendoFlatColorPicker");
  });
  </script>
```

[Functionality and Features](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/overview#functionality-and-features)
-----------------------------------------------------------------------------------------------------------------------------------------------------------

* [Views](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/views)—The FlatColorPicker enables you to choose between `gradient` or `palette` view types.
* [Contrast Tool](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/contrast-tool)—The FlatColorPicker enables you to configure the contrast tool.
* [Formats](https://www.telerik.com/aspnet-core-ui/documentation/html-helpers/editors/flatcolorpicker/rgb-hex-formats)—The FlatColorPicker provides support for both HEX and RGB coloring formats.
