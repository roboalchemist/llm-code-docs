# Exercise 6.2: User Journey Display

The User Journey displays the user’s journey through the conversation with the agent. This can be used to analyze the user’s experience with the agent that can also be used to improve the conversation flow.&#x20;

* To navigate to the User Journey navigate to **Agent-> Monitor -> User Journey**
