# Modify the Lambda function

1. Open your Lambda function and click on **Configuration**.
2. Select **Function URL** and then **Create function URL**.

   <figure><img src="https://2533899886-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MdwVZ6HOZriajCf5nXH%2Fuploads%2Fgit-blob-df8168f45f91924a8111e0bde5e7374cb9bb0284%2FScreenshot%202022-10-24%20at%2010.38.03%20AM.png?alt=media" alt=""><figcaption></figcaption></figure>
3. Change **Auth type** to **NONE** and save.

   <figure><img src="https://2533899886-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MdwVZ6HOZriajCf5nXH%2Fuploads%2Fgit-blob-732f3af1daa09764cad8db8c3adaf8b4f2efa9a8%2FScreenshot%202022-10-24%20at%2010.38.13%20AM.png?alt=media" alt=""><figcaption></figcaption></figure>
4. Copy the function URL to use in the next step of the guide.
5. The function URL is now ready for use and a resource-based policy has been added that you can verify under **Permissions**.
