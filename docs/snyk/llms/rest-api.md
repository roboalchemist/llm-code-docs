# REST API

This section provides an [introduction to the REST API](https://docs.snyk.io/snyk-api/rest-api/about-the-rest-api) and a resource to help you [get started](https://docs.snyk.io/snyk-api/rest-api/getting-started-with-the-rest-api).

For more information, see [Authentication for API](https://docs.snyk.io/snyk-api/authentication-for-api) and the [Changelog](https://docs.snyk.io/snyk-api/changelog) of updates to the [Reference](https://docs.snyk.io/snyk-api/reference).

{% hint style="info" %}
Note that the date updated in the changelog is not a valid version number. The valid version number precedes the date updated in each section. For example, in the section "2024-10-15 - Updated 2025-02-11," `2024-10-15` is the valid version number.
{% endhint %}

For more information about changes in the APIs, see [API End of Life (EOL) process and migration guides](https://docs.snyk.io/snyk-api/api-end-of-life-eol-process-and-migration-guides).

For help finding information about how to use the Snyk API, see [API endpoints index and notes](https://docs.snyk.io/snyk-api/api-endpoints-index-and-tips).
