# Organization-level integrations

Snyk provides seamless integrations with various source control management systems such as GitHub, GitLab, Bitbucket, and Azure Repos at the Organizational level. These integrations enable you to automatically scan for vulnerabilities and receive actionable insights to enhance the security of your repositories. By integrating at this level, you can ensure comprehensive protection for all your Projects within the Organization.

Snyk can integrate with the following SCMs at the Organization-level to help you track, monitor, and fix the issues and vulnerabilities in your code:

* [GitHub](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/github)
* [GitHub Enterprise](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/github-enterprise)
* [GitHub Cloud App](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/github-cloud-app)
* [GitHub Server App](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/github-server-app)
* [GitHub Read-only Projects](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/github-read-only-projects)
* [GitLab](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/gitlab)
* [Bitbucket Cloud](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/bitbucket-cloud)
* [Bitbucket Cloud App](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/bitbucket-cloud-app)
* [Bitbucket Data Center/Server](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/bitbucket-data-center-server)
* [Azure Repositories (TFS)](https://docs.snyk.io/developer-tools/scm-integrations/organization-level-integrations/azure-repositories-tfs)
