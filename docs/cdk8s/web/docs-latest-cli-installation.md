Install the cdk8s CLI[¶](https://cdk8s.io/docs/latest/cli/installation/#install-the-cdk8s-cli "Permanent link")
===============================================================================================================

This page describes the different ways you can install the Cloud Development Kit for Kubernetes (cdk8s) command-line interface (CLI) tool on your local machine.

Installation[¶](https://cdk8s.io/docs/latest/cli/installation/#installation "Permanent link")
---------------------------------------------------------------------------------------------

### Homebrew[¶](https://cdk8s.io/docs/latest/cli/installation/#homebrew "Permanent link")

To install the cdk8s CLI with [Homebrew](https://brew.sh/), run the following command:

```
brew install cdk8s
```

### npm[¶](https://cdk8s.io/docs/latest/cli/installation/#npm "Permanent link")

To install the cdk8s CLI with [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm), run the following command:

```
npm i -g cdk8s-cli
```

### Yarn[¶](https://cdk8s.io/docs/latest/cli/installation/#yarn "Permanent link")

To install the cdk8s CLI with [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable), run the following command:

```
yarn global add cdk8s-cli
```

Check version[¶](https://cdk8s.io/docs/latest/cli/installation/#check-version "Permanent link")
-----------------------------------------------------------------------------------------------

To check your version, run the following command:

```
cdk8s --version
```

 You should see something similar to the following: 

```
2.2.31
```
