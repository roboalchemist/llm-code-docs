# Get an action run's approvers

```
GET 
/v1/actions/runs/:run_id/approvers
```

This route allows you to fetch the approvers of a specific action run.<br /><br />To learn more about manual approval for actions, check out the [documentation](https://docs.port.io/create-self-service-experiences/set-self-service-actions-rbac/#configure-manual-approval-for-actions).
