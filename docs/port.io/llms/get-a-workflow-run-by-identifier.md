# Get a workflow run by identifier

```
GET 
/v1/workflows/runs/:identifier
```

This route allows you to retrieve detailed information about a specific workflow run, including its status and node runs.<br />To learn more about workflows, see the [documentation](https://docs.port.io/workflows/overview).
