# Update workflow node run acknowledgement

```
PATCH 
/v1/org/:org_id/workflows/runs/ack
```

This route allows you to acknowledge a previously claimed workflow node run to confirm it has been processed. Acknowledgement prevents the node run from becoming visible again after the visibility timeout expires.<br />To learn more about workflows, see the [documentation](https://docs.port.io/workflows/overview).
