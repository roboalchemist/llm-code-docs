# Update auto-discovery invocation suggestion

```
PATCH 
/v1/ai/entities-auto-discovery/:invocation_id/suggestions/:entity_identifier
```

This route allows you to update a suggestion for an auto-discovery invocation. If a suggestion is not accurate, you can update it with the correct information.<br />To learn more about the auto-discovery feature, see the [documentation](https://docs.port.io/build-your-software-catalog/catalog-auto-discovery).
