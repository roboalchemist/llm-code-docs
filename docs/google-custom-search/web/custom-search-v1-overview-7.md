**Uwaga:** interfejs Custom Search JSON API jest niedostępny dla nowych klientów. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=pl) to korzystna alternatywa do wyszukiwania w maksymalnie 50 domenach. Jeśli Twój przypadek użycia wymaga wyszukiwania w całej sieci, [skontaktuj się z nami](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=pl), aby wyrazić zainteresowanie naszym rozwiązaniem do wyszukiwania w całej sieci i uzyskać więcej informacji na jego temat.
Obecni klienci korzystający z interfejsu Custom Search JSON API mają czas na przejście na inne rozwiązanie do **1 stycznia 2027 roku**.

Interfejs Custom Search JSON API umożliwia tworzenie witryn i aplikacji, które programowo pobierają i wyświetlają wyniki wyszukiwania z Wyszukiwarki niestandardowej. Ten interfejs API umożliwia wysyłanie żądań REST w celu uzyskiwania wyników **wyszukiwania w internecie** lub **wyszukiwania obrazów** w formacie JSON.

Format danych
-------------

Interfejs Custom Search JSON API może zwracać wyniki w formacie danych [JSON](https://developers.google.com/custom-search/docs/glossary?hl=pl#json).

Powiązane dokumenty
-------------------

Interfejs Custom Search JSON API korzysta ze [specyfikacji OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Wymagania wstępne
-----------------

### Identyfikator wyszukiwarki

Zanim zaczniesz korzystać z interfejsu Custom Search JSON API, musisz utworzyć i skonfigurować Wyszukiwarkę programowalną. Jeśli nie masz jeszcze utworzonej Wyszukiwarki niestandardowej, możesz zacząć od odwiedzenia [panelu sterowania Wyszukiwarki niestandardowej](https://programmablesearchengine.google.com/controlpanel/all?hl=pl).

Aby dowiedzieć się więcej o różnych opcjach konfiguracji, zapoznaj się z tym [samouczkiem](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=pl).

Po utworzeniu Wyszukiwarki niestandardowej odwiedź [Centrum pomocy](https://support.google.com/programmable-search/answer/2649143?hl=pl), aby dowiedzieć się, jak znaleźć identyfikator wyszukiwarki.

### Klucz interfejsu API

Interfejs Custom Search JSON API wymaga użycia klucza interfejsu API.

Ceny
----

Poniższe ceny obowiązują tylko dotychczasowych klientów interfejsu Custom Search JSON API do momentu wycofania usługi 1 stycznia 2027 r. Ten interfejs API jest niedostępny dla nowych klientów. Interfejs Custom Search JSON API umożliwia bezpłatne wykonanie 100 zapytań dziennie. Jeśli potrzebujesz więcej, możesz zarejestrować się, aby korzystać z obsługi [płatności](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=pl) w Konsoli interfejsów API. Dodatkowe żądania kosztują 5 USD za 1000 zapytań, maksymalnie 10 tys.zapytań dziennie.

Monitorowanie
-------------

Podstawowe monitorowanie interfejsu Custom Search JSON API jest dostępne w[panelu informacyjnym interfejsów API w konsoli Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=pl). Do bardziej zaawansowanego monitorowania dostępny jest [pakiet operacyjny Google Cloud](https://cloud.google.com/products/operations?hl=pl) (wcześniej Stackdriver).

Dzięki Google Cloud Operations możesz tworzyć [panele niestandardowe](https://cloud.google.com/monitoring/charts?hl=pl), [konfigurować alerty](https://cloud.google.com/monitoring/alerts?hl=pl) i [uzyskiwać dostęp do danych o wskaźnikach w sposób programowy](https://cloud.google.com/monitoring/api/v3?hl=pl). Aby uzyskać dostęp do danych o wykorzystaniu interfejsu Custom Search JSON API w Google Cloud Operations, w Kreatorze zapytań wybierz „Typ zasobu: Wykorzystany interfejs API” i przefiltruj według „service = 'customsearch.googleapis.com'”.

Więcej informacji o różnych funkcjach monitorowania i alertów udostępnianych przez panel interfejsów API i pakiet Google Cloud Operations znajdziesz w artykule [Monitorowanie wykorzystania interfejsu API](https://cloud.google.com/apis/docs/monitoring?hl=pl).
