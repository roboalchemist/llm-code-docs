**Catatan:** Custom Search JSON API tidak lagi tersedia untuk pelanggan baru. [Vertex AI Search](https://cloud.google.com/use-cases/site-search?hl=id) adalah alternatif yang lebih baik untuk menelusuri hingga 50 domain. Atau, jika kasus penggunaan Anda memerlukan penelusuran web penuh, [hubungi kami](https://docs.google.com/forms/d/e/1FAIpQLSfYcIpOzj57MoetWyeFpJf9Cf4yezG5HDq3VsqbJhlngp5pXw/viewform?hl=id) untuk menyatakan minat Anda dan mendapatkan informasi selengkapnya tentang solusi penelusuran web penuh kami.
Pelanggan Custom Search JSON API yang sudah ada memiliki waktu hingga **1 Januari 2027** untuk beralih ke solusi alternatif.

Custom Search JSON API memungkinkan Anda mengembangkan situs dan aplikasi untuk mengambil dan menampilkan hasil penelusuran dari Programmable Search Engine secara terprogram. Dengan API ini, Anda dapat menggunakan permintaan RESTful untuk mendapatkan hasil **penelusuran web** atau **penelusuran gambar** dalam format JSON.

Format data
-----------

Custom Search JSON API dapat menampilkan hasil dalam format data [JSON](https://developers.google.com/custom-search/docs/glossary?hl=id#json).

Dokumen terkait
---------------

Custom Search JSON API menggunakan [Spesifikasi OpenSearch 1.1](https://github.com/dewitt/opensearch/blob/master/opensearch-1-1-draft-6.md).

Prasyarat
---------

### ID mesin telusur

Sebelum menggunakan Custom Search JSON API, Anda harus membuat dan mengonfigurasi Programmable Search Engine terlebih dahulu. Jika belum membuat Programmable Search Engine, Anda dapat memulainya dengan membuka [panel kontrol Programmable Search Engine](https://programmablesearchengine.google.com/controlpanel/all?hl=id).

Ikuti [tutorial](https://developers.google.com/custom-search/docs/tutorial/creatingcse?hl=id) untuk mempelajari lebih lanjut berbagai opsi konfigurasi.

Setelah membuat Programmable Search Engine, buka [pusat bantuan](https://support.google.com/programmable-search/answer/2649143?hl=id) untuk mempelajari cara menemukan ID Mesin telusur Anda.

### Kunci API

Custom Search JSON API memerlukan penggunaan kunci API.

Harga
-----

Harga berikut hanya berlaku untuk pelanggan Custom Search JSON API yang sudah ada hingga penghentian layanan pada 1 Januari 2027. API ini tidak tersedia untuk pelanggan baru. Custom Search JSON API menyediakan 100 kueri penelusuran per hari secara gratis. Jika memerlukan lebih banyak, Anda dapat mendaftar ke [penagihan](https://cloud.google.com/billing/docs/how-to/manage-billing-account?hl=id) di Konsol API. Permintaan tambahan dikenai biaya $5 per 1.000 kueri, hingga 10 ribu kueri per hari.

Pemantauan
----------

Pemantauan dasar untuk Custom Search JSON API tersedia melalui [Dasbor API Konsol Cloud Platform](https://console.cloud.google.com/apis/dashboard?hl=id). Untuk pemantauan yang lebih canggih, [Google Cloud Operations Suite](https://cloud.google.com/products/operations?hl=id) (sebelumnya bernama Stackdriver) tersedia.

Dengan Google Cloud Operations, Anda dapat membuat [dasbor kustom](https://cloud.google.com/monitoring/charts?hl=id), [menyiapkan pemberitahuan](https://cloud.google.com/monitoring/alerts?hl=id), dan [mengakses data metrik secara terprogram](https://cloud.google.com/monitoring/api/v3?hl=id). Untuk mengakses data penggunaan Custom Search JSON API di Google Cloud Operations, pilih "Jenis resource: API yang digunakan" dan filter "service = 'customsearch.googleapis.com'" di Pembuat Kueri.

Lihat [Memantau Penggunaan API Anda](https://cloud.google.com/apis/docs/monitoring?hl=id) untuk mengetahui pembahasan tentang berbagai kemampuan pemantauan dan pemberitahuan yang disediakan oleh Dasbor API dan Google Cloud Operations Suite.
