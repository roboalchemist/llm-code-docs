# Identity and Access Management documentation

[Read product documentation](https://docs.cloud.google.com/iam/docs/overview)

Identity and Access Management (IAM) lets you create and manage permissions for Google Cloud resources. IAM unifies access control for Google Cloud services into a single system and presents a consistent set of operations. 

[Go to the Identity and Access Management product page for more.](https://cloud.google.com/security/products/iam)
