Generating
==========

[](https://github.com/hexojs/site/edit/master/source/docs/generating.md "Improve this doc")

Generating static files with Hexo is quite easy and fast.

$ hexo generate

### [](https://hexo.io/docs/generating#Watch-for-File-Changes "Watch for File Changes")Watch for File Changes[](https://hexo.io/docs/generating#Watch-for-File-Changes)

Hexo can watch for file changes and regenerate files immediately. Hexo will compare the SHA1 checksum of your files and only write if file changes are detected.

$ hexo generate --watch

### [](https://hexo.io/docs/generating#Deploy-After-Generating "Deploy After Generating")Deploy After Generating[](https://hexo.io/docs/generating#Deploy-After-Generating)

To deploy after generating, you can run one of the following commands. There is no difference between the two.

$ hexo generate --deploy

$ hexo deploy --generate
