Entity Framework Core tools reference
=====================================

Feedback

 Summarize this article for me 

In this article
---------------

The Entity Framework Core tools help with design-time development tasks. They're primarily used to manage Migrations and to scaffold a `DbContext` and entity types by reverse engineering the schema of a database.

Either of the following tools can be installed, as both tools expose the same functionality:

*   The [EF Core Package Manager Console tools](https://learn.microsoft.com/en-us/ef/core/cli/powershell) run in the [Package Manager Console](https://learn.microsoft.com/en-us/nuget/tools/package-manager-console) in Visual Studio. We recommend using these tools if you are developing in Visual Studio as they provide a more integrated experience.

*   The [EF Core .NET command-line interface (CLI) tools](https://learn.microsoft.com/en-us/ef/core/cli/dotnet) are an extension to the cross-platform [.NET CLI tools](https://learn.microsoft.com/en-us/dotnet/core/tools/). These tools require a .NET SDK project (one with `Sdk="Microsoft.NET.Sdk"` or similar in the project file).
