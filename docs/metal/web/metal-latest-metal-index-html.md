[§](https://docs.rs/metal/latest/metal/index.html#-deprecated-)⚠️ Deprecated ⚠️
-------------------------------------------------------------------------------

Use of this crate is deprecated as the [`objc`](https://crates.io/crates/objc) ecosystem of mac system bindings are unmaintained. For new development, please use [`objc2`](https://crates.io/crates/objc2) and [`objc2-metal`](https://crates.io/crates/objc2-metal) instead. We will continue to merge basic PRs and keep things maintained, at least as long as it takes to migrate [`wgpu`](https://crates.io/crates/wgpu) to the `objc2` ecosystem [PR 5641](https://github.com/gfx-rs/wgpu/pull/5641).
