# Create a Dynamic Release Bundle (v1)

You can distribute a dynamic Release Bundle, which is a Release Bundle v1 that you can create, sign, and distribute on the fly using the [Dynamic Release Bundle REST API](/reference/distributedynamicreleasebundlev1). The Release Bundle version is created instantly and distributed to the selected Distribution Nodes. The distribution process for a Dynamic Release Bundle is the same as a regular Release Bundle v1.
