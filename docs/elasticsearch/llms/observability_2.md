# Troubleshoot Observability
Learn how to troubleshoot common issues on your own or ask for help.

## Tools

Use the following tools in Kibana to troubleshoot issues across Observability apps:
- [Explore data](https://www.elastic.co/docs/troubleshoot/observability/explore-data): Select and filter result data in any dimension, and look for the cause or impact of performance problems.
- [Inspect](https://www.elastic.co/docs/troubleshoot/observability/inspect): View information about all requests that were made to collect the data displayed on the current page.
