# Snapshots

# Permissions

Users can be given **manage** or **upload** permissions.

![477](https://files.readme.io/6ad23bd-image.png "image.png")

The following table describes the permissions required. Operations that require **view** permission are available to all users in a Snapshots-enabled site.
