#### User interface [](https://docs.gtk.org/#user-interface)

##### GTK

GTK is the primary library used to construct user interfaces. It provides user interface controls and signal callbacks to respond to user actions.

[GTK API reference](https://docs.gtk.org/gtk4/)

##### GSK

An intermediate layer which provides a rendering API implemented using Cairo, OpenGL or Vulkan.

[GSK API reference](https://docs.gtk.org/gsk4/)

##### GDK

An intermediate layer which isolates GTK from the details of the windowing system.

[GDK API reference](https://docs.gtk.org/gdk4/)

##### Pango

Pango is the core text and font handling library used in GTK applications. It has extensive support for the different writing systems used throughout the world.

[Pango API reference](https://docs.gtk.org/Pango/)

##### Cairo

Cairo is a 2D graphics library with support for multiple output devices. It is designed to produce consistent, high quality output on all media.

[Cairo API reference](https://www.cairographics.org/manual/)

#### Core libraries [](https://docs.gtk.org/#core-libraries)

##### GIO

GIO provides a portable, modern and easy-to-use file system abstraction API for accessing local and remote files; a set of low and high level abstractions over the [DBus](https://www.freedesktop.org/wiki/Software/dbus/) IPC specification; an application settings API; portable networking abstractions; and additional utilities for writing asynchronous operations without blocking the user interface of your application.

[GIO API reference](https://docs.gtk.org/gio/)

[GIO Unix-specific API reference](https://docs.gtk.org/gio-unix/)

[GIO Windows-specific API reference](https://docs.gtk.org/gio-win32/)

##### GIRepository

GIRepository provides access to introspected API bindings built using gobject-introspection.

[GIRepository API reference](https://docs.gtk.org/girepository/)
