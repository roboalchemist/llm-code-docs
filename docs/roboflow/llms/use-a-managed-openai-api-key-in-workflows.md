# Use a Managed OpenAI API Key in Workflows

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FWgfgOSKE4iAltGjroDa8%2FScreenshot%202025-06-23%20at%2011.37.45.png?alt=media&#x26;token=53a91297-4c06-417c-a3ed-2d433d185eeb" alt=""><figcaption></figcaption></figure>

You can now use a Roboflow-managed OpenAI API Key in your Workflows.

When you use the Roboflow-managed key, you will be billed based on usage according to our [Credits-based billing](https://roboflow.com/credits).

To use a Roboflow-managed key, set the "Api Key" field in a OpenAI Workflow block to "Roboflow Managed API Key".
