# RF-DETR - New Sizes Available

New RF-DETR (Roboflow Detection Transformer) sizes were added with support for model training and deployment. This delivers better flexibility when choosing the right mix of accuracy and latency for your computer vision project. &#x20;

New model sizes released:&#x20;

* RF-DETR Segmentation: Nano, Small, Medium, Large, X Large, 2X Large
* RF-DETR Object Detection: X Large and 2X Large&#x20;
