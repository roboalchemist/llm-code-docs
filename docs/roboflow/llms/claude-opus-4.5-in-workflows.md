# Claude Opus 4.5 in Workflows

We’ve added Claude Opus 4.5 to the Anthropic block. We also introduced API key passthrough, allowing you to run Anthropic models (including Opus 4.5, Sonnet 4.5, and Haiku 4.5) directly through Roboflow. This consolidates billing and access, eliminating the need to manage a separate Anthropic API key.
