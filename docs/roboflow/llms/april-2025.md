# April 2025

### Model Training

You can now train and deploy RF-DETR models with Roboflow Inference.

### Deployments

* Moondream 2 is now available in Inference and Workflows.
* SmolVLM is now available in Inference and Workflows.
* You can upload RF-DETR weights to Roboflow for deployment with Inference and our cloud deployment offerings.
* Speed improvements to Inference.

### Workflows

* Enterprise customers can now send data to Microsoft SQL Server with the Microsoft SQL Server Workflow block.
