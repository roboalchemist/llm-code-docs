# Use Label Assist with Classification Projects

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FbVn7naZ9Zt10Kf38q8t6%2FScreenshot%202025-05-29%20at%2010.27.15.png?alt=media&#x26;token=d0ca88ab-eefe-40ee-969a-0dab2e99364c" alt=""><figcaption><p>The Label Assist widget in the Roboflow Anotate interface.</p></figcaption></figure>

You can now use [Label Assist](https://docs.roboflow.com/annotate/ai-labeling/model-assisted-labeling) with Classification Projects. Label Assist lets you use versions of your model to annotate images in your dataset.&#x20;

To access this feature, click the magic wand icon in the tray on the right side of your Roboflow Annotate window. You will then be able to select a previous model you have trained for use with Label Assist.
