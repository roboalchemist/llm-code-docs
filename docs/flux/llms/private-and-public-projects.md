# Private and public projects

Let's talk about public projects and private projects. This distinction is important to identify which [Flux plan](https://www.flux.ai/p/pricing) is right for you.

**What is a private project?**

Projects and parts are considered private when they can be accessed only by you or individually selected people. To keep projects private, **the option "PRIVATE - Only you can view an edit" must be selected** in the [share dialog](https://docs.flux.ai/flux/reference/reference-sharing-and-permissions). You can still add people individually through the Advanced menu.

To quickly find out if a project is **private**, look for a padlock icon next to the share button:

![](https://uploads.developerhub.io/prod/86Yw/pe6dcsqx5hclroj90a6uw50shcbsz5xi2l9k9jhel1cr97ch15l917euwr9v1wko.png)

Note: new projects are private by default.

**What is a public project?**

Projects are considered **public when they can be accessed by anyone**. This means you've selected one of the options below in the share dialog:

- Anyone can edit
- Anyone can view
- Anyone can comment

To quickly find out if a project is **public**, look for one of the following icons next to the share button:
