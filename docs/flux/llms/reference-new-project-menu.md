# Project Launcher

A launching pad for your next project.

![](https://uploads.developerhub.io/prod/86Yw/y8g243rfuq0xmbk67lc7e5h1ns04bksinvejwqhl58p5l7ido0dcoqavyzj3eekq.png)

## Overview

Flux is built for re-usability from the ground up. That means any project you can find in Flux can be used as a starting point for your work; wether someone else in the community has made a  project you can use as a base for yours, or you create your own templates to get started quicker. 

The new project dialog allows you to explore what others are doing, re-use your own projects or start a new blank project.

## Templates

### All Templates

Never start from scratch. In Flux, every public project with `#template` in its description can be used as a template. This approach gets you closer to your goals more quickly with less work and less risk.

![](https://uploads.developerhub.io/prod/86Yw/cghw4n080oucjbltfkn0e327ciplvp8a9p54cg55cpec91obel8osrkep4dgixjd.png)

### My Templates

Creating your own templates are a handy way of creating similar parts or projects with similar characteristics. You can include PCB layout rules, parts, or terminals that will be automatically added to projects created from the template. [Learn more about using templates](https://docs.flux.ai/flux/tutorials/reusing-community-projects#create-your-own-templates)

![](https://uploads.developerhub.io/prod/86Yw/c9ac5r5rydnd1626ucicf6qoux1xgq6tfb6of27tdqo8svw8w9dsgglicabna8gi.png)

### Starred Templates

Use the [Starred](https://docs.flux.ai/flux/reference/star-project-reference) templates option to filter by templates you've starred in the past.

![](https://uploads.developerhub.io/prod/86Yw/txz8lo7q1x5mpqimktl6grj6x76n3jse3ptml04c10ldjlo57aeretbozjqxkcj0.png)
