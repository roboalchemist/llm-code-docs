# Grafbase Login Command

Log in to your Grafbase account.

**Usage:**

```bash
grafbase login
```

## Examples

Log in to your Grafbase account:

```bash
grafbase login
```

This command will open a browser window where you can log in to your Grafbase account. After logging in, you will be redirected back to the CLI.
