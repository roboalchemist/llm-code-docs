Docs for Developers
===================

MantisBT [source code](https://github.com/mantisbt/mantisbt) and [plugins](https://github.com/mantisbt/mantisbt-plugins) are hosted on GitHub. Detailed information on how to use Git with MantisBT can be found in the Mantis Bug Tracker Developers Guide, and the MantisBT Wiki provides various links and resources for MantisBT contributors. Help with setting up Git and learning basic usage can be found in GitHub's Help pages.

*   [Developer's Guide](https://mantisbt.org/docs/master/en-US/Developers_Guide)

Docs for Translators
====================

MantisBT ships in [50 languages](https://github.com/mantisbt/mantisbt/tree/master/lang) providing it a great penetration around the world. Anyone can contribute improvements to existing languages or even new languages via [translatewiki.net](https://translatewiki.net/wiki/Translating:MantisBT). For more details check the ['Translators How To Guide'](https://mantisbt.org/wiki/doku.php/mantisbt:translationshowto).

#### About MantisBT

MantisBT is a popular open source, web-based bug tracking system. Built on PHP, it supports Linux, Windows and macOS on the server side. Compatible with Chrome, Firefox, Safari, Opera and Edge. Released under the terms of the GNU General Public License (GPL).
