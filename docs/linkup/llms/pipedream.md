# Pipedream

> How to use Linkup in Pipedream

## Overview

Linkup can be used with [Pipedream](https://pipedream.com) as an Action to get contextual information from the internet.

## Installation

<Steps>
  <Step title="Create your project">
    1. Open your Pipedream account and click on **New project**

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/new_project.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=f6dd99f841acb44b6d0a159abde9b4fd" alt="Pipedream" width="2968" height="228" data-path="pages/integrations/pipedream/assets/new_project.png" />

    2. Name your project and click on **Create Project**

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/linkup_project.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=a10ba1afe5c59f0af7f4272206059dbe" alt="Pipedream" width="1364" height="590" data-path="pages/integrations/pipedream/assets/linkup_project.png" />

    3. Click on **New** > **Workflow**.

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/click_on_workflow.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=e6e6f37338aa6236bb46645ca28a1e54" alt="Pipedream" width="2558" height="1278" data-path="pages/integrations/pipedream/assets/click_on_workflow.png" />

    4. Name your workflow and click on **Create Workflow**

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/name_workflow.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=8f2704a5b0b4716b8b17a8fe053866e5" alt="Pipedream" width="1470" height="1446" data-path="pages/integrations/pipedream/assets/name_workflow.png" />
  </Step>

  <Step title="Select your Linkup Action">
    1. On the workflow page, click on the `+` icon to add the Linkup Action

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/add_action.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=520d718f29c5906ad55dbe70913db591" alt="Pipedream" width="1926" height="688" data-path="pages/integrations/pipedream/assets/add_action.png" />

    2. Search for **Linkup** and Click on it

        <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/linkup_tool.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=c67dfa0bc4157376530fd799301087a8" alt="Pipedream" width="2928" height="1292" data-path="pages/integrations/pipedream/assets/linkup_tool.png" />

    3. Select the Linkup Action you want to use
  </Step>

  <Step title="Connect your Linkup Account">
    1. Get your Linkup API Key

    <Card title="Get your API key" icon="key" href="https://app.linkup.so" horizontal="True">
      Create a Linkup account for free to get your API key.
    </Card>

    2. Go to create a Linkup account, enter your API Key and choose your connection Name

    <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/connect_linkup.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=e84d9d7f9270b0c9261b15632f99cee9" alt="Pipedream" width="2874" height="1462" data-path="pages/integrations/pipedream/assets/connect_linkup.png" />
    <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/linkup_api_key.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=16902cfc185f5ddf59778f1c1cf51348" alt="Pipedream" width="1522" height="672" data-path="pages/integrations/pipedream/assets/linkup_api_key.png" />
  </Step>

  <Step title="Run your Action">
    1. Enter the Query you want to search for and the depth you want to use

    <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/enter_query.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=397741b35567702cb0754e18faf655e4" alt="Pipedream" width="2602" height="754" data-path="pages/integrations/pipedream/assets/enter_query.png" />
    <img src="https://mintcdn.com/linkup-8b5c238e/ht3XJF6azWwbELaV/pages/integrations/pipedream/assets/choose_depth.png?fit=max&auto=format&n=ht3XJF6azWwbELaV&q=85&s=3979493875042eaa992880c063a93a9c" alt="Pipedream" width="2594" height="1106" data-path="pages/integrations/pipedream/assets/choose_depth.png" />

    2. ✅ Click on **Test** to run your action
  </Step>
</Steps>

You are all set to use Linkup in your Pipedream workflow! Visit the [Concepts](/pages/documentation/get-started/concepts) page to learn more about the different Linkup parameters.
