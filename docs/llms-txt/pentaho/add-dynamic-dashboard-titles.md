# Add dynamic dashboard titles

While in dashboard, each panel has the ability to show a title with current parameters.

1. Click the **Edit** (pencil) icon in the top toolbar.

   The **Edit** pane appears at the bottom of the screen.
2. Click the report that has the parameters you are going to change.

   You will need to go through this process for each report to which you want to add parameters.
3. Click the **Add Parameter** button.

   Placed parameter appears after the title.

   ![Editing parameter](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-4347de3f0d58a577bc9e575af4a87036183d910a%2FPDD_parameter_added_callout_ONYX.png?alt=media)
4. Click **Apply**.
5. Close the **Edit** pane by clicking the **Edit** (pencil) icon.

The title now shows the parameters applied to it.
