# Change page format

Perform the following steps to change the page format:

1. Click the **General** tab, then click **Page Setup**.
2. In the Page Setup dialog box, choose the correct page format for your report. Page margin measurements are displayed in the lower part of the dialog box. You can adjust the measurements as needed.
3. Click **OK** to save your changes.
