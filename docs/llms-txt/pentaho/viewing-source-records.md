# Viewing source records

View source records to review the data elements in the source record or to obtain information for tracing the record. For example, a Data Steward can trace a source record by using the Source Record Table Key and a Source Application Identifier.

Source records contain the raw data that you collect from specified data locations, such as CRMs, ERPs, and SCMs. The raw data in Source Records is cleaned, standardized, and stored as unmastered data in a data lake or data warehouse so that it can be processed into Master Records.

Perform the following steps to view source records:

1. On the left navigation menu, click **Master Data**. The Master Data page opens with the Master Records listed in a table.
2. Click the **Source** tab. The Source Data page appears with the source records listed in a table. Each source record has a Source Record Table Key and a Source Application Identifier, which are unique values that the data steward can use to trace the record. All other data elements in the source data table are configured by the user.
