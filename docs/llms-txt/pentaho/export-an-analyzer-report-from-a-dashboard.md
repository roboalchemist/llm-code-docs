# Export an Analyzer report

You can export an Analyzer report from a dashboard as a PDF, CSV, or an Excel workbook file.

1. In the dashboard toolbar, click **Actions**.
2. Select one of the following available output formats from the menu that displays:
   * **Export to PDF**
   * **Export to CSV**
   * **Export to Excel Workbook**\
     Your selection is the default export format until you change it.

PUC generates the report and downloads it to the browser's default download folder.
