# Java virtual machine

Pentaho software requirements for Java Runtime Environment (JRE).

| Pentaho Software     | Certified                                                            | Supported                                                                                                                                     |
| -------------------- | -------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| All Pentaho software | <ul><li>Oracle Java 11.0.13</li><li>Oracle OpenJDK 11.0.13</li></ul> | <ul><li>Oracle Java 8.x & 11.x</li><li>Oracle OpenJDK 8.x & 11.x</li><li>Eclipse Temurin by Adoptium</li><li>Zulu from Azul Systems</li></ul> |

\*\*Note:\*\* Some Hadoop clusters using Java 8 may not be fully compatible when running Pentaho with Java 11, see \[Compatability issues running Pentaho on Java 11 with your Hadoop cluster]\(../Pentaho%20Java%2011%20and%20Hadoop%20cluster%20compatibility%20(Components%20reference%20cp).md) for details.
