# Step 1: Download files

To manually install Pentaho Business Analytics design tools, you must first download the compressed files that contain the software. Pentaho Business Analytics design tools include Aggregation Designer, Metadata Editor, Report Designer, and Schema Workbench.

Complete the following steps to download files for client tools from the [Support Portal](https://support.pentaho.com/home).

1. On the [Support Portal](https://support.pentaho.com/hc/en-us) home page, sign in using the Pentaho support username and password provided in your Pentaho Welcome Packet.
2. In the Pentaho card, click **Download**.

   The Downloads page opens.
3. In the **10.x** list, click **See all \<number> articles** to see the full list of **10.x** downloads.
4. On the **10.x** page, click **Pentaho 10.2 GA Release**.
5. Scroll to the bottom of the Pentaho 10.2 GA Release page.
6. In the file component section, navigate to the `Client Tools` folder.
7. Open the folder for each design tool, and then download the following files from each folder.

   The following table lists the design tool folders and the files to download from each folder.

| Folder                 | Files to download                                                                                                                                                                                                                                                                                                                                                                     |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Aggregation Designer` | `pad-ee-10.2.0.0-<build number>.zip`                                                                                                                                                                                                                                                                                                                                                  |
| `Metadata Editor`      | `pme-ee-10.2.0.0-<build number>.zip`**Note:** If you plan to use HIVE as data source for building your model or report, you must also download `pme-ee-10.2.0.0-<build number>-hadoop-addon.zip`.                                                                                                                                                                                     |
| `Report Designer`      | <p><code>prd-ee-10.2.0.0-\<build number>.zip</code><strong>Notes:</strong></p><ul><li>If you plan to use HIVE as data source for building your model or report, you must also download <code>prd-ee-10.2.0.0-\<build number>-hadoop-addon.zip</code>.</li><li>For <code>prd-ee</code> Mac OS users, the unique file is included as part of the <code>prd-ee</code> package.</li></ul> |
| `Schema Workbench`     | `psw-ee-10.2.0.0-<build number>.zip`                                                                                                                                                                                                                                                                                                                                                  |
