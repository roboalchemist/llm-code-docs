# Adding data sources in bulk

Add multiple data sources at one time in the Pentaho Data Mastering by importing a JSON file that contains all the information for the data sources.

Perform the following steps to create multiple data sources at one time:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Resources** card, click **Data Sources**.

   The Data Sources page opens.
3. Click **Add Data Source** and select **Import Data Source(s)**.

   The Import Data Source(s) dialog box opens.
4. Browse to and select the JSON file that contains the data sources information to import.

   **Tip:** You can drag and drop the JSON file into the Import Data Source(s) dialog box.
5. Click **Continue**.

   A confirmation message is shown in the top-right corner of the page and the new data sources are added to the data source table.
