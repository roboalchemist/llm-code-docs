# Starting PDI tools and utilities

As part of the installation, we provide different ways to help you start each tool and utility, depending on the operating system you use. This table shows you the options.

| Operating System  | Installation Method                             | Start the Tools and Utilities                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------- | ----------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Microsoft Windows | Pentaho Installation Wizard                     | <ul><li><strong>PDI client</strong></li></ul><p>Click <strong>Start</strong> > <strong>Programs</strong> > <strong>Pentaho Enterprise Edition</strong> > <strong>Design Tools</strong> and double-click on <strong>Data Integration</strong> the item to launch the PDI client. For these tools and utilities, you must navigate to a directory to launch the batch file.</p><ul><li><strong>Carte</strong></li></ul><p><code>pentaho\design-tools\data-integration\carte.bat</code></p><ul><li><strong>Kitchen</strong></li></ul><p><code>pentaho\design-tools\data-integration\kitchen.bat</code></p><ul><li><strong>Pan</strong></li></ul><p><code>pentaho\design-tools\data-integration\pan.bat</code></p>   |
| Microsoft Windows | Pentaho Archive or Manual Installation          | <p>Create shortcuts to the .bat file or each design tool and add them to the <a href="http://windows.microsoft.com/en-us/windows7/customize-the-start-menu">Start menu</a>. Or, navigate to the one of these directories and run the design tool's batch file to launch it.- <strong>PDI client</strong></p><p><code>pentaho\design-tools\data-integration\spoon.bat</code></p><ul><li><strong>Carte</strong></li></ul><p><code>pentaho\design-tools\data-integration\carte.bat</code></p><ul><li><strong>Kitchen</strong></li></ul><p><code>pentaho\design-tools\data-integration\kitchen.bat</code></p><ul><li><strong>Pan</strong></li></ul><p><code>pentaho\design-tools\data-integration\pan.bat</code></p> |
| Linux             | Pentaho Wizard, Archive, or Manual Installation | <p>Navigate to one of these directories and run the design tool shell file to launch it.- <strong>PDI client</strong></p><p><code>pentaho/design-tools/data-integration/spoon.sh</code></p><ul><li><strong>Carte</strong></li></ul><p><code>pentaho/design-tools/data-integration/carte.sh</code></p><ul><li><strong>Kitchen</strong></li></ul><p><code>pentaho/design-tools/data-integration/kitchen.sh</code></p><ul><li><strong>Pan</strong></li></ul><p><code>pentaho/design-tools/data-integration/pan.sh</code></p>                                                                                                                                                                                        |
| Mac OS            | Pentaho Wizard, Archive, or Manual Installation | <p>Navigate to one of these directories and run the design tool application file to launch it.- <strong>PDI client</strong></p><p><code>pentaho/design-tools/data-integration/data integration.app</code></p><ul><li><strong>Carte</strong></li></ul><p><code>pentaho/design-tools/data-integration/carte.app</code></p><ul><li><strong>Kitchen</strong></li></ul><p><code>pentaho/design-tools/data-integration/kitchen.app</code></p><ul><li><strong>Pan</strong></li></ul><p><code>pentaho/design-tools/data-integration/pan.app</code></p>                                                                                                                                                                   |
