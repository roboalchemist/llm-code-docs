# Flag a filter as a dashboard parameter

You can flag a filter constraint as a Dashboard parameter. The parameter name will appear on the **Parameters**tab in Dashboard Designer when you place the report into a dashboard panel. You can also edit parameters while working in Report Designer.

Perform the following steps to flag a filter:

1. After you create your filter, enter the parameter name in the **Parameter Name** text box.
2. Click **OK**.
3. Save your report.

The parameter is ready to be used in Dashboard Designer. To see if the parameter appears in Dashboard Designer, insert your report into a dashboard panel. The parameter name will appear on the **Parameters** tab.
