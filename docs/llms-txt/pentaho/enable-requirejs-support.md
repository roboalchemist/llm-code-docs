# Enable RequireJS support

To include RequireJS, in the Settings dialog box, select the **RequireJS Support** check box when you save a dashboard.

**Note:** In Pentaho 7.0, new CDE dashboards will default to RequireJS dashboards. Regardless of the default behavior for new dashboards, your previously created non-RequireJS dashboards will render the same as before, and you can continue editing them without upgrading the dashboards to RequireJS support.
