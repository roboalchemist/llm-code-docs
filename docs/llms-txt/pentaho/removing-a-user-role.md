# Removing a user role

Remove a user role that you no longer want to use for controlling access and permissions in Pentaho Data Mastering application.

You must have admin privileges to remove user roles.

Perform the following steps to remove a user role:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Users** card, click **Roles**.

   The Roles page opens with the user roles shown in a table.
3. At the end of the row with the user role that you want to remove, click **Remove**.

   The Remove Role confirmation dialog box appears.
4. Click **Remove**.

   A confirmation notification message with the status of the removal is displayed in the top-right corner of the page.
