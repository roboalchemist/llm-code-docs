# Metadata injection support

You can use the Metadata Injection supported fields with [ETL metadata injection](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/pdi-transformation-steps-reference-overview/etl-metadata-injection) step to pass metadata to your transformation at runtime. The following **Value** fields of the Microsoft Excel writer step support metadata injection:

* **Name**
* **Type**
* **Format**
* **Style from cell**
* **Field title**
* **Header/Footer style from cell**
* **field contains formula**
* **Hyperlink**
* **Cell comment (XLSX)**
* **Cell comment author (XLSX)**
