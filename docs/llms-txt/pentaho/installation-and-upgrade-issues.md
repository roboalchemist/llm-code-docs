# Troubleshooting possible installation and upgrade issues

Follow the suggestions in these topics to help resolve common issues associated with installing and upgrading your Pentaho system:

* [User Console themes render improperly after upgrade](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/user-console-themes-render-improperly-after-upgrade)
* [Context XML changes do not take effect after deploying a WAR](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/context-xml-changes-do-not-take-effect-after-deploying-a-war)
* [No search manager configured for the Java repository](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/no-search-manager-configured-for-the-java-repository)
* [Cannot create Jackrabbit tables in MySQL](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/cannot-create-jackrabbit-tables-in-mysql)
* [Unable to use the database init scripts for PostgreSQL](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/unable-to-use-the-database-init-scripts-for-postgresql)
* [Not able to login or run transformations](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/not-able-to-login-or-run-transformations)
* [Error when using sample transformations or jobs](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/error-when-using-sample-transformations-or-jobs)
* [Missing Quartz library database error](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-and-upgrade-issues/missing-quartz-library-database-error)

See the [**Administer Pentaho Data Integration and Analytics**](https://app.gitbook.com/o/PtpmPYUKgAsUWgv8SVUt/s/TSPdUdWBWVGi0uurnXBG/) document for additional troubleshooting information.
