# Download and install DockMaker command line tool

Download and install the DockMaker command line tool so that you can build and deploy Pentaho containers.

Complete the following steps to download and install the DockMaker command line tool.

1. On the [Support Portal](https://support.pentaho.com/hc/en-us) home page, sign in using the Pentaho support username and password provided in your Pentaho Welcome Packet.
2. In the Pentaho card, click **Download**.

   The Downloads page opens.
3. In the **10.x** list, click **Pentaho 10.2 GA Release**.
4. Scroll to the bottom of the Pentaho 10.2 GA Release page.
5. In the folder widget, navigate to the `Utilities and Tools\DockMaker Tool` folder.
6. Download the `dock-maker-10.2.0.0-<build number>.zip` file.
7. Unpack the `dock-maker-10.2.0.0-<build number>.zip` file into a temporary directory.

   The temporary directory contains scripts for installing the DockMaker command line tool.
8. To install the DockMaker command line tool, run one of the following scripts:
   * `DockMaker.sh` for Linux
   * `DockMaker.bat` for Windows
