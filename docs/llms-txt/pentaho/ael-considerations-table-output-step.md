# AEL considerations

When using the **Table output** step with [Adaptive Execution Layer](https://docs.pentaho.com/pdia-data-integration/9.3-data-integration/advanced-topics-pentaho-data-integration-overview/adaptive-execution-layer-cp-landing-page) (AEL), consider the following factors for Hive database connections, which can affect performance and results.

* You can use Hortonworks to access Hive managed tables in ORC format, as described in the **Administer Pentaho Data Integration and Analytics** document.
