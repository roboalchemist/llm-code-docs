# Security tab

![Security tab](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-98dd341e38136f288caac798c10af99a8f9f7193%2FPDITransStep_AMQPConsumer_SecurityTab.png?alt=media)

The **Security** tab allows you to define authentication credentials for the AMPQ broker. This tab includes the following options:

| Option                  | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Username**            | Specify the user name required to access the AMQP broker.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Password**            | Specify the password associated with the *Username*.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Use secure protocol** | Select this option if you want to define SSL properties for the connection.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **SSL Properties**      | <p><em>Context Algorithm</em>: Specify the name of the secure protocol you are using.</p><p><em>Key Store Password</em>: Specify the password for the key store you want this connection to use.</p><p><em>Key Store Path</em>: Specify the file path location of the key store you want this secure connection to use.</p><p><em>Key Store Type</em>: Specify the identifying name or string for the type of key store.</p><p><em>Trust Store Password</em>: Specify the password for the trust store object that you want this secure connection to use.</p><p><em>Trust Store Path</em>: Specify the file path location of the trust store certificates you want this secure connection to use.</p><p><em>Trust Store Type</em>: Specify the format of the trust store.</p> |
