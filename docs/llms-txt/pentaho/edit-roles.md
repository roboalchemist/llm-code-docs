# Edit roles

1. Click **Roles**.

   The list of available roles appear.
2. Select the role you want to edit and click the **Edit** icon.

   The Edit Role dialog box appears.
3. Make the appropriate changes.
4. Click **OK** to save your changes and exit the Edit Role dialog box.
