# General

The following fields and button are general to this transformation step:

* **Step name**: Specify the unique name of the Avro Input step on the canvas. You can customize the name or leave it as the default.

You can use **Preview** to display the rows generated by this step. The Avro Input step determines what rows to input based on the information you provide in the option tabs. This preview function helps you to decide if the information provided accurately models the rows you are trying to retrieve.
