# Step 3: (Optional) Unpack the plugin files

If you downloaded plugin files for Pentaho components, you must unpack the files and move the contents to the correct directory so that you can use the components.

1. Locate the Pentaho component plugin files that you downloaded in [Step 1: Download files](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/archive-installation/archive-installation-process/prepare-your-windows-environment-for-an-archive-install/download-and-unpack-installation-files/step-1-download-files-reuse).
2. Unpack the following Pentaho component plugin files.

   The following table lists the plugin files that you must unpack for each Pentaho component.

   | Component                    | File                                        |
   | ---------------------------- | ------------------------------------------- |
   | Dashboard Designer Plugin    | `pdd-plugin-ee-10.2.0.0-<build number>.zip` |
   | Interactive Reporting Plugin | `pir-plugin-ee-10.2.0.0-<build number>.zip` |
   | Pentaho Analyzer Plugin      | `paz-plugin-ee-10.2.0.0-<build number>.zip` |
3. Move the contents of the Pentaho component plugin files to the `\pentaho\server\pentaho-server\pentaho-solutions\system` directory.
