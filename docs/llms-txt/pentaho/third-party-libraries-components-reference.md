# Third-party libraries

Pentaho software is compatible with the following third-party web framework, file system, engine, and utility libraries:

* AngularJS 1.7.8
* HTTPClient 4.5.9
* Apache VFS 2.3
* Apache Axis2 1.7.9
* Apache Log4j 2.17.1
