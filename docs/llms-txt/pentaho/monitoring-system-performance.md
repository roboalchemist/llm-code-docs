# Performance monitoring

You can use logging to monitor and review system performance. Pentaho provides you with several methods to monitor system performance. PDI logging offers you summarized information on transformations and jobs, while Pentaho logging provides detailed information about exceptions, errors, and debugging. You can maintain Pentaho logging through various tools. You can also monitor performance using the Pentaho Operations Mart. Third-party logging such as SNMP is also supported.

For instructions for using logging to monitor system performance, see the following topics:

* [PDI logging](https://docs.pentaho.com/pdia-admin/10.2-admin/optimize-the-pentaho-system/monitoring-system-performance/pdi-logging)
* [Maintain logging](https://docs.pentaho.com/pdia-admin/10.2-admin/optimize-the-pentaho-system/monitoring-system-performance/maintain-logging)
* [Pentaho Operations Mart](https://docs.pentaho.com/pdia-admin/10.2-admin/optimize-the-pentaho-system/monitoring-system-performance/pentaho-operations-mart)
* [Third-party monitoring with SNMP](https://docs.pentaho.com/pdia-admin/10.2-admin/optimize-the-pentaho-system/monitoring-system-performance/third-party-monitoring-with-snmp)
* [Tracking access to sensitive data](https://docs.pentaho.com/pdia-admin/10.2-admin/optimize-the-pentaho-system/monitoring-system-performance/logging-access-to-sensitive-data-with-pentaho-business-analytics)
