# Before you begin

PDI contains all the job entries and transformation steps required for working with Hadoop and MongoDB. Your cluster administrator can set up the Pentaho Server to communicate with most Hadoop distributions. See [Set up Pentaho to connect to a Hadoop cluster](https://github.com/pentaho/documentation/blob/main/PDIA/10.2/Install/Pentaho%20configuration/Pentaho%20Configuration%20-%20Config%20Overview%20\(article%20cp\)/Tasks%20to%20be%20Performed%20by%20an%20IT%20Administrator%20-%20Config%20Overview/Set%20up%20the%20Pentaho%20Server%20to%20connect%20to%20a%20Hadoop%20cluster=GUID-0866C02D-7364-4F94-9D0E-4C1EEA19FC41=12=en=.md) for more information. For a list of supported big data technology, including which configurations of Hadoop are currently supported, see the [Components Reference](https://docs.pentaho.com/install/10.2-install/components-reference).
