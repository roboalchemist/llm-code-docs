# Use Data Optimizer

Use these topics to understand the basic tiering and purging workflow tasks in Data Optimizer:

* [Tour the Home page](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdo-getting-started/tour-of-the-home-page-use-pdc-quick-start) to view recent data operations and [explore your data](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdso-use-data-storage-optimizer/pdso-exploring-your-data).
* Use [Dashboards](https://app.gitbook.com/s/RAKLVv06oBKpy9VLbw7P/pdc-dashboards-user-guide-cp) to gain insights into your data.
* [Assign data temperatures](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdso-use-data-storage-optimizer/pdso-assign-data-temperatures) to classify your data using Data Catalog.
* Use [Rules](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdso-management/pdso-use-rules) to perform tiering or purging. Also see the [examples](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdso-use-data-storage-optimizer/pdso-workflows).
* Use [Data Operations](https://docs.pentaho.com/pdc-10.2-data-optimizer/pdso-management/pdso-data-operations) to confirm file movements or purges.
