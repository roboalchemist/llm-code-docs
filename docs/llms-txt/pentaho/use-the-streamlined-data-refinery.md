# Use the Streamlined Data Refinery

Learn how to use the SDR Sample form and the PDI steps and job entries used to create SDR models.

The best method to learn the SDR is to use the SDR Sample Form to create Analyzer reports. [Learn more](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/use-the-streamlined-data-refinery/how-to-use-the-sdr-sample-form).
