# Using the Pentaho Upgrade Installer

You can upgrade an archive or manual installation of Pentaho products by using the Pentaho Upgrade Installer.

For instructions on using the Pentaho Upgrade Installer to upgrade Pentaho products, see the procedure for the type of installation you have.

* [Use the Pentaho Upgrade Installer to upgrade archive installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-archive-installation)To upgrade an archive installation, use the Pentaho Upgrade Installer to guide you through the upgrade process step-by-step.
* [Use the Pentaho Upgrade Installer to upgrade manual installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-manual-installation)To upgrade a manual installation of Pentaho products, use the Pentaho Upgrade Installer to install upgrade files into an empty folder, and then copy the upgrade files into your manual installation.
