# Other Fields

Enter information for the remaining fields on the **Content** tab.

| Option              | Description                                                                                                                                                                                              |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Time out**        | Specify the interval in milliseconds before step timeout occurs. 60000 ms is the default.                                                                                                                |
| **Use compression** | Select to compress (`.gzip`) the data when connecting between PDI and Salesforce.                                                                                                                        |
| **Limit**           | <p>Specify the maximum number of records to retrieve. Zero records is the default.</p><p>When you set the limit to <code>0</code>, there is no limit on the number of records that can be retrieved.</p> |
