# Using the Pentaho Upgrade Installer in console or silent mode

If you are upgrading in an environment that does not support the display of a graphical interface, use the Pentaho Upgrade Installer in silent mode or console mode from a command line. The command line for silent mode and console mode has specific parameters.

**Note:** The silent mode and console mode of the Pentaho Upgrade Installer is not supported for macOS.
