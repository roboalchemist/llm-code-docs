# Set up emails for scheduled reports

A convenient way you can share reports is to specify an email server to be used by the Pentaho Server to send scheduled reports to recipients, yet you do not need to have an email server to configure the Pentaho Server to send scheduled reports to recipients. If you want to get started quickly or do not have information about your email server, skip this task for now. You can always come back to it later.
