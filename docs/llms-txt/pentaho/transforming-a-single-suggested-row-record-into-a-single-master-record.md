# Transforming a single Suggested Row Record into a single Master Record

Transform a single Suggested Row Record into a single Master Record when there is only one source record identified for the Master Record.

Perform the following steps to transform a single Suggested Row Record into a single Master Record:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. Scroll down the page to view the Suggested Row Records table.
3. Select the checkbox for the Suggested Row Record that you want to transform into a Master Record.
4. Click **Selected** and then select **Create Single Master**. The **Create Master** confirmation box appears.
5. Insert a comment about why you are transforming a single Suggested Row Record into a single Master Record and click **Create Master**.

   The Suggested Row Record is transformed into a Master Record, removed from the **Suggested Rows** table, and added to the **Master Record** table as a Master Record with the **Draft** status.
