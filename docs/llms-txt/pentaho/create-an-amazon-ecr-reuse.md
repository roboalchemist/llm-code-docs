# Create an Amazon ECR

Before pushing the Pentaho image to AWS, you need to create an Amazon ECR.

1. Create an ECR repository to load the Pentaho image.

   For information on how to create an Amazon ECR, see [instructions for creating a private repository](https://docs.aws.amazon.com/AmazonECR/latest/userguide/repository-create.html) on AWS.
2. Record the name of the ECR repository that you have created in the [Worksheet for AWS hyperscaler](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/hyperscalers-landing-page/installing-pentaho-on-aws/running-pdi-cli-on-aws/worksheet-for-aws-hyperscaler-common).
