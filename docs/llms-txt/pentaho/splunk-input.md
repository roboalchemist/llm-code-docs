# Splunk Input

The Splunk Input transformation step enables you to connect to a Splunk server, enter a Splunk query, and get results back for use within a transformation. Once you have completed those steps, you can stream data from Splunk into your transformation. To learn more about Splunk see their [online documentation](http://docs.splunk.com/Documentation).
