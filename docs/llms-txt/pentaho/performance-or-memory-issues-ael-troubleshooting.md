# Performance or memory issues

If you experience performance or memory issues while running your PDI transformation on the Spark engine, your transformation may not be efficiently using Spark execution resources.

To resolve or minimize the issue, apply and adjust application and PDI step Spark tuning parameters. See [About Spark tuning in PDI](https://docs.pentaho.com/install/9.3-install/pentaho-configuration/tasks-to-be-performed-by-an-it-administrator/set-up-the-adaptive-execution-layer-ael/advanced-topics/spark-tuning-landing-page-cp/about-spark-tuning-in-pdi-cp) for details.
