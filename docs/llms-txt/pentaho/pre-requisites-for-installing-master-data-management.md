# Pre-requisites for installation

To install Pentaho Data Mastering, you must meet configuration requirements, install required software, and set up required accounts. Review the following requirements before you begin the installation process:

* [Server configuration](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/pre-requisites-for-installing-master-data-management/server-configuration)
* [Network configuration](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/pre-requisites-for-installing-master-data-management/network-configuration)
* [Required software and accounts](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/pre-requisites-for-installing-master-data-management/required-software-and-accounts)

After you have met configuration requirements, installed required software, and set up required accounts, make a note of the following information, which you must enter during the installation process:

* Your server’s IP address
* If you plan to be online for the installation, your JFrog Artifactory user ID and an Access token that you generated in Artifactory.
* The service image tag numbers for the version of Pentaho Data Mastering that you plan to install.
