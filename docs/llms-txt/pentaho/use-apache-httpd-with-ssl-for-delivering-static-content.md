# Use Apache HTTPd with SSL for delivering static content

To improve performance, you may want to use the Apache HTTPd Web server to handle delivery of static content and facilitation of socket connections.

For more information, see [Use the Apache web server (HTTPd) for socket handling](https://docs.pentaho.com/pdia-admin/10.2-admin/secure-the-pentaho-system/user-security/advanced-security-providers/ssl-security#use-the-apache-web-server-httpd-for-socket-handling).
