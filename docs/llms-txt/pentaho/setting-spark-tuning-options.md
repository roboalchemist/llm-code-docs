# Setting PDI step Spark tuning options

After you open the **Spark tuning parameters** dialog box, select the parameter and click in the **Value** column. Use this article and the best practices described in the [Spark documentation](https://spark.apache.org/docs/2.3.0/rdd-programming-guide.html) to adjust the value for your transformation. You can control the values in the Spark tuning parameters using PDI variables.
