# Sqoop Export

You can use the Sqoop Export job to export data from Hadoop into an RDBMs using Apache Sqoop. You can create, edit, and select a Hadoop cluster configuration to use. Hadoop cluster configurations settings can be reused in transformation steps and job entries that support this feature. This job has two setup modes:

* The **Quick Setup** mode provides the minimum options necessary to perform a successful Sqoop export. (Default)
* The **Advanced Options** mode provides more options to manage your Sqoop export. The **Advanced Options** mode also has a command line view which allows you to reuse an existing Sqoop command from the command line.

For additional information about Apache Sqoop, see <http://sqoop.apache.org/>.
