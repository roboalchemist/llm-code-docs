# Database connection and warehouse

In this section, specify the Snowflake database connection and identify the virtual warehouse you want to create. For more information about naming virtual warehouses, see the [Snowflake](https://docs.snowflake.net) documentation.

| Option                  | Description                                                                                                                                                                                                                                                                                                                                                                                                                |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Database connection** | <p>Select an existing Snowflake database from the list.If you do not have an existing connection, click <strong>New</strong>. If you need to modify an existing connection, click <strong>Edit</strong>.</p><p><strong>Note</strong>: If timeout errors occur, see <a href="../../../data-integration-issues/snowflake-timeout-errors-general-issues-in-troubleshooting">Snowflake timeout errors</a> to troubleshoot.</p> |
| **Warehouse name**      | Specify a valid identifier for the virtual warehouse you are creating. The name must be unique for your Snowflake account.                                                                                                                                                                                                                                                                                                 |
| **Warehouse comment**   | Specify a description or other information to identify or distinguish this virtual warehouse or its purpose.This information is useful in configurations that include multiple virtual warehouses.                                                                                                                                                                                                                         |
