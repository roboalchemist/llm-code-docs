# Delete a semantic model

Delete a semantic model that you no longer need for analysis or reporting.&#x20;

Complete the following steps to delete a semantic model:&#x20;

1. Log into the **Pentaho User Console** (PUC). &#x20;
2. Open the Semantic Model Editor by taking one of the following actions:

   * If you are using the Modern Design of PUC, in the menu on the left side of the page, click **Semantic Model Editor**.
   * If you are using the Classic Design of PUC, click **File** > **Semantic Model Editor**.&#x20;

   The **Semantic Model Editor** opens.&#x20;
3. In the **Semantic Models** list, navigate to the model you want to delete by searching or scrolling through the list.&#x20;
4. In the row that contains the semantic model you want to delete, click the **More Actions** icon, and select **Delete**. The **Delete Model** confirmation window opens.&#x20;
5. Click **Delete**. The semantic model is deleted.&#x20;
