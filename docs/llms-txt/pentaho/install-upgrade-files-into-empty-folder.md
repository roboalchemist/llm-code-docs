# Install upgrade files into empty folder

Use the Pentaho Upgrade Installer to install upgrade files into an empty folder so that you can copy those upgrade files into your manual installation.

Perform the following steps to install upgrade files into an empty folder.

1. Navigate to a location that is separate from your current Pentaho installation.
2. Create an empty folder named `pentaho-upgrade-files`.
3. Run the update executable file for the Pentaho Upgrade Installer that you downloaded from the [Support Portal](https://support.pentaho.com/hc/en-us). After processing, the Introduction window opens.
4. Click **Next**.
5. In the License Terms window, read through the license agreement and select the checkbox at the end of the agreement to acknowledge the terms.
6. Click **Next**.
7. In the Welcome window, select **Upgrade** and then click **Next**.
8. In the **Folder** text box of the Choose Folder window, specify the `pentaho-upgrade-files` folder that you created the previous step, and then click **Next**.
9. When prompted for the location of the hash files zip archive, specify the path to the archive, select the archive, and then click **Next**.

   **Note:** You can download the hash files zip archive from the [Support Portal](https://support.pentaho.com/home)

The upgrade files are installed into the empty folder, `pentaho-upgrade-files`.

Continue upgrading your manual installation of Pentaho products by completing the procedure, [Update files in manual installation](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/using-the-pentaho-upgrade-installer/use-the-pentaho-upgrade-installer-to-upgrade-manual-installation/update-files-in-manual-installation).
