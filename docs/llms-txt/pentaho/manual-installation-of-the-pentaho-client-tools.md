# Manual installation of the Pentaho Client Tools

After you install and start the Pentaho Server, you can install the Pentaho client tools on different workstations.

* [Installation of the Pentaho design tools](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-of-the-pentaho-design-tools)
* [Install PDI tools and plugins](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-of-the-pentaho-design-tools/install-the-pdi-tools-and-plugins)

After you have installed both the Pentaho Server and client tools, configure the Pentaho Server and design tools so that you can install licenses, log on to the Pentaho Server, set up data sources, and choose a security method. For information on configuring the server and design tools, see [Pentaho configuration](https://docs.pentaho.com/install/10.2-install/pentaho-configuration).

See the **Administer Pentaho Data Integration and Analytics** document for instructions on setting up your licenses and optional Operations Mart logging.
