# Enable default hyperlinks

After creating the annotation in the Mondrian schema file you will need to reimport it:

Perform the following steps to reimport the schema file:

1. Click **Manage Data Sources**in the Pentaho User Console.
2. In the **Manage data sources** dialog box select **Import Analysis** from the **More actions and options** menu.

   ![Manage data sources dialog box](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-83ed2795fd8b5e538db974b452710dc03297c78c%2FPAZ%20Manage%20data%20sources%20dialog%20box.png?alt=media)
3. Click the **Browse** icon In the Manage Data Sources dialog box and choose **Import Analysis**.
4. Select your schema file, then click **Import**.

The hyperlinks are enabled for the specified dimensions for new or existing reports associated with the edited cube.
