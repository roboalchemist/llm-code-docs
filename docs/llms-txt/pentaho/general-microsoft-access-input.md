# Microsoft Access input

The default value of the **Step Name** field is **Microsoft Access input**. You can customize this name or leave it as the default.

You can click **Preview rows** to examine the contents of the Access input file you have specified in the **File or directory** field. The Microsoft Access input step determines what rows to input based on the information you provide in the options tabs. This preview function helps you to decide if the information provided accurately models the rows you are trying to retrieve.
