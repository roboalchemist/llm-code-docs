# Transforming multiple Suggested Row Records into multiple Master Records

Transform multiple Suggested Row Records into multiple Master Records when you want each of the Suggested Row Records to be a separate Master Record.

Performthe following steps to transform multiple Suggested Rows Records into multiple Master Records:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. Scroll down the page to view the Suggested Row Records table.
3. Select the checkboxes for the Suggested Row Records that you want to transform into multiple Master Records.
4. Click **Selected** > **Create Multiple Master**. The **Create Master** confirmation dialog box appears.
5. Inserta comment about why you are transforming multiple Master Records from multiple Suggested Row Records and click **Create Master**.

   The selected Suggested Row Records are transformed into multiple Master Records, removed from the Suggested Rows table, and added to the **Master Record** table as Master Records with the **Draft** status.
