# Install in Hadoop Cluster

Pentaho Data Optimizer is a File System in Userspace (FUSE) file system purpose-built for Hadoop Distributed File System (HDFS) DataNodes, to enable seamless HDFS data tiering to Hitachi Content Platform, Virtual Storage Platform One Object, or Simple Storage Service (S3) object storage.

This guide covers the installation, configuration, and management of Data Optimizer, including best practices and troubleshooting techniques.
