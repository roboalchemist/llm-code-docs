# Query-metadata

The following attributes belong to the **query-metadata** property:

| Attribute Name   | Purpose                                                                             | Values                                                                      |
| ---------------- | ----------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| **data-format**  | Use the number or date formatting from the Report Design Wizard or the data source. | Boolean; default is true if you are using the wizard, false if you are not. |
| **style-format** | Use the style formatting from the Report Design Wizard or the data source.          | Boolean; default is true if you are using the wizard, false if you are not. |
