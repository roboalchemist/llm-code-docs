# Use the Pentaho Marketplace to manage plugins

**Pentaho Data Integration** has plugins that are installed by default. You can use the Pentaho Marketplace to download, install, and share plugins developed by Pentaho and members of the user community. The Marketplace presents a list of plugins, indicates whether they have been installed, and displays other information about the plugin, such as where to obtain technical support.
