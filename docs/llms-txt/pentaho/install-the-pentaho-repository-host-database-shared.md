# Install the Pentaho Repository host database

The Pentaho Repository houses data needed for Pentaho tools to provide scheduling and security functions. The repository also stores metadata and models for reports that you create.

Perform the following steps before specifying the host database for the Pentaho Repository:

1. Choose which type of database to use for the Pentaho Repository's host database, then install it. Pentaho supports these databases:
   * PostgreSQL
   * MySQL
   * MariaDB
   * Oracle
   * MS SQL Server
2. Check the [Components Reference](https://docs.pentaho.com/install/10.2-install/components-reference) to section to determine which versions of the databases Pentaho currently supports.
3. Download and install the database of your choice.
4. Verify that the database is installed correctly.
