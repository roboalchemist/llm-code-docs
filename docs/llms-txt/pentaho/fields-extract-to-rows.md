# Fields

| Field                      | Description                                                                                                                                                                                                                                                                    |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Hierarchical data path** | Complete path of the field name in the hierarchical field source. See [Hierarchical data path specifications](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/hierarchical-data/hdt-path-specification) |
| **Output field name**      | Name of the field that maps to the corresponding field in the hierarchical input source.                                                                                                                                                                                       |
| **Type**                   | Data type of the generated output field.                                                                                                                                                                                                                                       |
| **Path field name**        | (Optional) Adds the hierarchical path as a new output field.                                                                                                                                                                                                                   |
