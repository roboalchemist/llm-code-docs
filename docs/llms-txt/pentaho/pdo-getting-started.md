# Get started

Data Optimizer integrates with the Pentaho Data Catalog application. After installing Data Optimizer, you are ready to start identifying, classifying, and tiering or purging your files.

You can use Data Optimizer if you are a managed services provider identifying data tiering candidates, a storage administrator reviewing the capacity, utilization, and cost effectiveness of storage, or a backup administrator implementing a disk base solution to increase backup and restore success rates.

To begin using Data Optimizer, you must log in. User authentication, roles, and permissions for Data Optimizer are provided by Data Catalog. Use the login credentials shared by your Data Catalog administrator for access. Navigate to the Data Optimizer application using the **App Switcher**.
