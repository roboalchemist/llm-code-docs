# General

Use the Unique Rows step to define parameters for your output and how you want to sort duplicate rows.

![Unique rows step](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-18e3d9c2a527c5fa174378949e8f56f4e0c0cefd%2FPDI_TransStep_Dialog_UniqueRows.png?alt=media)

Enter the following information for the step:

* **Step name**: Specify the unique name of the transformation on the canvas. You can customize the name or leave it as the default.
