# Hierarchical JSON Output

You can use the Hierarchical JSON output step to convert hierarchical data from a previous step into JSON format. See [Hierarchical data](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/hierarchical-data) for an overview of hierarchical data in Pentaho.

**Note:** This step only accepts hierarchical data
