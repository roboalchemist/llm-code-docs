# Change the format of the data values

You can change the format of the data in each column. Click inside a column to apply a format, then click the **Formatting** tab to choose the appropriate format for the values in your column.

![Change Data Formats](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-f684ebdc1c6bd4bae9fe664b68d689869eb7daf5%2FCustomize_PIR_Change_Data_Formats.png?alt=media)

**Note:** In some instances, you might notice that your data is already formatted. For example, money symbols are already included in sales figures. Such formatting is inherited from metadata associated with the data source. However, you can override inherited formats whenever you want.
