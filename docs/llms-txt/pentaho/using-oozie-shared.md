# Oozie configuration

If you are using Oozie on your cluster, you must configure the platform and the server. By default, the Oozie user runs the Oozie jobs. If you use PDI to start an Oozie job, you must set up the PDI user to execute the program in proxy. This setup consists of the following tasks:

* Set up Oozie on a cluster
* Set up Oozie on the server
