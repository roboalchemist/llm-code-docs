# Connection tab

![Query metadata from Database step Connection tab](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-9cd1664c8b70361fc0b98564aab0075cc10ab64a%2FPDI_Query_Metadata_from_Database_Step_Connection_Tab.png?alt=media)

You use the **Connection** tab to connect to a JDBC database. You have two options when selecting a connection in the **Query metadata from Database** step.

| Option                  | Description                                                                                                                                                                                                                                                                                                                                                                                            |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Select connection**   | <p>Select a <strong>Connection</strong> from the dropdown menu. You can use the following options to perform additional actions:- <strong>New</strong></p><p>Create a new database connection that becomes available in the dropdown.</p><ul><li><strong>Edit</strong></li></ul><p>Edit a selected connection.</p><ul><li><strong>Wizard</strong></li></ul><p>Open the database connection wizard.</p> |
| **From field**          | Select a **Connection** from a display of available fields in the datastream.                                                                                                                                                                                                                                                                                                                          |
| **Remove input fields** | In the**From field** option, click **Remove input fields** to omit the fields from prior steps in the transformation data stream.                                                                                                                                                                                                                                                                      |
