# Configure the design tools and utilities

Before using design tools and utilities, you need to perform configuration tasks for each workstation running these tools.

**Note:** Depending on how these tools and utilities were [installed](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp), they might be located on different machines other than the Pentaho Server.
