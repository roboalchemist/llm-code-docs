# AEL considerations

When using the Kafka Consumer step with the [Adaptive Execution Layer](https://docs.pentaho.com/pdia-data-integration/9.3-data-integration/advanced-topics-pentaho-data-integration-overview/adaptive-execution-layer-cp-landing-page), the following may affect performance and results.

* When using this step with HDP 3.x on AEL Spark, there is a known JAR conflict. For solutions, see the AEL troubleshooting section in the **Administer Pentaho Data Integration and Analytics** document.
