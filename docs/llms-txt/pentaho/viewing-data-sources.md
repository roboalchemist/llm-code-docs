# Viewing data source details

View data source details to review the connection information and associated database type that is associated with a data source.

Perform the following steps to view data source details:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Resources** card, click **Data Sources**.

   The Data Sources page opens with the list of data sources shown in a table.
3. At the end of the row for a data source record you want to view, click the more actions icon and select **View Details**.
