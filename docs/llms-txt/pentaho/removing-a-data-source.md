# Removing a data source

Remove a data source when you no longer want to process the raw data from that source in the Pentaho Data Mastering application.

You must have admin privileges to remove a data source.

Perform the following steps to remove a data source:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Resources** card, click **Data Sources**.

   The Data Sources page opens with the list of existing data sources shown in a table.
3. At the end of the data source row that you want to remove, click the more actions icon and select **Remove**.

   The Remove Data Sources confirmation box appears.
4. Click **Remove**.

   A confirmation message is shown at the top of the page and the data source record is removed from the data source table.
