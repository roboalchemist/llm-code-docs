# PDI types

The Parquet to PDI data type values are as shown in the table below:

| Parquet Type         | PDI Type  |
| -------------------- | --------- |
| ByteArray            | Binary    |
| Boolean              | Boolean   |
| Double               | Number    |
| Float                | Number    |
| FixedLengthByteArray | Binary    |
| Decimal              | BigNumber |
| Date                 | Date      |
| Enum                 | String    |
| Int8                 | Integer   |
| Int16                | Integer   |
| Int32                | Integer   |
| Int64                | Integer   |
| Int96                | Timestamp |
| UInt8                | Integer   |
| UInt16               | Integer   |
| UInt32               | Integer   |
| UInt64               | Integer   |
| UTF8                 | String    |
| TimeMillis           | Timestamp |
| TimestampMillis      | Timestamp |
