# Design report headers and footers

Double-click the upper or lower corners of the report to design your report headers and footers.

![Design Report Headers and Footers](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-fa74b4b78638d644632aefc379d4d5a590ae8584%2FCustomize_PIR_Design_Report_Headers_and_Footers.png?alt=media)

Use the dialog box to adjust any of the following options:

* Delete any existing words, or enter a new header and press Enter.
* Click **Date** and press Enter to insert a date and time stamp in the page header. You can customize the date and page count by adding your own text, such as `Today: $(report.date)`, instead of using the default $(report.date).
* Click **#/Pages** and press Enter to insert the number of pages in the report.
