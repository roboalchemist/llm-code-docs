# AMQP Producer

The Advanced Message Queuing Protocol (AMQP) Producer step creates and publishes a [stream](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/streaming-analytics) of AMQP messages to a compatible AMQP 0-9-1 broker.
