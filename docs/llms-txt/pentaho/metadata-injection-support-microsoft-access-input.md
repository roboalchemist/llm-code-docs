# Metadata injection support

You can use the Metadata Injection supported fields with the [ETL metadata injection](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/pdi-transformation-steps-reference-overview/etl-metadata-injection) step to pass metadata to your transformation at runtime. The following fields and values of the Microsoft Access input step support metadata injection:

**File tab**

* Filename is defined in a field
* Get filename from field
* File or directory
* Regular Expression
* Exclude Regular Expression
* Required
* Include subfolders

**Content tab**

* Table
* Include filename in output?
* Include tablename in output?
* Include rownum in output?
* Filename fieldname
* Tablename fieldname
* Rownum fieldname
* Reset Rownum per file?
* Limit
* Add filename to result

**Fields tab**

* Name
* Column
* Type
* Length
* Precision
* Trim type
* Format
* Currency
* Decimal
* Group
* Repeat
