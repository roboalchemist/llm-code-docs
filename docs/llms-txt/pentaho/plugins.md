# Plugins

You can activate several plugins in the Pentaho User Console.

* [Pentaho Analyzer](https://docs.pentaho.com/pba/10.2-analytics/pentaho-analyzer-cp): Conceptualize your data, add query parameters and filters, drill-down links, and create visualizations.
* [Pentaho Interactive Reports](https://docs.pentaho.com/pba/10.2-analytics/pentaho-interactive-reports-cp): Create simple and on-demand operational reports without depending on IT or report developers.
* [Pentaho Dashboard Designer](https://docs.pentaho.com/pba/10.2-analytics/pentaho-dashboard-designer-cp): Create dashboards that help decision makers gain critical knowledge at a glance.

You can also use **Pentaho CTools** (a set of community-driven tools) to help you develop and define your data structure, which you can then use in dashboards.
