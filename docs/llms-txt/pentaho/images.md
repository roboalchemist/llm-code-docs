# Images

The following attributes belong to the **images** property:

| Attribute Name           | Purpose                                               | Values                                                                                                        |
| ------------------------ | ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **image-encode-quality** | Controls the JPEG encoding quality                    | A decimal percentage value between 0 and 1. The closer to 1, the higher the quality and larger the file size. |
| **image-encode-type**    | Determines the image format of PDF and HTML graphics. | Possible values are: `.jpg` and `.png`. Default is .png.                                                      |
