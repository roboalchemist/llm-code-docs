# Viewing a data source

Perform the following steps to view a data source:

1. On the **Menu**, click **Management**.

   The Manage Your Environment page opens.
2. On the **Data Sources** card, click **Imported**.

   The Data Sources page appears.

The table on the page displays the information about your available data sources.
