# Install and configure the Streamlined Data Refinery

You can install the SDR sample, along configuring the SDR for your environment, with the following information.

* The [Installing and configuring the SDR sample](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/installing-and-configuring-the-sdr-sample) walks you through each part of the process of installing the SDR sample to use.
* If you are not using Postgres and a default installation, you will need to configure a few KTRs in Spoon to use the SDR form. The [Configure KTR files for your environment](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/configure-ktr-files-for-your-environment) article shows you how.
* The [Clean up the All Requests Processed list](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/clean-up-the-all-requests-processed-list) article walks you through deleting items from the **All Requests Processed** list to clean your **Latest Requests** list.
* If you are using Vertica, use the [Install the Vertica JDBC driver](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/install-the-vertica-jdbc-driver) article to install the Vertica JDBC driver.
* Read the [Use Hadoop with the SDR](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/use-hadoop-with-the-sdr) article to set up the Streamlined Data Refinery to use Hadoop.
* Read the [App endpoints for SDR forms](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/app-endpoints-for-sdr-forms) article to learn about a few API endpoints that you can use to run the app.
