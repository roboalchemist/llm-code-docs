# Restoring a Pentaho Upgrade Installer backup of archive installation

When you upgrade an archive installation, the Pentaho Upgrade Installer automatically generates a backup of your existing Pentaho products during the upgrade process to the Pentaho 10.2 products. You can use this backup to restore your Pentaho products back to the earlier version.
