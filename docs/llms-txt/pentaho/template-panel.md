# Template panel

When you create new files or replace existing files, you can choose to create a copy of an existing template file instead. Make sure that the template file is the same type as the output file, either `.xls` or `.xlsx`. This panel includes fields for using template files and sheets in your Excel output file.

| Field                                     | Description                                                                                                                                                                                                                                                                                     |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use template when creating new files**  | <p>Specify whether to use a specified Excel template to create the output file.</p><ul><li>Select this option to use an template when creating the output file.</li><li>Clear this option if you do not want to use a template.</li></ul>                                                       |
| **Template file**                         | If you selected **Use template when creating new files**, above, then enter the name of the template file you want to use or click **Browse** and navigate to it.                                                                                                                               |
| **Use template when creating new sheets** | <p>Decide whether to use a specified Excel template sheet to create the output sheet.</p><ul><li>Select this option to specify a template sheet for creation of the output sheet.</li><li>Clear this option if you do not want to specify a template sheet.</li></ul>                           |
| **Template sheet**                        | If you selected **Use template when creating new sheets**, above, then enter the template sheet name you want to use or click **Browse** and navigate to it.                                                                                                                                    |
| **Hide Template sheet**                   | <p>Decide if you want the template sheet to be visible when the MS Excel output file is opened.</p><ul><li>Select this option to hide the template sheet from users who open the output file.</li><li>Clear this option to show the template sheet to users who open the output file.</li></ul> |
