# Install Pentaho software

Install the Pentaho software prior to using the Streamlined Data Refinery.

1. Install the latest version of Pentaho software.

   If you are installing Pentaho using the Evaluation method with Postgres, choose the default option for installation, and then enter `password` for the Postgres password information.

   **Note:** See the **Install Pentaho Data Integration and Analytics** document for instructions on installung the latest version of Pentaho software.
2. Log in to the Pentaho User Console (PUC), and verify that it is working properly.
3. Log out of PUC, then stop the Pentaho Server.

If you are using Vertica, you will need to [install the Vertica JDBC driver](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/advanced-topics-pentaho-data-integration-overview/work-with-the-streamlined-data-refinery/install-and-configure-the-streamlined-data-refinery/install-the-vertica-jdbc-driver) at this point.
