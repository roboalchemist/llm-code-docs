# View installed plugins and versions

Complete these steps to view the installed plugins and their versions:

1. Start the PDI client (Spoon).
2. Select **Tools** > **Show plugin information**.

   The Plugin browser window appears. The name of the plugin appears in the **Detected** plugins section of the page.
3. You can filter the installed plugins by type. To view the different types, select one from the **Plugin type** drop-down menu.
4. When complete, click the **Close** button to close the window.
