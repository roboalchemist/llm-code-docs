# Adding JBoss logging

Optionally, you can add Pentaho application-level logging to the JBoss logging subsystem. Make sure that the Pentaho Server is stopped before you begin, and [start the server](https://docs.pentaho.com/install/9.3-install/pentaho-installation-overview-cp/manual-installation/manual-installation-process/prepare-jboss-connections-and-web-app-servers/prepare-jboss-web-application-servers/step-5-configure-pentaho-settings-for-jboss/add-jboss-logging/configure-and-start-the-pentaho-server-after-manual-installation) after you save the file.
