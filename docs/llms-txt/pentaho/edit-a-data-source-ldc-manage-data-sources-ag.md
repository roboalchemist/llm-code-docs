# Edit a data source

With applicable permissions, you can edit a data source as needed.

Perform the following steps to edit a data source:

1. Click **Management** in the left navigation menu and on the **Resources** card, click **Data Sources**.

   The **Manage Data Sources & Data Centers** page opens.
2. Locate the data source that you want to edit and then click **View** at the right end of the row for the data source.

   The Edit Data source page opens.
3. You can edit the following additional connection information. \
   **Note**: The fields may vary based on the type of data source you have selected.
   * Data Source Description
   * &#x20;Affinity
   * Configuration details like User Name, Password, URI, Host, Port, and so on.
   * Physical Location
   * Storage optimization options
   * Note
4. Verify the updated information, and then click **Test Connection** to verify your connection to the specified data source.
5. Click **Save Data Source**.

The data source is updated.
