# Home

After you log in to the User Console, you will see the Home perspective, which serves as the portal for you to access Pentaho tools and features. Home contains easy access buttons so that you can browse your existing files, create new reports, dashboards, data sources, view Pentaho documentation, and quickly open recently viewed or favorite files.
