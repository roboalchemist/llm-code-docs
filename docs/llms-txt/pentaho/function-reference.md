# Function reference

You can click the **fx** button in the upper left corner of the pane to add a new function. Delete a function by clicking it, then pressing the Delete key, or by right-clicking it and selecting **Delete** from the context menu.

Every function available in Report Designer is defined and categorized according to the group it belongs to in the following topics:

* [Common functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/common-functions)
* [Report functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/report-functions)
* [Summary functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/summary-functions)
* [Running functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/running-functions)
* [Advanced functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/advanced-functions)
* [Chart Data functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/chart-data-functions)
* [Image functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/image-functions)
* [Script functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/script-functions)
* [Deprecated functions](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/function-reference/deprecated-functions)
