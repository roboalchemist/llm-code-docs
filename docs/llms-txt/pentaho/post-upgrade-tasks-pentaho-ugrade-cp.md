# Post-upgrade tasks

After you upgrade your Pentaho products to version 10.2, consider whether you should perform any of the following optional tasks depending on your Pentaho configuration.

* [**Apply customizations**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/apply-customizations-post-upgrade-tasks)

  If you previously customized items from the default Pentaho setup, you might want to merge your customizations into Pentaho 10.2 versions of those items.
* [**Update the default documentation version link**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/update-the-default-documentation-version-link-post-upgrade-tasks)

  If you link to specific documentation articles in your Pentaho configuration, update the default documentation version in those links.
* [**Install Ops Mart**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/install-ops-mart-post-upgrade-tasks)

  If you are interested in a centralized data mart for your PDI job or PDI transformation logs, you can install DI Ops Mart.
* [**Install drivers for your Hadoop clusters**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/install-drivers-for-your-hadoop-clusters-post-upgrade-tasks)

  If you use Hadoop clusters with your Pentaho products, you must re-install the drivers.
* [**Apply your plugins**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/apply-your-plugins-post-upgrade-tasks)

  If you use plugins with your Pentaho products, you must re-apply the plugins.
* [**Set up password encryption after upgrading**](https://docs.pentaho.com/install/10.2-install/pentaho-upgrade-cp/post-upgrade-tasks-pentaho-ugrade-cp/setting-up-password-encryption-after-upgrading-pentaho-upgrade-cp)

  To encrypt passwords currently stored as plain text in configuration files, you must set up your upgraded 10.2 Pentaho Server for password encryption.
