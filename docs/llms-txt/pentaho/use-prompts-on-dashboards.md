# Use prompts on dashboards

Prompts display a subset of data based on the dashboard user's point of view. For example, a European user may only want to see EMEA region sales data, whereas the dashboard's default region is the NA region. To ensure that a prompt links to the correct content in a dashboard, the content to which you are linking (data table or chart) must contain at least one [parameter](https://docs.pentaho.com/pba/10.2-analytics/pentaho-dashboard-designer-cp/advanced-topics/set-dashboard-parameters).
