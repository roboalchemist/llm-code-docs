# Google Analytics v4

You can use the Google Analytics v4 step to access your Google Analytics data to generate reports or to populate your data warehouse. This step queries Google Analytics properties using the [Google Analytics API v4](https://developers.google.com/analytics/devguides/reporting/data/v1) and places the resulting dimension and metric values on the step output stream.
