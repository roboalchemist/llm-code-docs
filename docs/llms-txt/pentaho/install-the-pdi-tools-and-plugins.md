# Install PDI tools and plugins

You can integrate your data by using Pentaho Data Integration (PDI) to develop and deploy ETL programs. You use the PDI design tool, known as the PDI client (formerly known as Spoon), to develop the ETL programs, along with several PDI utilities and plugins.

Install the PDI client, utilities, and plugins by downloading a ZIP file and extracting the installation files for each component. If you have the PDI client already installed, you can download just the plugins by using the ZIP file or the PDI Marketplace in PDI. See [Step 4: Install PDI plugins](https://docs.pentaho.com/install/10.2-install/pentaho-installation-overview-cp/installation-of-the-pentaho-design-tools/install-the-pdi-tools-and-plugins/install-pdi-plugins) for further details.
