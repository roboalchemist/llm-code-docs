# Viewing users

View the list of users to see who can access the Pentaho Data Mastering application.

You must have admin privileges to view users.

Perform the following steps to view users:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Users** card, click **Users**.

   The Users page opens with the list of users shown in a table that contains the following information:

   | Field          | Description                                                          |
   | -------------- | -------------------------------------------------------------------- |
   | **ID**         | User ID generated by the system.                                     |
   | **Names**      | Name of the user.                                                    |
   | **Roles**      | Role assigned to the user. If no role is assigned, then NA is shown. |
   | **Created At** | Date and time of user creation.                                      |
