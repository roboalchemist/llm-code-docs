# Pentaho Data Integration Plugins

As part of our effort to simplify deployment and upgrades, we will release new features via plugins where possible. Customers and Partners can easily download and deploy these plugins without requiring a full upgrade of the their deployment.

The following plugins are currently available. Click below for installation and usage instructions. Note that these are not available for the Developer (Community) Edition.

* [**Databricks Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins)
* [**Elasticsearch REST Bulk Insert Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/elasticsearch-rest-bulk-insert)
* [**Google Analytics v4 Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/google-analytics-v4)
* [**Hierarchical Data Types (HDT) Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/hierarchical-data-types-hdt)
* [**Kafka Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/kafka-plugin)
* [**Open Lineage**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/openlineage-plugin)
* [**Salesforce Bulk Operation Plugin**](https://docs.pentaho.com/pdia-data-integration/pentaho-data-integration-plugins/salesforce-bulk-operations-plugin)
