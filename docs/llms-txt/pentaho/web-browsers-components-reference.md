# Web browsers

Pentaho supports major versions of web browsers that are publicly available six weeks prior to the finalization of a Pentaho release.

| Pentaho Software                                                                                                  | Supported Browsers                                                                                                                                                                                                                                                               | Tested Browsers                                                                                                                                                                                                                                          |
| ----------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <p>Pentaho User Console (PUC)</p><p>(Pentaho recommends 2GB RAM for the web client.)</p>                          | <ul><li>Apple Safari 14 (On macOS only) and later</li><li>Google Chrome 91 and later</li><li>Microsoft Edge 9 and later</li><li>Microsoft Internet Explorer 11 and later (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89 and later</li></ul> | <ul><li>Apple Safari 14 (On macOS only)</li><li>Google Chrome 91.0.4472.114</li><li>Microsoft Edge 9.1.0.864.64</li><li>Microsoft Internet Explorer 11 (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89.0.2</li></ul> |
| Pentaho Report Designer                                                                                           | <ul><li>Apple Safari 14 (On macOS only) and later</li><li>Google Chrome 91 and later</li><li>Microsoft Edge 9 and later</li><li>Microsoft Internet Explorer 11 and later (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89 and later</li></ul> | <ul><li>Apple Safari 14 (On macOS only)</li><li>Google Chrome 91.0.4472.114</li><li>Microsoft Edge 9.1.0.864.64</li><li>Microsoft Internet Explorer 11 (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89.0.2</li></ul> |
| Pentaho Data Integration (PDI) client\*                                                                           | <ul><li>Apple Safari 14 (On macOS only) and later</li><li>Google Chrome 91 and later</li><li>Microsoft Edge 9 and later</li><li>Microsoft Internet Explorer 11 and later (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89 and later</li></ul> | <ul><li>Apple Safari 14 (On macOS only)</li><li>Google Chrome 91.0.4472.114</li><li>Microsoft Edge 9.1.0.864.64</li><li>Microsoft Internet Explorer 11 (Does not render PUC correctly using Compatibility Mode)</li><li>Mozilla Firefox 89.0.2</li></ul> |
