# Edit access to a semantic model

Edit access to a semantic model to control whether a user or role can view, edit, or share the model with other users.

Complete the following steps to edit access to a semantic model for a user or user role:

1. Log into the **Pentaho User Console** (PUC). &#x20;
2. Open the Semantic Model Editor by taking one of the following actions:

   * If you are using the Modern Design of PUC, in the menu on the left side of the page, click **Semantic Model Editor**.
   * If you are using the Classic Design of PUC, click **File** > **Semantic Model Editor**.&#x20;

   The **Semantic Model Editor** opens.&#x20;
3. In the **Semantic Models** list, navigate to the model you want to edit.&#x20;
4. In the row that contains the semantic model you want to share, click the **More Actions** icon, and select **Share**. The **Share&#x20;*****Name*****&#x20;Model** window opens.&#x20;
5. Click one of the following tabs:
   * **All:** List of all users and roles.
   * **Users:** List of only users.
   * **Roles:** List of only roles.
6. Find the user or role you want to edit, and then click the **Edit Permissions** icon. The **Share&#x20;*****Name*****&#x20;Model** window opens with only the permissions for the user or role you are editing.
7. Take one or more of the following actions:&#x20;
   * To remove all access to the model, click **Remove Access,** and then in the **Remove permissions** dialog box, click **Remove**.
   * To configure all access permissions at once, in the **Set all to** list, select one of the following options, and then on the left side of the user or role's name, click the **Go Back** arrow to return to the **Share&#x20;*****Name*****&#x20;Model** window and click **Apply**.&#x20;
     * **No permissions**&#x20;
     * **Full permission**
   * To add or remove permissions individually, turn the following toggles on or off, and then on the left side of the user or role's name, click the **Go Back** arrow to return to the **Share&#x20;*****Name*****&#x20;Model** window and click **Apply:**&#x20;

     * **Change Connection of Semantic Models**
     * **Delete Semantic Models**
     * **Edit Semantic Models**
     * **Share Semantic Models**
     * **View Semantic Models**

     <div data-gb-custom-block data-tag="hint" data-style="info" class="hint hint-info"><p><strong>Note:</strong> If you turn off the toggles to remove all permissions, the <strong>Remove permissions</strong> dialog box opens. Click <strong>Remove</strong> to finish removing all permissions from the user or role.</p></div>
