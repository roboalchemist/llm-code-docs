# Removing a match rule

Remove a match rule that you no longer want to use for controlling how Pentaho Data Mastering identifies matches in the data from different data sources.

You must have admin privileges to remove a match rule.

Perform the following steps to remove a match rule:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the **Match Rules** card, click **Match Rules**.

   The Match Rules page opens with the list of match rules shown in a table.
3. At the end of the row for a match rule record that you want to remove, click the more actions icon and select **Remove**.

   The Remove Match Rule confirmation box appears.
4. Click **Remove**.

   A confirmation notification message is shown at the top of the page and the match rule record is removed from the match rule table.
