# Viewing user roles

View user roles to see what user roles are available in the Pentaho Data Mastering application and the permissions that are assigned to those roles.

You must have admin privileges to view users roles.

Perform the following steps to view user roles:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. In the Roles card, click Users.

   The Roles page opens with a list of roles shown in a table that contains the following information.

| Field       | Description                                                                                                                                                                                   |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name        | Name of the user role. **Tip:** In your identity management tool, the user role name is appended with the prefix, `MDM`. For example, the admin role is `MDM_Admin` in the Keycloak IAM tool. |
| Description | Description of the user role.                                                                                                                                                                 |
| Permissions | Permissions given to the user role.                                                                                                                                                           |
| Created At  | Date and time of the user role creation.                                                                                                                                                      |
| Updated At  | Date and time of the last update to the user role.                                                                                                                                            |
