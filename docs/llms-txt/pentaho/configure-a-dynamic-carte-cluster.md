# Configure a Dynamic Carte Cluster

This procedure is only necessary for dynamic cluster scenarios in which one Carte server will control multiple slave Carte instances.

**Note:** The following instructions explain how to create `carte-master-config.xml` and `carte-slave-config.xml` files. You can rename these files if you want, but you must specify the content in the files as per the instructions.
