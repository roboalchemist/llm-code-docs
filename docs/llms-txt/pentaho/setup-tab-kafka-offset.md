# Setup tab

![Setup tab for Kafka Offset job](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-0db59e19749d2d25d2cfe4f5528e08c669e3e53f%2FPDI%20Kafka%20Offset%20dialog%20box%20Setup%20tab.png?alt=media)

In this tab, define the connections used for receiving messages, topics to which you want to subscribe, and the consumer group for the topics.

| Option                | Description                                                                                                                                                                                                                                                                                                                                                                                                                      |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Bootstrap servers** | Specify the **Bootstrap servers** from which you want to receive the Kafka streaming data.                                                                                                                                                                                                                                                                                                                                       |
| **Topics**            | Enter the name of each Kafka topic from which you want to consume streaming data (messages). You must include all topics that you want to consume.                                                                                                                                                                                                                                                                               |
| **Consumer group**    | <p>Enter the name of the group of which you want this consumer to be a member. Each <strong>Kafka Consumer</strong> step will start a single thread for consuming.</p><p>When part of a consumer group, each consumer is assigned a subset of the partitions from topics it has subscribed to, which locks those partitions. Each instance of a <strong>Kafka Consumer</strong> step will only run a single consumer thread.</p> |
