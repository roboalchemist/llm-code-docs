# View hidden fields

If you want to view hidden fields in your Analyzer report, use the **Show Hidden Fields** option from the **View** menu.

1. In the **Available fields** list, click the Down Arrow next to the **View** option.
2. From the context menu which appears, click **Show Hidden Fields**.

   The hidden fields now show in gray in the **Available fields** list.

   ![Show hidden fields](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-4cda96716b03e357b3bce32b2b82db6177e8744f%2FAnalyzer_ShowHidden.png?alt=media)
3. To prevent the hidden fields from appearing in the list, clear the **Show Hidden Fields** option.

   ![Show Hidden Fields option](https://591371677-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FFgwF7oC7Y6b7XWdahykf%2Fuploads%2Fgit-blob-6b3c8e3a94f10d88db4b3ec61c33884ad4ad08df%2FAnalyzer_ShowHidden_Unselect.png?alt=media)

   **Note:** You can use hidden fields to create and edit the formulas for calculated measures. When you select the **Show Hidden Fields** option in the **View** menu for the**Available Fields** list, measures set as 'hidden' are available for selection in the calculated measure formula field. To view these hidden measures, you need the **Manage Data Sources** permission.
