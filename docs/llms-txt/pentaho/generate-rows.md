# Generate rows

The Generate rows step outputs a specified number of rows. By default, the rows are empty; however, they can contain a number of static fields. You can use this step to generate a fixed number of rows for testing purposes, for example, if you want to produce exactly 12 rows for 12 months. You can also use the step to generate one row as an initiating point for your transformation. For example, you could generate one row that contains two or three field values that can be used to parametrize your SQL and then generate usable rows.
