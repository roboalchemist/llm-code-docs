# Manage data identification methods

Pentaho Data Catalog uses data identification methods called [dictionaries](https://docs.pentaho.com/pdc-admin/pdc-manage-data-identification-methods/pdc-dictionaries) and [data patterns](https://docs.pentaho.com/pdc-admin/pdc-manage-data-identification-methods/pdc-data-patterns) to help you identify data.
