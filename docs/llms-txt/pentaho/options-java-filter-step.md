# Options

![](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-5a2164034158dc3a26827a31bacbab42bca283d1%2FPDI_TransStep_Java_filter_dialog_w650.png?alt=media)

The Java filter step includes the following settings:

| Option                                                | Description                                                                                                                                                                                                                                                                             |
| ----------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Destination step for matching rows (optional)**     | Specify the name of the step to receive rows evaluated as matching by the **Condition (Java expression)** option.                                                                                                                                                                       |
| **Destination step for non-matching rows (optional)** | Specify the name of the step to receive rows evaluated as non-matching by the **Condition (Java expression)** option.                                                                                                                                                                   |
| **Condition (Java expression)**                       | Specify a Java expression for filtering the data. See [Filter expression examples](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/pdi-transformation-steps-reference-overview/java-filter-pdi-step/filter-expression-examples-java-filter-step) for code samples. |
