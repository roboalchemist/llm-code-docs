# SSTable Output

The SSTable Output step writes to a filesystem directory as an [Apache Cassandra SSTable](http://cassandra.apache.org/doc/latest/architecture/storage_engine.html#sstables) using CQL (Cassandra Query Language) version 3.x.

**Note:** This step supports Cassandra 2.2 and later.
