# Service pack information

Perform the following steps to find a list of issues that have been resolved in a particular service pack:

1. On the [Support Portal](https://support.pentaho.com/home) home page, sign in using your Pentaho support username and password.
2. On the Pentaho card, click **Download**.
3. Select your service pack in the **Downloads** section.
4. Scroll down to the bottom of the page and locate the download window.
5. Locate the \<SP version #>-fixes.txt file. For example, SP-9.3.0.4-739-Fixes.txt.
6. Click the file name in the window to view the file or click the **More Options** icon to download the file to your local computer. You can also save the file as a PDF file on your Box account by choosing the **Integrations** > **Create PDF with Adobe Acrobat**.
