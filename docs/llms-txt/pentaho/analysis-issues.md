# Troubleshooting possible analytics issues

Follow the suggestions in these topics to help resolve a common issue with Pentaho Business Analytics:

* Multi-Byte characters do not appear in PDFs exported from Analyzer
* Geo Maps partially rendering
* Cannot change Administrator password in PUC

See the **Administer Pentaho Data Integration and Analytics** document for additional troubleshooting information.
