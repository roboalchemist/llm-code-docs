# Style properties reference

You can click on any element, and the **Style** pane will adjust to show all of the available style properties, listed by group.

**Note:** You cannot edit any Style or Attributes options for any selected report elements in the workspace while the **Data** tab has focus. Click the **Structure** tab to see the **Style** and **Attributes** panes for selected elements.

Every report element can be customized according to parameters in the following topics:

* [Font styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/font-styles)
* [Text styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/text-styles)
* [Text Spacing styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/text-spacing-styles)
* [Padding styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/padding-styles)
* [Object styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/object-styles)
* [Size and Position styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/size-and-position-styles)
* [Links styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/links-styles)
* [Excel styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/excel-styles)
* [Sparkline styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/sparkline-styles)
* [Page Behavior styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/page-behavior-styles)
* [Border styles](https://docs.pentaho.com/pba-report-designer/10.2-report-designer/style-properties-reference/border-styles)
