# Metadata injection support

You can use this step with [ETL metadata injection](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/pdi-transformation-steps-reference-overview/etl-metadata-injection) to pass metadata to your transformation at runtime.

The following fields of the Row Normaliser step support metadata injection:

* **Fieldname**
* **Type**
* **new field**
