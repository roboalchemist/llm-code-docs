# Method 2: Retrieve the ID from the PDI plugin registry

If you are a developer, you can retrieve the step ID from the PDI plugin registry as described in **Dynamically build transformations** in the **Administer Pentaho Data Integration and Analytics** document.

If you have created your own PDI transformation step plugin, the step ID is one of the annotation attributes that the developer supplies.
