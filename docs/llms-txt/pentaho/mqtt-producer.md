# MQTT Producer

The MQTT Producer step allows you to publish messages in near-real-time to an MQTT broker. Within a transformation, the MQTT Producer step publishes a [stream of records](https://docs.pentaho.com/pdia-data-integration/10.2-data-integration/streaming-analytics) to one MQTT topic.

For more information about the MQTT protocol, see: <https://www.hivemq.com/mqtt/>
