# Transforming multiple Suggested Row Records into a single Master Record

Transform multiple Suggested Row Records into a single Master Record when there are multiple source records identified for a Master Record, but you only want one Master Record for the data.

Perform the following steps to merge multiple Suggested Rows Records and transform the merged records into a single Master Record:

1. On the left navigation menu, click **Master Data**. The **Master Data** page opens.
2. Scroll down the page to view the Suggested Row Records table.
3. Select the checkboxes for the Suggested Row Records that you want to merge and transform into a single Master Record.
4. Click **Selected** > **Create Single Master**. The **Create Master** confirmation dialog box appears.
5. Insert a comment about why you are transforming multiple Suggested Row Records into a single Master Record and click **Create Master**. The selected Suggested Row Records are merged and transformed into a single Master Record, removed from the **Suggested Rows** table, and added to the **Master Record** table as a Master Record with the **Draft** status.
