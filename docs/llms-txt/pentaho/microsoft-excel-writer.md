# Microsoft Excel writer

The Microsoft Excel writer step writes incoming rows from PDI out to an MS Excel file and supports both the `.xls` and `.xlsx` file formats. The `.xls` files use a binary format which is better suited for simple content, while the .`xlsx` files use the Open XML format which works well with templates since it can better preserve charts and miscellaneous objects.
