# ORC types

The table below shows how a PDI type is converted to an applicable ORC type.

| PDI Type    | ORC Type  |
| ----------- | --------- |
| InetAddress | String    |
| String      | String    |
| TimeStamp   | TimeStamp |
| Binary      | Binary    |
| BigNumber   | Decimal   |
| Boolean     | Boolean   |
| Date        | Date      |
| Integer     | Integer   |
| Number      | Double    |
