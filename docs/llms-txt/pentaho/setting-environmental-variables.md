# Setting environmental variables

To install Pentaho Data Mastering, you must set environmental variables in the Docker container.

The following environmental variables are required to install Pentaho Data Mastering with either the online installation process or the offline installation process:

* SERVER\_IP
* MDM\_DB\_TAG
* MDM\_BE\_TAG
* MDM\_POSTGRES\_TAG
* OBS\_OPENOBSERVE\_TAG
* OBS\_NODEEXP\_TAG

The following additional environmental variables must be set for only the online installation process:

* JFROG\_USER
* JFROG\_TOKEN

Complete the steps in one of the following procedures to set environmental variables:

* [Setting environmental variables for online installation](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/setting-environmental-variables/setting-environmental-variables-for-online-installation)
* [Setting environmental variables for offline installation](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/setting-environmental-variables/setting-environmental-variables-for-offline-installation)
