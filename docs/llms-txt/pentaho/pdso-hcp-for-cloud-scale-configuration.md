# VSP One Object configuration

If you are using VSP One Object and you are configuring more than one hundred Data Optimizer instances, you need to increase the maximum number of buckets allowed for your user.

See [**Hitachi Content Platform configuration documentation**](https://docs.hitachivantara.com/home) for more information.
