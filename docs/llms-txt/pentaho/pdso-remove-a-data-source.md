# Disabling a data source

If your role allows it, you can disable a data source when it is no longer needed.

Perform the following steps to disable a data source:

1. On the **Menu**, click **Management**.

   The Manage Your Environment page opens.
2. On the **Data Sources** card, click **Imported**.

   The Data Sources page opens showing the list of existing data sources.
3. Locate the data source that you want to disable.
4. Click **More options** (![](https://2613472723-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FTt1SdXx3alZu8NJiPBj6%2Fuploads%2Fgit-blob-df92ee60a4db655bf59fd4e197fbbcd4fdbde731%2FPDSO%20More%20options.png?alt=media)) and select **Disable**.

The data source is disabled, and the listed status is **Disabled**.
