# Servers tab

![ElasticSearch Bulk Insert Servers Tab](https://3411831820-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FAYwCj9fPr1B2pjC11IOQ%2Fuploads%2Fgit-blob-ea86e089044e3d2d7829db265a771d5261c64739%2FPDI_TransStep_ElasticSearch_Bulk_Insert_Servers_Tab.png?alt=media)

| Option              | Description                                                                 |
| ------------------- | --------------------------------------------------------------------------- |
| **#**               | Number of the server entry.                                                 |
| **Address**         | IP address of the server you want to connect to.                            |
| **Port**            | Port number for the server you want to connect to.                          |
| **Test Connection** | Verifies that the connection can be made to the servers listed in this tab. |
