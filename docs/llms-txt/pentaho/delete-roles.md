# Delete roles

1. Click **Roles**.

   The list of available roles appears.
2. Select the role you want to delete from the **Available** list.
3. Click the **X** icon next to **Available**.

   A security message appears.
4. Click **Yes** to remove the role.

   The specified role is deleted.
