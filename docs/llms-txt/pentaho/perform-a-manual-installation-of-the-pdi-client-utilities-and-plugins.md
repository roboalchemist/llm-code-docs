# Perform a manual installation of the PDI client, utilities, and plugins

You can install the PDI client, utilities, and plugins by downloading a ZIP file and extracting the installation files for each component.

If you have the PDI client already installed, you can download just the plugins via the ZIP file or by using the PDI Marketplace in PDI. See [Install PDI plugins](https://docs.pentaho.com/install/9.3-install/pentaho-installation-overview-cp/installation-of-the-pentaho-design-tools/install-the-pdi-tools-and-plugins/perform-a-manual-installation-of-the-pdi-client-utilities-and-plugins/install-pdi-plugins) for further details.
