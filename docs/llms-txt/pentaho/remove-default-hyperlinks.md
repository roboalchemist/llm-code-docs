# Remove default hyperlinks

You can remove default hyperlinks by deleting the annotation containing the hyperlinks in the Mondrian schema file. Once hyperlinks are predefined in the schema, they cannot be disabled from within the Analyzer report. However, they are still editable if changes are necessary.
