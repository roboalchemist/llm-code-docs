# Installing Pentaho Data Mastering

To set up the Pentaho Data Mastering application, you must install the Pentaho Data Mastering server, components for the OpenObserve platform, and the Keycloak Identity and Access Management (IAM) server.

You can install Pentaho Data Mastering online, while connected to the internet, or offline, with files that you download before the installation process. The online installation process requires you to download files from a JFrog Artifactory repository. The offline installation process requires you to download files from Hitachi Content Platform Anywhere.

Before you begin, verify that you have completed the prerequisites:

[Pre-requisites for installation](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/pre-requisites-for-installing-master-data-management)

To install Pentaho Data Mastering, complete the following procedures in order:

1. [Downloading files and drivers](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/downloading-files-and-drivers)
2. [Installing configuration pre-requisites](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/installing-configuration-pre-requisites)
3. [Setting environmental variables](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/setting-environmental-variables)
4. [Installing the OpenObserve APM tool components](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/installing-the-openobserve-apm-tool-components)
5. [Setting up the Keycloak IAM server](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/setting-up-the-keycloak-iam-server)
6. [Installing the Pentaho Data Mastering server](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/installing-the-pentaho-data-mastering-server)
7. [Validating installation of Pentaho Data Mastering](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/validating-installation-of-pentaho-data-mastering)
8. [Validating installation of the OpenObserve tool](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/validating-installation-of-the-openobserve-tool)

After you have installed Pentaho Data Mastering, see the following related topics:

1. [Frequently used commands to verify Docker container details](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/frequently-used-commands-to-verify-docker-container-details)
2. [Uninstalling components for Pentaho Data Mastering](https://docs.pentaho.com/pentaho-data-mastering/installing-pentaho-data-mastering/uninstalling-components-for-pentaho-data-mastering)
