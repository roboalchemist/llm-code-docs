# Remove tab

![Remove tab in Select values](https://2745965000-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FvZoINBz5C5WnDZQlbOJQ%2Fuploads%2Fgit-blob-77c63187139a6934af150b6c36a09190f0510c2c%2FPDI_SelectValues_Remove_Tab.png?alt=media)

This tab allows you to remove fields from the input stream. Click **Get fields to remove** to add fields from the incoming PDI stream.

**Note:** Field removals can slow down the process since the field order gets internally changed.
