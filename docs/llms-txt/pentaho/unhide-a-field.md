# Unhide a field

In your Analyzer report, you can unhide fields which were previously hidden.

1. If you have not already done so, select the **Show Hidden Fields** option from the **View** menu to display all the hidden fields.
2. In the **Available fields** list, hover your mouse over the grayed-out field you want to show and click the Down Arrow.
3. From the context menu which appears, click **Set Hidden** so this option is no longer selected.

   This field is now visible in the list and available for selection in the report.
