# Install drivers for your Hadoop clusters

If you use Hadoop clusters with your Pentaho products, after upgrading to version 10.2, you must install the drivers for the Hadoop clusters you want to access. To conserve memory and storage resources, these drivers are not automatically installed for your upgraded version 10.2 Pentaho products. You can install the specific drivers manually or through the PDI client. For instructions, see [Set up Pentaho to connect to a Hadoop cluster](https://docs.pentaho.com/install/10.2-install/pentaho-configuration/tasks-to-be-performed-by-an-it-administrator/set-up-the-pentaho-server-to-connect-to-a-hadoop-cluster).
