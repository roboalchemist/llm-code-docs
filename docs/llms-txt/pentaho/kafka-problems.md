# Kafka problems

| Symptoms                 | Common Causes                                                                                                                    | Common Resolutions                                                                                                                                                                            |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Cannnot connect to Kafka | <ul><li>Bootstrap server information is incorrect.</li><li>Specified bootstrap server is down.</li><li>Firewall issue.</li></ul> | <ul><li>Verify that the bootstrap server was correctly entered.</li><li>Verify that the bootstrap server is running.</li><li>Verify that a firewall is not blocking the connection.</li></ul> |
