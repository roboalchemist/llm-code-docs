# wandb disabled

Disable W\&B.

## Usage

```bash  theme={null}
wandb disabled [OPTIONS]
```

## Options

| Option      | Description                          |
| :---------- | :----------------------------------- |
| `--service` | Disable W\&B service (default: True) |
