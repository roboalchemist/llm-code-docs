# wandb job describe

Describe a launch job. Provide the launch job in the form of: entity/project/job-name:alias-or-version

## Usage

```bash  theme={null}
wandb job describe JOB
```

## Arguments

| Argument | Description              | Required |
| :------- | :----------------------- | :------- |
| `JOB`    | No description available | Yes      |
