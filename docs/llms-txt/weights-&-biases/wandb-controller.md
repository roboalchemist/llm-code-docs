# wandb controller

Run the W\&B local sweep controller

## Usage

```bash  theme={null}
wandb controller SWEEP_ID [OPTIONS]
```

## Arguments

| Argument   | Description              | Required |
| :--------- | :----------------------- | :------- |
| `SWEEP_ID` | No description available | Yes      |

## Options

| Option      | Description                             |
| :---------- | :-------------------------------------- |
| `--verbose` | Display verbose output (default: False) |
