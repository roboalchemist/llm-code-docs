# Interface: Buffer

Represents a buffer of data.

The `Buffer` interface provides a set of properties that describe a buffer of data, including a handle and the buffer itself.

## Properties[#](#properties)

| Property | Type |
| --- | --- |
| `handle` | `string` |
| `buffer` | `Uint8Array` |
