# Type Alias: EffectType

```
type EffectType =  | EffectTypeShorthand  | EffectTypeLonghand;
```

The block type IDs for the effect blocks. These are the IDs used to create new effects using `cesdk.engine.block.createEffect(id)`. Refer to [EffectTypeShorthand](https://img.ly/docs/cesdk/vue/api/cesdk-js/type-aliases/effecttypeshorthand/) and [EffectTypeLonghand](https://img.ly/docs/cesdk/vue/api/cesdk-js/type-aliases/effecttypelonghand/) for more details.
