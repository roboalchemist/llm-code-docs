# Interface: \_UBQExportAudioOptions

Specifies options for exporting audio design blocks to various formats.

The `UBQExportAudioOptions` interface provides a set of properties that control the behavior and quality of the exported audio content. These options include settings for sample rate and number of channels.

Methods for configuring export settings for audio design blocks.

## Properties[#](#properties)

| Property | Type |
| --- | --- |
| `sampleRate` | `number` |
| `numberOfChannels` | `number` |
| `skipEncoding?` | `boolean` |
