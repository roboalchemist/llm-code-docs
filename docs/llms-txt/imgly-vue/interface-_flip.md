# Interface: \_Flip

Specifies the horizontal and vertical flip states of a design block.

The `Flip` interface provides a set of properties that indicate whether the design block is flipped horizontally or vertically.

Methods for configuring the flip states of a design block.

## Properties[#](#properties)

| Property | Type |
| --- | --- |
| `horizontal` | `boolean` |
| `vertical` | `boolean` |
