# Getint Version 1.91

### Release date: October 12, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This update focuses on improved stability, security, and app compatibility. Highlights include restored Azure → Jira migration support for dependencies, enhanced HubSpot pagination, and key fixes across GitHub, Jira, Salesforce, and Trello integrations.

#### New Features and Improvements

* **Comments Tab**: "Edit" and "Exclude" options are now hidden unless explicitly enabled.
* **HubSpot Ticket Owners**: Improved pagination for more reliable and consistent data fetching.
* **Security Enhancements**: Upgraded core libraries to resolve high-severity Snyk vulnerabilities and strengthen overall platform security.

#### **App Changes and Bug Fixes**

* **Azure → Jira Migration**: Restored support for the Dependencies feature, which wasn't working properly.
* **GitHub**: Improved compatibility with repositories that have Issues disabled.
* **Jira**: Fixed a bug where field options would reappear after clearing mappings.
* **Salesforce**: Added support for Multiple-Select Picklist fields.
* **Trello**: Resolved an issue that prevented new connections from being established.
