# Getint Version 1.100

### Release date: February 9, 2026 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

Enhanced platform security and scripting, plus app-specific fixes including Asana MultiSelect, ClickUp due dates, and improved GitLab handling.

#### New Features and Improvements

* **Intercom widget** added for Forge apps.
* **Enhanced endpoints for security validation**.
* **Cache invalidation time** is now controllable via custom properties.
* The delete/update comments feature now appears only for **supported connectors**.
* **Left and right items** are now available for status update scripting hooks.

#### **App Changes and Bug Fixes**

* Bugfix for **clearing mapped options**.
* **Airtable**: Improved support for custom text fields. Adding support for rich text formatting.
* **Asana**: NullPointerException fixed for MultiSelect fields.
* **Asana-GitHub**: Markdown conversion improved between these connectors.
* **ClickUp**: Due date support added.
* **Git**: Commit messages longer than 1000 characters are now truncated.
* **GitLab**: "Delete attachments" option removed from comments & attachments section.
* **Jira-Jira (Forge)**: An additional mechanism for license verification has been implemented.
* **Monday**: Quickbuild no longer maps illogical work item types.
