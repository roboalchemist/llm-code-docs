# Getint Version 1.80

### Release date: April 24, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces a range of powerful new features, significant performance enhancements, and essential bug fixes—all aimed at delivering a smoother, more efficient, and highly reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Optimized auto-mapping logic for more precise and reliable field matching.
* Refined display of merged fields to enhance visibility and usability.
* Introduced advanced update strategies to manage simultaneous changes efficiently.

#### **App Changes and Bug Fixes**

* Resolved integration filter issue that previously caused incorrect app list displays.
* **Asana**: Incoming syncs now automatically convert HTML to plain text.
* **HubSpot**: HubSpot now supports attachments and includes retry-after handling to better manage API rate limits.
* **Jira**: Issue ID can now be mapped, and migration prioritizes creation time instead of update time for improved accuracy.
* **Jira Cloud**: Now features a new side panel with sync insights, allowing users to view status, timestamp, linked ID, and URL, with further enhancements coming soon. Additionally, it has been updated to a new search API endpoint for improved efficiency.
* **Salesforce**: Addressed OAuth issues affecting specific edge cases.
* **ServiceNow**: Resolved attachment syncing issues when using OAuth.
