# Getint Version 1.77

### Release date: March 13, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release brings a variety of new features, important performance improvements, and crucial bug fixes, all designed to provide a better and more reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Optimized sync log writing to prevent Getint from hanging.
* Added new date field operators for UI filters: IS\_AFTER, IS\_BEFORE.
* Script engine now supports Java 1.8+.
* Large Jira/ServiceNow user lists now have pagination and filtering.

#### **App Changes and Bug Fixes**

* Fixed import/export issues.
* Fixed attachment sync issue with accented filenames.
* Getint On-Premise now requires Java 1.8+.
* **Freshdesk/Freshservice**: Added support for cascading fields.
* **Git**: Updated the licensing model, and enabled syncing for commits, branches, and pull requests from public GitHub repositories.
* **Jira**: Added support for "Assignee emails" as a dedicated field.
* **Monday.com**: Resolved issues with Group field mapping, and implemented support for subitem comments.
* **Notion**: Introduced a 2,000-character limit for text fields, and added a single-select option for the assignee field.
