# Getint Version 1.61

### Release date: June 15, 2024 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release brings a variety of new features, important performance improvements, and crucial bug fixes, all designed to provide a better and more reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Enhanced security for cloud infrastructure.
* Added ability to search runs and syncs by ID.

#### **App Changes and Bug Fixes**&#x20;

* Fixed issues in Docker `destroy.sh` script
* **Azure**: Improved Quick Build automappings.
* **Freshdesk/Freshservice**: Fixed license reading issues.
* **HubSpot**: Improved Quick Build for Ticket type.
* **Monday**: Fixed comment sync order.
* **Notion**: Resolved paragraph field sync and status syncing issues.
* **Zendesk**: Fixed comment pagination issues.
