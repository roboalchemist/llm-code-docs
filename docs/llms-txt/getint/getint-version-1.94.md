# Getint Version 1.94

### Release date: November 14, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release delivers platform enhancements for status mapping, attachment handling, scripting, and credentials, along with fixes for Jira–Asana attachments and Monday.com board fetching.

#### New Features and Improvements

* Improved status mapping for multiple projects by including project names, ensuring correct status association.
* Introduced a custom property to set a minimum creation date for attachments to be migrated, helping avoid duplicate imports.
* Added support for `api.sync()` in the scripting engine, enabling advanced workflow automation for syncing cloned issues.
* Improved null value handling in connectors to prevent unnecessary updates when fields are empty.
* **Dynamic Credentials**: Improved connection handling and caching for scripts.

#### **App Changes and Bug Fixes**

* Fixed an issue where attachments were not syncing from Jira to Asana unless added with a comment.
* Resolved an error that occurred when fetching a large number of boards from Monday.com.
