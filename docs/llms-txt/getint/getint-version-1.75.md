# Getint Version 1.75

### Release date: February 13, 2025 <a href="#release-date-february-13-2025" id="release-date-february-13-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces an array of new features, significant performance enhancements, and essential bug fixes, all aimed at delivering a smoother and more efficient user experience when using Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Added an option to disable multi-mapping fields for better customization.
* Enhanced logging to improve visibility into license-related issues.
* Clearer guidance on setting migration time ranges.
* New support for concatenating multiple fields, adding more flexibility to data mapping. More information about this feature [here.](https://docs.getint.io/guides/integration-synchronization/how-to-map-fields#concatenating-fields)

#### Bug Fixes <a href="#bug-fixes" id="bug-fixes"></a>

* Fixed an issue where duplicate settings triggered incorrect license errors.
* Resolved a bug where the field mapping direction was being reset unexpectedly.
* **Jira DC**: Fixed an issue where the full list of available fields wasn't loading correctly.
