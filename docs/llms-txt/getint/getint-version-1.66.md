# Getint Version 1.66

### Release date: August 26, 2024 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This update introduces several improvements to enhance performance, usability, and integration across platforms and applications.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Improved authentication processes and optimized the integrations list page for better performance.
* Notifications are now fully editable, mapped options now retain default mapping when copied, and app connections now follow a stepped wizard with a back option.

#### **App Changes and Bug Fixes**

* Resolved issues with running integrations at specific hours and loading options for specific data types.
* **HubSpot**: Improved ticket sync triggers.
* **Jira DC**: Fixed parent link in hierarchy, resolved license error in the database, improved resync using items from other integrations, added support for notification edits, and introduced custom properties for better Getint control.
* **Notion**: Fixed comment creation with special characters and improved support for custom date fields.
* **ServiceNow**: Resolved issue with fetching issue types.
* **Wrike**: Support for US Wrike data centers was added.
