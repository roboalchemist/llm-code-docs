# Getint Version 1.81

### Release date: May 07, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces a range of powerful new features, significant performance enhancements, and essential bug fixes—all aimed at delivering a smoother, more efficient, and highly reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Introduced a script-level mechanism to identify an item's counterpart, making mappings more flexible and dynamic.
* Added support for dedicated folders to store run logs in Getint On-Premise installations, improving traceability and debugging.
* Launched a shared script library, allowing reusable scripts and utility functions across integrations for cleaner, more maintainable configurations.

#### **App Changes and Bug Fixes**

* Resolved integration filter issue that previously caused incorrect app list displays.
* **ClickUp**: Added support for syncing custom statuses to ensure workflows are accurately reflected across platforms.
* **HubSpot**: Improved comment-fetching capabilities to support longer conversations, alongside a fix for ticket status mapping in auto-build processes, ensuring greater accuracy.
* **Jira DC Apps**: Now exclusively supports Jira 9.0.0+, while also resolving dependency compatibility issues that previously blocked plugin installation. Additionally, custom fields now fetch properly across the entire Jira instance, improving data consistency.
