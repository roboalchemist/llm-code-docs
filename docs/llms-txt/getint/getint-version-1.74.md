# Getint Version 1.74

### Release date: January 30, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces a range of powerful new features, significant performance enhancements, and essential bug fixes—all aimed at delivering a smoother, more efficient, and highly reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Enhanced performance optimization for integration threads, ensuring faster and more efficient processing.

#### **App Changes and Bug Fixes**

* **Azure**: Updated endpoint version to improve field fetching.
* **Freshservice**: Added support for custom paragraph fields.
* **Git**: Integrations now visible in the list; improved GitHub commit fetching and sync logs.
* **HubSpot**: Enhanced pipeline stages and support for date/number fields.
* **Jira**: Resolved cascading field issues for Cloud/DC, improved multi-choice field sync, and enabled loading of fields unavailable in Create Meta.
* **Jira DC**: Improved Shared Mappings support.
* **Jira/Freshservice**: Cached comment authors for improved sync performance.
* **Salesforce**: Optimized Chatter comments sync.
