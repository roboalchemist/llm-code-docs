# Getint Version 1.82

### Release date: May 26, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

Getint now supports one-to-many link mapping, making data connections more flexible. Jira improves proxied connections and asset fetching. Azure DevOps fixes HTML link formatting, while Salesforce enhances search and performance. HubSpot, ClickUp, and Zendesk bring updates for better syncing and more useful comments.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Added support for one-to-many link mapping.

#### **App Changes and Bug Fixes**

* **Azure DevOps**: Fixed HTML field link formatting.
* **ClickUp**: Sync now supports custom statuses.
* **HubSpot**: Custom object sync support.
* **Jira**: Improved support for proxied connections and asset fetching now supports pagination.
* **Salesforce**: Performance optimizations, custom SOQL query support for search, support for reference/lookup fields, and the option to filter comments by privacy.
* **Zendesk**: Synced comments now include author names in footers.
