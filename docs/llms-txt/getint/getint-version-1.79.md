# Getint Version 1.79

### Release date: April 12, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release brings a variety of new features, important performance improvements, and crucial bug fixes, all designed to provide a better and more reliable experience with Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Paginated dropdowns now work more smoothly, making it easier to navigate and select options.
* Multi-threading mode is now available for on-premise deployments, allowing tasks to run faster.
* The scripting engine has been upgraded with new features:
  * A new AFTER SYNC hook for post-sync actions.
  * Access to state.leftItem and state.rightItem in ON BEFORE REQUEST.
  * Better support for JSON with the api.log() function.
  * Global scripts are now kept in a separate .js file for better organization.
  * The Monaco editor now includes auto-completion to make writing scripts quicker and simpler.

#### **App Changes and Bug Fixes**

* **Azure/Asana**: Cleaned up description formatting by removing raw HTML tags.
* **Git**: Corrected how branch URLs are resolved for Azure repositories.
* **HubSpot**: Raised the limits for fetching companies and contacts.
* **Jira**: Improved how sprint fields are extracted for issues.
* **Jira DC**: Fixed issues with handling dedicated licenses.
* **Monday**: Resolved errors that occurred when extracting field values.
* **Salesforce**: Added a configurable header to help manage reassignment logic.
