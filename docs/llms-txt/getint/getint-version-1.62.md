# Getint Version 1.62

### Release date: June 26, 2024 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

In the latest round of updates, significant improvements have been introduced to streamline integrations and enhance platform functionality. A new feature now allows users to replace connections for specific integrations, ensuring greater flexibility in managing workflows. Additionally, warning messages have been refined to display the property names that triggered issues, such as missing mappings or conversion errors, making troubleshooting more efficient.

These updates reinforce a commitment to seamless integrations, user-friendly functionality, and ongoing improvements across multiple platforms.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Added a feature to replace the connection for specific integrations.
* Improved warning messages to display property names that caused issues, such as missing mappings or conversion errors.

#### **App Changes and Bug Fixes**&#x20;

* **Azure**: Enhanced quick build functionality.
* **Freshservice**: Resolved license verification issues for the Freshservice Jira Cloud app.
* **Jira**: Fixed a bug preventing the use of the quick build feature, and addressed an issue where labels containing only numbers failed to sync.
* **Jira DC Apps**: Fixed a bug in request body parsing that prevented modifications in older integrations.
* **Notion**: Added support for custom date and datetime fields.
* **Zendesk**: Improved quick build performance.
