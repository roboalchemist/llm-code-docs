# Getint Version 1.98

### Release date: January 20, 2026 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release focuses on improving sync reliability and visibility across the platform, with enhanced resync capabilities for Many-to-Many integrations and fixes to sync completion and status accuracy. It also includes targeted improvements and bug fixes for Jira, GitHub, and Zendesk integrations.

#### New Features and Improvements

* **UI Resync for Many-to-Many Integrations**: Items synced through Many-to-Many integrations can now be resynced directly from the UI.
* **Improved Sync Completion and Status Accuracy**: Fixed issues that prevented some syncs from finishing and improved the accuracy of sync status presentation in the UI.

#### **App Changes and Bug Fixes**

* **GitHub**: Improved the process of fetching GitHub projects, resolving issues that occurred when no personal projects were available.
* **Jira**: Disabled editing of connections using the Atlassian OAuth method to prevent invalid or unintended modifications.
* **Zendesk**: Fixed an issue with status synchronization to ensure ticket status updates are reflected accurately.
