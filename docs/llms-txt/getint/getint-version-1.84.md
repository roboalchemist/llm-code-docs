# Getint Version 1.84

### Release date: June 20, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

We’re excited to announce key enhancements across platforms and apps designed to improve performance, reliability, and data handling. This release focuses on sync accuracy, integration stability, and better field support.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* **Smarter Comment Sync**: Improved direction logic for comment synchronization ensures comments now flow accurately between integrated platforms based on configuration.
* **Dependency Link Deletion Support**: You can now delete dependency links directly via sync, offering more control over relationship management.
* **Performance Boost**: Increased database pool size to better handle heavy loads and improve overall responsiveness in high-volume environments.

#### **App Changes and Bug Fixes**

* **Attachment Sync Fixes**: Resolved issues in one-way sync scenarios where attachments weren’t properly synchronized or deleted, ensuring cleaner data integrity.
* **Azure DevOps**: Addressed problems fetching **work item links**, enabling better visibility across linked tasks.
* **Git**:
  * Now supports **pull request statuses**, making it easier to track code review stages.
  * Improved **connection setup** and overall integration usability for a more intuitive onboarding experience.
* **Jira Data Center**: Fixed import issues when integrating with **existing connections**, enhancing reliability for enterprise-scale workflows.
* **Salesforce**: Introduced support for **currency fields**, enhancing financial data tracking and consistency across integrations.
* **ServiceNow**: Added support for setting **parent fields**, giving users more flexibility in structuring incidents and requests.
