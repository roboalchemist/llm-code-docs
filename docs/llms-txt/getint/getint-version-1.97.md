# Getint Version 1.97

### Release date: December 22, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release delivers stronger integration performance across both platforms and apps. *Many Projects* gain new routing filters and safer sync behavior, while Jira and Salesforce now handle API rate limits with retry logic to prevent duplicates. GitHub adds migration functionality, making transitions smoother and more reliable. Together, these updates reduce errors and improve efficiency across connected systems.

#### New Features and Improvements

* **Enhanced Routing Filters**: Added new filtering options within *Many Projects* integrations to improve control and flexibility.
* **Improved Sync Safety**: Disabled the *recreate* option for items synced by *Many Projects* to prevent unintended behavior.

#### **App Changes and Bug Fixes**

* **Jira / Salesforce**: Introduced support for API rate limits and implemented retry logic to handle request failures, ensuring stability and preventing duplicate item creation.
* **GitHub**: Added migration functionality to streamline project transitions.
