# Getint Version 1.76

### Release date: February 24, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces an array of new features, significant performance enhancements, and essential bug fixes, all aimed at delivering a smoother and more efficient user experience when using Getint.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Added search functionality for dropdown options by ID.
* Enabled HARD RESTART for integration threads.
* Enhanced notifications with more detailed information.

#### App Changes and Bug Fixes <a href="#app-changes-and-bug-fixes" id="app-changes-and-bug-fixes"></a>

* **Asana**: Implemented support for multi-select custom fields.
* **GitHub**: Addressed a bug in assignee synchronization and enhanced the quick build functionality.
* **GitLab**: Enabled fetching attachments from descriptions.
* **HubSpot**: Increased fetch limits for contacts and companies, improved ticket status retrieval, and added support for deal line items.
* **Jira**: Set the "Linked Issues" field to readonly, displaying all linked issues as text.
* **Salesforce**: Fixed a bug preventing the transmission of transition fields.
* **ServiceNow**: Resolved issues with comment filtering, enhanced fetch limits, and introduced support for attachment deletion. A new parent ticket reference number field was added.
* **Trello**: Optimized quick build performance.
