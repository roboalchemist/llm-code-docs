# Getint Version 1.96

### Release date: December 10, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release introduces **Dynamic Fields** for advanced field mapping and scripting, along with usability and validation improvements across the platform. App updates include fixes for **GitHub** quickbuild and **Jira** work item property updates.

#### New Features and Improvements

* **Dropdown usability fix**: Reopening dropdowns now preserves filter input.
* **Dynamic Fields**: A major enhancement that allows users to manage unsupported or custom fields through scripting.
  * Fields can be defined and mapped in integrations, with values calculated dynamically at runtime.
  * Enables advanced field mapping, supports complex business logic, and reduces the need for custom development.
  * Supported types: **String**, **IdLabel**, **Number**, and **Date**.
* **Multi-field mapping improvements**: Duplicate field mappings are now automatically prevented before configuration is applied.
* **Shared mappings validation**: Users can only select shared mappings that match the integration direction.

#### **App Changes and Bug Fixes**

* **GitHub**: Fixed quickbuild for Projects-type integrations.
* **Jira**: Work item properties now update correctly when items move between projects.
