# Getint Version 1.68

### Release date: October 18, 2024 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This update introduces several improvements to enhance performance, usability, and integration across platforms and applications.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* Getint OnPremise now supports Java 17+.
* Integrations change history is now available for easier troubleshooting.
* Shared Mappings feature introduced in beta—an efficient way to centrally manage field configurations across integrations.
* Value extraction improvements when using **Label From Other Side**.

#### **App Changes and Bug Fixes**

* **Freshdesk**: Added support for Agent, Group, and Type fields. Fixed the attachment synchronization issue caused by an incorrect endpoint. Possible requesters can now be mapped.
* **Jira DC**: Bugfix for fetching all assignable users for a given Jira project. Bugfix for missing libraries required for the quick-build feature.
* **Notion**: Added support for numerical fields.
* **Salesforce**: Added support for Chatter/feed comments synchronization.
