# Getint Version 1.99

### Release date: January 29, 2026 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

The latest update to Getint focuses on infrastructure security, improved authentication stability for major integrations, and expanded regional support for Forge. Version 1.99 introduces a modernized frontend build system and refined logic for handling complex field mappings across our supported apps.

#### New Features and Improvements

* **Improved OAuth token refresh logic** for Salesforce, ServiceNow, and Jira, now with retry handling for expired tokens.
* Dynamic fields upgraded to use an **improved REST client** for fetching data from HTTP endpoints.
* **Fixed UI Sync status display** to accurately reflect errors.
* The frontend build has been migrated from **Create React App** to **Vite**, resolving security alerts.

#### **App Changes and Bug Fixes**

* **Asana**: QuickBuild improved with auto-mapping for Asana types.
* **Forge**: Data Residency expanded to support Europe and USA regions.
* **Freshdesk**: Fixed merging of Description fields with others.
* **Freshservice**: Requester field now retrieves both agents and customers.
* **Jira**: Fixed transitions to statuses with parentheses in names, and transition fields no longer skipped if absent from the edit screen; errors now surface directly from Jira.
* **Jira - GIT**: OAuth authentication is now available for enhanced security.
* **Jira DC**: Added logic for updating user roles.
