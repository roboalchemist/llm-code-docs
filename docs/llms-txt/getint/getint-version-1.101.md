# Getint Version 1.101

### Release date: February 17, 2026 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

This release focuses on improving platform reliability, simplifying debugging, and strengthening security with new authentication features.

#### New Features and Improvements

* Introduced multi-factor authentication (2FA) support for **app.getint.io**, adding an extra layer of security for all user accounts.
* Improved script usage with minor enhancements for better reliability.
* **Enhanced dynamic credentials**: logs can now be printed for improved debugging and visibility during execution.
* Refined null-value handling for dropdown fields, allowing mapping and defaulting  `"null"` where appropriate.

#### **App Changes and Bug Fixes**

* **Jira**: Added support for mapping null values in dropdown fields, improving integration reliability and alignment with platform-wide enhancements.
