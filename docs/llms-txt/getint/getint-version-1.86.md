# Getint Version 1.86

### Release date: July 25, 2025 <a href="#release-date-february-24-2025" id="release-date-february-24-2025"></a>

### Overview <a href="#overview" id="overview"></a>

August update brings faster performance, a better code editor with syntax highlighting and full‑screen mode, and reduced latency. Fixes include improved sprint syncing in Jira, correct handling of attachments in Asana, and better encoding for Airtable and Azure.

#### New Features and Improvements <a href="#new-features-and-improvements" id="new-features-and-improvements"></a>

* **Backend Performance**: Optimized data handling to lower latency, especially for large‑scale integrations.
* **Code Editor Upgrades**: Now with syntax highlighting, line numbers, smarter auto‑complete, and a distraction‑free full‑screen mode for smoother scripting.
* **JavaScript API – Run‑Scoped Caching**: Added the ability to cache data during a single run, reducing repeated API calls and significantly improving script performance.

#### **App Changes and Bug Fixes**

* **Airtable**: Fixed mismatches between field names and IDs.
* **Asana**: Fixed incorrect encoding of attachment filenames.
* **Azure**: Corrected attachment filename encoding issues.
* **GitHub**: Added support for GitHub Projects.
* **HubSpot**: Prevented duplicate attachments during sync.
* **Jira**: Added validation for create/edit meta fields before syncing, improved user caching to speed up comment handling, and introduced an option to specify a board for Sprint sync.
