# Overview


`OpenAPI (Swagger) Extensions` are enhancements or customizations to the OpenAPI specification designed to meet specific needs or provide additional functionality. The OpenAPI specification allows developers to describe all details of a RESTful API, but sometimes the standard specification cannot fully cover all scenarios, necessitating the use of extensions as a supplement.

`Apidog OpenAPI (Swagger) Extensions` are custom properties added to the OpenAPI (Swagger) specification by Apidog to enhance the functionality and organization of API documentation. They do not affect the actual functionality of the API but can significantly improve the API documentation process and project management.
