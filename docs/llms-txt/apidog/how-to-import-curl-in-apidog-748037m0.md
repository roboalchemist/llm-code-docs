# How to import cURL in Apidog?

1. Hover over or click the `+` button adjacent to the search box on the left side. From the drop-down menu, select "Import cURL". Alternatively, you can use the shortcut key `Ctrl`(`⌘`) + `I`.
<Background>
   ![](https://assets.apidog.com/uploads/help/2023/07/18/b727caa5fa00c20a4d3ecb30895b8110.png)
</Background>
2. In the dialog that appears, paste the cURL format data previously copied from your selected tool.
<Background>
   ![](https://assets.apidog.com/uploads/help/2023/07/18/1d6577412a293005a1aa0b365fb46c6d.png)
</Background>
3. Click the 'OK' button. You will see the captured packet data loaded into the quick debugging API interface.
<Background>
   ![](https://assets.apidog.com/uploads/help/2023/07/18/eb9d7af89e7536ba0a0474d8de7c10ab.png)
</Background>
4. This quick debugging interface allows you to directly test and debug the API. Once confirmed, click the 'save' button to save the API configuration permanently.
<Background>
   ![](https://assets.apidog.com/uploads/help/2023/07/18/e7e296b83c5bcb070757038f48fa2c44.png)
</Background>
This streamlined process ensures efficient handling and reutilization of captured web traffic data.
