# Custom Layouts

Customize your documentation site's layout to create an optimal user experience. Page layout settings allow you to add navigation menus, banners, login buttons, and other convenient entry points to your published documentation sites. These customization options help you create professional, branded documentation that aligns with your organization's needs.

</Background>


## Top Navigation

Customize your navigation menu with three types of navigation options: **Link**, **Folder**, and **Multiple Menus**.


<Background>
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/364865/image-preview)
</Background>


The top navigation right-side supports adding **Button**, **Icon**, and **Link** components for additional functionality.
