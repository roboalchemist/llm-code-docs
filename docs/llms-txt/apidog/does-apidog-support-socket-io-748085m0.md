# Does Apidog support Socket.IO?

Yes, Apidog does support Socket.IO! 

> Socket.IO support is available in [Apidog version 2.7.0 or higher](https://apidog.com/download). 

Apidog offers comprehensive Socket.IO testing and documentation capabilities. Here are the key features:

1. Socket.IO Connection
   
   - Supports both ws:// and wss:// protocols
   - Compatible with Socket.IO v2, v3, and v4 (v4 by default)
   - Allows custom handshake paths and parameters
  
2. Event Management

   - Listen to multiple events simultaneously
   - Default listening to the message event
   - Dynamic event adding/removing

3. Message Handling
   
   - Send messages with single or multiple arguments
   - Support for JSON, text, and Binary data
   - Acknowledgment (Ack) support for callback handling

4. Documentation
   
   - Save Socket.IO endpoints in project structure
   - Generate comprehensive documentation
   - Share endpoints with team members
   
5. Advanced Features
   
   - Variable support in arguments
   - Customizable handshake parameters (Headers, Cookies, URL params)
   - Request configuration settings

[You can learn more about how to debug Socket.IO using Apidog here.](https://docs.apidog.com/socket-io-865459m0.md)
