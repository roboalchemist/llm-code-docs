# Associated Test Scenarios

Endpoints can be imported as test steps in automated test scenarios. When an endpoint is imported as a test step, it is covered by the automated test.

## Viewing Associated Test Scenarios

Click the association icon at the top-right corner of the endpoint specification to see which test scenarios are associated with this endpoint and the total number of test scenarios that use it, giving you an overview of its test coverage.

<Background>
![Endpoint imported as test step](https://api.apidog.com/api/v1/projects/544525/resources/348290/image-preview)
</Background>

Clicking a test scenario takes you to the test scenario detail page, where related steps using the endpoints are highlighted for easy viewing.
