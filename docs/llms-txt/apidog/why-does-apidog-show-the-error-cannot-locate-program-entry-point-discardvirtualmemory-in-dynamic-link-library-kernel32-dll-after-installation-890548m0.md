# Why does Apidog show the error "Cannot locate program entry point DiscardVirtualMemory in dynamic link library KERNEL32.dll" after installation?

This error usually occurs because the version of your Windows system is not compatible with the version of Apidog you have installed. Apidog has prepared compatible versions for systems like Windows 7 and 8. Please click the following link and select the appropriate installation package for your operating system.：
