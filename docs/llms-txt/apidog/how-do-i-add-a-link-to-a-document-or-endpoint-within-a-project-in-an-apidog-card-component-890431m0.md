# How do I add a link to a document or endpoint within a project in an Apidog card component?

You can use the following methods to create a jump link within a card component.
```js
<Card title= "Click to Jump" href="https://docs.apidog.com/5092448m0.md">
   Clicking this card will jump to the document page.
</Card>

<Card title="Click to Jump" href="https://docs.apidog.com/172828955e0.md">
   Clicking this card will jump to the interface page.。
</Card>

<Card title="Click to Jump" href="https://docs.apidog.com/41143239f0.md">
   Clicking this card will jump to the directory page.。
</Card>
```

## How to quickly get documents, endpoints, and folder paths?
Click the "Insert Endpoint/Document in Project" button in Apidog, and after selecting the resource insertion, the resource path will be displayed on the page.
