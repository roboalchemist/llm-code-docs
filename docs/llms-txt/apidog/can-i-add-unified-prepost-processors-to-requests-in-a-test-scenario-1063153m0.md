# Can I add unified pre/post processors to requests in a test scenario?

Yes, Apidog supports configuring unified pre/post processors for test scenarios. Once enabled, all requests within the scenario will automatically execute the same pre/post processors when the scenario runs.

To enable this feature:

1. Go to Project Settings.

<Background>  
![img_v3_02mu_2a6d3e2f-8a5b-407c-a0f9-94a28dc692fg.png](https://api.apidog.com/api/v1/projects/544525/resources/356103/image-preview)
</Background>


2. Turn on the "pre/post processors for Scenarios" switch.

<Background>
![img_v3_02mu_47a53922-efbe-4463-a4ff-11f1822991bg.png](https://api.apidog.com/api/v1/projects/544525/resources/356105/image-preview)
</Background>

After enabling the switch, you can add your desired actions directly from the Scenario Details page.
