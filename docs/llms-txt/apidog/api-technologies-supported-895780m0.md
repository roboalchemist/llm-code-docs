# API Technologies Supported

Modern software ecosystems rely on diverse API technologies, each serving specific use cases. REST APIs power web services, GraphQL optimizes data fetching, WebSockets enable real-time communication, and gRPC delivers high-performance microservices. Apidog provides comprehensive documentation support for all major API technologies, ensuring your team can document any API type with the same powerful toolset.

Apidog supports writing documentation for almost all types of APIs, including `REST`, `SOAP`, `GraphQL`, `gRPC`, `WebSocket`, `SSE`, and more.

Explore our collection of documentation best practices and real-world API examples to learn how to effectively document each API technology.
