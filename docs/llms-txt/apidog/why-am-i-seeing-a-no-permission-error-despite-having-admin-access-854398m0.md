# Why am I seeing a 'No Permission' error despite having admin access?

**Q: If I am the admin of a project, is there any chances that I will not be able to access the project due to permission issue?**
A: Yes, there can be such issue if you are on a free plan and added the maximum numbers of member (5) for a single project and if the project number exceeds numbr (5) will result the permission issue.

**Q: How to get rid of the issue?**
 A: You can remove one or multiple members from the project and it will restore your accessibility or you can proceed with [Purchasing Apidog Plans](https://apidog.com/pricing/). You can check the current members if you come back to application home and find the project , there you can click on members to find out the members you have added to the ptoject and if you are free plan and have added 5 maximum users, you need to remove them or limit the Project count to 5 to restore permission.
