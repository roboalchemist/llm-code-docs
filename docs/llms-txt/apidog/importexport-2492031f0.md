# Import/Export

## Docs

- [How to import API data into Apidog?](https://docs.apidog.com/how-to-import-api-data-into-apidog-748036m0.md): <Steps>
- [How to import cURL in Apidog?](https://docs.apidog.com/how-to-import-curl-in-apidog-748037m0.md): 1. Hover over or click the `+` button adjacent to the search box on the left side. From the drop-down menu, select "Import cURL". Alternatively, you can use the shortcut key `Ctrl`(`⌘`) + `I`.
- [How to migrate Postman environments to Apidog?](https://docs.apidog.com/how-to-migrate-postman-environments-to-apidog-748038m0.md): 1. Open Postman and go to the "Environments" tab.
- [How to auto - group endpoints on Swagger/OpenAPI import?](https://docs.apidog.com/how-to-auto-group-endpoints-on-swaggeropenapi-import-980378m0.md): When importing a Swagger/OpenAPI file in Apidog, the grouping of APIs is automatically completed based on the endpoint's tags. The system will set the first tag of the endpoint as the folder of the endpoint.
