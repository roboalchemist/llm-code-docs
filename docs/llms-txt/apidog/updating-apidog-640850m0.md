# Updating Apidog

Apidog routinely receives updates to enhance its functionality and user experience. Keeping your application up-to-date ensures you have access to the latest features, security patches, and performance improvements.

## Check the Current Version

To determine which version of Apidog you're currently using:

1. Click on your avatar in the upper right corner and select **Settings**.

<Background>
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/371451/image-preview)
</Background>



2. Navigate to **About** within the settings menu to check the version number of Apidog.

:::info[]
The About section displays your current version number, build date, and other system information that may be helpful when reporting issues or checking compatibility.
:::

## View the Changelog

To understand the enhancements and changes in each new release of Apidog:

1. Go to **Settings** and select **About**.

2. Click on **Changelog** to get detailed information about the updates included in the latest version.

:::info[]
The changelog provides a comprehensive list of new features, bug fixes, and improvements in each release, helping you understand what's changed and how it might affect your workflow.
:::

## Check for Updates

- Once you're familiar with the changes from the changelog, you can update your Apidog software:

- From the **About** section, click on **Check Update** to check if there's an update you can install.
- When there's an update, you can click on **Install Update** to install the latest version.

:::tip[]
You can receive notifications when updates are available by checking the reminder check box under **Software Update**.
:::

By keeping Apidog up-to-date, you ensure access to the latest tools, security updates, and improved features that can significantly enhance your experience and productivity.
