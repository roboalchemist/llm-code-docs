# Sharing Endpoints with API Versions

The Quick Share feature allows you to create online shares for endpoints specific to a particular API version, enabling targeted collaboration with different stakeholders.

When creating a new share, you can select the scope of endpoints for Quick Share. Choose a specific API version, then select all or some endpoints from that version to generate an online share link.

<Background>
![Quick Share API Version Endpoints](https://api.apidog.com/api/v1/projects/544525/resources/371667/image-preview)
</Background>
Users accessing the online share link will view only the content you've shared from the specific API version.
