# Music

Module to generate music related entries.

## Module Overview

Generate random music content.

For a random album name, use `{{$music.album}}`.

For a random artist, use `{{$music.artist}}`.

For a random genre, use `{{$music.genre}}`.

For a random song name, `{{$music.songName}}`.

All data types may be localized.

---

## album

Returns a random album name.

**Returns**: string

**Examples**

```js
{{$music.album}}  // 'El Viaje De Copperpot'
```
---

## artist

Returns a random artist name.

**Returns**: string

**Examples**

```js
{{$music.artist}}  // 'Chayanne'
```
---

## genre

Returns a random music genre.

**Returns**: string

**Examples**

```js
{{$music.genre}}  // 'Acid Jazz'
```
---

## songName

Returns a random song name.

**Returns**: string

**Examples**

```js
{{$music.songName}}  // 'Bad Romance'
```
