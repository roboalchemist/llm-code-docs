# Does Apidog support request code for custom endpoints?

Apidog supports customizing sample code in the endpoint. You can do this by following these steps:

1. Go to Project Settings -> Function Settings -> Endpoint Function Settings and turn on the sample code switch in the endpoint field


<Background>  
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/352421/image-preview)
</Background>

2. Add the sample code of the API on the API Editing page


<Background>
 
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/352429/image-preview)
</Background>
3. After saving, the sample code can be displayed in the API documentation
