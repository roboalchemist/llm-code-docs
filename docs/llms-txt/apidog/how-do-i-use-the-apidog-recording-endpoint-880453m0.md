# How do I use the Apidog recording endpoint?

Apidog does not currently support the automatic recording endpoint function, but there are plans to support it in the future.

There is currently a quick way to record: open DevTools while testing the endpoint in the browser. Once the operation is complete, right-click to export the HAR file and import it into Apidog with one click. Create a test scenario and import all requests to play back at any time. For details about how to import a HAR file, see Importing a `.har` file.
