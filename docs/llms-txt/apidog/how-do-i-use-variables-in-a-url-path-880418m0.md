# How do I use variables in a URL path?

In Apidog, the variable part of a path is usually defined as a path parameter. For example, if you type `{petId}`, you can see the Path parameter below.
At this point, you can define the parameter value as a reference global/environment variable value, for example, `{{id}}`. When you send a request, the path in the actual request is replaced with the actual value of the variable id.
