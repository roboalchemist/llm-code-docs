## Platform Availability

- **Desktop version**: Full keyboard shortcut support across all modules
- **Web version**: Limited support, primarily for core editing and request functions
- Some shortcuts (like tab management and window controls) are only available in the desktop version due to browser limitations

:::tip
You can customize many of these shortcuts in **Settings** → **Keyboard Shortcuts** to match your preferred workflow.
:::
