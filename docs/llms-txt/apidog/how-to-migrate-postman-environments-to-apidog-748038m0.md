# How to migrate Postman environments to Apidog?

1. Open Postman and go to the "Environments" tab.

<Background>
![image](https://assets.apidog.com/blog-learn/2023/07/image-23.png)
</Background>

2. Select the environment you want to export from the list. Click on the "..." button next to the environment name, and choosing the "Export" option.

<Background>
![image](https://assets.apidog.com/blog-learn/2023/07/image-22.png)
</Background>

This will generate a JSON file containing the environment variables and their values. Select the location where you want to save the exported file.

3. Go to the "Environment Management" section in the top right corner of the Apidog interface and select "Import Postman Environment.

<Background>
![image](https://assets.apidog.com/blog/2023/07/image-52.png)
</Background>

4. Simply select the downloaded Postman Environment file and upload it to Apidog. This will allow you to seamlessly import your Postman environment into Apidog.

<Background>
![image](https://assets.apidog.com/blog/2023/07/image-51.png)
</Background>
