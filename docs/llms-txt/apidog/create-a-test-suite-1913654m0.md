# Create A Test Suite

> Apidog version 2.8.0 or later is required.

You can quickly create a new test suite in the folder tree.

## Steps

<Steps>
  <Step>
  Upon opening Apidog, navigate to the `Tests` module, and then find `Test Suite`.
      
<Background>
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/369678/image-preview)
</Background>

  </Step>
  <Step>
  Click the `+ New` button (or click the `...` menu next to the folder and select `Create Test Suite`).
  </Step>
    
  <Step>
  Fill in the test suite name in the pop-up and set basic information, such as priority.
      
<Background>
<img src="https://api.apidog.com/api/v1/projects/544525/resources/369679/image-preview" style="width: 460px" />
</Background>

  </Step>
    <Step>
    Click `Continue` to create successfully and enter the test suite design page.
        
<Background>
![image.png](https://api.apidog.com/api/v1/projects/544525/resources/369680/image-preview)
</Background>

  </Step>
</Steps>
