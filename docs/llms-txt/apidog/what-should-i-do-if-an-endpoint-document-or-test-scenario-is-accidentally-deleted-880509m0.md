# What should I do if an endpoint, document, or test scenario is accidentally deleted?

Apidog supports the recycle bin function, which allows you to easily restore accidentally deleted APIs, floders, schemas, test scenarios, and other resources.

1. Click on the Recycle Bin icon in the bottom right corner of the Apidog page.

2. Select the resources to be restored (multiple selections are supported)

3. Just click on the "Recover" button.

In addition, if you mistakenly change the endpoint, document, or schemas, you can revert to the previous version through history:

1. Open the endpoint that needs to be restored.

2. Click the "History" icon in the top right corner.

3. Select the previous version you want to restore and click "Restore this version".

Note: Restoring from the Recycle Bin restores the original version, while restoring from the change history creates a new version.

For more information, see [Data backup](https://docs.apidog.com/backing-up-data-640829m0.md)
