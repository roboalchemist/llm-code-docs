# How to Set the Server Host for the General Runner in Apidog Web Version?

Due to browser security restrictions, the Web version cannot access insecure HTTP from an HTTPS page. As a result, setting the server host for the General Runner may fail in the Web version. 

Please configure it in the [Apidog client](https://docs.apidog.com/644408m0.md) instead.
