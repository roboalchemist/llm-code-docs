# Data Storage and Security

Apidog provides comprehensive data storage and security measures to protect your information across different deployment options.

## Data Storage by Service Type

| Service | Storage Location | Infrastructure | Data Control |
|---------|------------------|----------------|--------------|
| **Apidog SaaS Global** | United States | AWS (Amazon Web Services) | Apidog-managed |
| **Apidog SaaS Europe** | Ireland | AWS Ireland | Apidog-managed |
| **Apidog On-Premises** | Your servers | Self-hosted | Fully customer-controlled |


## Data Security Measures

Apidog employs various security measures to safeguard your data across all deployment options.

For detailed information, see our [Security Measures](https://legal.apidog.com/security-measures-645653m0).
