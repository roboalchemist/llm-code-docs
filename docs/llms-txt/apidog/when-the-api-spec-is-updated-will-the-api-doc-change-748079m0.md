# When the API spec is updated, will the API doc change?

Apidog's documentation content is updated in real-time, always reflecting changes in the API spec.

If you don't want the API doc to change in real-time, you can use the Sprint branch feature. By modifying the API definition in a branch, changes will only affect the documentation when merged into main.
