# API Hub

Apidog provides an API Hub featuring thousands of mainstream APIs from popular services and platforms. The API Hub allows you to browse API documentation, test endpoints directly, and clone APIs into your own projects for customization and integration.

In the API Hub, you can explore available APIs and interact with them in several ways:

- **Test Directly**: Click **Open in Apidog** to access the API's editing and debugging interface

## Contributing to API Hub

When you publish API documentation, you have the option to contribute it to the API Hub, making it available to the broader Apidog community.

To contribute your API:

1. Navigate to **Share Docs** → **Publish**
2. When clicking **Publish**, enable **Publish to API Hub**
3. The Apidog operations team will review your submission and decide whether to include it in the API Hub
