# Phone

Module to generate phone-related data.

## Module Overview

For a phone number, use [Number](https://docs.apidog.com/number-649453m0.md) Many locales provide country-specific formats.

---

## imei

Generates IMEI number.

**Returns**: string

**Examples**

```js
{{$phone.imei}}  // '73-603797-653587-5'
```
---

## number

Generates a random phone number.

**Parameters**

| Name | Type | Default | Description |
| --- | --- |--- | --- |
| style | 'human' \| 'national' \| 'international'| `human` |Style of the generated phone number:- `'human'`: (default) A human-input phone number, e.g. `555-770-7727` or `555.770.7727 x1234`- `'national'`: A phone number in a standardized national format, e.g. `(555) 123-4567`.- `'international'`: A phone number in the E.123 international format, e.g. `+15551234567`|

**Returns**: string

**Examples**

```js
{{$phone.number}} // '1-203-635-9321 x1495'

{{$phone.number(style='human')}} // '(343) 464-9991'

{{$phone.number(style='national')}}  // '(549) 994-3486'

{{$phone.number(style='international')}}  // '+14642309454'
```
