# How to send a SOAP/WebService request in Apidog?

Web services often utilize the Simple Object Access Protocol (SOAP). SOAP is an XML-based communication protocol that enables seamless interaction between diverse platforms and programming languages. 

For comprehensive examples and further details, please consult the [WebService documentation](https://webservice.apidog.io/).
