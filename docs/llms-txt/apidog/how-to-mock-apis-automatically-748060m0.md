# How to mock APIs automatically?

<Steps>
  <Step>
    Specify an endpoint or import an API spec; this endpoint needs to have a specified response.
  </Step>
  <Step>
   Copy the mock URL. 
      In `DESIGN` mode, it's in the "API" tab; 
<p style="text-align: center">
    <img src="https://api.apidog.com/api/v1/projects/544525/resources/343578/image-preview" style="width: 640px" />
</p>
      in `DEBUG` mode, it's in the "Mock" tab.
<p style="text-align: center">
    <img src="https://api.apidog.com/api/v1/projects/544525/resources/343579/image-preview" style="width: 640px" />
</p>
  </Step>
  <Step>
    Access this URL to obtain mock data. Refreshing will update the data.
<p style="text-align: center">
    <img src="https://api.apidog.com/api/v1/projects/544525/resources/343580/image-preview" style="width: 440px" />
</p>
  </Step>
</Steps>
