# Why Can't I Open Apidog After a Windows System Update?

1. Uninstall the software
- Go to Control Panel > Programs and Features, Locate the software and uninstall it.
- Note: Back up any important configuration files or personal data before uninstalling.
2. Reinstall Apidog.
- Download the latest installer from the official website -[Download & Register](https://docs.apidog.com/644408m0.md)
- During installation: Choose your preferred directory & select **Install for all users** (if available) to ensure proper permissions.

This should resolve the issue by resetting the installation with correct permissions. If the issue exists, check for additional system updates or contact here- .[Apidog Support Center](https://docs.apidog.com/apidog-support-center-748035m0.md)
