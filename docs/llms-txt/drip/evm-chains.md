# EVM Chains

> Activate ERC-721 NFTs, ERC-1155 NFTs, and ERC-20 Tokens on EVM Chains

### Add an Activation

After selecting an EVM Chain from the dropdown in the [`🔥 Manage Activations`](/admin-settings/admin-dashboard/admin-settings/web3-settings/manage-activations) menu, a new embed will appear:

<Frame caption="Ethereum Collection Activations Menu">

</Frame>

Select the type of NFT/Token you would like to add from the dropdown menu

### Remove an Activation

To remove a collection, go to the **Collection Setting Menu** and click the `🗑️ Remove A Collection` option.

<Frame caption="Select Remove an Activation">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_fcc30a.jpg?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=05ca4576114d50acbaefc3dfb2149822" width="991" height="870" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_fcc30a.jpg" />
</Frame>

Now select the collection you would like to remove from the dropdown menu

<Frame caption="Select the collection you want to remove">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_01ddb4.jpg?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=4c5a7f71718d9567a6bb04f80a02adc4" width="816" height="166" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_01ddb4.jpg" />
</Frame>

A new embed will appear, click on `🗑 Confirm Removal` \*\*\*\*to remove the collection. \*\*This cannot be undone\*\*!

<Frame caption="Click on Confirm Removal">
