# Overview

> The embed is the main body of the message

### Get Started

To edit the embed content, use the command `/create type:✨ Magic Embed` and select the `📝Edit Embed` option.

<Frame caption="Edit Embed Select Menu Magic Embed">
  <img src="https://mintcdn.com/driprewards/PdG1Mk4gy7bbF954/images/drip-features/magic-embed-builder/img_ccf319.png?fit=max&auto=format&n=PdG1Mk4gy7bbF954&q=85&s=769ef564a899bf51af4f0e306488aff7" width="798" height="308" data-path="images/drip-features/magic-embed-builder/img_ccf319.png" />
</Frame>

Now you see all the possible customizations, each of which is explained in [#embed-customizations-options](/drip-features/magic-embed-builder/embeds#embed-customizations-options)

## ♻️ Delete Embed

This option allows you to remove the embed from the message.

<Frame caption="Delete Embed Content">
  <img src="https://mintcdn.com/driprewards/PdG1Mk4gy7bbF954/images/drip-features/magic-embed-builder/img_fe093e.gif?fit=max&auto=format&n=PdG1Mk4gy7bbF954&q=85&s=67a3ccff188f48247649b4e08a8be0cb" width="510" height="461" data-path="images/drip-features/magic-embed-builder/img_fe093e.gif" />
</Frame>

Once you click the option, the embed is removed from the message.

<Frame caption="Updated Message">
  <img src="https://mintcdn.com/driprewards/PdG1Mk4gy7bbF954/images/drip-features/magic-embed-builder/img_fd302b.png?fit=max&auto=format&n=PdG1Mk4gy7bbF954&q=85&s=04bd82cf8f9426813dbf99444996eba8" width="934" height="278" data-path="images/drip-features/magic-embed-builder/img_fd302b.png" />
</Frame>

<Warning>
  Removing the embed is irreversible, all modifications will be lost
</Warning>

<Info>
  After you remove the embed, a new option is displayed to add a new embed. This will create the default embed.

    <img src="https://mintcdn.com/driprewards/PdG1Mk4gy7bbF954/images/drip-features/magic-embed-builder/img_7cec6d.png?fit=max&auto=format&n=PdG1Mk4gy7bbF954&q=85&s=06c37a3cb125a520fdbf12cec1dd76f0" alt="" width="398" height="114" data-path="images/drip-features/magic-embed-builder/img_7cec6d.png" />
</Info>
