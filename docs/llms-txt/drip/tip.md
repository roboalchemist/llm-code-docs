# Tip

> Tip your fellow members with your Points

To use this command type `/tip user: name amount: 999 note: Note` in the message box where U**ser** is the user you want to tip, **Amount** is the number of Points you want to tip and **Note** is an optional message you want to add.

<Frame caption="Write The Tip Command">
  <img src="https://mintcdn.com/driprewards/_LyuiJDsEr1kmn5T/images/drip-features/slash-commands/img_4d4551.png?fit=max&auto=format&n=_LyuiJDsEr1kmn5T&q=85&s=1d6b948f8e6a07ede2796a1c8592912a" width="1014" height="92" data-path="images/drip-features/slash-commands/img_4d4551.png" />

Once you have entered the command, click enter and you will see the following message:

<Frame caption="Click Confirm To Send The Tip">
  <img src="https://mintcdn.com/driprewards/_LyuiJDsEr1kmn5T/images/drip-features/slash-commands/img_a75af3.png?fit=max&auto=format&n=_LyuiJDsEr1kmn5T&q=85&s=1ff4c88b28f2f875dcbfcc11584e463b" width="540" height="358" data-path="images/drip-features/slash-commands/img_a75af3.png" />
</Frame>

Click the `✔️ Confirm` button to confirm and send your tip. Now your tip will appear in the channel:

<Frame caption="Success Message">
  <img src="https://mintcdn.com/driprewards/_LyuiJDsEr1kmn5T/images/drip-features/slash-commands/img_97d15c.png?fit=max&auto=format&n=_LyuiJDsEr1kmn5T&q=85&s=6c66786e37dbe76301a4470ead769b3d" width="758" height="426" data-path="images/drip-features/slash-commands/img_97d15c.png" />
</Frame>
