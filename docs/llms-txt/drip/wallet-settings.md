### Adding a new wallet

You can add up to 10 wallets. To add a wallet use the green `➕` button.

<Frame caption="Add a Wallet">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_708793.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=6c51d6a35bd987a5a227afdc1ef39baa" width="988" height="1058" data-path="images/drip-features/user-dashboard/user-settings/img_708793.png" />
</Frame>

After using the button, you will get 2 options to add a wallet.

<Frame caption="Link Wallet">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_24d0b1.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=d91dd8b73811aeb6d9db48f2476aa505" width="507" height="422" data-path="images/drip-features/user-dashboard/user-settings/img_24d0b1.png" />
</Frame>

* Link wallet using a traditional wallet connect via a message signature

* Link wallet using an Marketplace bio verification system **(most secure)**

### Traditional Wallet Linking

If you choose the Traditional `Link Wallet` option, the following message is displayed:

<Frame caption="Link Wallet Using Connect Page Message">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_cea243.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=e45be08cceb9fa5060ec651840011e86" width="528" height="318" data-path="images/drip-features/user-dashboard/user-settings/img_cea243.png" />
</Frame>

Use the Link Wallet button to go to the Wallet Connect page:

<Frame caption="Wallet Connect Page">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_983669.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=7e4d5a28a582dd578d92598ac87063ee" width="2304" height="1170" data-path="images/drip-features/user-dashboard/user-settings/img_983669.png" />
</Frame>

Click the Link Wallet button and choose the right wallet option.

Then, return to this page and use the `Link Wallet` button again. Check if the message matches the message from Discord. Sign only if it does.

Now your wallet is connected to your Discord account.

### Marketplace Bio Verification

Marketplace `Bio Verification` is the most secure linking option available because the signing happens on a website many people trust. For this example we will use OpenSea. If you choose this method, you will get the following window where you must enter your Marketplace username OR wallet address:

<Frame caption="Opensea Bio Verification">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_1fb548.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=e624115d9e67b056917cf74934bdf5d6" width="874" height="566" data-path="images/drip-features/user-dashboard/user-settings/img_1fb548.png" />
</Frame>

Once you click **Submit**, you will be given a secret code to paste into your Marketplace bio.

<Frame caption="Marketplace Bio Verification Code">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_4cd105.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=2d22303bfdc92b11552544ab222bdaa0" width="520" height="290" data-path="images/drip-features/user-dashboard/user-settings/img_4cd105.png" />
</Frame>

Now go to your OpenSea profile and click the 3 dots on the right side of your screen.

<Frame caption="Opensea Settings">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_2d07b1.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=09656cb0d724eed614dddb56d89e4427" width="1903" height="916" data-path="images/drip-features/user-dashboard/user-settings/img_2d07b1.png" />

Paste the code into the bio text field and click **save**.

<Frame caption="Opensea Settings Bio">

Now go back to your discord and click the ✅**Verify** button. A message similar to the one below should be displayed.

### Unlinking a Wallet

To unlink a wallet, use the "Select the wallet you want to use" dropdown menu to choose the wallet you want to unlink.

<Frame caption="Wallet Connect Options">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_cd0080.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=109fa5f851853506c286653b8d9430a0" width="431" height="96" data-path="images/drip-features/user-dashboard/user-settings/img_cd0080.png" />
</Frame>

You will see that your message has been edited and displays the wallet you selected:

<Frame caption="Select Wallet Message">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_81802e.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=dd749d5d5281b3bb26996d64f99a8e9c" width="444" height="137" data-path="images/drip-features/user-dashboard/user-settings/img_81802e.png" />
</Frame>

Click the ❌ button to unlink the selected wallet from your account.

### Setting Primary Wallet

The primary wallet is the wallet that is visible to server admins when they need to send out a prize or a purchased store item.

To set a wallet as your primary wallet, use the "Select the wallet you want to use" dropdown menu to choose the wallet you want to set as your primary wallet.

<Frame caption="Wallet Settings Options">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_85aa53.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=5f8d422cc34aa593622a97c4c8f674b2" width="431" height="96" data-path="images/drip-features/user-dashboard/user-settings/img_85aa53.png" />
</Frame>

You will see that your message has been edited and displays the wallet you selected:

<Frame caption="Select Wallet Message">
  <img src="https://mintcdn.com/driprewards/93FznksNZY37vUgd/images/drip-features/user-dashboard/user-settings/img_d818e3.png?fit=max&auto=format&n=93FznksNZY37vUgd&q=85&s=ba44c059f7afeeeeedcce476c677421a" width="444" height="137" data-path="images/drip-features/user-dashboard/user-settings/img_d818e3.png" />
</Frame>

Click the ⭐ button to set the selected wallet as your primary wallet.

<Info>
  If a member has only 1 wallet, this will automatically be their primary
  wallet.
</Info>
