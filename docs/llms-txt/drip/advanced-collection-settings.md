# Advanced Collection Settings

> There are various Advanced Settings available for Generator and Multiplier Collections

Advanced Collection Settings can be accessed via the [`🔥 Manage Activations`](/admin-settings/admin-dashboard/admin-settings/web3-settings/manage-activations) Menu

<Tabs>
  <Tab title="Attribute Override (Generators & Multipliers)">
    Generators and multipliers can be edited in even more detail. You can have generators with a particular attribute generate more or fewer tokens than others. Or you can make multipliers with a specific attribute have a greater effect on your generators.

    1. Click `📝 Attribute Override`

    <Frame caption="Click on Attribute Override">
      <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_fec1ce.gif?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=029b339d62ff6c25a89b53ee3de5f119" width="397" height="312" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_fec1ce.gif" />
    </Frame>

    2. A new embed will appear, click on `📝 Add Property`

    <Frame caption="Click on Add Property">
      <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_01ab24.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=83813bc5075fdd07863c17a37eddefb6" width="417" height="358" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_01ab24.png" />
    </Frame>

    3. A new embed will appear, choose the property you want to edit from the dropdown menu.

    <Frame caption="Select a property to edit">
      <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_5b69a0.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=bafcf40820239730c9448bdbae336b97" width="507" height="323" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_5b69a0.png" />
    </Frame>

    4. Choose an attribute to edit from the dropdown menu

    <Frame caption="Select an attribute to edit">
      <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_5a0a57.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=fba124bef1a524028d897850dcd03b8f" width="493" height="398" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_5a0a57.png" />
    </Frame>

    5. A new form will appear, enter the new value for the attribute override

    <Frame caption="Edit Attribute Value">
      <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_cb07b3.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=edb65fcbc5439ec558247f8c31b234bc" width="874" height="552" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_cb07b3.png" />
    </Frame>

    The attribute override will now appear in the Advanced Attribute Settings Menu

    <Frame caption="New Attribute Override">
      <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_41b3af.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=72aeda676185fb6177c4e05a3ef1c8ac" width="497" height="415" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_41b3af.png" />
    </Frame>
  </Tab>

  <Tab title="Set Max Balance (Generators)">
    To set a maximum balance that an NFT can generate, click on the `📝 Set Max Balance` button

    <Frame caption="Click on Set Max Balance">
      <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_a6de75.jpg?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=111f5b53ab3246e62ebb7d6954a9da4e" width="1010" height="840" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_a6de75.jpg" />
    </Frame>

    A new form will appear, enter the new collection max balance

    <Frame caption="Enter a new max balance">
      <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_65593f.jpg?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=2c5b0f53e34c28a96ebd477f2974fd50" width="866" height="548" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_65593f.jpg" />
    </Frame>
  </Tab>

  <Tab title="Re-Sync">
    This allows you to update collections in case of changes to metadata, collection size, etc. To do so, click on the `🔄 Re-Sync` button

    <Frame caption="Click on Re-Sync">
      <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_43eb97.jpg?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=53b37c525c43a64688a03c8d2955536f" width="1005" height="830" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_43eb97.jpg" />
    </Frame>

    A new embed will appear for you to confirm a Re-Sync. You can do so by clicking the `🔄 Re-Sync` button again. This can only be done once a day.

    <Frame caption="Confirm a Re-Sync">
      <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_aa9df4.jpg?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=7e2a766b0238e28b32523db8d5e5477f" width="986" height="612" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_aa9df4.jpg" />
    </Frame>
  </Tab>
</Tabs>
