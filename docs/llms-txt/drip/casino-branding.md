# Casino Branding

> Configure the casino branding for your server

<Warning>This feature is only available for Custom Plans</Warning>

After selecting `🃏``Casino Branding` \*\*\*\*from the dropdown in [Game Settings,](/admin-settings/admin-dashboard/admin-settings/game-settings) you will see one option. Choose this option from the dropdown menu to edit:

* 📷 **Casino Images:** Configure the casino images for your server

  <img src="https://mintcdn.com/driprewards/C8C-yxJTuKgUzgAP/images/admin-settings/admin-dashboard/admin-settings/game-settings/img_38ef66.jpg?fit=max&auto=format&n=C8C-yxJTuKgUzgAP&q=85&s=f5fe5d7cc2038bd2f3d353f691ac2cdd" width="986" height="532" data-path="images/admin-settings/admin-dashboard/admin-settings/game-settings/img_38ef66.jpg" />

A new embed will appear and should look something like this unless the media has already been changed:

<Frame caption="Edit Casino Images">
  <img src="https://mintcdn.com/driprewards/C8C-yxJTuKgUzgAP/images/admin-settings/admin-dashboard/admin-settings/game-settings/img_381311.png?fit=max&auto=format&n=C8C-yxJTuKgUzgAP&q=85&s=c9f324a09b4a02d135d52ae46ff24086" width="792" height="1164" data-path="images/admin-settings/admin-dashboard/admin-settings/game-settings/img_381311.png" />
</Frame>

Drip Rewards comes with its own Casino branding but you can change the various media by clicking the blue arrows at the bottom of the embed and then selecting `Edit Image.` A form will appear for you to edit the URL for the media you would like to use

<Frame caption="Enter an Image URL">
  <img src="https://docs.drip.re/~gitbook/image?url=https%3A%2F%2F2728597434-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FxUaE8LDXTeS85PkKxJJa%252Fuploads%252FUzDr5lEDILyCiYLbk8Nu%252F%2523%25F0%259F%259B%25A0%25E2%2594%2586admin%2520Drip%2520Documentation%2520-%2520Discord%25202024-02-16%2520at%252012.47.07%2520PM.jpg%3Falt%3Dmedia%26token%3D980e8d73-30a5-491e-8810-24d2659b9859&width=768&dpr=4&quality=100&sign=14f8d44&sv=1" />

After clicking submit, your new media should appear! You can reset the media by clicking the green `Reset` button
