# Prizes

> Create raffles for your members to spend tokens on!

### Creating a Prize

To create a Prize, use the command `/create` and select `🎁 Prize`.

  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_b9cae8.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=28537f27abc95745c153ca1ad7df9324" width="564" height="154" data-path="images/drip-features/img_b9cae8.png" />

#### Edit Giveaway

This option allows you to edit the embed with the information of your Giveaway. You can edit the following:

* Giveaway Title

* Giveaway Description

* Giveaway Image

* Giveaway Thumbnail

* More Info Link

<Frame caption="Edit Prize Form">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_304abe.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=b76673845fa297231a8b2b39c48ae95d" width="878" height="1296" data-path="images/drip-features/img_304abe.png" />

### Edit Settings

Here you can configure the settings of your item:

* Entry Limit in Points

* Min entry in Points *(Leave blank if no minimum)*

#### Edit End Time

Here you can set the end time of your giveaway. There are some preset options you can use or you can set a custom time.

<Frame caption="End Time Options Prize">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_61767a.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=2665eb359c99072eda786c0eed1959db" width="413" height="252" data-path="images/drip-features/img_61767a.png" />
</Frame>

If you would like to set a custom end date, there are 2 options:

<Tabs>
  <Tab title="Relative Time Format">
    To set a custom end date by using the **Relative Time Format**, scroll down in the select menu and click the option `⚙️Custom`.

    Once you click the option, enter the time in the correct format and click `Submit`.

    Time format options:

    * 'y' for years
    * 'M' for months (Capital)
    * 'w' for weeks
    * 'd' for days
    * 'h' for hours
    * 'm' for minutes (Lowercase)

    <Frame caption="Relative Time Format">
      <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_21bf89.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=984ac8175ffa46c605f59dc3faa33c7a" width="868" height="558" data-path="images/drip-features/img_21bf89.png" />
    </Frame>
  </Tab>

  <Tab title="Custom Timestamp">
    <Frame caption="Custom Timestamp Option">
      <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_9f4ba7.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=7a48395d65e79c85a3db0346b8d6da1a" width="413" height="252" data-path="images/drip-features/img_9f4ba7.png" />
    </Frame>

    A custom timestamp gives you absolute control over when your giveaway ends down to the second.

    **How to get the timestamp?**

    Go to [this website](https://timestampgenerator.com/), and select the date and time you want your giveaway to end. Then, click `Submit`. In `Common Date Formats`, copy the `timestamp`. Now click `Enter Timestamp` below and paste it into the form.

    <Frame caption="Timestamp Generator">
      <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_45514f.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=225777597531a617ae6df2a54b62ccb3" width="866" height="554" data-path="images/drip-features/img_45514f.png" />
    </Frame>
  </Tab>
</Tabs>

### Copy Giveaway

To copy an already existing Giveaway, copy the message ID from the Giveaway you want to duplicate.

<Frame caption="Copy Message ID Prize">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_bed451.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=bccba6dbe3060d83eb7882d881915af3" width="944" height="1560" data-path="images/drip-features/img_bed451.png" />
</Frame>

<Info>
  If you don't see the option to copy the message ID, enable Developer Mode
  under Settings -> Advanced -> Developer Mode
</Info>

Now use the `/create` command and select `🎁 Prize`. Click the `💾Copy Existing` button and paste the ID from the message you copied earlier.

<Frame caption="Paste Prize ID Form">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_0b3f4d.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=2454674a1d77538c67ab0ed456187f4d" width="870" height="562" data-path="images/drip-features/img_0b3f4d.png" />
</Frame>

Now you see that the entire configuration of the Giveaway is copied.

### Edit Giveaway

To edit an already existing Store Item: **right-click the message** -> **Apps** -> **Edit**

<Frame caption="Edit Prize Button">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_afd8b0.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=68c2f77afc8f3286a62e7d52327e9e35" width="1374" height="1458" data-path="images/drip-features/img_afd8b0.png" />
</Frame>

When clicking **Edit**, you will now be able to edit the message. For details, follow the steps in [Creating a Prize](/drip-features/prizes#creating-a-prize).

To enter a giveaway, click `🎁 Enter` -> Amount of Points you want to enter

### Draw Prize

A prize can only be drawn by a **prize master** or **administrator.** To do this go to the Prize, click `🎉Draw Prize.`

<Frame caption="Click &#x22;Draw Prize&#x22; to pick a winner">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_b765b9.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=2a6a4bc3018157781bd2a2eb3ec741f7" width="996" height="580" data-path="images/drip-features/img_b765b9.png" />
</Frame>

Once you have confirmed, the bot will choose a winner from the members who have entered. Each entry is a chance to win, the more they enter the greater their chance of winning.

<Frame caption="Draw a Prize">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_5f6615.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=a5b414ff676c90baee6c4f8f074b80f1" width="1046" height="882" data-path="images/drip-features/img_5f6615.png" />
</Frame>

Once a winner is chosen, the message will be updated.

<Frame caption="Winner Chosen">
  <img src="https://mintcdn.com/driprewards/Xh33JGW0jVY0ds3p/images/drip-features/img_47737c.png?fit=max&auto=format&n=Xh33JGW0jVY0ds3p&q=85&s=7da52e59b18d7f4d3feb9dff35928d91" width="1220" height="1118" data-path="images/drip-features/img_47737c.png" />
</Frame>

<Info>
  A prize can always be redrawn by following the exact same steps from above.
</Info>
