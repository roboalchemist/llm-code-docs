# Meme Maker

> Set up the meme maker for your collection(s)

## Set Up a Meme

To create a new meme, add the collection you want to create a new meme from. To do this, click the `⚡ Add a New Collection` option.

<Frame caption="Click Add a New Collection">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_c2dec8.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=76fc634431519612859da6d491a903ed" width="998" height="616" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_c2dec8.png" />
</Frame>

Select the collection you want to use next.

<Frame caption="Choose the Correct Collection">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_8d3cc3.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=c3a8caac8590817f37237442fc94333e" width="998" height="558" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_8d3cc3.png" />
</Frame>

Now click the `Set Attribute` button and select the attribute you want to use to base your memes on.

<Frame caption="Click Set Attribute">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_3072bc.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=fd2ca379e675c947af6d0b4fd9f049ee" width="1000" height="700" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_3072bc.png" />
</Frame>

<Frame caption="Choose the Attribute">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_cc6c2e.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=e32b75759bd2871b901270c8a7805479" width="1002" height="964" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_cc6c2e.png" />
</Frame>

After picking the correct attribute, you can now create a new meme. Do this by clicking `Create a new meme` from the dropdown menu at the bottom of the embed. A pop-up will appear asking you to enter the name of the new meme.

<Frame caption="Click Create a new meme">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ea5b4e.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=571ca77a0a235626aacedb10c13b7fb9" width="792" height="160" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ea5b4e.png" />
</Frame>

<Frame caption="Fill In the Name Of the Meme">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_34b10a.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=dffa6e2cf1407fd83973f30e9f62e5e5" width="870" height="560" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_34b10a.png" />
</Frame>

Then choose a property you want to set and fill in the URL to the overlay.

<Frame caption="Choose a Trait">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_a9922c.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=1245525da38d94d958fa423a2aec67e1" width="794" height="640" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_a9922c.png" />
</Frame>

<Frame caption="Fill In the URL To the Overlay">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_99f183.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=344f069ff0b9d4e226b7c3bedb80e084" width="874" height="562" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_99f183.png" />
</Frame>

<Info>
  The overlay should be a PNG with all the right dimensions to fit the NFT image
</Info>

<Warning>
  We suggest using an external image host, like[
  https://imgtr.ee/](https://imgtr.ee/) or
  [https://imgur.com](https://imgur.com) for hosting images
</Warning>

## Remove a Meme

To remove a meme, click the `🗑️ Remove` button.

<Frame caption="Click Remove Button">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_30c496.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=b49546974114451e63779e35f33b4adc" width="822" height="264" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_30c496.png" />
</Frame>

To be safe, another confirmation will be shown if you still want to delete the meme. Then click the `Confirm` button.

<Frame caption="Click Confirm">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ea5957.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=9917d3c21e49137b243b6fa404cf922e" width="996" height="680" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ea5957.png" />
</Frame>

## Send Meme Maker

To send the **Meme Maker** message, use the `📤 Send Meme Maker` button.

<Frame caption="Click the Send Meme Maker button">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ab634e.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=de884c1af2ea87efcd161bbd17cf6a44" width="818" height="262" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_ab634e.png" />
</Frame>

This opens a new drop-down menu where you must choose the channel to which the message should be sent.

<Frame caption="Choose a Channel You Want To Send The Meme Maker To">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_d20871.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=72628c3d06ed04410632a31230731017" width="796" height="532" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_d20871.png" />
</Frame>

## Use the Meme Maker

To use the meme maker, use the `Create Meme` button.

<Frame caption="Click the Create meme button">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_1defdf.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=5fec0d41aed75786394c81c1f243d3f5" width="858" height="426" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_1defdf.png" />
</Frame>

If multiple collections are activated, you will see multiple options. Click on the collection you want to use.

<Frame caption="Click the Collection You Want To Use">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_9e20d7.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=bbb3098fc86c89dc3dcf07a3328e4561" width="409" height="186" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_9e20d7.png" />
</Frame>

Now choose the NFT for which you want to create a meme.

<Frame caption="Choose the NFT You Want To Use">
  <img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_53dbf1.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=38024c284100897c16fde99b22df266f" width="794" height="388" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_53dbf1.png" />
</Frame>

As a final step, select the meme you want to create.

<Frame caption="Choose the Meme">
  <img src="https://mintcdn.com/driprewards/kUGFp_25nD2Z7HmN/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_8e800d.png?fit=max&auto=format&n=kUGFp_25nD2Z7HmN&q=85&s=6d1267e1142cdee8b7a543edc9378e87" width="926" height="548" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_8e800d.png" />
</Frame>

After a few seconds, your meme is ready to use

<img src="https://mintcdn.com/driprewards/fZiObPJYfg75-m3M/images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_102024.png?fit=max&auto=format&n=fZiObPJYfg75-m3M&q=85&s=305f25200685bb75ff8e1348979e6d26" alt="" width="834" height="806" data-path="images/admin-settings/admin-dashboard/admin-settings/web3-settings/img_102024.png" />
