# Check Alert History

You can see the alert history to see if there are any alerts triggered in the past. To see the alert history, do the following:

1. From the left navigation bar, choose **Alert Management**, then select **Alert History** from the secondary menu.

2. In the alert history table, search for the alert you want to check.

   <!-- -->

   1. You can know if an alert message is sent successfully
   2. You can know the alert severity, used channel

3. Click on the **Detail** button to see the specific alert message request and response.
