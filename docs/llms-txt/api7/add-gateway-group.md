# Add a Gateway Group

This tutorial walks you through adding a new [gateway group](https://docs.api7.ai/enterprise/3.3.x/key-concepts/gateway-groups.md).

## Prerequisites[â](#prerequisites "Direct link to Prerequisites")

1. [Install API7 Enterprise](https://docs.api7.ai/enterprise/3.3.x/getting-started/install-api7-ee.md).

## Steps[â](#steps "Direct link to Steps")

1. Select **Gateway Groups** from the side navigation bar and then click **Add Gateway Group**.
2. In the dialog box, do the following:

* In the **Type** field, choose `API7 Gateway`.
* In the **Name** field, enter `Production`.
* Click **Add**.

Continue to the [add a gateway instance](https://docs.api7.ai/enterprise/3.3.x/getting-started/add-gateway-instance.md) tutorial to use the newly created gateway group.
