# Variables

API7 Cloud supports configuring variables in some cases, like a consistent hash key and log format. Variables are used as interpolation and will be replaced with the actual value with the API request runtime.

API7 Cloud supports variables provided by [NGINX](https://nginx.org/) and [Apache APISIX](https://apisix.apache.org/).

Refer to the [built-in variables](https://docs.api7.ai/apisix/reference/built-in-variables.md) for more information.
