# Selecting and Navigating Groups

## Double-click

Click on a group on the [Stage](/editor/interface-overview/stage) to select it. To select an object in a group, double-click on the object you want to select. This takes you down one level in the hierarchy and allows you to select any object on that level.

<img src="https://mintcdn.com/rive/jA1HsvaDbvTOHjR6/images/editor/fundamentals/DoubleClick1.gif?s=aeb6f53b9a9b2235a38746c72d2c74c0" alt="Double Click1 Gi" width="800" height="459" data-path="images/editor/fundamentals/DoubleClick1.gif" />

You can continue to double-click on other nested groups and shapes to navigate down the [Hierarchy](/editor/interface-overview/hierarchy).

<img src="https://mintcdn.com/rive/jA1HsvaDbvTOHjR6/images/editor/fundamentals/DoubleClick2.gif?s=02e260f2fd260305376e9d87da48a675" alt="Double Click2 Gi" width="800" height="459" data-path="images/editor/fundamentals/DoubleClick2.gif" />

## Enter and Esc shortcuts

Use the `Enter` key to navigate down a level in the Hierarchy to the selected object's child.

<img src="https://mintcdn.com/rive/jA1HsvaDbvTOHjR6/images/editor/fundamentals/EnterAndEscape.gif?s=1c2f4e880532c768a8c848b7817d49a8" alt="Enter And Escape Gi" width="800" height="459" data-path="images/editor/fundamentals/EnterAndEscape.gif" />

Use the `Esc` key to quickly navigate up the Hierarchy to select the object's parent.

## Deep Select

Hold `⌘` in macOS or `Ctrl` in Windows, click directly on a shape to select it, no matter where you are in the Hierarchy. This allows you to click through all groups and directly select a shape.

<img src="https://mintcdn.com/rive/jA1HsvaDbvTOHjR6/images/editor/fundamentals/DeepSelect.gif?s=062a5ba7607d988971e45f6759560406" alt="Deep Select Gi" width="800" height="459" data-path="images/editor/fundamentals/DeepSelect.gif" />
