# Getting Started

> Add Rive to your game project.

Rive supports popular game engines and is easy to integrate with custom engines. New to Rive? Start with the [Rive Editor](/editor/fundamentals/overview) to create your graphics, then [export for runtime](/editor/exporting/exporting-for-runtime) when you're ready to bring them into your game.
