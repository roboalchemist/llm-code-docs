# Denoise VM Results by Configuring Your Environments

Aikido helps you manage virtual machines by denoising results and reducing scanning frequency for non-production environments.

> If the purpose of your connected cloud is set to **"mixed,"** you need to configure VM environments manually.

### How to configure your VM environments <a href="#how-to-configure-your-vm-environments" id="how-to-configure-your-vm-environments"></a>

**Step 1:** Navigate to the [**Virtual Machines** page](https://app.aikido.dev/virtual-machines)

**Step 2:** Click the **triple dots** menu for the VM you want to configure, and select **Edit Configuration**.

![Critical vulnerability detected in SSH jump host with scan and configuration options.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-20b519b8ac07145c9fb0a3ac3e7ba9c5767379d6%2Fdenoise-vm-results-by-configuring-your-environments_bba664dc-0056-48f4-8231-fc38babba788.png?alt=media)

**Step 3:** Adjust the **Environment** to match the desired environment type (e.g., staging, development, production).

![Virtual machine configuration editing for SSH database access in a staging environment.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-7cbb3dafbfd6efff72357b1372705dc837a3b109%2Fdenoise-vm-results-by-configuring-your-environments_18e1e6ed-36c1-41b6-b6a9-5e228ad8268f.png?alt=media)

**Step 4:** Save your changes to apply the new environment settings. The environment configuration label will be shown on the overview.
