# CLI

The Acceldata ADOC CLI (Command Line Interface) is a software tool that allows users to access the features of the Acceldata product through various interfaces, such as APIs, while also using the standard user interface (UI) provided by the product.

Acceldata ADOC CLI is a powerful tool that provides users with a way to interact with the Acceldata product in a more customized and efficient way. It enables users to streamline their workflows, while also providing a range of features depending on user requirements.
