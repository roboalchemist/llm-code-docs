# Policy Metadata & Configuration

Policy Metadata and Configuration APIs provide a set of endpoints that allow users to view, track, and manage the operational state of reliability policies in Acceldata.  

While policies define the rules and logic for data quality, reconciliation, or drift detection, the metadata APIs let you:

- View execution history across different policy types
- Retrieve detailed results for any execution
- Understand which assets are associated with each policy
- Manage tagging and export/import of policies (when needed)

These APIs are especially useful for **operational visibility**, **monitoring**, and **auditing** — without modifying the actual policy configuration.
