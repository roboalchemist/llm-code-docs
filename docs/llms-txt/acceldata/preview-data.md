# Preview Data

Previewing data lets you **validate the contents of an asset quickly** by fetching a small sample (up to 100 rows). Instead of scanning an entire dataset, which might contain billions of records, you can trigger an asynchronous sample request and retrieve results once ready. This step is especially useful for troubleshooting, quick validation, and exploratory analysis.
