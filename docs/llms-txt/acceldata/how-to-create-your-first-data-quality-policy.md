# Create and Execute a Policy

Perform the following steps to create and execute your first sample policy:

1. Click  **Reliability** icon **-&gt;** **Manage Policies** **-&gt;** **Add Policy -&gt;** Select **Data Quality.**
2. Select the table under the required asset from the **Asset List**. Data Quality Policy page appears. 
3. Click **Configure Rules** -&gt; With **Row Based** Rule Definition as **Null Values**, select any Column Name -&gt; Click **Add.**

![](https://uploads.developerhub.io/prod/Yoq2/vi5108a6bcl9cxxci6d9q7jkgbjawsa5ehbm3jmonhw9dbcmr3bj9ynl9gfyfex5.png)

4. Click **Configure Alerts and Notifications** **-&gt;** **Configure Execution Details -&gt;** **Overall Summary -&gt;** Provide Policy Name and click **Save Policy.** 
5. Click **Run** in the top right corner of the window and then **Execute**.

![](https://uploads.developerhub.io/prod/Yoq2/gj91x4msb852t1jungzwu20ajye1sqkp2nwd27surd9gbu5nwn9iatze0mcqwnng.png)

You have successfully created and executed your first policy.
