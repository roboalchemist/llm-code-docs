# Replit App in ChatGPT

> Learn how to build a Replit App directly from ChatGPT.

‚è∞ *Estimated time: 5 minutes*

Learn how to build Replit Apps directly from ChatGPT using the Replit integration. This integration brings Replit's development environment into your ChatGPT workflow, letting you:

* Create full-stack applications from natural language descriptions
* Bootstrap projects without leaving ChatGPT
* Seamlessly transition to Replit for further development

## ChatGPT Apps

Replit is the fastest way to turn your ideas into a live app, and now you can do it without leaving ChatGPT.

Apps in ChatGPT let you connect external tools and services directly into the conversation. Instead of switching between tabs or copying context around, you can ask ChatGPT to take action inside the tools you already use.

The Replit app allows you to create and bootstrap real applications directly from ChatGPT, with the code living in your Replit Workspace.

Follow the steps below to build your first Replit App in ChatGPT, or try it now by clicking the button below.

<Card title="Try it now" icon="rocket" href="https://chatgpt.com?prompt=build+me+a+cool+app+with+%40replit">
  Build an app with Replit directly in ChatGPT
</Card>

<Steps>
  <Step title="Open a conversation in ChatGPT">
    Start a new conversation or continue an existing one in ChatGPT.
  </Step>

  <Step title="Describe what you want to build and invoke @Replit">
    In the chat input, describe your app idea and mention `@Replit` to invoke the Replit app.

    Example prompt:

    ```text  theme={null}
    Create a beautiful app to split my restaurant bill @Replit
    ```
