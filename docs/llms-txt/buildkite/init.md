# Buildkite CLI init command

The `bk init` command allows you to initialize a pipeline file with Buildkite Pipelines from the command line.

Initialize a pipeline.yaml file

```bash
bk init [flags]
```

## Flags

| Flag | Description |
| --- | --- |
| `--debug` | Enable debug output for REST API calls |
