# Buildkite CLI version command

The `bk version` command allows you to display which version of the Buildkite CLI you're using from the command line.

Print the version of the CLI being used

```bash
bk version [flags]
```

## Flags

| Flag | Description |
| --- | --- |
| `--debug` | Enable debug output for REST API calls |
