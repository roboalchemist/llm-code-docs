# Package ecosystems overview

Buildkite Package Registries supports the following language and package ecosystems:

<ul>
<li><a href="/docs/package-registries/ecosystems/alpine">Alpine (apk)</a></li>
<li>
<a href="/docs/package-registries/ecosystems/oci">OCI (Docker)</a> images</li>
<li><a href="/docs/package-registries/ecosystems/debian">Debian/Ubuntu (deb)</a></li>
<li><a href="/docs/package-registries/ecosystems/files">Files (generic)</a></li>
<li>Helm (<a href="/docs/package-registries/ecosystems/helm-oci">OCI</a> or <a href="/docs/package-registries/ecosystems/helm">Standard</a>) charts</li>
<li>
<a href="/docs/package-registries/ecosystems/hugging-face">Hugging Face</a> models</li>
<li>Java (<a href="/docs/package-registries/ecosystems/maven">Maven</a> or Gradle using <a href="/docs/package-registries/ecosystems/gradle-kotlin">Kotlin</a> or <a href="/docs/package-registries/ecosystems/gradle-groovy">Groovy</a>)</li>
<li><a href="/docs/package-registries/ecosystems/javascript">JavaScript (npm)</a></li>
<li><a href="/docs/package-registries/ecosystems/nuget">NuGet</a></li>
<li><a href="/docs/package-registries/ecosystems/python">Python (PyPI)</a></li>
<li><a href="/docs/package-registries/ecosystems/red-hat">Red Hat (RPM)</a></li>
<li><a href="/docs/package-registries/ecosystems/ruby">Ruby (RubyGems)</a></li>
<li>
<a href="/docs/package-registries/ecosystems/terraform">Terraform</a> modules</li>
</ul>
