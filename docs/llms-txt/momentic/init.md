# init

Initialize a new Momentic project and create a
[configuration file](/cli/configuration).

<CodeGroup>
  ```bash npm theme={null}
  npx momentic init
  ```

  ```bash yarn theme={null}
  yarn dlx momentic init
  ```

  ```bash pnpm theme={null}
  pnpm dlx momentic init
  ```
</CodeGroup>
