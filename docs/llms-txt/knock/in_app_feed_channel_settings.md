### InAppFeedChannelSettings

In-app feed channel settings. Only used as configuration as part of a workflow channel step.

#### Attributes

- **link_tracking** (boolean) - Whether to track link clicks on in-app feed notifications.

#### Example

```json
{
  "link_tracking": true
}
```
