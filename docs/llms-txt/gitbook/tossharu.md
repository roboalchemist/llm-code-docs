# 共有とソーシャル

### ソーシャルアカウント

カスタマイズメニューの **共有** タブから、サイトのメタデータおよびフッターにソーシャルアカウントを追加できます。

クリック **+ アカウントを追加** を選択し、X、GitHub、LinkedIn、Discord、Blueskyなど、含まれているオプションの一覧からアカウントタイプを選びます。このメニューでソーシャルアカウントを追加すると、サイトのメタデータにも追加されます。&#x20;

各ソーシャルアカウントについて、メタデータに影響を与えることなく、ドキュメントサイトのフッターでの表示をオンまたはオフに切り替えることができます。
