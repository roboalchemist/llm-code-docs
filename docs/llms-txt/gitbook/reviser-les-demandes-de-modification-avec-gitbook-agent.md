# Réviser les demandes de modification avec GitBook Agent

Ainsi que [créer du contenu](https://gitbook.com/docs/documentation/fr/gitbook-agent/write-and-edit-with-ai), GitBook Agent peut examiner vos demandes de modification pour trouver et corriger rapidement des erreurs, suggérer des améliorations et plus encore.

L’Agent comprend le contexte du reste du contenu de vos docs, et vous pouvez aussi ajouter [le guide de style de votre équipe](https://gitbook.com/docs/documentation/fr/quest-ce-que-gitbook-agent#add-a-style-guide-and-custom-instructions) pour s’assurer qu’il examine vos modifications tout en respectant le style de votre équipe

### Examiner les demandes de modification avec GitBook Agent

Vous pouvez demander une révision à GitBook Agent dans n’importe quelle [demande de modification ouverte](https://gitbook.com/docs/documentation/fr/collaboration/change-requests/ecran-des-demandes-de-modification) dans votre organisation.

Lors de la consultation d’une demande de modification ouverte, vous pouvez sélectionner GitBook Agent en tant que relecteur. Une fois sélectionné comme relecteur, GitBook Agent parcourra vos modifications et vérifiera l’exactitude, la clarté, et plus encore — en tenant compte du contexte de votre site publié et du guide de style.

Il proposera d’autres modifications si nécessaire — et si vous souhaitez que l’Agent les mette en œuvre, vous pouvez lui demander de le faire soit [via la fenêtre de chat](https://gitbook.com/docs/documentation/fr/write-and-edit-with-ai#open-gitbook-agent-chat-window-in-any-change-request) ou en [le mentionnant dans un commentaire](https://gitbook.com/docs/documentation/fr/write-and-edit-with-ai#tag-gitbook-agent-in-a-comment).

Une fois que vous êtes satisfait de vos modifications, vous pouvez [fusionner la demande de modification](https://gitbook.com/docs/documentation/fr/collaboration/change-requests#merge) pour publier les modifications en direct.

### Mettre à jour les demandes de modification avec GitBook Agent

GitBook Agent travaille avec vous comme un autre membre de votre équipe. [À l’intérieur d’une demande de modification](https://gitbook.com/docs/documentation/fr/collaboration/change-requests/demandes-de-modification-dans-un-espace), vous pouvez [ouvrir la fenêtre de chat GitBook Agent](https://gitbook.com/docs/documentation/fr/write-and-edit-with-ai#open-gitbook-agent-chat-window-in-any-change-request) pour commencer à collaborer.

Vous pouvez lui demander de faire des choses comme :

* Rédiger de nouveaux contenus ou développer des sections existantes
* Corriger les fautes de frappe et les erreurs d’orthographe
* Réécrire des explications peu claires ou obsolètes
* Relire les pages pour en vérifier l’exactitude, la clarté et l’exhaustivité
* Générer ou mettre à jour des exemples de code
* Identifier les documents manquants ou les lacunes dans les workflows.

En savoir plus sur [la rédaction avec GitBook Agent ici](https://gitbook.com/docs/documentation/fr/gitbook-agent/write-and-edit-with-ai).
