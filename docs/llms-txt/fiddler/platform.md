# Monitoring Platform

Fiddler Monitoring helps you detect performance issues in your production ML models and LLM applications. It provides five built-in metric types, plus support for custom metrics that you can monitor and track.

* [**Data Drift**](https://docs.fiddler.ai/observability/platform/data-drift-platform)
* [**Performance**](https://docs.fiddler.ai/observability/platform/performance-tracking-platform)
* [**Data Integrity**](https://docs.fiddler.ai/observability/platform/data-integrity-platform)
* [**Traffic**](https://docs.fiddler.ai/observability/platform/traffic-platform)
* [**Statistic**](https://docs.fiddler.ai/observability/platform/statistics)
* [**Custom Metrics**](https://docs.fiddler.ai/observability/platform/custom-metrics)

These metrics can be visualized through Fiddler's Charts and Dashboards, alerted on using Alerts, and extended with Custom Metrics, allowing for quick root cause analysis alerts.
