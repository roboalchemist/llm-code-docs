# cPanel/WHM Automation Plugin

The SMTP2GO cPanel/WHM Automation Plugin is designed for hosting providers who manage many customer domains and email services. The plugin is designed to complete the following tasks:

* Add Verified Senders to your SMTP2GO App dashboard.
* Update the cPanel-managed DNS with the two required CNAME entries for Sender Domain verification.
* Verify the DNS changes have been propagated.
* Add an SMTP User and complex password to your SMTP2GO App dashboard.
* Update the cPanel EXIM sending configuration files to allow email to be sent via SMTP2GO’s infrastructure.

For detailed setup instructions and information, view our [cPanel/WHM Automation Plugin knowledgebase](https://support.smtp2go.com/hc/en-gb/articles/4405809053977-cPanel-WHM-Automation-Plugin) article.
