# Official Libraries

SMTP2GO's [GitHub repository](https://github.com/smtp2go-oss) contains open-source software integrations that you can use to integrate our email service into your project. The available libraries include:

* [.NET (Visual Studio NuGet)](https://www.nuget.org/packages/Smtp2Go.ApiClient/)
* [Go](https://github.com/smtp2go-oss/smtp2go-go)
* [Python](https://github.com/smtp2go-oss/smtp2go-python)
* [Node.js](https://github.com/smtp2go-oss/nodejs)
* [PHP](https://github.com/smtp2go-oss/smtp2go-php)
* [Ruby / Ruby on Rails](https://github.com/smtp2go-oss/smtp2go-ruby)
* [Rust](https://github.com/smtp2go-oss/smtp2go-rust)
* [Django](https://github.com/smtp2go-oss/smtp2go-django)
* [Wordpress Plugin](https://github.com/smtp2go-oss/wordpress-plugin)

If you require assistance or would like to submit a request for additional libraries to be added to the repository, please [contact](https://www.smtp2go.com/contact/) our support team. Additional code examples can be found under the “Code and Frameworks” section of our [setup guides page](https://www.smtp2go.com/setupguide/).
