# WordPress Plugin

SMTP2GO’s official WordPress plugin can be set up in minutes and replaces the default WordPress wp\_mail() function. Once enabled, all outgoing mail through your WordPress installation will be sent via the SMTP2GO API.

Utilizing the plugin will ensure your emails are delivered with the highest sending reputation and provide you with a range of additional reporting and features accessible in the SMTP2GO App or via the API. Notably, you’ll gain valuable insights into every aspect of your email’s life cycle, enabling you to track delivery rates, opens, clicks, bounce rates and more.

For detailed setup instructions, view our [WordPress Plugin knowledgebase article](https://support.smtp2go.com/hc/en-gb/articles/900000195666-SMTP2GO-WordPress-Plugin) or view the plugin in the [WordPress Plugins directory](https://wordpress.org/plugins/smtp2go/).

If you have questions or need assistance with setting up or using our plugin, please [contact](https://www.smtp2go.com/contact/) our support team.
