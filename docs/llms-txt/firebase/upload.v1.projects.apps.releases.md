# REST Resource: upload.v1.projects.apps.releases

- [Resource](https://firebase.google.com/docs/reference/app-distribution/rest/v1/upload.v1.projects.apps.releases#RESOURCE_REPRESENTATION)
- [Methods](https://firebase.google.com/docs/reference/app-distribution/rest/v1/upload.v1.projects.apps.releases#METHODS_SUMMARY)

## Resource

This resource is used to upload a binary, which may create or update a [Release](https://firebase.google.com/docs/reference/app-distribution/rest/v1/projects.apps.releases#Release). There is no persistent data associated with this resource.

|                                                                  ## Methods                                                                  ||
|---------------------------------------------------------------------------------------------------------------------------|-------------------|
| ### [upload](https://firebase.google.com/docs/reference/app-distribution/rest/v1/upload.v1.projects.apps.releases/upload) | Uploads a binary. |
