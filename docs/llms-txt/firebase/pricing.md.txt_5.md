# Firebase Studio pricing, quotas, and limits

Access to Firebase Studio is available at no cost, but you can increase the
number of workspaces you can create by joining the
[Google Developer Program](https://developers.google.com/profile/u/_/dashboard).

Certain integrations (like Firebase App Hosting) may require a
Cloud Billing account.

## Google Developer Program summary

The Google Developer Program is an optional, but recommended program to help
you build new projects, advance your career, and expand your network. This table
lists its specific benefits for Firebase Studio users.

| **Feature** | **No Google Developer Program profile** | **Google Developer Program (Standard plan)** | **Google Developer Program (Premium plan)** |
|---|---|---|---|
| **Firebase Studio workspaces** *[See important note about Flutter and React Native + Expo workspaces](https://firebase.google.com/docs/studio/pricing#flutter-react-native)* | 3 per user | 10 per user | 30 per user |
| **Gemini quota for the App Prototyping agent** | Standard | Standard | Increased for the App Prototyping agent |
| **Pricing** | No cost | No cost | Subscription |

## Important notes on Flutter and React Native + Expo workspaces

If you're not subscribed to a
[Google Developer Program premium plan](https://developers.google.com/profile/u/_/plans-and-pricing),
you're limited to two total workspaces across both Flutter and React Native +
Expo. This means you can have a maximum of:

- Two Flutter workspaces OR
- Two React Native + Expo workspaces OR
- One Flutter and one React Native + Expo workspace

Once you have reached the maximum of two workspaces, you must delete an existing
workspace before you can create a new one.

### What if I already have more than two workspaces?

If you have more than two Flutter or React Native + Expo workspaces, you will
retain access to them. However, you won't be able to create any new workspaces
in these categories until you delete enough to fall under the two workspace
limit.

To increase the number of Flutter and React Native + Expo workspaces you can
create, join the [Google Developer Program premium plan](https://developers.google.com/profile/u/_/plans-and-pricing).

## Cloud Billing and paid services

Certain integrations, such as Firebase App Hosting, may require a
Cloud Billing account. If you create a Firebase project with
Firebase Studio and link a billing account to that project, the following
occurs:

- The Firebase project is automatically upgraded to the [pay-as-you-go Blaze pricing plan](https://firebase.google.com/pricing).
- You will be charged for any usage of paid services beyond the no-cost quota.
- Your Gemini Developer API usage is upgraded to the paid tier.
