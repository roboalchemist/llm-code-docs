# GeneratedConnector


```
interface GeneratedConnector<T : GeneratedConnector<T>>
```

<br />

*** ** * ** ***

The interface to be implemented by the over-arching "connector" classes that are generated by the Firebase CLI code generation.

### Safe for concurrent use

All methods and properties of `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/dataconnect/generated/GeneratedConnector` are thread-safe and may be safely called and/or accessed concurrently from multiple threads and/or coroutines.

### Stable for inheritance

The `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/dataconnect/generated/GeneratedConnector` interface *is* stable for inheritance in third-party libraries, as new methods will not be added to this interface and contracts of the existing methods will not be changed, except possibly during major version number changes.
