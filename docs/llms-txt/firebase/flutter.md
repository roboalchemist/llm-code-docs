## Discover Firebase for Flutter

Firebase provides a robust set of Flutter plugins that connect your Flutter application to our services.

Integrating Firebase with your Flutter apps lets you**get to market** and deliver value to your users,**improve app quality** in less time with less effort, and**optimize your app experience**to keep users happy.

## Get to know Firebase for Flutter

Ready to connect your Flutter apps with Firebase? If you're new to using Firebase in Flutter we recommend starting with the**Get to know Firebase for Flutter** [codelab](https://firebase.google.com/codelabs/firebase-get-to-know-flutter)and video.

Then, for detailed setup instructions, follow our guide to[Adding Firebase to your Flutter app](https://firebase.google.com/docs/flutter/setup).

## Product plugins

The Firebase plugins for Flutter provide a broad range of client- and server-side functionality.  

Investigate plugins that let you build, release, monitor and engage users.  
![](http://firebase.google.com/static/docs/images/analytics_light.png)  

### Analytics

Google Analytics is a free app measurement solution that provides insight on app usage and user engagement. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/analytics/get-started?platform=flutter).  
![](http://firebase.google.com/static/docs/images/app_check_light.png)  

### App Check

Firebase App Check helps protect your backend resources from abuse such as billing fraud or phishing. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/app-check/flutter/default-providers).  
![](http://firebase.google.com/static/docs/images/auth_light.png)  

### Authentication

Firebase Authentication provides backend services and easy-to-use SDKs to authenticate users to your app. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/auth/flutter/start).  
![](http://firebase.google.com/static/docs/images/cloudfirestore_light.png)  

### Cloud Firestore

Firestore is a flexible, scalable NoSQL cloud database to store and sync data. It keeps your data in sync across client apps through realtime listeners, and offers offline support. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/firestore/quickstart).  
![](http://firebase.google.com/static/docs/images/functions_light.png)  

### Cloud Functions

Cloud Functions for Firebase let you automatically run backend code in response to events triggered by Firebase features and HTTPS requests. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started deploying functions](https://firebase.google.com/docs/functions/get-started)and[calling them from your app](https://firebase.google.com/docs/functions/callable).  
![](http://firebase.google.com/static/docs/images/cloudmessaging_light.png)  

### Cloud Messaging

Firebase Cloud Messaging (FCM) is a cross-platform messaging solution that lets you reliably send messages at no cost. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/cloud-messaging/flutter/client).  
![](http://firebase.google.com/static/docs/images/cloudstorage_light.png)  

### Cloud Storage

Cloud Storage for Firebase is designed to help you quickly and easily store and serve user-generated content, such as photos and videos. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/storage/flutter/start).  
![](http://firebase.google.com/static/docs/images/crashlytics_light.png)  

### Crashlytics

Firebase Crashlytics helps you to collect analytics and details about crashes and errors that occur in your app. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/crashlytics/flutter/get-started).  
![](http://firebase.google.com/static/docs/images/inappmessaging.png)  

### In-App Messaging

Firebase In-App Messaging helps you engage your app's active users by sending them targeted, contextual messages that encourage them to use key app features. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/in-app-messaging/get-started?platform=flutter).  
![](http://firebase.google.com/static/docs/images/hosting_light.png)  

### Installations

Firebase Installations lets you manage the installation of your app on a user's device. The Firebase installations service (FIS) provides a Firebase installation ID (FID) for each installed instance of a Firebase app. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/projects/manage-installations).  
![](http://firebase.google.com/static/docs/images/mldiscovery_light.png)  

### ML Model Downloader

If you use custom TensorFlow Lite models, Firebase ML can help you ensure your users are always using the best-available version of your custom model. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/ml/flutter/use-custom-models).  
![](http://firebase.google.com/static/docs/images/performance_light.png)  

### Performance Monitoring

Firebase Performance Monitoring is a service that helps you to gain insight into the performance characteristics of your iOS, Android, and web apps. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/perf-mon/flutter/get-started).  
![](http://firebase.google.com/static/docs/images/rtdb_light.png)  

### Realtime Database

The Firebase Realtime Database is a cloud-hosted database. Data is stored as JSON and synchronized in realtime to every connected client. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/database/flutter/start).  
![](http://firebase.google.com/static/docs/images/config_light.png)  

### Remote Config

Firebase Remote Config is a cloud service that lets you change the behavior and appearance of your app without requiring users to download an app update. Check[platform support](https://firebase.google.com/docs/flutter/setup#available-plugins)then[get started](https://firebase.google.com/docs/remote-config/get-started?platform=flutter).
