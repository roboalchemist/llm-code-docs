# GenerativeModel

    @available(iOS 15.0, macOS 12.0, tvOS 15.0, watchOS 8.0, *)
    public final class GenerativeModel : Sendable

A type that represents a remote multimodal model (like Gemini), with the ability to generate
content based on various input types.

  ### [generateContent(_:)](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Classes/GenerativeModel#/s:10FirebaseAI15GenerativeModelC15generateContentyAA08GenerateF8ResponseVAA18PartsRepresentable_pd_tYaKF)

  `
  `  
  Generates content from String and/or image inputs, given to the model as a prompt, that are
  representable as one or more [Part](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Protocols/Part.html)s.

  Since [Part](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Protocols/Part.html)s do not specify a role, this method is intended for generating content from
  [zero-shot](https://developers.google.com/machine-learning/glossary/generative#zero-shot-prompting)
  or "direct" prompts. For
  [few-shot](https://developers.google.com/machine-learning/glossary/generative#few-shot-prompting)
  prompts, see `generateContent(_ content: [ModelContent])`.  
  Throws
  A [GenerateContentError](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Enums/GenerateContentError.html) if the request failed.

  ### [countTokens(_:)](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Classes/GenerativeModel#/s:10FirebaseAI15GenerativeModelC11countTokensyAA05CountF8ResponseVAA18PartsRepresentable_pd_tYaKF)

  Runs the model's tokenizer on String and/or image inputs that are representable as one or more
  [Part](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Protocols/Part.html)s.

  Since [Part](https://firebase.google.com/docs/reference/swift/firebaseai/api/reference/Protocols/Part.html)s do not specify a role, this method is intended for tokenizing
  [zero-shot](https://developers.google.com/machine-learning/glossary/generative#zero-shot-prompting)
  or "direct" prompts. For
  [few-shot](https://developers.google.com/machine-learning/glossary/generative#few-shot-prompting)
  input, see `countTokens(_ content: @autoclosure () throws -> [ModelContent])`.
