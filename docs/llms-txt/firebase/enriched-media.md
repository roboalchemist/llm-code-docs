UseCloud FirestoreandFirebase Extensionsto unlock the value of your audio and image resources using AI models.

## Classify images

Extract labels from images and save to Firestore using Cloud Vision API.[Learn more](https://extensions.dev/extensions/googlecloud/storage-label-images).

## Convert speech to text from audio files

Transcribe audio files in Cloud Storage to .txt files using Cloud Speech To Text.[Learn more](https://extensions.dev/extensions/googlecloud/speech-to-text).

## Perform optical character recognition (OCR)

Extract text from images and save to Firestore using Cloud Vision API.[Learn more](https://extensions.dev/extensions/googlecloud/storage-extract-image-text).
