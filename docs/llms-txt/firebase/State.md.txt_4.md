# State

Defining State used in the message State

| Enums ||
|---|---|
| `STATE_UNSPECIFIED` | The experiment state is unspecified. |
| `PENDING` | The experiment is pending. |
| `RUNNING` | The experiment is running. |
| `DONE` | The experiment is done. |
| `EXPIRED` | The experiment is expired. |
