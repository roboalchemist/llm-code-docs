# Source: https://firebase.google.com/docs/firestore/solutions/bigquery.md.txt

[BigQuery](https://cloud.google.com/bigquery/docs) is a fully managed,
petabyte-scale, and cost-effective analytics data warehouse that lets you run
analytics over vast amounts of data. You can use the following
Firebase Extensions to connect Cloud Firestore with BigQuery:

## Stream Cloud Firestore data to BigQuery

Send real-time, incremental updates from a Cloud Firestore collection to BigQuery.
[Learn more](https://extensions.dev/extensions/firebase/firestore-bigquery-export).

## Export BigQuery query results to Cloud Firestore

Schedule and export BigQuery query results in Cloud Firestore for
real-time delivery. [Learn more](https://extensions.dev/extensions/googlecloud/bigquery-firestore-export).
