# FirebaseAuth

public final class **FirebaseAuth** extends [AbstractFirebaseAuth](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/auth/AbstractFirebaseAuth)  
This class is the entry point for all server-side Firebase Authentication actions.

You can get an instance of FirebaseAuth via [getInstance(FirebaseApp)](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/auth/FirebaseAuth#getInstance(com.google.firebase.FirebaseApp)) and
then use it to perform a variety of authentication-related operations, including generating
custom tokens for use by client-side code, verifying Firebase ID Tokens received from clients, or
creating new FirebaseApp instances that are scoped to a particular authentication UID.
