# AnalyticsCallOptions interface

Additional options that can be passed to Analytics method calls such as `logEvent`, etc.

**Signature:**  

    export interface AnalyticsCallOptions 

## Properties

|                                                       Property                                                       |  Type   |                                             Description                                             |
|----------------------------------------------------------------------------------------------------------------------|---------|-----------------------------------------------------------------------------------------------------|
| [global](https://firebase.google.com/docs/reference/js/analytics.analyticscalloptions.md#analyticscalloptionsglobal) | boolean | If true, this config or event call applies globally to all Google Analytics properties on the page. |

## AnalyticsCallOptions.global

If true, this config or event call applies globally to all Google Analytics properties on the page.

**Signature:**  

    global: boolean;
