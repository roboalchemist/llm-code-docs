## Properties

### FilteredReason

```c#
string Firebase::AI::ImagenGenerationResponse< T >::FilteredReason
```
The reason, if any, that generated images were filtered out.

This property will only be populated if fewer images were generated than were requested, otherwise it will be `null`. Images may be filtered out due to the `SafetyFilterLevel`, the `PersonFilterLevel`, or filtering included in the model. The filter levels may be adjusted in your `https://firebase.google.com/docs/reference/unity/struct/firebase/a-i/imagen-safety-settings#struct_firebase_1_1_a_i_1_1_imagen_safety_settings`. See the [Responsible AI and usage guidelines for Imagen](https://cloud.google.com/vertex-ai/generative-ai/docs/image/responsible-ai-imagen) for more details.

### Images

```c#
IReadOnlyList< T > Firebase::AI::ImagenGenerationResponse< T >::Images
```
The images generated by Imagen; see `https://firebase.google.com/docs/reference/unity/struct/firebase/a-i/imagen-inline-image#struct_firebase_1_1_a_i_1_1_imagen_inline_image`.

Important: The number of images generated may be fewer than the number requested if one or more were filtered out; see `FilteredReason`.
