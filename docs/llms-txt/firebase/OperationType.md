# OperationType

The type of index operation.

|                                                 Enums                                                 ||
|------------------------------|-------------------------------------------------------------------------|
| `OPERATION_TYPE_UNSPECIFIED` | Unspecified. Never set by server.                                       |
| `CREATING_INDEX`             | The operation is creating the index. Initiated by a `CreateIndex` call. |
