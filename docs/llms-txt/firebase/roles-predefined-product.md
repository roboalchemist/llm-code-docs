# Source: https://firebase.google.com/docs/projects/iam/roles-predefined-product.md.txt

These roles grant full read/write or read-only access to*specific*Firebase products.

Assign these roles to project members using the[Google Cloudconsole](https://console.cloud.google.com/iam-admin/iam).
| **Note:** The following permissions are in*all* the Firebase product-level predefined roles:  
| - firebase.clients.get  
| - firebase.clients.list  
| - firebase.projects.get  
| - resourcemanager.projects.get  
| - resourcemanager.projects.list
