# GenerativeModel

    @available(iOS 15.0, macOS 12.0, tvOS 15.0, watchOS 8.0, *)
    public final class GenerativeModel : Sendable

A type that represents a remote multimodal model (like Gemini), with the ability to generate
content based on various input types.

  ` Generates content from String and/or image inputs, given to the model as a prompt, that are
  representable as one or more `https://firebase.google.com/docs/reference/swift/firebasevertexai/api/reference/Protocols/Part.html`s.

  Since `https://firebase.google.com/docs/reference/swift/firebasevertexai/api/reference/Protocols/Part.html`s do not specify a role, this method is intended for generating content from
  [zero-shot](https://developers.google.com/machine-learning/glossary/generative#zero-shot-prompting)
  or "direct" prompts. For
  [few-shot](https://developers.google.com/machine-learning/glossary/generative#few-shot-prompting)
  prompts, see `generateContent(_ content: [ModelContent])`.
  Throws
  A `https://firebase.google.com/docs/reference/swift/firebasevertexai/api/reference/Enums/GenerateContentError.html` if the request failed.
