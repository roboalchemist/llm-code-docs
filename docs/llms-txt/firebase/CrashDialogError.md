# CrashDialogError

- [JSON representation](https://firebase.google.com/docs/test-lab/reference/toolresults/rest/v1beta3/CrashDialogError#SCHEMA_REPRESENTATION)

Crash dialog was detected during the test execution

|        JSON representation         |
|------------------------------------|
| ``` { "crashPackage": string } ``` |

|                                  Fields                                  ||
|----------------|----------------------------------------------------------|
| `crashPackage` | `string` The name of the package that caused the dialog. |
