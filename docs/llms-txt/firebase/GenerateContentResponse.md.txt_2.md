## Public fields

### candidates

```
public final @NonNull List<@NonNull Candidate> candidates
```

The list of `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/Candidate` responses generated by the model.

### functionCalls

```
public final @NonNull List<@NonNull FunctionCallPart> functionCalls
```

Convenience field to list all the `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/FunctionCallPart`s in the response.

The value is an empty list if the response contains no `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### inferenceSource

```
public final @NonNull InferenceSource inferenceSource
```

The source of the inference for this response.

### inlineDataParts

```
public final @NonNull List<@NonNull InlineDataPart> inlineDataParts
```

Convenience field representing all the `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/InlineDataPart`s in the first candidate.

This also includes any `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/ImagePart`, but they will be represented as `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/InlineDataPart` instead.

The value is an empty list if the response contains no `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### promptFeedback

```
public final PromptFeedback promptFeedback
```

Feedback about the prompt send to the model to generate this response. When streaming, it's only populated in the first response.

### text

```
public final String text
```

Convenience field representing all the text parts in the response as a single string.

The value is null if the response contains no valid text `https://firebase.google.com/docs/reference/android/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### thoughtSummary

```
public final String thoughtSummary
```

Convenience field representing all the text parts in the response that are marked as thoughts as a single string, if they exist.

Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### usageMetadata

```
public final UsageMetadata usageMetadata
```

Information about the number of tokens in the prompt and in the response.
