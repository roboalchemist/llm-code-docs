Firebase Authentication.

## Functions

|                                                   Function                                                   |                                                                                                                                                                                                                                                          Description                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getAuth(app)](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.md#getauth_8a40afc) | Gets the [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service for the default app or a given app.`getAuth()` can be called with no arguments to access the default app's [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service or as `getAuth(app)` to access the [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service associated with a specific app. |

## getAuth(app)

Gets the [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service for the default app or a given app.

`getAuth()` can be called with no arguments to access the default app's [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service or as `getAuth(app)` to access the [Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class) service associated with a specific app.

**Signature:**  

    export declare function getAuth(app?: App): Auth;

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| app       | App  |             |

**Returns:**

[Auth](https://firebase.google.com/docs/reference/admin/node/firebase-admin.auth.auth.md#auth_class)

### Example 1

    // Get the Auth service for the default app
    const defaultAuth = getAuth();

### Example 2

    // Get the Auth service for a given app
    const otherAuth = getAuth(otherApp);
