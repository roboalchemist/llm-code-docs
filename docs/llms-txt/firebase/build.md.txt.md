## Featured products

[![](http://firebase.google.com/static/docs/images/cloudfirestore_dark.png)](http://firebase.google.com/docs/firestore)

### [Cloud Firestore](http://firebase.google.com/docs/firestore)

Firestore is a flexible, scalable database for mobile, web, and server development from Firebase and Google Cloud. It keeps your data in sync across client apps through realtime listeners and offers offline support for mobile and web so you can build responsive apps that work regardless of network latency or Internet connectivity. [Introduction](http://firebase.google.com/docs/firestore) [![](http://firebase.google.com/static/docs/images/functions_dark.png)](http://firebase.google.com/docs/functions)

### [Cloud Functions](http://firebase.google.com/docs/functions)

Cloud Functions for Firebase is a serverless framework that lets you automatically run backend code in response to events triggered by Firebase features and HTTPS requests. Your code is stored in Google servers so there's no need to manage and scale your own. [Introduction](http://firebase.google.com/docs/functions) [![](http://firebase.google.com/static/docs/images/rtdb_dark.png)](http://firebase.google.com/docs/database)

### [Realtime Database](http://firebase.google.com/docs/database)

The Firebase Realtime Database is a cloud-hosted database. Data is stored as JSON and synchronized in realtime to every connected client, with cross-platform support for iOS, Android, Web and more. [Introduction](http://firebase.google.com/docs/database)

### Introducing Firestore

Cloud Firestore is a NoSQL document database that simplifies storing, syncing, and querying data for your mobile and web apps at global scale. Its client libraries provide live synchronization and offline support, while its security features and integrations with the Firebase and Google Cloud platforms accelerate building truly serverless apps. [Video](https://www.youtube.com/watch?v=8sGY55yxicA)

### Introducing Firebase Authentication

Firebase Authentication makes authentication easy for end users and developers. It allows you to focus on your users, and not the sign-in infrastructure to support them. [Video](https://www.youtube.com/watch?v=p5-BDRCAkMI)

### Introducing Firebase ML

Developers are increasingly relying on machine learning to enhance their app's user experience, and only with finely-tuned machine learning models can they deliver those powerful features to delight their users.
