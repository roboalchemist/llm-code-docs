# PopupRedirectResolver interface

A resolver used for handling DOM specific operations like [signInWithPopup()](https://firebase.google.com/docs/reference/js/auth.md#signinwithpopup_770f816) or [signInWithRedirect()](https://firebase.google.com/docs/reference/js/auth.md#signinwithredirect_770f816).
