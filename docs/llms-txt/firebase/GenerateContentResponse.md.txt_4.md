## Public properties

### candidates

```
val candidates: List<Candidate>
```

The list of `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/Candidate` responses generated by the model.

### functionCalls

```
val functionCalls: List<FunctionCallPart>
```

Convenience field to list all the `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/FunctionCallPart`s in the response.

The value is an empty list if the response contains no `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### inferenceSource

```
val inferenceSource: InferenceSource
```

The source of the inference for this response.

### inlineDataParts

```
val inlineDataParts: List<InlineDataPart>
```

Convenience field representing all the `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/InlineDataPart`s in the first candidate.

This also includes any `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/ImagePart`, but they will be represented as `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/InlineDataPart` instead.

The value is an empty list if the response contains no `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### promptFeedback

```
val promptFeedback: PromptFeedback?
```

Feedback about the prompt send to the model to generate this response. When streaming, it's only populated in the first response.

### text

```
val text: String?
```

Convenience field representing all the text parts in the response as a single string.

The value is null if the response contains no valid text `https://firebase.google.com/docs/reference/kotlin/com/google/firebase/ai/type/GenerateContentResponse#candidates()`.

Any part that's marked as a thought will be ignored. Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### thoughtSummary

```
val thoughtSummary: String?
```

Convenience field representing all the text parts in the response that are marked as thoughts as a single string, if they exist.

Learn more about [thinking](https://firebase.google.com/docs/ai-logic/thinking?api=dev).

### usageMetadata

```
val usageMetadata: UsageMetadata?
```

Information about the number of tokens in the prompt and in the response.
