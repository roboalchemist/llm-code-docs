# Get started with a Firebase Studio template

Firebase Studio supports many frameworks and backends and provides a large
number of [pre-configured
templates](http://github.com/firebase-studio/templates) and [sample
apps](https://github.com/firebase-studio/templates?category=solutions)
for a multitude of frameworks, languages, databases, and APIs.

All of the templates below---and, most importantly, a blank template that
you can use for any framework you want---are available in
Firebase Studio and on
[GitHub](https://github.com/firebase-studio/templates). You
can also find
[Community templates](https://github.com/firebase-studio/community-templates)
on GitHub, which allow contributions from the Open Source community.

## Get started with a template

<button value="official" default="">Official templates</button> <button value="community">Community templates</button>

To start a Firebase Studio project from a template:

1. Log into your Google Account and open
   [Firebase Studio](https://firebase.google.com/docs/studio/https://studio.firebase.google.com/).

2. If you see a template you'd like to use, select it. Otherwise, click
   **See all templates** and select a template that matches what you'd like to
   use.

3. When prompted, enter a name for your workspace and, depending on the type
   of project you selected, any other needed information. For example, if you
   chose a Python Flask project, you can choose a package manager (poetry or
   pip) and a server type (API server or web server).

4. Click **Create** . Firebase Studio creates a new workspace based on
   the template you chose.

Note that templates can include:

- **Custom AI rules files** that add context for Gemini to better understand and process changes for the type of app you're developing. These files can be modified or deleted to adjust how you prefer to interact with Gemini. [Learn more about where to find and edit AI rules
  files](https://firebase.google.com/docs/studio/set-up-gemini#custom-instructions).
- **MCP servers** that provide additional tools and data sources for Gemini to use. [Learn more about where to find or change which
  MCP servers Gemini can connect to](https://firebase.google.com/docs/studio/mcp-servers).
