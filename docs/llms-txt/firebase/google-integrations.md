## Use theFirebase Studiopanel

With fully-integrated workflows fromFirebase Studio, you can add Google and Firebase APIs and services to your app with the click of a button.

1. Open[Firebase Studio](https://studio.firebase.google.com/).

2. InCodeview, click theFirebase Studioicon on the right side of the workspace.

3. Expand the**Google Integrations**section to see the list of available workflows.

4. Click the action you want to take for the API or service you want to add to your app.

5. Follow the prompts to authenticate your account, select a billing project (if required), and add the selected service to your app.

   TheFirebase Studiopanel provides the following integrations:
   - [Publish your app withFirebase HostingorCloud Run](https://firebase.google.com/docs/studio/deploy-app).
   - EnableGeminiDeveloper API and get aGemini APIkey.
   - Set up the[Google Maps API](https://mapsplatform.google.com/)and get a Maps API key.
   - Use[Google CloudSecret Manager](https://cloud.google.com/security/products/secret-manager)to securely store API keys and credentials.
