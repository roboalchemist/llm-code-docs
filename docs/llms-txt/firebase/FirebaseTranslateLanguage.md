## Public Methods

#### public static [Set](https://developer.android.com/reference/java/util/Set.html)\<[Integer](https://developer.android.com/reference/java/lang/Integer.html)\>
**getAllLanguages** ()

Returns the set of all [FirebaseTranslateLanguage.TranslateLanguage](https://firebase.google.com/docs/reference/android/com/google/firebase/ml/naturallanguage/translate/FirebaseTranslateLanguage.TranslateLanguage) values that are supported by the
Translate API.  

#### public static [String](https://developer.android.com/reference/java/lang/String.html) **languageCodeForLanguage** (int language)

Returns the BCP-47 language code for the given [FirebaseTranslateLanguage.TranslateLanguage](https://firebase.google.com/docs/reference/android/com/google/firebase/ml/naturallanguage/translate/FirebaseTranslateLanguage.TranslateLanguage).  

#### public static [Integer](https://developer.android.com/reference/java/lang/Integer.html) **languageForLanguageCode** ([String](https://developer.android.com/reference/java/lang/String.html) languageCode)

Returns the [FirebaseTranslateLanguage.TranslateLanguage](https://firebase.google.com/docs/reference/android/com/google/firebase/ml/naturallanguage/translate/FirebaseTranslateLanguage.TranslateLanguage) for a given BCP-47 language
code, or `null` if the language code is invalid or not supported by the
Translate API.
