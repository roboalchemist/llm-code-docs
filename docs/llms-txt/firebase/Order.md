# Order

The supported orderings.

|                                 Enums                                 ||
|---------------------|--------------------------------------------------|
| `ORDER_UNSPECIFIED` | The ordering is unspecified. Not a valid option. |
| `ASCENDING`         | The field is ordered by ascending field value.   |
| `DESCENDING`        | The field is ordered by descending field value.  |
