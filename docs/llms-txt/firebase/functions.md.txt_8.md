# Cloud Functions for Firebase API Reference

The SDK supports
[Node.js](https://firebase.google.com/docs/reference/functions/2nd-gen/node/firebase-functions) and
[Python](https://firebase.google.com/docs/reference/functions/2nd-gen/python) across the 2nd gen
feature set.

The SDK also includes Node.js support for Cloud Functions for Firebase
[(1st gen)](https://firebase.google.com/docs/reference/functions/firebase-functions).
See the [version comparison](https://firebase.google.com/docs/functions/version-comparison)
for more information on the differences between the two versions.
The [`firebase-functions-test`](https://firebase.google.com/docs/reference/functions/test/test) SDK serves
as a Node.js testing companion to both versions.
