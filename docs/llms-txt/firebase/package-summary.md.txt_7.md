# com.google.firebase.projectmanagement

### Classes

|---|---|
| [AndroidApp](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/AndroidApp) | An instance of this class is a reference to an Android App within a Firebase Project; it can be used to query detailed information about the App, modify the display name of the App, or download the configuration file for the App. |
| [AndroidAppMetadata](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/AndroidAppMetadata) | Contains detailed information about an Android App. |
| [FirebaseProjectManagement](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/FirebaseProjectManagement) | This class is the entry point for all Firebase Project Management actions. |
| [IosApp](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/IosApp) | An instance of this class is a reference to an iOS App within a Firebase Project; it can be used to query detailed information about the App, modify the display name of the App, or download the configuration file for the App. |
| [IosAppMetadata](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/IosAppMetadata) | Contains detailed information about an iOS App. |
| [ShaCertificate](https://firebase.google.com/docs/reference/admin/java/reference/com/google/firebase/projectmanagement/ShaCertificate) | Information about an SHA certificate associated with an Android app. |
