# ChangeType

Specifies how the index is changing.

|                                  Enums                                   ||
|---------------------------|-----------------------------------------------|
| `CHANGE_TYPE_UNSPECIFIED` | The type of change is not specified or known. |
| `ADD`                     | The single field index is being added.        |
| `REMOVE`                  | The single field index is being removed.      |
