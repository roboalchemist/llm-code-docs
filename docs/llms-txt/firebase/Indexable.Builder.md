# Indexable.Builder

public static class **Indexable.Builder** extends [IndexableBuilder](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/builders/IndexableBuilder)\<[Indexable.Builder](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable.Builder)\>  
The builder for [Indexable](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable).

Convenience methods to construct indexables for common data types are available via
[Indexables](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/builders/Indexables).

Note that the following limitations are automatically applied:

- The name field must be set for the root [Indexable](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable). Refer to the [IndexableBuilder.setName(String)](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/builders/IndexableBuilder#setName(java.lang.String)) method.
- The number of characters in a string value is limited to [Indexable.MAX_STRING_LENGTH](https://firebase.google.com/docs/reference/android).
- The number of values for a single field is limited to [Indexable.MAX_REPEATED_SIZE](https://firebase.google.com/docs/reference/android).
- The depth of nested values is limited to [Indexable.MAX_NESTING_DEPTH](https://firebase.google.com/docs/reference/android).
- The number of set fields on an [Indexable](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable) is limited to [Indexable.MAX_NUMBER_OF_FIELDS](https://firebase.google.com/docs/reference/android).
- The total byte size of an [Indexable](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable) is limited to [Indexable.MAX_BYTE_SIZE](https://firebase.google.com/docs/reference/android).
