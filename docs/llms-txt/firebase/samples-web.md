## Quickstart - Friendly Eats

This sample app demonstrates building a simple restaurant recommendation service usingFirebase. This demonstrates features such as compound queries, client-side transactions, subcollections, and offline persistence.

For more information, see the[sample page](https://firebaseopensource.com/projects/firebase/friendlyeats-web/).
