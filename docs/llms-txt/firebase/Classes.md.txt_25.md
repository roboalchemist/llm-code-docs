# Classes

The following classes are available globally.
- `


  ### [Functions](https://firebase.google.com/docs/reference/swift/firebasefunctions/api/reference/Classes/Functions)


  ` `Functions` is the client for Cloud Functions for a Firebase project.

  #### Declaration

  Swift

      @objc(FIRFunctions)
      open class Functions : NSObject, @unchecked Sendable

- `


  ### [HTTPSCallableResult](https://firebase.google.com/docs/reference/swift/firebasefunctions/api/reference/Classes/HTTPSCallableResult)

  #### Declaration

  Swift

      @objc(FIRHTTPSCallableResult)

- `


  ### [HTTPSCallable](https://firebase.google.com/docs/reference/swift/firebasefunctions/api/reference/Classes/HTTPSCallable)


  ` A `HTTPSCallable` is a reference to a particular Callable HTTPS trigger in Cloud Functions.

  #### Declaration

  Swift

      @objc(FIRHTTPSCallable)
      public final class HTTPSCallable : NSObject, Sendable
