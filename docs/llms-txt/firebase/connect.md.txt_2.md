# Connect with us

If you experience issues while you're working in Firebase Studio, try
refreshing the page. This action resolves many errors. See [Firebase Studio
troubleshooting \& FAQ](https://firebase.google.com/docs/studio/troubleshooting) for troubleshooting help, a
list of known issues, and answers to frequently asked questions.

If you're still stuck, [ask a question in the discussion
forum](https://community.firebasestudio.dev/) or [contact Firebase
Support](https://firebase.google.com/support/troubleshooter/studio).

To check on real-time Firebase Studio service availability, check out the
[Firebase status dashboard](https://status.firebase.google.com).

If you have a feature request:

- To share your ideas and feature requests and vote on ideas, [join the
  discussion on UserVoice](http://firebase.uservoice.com).

- To file a feature request with Support, visit the
  [Firebase Support
  site](https://firebase.google.com/support/troubleshooter/report).
