# Gemini in Firebase

Streamline your development process with Gemini in
Firebase, an AI-powered collaborative assistant.  
Gemini in Firebase is an AI-powered collaborative assistant that can streamline your development process, reduce debugging time, and help you build high-quality apps with confidence.

<br />

Get instant answers to questions about Firebase products and features,
generate code for faster development, and shorten your troubleshooting process
with new insights from Gemini in Firebase using a natural-language chat
interface in the Firebase console and other Firebase development
environments.

[Get started](https://firebase.google.com/docs/gemini-in-firebase/try-gemini)

## How Gemini in Firebase uses your data

### Gemini in Firebase

If you enable Gemini in Firebase in the Firebase console, Firebase CLI, or Firebase MCP
server, it will have access to your Firebase environment, and responses may be tailored to your
project.

- **If you're using Gemini in Firebase
  (no-cost):** Use of Gemini in Firebase (no-cost) is
  governed by the [Google
  Terms of Service](https://policies.google.com/terms) and the [Generative
  AI prohibited use policy](https://policies.google.com/terms/generative-ai/use-policy), so Gemini in Firebase may use your
  prompts and its responses to train its model.

  Learn more about
  Gemini in Firebase (no-cost) at [Quotas
  and pricing](https://firebase.google.com/docs/gemini-in-firebase#quotas-and-pricing).
- **If you're using Gemini in Firebase with a [Gemini Code Assist
  Standard or Enterprise license](https://codeassist.google/products/business):** Use is governed under
  the [Google Cloud Terms of
  Service](https://cloud.google.com/terms) and the Generative AI Services terms in [Service Specific
  Terms](https://cloud.google.com/terms/service-terms), so Gemini in Firebase does *not* use its
  responses, your prompts, or your data to train its model.

  For more information about how Gemini in Firebase and Gemini Code Assist
  use your data, see [How
  Gemini for Google Cloud uses your data](https://cloud.google.com/gemini/docs/discover/data-governance).  

  For more information
  about Gemini Code Assist, see [Gemini Code Assist
  overview](https://cloud.google.com/gemini/docs/codeassist/overview).

## Quotas and pricing

This section describes the quotas and pricing structure for
Gemini in Firebase.

### Quotas and limits

You can view your current quotas on the
[Quotas page for the Gemini for Google Cloud API](https://console.cloud.google.com/apis/api/cloudaicompanion.googleapis.com/quotas?project=_):

1. From the [Google Cloud console](https://console.cloud.google.com), select **Enabled APIs \& services**.
2. Search for, then click **Gemini for Google Cloud API**.
3. Click **Quotas \& system limits**.

Gemini for Google Cloud API quotas appear. Gemini in Firebase uses the
"Chat API requests per day per user" quota.

To request a quota increase:

1. Select the quota you want to increase, and click **Edit request**.
2. Update the **New value** text field with the quota you'd like to
   request, then click **Submit**.

   The Google Cloud team will evaluate your request and respond by email.

### Pricing

Pricing differs based on the version of Gemini in Firebase you're using and
your account type:

- For users who are not using Google Workspace (formerly G Suite), Gemini in Firebase is available at no-cost when used in the Firebase console and in IDE tooling, like the Data Connect VS Code extension for Visual Studio Code.
- Google Workspace (formerly G Suite) users are required to have a [Gemini Code Assist](https://cloud.google.com/products/gemini/code-assist) Standard or Enterprise license to access Gemini in Firebase in the Firebase console, which can be set up by your [Google Workspace
  administrator](https://cloud.google.com/gemini/docs/codeassist/request-license). Learn more at [Gemini in Firebase with a
  Gemini Code Assist license](https://firebase.google.com/docs/gemini-in-firebase/set-up-gemini#code-assist).

Terms and conditions and how your data is used vary based on the plan you use.
Learn more at [How Gemini in Firebase uses your
data](https://firebase.google.com/docs/gemini-in-firebase#how-gemini-in-firebase-uses-your-data).
