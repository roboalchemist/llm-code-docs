# AggregateRatingBuilder

public final class **AggregateRatingBuilder** extends [IndexableBuilder](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/builders/IndexableBuilder)\<[AggregateRatingBuilder](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/builders/AggregateRatingBuilder)\>  
Builder to construct an [Indexable](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/Indexable)
for aggregate rating.

The aggregate rating can only be used as a parameter of other builders, and it cannot be
passed to [FirebaseAppIndex.update(Indexable...)](https://firebase.google.com/docs/reference/android/com/google/firebase/appindexing/FirebaseAppIndex#update(com.google.firebase.appindexing.Indexable...)) directly.

For reference, see: <https://schema.org/AggregateRating>.
