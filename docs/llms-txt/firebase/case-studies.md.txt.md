Here are success stories of some Firebase Authentication integrations:

## Case Studies

### Fabulous

[![Fabulous case study](https://firebase.google.com/static/docs/auth/case-studies/fabulous.png)](https://firebase.google.com/static/docs/auth/case-studies/fabulous.pdf)

[Fabulous](https://play.google.com/store/apps/details?id=co.thefabulous.app) implemented Firebase Authentication in one afternoon, making use of
anonymous authentication, email and password based sign-in, Google Sign-in, and
Facebook Login.

### Rave

[![Rave case study](https://firebase.google.com/static/docs/auth/case-studies/rave.png)](https://firebase.google.com/static/docs/auth/case-studies/rave.pdf)

Rave ([iOS](https://itunes.apple.com/us/app/wemesh/id929775122),
[Android](https://play.google.com/store/apps/details?id=com.wemesh.android))
was able to implement Firebase Authentication in one to two days on the client
and in one hour on the server. This allowed Rave to quickly set up and run a
multi-platform login system.
