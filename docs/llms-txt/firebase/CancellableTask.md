### cancel

```
abstractÂ funÂ cancel():Â Boolean
```

Attempts to cancel the task. A canceled task cannot be resumed later. A canceled task calls back on listeners subscribed to [addOnFailureListener](https://developers.google.com/android/reference/com/google/android/gms/tasks/Task.html#addOnFailureListener-com.google.android.gms.tasks.OnFailureListener-) with an exception that indicates the task was canceled.  

|                                      Returns                                       |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html) | true if this task was successfully canceled or is in the process of being canceled. Returns false if the task is already completed or in a state that cannot be canceled. |
