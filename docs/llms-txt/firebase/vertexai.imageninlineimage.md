# ImagenInlineImage interface

> This API is provided as a preview for developers and may change based on feedback that we receive. Do not use this API in a production environment.

An image generated by Imagen, represented as inline data.

**Signature:**  

    export interface ImagenInlineImage

## ImagenInlineImage.mimeType

> This API is provided as a preview for developers and may change based on feedback that we receive. Do not use this API in a production environment.

The MIME type of the image; either `"image/png"` or `"image/jpeg"`.

To request a different format, set the `imageFormat` property in your [ImagenGenerationConfig](https://firebase.google.com/docs/reference/js/vertexai.imagengenerationconfig.md#imagengenerationconfig_interface).

**Signature:**  

    mimeType: string;
