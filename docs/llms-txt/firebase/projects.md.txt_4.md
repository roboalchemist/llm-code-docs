# REST Resource: projects

## Resource

There is no persistent data associated with this resource.

| ## Methods ||
|---|---|
| ### `https://firebase.google.com/docs/reference/rest/storage/rest/v1alpha/projects/deleteDefaultBucket` | Unlinks and deletes the default bucket for the specified Firebase project. |
| ### `https://firebase.google.com/docs/reference/rest/storage/rest/v1alpha/projects/getDefaultBucket` | Gets the default bucket for the specified Firebase project. |
