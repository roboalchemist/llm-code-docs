# REST Resource: availableProjects

## Resource

There is no persistent data associated with this resource.

|                                                                                                                                           ## Methods                                                                                                                                            ||
|----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ### [list](https://firebase.google.com/docs/reference/firebase-management/rest/v1beta1/availableProjects/list) | Lists each [Google Cloud `Project`](https://cloud.google.com/resource-manager/reference/rest/v1/projects) that can have Firebase resources added and Firebase services enabled. |
