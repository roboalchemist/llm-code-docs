# Exercise 1.4: Changing welcome message

To change the welcome message you can click the Greeting in the built-in skills section.&#x20;

Once you are in the greeting message, delete the text outpu&#x74;**,** and then add a card. In the description section of the card, add a short welcome message to your agent. For example:
