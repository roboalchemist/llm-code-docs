# Privacy

You can set your Agent's Personally Identifiable Information (PII) settings in the `Privacy` page.&#x20;

To access the privacy settings:

1. Click on your profile icon and select **Settings**.
2. Navigate to the **Privacy** section.

You can configure two predefined intents available in every agent, allowing users to gain insights into their stored PII data

![](https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-LpXFTiTgns4Ml77XGi3%2Fuploads%2FC54ZRQT4E1WwsWnlrTkR%2FScreenshot%202025-03-18%20at%2010.57.49%E2%80%AFAM.png?alt=media\&token=2c259c98-3554-456a-b1b9-1d13b5201429)

You can view the default system PII entities under the `Default system PII entities` tab, which lists all the system entities that are masked by default.

You can request the PII data under the `PII data request` tab.

<figure><img src="https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-LpXFTiTgns4Ml77XGi3%2Fuploads%2FNWsvaO3vcY01rHVgbEZg%2FScreenshot%202025-03-18%20at%2011.14.21%E2%80%AFAM.png?alt=media&#x26;token=85ce9e96-4569-4558-b545-9b5f64cb1fca" alt=""><figcaption></figcaption></figure>

1. Click `Request` to export user queries containing PII data. You can specify the list of users for whom this request applies, fill in all the required details, and click `Submit`.

<figure><img src="https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-LpXFTiTgns4Ml77XGi3%2Fuploads%2FPSnJtwFoWRaKQIE6Bw92%2FScreenshot%202025-03-18%20at%2011.11.57%E2%80%AFAM.png?alt=media&#x26;token=eccc2bac-db04-4b3e-83cc-e696d79b919b" alt=""><figcaption></figcaption></figure>

2. Click `Request` to delete user queries containing PII data. Fill in all the required details, then click `Submit`**.**
