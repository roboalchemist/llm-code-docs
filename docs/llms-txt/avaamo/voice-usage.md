# Voice Usage

The `Voice Usage` feature helps you to track the anticipated voice billing usage associated with the companyâ€™s license configuration. It allows you to track monthly incoming and outgoing calls on your account and export usage reports for further analysis.

Click your `Profile` icon at the top-right corner, Navigate to `Settings -> Usage Reports` to access the `Voice Usage` option.&#x20;

### How to use this page?

{% hint style="info" %}
**Notes**:&#x20;

* The `Voice Usage` page is available only for users with the `Settings` role.
* All the data in the page is captured in the UTC timezone.
  {% endhint %}

<figure><img src="https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-LpXFTiTgns4Ml77XGi3%2Fuploads%2FKm0lhfYrcOHzQ1r9ufPk%2FScreenshot%202025-02-24%20at%203.06.37%E2%80%AFPM.png?alt=media&#x26;token=427a13f0-e5ff-417e-a7c5-ddf298dbd199" alt=""><figcaption></figcaption></figure>

* You can view voice usage data for the past 12 months. To export data in a CSV format, use the `Export All` option in the top-right corner of the window.
* Data for each month with segment and message summary is displayed. Click the month to view the complete history of Voice usage for each day of the month. C The following information is displayed in the detailed view for each date:
  * **Incoming IVR calls**
  * **Outgoing IVR calls**
* Click `Export` to export the specific month's data in a CSV format.
