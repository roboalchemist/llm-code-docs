# Exercise 6.1: Analytics Dashboard

In the Agent Configuration menu, click Monitor to understand and view the analytics of the agent.&#x20;

* To find your top intents in the Agent, navigate to **Agent-> Monitor -> Analytics**

![](https://lh5.googleusercontent.com/N6_6-rbF7LH1pIBCrSSIP1XCQAEf28lYcLDZIHokwMonmPBTdm_esMwizoEnmozm4TUuabXoq54ETmL5l1pvXQRcoH420pS0mi6McGVSi95T3YkPGWG-hvxmbRWVNZpfkG-fOPsA)

* On this dashboar&#x64;**,** you will see different conversation metrics associated with the agent. You can see the total queries, Active users, Channels, User Feedback, Top dialog skills, and top Q\&A intent.
* Observe the analytics for your agent and identify the top intents.
