# View user information

Live agents can click the information icon next to the user name at the top to view the user details.

<figure><img src="https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-LpXFTiTgns4Ml77XGi3%2Fuploads%2Fk9oom83aNnV1LWOAObZ2%2Fimage.png?alt=media&#x26;token=40385f64-26dd-400a-b168-a0970f2331a2" alt=""><figcaption></figcaption></figure>

Live agents can view all the following details:

* Channel of communication
* Agent Name
* All user property information such as first name, last name, or any custom user properties that are available and set for the user. This provides additional details about the user to the live agents. Live agents can use these details to personalize their interactions with users.

{% hint style="success" %}
**Key points**: The following are some of the ways Avaamo Conversational AI Platform allows collecting user information:

* Using the [Collect user information toggle](https://docs.avaamo.com/user-guide/how-to/build-agents/configure-agents/deploy/web-channel/configure-web-channel#collect-user-information) in the Web channel
* Setting the user properties in the JS code. See [User.setProperty](https://docs.avaamo.com/user-guide/how-to/build-skills/create-skill/customize-your-skill/reference-library/user.setproperty) or [User.setProperties](https://docs.avaamo.com/user-guide/how-to/build-skills/create-skill/customize-your-skill/reference-library/user.setproperties), for more information.
