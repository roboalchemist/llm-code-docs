# Exercise 6.3: Query Insights

The query insights help to give you a closer look into the conversations that users have with your agent.&#x20;

* To navigate to Query Insights navigate to **Agent-> Monitor -> Query Insights**.

![](https://2934665269-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2Fv5%2F-MCQ5cA57CNCjlqZsBu8%2F-MCQ9svAPdPrcsRw6e7n%2F23.jpeg?generation=1594962461619755\&alt=media)

* Select a date range, the agent, and filter the conversations to see each query and its associated insight in detail.
