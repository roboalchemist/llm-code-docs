# New Tutorial: Using Custom Objects with the API

We published a  step-by-step tutorial on using the  API to create and maintain custom Objects! You'll learn how to:

* Create and maintain Custom Object Schemas
* Add and maintain records for your schemas
* Start automations based on new or updated Custom Objects
* Send emails that include Custom Object record data
