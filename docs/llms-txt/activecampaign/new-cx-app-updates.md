# New CX App Commands

ActiveCampaign now supports validating webhooks, as well as many other commands including:

### `!_comment` [Leave comments in your CX App](https://developers.activecampaign.com/docs/commands-1#_comment-command-a-namecommenta)

### `verification` [Validate webhooks if required by a third party](https://developers.activecampaign.com/docs/data_intake#verification-object-optional-a-namedataintake-verificationa)

### `!sha`[Encrypt your data](https://developers.activecampaign.com/docs/commands-1#sha-command-a-nameshaa)

### `!respond` [Define the HTTP response expected by third-party system](https://developers.activecampaign.com/docs/commands-1#respond-command-a-nameresponda)
