# Variables

Use variables when messaging specific mailing lists.

Variables are used for global/universal content across templates and messages. Variables are assigned to individual lists.

**Example:**
A sports league places each player in a mailing list specific to that player's team. A variable with the tag team\_name could be created, so any email to the players the that team/mailing list would contain that specific team's name.

<Callout icon="🚧">
  **Management limitation for locked variables**

  In ActiveCampaign, you can lock variables to prevent unauthorized editing or being overwritten by third-party integrations. Only admin users can lock or unlock a message variable.
</Callout>
