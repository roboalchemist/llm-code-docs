# Is it GPT-4o?

Wondering which AI model you're chatting with on Bito? Let's clear it up.

Bito AI Chat uses the most capable AI models available, including GPT-4o, Claude Sonnet 3.5, DeepSeek-V3, o3-mini, and similar cutting-edge models. These advanced models are great for tasks that require long, complex prompts and advanced reasoning.

Bito automatically selects the relevant AI models to provide you with the best possible responses for your queries.

At Bito, we use OpenAI's API to integrate both GPT-4o and o3-mini AI models.

If you ask the OpenAI GPT-4o API which model it is, it will still say GPT-3 because GPT-4o was trained using information only up to 2021. So, it might not always recognize itself as GPT-4o. This is a small issue on their part, not ours.

As a result, when you chat with Bito, the chatbox might still display "GPT-3" even though you're using GPT-4o or other advanced models.

You can read more about this issue on the official OpenAI's community forum.
