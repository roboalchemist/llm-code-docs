# slowTestThreshold&#x20;

* **Type**: `number`
* **Default**: `300`
* **CLI**: `--slow-test-threshold=<number>`, `--slowTestThreshold=<number>`

The number of milliseconds after which a test or suite is considered slow and reported as such in the results.
