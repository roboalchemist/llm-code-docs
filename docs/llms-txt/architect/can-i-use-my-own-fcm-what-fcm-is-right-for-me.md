# Can I use my own FCM? What FCM is right for me?

If you have an existing FCM relationship that you want to use with Architect, please [contact-us](https://docs.architect.co/user-guide/contact-us "mention") and we will explore the available options! We are often able to facilitate this.&#x20;

If you're creating a new account, the onboarding process will guide you through the FCM account opening process.&#x20;
