# Microsoft Excel Add-In

For users of the Architect trading platform who want access to trading functionality via Excel, we provide an Excel Add-In. This add-in also allows Architect users to connect and see prices, positions, balances, and PnL.

Users should already have an account with Architect, along with an API key and secret.
