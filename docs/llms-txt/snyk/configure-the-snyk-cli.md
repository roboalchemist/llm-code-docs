# Configure the Snyk CLI

You can use [environment variables](https://docs.snyk.io/developer-tools/snyk-cli/configure-the-snyk-cli/environment-variables-for-snyk-cli) to configure the Snyk CLI.

You can also set variables to configure the Snyk CLI to [connect to the Snyk API](https://docs.snyk.io/developer-tools/snyk-cli/configure-the-snyk-cli/configure-snyk-cli-to-connect-to-snyk-api).

You can [configure Snyk CLI to be used behind a Proxy](https://docs.snyk.io/developer-tools/snyk-cli/configure-the-snyk-cli/proxy-configuration-for-snyk-cli).
