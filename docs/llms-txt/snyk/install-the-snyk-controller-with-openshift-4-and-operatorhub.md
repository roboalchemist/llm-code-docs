# Install the Snyk Controller with OpenShift 4 and OperatorHub

The Snyk Controller V2 is currently not available in the OpenShift Marketplace nor available as a community version.

You can still install Snyk Controller V2 in OpenShift using the Helm install instructions. See [Install the Snyk Controller with Helm (Azure and Google Cloud Platform)](https://docs.snyk.io/scan-with-snyk/snyk-container/kubernetes-integration/install-the-snyk-controller/install-the-snyk-controller-with-helm-azure-and-google-cloud-platform).
