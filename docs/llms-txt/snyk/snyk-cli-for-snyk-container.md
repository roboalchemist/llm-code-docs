# Snyk CLI for Snyk Container

{% hint style="info" %}
To use the Snyk CLI, ensure you [install ](https://docs.snyk.io/developer-tools/snyk-cli/install-or-update-the-snyk-cli)and [authenticate](https://docs.snyk.io/developer-tools/snyk-cli/commands/auth).
{% endhint %}

The Snyk Container Command Line Interface or [Snyk CLI](https://docs.snyk.io/developer-tools/snyk-cli) helps you find and fix vulnerabilities in container images on your local machine.

To use Snyk Container from the CLI, see the following pages:

* [Scan and monitor images](https://docs.snyk.io/developer-tools/snyk-cli/scan-and-maintain-projects-using-the-cli/snyk-cli-for-snyk-container/scan-and-monitor-images)
* [Understand Snyk Container CLI results](https://docs.snyk.io/developer-tools/snyk-cli/scan-and-maintain-projects-using-the-cli/snyk-cli-for-snyk-container/understand-snyk-container-cli-results)
* [Advanced use of Snyk Container CLI](https://docs.snyk.io/developer-tools/snyk-cli/scan-and-maintain-projects-using-the-cli/snyk-cli-for-snyk-container/advanced-use-of-snyk-container-cli)

For more information, see [Detect application vulnerabilities in container images](https://docs.snyk.io/scan-with-snyk/snyk-container/use-snyk-container/detect-application-vulnerabilities-in-container-images).
