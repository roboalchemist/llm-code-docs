# Quickstart guides

This section lists quickstart guides for common AI agents. Snyk supports any AI agent and ADE that integrates with a local MCP server.

For a general quickstart guide with steps that apply to any coding assistant, visit [Getting started with Snyk Studio](https://docs.snyk.io/integrations/snyk-studio-agentic-integrations/getting-started-with-snyk-studio).
