# Usage analytics

Snyk provides Snyk Studio usage analytics in the following report types:

* [Snyk Studio Adoption](https://docs.snyk.io/manage-risk/analytics/redesigned-analytics#snyk-studio-adoption)
* [Developer IDE and CLI usage](https://docs.snyk.io/manage-risk/reporting/available-snyk-reports#developer-ide-and-cli-usage)

Select the report name for more details on the report type.

{% hint style="warning" %}
Data is only be available for Agentic scans with CLI version 1.1298.1 or later.
{% endhint %}
