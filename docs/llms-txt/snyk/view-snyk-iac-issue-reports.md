# View Snyk IaC issue reports

Set the **Issue Type** filter on [Snyk reports](https://docs.snyk.io/manage-risk/reporting/legacy-reports) to **Configuration** to view issues in your IaC configuration files.

<figure><img src="https://2533899886-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MdwVZ6HOZriajCf5nXH%2Fuploads%2FV2p8Yr3wRpLedkHKHxfp%2FIssue%20type%20filter%20-%20Configuration.png?alt=media&#x26;token=f25dac7e-3d8b-4975-8cd1-adb9cadc9b82" alt=""><figcaption><p>The <strong>Issue type</strong> filter set to <strong>Configuration</strong>.</p></figcaption></figure>

You can also use this filter on the **Issues** page to view your IaC issues across all of your Projects.

## API access to IaC issues

You can set the **product\_name** filter for the [Export](https://docs.snyk.io/snyk-api/reference/export) API to `Snyk IaC` to retrieve a list of IaC issues on your Org or Group.
