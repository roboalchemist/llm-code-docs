# Container

## Description

The `snyk container` commands test and continuously monitor container images for vulnerabilities and generate an SBOM for a container image.

For more information, see [Snyk CLI for Snyk Container](https://docs.snyk.io/snyk-cli/scan-and-maintain-projects-using-the-cli/snyk-cli-for-snyk-container)

## `snyk container` commands and the help docs

The `snyk container` commands are listed here with the help options:

* [`container test`](https://docs.snyk.io/developer-tools/snyk-cli/commands/container-test), `container test --help`: tests for any known vulnerabilities
* [`container monitor`](https://docs.snyk.io/developer-tools/snyk-cli/commands/container-monitor), `container monitor --help`: captures the container image layers and dependencies and monitors for vulnerabilities on [snyk.io](https://snyk.io)
* [`container sbom`](https://docs.snyk.io/developer-tools/snyk-cli/commands/container-sbom), `container sbom --help`: generates an SBOM for a container image
