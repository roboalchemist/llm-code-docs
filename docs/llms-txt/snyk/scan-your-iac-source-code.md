# Scan your IaC source code

Using Snyk IaC, you can:

* [Scan Terraform files](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/scan-terraform-files)
* [Scan CloudFormation files](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/scan-cloudformation-files)
* [Scan Kubernetes configuration files](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/scan-kubernetes-configuration-files)
* [Scan ARM configuration files](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/scan-arm-configuration-files)
* [Scan Serverless files](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/scan-serverless-files)

This section also has a list of [supported IaC and cloud providers](https://docs.snyk.io/scan-with-snyk/snyk-iac/supported-iac-languages-cloud-providers-and-cloud-resources) and instructions for [disabling IaC scans per Organization](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code/disable-iac-scans-per-organization-current-iac).
