# Group and Organization settings

## Group settings

At the Group level, select **Settings** to manage Group settings:

<figure><img src="https://2533899886-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MdwVZ6HOZriajCf5nXH%2Fuploads%2Fgit-blob-82ead02ec6a78747be232b51b927c605df4bf7e7%2FGroup%20Settings.png?alt=media" alt=""><figcaption></figcaption></figure>

From the Group settings menu, select **Settings** and then select one of the Group settings options:

* **General**: See [Group general settings](https://docs.snyk.io/snyk-platform-administration/groups-and-organizations/groups/group-general-settings) for details.
* **SSO**: See [Self-Serve Single Sign-On (SSO)](https://docs.snyk.io/implementation-and-setup/enterprise-setup/single-sign-on-sso-for-authentication-to-snyk/configure-self-serve-single-sign-on-sso) for details.
* **Member roles**: See [Manage Member Roles](https://docs.snyk.io/snyk-platform-administration/user-roles/user-role-management) for details.
* **Service accounts**: See [Service accounts](https://docs.snyk.io/implementation-and-setup/enterprise-setup/service-accounts) for details.
* **Notifications**: See [Manage notifications](https://docs.snyk.io/snyk-platform-administration/manage-notifications) for details.
* **Usage**: See [Usage setting](https://docs.snyk.io/snyk-platform-administration/groups-and-organizations/usage-settings)s for details.
* **Your plan and billing**: Billing is through a contract and cannot be changed by using this option. Contact your sales representative or [Snyk support](https://support.snyk.io) if you have billing-related requests or issues.
* **Available plans**: Billing is through a contract and cannot be changed by using this option. Contact your sales representative or [Snyk support](https://support.snyk.io) if you have billing-related requests or issues. For more information, see the [Plans and pricing](https://snyk.io/plans/) page on the Snyk website.
* **Snyk Preview**: Preview upcoming and experimental features before they are released. See the [Snyk Preview](https://docs.snyk.io/snyk-platform-administration/snyk-preview) documentation for details.
* **Snyk IaC**: Enable rules to add checks to Snyk rules when you are scanning configuration files with the Snyk CLI. See the [IaC documentation](https://docs.snyk.io/scan-with-snyk/snyk-iac) for details.

## Organization settings

At the Organization level, select **Settings** to manage Organization settings:

<figure><img src="https://2533899886-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MdwVZ6HOZriajCf5nXH%2Fuploads%2Fgit-blob-21c8141de4fba88af1cfd2ec40ec6554b7010088%2FOrg%20Settings.png?alt=media" alt=""><figcaption></figcaption></figure>

From the Organization setting menu, select **Settings** and then select one of the Organization settings options:

* **General**: See [Organization general settings](https://docs.snyk.io/snyk-platform-administration/groups-and-organizations/organizations/organization-general-settings) for details.
* **Service accounts**: See [Service accounts](https://docs.snyk.io/implementation-and-setup/enterprise-setup/service-accounts) for details.
* **Integrations**: Set up integrations and see a list of those configured for your Organization; click the name of the configured integration to see the details. See [Integrate with Snyk](https://docs.snyk.io/integrations/integrate-with-snyk) for information about available Snyk integrations.
* **Snyk Open Source**: Enable Reachable vulnerabilities analysis and edit language settings; see [Snyk Open Source - supported languages and package managers](https://docs.snyk.io/supported-languages/supported-languages-package-managers-and-frameworks) for details.
* **Snyk Code**: Enable Snyk Code; see the [Snyk Code](https://docs.snyk.io/scan-with-snyk/snyk-code) documentation for details.
* **Snyk IaC:** Enable Snyk Iac, detecting configuration files, and rules. Select severity levels for configurations scanned. See the [IaC ](https://docs.snyk.io/scan-with-snyk/snyk-iac/scan-your-iac-source-code)documentation for details.
* **Usage**: See the [Usage settings](https://docs.snyk.io/snyk-platform-administration/groups-and-organizations/usage-settings) page for details.
* **Notifications**: See the [Manage notifications](https://docs.snyk.io/snyk-platform-administration/manage-notifications) page for details.
* **Snyk Preview**: See the [Snyk Preview page](https://docs.snyk.io/snyk-platform-administration/snyk-preview) for details.
