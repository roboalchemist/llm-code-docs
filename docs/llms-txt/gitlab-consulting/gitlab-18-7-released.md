# GitLab 18.7 Release: AI Code Suggestions, Compliance Reporting and More
<h2 id="whats-new-in-gitlab-187-enhanced-developer-experience-and-advanced-ai-capabilities">What’s New in GitLab 18.7: Enhanced Developer Experience and Advanced AI Capabilities</h2>
<p>The release of GitLab 18.7 brings a new wave of improvements, focusing on increased productivity, smarter collaboration, and enhanced security. With significant upgrades across AI-powered development, security tools, and platform efficiency, this update supports teams in shipping software faster and more securely.</p>
<h3 id="code-suggestions-now-generally-available">Code Suggestions Now Generally Available</h3>
<p>GitLab 18.7 marks the general availability of code suggestions for SaaS Premium and Ultimate customers. Developers now benefit from improved inline code completions driven by AI, optimising productivity and reducing boilerplate coding. With a focus on user privacy and enterprise-grade control, these suggestions are generated securely within GitLab’s infrastructure.</p>
<h3 id="new-issues-and-merge-requests-widgets">New Issues and Merge Requests Widgets</h3>
<p>Track progress at a glance with new issue and merge request widgets. These widgets streamline visibility across key features such as code quality, SAST, and test reports. Enable stakeholders to respond rapidly and developers to receive actionable feedback earlier in the development cycle.</p>
<h3 id="pci-dss-and-fedramp-reports-now-available">PCI DSS and FedRAMP Reports Now Available</h3>
<p>Compliance reporting gets a boost in 18.7, now supporting downloadable reports for PCI DSS and FedRAMP. This helps regulated industries easily meet audit requirements without extensive manual configuration, streamlining security validation and governance.</p>
<h3 id="fine-grained-project-access-control">Fine-Grained Project Access Control</h3>
<p>Gain more control over project visibility with customisable project access levels. Whether you need to restrict access for sensitive environments or allow broader collaboration on public projects, GitLab 18.7 adds enhanced flexibility for managing who can see and contribute to your work.</p>
<h3 id="gitlab-duo-chat-enhancements">GitLab Duo Chat Enhancements</h3>
<p>The AI-powered GitLab Duo Chat now supports questions across issues, merge requests, and snippets, providing contextual support to developers where it&rsquo;s needed most. This helps reduce context switching and brings faster answers directly within the GitLab UI.</p>
<h3 id="plan-build-and-secure--more-efficiently-than-ever">Plan, Build, and Secure — More Efficiently Than Ever</h3>
<p>From redesigned DevSecOps reports to enhanced pipeline run logs, GitLab 18.7 helps your team build secure, high-quality software at scale. Additional improvements include improved IDE integrations, more intelligent automated testing workflows, and streamlined board filtering for agile planning.</p>
