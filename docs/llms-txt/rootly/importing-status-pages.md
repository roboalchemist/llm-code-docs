# Importing Status Pages

> Import your existing status pages from StatusPage.io into Rootly for integrated incident communication management.

## Importing Status Pages

### Overview

You can import your existing status pages from statuspage.io. This requires that StatusPageIO integration has already been set up. If you have not that yet please refer to our guide [here](/integrations/status-page-io "here").

### Steps

1. Within our web experience navigate to *Configuration > Status pages*

  <img alt="Document Image" src="https://mintcdn.com/rootly/kMnLuRjX9XFq2H7j/images/integrations/status-page-io/importing-status-pages/images-1.webp?fit=max&auto=format&n=kMnLuRjX9XFq2H7j&q=85&s=a23a4e1cf162ab3aa24f5cf57a235e4d" width="827" height="210" data-path="images/integrations/status-page-io/importing-status-pages/images-1.webp" />
</Frame>

2. On the status pages configuration page click *Import from statuspage.io*

<Frame>
  <img alt="Document Image" src="https://mintcdn.com/rootly/kMnLuRjX9XFq2H7j/images/integrations/status-page-io/importing-status-pages/images-2.webp?fit=max&auto=format&n=kMnLuRjX9XFq2H7j&q=85&s=705ab7702c99c6ce8bdea4d2259f0de9" width="834" height="354" data-path="images/integrations/status-page-io/importing-status-pages/images-2.webp" />
</Frame>

3. Within the popup modal select the StatusPageIO page you want to import into Rootly. Then click import pages

<Frame>
  <img alt="Document Image" src="https://mintcdn.com/rootly/kMnLuRjX9XFq2H7j/images/integrations/status-page-io/importing-status-pages/images-3.webp?fit=max&auto=format&n=kMnLuRjX9XFq2H7j&q=85&s=b866f76c994d5a406e9b4087dc07926d" width="886" height="891" data-path="images/integrations/status-page-io/importing-status-pages/images-3.webp" />
</Frame>

4. You should see the newly imported page within the list
