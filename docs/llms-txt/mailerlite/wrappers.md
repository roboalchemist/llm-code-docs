# Wrappers

Official wrappers for MailerLite API:

* [MailerLite PHP SDK](https://github.com/mailerlite/mailerlite-api-v2-php-sdk)

Community contributions:

* [MailerLite Ruby SDK](https://github.com/jpalumickas/mailerlite-ruby) by [@jpalumickas](https://github.com/jpalumickas/)
* [MailerLite C# SDK](https://github.com/djhvscf/mailerlite-sdk-csharp) by [@djhvscf](https://github.com/djhvscf)
* [MailerLite Python SDK](https://github.com/skoudoro/mailerlite-api-python) by [@skoudoro](https://github.com/skoudoro)
* [MailerLite Node.js SDK](https://github.com/zygos/mailerlite-api-v2-node) by [@zygos](https://github.com/zygos) and [@pedrogaudencio](https://github.com/pedrogaudencio)

You are welcome to create any other SDK in your preferred language!
