# /subscribers/(:id or :email)/activity/:type

Get activity of selected subscriber by specified type (opens, clicks, etc) [Rate limited]

This endpoint is the same as [Actions of Subscriber](https://developers-classic.mailerlite.com/docs/activity-of-single-subscriber) but it allows to get actions by specified `type`.
