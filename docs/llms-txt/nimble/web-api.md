### Overview

The Nimble Web API is Nimble’s multi-purpose solution for turning any website into a ready-to-use real-time pipeline. Requiring only a URL to operate, the Nimble Web API manages every step of the data collection process from access to delivery. Customers use the Nimble Web API to collect many types of web data, including:

* [x] AI training data collection
* [x] Sales and lead enrichment
* [x] Cross-industry research data gathering
* [x] Social media sentiment analysis
* [x] and more!

### Features

The Web API is designed to be versatile and universal, and comes built-in with the following features:

* [x] [Real-time data](https://docs.nimbleway.io/nimble-sdk/nimble-web-api-functions/realtime-async-and-batch-request#example) gathering with a single API call
* [x] Flexible [delivery methods](https://docs.nimbleway.io/nimble-sdk/web-api/nimble-web-api-quick-start-guide/delivery-methods) including multiple Cloud storage providers
* [x] [Batch processing](https://docs.nimbleway.io/nimble-sdk/web-api/nimble-web-api-quick-start-guide/batch-processing) of up to 1,000 URLs in a single request
* [x] Support for user-defined custom [parsing templates](https://docs.nimbleway.io/nimble-sdk/web-api/nimble-web-api-functions/data-parsing/parsing-templates)
* [x] [Page interactions](https://docs.nimbleway.io/nimble-sdk/web-api/nimble-web-api-functions/page-interaction) that enable clicking, scrolling, form submissions, and more for particularly hard-to-reach data.
* [x] Built-in proxies thanks to native integration with [Nimble IP](https://docs.nimbleway.io/nimble-sdk/proxy-api/nimble-ip-overview) - our premium IP network.
