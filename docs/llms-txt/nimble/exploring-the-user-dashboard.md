# Exploring the User Dashboard

{% embed url="<https://www.loom.com/share/c9793fcd2d024f09a5ada6ae8b4119b7?sid=7b718304-e46d-4add-b4c0-f3872cf19f56>" %}

The Nimble Dashboard is made up of the following pages:

* [x] **Dashboard** - A general overview of your **account status**, with quick summaries of the usage across products, **top origin countries**, and **most requested domains**.
* [x] **Pipelines** - Create, disable, and archive pipelines. View each pipeline's **usage statistics** and **authentication details**, and set **budget limits** and other pipeline-level settings.
* [x] **Analytics** - Use the **Current vs Past** report to compare any time period to the one before it, or to the same period the year before. Use the **Performance Comparison** report to create and compare custom graphs for any product, pipeline, country, and domain.
* [x] **Playground** - Build **custom queries** for any Nimble product with an easy-to-use graphical interface, and run them inside the Playground to **view their output**.
* [x] **Billing** - View your balance, payment history, payment method, and active plan details.
* [x] **Account Settings** - View your account **contact details** and users, and manage **authenticated IPs.**
