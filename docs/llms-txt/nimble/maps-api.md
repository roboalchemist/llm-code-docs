## Overview

The Maps API allows users to easily search popular mapping platforms for a variety of geographically-relevant points of interest including businesses, attractions, natural features, and more. The Maps API is not a standalone API, but rather a branch of the SERP API, and uses the same endpoints to interface with users.

The Maps API make it easy for users to:

* [x] Search a geographic location for points of interest.
* [x] Collect data about a place including an address, description, image, and more.
* [x] Gather reviews left by users including a rating, review body, reviewer details, and more.

## Features

* [x] Search for places around a set of geographic coordinates using text queries
* [x] Collect and structure information about relevant places
* [x] Collect and structure review data about relevant places
* [x] Easily handle pagination through search and review pages
