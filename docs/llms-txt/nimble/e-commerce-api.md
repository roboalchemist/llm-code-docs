### Overview

The Nimble E-commerce API was designed to help users quickly and easily access product data from the web’s largest online markets. The E-commerce API comes built-in with all the tools needed to access, collect, and deliver detailed product data, and needs only a simple product URL to begin. Customers use the E-commerce API to:

* [x] Aggregate and compare product data between marketplaces
* [x] Monitor changes in a product's price over time
* [x] Compare product information across locations, time, or other variables
* [x] Collect product reviews for sentiment analysis
* [x] and more!

### Features

The E-commerce API is designed to provide fast yet granular access to product data and supports:

* [x] Real-time data gathering for [products](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/e-commerce-api/real-time-ecommerce-request) and [product searches](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/broken-reference)
* [x] Flexible [delivery methods](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/e-commerce-api/delivery-methods) including multiple Cloud storage providers
* [x] [Batch processing](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/e-commerce-api/batch-processing) of up to 1,000 URLs in a single request
* [x] AI-powered parsing for granular data extraction
* [x] Built-in proxies thanks to native integration with [Nimble IP](https://docs.nimbleway.io/nimble-sdk/proxy-api/nimble-ip-overview) - our premium IP network.
