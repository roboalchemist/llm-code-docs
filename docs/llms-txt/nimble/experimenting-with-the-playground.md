# Experimenting with the Playground

{% embed url="<https://www.loom.com/share/5994b4c1dc874184a41c45e67eb2f168?sid=63de2659-80f7-473c-9561-20defbc8233c>" %}

The Nimble Playground was designed to help both new and veteran Nimble users learn and experiment with our products.&#x20;

It includes an easy-to-use and accessible user interface that walks you through the settings and parameters available for each product, and writes sample request code in real time based on the selected settings.

In many cases, the playground also supports running the request inside the interface, and displays a graphical and a printout of the output.

Requests made through the Playground are also free of charge, and don't count towards your existing quota so you can feel free to learn and test our platform!
