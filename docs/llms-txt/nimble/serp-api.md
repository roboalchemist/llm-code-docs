## Overview

The Nimble SERP API makes collecting search engine data a breeze by providing a specialized solution that streamlines data gathering from SERPs. With little more than a search phrase, the SERP API returns detailed search engine data and internally manages the entire data collection process. Customers use the SERP API for many uses cases, including:

* [x] Monitoring their search engine rankings
* [x] Keeping abreast of changes in their competitors' SEO strategies
* [x] Optimizing their SEM strategy and spend with real-time data
* [x] Discovering new keywords, search terms, and competitive spaces
* [x] and more!

## Features

* [x] Gather updated SERP listings for search terms in [real-time](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/serp-api/real-time-search-request)
* [x] Flexible [delivery methods](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/serp-api/delivery-methods) including multiple Cloud storage providers
* [x] [Batch processing](https://docs.nimbleway.io/nimble-sdk/web-api/vertical-endpoints/serp-api/batch-processing) of up to 1,000 URLs in a single request
* [x] AI-powered parsing for granular data extraction
* [x] Built-in proxies thanks to native integration with [Nimble IP](https://docs.nimbleway.io/nimble-sdk/proxy-api/nimble-ip-overview) - our premium IP network.
