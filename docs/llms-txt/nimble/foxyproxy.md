# FoxyProxy

### About FoxyProxy

FoxyProxy is a simple and free browser extension that lets you easily use Nimble IP on Firefox or Chrome. While setting up your proxy server connection can be done without an extension, FoxyProxy provides a simple and flexible interface as well as additional options that browsers lack.

### Installing FoxyProxy

Installing FoxyProxy is very straightforward. Simply follow the steps on your browser of choice:

[Chrome download page](https://chrome.google.com/webstore/detail/foxyproxy-standard/gcknhkkoolaabfmlnjonogaaifnjlfnp)

[Firefox download page](https://addons.mozilla.org/en-US/firefox/addon/foxyproxy-standard/)

### Chrome setup

1. After downloading, click the extensions button on the upper right of Chrome and pin the extension to the toolbar

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/8gAokrnuUrEsv4o629IW/Foxyproxy%20-%20Nimble%20IP%20-%201.webp" alt=""><figcaption></figcaption></figure>

2\. Next, click the FoxyProxy icon to open the quick activation menu, and select “Options” to modify the proxy settings.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/KBd5BIY98P4q4Momyvo5/Foxyproxy%20-%20Nimble%20IP%20-%202.png" alt=""><figcaption></figcaption></figure>

3\. On the left navigation menu, go to “Proxies”.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/953nEJrCGaSgwl5urRQ4/Foxyproxy%20-%20Nimble%20IP%20-%203.png" alt=""><figcaption></figcaption></figure>

4\. On the options menu to the right, select “Add New Proxy”.

5\. In the Proxy Settings dialog that opens, enter the following credentials:&#x20;

&#x20;    a. Host ⇒ **ip.nimbleway.com**

&#x20;    b. Port ⇒ **7000**

&#x20;    c. Check the option to “Save Login Credentials” (click ‘OK’ on the message that pops up, explaining how FoxyProxy handles usernames and passwords).

&#x20;    d. Username ⇒ Nimble’s pipeline ***username string***

&#x20;    e. Password ⇒ Nimble’s pipeline ***password***

{% hint style="info" %}
Your Nimble account manager will provide you with the username and password for your default pipeline.
{% endhint %}

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/pJOQd1ILTGfJAOzQW7sy/Foxyproxy%20-%20Nimble%20IP%20-%204.png" alt=""><figcaption></figcaption></figure>

6\. Go to the “General” tab and set a name for your Proxy (we used Nimble).

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/en0WFCi5lZoLOeqbgLLI/Foxyproxy%20-%20Nimble%20IP%20-%205.png" alt=""><figcaption></figcaption></figure>

7\. There are two ways to enable the proxy,

&#x20;    a. From the FoxyProxy options page, change the proxy mode from “Disable FoxyProxy” to “Use proxy *yourProfile* for all URLs.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/MWb2XRVV66w4KS3cAW2G/Foxyproxy%20-%20Nimble%20IP%20-%206.webp" alt=""><figcaption></figcaption></figure>

&#x20;    b. Alternatively, you can use FoxyProxy’s quick menu to activate the proxy from any page. Click on the FoxyProxy icon in the toolbar, and then select your proxy profile.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/ZSFUf5rQSKKCwh0FyEgB/Foxyproxy%20-%20Nimble%20IP%20-%207.webp" alt=""><figcaption></figcaption></figure>

### Firefox setup

1. After downloading the add-on, Firefox will automatically add a shortcut to the toolbar. Simply click the FoxyProxy icon to open the quick activation menu, and then click “Options” to modify the proxy settings.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/7EAEgX7oEa5SHrzxDv9n/Foxyproxy%20-%20Nimble%20IP%20-%208.png" alt=""><figcaption></figcaption></figure>

2\. On the left navigation menu, click “Add” to create a new proxy profile.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/wrf8zdc1nOaX2J1y9ta4/Foxyproxy%20-%20Nimble%20IP%20-%209.webp" alt=""><figcaption></figcaption></figure>

3\. In the “Add Proxy” page, set a name for this proxy setup (we used Nimble), then enter the following details:

&#x20;    a. Proxy IP address or DNS name ⇒ ip.nimbleway.com

&#x20;    b. Port ⇒ 7000

&#x20;    c. Username ⇒ Nimble’s pipeline ***username string***

&#x20;     d. Password ⇒ Nimble’s pipeline ***password***

&#x20;     e. Finally, click save.<br>

{% hint style="info" %}
Your Nimble account manager will provide you with the username and password for your default pipeline.
{% endhint %}

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/FO32FAzspQCR25v1GUUl/Foxyproxy%20-%20Nimble%20IP%20-%2010.webp" alt=""><figcaption></figcaption></figure>

4\. There are two ways to enable the proxy:

&#x20;    a. From the FoxyProxy options page, change “Turn Off (Use Firefox Settings)” to the title you set for your proxy profile (we used “Nimble”).

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/zWhmPyP1Yh2UPM2ZsULS/Foxyproxy%20-%20Nimble%20IP%20-%2011.webp" alt=""><figcaption></figcaption></figure>

&#x20;    b. Alternatively, you can use FoxyProxy’s quick menu to activate the proxy from any page. Click on the FoxyProxy icon in the toolbar, and then select your proxy profile.

<figure><img src="https://content.gitbook.com/content/ZNy4aqrBN53tR8DpTEuW/blobs/uDwObPAyZ0ZYubQQHX1r/Foxyproxy%20-%20Nimble%20IP%20-%2012.png" alt=""><figcaption></figcaption></figure>

You can test that the proxy is working correctly by clicking on “What’s My Ip?”. On the resulting page, the displayed location will differ from your actual location, confirming that Nimble IP is channeling your requests.
