# Tools and Integrations

<details>

<summary>Does Nimble IP work with mobile devices?</summary>

Yes! To learn how to configure Android and iOS devices to use Nimble IP, check out our [integration guides](https://docs.nimbleway.io/nimble-sdk/proxy-api/integration-guides).

</details>

<details>

<summary>Can I use Nimble IP with 3rd party apps?</summary>

Nimble IP was designed to be fully compatible with the proxy management tools available on the market. We support both username/password and IP whitelist authentication for compatibility with any use case.

Be sure to see our [Integration Guides](https://docs.nimbleway.io/nimble-sdk/proxy-api/integration-guides) page for some examples on integrating Nimble IP with 3rd party apps.

</details>

<details>

<summary>Can I use Nimble IP on multiple devices?</summary>

Yes. Nimble IP works on a per-request basis, and there is no limit on the number of devices being used.

</details>

<details>

<summary>Which ports and protocols does Nimble IP support?</summary>

We currently support HTTP ports 80 and 443, and we’re working on adding SOCKS5 support.

</details>

<details>

<summary>How many sessions can be opened simultaneously?</summary>

Nimble’s infrastructure is designed to handle any number of threads, so there’s no limit on the number of sessions running at the same time.

</details>
