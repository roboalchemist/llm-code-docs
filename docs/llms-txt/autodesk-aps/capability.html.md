<h1>API Capabilities</h1>
<img alt="../../../_images/capabilities.png" src="../../../_images/capabilities.png">
<p>Using the AEC Data Model API, you can perform the following actions:</p>
<ul class="simple">
<li>Retrieve all hubs that match a specific criterion, such as the name of the hub that you want. Only exact matches are currently supported.</li>
<li>Retrieve a hub using a hub ID.</li>
<li>Retrieve all projects that match the filter criteria from a specified hub.</li>
<li>Retrieve a project using a project ID from a specified hub.</li>
<li>Retrieve all top-level folders within a specified project.</li>
<li>Retrieve all subfolders within a specified folder that meet the filter criteria.</li>
<li>Retrieve a folder using the folder ID.</li>
<li>Retrieve the latest elementGroup data based on elementGroup ID.</li>
<li>Retrieve elementGroup by version number and elementGroup ID.</li>
<li>Retrieve elementGroups in the given hub using additional RSQL filters, if provided.</li>
<li>Retrieve elementGroups in the given project using additional RSQL filters, if provided.</li>
<li>Retrieve elementGroups in the given folder using additional RSQL filters, if provided.</li>
<li>Retrieve element using element ID.</li>
<li>Retrieve elements from given hub using additional RSQL filters, if provided.</li>
<li>Retrieve elements from given project using additional RSQL filters, if provided.</li>
<li>Retrieve elements from given folder using additional RSQL filters, if provided.</li>
<li>Retrieve elements from given elementGroup using additional RSQL filters, if provided.</li>
<li>Retrieve elements from given elementGroup and version by using additional RSQL filters, if provided.</li>
<li>Retrieve distinct property values for ElementGroups by ID and name. For more information, refer <a class="reference external" href="en/docs/aecdatamodel/v1/tutorials/tutorial-02/distinctvaluesquery/">Get Distinct Values of Properties</a>.</li>
</ul>
<p>For detailed information on the supported queries, objects, and inputs, refer to the <a class="reference external" href="en/docs/aecdatamodel/v1/reference/queries/">Reference Guide</a> section.</p>
