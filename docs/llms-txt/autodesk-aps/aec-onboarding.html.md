<h1>Onboarding to AEC Data Model</h1>
<p>This topic outlines the requirements to get started with onboarding. If you have an Autodesk Docs subscription, you can enable your ACC account to extract data to AEC Data Model at no additional cost. Access to this functionality is being rolled out and will be available in the account settings in the coming days for you to enable.</p>
<section id="activate-your-acc-account-to-extract-data-to-aec-data-model">
<a href="#activate-your-acc-account-to-extract-data-to-aec-data-model"><h2>Activate your ACC Account to extract data to AEC Data Model</h2></a>
<p>Enable your ACC account for AEC Data Model by performing the following steps:</p>
<ol class="arabic">
<li>Log on to the <a class="reference external" href="https://acc.autodesk.com/projects">ACC web site</a> .</li>
<li>You can access the <strong>Account Admin Settings</strong> page in the following two ways:</p>
<ul>
<li>From the ACC Home page, click <strong>Account Admin</strong>.</p>
<blockquote>
<div><img alt="../../../_images/acc-admin1.png" src="../../../_images/acc-admin1.png">
<p>Click <strong>Settings</strong> to access the <strong>Account Admin Settings</strong> page.</p>
<img alt="../../../_images/acc-admin4.png" src="../../../_images/acc-admin4.png">
</div></blockquote>
</li>
<li>From the ACC Docs page, if you have access to ACC Docs, click <strong>Settings</strong> to access the <strong>Account Admin Settings</strong> page.</p>
<blockquote>
<div><img alt="../../../_images/acc-admin2.png" src="../../../_images/acc-admin2.png">
</div></blockquote>
</li>
</ul>
</li>
<li>On the <strong>Account Admin Settings</strong> page, locate the <strong>Activate</strong> button. It is not in the activated state by default.</p>
<blockquote>
<div><img alt="../../../_images/acc-activ-switch.png" src="../../../_images/acc-activ-switch.png">
</div></blockquote>
</li>
<li>Click the <strong>Activate</strong> button to enable the AEC Data Model capabilities for your account.</p>
<blockquote>
<div><img alt="../../../_images/acc-switch-enabled.png" src="../../../_images/acc-switch-enabled.png">
</div></blockquote>
</li>
</ol>
