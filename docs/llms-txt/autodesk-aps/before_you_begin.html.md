<section id="step-1-prerequisites">
<a href="#step-1-prerequisites"><h2>Step 1: Prerequisites</h2></a>
<blockquote>
<div><ul class="simple">
<li>Make sure that you have enabled your ACC Account for extraction to the AEC Data Model. For more information, refer <a class="reference external" href="/en/docs/aecdatamodel/v1/developers_guide/onboarding">AEC Data Model Onboarding</a>.</li>
<li>Make sure you are working with Revit files that are version 2024 or later.</li>
<li>Make sure that you have an Autodesk Docs license and are using the AMER, EMEA or AUS instance of your Autodesk Construction Cloud account.</li>
<li>Make sure you have access to the necessary files in your Autodesk Construction Cloud project.</li>
<li>Make sure you select the <strong>AEC Data Model API</strong> and <strong>Data Management API</strong> services to use the AEC Data Model API.</li>
</ul>
</div></blockquote>
</section>
