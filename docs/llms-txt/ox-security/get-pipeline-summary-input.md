# getPipelineSummaryInput

Input parameters for retrieving pipeline summary data with filtering and pagination.

### Examples

```graphql
input GetPipelineSummaryInput {
  limit: Int
  offset: Int
  owners: [String]
  tagIds: [String]
  dateRange: DateRange
  filters: PipelineSummaryFilters
  openItems: [FilterTypes]
}
```
