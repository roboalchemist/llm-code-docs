# cicdIssuesInput

Defines the input parameters for querying CI/CD issues.

### Examples

```graphql
input CICDIssuesInput {
  search: [AutoCompleteSearch]
  scanID: String
  limit: Int!
  offset: Int!
  filters: CICDIssueFilters
  sort: CICDIssuesSort
  owners: [String]
  tagIds: [String]
  inventoryFilters: [InventoryTypes]
  conditionalFilters: [ConditionalFilters]
  openItems: [FilterTypes]
  dateRange: DateRange
  limitAggItems: Int
  offsetAggItems: Int
  topOffset: Int
  issueId: String
  scrollDirection: String
  topLevelSearch: String
}
```
