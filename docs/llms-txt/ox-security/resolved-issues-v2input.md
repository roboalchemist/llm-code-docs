# resolvedIssuesV2input

Input for resolved issues V2 with conditional filters support.

### Examples

```graphql
input ResolvedIssuesV2Input {
  scanID: String
  limit: Int!
  offset: Int!
  filters: IssueFilters
  sort: RIssuesSort
  owners: [String]
  tagIds: [String]
  dateRange: DateRange
  search: [AutoCompleteSearch]
  openItems: [FilterTypes]
  conditionalFilters: [ConditionalFilters]
  topOffset: Int
  issueId: String
  scrollDirection: String
  restrictToCategories: [Int]
  topLevelSearch: String
}
```
