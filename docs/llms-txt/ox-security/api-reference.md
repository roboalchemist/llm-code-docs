# API Reference

The **OX Security API** provides developers with secure, programmatic access to OX Security’s platform, enabling seamless integration with your existing security workflows.

With this API you can:

* **Automate security operations:** Retrieve and manage security findings, vulnerabilities, and risk insights.
* **Enforce security policies:** Set up and monitor compliance rules across your development pipeline.
* **Integrate with third-party tools:** Connect OX Security with CI/CD pipelines, ticketing systems, and other security solutions.
