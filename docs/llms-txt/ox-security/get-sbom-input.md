# getSbomInput

Defines input parameters for retrieving SBOM libraries with enhanced filtering and pagination support.

### Examples

```graphql
input GetSbomInput {
  scanId: String
  conditionalFilters: [ConditionalFilters]
  search: String
  limit: Int
  offset: Int
  sort: SbomLibSortInput
  owners: [String]
  tagIds: [String]
  forCSV: Boolean
  appPage: Boolean
  cursorValue: String
}
```
