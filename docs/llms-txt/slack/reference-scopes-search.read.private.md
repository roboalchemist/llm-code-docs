# search:read.private scope

Search a workspace's content in private channels

## Facts

## Supported token types

[`User`](/authentication/tokens#user)

## Usage info {#usage-info}

This scope allows [apps using AI features](/ai/) to query messages and files in private channels to help them answer user queries. Users must consent to use within the Slack client before the API can search with this scope, and users may revoke consent after it has been initially given.
