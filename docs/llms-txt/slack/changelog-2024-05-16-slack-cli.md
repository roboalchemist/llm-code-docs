# Release: Slack CLI v2.24.0

May 16, 2024

Version `2.24.0` of the developer tools for the Slack automations platform has landed!

* We've renamed the `function distribute` command to `function access` (however, the `distribute` subcommand has been added as an alias so you can still use `function distribute`). Refer to [custom function access](/tools/deno-slack-sdk/guides/creating-custom-functions#access) for more details.

* We've fixed several bugs that address some messaging, formatting, and emoji in the CLI for easier reading.
