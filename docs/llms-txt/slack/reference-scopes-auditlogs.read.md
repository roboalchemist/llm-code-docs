# auditlogs:read scope

View events from all workspaces, channels and users (Enterprise orgs only)

## Facts

## Supported token types

[`User`](/authentication/tokens#user)

## Usage info {#usage-info}

This scope enables the [Audit Logs API](/admins/audit-logs-api/) for an [Enterprise](/enterprise) organization.

Apps requesting this permissions **must** be installed by the Owner of an Enterprise organization.
