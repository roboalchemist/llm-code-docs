# Release: Slack CLI v3.9.0

October 21, 2025

Version `3.9.0` of the developer tools for the Slack platform has arrived!

* Slack CLI now supports [Work Objects](/messaging/work-objects-overview). The manifest values for `entity_types` are now gathered when reading an app manifest.
* When reinstalling the Slack CLI after some time, we now avoid blocking the installation script with an automatic prompt to update to the latest version.
