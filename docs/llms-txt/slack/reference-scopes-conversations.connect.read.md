# conversations.connect:read scope

Receive Slack Connect invite events sent to the channels your Slack app is in

## Facts

## Supported token types

[`Bot`](/authentication/tokens#bot)

[`Legacy Bot`](/authentication/tokens#legacy-bot)

## Usage info {#usage-info}

This scope allows your app to receive events when someone invites you to join a [Slack Connect](/apis/slack-connect/) conversation.
