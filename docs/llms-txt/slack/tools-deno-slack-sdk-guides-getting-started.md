<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id=install-cli>Step 1: Install the Slack CLI<a href=#install-cli class=hash-link aria-label="Direct link to Step 1: Install the Slack CLI" title="Direct link to Step 1: Install the Slack CLI" translate=no>â€‹</a></h2>
<p>Refer to <a class="" href=/tools/slack-cli/guides/installing-the-slack-cli-for-mac-and-linux>these instructions</a> to install the latest version of the Slack CLI. The instructions will guide you through installing the Slack CLI and all required dependencies.</p>

slash command, and a prompt to enter a challenge code:</p>

<p>Copy the slash command and paste it into any Slack conversation in the workspace you will be developing in.</p>
<p>When you send the message containing the slash command, a modal will pop up, prompting you to grant certain permissions to the Slack CLI. Click the Confirm button in the modal to move to the next step.</p>

<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id=use>Step 5: Use your app<a href=#use class=hash-link aria-label="Direct link to Step 5: Use your app" title="Direct link to Step 5: Use your app" translate=no>â€‹</a></h2>
<p>Grab the <code>Shortcut URL</code> you generated in the previous step and paste it in a
public channel in your workspace. You will see the shortcut unfurl with a
"Start Workflow" button. Click the button to execute the shortcut.</p>
