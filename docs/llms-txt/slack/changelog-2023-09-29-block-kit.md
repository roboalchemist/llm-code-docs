# Rich text formatting for Block Kit surfaces

September 29, 2023

Ever wanted to include the formatting available in the message composer to your form modals or Block Kit surfaces? You can now use the [rich text input](/reference/block-kit/block-elements/rich-text-input-element) to collect formatted answers from users and also post Block Kit payloads that include [rich text blocks](/reference/block-kit/blocks/rich-text-block). Try them out in [Block Kit Builder](https://app.slack.com/block-kit-builder) now. Hint: Check-out the "edit" functionality (hover and click the pencil icon) to compose rich text blocks in-line.
