# chat:write.customize scope

Send messages as your Slack app with a customized username and avatar

## Facts

## Supported token types

[`Bot`](/authentication/tokens#bot)

[`Legacy Bot`](/authentication/tokens#legacy-bot)

## Usage info {#usage-info}

This scope is only available to [Slack apps](/quickstart). Check the [`chat.postMessage`](/reference/methods/chat.postMessage#authorship) documentation for more information on how to use this scope.

To use this scope, your app must also request [`chat:write`](/reference/scopes/chat.write).
