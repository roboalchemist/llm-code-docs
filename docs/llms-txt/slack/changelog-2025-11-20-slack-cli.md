# Release: Slack CLI v3.9.2

November 20, 2025

Version `3.9.2` of the developer tools for the Slack platform has landed!

* We made a few updates behind the scenes to keep things running smoothly.
* The `deno-slack-hooks` version was updated to `v1.5.0`. This version adds support for bundling `deno-slack-sdk` projects with the new `deno bundle` command.
