# Slack CLI v1.17.0 and below is now deprecated

June 28, 2023

Beginning **June 28, 2023**, version `v1.17.0` and below of the CLI is deprecated. We recommend reinstalling the CLI to get the latest version. [Learn more](/tools/deno-slack-sdk/guides/getting-started).
