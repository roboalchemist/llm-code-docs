## New supported markdown types for rich text in Block Kit

March 6, 2026

The following new supported markdown types for rich text when using Block Kit in your apps are being rolled out:

* syntax-highlighted code blocks (language-specific)
* tables
* task lists
* dividers
* variable-sized headers

Check out our [rich text block](/reference/block-kit/blocks/rich-text-block) and [markdown block](/reference/block-kit/blocks/markdown-block) reference for more details!
