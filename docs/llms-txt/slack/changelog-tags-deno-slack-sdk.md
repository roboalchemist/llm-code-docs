## Release: Deno Slack SDK v2.13.0 & v2.14.0

July 3, 2024

In the Deno Slack SDK `2.13.0` and `2.14.0` releases, we've added API methods to support Slack canvas and Slack Connect. See the Deno Slack API [`2.5.0`](/changelog#entry-june_2024_2), [`2.6.0`](/changelog#entry-june_2024_3), and [`2.7.0`](/changelog#entry-july_2024_0) updates for more details.
