# The files.upload method is now deprecated

May 16, 2024

As of **May 16, 2024**, newly-created Slack apps are no longer able to access the [`files.upload`](/reference/methods/files.upload) API method. [Learn how](/changelog/2024-04-a-better-way-to-upload-files-is-here-to-stay) to use our new asynchronous upload flow to migrate your existing apps and integrations by **March 11, 2025**.
