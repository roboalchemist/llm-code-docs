# The Slack CLI is now open source

April 10, 2025

Long a staple of apps built with the Deno Slack SDK—and now with recent support for the Bolt framework—we're proud to make the ever-growing Slack CLI part of our collection of open source tooling.

We love contributions from our community, so we encourage you to explore and interact with the [GitHub repo](https://github.com/slackapi/slack-cli). Contributions, bug reports, and any feedback are all helpful; let us nurture the Slack CLI together to help make building Slack apps more pleasant for everyone.

Not familiar with the Slack CLI? Visit the docs and follow the [installation guide](/tools/slack-cli/guides/installing-the-slack-cli-for-mac-and-linux/) to begin your journey.
