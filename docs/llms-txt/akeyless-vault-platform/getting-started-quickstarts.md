# Quickstarts

Quick starts are short, task-focused guides that help you complete common Akeyless workflows end to end. They are designed to be followed in order if you are new to the platform, but each guide can also be used independently. They are *not* designed to instruct you in every aspect of the process.

## Before You Begin

Depending on which Quickstart you follow, you may need:

* A modern web browser and internet access
* Access to an email account to receive Akeyless signup and verification messages
* Basic familiarity with command-line tools
* Access to install command-line tools and applications

> ℹ️ **Note:** Some familiarity with cloud providers, containerization, and Kubernetes will be helpful, but is not required.

Each Quickstart lists its own prerequisites in more detail.

***

## Choosing Where to Start

If you are completely new to Akeyless, we recommend performing each Quickstart in the order in which they appear in the left-side navigation menu.

The first is [Creating an Akeyless Account Quickstart](https://docs.akeyless.io/docs/account-quickstart).
