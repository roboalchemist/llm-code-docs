# Red Hat OpenShift

Red Hat OpenShift is an enterprise-ready Kubernetes container platform with full-stack automated operations to manage hybrid cloud, multi-cloud, and edge deployments.

Akeyless official [Kubernetes Plugins](https://docs.akeyless.io/docs/kubernetes-plugins) fully supports Red Hat OpenShift and can be deployed on OpenShift platform.
