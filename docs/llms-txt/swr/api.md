</ul>
</li>

<li><code>onError(err, key, config)</code>: callback function when a request returns an error</li>
<li><code>onErrorRetry(err, key, config, revalidate, revalidateOps)</code>: handler for error retry</li>
<li><code>onDiscarded(key)</code>: callback function when a request is ignored due to race conditions</li>

<li><code>use</code>: array of middleware functions <a class="font-normal text-primary no-underline" href="/en/docs/middleware">(details)</a></li>
</ul>
