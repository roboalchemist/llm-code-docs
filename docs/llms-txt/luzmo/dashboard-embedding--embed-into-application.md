# Embed a dashboard using an Embed token

After [generating an embed authorization token server-side](guide/dashboard-embedding--generating-an-authorization-token.md) that provides access to a dashboard and its underlying datasets, you can use that Embed key-token pair to securely embed the dashboard using one of Luzmo's Embed libraries.
