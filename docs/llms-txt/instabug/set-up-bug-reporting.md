# Set up Bug Reporting

{% hint style="info" %}

### Integrating and Showing Luciq

To be able to use Luciq's Bug Reporting and Feedback product, you must first [integrate the SDK](https://docs.luciq.ai/android/set-up-luciq-for-android/integrate-luciq-on-android). You should also understand how to [show the SDK](https://docs.luciq.ai/android/set-up-luciq-for-android/set-up-bug-reporting/showing-luciq).
{% endhint %}

Your users and beta testers can show Luciq Bug Reporting from any page in your app. When they select "Report a bug", "Suggest an improvement", or "Ask a question" from the default [Prompt Options](https://docs.luciq.ai/android/set-up-luciq-for-android/sdk-customizations/prompt-options) menu that pops up when they show Luciq, these reports are sent to the bugs page of your dashboard with all the information and logs you need to pinpoint any issues.
