# Logging

Luciq offers various types of logging methods. The logs we collect are:

* Console Logs
* Luciq Logs
* Network Logs
* User Steps
* User Events

{% hint style="warning" %}

#### 🚧 Console Logging on iOS 10

Due to changes made by Apple on iOS 10+, certain workarounds are required to capture console logs. These are detailed [here](https://docs.luciq.ai/docs/ios-logging#section-console-logs-on-ios-10).
{% endhint %}
