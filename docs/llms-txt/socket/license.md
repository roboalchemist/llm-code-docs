# License

| Severity | Alert Type                                                                       | Description                                                                                               |
| :------- | :------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------- |
| High     | [Explicitly Unlicensed Item](https://socket.dev/alerts/explicitlyUnlicensedItem) | Something was found which is explicitly marked as unlicensed                                              |
| Medium   | [Misc. License Issues](https://socket.dev/alerts/miscLicenseIssues)              | A package's licensing information has fine-grained problems                                               |
| Low      | [Copyleft License](https://socket.dev/alerts/copyleftLicense)                    | Copyleft license information was found                                                                    |
| Low      | [No License Found](https://socket.dev/alerts/noLicenseFound)                     | License information could not be found                                                                    |
| Low      | [Non-permissive License](https://socket.dev/alerts/nonpermissiveLicense)         | A license not known to be considered permissive was found                                                 |
| Low      | [Unidentified License](https://socket.dev/alerts/unidentifiedLicense)            | Something that seems like a license was found, but its contents could not be matched with a known license |
| Low      | [Ambiguous License Classifier](https://socket.dev/alerts/ambiguousClassifier)    | An ambiguous license classifier was found.                                                                |
| Low      | [License exception](https://socket.dev/alerts/licenseException)                  | Contains a known deprecated SPDX license exception.                                                       |
