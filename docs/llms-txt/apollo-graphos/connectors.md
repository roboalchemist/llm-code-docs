# Rover connector Commands

Rover provides a set of `rover connector` commands for managing [Apollo Connectors](https://www.apollographql.com/docs/graphos/connectors). These allow you to run and analyze API requests, generate schemas based on that analysis, and create automated tests for your connectors.

For a complete reference, see the [CLI tools for connectors](https://www.apollographql.com/docs/graphos/connectors/tooling/cli-tools).
