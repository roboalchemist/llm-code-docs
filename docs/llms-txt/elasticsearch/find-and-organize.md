# Find and organize content
When using Elastic, you create, manage, and export various types of content, called objects. From data views that let your organize your Elasticsearch data into logical subsets, to dashboards and reports that let you visualize and share insights with others.
This section describes what those objects are, how you can organize them, and how you can find them quickly when navigating the user interface.
- [Data views](https://www.elastic.co/docs/explore-analyze/find-and-organize/data-views)
- [Saved objects](https://www.elastic.co/docs/explore-analyze/find-and-organize/saved-objects)
- [Files](https://www.elastic.co/docs/explore-analyze/find-and-organize/files)
- [Reports](https://www.elastic.co/docs/explore-analyze/find-and-organize/reports)
- [Tags](https://www.elastic.co/docs/explore-analyze/find-and-organize/tags)
- [Finding apps and objects](https://www.elastic.co/docs/explore-analyze/find-and-organize/find-apps-and-objects)
