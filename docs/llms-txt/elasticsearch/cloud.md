# Cloud
This section contains reference information about RESTful APIs, third-party dependencies, supported regions, and more for the following types of deployment:
- Elastic Cloud Hosted
- Elastic Cloud Enterprise
- Elastic Cloud on Kubernetes
- Elastic Cloud Control (ECCTL)

<note>
  For similar information about Elastic Cloud Serverless, refer to the [serverless documentation](https://www.elastic.co/docs/deploy-manage/deploy/elastic-cloud/serverless).
</note>

Learn more about the various [deployment options available](https://www.elastic.co/docs/deploy-manage/deploy), their advantages, and how to get started.

## APIs

Depending on the deployment option that you chose, you can use the following APIs to manage your environment:
- [Elastic Cloud API](https://www.elastic.co/docs/api/doc/cloud)
- [Elastic Cloud Billing Service API](https://www.elastic.co/docs/api/doc/cloud-billing)
- [Elastic Cloud Serverless API](https://www.elastic.co/docs/api/doc/elastic-cloud-serverless)
- [Elastic Cloud Enterprise API](https://www.elastic.co/docs/api/doc/cloud-enterprise)
