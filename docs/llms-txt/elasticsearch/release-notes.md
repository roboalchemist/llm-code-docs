# Elastic release notes
Stay up to date with the latest changes, fixes, known issues, and deprecations in Elastic products.
Release notes cover all the latest Elastic product changes, including the following:
- Elastic Stack 9.0.0 and later, including the most recent 9.3.1 release
- Elastic Cloud Serverless, including updates to Elasticsearch, and Observability and Elastic Security solutions
