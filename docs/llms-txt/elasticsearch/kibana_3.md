# Kibana
This section contains reference information for configuring Kibana and its apps, including:
- Application, management, and advanced settings
- Audit events
- Connectors
- Plugins
- Command line tools
- Osquery fields and packs

<note>
  Available and supported settings can vary depending on the type of environment where Kibana is deployed. For example, some settings available on a self-managed installation are not available or can be managed differently when Kibana is deployed on Elastic Cloud Hosted. While browsing these pages, look for indicators such as ![logo cloud](https://doc-icons.s3.us-east-2.amazonaws.com/logo_cloud.svg "Supported on Elastic Cloud Hosted").
</note>

Refer to [Install Kibana](https://www.elastic.co/docs/deploy-manage/deploy/self-managed/install-kibana) for an overview and setup instructions.
