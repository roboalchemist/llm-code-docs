# Create Incident or Ticket

The **Create Incident and Ticket** category lets you create an incident in various systems, such as:

* [ServiceNow - Create Incident](/docs/create-servicenow-incident)
* [Jira Service Management - Create Ticket](/docs/create-jira-service-desk-ticket)
* [Zendesk  -Create Ticket](/docs/create-zendesk-ticket)
* And more.

Users can track the progress of tickets/incidents, created in third-party ticketing systems (such as ServiceNow) using Enforcement Actions, directly from Axonius. Learn more on [how to monitor third-party tickets from within the Axonius application](/docs/monitoring-third-party-tickets).
