# What's New in Axonius 6.1.31

#### Release Date: September 8th 2024

These Release Notes contain new features and enhancements added in version 6.1.31.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Assets Pages

The following features were added to all assets pages:

### Filter Complex Field Tables by Any Field Value

You can add filters for field for a Complex Field Table in the Asset Profile. This allows you to easily adjust and focus the data displayed in Complex Field Tables.
![ComplexTableFilter](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/ComplexTableFilter.png)

## Devices Page New Features and Enhancements

The following new features and enhancements were added to the **Devices** page.

### Support for Oracle Solaris OS

[Devices in Axonius](/docs/devices-page#os-end-of-life-end-of-support-and-latest-os-version) now support the Oracle Solaris Operating System. In addition, 'End of Life', 'End of Support', 'Latest Version', and 'Is Latest Version'  are supported for this operating system.

## Software Management Module New Features and Enhancements

The following new features and enhancements were added to the Software Management Module:

### New Predefined Software Versions View

In the Software Page, users can now choose to expand rows by a new predefined 'Software Versions' view.

## Activity Log New Features and Enhancements

User updates are now audited in more details in  **Activity Logs**.

## Enforcement Center New Features and Enhancements

The following new features and enhancements were added to the Enforcement Center:

### Enhancements to Axonius Actions

In the [Axonius - Send Email per Asset](/docs/email-send-per-asset) Enforcement Action, it is now possible to upload the custom company logo so that it appears in the header of emails sent by the Enforcement Center.
