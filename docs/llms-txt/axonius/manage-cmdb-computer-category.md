# Manage CMDB Assets Category

The **Manage CMDB Assets** category lets you create or update CMDB assets.
This category, include actions such as:

* [Create ServiceNow Asset ](/docs/create-servicenow-computer)
* [Update ServiceNow Asset](/docs/update-servicenow-computer)
* [Create Cherwell Asset](/docs/create-cherwell-computer)
* [Update Cherwell Asset](/docs/update-cherwell-computer)
* [Create Ivanti Service Manager Asset](/docs/create-ivanti-service-manager-computer)
* [Update Ivanti Service Manager Asset](/docs/update-ivanti-service-manager-computer)
* And more.

Use the navigation tree and the search bar to explore the various enforcement actions.
