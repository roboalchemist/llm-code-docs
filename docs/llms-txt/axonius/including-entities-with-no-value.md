# Including Entities with No Value

<Callout icon="📘" theme="info">
  Note

  The **Include entities with no value**  option is not available for Pivot line charts when charting Activity Logs or Adapter Fetch History.
</Callout>

When configuring many chart types, you can choose to include assets that have no value returned by the query(s) used for the chart.

When **Include entities with no value** is enabled, these entities are included in the chart data, and you can indicate these entities with a display name. In **Display Name**, enter a name for entities with no value. The default is "No Value".
