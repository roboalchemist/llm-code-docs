# Configuring CSV Export Settings

![ExportCSVSettings](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/ExportCSVSettings.png)

* **Add time zone indication to date field names** *(required, default: false)* - Select whether to add a time zone label to date field columns appearing in CSV reports.
* **Export CSV Separator** - Specify the character used to separate items in CSV exports. By default, the separator is a comma (,).
