# Storage Overview

The Storage Category is a category for Storage related assets.
They include:

* **Object Storage** - Assets describing storage services that manage and organize data as discrete units (objects), typically fetched from cloud or cloud security adapters.

* **File Systems** - Assets describing file systems, typically fetched from cloud or cloud security adapters.

* **Disks** - Assets describing storage services based on disks (such as EBS volumes), typically fetched from cloud or cloud security adapters.

Each storage type is presented on its own page.

* All the Assets pages have similar capabilities as described in [Assets Page](/docs/assets-page).
