# System Charts

The system charts appear on the Axonius dashboard which is the default dashboard:

* **[Device Discovery](/docs/device-discovery-chart)**
* **[User Discovery](/docs/user-discovery-chart)**
* **[Adapter Connections Status](/docs/adapter-connections-status-chart)**
* [**System Lifecycle**](/docs/system-lifecycle-chart)
* [**Discovery Log**](/docs/system-lifecycle-chart#discovery-log-chart)

These charts address some of the most common and basic questions regarding asset management:

* How many devices do I have?
* How many users do I have in my network?
* When was the last Discovery?
* What adapters are connected?
