# Configuring Lifecycle Settings

<br />

The **Lifecycle Settings** enable configuring the global discovery cycle parameters and retention settings.

**To open the Lifecycle Settings:**

1. From the top right corner of any page, click ![](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image\(269\).png). The **System Settings** page opens.
2. In the Categories/Subcategories pane of the System Settings page, expand **Lifecycle**, and select one of the following:

   * **[Discovery](/docs/configuring-discovery-settings)** - to configure discovery settings (opens by default when opening the system settings)
   * **[Retention](/docs/configuring-retention-settings)** - to configure retention settings
   * **[Advanced Settings](/docs/configuring-advanced-lifecycle-settings)** - to disable all scheduled activities (including Discovery cycles, Adapter fetches, Enforcement actions, and Reports)
