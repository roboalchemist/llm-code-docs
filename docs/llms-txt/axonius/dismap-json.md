# Dismap - JSON

Dismap is an asset discovery and identification tool.

The Dismap JSON adapter  imports  JSON files created by Dismap scans.with information about devices and open ports for these devices

The adapter parameters are as same as the [CSV adapter parameters](/docs/csv).

The functionality of this adapter is as same as the [CSV adapter](/docs/csv).
However, only device import is performed.
