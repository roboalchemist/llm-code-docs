# What's New in Axonius 6.1.30

#### Release Date: September 1st 2024

These Release Notes contain new features and enhancements added in version 6.1.30.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Dashboard New Features and Enhancements

The following new features and enhancements were added to the Dashboards:

### Chart Enhancements

#### Rich Text Formatting in Dashboards

You can now format [Text Charts](/docs/text-only) in Axonius dashboards with bold, italics, underline, custom font sizes, colors and alignments. This allows for clearer, better-structured narratives, highlighting key insights, and more effective content organization - resulting in dashboards that are both informative and visually streamlined.

## Assets Pages

The following features were added to all assets pages:

### Dedicated Upper Tab for Asset Profile Dashboards

Asset Profile dashboards now appear in the new ['Dashboards' tab](/docs/asset-profile-page#dashboards-tab) on top of the Asset Profile page.

## Activity Log New Features and Enhancements

The following updates were made to the activity logs:

### User Details Included in Activity Log Messages

When user details are being edited, the specific changes are included in the [activity log](/docs/activity-logs-page) messages.

## Administrator Settings New Features and Enhancements

The following updates were made to various Administrator settings:

### Rotating the API Key of Service Accounts

The API key of [service accounts](/docs/manage-service-accounts#resetting-the-api-key-of-a-service-account) can now be rotated on demand.
