# Exporting Chart Data to CSV

You can export data from some charts to a CSV file.

**To download the table data to a CSV file:**

1. Configure the table to display the data you want to explore.
2. When viewing the chart on a dashboard, click the chart's 3-dot menu and select **Export Data to CSV**.

   <Image align="center" alt="ChartExportToCSVMenu.png" border={false} width="400px" src="https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/img/Dashboards/ChartExportToCSVMenu.png" />
3. The file is downloaded to your Downloads folder.
