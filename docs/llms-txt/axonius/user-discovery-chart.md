# User Discovery Chart

The **User Discovery** chart, similar to the **Device Discovery** chart, lists the number of users seen by each connected adapter (top 10 adapters), the total and unique number of users after correlation.

If there are more than 10 connected user adapters, use the pagination buttons to view all the connected adapters.

This answers the question: How many users do I have?

![UserDiscovery\_24](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/UserDiscovery_24.png)

The chart displays the same information as the **Device Discovery** chart.

To view all users per specific adapter, click that row in the chart. The **Users page** opens, filtered to results of that adapter.

The **User Discovery** chart is permanent and cannot be edited or deleted.
