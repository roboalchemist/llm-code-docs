# Axonius Software Catalog

The Axonius Catalog enriches Software with information to enable users to gain more comprehensive data about the software in their organization. This information is updated and expanded on an ongoing basis. Axonius Catalog enrichments are represented by an Axonius icon in the Adapter Connections column ![](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/AxoniusCAtalog.png) .

### End-of-life and End-of-support

Software is enriched with the date that the software reached the end-of-life and the date that the software reached the end-of-support. The End-of-life and End-of-support fields allow you to use Axonius to track information about End-of-life and End-of-support for Software Versions.
The relevant fields are displayed on the **Software** page and the **Software Versions** tab of the **Software Profile** page.

* End-of-life (date)
* Is End-of-life (either Yes or No)

<Callout icon="📘" theme="info">
  Note

  If software is known to not have an EOL date, "Is EOL" will be 'No', ensuring accurate reporting and management of your software assets.
</Callout>

* End-of-support (date)

* Is End-of-support (either Yes or No)

### Software Name and Vendor

The Axonius Catalog contains enrichments for Software Name and Software Vendor for a wide range of Software. When they exist they are presented as an additional Software Name and Software Vendor from the Axonius catalog source. In addition, when they exist, they are the value for the Preferred Software Name and Preferred Vendor Name respectively, otherwise Axonius selects the best naming option from all the fetched names.
