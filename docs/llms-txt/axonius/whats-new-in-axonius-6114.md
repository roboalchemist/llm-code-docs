# What's New in Axonius 6.1.14

#### Release Date: May 12th 2024

These Release Notes contain new features and enhancements added in version 6.1.14.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Software Management Module New Features and Enhancements

The following new features and enhancements were added to the **Software Management Module**:

#### Associated Devices in CSV Export on Software Versions Tab

In [CSV Export](/docs/software-profile#exporting-software-profile-data-to-csv) on the **Software Versions** tab of **Software Profile** page it is now possible to export associated devices, and select required device fields.

## Asset Graph New Features and Enhancements

### Expanded Asset Profile Details

The Asset Profile Details drawer in the Asset Graph has been expanded provide additional information about any Asset node in the same context of the graph. It provides similar functionality as the Asset Profile when accessed from an asset page.

![AssetGraph-DeviceDetailsIndv.png](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/AssetGraph-DeviceDetailsIndv.png)

## Query Management New Features and Enhancements

The following new features and enhancements were added to the Queries:

### Free Search for Query Name in the Queries Page

On the [Queries](/docs/managing-queries) page, a new Search field allows users to perform a free search for Query names.

## Administrator Settings New Features and Enhancements

The following updates were made to various Administrator settings:

### Preferred Fields

* Added a new preferred field that shows Combined OS fields, showing the OS, Type and Distribution. For  Windows, the Version and Edition may also be displayed
* Added the capability to configure preferred serial number for VMWare formats.
