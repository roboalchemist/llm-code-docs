# Network Inspector Requirements

This topic details the requirements for installing the Axonius Network Inspector device.

## Prerequisites

Before you start the installation, make sure that the following prerequisites are in place:

1. You have received an email from your account manager containing the **Username** and **Password** for the admin user.
2. You have reviewed the [Network Inspector specifications](/docs/network-inspector-specifications) and prepared rack space and appropriate power supply for the device.
3. If you will be assigning static IP addresses, you have the designated addresses for the Management and iDRAC communication (Management IP + Gateway, iDRAC IP + Gateway).
4. Internet access and cloud access for the Axonius Network Inspector have been configured on your firewall with specific US and/or EU domains and IP addresses, as well as for TCP and UDP communication, as detailed on the dedicated <Anchor label="Axonius support page" target="_blank" href="https://support.axonius.com/hc/en-us/articles/35991260012055-Network-Inspector-Requirements">Axonius support page</Anchor>.
