# Filtering a Chart

You can configure a single chart to update based on filter or query criteria.

<Callout icon="📘" theme="info">
  Note

  This filter is NOT persistent. It does NOT remain when you leave the dashboard and will NOT be applied when viewed again.
</Callout>

**To filter a chart**

1. Select the tab of an existing Dashboard.
2. Hover over a chart and click the filter icon.

<Image alt="DynamicaChatHeadnig" width="300px" src="https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/DynamicaChatHeadnig.png" />

3. Do one or more of the following:
   * Select a historical date (fetch results from a specific date).
   * Enter a search term to find chart values.
   * Select an Asset
   * Configure a query - see [Creating Queries with the Query Wizard](/docs/query-wizard-and-query-filter)
     1. Select an asset module and then select a query or field.

![ChartAssetFilter](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/ChartAssetFilter.png)

4. Click **Apply**. The filter settings are applied to the chart.
