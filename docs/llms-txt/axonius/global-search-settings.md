# Global Search Settings

You can set which asset types should be included in Global Search or excluded from it, and for each included asset type, configure its searchable fields.

Navigate to **System Settings** `>` **Data** `>` **Global Search Fields**.

For more information, see [Customizing Global Search Settings](https://docs.axonius.com/axonius-help-docs/docs/customizing-global-search-settings).
