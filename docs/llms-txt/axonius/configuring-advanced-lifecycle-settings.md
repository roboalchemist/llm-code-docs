# Configuring Advanced Lifecycle Settings

Use this page to pause all scheduled activities.

**To open the Advanced Lifecycle Settings:**

1. From the top right corner of any page, click ![image.png](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image\(269\).png). The **System Settings** page opens.
2. In the Categories/Subcategories pane of the System Settings page, expand **Lifecycle**, and select **Advanced Lifecycle Settings**.

**Disable all scheduled activities (including Discovery cycles, Adapter fetches, Enforcement actions and Reports)** - Admin users can toggle on this setting to pause all scheduled activities. This includes all scheduled activites including  Discovery cycles, Adapter fetches, Enforcement actions, and Reports. This setting is only available to admin users. Before enabling this setting, the system requests your confirmation to disable all scheduled activities in the system. Once you confirm, a banner is displayed on the system notifying you that activites are paused.
