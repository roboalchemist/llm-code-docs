# Installing Axonius OVA File on a Local VMware Workstation

To install the Axonius OVA file on a **local VMware Workstation** rather than on VMware ESXi (formerly ESX) or on VMware vCenter do as follows:

1. Open VMWare Workstation. From the File menu bar, click Open.

<Image align="center" border={false} src="https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image(636).png" />

2. Choose the Axonius OVA file and click Open. Specify a local path for storage and then click Import.

<Image align="center" border={false} src="https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image(637).png" />

3. When the import process is completed, power on the local virtual machine.

<Image align="center" border={false} src="https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image(638).png" />

4. Complete Axonius configuration steps. For details, see [Configuring the Axonius Platform](/docs/configuring-the-axonius-platform).
