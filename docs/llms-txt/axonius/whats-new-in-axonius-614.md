# What's New in Axonius 6.1.4

#### Release Date: March 3rd 2024

These Release Notes contain new features and enhancements added in version 6.1.4

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Dashboard New Features and Enhancements

The following new features and enhancements were added to the Dashboards:

### Chart Enhancements

#### Pivot Chart - Set Calculation Functions Per Measure

It is now possible to apply a [separate calculation function to each measure](/docs/adv-pivot-chart#6-calculation) in a Pivot Chart. This allows users more flexibility in creating  charts.

#### Exclude Date Range From Timeline Chart

It is now possible to [exclude specific date points](/docs/query-timeline-chart) from timeline charts, allowing users to focus-in on more meaningful parts of the timeline.

## Assets Pages

The following features were added to all assets pages:

### Asset Profile Dashboards

[Asset Profile Dashboards](/docs/asset-profile-dashboards) enable users to create custom dashboards within the Asset Profile, in the context of the selected asset. Dashboards can be created on the same asset type, or from other asset types. For example, it is possible to create a dashboard that shows segmentation of vulnerabilities by severity in a device.

![DashboardonAssetProfile](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/DashboardonAssetProfile.png)

## Asset Graph New Features and Enhancements

The following features were added to the Asset Graph:

### Actions Appear in Upper-Right when Multiple Nodes are Selected

When users select multiple nodes on an [Asset Graph](/docs/asset-graph), the Actions appear in the upper-right corner.
![AssetGraphActions.png](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/AssetGraphActions.png)

## Administrator Settings New Features and Enhancements

The following updates were made to various Administrator settings:

#### Syslog/SMTP/LDAP Server

It is now possible to configure the tunnel to use for the Syslog Server, for SMTP and for the LDAP connection. This is configured for each of these settings on the relevant System Settings page.
