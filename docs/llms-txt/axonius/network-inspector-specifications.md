# Network Inspector Specifications

This topic details the Axonius Network Inspector device specifications and interface panel information.

The Axonius Network Inspector device could be any of the following devices, manufactured by Dell Technologies and Compulab:

* Dell PowerEdge R250
* Dell PowerEdge R260
* Dell PowerEdge R650
* Dell PowerEdge R660
* Fitlet3 small form factor

These devices incorporate a hardened Linux operating system.

<Callout icon="📘" theme="info">
  Note

  * The SFP+ interfaces come equipped with removable SR (Short Range) multimode fiber transceivers by default. These transceivers are designed for use with multimode fiber optic cables.

  * While you can install additional network cards, be aware that doing so may reduce the overall SPAN (Switched Port Analyzer) throughput. This is because the additional cards can consume system resources needed for SPAN functionality.
</Callout>
