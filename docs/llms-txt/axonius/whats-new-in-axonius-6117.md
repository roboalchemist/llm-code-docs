# What's New in Axonius 6.1.17

#### Release Date: June 2nd 2024

These Release Notes contain new features and enhancements added in version 6.1.17.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Assets Pages

## Devices   Page  New Features and Enhancements

The following new features and enhancements were added to the **Devices**  page.

### OS Enrichments

It is now possible to query the OS by the "Latest OS Version", to find out what the latest version of the operating system  is. To find out whether the latest version is installed on the system it is now possible to use the field "Is Latest OS Version"

## Vulnerability Management Module New Features and Enhancements

The following new features and enhancements were added to the Vulnerability Management Module:

**New  Remediation Field**
The new Remediation complex field shows all the available remediations from various adapters and enrichments in one field, on the Vulnerabilities page, and on the Vulnerabilities Profile page.
The  Vulnerabilities Profile page displays all the existing remediations broken down in a dedicated table.

## Administrator Settings New Features and Enhancements

The following updates were made to various Administrator settings:

### Custom Enrichment Management

From the Custom Enrichment Management table, you can now [delete one or more Custom Enrichment Fields](/docs/managing-custom-enrichment#deleting-custom-enrichment-fields) using the **Delete** action.
