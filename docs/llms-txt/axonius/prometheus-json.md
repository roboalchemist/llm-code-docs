# Prometheus JSON

Prometheus JSON is an integration or exporter that allows Prometheus to scrape and collect metrics from JSON-formatted data sources for monitoring and alerting purposes.

The adapter parameters are as same as the [CSV adapter parameters](/docs/csv).

The functionality of this adapter is as same as the [CSV adapter](/docs/csv). However, only device import is performed.
