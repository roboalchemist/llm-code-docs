# Axonius Deployment Options

The Axonius Platform offers two deployment options:

* **[Axonius-hosted (SaaS)](/docs/saas-deployment)** - Axonius is deployed on an AWS EC2 instance that is fully separated from other customers’ environments. The AWS EC2 instance can be hosted on any of the available Amazon EC2 Regions.

* **[Customer-hosted (on-premises / private cloud)](/docs/on-premise-and-private-cloud-deployments)** - Axonius is deployed on a virtual appliance that is part of your organization’s internal network.
