# Compute Overview

The Compute category is a category for Compute related assets.
They include:

* **[Devices](/docs/devices-page)** - Bare metal and virtual devices, including workstations, mobile devices, servers, and virtual machines.

* **Compute Services** - Assets supporting compute services, such as Container Registry or Kubernetes pods, typically fetched from cloud or cloud security adapters​.

* **Databases** - Database assets, like AWS RDS or DynamoDB, typically fetched from cloud or cloud security adapters.

* **Containers** - Container assets, typically fetched from cloud or cloud security adapters.

* **Serverless Functions** - Serverless functions, typically fetched from cloud or cloud security adapters.

* **Compute Images** - Virtualization/Cloud images, such as AMIs, GCE Images, or OCI Images.

* **Configurations** - Assets representing configurations and parameters, such as passwords, database URLs, or variables.

Each Compute type is presented on its own page.

* All the Assets pages have similar capabilities as described in [Assets Page](/docs/assets-page).
