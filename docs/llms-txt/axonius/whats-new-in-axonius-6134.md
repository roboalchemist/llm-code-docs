# What's New in Axonius 6.1.34

#### Release Date: September 29th 2024

These Release Notes contain new features and enhancements added in version 6.1.34.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Asset Graph New Features and Enhancements

The following new features and enhancements were added to the Asset Graph:

### New Node Icons

The node icons on the [Asset Graph](/docs/asset-graph) have been updated.

## Enforcement Actions

### Enhancements to Axonius Actions

In the [Axonius - Send Email](/docs/send-email) Enforcement Action, it is now possible to do the following:

* Upload the custom company logo so that it appears in the header of emails sent by the Enforcement Center.
* When the Enforcement Action is based on **Adapters Fetch History** assets, the **Adapter Name** column in the Query Results table displays the adapter names and their icons. When the **Hide adapter icons in table** option is enabled, the **Adapter Name** column appears in the first column of the Query Results table but only shows the adapter name; not the adapter icons.
