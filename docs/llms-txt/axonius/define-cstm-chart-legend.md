# Defining a Custom Label for Chart Legends

This feature is available for the following chart types:

* [Query Intersection](/docs/query-intersection-chart)
* [Query Comparison](/docs/query-comparison-chart)
* [Matrix Data Chart](/docs/matrix-data-chart)
* [Query Timeline Chart](/docs/query-timeline-chart)

By default, Axonius applies the query name as legends in charts. You can define custom chart labels to something more meaningful.

**To define custom legend labels**

1. From the chart More menu (three dots), select **Edit**.
2. Select the queries you want represented in the chart.
3. At the far right of the query settings, click ![CustomLegendLabelButton.png](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/CustomLegendLabelButton.png).
4. In the text box, enter a value for the label.
5. Click **Save**.
