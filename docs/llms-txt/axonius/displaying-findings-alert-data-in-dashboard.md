# Displaying Findings Alert Data in Dashboard

You can display Findings Alerts data in the [Dashboard](/docs/working-with-dashboard-spaces) with the [Pivot Chart](/docs/adv-pivot-chart) or the [ Field Segmentation chart](/docs/field-segmentation-chart). For example, you can display a segmentation of alert severity in your organization. If the chart shows that there's a high frequency of urgent alerts, that would indicate that there may be a systematic issue that needs to be dealt with.

<Callout icon="📘" theme="info">
  Note

  You cannot filter or open a Findings Alerts query in the Chart Wizard. Such queries can only be managed in the [Findings](/docs/findings-overview) module.
</Callout>
