# Viewing System Information

To view the system information, on the top right corner of any page, click ![image.png](https://raw.githubusercontent.com/Axonius/ax-docs-pub/refs/heads/main/Images/image\(269\).png), and in the **System Settings** navigation pane that opens, select **About**.

The following information is displayed:

* **Build Date** - The build date of the currently installed version of Axonius.
* **Customer ID** - Your customer ID.
* **Installed Version** - The version number of the currently installed instance of Axonius.
* **Release Notes** - Click to open the release notes for the installed version in a new browser tab.
