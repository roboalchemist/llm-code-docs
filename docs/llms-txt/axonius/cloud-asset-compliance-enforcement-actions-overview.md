# Cloud Asset Compliance Enforcement Actions

In the **Cloud Asset Compliance** page, you can take various actions on the benchmark results table data.

To take an action, in the **Cloud Asset Compliance** page, click the **Enforce** on the right side of the page just above the table, and select the desired action.

<Callout icon="📘" theme="info">
  NOTE

  **[Axonius Security Policy Action Center](/docs/ec-overview)** is required to enforce actions for cloud assets.
</Callout>

Axonius supports the following actions:

* **[Send Email](/docs/cac-send-email)** - Sends the CSV with the filtered compliance results by email to a specified recipient list.
* **[Create Jira Issue](/docs/cac-create-jira-issue)** - Creates a Jira issue with the CSV with the filtered compliance results attached to the issue.
