# What's New in Axonius 6.1.13

#### Release Date: May 5th 2024

These Release Notes contain new features and enhancements added in version 6.1.12 and 6.1.13.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Dashboard New Features and Enhancements

The following new features and enhancements were added to the Dashboards:

### Dashboard Management Enhancements

#### Set Dashboard Access Level as Part of the Import

The capability was added to define the access level of [imported dashboards](/docs/importing-and-exporting-dashboards). This enables importing dashboards as private, public, restricted and shared across Data Scopes, including an option to select whether or not to overwrite an existing dashboard. When importing multiple dashboard, the same access permissions are applied to them all.

## Case Management New Features and Enhancements

### Create New Case Enforcement Action

It is now possible to create a new case using the [**Create new case** enforcement action](/docs/create-case). This is in addition to the already existing method of [creating a new Case from the Case Management page](/docs/creating-a-new-case).

## Adapter Pages and Adapter Interface New Features and Enhancements

The following updates were made to the common functionality across all adapters:

### Ingestion Rule Support of Regex Options

New  match\_regex and not\_match\_regex operators were added to Ingestion Rules.

## Activity Log New Features and Enhancements

Activity logs for CSV Export now show the number of assets or records exported.
