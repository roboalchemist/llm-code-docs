# Trello Ticket Created

Axonius supports **Trello Ticket Created** as an event in a Workflow. Whenever a ticket is opened in Trello, an event is created. All Workflows configured with this event are then triggered.

Before including this event in a Workflow the first time, you need to enable events in the [Atlassian adapter](/docs/atlassian-jira-software). Learn how to [enable events in the Atlassian adapter](/docs/configuring-workflows-events-settings#enabling-events-in-an-adapter) from **System Settings**.
