# Queries

Axonius uses a sophisticated set of highly detailed queries to understand how assets adhere to their policies. You can define a wide variety of filters, from which you can easily drill down to the assets that match the required search criteria.

Use the [Query Wizard](/docs/query-wizard-and-query-filter) to create detailed and granular queries. Use [Basic mode](/docs/basic-query-mode) to quickly create queries from filters. Queries are the basis for many Axonius capabilities, such as Dashboards, Reports, and Enforcement Sets. You can also create and save queries from the Vulnerabilities and Software pages. Vulnerabilities and Software queries are multi-level queries to provide vulnerability and software contextualization to help inform prioritization, refer to [ Creating Queries on Vulnerabilities](/docs/vulnerabilities#creating-queries-on-vulnerabilities) and [Creating Queries on Software](/docs/software#creating-queries-on-software).

You can see all of the Queries in the system on the [Queries](/docs/queries) page.

Use [System Queries](/docs/creating-queries-filters) on various pages in the system to create sets of filters to easily view data about the running of the system.
