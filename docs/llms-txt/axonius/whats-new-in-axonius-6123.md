# What's New in Axonius 6.1.23

#### Release Date: July 14th 2024

These Release Notes contain new features and enhancements added in version 6.1.23.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Assets Pages

The following features were added to all assets pages:

### Create Queries using Basic Query Mode

A new Basic mode was introduced for the Query Wizard. It uses filters to create simple queries. Queries are created by selecting fields and field values from the lists. When Basic mode is selected, it remains selected for that asset type until changed. The filters selected for a specific asset type also remain selected until changed.

## Administrator Settings New Features and Enhancements

The following updates were made to various Administrator settings:

### Any User Can Move to Another Data Scope

Users assigned Data Scopes other than Global can now also move to any other Data Scope for which they have permission.

## Enforcement Center New Features and Enhancements

The following new features and enhancements were added to the Enforcement Center:

### Enforcement Center Now Available to Data Scopes with Excluded Fields

The limitation of the Enforcement Center not being available within a [Data Scope with excluded fields](/docs/data-scope-management) has been removed. The Enforcement Center is now available when fields are excluded.

### Dynamic Value Statement Updates

The following updates were made to the Dynamic Value statement functionality:

* Limitation: The dynamic value statement autocomplete feature is temporarily not available in this version.
