# Active Directory Related Enforcement Actions

Using Axonius to provide visibility and searchability into AD brings Enforcement Center to the fore. Devices can be 'decommissioned', moved and/or segmented based on non-configuration criteria; and user accounts can be modified, enabled, or disabled.
