# What's New in Axonius 6.1.24

#### Release Date: July 21st 2024

These Release Notes contain new features and enhancements added in version 6.1.24.

* Read [**What's New in Axonius 6.1**](/docs/whats-new-in-axonius-610) to see all Axonius 6.1 features.

* Axonius adds and updates adapters and enforcement actions all the time. Follow [Ongoing updates to adapters and enforcement actions in Axonius 6.1](/docs/axonius-61-ongoing-adapter-and-enforcement-action-updates).

## Dashboard New Features and Enhancements

The following new features and enhancements were added to the Dashboards:

### Chart Enhancements

#### Column Sorting for Asset Table Charts

Asset Tables can now be sorted by any of their columns in ascending or descending order. Users can also set the default sorting for the table in the Chart Wizard.

## Assets Pages

The following features were added to all assets pages:

### Devices

#### EOL/EOS and Latest Version Support for PanOS

[Devices in Axonius](/docs/devices-page#os-end-of-life-end-of-support-and-latest-os-version) now support End of Life, End of Support, Latest Version, and Is Latest Version for the PanOS operating system.

## Query Management New Features and Enhancements

The following new features and enhancements were added to the Queries:

### Imported Queries are Accessible Based On the Data Scopes of the Current User

When a user imports queries to Axonius, the queries are now granted the same data scopes that the user has access to.

## Adapter Pages and Adapter Interface New Features and Enhancements

The following updates were made to the common functionality across all adapters:

### Adapter Interface

**Unique Connection Labels**
The option was added to require a unique connection label for each connection on every adapter. This setting is enabled on the **UI** page under **System Settings**.

## General Updates

### Developers REST API Updates and Enhancements

Axonius is excited to announce the beta release of API v2 which will be available by default on all Axonius instances running version 6.1.24 and above.  APIv2 is designed to enhance your REST API experience. This new version features a more intuitive interface with cleaner and more intuitive endpoints and simplified parameter names.

**Key Features of API v2:**

* Streamlined Endpoints: The initial release includes around 40 endpoints, each focused on the most vital parameters needed to achieve your goals. More endpoints will be added over time.
* Standardized and Developer-Friendly: The new API adheres to industry standards, making it easier to use and developer-friendly, which in turn speeds up development for Axonius developers.
* Comprehensive Documentation: Our detailed and cohesive documentation helps users quickly understand which endpoints to use and how to write the necessary code to achieve their objectives.
* Clear Error Messages: Clear and simple error messages, making it easier to understand and resolve issues.

Interested in joining our Early Adoption Program? Open a [Zendesk ticket](http://support.axonius.com) and select the APIv2 option, to request access to the documentation.
