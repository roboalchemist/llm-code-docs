# Axonius Network Inspector Deployment

Collecting asset data from IoT devices on a network, such as a healthcare system network, requires the **Axonius Network Inspector**, which is a physical server appliance. This device connects to the network to gather data for the Axonius platform.

The Axonius Network Inspector requires **deployment and configuration** as detailed in the following subsections, which include the device specifications, requirements, initial setup, and troubleshooting.

The device is managed through the embedded appliance dashboard UI, used for configuration, monitoring, and troubleshooting.

<Callout icon="📘" theme="info">
  Note

  The Axonius Network Inspector device works only with Axonius-hosted (SaaS), and not with the customer-hosted (on-premises / private cloud) deployment of Axonius. For more information, see [Axonius-hosted (SaaS)](https://docs.axonius.com/axonius-help-docs/docs/saas-deployment) and [Axonius Deployment Options](https://docs.axonius.com/axonius-help-docs/docs/axonius-deployment-options).
</Callout>
