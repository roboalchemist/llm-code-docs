# Logitech Sync

Logitech Sync is a device management platform that provides remote monitoring, configuration and firmware updates for conferencing and workspace devices via a unified web interface.

The Logitech Sync adapter provides Axonius with unified visibility into conferencing and workspace device health, firmware status, and room associations to ensure consistent performance across all managed environments.

## Asset Types Fetched

* Devices

## Data Retrieved through the Adapter

The adapter retrieves information regarding conferencing and workspace hardware. The retrieved data may include:

* **Devices** - Hardware identification (ID, name), firmware versions, serial numbers, and real-time health status.
* **Room Associations** - Metadata about the physical workspace, such as room names, organizational groups, and locations.

## Before You Begin

### Required Ports

* TCP port 443

### Authentication Methods

* Organization ID
* Client Certificate
* Private Key

### Required Permissions

The connection requires a Client Certificate and Private Key generated by a user with the **Sync Admin** role, which provides the high administrative privileges necessary to access organizational device data. See <Anchor label="Generating the API Certificate" target="_blank" href="/docs/logitech-sync#generating-the-api-certificate">Generating the API Certificate</Anchor>.

### APIs

Axonius uses the <Anchor label="Logitech Sync Cloud API" target="_blank" href="https://hub.sync.logitech.com/syncguides/post/1-9-getting-started--sync-cloud-api-postman-k9wIGytMJfIYs5c">Logitech Sync Cloud API</Anchor> to retrieve asset data.

### Generating the API Certificate

1. Log in to the <Anchor label="Logitech Sync Cloud portal" target="_blank" href="https://sync.logitech.com/">Logitech Sync Cloud portal</Anchor> with an administrator account.
2. Navigate to **System** and select the **Sync Cloud API** tab.
3. Click **Generate** new certificate.
4. Securely download and store both the **Client Certificate** and the **Private Key** files.
5. Locate and record your **Organization ID** (Org ID) from same page.

### Supported from Version

This adapter is supported from Axonius version 8.0.11.

## Connection Parameters

To connect the adapter in Axonius, provide the following parameters.

### Required Parameters

1. **Host Name or IP Address** - Enter the hostname of the Logitech Sync API server.
2. **Organization ID** - Enter the unique Organization ID retrieved from the Logitech Sync Portal.
3. **Client Certificate** - Upload the certificate file obtained from the Logitech Sync Portal.
4. **Private Key** - Upload the corresponding private key file for the client certificate.

### Optional Parameters

1. **Verify SSL** - Select whether to verify the SSL certificate of the server against the CA database inside of Axonius.  For more details, see [SSL Trust & CA Settings](/docs/certificate-settings#ssl-trust-ca-settings).

2. **HTTPS Proxy** - Enter an HTTPS proxy address to connect the adapter to a proxy instead of directly connecting it to the domain.

3. **HTTPS Proxy User Name** - Enter the user name to use when connecting to the value supplied in **Host Name or IP Address** via the value supplied in **HTTPS Proxy**.

4. **HTTPS Proxy Password** - Enter the password to use when connecting to the server using the **HTTPS Proxy**.

To learn about additional optional/common adapter connection parameters, see [Adding a New Adapter Connection](/docs/adding-a-new-adapter-connection).
