# Furby 2012 Teardown

## Remove Face Plate

![](https://cdn-learn.adafruit.com/assets/assets/000/002/485/medium800/projects_PA102047.jpg?1396783651)

We couldn't figure out a way to get Furby's inner face plate off cleanly, so we made one cut with shears, after which it released easily.

![](https://cdn-learn.adafruit.com/assets/assets/000/002/486/medium800/projects_PA102048.jpg?1396783673)

![](https://cdn-learn.adafruit.com/assets/assets/000/002/573/medium800/projects_PA102051.jpg?1396784700)

Remove the piezo sensor (orange wires) from this inner faceplate.

![](https://cdn-learn.adafruit.com/assets/assets/000/002/574/medium800/projects_PA102052.jpg?1396784724)

This is as far as we took Furby apart! Here's a video of how functional it still is, although we managed to bust part of one of Furby's LCD eyes.

https://www.youtube.com/watch?v=qX0_YwA1KTc
