# 1.8" TFT Display Breakout and Shield

## Breakout Assembly

![](https://cdn-learn.adafruit.com/assets/assets/000/019/555/medium800/lcds___displays_358top_ORIG.jpg?1409944057)

## Prepare the header strip:

Cut the strip to length if necessary. It will be easier to solder if you insert it into a breadboard - **long pins down**

![lcds___displays_header.jpg](https://cdn-learn.adafruit.com/assets/assets/000/019/556/medium640/lcds___displays_header.jpg?1409944088)

## Add the breakout board:

Place the breakout board over the pins so that the short pins poke through the breakout pads

![lcds___displays_place.jpg](https://cdn-learn.adafruit.com/assets/assets/000/019/557/medium640/lcds___displays_place.jpg?1409944108)

## And Solder!

Be sure to solder all pins for reliable electrical contact.  
  
_(For tips on soldering, be sure to check out our_ [_Guide to Excellent Soldering_](http://learn.adafruit.com/adafruit-guide-excellent-soldering)_)._

![lcds___displays_solder1.jpg](https://cdn-learn.adafruit.com/assets/assets/000/019/558/medium640/lcds___displays_solder1.jpg?1409944128)

![lcds___displays_solder2.jpg](https://cdn-learn.adafruit.com/assets/assets/000/019/559/medium640/lcds___displays_solder2.jpg?1409944155)

You're done! Check your solder joints visually and continue onto the next steps

![lcds___displays_done.jpg](https://cdn-learn.adafruit.com/assets/assets/000/019/560/medium640/lcds___displays_done.jpg?1409944169)
