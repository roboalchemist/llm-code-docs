# Developing locally

Copy Page

Airbyte doesn't currently accept platform contributions.

For connector contributions:

* In most cases, use the [Connector Builder](/platform/connector-development/connector-builder-ui/overview.md).

* For complex situations the Connector Builder can't support, see [Developing connectors locally](/platform/connector-development/local-connector-development.md).
