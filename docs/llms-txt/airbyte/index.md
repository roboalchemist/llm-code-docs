# Airbyte documentation

Airbyte is an open source data integration, activation, and agentic data platform. Use our data replication platform to consolidate data from hundreds of sources into your data warehouses, data lakes, and databases. Then, move data into the operational tools where work happens, like CRMs, marketing platforms, and support systems.
