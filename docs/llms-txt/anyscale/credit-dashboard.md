# Credit history dashboard

The credit history dashboard provides a complete view of all Anyscale credits granted to your organization. Use the credit history dashboard to monitor credit balances, track usage and grant activity, evaluate the need for early renewals, and forecast future consumption. You must be an organization owner to view the credit history dashboard. See [Organization owner](/administration/organization/permissions.md#org-owner).

note

This feature is a beta release and is in active development. Reach out with any feedback or suggestions.

Anyscale credits include free credits granted by Anyscale and prepaid credits associated from contractual agreements. Anyscale bills pay-as-you-go customers based on usage. To find out more about purchasing credits, contact your account representative or [Anyscale support](mailto:support@anyscale.com).

For granular insights on usage that include breakdowns by project, user, or cloud provider, use the usage dashboard. See [Usage dashboard](/administration/billing/usage-dashboard.md).

## Access the credit history dashboard[​](#access-the-credit-history-dashboard "Direct link to Access the credit history dashboard")

To view the credit history dashboard, complete the following steps:

1. [Log in to the Anyscale console](https://console.anyscale.com/).
2. Click the user icon.
3. Click **Organization settings**.
4. Click **Billing**. The **Credit history** dashboard displays.

## Use the credit history dashboard[​](#use-the-credit-history-dashboard "Direct link to Use the credit history dashboard")

The credit history dashboard displays an overview of your credit history with the following information:

| Field           | Description                                                          |
| --------------- | -------------------------------------------------------------------- |
| Current balance | The current balance of unused credits in your Anyscale organization. |
| Amount spent    | The total amount of credits used from currently active grants.       |
| Total granted   | The total number of credits granted under the current agreement.     |

Below this summary, you can view the following breakdown views of your credits:

| View        | Description                                                        |
| ----------- | ------------------------------------------------------------------ |
| In use      | The credit grants currently in use by your organization.           |
| Pending     | Scheduled credit grants that become available on a specified date. |
| Expired     | Credit grants that have expired, which may include unused credits. |
| All credits | A full overview of all credit grants for your organization.        |

Click on the name of a credit grant to open the credit ledger.

## Use the credit ledger[​](#use-the-credit-ledger "Direct link to Use the credit ledger")

You can view details for each credit grants using the **Ledger** tab.

The ledger tab contains the following information:

| Type                  | Amount                              | Balance                      | Timestamp                         |
| --------------------- | ----------------------------------- | ---------------------------- | --------------------------------- |
| **Credits available** | Credits added to your organization. | Total balance for the grant. | Last date for added credits.      |
| **Credit usage**      | Total credits used.                 | Remaining balance.           | Date of last usage for the grant. |

Some organizations receive grants in multiple disbursements with different start and end dates. View these details in the **Grants** tab.
