# Anyscale platform release notes - January 2026

Use this page to learn about new features and improvements to the Anyscale platform released in January 2026, including the Anyscale console, CLI, and SDK.

To learn about new features added as part of Ray releases, see [Ray release highlights](https://github.com/ray-project/ray/releases).

To learn about Python and system libraries installed in each Anyscale base image, see [Anyscale base images](/reference/base-images.md).

To view platform release notes from other months, see [All platform release notes](/release-notes.md#all-platform).

## Anyscale CLI version 0.26.82 released[​](#anyscale-cli-version-02682-released "Direct link to Anyscale CLI version 0.26.82 released")

**January 27, 2026**

A new version of the Anyscale CLI and SDK is now available. See [Version 0.26.82 release notes](/release-notes/cli-sdk.md#0-26-82).

## Declarative compute configs are now in beta for Kubernetes-backed clouds[​](#declarative-compute-configs-are-now-in-beta-for-kubernetes-backed-clouds "Direct link to Declarative compute configs are now in beta for Kubernetes-backed clouds")

**January 27, 2026**

You can now use declarative, cloud-agnostic syntax to define compute requirements for your Anyscale workloads backed by Kubernetes. Anyscale dynamically defines pod shapes based on your requirements. This feature is in beta release. See [Declarative compute configs](/configuration/compute/declarative.md).

## Dark mode in the Anyscale console[​](#dark-mode-in-the-anyscale-console "Direct link to Dark mode in the Anyscale console")

**January 22, 2026**

The Anyscale console now offers dark mode. Change the theme for your user account in the [account settings](https://console.anyscale.com/account-settings).

## Alerts for long-running workspaces[​](#alerts-for-long-running-workspaces "Direct link to Alerts for long-running workspaces")

**January 21, 2026**

Organization owners can now configure resource notifications to send alerts for long-running workspaces. See [Custom job and service notifications](/administration/resource-management/custom-notifications.md).

## Actor dashboard is now in beta[​](#actor-dashboard-is-now-in-beta "Direct link to Actor dashboard is now in beta")

**January 20, 2026**

Use the Anyscale actor dashboard to view details about Ray actors in your jobs and workspaces. Metrics persist after cluster termination. This feature is in beta release. See [Anyscale actor dashboard](/monitoring/actor-dashboard.md).

## Anyscale operator for Kubernetes version 1.3.2 released[​](#anyscale-operator-for-kubernetes-version-132-released "Direct link to Anyscale operator for Kubernetes version 1.3.2 released")

**January 20, 2026**

Anyscale has released version 1.3.2 of the Anyscale operator for Kubernetes. See release details on the [Anyscale operator GitHub page](https://github.com/anyscale/helm-charts/releases/tag/anyscale-operator-1.3.2).
