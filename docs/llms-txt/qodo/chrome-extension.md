# Chrome Extension

{% hint style="warning" %}
**Git Integration (formerly Qodo Merge)** – AI code review agents for pull requests.\
This documentation describes the Qodo v1 experience. For the Qodo v2 documentation, click [here.](https://app.gitbook.com/s/yXEFCLH7CkXrROa2gOqv/)
{% endhint %}

{% hint style="info" %}
**Supported browsers:** The extension is supported on all Chromium-based browsers, including Google Chrome, Arc, Opera, Brave, and Microsoft Edge.

**The Chrome Extension is only available for GitHub Cloud users.**
{% endhint %}

The [Qodo Chrome extension](https://chromewebstore.google.com/detail/pr-agent-chrome-extension/ephlnjeghhogofkifjloamocljapahnl) adds AI-assisted tools directly to your GitHub pull request interface.

### **Features**

* **Context-aware AI chat for pull request code**
  * Adds a chat interface to pull requests that understands the code and PR context.
* **Extension toolbar with multiple AI feedback options.**
  * Includes a toolbar on the PR page with quick access to multiple Qodo tools.
* **Custom filters and additional review capabilities.**
  * Enables advanced filtering options for pull requests using Qodo metadata.

### **Availability**

**Public repositories**: All extension features are available by default.

**Private repositories**: Requires the Qodo GitHub App to be installed in addition to the extension.

### **Installation**

Installation is simple and requires no setup beyond adding the extension to your browser:

{% embed url="<https://chromewebstore.google.com/detail/pr-agent-chrome-extension/ephlnjeghhogofkifjloamocljapahnl>" %}

For private repositories, you will need to install [Qodo Git interface](https://github.com/apps/qodo-merge-pro) in addition to the extension.
