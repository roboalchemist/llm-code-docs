# Invite-Only Organizations

{% hint style="success" %}
To enable an invite-only organization, contact Qodo support.
{% endhint %}

Invite-only organizations restrict access so that users can join only through an invitation.

In invite-only organizations:

* Users who sign up to Qodo using a company email domain are **not automatically added** to the organization.
* Instead, an access request is sent to an organization admin for approval.
* Once approved, the user receives an invitation to join the organization.

**Signing up to an invite-only organization**

To access an invite-only organization:

* Use the link provided in the invitation email from your admin.
* Complete the sign-up process using the invited email address.

If a user attempts to sign up without an invitation, access to the organization is blocked until an admin approves the request and sends an invitation.
