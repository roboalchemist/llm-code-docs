# Connect Qodo Git integration

Connecting the Qodo Git integration enables Qodo to analyze your repository and provide feedback directly in your pull/merge requests.

\
Make sure your Git user account is connected to Qodo (see [Connect a Git Provider Account)](https://docs.qodo.ai/qodo-documentation/management-portal/account-preferences#connect-a-git-provider-account).<br>

1. Navigate to **Integrations.** A list of supported repositories will be displayed.
2. Select your repository hosting service (GitHub, GitLab, or Bitbucket).
3. Click **Connect** and authorize Qodo to access your repository.
4. Configure the integration settings as needed.

Once connected, Qodo will begin analyzing your repository and providing feedback in Git.
