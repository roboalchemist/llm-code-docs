# Help

{% hint style="warning" %}
**Git Integration (formerly Qodo Merge)** – AI code review agents for pull requests.\
This documentation describes the Qodo v1 experience. For the Qodo v2 documentation, click [here.](https://app.gitbook.com/s/yXEFCLH7CkXrROa2gOqv/)
{% endhint %}

The `help` tool provides a list of all the available tools and their descriptions. It also enables to trigger each tool by checking the relevant box.

### How to use the `help` tool <a href="#how-to-use-the-help-tool" id="how-to-use-the-help-tool"></a>

**Manual usage**Comment on the PR:/help

#### Interactive options <a href="#interactive-options" id="interactive-options"></a>

The `/help` command not only lists available tools and their descriptions, but also enables immediate tool invocation through interactive checkboxes.When a user checks a tool's checkbox, Qodo instantly triggers that tool without requiring additional commands. This transforms the standard help menu into an interactive launch pad for all Qodo capabilities, eliminating context switching by keeping developers within their PR workflow.
