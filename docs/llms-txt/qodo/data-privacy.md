# Data Privacy

{% hint style="warning" %}
**Git Integration (formerly Qodo Merge)** – AI code review agents for pull requests.\
This documentation describes the Qodo v1 experience. For the Qodo v2 documentation, click [here.](https://app.gitbook.com/s/yXEFCLH7CkXrROa2gOqv/)
{% endhint %}

Qodo is designed with your code’s privacy in mind:

* The Chrome extension does **not** send your code to any external servers.
* For **private repositories**, all requests are routed through your existing Qodo integration.
  * User identity and permissions are verified before any response is generated.
