# How to Add Repositories to the Context Engine (RAG) Index

Add specific repositories to [the Context Engine](https://docs.qodo.ai/qodo-documentation/qodo-gen/code-intelligence/qodo-aware) to provide context for Qodo.

This will help it better understand your company's code and answer more complex questions about your projects.

### What is the Context Engine? <a href="#what-is-qodo-aware" id="what-is-qodo-aware"></a>

The Context Engine is the code intelligence engine behind the Qodo platform. It connects to all your repos, analyzes architecture and patterns, and powers AI agents that help you search, understand, and solve technical problems.

Think of it as an AI-powered Principal Engineer, that understands your entire system and helps you move faster with confidence.

### How to add repositories to the Context Engine?

1. **Open and enable the CodiumAI Semantic Indexer GitHub app.**\
   This app is only available to administrators of enterprise orgs that have the Context Engine enabled in their contract.\
   Contact Qodo support if you'd like to learn more.

2. **Go to the App settings page**.
3. **Select the wanted repositories from the list:** Scroll down to **Repository access** and choose the repositories you'd like to add as context from the dropdown list.
