# Data Sharing

{% hint style="warning" %}
**Git Integration (formerly Qodo Merge)** – AI code review agents for pull requests.\
This documentation describes the Qodo v1 experience. For the Qodo v2 documentation, click [here.](https://app.gitbook.com/s/yXEFCLH7CkXrROa2gOqv/)
{% endhint %}

Similar to other popular generative-AI tools, we temporarily send code snippets to our servers to generate results. You can learn more in our [terms of use](https://www.qodo.ai/terms) and [privacy policy](https://www.qodo.ai/privacy-policy).

* **Zero data retention:** Your code and data are **never stored** or used for **training**.
* In some cases, Qodo’s own models may be used — you’ll be notified if so.
* **Qodo's Git interface activates on-demand.** It extracts and analyzes only data relevant to the executed command and queried pull request.
* **Qodo's Chrome Extension** only communicates with Qodo and the LLM provider, no other third party. It does not perform browser data collection of any sort.
