# Thoropass Integration

The Thoropass integration automatically pushes vulnerabilities to Thoropass for SOC2 and ISO27001:2022

### How to enable the Thoropass Integration. <a href="#how-to-enable-the-thoropass-integration" id="how-to-enable-the-thoropass-integration"></a>

**Step 1.** Go to [**Settings > Integrations > Thoropass**](https://app.aikido.dev/settings/integrations/compliance/thoropass) and click 'Add Thoropass integration'.

![Missing Thoropass integration; option to add and enable automatic vulnerability push.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-80eb61ac931fc9b871879098a6de0b4c665900ad%2Fthoropass-compliance-integration_2ad2f45c-83ce-440b-8326-7d3437f5b609.png?alt=media)

**Step 2.** Authorize Aikido Access in your Thoropass Account

![Aikido requests read/write access to vulnerabilities in your Thorapass account.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-84bc9839af182fcb1f07927fbf24030cb9f2ab4e%2Fthoropass-compliance-integration_ea3e30fb-51cd-414d-aa73-0b7557f91475.png?alt=media)

**Step 3.** You are all set. Aikido will actively push all vulnerabilities to Thoropass on a daily basis. You can also see the active Aikido integration [inside the Thoropass App](https://app.thoropass.com/integrations).
