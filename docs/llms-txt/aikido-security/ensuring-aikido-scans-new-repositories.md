# Ensuring Aikido Scans New Repositories

By default, Aikido will not enable scanning on newly created repositories that you did not previously enable. You can change this default behaviour.

**Step 1**. Go to [Settings/ repositories](https://app.aikido.dev/settings/integrations/repositories)

**Step 2.** Click 'Actions' dropdown on the top right and select **Repository Settings**.

![Actions dropdown menu with options for repository settings and private registry connections.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-75eb3f6c277489f49f65fb9ea5018923cb511f61%2Fensuring-aikido-scans-new-repositories_46bb8898-183a-48a0-a99c-13d2af863789.png?alt=media)

**Step 3**. Enable the slider to make sure any new repository added from that point on will be scanned.

![Enable auto-activation and scanning for new repositories in repository configuration settings.](https://3149773201-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FyKbzcQGrx7UtrG0nPZZ7%2Fuploads%2Fgit-blob-07aed80a9fd5596da144424e804d7126cf479f92%2Fensuring-aikido-scans-new-repositories_fb601d1e-1263-4642-bdb1-2d882a5d244d.png?alt=media)
