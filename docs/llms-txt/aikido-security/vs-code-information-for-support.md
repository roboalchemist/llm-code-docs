# VS Code - Information for Support

When contacting Aikido Support about the VS Code, Windsurf, Cursor or other variant IDE plugin, please include the information below. This helps us diagnose issues faster and avoid unnecessary back and forth.

### Issue reproducibility

* Is the issue consistently reproducible?
* Does it occur with a single file or with multiple files?
  * If possible share the file with us
* [Did you check connection to Aikido platform?](https://help.aikido.dev/ide-plugins/troubleshooting/troubleshooting-ide-plugin-connectivity)

### Plugin & Repository information

* Plugin version \
  Settings > Extensions -> Aikido Security

* Operating System
* IDE Version
* Link to repository in Aikido
* Usage of Docker, WSL or any other remote development environment

### Review IDE logs

1. Open Output window: `View > Output`
2. Select `Aikido` logs
3. Copy and paste into text file and share
