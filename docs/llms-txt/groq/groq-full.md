- **Type:** `object`

## Chat Completion Document

- **Description:** A document that can be referenced by the model while generating responses.
- **Properties:** `id`, `source`
- **Type:** `object`

## Chat Completion Message

- **Description:** The contents of the assistant message.
- **Properties:** `content`, `function_call`
- **Type:** `object`

## Chat Completion Response Message

- **Description:** A chat completion message generated by the model.
- **Properties:** `annotations`, `content`, `executed_tools`, `function_call`, `reasoning`, `role`, `tool_calls`
- **Type:** `object`

## Chat Completion Role

- **Description:** The role of the author of a message.
- **Properties:** `system`, `user`, `assistant`, `tool`, `function`
- **Type:** `string`

## Chat Completion Stream

- **Description:** A chat completion delta generated by streamed model responses.
- **Properties:** `annotations`, `content`, `executed_tools`, `function_call`, `reasoning`, `role`, `tool_calls`
- **Type:** `object`

## Chat Completion Token Logprob

- **Description:** A list of tokens with log probabilities.
- **Properties:** `bytes`, `logprob`
- **Type:** `array`

## Chat Completion Tool

- **Description:** Controls which (if any) tool is called by the model.
- **Properties:** `none`, `auto`, `required`
- **Type:** `object`

## Chat Completion Response

- **Description:** Represents a chat completion response returned by model, based on the provided input.
- **Properties:** `choices`, `created_at`, `error`, `id`, `incomplete_details`, `instructions`, `max_output_tokens`, `max_tool_calls`, `metadata`, `model`, `object`, `output`, `parallel_tool_calls`, `previous_response_id`, `reasoning`, `response_format`, `stream`, `temperature`, `tool_choice`, `tools`, `top_logprobs`, `top_p`, `truncation`, `usage`
- **Type:** `object`

## Chat Completion Stream Response

- **Description:** Represents a streamed chunk of a chat completion response returned by model, based on the provided input.
- **Properties:** `choices`, `created_at`, `error`, `id`, `incomplete_details`, `instructions`, `max_output_tokens`, `max_tool_calls`, `metadata`, `model`, `object`, `output`, `parallel_tool_calls`, `previous_response_id`, `reasoning`, `response_format`, `stream`, `temperature`, `tool_choice`, `tools`, `top_logprobs`, `top_p`, `truncation`, `usage`
- **Type:** `object`

## Chat Completion Response Message

- **Description:** A message input to the model with a role indicating instruction following hierarchy.
- **Properties:** `content`, `role`, `status`
- **Type:** `object`

## Chat Completion File

- **Description:** The `File` object represents a document that has been uploaded.
- **Properties:** `bytes`, `created_at`, `filename`, `id`, `object`, `purpose`
- **Type:** `object`

## Function Citation

- **Description:** A citation referencing a specific document that was provided in the request.
- **Properties:** `document_id`, `end_index`, `start_index`
- **Type:** `object`

## Embedding

- **Description:** Represents an embedding vector returned by embedding endpoint.
- **Properties:** `embedding`, `index`, `object`
- **Type:** `object`
