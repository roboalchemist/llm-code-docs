# Xpander

> Integrate Linkup with Xpander to enhance your AI workflows

## Overview

Linkup can be used in [Xpander.ai](http://Xpander.ai) to get contextual information from the internet in a workflow.

## Installation

<Steps>
  <Step title="Open Xpander">
    1. Open your Xpander account.
    2. Go to the **Agents** tab.
    3. Click **New Agent**
    4. Name your agent (e.g. “Linkup Agent”)

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.45.34_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=83ddb4d3a970d6dbc4216fb49e70e8c1" alt="Step 1" width="2940" height="1654" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.45.34_AM.png" />
  </Step>

  <Step title="Open the Tools Panel">
    1. Inside the agent editor, click on the **Tools** tab
    2. Click **+ Add Tools**

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.46.08_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=994153504e8a459f680ef0dd4a0bf64c" alt="Step 2" width="2940" height="1672" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.46.08_AM.png" />
  </Step>

  <Step title="Select Linkup">
    1. From the list, choose [**Linkup.so**](http://Linkup.so) (auth required)

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.47.21_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=f06b09ef6548d67c529dd5466d273116" alt="Step 3" width="802" height="664" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.47.21_AM.png" />

    2. Choose “API Key” under Authentication Mode
    3. Enter your Linkup API Key
    4. Choose “Bearer”

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_11.11.30_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=12c197f2e50fc36b3e85f4bb16bbf12a" alt="Step 4" width="1152" height="1464" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_11.11.30_AM.png" />

    5. Select the action: `Retrieve Web Content by Query`

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.47.54_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=3346c9153f7cca8f655644076fdeb0d9" alt="Step 5" width="802" height="540" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_10.47.54_AM.png" />

    6. Click **Add**
  </Step>

  <Step title="Confirm the Tool is Active">
    1. Linkup should now appear in your Tools list
    2. You don’t need to connect it to other tools manually
    3. Xpander AI will automatically detect when to use Linkup

        <img src="https://mintcdn.com/linkup-8b5c238e/oh3ymISFMHZ8Z97k/pages/integrations/xpander/assets/Screenshot_2025-07-15_at_11.14.31_AM.png?fit=max&auto=format&n=oh3ymISFMHZ8Z97k&q=85&s=4d7c2a2c37544ce74c51073d01409883" alt="Step 6" width="1162" height="654" data-path="pages/integrations/xpander/assets/Screenshot_2025-07-15_at_11.14.31_AM.png" />
  </Step>
</Steps>

You are all set to use Linkup in your Xpander workflow! Visit the [Concepts](/pages/documentation/get-started/concepts) page to learn more about the different Linkup parameters.
