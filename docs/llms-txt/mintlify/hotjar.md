# HotJar

> Capture user feedback and heatmaps with HotJar.

Add the following to your `docs.json` file to send analytics to HotJar.

```json Analytics options in docs.json theme={null}
"integrations": {
    "hotjar": {
        "hjid": "required",
        "hjsv": "required"
    }
}
```
