# Interface: Font

Represents a font.

The Font interface defines the structure of a font within the Creative Editor SDK. It includes properties for the uri, subFamily, weight, and style.

## Properties[#](#properties)

| Property | Type |
| --- | --- |
| `uri` | `string` |
| `subFamily` | `string` |
| `weight?` | [`FontWeight`](https://img.ly/docs/cesdk/sveltekit/api/cesdk-js/type-aliases/fontweight/) |
| `style?` | [`FontStyle`](https://img.ly/docs/cesdk/sveltekit/api/cesdk-js/type-aliases/fontstyle/) |
