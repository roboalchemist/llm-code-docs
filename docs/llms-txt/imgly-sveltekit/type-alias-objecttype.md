# Type Alias: ObjectType

```
type ObjectType =  | ObjectTypeShorthand  | ObjectTypeLonghand;
```

The block type IDs for all blocks types in the Creative Engine. Those are the types that can be passed to `cesdk.engine.block.findByType(type)` for example. Refer to [ObjectTypeShorthand](https://img.ly/docs/cesdk/sveltekit/api/cesdk-js/type-aliases/objecttypeshorthand/) and [ObjectTypeLonghand](https://img.ly/docs/cesdk/sveltekit/api/cesdk-js/type-aliases/objecttypelonghand/) for more details.
