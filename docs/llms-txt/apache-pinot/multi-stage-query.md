# Multi-stage query

The general explanation of the multi-stage query engine is provided in the [Multi-stage query engine](https://docs.pinot.apache.org/reference/multi-stage-engine) reference documentation. This section provides a deep dive into the multi-stage query engine. Most of the concepts explained here are related to the internals of the multi-stage query engine and users don't need to know about them in order to write queries. However, understanding these concepts can help you to take advantage of the engine's capabilities and to troubleshoot issues.
