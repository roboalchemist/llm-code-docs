# Binary Functions

[**SHA(bytesCol)**](https://docs.pinot.apache.org/functions-1/sha)\
Return SHA-1 digest of binary column(`bytes` type) as hex string

[**SHA256(bytesCol)**](https://docs.pinot.apache.org/functions-1/sha256)\
Return SHA-256 digest of binary column(`bytes` type) as hex string

[**SHA512(bytesCol)**](https://docs.pinot.apache.org/functions-1/sha512)\
Return SHA-512 digest of binary column(`bytes` type) as hex string

[**MD5(bytesCol)**](https://docs.pinot.apache.org/functions-1/md5)\
Return MD5 digest of binary column(`bytes` type) as hex string

[**toBase64(bytesCol)**](https://docs.pinot.apache.org/functions-1/base64)\
Return the Base64-encoded string of binary column(`bytes` type)

[**fromUtf8(bytesCol)**](https://docs.pinot.apache.org/functions-1/utf8)\
Return the UTF8-encoded string of binary column(`bytes` type)
