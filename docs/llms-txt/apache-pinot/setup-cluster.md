# Set up cluster

To set up a Pinot cluster, follow these steps

1. [Create a cluster](https://docs.pinot.apache.org/basics/concepts/components/cluster#setup-a-pinot-cluster)
2. [Start Controller](https://docs.pinot.apache.org/basics/concepts/components/cluster/controller#starting-a-controller)instances
3. [Start Broker](https://docs.pinot.apache.org/basics/concepts/components/cluster/broker#starting-a-broker) instances
4. [Start Server](https://docs.pinot.apache.org/basics/concepts/components/cluster/server#starting-a-server) instances
