# SPACE

Builds a string consisting of the specified number of blank spaces.

## Syntax

```sqlsyntax
SPACE(<n>)
```

## Arguments

`n`
:   The number of blank spaces used to build the string.

## Examples

```sqlexample
SELECT SPACE(3);
```
