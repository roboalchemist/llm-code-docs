# SYSTEM$GET_DEBUG_STATUS

Returns the [session debug mode](../../developer-guide/native-apps/installing-testing-application.md) status of the current session.

## Syntax

```sqlsyntax
SYSTEM$GET_DEBUG_STATUS()
```
