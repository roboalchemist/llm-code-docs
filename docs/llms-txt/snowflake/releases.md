# Snowflake Container Runtime release notes

Snowflake provides container runtime environments for both CPU and GPU compute. This container runtime contains a base compute environment and libraries, along with common Python packages, including those for machine learning. Container runtimes are versioned, allowing you to choose a specific runtime for your ML Job workloads.

Each Snowflake container runtime can include different versions of Python, with various levels of package support. Container Runtime release notes inform you of the Python and package versions available on each runtime, in addition to the usual information about improvements, features, and important security or bug fixes.

> **Important:**
>
> We recommend you update to the latest container runtime environments whenever possible, to make sure you have important
> security fixes and product updates.
