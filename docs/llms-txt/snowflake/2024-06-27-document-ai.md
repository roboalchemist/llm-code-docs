# June 27, 2024 — Document AI release notes

With this release, we are pleased to announce the increase in number of documents that you can process
with [<model_build_name>!PREDICT](../../../sql-reference/classes/document-intelligence/methods/predict.md) in one query.

You can now process a maximum of 1000 documents in one query, instead of 20.

For more information, see [Use the extracting query](../../../user-guide/snowflake-cortex/document-ai/extract-information.md).
