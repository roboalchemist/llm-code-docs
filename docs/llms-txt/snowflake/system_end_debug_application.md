# SYSTEM$END_DEBUG_APPLICATION

Disables [session debug mode](../../developer-guide/native-apps/installing-testing-application.md) for a Snowflake Native App.

## Syntax

```sqlsyntax
SYSTEM$END_DEBUG_APPLICATION()
```
