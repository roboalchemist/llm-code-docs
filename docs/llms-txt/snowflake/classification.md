# CLASSIFICATION (SNOWFLAKE.ML)

Classification commands enable you to manage [classification](../../user-guide/ml-functions/classification.md)
models in your account. Classification uses machine learning algorithms to sort data into different classes using patterns
detected in training data.

## Classification commands

* [ALTER SNOWFLAKE.ML.CLASSIFICATION](classification/commands/alter-classification.md)
* [CREATE SNOWFLAKE.ML.CLASSIFICATION](classification/commands/create-classification.md)
* [DROP SNOWFLAKE.ML.CLASSIFICATION](classification/commands/drop-classification.md)
* [SHOW SNOWFLAKE.ML.CLASSIFICATION](classification/commands/show-classification.md)
