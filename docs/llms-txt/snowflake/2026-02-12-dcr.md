# Feb 12, 2026: Snowflake Data Clean Rooms updates

## Clean Rooms API Version: 13.2

The following new features and enhancements are now available in Snowflake Data Clean Rooms:

* **Custom registries:** Collaboration Data Clean Rooms users can now add custom registries to organize and manage access to resources in their collaborations. To learn more, read [Registries](../../../user-guide/cleanrooms/v2/registries.md).
* Updates to private preview features.
