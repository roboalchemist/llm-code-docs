# Nov 20, 2025: New versions of Streamlit supported in Streamlit in Snowflake (General availability)

The following versions of the Streamlit open-source library are now supported in Streamlit in Snowflake:

* 1.51.0
* 1.50.0
* 1.49.1
* 1.48.0
* 1.47.0
