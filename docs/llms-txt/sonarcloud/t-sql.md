# T-SQL

### Language-specific properties <a href="#language-specific-properties" id="language-specific-properties"></a>

To discover and update the T-SQL-specific properties, navigate in SonarQube Cloud to *Your Project* > **Administration** > **General Settings** > **Languages** > **T-SQL**. See the [analysis-parameters](https://docs.sonarsource.com/sonarqube-cloud/advanced-setup/analysis-parameters "mention") page for more information about specific properties.

### Important Note <a href="#important-note" id="important-note"></a>

With the default configuration, only files with the `.tsql` are analyzed as T-SQL, and files with the `.sql` file extension are analyzed as PL/SQL. This behavior is defined in Project **Administration** > **General Settings > T-SQL > File Suffixes** and *Your Project* > **Administration** > **General Settings** > **Languages** > **PL/SQL** > **File Suffixes**. You can override these properties at the project level.
