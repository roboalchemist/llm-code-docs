# Deleting your account

Before you can delete your user account, you must first manually unassign yourself from all your issues and remove your issue comment. See [#assigning](https://docs.sonarsource.com/sonarqube-cloud/managing-your-projects/issues/editing#assigning "mention") for more details.

{% hint style="info" %}
If you are the only administrator for an organization of which you are a member, the administrator permissions must be transferred before you can delete your SonarQube Cloud account. See [#transferring-ownership-of-organization](https://docs.sonarsource.com/sonarqube-cloud/administering-sonarcloud/managing-organization/users-and-permissions/organization-permissions#transferring-ownership-of-organization "mention") for more information.
{% endhint %}

To delete your user account:

1. Select your account menu in the top right corner of the SonarQube Cloud interface.
2. In the menu, select **My account**. The **Profile** page opens.
3. In the **Delete your SonarQube Cloud account** section, select **Delete account**.
