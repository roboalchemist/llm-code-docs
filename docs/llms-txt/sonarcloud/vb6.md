# VB6

To discover and update the VB6-specific properties, navigate in SonarQube Cloud to *Your Project* > **Administration** > **General Settings** > **Languages** > **Visual Basic**. See the [analysis-parameters](https://docs.sonarsource.com/sonarqube-cloud/advanced-setup/analysis-parameters "mention") page for more information about specific properties.

{% hint style="info" %}
Although `.cls` files can contain VB6 code, they are not analyzed by default in order to avoid clashes with other plugins.
{% endhint %}
