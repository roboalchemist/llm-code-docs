# Moving to another SonarQube Server edition

To move to another SonarQube Server edition during an update:

* Follow the instructions in the respective update section below on the [update](https://docs.sonarsource.com/sonarqube-server/server-update-and-maintenance/update/update "mention") page by using the appropriate edition file or Docker image tag:
  * **Updating a ZIP file instance**
  * **Upgdating a Docker image instance**
  * **Updating a Helm chart instance**

To move to another SonarQube Server edition without updating your SonarQube Server version:

* Follow the instructions in the respective update section on the [update](https://docs.sonarsource.com/sonarqube-server/server-update-and-maintenance/update/update "mention") page:
  * By using the appropriate edition file or Docker image tag.
  * Without navigating to `http://<yourSonarQubeServerURL>/setup`.
