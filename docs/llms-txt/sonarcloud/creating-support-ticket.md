# Creating support ticket

If your license includes Sonar support, a **Support** tab will be visible on the **Administration** > **Support** page, providing guidance for interacting with the support team.

Click **Download** to collect the support information file of your instance. Make sure to provide this file for any interaction with Sonar Support.
