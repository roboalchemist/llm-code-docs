# Scala

### Language-specific properties <a href="#language-specific-properties" id="language-specific-properties"></a>

To discover and update the Scala-specific properties, navigate in SonarQube Cloud to *Your Project* > **Administration** > **General Settings** > **Languages** > **Scala**. See the [analysis-parameters](https://docs.sonarsource.com/sonarqube-cloud/advanced-setup/analysis-parameters "mention") page for more information about specific properties.
