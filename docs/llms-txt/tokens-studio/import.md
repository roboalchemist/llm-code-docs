# Import from Figma Guide

## Import from Figma guide&#x20;

You can create Design Tokens in Tokens Studio by importing your Styles or Variables from Figma into the Plugin. This allows Tokens Studio to be the single place to manage Tokens, Styles, and Variables in Figma.

Each time you Import from Figma, you choose:

* To import Variables or Variables.&#x20;
* Which Types of Variables or Styles to import at that time.&#x20;

### Variable Types as Token Types

When importing variables:

* Color Variables -  creates [Color Tokens](https://docs.tokens.studio/manage-tokens/token-types/color).
* Boolean Variables - creates [Boolean Tokens](https://docs.tokens.studio/manage-tokens/token-types/boolean).&#x20;
* String/Text Variables - creates [Text Tokens](https://docs.tokens.studio/manage-tokens/token-types/text).
* Number Variables - created [Number](https://docs.tokens.studio/manage-tokens/token-types/number) or [Dimension Tokens](https://docs.tokens.studio/manage-tokens/token-types/dimension).&#x20;

When you import variables from Figma, you may wish to manually adjust the Token Type.

→ [Jump to the guide on Variable Types and Token Types](https://docs.tokens.studio/figma/import/variables/token-types)

### Style Types as Token Types

When importing styles:

* Color Styles - creates [Color Tokens](https://docs.tokens.studio/manage-tokens/token-types/color).
* Text Styles - creates [Typography Composite Tokens](https://docs.tokens.studio/manage-tokens/token-types/typography) and optional Text Property Tokens.
* Shadows - creates [Box Shadow Tokens](https://docs.tokens.studio/manage-tokens/token-types/box-shadow) from Effects Styles.
