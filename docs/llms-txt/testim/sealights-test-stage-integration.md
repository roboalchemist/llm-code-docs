# Sealights test stage integration

We've added support for custom team stage names to the Sealights integration to both CLI and Scheduler. You can use  `--sealights-test-stage [sealights-test-stage-name]` to specify custom test stage names in commands. Or just enter your custom test stage name in the Scheduler configuration screen.
