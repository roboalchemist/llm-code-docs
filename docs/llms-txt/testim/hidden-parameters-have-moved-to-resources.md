# Hidden Parameters have moved to Resources!

From now on, you can find and manage [hidden parameters](https://help.testim.io/docs/hidden-parameters) in the new **Resources** tab. This move also includes new capabilities to manage your hidden parameters and makes is simpler to create tests.
