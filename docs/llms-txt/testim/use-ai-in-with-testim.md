# Use AI in with Testim

Take a look at the AI tools we have on offer to make your life easier.

We've integrated AI throughout Testim in ways designed to make your life easier. Take a look at the AI tools we offer with Testim to learn how can speed up your testing and find answers to your most technical questions.

## Learn how to use Testim

Need help with Testim software? Whether it's something basic, such as instructions on how to validate tests, or a more complex query, such as tips on how to stabilize your test runs, you can ask the **Testim Copilot Help Assistant**.

<Image align="center" border={false} caption="Ask the Testim Copilot Help Assistant any questions you have." src="https://files.readme.io/8e5294d6ea7cbd887a05459b39c7dee73ddf2212257292f80296c142284439a0-8243ad1-image_7.png" width="70% " />

Use your copilot to get relevant information for your Testim questions, including links to documentation and other resources. Just click the **Help** button to get started.

Learn more about the [Testim Copilot Help Assistant](https://help.testim.io/docs/help-ai-assistant#/).

## Get coding help

Need help writing, understanding, or fixing the JavaScript code in your tests? Use the **Testim Copilot Coding Assistant** to get help on any test steps that use custom JavaScript code.

<Image align="center" border={false} caption="Your copilot can help, whether you need to generate, fix, or have some JavaScript code explained to you." src="https://files.readme.io/f5ab918178dd85bd78d2fbaad1bc0fb2bbdcfbefb7a309f197b69e325ced3272-e2c25d1-codingassistant.png" width="50% " />

Learn more about the [Testim Copilot Coding Assistant](https://help.testim.io/docs/coding-assistant).
