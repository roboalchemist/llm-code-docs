# Keyboard shortcuts

Keyboard shortcuts for quicker and easier test authoring

Testim has a set of shortcuts you can use to speed up your test authoring. Using these shortcuts will allow you to add various validations during test recording.

| Action Name                                                              | Windows           | Mac               | Available in |
| :----------------------------------------------------------------------- | :---------------- | :---------------- | :----------- |
| [Add Hover Step](https://help.testim.io/docs/hover-step)                 | Alt + H           | Option + H        | Editor + AUT |
| [Validate element text](https://help.testim.io/docs/validate-element-text)                       | Alt + V           | Option + V        | Editor + AUT |
| [Validate element visible](https://help.testim.io/docs/validate-element-visible)                 | Alt + L           | Option + L        | Editor + AUT |
| [Add custom validations and actions](https://help.testim.io/docs/custom-code)                    | Alt + J           | Option + J        | Editor       |
| [Add new group](https://help.testim.io/docs/groups#creating-a-group)     | Alt + G           | Option + G        | Editor       |
| Save test                                                                | Ctrl + S          | Cmd + S           | Editor       |
| Create new test                                                          | Alt + N           | Option + N        | Editor       |
| [Validate Element Visualization](https://help.testim.io/docs/validate-element-visualization)     | Alt + E           | Option + E        | Editor + AUT |
| [Validate Viewport Visualization](https://help.testim.io/docs/validate-viewport-visualization)   | Alt + P           | Option + P        | Editor       |
| [Validate Full-page Visualization](https://help.testim.io/docs/validate-full-page-visualization) | Alt + D           | Option + D        | Editor       |
| Select all (in lists)                                                    | Ctrl + A          | Cmd + A           | Editor       |
| [Toggle breakpoint](https://help.testim.io/docs/stop-pause-debug-tests)  | Ctrl + B          | Cmd + B           | Editor       |
| [Copy Step](https://help.testim.io/docs/editing-your-tests#copying-a-step)                       | Ctrl + C          | Cmd + C           | Editor       |
| [Cut Step](https://help.testim.io/docs/editing-your-tests#cutting-a-step)                        | Ctrl + X          | Cmd + X           | Editor       |
| [Paste Step](https://help.testim.io/docs/editing-your-tests#pasting-a-step)                      | Ctrl + V          | Cmd + V           | Editor       |
| [Delete Step](https://help.testim.io/docs/editing-your-tests#deleting-steps)                     | Delete, Backspace | Delete, Backspace | Editor       |
| [Add extract value step](https://help.testim.io/docs/extract-text)       | Alt + X           | Option + X        | Editor       |
| [Wait For Element Visible](https://help.testim.io/docs/wait-for#wait-for-element-visible)        | Alt + M           | Option + M        | Editor       |
| [Wait for element text](https://help.testim.io/docs/wait-for#wait-for-element-text)              | Alt + T           | Option + T        | Editor       |
