# Project users management

Invite & remove users from your project

You can manage the teammates that will be assigned to your project and their roles. On the **Project teammates** screen you can view a list of users that are currently assigned to your project. Any team member can invite new project teammates. A project owner can remove teammates and add/remove project owners.

## Adding a teammate to a Project

You can add a new teammate to your project. New teammates are able to view, edit, and run all the tests in your project, see test results, and invite other users.

:fa-arrow-right: **To add a teammate:**

1. In the main navigation, click the **Users** icon (on the Main navigation menu).

The **Project teammates** page is shown.

3. Enter the work email address (followed by a *space* or *enter*) for each teammate you wish to invite to the project.
4. Click **Invite**.

An invitation to join the project is sent to the address/es you entered, and they are added to the list on the **Project teammates** page. A user who has not yet accepted the invitation will appear as *Pending*.

![](https://files.readme.io/a651531-c664f51-Screen_Shot_2021-11-28_at_8.43.33.png "c664f51-Screen_Shot_2021-11-28_at_8.43.33.png")

## Removing a teammate

A project owner can remove a teammate from your project. This procedure will only remove a user from the project, but not from the company. To remove the user from the company’s user list, see [Projects and users management (company level)](https://help.testim.io/docs/project-and-user-management). (A teammate who is the only project owner cannot be removed from the project.)

> 📘
>
> The teammate's personal information will be automatically deleted within 90 days. If you want to delete it sooner, please contact support - [support@testim.io](mailto:support@testim.io).

:fa-arrow-right: **To remove a teammate:**

1. In the main navigation, click the **Users** icon.

![](https://files.readme.io/62ec3ab-users.png "users.png")

The **Project teammates** page is shown.

2. Select one or more teammates you wish to remove. Use **CTRL/CMD + click** to select more than one teammate.\
   3\. Click the **Delete User** button. Alternatively you can right click the selected teammates and select "Delete users".

![](https://files.readme.io/aec7e20-select-delete.png "select-delete.png")

![](https://files.readme.io/843e879-4a1895b-rightclickdelete.png "4a1895b-rightclickdelete.png")

The **Remove From Project** confirmation is shown.

3. Click **OK**.

![](https://files.readme.io/19e6767-confirmdelete.png "confirmdelete.png")

The selected teammates are removed from the project.

## Assigning and removing project owners

A project can have multiple project owners. A project owner can add and remove project owners on the **Project teammates** page.

:fa-arrow-right: **To add a teammate as project owner:**

1. In the main navigation, click the **Users** icon.

![](https://files.readme.io/e402149-Testim_446a.png "Testim 446a.png")

The **Project teammates** page is shown.

2. Right click on the teammate you wish to add as project owner, and select **Add Project Owner**.

![](https://files.readme.io/4feb2ce-6d483e0-Screen_Shot_2021-11-28_at_9.13.20.png "6d483e0-Screen_Shot_2021-11-28_at_9.13.20.png")

The teammate is added as a project owner.

:fa-arrow-right: **To remove a teammate as project owner:**

1. In the main navigation, click the **Users** icon.

![](https://files.readme.io/08f10f6-Testim_446a.png "Testim 446a.png")

The **Project teammates** page is shown.

2. Hover over the teammate you wish to remove as project owner, and click **Remove Project Owner**.

![](https://files.readme.io/4ad504b-cc07442-Screen_Shot_2021-11-28_at_9.14.37.png "cc07442-Screen_Shot_2021-11-28_at_9.14.37.png")

The **Remove Project Owner** confirmation is shown.

3. Click **Remove**.

![](https://files.readme.io/c1cc343-Testim_452a.png "Testim 452a.png")

The teammate is removed as project owner.
