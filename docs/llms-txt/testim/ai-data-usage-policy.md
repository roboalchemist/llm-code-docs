# AI Data Usage Policy

Testim AI solutions integrate the power of Microsoft Azure OpenAI Service with Testim products. Microsoft provides enterprise level data privacy and data security compliance. It maintains a multi-jurisdictional AI compliance program for its AI services. Copilot is designed to enhance user productivity in testing and accelerate the delivery of high-quality software without compromising the privacy of your data.

Tricentis is focused on ensuring responsible and trustworthy use of AI in our products. We were guided by the AI principles from the Organization for Economic Cooperation and Development (OECD), which include transparency, privacy adherence, human control, fair application and accountability. Additionally, we adhere in good faith to the Code of Conduct for Azure OpenAI Service.

## Affected Components

Testim offers the following features:

* [Testim Help Assistant](https://help.testim.io/docs/help-ai-assistant) - uses generative AI technology to assist users on how to use the software. You can ask questions (using prompts) and receive relevant help information in response.
* [Testim Copilot Coding Assistant](https://help.testim.io/docs/coding-assistant) - helps you write, understand, or fix the JS code that is part of any test step that involves writing custom JS code.
* [Agentic Test Automation for Salesforce](https://help.testim.io/docs/use-agentic-test-automation-for-salesforce#/) - uses generative AI technology and advanced Salesforce prebuilt steps to create and run tests in Salesforce environments.

## Customer Data Usage

Customer data is not used to train Microsoft Azure OpenAI Service models. Microsoft temporarily stores input and output content to provide meaningful human oversight and to monitor for and prevent abusive uses or outputs of the Azure OpenAI Service. Tricentis uses and analyzes inputs and output content, including customer feedback and data about how our customers use the AI solutions. This information is used to provide benchmarking information, improve AI solutions, and to develop new features, products, and services. Additionally, Tricentis provides meaningful human oversight to monitor for and prevent abusive uses or outputs of the AI solutions.

## Data Retention

The data in the user prompts, which may include code snippets, additional chat history data, or Salesforce metadata depending on the tool you use, is stored for a period of up to 24 hours. The prompt content is not used for model training purposes. The prompt data is isolated from other data and maintained for the sole purpose of maintaining the specific chat session. A unique and pseudo-random session ID for the chat session is stored. This means that the chat data cannot be directly traced back to the identity of the user/company that submitted the prompt.

## Data Processing

User prompts are used for the chat sessions only and not used for model training purposes, or any other purposes. Additional specialized context is added to the user prompt and submitted to the LLM through API.

Our [Data Processing Addendum](https://www.tricentis.com/legal-information/data-processing-addendum) applies to all Tricentis products and services, including Copilot solutions. Tricentis is committed to processing and transferring your data in accordance with global data protection laws.

## Data Location

All data is stored and processed in the US.

## Opting in/out

### Opting in:

**Help Assistant** - Doesn't require licenses and is available to all users.

**Testim Copilot Coding Assistant** - Requires company registration and individual acknowledgement to opt in:

* Company owners have to register to the service and assign available seats to specific users to give them access.
* Users can activate the Copilot interface on an individual basis. For example, by selecting the **Write code with AI** button.
* Before the first use, users must click **Got It** on the legal notice to acknowledge the terms of service.

**Agentic Test Automation for Salesforce** - Users can opt in on an individual basis:

* To activate, users can select the **Agentic Test Automation** button, then review and accept the terms of service to start using the tool. After they accept the terms of service, users are granted a free trial for two months.
* When your trial ends, your organization can purchase AI Credits, which are used across Tricentis tools and consumed according to the computational difficulty of your query. Select **Manage Credits** to inquire about buying credits for your organization.

### Opting out:

For Testim Copilot Coding Assistant, Company owners can opt out specific users by using the Copilot seats management interface.
