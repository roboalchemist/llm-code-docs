# Log in

> 📘 Salesforce Step
>
> This is a Salesforce step.

The Log in step logs into Salesforce using the selected [Persona](https://help.testim.io/docs/create-a-persona-and-add-users). All [Salesforce Steps](https://help.testim.io/docs/salesforce-steps), except the steps under the API Operations category, require you to be logged in to the Salesforce environment.

:fa-arrow-right: **To add a Log in step:**

1. In the editor, add a step by clicking the **+** button.
2. Under the **Salesforce steps** tab, click **Common operations** and select Log in step.\
   The **Log in** step is added and the following **Object** properties is displayed.

   <Image align="center" src="https://files.readme.io/793c6ea-2024-05-05_16-19-27.png" />
3. Under **Select login persona**, select the desired persona from the drop-down menu.
4. If you cannot find the desired persona, create a new one by following the instructions in [Creating a persona](https://help.testim.io/docs/create-a-persona-and-add-users).
5. When finished, click **Save**.
