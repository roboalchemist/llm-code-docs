# Why do you need Testim extension?

Testim uses Smart Locators for the authoring and execution of tests. We scan the entire DOM, extract its elements and score their attributes. The Testim Extension is an essential component for gaining access to the DOM and is required for authoring new tests by recording user flows.

![](https://files.readme.io/6829989-testim_extension.PNG "testim_extension.PNG")

To learn more about what you can do with the Testim Extension, see [Testim Extension - Overview](https://help.testim.io/docs/testim-extension-overview).

## Security disclaimer

Testim extension does not track user browsing outside the application and does not collect private information such as IP address and passwords. We do not alter source code. Any communication between our client and our servers is performed through a secure connection. We conform to Chrome's guidelines for extensions.
