# Close console tabs

The **Close console tabs** step closes all the console tabs in Salesforce console application. When testing a console application, it is recommended to place this step at the beginning of the test to ensure that you are beginning the test with a "clean slate".

:fa-arrow-right: **To add a Close Console Tabs step:**

1. In the editor, add a step by clicking the **+** button.
2. Under the **Salesforce steps** tab, click **Common operations** and select **Close console tabs** step.
3. When done, click **Save** to save the test.
