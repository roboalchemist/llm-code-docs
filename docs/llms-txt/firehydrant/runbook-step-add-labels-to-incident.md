# Add Labels to Incident

<Image alt="Add labels to incident step" align="center" width="650px" src="https://files.readme.io/74acf0d-image.png">
  Add labels to incident step
</Image>

[Incident Labels](https://docs.firehydrant.com/docs/incident-labels) are a way to track custom key-value pairs of data on incidents and then filter with them in the [Incidents page](https://app.firehydrant.io/incidents) and [Analytics page](https://app.firehydrant.io/analytics). This Runbook step will allow you to set these labels as part of Runbook automation, making it even easier to record information based on FireHydrant's extensive array of conditions.

To add this Runbook step, edit any Runbook, click "+ Add Step" at the bottom, and search for "labels." Each key-value pair should be in the format of `key=value` and on a separate line.
