# Archive Slack Incident Channel

> 📘 Note:
>
> This step is only available for Slack today.

<Image alt="Archive Incident Channel Runbook step" align="center" src="https://files.readme.io/9aa3cfe-image.png">
  Archive Incident Channel Runbook step
</Image>

This step automatically archives the incident channel based on the conditions you set. By default, the step will be set to execute **When incident milestone is "Retrospective Completed"**. However, you can change this as needed.

<Image alt="The default condition for this step is &#x22;When milestone is Retrospective Completed&#x22;" align="center" src="https://files.readme.io/d0903a5-image.png">
  The default condition for this step is "When milestone is Retrospective Completed"
</Image>
