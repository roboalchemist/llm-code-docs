# Other Analytics

Analytics and data exports are also available on various pages throughout the application on top of the primary **Analytics** section.

## Service Catalog

<Image alt="Analytics embedded into each Component's page" align="center" width="650px" src="https://files.readme.io/6a89117-CleanShot_2024-07-03_at_16.12.502x.png">
  Analytics embedded into each Component's page
</Image>

On the page for each component in the Service Catalog, there is an **Analytics** tab you can navigate to which will display the same charts as the MTTx Analytics, but with data for that specific component only.

This is available on Service, Functionalities, and Environments pages. For more information about the specific charts provided, visit [MTTx Analytics](https://docs.firehydrant.com/docs/analytics-mttx-analytics).

## Teams

<Image alt="Analytics embedded into each Team's page" align="center" width="650px" src="https://files.readme.io/5292585-CleanShot_2024-07-03_at_16.13.542x.png">
  Analytics embedded into each Team's page
</Image>

Likewise, Teams also have an analytics tab embedded into their respective pages. This shows data pared down to that specific Team (e.g., any incidents where this Team was assigned.

For more information, visit [MTTx Analytics](https://docs.firehydrant.com/docs/analytics-mttx-analytics).
