# End Linked Zoom Meetings

<Image alt="End Linked Zoom Meetings step" align="center" width="650px" src="https://files.readme.io/94af4da-Runbook_Step_-_End_Linked_Zoom_Meetings.jpg">
  End Linked Zoom Meetings step
</Image>

When one or more Zoom meetings are linked to a FireHydrant incident, sometimes the meetings continue even after everyone has left. This runbook will iterate through the list of attached meetings and preemptively end them so that meeting recordings can be processed and then attached as links to the incident.

This step requires no configuration and only requires that Zoom exists.
