<p>In this recipe, we will be building a simple e-commerce assistant agent with the <a href="https://docs.weaviate.io/agents" target="_blank" rel="noopener noreferrer" class="">Weaviate Query Agent</a>. This agent will have access to a number of Weaviate collections, and will be capable of answering complex queries about brands and clothing items, accessing information from each collection.</p>
<blockquote>
<p>üìö You can read and learn more about this service in our <a href="https://weaviate.io/blog/query-agent" target="_blank" rel="noopener noreferrer" class="">&quot;Introducing the Weaviate Query Agent&quot;</a> blog.</p>
</blockquote>
<p>To get started, we&#x27;ve prepared a few open datasets, available on Hugging Face. The first step will be walking through how to populate your Weaviate Cloud collections.</p>
<ul>
<li class=""><a href="https://huggingface.co/datasets/weaviate/agents/viewer/query-agent-ecommerce" target="_blank" rel="noopener noreferrer" class=""><strong>E-commerce:</strong></a> A dataset that lists clothing items, prices, brands, reviews etc.</li>
<li class=""><a href="https://huggingface.co/datasets/weaviate/agents/viewer/query-agent-brands" target="_blank" rel="noopener noreferrer" class=""><strong>Brands:</strong></a> A dataset that lists clothing brands and information about them such as their parent brand, child brands, average customer rating etc.</li>
</ul>
<p>Additionally, we also have access to some other unrelated datasets which you can use to add more capabilities and variety to other agents later on in the recipe:</p>
