<ul>
<li class=""><code>Get</code> queries are used for retrieving data objects.</li>
<li class=""><code>Aggregate</code> queries can be used to retrieve metadata, or aggregated data.</li>
<li class="">Operators such as <code>nearText</code> or <code>nearVector</code> can be used for vector queries.</li>
<li class="">Scalar filters can be used for exact filtering, taking advantage of inverted indexes.</li>
<li class="">Vector and scalar filters can be combined, and are available on both <code>Get</code> and <code>Aggregate</code> queries</li>
</ul>

<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="how-is-certainty-calculated">How is certainty calculated?<a href="#how-is-certainty-calculated" class="hash-link" aria-label="Direct link to How is certainty calculated?" title="Direct link to How is certainty calculated?" translate="no">â€‹</a></h3>
