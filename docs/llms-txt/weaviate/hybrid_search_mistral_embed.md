<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="hybrid-search">Hybrid Search<a href="#hybrid-search" class="hash-link" aria-label="Direct link to Hybrid Search" title="Direct link to Hybrid Search" translate="no">​</a></h2>
<p>The <code>alpha</code> parameter determines the weight given to the sparse and dense search methods. <code>alpha = 0</code> is pure sparse (bm25) search, whereas <code>alpha = 1</code> is pure dense (vector) search.</p>
<p>Alpha is an optional parameter. The default is set to <code>0.75</code>.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="hybrid-search-only">Hybrid Search only<a href="#hybrid-search-only" class="hash-link" aria-label="Direct link to Hybrid Search only" title="Direct link to Hybrid Search only" translate="no">​</a></h3>

Notice <code>alpha</code> is set to <code>0.80</code>, which means it is weighing the vector search results more than bm25. If you were to set <code>alpha = 0.25</code>, you would get different results.</p>
