<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="architecture">Architecture<a href="#architecture" class="hash-link" aria-label="Direct link to Architecture" title="Direct link to Architecture" translate="no">​</a></h2>
<p>The Personalization Agent is provided as a service on Weaviate Cloud.</p>
<p>When a user-specific recommendations request is made, the Personalization Agent analyses the user profile and any other known context to autonomously carry out the searches itself. The context may include data about the previous user interactions, information about the user themselves, and any other relevant information.</p>
<p>The Personalization Agent uses the contextual information to not only retrieve the most relevant recommendations, but also to rank them for the user.</p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="personalization-agent-visualized-workflow">Personalization Agent: visualized workflow<a href="#personalization-agent-visualized-workflow" class="hash-link" aria-label="Direct link to Personalization Agent: visualized workflow" title="Direct link to Personalization Agent: visualized workflow" translate="no">​</a></h2>
<p><img decoding="async" loading="lazy" alt="Weaviate Personalization Agent at a high level" src="/assets/images/personalization_agent_architecture_light-7af9ae0df988efda70805b023e0d7de0.png#gh-light-mode-only" title="Weaviate Personalization Agent at a high level" width="3611" height="1840" class="img_ev3q">
<img decoding="async" loading="lazy" alt="Weaviate Personalization Agent at a high level" src="/assets/images/personalization_agent_architecture_dark-a157453988d8b7912d439f9529f53c83.png#gh-dark-mode-only" title="Weaviate Personalization Agent at a high level" width="3611" height="1840" class="img_ev3q"></p>
<p>The Personalization Agent works as follows at a high level:</p>
<ul>
<li class="">Create a Weaviate-managed user collection, which will store each user&#x27;s profiles &amp; previous interactions for each user.</li>
<li class="">When a request for personalized recommendations is made, the Personalization Agent fetches the user data, and analyze it to determine patterns and preferences.</li>
<li class="">Perform initial searches in Weaviate based on the analysis to retrieve the most relevant recommendations.</li>
<li class="">Use appropriate generative models to determine any additional search strategies, and to re-rank fetched data as required.</li>
<li class="">Perform additional searches in Weaviate as needed to retrieve the final set of recommendations.</li>
<li class="">Combine and rank the recommendations based on the user&#x27;s profile and preferences.</li>
</ul>
<p>Then, the Personalization Agent returns the user-specific recommendations in the response. The response will also include intermediate outputs, such as the underlying search results from Weaviate.</p>
<p>Let&#x27;s dive into a little more detail about the Personalization Agent.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="user-profiles">User profiles<a href="#user-profiles" class="hash-link" aria-label="Direct link to User profiles" title="Direct link to User profiles" translate="no">​</a></h3>
<p>The Personalization Agent uses user profiles to provide personalized recommendations. This information is stored in a collection in your Weaviate instance under a specific name. The user profile may include the user&#x27;s preferences and previous interactions, such as their likes and dislikes.</p>
<p><img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - User Data Collection" src="/assets/images/personalization_agent_users_light-53c85e2b7eed8a46031de53ea9c6ad5f.png#gh-light-mode-only" title="Weaviate Personalization Agent - User Data Collection" width="3497" height="1180" class="img_ev3q">
<img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - User Data Collection" src="/assets/images/personalization_agent_users_dark-84ff5a032a67316f2eb053cde8163896.png#gh-dark-mode-only" title="Weaviate Personalization Agent - User Data Collection" width="3497" height="1180" class="img_ev3q"></p>
<p>As shown here, the user data collection can be updated over time. It can be updated with information about new users, or with new information about existing users.</p>
<p>This will help the Personalization Agent continue to learn and provide the most relevant, up-to-date recommendations to each user.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="recommendations">Recommendations<a href="#recommendations" class="hash-link" aria-label="Direct link to Recommendations" title="Direct link to Recommendations" translate="no">​</a></h3>
<p>There are two major components to the Personalization Agent&#x27;s recommendations, the searches it performs to retrieve the recommendations, and the ranking of the recommendations.</p>
<h4 class="anchor anchorTargetStickyNavbar_Vzrq" id="searches">Searches<a href="#searches" class="hash-link" aria-label="Direct link to Searches" title="Direct link to Searches" translate="no">​</a></h4>
<p>The Personalization Agent performs searches in Weaviate to retrieve the most relevant recommendations for the user from the specified collections.</p>
<p><img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - Searches" src="/assets/images/personalization_agent_search_light-3659b1529c6bd9b217d045228662a7bf.png#gh-light-mode-only" title="Weaviate Personalization Agent - Searches" width="3611" height="1840" class="img_ev3q">
<img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - Searches" src="/assets/images/personalization_agent_search_dark-c85f7eef239f355751b3de9f145ea60f.png#gh-dark-mode-only" title="Weaviate Personalization Agent - Searches" width="3611" height="1840" class="img_ev3q"></p>
<p>The diagram depicts that the search process may be based on a number of factors:</p>
<ul>
<li class="">The user&#x27;s profile and preferences, fetched from the user data collection.</li>
<li class="">The user&#x27;s previous interactions, fetched from the user data collection.</li>
<li class="">The recommendation context, such as the type of recommendations requested or any other relevant information.</li>
<li class="">Additional search strategies, as determined by the Personalization Agent.</li>
</ul>
<p>The Personalization Agent may perform multiple searches in Weaviate to retrieve the most relevant recommendations, before combining and ranking them.</p>
<h4 class="anchor anchorTargetStickyNavbar_Vzrq" id="re-ranking">(Re-)Ranking<a href="#re-ranking" class="hash-link" aria-label="Direct link to (Re-)Ranking" title="Direct link to (Re-)Ranking" translate="no">​</a></h4>
<p>The Personalization Agent uses multiple factors to rank the recommendations it retrieves from Weaviate, so that the final result set is tailored to the user&#x27;s preferences.</p>
<p><img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - (re)rank" src="/assets/images/personalization_agent_rank_light-9ea0086b9b099898d25a9a2b4381ea3a.png#gh-light-mode-only" title="Weaviate Personalization Agent - (re)rank" width="3611" height="1840" class="img_ev3q">
<img decoding="async" loading="lazy" alt="Weaviate Personalization Agent - (re)rank" src="/assets/images/personalization_agent_rank_dark-5522d46f23eabeef9548e2932bf58dfb.png#gh-dark-mode-only" title="Weaviate Personalization Agent - (re)rank" width="3611" height="1840" class="img_ev3q"></p>
<p>The rankings may be based on a number of factors:</p>
<ul>
<li class="">The user&#x27;s profile and preferences, fetched from the user data collection.</li>
<li class="">The user&#x27;s previous interactions, fetched from the user data collection.</li>
<li class="">The recommendation context, such as the type of recommendations requested or any other relevant information.</li>
<li class="">Additional ranking strategies, as determined by the Personalization Agent.</li>
</ul>
<p>This process ranks the combined result set as a whole, before serving them back in the response.</p>

<p>Here is an overview of how to use the this Weaviate Agent. For more detailed information, refer to the <a class="" href="/agents/personalization/usage">Usage</a> page.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="prerequisites">Prerequisites<a href="#prerequisites" class="hash-link" aria-label="Direct link to Prerequisites" title="Direct link to Prerequisites" translate="no">​</a></h3>
<p>This Agent is available exclusively for use with a Weaviate Cloud instance, and a supported version of the Weaviate client library.</p>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="example-usage">Example Usage<a href="#example-usage" class="hash-link" aria-label="Direct link to Example Usage" title="Direct link to Example Usage" translate="no">​</a></h3>
<p>To use the Personalization Agent, instantiate it with the following inputs:</p>
<ul>
<li class="">An instance of the Weaviate client (e.g. the <code>WeaviateClient</code> object in Python), connected to a Weaviate Cloud instance.</li>
<li class="">Name of the target collection to get personalized items from.</li>
<li class="">A list of user properties to base the personalization on.</li>
</ul>
<p>Note that the Personalization Agent persists. This means that if you have previously created a Personalization Agent, you can connect to it again without having to create a new one.</p>
