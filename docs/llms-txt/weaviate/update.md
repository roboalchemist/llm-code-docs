<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="update-object-properties">Update object properties<a href="#update-object-properties" class="hash-link" aria-label="Direct link to Update object properties" title="Direct link to Update object properties" translate="no">​</a></h2>
<p>This operation replaces the entire value of the specified properties only, leaving the unspecified properties. Provide the collection name, the object id, and the properties to update.</p>
<p>If you update the value of a <em>previously vectorized</em> property, Weaviate re-vectorizes the object automatically. This also reindexes the updated object.</p>
<p>However, if you add a <em>new</em> property to your collection definition, Weaviate only vectorizes the new objects. Weaviate doesn&#x27;t re-vectorize and re-index existing objects when a new property is defined, only when an existing property is updated.</p>
