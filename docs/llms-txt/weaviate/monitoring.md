<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="nodes-api-endpoint"><code>nodes</code> API Endpoint<a href="#nodes-api-endpoint" class="hash-link" aria-label="Direct link to nodes-api-endpoint" title="Direct link to nodes-api-endpoint" translate="no">​</a></h2>
<p>To get collection details programmatically, use the <a class="" href="/deploy/configuration/nodes"><code>nodes</code></a> REST endpoint.</p>
<!-- -->
<p>The <code>nodes</code> endpoint returns an array of nodes. The nodes have the following fields:</p>
<ul>
<li class=""><code>name</code>: Name of the node.</li>
<li class=""><code>status</code>: Status of the node (one of: <code>HEALTHY</code>, <code>UNHEALTHY</code>, <code>UNAVAILABLE</code>, <code>INDEXING</code>).</li>
<li class=""><code>version</code>: Version of Weaviate running on the node.</li>
<li class=""><code>gitHash</code>: Short git hash of the latest commit of Weaviate running on the node.</li>
<li class=""><code>stats</code>: Statistics for the node.<!-- -->
<ul>
<li class=""><code>shardCount</code>: Total number of shards on the node.</li>
<li class=""><code>objectCount</code> Total number of indexed objects on the node.</li>
</ul>
</li>
<li class=""><code>shards</code>: Array of shard statistics. To see <code>shards</code> details, set <code>output == verbose</code>.<!-- -->
<ul>
<li class=""><code>name</code>: Name of the shard.</li>
<li class=""><code>class</code>: Name of the collection stored on the shard.</li>
<li class=""><code>objectCount</code>: Number of indexed objects on the shard.</li>
<li class=""><code>vectorQueueLength</code>: Number of objects waiting to be indexed on the shard. (Available starting in Weaviate <code>1.22</code> when <code>ASYNC_INDEXING</code> is enabled.)</li>
</ul>
</li>
</ul>
