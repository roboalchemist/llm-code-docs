<p>In this recipe, we will use a Weaviate <a href="https://docs.weaviate.io/agents/transformation" target="_blank" rel="noopener noreferrer" class=""><code>TransformationAgent</code></a> to enhance our data in Weaviate. We will then use the agent to create additional properties for eaach of our objects in the collection.</p>
<blockquote>
<p>‚ö†Ô∏è The Weaviate Transformation Agent is designed to modify data in Weaviate in place. <strong>While the Agent is in technical preview, do not use it in a production environment.</strong> The Agent may not work as expected, and the data in your Weaviate instance may be affected in unexpected ways.</p>
</blockquote>
<p>The <code>TransformationAgent</code> is able to access a Weaviate collection of your chosing, and perform operations on the objects within it. However, each operation for the agent can be defined in natural language. The agent will then use an LLM to complete the instructions in the operation.</p>
<blockquote>
<p>üìö You can learn more about the new <code>TransformationAgent</code>, you can read our accompanyin <a href="https://weaviate.io/blog/transformation-agent" target="_blank" rel="noopener noreferrer" class="">&quot;Introducing the Weaviate Transformation Agent&quot;</a> blog</p>
</blockquote>
<p>To get started, we&#x27;ve prepared an open datasets, available on Hugging Face. The first step will be walking through how to populate your Weaviate Cloud collections.</p>
<ul>
