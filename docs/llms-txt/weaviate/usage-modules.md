<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="what-it-does">What it does:<a href="#what-it-does" class="hash-link" aria-label="Direct link to What it does:" title="Direct link to What it does:" translate="no">​</a></h3>
<ul>
<li class="">Periodically collecting usage data.</li>
<li class="">Uploading JSON reports to  either S3 or GCS.</li>
<li class="">Supports runtime configuration overrides.</li>
<li class="">Includes metrics and logging.</li>
<li class="">Verifies storage permissions before uploading.</li>
</ul>

<ul>
<li class=""><code>weaviate_usage_{gcs|s3}_operations_total</code>: Total number of operations for module labels (<code>operation</code>:collect<!-- -->/upload,  status: success, error).</li>

<li class=""><code>weaviate_usage_{gcs|s3}_uploaded_file_size_bytes</code>: Size of the uploaded usage file in bytes.</li>
</ul>
