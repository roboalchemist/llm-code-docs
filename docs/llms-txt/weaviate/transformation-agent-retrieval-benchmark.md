<p>Traditionally, AI system evaluation has relied heavily on human-written evaluation sets. Most notably, this process demands substantial time and resource investment, preventing most developers from creating and properly evaluating their AI systems.</p>
<p>The Weaviate Transformation Agent offers a breakthrough in AI evaluation by enabling the rapid generation of synthetic evaluation datasets!</p>
<p>In this notebook, we generate <strong>2,100 synthetic questions for the Weaviate Blogs (1 for each) in 63 seconds!</strong></p>
<p>We then use this dataset to report embedding recall (finding the source document in the haystack) between the Snowflake <a href="https://arxiv.org/abs/2412.04506" target="_blank" rel="noopener noreferrer" class="">Arctic 2.0</a> and <a href="https://arxiv.org/abs/2405.05374" target="_blank" rel="noopener noreferrer" class="">Arctic 1.5</a> embedding models. We find the following result:</p>
<table><thead><tr><th>Model</th><th>Recall@1</th><th>Recall@5</th><th>Recall@100</th></tr></thead><tbody><tr><td>Arctic 1.5</td><td>0.7995</td><td>0.9245</td><td>0.9995</td></tr><tr><td>Arctic 2.0</td><td>0.8412</td><td>0.9546</td><td>0.9995</td></tr></tbody></table>
