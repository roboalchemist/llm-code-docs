this page</button></div><div class="theme-doc-markdown markdown"><header><h1>Cluster status</h1></header><p><a href="https://weaviate.io/go/console?utm_source=docs&amp;utm_content=cloud" target="_blank" rel="noopener noreferrer" class="">Weaviate Cloud (WCD)</a> exposes two ways of checking the cluster status:</p>
<ul>
<li class="">A <strong><a href="#select-a-cluster" class="">web interface</a></strong> in the Weaviate Cloud console.</li>
<li class="">An <strong><a href="#api-endpoint" class="">API endpoint</a></strong> that provides information about your Weaviate instance.</li>
</ul>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="select-a-cluster">Select a cluster<a href="#select-a-cluster" class="hash-link" aria-label="Direct link to Select a cluster" title="Direct link to Select a cluster" translate="no">​</a></h2>
<div style="position:relative;padding-bottom:calc(54.10879629629629% + 50px);height:0"><iframe id="qp7x9q6fjr" src="https://app.guideflow.com/embed/qp7x9q6fjr" width="100%" height="100%" style="overflow:hidden;position:absolute;border:none" scrolling="no" allow="clipboard-read; clipboard-write" allowfullscreen="" allowtransparency="true"></iframe><script src="https://app.guideflow.com/assets/opt.js" data-iframe-id="qp7x9q6fjr"></script></div>
<br>
<details class="details_lb9f alert alert--info details_b_Ee" data-collapsed="true"><summary>Select your cluster</summary><div><div class="collapsibleContent_i85q"><ol>
<li class="">Open the <a href="https://weaviate.io/go/console?utm_source=docs&amp;utm_content=cloud" target="_blank" rel="noopener noreferrer" class="">Weaviate Cloud console</a>.</li>
<li class="">Open the <code>Clusters</code> list from the left sidebar.</li>
<li class="">Select the cluster you want to inspect.</li>
<li class="">The cluster details panel will open to the right.</li>
</ol></div></div></details>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="cluster-status-information">Cluster status information<a href="#cluster-status-information" class="hash-link" aria-label="Direct link to Cluster status information" title="Direct link to Cluster status information" translate="no">​</a></h2>
<p>You can see the following cluster statistics at the top of the page:</p>
<ul>
<li class=""><code>Dimensions stored</code></li>
<li class=""><code>Object count</code></li>
<li class=""><code>Cluster expires on</code> (only for Sandbox cluster)</li>
</ul>
<p>You can find the following information on the cluster details page:</p>
<ul>
<li class=""><code>Name of the cluster</code></li>
<li class=""><code>REST Endpoint</code></li>
<li class=""><code>gRPC Endpoint</code></li>
<li class=""><code>API Keys</code></li>
<li class=""><code>Roles</code></li>
<li class=""><code>Weaviate Database version</code></li>
<li class=""><code>Cloud provider</code></li>
<li class=""><code>Cloud region</code></li>
<li class=""><code>Created at</code></li>
<li class=""><code>Type</code> (Shared Cloud or Sandbox)</li>
<li class=""><code>Backup retention days</code> (only for Shared Cloud cluster)</li>
<li class=""><code>Availability SLA</code> (only for Shared Cloud cluster)</li>
</ul>
<p>Clusters also have an <code>Advanced configuration</code> section to configure or check the status of these additional settings:</p>
<ul>
<li class=""><code>Enable auto schema generation</code></li>
<li class=""><code>Allow all CORS origins</code></li>
</ul>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="api-endpoint">API Endpoint<a href="#api-endpoint" class="hash-link" aria-label="Direct link to API Endpoint" title="Direct link to API Endpoint" translate="no">​</a></h2>
<p>To get cluster details programmatically, use the <a class="" href="/deploy/configuration/nodes"><code>nodes</code></a> REST endpoint.</p>
<!-- -->
<p>The <code>nodes</code> endpoint returns an array of nodes. The nodes have the following fields:</p>
<ul>
<li class=""><code>name</code>: Name of the node.</li>
<li class=""><code>status</code>: Status of the node (one of: <code>HEALTHY</code>, <code>UNHEALTHY</code>, <code>UNAVAILABLE</code>, <code>INDEXING</code>).</li>
<li class=""><code>version</code>: Version of Weaviate running on the node.</li>
<li class=""><code>gitHash</code>: Short git hash of the latest commit of Weaviate running on the node.</li>
<li class=""><code>stats</code>: Statistics for the node.<!-- -->
<ul>
<li class=""><code>shardCount</code>: Total number of shards on the node.</li>
<li class=""><code>objectCount</code> Total number of indexed objects on the node.</li>
</ul>
</li>
<li class=""><code>shards</code>: Array of shard statistics. To see <code>shards</code> details, set <code>output == verbose</code>.<!-- -->
<ul>
<li class=""><code>name</code>: Name of the shard.</li>
<li class=""><code>class</code>: Name of the collection stored on the shard.</li>
<li class=""><code>objectCount</code>: Number of indexed objects on the shard.</li>
<li class=""><code>vectorQueueLength</code>: Number of objects waiting to be indexed on the shard. (Available starting in Weaviate <code>1.22</code> when <code>ASYNC_INDEXING</code> is enabled.)</li>
</ul>
</li>
</ul>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="enabled-modules">Enabled modules<a href="#enabled-modules" class="hash-link" aria-label="Direct link to Enabled modules" title="Direct link to Enabled modules" translate="no">​</a></h2>
<p>Each Weaviate instance has a set of enabled modules. This list will vary depending on the instance, due to the Weaviate version and Weaviate Cloud policies.</p>
<p>To see the list of enabled modules on a Weaviate Cloud instance:</p>
<!-- -->
<div class="row"><div class="col col--4"><ol><li>Open the Weaviate Cloud console.</li><li><p>Click on <code>Clusters</code> and select the cluster you want to check.</p></li><li><p>Scroll to the <code>Available modules</code> section under<!-- --> <!-- -->
<code>Advanced options</code>.</p></li><li><p>You can click on a module to open its documentation.</p></li></ol></div><div class="col col--8"><div class="card"><div class="card__image"><img src="/assets/images/weaviate-cloud-available-modules-b56cc7049b2e79c2bdae257bb9f7ff22.png" alt="Create a cluster"></div><div class="card__body">Available modules for your cluster.</div></div></div></div>
