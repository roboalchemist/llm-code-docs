<p>Weaviate&#x27;s integration with Cohere&#x27;s APIs allows you to access their models&#x27; capabilities directly from Weaviate.</p>
<p><a href="#configure-the-vectorizer" class="">Configure a Weaviate vector index</a> to use a Cohere embedding model, and Weaviate will generate embeddings for various operations using the specified model and your Cohere API key. This feature is called the <em>vectorizer</em>.</p>
<p>At <a href="#data-import" class="">import time</a>, Weaviate generates multimodal object embeddings and saves them into the index. For <a href="#vector-near-text-search" class="">vector</a> and <a href="#hybrid-search" class="">hybrid</a> search operations, Weaviate converts text queries into embeddings. <a href="#vector-near-media-search" class="">Multimodal search operations</a> are also supported.</p>
<p><img decoding="async" loading="lazy" alt="Embedding integration illustration" src="/assets/images/integration_cohere_embedding-9242506b42f2f7ab5f654b8d1cadbcb8.png" width="1354" height="430" class="img_ev3q"></p>
<h2 class="anchor anchorTargetStickyNavbar_Vzrq" id="requirements">Requirements<a href="#requirements" class="hash-link" aria-label="Direct link to Requirements" title="Direct link to Requirements" translate="no">​</a></h2>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="weaviate-configuration">Weaviate configuration<a href="#weaviate-configuration" class="hash-link" aria-label="Direct link to Weaviate configuration" title="Direct link to Weaviate configuration" translate="no">​</a></h3>
<p>Your Weaviate instance must be configured with the Cohere vectorizer integration (<code>multi2vec-cohere</code>) module.</p>
<details class="details_lb9f alert alert--info details_b_Ee" data-collapsed="true"><summary>For Weaviate Cloud (WCD) users</summary><div><div class="collapsibleContent_i85q"><p>This integration is enabled by default on Weaviate Cloud (WCD) instances.</p></div></div></details>
<details class="details_lb9f alert alert--info details_b_Ee" data-collapsed="true"><summary>For self-hosted users</summary><div><div class="collapsibleContent_i85q"><ul>
<li class="">Check the <a class="" href="/deploy/configuration/meta">cluster metadata</a> to verify if the module is enabled.</li>
<li class="">Follow the <a class="" href="/weaviate/configuration/modules">how-to configure modules</a> guide to enable the module in Weaviate.</li>
</ul></div></div></details>
<h3 class="anchor anchorTargetStickyNavbar_Vzrq" id="api-credentials">API credentials<a href="#api-credentials" class="hash-link" aria-label="Direct link to API credentials" title="Direct link to API credentials" translate="no">​</a></h3>
<p>You must provide a valid Cohere API key to Weaviate for this integration. Go to <a href="https://cohere.com/" target="_blank" rel="noopener noreferrer" class="">Cohere</a> to sign up and obtain an API key.</p>
<p>Provide the API key to Weaviate using one of the following methods:</p>
<ul>
<li class="">Set the <code>COHERE_APIKEY</code> environment variable that is available to Weaviate.</li>
<li class="">Provide the API key at runtime, as shown in the examples below.</li>
</ul>

<li class="">Join the (prepended) property values with spaces</li>

<li class="">Convert the produced string to lowercase</li>
