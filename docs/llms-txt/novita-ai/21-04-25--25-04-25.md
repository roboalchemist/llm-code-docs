# April 25, 2025 Product Updates

* **New Models: The GLM-4-0414 Series and the Qwen2.5-7B-Instruct Model**

## **GPUs Updates**

### **Improvements**

* **Optimized Storage Architecture for GPU Container Instances**

  To address the overlap between system disks and local disks, we have updated the instance storage policy. Effective immediately, the option to configure **a local disk for new instances has been discontinued**. Instances and templates that have already been created with local disk settings will remain unaffected.

  Following these changes, instances now support **system disk expansion**. For instances that have existing local disks, users can now unmount or resize the local disk space. When local storage is unmounted, all associated data will be automatically migrated to the system disk to ensure data continuity.\
  For more details, please refer to the [Instance Management](https://novita.ai/gpus-console/instances) page.

  Updated time: April 24, 2025
