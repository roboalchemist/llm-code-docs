# July 11, 2025 Product Updates

## **Model APIs Updates**

### **New Features**

* **MiniMax Speech-02 and MiniMax Voice Cloning Now Available on Novita**

  Novita now supports **MiniMax Speech-02-HD** and **MiniMax Speech-02-Turbo** audio generation models, along with the **MiniMax Voice Cloning** model.

  **MiniMax Speech-02** features powerful generalization capabilities, effortlessly handling speech across 32 languages, various accents, and a wide range of emotional tones. Since its release in May, it has consistently ranked No.1 globally on the Artificial Analysis leaderboard.

  **MiniMax Voice Cloning** is designed for use cases such as IP voice replication and timbre cloning, offering fast and accurate voice reproduction. It supports both mono and stereo voice cloning, enabling quick replication of a target voice from an audio sample.

  Get started now: [**MiniMax Speech-02-HD**](https://novita.ai/docs/api-reference/model-apis-minimax-speech-02-hd)\[[**********; MiniMax Speech-02-Turbo; MiniMax Voice Cloning.**********](https://novita.ai/docs/api-reference/model-apis-minimax-voice-cloning)]\([https://novita.ai/docs/api-reference/model-apis-minimax-speech-02-turbo](https://novita.ai/docs/api-reference/model-apis-minimax-speech-02-turbo))

  Updated time: July 10, 2025
* **GLM-4.1V-9B-Thinking Now Available on Novita**

  As a lightweight model with 9 billion parameters, **GLM-4.1V-9B-Thinking** delivers impressive performance—matching or even surpassing the 72B-parameter Qwen-2.5-VL-72B in 18 out of 28 authoritative benchmark evaluations.

  It excels in image-text understanding, scientific and mathematical reasoning, and video comprehension tasks. The model also supports ultra-high-resolution images up to 4K and accepts inputs with arbitrary aspect ratios.

  Get started now: [**GLM-4.1V-9B-Thinking**](https://novita.ai/models/llm/thudm-glm-4.1v-9b-thinking)**.**

  Updated time: July 10, 2025
* **Kimi-K2-Instruct Now Available on Novita**

  **Kimi K2** is a state-of-the-art Mixture-of-Experts (MoE) language model featuring 32 billion active parameters and 1 trillion total parameters.

  Trained with the Muon optimizer, Kimi K2 delivers exceptional performance in frontier knowledge tasks, reasoning, and coding. It is also carefully optimized for agent capabilities—designed specifically for tool use, advanced reasoning, and autonomous problem solving.

  Get started now: [**Kimi-K2-Instruct**](https://novita.ai/models/llm/thudm-glm-4.1v-9b-thinking)

  Updated time: July 11, 2025

## **GPUs Updates**

### **New Features**

* **Instance migration supports migrating system disks**

  Supports instance migration and also supports migrating the system disk. After migration, instance information (such as instance ID, egress IP, etc.) will not change. After migration, the mirroring will be stored in the Novita official mirroring repository. If the system disk is migrated, the system disk data will be packaged into a mirroring and migrated together to the new resource. The new mirroring contains the system disk data and is ready to use immediately.

  Updated time: July 7, 2025
* **GPU Instance metrics**

  Supports real-time monitoring of the following data:

  * CPU Utilization
  * Memory Utilization
  * GPU Utilization
  * Video Memory Utilization
  * System Disk Utilization

  Updated time: July 9, 2025
