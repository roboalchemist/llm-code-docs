# July 25, 2025 Product Updates

## **Model APIs Updates**

### **New Features**

* **New Model Available: Qwen3 Coder 480B A35B Instruct**

  Novita now supports **Qwen3 Coder 480B A35B Instruct**, the cutting-edge open-source coding model from the Qwen team. This model delivers performance on par with Claude Sonnet for agentic coding, browser automation, and other core development tasks, enabling highly efficient code generation and the handling of complex problems.

  Get started now: [**Qwen3 Coder 480B A35B Instruct**](https://novita.ai/models/llm/qwen-qwen3-coder-480b-a35b-instruct)

  Updated time: July 23, 2025
* **New Model Available: Qwen3 235B A22b Thinking 2507**

  Novita now supports Qwen3 235B A22B Instruct 2507, a powerful Mixture-of-Experts (MoE) model designed for ultimate performance and efficiency. By activating just 22B of its 235B parameters during inference, it dynamically handles diverse tasks with incredible speed.

  With its native 262k-token context window, the model excels at demanding use cases, including analyzing long documents, comprehending full code repositories, and performing complex multi-paragraph reasoning.

  Get started now: [**Qwen3 235B A22b Thinking 2507**](https://novita.ai/models/llm/qwen-qwen3-235b-a22b-thinking-2507)

  Updated time: July 23, 2025
