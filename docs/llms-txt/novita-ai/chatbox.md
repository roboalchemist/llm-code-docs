# ChatBox

With the Novita AI & Chatbox integration, you gain comprehensive access to Novita AI’s powerful suite of LLMs, including GLM 4.5, Deepseek, Qwen, and more.

This guide will walk you through integrating Novita AI’s LLMs with the ChatBox platform.

## What is ChatBox

ChatBox is an open-source, cross-platform AI chat application designed to interact with a variety of LLMs. It supports a wide range of LLM providers including OpenAI, Claude, Google Gemini, and Novita AI. With features like multi-language support, advanced prompt management, image generation via DALL·E 3, and collaboration tools, Chatbox enables developers and users alike to create and manage AI workflows efficiently.

## Integration Steps

### Step 1 Open Settings

Open the Chatbox app and find the **Settings** at the bottom left.

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/opensettings.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=0175ad0bbfa42d185a46040a68d43d1e" alt="opensettings.png" width="2596" height="1378" data-path="images/opensettings.png" />

### Step 2 Locate Novita AI

The Settings page defaults to the Model Providers tab. Scroll through the model list to find Novita AI.

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/locatenovita.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=b5acd768a1fc4936c75b9897ff70fc40" alt="locatenovita.png" width="1630" height="1024" data-path="images/locatenovita.png" />

### Step 3 Model Selection

Select Novita AI to view the model selection section, where several default models are available, including models supported by Novita such as deepseek-v3-0324, deepseek-r1-0528, glm 4.5, and more. For the complete list of available models, visit [https://novita.ai/models](https://novita.ai/models).

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/modelselection.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=1ad29e7bcd22e3ad12295c393eee6151" alt="modelselection.png" width="1630" height="1020" data-path="images/modelselection.png" />

Click **Fetch** to open the model list. From there, you can add models to the default list or remove existing ones as needed.

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/fetch.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=29017b9763749c6c3aa864514005140f" alt="fetch.png" width="1635" height="1026" data-path="images/fetch.png" />

### Step 4 Check API Key

Here, you can enter your API key. After entering it, click **Check**. If you see the message **“Connection successful”**, it means you're all set to start using it.

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/apikey.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=c17cb31bd89ac247aab79d1763845447" alt="apikey.png" width="1639" height="969" data-path="images/apikey.png" />

### Step 5 Start Using

Start a new chat. Enter "Hello" and receive a reply. You should see that the response is provided by Novita AI.

<img src="https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/startnewchat.png?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=809d8030413e089af2ffa7f9b4c2363e" alt="startnewchat.png" width="1915" height="1020" data-path="images/startnewchat.png" />

## Embedded Model Use

### Step 1 Locate Knowledge Base

Click **"Settings"**, then **"Knowledge Base"**. If there’s nothing there yet, you’ll be prompted to create a new one (the screenshot shows one already created).

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/locatekb.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=ebd5b18104ec7a59055317a004d7cf93" alt="locatekb.png" width="1911" height="1016" data-path="images/locatekb.png" />

### Step 2 Create Knowledge Base

When creating a new knowledge base, remember to select your desired **embedding model**, then click **"Create"**.

<img src="https://mintcdn.com/novitaai/bnRhXPrVKQGiPndx/images/createkb.png?fit=max&auto=format&n=bnRhXPrVKQGiPndx&q=85&s=7098df3e6d2ab955ab1e821451d32b95" alt="createkb.png" width="1913" height="1027" data-path="images/createkb.png" />

### Step 3 Start Using

Return to the **chat page**, and click the **knowledge base icon** below the input box. Then, select the knowledge base you just created. Now start your conversation with Novita AI LLM on ChatBox.

<img src="https://mintcdn.com/novitaai/OIO9nfACcX0vhcTI/images/startkb.png?fit=max&auto=format&n=OIO9nfACcX0vhcTI&q=85&s=b89f5c9fa504a958bf9f645cd85aa873" alt="startkb.png" width="1911" height="1017" data-path="images/startkb.png" />
