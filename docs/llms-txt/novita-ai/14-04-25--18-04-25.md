# April 18, 2025 Product Updates

## **New Features**

* **Sign Up with Google, GitHub, or Hugging Face Now Supported**

  Users can now register quickly and securely using their existing **Google**, **GitHub**, or **Hugging Face** accounts—no need to create a new password.

  Updated time: April 17, 2025
* **Rate Limit Display Added for LLM Usage in Console**

  The console now displays **rate limit information** for large language models (LLMs), allowing users to easily view the default request limits for each model and better plan their usage.

  In addition, a new rate limit increase request entry has been added, enabling users to submit requests for higher limits based on specific business needs.

  Updated time: April 17, 2025
* **New Video Model Added: Wan 2.1 I2V**

  The **Wan 2.1 I2V (Image-to-Video)** model is now available on the platform. It enables the generation of continuous video sequences from a single input image.

  This model delivers strong performance in **motion consistency**, **visual stability**, and **detail fidelity**, making it well-suited for **creative content generation**, **virtual character animation**, and related use cases.

  Updated time: April 16, 2025
* **Pricing Update for Video Model Wan 2.1 T2V**

  We’ve introduced a **resolution-based pricing model** for the **Wan 2.1 T2V (Text-to-Video)** model.\
  Users are now charged based on the output video resolution, enabling more flexible and cost-effective usage based on quality requirements.

  Updated time: April 16, 2025
* **Function Calling Now Supported on meta-llama/llama-4-maverick-17b-128e-instruct-fp8**

  The model `meta-llama/llama-4-maverick-17b-128e-instruct-fp8` supports the function call feature. For usage instructions, please refer to the [documentation](https://novita.ai/docs/guides/llm-function-calling).

  Updated time: April 14, 2025
* **Function Calling Now Supported on meta-llama/llama-4-scout-17b-16e-instruct**

  The model `meta-llama/llama-4-scout-17b-16e-instruct` supports the function call feature. For usage instructions, please refer to the [documentation](https://novita.ai/docs/guides/llm-function-calling).

  Updated time: April 14, 2025
