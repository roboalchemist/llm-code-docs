# August 1, 2025 Notice

## **Model APIs Updates**

### **New Features**

* **New Model Available: Seedream 3.0 Text to Image**

  The **Seedream 3.0 Text to Image** model is a new generation text-to-image model built on ByteDance’s large visual model. It offers accurate understanding of textual input and the ability to generate images with high text-image alignment and aesthetic quality. The model supports direct output at resolutions below 2048, with significantly improved performance across various scenarios.

  Get started now: [**Seedream 3.0 Text to Image**](https://novita.ai/docs/api-reference/model-apis-seedream-3-0-t2i)
