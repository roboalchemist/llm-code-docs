# Introduction

<Tip>
  **Use Novita Docs with your AI agent**

  **Quick start — paste into your agent chat (no install needed):**

  ```
  Read https://novita.ai/docs/skill.md and follow the instructions to integrate Novita AI.
  ```

  **Optional install — if your agent runner supports skills (recommended for frequent use):**

  ```bash  theme={"system"}
  npx skills add novitalabs/novita-skills --skill novita-docs
  ```

  If your agent can run shell commands, you can ask it to run the install command for you.

  [View on GitHub →](https://github.com/novitalabs/novita-skills)
</Tip>
