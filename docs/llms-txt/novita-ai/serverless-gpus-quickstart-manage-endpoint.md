# Manage Serverless Endpoint

## Check Serverless Endpoint

Go to the <a href="https://novita.ai/gpu-instance/console/serverless" target="_blank">Serverless Console</a> to check all the Serverless Endpoints you have created.

Click the **"Dropdown"** button at the bottom right corner of the target serverless endpoint to view the number of running Workers, Worker logs, and advanced operation options for the serverless endpoint.

## Modify Serverless Endpoint

Click the **"Modify"** button to modify the configuration of the target serverless endpoint in the subsequent pop-up window. Except for the "App Name", all other configurations can be modified. For details, refer to: [Configure Serverless Endpoint](/guides/serverless-gpus-quickstart-create-endpoint#step-3-configure-serverless-endpoint).
