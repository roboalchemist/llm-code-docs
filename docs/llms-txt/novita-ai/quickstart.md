# Quickstart

Novita AI is the go-to inference platform for AI developers seeking a low-cost, reliable, and simple solution for shipping AI models. This is a quickstart guide to help you set up your account and get started.

<Tip>
  If you have questions, please check our [FAQs](/guides/faq) first or [reach out to our support team on Discord](https://discord.gg/YyPRAzwp7P).
</Tip>

## 1. Log into [novita.ai](https://novita.ai/user/login?utm_source=getstarted)

You can log in using Google or GitHub authentication, and a new account will be automatically created on your first login. You can also [sign up using your email address](https://novita.ai/user/register?utm_source=getstarted).

## 2. Manage API Key

Novita AI authenticates API access using Bearer authentication with an API Key in the request header, e.g., "Authorization: Bearer \{API Key}".

If you need to access APIs, you can go to [Key Management](https://novita.ai/settings/key-management?utm_source=getstarted) setting page to create or manage your API keys.

## 3. Maintain sufficient credit balance in your account

We offer new users a voucher with some credit to try our products. To add more credit, please visit [Billing and Payments](https://novita.ai/billing?utm_source=getstarted) and follow the guide on [Payment Methods](/guides/payment-methods).

Also, to avoid any interruption to your service, please ensure your account has sufficient credit balance, and [Setup Automatic Top-up](/guides/auto-top-up) is a recommended practice.
