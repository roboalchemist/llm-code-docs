# February 29, 2024

We've launched some new features lately, here are the details:

1. You can now check if the civitai models exist on novita.ai. Simply copy the civitai model page URL and paste it into the search box at [https://novita.ai/model-api/model](https://novita.ai/model-api/model).

2. SDXL Lightning LoRAs are now supported in the API /v3/async/txt2img. Available models include sdxl\_lightning\_2step\_lora, sdxl\_lightning\_4step\_lora, and sdxl\_lightning\_8step\_lora.
