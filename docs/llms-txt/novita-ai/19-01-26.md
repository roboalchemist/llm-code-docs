# January 19, 2026 Product Updates

## **GPUs Updates**

### **New Features**

* **Added Automatic Instance Migration for newly GPU instance**

  The GPU **Automatic Instance Migration** entry has been relocated to the **Instance Management** page, and now supports configuring migration policies at the **instance** level. To ensure business continuity, **Automatic Instance Migration** will be enabled by default for **newly created resources starting at 2026**. You can manage migration settings via `More Operations`→  `Automatic Instance Migration` or configure migration attributes through the **OpenAPI**.
