# Jupyter Lab

Jupyter Lab is an interactive notebook that supports over 40 programming languages. Jupyter Lab is an open-source web application that facilitates the creation and sharing of literate programming documents, supporting real-time code execution, mathematical equations, visualizations, and markdown editing.

The official templates of GPU instance come with a pre-installed Jupyter Lab environment. To enable this feature, please checkout the **"Start Jupyter Notebook"** option when deploying the instance.

<Warning>
  Some official templates may not support the Jupyter Lab environment. If you have specific requirements, please reach out to us through the <a href="https://discord.gg/YyPRAzwp7P" target="_blank">Discord channel</a>.
</Warning>

## Accessing Jupyter Lab

While the GPU instance is running, you can click the **"Connect"** button in the bottom right corner of the instance, then click **"Connect to Jupyter Lab"** in the pop-up window to access the Jupyter Lab webpage deployed on that instance.
