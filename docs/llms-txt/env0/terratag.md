# Terratag

[Terratag](https://www.terratag.io/) is a powerful CLI tool developed by env zero that simplifies the process of applying tags or labels across your entire Infrastructure-as-Code (IaC) codebase.

## What is Terratag?

Terratag automatically applies consistent tags or labels to AWS, GCP, and Azure resources across your Terraform or Terragrunt files. This tool eliminates the complexity of manual tagging and reduces the risk of human error when managing resource metadata at scale.

## Key features

Terratag provides several capabilities that enhance your Infrastructure-as-Code workflows:

* **Automated tagging**: Apply tags across entire sets of Terraform or Terragrunt files
* **Multi-cloud support**: Works with AWS, GCP, and Azure resources
* **Retroactive tagging**: Tag previously deployed IaC resources
* **Error reduction**: Eliminates manual tagging mistakes
* **Cost management**: Enable better cost allocation and reporting through consistent tagging
* **Organization**: Improve resource organization and governance

## How it works

Terratag scans your Terraform or Terragrunt files and automatically adds the specified tags to all supported resources. This ensures consistent tagging across your entire infrastructure without requiring manual updates to each resource definition.
