# 🧠 Drift Cause Analysis: Understand Why Drift Happened

> Drift Cause Analysis helps you move beyond detection by explaining what most likely caused each drifted resource.   env zero uses AI to analyze context from your environment, code, and history to identify causes like manual changes, unapplied commits, variable updates, and version mismatches. You'll see a clear explanation next to each drifted resource, along with links to related commits or cloud events, when available.
