# Enable env zero Secrets

> Control whether users can store sensitive data like variables and SSH keys in env zero

Sensitive data storage within the env zero platform is governed by organizational policies, allowing administrators to disable this feature to prevent users from storing sensitive information if required.

This policy applies to variables, SSH keys, and deployment credentials.

To modify the policy, navigate to \*\*Organization Settings \*\*and then select **Policies**.

<Note>
  At this stage, only organizations using self-hosted agents can modify this policy.
</Note>
