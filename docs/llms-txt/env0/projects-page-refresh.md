# ✨ A Freshly Updated Projects Page: More Useful, More Powerful, and Easier to Navigate

> The Projects page in env zero is refreshed to make it easier to understand, organize, and act on your projects at a glance.

We have rolled out a refreshed **Projects** page with improvements designed to make it easier to understand, organize, and act on your projects at a glance.

## What's new

### 🔍 Get a clearer project overview

* See richer project metadata at a glance
* View project details, environment status, and creation and update dates
* Understand project state without opening each project
* Follow project hierarchies more easily with sub projects shown directly in the list

### 🏷️ Organize projects with tags

* Group related projects using tags
* Reuse tags across projects
* Filter projects by tags to focus on what matters

### 🎨 Work more efficiently with a clearer UI

* Scan projects more easily with an improved layout
* Move faster with more consistent interactions

These updates are part of an ongoing effort to make project management in env zero more scalable and friendly, especially as your number of projects grows.

As always, we would love your feedback. 🙏🏼
