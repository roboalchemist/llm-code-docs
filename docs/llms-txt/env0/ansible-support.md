# 🅰️ Ansible Support

> Ansible is an open-source automation tool used for configuration management, application deployment, and task automation. It allows users to define their systems and infrastructure as code using simple, human-readable YAML files.
We are excited to announce that Ansible now gets first-class support in env zero!

[Ansible](https://www.ansible.com/) is an open-source automation tool used for configuration management, application deployment, and task automation. It allows users to define their systems and infrastructure as code using simple, human-readable YAML files. We are excited to announce that Ansible now gets first-class support in env zero!

## ✨ Ansible - How does it work ✨

Deploy your playbooks as env zero environments, with all the benefits you know and love, like Approval Flow, Plan on Pull Request, Drift Detection, TTL, Scheduling, and a whole lot more!

In order to manage your Ansible deployments using env zero, you will need to create a new Ansible template, or run an environment directly from a VCS and choose the Ansible type:
