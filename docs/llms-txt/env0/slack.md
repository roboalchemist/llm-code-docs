# Slack Integrations

> Connect env zero to Slack for deployment notifications using Incoming Webhooks integration

## Prerequisites - Slack

To start using Slack notifications with env zero, you must first install the Slack Incoming Webhooks integration.

1. Navigate to the [Incoming WebHooks](https://env0.slack.com/apps/A0F7XDUAZ-incoming-webhooks) page in the Slack App Directory.
2. Click on "Add to Slack"
3. Select the channel you'd like env zero to post to and click "Add Incoming WebHooks Integration"
4. Save the `Webhook URL` you will be shown.
