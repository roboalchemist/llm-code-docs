# Using the Lightlytics Plugin

> Run Lightlytics simulations through env zero deployments using the Lightlytics plugin

## Overview

This plugin executes Lightlytics and lets you run your Lightlytics Simulations through your env zero deployment.

For more information, check out the [Lightlytics Plugin git repository](https://github.com/lightlytics/env0-lightlytics-plugin).
