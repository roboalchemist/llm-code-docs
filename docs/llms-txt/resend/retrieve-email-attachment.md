# Retrieve Attachment

> Retrieve a single attachment from a sent email.

## Path Parameters

<ParamField path="id" type="string" required>
  The Attachment ID.
</ParamField>

<ParamField path="email_id" type="string" required>
  The Email ID.
</ParamField>
