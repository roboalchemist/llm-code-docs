# JetBrains IDEs (Private/Enterprise-SaaS)

Once you’ve joined your team’s Tabnine account, the next step is to install Tabnine from the JetBrains Marketplace.

\
This article will guide you through onboarding using the Tabnine Enterprise plugin.

## Install from the JetBrains Marketplace <a href="#install-from-marketplace" id="install-from-marketplace"></a>

In your JetBrains IDE, go to S**ettings** > P**lugins** and install the `Tabnine Enterprise (Self-Hosted)` plugin from the JetBrains plugin marketplace:

1. In your JetBrains IDE, go to **Settings > Plugins.**
2. Search for **Tabnine Enterprise (Self-Hosted)** and select it (don't mix it with the other Tabnine extension).
3. Click **Install.**

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2FRA3Fgq8XFnxuCzHSvNXz%2Fimage.png?alt=media&#x26;token=2afaf9a5-6022-49b0-89a9-2e289b63d301" alt=""><figcaption></figcaption></figure>

4. Once the plugin is installed, click **Restart IDE.**

## Configure the Tabnine Server URL

Once you restart the IDE and the plugin is initialized, you’ll be prompted to set the URL of your Tabnine Enterprise Server. If you don’t have the URL, contact your Tabnine admin within your organization.

Enter the URL and click **OK:**

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2Fgit-blob-e525645d4ee2b7256d5db3d4d95b5aa8a884ef53%2Fjb3%20(1).webp?alt=media" alt="" width="375"><figcaption></figcaption></figure>

If for some reason you are not prompted, you can set the server URL by clicking on Settings (⚙), then select Settings in the dropdown menu.

Search for “Tabnine,” then navigate to **Tools** on the left-side menu. Select **Tabnine**.

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2FU2jg8szNTcTgnU9W9EC7%2Fimage.png?alt=media&#x26;token=411e6aa5-8106-47be-aaba-0bbc3cb6b5a2" alt=""><figcaption></figcaption></figure>

In the settings, under Engine, type/paste your Tabnine Enterprise URL, [https://console.tabnine.com](https://console.tabnine.com/), in the adjacent bar. Then press <mark style="background-color:blue;">**OK**</mark>.

## Sign in to Tabnine

Next, navigate to the Tabnine prompt on the bottom menu in order to sign in.&#x20;

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2FKLYoT6mCNLwuU1GhQjAx%2Funknown.png?alt=media&#x26;token=a91fa7e0-3fbc-4622-8db0-d4e9915dc103" alt=""><figcaption></figcaption></figure>

1. Click the status bar and select **Sign in to Tabnine**:

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2F5CCUznUlsdsYYssf5ytj%2Fimage.png?alt=media&#x26;token=3bf29d38-7a43-45e2-8537-c0e14749c54f" alt=""><figcaption></figcaption></figure>

2. In the browser window that opens, click **Sign In:**

<figure><img src="https://3436682446-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FY2qxVf5VTm3fmwP4B4Gx%2Fuploads%2Fgit-blob-4cd77c284020ff52e597ba49996e199e8a54eb9d%2FSSO%20sign%20in.png?alt=media" alt=""><figcaption></figcaption></figure>

If your organization doesn't have SSO, you'll see the following screen. Sign in using your user credentials:
