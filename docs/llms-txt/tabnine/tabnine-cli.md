# Tabnine CLI

Tabnine CLI is your AI-powered coding assistant in the terminal.

{% hint style="warning" %}
**Prerequisite:** Tabnine CLI requires **Tabnine Agents** to be enabled for your team. Ask your admin to enable Agents in the Admin Console.
{% endhint %}
