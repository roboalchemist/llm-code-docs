# Create workflow node permissions

```
POST 
/v1/workflows/:workflow_identifier/nodes/:node_identifier/permissions
```

This route allows you to define permissions for a specific node in a workflow using rules and combinators.<br />To learn more about workflows, see the [documentation](https://docs.port.io/workflows/overview).
