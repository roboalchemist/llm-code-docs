```
GET 
/v1/blueprints/:blueprint_identifier/permissions
```

This route allows you to retrieve the permissions of a blueprint.<br /><br />To learn more about permissions, check out the [documentation](https://docs.port.io/build-your-software-catalog/set-catalog-rbac/examples).
