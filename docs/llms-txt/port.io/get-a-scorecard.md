# Get a scorecard

```
GET 
/v1/blueprints/:blueprint_identifier/scorecards/:scorecard_identifier
```

This route allows you to fetch a specific scorecard. A scorecard is a set of rules that define the quality of a blueprint.<br /><br />To learn more about scorecards, check out the [documentation](https://docs.port.io/scorecards/overview).
