# Get a blueprint's scorecards

```
GET 
/v1/blueprints/:blueprint_identifier/scorecards
```

This route allows you to fetch all scorecards for a given blueprint. A scorecard is a set of rules that define the quality of a blueprint.<br /><br />To learn more about scorecards, check out the [documentation](https://docs.port.io/scorecards/overview).
