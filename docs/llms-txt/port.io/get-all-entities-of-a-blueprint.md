# Get all entities of a blueprint

```
GET 
/v1/blueprints/:blueprint_identifier/entities
```

This route allows you to fetch all entities in your software catalog based on a given blueprint.<br /><br />To learn more about entities, check out the [documentation](https://docs.port.io/build-your-software-catalog/sync-data-to-catalog/#entities).
