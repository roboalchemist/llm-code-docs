# Get an action's permissions

```
GET 
/v1/actions/:action_identifier/permissions
```

This route allows you to fetch the permissions of a self-service action.<br /><br />To learn more about action RBAC, check out the [documentation](https://docs.port.io/actions-and-automations/create-self-service-experiences/set-self-service-actions-rbac/).
