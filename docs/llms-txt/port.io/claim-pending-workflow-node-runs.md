# Claim pending workflow node runs

```
POST 
/v1/org/:org_id/workflows/runs/claim-pending
```

This route allows you to claim pending workflow node runs for processing. Claimed runs will not appear from this endpoint until their visibility timeout expires.<br />To learn more about workflows, see the [documentation](https://docs.port.io/workflows/overview).
