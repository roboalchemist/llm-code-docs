# Get an action run's details

```
GET 
/v1/actions/runs/:run_id
```

This route allows you to fetch the details of an action run.<br /><br />To learn more about action runs, check out the [documentation](https://docs.port.io/create-self-service-experiences/reflect-action-progress/).<br />

Version parameter

Set the `version` parameter to `v2` for the latest version of the API.
