# Review auto-discovery invocation suggestions

```
POST 
/v1/ai/entities-auto-discovery/:invocation_id/review
```

This route allows you to review suggestions for an auto-discovery invocation. You can accept or reject individual suggestions, or accept or reject all suggestions at once.<br />To learn more about the auto-discovery feature, see the [documentation](https://docs.port.io/build-your-software-catalog/catalog-auto-discovery).
