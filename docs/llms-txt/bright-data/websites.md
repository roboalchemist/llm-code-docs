  <Accordion title="How do I scrape Public Web Data from LinkedIn?">
    When it comes to gathering public web data from LinkedIn, you have two straightforward options to choose from. Both methods are effective, depending on your specific needs and resources.

    **Option 1: Ready-Made Datasets**

    The first option is to utilize ready-made datasets available on the Datasets Marketplace. You can access the Datasets Marketplace by going to "Web Data" from the sidebar, then selecting "Datasets Market" from the top menu.

    These datasets are pre-compiled collections of LinkedIn data, such as user profiles or company information. This option is ideal for those who want to save time and effort, as you can simply download the data you need without having to scrape it yourself. It’s a great solution for businesses or researchers looking for quick access to large volumes of data with minimal technical overhead.

    **Option 2: Use the Web Scraping API**

    This option lets you easily scrape LinkedIn in real-time via API. You can instantly scrape the information you need by going [here](https://brightdata.com/cp/scrapers/browse?domain=linkedin.com)

    Information is readily available for LinkedIn people profiles, company information etc.

    **Option 3: Unlocker API or Browser API**

    The third option is to actively scrape the data using either the Unlocker API or the Browser API to gather LinkedIn data on your own. This approach allows for more flexibility and customization. With Unlocker API or Browser API, you can target specific data points and update your data in real-time. This method is perfect for users who need tailored datasets that aren’t available in pre-made form or require fresh, up-to-date information.

    All methods have their own advantages. Ready-made datasets provide convenience and efficiency, while web scraping tools offer control and customization. Choose the one that best fits your needs, and get started with accessing LinkedIn data today!
  </Accordion>

  <Accordion title="How do I scrape Public Web Data from Google?">
    When it comes to gathering public web data from Google, there is a straightforward and powerful option available to you - the **SERP API**

    The most effective way to collect data from Google is by using the SERP API. The SERP API (Search Engine Results Page API) allows you to retrieve search engine results directly from Google without having to scrape the site manually. This option is particularly useful for anyone needing access to Google's search data, such as rankings, snippets, images, maps, and other search features.

    You can read more about SERP API and the customisation options in the [SERP API documentation](https://docs.brightdata.com/scraping-automation/serp-api/introduction).

    Using the SERP API offers several advantages. Firstly, it simplifies the process of collecting search data, allowing you to focus on analyzing results rather than dealing with the technicalities of web scraping. Additionally, it provides a reliable way to access fresh, real-time search data, ensuring that the information you receive is up-to-date and accurate.

    The SERP API is also highly customizable, letting you specify parameters such as keywords, location, language, and device type. This flexibility makes it a powerful tool for marketers, SEO specialists, and researchers who need specific insights into Google's search results.

    While using the SERP API requires some setup and understanding of API requests, it's a more streamlined and reliable method than traditional web scraping, which can be challenging due to Google's frequent updates and anti-scraping measures.

    If you're looking to gather Google search data efficiently and effectively, the SERP API is the way to go. Start exploring its capabilities today to unlock valuable insights from Google search results!
