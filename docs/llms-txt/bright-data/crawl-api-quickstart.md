# Crawl API Overview

> Crawl API allows you to extract, map, and transform content from any domain into structured, LLM-compatible data. Whether you're building AI training datasets, performing SEO audits, or integrating content into your workflows, Crawl API delivers clean, ready-to-use data in formats like HTML, Markdown, and JSON.

## Key Features

* Crawl Any Domain. Extract Any Data.
* Discover and map full domain structures with a single API call
* Collect static and dynamic content in Markdown, HTML, or plain text
* Automate at scale with scheduler support and webhook delivery
* Developer-first: works with Python, Node.js, BeautifulSoup, Cheerio, and more
* No-code option available via our Control Panel

## Use Cases

* AI/LLM training data collection
* SEO audits and site structure analysis
* Competitive research and content aggregation
* Compliance and accessibility checks
* Content migration and archiving
