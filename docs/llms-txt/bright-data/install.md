# How to install the extension

> Learn how to install the Bright Data extension: download, add to Chrome, pin it, and sign in to start using it with your Bright Data account.

<Info>Make sure you have a Bright Data account or Create new Bright Data account in [brightdata.com](https://brightdata.com/)</Info>

<Steps>
  <Step title="Sign in" />

  <Step title="Download the extension">
    You can find the extension link [here](https://chromewebstore.google.com/u/1/detail/bright-data/efohiadmkaogdhibjbmeppjpebenaool?hl=en).
  </Step>

  <Step title="Add to Chrome" />

  <Step title="Go to browser settings and Manage Extension" />

  <Step title="Make sure its On and go to &#x22;Details&#x22;" />

  <Step title="Toggle On &#x22;Pin to toolbar" />

  <Step title="Once the &#x22;Pin to toolbar&#x22; turn on click see on the Bright Data icon in tool bar" />

  <Step title="Sign in with your account">
    Once you Bright Data account is active click "Sign in with you account" and type your credentials in the new Control panel tab that will be open.
  </Step>

  <Step title="Click the extension icon, turn it on and start working with the extension." />
</Steps>
