# Five-Minute How-To: Easy Scraper

This guide helps you collect your first dataset using **Easy Scraper** in under five minutes.\
No code. No setup. Just select a scraper, run it, and download your data.

## What you’ll do in 5 minutes

* Open Easy Scraper
* Select a ready-made scraper
* Enter inputs
* Run the scraper
* Download the results

## Step 1: Open Easy Scraper

1. Log in to your Bright Data account.
2. From the dashboard, go to **Easy Scraper**.
3. You’ll see a list of available scrapers such as Amazon, Airbnb, Instagram, and LinkedIn.

## Step 2: Choose a Scraper

1. Click the scraper that matches the website you want to collect data from.
2. This opens the scraper configuration page.

Each scraper is prebuilt for a specific platform and data type.

## Step 3: Enter Inputs

1. Provide the required inputs for the selected scraper, such as:
   * Keywords
   * URLs
   * Profiles, posts, or listing links (depending on the scraper)
2. Make sure the inputs are valid and complete.

## Step 4: Optional Result Limit

1. Set a maximum number of records if needed.
2. Leave it empty to collect all available data.

This is useful for testing or small sample collections.

## Step 5: Start Collecting

1. Click **Start Collecting**.
2. You will be redirected to the **Logs** page.

From the Logs page, you can view:

* Job status
* Snapshot ID
* Number of records collected
* File size
* Any errors

## Step 6: Download Your Data

1. Once the job is completed, download the dataset.
2. Available formats include:
   * JSON
   * CSV
   * NDJSON
   * JSON Lines

Choose the format that best fits your workflow.
