# Log Exports

Cloudsmith allows you to export [Client Logs](https://help.cloudsmith.io/docs/client-logs) and [Audit Logs](https://help.cloudsmith.io/docs/repo-audit-logs) This enables you then to ingest these logs into another reporting or analytics tool, and helps provide greater insights into package usage and user activity.
