## Create a Team

Creating a team requires that you have first created an organization (See [Organizations](https://help.cloudsmith.io/docs/organisations) for full details).

To create a team:

1. Navigate to Teams in the main menu
2. Click "New Team"

<img src="https://files.readme.io/48ff70609da4229414aab967c6a56d28ff1be5bcc389fc52e4db15780537e394-app.cloudsmith.com_login_totp_two_factor_tokeneyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ0b2tlbl90eXBlIjoiMmZhX3JlcXVpcmVkIiwiZXhwIjoxNzI4MTE5Njg2LCJpYXQiOjE3MjgwMzMyODYsImp0aSI6ImRiMDg4NTA3ZjcyZDQwNTk4YjU3YWY5ZTJiOGFhOGY2IiwidXNlcl9pZCI6ImND.png" border="true" caption="New Team button" sizing="100" />

You are then presented with the Add New Team form:

<img src="https://files.readme.io/2ac56ccf935b94a15120df046b11be3dfd2053ebdda9ea0cae2f34703845e53d-app.cloudsmith.com_login_totp_two_factor_tokeneyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ0b2tlbl90eXBlIjoiMmZhX3JlcXVpcmVkIiwiZXhwIjoxNzI4MTE5Njg2LCJpYXQiOjE3MjgwMzMyODYsImp0aSI6ImRiMDg4NTA3ZjcyZDQwNTk4YjU3YWY5ZTJiOGFhOGY2IiwidXNlcl9pZCI6_1.png" border="true" caption="Add New Team form" />

A Team name is required while the description is optional. "Visibility" controls if non-members can see the team or not:

|         |                                  |
| :------ | :------------------------------- |
| Visible | non-members can view the team    |
| Hidden  | non-members cannot view the team |
