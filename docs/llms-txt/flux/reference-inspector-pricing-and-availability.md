# Pricing and availability

Read this [tutorial](https://docs.flux.ai/flux/tutorials/components-procurement) if you want to learn more about pricing and availability. 

If the selected object contains a [property](https://docs.flux.ai/flux/reference/reference-inspector-properties) called "Manufacturer Part Number", Flux will automatically search for pricing and availability in leading distributors:

- [Mouser](http://mouser.com/)
- [Digikey](https://digikey.com/)
- [LCSC](https://www.lcsc.com/)

![](https://uploads.developerhub.io/prod/86Yw/excu2e0ghzgccpj56itoe41jelkn9dabwxa3s1h3nswl7drieyykn6zizqmpcw3j.png)

**Pro tip:** Pricing information is also available when searching for a part in the library. Just make sure you [expand the library drawer.](https://docs.flux.ai/flux/reference/reference-library#expanding-the-library)

![](https://uploads.developerhub.io/prod/86Yw/dceioejc284hcxec0k5vsmx8jso1siice2uhgrpqpldtmnppe892kmyqynnlhrmf.gif)

> If the part does not contain a "Manufacturer Part Number" property, pricing information will not be shown.
