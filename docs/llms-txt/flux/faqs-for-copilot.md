# Flux AI FAQs


### Can Flux read datasheets or suggest component choices?

Yes. Flux uses AI to process datasheet information and provide recommendations for PCB layout and component selection. It can suggest parts based on specifications and flag potential mismatches

### Can I use JavaScript or scripting to automate tasks?

No, apart from writing simulation models, Flux Editor does not support scripting for automating PCB design.

### Will Flux optimize my schematic or PCB layout automatically?

Flux provides recommendations and assists with tasks like:

- Searching the library
- Performing calculations
- Add, remove, or replace components in a project
- Re-assign designators and component values.
- Assisting with wiring
- Analyzing images However, **you retain final control** over design decisions.
