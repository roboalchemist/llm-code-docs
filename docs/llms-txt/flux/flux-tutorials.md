## Tutorial Categories

We've organized our tutorials into categories based on common tasks and goals. Browse the categories below to find what you need:

### Getting Started

- [Flux Walkthrough Project](https://docs.flux.ai/flux/Introduction/flux-walkthrough-project) - Learn the basics of Flux with a guided project
- [PCB Editor Overview](https://docs.flux.ai/flux/reference/reference-pcb-editor) - Understand the main interface for PCB design
- [Schematic Editor Basics](/tutorials/schematic-editor) - Get familiar with creating circuit schematics

### Design Fundamentals

- [Component Placement](https://docs.flux.ai/flux/tutorials/component-placement) - Learn how to effectively place components
- [Routing Tutorial](https://docs.flux.ai/flux/tutorials/routing-tutorial) - Master the art of PCB trace routing
- [Board Outline Creation](https://docs.flux.ai/flux/tutorials/tutorial-board-outline-shape) - Create custom board shapes

### Advanced Features

- [Layout Rules Deep Dive](https://docs.flux.ai/flux/tutorials/tutorial-layout-rules-deep-dive) - Understand Flux's powerful rule system
- [Working with Ground Signals](https://docs.flux.ai/flux/tutorials/working-with-ground--basics--multiple-grounds--and-ground-fill-m) - Learn about ground planes and fills
- [Modules and Sublayouts](https://docs.flux.ai/flux/tutorials/tutorial-parts-and-sublayouts) - Reuse design sections efficiently

### Collaboration and Workflow

- [Collaboration Deep Dive](https://docs.flux.ai/flux/tutorials/tutorial-collaboration-deep-dive) - Work with others on PCB designs
- [Version Control](https://docs.flux.ai/flux/tutorials/version-control---deep-dive) - Track changes to your designs
- [Reusing Community Projects](https://docs.flux.ai/flux/tutorials/reusing-community-projects) - Leverage existing designs

### Manufacturing and Production

- [Preparing for Manufacturing](/tutorials/preparing-for-manufacturing) - Get your design ready for production
- [Gerber Export](https://docs.flux.ai/flux/reference/gerber-export) - Generate manufacturing files
- [Components Procurement](https://docs.flux.ai/flux/tutorials/components-procurement) - Manage your bill of materials

### AI-Assisted Design

- [AI for Hardware Design](https://docs.flux.ai/flux/tutorials/ai-for-hardware-design) - Leverage AI in your design process
- [Getting Started with Flux](https://docs.flux.ai/flux/tutorials/getting-started-copilot) - Use Flux's AI assistant
