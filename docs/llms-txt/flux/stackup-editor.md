# Stackup Editor

Efficiently create and modify stackups in a simple and intuitive interface.

![](https://uploads.developerhub.io/prod/86Yw/chxbzt4j9oh8wlevufjqep5h4yx2rldktj3h508tvwgbkx1epxegezivyzjfvh0n.png)

## Overview

Access to Community Resources: Users can leverage the expertise of Flux’s community, which includes over 175,000 electrical engineers. The platform offers project templates and resources, allowing for the sharing of knowledge and best practices within the same tool.

Central Hub for the Hardware Industry: Fabrication houses are empowered to input their stackup options directly into Flux as templates. This direct input from the fabrication houses ensures that the stackups are optimized for production, aligning the design process with manufacturing realities.

Layer Management and Customization: The Stackup Editor allows users to create and rearrange layers within the tool. It provides the capability to save and load stackup presets from various users and fabrication houses. Additionally, users can effortlessly tweak and change layer values, offering flexibility and customization in the stackup design.

## Getting Started

### Selecting a Stackup Template

Flux comes with a few stackup templates that you can use as a base for your customized template. To select a template, click on the arrow in the top-right corner of the menu. 

Once a template has been selected you can use the stackup editor to modify any properties or add/remove any layers. 

![](https://uploads.developerhub.io/prod/86Yw/7ampqsl642ahordd37v2mxl3ixnsrmowbw6c2doq6b8uee74ar68fi9oki52cao2.png)

### Adding and Removing Layers

To add a new layer to your stackup, click on the "+Layer" button at the bottom-left corner of the menu. To remove a layer, select the target layer and click on the trash can button on the left of the layer's name.

![](https://uploads.developerhub.io/prod/86Yw/yazt2bv9fkxqukq9kgje45dg3if1twxnoksmhbg9mkfyqexn1w5261ouavrq6th2.gif)

### Editing Layer Properties

Every layer's properties can be modified by clicking on the target property. Some of those properties will have preloaded options, and some others can accept any input.

![](https://uploads.developerhub.io/prod/86Yw/wp6klh64daz3n4wi363n9xw3rzgs48g10vtupw1fmkg1e4pme8g66l48um921y9q.gif)

### Rearranging Layers

To change a layer's position in the stackup, hover over the target layer and click and drag the six dots button.

![](https://uploads.developerhub.io/prod/86Yw/y8d59sxkmxkz3dfd2sr1ykmlebgp76rjej6mxcynpee227svd1roj5yhfv2cxiom.gif)
