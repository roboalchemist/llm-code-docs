# Object-Specific Layout Rules


**As the name suggests, object-specific PCB rules apply only to the selected object**. Only [Selector-Based Layout Rules](https://docs.flux.ai/flux/reference/pcb-layout-rule-selectors) can be applied to multiple objects.

### Quick Method: Using the Toolbar (Recommended)
For individual objects, the fastest way to modify layout properties is to use the [toolbar](https://docs.flux.ai/flux/tutorials/toolbar), which appears automatically when you select an object in the PCB editor.

### Traditional Method: Using the Inspector Panel
To add an object-specific rule through the inspector panel:

- Select an object
- Find the "Layout rules" in the right-side menu
- Click on "Edit" -&gt; "Add"
- Search for the desired rule and add it.
