# Comments

## View comments

Comments in a project are pinned to the point of the canvas selected by the commenter. How comments display depends on your zoom level of the canvas and the amount of comments. 

Pins: Individual comments are represented by a pin icon . Hover over a pin to preview the comment. **Click the pin to open the comment modal**, where you can reply

![](https://uploads.developerhub.io/prod/86Yw/teu51tut7edaoc6mhn6c1wsm566004dw6pbfau14rnilwf0rslig8s083nwzw6ag.png)

## Add comments

To add a comment, **right-click** exactly where you want to add the comment and select **"Insert Comment"**

![](https://uploads.developerhub.io/prod/86Yw/gt6zfw3z0hf5txj5i3wno71c20cltsesyrrzf999p1lt78idz7n309w4idwhvpc5.png)

## Comments Menu

The comments menu provides a summarized view of all the comments in your project. To open the comments menu, **click on the icon in the bottom-left corner of the Schematic or PCB editors.**

![](https://uploads.developerhub.io/prod/86Yw/08jtrojohwhywwymyppi74qa4x0qsyf6mfo0tg9xmud9c4uuw4utco5hwbyysp8p.png)

## Comment options

To access the comment options, **click on the three dot icon in the top right of the comment modal.**

### Pin a comment thread

Pinning a comment will keep that thread open at all times so its easier for you to reference it while you make changes to your designs.

![](https://uploads.developerhub.io/prod/86Yw/0utn9r0gvf7v97vafhz6ai97b3r8vdxxiv082gfuvg282t6ec7x9vs63dghzw4g9.png)

### Delete Comment

Deletes the comment

![](https://uploads.developerhub.io/prod/86Yw/ugda247t5nqqd9gi31mqntla6s3vabp4th5gr40ilp7e6dtofissbou0bbfz82q0.png)

### Resolve Comment

Marks the comment as resolved without deleting it.

![](https://uploads.developerhub.io/prod/86Yw/2iqfjf5wu1yjhvxtf8uwywphwwz8fzbwx6apki78h2mynae7rfnsh0t72lt9jfuv.png)

## Hide all comments

Comments are always visible on the canvas by default, whether or not you're in comment mode. If you rather hide comments from your canvas and only see them when in comment mode, you can do so at any time.

**To hide comments, use the keyboard shortcut** ⇧C.

## Embedding content in a comment

You can embed external URLs in comments like YouTube videos or images by simple pasting the url in the comment.

![](https://uploads.developerhub.io/prod/86Yw/jrcfpvsc95l9z46twa1n35cwipi13dh3v3aprxtrhuojbnvsj2yrjvb4sdj4poh2.png)
