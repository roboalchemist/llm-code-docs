# Star a project (favorite)

Favorite projects, parts, and modules that you like.

![](https://uploads.developerhub.io/prod/86Yw/97x1v68wcpj3v9qql6kz362nlu2nd56tdfj0oglmkupcwmiaazujc8s0xez13y4x.png)

## Overview

You can star (favorite) any project you find interesting by clicking on the "star" icon in the top left corner. You'll be able to quickly find all your favorite parts and modules from the [library filter.](https://docs.flux.ai/flux/reference/reference-library#filtering)

Starred projects rank higher in search results, so it's a great tool for curating high-quality projects
