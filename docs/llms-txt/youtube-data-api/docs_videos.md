# Videos

All videos uploaded via the [videos.insert](https://developers.google.com/youtube/v3/docs/videos/insert) endpoint from unverified API projects created after 28 July 2020 will be restricted to private viewing mode. To lift this restriction, each API project must [undergo an audit](https://support.google.com/youtube/contact/yt_api_form) to verify compliance with the [Terms of
Service](https://developers.google.com/youtube/terms/api-services-terms-of-service). Please see the [API
Revision History](https://developers.google.com/youtube/v3/revision_history#release_notes_07_28_2020) for more details.
A **video** resource represents a YouTube video.

## Methods

The API supports the following methods for `videos` resources:

[getRating](https://developers.google.com/youtube/v3/docs/videos/getRating)
:   Retrieves the ratings that the authorized user gave to a list of specified videos.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/getRating#usage).

[list](https://developers.google.com/youtube/v3/docs/videos/list)
:   Returns a list of videos that match the API request parameters.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/list#usage).

[insert](https://developers.google.com/youtube/v3/docs/videos/insert)
:   Uploads a video to YouTube and optionally sets the video's metadata.

[update](https://developers.google.com/youtube/v3/docs/videos/update)
:   Updates a video's metadata.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/update#usage).

[delete](https://developers.google.com/youtube/v3/docs/videos/delete)
:   Deletes a YouTube video.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/delete#usage).

[rate](https://developers.google.com/youtube/v3/docs/videos/rate)
:   Add a like or dislike rating to a video or remove a rating from a video.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/rate#usage).

[reportAbuse](https://developers.google.com/youtube/v3/docs/videos/reportAbuse)
:   Report a video for containing abusive content.
    [Try it now](https://developers.google.com/youtube/v3/docs/videos/reportAbuse#try-it).
