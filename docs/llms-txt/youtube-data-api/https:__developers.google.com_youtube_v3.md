## Add YouTube functionality to your app

[![](http://developers.google.com/static/youtube/images/youtube_home_page_data_api.png)](http://developers.google.com/youtube/v3/getting-started)  

### [Add YouTube functionality to your site](http://developers.google.com/youtube/v3/getting-started)

With the YouTube Data API, you can add a variety of YouTube
features to your application. Use the API to
upload videos, manage playlists and subscriptions, update
channel settings, and more.  
[Get started](http://developers.google.com/youtube/v3/getting-started) [Implementation guide](http://developers.google.com/youtube/v3/guides/implementation)  
[![](http://developers.google.com/static/youtube/images/yt-data-api-search.png)](http://developers.google.com/youtube/v3/docs/search/list)  

### [Search for content](http://developers.google.com/youtube/v3/docs/search/list)

Use the API to search for videos matching specific
search terms, topics, locations, publication dates, and much
more. The APIs `search.list` method also
supports searches for playlists and channels.  
[Search for content](http://developers.google.com/youtube/v3/docs/search/list)
