# Subscriptions

A **subscription** resource contains information about a YouTube user subscription. A subscription notifies a user when new videos are added to a channel or when another user takes one of several actions on YouTube, such as uploading a video, rating a video, or commenting on a video.

## Methods

The API supports the following methods for `subscriptions` resources:

[list](https://developers.google.com/youtube/v3/docs/subscriptions/list)
:   Returns subscription resources that match the API request criteria.
    [Try it now](https://developers.google.com/youtube/v3/docs/subscriptions/list#usage).

[insert](https://developers.google.com/youtube/v3/docs/subscriptions/insert)
:   Adds a subscription for the authenticated user's channel.
    [Try it now](https://developers.google.com/youtube/v3/docs/subscriptions/insert#usage).

[delete](https://developers.google.com/youtube/v3/docs/subscriptions/delete)
:   Deletes a subscription.
    [Try it now](https://developers.google.com/youtube/v3/docs/subscriptions/delete#usage).

## Resource representation

The following JSON structure shows the format of a `subscriptions` resource:  

```text
{
  "kind": "youtube#subscription",
  "etag": etag,
  "id": string,
  "snippet": {
    "publishedAt": datetime,
    "channelTitle": string,
    "title": string,
    "description": string,
    "resourceId": {
      "kind": string,
      "channelId": string,
    },
    "channelId": string,
    "thumbnails": {
      (key): {
        "url": string,
        "width": unsigned integer,
        "height": unsigned integer
      }
    }
  },
  "contentDetails": {
    "totalItemCount": unsigned integer,
    "newItemCount": unsigned integer,
    "activityType": string
  },
  "subscriberSnippet": {
    "title": string,
    "description": string,
    "channelId": string,
    "thumbnails": {
      (key): {
        "url": string,
        "width": unsigned integer,
        "height": unsigned integer
      }
    }
  }
}
```
