# What should I name my target?

We typically use targets to differentiate between development and production runs of dbt, naming the targets `dev` and `prod`, respectively. Check out the docs on [managing environments in dbt Core](https://docs.getdbt.com/docs/local/dbt-core-environments.md) for more information.
