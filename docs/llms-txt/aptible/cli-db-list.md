# aptible db:list

This command lists [Databases](/core-concepts/managed-databases/overview) in an [Environment](/core-concepts/architecture/environments).

# Synopsis

```
Usage:
  aptible db:list

Options:
  --env, [--environment=ENVIRONMENT]
```
