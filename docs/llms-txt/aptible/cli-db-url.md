# aptible db:url

This command prints [Database Credentials](/core-concepts/managed-databases/connecting-databases/database-credentials) (which are displayed as Database URLs).

# Synopsis

```
Usage:
  aptible db:url HANDLE

Options:
  --env, [--environment=ENVIRONMENT]
  [--type=TYPE]
```
