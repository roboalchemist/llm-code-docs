# aptible config:rm

This command is an alias to [`aptible config:unset`](/reference/aptible-cli/cli-commands/cli-config-unset).

## Synopsis

```
Usage:
  aptible config:rm [--app APP] [VAR1][VAR2][...]

Options:
 [--app=APP]
 [--environment= ENVIRONMENT]
  -r, [--remote=REMOTE]
```
