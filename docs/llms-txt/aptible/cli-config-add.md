# aptible config:add

This command is an alias to [`aptible config:set`](/reference/aptible-cli/cli-commands/cli-config-set).

# Synopsis

```javascript  theme={null}
Usage:
  aptible config:add [--app APP] [VAR1=VAL1] [VAR2=VAL2] [...]

Options:
      [--app=APP]
  --env, [--environment=ENVIRONMENT]
  -r, [--remote=REMOTE]
```
