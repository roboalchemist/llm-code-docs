# Custom Synchronization

## Overview

The Custom Synchronization StackPack has everything you need to automatically synchronize topology information from an external topology source.

**Use the Custom Synchronization StackPack if you want to connect an external topology source that isn't listed as a standard StackPack.**

Custom synchronization is a [StackState curated integration](https://archivedocs.stackstate.com/5.1/stackpacks/about_integrations#stackstate-curated-integrations).

## Next steps

Read up on [configuring synchronizations](https://archivedocs.stackstate.com/5.1/configure/topology/sync), follow the [Push-Integration tutorial](https://archivedocs.stackstate.com/5.1/develop/tutorials/push_integration_tutorial) or read the [Custom Synchronization developer guide](https://archivedocs.stackstate.com/5.1/develop/developer-guides/custom_synchronization_stackpack).

## Open-source

The Custom Synchronization StackPack is open-source and can be found [on StackState's gitHub page](https://github.com/StackVista/stackpack-autosync).
