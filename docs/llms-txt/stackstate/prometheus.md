# Prometheus

## Overview

StackState can be configured to pull telemetry data from an external Prometheus system. This integration isn't set up using a StackPack.

➡️ [Learn how to configure a Prometheus mirror](https://archivedocs.stackstate.com/5.1/configure/telemetry/data-sources/prometheus-mirror).
