# Keyboard shortcuts

The keyboard shortcuts below can be used to navigate the StackState UI.

| Keyboard shortcut      | Action                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `CTRL` + `SHIFT` + `F` | Search for a component by name in the [Topology Perspective](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/perspectives/topology-perspective).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `CTRL` + `S`           | Overwrite previously saved settings for a [view](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/views/about_views). This will update the saved view to include any changes made to the [view filters](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/filters) or [visualization settings](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/views/visualization_settings). Note that in [Explore mode](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/explore_mode), settings can only be saved as a new view (`CTRL` + `SHIFT` + `S`). It isn't possible to save changes made to a [subview](https://archivedocs.stackstate.com/5.1/use/views/about_views#subview). |
| `CTRL` + `SHIFT` + `S` | Save the current view settings as a new view. The new view will include all currently applied [view filters](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/filters) and [visualization settings](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/views/visualization_settings). Note that it isn't possible to save changes made to a subview as a new view.                                                                                                                                                                                                                                                                                                                          |
| `CTRL` + `SHIFT` + `R` | Reset any unsaved changes and revert to the original view, subview or explore mode. This will reset all unsaved changes made to the [view filters](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/filters) or [visualization settings](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/views/visualization_settings).                                                                                                                                                                                                                                                                                                                                                                  |
| `SHIFT` + `1`          | In a view, subview or explore mode, open the [Topology Perspective](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/perspectives/topology-perspective).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `SHIFT` + `2`          | In a view, subview or explore mode, open the [Events Perspective](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/perspectives/events_perspective).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `SHIFT` + `3`          | In a view, subview or explore mode, open the [Metrics Perspective](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/perspectives/metrics-perspective).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `SHIFT` + `4`          | In a view, subview or explore mode, open the [Traces Perspective](https://archivedocs.stackstate.com/5.1/use/stackstate-ui/perspectives/traces-perspective).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
