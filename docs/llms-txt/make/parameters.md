# Static parameters

Static parameters are only used for trigger modules.

For all other modules, the Static Parameters tab is not applicable and should be left empty. All fields should be added in the Mappable Parameters tab instead.

Static parameters use [parameters syntax](https://developers.make.com/custom-apps-documentation/block-elements/parameters).
