# Remove a user from an organization

The **Administration > Users** page lets you manage a specific user's relationship with organizations and teams. Removing a user from an organization requires selecting **---** as their organization or team role.

1. Go to **Administration > Users** and use the filters to find the user you want to remove.
2. Click **Detail** for that user.
3. A redirect takes you to a page where you see a list of all organizations that the user belongs to. Use the **Role** menu and select **---**.
