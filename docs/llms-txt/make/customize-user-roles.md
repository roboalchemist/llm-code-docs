# Customize user roles

Only users with the instance-level role `SA` can customize instance-wide, organization, and team roles by enabling or disabling specific scopes. See the following reference chart for the default permission settings.

1. Go to **Administration > Roles**.
2. Click on a tab such as **Admin-system** or **Admin-developers**.
3. Click the toggle to enable or disable scopes for each role as needed. Enable the `access monitoring` scope for the `BI` role.

The toggle stays on or off to indicate the current status.
