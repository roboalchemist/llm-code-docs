# Rebrand your instance

The system settings page lets you customize the appearance, name, support, and available features of your instance. To access and edit these settings, go to **Administration > System Settings** and find the corresponding field.

### Name and appearance

The following fields let you customize the name and appearance of your instance.

<table data-header-hidden><thead><tr><th width="267"></th><th></th></tr></thead><tbody><tr><td><strong>Brand title</strong></td><td>The name that appears in the browser tab and in email communication to your end user.</td></tr><tr><td><strong>Brand theme</strong></td><td>Select one of the available colors. This color applies to the login page, public side navigation bar, and other UI features.</td></tr><tr><td><strong>Brand favicon</strong></td><td>Upload or drag and drop a <code>.ico</code> file to appear in browser tabs when your users have it open.</td></tr><tr><td><strong>Brand logo (for dark background)</strong></td><td><p>Upload or drag and drop a .png file to appear against dark backgrounds.</p><p>Recommended size is 512 x 512 px with a square image.</p><p>The logo you upload here appears on the login screen.</p></td></tr><tr><td><strong>Brand logo (for light background)</strong></td><td><p>Upload or drag and drop a .png file to appear against light backgrounds.</p><p>Recommended size is 512 x 512 px with a square image.</p></td></tr></tbody></table>

{% hint style="warning" %}
The name and appearance settings do not impact the administration interface. You need to click Leave administration to confirm the changes. Recent changes may take several minutes to appear in your UI.
{% endhint %}

### Support root URLs and emails

Support root URLs provide a base URL for UI links to support documentation and tickets.

<table data-header-hidden><thead><tr><th width="272"></th><th></th></tr></thead><tbody><tr><td><strong>Support email</strong></td><td>The name and appearance settings do not impact the administration interface. You need to click <strong>Leave administration</strong> to confirm the changes. Recent changes may take several minutes to appear in your UI.</td></tr><tr><td><strong>SDK App Help Root</strong></td><td>This URL only impacts the custom app editor and is not necessary if the Apps Platform is disabled. Enter the URL for documentation on creating custom apps.</td></tr><tr><td><strong>Help Center Root</strong></td><td>This URL provides a base URL for in-product user support. Your users encounter this when they click on the <strong>?</strong> when editing module settings.</td></tr><tr><td><strong>Open Ticket Root</strong></td><td>This URL links to your customer support ticketing system. When users click on Contact Support, a new tab opens with the URL you enter in this field.</td></tr></tbody></table>
