		  		  <h1 class="entry-title">Introducing Accelerated PyTorch Training on Mac</h1>

						<div id="single-below-header" data-hide-on-mobile="false">
				<span class="meta-author vcard author"><span class="fn"><span class="author-leading">By</span> <a href="#" title="Posts by PyTorch" rel="author">PyTorch</a></span></span><span class="meta-date date published">May 18, 2022</span><span class="meta-date date updated rich-snippet-hidden">November 15th, 2024</span><span class="meta-comment-count"><a href="https://pytorch.org/blog/introducing-accelerated-pytorch-training-on-mac/#respond">No Comments</a></span>			</div><!--/single-below-header-->

<article id="post-1531" class="post-1531 post type-post status-publish format-standard category-blog">
  
  <div class="inner-wrap">

		<div class="post-content" data-hide-featured-media="1">
      
        <div class="content-inner">
<p>In collaboration with the Metal engineering team at Apple, we are excited to announce support for GPU-accelerated PyTorch training on Mac. Until now, PyTorch training on Mac only leveraged the CPU, but with the upcoming PyTorch v1.12 release, developers and researchers can take advantage of Apple silicon GPUs for significantly faster model training. This unlocks the ability to perform machine learning workflows like prototyping and fine-tuning locally, right on Mac.</p>



<figure class="wp-block-image"><img fetchpriority="high" decoding="async" width="8192" height="2436" src="https://pytorch.org/wp-content/uploads/2024/11/intro-graphic-accelerated-pytorch-training-revised.png" alt="" class="wp-image-1532" srcset="https://pytorch.org/wp-content/uploads/2024/11/intro-graphic-accelerated-pytorch-training-revised.png 8192w, https://pytorch.org/wp-content/uploads/2024/11/intro-graphic-accelerated-pytorch-training-revised-300x89.png 300w, https://pytorch.org/wp-content/uploads/2024/11/intro-graphic-accelerated-pytorch-training-revised-1024x305.png 1024w" sizes="(max-width: 8192px) 100vw, 8192px" /></figure>



<h2 class="wp-block-heading" id="metal-acceleration">Metal Acceleration<a href="https://pytorch.org/blog/introducing-accelerated-pytorch-training-on-mac/#metal-acceleration"></a></h2>



<p>Accelerated GPU training is enabled using Apple’s Metal Performance Shaders (MPS) as a backend for PyTorch. The MPS backend extends the PyTorch framework, providing scripts and capabilities to set up and run operations on Mac. MPS optimizes compute performance with kernels that are fine-tuned for the unique characteristics of each Metal GPU family. The new device maps machine learning computational graphs and primitives on the MPS Graph framework and tuned kernels provided by MPS.</p>



<h2 class="wp-block-heading" id="training-benefits-on-apple-silicon">Training Benefits on Apple Silicon<a href="https://pytorch.org/blog/introducing-accelerated-pytorch-training-on-mac/#training-benefits-on-apple-silicon"></a></h2>



<p>Every Apple silicon Mac has a unified memory architecture, providing the GPU with direct access to the full memory store. This makes Mac a great platform for machine learning, enabling users to train larger networks or batch sizes locally. This reduces costs associated with cloud-based development or the need for additional local GPUs. The Unified Memory architecture also reduces data retrieval latency, improving end-to-end performance.</p>



<p>In the graphs below, you can see the performance speedup from accelerated GPU training and evaluation compared to the CPU baseline:</p>



<figure class="wp-block-image aligncenter size-large"><img decoding="async" src="https://pytorch.org/wp-content/uploads/2024/11/METAPT-002-BarGraph-02.gif"/></figure>



<p class="has-text-align-center">Accelerated GPU training and evaluation speedups over CPU-only (times faster)</p>



<h2 class="wp-block-heading" id="getting-started">Getting Started<a href="https://pytorch.org/blog/introducing-accelerated-pytorch-training-on-mac/#getting-started"></a></h2>



<p>To get started, just install the latest&nbsp;<a href="https://pytorch.org/get-started/locally/">Preview (Nightly) build</a>&nbsp;on your Apple silicon Mac running macOS 12.3 or later with a native version (arm64) of Python.</p>



<p>You can also learn more about Metal and MPS on&nbsp;<a href="https://developer.apple.com/metal/" target="_blank" rel="noreferrer noopener">Apple’s Metal page</a>.</p>



<p>*&nbsp;<em>Testing conducted by Apple in April 2022 using production Mac Studio systems with Apple M1 Ultra, 20-core CPU, 64-core GPU 128GB of RAM, and 2TB SSD. Tested with macOS Monterey 12.3, prerelease PyTorch 1.12, ResNet50 (batch size=128), HuggingFace BERT (batch size=64), and VGG16 (batch size=64). Performance tests are conducted using specific computer systems and reflect the approximate performance of Mac Studio.</em></p>
