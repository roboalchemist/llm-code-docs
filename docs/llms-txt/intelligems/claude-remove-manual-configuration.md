# Claude: Remove Manual Configuration

## Overview

If you were one of our early adopters to our Claude <> Intelligems MCP Server, you may have been instructed to manually add Intelligems to your claude account under the Developers Section. Before following [these steps](https://docs.intelligems.io/developer-resources/mcp-server/claude) to migrate to our latest Claude setup, first remove the manual configuration to prevent duplicate tool listings or conflicts.

#### Remove the Manual Configuration

1. Open the [Claude Desktop](https://docs.intelligems.io/developer-resources/mcp-server/claude#claude-desktop) app.
2. Click your profile name in the sidebar and select Settings.
3. Go to the Developer tab, look under Local MCP servers, and click the trashcan button next to "intelligems".
