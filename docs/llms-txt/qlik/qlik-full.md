1) Ask/confirm: Cloud vs Client-managed + product + version.
2) Start with an overview/concepts page to define terms and scope.

5) If behavior differs by version, explicitly call out differences and link to the relevant version docs.
6) Always link to the exact version-specific page when applicable.
7) Do not link to search results.
