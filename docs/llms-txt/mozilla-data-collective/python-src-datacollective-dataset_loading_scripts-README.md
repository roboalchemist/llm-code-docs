# Source: https://raw.githubusercontent.com/Mozilla-Data-Collective/datacollective-python/main/src/datacollective/dataset_loading_scripts/README.md

`load_dataset()` requires a certain dataset-specific logic in order to parse the data correctly from the downloaded files into a Pandas DataFrame.

This directory contains dataset loading scripts for different datasets hosted on Mozilla Data Collective to enable the `load_dataset()` functionality.
