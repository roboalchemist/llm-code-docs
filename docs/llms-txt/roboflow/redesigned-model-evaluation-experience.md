# Redesigned Model Evaluation Experience

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2Fx65Cw1BR0m4rbRD3EZhr%2FScreenshot%202025-07-14%20at%2015.25.05.png?alt=media&#x26;token=baa4df94-a983-4a72-94e8-affa6fce2c8c" alt=""><figcaption></figcaption></figure>

The Model Evaluation tool lets you see how well your model performs when validated against your dataset valid and/or test sets.

Model Evaluation has been redesigned to be more intuitive, and comes with several new features. These are:

1. [Metrics Explorer in Model Evaluation](https://docs.roboflow.com/changelog/explore-by-month/july-2025/metrics-explorer-in-model-evaluation)
2. [Apply Tags from the Confusion Matrix](https://docs.roboflow.com/changelog/explore-by-month/july-2025/apply-tags-from-the-confusion-matrix)
3. [Automated Model Improvement Recommendations](https://docs.roboflow.com/changelog/explore-by-month/july-2025/automated-model-improvement-recommendations)
