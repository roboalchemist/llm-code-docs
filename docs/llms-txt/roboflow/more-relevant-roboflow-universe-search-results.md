# More Relevant Roboflow Universe Search Results

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FKU9zbQJezUgZbDOt9NX8%2Fimage%20(17).png?alt=media&#x26;token=99592ecf-b200-4f4a-8021-cf0f2238be7e" alt=""><figcaption></figcaption></figure>

Roboflow Universe now uses a new hybrid search system that uses both the semantic content of datasets as well as keyword-based search to find results. This leads to more relevant results.

As part of this change, we have deprecated the "Subject" and "Metadata" search option. In our tests, the new hybrid search leads to consistently more relevant results than both of these previous options.

To try out this change, go to [Roboflow Universe](https://universe.roboflow.com/) and enter a search query.
