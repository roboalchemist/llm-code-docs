# Multi-Label Classification Label Tooltips in Image Library

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FDkygV4HB0dsSk4gcgGF9%2FScreenshot%202025-11-10%20at%2010.11.03.png?alt=media&#x26;token=fda79c31-a4de-453d-98c6-72e747b3391e" alt=""><figcaption></figcaption></figure>

The "Images" tab in Multi-Label Classification datasets now shows a list of colours that correspond to the classes used to label an image. You can hover over each colour to see the class name.

This feature allows you to see all of the labels applied to an image at a glance while navigating through your image library.
