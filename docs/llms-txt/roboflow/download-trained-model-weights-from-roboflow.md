# Download trained model weights from Roboflow

Customers on Basic and Growth plans can now download weights from models trained with Roboflow Train. Exports are available from model pages in your Roboflow dashboard. This feature is ideal to use your model in environments that Roboflow does not yet support.

Read our documentation to learn more about [how to download weights from Roboflow](https://docs.roboflow.com/deploy/download-roboflow-model-weights).
