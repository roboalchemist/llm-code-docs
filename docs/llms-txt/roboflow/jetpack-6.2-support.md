# JetPack 6.2 Support

Deployment Manager and Inference Server now support JetPack 6.2 on NVIDIA Jetson devices. This update moves the base OS to Ubuntu 22.04 and unlocks major performance gains through CUDA 12.6, TensorRT 10.3, and cuDNN 9.3. You can take advantage of improved inference speeds, better transformer model support, and enhanced INT8/FP16 quantization capabilities.
