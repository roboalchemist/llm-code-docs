# Scoped API Keys

{% hint style="warning" %}
This feature is only available to Enterprise customers.
{% endhint %}

Scoped API keys let you issue an API key that has specific permissions.

For example, you can issue an API key that allows you to manage images in a Project, but does not allow you to manage models.

To create a scoped API key, go to your [Settings - API keys](https://app.roboflow.com/settings/api):

<figure><img src="https://1284666567-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fe5GEiPeDoFksvZv1vH3A%2Fuploads%2FTPFFD0TbI3LMsU66c2HG%2FScreenshot%202025-05-20%20at%2009.37.59.png?alt=media&#x26;token=09e603b2-df8c-4330-9174-d53ccb7c4e30" alt=""><figcaption></figcaption></figure>

Next, click "Generate New Key":

<figure><img src="https://1284666567-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fe5GEiPeDoFksvZv1vH3A%2Fuploads%2FIEYz7vAHY411s6tOr0MS%2FScreenshot%202025-05-20%20at%2009.38.26.png?alt=media&#x26;token=de878d66-49c8-422c-a061-98aa02ffbd57" alt=""><figcaption></figcaption></figure>

This will add a new key to your Workspace.

To define the scope for the API key, click the edit icon in the table row of the API key you want to update.

A window will appear where you can set a name for and define the scope for your API key:

<figure><img src="https://1284666567-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2Fe5GEiPeDoFksvZv1vH3A%2Fuploads%2F1InCHtrrFeKaBUQi3VSm%2FScreenshot%202025-05-20%20at%2009.39.22.png?alt=media&#x26;token=10037628-c0db-4b19-bbef-729dd063b2e0" alt=""><figcaption></figcaption></figure>

You can customise API key scopes at any time.

If you do not have access to this feature and are on an enterprise plan, contact your Roboflow account representative.
