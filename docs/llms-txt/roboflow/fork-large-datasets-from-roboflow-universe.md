# Fork Large Datasets from Roboflow Universe

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2Fa7fnYMmXFy9ePNv3ivtB%2FScreenshot%202025-09-29%20at%2010.40.47.png?alt=media&#x26;token=c67d2367-b23a-44a7-8454-3e9a549c6909" alt=""><figcaption></figcaption></figure>

We have made several optimisations to the [Roboflow Universe](https://universe.roboflow.com) fork feature which lets you create a copy of a dataset on Roboflow Universe in your own Roboflow account. This change allows you to fork large datasets (i.e. datasets that contain over 1,000 images) in minutes rather than hours.

To try this feature, go to a dataset on Roboflow Universe that you want to copy into your account, go to the Images tab accessible from the left sidebar of Universe, then select "Fork Dataset".
