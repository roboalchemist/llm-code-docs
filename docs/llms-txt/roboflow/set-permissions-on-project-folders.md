# Set Permissions on Project Folders

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FXd02X1u7WspcsHA9Qvid%2Fimage.png?alt=media&#x26;token=bccd9717-c8bc-4547-9a4f-17fb62fb7926" alt="" width="563"><figcaption></figcaption></figure>

We are launching a feature in beta for Enterprise customers that lets you set fine-grained access controls for Project Folders. With this feature enabled, you can invite people with read or write (Editor) access to Projects within a Project Folder. This gives you greater control over how users can manage Projects within a folder.

If you are an Enterprise customer interested in using this feature, contact your account representative. You can also contact the [Roboflow sales team](https://roboflow.com/sales) to learn more about this feature.
