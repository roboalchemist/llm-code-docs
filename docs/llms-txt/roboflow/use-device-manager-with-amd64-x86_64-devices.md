# Use Device Manager with AMD64 (x86\_64) Devices

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2Fl4UNJqYIC5kV0x9mBGdp%2Fimage.png?alt=media&#x26;token=22b7600c-0dcf-46b2-8e5c-05bca1d2e674" alt=""><figcaption></figcaption></figure>

The Roboflow Device Manager provides everything you need to set up, deploy, and manage computer vision devices on the edge.

You can now use Device Manager with any device that uses the AMD64 Architecture.

To get started, open Device Manager, then follow our instructions to [Add a Device.](https://docs.roboflow.com/deploy/device-manager/add-a-device)
