# Qwen3 VL in Workflows

Qwen3-VL, the latest vision-language model in Alibaba Cloud’s Qwen series, is available in Workflows. With the Qwen3-VL block, you can deploy the model for various tasks (e.g., visual question answering, OCR, object grounding, etc.) without setting up an environment to run the model yourself.&#x20;
