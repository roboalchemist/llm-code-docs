# Workflow Block Governance

Enterprise administrators can control which workflow blocks are available to users in their workspace. This governance feature allows organizations to selectively disable blocks that connect to external AI services or other third-party integrations, ensuring compliance with internal security policies.

From Settings > Workflows, administrators can view all available workflow blocks and toggle each one on or off for the entire workspace, including blocks for external AI providers (OpenAI, Anthropic), communication services (Twilio, SMTP), and more.
