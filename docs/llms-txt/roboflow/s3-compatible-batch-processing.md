# S3-Compatible Batch Processing

Batch processing is now compatible with a wide range of S3-compatible cloud storage providers, including Cloudflare R2, Backblaze B2, Wasabi, DigitalOcean, and more. You can now point the Roboflow Inference CLI directly at your external buckets (e.g., s3://your-data/\*.jpg) to process massive volumes of images without moving your data.
