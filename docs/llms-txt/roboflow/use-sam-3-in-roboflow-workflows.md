# Use SAM 3 in Roboflow Workflows

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2F1kYjGPPupC0x4oUxohGM%2Fam3.png?alt=media&#x26;token=2d257803-a422-4ca7-83ae-11da59a497a5" alt=""><figcaption></figcaption></figure>

You can now use [SAM 3 in Roboflow Workflows](https://app.gitbook.com/u/s4CyJzhd1vZDZ4NRPss7hiInsak2). Roboflow Workflows has over 100 pre-built functions that let you prototype, test, integrate, and deploy pipelines to production for complex vision problems.

You can use the base SAM 3 model with custom text prompts in Workflows. You can also run SAM 3 models that you have fine-tuned on Roboflow.
