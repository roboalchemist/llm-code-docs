# Automated Model Improvement Recommendations

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FvyWPYtUueYRC1k8Voh1B%2FScreenshot%202025-07-14%20at%2015.24.09.png?alt=media&#x26;token=6c90955c-c5a0-49ea-a7ac-cb5eb0c51b00" alt=""><figcaption></figcaption></figure>

The Roboflow Model Evaluation tool now makes recommendations on how to improve your model. These recommendations are based on the results from your model confusion matrix.

The Model Evaluation tool can make recommendations based on:

* How to improve a model that predicts many false negatives.
* How to improve a model that predicts many false positives.
* What classes are often confused (mis-identified).
* What classes need more data to improve accuracy.
* When a test or validation set may be too small.
* And more.

You can see this feature by clicking the "Model Evaluation" button on any model trained on or uploaded to Roboflow.
