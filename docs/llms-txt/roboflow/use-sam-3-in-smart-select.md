# Use SAM 3 in Smart Select

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2F09LosD3t4FKTAUXwqxeN%2FScreenshot%202025-11-19%20at%2017.45.34.png?alt=media&#x26;token=197199b9-6f0f-48cc-8fc2-3e4239fa8695" alt=""><figcaption></figcaption></figure>

You can now use SAM 3 in Smart Select (Smart Polygon) to identify objects based on points in an image. To use this feature, click on the cursor icon in the left side of the Roboflow Annotate interface. This will enable a tool that lets you click anywhere on an image and calculate a segmentation mask that corresponds with the object you have clicked. This feature is powered by SAM 3.

You can extend an annotation by adding more points outside the highlighted region, or refine an anotation by clicking inside at the edges where you want to refine your annotation.
