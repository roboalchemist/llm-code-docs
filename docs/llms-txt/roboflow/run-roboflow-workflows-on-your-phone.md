# Run Roboflow Workflows on Your Phone

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FzkezDZKFMQzchEtwjF6m%2FIMG_3138.jpg?alt=media&#x26;token=cadb284b-5352-4610-928d-ed32acfbc4e9" alt=""><figcaption></figcaption></figure>

Roboflow Workflows now has a mobile view you can use to run your Workflows.

The mobile view is optimised to run as a web app. This means you can save the mobile view fora. Workflow to your home screen.

To access this feature, click the "Run From Anywhere" button in the Workflows interface. Then, scan the QR code to open your Workflow.
