# Workspace Homepage with Deployment Overview

The workspace homepage has been redesigned to provide immediate visibility into your deployment performance. You can now view interactive usage charts showing inference activity over time (from 1 week to all-time). The new layout also surfaces both deployed and undeployed models, making it easier to manage active projects and kickstart new ones with workflow templates.
