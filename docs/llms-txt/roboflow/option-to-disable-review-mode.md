# Option to Disable Review Mode

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2Fo3QMFmxZQEhT83CYcljJ%2FScreenshot%202025-07-14%20at%2015.13.40.png?alt=media&#x26;token=c47b703a-2107-4851-baa9-e01a7d9706b5" alt=""><figcaption></figcaption></figure>

You can now disable Review Mode from the Annotate page of a Roboflow Project.

By disabling Review Mode, the Review column will be removed from your dashboard. When you have finished annotating a job, you can move it immediately into your Dataset instead of moving to a review stage.

Whereas Review Mode is an essential part of many team workflows, disabling Review Mode is useful if you are working on a computer vision project yourself.
