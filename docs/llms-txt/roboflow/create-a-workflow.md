# Create a Workflow

### Create a Workflow

To create a Workflow, navigate to your workspace dashboard and click the "Workflows" tab:

On the Workflows page, click "Create a Workflow" to create a workflow:

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-8ee534d9d317fe203f2a74fee00a47e0b890b731%2FCreate%20a%20workflow.png?alt=media" alt=""><figcaption><p>Select "Create a Workflow"</p></figcaption></figure>

You will be redirected to your new Workflow in the Roboflow web application:

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-5fafa4f760a4bfd0e5b7019d1d27a12bfe86f50e%2FCleanShot%202025-01-29%20at%2013.12.13%402x.png?alt=media" alt=""><figcaption></figcaption></figure>

You can optionally start with a template that let you do, among other things:

1. Run a single model, count detected objects, and visualize results.
2. Run a detection model, crop results, and then run an OCR model for text recognition.
3. Detect an object then classify the region of the detection.

Select "Start from a Template" to browse our popular workflow templates.
