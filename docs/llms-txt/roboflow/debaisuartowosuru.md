# デバイスアラートを設定する

Roboflow Deployment Manager は、次のようなデバイスが期待どおりに動作していない可能性がある場合にメールでアラートを送信できます。

* デバイスが Roboflow Cloud との接続を失ったとき
* ディスク使用量が異常に高く、デバイスのパフォーマンスに影響を与える可能性があるとき
* 特定のストリームのカメラの FPS が低下しすぎたとき
* 特定のストリームの推論（Inference）FPS が低下しすぎたとき

デバイスアラートを設定するには、Deployment Manager ダッシュボードから該当デバイスのページに移動します。次に「Device Alerts」タブに移動して「Add Alert」をクリックします。
