# Deploy a Rapid Model

When you create a Rapid model, a [Workflow](https://docs.roboflow.com/workflows/what-is-workflows) will be automatically created to run your model. This Workflow will be powered by the [Roboflow Serverless API](https://docs.roboflow.com/deploy/serverless-hosted-api-v2).

Rapid will show code snippets in various programming languages that you can use to integrate your model into your applications:

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-1ec86d1ff8168451e22f3c816dfe60fed553bd0e%2FScreenshot%202025-11-10%20at%2011.08.39.png?alt=media" alt=""><figcaption></figcaption></figure>

You can customise your API to add new features by clicking the "Open Roboflow Workflows" link in the bottom right corner. This will open the Roboflow Workflow that powers your cloud API. You can customise your Workflow in whatever way you want. For example, you could add a ByteTrack block to add stateful object tracking capabilities to your API.
