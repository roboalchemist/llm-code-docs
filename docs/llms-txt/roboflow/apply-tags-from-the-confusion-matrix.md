# Apply Tags from the Confusion Matrix

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2Fs8ipJBOEpPFwPJ0xztDY%2FScreenshot%202025-07-14%20at%2015.27.27.png?alt=media&#x26;token=33133e46-0b0c-4a81-bdc4-306ecb554129" alt=""><figcaption></figcaption></figure>

You can now apply tags to images from the Confusion Matrix in the Model Evaluation interface.

Applying a tag means the images with the tag will be more easily searchable in your Dataset tab.

To apply a tag to images, first select a box in the Confusion Matrix. Then, click "Add Tags". You will be able to create a new tag or choose from an existing one.

You may apply a tag to images where there has been a false positive, for example. You can then search for that tag in your Dataset tab and correct images that are improperly labeled.
