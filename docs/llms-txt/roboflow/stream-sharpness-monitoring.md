# Stream Sharpness Monitoring

Deployment Manager now surfaces camera focus quality across your device fleet in real time. Each stream is assigned a sharpness score from 0–100%, visible as a dedicated column in the streams table view. Teams can configure threshold-based alerts to fire when sharpness drops below an acceptable level, making it easier to catch and resolve camera focus issues before they affect model performance.
