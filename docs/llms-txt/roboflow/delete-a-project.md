# Delete a Project

{% hint style="danger" %}
Keep in mind that deletions are permanent and irreversible. Please make sure you are confident deleting the project.
{% endhint %}

1. Click the three dots beside a project listing on the dashboard or the sidebar in your project
2. Click "Delete Project"
3. Follow the prompts given and confirm the deletion.
