# Merge Detections block available in Workflows

<figure><img src="https://cdn.prod.website-files.com/5f6bc60e665f54db361e52a9/67fd33fd8045cccfe8ea71cb_Screenshot%202025-04-14%20at%2017.12.03.png" alt=""><figcaption></figcaption></figure>

The DetectionsMerge block combines multiple detections into a single bounding box that encompasses all input detections. This is useful when you want to:

* Merge overlapping or nearby detections of the same object
* Create a single region that contains multiple detected objects
* Simplify multiple detections into one larger detection
