# Qwen3.5-VL in Workflows

Qwen3.5-VL is now available as a Workflows block and supported in Roboflow Inference. Multiple model sizes are available in the block (0.8B and 2B), with thinking mode supported in larger variants which allows for step-by-step reasoning on complex tasks.
