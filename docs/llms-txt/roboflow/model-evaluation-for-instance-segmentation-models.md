# Model Evaluation for Instance Segmentation Models

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FK7qXDWvVLtoVAv2vNMYp%2FScreenshot%202025-11-10%20at%2010.06.03.png?alt=media&#x26;token=3f98e450-552a-4f7a-ab28-878ec4678bd5" alt=""><figcaption></figcaption></figure>

The Roboflow [Model Evaluation](https://docs.roboflow.com/train/evaluate-trained-models) tool now supports instance segmentation projects. You can now use this tool to explore mAP by object size, identify the ideal confidence threshold based on precision/recall trade-off, view model accuracy by class, and more.

To use this feature, open a model in Roboflow and click on the "View Full Model Evaluation" button to view your model evaluation.

*Note: If your model was trained before the end of October 2025, a button may appear asking you to recompute your model evaluation. This will ensure our dashboard has all of the metrics now supported in our instance segmentation model evaluation suite.*
