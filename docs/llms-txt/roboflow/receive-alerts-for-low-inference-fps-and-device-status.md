# Receive Alerts for Low Inference FPS and Device Status

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FhzqJUCnJxALb2McqG0Qj%2FScreenshot%202025-09-29%20at%2020.25.57.png?alt=media&#x26;token=285b4f22-3732-4ec2-aa7b-3b30a5819311" alt=""><figcaption></figcaption></figure>

You can now set up alerts in Device Manager to be notified when:

* Your device is offline, and;
* The FPS at which your model is running is below a certain threshold.

You can configure alerts by clicking "Alerts" then the "Add Alert" button on a Device page within Device Manager.
