# Add Augmentations to Images

To add augmentations, [create a new dataset version](https://docs.roboflow.com/datasets/dataset-versions/create-a-dataset-version).

On Step 4 of the process, select the option to "Add Augmentation Step".

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-6d1e6e0820ab1b55e9db2605da72e0e869ea691e%2Fimage.png?alt=media" alt=""><figcaption></figcaption></figure>

Choose from the list of possible augmentation options and click Apply. Multiple augmentations can be added to the dataset version one at a time.

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-83744ae25e13b18ede58c7ed0816f8d9763f690a%2Fimage.png?alt=media" alt=""><figcaption></figcaption></figure>

On Step 5, select the number of images that you want to be created by augmentation.

<figure><img src="https://662926385-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-M6S9nPJhEX9FYH6clfW%2Fuploads%2Fgit-blob-08097d818bb093dea7e6a1e36c1894aa4d48dd3c%2Fimage.png?alt=media" alt=""><figcaption></figcaption></figure>
