# Draw polygon zones in Roboflow Workflows

You can now use an interactive editor in Roboflow Workflows to draw polygon zones for use in counting and objects in a region.

This feature is available for any block that supports polygon zones, including:

* Polygon Zone Annotator
* Time in Zone

With this feature, you no longer need to manually calculate coordinates to draw zones.

[Go to Workflows to start using this feature.](https://app.roboflow.com/)
