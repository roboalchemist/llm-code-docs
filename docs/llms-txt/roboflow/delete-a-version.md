# Delete a Version

If you want to clean up your project's version list or lower your project's generated image usage count.

{% hint style="info" %}
If you have hit your generated (not source) image limit, you can delete versions to lower your usage towards the generated image quota.
{% endhint %}

You can delete a version by clicking on the "Edit Dataset" button and clicking "Delete Version"
