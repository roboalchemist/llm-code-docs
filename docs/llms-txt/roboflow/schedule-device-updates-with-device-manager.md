# Schedule Device Updates with Device Manager

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FcVuiBmZ1ESZV0yY5LRaI%2FScreenshot%202025-09-29%20at%2020.29.04.png?alt=media&#x26;token=a3c523b1-30c9-46fe-9303-1a34e385fe96" alt=""><figcaption></figcaption></figure>

You can now set Maintenance Windows during which devices configured with Device Manager will update.

During a Maintenance Window, your server will run any available updates.

Running updates regularly helps ensure your device is running the latest software.

You can configure your set Maintenance Windows by clicking the three dots in the top corner of a Device page within Device Manager. Choose the Maintenance Window option. Then, a pop up will appear in which you can set your Maintenance Windows.
