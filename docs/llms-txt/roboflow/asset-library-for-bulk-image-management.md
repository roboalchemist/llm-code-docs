# Asset Library for Bulk Image Management

The new Asset Library provides a centralized interface to view, search, and manage all images across an entire Workspace. Engineers can locate specific data by filtering images by attributes such as project, metadata, tags, and class, or by utilizing advanced semantic and similarity search tools.

Once identified, users can select the required images to create a new project or route them directly into an existing project. This capability simplifies bulk data management, exploration, and curation across large-scale computer vision workflows.

[Learn more about the Asset Library](https://docs.roboflow.com/workspaces/asset-library)
