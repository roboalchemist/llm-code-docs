# Use an AI Assistant with Custom Python Blocks

<figure><img src="https://2667452268-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FMR3m936tBXGm5QsAcPwe%2Fuploads%2FzN9YlVW871Isw5bV9clE%2FScreenshot%202025-08-26%20at%2008.07.00.png?alt=media&#x26;token=d40b5e32-0421-4d19-814b-6d2e47ef1e48" alt=""><figcaption></figcaption></figure>

You can now use the Workflows AI Assistant inside the Custom Python Block. This block lets you write custom code that runs in a Workflow.

You can use this feature to help you build custom logic inside your Workflow.

To use this feature, add a Custom Python Block to your Workflow. Click on the block. You will then see an AI Assistant panel to which you can ask questions.
