  accounts with your system of records and to clear your receivables. This report contains both
  summarized and event-level details (including fees) of every payment event that impacted a deposit
  from your AIB funded Braintree account to your deposit bank account.

</span></p><h3 id="how-to-use-the-report" class="test_h3 markdown-h3"><span>How to Use the Report</span><a class="icon icon--link hash-link" href="#how-to-use-the-report" aria-label="How to Use the Report"><img src="https://www.paypalobjects.com/btdevdoc/bundles/4.3.24/_assets/anchor-PKYVW2VT.svg" alt="AnchorIcon"/></a></h3><h4 id="reconciling-deposit-amount-to-the-reported-amounts" class="test_h4 markdown-h4"><span>Reconciling deposit amount to the reported amounts</span><a class="icon icon--link hash-link" href="#reconciling-deposit-amount-to-the-reported-amounts" aria-label="Reconciling deposit amount to the reported amounts"><img src="https://www.paypalobjects.com/btdevdoc/bundles/4.3.24/_assets/anchor-PKYVW2VT.svg" alt="AnchorIcon"/></a></h4><p class="markdown-p"><span>
  To reconcile the line-item deposit records on your bank statement with the AIB Transaction Fee
  Report:
</span></p><ol class="list list--num"><li><span>
    Using the Transfer ID, match each deposit amount from AIB on your bank statement with the
    amounts found in the Net Disbursed columns of the Summary Records of the Transaction Fee Report
  </span></li><li><span>Locate the “Transaction Disbursement Key” for each deposit line-item</span></li><li><span>
    Use Transaction Disbursement Key for each deposit to identify the rows that belong to each
    deposit
  </span></li><li><span>
    For each unique Transaction Disbursement Key:
    </span><ul class="list list--disc"><li><span>
        Add the Settlement amount for each row and truncate the total to two decimals (do not round)
      </span></li><li><span>
        Add the Total Fee Amount for each row (For Gateway Transaction Records, only add if the
        “Transaction Fee Debit Date” = “Settlement Date”) and truncate the total to two decimals (do
        not round)
      </span></li><li><span>
        Add the Chargeback Amount for each row and truncate the total to two decimals (do not round)
      </span></li></ul></li><li><span>
    The sum for each group will match the amounts found in the corresponding "Summary" sections
  </span></li><li><span>
    Add all the amounts in the summary row to calculate the “Net Disbursed” for each Transaction
    Disbursement Key
  </span></li></ol><p class="markdown-p"><span>

  following sections.
