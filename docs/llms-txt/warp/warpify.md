# Warpify

1. [Subshells](https://docs.warp.dev/terminal/warpify/subshells), Warp supports enabling Warp features in subshells for bash, zsh, and fish.
2. [SSH](https://docs.warp.dev/terminal/warpify/ssh), Warp supports a tmux powered wrapper that enables Warp features in remote (SSH) sessions.
3. [SSH Legacy](https://docs.warp.dev/terminal/warpify/ssh-legacy), Warp supports a legacy wrapper that enables Warp features in remote (SSH) sessions.
