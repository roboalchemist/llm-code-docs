# Command Completions

1. [Completions](https://docs.warp.dev/terminal/command-completions/completions) will suggest commands, option names, and path parameters for you.
2. [Autosuggestions](https://docs.warp.dev/terminal/command-completions/autosuggestions) will automatically suggest commands as you type based on shell history and possible completions.
