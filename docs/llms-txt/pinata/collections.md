Click a collection to see:

* Collection metadata (image, description)
* List of CIDs with backup status

### CID Information

| Field     | Description                    |
| --------- | ------------------------------ |
| CID       | IPFS content identifier        |
| Source    | `metadata` or `image`          |
| Backed Up | Whether pinned to your account |
