Crawlee v3.16 is here, and the headline feature is the new `StagehandCrawler` — an AI-powered crawler that lets you interact with web pages using natural language instead of CSS selectors. On top of that, we've added async iterators for `Dataset` and `KeyValueStore`, a new `discoverValidSitemaps` utility, and made `handleCloudflareChallenge` more configurable.

Here's what's new:

* [StagehandCrawler — AI-powered browser automation](https://crawlee.dev/blog/crawlee-v3-16.md#stagehandcrawler--ai-powered-browser-automation)
* [Async iterators for Dataset and KeyValueStore](https://crawlee.dev/blog/crawlee-v3-16.md#async-iterators-for-dataset-and-keyvaluestore)
* [discoverValidSitemaps utility](https://crawlee.dev/blog/crawlee-v3-16.md#discovervalidsitemaps-utility)
* [Improved Cloudflare challenge handling](https://crawlee.dev/blog/crawlee-v3-16.md#improved-cloudflare-challenge-handling)
