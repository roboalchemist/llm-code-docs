# Configuring Ticket Attributes

Once you are happy with your Ticket categories, you need to set the following attributes for each category in the table on the Ticket screen:

<table data-header-hidden><thead><tr><th width="255.7947661263608">Setting</th><th>Description</th></tr></thead><tbody><tr><td>Setting</td><td>Description</td></tr><tr><td>When is it due?*</td><td>Select a value from the dropdown menu of Due Date ‘flavours’ (See <a href="../shared-standardised-settings-flavours/due-date-flavours">here </a>for more information).</td></tr><tr><td>Who does it go to?*</td><td>Select a value from the dropdown menu of Allocation ‘flavours’ (See <a href="../shared-standardised-settings-flavours/allocation-flavours">here </a>for more information).</td></tr><tr><td>From Email Addresses*</td><td>The email address from which any outgoing communications are sent out.<br>Multiple addresses can be added, and a single address set as the default. If multiple addresses as set, at runtime users will be presented with a dropdown choice. Whichever address is set as default will be auto-populated into the from address, but alternates can be selected.</td></tr><tr><td>Email BCC</td><td>Address to copy on all communications for the Ticket.</td></tr><tr><td>Keep Ticket Open</td><td>The number of days a Ticket will be kept open before being closed.</td></tr><tr><td>Allow Title Changes</td><td>Whether Service Agents are allowed to modify the Ticket Title if generated from email subject of initial incoming email.</td></tr><tr><td>Self Service</td><td>Whether or not this category should be displayed as as an option on Self Service submission forms. This allows you to show a subset of categories for end user submission.</td></tr></tbody></table>

*\*mandatory in order to set live.*

You can copy and paste attributes in the Ticket Screen. You can select data from multiple cells and paste that data into selected destination cells.&#x20;

{% hint style="info" %}
Please note that you cannot copy and paste data between columns and you cannot copy data from the Ticket Category column.
{% endhint %}

If you are copying just one cell from each column, you can paste it in any number of places:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2Fg7KsQxCVkZ2Kyzr6mYjd%2FCopy-and-Paste-1%20\(1\).gif?alt=media\&token=f50c1698-bfa2-46fa-b47d-160b6a4c8e63)

If you want to copy more that one cell from multiple columns, you must copy the same number of cells from each column:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FejnGMpcF1AV2Wsjo38wS%2FCopy-and-Paste-2-Cells%20\(1\).gif?alt=media\&token=c0e07f66-32dc-469a-b3a5-befe59d02aa0)

If the number of cells you select to paste data in is more than the number of cells you have copied, the data you have copied will repeat itself to fill the highlighted cells.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2Fudf5Og0akcNxhXFfDtAG%2FCopy-and-Paste-Not-enough-cells.gif?alt=media\&token=69879ea0-d4d8-463b-8ecf-8692004c6c7d)

If the number of cells you select to paste data in is less than the number of cells you have copied, you will only paste data to fill the highlighted cells i.e. if you copy the data from 4 cells but select to paste that data into 3 cells, then only the data from the first 3 cells you copied will be pasted.&#x20;

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FEXndFv1Nc4y1W2sLz6CG%2FCopy-Paste-too-few.gif?alt=media\&token=a77aeb65-1941-4817-b29a-dc81d3103a96)

If you select to paste data into multiple places, the data that is pasted will begin from the start of the first cell you copied.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FI8zRXZquAS4EtUooWXWF%2FCopy-and-Paste-multiple-places.gif?alt=media\&token=1b7a7ffa-534b-4f7b-8e70-69654bb40480)

The data from the first cell you select will be pasted first:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2Fk1WG3rZWBPK71WvW0ZPW%2FCopy-and-Paste-order.gif?alt=media\&token=e70968c9-8880-4c66-a373-cf72ca6f8918)
