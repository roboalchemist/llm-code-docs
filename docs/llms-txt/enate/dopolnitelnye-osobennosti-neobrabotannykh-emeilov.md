# Дополнительные особенности Необработанных емейлов

## Когда емейлы появляются в просмотре Необработанных емейлов?

Емейлы появятся в просмотре Необработанных емейлов в просмотре Вашей «Входящей почты» в Менеджере Работ, если они отвечают следующим условиям:

1. Ни в графе получателя, ни в графе копии не указаны электронные адреса, совпадающие с соответствующим емейл маршрутом.
2. Емейл адрес указан только для скрытой копии, но не для получателя или копии.
