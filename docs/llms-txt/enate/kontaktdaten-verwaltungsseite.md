# Die Kontaktverwaltungsseite

## Überblick

Auf der Kontakt Verwaltungsseite können Sie alle externen Kontakte an einem Ort sehen und verwalten.

{% hint style="info" %}
Note: you will only be able to access the Contact Management Page if you have been set up with the relevant permissions in Enate Builder. See [here ](https://docs.enate.net/enate-help/builder/builder-2021.1/user-management/service-agents#access-to-contact-management-page)for more information.
{% endhint %}

Sie können auf die Kontakt Verwaltungsseite vom Navigations-Link aus zugreifen. Ihre Kontakte und deren Information wird in einer Tabelle dargestellt.&#x20;

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FXeVwoIYScByu56fngxj2%2F7%20CM%20Page%20Clicking-on-Contact-Management.gif?alt=media\&token=376419ab-bc20-4a3d-bfc6-80b04b1e1e69)

Wenn Sie auf den Vornamen eines Kontakts klicken, werden Sie zu dessen [Kontakt-Aktivitätenseite](https://docs.enate.net/enate-help/deutsch/kontakte/die-aktivitaetsseite-kontakt) weitergeleitet.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F5ajD7nYiqMmtBXw5mwke%2F7%20CM%20Page%20Clicking-on-Contact-Activity-P.gif?alt=media\&token=fd663ef3-db7c-49ff-bd4e-144a7b9ee06d)

## Kontakte hinzufügen

### Einen Kontakt hinzufügen

Sie können auf der Kontaktverwaltungsseite Kontakte hinzufügen, indem Sie auf das „Kontakt hinzufügen“-Symbol klicken und die Angaben dazu in dem Pop-up-Fenster eintragen, das dann erscheint.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FlZC6Ly41HGz5HPEqHMSs%2F7A%20Adding-Contact-from-Contact-Mana.gif?alt=media&#x26;token=0c48234a-22e3-49da-bb63-e11f7cef962c" alt=""><figcaption></figcaption></figure>

### Mehrere Kontakte hinzufügen

Sie können über die Kontaktverwaltungsseite eine Excel-Liste mit Kontaktdaten importieren. Es gibt dafür eine Vorlage, die in allen von Enate angebotenen Sprachen zur Verfügung steht.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FPj4yu59EZ55usJBIzqYz%2F7A%20Bulk-Adding-Contacts.gif?alt=media&#x26;token=e54450bc-9e2f-4094-a365-278d7d107d34" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Wenn Sie Kontakte aus einer Excel-Vorlage importieren, müssen Sie eine E-Mail-Adresse angeben. Wenn Sie keine Unternehmen angeben, wird der Kontakt automatisch zur Kategorie „Global“ hinzugefügt. Hier finden Sie mehr Informationen über [Kontaktbegrenzung](https://docs.enate.net/enate-help/deutsch/kontakte-hinzufugen-bearbeiten-und-loschen#firmenname-zweck-der-externen-kontakte).
{% endhint %}

### Einen Kontakt bearbeiten

Um einen Kontakt zu bearbeiten, gehen Sie auf die Kontaktverwaltungsseite und rufen per Doppelklick auf den Kontakt das Pop-up-Fenster „Kontakt bearbeiten“ auf.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FEmBqhYM7clOpLIW7qKNd%2F7A%20Editing-a-Contact-in-Contact-Management%20Page.gif?alt=media&#x26;token=d67a48ca-4cc6-4724-a422-a873dd9fc2bb" alt=""><figcaption></figcaption></figure>

Sie können auch das Unternehmen, die Zeitzone, den Bürostandort und die bevorzugte Sprache Ihrer Kontakte massenweise bearbeiten, indem Sie den Kontakt links auswählen, woraufhin der „Bearbeiten“-Button erscheint.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F4JYkdeLMp909rgjAvAVu%2F7A%20Bulk-Editing-Contacts-in-Conta.gif?alt=media&#x26;token=8056b621-a434-46fb-bb8d-3bb3c1bb64d6" alt=""><figcaption></figcaption></figure>

### Einen Kontakt löschen

Um einen Kontakt zu löschen, gehen Sie auf die Kontaktverwaltungsseite und wählen den Kontakt links aus, woraufhin der „Löschen“-Button erscheint. Sie können mehrere Kontakte gleichzeitig löschen.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2Fy4cAdqiWLwD6eJy8StHG%2F7A%20Deleting-Conacts-from-Contact.gif?alt=media&#x26;token=bf638252-2871-43e0-b41e-8795aefdeb7b" alt=""><figcaption></figcaption></figure>

## Anpassung der Spalteneinstellungen

Sie können auswählen, welche Spalten angezeigt werden sollen, indem Sie auf das Zahnradsymbol klicken und aus Ihrer Liste der Standardspalten auswählen.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FaOenAbY6n5yMKIyTNlw6%2Fimage.png?alt=media\&token=3a3bb8e3-6a5f-463c-8abd-0d45a88a959b)

In der Spalte "Automatisch erstellt" können Sie sehen, wie die externen Kontakte erstellt wurden - ob sie automatisch oder manuell erstellt wurden. Bitte beachten Sie, dass ein automatisch erstellter Kontakt nicht mehr als automatisch erstellter Kontakt in der Spalte "Automatisch erstellt" auf der Seite "Kontaktverwaltung" angezeigt wird, sobald er bearbeitet wurde.

Sie können den Spalten benutzerdefinierte Spalten hinzufügen, indem Sie auf das Zahnradsymbol klicken und aus Ihrer Liste der benutzerdefinierten Datenspalten auswählen. Diese werden in Builder konfiguriert( klicken Sie [hier ](https://docs.enate.net/enate-help/builder/builder-2021.1/custom-data-and-custom-card-configuration)für weitere Informationen). Es können Felder aller Datentypen erstellt werden, mit Ausnahme von Tabellenfeldern und Langtextfeldern.

Sie können die Tabelle nach einer Spalte in alphabetischer Reihenfolge sortieren, indem Sie auf den Spaltentitel klicken, und Sie können zur ursprünglichen Spaltenauswahl zurückkehren, indem Sie auf "Zu Standardwerten zurückkehren" klicken.
