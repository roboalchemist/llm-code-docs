# Getting Started with Automation Anywhere

## Metabot Installation

Follow these steps to sync Automation Anywhere with Enate:

Logged into Automation Anywhere, click on the MetaBots icon.

<figure><img src="https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FCdzyC85BThlIjWVPu7YE%2Fimage.png?alt=media&#x26;token=53c3dba5-8b12-4ab2-b659-6fd767303e3a" alt=""><figcaption></figcaption></figure>

Click on 'Edit', then select 'Locate on Disk'. Then paste Enate Metabot to the opened location path

<figure><img src="https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FCxIsqOedaWD7E03E5LDf%2Fimage.png?alt=media&#x26;token=137fdd06-aded-496a-9bb2-858eac7f3bba" alt=""><figcaption></figcaption></figure>
