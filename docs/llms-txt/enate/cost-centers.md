# Cost Centers

As part of [system-level settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings) you can view, edit and create Cost Centers to identify where geographically a user or a contact are.

### Adding New Cost Centers <a href="#adding-new-contact-tags" id="adding-new-contact-tags"></a>

To create a new Cost Center, you must provide a name and a code followed by an optional description.
