# Locations

As part of [system-level settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings) you can view, edit and create Locations to identify where geographically a user or a contact are.

### Adding New Locations <a href="#adding-new-contact-tags" id="adding-new-contact-tags"></a>

To create a new Location, you must provide a name and then you can optional add a description, address, city, state, country and postal code
