# Agregar una Nota

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTM2MTYxMA==>" %}

Haga clic en la pestaña *Nota* para agregar una nota. Cualquiera puede agregar una nota interna a un Ticket, un Caso o una Acción – no es necesario que esté asignado a usted, y agregar una nota no le asignará automáticamente el trabajo a usted. Estas notas son para uso interno y los contactos externos no las verán nunca.

![](https://429105696-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-987003710%2Fuploads%2FAluKKgqjfgguWbSKVgze%2Fimage.png?alt=media\&token=1c7fd20b-11d5-4847-8890-c333fb84524b)

### Especificar tipo de Nota

Si lo desea, puede especificar el tipo de interacción en la *Nota,* p.ej. teléfono entrante / saliente, etc.

![](https://429105696-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-987003710%2Fuploads%2F6XjESJshNtzdrpTnkAWq%2Fimage.png?alt=media\&token=ce890d4a-6f76-452c-ab13-aef98eacf8a0)

### Agregar una Nota y resolver un Ticket

Si estamos agregando una Nota a un Ticket asignado a nosotros, tenemos la opción adicional de Agregar una Nota y Resolver la tarea, que resolverá la tarea cuando hagamos clic sobre ella.

![](https://429105696-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-987003710%2Fuploads%2Fu1FV4gJFiY3AqkxNF4aC%2Fimage.png?alt=media\&token=a4490103-6590-45db-95bf-3b42445c8190)

Después de agregar la Nota, la pantalla se actualizará mostrando la nueva nota en la línea cronológica.

![](https://429105696-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-987003710%2Fuploads%2FBDUjVESeWc60UnjhBHvS%2Fimage.png?alt=media\&token=fee08c3a-7b7d-4c28-af1a-2c9d8470d1f7)

### Cerrar un elemento con notas incompletas

Si ha comenzado a ingresar una nota, pero no la ha guardado, la pestaña mostrará un marcador ‘\*’ en la pestaña, que le alerta de este suceso.

![](https://429105696-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-987003710%2Fuploads%2FWPuSCVkQaPuG5bHH5IBp%2Fimage.png?alt=media\&token=16c02d1e-3c1c-4e04-a7cd-20e083d63de0)

Si posteriormente intenta cerrar la pestaña *Tarea* con estos elementos "*En progreso*" aún sin guardar, el sistema lo alertará para preguntarle si desea continuar.
