# Erweiterte Suche

Diese erweiterte Suchfunktion, die auf dem Bildschirm "Ansichten" verfügbar ist, bietet Ihnen eine flexible Möglichkeit, die Arbeitsaufträge in Ihrem Geschäftsbereich anzuzeigen, indem Sie mehrere verschiedene Kombinationen von Suchkriterien angeben können, so dass Sie die Daten Ihrer Arbeitsaufträge genau so anzeigen können, wie Sie es wünschen.

Sie können die Registerkarte 'Ansichten' aus dem Navigationsbereich aufrufen.  Wenn Sie den Mauszeiger über die Option "Ansichten" bewegen, sehen Sie auch eine Liste Ihrer [gespeicherten Ansichten](#ihre-ansichten-speichern) (falls Sie welche haben), die alphabetisch sortiert sind. Sie können gespeicherte Ansichten ganz einfach erstellen, wenn Sie mit Ihren Daten auf der Registerkarte "Ansichten" arbeiten, so dass Sie Ihre Suchkriterien nicht jedes Mal neu eingeben müssen.

## Auswählen, welche Datenspalten in den Suchergebnissen angezeigt werden sollen <a href="#b-auswaehlen-welche-datenspalten-in-den-suchergebnissen-angezeigt-werden-sollen" id="b-auswaehlen-welche-datenspalten-in-den-suchergebnissen-angezeigt-werden-sollen"></a>

Wie in Ihrem Arbeits Posteingangs- und Eigener Arbeitstabelle können Sie wählen, welche Datenspalten Sie in Ihrem Suchergebnisraster anzeigen oder ausblenden möchten, indem Sie auf das Zahnradsymbol oberhalb des Suchergebnisrasters klicken.

Es erscheint ein Popup-Fenster 'Anzeige- und Spalteneinstellungen', in dem Sie auswählen können, welche Datenspalten Sie für Ihre Suchergebnisse anzeigen möchten.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYtz7NIF5ed0SdI8MIe%2F-MYu-kenHwpZZG5g3Dsh%2FOrder-Views-Data-Columns.gif?alt=media\&token=05996cb2-015b-4df9-b8a7-611bf66e686b)

Anmerkung: Diese Einstellungen für benutzerdefinierte Datenspalten bleiben erhalten, wenn Sie mehrere Suchvorgänge durchführen und wenn Sie sich das nächste Mal wieder anmelden.
{% endhint %}

## Ihre Ansichten speichern

Sie können Ihre Ansichten auch speichern, um sie jederzeit wieder aufrufen zu können. Die Anzahl der Ansichten, die Sie speichern können, ist nicht begrenzt.

## Ansichten erweiterter Suchen in Excel-Dateien exportieren

Sie können Ihre Ansichtsdaten aus Ihren erweiterten Suchen als Excel-Dateien exportieren. Sehen Sie sich dieses Video an, um mehr zu erfahren

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTQwNzYzOQ==>" %}

Um Ihre Ansichtsdaten in eine Excel-Tabelle zu exportieren, geben Sie Ihre Suchkriterien in auf der Seite „Erweiterte Suche“ ein und klicken „Zum Export wechseln“ an.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FvNRSZu5WgaeSgqdt8W0D%2Fimage.png?alt=media&#x26;token=8d0dc489-8adb-4821-b6f4-4046ce6fff92" alt=""><figcaption></figcaption></figure>

Sie sehen dann eine Liste aller bereits exportierten Dateien inklusive der Informationen, wann sie heruntergeladen wurden, wie groß sie sind, wie viele Zeilen sie enthalten, wann der Export beantragt wurde, wann er begonnen hat und wann er abgeschlossen wurde.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F17lOK48fe63F4kjOK1Dy%2Fimage.png?alt=media&#x26;token=684b939e-0465-4a4b-a40e-d5666ded715d" alt=""><figcaption></figcaption></figure>

Bevor Sie Ihre Suche exportieren, können Sie den Namen, unter dem die Datei exportiert wird, bearbeiten.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FT5ON1mGyF4NvbiESimdl%2Fimage.png?alt=media&#x26;token=6eefa30f-007c-484b-b3e5-7b694af33d4e" alt=""><figcaption></figcaption></figure>

Sie können mit einem Klick auf „Spalten auswählen“ auch anpassen, welche Spalten Sie in Ihrem Excel-Bericht sehen möchten. Die in Ihrer Suche ausgewählten Spalten werden automatisch ausgewählt, aber Sie können auch diese Auswahl ändern.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FaiNEyjxMcZKvw8tHvYMn%2Fimage.png?alt=media&#x26;token=5d554745-e8ab-41c1-9480-8e6d7496acaa" alt=""><figcaption></figcaption></figure>

Klicken Sie dann auf „Exportieren“.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FXNZNHVCVkMPZVbsuUgWV%2Fimage.png?alt=media&#x26;token=8fbfdb59-3b72-42b1-913c-27f44b058463" alt=""><figcaption></figcaption></figure>

Ihre Datei wird als neue Zeile im Abschnitt „Exportierte Dateien“ neben einem Icon „In der Warteschlange“ erscheinen. Aktualisieren Sie die Tabelle.&#x20;

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F4Pi9HDmYhbHE3VUDj4tt%2Fimage.png?alt=media&#x26;token=1458eeba-58b4-468c-9a11-9a68d821653a" alt=""><figcaption></figcaption></figure>

Sobald Sie die Tabelle aktualisiert haben, erscheint ein grüner einzelner Haken, der bedeutet, dass der Export abgeschlossen wurde.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FokUTMi3cij0lenARE4Xe%2Fimage.png?alt=media&#x26;token=f9d72a0d-640b-4e3d-91de-50a83c3dda05" alt=""><figcaption></figcaption></figure>

Sollte es zu einem Fehler beim Exportieren gekommen sein, wir ein rotes Warn-Icon angezeigt.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FByAoxuJWcmkveeZ3uSnU%2Fimage.png?alt=media&#x26;token=5b835cee-a30a-4c55-a43c-3fdc7eb6a7a5" alt=""><figcaption></figcaption></figure>

Mit einem Klick auf den Dateinamen können Sie die Datei herunterladen.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FKZo1NdkaYzgCobaV6eBB%2Fimage.png?alt=media&#x26;token=c35b048d-bf46-45e4-85be-451ac8157dd4" alt=""><figcaption></figcaption></figure>

Sobald die Datei heruntergeladen wurde, wird aus dem Haken ein doppelter grüner Haken, der bedeutet, dass die Datei heruntergeladen worden ist.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FxrNVRiccpIvIslCrCiJy%2Fimage.png?alt=media&#x26;token=c1c250af-165b-4c6b-b281-da10a73610d7" alt=""><figcaption></figcaption></figure>

Die Uhrzeit und das Datum des Downloads erscheinen dann in der Spalte „Heruntergeladen am“.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FPkQmlwp92ChR4Y8aTjLQ%2Fimage.png?alt=media&#x26;token=ba619d39-4769-4be4-b0b0-e831161d3962" alt=""><figcaption></figcaption></figure>

Die Daten aus Ihrer erweiterten Suche wurden dann als .xlsx-Datei exportiert, inklusive der Spaltenüberschriften, die Sie ausgewählt haben.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F7Q9Iw03RIMdoeA2Qxanl%2Fimage.png?alt=media&#x26;token=b4c9113e-7fb6-411d-b873-162827a92cb1" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Bitte beachten Sie, dass die Spaltenüberschriften in dieser Erstausgabe nur auf Englisch verfügbar sein werden. Diese werden in einem anstehenden Update lokalisiert.

Beachten Sie des Weiteren bitte, dass Sie maximal eine Million Zeilen in einer Datei exportieren können.

Die Spaltenüberschriften in den Exportdateien unterscheiden sich von denen auf der Seite „Erweiterte Suche“.
{% endhint %}

{% hint style="info" %}
Zusätzliche Anmerkung: Der Zugriff zu dieser Excel-Export-Funktion ist für Teammitglieder und TeamleiterInnen nicht aktiviert. Um NutzerInnen Zugriff zu dieser Funktion zu geben, muss im Abschnitt „NutzerInnen-Management“ eine benutzerdefinierte Rolle erstellt werden, für die im Abschnitt Funktionszugriff für „Erweiterte Suchen“ die Erlaubnis „Kann erweiterte Suchansichten in Excel-Dateien exportieren“ erteilt wird.
{% endhint %}

## Kopieren/Einfügen in Excel <a href="#e-copy-paste-into-excel" id="e-copy-paste-into-excel"></a>

Sie können Informationen aus dem Ansichten-Raster in eine Excel-Tabelle kopieren und einfügen, indem Sie die zu kopierenden Informationen auswählen und die Tasten Strg C und Strg V drücken. Die Spaltentitel werden automatisch kopiert und ebenfalls eingefügt.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYuS6h2QRhLllsba5Dk%2F-MYuTgl-EXI_C_I4eJLQ%2FCopy-Views-to-Excel.gif?alt=media\&token=e89b9890-c260-48f0-867c-6ab7bd5293c4)

Sie können auch alle Informationen aus dem Ansichten-Raster kopieren und einfügen, indem Sie Strg-A drücken.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYuS6h2QRhLllsba5Dk%2F-MYuUled-D4l_4a1Z5V4%2FCtrl-A-Into-Excel-from-Views.gif?alt=media\&token=a1c81a29-3b81-4b31-aa1d-903abe8ba7a0)

{% hint style="info" %}
Hinweis: Es werden nur Informationen kopiert, die im Raster "Ansichten" geladen wurden.
{% endhint %}
