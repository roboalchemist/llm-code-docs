# Обзор типов задач

Существует три типа задач, над которыми пользователи могут работать в Enate:

* [**Заявки** ](https://docs.enate.net/enate-help/ru/obrabotka-zayavki)- Используется для моделирования однокомпонентной деятельности, например, запросов. Заявки являются самостоятельными и не являются частью бизнес-процесса. Могут быть повышен в приоритете, чтобы стать конкретным случаем.
* [**Случаи** ](https://docs.enate.net/enate-help/ru/obrabotka-sluchaya)- Используется для моделирования деятельности, состоящей из нескольких частей (т.е. бизнес-процессов).
* [**Действия** ](https://docs.enate.net/enate-help/ru/obrabotka-deistviya)- составные части случая, т.е. случай состоит из потока действий. Содержит набор инструкций, часто контрольный список действий для отслеживания прогресса в рамках Действия. Это могут быть как ручные (могут выполняться людьми и ботами), так и автоматические действия, например, автоматическая отправка сообщений по электронной почте.
