# Raport de vizualizare a echipei

Raportul de vizualizare a echipei afișează date despre activitatea desfășurată în Cozile de așteptare pe care le gestionați sau din care faceți parte. Raportul de vizualizare a echipei conține următoarele date disponibile:

| Tabel                                             | Domenii                                            | Descriere                                                                                                                                                                           |
| ------------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Activitate                                        | Utilizatori activi                                 | Numărul de utilizatori care sunt activi la o anumită dată selectată                                                                                                                 |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Număr total de ore înregistrate</p>     | Durata totală, exprimată în ore, petrecută de un agent cu toate articolele de lucru la care a lucrat în intervalul de date selectat                                                 |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Total utilizatori</p>                   | Numărul total de utilizatori responsabili pentru articolele de lucru dintr-o Coadă de așteptare                                                                                     |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Tip de activitate</p>                   | Tipul de activitate realizată de un agent cu privire la articolul de lucru (de ex., „Salvat” sau „Finalizat”)                                                                       |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Durata în sec</p>                       | Durata, exprimată în secunde, a timpului petrecut de un agent cu un articol de lucru. Dacă este disponibilă, se ia în considerare și durata introdusă manual.                       |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Data inițială</p>                       | Data de lansare a activității aferente articolului de lucru în format dată/oră                                                                                                      |
| <p> </p><p>Activitate</p>                         | <p> </p><p>Data inițială\_data</p>                 | Data de lansare a activității aferente articolului de lucru, doar în funcție de dată                                                                                                |

| Context                                           | Contract                                           | Nume contract                                                                                                                                                                       |
| Context                                           | Client                                             | Nume client                                                                                                                                                                         |

| Data                                              | Anul                                               | Anul datelor                                                                                                                                                                        |
| Defecte                                           | Numărul de articole de lucru                       | Numărul de articole de lucru care prezintă defecte                                                                                                                                  |
| <p> </p><p>Defecte</p>                            | <p> </p><p>Data la care s-a generat</p>            | Data la care a fost generat defectul pentru un articol de lucru                                                                                                                     |
| Defecte                                           | Categoria defectelor                               | Categoria defectului                                                                                                                                                                |
| Defecte                                           | Descriere                                          | Descrierea defectului                                                                                                                                                               |
| Defecte                                           | Status                                             | Menționează dacă defectul a fost sau nu rezolvat                                                                                                                                    |
| Feedback                                          | Evaluarea medie                                    | Media evaluărilor pentru fiecare articol de lucru                                                                                                                                   |
| Feedback                                          | Evaluare                                           | Evaluarea utilizatorilor pentru fiecare articol de lucru                                                                                                                            |

| Proces                                            | Proces                                             | Numele procesului din care face parte fiecare articol de lucru                                                                                                                      |
| Proces                                            | Tipul articolului de lucru                         | Tipul articolului de lucru (Tichet, Acțiune sau Caz)                                                                                                                                |
| <p> </p><p>Cozi de așteptare</p>                  | <p> </p><p>Coada de așteptare</p>                  | Numele Cozii de așteptare în care a apărut ultima dată articolul de lucru                                                                                                           |
| <p> </p><p>Status</p>                             | <p> </p><p>Status</p>                              | Statusul articolului de lucru (De făcut, În curs de desfășurare, În așteptare, Rezolvat, Închis)                                                                                    |

| Istoric al statusului de așteptare                | <p> </p><p>Data inițială\_data</p>                 | Data de lansare, doar în funcție de data la care articolul de lucru a fost În stare de așteptare                                                                                    |
| Istoric al statusului de așteptare                | <p> </p><p>Nume utilizator</p>                     | <p> </p><p>Agentul responsabil care a setat statusul articolului de lucru la „În așteptare”.</p>                                                                                    |
