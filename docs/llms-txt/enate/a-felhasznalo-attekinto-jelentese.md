# A felhasználó áttekintő jelentése

A felhasználó áttekintő jelentése az Ön saját munkájával kapcsolatosan jelenít meg adatokat.

A felhasználó áttekintő jelentése a következő elérhető adatkészleteket tartalmazza:

| Táblázat                                   | Mezők                                                | Leírás                                                                                                                                                     |
| ------------------------------------------ | ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <p> </p><p>Tevékenység</p>                 | <p> </p><p>Összes naplózott óra</p>                  | Az ügynök által az összes munkatételen eltöltött teljes idő (órában megadva) a kiválasztott dátumtartomány során                                           |
| <p> </p><p>Tevékenység</p>                 | <p> </p><p>Tevékenységtípus</p>                      | Egy ügynök által a munkatételen elvégzett tevékenység típusa („Elmentés” vagy „Befejezés”)                                                                 |
| <p> </p><p>Tevékenység</p>                 | <p> </p><p>Időtartam másodpercben</p>                | Egy ügynök által a munkatételen eltöltött idő (másodpercben megadva). Ha rendelkezésre áll, akkor figyelembe vesszük a manuálisan megadott időt.           |
| Tevékenység                                | StartDate                                            | A munkatétel-tevékenység kezdő dátuma dátum/idő formátumban megadva                                                                                        |
| Tevékenység                                | StartDate\_date                                      | A munkatétel-tevékenység kezdő dátuma csak dátum formátumban megadva                                                                                       |
| Tevékenység                                | Felhasználónév                                       | A tevékenységet elvégző felhasználó                                                                                                                        |
| Kontextus                                  | Szerződés                                            | A szerződés neve                                                                                                                                           |
| Kontextus                                  | Ügyfél                                               | Az ügyfél neve                                                                                                                                             |
| Kontextus                                  | Szolgáltatás                                         | A szolgáltatás neve                                                                                                                                        |
| Kontextus                                  | Szolgáltató                                          | A szolgáltató neve                                                                                                                                         |
| Dátum                                      | Dátum                                                | Az adatszűrés dátumtartománya                                                                                                                              |
| Dátum                                      | Hónap                                                | A dátumok hónapjai                                                                                                                                         |
| Dátum                                      | Hét                                                  | A dátumok hetei                                                                                                                                            |
| Dátum                                      | Év                                                   | A dátumok éve                                                                                                                                              |
| <p> </p><p>Hibák</p>                       | <p> </p><p>Munkatételek száma</p>                    | Azon munkatételek száma, amelyek hibásnak vannak jelölve                                                                                                   |
| <p> </p><p>Hibák</p>                       | <p> </p><p>Létrehozás dátuma</p>                     | Az a dátum, amelyen egy adott munkatétel hibáját észlelték                                                                                                 |
| Hibák                                      | Hibakategória                                        | A hiba kategóriája                                                                                                                                         |
| Hibák                                      | Leírás                                               | A hiba leírása                                                                                                                                             |
| Hibák                                      | Állapot                                              | Megmutatja, hogy sikerült-e megoldani a hibát vagy sem                                                                                                     |
| Folyamat                                   | Folyamat                                             | Annak a folyamatnak a neve, amelyhez minden munkatétel tartozik                                                                                            |
| Folyamat                                   | Munkatétel típusa                                    | A munkatétel típusa (jegy, művelet vagy ügy)                                                                                                               |
| <p> </p><p>Várólisták</p>                  | <p> </p><p>Várólista</p>                             | Annak a várólistának a neve, ahol a munkatétel utoljára jelen volt                                                                                         |
|                                            |                                                      |                                                                                                                                                            |
| <p> </p><p>Állapot</p>                     | <p> </p><p>Állapot</p>                               | Munkatétel állapota (elvégzendő, folyamatban, várakozó, megoldva, lezárva)                                                                                 |
| <p> </p><p>Állapot oka</p>                 | <p> </p><p>Állapot oka</p>                           | A munkatétel állapotváltozása mögött meghúzódó ok (újonnan létrehozott, új információ érkezett be, üzleti szabály által gátolt stb.)                       |
| <p> </p><p>Várakozó állapot előzményei</p> | <p> </p><p>Várakozások száma naponta</p>             | Az ügynök által egy adott napon várakozik állapotúra állított munkatételek száma (azok közül a munkatételek közül, amelyeken az ügynök dolgozik/dolgozott) |
| Várakozó állapot előzményei                | <p> </p><p>EndDate</p>                               | A várakozó állapot befejezési dátuma dátum/idő formátumban megadva                                                                                         |
| Várakozó állapot előzményei                | <p> </p><p>EndDate\_date</p>                         | A várakozó állapot befejezési dátuma csak dátum formátumban megadva                                                                                        |
| Várakozó állapot előzményei                | <p> </p><p>StartDate</p>                             | A várakozó állapot kezdő dátuma dátum/idő formátumban megadva                                                                                              |
| Várakozó állapot előzményei                | <p> </p><p>StartDate\_date</p>                       | A várakozó állapot kezdő dátuma csak dátum formátumban megadva                                                                                             |
| Várakozó állapot előzményei                | <p> </p><p>Felhasználónév</p>                        | <p> </p><p>Az ügynök, aki a munkatétel állapotát várakozó állapotúra állította</p>                                                                         |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Művelet</p>                               | Egy ügynök által egy adott napon lezárt és várakozó munkatételek összege                                                                                   |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>A mai napon hozzárendelt munkatételek</p> | A mai napon az ügynökhöz hozzárendelt munkatételek száma                                                                                                   |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Lezárt munkatételek</p>                   | Az ügynök által a kiválasztott dátumtartományban lezárt munkatételek száma                                                                                 |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>A mai napon lezárt munkatételek</p>       | A mai napon az ügynök által lezárt munkatételek száma                                                                                                      |
| Munkatételek                               | A mai napon esedékes munkatételek                    | A mai napon az ügynök által esedékes munkatételek száma                                                                                                    |
| Munkatételek                               | Megnyitott munkatételek                              | Az ügynök által megnyitott összes munkatétel száma                                                                                                         |
| Munkatételek                               | Késedelmes munkatételek                              | Az ügynök által esedékes összes munkatétel száma                                                                                                           |
| Munkatételek                               | Újra megnyitott munkatételek                         | Az ügynök számára újra megnyitott összes jegy száma                                                                                                        |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Megoldott munkatételek</p>                | Az ügynök által a kiválasztott dátumtartományban megoldott munkatételek száma                                                                              |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Elindított munkatételek</p>               | Az ügynök által a kiválasztott dátumtartományban elindított munkatételek száma                                                                             |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Kor napokban</p>                          | Kor naptári napokban megadva. Befejezett munkatételek esetében: (záró dátum – kezdő dátum), nyitott tételek esetében: (ma – kezdő dátum).                  |
| Munkatételek                               | Érintett rekordok száma                              | Az érintett rekordok száma összesen                                                                                                                        |
| Munkatételek                               | Hibák száma                                          | Az egyes munkatételek hibáinak száma (ha van hibája)                                                                                                       |
| Munkatételek                               | Újbóli elvégzések száma                              | Az egyes munkatételek újbóli elvégzéseinek száma (ha van ilyen)                                                                                            |
| Munkatételek                               | DueDate                                              | A munkatétel határideje dátum/idő formátumban megadva                                                                                                      |
| Munkatételek                               | DueDate\_date                                        | A munkatétel határideje csak dátum formátumban megadva                                                                                                     |
| Munkatételek                               | EndDate                                              | A munkatétel záró dátuma dátum/idő formátumban megadva                                                                                                     |
| Munkatételek                               | EndDate\_date                                        | A munkatétel záró dátuma csak dátum formátumban megadva                                                                                                    |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Hibás</p>                                 | Megmutatja, hogy az adott munkatétel hibás vagy sem (igen vagy nem)                                                                                        |
| Munkatételek                               | Referencia                                           | Az egyes munkatételek referenciaszáma                                                                                                                      |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Újra megnyitották</p>                     | Jegyek, amelyeket megnyitottak, miután megoldott állapotúak voltak                                                                                         |
| <p> </p><p>Munkatételek</p>                | <p> </p><p>Szolgáltatói megállapodás</p>             | Szolgáltatói megállapodás annak jelzésére, hogy a munkatétel esedékes-e vagy sem                                                                           |
| Munkatételek                               | StartDate                                            | A munkatétel kezdő dátuma dátum/idő formátumban megadva                                                                                                    |
