## Nutzerinformationen

Wenn Sie auf Ihren Namen oder Ihr Bild klicken, gelangen Sie zu den Nutzereinstellungen. Hier können Sie Ihre bevorzugte Sprache auswählen, ein Profilbild hinzufügen, Ihr Passwort ändern und Datum und Uhrzeit auswählen.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MZCX5sRzULdwwap_qSX%2F-MZCYxEqO7XSmdqDy7oY%2FClick-on-User-Settings.gif?alt=media\&token=1de23615-4868-4c1a-b12c-1241b67b243a)

### **Sprache** <a href="#sprache" id="sprache"></a>

Enate unterstützt derzeit die folgenden Sprachen:

* Britisches Englisch
* Amerikanisches Englisch
* Spanisch
* Portugiesisch (brasilianisch)
* Rumänisch
* Ungarisch
* Polnisch
* Russisch
* Französisch
* Deutsch

Wenn Sie eine nicht-englische Sprache wählen, werden die Benutzeroberfläche und die Beschriftungen in die gewählte Sprache geändert. Wenn Ihr Systemadministrator die Übersetzung für Fall-/Ticket-/Aktionsinformationen in diese Sprachen konfiguriert hat, werden alle Fall-/Ticket-/Aktionsinformationen ebenfalls in der angegebenen Sprache angezeigt. [Klicken Sie hier für weitere Informationen](https://docs.enate.net/enate-help/builder/builder-2021.1/adding-localisations).

### **Profilbild** <a href="#profilbild" id="profilbild"></a>

Wenn Sie auf 'Profilbild bearbeiten' klicken, wird die Bildsteuerung aufgerufen, um ein neues Bild von Ihrem Gerät zu holen. Wenn Sie auf die Schaltfläche Bearbeiten klicken, können Sie zur Bildauswahl navigieren. Sie können das vorhandene Bild auch über die Schaltfläche Löschen löschen.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWqVjt5Za-7iH_NtTb-%2F-MWqWhiUhTdshI8k3V4I%2Fimage.png?alt=media\&token=267c93f5-4389-4b87-8f2c-60c94a86024a)

### **Personalisierte E-Mail-Signatur** <a href="#personalisierte-e-mail-signatur" id="personalisierte-e-mail-signatur"></a>

Die E-Mail-Signaturfunktion von Enate ermöglicht es Ihnen, Ihre eigene Signatur für ausgehende E-Mails zu konfigurieren. Sie können den Abschnitt E-Mail-Signatur verwenden, um den Signaturinhalt zu definieren, der in ausgehenden E-Mails, die Sie schreiben, gesendet werden soll.

{% hint style="info" %}
Anmerkung: Die konfigurierte E-Mail-Signatur wird nur dann in die ausgehenden E-Mails aufgenommen, wenn das Kontrollkästchen "**Meine Signatur in ausgehenden E-Mails anzeigen"** am Ende dieses Abschnitts aktiviert ist.
{% endhint %}

Wenn Sie nach dem Erstellen einer Signatur auf einen der Links klicken, um den Abschnitt "Neue E-Mail verfassen" in einem beliebigen Workitem aufzurufen, wird die konfigurierte E-Mail-Signatur unten im E-Mail-Text angezeigt.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYzQmkKW2pv6XWGPf7q%2F-MYzWrCH41AY__7Gq2S6%2FEmail-Signature%20\(1\).gif?alt=media\&token=1d648cb4-1093-4dcb-8802-33b5e61074c5)

### Ihr Password ändern

Wenn Sie Ihr Passwort ändern möchten, geben Sie hier Ihr altes und das gewünschte neue Passwort ein und klicken Sie dann auf die Schaltfläche Speichern. Nach einer erfolgreichen Zurücksetzung Ihres Passworts werden Sie auf die Anmeldeseite weitergeleitet, um sich mit Ihrem neuen Passwort wieder einzuloggen.

### Andere Personalisierungsmöglichkeiten

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FoMw5TL2ocArpRiPzOVBE%2Fimage.png?alt=media&#x26;token=f4674a73-65d3-41b0-bea0-30ea367c33ac" alt=""><figcaption></figcaption></figure>

* Mit der Option "Meine Signatur in ausgehende E-Mails einfügen" können Sie festlegen, ob Ihre personalisierte Signatur in die von Ihnen gesendeten E-Mails eingefügt werden soll oder nicht.
* Mit der Option "Popup-Fenster zum Erstellen von Kontakten nicht anzeigen" können Sie festlegen, ob Sie jedes Mal ein Popup-Fenster zum Erstellen von Kontakten erhalten möchten, wenn Sie mit einem Kontakt kommunizieren, der noch nicht im System vorhanden ist.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FRGFYBIk31pv9hjAu3QbL%2Fimage.png?alt=media\&token=7531fd2a-6e26-4e6f-84cd-84871795d8ad)

* Mit der Option "Neue Informationen nicht anzeigen" können Sie festlegen, ob Sie jedes Mal benachrichtigt werden möchten, wenn Sie einen Arbeitsauftrag aktualisieren und nicht auf das Symbol für neue Informationen geklickt haben, um die letzten Änderungen an dem Arbeitsauftrag zu sehen.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F0PqPCkvZPYUaUXXckAo8%2Fimage.png?alt=media\&token=0c9f5662-a364-4b82-9a5b-82b1c70b880b)

* Mit der Option "Keine Warnung für globalen Kollegen anzeigen" können Sie festlegen, ob Sie jedes Mal benachrichtigt werden möchten, wenn Sie einen globalen Kollegen(d. h. einen Kollegen, der nicht mit einem bestimmten Unternehmen verbunden ist) zu einem Arbeitsauftrag hinzufügen.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FeGg5h2kGi4SShBgPepF0%2Fimage.png?alt=media\&token=81f36882-ae0b-46f4-8510-e6c5eaa62225)

* FALLS [Plus Addressing](https://docs.enate.net/enate-help/builder/builder-2021.1/email-mailbox-configuration/email-processing-modes-options#what-is-plus-addressing-and-why-is-it-useful-in-enate) in Ihrem System aktiviert wurde, um Ihnen bei der Kombination von E-Mails und Arbeitsaufträgen zu helfen, werden Sie mit der Option „Die Warnung ‚Details überprüfen‘ nicht anzeigen“ konfrontiert. Sie werden gefragt, ob Sie jedes Mal gewarnt werden wollen, wenn Sie eine E-Mail an mehr als eine Adresse schicken, die bereits mit einem Enate-Posteingang verknüpft ist, und Sie nicht die Details zu jedem Arbeitsauftrag geöffnet haben, der mit dieser E-Mail-Adresse erstellt oder verlinkt wird.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FFuMqcGvNX6BYN7Qd7B8g%2Fimage.png?alt=media&#x26;token=3e2cb127-7dde-43eb-b070-a84a47218f80" alt=""><figcaption></figcaption></figure>

* FALLS [Plus Addressing](https://docs.enate.net/enate-help/builder/builder-2021.1/email-mailbox-configuration/email-processing-modes-options#what-is-plus-addressing-and-why-is-it-useful-in-enate) in Ihrem System aktiviert wurde, um Ihnen bei der Kombination von E-Mails und Arbeitsaufträgen zu helfen, werden Sie mit der Option „Die Warnung ‘Routenadresse entfernen‘ nicht anzeigen“ konfrontiert. Sie werden gefragt, ob Sie jedes Mal gewarnt werden wollen, wenn Sie entscheiden, beim Senden einer E-Mail an mehr als eine:n Empfänger:n eine Adresse zu entfernen, die bereits mit einem Enate-Posteingang verknüpft ist.&#x20;

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FTe7rSt2uIz42H4kotj35%2Fimage.png?alt=media&#x26;token=b6da7d72-0115-466e-b3fe-c4e3158beb50" alt=""><figcaption></figcaption></figure>

* Die Option „Links in neuen Registerkarten im Browser öffnen“ lässt Sie entscheiden, wie Links, die Sie in Enate anklicken, geöffnet werden sollen – entweder in einer neuen, eigenen Registerkarte im Browser oder in der Registerkarte, in der Sie gerade arbeiten. Diese Option ist standardmäßig AUSgeschaltet, also werden Links nach dem Anklicken in der aktuellen Browser-Registerkarte geöffnet. Falls Ihnen lieber wäre, Links würden in eigenen Registerkarten im Browser geöffnet werden, können Sie diese Option AKTIVIEREN und Ihre Einstellungsänderung speichern. Wenn Sie von nun an Links anklicken, sehen Sie, dass eine neue, eigene Registerkarte für diesen Inhalt geöffnet wird.

## Senden rückgängig machen <a href="#undo-send" id="undo-send"></a>

Mit der Einstellung "Senden rückgängig machen" können Sie den Versand Ihrer E-Mails zeitlich verzögern. So haben Sie die Möglichkeit, den Versand einer E-Mail abzubrechen oder eine E-Mail zu überprüfen, bevor sie gesendet wird.

Sie können die Zeit für das Rückgängigmachen des Versands einstellen, indem Sie den Schieberegler auf die gewünschte Einstellung verschieben. [Sehen sie hier für weitere Informationen](https://docs.enate.net/enate-help/deutsch/e-mails/terminieren-von-e-mails-und-des-postausgangs#undo-send).

![](https://gblobscdn.gitbook.com/assets%2F-MR4uErt41EMkGUOTvyd%2F-M_1S1iwlAMMLQte13eb%2F-M_1U-zUN7LiK-ovkdMH%2Fimage.png?alt=media\&token=98e3add0-cf2d-4651-b79d-e98ebdf6ad89)

## Benachrichtigungseinstellungen

Mit Enates Benachrichtigungen sind Sie immer am neuesten Stand darüber, was mit Tickets, Aktionen und Fällen passiert. Sie können selbst entscheiden, wie Sie Benachrichtigungen erhalten wollen, welchen Aufträgen Sie folgen wollen, über welche Events Sie benachrichtigt werden wollen und wie Sie diese erhalten wollen.

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTM2MTY5Mg==>" %}

Hier können Sie auswählen, welche Benachrichtigungen Sie erhalten möchten und wie.

Zusätzlich zum entfolgen eines bestimmten Arbeitsauftrags können Sie die Benachrichtigungseinstellungsseite auch verwenden, um zu spezifizieren, welche Benachrichtigung Sie erhalten wollen. Hier gibt es zwei Teile:

* Zuerst spezifizieren Sie, welche Arbeitsaufträge Sie erhalten wollen, beispielsweise Aufträge in Ihrem eigenen Posteingang.
* Zusätzlich dazu, können Sie spezifizieren, über welche Events über diese Arbeitsaufträge Sie benachrichtigungen erhalten wollen.

Unten sehen Sie die Details ...

### So erhalte ich Benachrichtigungen - Verschiedene Arten

Für jedes Event, dem Sie folgen, werden Sie automatisch im Arbeitsmanager im [Benachrichtigungszentrum ](https://docs.enate.net/enate-help/deutsch/benachrichtigungen#benachrichtigungen-stummschalten)(greifen Sie über das Benachrichtigungssymbol im Header darauf zu) benachrichtigt — mit einem Zähler, mit dem die Anzahl der nicht gelesenen Benachrichtigungen angezeigt wird. Sie können zusätzlich auswählen, ob Sie per [E-Mail](https://docs.enate.net/enate-help/deutsch/benachrichtigungen#e-mail-benachrichtigungen) und/oder per [Browser-Pop-Up](https://docs.enate.net/enate-help/deutsch/benachrichtigungen#browser-pop-up-benachrichtigungen) benachrichtigt werden wollen.
