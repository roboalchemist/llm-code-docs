# Queues

As part of [system-level settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings) you can view, edit and create the Queues which exist in the system.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWq25-LnWlv1WFp-ufq%2F-MWq3vQaEa0WWxSpotDq%2Fimage.png?alt=media\&token=22768bfe-fbd9-4c75-8726-2fdb25e2b74d)

For each existing Queue you will be able to see:

* The Queue Name
* The number of people linked to the Queue (Members)
* Who are the Manager(s) of the Queue

Click the + icon to create a new Queue. Click on a Queue to edit it.&#x20;

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWq25-LnWlv1WFp-ufq%2F-MWq3zc2Mu26SwmwPE1R%2Fimage.png?alt=media\&token=e5616bde-8074-430c-8f55-c7bf0c81d2e9)

Clicking on the Show Activity button lets you see what edits have been made to the Queue and when, as well as when the Queue was created and by who.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWq25-LnWlv1WFp-ufq%2F-MWq43FwSRNfBACYgAHG%2Fimage.png?alt=media\&token=6c1cd0ef-dce5-40ec-933b-bac42af91398)
