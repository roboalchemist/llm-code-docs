# Recherche avancée

Cette fonction de recherche avancée disponible sur la page « Vues » vous offre un moyen flexible de visualiser les activités de votre secteur d’activité en vous permettant de spécifier plusieurs combinaisons différentes de critères de recherche, ce qui vous permet de visualiser exactement les données relatives à vos activités que vous souhaitez voir.

Vous pouvez accéder à l’onglet « Vues » à partir de la section Navigation. Lorsque vous passez votre souris sur l’option « Vues », vous verrez également une liste de vos [Vues enregistrées](#d-saving-your-views) (si vous en avez), classées par ordre alphabétique. Vous pouvez facilement créer des Vues enregistrées lorsque vous travaillez avec vos données dans l'onglet « Vues », ce qui vous évite de devoir saisir vos critères de recherche à chaque fois.

## Choisir les données à voir dans les résultats de la recherche <a href="#b-choisir-les-donnees-a-voir-dans-les-resultats-de-la-recherche" id="b-choisir-les-donnees-a-voir-dans-les-resultats-de-la-recherche"></a>

Comme pour les grilles de votre Boîte de réception et de vos Responsabilités, vous pouvez choisir les colonnes de données que vous souhaitez voir ou masquer dans votre grille de résultats.

Il suffit de cliquer sur le roue dentée au-dessus de la grille et, dans la fenêtre « Paramètres de l’affichage et des colonnes », de sélectionner les données standard du système et les champs de données personnalisées que vous souhaitez afficher.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYtz7NIF5ed0SdI8MIe%2F-MYu-kenHwpZZG5g3Dsh%2FOrder-Views-Data-Columns.gif?alt=media\&token=05996cb2-015b-4df9-b8a7-611bf66e686b)

{% hint style="info" %}
Note : Ces préférences de données personnalisées en colonnes resteront les mêmes lorsque vous effectuez des recherches multiples et quand vous vous reconnecterez ensuite.
{% endhint %}

## Exporter des vues de la Recherche avancée dans Excel

Vous pouvez exporter les résultats de vos Recherches avancées vers Excel. Regardez cette vidéo pour en savoir plus :

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTQwNzYzOQ==>" %}

Pour exporter vos données dans une feuille de calcul Excel, saisissez vos critères de recherche sur la page Recherche avancée, puis cliquez sur *Exportation*.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2F8WKSBXYQaKtYamGMHhDI%2Fimage.png?alt=media&#x26;token=afe0a169-8fda-44a7-a75b-0b6b384e1d10" alt=""><figcaption></figcaption></figure>

Vous obtiendrez alors une liste de tous vos fichiers exportés précédemment, ainsi que la date de leur téléchargement, leur taille, leur nombre total de lignes, la date à laquelle l’exportation du fichier a été demandée, la date à laquelle l’exportation a commencé et la date à laquelle elle s’est achevée.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2FZ0K6RmI9dX1TesRdb7dV%2Fimage.png?alt=media&#x26;token=3a2ba0b0-fea7-4e13-8609-2f4e48e5bf04" alt=""><figcaption></figcaption></figure>

Avant d’exporter votre recherche, vous pouvez modifier le nom du fichier sous lequel elle sera exportée.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2Ff2JSzWJN2kUmL6ZQOrZN%2Fimage.png?alt=media&#x26;token=80a5943e-dea2-4c57-bb44-dc84790b68af" alt=""><figcaption></figcaption></figure>

Vous pouvez également définir les colonnes que vous souhaitez voir apparaître dans votre fichier Excel en cliquant sur *Sélectionner les colonnes*. Les colonnes cochées lors de votre recherche seront automatiquement sélectionnées, mais vous pouvez également ajuster votre sélection ici.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2FasQPXb6fKtDZRGJ6HYVL%2Fimage.png?alt=media&#x26;token=4465336a-b7a2-4710-9a38-5ff562459021" alt=""><figcaption></figcaption></figure>

Cliquez ensuite sur *Exporter*.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2Fqd0ItJIxnD2HjqVb1OG8%2Fimage.png?alt=media&#x26;token=45ed01b0-9ad4-4f6e-932b-c1be331de8d0" alt=""><figcaption></figcaption></figure>

Votre fichier apparaîtra sous la forme d’une nouvelle ligne dans la section Fichiers exportés, accompagné d’une icône « en attente ». Cliquez pour actualiser la grille.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2Fwb2CH3C2YJ8O5aCCDXst%2Fimage.png?alt=media&#x26;token=0c9b181c-f274-4f1d-a656-1520a98b16c1" alt=""><figcaption></figcaption></figure>

Une fois la grille actualisée, l’icône se transforme en un petit V vert indiquant que l’exportation est terminée.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2F9KMoeeFeFcQ3WgX0U4UL%2Fimage.png?alt=media&#x26;token=1e29637e-b3c7-4fec-b3f7-cd5f5d6337b9" alt=""><figcaption></figcaption></figure>

Si une erreur s’est produite lors de l’exportation, une icône d’avertissement rouge apparaîtra à côté du fichier.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2FuNLq8AD1C9BKzlQKUfr2%2Fimage.png?alt=media&#x26;token=eb486db5-78a1-4bd8-9e15-1659fdadbd6c" alt=""><figcaption></figcaption></figure>

Vous pouvez télécharger le fichier en cliquant sur son nom.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2FIMNxKmhedFDkm8pVHFIQ%2Fimage.png?alt=media&#x26;token=1e06f2c8-d71c-417b-a5bf-531390234a33" alt=""><figcaption></figcaption></figure>

Une fois le fichier téléchargé, l’icône se transforme en deux petits V verts pour indiquer qu’il a été téléchargé.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2Fg8kAbJASebjpGmXlAHK9%2Fimage.png?alt=media&#x26;token=e0b075c3-48d2-47cb-863f-09e1401fe204" alt=""><figcaption></figcaption></figure>

L’heure et la date du téléchargement apparaissent alors dans la colonne Téléchargé le.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2FT2rdQHs9wd0yHM1ue2Na%2Fimage.png?alt=media&#x26;token=ddc9ec4e-2f30-4047-a5fe-dd18ff78fdd8" alt=""><figcaption></figcaption></figure>

Les données de votre recherche avancée auront été exportées dans un fichier .xlsx, avec les titres des colonnes sélectionnées.

<figure><img src="https://675373381-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-2056118458%2Fuploads%2F0CxwQCNvMtW4S4Wa03dQ%2Fimage.png?alt=media&#x26;token=b65905f6-2a37-4072-8536-93d6cf5e687c" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Attention, pour cette première version, les titres des colonnes n’apparaîtront qu’en anglais. Ils seront localisés dans une version ultérieure.

Par ailleurs, veuillez noter que vous pouvez exporter un million de lignes au maximum.

Les noms des colonnes dans le fichier exporté sont différents des noms affichés dans l’interface utilisateur de la page de recherche avancée.
{% endhint %}

{% hint style="info" %}
Remarque complémentaire : cette fonction d’exportation vers Excel n’est pas disponible pour les rôles standard de *Membre de l’équipe* et de *Chef d’équipe*. Pour donner accès à cette fonctionnalité aux utilisateurs, un rôle personnalisé doit être créé via la section *Gestion des utilisateurs* de Builder, avec l’option *Peut exporter les vues de la recherche avancée vers Excel* activée dans la section d’accès à la fonctionnalité *Recherche avancée*.
{% endhint %}

## Copier/coller dans Excel

Vous pouvez copier et coller les informations de la grille Vues dans une feuille de calcul Excel en sélectionnant les informations que vous souhaitez copier et en utilisant les touches Ctrl C et Ctrl V. Les informations copiées et collées incluent tous les filtres appliqués. Les titres des colonnes seront également copiés et collés automatiquement.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYuS6h2QRhLllsba5Dk%2F-MYuTgl-EXI_C_I4eJLQ%2FCopy-Views-to-Excel.gif?alt=media\&token=e89b9890-c260-48f0-867c-6ab7bd5293c4)

Vous pouvez également copier et coller toutes les informations de la grille de vues en utilisant Ctrl-A.

![](https://gblobscdn.gitbook.com/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYuS6h2QRhLllsba5Dk%2F-MYuUled-D4l_4a1Z5V4%2FCtrl-A-Into-Excel-from-Views.gif?alt=media\&token=a1c81a29-3b81-4b31-aa1d-903abe8ba7a0)

{% hint style="info" %}
Remarque : seules les informations qui ont été chargées dans la grille de Vues seront copiées.
