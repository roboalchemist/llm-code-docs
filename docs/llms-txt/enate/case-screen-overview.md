# Case Screen Overview

## Overview <a href="#a-overview-what-you-are-you-doing-on-the-caseflow-screen" id="a-overview-what-you-are-you-doing-on-the-caseflow-screen"></a>

The Case screen is where you create and configure a flow of Actions to make up your business process.&#x20;

The Case screen is made up of two main sections:

1. Flow Diagram - where you can see the Steps of the Case and configure the flow of Actions for the Case. [See here for more information about designing your Case](https://docs.enate.net/enate-help/builder/builder-2021.1/case-configuration/case-info-section).
2. Info Section - where you configure the settings for the Case and its Actions. The Case Info tab and the Action Info tab will display by default, but other tabs might appear depending on what type of Actions you add to your flow. [See here for more information about configuring your Case's settings.](https://docs.enate.net/enate-help/builder/builder-2021.1/case-configuration/case-info-section)

## Further Options in the Case Screen

If you have opened the process in view-only mode, click on the blue Edit icon in the toolbar to allow changes to be made.&#x20;

This will bring up the Save and copy buttons.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp2RqT3q66l_EgqNpy%2F-MWp4XouDVCeovg3uvFu%2Fimage.png?alt=media\&token=ed0de41b-2418-446c-a492-3652fbe6545f)

You can view the status of the Case from the 'Status' dropdown.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp2RqT3q66l_EgqNpy%2F-MWp4EG5xYUPX3V-PNSh%2Fimage.png?alt=media\&token=26ece953-e8a4-4889-be4d-52f40154de1f)

Steps in a Case flow have already been defined as part of Case TYPE definition under the Service Line and cannot be changed from the Case screen. See [here ](https://docs.enate.net/enate-help/builder/builder-2021.1/service-lines-screen/creating-new-case-types-in-a-service-line)for more information on this.

Each step in the flow diagram shows you the total number of Action processes that are configured for that step.&#x20;

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp_fQpjpz0j2866Jq0%2F-MWp_jLtnM7uEueIKE0l%2Fimage.png?alt=media\&token=986d29f6-991c-4656-a53a-a636869b20bc)

You can adjust the height of the info section:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2Fsbar4XYZkpca2w6sSaVY%2FAdjust-Info-Section-Height.gif?alt=media\&token=9f3fa29c-d474-4c3b-a4f8-74ebdae6f35d)

You can also use the ‘View’ dropdown link to change the view of the screen. You can: View the Flow only; View the Info Section only; or view both sections. Your preference will be saved when you switch between Cases and when you next log in.

The 'Version' dropdown lets you choose which version of the Process you want. By default you will see the currently draft (In Progress version), but can switch to the currently Live version and even previous now retired versions to compare process flow setup.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp2RqT3q66l_EgqNpy%2F-MWp4AUhgNPYTa6s3gUU%2Fimage.png?alt=media\&token=24b0071c-1713-4329-adae-84776629ede2)

You can also restore a retired version of a Case process by opening the Case in edit mode, clicking on the retired version you want to restore, then clicking on 'Restore'.

<figure><img src="https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2F4EnhiaK1UytPWIvPrRen%2F7-Restore-Case.gif?alt=media&#x26;token=9267ea8d-3d94-4040-ae90-86fe35bac06f" alt=""><figcaption></figcaption></figure>

When you restore a retired version, it will return it to a state of draft that you can then edit, save and set live.&#x20;

{% hint style="info" %}
Note that if you already have a draft version of the process in progress, any unsaved changes that you have made will be lost upon clicking 'Restore'.
{% endhint %}

The 'Show Activity' button lets you see the previous activity for the Case Process. You can see who created/deleted/last edited the Case Process, so you know who to go to if you need further information. The information will be highlighted in Green if the object has been created, restored, enabled or set live; in yellow if the object has been updated; and in red if the object has been deleted, disabled or retired.

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp2RqT3q66l_EgqNpy%2F-MWp4I3Zfl5gOqrSlXPW%2Fimage.png?alt=media\&token=ee746ff8-1d52-42b4-ada3-b5c9cb0d9982)

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWp2RqT3q66l_EgqNpy%2F-MWp4JTslcWl1iyukAHv%2Fimage.png?alt=media\&token=b6bc2329-01d0-4dc6-abea-baaef8d3ff21)

The Show in Self Service option lets you choose if you want the Case process to be available to Self Service users. If the icon is highlighted in blue, the Case will be available to Self Service users; if the icon is grey, the Case will not be available to Self Service users.
