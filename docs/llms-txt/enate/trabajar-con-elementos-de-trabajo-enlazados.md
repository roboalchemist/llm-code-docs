# Trabajar con Elementos de Trabajo Enlazados

Varias funciones de Enate le permiten gestionar las actividades que realizan diferentes equipos, permitiéndole crear y supervisar fácilmente las tareas, incluso cuando no tiene todos los permisos para las tareas de los demás.

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTM2MTg5NQ==>" %}

Puede:

* crear y enlazar tareas, incluso en áreas de negocio para las que no tiene permisos;
* compartir y añadir comunicaciones entre esas tareas.

Al lanzar un Caso o un Ticket desde uno ya existente, se crea una relación "Enlazada" entre las tareas. Las tareas creadas de este modo mantendrán un enlace con el Caso o Ticket original y se mostrarán en una [pestaña de "Enlaces"](#visualizar-las-tareas-enlazadas-la-pestana-de-enlaces) dentro de ellas, lo que facilita el seguimiento de un grupo de tareas que se relacionan entre sí.
