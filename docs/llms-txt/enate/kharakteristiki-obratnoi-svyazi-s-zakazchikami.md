# Характеристики обратной связи с заказчиками

## Получение отзывов заказчиков по электронной почте <a href="#a-poluchenie-otzyvov-zakazchikov-po-elektronnoi-pochte" id="a-poluchenie-otzyvov-zakazchikov-po-elektronnoi-pochte"></a>

Если для категории Заявок в Конструкторе установлена галочка «Обратная связь по запросам» (<mark style="color:blue;">тут</mark> Вы можете найти больше информации), то в исходящих сообщениях содержится ссылка в нижнем колонтитуле, по которой получатели могут дать обратную связь с оценкой.

![](https://3327152148-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWskNwXafCKXzHPD007%2F-MWsxOiqio9JJAIxd6MX%2Fimage.png?alt=media\&token=d0c5322d-4da4-42ae-a3ff-0e176768c087)

После нажатия на кнопку в отдельном окне браузера откроется форма обратной связи Enate (см. ниже). В форме предлагаются те же значки обратной связи, что и в предыдущем письме, но она позволяет пользователю изменить свой выбор при желании, а также добавить дополнительный комментарий.

![](https://3327152148-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWskNwXafCKXzHPD007%2F-MWsxSapI9dGWBxRMIJy%2Fimage.png?alt=media\&token=e207d0d5-2bdc-49e1-b597-5310d7b99c35)

После того как пользователь закончил предоставление обратной связи, он просто нажимает на кнопку «Отправить», которая закрывает окно обратной связи. Данные обратной связи впоследствии сохраняются, готовые для просмотра агентом из заявки.

## Опция повторного открытия Случаев во время открытия окна обратной связи <a href="#b-opciya-povtornogo-otkrytiya-sluchaev-vo-vremya-otkrytiya-okna-obratnoi-svyazi" id="b-opciya-povtornogo-otkrytiya-sluchaev-vo-vremya-otkrytiya-okna-obratnoi-svyazi"></a>

Случаи могут быть заданы с помощью окна обратной связи (например, 2 недели), позволяющего заявителю в течение некоторого времени вернуться до того, как Случай будет полностью, безвозвратно закрыт. В течение этого периода можно открыть дело заново. Случай автоматически откроется заново, если (в окне обратной связи) поступит электронное письмо или в него будет объединена заявка

Случай может быть открыт пользователем вручную с помощью кнопки "Открыть повторно", которая представляет завершенный случай, находящийся в окне обратной связи.

![](https://3327152148-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1956395681%2Fuploads%2FbRF7na79yc5DTEbmGxDT%2Fimage.png?alt=media\&token=19877d3a-8b90-4449-8eef-2985d4b72e8a)

Окно Обратной связи для случая устанавливается в Конструкторе как часть конфигурации процесса Случая.

## Отображение последних данных обратной связи через рабочие элементы <a href="#c-otobrazhenie-poslednikh-dannykh-obratnoi-svyazi-cherez-rabochie-elementy" id="c-otobrazhenie-poslednikh-dannykh-obratnoi-svyazi-cherez-rabochie-elementy"></a>

При работе с рабочими элементами, например, заявками, агенты могут видеть текущий рейтинг отзывов заказчиков через ссылку в панели ленты:

![](https://3327152148-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1956395681%2Fuploads%2FgMDryYstbpAoQg9jaOcF%2Fimage.png?alt=media\&token=2ef45be4-6eaf-420d-82e3-278932b56fff)

При нажатии на эту кнопку откроется всплывающее окно обратной связи, в котором будет показан рейтинг этого человека (если он решил ввести) и общий средний рейтинг, вычисленный из всех записанных элементов обратной связи по одному рабочему элементу.
