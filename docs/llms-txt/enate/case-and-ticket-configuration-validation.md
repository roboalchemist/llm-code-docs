# Case and Ticket Configuration Validation

## Validation Error Display <a href="#a-validation-error-display" id="a-validation-error-display"></a>

On saving your changes the system will determine whether the configuration is now in a valid or invalid state. If invalid, a warning sign will show. Click the warning marker to view a list of validation errors:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWpuW3lJIOZFs7NIFGR%2Fimage.png?alt=media\&token=07bf94a8-9b2c-47d4-9f05-aca4d8b16676)

Clicking on the each error link will take you to the relevant section of the Ticket / Case configuration screen to help you resolve the issue, and will highlight the item which needs to be resolved.

Case / Ticket processes that fail validation can be saved but cannot be run in Work Manager.

**More fundamental validation error messages will display with a red icon:**

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWpuaQ-g1mCC-Iwma3Z%2Fimage.png?alt=media\&token=4e07e515-5b0b-4aca-ad93-8cbbc15d6a2a)

**Fundamental validation errors must be resolved before the Case / Ticket process can be saved.**

## Explicit User Permission for setting Case/Ticket configuration Live <a href="#b-explicit-user-permission-for-setting-case-ticket-configuration-live" id="b-explicit-user-permission-for-setting-case-ticket-configuration-live"></a>

Access to promote Case or Ticket configuration to ‘Live’ is an explicit permission configurable in the 'Permissions' tab of the Create/Edit User popup in Builder:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWpurwnXtl04hnzpByX%2Fimage.png?alt=media\&token=554cf0c3-8694-452e-9780-fd6558651513)

Click [here ](https://docs.enate.net/enate-help/builder/builder-2021.1/user-management/service-agents)for more information about different levels of access in Builder.
