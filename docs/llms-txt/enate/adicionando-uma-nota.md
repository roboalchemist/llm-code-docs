# Adicionando uma Nota

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTM2MTYxMA==>" %}

Clique na aba Nota para adicionar uma nota. Qualquer um pode adicionar uma nota interna a um Ticket, Case ou Ação – você não precisa tê-los atribuídos à você, e adicionar uma nota não irá atribuir automaticamente o trabalho. Essas são notas para uso interno e contatos externos não as verão.

![](https://828797872-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-419833493%2Fuploads%2FLxebBgd01doGEqIi5yQn%2Fimage.png?alt=media\&token=7d61c9f9-121d-47c8-b1ff-7766ac8197b4)

### **Especificando o tipo de nota**

Se desejar, você pode especificar qual tipo de interação resultou na nota, como por exemplo: chamada recebida/enviada, etc.

![](https://828797872-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-419833493%2Fuploads%2FcG5nTU0covwpWA5IgikW%2Fimage.png?alt=media\&token=85ff8638-194a-4b89-97e4-4693b6e70efd)

### **Adicionando um nota e resolvendo um Ticket**

Se estamos adicionando uma nota a um Ticket atribuído a nós, temos a opção adicional de Adicionar Nota e Resolver item de trabalho, o que irá resolver o item quando clicarmos.

![](https://828797872-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-419833493%2Fuploads%2Fg15VgVUypmFGLsyy9S1T%2Fimage.png?alt=media\&token=a01915cb-663e-42e0-9d95-381f727a4f4d)

Ao adicionar uma nota, a tela será atualizada e a nova nota será mostrada na linha do tempo.

![](https://828797872-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-419833493%2Fuploads%2F44xKIEaolBGDn5ietYNp%2Fimage.png?alt=media\&token=ce7862a5-4740-4df2-940d-68f8e43b3f40)

#### **Fechando de um item com notas incompletas** <a href="#closing-an-item-with-incomplete-notes" id="closing-an-item-with-incomplete-notes"></a>

Se você começou a digitar uma nota mas não a salvou, a aba exibirá um '\*' alertando-o sobre isso.![](https://828797872-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-419833493%2Fuploads%2F24uDnaJ0bXVIfo4wLKkV%2Fimage.png?alt=media\&token=3236f8f2-7b9b-4e77-a0bc-95b79bc05713)

Se posteriormente você tentar fechar a aba do item de trabalho com este item "em andamento" ainda não salvo, o sistema o alertará perguntando se você deseja prosseguir.
