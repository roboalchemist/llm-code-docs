# Useful Links

Clicking on the user dropdown will show the following dropdown:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-887967055%2Fuploads%2FORLRGhWcIR1BTqDtsD1R%2Fimage.png?alt=media\&token=e856d6e8-b7b8-4117-b83b-a8cebf1ee194)

There are three useful navigation links:

1\. [**Enate Homepage**](https://www.enate.net/) – This is the Enate Company website. Here you can see latest resources, industry insight and product release info.

2\. [**Enate Academy**](https://www.enate.academy/learn) – Here you can sign up for different Enate training courses. See [**here**](https://docs.enate.net/enate-help/implementation/enate-academy-training-courses) for more information on the training courses available at Enate Academy.

3\. [**Enate Help**](https://docs.enate.net/builder2021-1/) – This is a link to Enate's Builder help files.

4\. [**Web API Docs**](https://docs.enate.net/enate-help/integrations/enate-integrations/apis) - This is a link to Enate's WebAPI documentation which includes information about Enate's APIs and contains some sample API integrations.

5\. [**Try Web API**](https://docvmdc.enate.local/Documentation/WebAPI/swagger/ui/index) - This is a link to Enate's Web API Swagger Index.

6\. [**Webhooks Docs**](https://docs.enate.net/enate-help/integrations/enate-integrations/webhooks) - This is a link to Enate's Webhook documentation which includes information about Enate's Webhooks, API Webhook Subscription, Webhook Troubleshooting and contains some sample Webhook integrations.
