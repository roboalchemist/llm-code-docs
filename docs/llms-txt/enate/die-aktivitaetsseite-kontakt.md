# Die Kontakte-Aktivitätenseite

Die Kontakte-Aktivitätenseite zeigt Ihnen Details zur Aktivität einer/s Service-Nutzerin/Nutzers an.

Die Kontakte-Aktivitätenseite erreichen Sie über:

**1. Die Schnellsuche**— indem Sie in der [Schnellsuche](https://docs.enate.net/enate-help/deutsch/schnellsuche) auf einen Nutzer/eine Nutzerin klicken

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYEeZvyo3kK3CdIt5hu%2F-MYEg6F9sJEf7DZGHI3u%2FQuickfind-to-Contact-Activity-Pa.gif?alt=media\&token=b757bf9c-9ac1-4d31-8ef8-932354634fec)

**2. Einen Arbeitsauftrag** – indem Sie in der [Kontaktkarte](https://docs.enate.net/enate-help/deutsch/kontakte/kontakte) eines Arbeitsauftrags auf einen Kontakt klicken

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYEeZvyo3kK3CdIt5hu%2F-MYEh5SNyUu3ZxSzpsVL%2FWork-Item-to-Contact-Activity.gif?alt=media\&token=245c922f-6aa9-495e-bdf9-c896129e8358)

**3. Die Kontaktverwaltungsseite** – indem Sie auf der [Kontaktverwaltungsseite](https://docs.enate.net/enate-help/deutsch/kontakte/kontaktdaten-verwaltungsseite) auf den Vornamen eines Kontakts klicken

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MYAf86bhlx18jwx2Lze%2F-MYAirfF8i7EXaVMTFQZ%2FClicking-on-Contact-Activity-Pag.gif?alt=media\&token=173bcab8-e7f1-440c-ac3e-76834bdff5c8)

Auf der Kontakte-Aktivitätenseite können Sie dann rechts die Kontaktdetails sehen und diese wenn nötig aktualisieren.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FjRKskRdgJ0NCV1MCx0Yc%2F7-CA-Edit-Contact-Details.gif?alt=media&#x26;token=b9c2b473-e0f4-4ef9-a971-7c174e005b1b" alt=""><figcaption></figcaption></figure>

Mit einem Klick auf die Arbeitsauftrags-Registerkarte rufen Sie alle Arbeitsaufträge auf, die mit dem Kontakt verknüpft sind. Jeder Arbeitsauftrag wird als Karte angezeigt, die die folgenden Informationen beinhaltet: die Referenznummer, den Titel des Arbeitsauftrages, das Fälligkeitsdatum, den zugewiesenen Mitarbeitenden, die Warteschlange und den Status. Mit einem Klick auf die Referenznummer und den Titel können Sie Arbeitsaufträge öffnen, um weitere Details angezeigt zu bekommen. Laufende Aufträge werden automatisch angezeigt, abgeschlossene Arbeitsaufträge können auch angezeigt werden.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FaJNR5gMgpKJzwLRpms00%2F7-CA-Page-Work-Items-Tab.gif?alt=media&#x26;token=77b2447c-5c88-42b5-a77b-eeb5c28745f8" alt=""><figcaption></figcaption></figure>

Wenn Sie auf die Registerkarte „Kommunikation“ klicken, werden Ihnen dem Nutzer/der Nutzerin und den Arbeitsaufträgen zugehörige Kommunikationen (E-Mails, Notizen) angezeigt. Sie können hier auch nach bestimmten Kommunikationen suchen.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FkUcprOZQCtJu6L8c0Vqa%2F7-CA-Page-Comms-Tab.gif?alt=media&#x26;token=2fe0c0e6-c778-4896-8776-f998ccc8395c" alt=""><figcaption></figcaption></figure>

Von der Kontakte-Aktivitätenseite aus können Sie auch einen neuen Arbeitsauftrag starten. Wenn Sie von der Kontakte-Aktivitätenseite aus einen neuen Arbeitsauftrag starten, wird der Kontakt automatisch zum Hauptkontakt, zum/zur Antragsteller/in und zum Betreff des Arbeitsauftrags. Wenn bei diesem der Standard-Tag eingestellt ist, wird der Kontakt zudem auch als solcher für den Arbeitsauftrag eingetragen. Bitte beachten Sie, dass die Option „Senden Sie automatisierte E-Mails“ beim Start eines Tickets automatisch ausgeschaltet ist.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FS9lklB2OfqdGe1NH3VR6%2F7-CA-Page-Start-New-Work-Item.gif?alt=media&#x26;token=a2e9dde7-c8ec-45d5-a885-09cde3d8dad3" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Ob Sie einen Arbeitsauftrag von hier aus starten können, hängt davon ab, ob er [im Builder für einen Start von der Kontakte-Aktivitätenseite aus konfiguriert wurde ](https://d.docs.live.net/enate-help/builder/builder-2021.1/service-matrix-screen/adding-a-new-ticket-case-to-a-service#c-opening-editing-setting-a-ticket-or-case-live), welchem [Unternehmen der Kontakt zugewiesen wurde](https://d.docs.live.net/enate-help/work-manager/work-manager-2021.1/managing-contacts/adding-editing-and-deleting-contacts#adding-contacts) als er im Arbeitsmanager erstellt wurde, und auch welchem [Unternehmen Sie als Nutzer/in zugeteilt wurden](https://docs.enate.net/enate-help/deutsch/kontakte-hinzufugen-bearbeiten-und-loschen#firmenname-zweck-der-externen-kontakte).
{% endhint %}

Sie können den Namen der Nutzerin/des Nutzers kopieren, indem Sie den „Kopieren“-Button in der Registerkarte anklicken.
