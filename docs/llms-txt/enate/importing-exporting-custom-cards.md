# Importing / Exporting Custom Cards

## &#x20;<a href="#importing-exporting-custom-cards" id="importing-exporting-custom-cards"></a>

#### Exporting Custom Cards <a href="#exporting-custom-cards" id="exporting-custom-cards"></a>

To export a card, select the desired card(s) from the main screen list and select the export link

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWpplkFMhVaG-IK_h03%2Fimage.png?alt=media\&token=eef8f7ad-4d97-44e5-97cb-062276803913)

These will automatically export to your downloads folder. as '.en8Card' files.

#### Importing Custom Cards <a href="#importing-custom-cards" id="importing-custom-cards"></a>

To import a card, select the import link and in the resulting screens select the '.en8Card' files you wish to import

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWpprPQBg89ijj7MHAQ%2Fimage.png?alt=media\&token=540897a0-6afa-4425-8df6-9689ef71ef8a)

If the card is a duplicate (shares same Card name) as an existing form you will be asked to confirm updating of that card..

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWpixQQ-XB_sHNapfgn%2F-MWppuKVM96alZQMdIDv%2Fimage.png?alt=media\&token=c04fe59c-abdb-442c-ae15-9a7edc7d49ca)
