# Maximum Limits in Enate

Enate's system administrators and builders can use the information provided here to understand the data limits for different objects. The intention is to help Enate clients understand the tested and supported data limits of the system. Additionally, warnings may start to be added into the system for end users once they reach the maximum limit for a certain type of data.

See the diagram below for information around specific recommended maximums for the most regularly uses types of data in the Enate system (expand image for more detailed view).
