# Csapatnézet-jelentés

A Csapatnézet-jelentés azokat a munkákat jeleníti meg, amelyek az Ön által kezelt vagy azokon a várólistákon szerepelnek, amelyeknek Ön is része.

A Csapatnézet-jelentés a következő elérhető adatkészleteket tartalmazza:

| Táblázat                    | Mezők                                   | Leírás                                                                                                                                                     |
| --------------------------- | --------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tevékenység                 | Aktív felhasználók                      | Az egy adott kiválasztott dátumon aktív felhasználók száma                                                                                                 |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>Összes naplózott óra</p>     | Az ügynök által az összes munkatételen eltöltött teljes idő órában megadva a kiválasztott dátumtartomány során                                             |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>Összes felhasználó</p>       | Az egy adott várólistán szereplő összes munkatételért felelős felhasználók száma összesen                                                                  |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>Tevékenységtípus</p>         | Egy ügynök által a munkatételen elvégzett tevékenység típusa (pl. „Elmentett” vagy „Befejezett”)                                                           |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>Időtartam másodpercben</p>   | Egy ügynök által a munkatételen eltöltött idő másodpercben megadva. Ha rendelkezésre áll, akkor figyelembe vesszük a manuálisan megadott időt.             |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>StartDate</p>                | A munkatételhez kapcsolódó tevékenység kezdő dátuma dátum/idő formátumban megadva                                                                          |
| <p> </p><p>Tevékenység</p>  | <p> </p><p>StartDate\_date</p>          | A munkatételhez kapcsolódó tevékenység kezdő dátuma csupán dátumformátumban megadva                                                                        |
| Tevékenység                 | Felhasználónév                          | A tevékenységet elvégző felhasználó                                                                                                                        |
| Kontextus                   | Szerződés                               | A szerződés neve                                                                                                                                           |
| Kontextus                   | Ügyfél                                  | Az ügyfél neve                                                                                                                                             |
| Kontextus                   | Szolgáltatás                            | A szolgáltatás neve                                                                                                                                        |
| Kontextus                   | Szolgáltató                             | A szolgáltató neve                                                                                                                                         |
| Dátum                       | Dátum                                   | Az adatszűrés dátumtartománya                                                                                                                              |

| Dátum                       | Hét                                     | A dátumok hetei                                                                                                                                            |
| Dátum                       | Év                                      | A dátumok éve                                                                                                                                              |
| Hibák                       | Munkatételek száma                      | A hibás munkatételek száma                                                                                                                                 |
| <p> </p><p>Hibák</p>        | <p> </p><p>Észlelés dátuma</p>          | Az a dátum, amelyen egy adott munkatétel hibáját észlelték                                                                                                 |
| Hibák                       | Hibakategória                           | A hiba kategóriája                                                                                                                                         |
| Hibák                       | Leírás                                  | A hiba leírása                                                                                                                                             |
| Hibák                       | Állapot                                 | Megmutatja, hogy sikerült-e megoldani a hibát vagy sem                                                                                                     |
| Visszajelzés                | Átl. értékelés                          | Az egyes munkatételek minősítéseinek átlaga                                                                                                                |
| Visszajelzés                | Értékelés                               | Az egyes munkatételek felhasználói értékelése                                                                                                              |
| Visszajelzés                | Referencia                              | A visszajelzéssel rendelkező munkatétel referenciaszáma                                                                                                    |
| Folyamat                    | Folyamat                                | Annak a folyamatnak a neve, amelyhez minden munkatétel tartozik                                                                                            |
| Folyamat                    | Munkatétel típusa                       | A munkatétel típusa (jegy, művelet vagy ügy)                                                                                                               |
| Várólisták                  | Várólista                               | Annak a várólistának a neve, ahol a munkatétel utoljára jelen volt                                                                                         |
| Állapot                     | Állapot                                 | Munkatétel állapota (elvégzendő, folyamatban, várakozó, megoldva, lezárva)                                                                                 |

| Várakozó állapot előzményei | StartDate                               | A várakozó állapot kezdő dátuma dátum/idő formátumban megadva                                                                                              |
| Várakozó állapot előzményei | StartDate\_date                         | A várakozó állapot kezdő dátuma csak dátum formátumban megadva                                                                                             |
| Várakozó állapot előzményei | Felhasználónév                          | Az ügynök, aki a munkatétel állapotát várakozó állapotúra állította                                                                                        |

| Munkatételek                | Elindított munkatételek                 | Az ügynök által a kiválasztott dátumtartományban elindított munkatételek száma                                                                             |

| Munkatételek                | Érintett rekordok száma                 | Az érintett rekordok száma összesen                                                                                                                        |

| Munkatételek                | Hibás                                   | Megmutatja, hogy az adott munkatétel hibás vagy sem (igen vagy nem)                                                                                        |
