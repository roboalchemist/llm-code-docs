# System-Wide Settings

To access system-wide settings, select the cog at the foot of the Builder toolbar:

![](https://3859925423-files.gitbook.io/~/files/v0/b/gitbook-legacy-files/o/assets%2F-MWYnDNwe3Cuo4zlGbs5%2F-MWq25-LnWlv1WFp-ufq%2F-MWq2rZe4SkvPdPZ9sfx%2Fimage.png?alt=media\&token=ae6579d9-537c-4c78-8819-a2361f8753a8)

Here you'll find settings which have a global effect across the system. The settings are grouped into the following sections:

* [General Settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/general-settings)
* [Defect Parties](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/defect-parties)
* [File Tags](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/file-tags)
* [Office 365 Integration](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/office-365-integration)
* [Suppliers](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/suppliers)
* [Queues](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/queues)
* [Contact Tags](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/contact-tags)
* [Departments](https://app.gitbook.com/o/-MMRJw5Faehepj28yGyJ/s/-MWYnDNwe3Cuo4zlGbs5-887967055/~/changes/1706/builder/builder-2021.1/system-wide-settings/departments)
* [Locations](https://app.gitbook.com/o/-MMRJw5Faehepj28yGyJ/s/-MWYnDNwe3Cuo4zlGbs5-887967055/~/changes/1706/builder/builder-2021.1/system-wide-settings/locations)
* [Cost Centers](https://app.gitbook.com/o/-MMRJw5Faehepj28yGyJ/s/-MWYnDNwe3Cuo4zlGbs5-887967055/~/changes/1706/builder/builder-2021.1/system-wide-settings/cost-centers)
* [Password Policy](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/password-policy)
* [SSO Settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/sso-settings)
* [Approval Settings](https://docs.enate.net/enate-help/builder/builder-2021.1/system-wide-settings/approval-settings)
