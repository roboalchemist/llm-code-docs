# Welcome to LlamaCloud

LlamaCloud is a hosted service for document processing and search, powered by
LlamaIndex. It consists of five primary components:

Parse

[ Get started ](./llamaparse/getting_started) [ Examples
](./llamaparse/examples)

Transform complex documents into LLM-ready structured data with support for
130+ file formats, multimodal parsing, and advanced table/chart extraction.

Extract

[ Get started ](./llamaextract/getting_started) [ Examples
](./llamaextract/examples)

Transform documents into well-typed structured data with customizable schemas
and batch processing capabilities.

Index

[ Get started ](./llamacloud/getting_started) [ Examples
](./llamacloud/examples)

Transform document collections into searchable knowledge bases with vector
database integration and customizable RAG pipelines.

Classify

[ Get started ](./llamaclassify/getting_started) [ Examples
](./llamaclassify/examples)

Automatically categorize documents using natural-language rules. Use as a pre-
processing step for extraction, parsing, or indexing.

BETA

Split

[ Get started ](./split/getting_started/) [ Examples ](./split/examples/)

Automatically segment concatenated PDFs into logical document sections using
AI-powered classification.

BETA

Agents

[ Get started ](/python/llamaagents/getting-started/)

Build and deploy workflow-driven agentic apps with durable APIs, optional UI
development, and rapid deployment.

BETA

Sheets

[ Get started ](./llamasheets/getting_started) [ Examples
](./llamasheets/tutorials/coding_agent)

Intelligently extract tables and metadata from messy spreadsheets.
