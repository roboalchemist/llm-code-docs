<h2 class="anchor anchorWithStickyNavbar_loeA" id="1-create-a-new-agent"><span class="step">1. </span> <!-- -->Create a new agent<a href="#1-create-a-new-agent" class="hash-link" aria-label="Direct link to 1. Create a new agent" title="Direct link to 1. Create a new agent">​</a></h2>
<p>To create a new agent, sign in to your Retool organization, and select the <strong>Agents</strong> tab in the top navigation bar.</p>
<ol>
<li>From the <strong>All agents</strong> page, click the <strong>+ Agent</strong> button.</li>
<li>Select <strong>Start from scratch</strong> and click <strong>Create</strong>.</li>
<li>In the <strong>Agent name</strong> field, enter <code>Retool Digest</code>.</li>
<li>In the <strong>Description</strong> field, enter <code>Retool release notes and changelog updates</code> and click <strong>Create</strong>.</li>
</ol>

<p>There are several kinds of <a class="glossary tool" target="_blank" rel="noopener noreferrer">tools</a> that Retool Agents can consume and use. For this tutorial, you&#x27;re going to add prebuilt tools, and create custom tools.</p>

<h3 class="anchor anchorWithStickyNavbar_loeA" id="add-core-tools">Add core tools<a href="#add-core-tools" class="hash-link" aria-label="Direct link to Add core tools" title="Direct link to Add core tools">​</a></h3>
<p><a class="glossary core-tool" target="_blank" rel="noopener noreferrer">Core tools</a> are created by Retool to provide you out-of-the-box tool functionality for common tasks, like updating a Google Calendar or Executing Code.</p>
<p>To add core tools:</p>
<ol>
<li>Click <strong>Add Tool</strong> and click the <strong>+</strong> on the <strong>Send Email</strong> <a class="glossary tool" target="_blank" rel="noopener noreferrer">tool</a> and the <strong>Get Webpage Content</strong> tool. A green check mark will appear to notify you that the tools have been selected.</li>
<li>Click the <strong>Add 2 tools</strong> button. The number changes depending on how many tools are selected.</li>
<li>The <strong>Send Email</strong> and <strong>Get Webpage Content</strong> tools are displayed in the <strong>Tools</strong> section of the <strong>Configuration</strong> tab.</li>
</ol>
