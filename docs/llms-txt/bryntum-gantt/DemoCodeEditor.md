# [DemoCodeEditor](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor)

A CodeEditor subclass, which is used for editing or viewing demo application code.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[mode](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#config-mode)
Editor mode

[appFolder](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#config-appFolder)
App folder location

[preferredSources](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#config-preferredSources)
Preferred source files. First matched file from the list will be shown on initial load.

[editor](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#config-editor)
An editor configuration object to be passed to https://microsoft.github.io/monaco-editor/docs.html#functions/editor.create.html

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isDemoCodeEditor](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#property-isDemoCodeEditor)
Identifies an object as an instance of [DemoCodeEditor](https://bryntum.com/docs/gantt/api/#Core/widget/DemoCodeEditor) class, or subclass thereof.

[isDemoCodeEditor](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#property-isDemoCodeEditor-static)
Identifies an object as an instance of [DemoCodeEditor](https://bryntum.com/docs/gantt/api/#Core/widget/DemoCodeEditor) class, or subclass thereof.

[monacoCodePath](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#property-monacoCodePath-static)
Returns default monaco code path

## Functions

Functions are methods available for calling on the class

[toggleCodeEditor](https://bryntum.com/docs/gantt/api/Core/widget/DemoCodeEditor#function-toggleCodeEditor-static)
Shows / hides code editor
