# [GanttState](https://bryntum.com/docs/gantt/api/Gantt/view/mixin/GanttState)

Mixin for Gantt that handles state. It serializes the following gantt properties, in addition to what is already stored by its superclass [TimelineState](https://bryntum.com/docs/gantt/api/#Scheduler/view/mixin/TimelineState):

* barMargin
* tickSize
* zoomLevel

See [GridState](https://bryntum.com/docs/gantt/api/#Grid/view/mixin/GridState) and [State](https://bryntum.com/docs/gantt/api/#Core/mixin/State) for more information on state.
