# [ResourceTimeRangesBase](https://bryntum.com/docs/gantt/api/Scheduler/feature/base/ResourceTimeRangesBase)

Abstract base class for ResourceTimeRanges and ResourceNonWorkingTime features. You should not use this class directly.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[tabIndex](https://bryntum.com/docs/gantt/api/Scheduler/feature/base/ResourceTimeRangesBase#config-tabIndex)
Specify value to use for the tabIndex attribute of range elements
