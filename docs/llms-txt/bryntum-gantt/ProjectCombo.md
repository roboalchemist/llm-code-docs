# [ProjectCombo](https://bryntum.com/docs/gantt/api/Scheduler/widget/ProjectCombo)

Combo that allows picking a dataset to use for a [ProjectModel](https://bryntum.com/docs/gantt/api/#Scheduler/model/ProjectModel). Each item holds a title and a load url to reconfigure the project with.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[project](https://bryntum.com/docs/gantt/api/Scheduler/widget/ProjectCombo#config-project)
Project to reconfigure when picking an item.

[displayField](https://bryntum.com/docs/gantt/api/Scheduler/widget/ProjectCombo#config-displayField)
Field used as projects title.

[valueField](https://bryntum.com/docs/gantt/api/Scheduler/widget/ProjectCombo#config-valueField)
Field used as projects load url.
