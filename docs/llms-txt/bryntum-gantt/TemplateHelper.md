# [TemplateHelper](https://bryntum.com/docs/gantt/api/Core/helper/TemplateHelper)

Helper for template strings, use as tag function

```
TemplateHelper.tpl`...`
```

## Functions

Functions are methods available for calling on the class

[tpl](https://bryntum.com/docs/gantt/api/Core/helper/TemplateHelper#function-tpl-static)
Tag function for template literals that does some basic cleanup

[docsTpl](https://bryntum.com/docs/gantt/api/Core/helper/TemplateHelper#function-docsTpl-static)
Tag function for template literals that does some basic cleanup. Version for docs that do not remove blank space, to keep code snippets formatting intact.
