# [NumberHelper](https://bryntum.com/docs/gantt/api/Core/helper/NumberHelper)

## Functions

Functions are methods available for calling on the class

Returns a [NumberFormat](https://bryntum.com/docs/gantt/api/#Core/helper/util/NumberFormat) instance built using the current [locale](https://bryntum.com/docs/gantt/api/#Core/localization/LocaleManager#property-locale).
