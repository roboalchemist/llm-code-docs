# [WBSColumn](https://bryntum.com/docs/gantt/api/Gantt/column/WBSColumn)

A calculated column which displays the _WBS_ (_Work Breakdown Structure_) for the tasks - the position of the task in the project tree structure.

While there is no `editor`, since the WBS is a calculated value, there is a `renumber` item in the `headerMenuItems` that allows the user to [refresh](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#function-refreshWbs) the WBS values.
