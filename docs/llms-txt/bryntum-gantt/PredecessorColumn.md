# [PredecessorColumn](https://bryntum.com/docs/gantt/api/Gantt/column/PredecessorColumn)

A column which displays, in textual form, the dependencies which link from tasks upon which the contextual task depends.

This type of column is editable by default. Default editor is a [DependencyField](https://bryntum.com/docs/gantt/api/#Gantt/widget/DependencyField).

This column will be ignored if using [CellCopyPaste](https://bryntum.com/docs/gantt/api/#Grid/feature/CellCopyPaste) to paste or [FillHandle](https://bryntum.com/docs/gantt/api/#Grid/feature/FillHandle) to fill values.
