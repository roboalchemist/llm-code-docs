# [DayAvailabilityStore](https://bryntum.com/docs/gantt/api/SchedulerPro/data/calendareditor/DayAvailabilityStore)

A store of week availability ranges provided on daily basis. Each store record represents a single day availability.

The class is used by [CalendarEditorWeekModel](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/calendareditor/CalendarEditorWeekModel) class which is a part of the [calendar editor](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarEditor) data level.

The store uses [DayAvailabilityModel](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/calendareditor/DayAvailabilityModel) model.
