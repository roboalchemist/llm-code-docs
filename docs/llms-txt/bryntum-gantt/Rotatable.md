# [Rotatable](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Rotatable)

A mixin that provides support for rotating a widget's primary element.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[rotate](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Rotatable#config-rotate)
Specify `'left'` to rotate the button content 90 degrees counter-clockwise or `'right'` for clockwise.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isRotatable](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Rotatable#property-isRotatable)
Identifies an object as an instance of [Rotatable](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Rotatable) class, or subclass thereof.

[isRotatable](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Rotatable#property-isRotatable-static)
Identifies an object as an instance of [Rotatable](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Rotatable) class, or subclass thereof.

[rotate](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Rotatable#property-rotate)
Set to `'left'` to rotate the button content 90 degrees counter-clockwise or `'right'` for clockwise.
