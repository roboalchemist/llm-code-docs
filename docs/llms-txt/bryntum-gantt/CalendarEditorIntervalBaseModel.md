# [CalendarEditorIntervalBaseModel](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel)

Base model for the calendar editor store.

## Fields

Fields belong to a Model class and define the Model data structure

[type](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel#field-type)
The interval type. It's either a **week** (specified default daily availability) or an **exception** (an availability override for a specific period of time, for example a holiday or a vacation).

[name](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel#field-name)
The interval name.

[calendar](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel#field-calendar)
The calendar this interval belongs to.

[startDate](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel#field-startDate)
The interval start date (if the intervals overrides a specific date range).

[endDate](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/CalendarEditorIntervalBaseModel#field-endDate)
The interval end date (if the intervals overrides a specific date range).
