# [RemoveChildAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveChildAction)

Action to record store remove child operation.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[parentModel](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveChildAction#config-parentModel)
Reference to a parent model a child model has been removed to.

[childModels](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveChildAction#config-childModels)
Children models removed.

[context](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveChildAction#config-context)
Map having children models as keys and values containing previous parentIndex.
