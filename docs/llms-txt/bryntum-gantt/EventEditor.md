# [EventEditor](https://bryntum.com/docs/gantt/api/Scheduler/view/EventEditor)

Base abstract class used by the [editorConfig](https://bryntum.com/docs/gantt/api/#Scheduler/feature/EventEdit#config-editorConfig) feature config for the editor dialog.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[titleRenderer](https://bryntum.com/docs/gantt/api/Scheduler/view/EventEditor#config-titleRenderer)
A Function (or _name_ of a function) which produces a customized Panel header based upon the event being edited.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isEventEditor](https://bryntum.com/docs/gantt/api/Scheduler/view/EventEditor#property-isEventEditor)
Identifies an object as an instance of [EventEditor](https://bryntum.com/docs/gantt/api/#Scheduler/view/EventEditor) class, or subclass thereof.

[isEventEditor](https://bryntum.com/docs/gantt/api/Scheduler/view/EventEditor#property-isEventEditor-static)
