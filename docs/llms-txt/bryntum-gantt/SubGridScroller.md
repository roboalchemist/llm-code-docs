# [SubGridScroller](https://bryntum.com/docs/gantt/api/Grid/util/SubGridScroller)

A Scroller subclass which handles scrolling in a SubGrid. Needs special treatment since the SubGrid itself only allows horizontal scrolling, while the vertical scrolling is done by an outer element containing all subgrids.
