# [ResourceCombo](https://bryntum.com/docs/gantt/api/Scheduler/widget/ResourceCombo)

A Combo subclass which selects resources, optionally displaying the [eventColor](https://bryntum.com/docs/gantt/api/#Scheduler/model/ResourceModel#field-eventColor) of each resource in the picker and in the input area.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[showEventColor](https://bryntum.com/docs/gantt/api/Scheduler/widget/ResourceCombo#config-showEventColor)
Show the [event color](https://bryntum.com/docs/gantt/api/#Scheduler/model/ResourceModel#field-eventColor) for each resource
