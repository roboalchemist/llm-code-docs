# [AIChatMessageModel](https://bryntum.com/docs/gantt/api/Core/data/AIChatMessageModel)

A ChatMessageModel defines a message in the [AIChatPanel](https://bryntum.com/docs/gantt/api/#Core/widget/chat/AIChatPanel) widget.

## Fields

Fields belong to a Model class and define the Model data structure

[confidence](https://bryntum.com/docs/gantt/api/Core/data/AIChatMessageModel#field-confidence)
A number between 0 and 1 indicating the confidence score when a remote message was provided by AI
