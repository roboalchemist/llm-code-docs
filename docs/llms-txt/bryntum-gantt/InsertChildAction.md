# [InsertChildAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction)

Action to record the fact of adding children models into a parent model.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[parentModel](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction#config-parentModel)
Reference to a parent model a child model has been added to.

[childModels](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction#config-childModels)
Children models inserted.

[insertIndex](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction#config-insertIndex)
Index a children models are inserted at

[orderedBeforeNode](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction#config-orderedBeforeNode)
Reference node in the ordered tree

[context](https://bryntum.com/docs/gantt/api/Core/data/stm/action/InsertChildAction#config-context)
Map having children models as keys and values containing previous parent of each model and index at the previous parent.
