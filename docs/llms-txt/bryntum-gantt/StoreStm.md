# [StoreStm](https://bryntum.com/docs/gantt/api/Core/data/stm/mixin/StoreStm)

Store mixin to make it compatible with [StateTrackingManager](https://bryntum.com/docs/gantt/api/#Core/data/stm/StateTrackingManager).

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[stm](https://bryntum.com/docs/gantt/api/Core/data/stm/mixin/StoreStm#config-stm)
Reference to STM manager

[ignoreRemoteChangesInSTM](https://bryntum.com/docs/gantt/api/Core/data/stm/mixin/StoreStm#config-ignoreRemoteChangesInSTM)
Set to `false` to not record transaction during `applyChangeset` call
