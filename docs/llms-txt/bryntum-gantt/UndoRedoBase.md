# [UndoRedoBase](https://bryntum.com/docs/gantt/api/Core/widget/base/UndoRedoBase)

Abstract base class used by UndoRedo widgets in Scheduler and TaskBoard.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[text](https://bryntum.com/docs/gantt/api/Core/widget/base/UndoRedoBase#config-text)
Configure as `true` to show "Undo" and "Redo" as button texts. The buttons always have a tooltip as a hint to the user as to their purpose.

[color](https://bryntum.com/docs/gantt/api/Core/widget/base/UndoRedoBase#config-color)
Button color for the undo and redo buttons. See [color](https://bryntum.com/docs/gantt/api/#Core/widget/Button#config-color).

[showZeroActionBadge](https://bryntum.com/docs/gantt/api/Core/widget/base/UndoRedoBase#config-showZeroActionBadge)
Configure as `true` to show "0" badge on the undo and redo buttons when they have no actions left to perform. By default when there are no actions, no badge is displayed.
