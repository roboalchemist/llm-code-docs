# [Fullscreen](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen)

Encapsulates the functionality related to switching cross-browser to full screen view and back.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[enabled](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#property-enabled-static)
True if the fullscreen mode is supported and enabled, false otherwise

[isFullscreen](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#property-isFullscreen-static)
True if fullscreen mode is currently active, false otherwise

## Functions

Functions are methods available for calling on the class

[request](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#function-request-static)
Request entering the fullscreen mode.

[exit](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#function-exit-static)
Exit the previously entered fullscreen mode.

[onFullscreenChange](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#function-onFullscreenChange-static)
Installs the passed listener to fullscreenchange event

[unFullscreenChange](https://bryntum.com/docs/gantt/api/Core/helper/util/Fullscreen#function-unFullscreenChange-static)
Uninstalls the passed listener from fullscreenchange event
