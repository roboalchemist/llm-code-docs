# [CurrentConfig](https://bryntum.com/docs/gantt/api/Scheduler/view/mixin/CurrentConfig)

Mixin that makes sure inline data & crud manager data are removed from current config for products using a project. The data is instead inlined in the project (by ProjectModel.js)
