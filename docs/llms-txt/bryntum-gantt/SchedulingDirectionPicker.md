# [SchedulingDirectionPicker](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/SchedulingDirectionPicker)

Combo box preconfigured with possible [scheduling direction](https://bryntum.com/docs/gantt/api/https://bryntum.com/products/gantt/docs/api/Gantt/model/TaskModel#field-direction) values. This picker doesn't support [multiSelect](https://bryntum.com/docs/gantt/api/#Core/widget/Combo#config-multiSelect).

This field can be used as an editor for a [Column](https://bryntum.com/docs/gantt/api/#Grid/column/Column).
