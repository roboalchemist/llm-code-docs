# [CalendarEditorDatePicker](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/calendareditor/CalendarEditorDatePicker)

A special date picker visualizing a calendar effect on displayed dates. The widget is used by [the calendar editor](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarEditor).

It basically highlights dates affected by some intervals by adding color badges representing the intervals.
