# [DesignableChartConfig](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig)

Subset of [Chart](https://bryntum.com/docs/gantt/api/#Chart/widget/Chart) configuration options also supported by [ChartDesigner](https://bryntum.com/docs/gantt/api/#Chart/widget/ChartDesigner).

Not to be used directly.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[showTitle](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-showTitle)
Whether to show the chart title.

[showSubtitle](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-showSubtitle)
Whether to show the chart subtitle.

[showLegend](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-showLegend)
Whether to show the chart's legend.

[chartPadding](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-chartPadding)
Padding around the entire chart area, in pixels.

[titlePadding](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-titlePadding)
Padding around the chart title, in pixels.

[subtitlePadding](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-subtitlePadding)
Padding around the chart subtitle, in pixels.

[background](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-background)
The chart background color, or `null` for transparent.

[axisColor](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-axisColor)
The stroke color used for the axis lines and ticks, or `null` for theme default.

[axisLabelColor](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-axisLabelColor)
The text color used for the axis labels / ticks, or `null` to use `axisColor`.

[gridColor](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-gridColor)
The stroke color used for the grid lines, or `null` for theme default.

[maxTickLabelRotation](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-maxTickLabelRotation)
The maximum tick rotation in degrees.

[minTickLabelRotation](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-minTickLabelRotation)
The minimum tick rotation in degrees.

[dataPointShape](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-dataPointShape)
The shape to use for data point markers, or `'none'` for no markers.

[titleFont](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-titleFont)
The font to use for the chart title.

Default:

```
{
   family    : 'sans-serif',
   size      : 24,
   bold      : true,
   italic    : false,
   color     : '#3A4750',
   darkColor : '#ddd',
   underline : false
}
```

[subtitleFont](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-subtitleFont)
The font to use for the chart subtitle.

Default:

```
{
   family    : 'sans-serif',
   size      : 20,
   bold      : true,
   italic    : false,
   color     : '#3A4750',
   darkColor : '#ddd',
   underline : false
}
```

[seriesLineThickness](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-seriesLineThickness)
The thickness of data series lines, in pixels.

[seriesLineDash](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-seriesLineDash)
The type of stroke to use for data series lines.

[seriesLineOpacity](https://bryntum.com/docs/gantt/api/Chart/widget/mixin/DesignableChartConfig#config-seriesLineOpacity)
The opacity of data series lines, from zero to one.
