# [ChatButton](https://bryntum.com/docs/gantt/api/Core/widget/chat/ChatButton)

A button that toggles the visibility of a linked [ChatPanel](https://bryntum.com/docs/gantt/api/#Core/widget/chat/ChatPanel).

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[chatPanel](https://bryntum.com/docs/gantt/api/Core/widget/chat/ChatButton#config-chatPanel)
The linked ChatPanel instance whose visibility will be toggled by this button

## Properties

Properties are getters/setters or publicly accessible variables on this class

[chatPanel](https://bryntum.com/docs/gantt/api/Core/widget/chat/ChatButton#property-chatPanel)
The linked ChatPanel instance whose visibility will be toggled by this button

[unscaledRect](https://bryntum.com/docs/gantt/api/Core/widget/chat/ChatButton#property-unscaledRect)
Returns a Rectangle representing this button's bounds ignoring any CSS scale transform. This ensures consistent popup alignment regardless of hover state.
