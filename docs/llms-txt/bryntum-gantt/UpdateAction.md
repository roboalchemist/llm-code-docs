# [UpdateAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/UpdateAction)

Action to record the fact that a model has been updated.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[model](https://bryntum.com/docs/gantt/api/Core/data/stm/action/UpdateAction#config-model)
Reference to a model which has been updated.

[newData](https://bryntum.com/docs/gantt/api/Core/data/stm/action/UpdateAction#config-newData)
Map of updated properties with new values.

[oldData](https://bryntum.com/docs/gantt/api/Core/data/stm/action/UpdateAction#config-oldData)
Map of updated properties with old values.
