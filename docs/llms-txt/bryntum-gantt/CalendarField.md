# [CalendarField](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/CalendarField)

A combo used to select the calendar for an event. This field can be seen in the [AdvancedTab](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/taskeditor/AdvancedTab)

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[calendarConsumerRecord](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/CalendarField#config-calendarConsumerRecord)
The consumer of the edited calendar. Can be a resource, an event or a project model. The value is required for ["Edit calendar" trigger](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarField#config-calendarEditor) functionality.

[calendarEditor](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/CalendarField#config-calendarEditor)
Configuration of the calendar editor triggered by clicking "Edit calendar" trigger. Provide `false` to this config to skip both the editor and the trigger constructing.

[store](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/CalendarField#config-store)
The store containing the calendars
