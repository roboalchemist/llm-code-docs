# [CalendarModel](https://bryntum.com/docs/gantt/api/Gantt/model/CalendarModel)

This class represents a calendar in the Gantt project. It contains a collection of the [CalendarIntervalModel](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/CalendarIntervalModel). Every interval can be either recurrent (regularly repeating in time) or static.

Please refer to the [calendars guide](https://bryntum.com/docs/gantt/api/#Gantt/guides/basics/calendars.md) for details
