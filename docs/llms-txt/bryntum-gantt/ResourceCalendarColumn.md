# [ResourceCalendarColumn](https://bryntum.com/docs/gantt/api/SchedulerPro/column/ResourceCalendarColumn)

A column that displays (and allows user to update) the current [calendar](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/CalendarModel) of the resource.

Default editor is a [CalendarField](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarField).
