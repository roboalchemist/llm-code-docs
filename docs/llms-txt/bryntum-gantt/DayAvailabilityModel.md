# [DayAvailabilityModel](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/DayAvailabilityModel)

Represents a particular day availability and implements a record of [DayAvailabilityStore](https://bryntum.com/docs/gantt/api/#SchedulerPro/data/calendareditor/DayAvailabilityStore) store (a part of the [calendar editor](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarEditor) data level).

## Fields

Fields belong to a Model class and define the Model data structure

[id](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/DayAvailabilityModel#field-id)
Day identifier. A zero-based index of the day (`0` - Sunday, `1` - Moday, `2` - Tuesday, `3` - Wedndesday, `4` -Thursday, `5` - Friday and `6` - Saturday).

[availability](https://bryntum.com/docs/gantt/api/SchedulerPro/model/calendareditor/DayAvailabilityModel#field-availability)
Day availability.
