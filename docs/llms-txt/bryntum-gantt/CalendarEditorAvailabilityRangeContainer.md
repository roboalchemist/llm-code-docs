# [CalendarEditorAvailabilityRangeContainer](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/calendareditor/CalendarEditorAvailabilityRangeContainer)

[store](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/calendareditor/CalendarEditorAvailabilityRangeContainer#config-store)
