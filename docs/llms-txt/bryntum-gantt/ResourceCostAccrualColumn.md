# [ResourceCostAccrualColumn](https://bryntum.com/docs/gantt/api/SchedulerPro/column/ResourceCostAccrualColumn)

A column that displays (and allows user to update) a resource [cost accrual](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/ResourceModel#field-costAccrual).
