# [MonthPicker](https://bryntum.com/docs/gantt/api/Core/widget/MonthPicker)

A Panel subclass which allows the selection of a month.

The [month](https://bryntum.com/docs/gantt/api/#Core/widget/MonthPicker#property-month) indicates and sets the currently selected month.

The [select](https://bryntum.com/docs/gantt/api/#Core/widget/MonthPicker#event-select) event is fired with a 0-indexed value when a different month is selected.

The [titleClick](https://bryntum.com/docs/gantt/api/#Core/widget/MonthPicker#event-titleClick) event is fired when the panel title is clicked.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[month](https://bryntum.com/docs/gantt/api/Core/widget/MonthPicker#config-month)
The month to use as the selected month (zero-based, 0-11). Defaults to the current month.
