# [WriteExcelFileProvider](https://bryntum.com/docs/gantt/api/Grid/feature/experimental/xlsproviders/WriteExcelFileProvider)

Provider class utilizing write-excel-file library to create xls file

[write](https://bryntum.com/docs/gantt/api/Grid/feature/experimental/xlsproviders/WriteExcelFileProvider#function-write-static)
