# [Badge](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Badge)

Mixin that allows a widget to display a badge (mostly done as css)

```
// show badge
button.badge = 5;

// hide badge
button.badge = null;
```

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[badge](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Badge#config-badge)
Initial text to show in badge.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isBadge](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Badge#property-isBadge)
Identifies an object as an instance of [Badge](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Badge) class, or subclass thereof.

[isBadge](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Badge#property-isBadge-static)
Identifies an object as an instance of [Badge](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Badge) class, or subclass thereof.

[badge](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Badge#property-badge)
Get/sets and display badge, set to null or empty string to hide.
