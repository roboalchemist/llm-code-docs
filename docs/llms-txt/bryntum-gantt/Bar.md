# [Bar](https://bryntum.com/docs/gantt/api/Grid/view/Bar)

Base class used by Header and Footer. Holds an element for each column. Not intended to be used directly.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isBar](https://bryntum.com/docs/gantt/api/Grid/view/Bar#property-isBar)
Identifies an object as an instance of [Bar](https://bryntum.com/docs/gantt/api/#Grid/view/Bar) class, or subclass thereof.

[isBar](https://bryntum.com/docs/gantt/api/Grid/view/Bar#property-isBar-static)
Identifies an object as an instance of [Bar](https://bryntum.com/docs/gantt/api/#Grid/view/Bar) class, or subclass thereof.

## Functions

Functions are methods available for calling on the class

[fixCellWidths](https://bryntum.com/docs/gantt/api/Grid/view/Bar#function-fixCellWidths)
Fix cell widths (flex or fixed width) after rendering. Not a part of template any longer because of CSP

[getBarCellElement](https://bryntum.com/docs/gantt/api/Grid/view/Bar#function-getBarCellElement)
Get the header or footer cell element for the specified column.
