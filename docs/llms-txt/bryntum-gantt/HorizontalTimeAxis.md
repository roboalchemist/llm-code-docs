# [HorizontalTimeAxis](https://bryntum.com/docs/gantt/api/Scheduler/view/HorizontalTimeAxis)

A visual horizontal representation of the time axis described in the [headers](https://bryntum.com/docs/gantt/api/#Scheduler/preset/ViewPreset#field-headers) field. Normally you should not interact with this class directly.
