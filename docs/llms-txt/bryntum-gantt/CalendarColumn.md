# [CalendarColumn](https://bryntum.com/docs/gantt/api/Gantt/column/CalendarColumn)

A column that displays (and allows user to update) the current [calendar](https://bryntum.com/docs/gantt/api/#Gantt/model/CalendarModel) of the task.

Default editor is a [CalendarPicker](https://bryntum.com/docs/gantt/api/#Gantt/widget/CalendarPicker).
