# [AIFilterField](https://bryntum.com/docs/gantt/api/Grid/widget/AIFilterField)

A text field specialized for use as the input field for the [AIFilter](https://bryntum.com/docs/gantt/api/#Grid/feature/ai/AIFilter) feature.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[progressText](https://bryntum.com/docs/gantt/api/Grid/widget/AIFilterField#config-progressText)
The text to show in the load mask when the AI is processing the filter request

[client](https://bryntum.com/docs/gantt/api/Grid/widget/AIFilterField#config-client)
The Grid instance this field is filtering. Defaults to the closest Grid instance found when the field is added to the DOM. If you are not adding the field to the Grid's DOM, you must set this property.
