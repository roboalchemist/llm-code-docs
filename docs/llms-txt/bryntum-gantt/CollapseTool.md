# [CollapseTool](https://bryntum.com/docs/gantt/api/Core/widget/panel/CollapseTool)

This ensures the correct icon is used to represent the [panel's](https://bryntum.com/docs/gantt/api/#Core/widget/Panel) [collapsed](https://bryntum.com/docs/gantt/api/#Core/widget/Panel#config-collapsed) state.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[autoAlign](https://bryntum.com/docs/gantt/api/Core/widget/panel/CollapseTool#config-autoAlign)
Set to `false` to disable automatic adjustment of the [align](https://bryntum.com/docs/gantt/api/#Core/widget/panel/CollapseTool#config-align) config based on the state of the panel's [collapsed](https://bryntum.com/docs/gantt/api/#Core/widget/Panel#config-collapsed) config and the [direction](https://bryntum.com/docs/gantt/api/#Core/widget/panel/PanelCollapser#config-direction).

If this is set to a function, that function is called passing the owning `Panel` instance and its return value is assigned to the [align](https://bryntum.com/docs/gantt/api/#Core/widget/panel/CollapseTool#config-align) config.
