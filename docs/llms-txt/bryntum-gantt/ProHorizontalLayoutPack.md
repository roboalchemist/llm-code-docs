# [ProHorizontalLayoutPack](https://bryntum.com/docs/gantt/api/SchedulerPro/eventlayout/ProHorizontalLayoutPack)

Handles layout of events within a row (resource) in horizontal mode. Packs events (adjusts their height) to fit available row height.

This layout supports grouping events inside the resource row. See [ProHorizontalLayout](https://bryntum.com/docs/gantt/api/#SchedulerPro/eventlayout/ProHorizontalLayout) for more info.
