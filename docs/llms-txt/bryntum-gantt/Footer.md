# [Footer](https://bryntum.com/docs/gantt/api/Grid/view/Footer)

Grid footer, used by Summary feature. You should not need to create instances manually.

## Properties

Properties are getters/setters or publicly accessible variables on this class

[isFooter](https://bryntum.com/docs/gantt/api/Grid/view/Footer#property-isFooter)
Identifies an object as an instance of [Footer](https://bryntum.com/docs/gantt/api/#Grid/view/Footer) class, or subclass thereof.

[isFooter](https://bryntum.com/docs/gantt/api/Grid/view/Footer#property-isFooter-static)
Identifies an object as an instance of [Footer](https://bryntum.com/docs/gantt/api/#Grid/view/Footer) class, or subclass thereof.

## Functions

Functions are methods available for calling on the class

[getFooter](https://bryntum.com/docs/gantt/api/Grid/view/Footer#function-getFooter)
Get the footer cell element for the specified column.

[contentTemplate](https://bryntum.com/docs/gantt/api/Grid/view/Footer#function-contentTemplate)
Footer template. Iterates leaf columns to create content. Style not included because of CSP. Widths are fixed up in [fixFooterWidths](https://bryntum.com/docs/gantt/api/#Grid/view/Footer#function-fixFooterWidths)

[fixFooterWidths](https://bryntum.com/docs/gantt/api/Grid/view/Footer#function-fixFooterWidths)
Fix footer widths (flex or fixed width) after rendering. Not a part of template any longer because of CSP
