# [ProjectLines](https://bryntum.com/docs/gantt/api/Gantt/feature/ProjectLines)

This feature draws the vertical lines in the schedule area, indicating project start/end dates and also its [status date](https://bryntum.com/docs/gantt/api/#Gantt/model/ProjectModel#field-statusDate). The latter can be disabled with the [showStatusDate](https://bryntum.com/docs/gantt/api/#Gantt/feature/ProjectLines#config-showStatusDate) config.

This feature is **enabled** by default.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[showStatusDate](https://bryntum.com/docs/gantt/api/Gantt/feature/ProjectLines#config-showStatusDate)
A boolean specifying whether to show the project's [statusDate](https://bryntum.com/docs/gantt/api/#Gantt/model/ProjectModel#field-statusDate)
