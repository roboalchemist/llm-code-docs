# [NoteColumn](https://bryntum.com/docs/gantt/api/Gantt/column/NoteColumn)

A column which displays a task's [note](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-note) field.

Default editor is a [TextAreaPickerField](https://bryntum.com/docs/gantt/api/#Core/widget/TextAreaPickerField).
