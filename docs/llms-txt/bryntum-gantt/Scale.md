# [Scale](https://bryntum.com/docs/gantt/api/Core/widget/graph/Scale)

Displays a scale with ticks and labels.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[horizontal](https://bryntum.com/docs/gantt/api/Core/widget/graph/Scale#config-horizontal)
Configure as `true` to create a horizontal scale. Scales are vertical by default.

[align](https://bryntum.com/docs/gantt/api/Core/widget/graph/Scale#config-align)
Side to align the scale to. Defaults to `bottom` for [horizontal](https://bryntum.com/docs/gantt/api/#Core/widget/graph/Scale#config-horizontal) Scales and `right` for vertical Scales.
