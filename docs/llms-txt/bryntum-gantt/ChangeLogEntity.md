# [ChangeLogEntity](https://bryntum.com/docs/gantt/api/SchedulerPro/model/changelog/ChangeLogEntity)

An immutable, serializable object that describes an entity instance (for example, a single task). ChangeLogEntity instances appear in change log entries to indicate which entity was affected by a [ChangeLogAction](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/changelog/ChangeLogAction).

## Properties

Properties are getters/setters or publicly accessible variables on this class

[id](https://bryntum.com/docs/gantt/api/SchedulerPro/model/changelog/ChangeLogEntity#property-id)
The unique id of the entity instance, e.g. 'TaskModel-1'.

[type](https://bryntum.com/docs/gantt/api/SchedulerPro/model/changelog/ChangeLogEntity#property-type)
The name of the Model type of the entity, e.g. 'TaskModel'.

[name](https://bryntum.com/docs/gantt/api/SchedulerPro/model/changelog/ChangeLogEntity#property-name)
A user-friendly name for the entity instance, e.g. 'My important task'.
