Child of TestDocsBase. This class is required to check how docs are generated.

Configs are options you supply in a configuration object when creating an instance of this class

[_testGrandChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/Child#config-_testGrandChildDescription)
Child node. This description is NOT used in the docs of GrandChild because description is specified in GrandChild.

[_testChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/Child#config-_testChildDescription)

[_testMixinChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/Child#config-_testMixinChildDescription)
Child node. This description is NOT used in the docs of GrandChild because description is specified in ChildMixin and there is no description in GrandChild.
