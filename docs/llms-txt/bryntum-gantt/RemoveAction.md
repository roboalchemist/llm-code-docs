# [RemoveAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAction)

Action to record the fact of models removed from a store.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[store](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAction#config-store)
Reference to a store that models have been removed from.

[modelList](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAction#config-modelList)
List of models removed from the store.

[context](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAction#config-context)
Models removing context.

[silent](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAction#config-silent)
Flag showing if undo/redo should be done silently i.e. with events suppressed
