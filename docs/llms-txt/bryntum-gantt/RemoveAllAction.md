# [RemoveAllAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAllAction)

Action to record a store "remove all" operation.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[store](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAllAction#config-store)
Reference to cleared store

[allRecords](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAllAction#config-allRecords)
All store records removed

[silent](https://bryntum.com/docs/gantt/api/Core/data/stm/action/RemoveAllAction#config-silent)
Flag showing if undo/redo should be done silently i.e. with events suppressed
