# [CalendarIntervalModel](https://bryntum.com/docs/gantt/api/Gantt/model/CalendarIntervalModel)

This class represents a calendar interval in the Gantt calendar. Every interval can be either recurrent (regularly repeating in time) or static.

Please refer to the [calendars guide](https://bryntum.com/docs/gantt/api/#Gantt/guides/basics/calendars.md) for details
