# [DragHelperContainer](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer)

Mixin for DragHelper that handles dragging elements between containers (or rearranging within)

## Functions

Functions are methods available for calling on the class

[initContainerDrag](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-initContainerDrag)
Initialize container drag mode.

[grabContainerDrag](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-grabContainerDrag)
Grab an element which can be dragged between containers.

[updateContainerDrag](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-updateContainerDrag)
Move the placeholder element into its new position on valid drag.

[finishContainerDrag](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-finishContainerDrag)
Finalize drag, fire drop.

[abortContainerDrag](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-abortContainerDrag)
Aborts a drag operation.

[updateContainerProxy](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-updateContainerProxy)
Updates the drag proxy position.

[moveNextTo](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-moveNextTo)
Positions element being dragged in relation to targetElement.

[revertPosition](https://bryntum.com/docs/gantt/api/Core/helper/mixin/DragHelperContainer#function-revertPosition)
Moves element being dragged back to its original position.
