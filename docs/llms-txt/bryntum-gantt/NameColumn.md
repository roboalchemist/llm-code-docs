# [NameColumn](https://bryntum.com/docs/gantt/api/Gantt/column/NameColumn)

A tree column showing (and allowing user to edit) the task's [name](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-name) field.

Default editor is a [TextField](https://bryntum.com/docs/gantt/api/#Core/widget/TextField).

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[showWbs](https://bryntum.com/docs/gantt/api/Gantt/column/NameColumn#config-showWbs)
Set to `true` to render the WBS code before the task name
