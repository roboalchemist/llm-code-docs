# [RecurrenceTab](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/taskeditor/RecurrenceTab)

Tab for editing an events recurrence rule, only shown when Scheduler Pro is configured to use recurring events (see [enableRecurringEvents](https://bryntum.com/docs/gantt/api/#Scheduler/view/mixin/RecurringEvents#config-enableRecurringEvents)).

Does not currently offer any customization options.
