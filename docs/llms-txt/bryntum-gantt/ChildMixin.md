# [ChildMixin](https://bryntum.com/docs/gantt/api/Core/test/docs/ChildMixin)

Mixin for children of TestDocsBase. This class is required to check how docs are generated.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[_testMixinGrandChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/ChildMixin#config-_testMixinGrandChildDescription)
ChildMixin node. This description is NOT used in the docs of GrandChild because description is specified in GrandChild.

[_testMixinChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/ChildMixin#config-_testMixinChildDescription)
ChildMixin node. This description is USED in the docs of GrandChild because there is no description in GrandChild and mixin is more important than relatives.

[_testMixinBaseDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/ChildMixin#config-_testMixinBaseDescription)
ChildMixin node. This description is USED in the docs of GrandChild because there is no description in GrandChild and mixin is more important than relatives.
