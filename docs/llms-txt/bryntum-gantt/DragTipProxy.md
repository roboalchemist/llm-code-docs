# [DragTipProxy](https://bryntum.com/docs/gantt/api/Core/util/drag/DragTipProxy)

This drag proxy manages a [tooltip](https://bryntum.com/docs/gantt/api/#Core/util/drag/DragTipProxy#config-tooltip) (or derived class) and aligns the tooltip to the current drag position adjusted by the [align](https://bryntum.com/docs/gantt/api/#Core/util/drag/DragTipProxy#config-align) config.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[align](https://bryntum.com/docs/gantt/api/Core/util/drag/DragTipProxy#config-align)
Controls how the tooltip will be aligned to the current drag position.

See [alignTo](https://bryntum.com/docs/gantt/api/#Core/helper/util/Rectangle#function-alignTo) for details.

[offset](https://bryntum.com/docs/gantt/api/Core/util/drag/DragTipProxy#config-offset)
The number of pixels to offset from the drag position.

[tooltip](https://bryntum.com/docs/gantt/api/Core/util/drag/DragTipProxy#config-tooltip)
The tooltip to be shown, hidden and repositioned to track the drag position.
