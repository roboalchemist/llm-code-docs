# [RecurrenceEditorPanel](https://bryntum.com/docs/gantt/api/Scheduler/view/recurrence/RecurrenceEditorPanel)

Panel containing fields used to edit a [recurrence model](https://bryntum.com/docs/gantt/api/#Scheduler/model/RecurrenceModel). Used by [RecurrenceEditor](https://bryntum.com/docs/gantt/api/#Scheduler/view/recurrence/RecurrenceEditor), and by the recurrence tab in Scheduler Pro's event editor.

Not intended to be used separately.

## Functions

Functions are methods available for calling on the class

[syncEventRecord](https://bryntum.com/docs/gantt/api/Scheduler/view/recurrence/RecurrenceEditorPanel#function-syncEventRecord)
Updates the provided recurrence model with the contained form data. If recurrence model is not provided updates the last loaded recurrence model.
