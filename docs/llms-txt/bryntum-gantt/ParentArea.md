# [ParentArea](https://bryntum.com/docs/gantt/api/Gantt/feature/ParentArea)

Highlights the area encapsulating all child tasks of a parent task in a semi-transparent layer. You can style these layer elements using the `b-parent-area` CSS class.

This feature is **disabled** by default. For info on enabling it, see [GridFeatures](https://bryntum.com/docs/gantt/api/#Grid/view/mixin/GridFeatures).

```
const gantt = new Gantt({
    features : {
        parentArea : true
    }
});
```
