# [DeadlineDateColumn](https://bryntum.com/docs/gantt/api/Gantt/column/DeadlineDateColumn)

A column showing the [deadlineDate](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-deadlineDate) field.

Default editor is a [DateField](https://bryntum.com/docs/gantt/api/#Core/widget/DateField).

If [format](https://bryntum.com/docs/gantt/api/#Gantt/column/DeadlineDateColumn#config-format) is omitted, Gantt's [displayDateFormat](https://bryntum.com/docs/gantt/api/#Scheduler/view/mixin/TimelineViewPresets#config-displayDateFormat) will be used as a default value and the format will be dynamically updated while zooming according to the [displayDateFormat](https://bryntum.com/docs/gantt/api/#Scheduler/preset/ViewPreset#field-displayDateFormat) value specified for the ViewPreset being selected.
