# [HorizontalLayoutStack](https://bryntum.com/docs/gantt/api/Scheduler/eventlayout/HorizontalLayoutStack)

Handles layout of events within a row (resource) in horizontal mode. Stacks events, increasing row height when to fit all overlapping events.

This layout is used by default in horizontal mode.
