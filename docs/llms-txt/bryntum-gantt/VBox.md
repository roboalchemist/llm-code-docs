# [VBox](https://bryntum.com/docs/gantt/api/Core/widget/layout/VBox)

A [Box](https://bryntum.com/docs/gantt/api/#Core/widget/layout/Box) layout that defaults to `horizontal : false`.

The following:

```
 layout : {
     type : 'vbox'
 }
```

Is equivalent to:

```
 layout : {
     type       : 'box',
     horizontal : false
 }
```
