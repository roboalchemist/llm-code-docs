# [VersionStore](https://bryntum.com/docs/gantt/api/SchedulerPro/data/VersionStore)

A [Store](https://bryntum.com/docs/gantt/api/#Core/data/Store) that contains the list of saved versions of the project, managed by the [Versions](https://bryntum.com/docs/gantt/api/#SchedulerPro/feature/Versions) feature. See also [ChangeLogStore](https://bryntum.com/docs/gantt/api/#SchedulerPro/data/ChangeLogStore).

You can provide a custom subclass of [VersionModel](https://bryntum.com/docs/gantt/api/#SchedulerPro/model/VersionModel) using the [versionModelClass](https://bryntum.com/docs/gantt/api/#SchedulerPro/feature/Versions#config-versionModelClass) config.
