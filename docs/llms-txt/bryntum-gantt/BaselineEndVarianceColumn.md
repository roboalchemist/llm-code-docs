# [BaselineEndVarianceColumn](https://bryntum.com/docs/gantt/api/Gantt/column/BaselineEndVarianceColumn)

A column that displays the task End Variance. The end variance field is "0 days" until the task start date varies from the baseline end date. This field is calculated as:

```
End Variance = End - Baseline End
```
