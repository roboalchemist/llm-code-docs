Chatbot</h1></header>
<p><a href="https://www.chatbase.co" target="_blank" rel="noopener noreferrer" class="">Chatbase</a> is a platform for building custom AI chatbots that can be embedded into websites for customer support, lead generation, and user engagement. These chatbots use RAG (Retrieval-Augmented Generation) to access your organization&#x27;s knowledge base and maintain conversations with users.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="multi-turn-vs-single-turn-testing">Multi-turn vs Single-turn Testing<a href="#multi-turn-vs-single-turn-testing" class="hash-link" aria-label="Direct link to Multi-turn vs Single-turn Testing" title="Direct link to Multi-turn vs Single-turn Testing" translate="no">​</a></h2>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="single-turn-systems">Single-turn Systems<a href="#single-turn-systems" class="hash-link" aria-label="Direct link to Single-turn Systems" title="Direct link to Single-turn Systems" translate="no">​</a></h3>
<p>Many LLM applications process each query independently, treating every interaction as a new conversation. Like talking to someone with no memory of previous exchanges, they can answer your current question but don&#x27;t retain context from earlier messages.</p>
<p>This makes single-turn systems inherently more secure since attackers can&#x27;t manipulate conversation history. However, this security comes at the cost of usability - users must provide complete context with every message, making interactions cumbersome.</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="multi-turn-systems-like-chatbase">Multi-turn Systems (Like Chatbase)<a href="#multi-turn-systems-like-chatbase" class="hash-link" aria-label="Direct link to Multi-turn Systems (Like Chatbase)" title="Direct link to Multi-turn Systems (Like Chatbase)" translate="no">​</a></h3>
<p>Modern conversational AI, including Chatbase, maintains context throughout the interaction. When users ask follow-up questions, the system understands the context from previous messages, enabling natural dialogue.</p>
<p>In Promptfoo, this state is managed through a <code>conversationId</code> that links messages together. While this enables a better user experience, it introduces security challenges. Attackers might try to manipulate the conversation context across multiple messages, either building false premises or attempting to extract sensitive information.</p>
