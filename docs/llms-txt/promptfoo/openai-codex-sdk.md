<ul>
<li class=""><code>never</code> - Never require approval</li>
<li class=""><code>on-request</code> - Require approval when requested</li>
<li class=""><code>on-failure</code> - Require approval after failures</li>
<li class=""><code>untrusted</code> - Require approval for untrusted operations</li>
</ul>

<h2 class="anchor anchorTargetStickyNavbar_tleR" id="comparison-with-claude-agent-sdk">Comparison with Claude Agent SDK<a href="#comparison-with-claude-agent-sdk" class="hash-link" aria-label="Direct link to Comparison with Claude Agent SDK" title="Direct link to Comparison with Claude Agent SDK" translate="no">​</a></h2>
<p>Both providers support code operations, but have different features:</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="openai-codex-sdk-1">OpenAI Codex SDK<a href="#openai-codex-sdk-1" class="hash-link" aria-label="Direct link to OpenAI Codex SDK" title="Direct link to OpenAI Codex SDK" translate="no">​</a></h3>
<ul>
<li class=""><strong>Best for</strong>: Code generation, structured output, reasoning tasks</li>
<li class=""><strong>Features</strong>: JSON schema support, thread persistence, Codex models</li>
<li class=""><strong>Thread management</strong>: Built-in pooling and resumption</li>
<li class=""><strong>Working directory</strong>: Git repository validation</li>
<li class=""><strong>Configuration</strong>: Focused on code tasks</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="claude-agent-sdk">Claude Agent SDK<a href="#claude-agent-sdk" class="hash-link" aria-label="Direct link to Claude Agent SDK" title="Direct link to Claude Agent SDK" translate="no">​</a></h3>
<ul>
<li class=""><strong>Best for</strong>: File manipulation, system commands, MCP integration</li>
<li class=""><strong>Features</strong>: Tool permissions, MCP servers, CLAUDE.md support</li>
<li class=""><strong>Thread management</strong>: Temporary directory isolation</li>
<li class=""><strong>Working directory</strong>: No Git requirement</li>
<li class=""><strong>Configuration</strong>: More options for tool permissions and system access</li>
</ul>
<p>Choose based on your use case:</p>
<ul>
<li class=""><strong>Code generation &amp; analysis</strong> → OpenAI Codex SDK</li>
<li class=""><strong>System operations &amp; tooling</strong> → Claude Agent SDK</li>
</ul>
