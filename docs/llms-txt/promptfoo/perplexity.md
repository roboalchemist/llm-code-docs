<p>The <a href="https://blog.perplexity.ai/blog/introducing-pplx-api" target="_blank" rel="noopener noreferrer" class="">Perplexity API</a> provides chat completion models with built-in search capabilities, citations, and structured output support. <p>Perplexity follows OpenAI&#x27;s chat completion API format - see our <a href="https://promptfoo.dev/docs/providers/openai" target="_blank" rel="noopener noreferrer" class="">OpenAI documentation</a> for the base API details.</p>

<li class="">Set the <code>PERPLEXITY_API_KEY</code> environment variable or specify <code>apiKey</code> in your config</li>
</ol>

<h2 class="anchor anchorTargetStickyNavbar_tleR" id="best-practices">Best Practices<a href="#best-practices" class="hash-link" aria-label="Direct link to Best Practices" title="Direct link to Best Practices" translate="no">​</a></h2>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="model-selection">Model Selection<a href="#model-selection" class="hash-link" aria-label="Direct link to Model Selection" title="Direct link to Model Selection" translate="no">​</a></h3>
<ul>
<li class=""><strong>sonar-pro</strong>: Use for complex queries requiring detailed responses with citations</li>
<li class=""><strong>sonar</strong>: Use for factual queries and cost efficiency</li>
<li class=""><strong>sonar-reasoning-pro/sonar-reasoning</strong>: Use for step-by-step problem solving</li>
<li class=""><strong>sonar-deep-research</strong>: Use for comprehensive reports (may take 30+ minutes)</li>
<li class=""><strong>r1-1776</strong>: Use for creative content not requiring search</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="search-optimization">Search Optimization<a href="#search-optimization" class="hash-link" aria-label="Direct link to Search Optimization" title="Direct link to Search Optimization" translate="no">​</a></h3>
<ul>
<li class="">Set <code>search_domain_filter</code> to trusted domains for higher quality citations</li>
<li class="">Use <code>search_recency_filter</code> for time-sensitive topics</li>
<li class="">For cost optimization, set <code>web_search_options.search_context_size</code> to &quot;low&quot;</li>
<li class="">For comprehensive research, set <code>web_search_options.search_context_size</code> to &quot;high&quot;</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="structured-output-tips">Structured Output Tips<a href="#structured-output-tips" class="hash-link" aria-label="Direct link to Structured Output Tips" title="Direct link to Structured Output Tips" translate="no">​</a></h3>
<ul>
<li class="">When using structured outputs with reasoning models, responses will include a <code>&lt;think&gt;</code> section followed by the structured output</li>
<li class="">For regex patterns, ensure they follow the supported syntax</li>
<li class="">JSON schemas cannot include recursive structures or unconstrained objects</li>
</ul>
