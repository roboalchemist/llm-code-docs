<h2 class="anchor anchorTargetStickyNavbar_tleR" id="supported-models">Supported Models<a href="#supported-models" class="hash-link" aria-label="Direct link to Supported Models" title="Direct link to Supported Models" translate="no">​</a></h2>
<p>The Alibaba provider includes support for the following model formats:</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="qwen-3-flagship">Qwen 3 Flagship<a href="#qwen-3-flagship" class="hash-link" aria-label="Direct link to Qwen 3 Flagship" title="Direct link to Qwen 3 Flagship" translate="no">​</a></h3>
<ul>
<li class=""><code>qwen3-max</code> - Next-generation flagship with reasoning and tool integration</li>
<li class=""><code>qwen3-max-preview</code> - Preview version with thinking mode support</li>
<li class=""><code>qwen3-max-2025-09-23</code> - September 2025 snapshot</li>
<li class=""><code>qwen-max</code> - 32K context (30,720 in, 8,192 out)</li>
<li class=""><code>qwen-max-latest</code> - Always updated to latest version</li>
<li class=""><code>qwen-max-2025-01-25</code> - January 2025 snapshot</li>
<li class=""><code>qwen-plus</code> / <code>qwen-plus-latest</code> - 128K-1M context (thinking &amp; non-thinking modes)</li>
<li class=""><code>qwen-plus-2025-09-11</code>, <code>qwen-plus-2025-07-28</code>, <code>qwen-plus-2025-07-14</code>, <code>qwen-plus-2025-04-28</code>, <code>qwen-plus-2025-01-25</code> - Dated snapshots</li>
<li class=""><code>qwen-flash</code> / <code>qwen-flash-2025-07-28</code> - Latency-optimized general model</li>
<li class=""><code>qwen-turbo</code> / <code>qwen-turbo-latest</code> / <code>qwen-turbo-2025-04-28</code> / <code>qwen-turbo-2024-11-01</code> - Fast, cost-effective (being replaced by qwen-flash)</li>
<li class=""><code>qwen-long-latest</code> / <code>qwen-long-2025-01-25</code> - <strong>10M context</strong> for long-text analysis, summarization, and extraction</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="qwen-3-omni--realtime">Qwen 3 Omni &amp; Realtime<a href="#qwen-3-omni--realtime" class="hash-link" aria-label="Direct link to Qwen 3 Omni &amp; Realtime" title="Direct link to Qwen 3 Omni &amp; Realtime" translate="no">​</a></h3>
<ul>
<li class=""><code>qwen3-omni-flash</code> / <code>qwen3-omni-flash-2025-09-15</code> - Multimodal flagship with speech + vision support (thinking &amp; non-thinking modes)</li>
<li class=""><code>qwen3-omni-flash-realtime</code> / <code>qwen3-omni-flash-realtime-2025-09-15</code> - Streaming realtime with audio stream input and VAD</li>
<li class=""><code>qwen3-omni-30b-a3b-captioner</code> - Dedicated audio captioning model (speech, ambient sounds, music)</li>
<li class=""><code>qwen2.5-omni-7b</code> - Qwen2.5-based multimodal model with text, image, speech, and video inputs</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="reasoning--research">Reasoning &amp; Research<a href="#reasoning--research" class="hash-link" aria-label="Direct link to Reasoning &amp; Research" title="Direct link to Reasoning &amp; Research" translate="no">​</a></h3>
<ul>
<li class=""><code>qwq-plus</code> - Alibaba&#x27;s reasoning model (commercial)</li>
<li class=""><code>qwq-32b</code> - Open-source QwQ reasoning model trained on Qwen2.5</li>
<li class=""><code>qwq-32b-preview</code> - Experimental QwQ research model (2024)</li>
<li class=""><code>qwen-deep-research</code> - Long-form research assistant with web search</li>
<li class=""><code>qvq-max</code> / <code>qvq-max-latest</code> / <code>qvq-max-2025-03-25</code> - Visual reasoning models (commercial)</li>
<li class=""><code>qvq-72b-preview</code> - Experimental visual reasoning research model</li>
<li class=""><strong>DeepSeek models</strong> (hosted by Alibaba Cloud):<!-- -->
<ul>
<li class=""><code>deepseek-v3.2-exp</code> / <code>deepseek-v3.1</code> / <code>deepseek-v3</code> - Latest DeepSeek models (671-685B)</li>
<li class=""><code>deepseek-r1</code> / <code>deepseek-r1-0528</code> - DeepSeek reasoning models</li>
<li class=""><code>deepseek-r1-distill-qwen-{1.5b,7b,14b,32b}</code> - Distilled on Qwen2.5</li>
<li class=""><code>deepseek-r1-distill-llama-{8b,70b}</code> - Distilled on Llama</li>
</ul>
</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="vision--multimodal">Vision &amp; Multimodal<a href="#vision--multimodal" class="hash-link" aria-label="Direct link to Vision &amp; Multimodal" title="Direct link to Vision &amp; Multimodal" translate="no">​</a></h3>
<p><strong>Commercial:</strong></p>
<ul>
<li class=""><code>qwen3-vl-plus</code> / <code>qwen3-vl-plus-2025-09-23</code> - High-res image support with long context (thinking &amp; non-thinking modes)</li>
<li class=""><code>qwen3-vl-flash</code> / <code>qwen3-vl-flash-2025-10-15</code> - Fast vision model with thinking mode support</li>
<li class=""><code>qwen-vl-max</code> - 7.5K context, 1,280 tokens/image</li>
<li class=""><code>qwen-vl-plus</code> - High-res image support</li>
<li class=""><code>qwen-vl-ocr</code> - OCR-optimized for documents, tables, handwriting (30+ languages)</li>
</ul>
<p><strong>Open-source:</strong></p>
<ul>
<li class=""><code>qwen3-vl-235b-a22b-thinking</code> / <code>qwen3-vl-235b-a22b-instruct</code> - 235B parameter Qwen3-VL</li>
<li class=""><code>qwen3-vl-32b-thinking</code> / <code>qwen3-vl-32b-instruct</code> - 32B parameter Qwen3-VL</li>
<li class=""><code>qwen3-vl-30b-a3b-thinking</code> / <code>qwen3-vl-30b-a3b-instruct</code> - 30B parameter Qwen3-VL</li>
<li class=""><code>qwen3-vl-8b-thinking</code> / <code>qwen3-vl-8b-instruct</code> - 8B parameter Qwen3-VL</li>
<li class=""><code>qwen2.5-vl-{72b,7b,3b}-instruct</code> - Qwen 2.5 VL series</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="audio--speech">Audio &amp; Speech<a href="#audio--speech" class="hash-link" aria-label="Direct link to Audio &amp; Speech" title="Direct link to Audio &amp; Speech" translate="no">​</a></h3>

</ul>
