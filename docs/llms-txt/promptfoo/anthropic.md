<h3 class="anchor anchorTargetStickyNavbar_tleR" id="options">Options<a href="#options" class="hash-link" aria-label="Direct link to Options" title="Direct link to Options" translate="no">​</a></h3>
<p>The Anthropic provider supports several options to customize the behavior of the model. These include:</p>
<ul>
<li class=""><code>temperature</code>: Controls the randomness of the output.</li>
<li class=""><code>max_tokens</code>: The maximum length of the generated text.</li>
<li class=""><code>top_p</code>: Controls nucleus sampling, affecting the randomness of the output.</li>
<li class=""><code>top_k</code>: Only sample from the top K options for each subsequent token.</li>
<li class=""><code>tools</code>: An array of tool or function definitions for the model to call.</li>
<li class=""><code>tool_choice</code>: An object specifying the tool to call.</li>
<li class=""><code>extra_body</code>: Additional parameters to pass directly to the Anthropic API request body.</li>
</ul>

<h3 class="anchor anchorTargetStickyNavbar_tleR" id="additional-capabilities">Additional Capabilities<a href="#additional-capabilities" class="hash-link" aria-label="Direct link to Additional Capabilities" title="Direct link to Additional Capabilities" translate="no">​</a></h3>
<ul>
<li class=""><strong>Caching</strong>: Promptfoo caches previous LLM requests by default.</li>
<li class=""><strong>Token Usage Tracking</strong>: Provides detailed information on the number of tokens used in each request, aiding in usage monitoring and optimization.</li>
<li class=""><strong>Cost Calculation</strong>: Calculates the cost of each request based on the number of tokens generated and the specific model used.</li>
</ul>
