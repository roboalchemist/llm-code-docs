<h2 class="anchor anchorTargetStickyNavbar_tleR" id="troubleshooting">Troubleshooting<a href="#troubleshooting" class="hash-link" aria-label="Direct link to Troubleshooting" title="Direct link to Troubleshooting" translate="no">​</a></h2>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="common-issues">Common Issues<a href="#common-issues" class="hash-link" aria-label="Direct link to Common Issues" title="Direct link to Common Issues" translate="no">​</a></h3>
<p><strong>Agent conversations timeout:</strong></p>
<ul>
<li class="">Increase <code>maxTurns</code> and <code>timeout</code> in config</li>
<li class="">Simplify evaluation criteria</li>
<li class="">Use faster LLM models</li>
</ul>
<p><strong>High costs during testing:</strong></p>
<ul>
<li class="">Use <code>gpt-4o-mini</code> instead of <code>gpt-4o</code></li>
<li class="">Enable caching for repeated tests</li>
<li class="">Implement LLM cascading</li>
<li class="">Test with shorter prompts first</li>
</ul>
<p><strong>Evaluation criteria always failing:</strong></p>
<ul>
<li class="">Start with simple, objective criteria</li>
<li class="">Lower passing thresholds during development</li>
<li class="">Review agent transcript to understand behavior</li>
<li class="">Add more specific criteria descriptions</li>
</ul>
<p><strong>Audio quality issues:</strong></p>
<ul>
<li class="">Try different <code>outputFormat</code> settings</li>
<li class="">Adjust voice settings (stability, similarity_boost)</li>
<li class="">Test with different models</li>
<li class="">Consider using Turbo over Flash for quality</li>
</ul>
