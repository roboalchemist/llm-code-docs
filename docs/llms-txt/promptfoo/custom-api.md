<h2 class="anchor anchorTargetStickyNavbar_tleR" id="supported-file-formats-and-examples">Supported File Formats and Examples<a href="#supported-file-formats-and-examples" class="hash-link" aria-label="Direct link to Supported File Formats and Examples" title="Direct link to Supported File Formats and Examples" translate="no">​</a></h2>

<ul>
<li class=""><a href="https://github.com/promptfoo/promptfoo/tree/main/examples/custom-provider" target="_blank" rel="noopener noreferrer" class="">CommonJS Provider</a> - (<code>.js</code>, <code>.cjs</code>) - Uses <code>module.exports</code> and <code>require()</code></li>

<li class=""><a href="https://github.com/promptfoo/promptfoo/tree/main/examples/custom-provider-typescript" target="_blank" rel="noopener noreferrer" class="">TypeScript Provider</a> - (<code>.ts</code>) - Provides type safety with interfaces</li>
<li class=""><a href="https://github.com/promptfoo/promptfoo/tree/main/examples/custom-provider-embeddings" target="_blank" rel="noopener noreferrer" class="">Embeddings Provider</a> (commonjs)</li>
</ul>
