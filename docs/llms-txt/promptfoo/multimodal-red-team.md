<p>The key elements in this configuration:</p>
<ul>

<li class=""><code>defaultTest.options.transformVars</code>: Adds image preprocessing that removes the &quot;data<!-- -->:binary<!-- -->/octet-stream;base64,&quot; prefix from image data - this is specifically required for Amazon Bedrock Nova models but may not be needed for other providers</li>
<li class=""><code>redteam.injectVar: image</code>: Specifies that the image variable will receive unsafe content</li>
<li class=""><code>redteam.plugins</code>: Uses the unsafebench plugin with specific categories of unsafe images</li>
<li class=""><code>redteam.purpose</code>: Provides context for the model&#x27;s role as an assistant with ethical boundaries</li>
</ul>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="how-the-unsafebench-plugin-works">How the UnsafeBench Plugin Works<a href="#how-the-unsafebench-plugin-works" class="hash-link" aria-label="Direct link to How the UnsafeBench Plugin Works" title="Direct link to How the UnsafeBench Plugin Works" translate="no">​</a></h3>
<p>The UnsafeBench plugin:</p>
<ol>
<li class="">Fetches images labeled as &quot;unsafe&quot; from the UnsafeBench dataset</li>
<li class="">Automatically converts images to base64 format (even if originally provided as URLs)</li>
<li class="">Filters images by specified categories (Violence, Sexual, Hate, etc.)</li>
<li class="">Injects these images into your prompts for testing</li>
<li class="">Allows for automated evaluation of model responses</li>
</ol>
<h4 class="anchor anchorTargetStickyNavbar_tleR" id="image-format-handling">Image Format Handling<a href="#image-format-handling" class="hash-link" aria-label="Direct link to Image Format Handling" title="Direct link to Image Format Handling" translate="no">​</a></h4>
<p>Some providers like Amazon Bedrock Nova require special handling for image data:</p>
<ul>
<li class="">The <code>transformVars</code> function removes any &quot;data<!-- -->:binary<!-- -->/octet-stream;base64,&quot; prefix from image data</li>
<li class="">This transformation is needed specifically for Nova models but generally not required for other providers like OpenAI or Anthropic</li>
<li class="">The function runs before the prompt template is filled with variables, ensuring the image data is in the correct format</li>
</ul>
