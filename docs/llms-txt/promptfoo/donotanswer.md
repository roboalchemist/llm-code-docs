<h2 class="anchor anchorTargetStickyNavbar_tleR" id="overview">Overview<a href="#overview" class="hash-link" aria-label="Direct link to Overview" title="Direct link to Overview" translate="no">â€‹</a></h2>
<p>The DoNotAnswer plugin tests how well LLMs handle harmful queries. The dataset contains questions that responsible AI systems should refuse to answer, across categories like:</p>
<ul>
<li class="">Discrimination and hate speech</li>
<li class="">Privacy violations</li>
<li class="">Adult content</li>
<li class="">Social stereotypes</li>
<li class="">Human-chatbot interaction issues</li>
</ul>
<p>Each test includes metadata about the risk area and type of harm.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="how-it-works">How it works<a href="#how-it-works" class="hash-link" aria-label="Direct link to How it works" title="Direct link to How it works" translate="no">â€‹</a></h2>
<p>The plugin:</p>
<ul>
<li class="">Loads harmful queries from the dataset</li>
<li class="">Selects and shuffles a specified number of queries</li>
<li class="">Tests if the model appropriately refuses each harmful request</li>
<li class="">Uses LLM-as-a-judge to evaluate responses based on harm categories</li>
</ul>
