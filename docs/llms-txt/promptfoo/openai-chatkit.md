<h3 class="anchor anchorTargetStickyNavbar_tleR" id="playwright-not-installed">Playwright not installed<a href="#playwright-not-installed" class="hash-link" aria-label="Direct link to Playwright not installed" title="Direct link to Playwright not installed" translate="no">​</a></h3>
<div class="language-text codeBlockContainer_mQmQ theme-code-block" style="--prism-color:#393A34;--prism-background-color:#f6f8fa"><div class="codeBlockContent_t_Hd"><pre tabindex="0" class="prism-code language-text codeBlock_RMoD thin-scrollbar" style="color:#393A34;background-color:#f6f8fa"><code class="codeBlockLines_AclH"><span class="token-line" style="color:#393A34"><span class="token plain">Error: Playwright browser not installed</span><br></span></code></pre></div></div>
<p>Run <code>npx playwright install chromium</code></p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="timeout-errors">Timeout errors<a href="#timeout-errors" class="hash-link" aria-label="Direct link to Timeout errors" title="Direct link to Timeout errors" translate="no">​</a></h3>
<ol>
<li class="">Increase the timeout: <code>timeout: 180000</code></li>
<li class="">Reduce concurrency: <code>--max-concurrency 1</code></li>
<li class="">Test the workflow manually in Agent Builder</li>
</ol>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="empty-responses">Empty responses<a href="#empty-responses" class="hash-link" aria-label="Direct link to Empty responses" title="Direct link to Empty responses" translate="no">​</a></h3>
<ol>
<li class="">Verify the workflow works in Agent Builder</li>
<li class="">Check that the workflow version exists</li>
<li class="">Increase timeout for slow responses</li>
</ol>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="high-memory-usage">High memory usage<a href="#high-memory-usage" class="hash-link" aria-label="Direct link to High memory usage" title="Direct link to High memory usage" translate="no">​</a></h3>
<p>Reduce <code>poolSize</code> or <code>--max-concurrency</code>. Each browser context consumes memory.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="architecture">Architecture<a href="#architecture" class="hash-link" aria-label="Direct link to Architecture" title="Direct link to Architecture" translate="no">​</a></h2>
<p>The provider:</p>
<ol>
<li class="">Starts a local HTTP server with the ChatKit embed</li>
<li class="">Acquires a browser context from the pool</li>
<li class="">Waits for ChatKit to initialize via the OpenAI session API</li>
<li class="">Sends messages through the ChatKit JavaScript API</li>
<li class="">Extracts responses from the DOM</li>
<li class="">Processes approval steps if configured</li>
<li class="">Returns the response and releases the context back to the pool</li>
</ol>
<p>ChatKit workflows require browser automation because they don&#x27;t expose a direct API.</p>
