<h2 class="anchor anchorTargetStickyNavbar_tleR" id="troubleshooting">Troubleshooting<a href="#troubleshooting" class="hash-link" aria-label="Direct link to Troubleshooting" title="Direct link to Troubleshooting" translate="no">​</a></h2>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="502-bad-gateway-errors">502 Bad Gateway Errors<a href="#502-bad-gateway-errors" class="hash-link" aria-label="Direct link to 502 Bad Gateway Errors" title="Direct link to 502 Bad Gateway Errors" translate="no">​</a></h3>
<p>If you encounter 502 Bad Gateway errors when using the xAI provider, this typically indicates:</p>
<ul>
<li class="">An invalid or missing API key</li>
<li class="">Server issues on x.ai&#x27;s side</li>
</ul>
<p>The xAI provider will provide helpful error messages to guide you in resolving these issues.</p>
<p><strong>Solution</strong>: Verify your <code>XAI_API_KEY</code> environment variable is set correctly. You can obtain an API key from <a href="https://x.ai/" target="_blank" rel="noopener noreferrer" class="">https://x.ai/</a>.</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="controlling-retries">Controlling Retries<a href="#controlling-retries" class="hash-link" aria-label="Direct link to Controlling Retries" title="Direct link to Controlling Retries" translate="no">​</a></h3>
<p>If you&#x27;re experiencing timeouts or want to control retry behavior:</p>
<ul>
<li class="">To disable retries for 5XX errors: <code>PROMPTFOO_RETRY_5XX=false</code></li>
<li class="">To reduce retry delays: <code>PROMPTFOO_REQUEST_BACKOFF_MS=1000</code> (in milliseconds)</li>
</ul>
