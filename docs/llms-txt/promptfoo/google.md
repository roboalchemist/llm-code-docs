markdown"><div style="position:relative"><header><h1>Google AI / Gemini</h1></header>
<p>The <code>google</code> provider enables integration with Google AI Studio and the Gemini API. It provides access to Google&#x27;s state-of-the-art language models with support for text, images, and video inputs.</p>
<p>If you are using Vertex AI instead of Google AI Studio, see the <a class="" href="/docs/providers/vertex/"><code>vertex</code> provider</a>.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="authentication">Authentication<a href="#authentication" class="hash-link" aria-label="Direct link to Authentication" title="Direct link to Authentication" translate="no">​</a></h2>
<p>To use the Google AI Studio API, you need to authenticate using an API key. Follow these steps:</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="1-get-an-api-key">1. Get an API Key<a href="#1-get-an-api-key" class="hash-link" aria-label="Direct link to 1. Get an API Key" title="Direct link to 1. Get an API Key" translate="no">​</a></h3>
<ol>
<li class="">Visit <a href="https://aistudio.google.com/" target="_blank" rel="noopener noreferrer" class="">Google AI Studio</a></li>
<li class="">Click on &quot;Get API key&quot; in the left sidebar</li>
<li class="">Create a new API key or use an existing one</li>
<li class="">Copy your API key</li>
</ol>
<p><strong>Security Note:</strong> Never commit API keys to version control. Always use environment variables or a <code>.env</code> file that&#x27;s added to <code>.gitignore</code>.</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="2-configure-authentication">2. Configure Authentication<a href="#2-configure-authentication" class="hash-link" aria-label="Direct link to 2. Configure Authentication" title="Direct link to 2. Configure Authentication" translate="no">​</a></h3>
