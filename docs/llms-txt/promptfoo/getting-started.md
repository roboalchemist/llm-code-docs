<li class="">Compare same models with different temperatures (see <a href="https://github.com/promptfoo/promptfoo/tree/main/examples/gpt-4o-temperature-comparison" target="_blank" rel="noopener noreferrer" class="">GPT temperature comparison</a>)</li>
<li class="">Compare Llama vs. GPT (see <a class="" href="/docs/guides/compare-llama2-vs-gpt/">Llama vs GPT benchmark</a>)</li>
<li class="">Compare Retrieval-Augmented Generation (RAG) with LangChain vs. regular GPT-4 (see <a class="" href="/docs/configuration/testing-llm-chains/">LangChain example</a>)</li>
</ul>
