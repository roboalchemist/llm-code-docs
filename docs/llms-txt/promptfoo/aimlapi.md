<p>AI/ML API&#x27;s endpoints are compatible with OpenAI&#x27;s API, which means all parameters available in the <a class="" href="/docs/providers/openai/">OpenAI provider</a> work with AI/ML API.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="setup">Setup<a href="#setup" class="hash-link" aria-label="Direct link to Setup" title="Direct link to Setup" translate="no">​</a></h2>
<p>To use AI/ML API, you need to set the <code>AIML_API_KEY</code> environment variable or specify the <code>apiKey</code> in the provider configuration.</p>
