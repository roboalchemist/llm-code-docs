<h2 class="anchor anchorTargetStickyNavbar_tleR" id="keyboard-shortcuts">Keyboard Shortcuts<a href="#keyboard-shortcuts" class="hash-link" aria-label="Direct link to Keyboard Shortcuts" title="Direct link to Keyboard Shortcuts" translate="no">‚Äã</a></h2>
<table><thead><tr><th>Shortcut</th><th>Action</th></tr></thead><tbody><tr><td><code>Ctrl+K</code> / <code>Cmd+K</code></td><td>Open eval selector</td></tr><tr><td><code>Esc</code></td><td>Clear search</td></tr><tr><td><code>Shift</code> (hold)</td><td>Show extra cell actions</td></tr></tbody></table>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="toolbar">Toolbar<a href="#toolbar" class="hash-link" aria-label="Direct link to Toolbar" title="Direct link to Toolbar" translate="no">‚Äã</a></h2>
<ul>
<li class=""><strong>Eval selector</strong> - Switch between evals</li>
<li class=""><strong>Zoom</strong> - Scale columns (50%-200%)</li>
<li class=""><strong>Display mode</strong> - Filter: All, Failures, Passes, Errors, Different, Highlights</li>
<li class=""><strong>Search</strong> - Text or regex</li>
<li class=""><strong>Filters</strong> - By metrics, metadata, pass/fail. Operators: <code>=</code>, <code>contains</code>, <code>&gt;</code>, <code>&lt;</code></li>
</ul>
<p><img decoding="async" loading="lazy" alt="Display mode dropdown" src="/assets/images/web-ui-display-mode-1088ff1003fc319d9833b292bb6b885e.png" width="1400" height="900" class="img_SS3x"></p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="table-settings">Table Settings<a href="#table-settings" class="hash-link" aria-label="Direct link to Table Settings" title="Direct link to Table Settings" translate="no">‚Äã</a></h2>
<p><img decoding="async" loading="lazy" alt="Table Settings dialog" src="/assets/images/web-ui-table-settings-ec4356f4ead509dca1b4ae71b714a82a.png" width="1400" height="900" class="img_SS3x"></p>
<ul>
<li class=""><strong>Columns</strong> - Toggle variable and prompt visibility</li>
<li class=""><strong>Truncation</strong> - Max text length, word wrap</li>
<li class=""><strong>Rendering</strong> - Markdown, JSON prettification</li>
<li class=""><strong>Inference details</strong> - Tokens, latency, cost, tokens/sec</li>
<li class=""><strong>Media</strong> - Image size limits; double-click for lightbox</li>
</ul>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="cell-actions">Cell Actions<a href="#cell-actions" class="hash-link" aria-label="Direct link to Cell Actions" title="Direct link to Cell Actions" translate="no">‚Äã</a></h2>
<p>Hover to reveal actions. Hold <code>Shift</code> for more:</p>
<table><thead><tr><th></th><th>Action</th><th>Description</th></tr></thead><tbody><tr><td>üîç</td><td>Details</td><td>Full output, prompt, variables, grading results</td></tr><tr><td>üëç</td><td>Pass</td><td>Mark as passed (score = 1.0)</td></tr><tr><td>üëé</td><td>Fail</td><td>Mark as failed (score = 0.0)</td></tr><tr><td>üî¢</td><td>Score</td><td>Set custom score (0-1)</td></tr><tr><td>‚úèÔ∏è</td><td>Comment</td><td>Add notes</td></tr><tr><td>‚≠ê</td><td>Highlight</td><td>Mark for review (<code>Shift</code>)</td></tr><tr><td>üìã</td><td>Copy</td><td>Copy to clipboard (<code>Shift</code>)</td></tr><tr><td>üîó</td><td>Share</td><td>Link to this output (<code>Shift</code>)</td></tr></tbody></table>
<p>Ratings and comments persist and are included in exports‚Äîuse them to build training datasets.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="eval-actions">Eval Actions<a href="#eval-actions" class="hash-link" aria-label="Direct link to Eval Actions" title="Direct link to Eval Actions" translate="no">‚Äã</a></h2>
<p><img decoding="async" loading="lazy" alt="Eval actions menu" src="/assets/images/web-ui-eval-actions-628bf5c780e6c72e95a649ae7e207a0c.png" width="1400" height="900" class="img_SS3x"></p>
<ul>
<li class=""><strong>Edit name</strong> - Rename eval</li>
<li class=""><strong>Edit and re-run</strong> - Open in eval creator</li>
<li class=""><strong>Compare</strong> - Diff against another eval (green = added, red = removed)</li>
<li class=""><strong>View YAML</strong> - Show config</li>
<li class=""><strong>Download</strong> - Opens export dialog:</li>
</ul>
<table><thead><tr><th>Export</th><th>Use case</th></tr></thead><tbody><tr><td>YAML config</td><td>Re-run the eval</td></tr><tr><td>Failed tests only</td><td>Debug failures</td></tr><tr><td>CSV / JSON</td><td>Analysis, reporting</td></tr><tr><td>DPO JSON</td><td>Preference training data</td></tr><tr><td>Human Eval YAML</td><td>Human labeling workflows</td></tr><tr><td>Burp payloads</td><td>Security testing (red team only)</td></tr></tbody></table>
<ul>
<li class=""><strong>Copy</strong> - Duplicate eval</li>
<li class=""><strong>Share</strong> - Generate URL (see <a href="#sharing" class="">Sharing</a>)</li>
<li class=""><strong>Delete</strong></li>
</ul>
