class="theme-doc-markdown markdown"><div style="position:relative"><header><h1>Voyage AI</h1></header>
<p><a href="https://www.voyageai.com/" target="_blank" rel="noopener noreferrer" class="">Voyage AI</a> is Anthropic&#x27;s <a href="https://docs.anthropic.com/en/docs/embeddings" target="_blank" rel="noopener noreferrer" class="">recommended</a> embeddings provider. It supports <a href="https://docs.voyageai.com/docs/embeddings" target="_blank" rel="noopener noreferrer" class="">all models</a>. Latest models include:</p>
<ul>
<li class="">voyage-3-large (state-of-the-art general-purpose, Jan 2025)</li>
<li class="">voyage-3.5 and voyage-3.5-lite (improved quality, May 2025)</li>
<li class="">voyage-3 and voyage-3-lite (general-purpose with 32K context)</li>
<li class="">voyage-multimodal-3 (text + images)</li>
<li class="">voyage-context-3 (contextualized chunks)</li>
<li class="">voyage-code-3 (code-specific retrieval)</li>
</ul>
<p>To use it, set the <code>VOYAGE_API_KEY</code> environment variable.</p>
<p>Use it like so:</p>
