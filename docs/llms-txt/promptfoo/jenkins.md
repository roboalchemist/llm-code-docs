<p>Here&#x27;s an example of a more advanced pipeline with additional features:</p>
<p>The advanced configuration includes several important improvements:</p>
<ul>
<li class=""><strong>Build timeouts</strong>: The <code>timeout</code> option ensures builds don&#x27;t run indefinitely (1 hour limit)</li>
<li class=""><strong>Timestamps</strong>: Adds timestamps to console output for better debugging</li>
<li class=""><strong>SCM polling</strong>: Automatically checks for changes every 15 minutes using <code>pollSCM</code></li>
<li class=""><strong>Conditional execution</strong>: Only runs evaluations when files in <code>prompts/</code> directory change</li>
<li class=""><strong>Email notifications</strong>: Sends emails to developers on pipeline failures</li>
<li class=""><strong>Workspace cleanup</strong>: Automatically cleans up workspace after each run</li>
<li class=""><strong>Artifact management</strong>: Archives both JSON and HTML reports with fingerprinting</li>
<li class=""><strong>Better error handling</strong>: More robust error catching and build status management</li>
</ul>

<h2 class="anchor anchorTargetStickyNavbar_tleR" id="troubleshooting">Troubleshooting<a href="#troubleshooting" class="hash-link" aria-label="Direct link to Troubleshooting" title="Direct link to Troubleshooting" translate="no">​</a></h2>
<p>Common issues and solutions:</p>
<ol>
<li class="">
<p><strong>Permission issues:</strong></p>
<ul>
<li class="">Ensure Jenkins has appropriate permissions to install global npm packages</li>
<li class="">Verify cache directory permissions</li>
<li class="">Check API key credential permissions</li>
</ul>
</li>
<li class="">
<p><strong>Pipeline timeout:</strong></p>
<ul>
<li class="">Adjust the timeout in pipeline options</li>
<li class="">Consider splitting evaluations into smaller batches</li>
<li class="">Monitor API rate limits</li>
</ul>
</li>
<li class="">
<p><strong>Cache problems:</strong></p>
<ul>
<li class="">Verify cache path exists and is writable</li>
<li class="">Check disk space availability</li>
<li class="">Clear cache if needed: <code>rm -rf ~/.promptfoo/cache/*</code></li>
</ul>
</li>
<li class="">
<p><strong>Node.js issues:</strong></p>
<ul>
<li class="">Ensure Node.js is installed on the Jenkins agent</li>
<li class="">Verify npm is available in PATH</li>
<li class="">Consider using <code>nodejs</code> tool installer in Jenkins</li>
</ul>
</li>
</ol>
