class="theme-doc-markdown markdown"><div style="position:relative"><header><h1>Image Jailbreaking</h1></header>
<p>The Image strategy converts prompt text into an image and then encodes that image as a base64 string. This approach enables testing how AI systems handle images of text, which may potentially bypass text-based guardrails / content filters or lead to different behaviors than when processing plain text.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="why-use-this-strategy">Why Use This Strategy<a href="#why-use-this-strategy" class="hash-link" aria-label="Direct link to Why Use This Strategy" title="Direct link to Why Use This Strategy" translate="no">​</a></h2>
<p>This strategy helps security researchers and AI developers:</p>
<ol>
<li class=""><strong>Test model capabilities</strong>: Assess if models can extract and process text from base64-encoded images</li>
<li class=""><strong>Evaluate security measures</strong>: Determine if image-encoded text can bypass content filters that typically scan plain text</li>
<li class=""><strong>Assess multi-modal behavior</strong>: Identify differences in how models respond to the same content in different formats</li>
<li class=""><strong>Discover inconsistencies</strong>: Reveal potential vulnerabilities by comparing text-based and image-based processing pathways</li>
</ol>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="how-it-works">How It Works<a href="#how-it-works" class="hash-link" aria-label="Direct link to How It Works" title="Direct link to How It Works" translate="no">​</a></h2>
<p>The strategy performs the following operations:</p>
<ol>
<li class="">Takes the original text from your test case</li>
<li class="">Renders the text onto a blank PNG image with a white background and black text</li>
<li class="">Converts the PNG image to a base64 string</li>
