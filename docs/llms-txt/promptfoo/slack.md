<h3 class="anchor anchorTargetStickyNavbar_tleR" id="slack-app-setup">Slack App Setup<a href="#slack-app-setup" class="hash-link" aria-label="Direct link to Slack App Setup" title="Direct link to Slack App Setup" translate="no">​</a></h3>
<ol>
<li class="">
<p><strong>Create a Slack App</strong></p>
<ul>
<li class="">Go to <a href="https://api.slack.com/apps" target="_blank" rel="noopener noreferrer" class="">api.slack.com/apps</a></li>
<li class="">Click &quot;Create New App&quot; → &quot;From scratch&quot;</li>
<li class="">Give your app a name and select your workspace</li>
</ul>
</li>
<li class="">
<p><strong>Configure Bot Token Scopes</strong></p>
<ul>
<li class="">Navigate to &quot;OAuth &amp; Permissions&quot; in your app settings</li>
<li class="">Under &quot;Scopes&quot; → &quot;Bot Token Scopes&quot;, add these REQUIRED scopes:<!-- -->
<ul>
<li class=""><code>chat:write</code> - to send messages</li>
<li class=""><code>channels:history</code> - to read public channel messages</li>
<li class=""><code>groups:history</code> - to read private channel messages</li>
<li class=""><code>im:history</code> - to read direct messages</li>
<li class=""><code>channels:read</code> - to access public channel information</li>
<li class=""><code>groups:read</code> - to access private channel information</li>
<li class=""><code>im:read</code> - to access direct message information</li>
</ul>
</li>
</ul>
<p><strong>Note</strong>: All scopes are required for the provider to work properly across different channel types.</p>
</li>
<li class="">
<p><strong>Install App to Workspace</strong></p>
<ul>
<li class="">Go to &quot;Install App&quot; in your app settings</li>
<li class="">Click &quot;Install to Workspace&quot;</li>
<li class="">Copy the &quot;Bot User OAuth Token&quot; (starts with <code>xoxb-</code>)</li>
</ul>
</li>
<li class="">
<p><strong>Invite Bot to Channel</strong></p>
<ul>
<li class="">In Slack, go to the channel where you want to use the bot</li>
<li class="">Type <code>/invite @YourBotName</code></li>
</ul>
</li>
</ol>

<h3 class="anchor anchorTargetStickyNavbar_tleR" id="best-practices-for-bot-testing">Best Practices for Bot Testing<a href="#best-practices-for-bot-testing" class="hash-link" aria-label="Direct link to Best Practices for Bot Testing" title="Direct link to Best Practices for Bot Testing" translate="no">​</a></h3>
<ol>
<li class=""><strong>Use dedicated test channels</strong> to avoid disrupting production</li>
<li class=""><strong>Set appropriate timeouts</strong> - bots may take longer to respond than humans</li>
<li class=""><strong>Test edge cases</strong> including malformed inputs and prompt injection attempts</li>
<li class=""><strong>Monitor rate limits</strong> when running many tests</li>
<li class=""><strong>Use assertions</strong> to verify both content and format of responses</li>
<li class=""><strong>Test at different times</strong> to ensure consistent performance</li>
</ol>
