<h2 class="anchor anchorTargetStickyNavbar_tleR" id="how-to-select-plugins">How to Select Plugins<a href="#how-to-select-plugins" class="hash-link" aria-label="Direct link to How to Select Plugins" title="Direct link to How to Select Plugins" translate="no">‚Äã</a></h2>
<p>Begin by assessing your LLM application&#x27;s architecture, including potential attack surfaces and relevant risk categories. Clearly define permissible and prohibited behaviors, extending beyond conventional security or privacy requirements. We recommend starting with a limited set of plugins to establish baseline insights, then gradually adding more as you refine your understanding of the model&#x27;s vulnerabilities. Keep in mind that increasing the number of plugins lengthens test durations and requires additional inference.</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="single-user-andor-prompt-and-response">Single User and/or Prompt and Response<a href="#single-user-andor-prompt-and-response" class="hash-link" aria-label="Direct link to Single User and/or Prompt and Response" title="Direct link to Single User and/or Prompt and Response" translate="no">‚Äã</a></h3>

<ul>
<li class=""><code>output</code>: The example output to grade</li>
<li class=""><code>pass</code>: Whether the output passes the test</li>
<li class=""><code>score</code>: The score for the output</li>
<li class=""><code>reason</code>: A brief explanation for the score</li>
</ul>

<h2 class="anchor anchorTargetStickyNavbar_tleR" id="custom-business-logic">Custom Business Logic<a href="#custom-business-logic" class="hash-link" aria-label="Direct link to Custom Business Logic" title="Direct link to Custom Business Logic" translate="no">‚Äã</a></h2>
<p>Promptfoo also supports custom test cases to reflect more nuanced requirements, whether you have certain benchmarks you want to test against or have specific behavioral standards defined by your legal, compliance, marketing, or brand teams. This could include enforcing policies such as an AI code of conduct or preventing an LLM from responding in ways that might breach legal requirements, such as in the <a class="" href="/blog/eu-ai-act/">EU AI Act</a>.</p>

<p>You can also include custom prompts using the <a class="" href="/docs/red-team/plugins/intent/">intents</a>, which are passed directly to the target. The intent red teaming plugin is designed to test whether an AI system can be manipulated to exhibit specific behaviors or responses.</p>
