markdown"><div style="position:relative"><header><h1>Cloudflare Workers AI</h1></header>
<p>This provider supports the <a href="https://developers.cloudflare.com/workers-ai/models/" target="_blank" rel="noopener noreferrer" class="">models</a> provided by Cloudflare Workers AI, a serverless edge inference platform that runs AI models closer to users for low-latency responses.</p>
<p>The provider uses Cloudflare&#x27;s OpenAI-compatible API endpoints, making it easy to migrate between OpenAI and Cloudflare AI or use them interchangeably.</p>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="required-configuration">Required Configuration<a href="#required-configuration" class="hash-link" aria-label="Direct link to Required Configuration" title="Direct link to Required Configuration" translate="no">â€‹</a></h2>
<p>Set your Cloudflare account ID and API key as environment variables:</p>
