type="button" class="clean-btn tocCollapsibleButton_iI2p">On this page</button></div><div class="theme-doc-markdown markdown"><div style="position:relative"><header><h1>OWASP Top 10 for Agentic Applications</h1></header>
<p>The <a href="https://genai.owasp.org/resource/owasp-top-10-for-agentic-applications/" target="_blank" rel="noopener noreferrer" class="">OWASP Top 10 for Agentic Applications</a> was announced during Black Hat Europe 2025 and the OWASP Agentic Security Summit. It represents the most critical security risks for AI agent applications.</p>
<p>Unlike traditional LLM applications, agentic systems introduce unique security challenges due to their:</p>
<ul>
<li class=""><strong>Autonomous decision-making</strong>: Agents independently determine steps to achieve goals</li>
<li class=""><strong>Persistent memory</strong>: Both short-term and long-term memory across sessions</li>
<li class=""><strong>Tool and API access</strong>: Direct interaction with external systems</li>
<li class=""><strong>Multi-agent coordination</strong>: Complex inter-agent communication and delegation</li>
</ul>
<h2 class="anchor anchorTargetStickyNavbar_tleR" id="the-top-10-risks">The Top 10 Risks<a href="#the-top-10-risks" class="hash-link" aria-label="Direct link to The Top 10 Risks" title="Direct link to The Top 10 Risks" translate="no">​</a></h2>
<table><thead><tr><th>ID</th><th>Risk Name</th><th>Description</th></tr></thead><tbody><tr><td>ASI01</td><td>Agent Goal Hijack</td><td>Attackers alter agent objectives through malicious content</td></tr><tr><td>ASI02</td><td>Tool Misuse and Exploitation</td><td>Agents use legitimate tools in unsafe ways</td></tr><tr><td>ASI03</td><td>Identity and Privilege Abuse</td><td>Agents inherit or escalate high-privilege credentials</td></tr><tr><td>ASI04</td><td>Agentic Supply Chain Vulnerabilities</td><td>Compromised tools, plugins, or external components</td></tr><tr><td>ASI05</td><td>Unexpected Code Execution</td><td>Agents generate or run code/commands unsafely</td></tr><tr><td>ASI06</td><td>Memory and Context Poisoning</td><td>Attackers poison agent memory systems and RAG databases</td></tr><tr><td>ASI07</td><td>Insecure Inter-Agent Communication</td><td>Multi-agent systems face spoofing and tampering</td></tr><tr><td>ASI08</td><td>Cascading Failures</td><td>Small errors propagate across planning and execution</td></tr><tr><td>ASI09</td><td>Human Agent Trust Exploitation</td><td>Users over-trust agent recommendations</td></tr><tr><td>ASI10</td><td>Rogue Agents</td><td>Compromised agents act harmfully while appearing legitimate</td></tr></tbody></table>

<li class="">Forging false consensus messages to exploit authentication weaknesses</li>
<li class="">Strategically planting false data that cascades through agent networks</li>
</ul>

<li class="">Implanting false information that progressively worsens through self-reinforcement</li>
</ul>
