<h3 class="anchor anchorTargetStickyNavbar_tleR" id="special-csv-columns">Special CSV Columns<a href="#special-csv-columns" class="hash-link" aria-label="Direct link to Special CSV Columns" title="Direct link to Special CSV Columns" translate="no">​</a></h3>
<table><thead><tr><th>Column</th><th>Purpose</th><th>Example</th></tr></thead><tbody><tr><td><code>__expected</code></td><td>Single assertion</td><td><code>contains: Paris</code></td></tr><tr><td><code>__expected1</code>, <code>__expected2</code>, ...</td><td>Multiple assertions</td><td><code>equals: 42</code></td></tr><tr><td><code>__description</code></td><td>Test description</td><td><code>Basic math test</code></td></tr><tr><td><code>__prefix</code></td><td>Prepend to prompt</td><td><code>You must answer: </code></td></tr><tr><td><code>__suffix</code></td><td>Append to prompt</td><td><code> (be concise)</code></td></tr><tr><td><code>__metric</code></td><td>Display name in reports (does not change assertion type)</td><td><code>accuracy</code></td></tr><tr><td><code>__threshold</code></td><td>Pass threshold (applies to all asserts)</td><td><code>0.8</code></td></tr><tr><td><code>__metadata:*</code></td><td>Filterable metadata</td><td>See below</td></tr><tr><td><code>__config:__expected:&lt;key&gt;</code> or <code>__config:__expectedN:&lt;key&gt;</code></td><td>Set configuration for all or specific assertions</td><td><code>__config:__expected:threshold</code>, <code>__config:__expected2:threshold</code></td></tr></tbody></table>
<p>Using <code>__metadata</code> without a key is not supported. Specify the metadata field like <code>__metadata:category</code>.
If a CSV file includes a <code>__metadata</code> column without a key, Promptfoo logs a warning and ignores the column.</p>
