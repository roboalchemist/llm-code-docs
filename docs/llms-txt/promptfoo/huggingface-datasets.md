<h3 class="anchor anchorTargetStickyNavbar_tleR" id="authentication-errors">Authentication errors<a href="#authentication-errors" class="hash-link" aria-label="Direct link to Authentication errors" title="Direct link to Authentication errors" translate="no">​</a></h3>

<h3 class="anchor anchorTargetStickyNavbar_tleR" id="dataset-not-found">Dataset not found<a href="#dataset-not-found" class="hash-link" aria-label="Direct link to Dataset not found" title="Direct link to Dataset not found" translate="no">​</a></h3>
<p>Verify the dataset path format: <code>owner/repo</code> (e.g., <code>rajpurkar/squad</code>)</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="empty-results">Empty results<a href="#empty-results" class="hash-link" aria-label="Direct link to Empty results" title="Direct link to Empty results" translate="no">​</a></h3>
<p>Check that the specified split exists for the dataset. Try <code>split=train</code> if <code>split=test</code> returns no results.</p>
<h3 class="anchor anchorTargetStickyNavbar_tleR" id="performance-issues">Performance issues<a href="#performance-issues" class="hash-link" aria-label="Direct link to Performance issues" title="Direct link to Performance issues" translate="no">​</a></h3>
<p>Add the <code>limit</code> parameter to reduce the number of rows loaded: <code>&amp;limit=100</code></p>
