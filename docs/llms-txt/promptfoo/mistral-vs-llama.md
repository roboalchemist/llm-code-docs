<h2 class="anchor anchorTargetStickyNavbar_tleR" id="run-the-comparison">Run the comparison<a href="#run-the-comparison" class="hash-link" aria-label="Direct link to Run the comparison" title="Direct link to Run the comparison" translate="no">​</a></h2>

<h2 class="anchor anchorTargetStickyNavbar_tleR" id="conclusion">Conclusion<a href="#conclusion" class="hash-link" aria-label="Direct link to Conclusion" title="Direct link to Conclusion" translate="no">​</a></h2>
<p>On this limited dataset, Mistral, Mixtral score 75%, but Llama2 scores 50%. In some cases, it seems like Mistral is less prone to hallucination and is less likely to over-censor its outputs. But these are just a handful of use cases - far from conclusive.</p>
<p>Contrast this with generic public benchmarks, which show that Llama3 &gt;&gt; Mixtral 8x7B &gt;&gt; Llama2 70B &gt; Mistral 7B &gt;&gt; Llama2 7B.</p>
