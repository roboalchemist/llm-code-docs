# Tour progression - Docs

**Product tours is in private alpha**

Product Tours is currently in private alpha. [Share your thoughts](https://app.posthog.com/external_surveys/019af5f5-a50e-0000-b10f-e8c30c0b73a0) and we'll reach out with early access.

**Currently only available on the web. Requires `posthog-js` >= v1.324.0.**

Product Tours can progress in one of two ways: **button click** or **element click**.

These options are available in the **Advance on...** section of tour step settings in both the Toolbar and PostHog's main app.

## Next button

This is the default behavior for new tour steps. We show a button that takes the user to the next step in your tour.

To learn more about customizing this button and its behavior, see [Button actions](/docs/product-tours/button-actions.md).

![Tour step with next button](https://res.cloudinary.com/dmukukwp6/image/upload/Screenshot_2026_02_05_at_9_43_34_PM_3f352cf68b.png)

## Element click

For more interactive tours, you can choose **Element click** as a step's progression trigger.

With this option we will not show a **Next** button by default, and users get to the next step by clicking on the current step's element.

![Tour step with element click](https://res.cloudinary.com/dmukukwp6/image/upload/Screenshot_2026_02_05_at_9_43_51_PM_1bc40810ca.png)
