# Configure dashboard with AI - Docs

You can use PostHog AI to find the best events to configure your Customer Analytics dashboard with.

For each event in the [configuration page](/docs/customer-analytics/configure-your-dashboard.md), there's a sparkle button right beside the event name. Clicking the button will pre-fill PostHog AI with a prompt to identify that specific event.

Each dashboard event has a different button, because there is a different prompt for each event.

![Configure dashboard events with AI](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_19_T12_57_26_738_Z_f7ba279249.png)![Configure dashboard events with AI](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_19_T12_57_52_863_Z_f27de8d77e.png)

After PostHog AI finishes the research, it will provide a list of events that can be used to configure the dashboard.

![Configure with AI chat](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_20_T11_00_46_338_Z_7efa8b9f06.png)![Configure with AI chat](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_20_T11_01_05_407_Z_52cbf52f4e.png)

To assign the event to the dashboard, copy the name of the event provided and search it in the event selector.

![Use chat event suggestion](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_19_T14_24_01_132_Z_0f27630c56.png)![Use chat event suggestion](https://res.cloudinary.com/dmukukwp6/image/upload/q_auto,f_auto/pasted_image_2025_12_19_T14_23_43_640_Z_9a7d006df5.png)
