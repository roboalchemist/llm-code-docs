# Users

The **Users** page lists the users configured to access Portainer and allows you to add new users and manage existing users.

{% hint style="info" %}
For management of user roles in Portainer Business Edition, refer to the [Roles](https://docs.portainer.io/admin/user/roles) section.
{% endhint %}
