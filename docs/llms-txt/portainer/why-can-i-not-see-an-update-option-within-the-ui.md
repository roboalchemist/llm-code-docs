# Why can I not see an update option within the UI?

If you are running any form of ad blocking, DNS filtering, or network proxy between the Portainer Server and the internet, Portainer may be unable to reach the update service. When this happens, Portainer cannot check for the latest available version, and the update option is not shown in the UI.&#x20;

Refer to the [update documentation](https://docs.portainer.io/2.33-lts/start/upgrade) for the available options on how to update Portainer in your environment.
