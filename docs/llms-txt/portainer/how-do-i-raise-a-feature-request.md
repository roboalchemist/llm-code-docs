# How do I raise a feature request?

You can request new features in this [GitHub](https://github.com/portainer/portainer/issues/new?assignees=\&labels=\&template=Feature_request.md\&title=) repo. When you click through, our template makes it easy to record all of the details. We also publish a list of [open](https://github.com/portainer/portainer/issues?q=is%3Aissue+is%3Aopen+label%3Abug%2Fconfirmed) feature requests in GitHub. Check to see if someone has already requested the feature you want, and give it a thumbs up.

Learn how we prioritize feature development [here](https://docs.portainer.io/faqs/contributing/how-do-you-decide-which-bugs-and-features-to-work-on-first).
