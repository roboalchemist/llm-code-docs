# API documentation

Portainer exposes an HTTP API that you can use to automate everything you do via the Portainer UI. You can also use Portainer as a gateway (HTTP queries against the Portainer API) to the underlying Docker/Kubernetes API.

{% hint style="info" %}
You will need an access token in order to use the Portainer API. If you have not already set up an access token for the API, we have [instructions on how to do so](https://docs.portainer.io/api/access).
{% endhint %}

You can find our API documentation here:

* [Business Edition (BE) 2.39.0 API Documentation](https://api-docs.portainer.io/?edition=ee\&version=2.39.0)&#x20;
* [Community Edition (CE) 2.39.0 API Documentation](https://api-docs.portainer.io/?edition=ce\&version=2.39.0)
