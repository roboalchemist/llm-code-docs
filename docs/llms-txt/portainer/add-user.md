# Add a user to a team

From the menu expand **User-related** then select **Teams**.&#x20;

<figure><img src="https://content.gitbook.com/content/xdTQRpMuktD2l0URtOJO/blobs/VWNa71HazRsWm5oHsHej/Add-team-new.gif" alt=""><figcaption></figcaption></figure>

Select the team you want to add users to then click **Add** next to the user.
