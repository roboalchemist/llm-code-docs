# Policies

To view any policies that apply to the selected environment, from the menu expand **Host** then select **Policies**.&#x20;

<figure><img src="https://2837857291-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FxdTQRpMuktD2l0URtOJO%2Fuploads%2FHlelejMSzALGlKc7iQfM%2F2.39-docker-policies-1.gif?alt=media&#x26;token=4b405574-e012-4ea7-b2ce-cffed34a4039" alt=""><figcaption></figcaption></figure>

Policy details from this view are read-only. As an admin user, you can select the **Edit** button next to the policy name to manage the policy. See the [Policies](https://docs.portainer.io/admin/environments/policies) documentation for details.
