# Namespaces

{% hint style="info" %}
Namespaces used to be called 'resource pools' prior to Portainer CE 2.6.0.
{% endhint %}

In Kubernetes, a single physical cluster can support multiple virtual clusters. These are known as namespaces.&#x20;
