# Reset a user's password

From the menu expand **User-related** then select **Users**.&#x20;

<figure><img src="https://content.gitbook.com/content/xdTQRpMuktD2l0URtOJO/blobs/KPcKF5qEYDALdhHAMFe3/Add-user-new.gif" alt=""><figcaption></figcaption></figure>

Click the username of the user whose password you want to reset. In the **Change user password** section, enter a new strong password, re-enter the password to confirm it then click **Update password**.

<figure><img src="https://content.gitbook.com/content/xdTQRpMuktD2l0URtOJO/blobs/Edmf6BKL39Qqc98H0Yf3/2.15-settings-users-changepw.png" alt=""><figcaption></figcaption></figure>
