### Frequently Asked Questions

Have another question? Join our community on GitHub

1. **What are Claude Agent Skills?**
   - Agent Skills are modular capabilities that extend Claude's functionality through filesystem-based commands. They use progressive disclosure architecture (metadata → instructions → resources) and work across Claude API, Claude.ai, and Claude Code.

2. **How do I install a Skill?**
   - Download the SKILL.md file from our marketplace. On Claude Code, use `claude code skills add path/to/SKILL.md`. On Claude.ai, upload via the Skills interface. Full instructions in our Getting Started guide.

3. **Can I create my own Skills?**
   - Yes! Our 'Create Your First Skill' tutorial walks you through the complete process. Learn progressive disclosure patterns, evaluation strategies, and best practices from real examples.

4. **Are these Skills production-ready?**
   - All Skills in our marketplace are tested and documented. We only feature Skills that have been used in real workflows. Each includes complete documentation, evaluation criteria, and usage examples.
