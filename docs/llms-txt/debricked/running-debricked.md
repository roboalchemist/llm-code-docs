# Running OpenText Core SCA

You can run OpenText Core SCA in any of the following ways:

* **Web**: [The OpenText Core SCA web app](https://debricked.com/app/en/login) enables a browser-based experience, providing all functions, including security and license scanning, automation, reporting, and more.
* **CLI**: [The OpenText Core SCA Command Line Interface](https://docs.debricked.com/tools-and-integrations/cli/debricked-cli) enhances open-source security and license compliance for your project through the command prompt. It allows you to conduct vulnerability scans on your local machine and seamlessly integrate OpenText Core SCA into your pipeline.
* **API**: [The OpenText Core SCA APIs](https://docs.debricked.com/tools-and-integrations/debricked-apis) allow you to integrate our service into your code, CI pipelines, and more.
* **Browser**: [The OpenText Core SCA Select Browser Extension](https://docs.debricked.com/product/open-source-select/debricked-select-browser-extension) integrates the **Select** option directly into your browser, allowing you to examine an open-source project's health metrics and license information without leaving the page.
