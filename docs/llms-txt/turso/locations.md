# db locations

You can fetch a list of supported locations where databases can be located by running the following command:

```bash  theme={null}
turso db locations
```

## Flags

| Flag                     | Description                                                               |
| ------------------------ | ------------------------------------------------------------------------- |
| `-l`, `--show-latencies` | Display latencies from your current location to each of Turso's locations |
