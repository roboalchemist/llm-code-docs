# Audit Log

## Overview

Audit Logging, available within Settings, displays a log of internal user activities within your organization.  This allows you to keep track of exactly which sessions specific employees have been viewing.  Please note that this feature is only available on the Professional plan.

Only Admin users will have the ability to view the logs and enable the setting.
