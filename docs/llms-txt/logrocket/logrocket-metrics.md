## Metric Configuration Options

### Time Range, Intervals, and Comparison Time Ranges

The time range, interval, and comparison time range are saved with the metric. This allows you to consistently view a particular metric for either a specific date range (i.e Feb 2nd - Feb 4th) or relative date range (i.e last week.)

When arriving at a metric from a dashboard, the dashboard time range will be shown in the metric. From here, you can either save the new time information, or reset the metric to return to the saved time range of the chart.
