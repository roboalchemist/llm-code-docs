# BugHerd

Integrating LogRocket with BugHerd

If you are using [BugHerd](https://bugherd.com/) to collect website feedback, you can view LogRocket user sessions directly from your BugHerd tasks, providing a detailed look at user behavior, and helping to identify and fix website issues more efficiently based on the feedback collected using BugHerd.

<Image align="center" width="50% " src="https://files.readme.io/3d83913-unnamed_1.png" />

<br />

There's no installation needed to use the BugHerd and LogRocket integration. If you have LogRocket and BugHerd installed on your website or web application, BugHerd will automatically detect your LogRocket session, connect the two tools, and generate the "View session" link for you to view.
