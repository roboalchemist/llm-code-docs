# Tables

## Overview

Tables allow you to view aggregate data elements for various pieces of information.

You can use these charts to answer questions such as *"What is the session breakdown per browser?"* and *"What are the most commonly rage clicked elements in my app?"*

To use these, simply select "Table" followed by the metric that you are interested in measuring.

<Image align="center" className="border" border={true} src="https://files.readme.io/4b5fcfeaac4dc9eed4828db07442ce47e959c9e0baec9c92aa57da733c992d96-Tables_Overview_1.png" />

You can layer filters on top of the tables as well, for more granular insights.

After the table is saved, you'll see a list of different types on the lefthand side with the session count on the righthand side:

<Image align="center" src="https://files.readme.io/451bfdc9acbdc057f84638a5ceef6287bbef79ea786344b2f7f75d6ecfb3a65c-Tables_Overview_2.png" />

You can click each type on the left to filter the sessions displayed in the list on the right.

## Donut Chart View

For most table types (excluding Network Request Durations, Initial Page Load Durations, and Issues), we support a Donut Chart view of the table.

This is automatically available within the full detail view for supported tables:

<Image align="center" src="https://files.readme.io/972f5dbc741087b8e65599f9a9a60c6e5974be32385109afc460829ab4dc6a70-Donut_Chart_View_1.png" />

You'll see the donut chart at the top of the page, representing the top 20 elements in the table if the table contains more.  You can hover over each section of the chart to see some additional data.  Each section is also clickable, which will filter the session list on the bottom right to the relevant sessions.

You can also choose to view certain table as a Donut Chart within a dashboard view.  To toggle this, go to the Edit menu in the top righthand corner of the chart and click "View as donut chart".

<Image align="center" src="https://files.readme.io/08e90f05a040c329118432795d1588a001f899e2aaa28d89b3d344e41b7128a8-Donut_Chart_View_2.png" />

You'll see the same data represented as a donut chart, with mouseovers giving you detailed data on each section of the chart.

<Image align="center" src="https://files.readme.io/64eca595fbba2b275680d4a6a2e706e598b00212076eceb42a543c06830d2506-Donut_Chart_View_3.png" />
