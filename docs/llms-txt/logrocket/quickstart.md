# Quickstart

Getting started is easy:

1. Visit [https://app.logrocket.com](https://app.logrocket.com) to get an app ID.
2. Install LogRocket via NPM or script tag. `LogRocket.init()` must be called client-side, not server-side.

```javascript NPM
$ npm i --save logrocket

// Code:
import LogRocket from 'logrocket';
LogRocket.init(YOUR_APP_ID);
```

```html Script Tag
Add to your HTML:

<script src="https://cdn.logr-in.com/LogRocket.min.js" crossorigin="anonymous"></script>
<script>window.LogRocket && window.LogRocket.init(YOUR_APP_ID);</script>
```

3. (Optional) Install plugins for deeper integrations with your stack:
   1. [Redux middleware](https://docs.logrocket.com/v3.0/reference/redux-middleware-1)
   2. [ngrx middleware](https://docs.logrocket.com/v3.0/reference/ngrx-middleware-1)
   3. [Vuex plugin](https://docs.logrocket.com/v3.0/reference/vuex)
