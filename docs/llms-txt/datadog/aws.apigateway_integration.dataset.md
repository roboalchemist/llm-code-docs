# API Gateway Integration

API Gateway Integration in AWS defines how an API method connects to a backend endpoint or service. It specifies the type of integration, such as HTTP, AWS service, Lambda function, or mock, and manages request/response transformations, timeouts, and credentials. This resource is essential for linking API Gateway methods to the actual business logic or data sources.

```
aws.apigateway_integration
```
