# Ad Domain

This table represents the ad_domain resource from Microsoft Azure.

```
azure.ad_domain
```

| _key                                 | core | string        |
| authentication_type                  | core | string        | Indicates the configured authentication type for the domain. The value is either Managed or Federated.
