# Transcribe Call Analytics Job

Transcribe Call Analytics Job in AWS is a resource that provides details about an analytics job run by Amazon Transcribe. It processes call audio to generate insights such as speaker separation, sentiment, and conversation characteristics. The job resource includes information like job status, input and output data locations, and analysis results.

```
aws.transcribe_call_analytics_job
```
