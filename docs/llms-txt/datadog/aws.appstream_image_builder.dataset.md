# AppStream 2.0 Image Builder

AppStream 2.0 Image Builder is a managed AWS resource that provides a virtual machine environment for creating and customizing images used in Amazon AppStream 2.0. It allows you to install applications, configure settings, and optimize the desktop experience before publishing the image for use in streaming fleets.

```
aws.appstream_image_builder
```
