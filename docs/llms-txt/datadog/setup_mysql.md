| MySQL 5.6     | yes        | yes           | yes                             | yes   |
| MySQL 5.7     | yes        | yes           | yes                             | yes   | yes |

| MariaDB 10.6  | yes        | yes           |
| MariaDB 10.11 | yes        | yes           |
| MariaDB 11.4  | yes        |
