# Ensure that logging for Azure Key Vault is Enabled
 
## Description{% #description %}

This rule checks if Azure Key Vault has a diagnostic setting enabled. Diagnostic settings allow you to send logs and metrics to Azure Monitor, Azure Storage, or Azure Event Hubs.

## Remediation{% #remediation %}

To enable diagnostic settings for Azure Key Vault, see [Enable Key Vault logging](https://learn.microsoft.com/en-us/azure/key-vault/general/howto-logging).
