# Certificate Template

A Certificate Template in Google Cloud is a reusable configuration that defines parameters and policies for issuing X.509 certificates through Certificate Authority Service. It specifies details such as key usage, identity constraints, lifetime, and allowed issuers, ensuring consistent and secure certificate issuance across projects and environments.

```
gcp.privateca_certificate_template
```
