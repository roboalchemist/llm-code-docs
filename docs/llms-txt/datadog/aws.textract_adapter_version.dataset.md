# Textract Adapter Version

Textract Adapter Version in AWS represents the response structure for retrieving details about a specific adapter version in Amazon Textract. It provides metadata and configuration information related to the adapter version, enabling users to manage and track different iterations of Textract adapters used for document analysis workflows.

```
aws.textract_adapter_version
```
