# Compute Autoscaler

Compute Autoscaler in Google Cloud automatically adjusts the number of virtual machine instances in a managed instance group based on load or custom metrics. It helps ensure applications have the right amount of resources to handle traffic while minimizing costs by scaling down when demand is low.

```
gcp.compute_autoscaler
```
