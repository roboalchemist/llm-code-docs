# Apigee Organization

Apigee Organization in Google Cloud is the top-level container for managing Apigee API management resources. It defines the administrative boundary for environments, API proxies, developers, and analytics data. Each organization is associated with a specific Google Cloud project and region, providing centralized control over API lifecycle, security, and traffic management.

```
gcp.apigee_organization
```
