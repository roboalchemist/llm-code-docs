# Transcribe Medical Scribe Job

Transcribe Medical Scribe Job in AWS is a resource that provides details about a medical transcription job created through Amazon Transcribe Medical. It returns information such as the job status, transcription output location, and metadata related to the medical scribe process. This helps healthcare providers and applications retrieve and manage automated medical transcription results securely and at scale.

```
aws.transcribe_medical_scribe_job
```
