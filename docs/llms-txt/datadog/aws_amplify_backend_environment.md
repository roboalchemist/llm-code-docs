**Type**: `STRING`

## `backend_environment_arn`{% #backend_environment_arn %}

**Type**: `STRING`**Provider name**: `backendEnvironmentArn`**Description**: The Amazon Resource Name (ARN) for a backend environment that is part of an Amplify app.

## `create_time`{% #create_time %}

**Type**: `TIMESTAMP`**Provider name**: `createTime`**Description**: The creation date and time for a backend environment that is part of an Amplify app.

## `deployment_artifacts`{% #deployment_artifacts %}

**Type**: `STRING`**Provider name**: `deploymentArtifacts`**Description**: The name of deployment artifacts.

## `environment_name`{% #environment_name %}

**Type**: `STRING`**Provider name**: `environmentName`**Description**: The name for a backend environment that is part of an Amplify app.

## `stack_name`{% #stack_name %}

**Type**: `STRING`**Provider name**: `stackName`**Description**: The AWS CloudFormation stack name of a backend environment.

## `tags`{% #tags %}

**Type**: `UNORDERED_LIST_STRING`

## `update_time`{% #update_time %}

**Type**: `TIMESTAMP`**Provider name**: `updateTime`**Description**: The last updated date and time for a backend environment that is part of an Amplify app.
