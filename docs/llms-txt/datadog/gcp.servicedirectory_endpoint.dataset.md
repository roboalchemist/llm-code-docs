# Service Directory Endpoint

A Service Directory Endpoint in Google Cloud represents a network endpoint that provides a specific service instance within a namespace. It contains details such as IP address, port, and metadata, allowing clients to discover and connect to services dynamically. This helps manage service discovery across environments and simplifies communication between distributed applications.

```
gcp.servicedirectory_endpoint
```
