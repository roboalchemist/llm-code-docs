# Sophos Central Cloud alert

{% alert level="danger" %}
This rule is part of a beta feature. To learn more, [contact Support](https://docs.datadoghq.com/help/).
{% /alert %}
Classification:attackTactic:[TA0002-execution](https://attack.mitre.org/tactics/TA0002)Technique:[T1059-command-and-scripting-interpreter](https://attack.mitre.org/techniques/T1059)
## Goal{% #goal %}

Monitor and respond to alerts generated by Sophos Central Cloud to maintain security posture.

## Strategy{% #strategy %}

Trigger notifications for any alerts issued by Sophos Central Cloud.

## Triage and response{% #triage-and-response %}

1. Assess the alert details, focusing on the severity level `{{@log_message.severity}}`.
1. Follow the recommended course of action as described in `{{@log_message.description}}`.
1. Adhere to organizational protocols to effectively address and mitigate the alert.
