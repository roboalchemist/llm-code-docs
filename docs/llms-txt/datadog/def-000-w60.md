# Default to Microsoft Entra authorization in the Azure portal should be enabled

## Description{% #description %}

Enable `Default to Microsoft Entra authorization` in the Azure portal for Storage Accounts to improve security by using OAuth authentication instead of shared keys.

## Remediation{% #remediation %}

Enable default to Microsoft Entra authorization in the Storage Account Configuration settings. See [Azure Storage Account OAuth authentication](https://learn.microsoft.com/en-us/azure/storage/common/authorize-data-access) for more information.
