## `account_id`{% #account_id %}

**Type**: `STRING`

## `branch_name`{% #branch_name %}

**Type**: `STRING`**Provider name**: `branchName`**Description**: The name for a branch that is part of an Amplify app.

## `create_time`{% #create_time %}

**Type**: `TIMESTAMP`**Provider name**: `createTime`**Description**: A timestamp of when Amplify created the webhook in your Git repository.

## `description`{% #description %}

**Type**: `STRING`**Provider name**: `description`**Description**: The description for a webhook.

## `tags`{% #tags %}

**Type**: `UNORDERED_LIST_STRING`

## `update_time`{% #update_time %}

**Type**: `TIMESTAMP`**Provider name**: `updateTime`**Description**: A timestamp of when Amplify updated the webhook in your Git repository.

## `webhook_arn`{% #webhook_arn %}

**Type**: `STRING`**Provider name**: `webhookArn`**Description**: The Amazon Resource Name (ARN) for the webhook.

## `webhook_id`{% #webhook_id %}

**Type**: `STRING`**Provider name**: `webhookId`**Description**: The ID of the webhook.

## `webhook_url`{% #webhook_url %}

**Type**: `STRING`**Provider name**: `webhookUrl`**Description**: The URL of the webhook.
