# OpenSearch domains should have the latest software update installed

## Description{% #description %}

This control checks whether an Amazon OpenSearch Service domain has the latest software update installed.

## Remediation{% #remediation %}

To identify and install the latest software updates for your Amazon OpenSearch Service domains, refer to the [Updating an Amazon OpenSearch Service Domain](https://docs.aws.amazon.com/opensearch-service/latest/developerguide/upgrade-domains.html) section of the Amazon OpenSearch Service Developer Guide.
