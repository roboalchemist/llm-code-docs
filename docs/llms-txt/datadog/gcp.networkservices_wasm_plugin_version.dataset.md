# Wasm Plugin Version

Wasm Plugin Version in Google Cloud represents a specific version of a WebAssembly (Wasm) plugin used within GCP services such as API Gateway or Envoy-based proxies. It defines the versioned configuration and executable code for extending service behavior using Wasm modules, enabling custom logic, filters, or policies to run securely and efficiently within the GCP environment.

```
gcp.networkservices_wasm_plugin_version
```
