# CodeBuild Project

An AWS CodeBuild Project is a configuration that defines how source code is built and tested in a fully managed build service. It specifies details such as the source repository, build environment, runtime settings, and output artifacts. Each project runs builds in isolated containers, scales automatically, and integrates with other AWS services like CodePipeline, S3, and CloudWatch.

```
aws.codebuild_project
```
