# Cloud Asset Feed

Cloud Asset Feed in Google Cloud lets you receive real-time updates about changes to your cloud resources and policies. It continuously streams asset metadata such as creation, deletion, or modification events to a specified destination like Pub/Sub. This helps maintain up-to-date visibility, automate compliance checks, and trigger workflows based on asset changes.

```
gcp.cloudasset_feed
```
