# Azure should use the latest Java version available

## Description{% #description %}

New versions of Java software are periodically released to address security vulnerabilities and introduce new functionalities. To ensure optimal security and take advantage of the latest features, it's recommended to keep web apps updated with the latest Java version. Organizations should assess each update for compatibility and ensure it integrates smoothly with other software components.

Current version: OpenJDK 21 LTS

## Remediation{% #remediation %}

For guidance on updating your Azure Web App to the latest version of Java, please visit the [Azure Release and Servicing Roadmap for Java](https://learn.microsoft.com/en-us/java/openjdk/support#release-and-servicing-roadmap).
