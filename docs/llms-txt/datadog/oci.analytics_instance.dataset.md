# Analytics Instance

An Analytics Instance in Oracle Cloud Infrastructure is a managed service that provides a scalable environment for running Oracle Analytics Cloud. It enables users to prepare, analyze, and visualize data from various sources without managing underlying infrastructure. The service supports advanced analytics, machine learning integration, and interactive dashboards, making it easier for organizations to gain insights and share data-driven results securely.

```
oci.analytics_instance
```
