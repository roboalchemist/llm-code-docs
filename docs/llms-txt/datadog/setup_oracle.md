# Setting up Oracle

| Oracle 11g  | yes | yes | yes     | yes                 | yes                          | yes |
| Oracle 12c  | yes | yes | yes     | yes                 | yes                          | yes |

| Oracle 19c  | yes | yes | yes     | yes                 | yes                          | yes |
| Oracle 21c  | yes | yes | yes     | yes                 | yes                          | yes |
