# Setting up Amazon DocumentDB

### Amazon DocumentDB versions supported{% #amazon-documentdb-versions-supported %}

| Instance-based cluster |
| ---------------------- |
| 4.0.0                  | yes |
| 5.0.0                  | yes |

**Note**: Elastic clusters are not supported for these versions. Amazon DocumentDB version 3.6.0 is also not supported.

For setup instructions, select your hosting type:

- [Amazon DocumentDB](https://docs.datadoghq.com/database_monitoring/setup_documentdb/amazon_documentdb)
