# FSx File System

FSx File System is a fully managed AWS service that provides high-performance file storage built on popular file systems like Windows File Server, Lustre, NetApp ONTAP, and OpenZFS. It is designed for workloads that require fast, scalable, and feature-rich file storage, such as enterprise applications, big data analytics, machine learning, and media processing. FSx handles infrastructure management tasks like hardware provisioning, patching, and backups, allowing users to focus on their applications while benefiting from seamless integration with other AWS services and on-premises environments.

```
aws.fsx_file_system
```
