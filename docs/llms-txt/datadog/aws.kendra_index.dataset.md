# Amazon Kendra Index

Amazon Kendra Index is a managed search service resource that stores and organizes indexed data for intelligent search. It enables users to ingest documents from multiple data sources, apply access controls, and use machine learning to deliver accurate, context-aware search results across enterprise content.

```
aws.kendra_index
```
