# Azure Synapse Analytics Workspace

Azure Synapse Analytics Workspace is a cloud-based analytics service that brings together big data and data warehousing. It provides a unified environment to ingest, prepare, manage, and serve data for business intelligence and machine learning. The workspace integrates with various data sources, supports on-demand or provisioned query models, and offers built-in security and monitoring. It enables teams to analyze large volumes of data efficiently and collaborate across different analytics workloads.

```
azure.synapse_workspace
```
