# Comprehend PII Entities Detection Job Properties

Comprehend PII Entities Detection Job Properties in AWS describe the details of a job that identifies and processes personally identifiable information (PII) within text. It includes information such as job status, input and output data configuration, language code, job name, and timestamps. This resource helps track and manage PII detection jobs, ensuring sensitive data is recognized and handled appropriately.

```
aws.comprehend_pii_entities_detection_job
```
