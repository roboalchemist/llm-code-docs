# ECS Task Definition

An ECS Task Definition in AWS is a blueprint that describes how Docker containers should run within Amazon ECS. It specifies details such as container images, CPU and memory requirements, networking, logging, and IAM roles. Task definitions can include multiple containers and define how they interact, making them the foundation for running tasks and services in ECS.

```
aws.ecs_task_definition
```
