# ElastiCache Reserved Cache Node

An ElastiCache Reserved Cache Node is a billing option in Amazon ElastiCache that allows you to reserve a specific cache node type for a one- or three-year term. By committing to a reserved node, you receive significant cost savings compared to on-demand pricing. This option is ideal for predictable, steady-state workloads where cache usage is consistent over time.

```
aws.elasticache_reserved_cache_node
```
