# Target gRPC Proxy

A Target gRPC Proxy in Google Cloud is a load balancing resource that routes incoming gRPC requests to backend services. It works with global external load balancers to distribute traffic across multiple backend instances, ensuring high availability and efficient request handling for gRPC-based applications.

```
gcp.compute_target_grpc_proxy
```
