# Route 53 Resolver Endpoint

Route 53 Resolver Endpoint is an AWS resource that enables hybrid cloud DNS resolution between on-premises networks and Amazon VPCs. It allows you to create inbound endpoints to forward DNS queries from on-premises to Route 53 Resolver, and outbound endpoints to forward queries from VPCs to external DNS servers. This helps integrate DNS resolution across environments securely and at scale.

```
aws.route53resolver_resolver_endpoint
```
