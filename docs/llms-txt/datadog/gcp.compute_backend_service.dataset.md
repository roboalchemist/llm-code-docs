# Backend Service

A Backend Service in Google Cloud is a configuration resource that defines how incoming traffic is distributed to backend instances or groups. It specifies load balancing settings such as health checks, session affinity, and connection draining. Backend Services are used with HTTP(S), TCP, and SSL proxy load balancers to ensure reliable and efficient traffic routing to applications running on Compute Engine or other backends.

```
gcp.compute_backend_service
```
