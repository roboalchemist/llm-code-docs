# Public network access should be disabled for Azure Storage Accounts
 
## Description{% #description %}

Disable public network access for Azure Storage Accounts to improve security.

## Remediation{% #remediation %}

Disable public network access in the Storage Account Networking settings. See [Azure Storage account public network access](https://learn.microsoft.com/en-us/azure/storage/common/storage-network-security?tabs=azure-portal#public-network-access) for more information.
