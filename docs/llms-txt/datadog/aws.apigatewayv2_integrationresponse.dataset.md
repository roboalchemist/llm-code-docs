# API Gateway V2 Integration Response

API Gateway V2 Integration Response in AWS defines how the API Gateway should handle the response from an integration, such as a Lambda function or HTTP backend. It specifies the mapping of integration response parameters to method responses, including status codes, headers, and payload transformations. This allows developers to control how backend responses are returned to clients through the API.

```
aws.apigatewayv2_integrationresponse
```
