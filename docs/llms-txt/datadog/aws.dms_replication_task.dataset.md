# DMS Replication Task

An AWS DMS Replication Task defines the actual migration or replication job that moves data between a source and target database. It specifies the type of migration, such as full load, ongoing replication, or both, and includes settings for table mappings, transformations, and task behavior. This resource is central to controlling how data is transferred and kept in sync during database migrations.

```
aws.dms_replication_task
```
