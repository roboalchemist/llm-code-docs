# AppSync GraphQL APIs should have field-level logging enabled
 
## Description{% #description %}

Enable field-level logging for your AWS AppSync GraphQL APIs

By enabling field-level logging for your AWS AppSync GraphQL APIs, you gain detailed insights into the operations and security of your application, which is essential for maintaining a robust, secure, and efficient system.

## Remediation{% #remediation %}

Follow the [Using CloudWatch to monitor and log GraphQL API data](https://docs.aws.amazon.com/appsync/latest/devguide/monitoring.html) documentation to learn how to enable field-level logging from the AWS Management Console.
