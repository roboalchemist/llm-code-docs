# Transcribe Medical Transcription Job

Transcribe Medical Transcription Job in AWS provides details about a medical transcription job created with Amazon Transcribe Medical. It returns information such as the job status, transcription output location, and metadata about the audio input. This resource is used to track and retrieve results of medical speech-to-text processing, enabling healthcare applications to convert clinical conversations into structured text securely and accurately.

```
aws.transcribe_medical_transcription_job
```
