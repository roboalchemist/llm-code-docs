# DataSync tasks should have logging enabled
 
## Description{% #description %}

DataSync tasks should have logging enabled. CloudWatch logging provides visibility into data transfer operations, including files transferred and any errors encountered, for troubleshooting and compliance auditing.

## Remediation{% #remediation %}

Enable CloudWatch logging for your DataSync task. Consult the [AWS Documentation](https://docs.aws.amazon.com/datasync/latest/userguide/configure-logging.html) for specific enablement instructions.
