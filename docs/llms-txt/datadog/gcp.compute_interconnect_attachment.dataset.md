# Interconnect Attachment

An Interconnect Attachment in Google Cloud is a regional resource that connects your Virtual Private Cloud (VPC) network to an on-premises network through Dedicated Interconnect or Partner Interconnect. It acts as the logical connection point where traffic enters or exits your VPC over the interconnect, enabling high-bandwidth, low-latency hybrid connectivity.

```
gcp.compute_interconnect_attachment
```
