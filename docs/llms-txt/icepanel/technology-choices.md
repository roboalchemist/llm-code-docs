# Technology choices

IcePanel allows you to attach technology choices to objects and connections so your audience can quickly see everything from service providers to languages, frameworks, gateways, and more. Add tech choices from our existing catalogue, or upload a custom technology.

{% hint style="info" %}
Private technology choices are only available on paid plans.
{% endhint %}

## Adding an existing technology

To add a new technology:

1. Click on the `Icon` button on the top-left of the right-hand details panel OR the `+ Add technology` button lower in the panel.
2. Search for the technology and add it.

Technologies added will contain a link to the website, description, docs URL (optional), and an updates URL (optional).

To remove a technology, navigate the technology section in the details panel, click on the `...` menu option, and click on `Remove`.

### Setting the technology icon

Each object can have 1 technology icon added that will appear in the diagram and exports. This adds a quick visual indicator for viewers to understand the different parts of your diagram. If an object has no icon, the first technology added will automatically be set as the icon. This can be changed as you add more technologies.

To change the technology icon:

1. Click on the object or connection.
2. Click the icon button on the top-left of the details panel OR the technology property lower in the panel.
3. Click on the `...`menu button and select `Select as icon` .

## Adding a new technology&#x20;

You can submit technologies not in the catalogue for private or public use.

{% hint style="info" %}
Public submissions must be approved by IcePanel, but you'll still be able to use them privately.
{% endhint %}

To submit a new technology:

1. Search for the technology in the right-hand details panel.
2. Enter the URL of the technology and click on `Add technology`.  Alternatively, click on `No website for this technnology`if there's no website.
3. Follow the steps to add the new technology. IcePanel uses a combination of AI and [Brandfetch](https://brandfetch.com/) to pull in the relevant information about the technology, saving you time from manual entry.
4. In the final step, you can choose whether to use the technology for private (only within your organization) or public use.

## Viewing your technologies

Navigate to the technologies tab from the home screen to see a breakdown of all the different technologies you use in the landscape or domain. The tabs are ordered by the technologies assigned to the most objects and link to a list of the model objects.

## Managing your technologies

{% hint style="info" %}
Only admins can review and manage technology submissions.
{% endhint %}

Admins in the org can manage internal technologies and submissions in `Manage organization` .

From the technologies section, admins can:

* Edit technologies (name, description, URL links, icons).
* Set private technologies to the public (requires review by IcePanel).
* Delete technologies.
* Add new technologies.
