# Get project info

> Retrieve project configuration and environment details

## OpenAPI

````yaml get /v1/connect/{project_id}/projects/info

  title: Pipedream REST API

  - url: https://api.pipedream.com

  /v1/connect/{project_id}/projects/info:
    parameters:
      - name: project_id

        required: true

          type: string
          pattern: ^proj_[a-zA-Z0-9]+$
````
