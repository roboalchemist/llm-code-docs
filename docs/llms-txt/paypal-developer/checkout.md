# PayPal Checkout Integration | PayPal Developer

## Overview

Use the [JavaScript SDK](/sdk/js/) to present payment buttons to your buyers so they can pay with PayPal, debit and credit cards, Pay Later options, Venmo, and alternative payment methods.

See PayPal's [Custom Integration](https://www.paypal.com/us/business/accept-payments/checkout/integration#standard-checkout) page for more information about capabilities, pricing, and other merchant resources.

You can get started quickly with this 15-minute copy-and-paste integration.

[Read the guide](/docs/checkout/standard/)
