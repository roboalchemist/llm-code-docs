# Review transactions

Transactions that are flagged for review will go into a single queue. The **Review** tab displays the filters that flagged the transaction, the time that the transaction occurred, and the risk score.

![Review,tab](https://www.paypalobjects.com/devdoc/Review_View.png)

## Review a transaction

You can review transactions using the following steps:

1. From the **Review** tab, select the **Transaction ID** of the transaction you'd like to review.
2. Review all the details and select **Submit** after making a decision.
   ![image](https://www.paypalobjects.com/devdoc/Review_Single_Case_View.png)

**Note**: You have 30 days to review a transaction. It will only be available for review during this period.

To view a detailed report of attempted and declined transactions, see [Attempts and Declines Report User Guide](/docs/reports/attempts-declines/).
