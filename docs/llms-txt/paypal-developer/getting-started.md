# Getting started with FPA

The Getting Started guide offers a comprehensive overview of onboarding with Fraud Protection Advanced (FPA). It includes the following scenarios:

## Activating FPA for direct merchants

This flow is for existing merchants who have not signed up for FPA. Follow the [guide](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/getting-started/activate/) to complete the process.

## Upgrading to FPA

This flow applies to existing merchants looking to upgrade from the Fraud Protection Standard tool to the Fraud Protection Advanced tool. Follow the [guide](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/getting-started/upgrade/) to complete the process.

---

After onboarding to FPA, set up FPA for your account.

For more information on how you can set up FPA, see the following:

- [Create and set up filters](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/filters/)
- [Manage lists](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/lists/)
- [Review transactions](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/review/)
- [Monitor FPA activity](https://developer.paypal.com/docs/checkout/advanced/customize/fraud-protection/fraud-protection-advanced/monitoring/)
