# Add package tracking to orders

## API

**Current**

Last updated: May 13th, 11:13am

There are 2 ways to add package tracking to orders. The integration method you use depends on your checkout integration:

- If you are using the Orders v2 API, integrate package tracking directly through the [Orders v2 API](/docs/tracking/orders-api/).
- If you have a legacy integration that doesn't use the Orders v2 API, integrate package tracking through the [Add Shipping Tracking API](/docs/tracking/tracking-api/).
