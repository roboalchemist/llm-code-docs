# Monitor FPA Activity

The **Activity** tab logs all updates made to the tool, including changes to filters, lists, and case reviews. This feature allows you to see which users made changes and when they occurred.

![Image 1](https://www.paypalobjects.com/devdoc/Monitor_FPA_Activity2.png)

Select the expand icon to see the details of any item.

![Image 2](https://www.paypalobjects.com/devdoc/Monitor_FPA_Activity.png)

You can also select **Advanced Search** to search for specific audit changes.

![Image 3](https://www.paypalobjects.com/devdoc/Monitor_Activity_AdvancedSearch.png)
