# Eligibility

This page covers the countries, currencies, card brands, and payment methods that Expanded Checkout supports.

**info**
If you have an Express Checkout or PayPal Checkout integration, PayPal recommends upgrading to Expanded Checkout using the [JavaScript SDK](/sdk/js/configuration/). Visit [Upgrade your Checkout integration](/docs/checkout/advanced/upgrade/) for more information.

## Eligibility
