###### Note

This Robot only accepts videos.

###### Warning

Vimeo's API limits the number of concurrent uploads per minute based on your Vimeo account plan. To see how many videos can be uploaded at once based on your plan, click the following [link](https://developer.vimeo.com/guidelines/rate-limiting#table-1).

## Installation

Since Vimeo works with OAuth, you will need to generate [Template Credentials](/c/template-credentials/) to use this Robot.

To change the `title` or `description` per video, we recommend to [inject variables into your Template](/docs/topics/templates.md).
