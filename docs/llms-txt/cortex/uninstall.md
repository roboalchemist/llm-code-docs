# Uninstall

## Uninstall (if installed with pip)

```bash
pip uninstall cortex
rm -rf ~/.cortex
```

## Uninstall (if installed without pip)

```bash
rm /usr/local/bin/cortex
rm -rf ~/.cortex
```
