<p>Our verification API allows you to create Custom Integrations to add email verification to any part of your software. We offer solutions for verifying individual emails as well as lists containing thousands or even millions of emails.</p>
<h2 class="heading heading-2 header-scroll"><div class="heading-anchor anchor waypoint" id="first-steps"></div><div class="heading-text">First Steps</div><a aria-label="Skip link to First Steps" class="heading-anchor-icon fa fa-regular fa-anchor" href="#first-steps"></a></h2>
<p>Starting with Version 4 of our API, accounts can now have multiple unique API keys. These API keys are attached to Custom Integrations, allowing advanced reporting and advanced settings.</p>
<p>To get started, log into your account and create a new <a href="https://app.neverbounce.com/apps/custom-integration/new">Custom Integration App</a>. After creation, you&#x27;ll be taken to the newly created app&#x27;s overview page, where you&#x27;ll find your app&#x27;s API key. This key will be used to authenticate your requests to our API.</p>
<h2 class="heading heading-2 header-scroll"><div class="heading-anchor anchor waypoint" id="not-a-developer"></div><div class="heading-text">Not a developer?</div><a aria-label="Skip link to Not a developer?" class="heading-anchor-icon fa fa-regular fa-anchor" href="#not-a-developer"></a></h2>
