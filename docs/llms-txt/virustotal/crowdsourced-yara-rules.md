# Crowdsourced YARA Rules

If a sample matched any of our open source community YARA rules , you will see the following section on the [file report](https://www.virustotal.com/gui/file/275a021bbfb6489e54d471899f7db9d1663fc695ec2fe2a2c4538aabf651fd0f/detection):

[![Crowdsourced Yara report](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/crowdsourcedyara_report_20231108.png)](https://www.virustotal.com/gui/file/275a021bbfb6489e54d471899f7db9d1663fc695ec2fe2a2c4538aabf651fd0f/detection)

1. Rule name.
2. Author of the Ruleset.
3. Ruleset name.
4. Github link.
5. Rule description.
6. You can click to view the Ruleset.
7. You can use VirusTotal Intelligence to search for other matches of the same rule.

[![Crowdsourced Yara details](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/crowdsourcedyara_details_20231108.png)](https://www.virustotal.com/gui/file/275a021bbfb6489e54d471899f7db9d1663fc695ec2fe2a2c4538aabf651fd0f/detection)

1. If you scroll through the Ruleset this link will return the cursor back to the matched rule.
2. Copy the Ruleset to the clipboard.
3. Import the Ruleset to Livehunt.
4. Import the Ruleset to Retrohunt.
5. The matched rule is highlighted.
