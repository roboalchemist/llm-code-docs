# Feeds

We'll use the files feed as an example, but all the views are similar.

In the [IoC stream view](https://www.virustotal.com/gui/ioc-notifications/iocs/files), all the notifications are listed **sorted by the matching date in descending order** as shown in the image below. Notifications are **deleted automatically after 30 days**.

1. You can **filter the notifications** by a searching string, specific source name, etc.
2. You can click to **filter the notifications** based on the date they matched on.
3. You can click to **filter the notifications** based on the source type: Collection, Hunting ruleset, Retrohunt job or Threat Actor.
4. **List with matches notifications** generated by your subscription.
5. You can **click on the source** of an IoC to filter by that source.
6. You can **sort by the matching date** in ascendent or descendent order.
7. You can export IOCs by **downloading the matching files** or **copying the hashes to the clipboard**.
8. In the Actions menu you can
   1. **Create a diff session** with the selected files.
   2. **Create a VT Graph** with the selected files.
   3. **Find commonalities** in the selected files.
   4. Add the selected files to **a collection**.
9. Refresh the data
10. Delete selected or all notifications.
11. Feature Icons:
    1. If your YARA rule is based on strings, if you hover the eye icon , you can see the **Match context**.\
       ![IOC Stream Eye Icon](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/images/iocstream_eyeicon_20231031.png)
    2. If the matching file trigger network connections, if you hover the globe icon, you can see the **Related network locations.**\
       ![IOC Stream Globe Icon](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/images/iocstream_globeicon_20231031.png)
    3. If the matching file comes from a known distributor, if you hover the source icon, you will see the **known distributor name**.\
       ![IOC Stream Source Icon](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/images/iocstream_sourceicon_20231031.png)
    4. If the matching file have more than one name on its different submissions, if you hover the last name you can **see the list of names**.\
       ![IOC Stream Last name](https://storage.googleapis.com/vtdocresources/guides/vt-hunting/images/iocstream_lastname_20231031.png)

For livehunt rules, additionally, you will **receive notifications in your email if you configured** your ruleset as described in the previous section. You can also **access your notifications via the [VirusTotal API](https://docs.virustotal.com/reference/ioc-stream-introduction)**.
