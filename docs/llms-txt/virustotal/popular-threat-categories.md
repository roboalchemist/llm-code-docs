# Popular Threat Categories

List of malware categories commonly used in AV verdicts (e.g., trojan, dropper, ...).

This is a short list of malware categories commonly used in AV verdicts (e.g., trojan, dropper, ...). The AV verdicts from every file are processed for these kind of categories, normalized (e.g., "ransom" becomes "ransomware"), and set as as part of their [popular\_threat\_classification](#popular_threat_classification).
