# What type of files are supported by code insight?

These are the file types supported by [VirusTotal Code Insight](https://blog.virustotal.com/2023/04/introducing-virustotal-code-insight.html) :

APPLESCRIPT, AWK, C, CPP, DOC, DOCX, FORTRAN, GOLANG, JAVA, JAVASCRIPT, MAKEFILE, OBJETIVEC, PASCAL, PERL, PHP, POWERSHELL, PPT, PPTX, PYTHON, RUBY, SCRIPT, SHELLSCRIPT, SQL, TEXT, XLS, XLSX, ZIP.

The file limit is 1mb but be aware that for document or compressed files, the limit is applied to the bundled script not the parent file.
