# tlsh

Trend Micro's TLSH hash

`tlsh` is a hash used by [Trend Micro](https://github.com/trendmicro/tlsh) which can be used for similarity comparisons.

```json TLSH attribute
{
    "data": {
        "attributes": {
            "tlsh": "<string>"
        }
    }
}
```
```json Example
{
  "data": {
    "attributes": {
      "tlsh": "T1C7364B3AFD423955C3D462BB317E84C3334323B832E93133A9213B247A3F5CF3E3A359"
    }
  }
}
```
