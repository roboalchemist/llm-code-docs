# What does the green circle with a white tick mark icon mean?

VirusTotal makes use of the ![](https://storage.googleapis.com/vtdocresources/guides/faq/images/antivirusproducts_greencircle_20231121.png) checkmark symbol to indicate that the given file was not detected in any way by the antivirus under consideration. We do not use the word "clean" or "innocuous" because antivirus solutions do not tell you whether a file is *goodware*, they just flag maliciousness.

Other symbols that may appear are:

![](https://storage.googleapis.com/vtdocresources/guides/faq/images/antivirusproducts_positive_20231121.png) The antivirus under consideration detected this resource.\
![](https://storage.googleapis.com/vtdocresources/guides/faq/images/antivirusproducts_timeout_20231121.png) Timeout while running this antivirus.\
![](https://storage.googleapis.com/vtdocresources/guides/faq/images/antivirusproducts_notsupported_20231121.png) Unable to process file type.
