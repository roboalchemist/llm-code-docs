# How can I automate scans?

VirusTotal provides an [API](https://virustotal.readme.io/reference/overview) for automating analysis tasks, you can find more information in the VirusTotal [API documentation](https://virustotal.readme.io/reference/overview).

You can also check the list of [API Scripts](https://virustotal.readme.io/docs/api-scripts-and-client-libraries) developed by the community.
