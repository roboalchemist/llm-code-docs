# crowdsourced_ids_stats

IDS results stats.

IDS results stats. Dictionary containing:

* `high`: <*integer*>: number of IDS matched rules having a `high` severity.
* `info`: <*integer*>: number of IDS matched rules having a `info` severity.
* `low`: <*integer*>: number of IDS matched rules having a `low` severity.
* `medium`: <*integer*>: number of IDS matched rules having a `medium` severity.
