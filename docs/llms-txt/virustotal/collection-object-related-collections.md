# 🔀🔒 related_collections

Related collections for a given collection.

The *related\_collections* relationship returns a list of ***collections related to the collection***.

This relationship can be retrieved using the [relationships API endpoint](https://virustotal.readme.io/reference/get-collections-relationship). The response contains a list of [Collections](https://virustotal.readme.io/reference/collections-object) objects.

```json /collections/{id}/related_collections
{
  "data": [
    <COLLECTION_OBJECT>,
    <COLLECTION_OBJECT>,
    ...
  ],
  "links": {
    "next": <string>,
    "self": <string>
  },
  "meta": {
    "count": <int>,
    "cursor": <string>
  }
}
```
