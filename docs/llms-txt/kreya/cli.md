# CLI

The Kreya CLI `kreyac` provides a command line interface to perform operations with a Kreya project.

![An animation showcasing the CLI execution]()

## Installation[​](#installation "Direct link to Installation")

### Docker[​](#docker "Direct link to Docker")

A docker image for kreyac is available at [riok/kreyac](https://hub.docker.com/r/riok/kreyac).

### Download[​](#download "Direct link to Download")

The kreyac CLI binary can also be downloaded on the [downloads page](/downloads.md).

caution

The kreyac CLI does not update itself automatically.
