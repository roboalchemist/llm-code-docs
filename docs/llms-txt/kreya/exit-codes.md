# exit-codes

Show exit codes used by kreyac

```
kreyac exit-codes
```

## Exit codes used by kreyac[​](#exit-codes-used-by-kreyac "Direct link to Exit codes used by kreyac")

| Exit code | Description                |
| --------- | -------------------------- |
| 0         | Ok                         |
| 1         | UnknownError               |
| 2         | InitializationFailed       |
| 3         | Cancelled                  |
| 101       | ProjectNotFound            |
| 102       | OperationNotFound          |
| 103       | EnvironmentNotFound        |
| 104       | CollectionNotFound         |
| 105       | ScriptNotFound             |
| 201       | FailedTests                |
| 202       | OperationInvocationFailed  |
| 203       | CollectionInvocationFailed |
| 204       | ScriptInvocationFailed     |
