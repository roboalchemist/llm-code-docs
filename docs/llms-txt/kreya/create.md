# project create

Creates a new project

```
kreyac project create [ProjectPath]
```

## Example[​](#example "Direct link to Example")

```
kreyac project create my-project
```

## Arguments[​](#arguments "Direct link to Arguments")

### ProjectPath `required`[​](#projectpath-required "Direct link to projectpath-required")

Path where the project should be created.
