# Bottom Bar

* [Bubble.io Plugin](#bubble.io-plugin)
* [JavaScript SDK](#javascript-sdk)

### 🧋 Bubble.io Plugin

#### \[Action] Natively **- Show Bottom Bar**

This action shows the Bottom Bar and reloads the WebView.

#### \[Action] Natively - Hide Bottom Bar

This action hides the Bottom Bar and reloads the WebView.

### 🛠 JavaScript SDK

#### Show Bottom Bar

<pre class="language-javascript" data-overflow="wrap"><code class="lang-javascript"><strong>window.natively.showTabBar();
</strong></code></pre>

#### Hide Bottom Bar

```javascript
window.natively.hideTabBar();
```
