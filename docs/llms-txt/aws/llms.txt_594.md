## [Getting Started with the AWS Mobile SDK for .NET and Xamarin](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-xamarin.html)

- [Store and Retrieve Files with Amazon S3](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-store-retrieve-s3-transferutility.html): Amazon Simple Storage Service (Amazon S3) provides mobile developers with secure, durable, highly-scalable object storage.
- [Sync User Data with Cognito Sync](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-sync-data.html): Amazon Cognito Sync makes it easy to save mobile user data, such as app preferences or game state in the AWS Cloud without writing any backend code or managing any infrastructure.
- [Store and Retrieve Data with DynamoDB](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-store-retrieve-data.html): Amazon DynamoDB is a fast, highly scalable, highly available, cost-effective, non-relational database service.
- [Tracking App Usage Data with Amazon Mobile Analytics](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-analytics.html): Amazon Mobile Analytics allows you to measure app usage and app revenue.
- [Receive Push Notifications using SNS (Xamarin iOS)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-sns-ios.html): This document explains how to send push notifications to a Xamarin iOS application using Amazon Simple Notification Service (SNS) and the AWS Mobile SDK for .NET and Xamarin.
- [Receive Push Notifications using SNS (Xamarin Android)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/getting-started-sns-android.html): The tutorial explains how to send push notifications to a Xamarin Android application using Amazon Simple Notification Service (SNS) and the AWS Mobile SDK for .NET and Xamarin.


## [Amazon Simple Storage Service (S3)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/s3.html)

- [Using the S3 Transfer Utility](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/s3-integration-transferutility.html): The S3 Transfer Utility makes it easier to upload and download files to S3 from your Xamarin application.
- [Using the Service Level S3 APIs](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/s3-integration-lowlevelapi.html): In addition to using the S3 TransferUtility, you can also interact with S3 using the low-level S3 APIs.


## [Amazon DynamoDB](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/dynamodb.html)

- [Using the Document Model](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/dynamodb-integration-docmodel.html): The Document Model provides wrapper classes around the low-level .NET API.
- [Using the Object Persistence Model](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/dynamodb-integration-objectpersistencemodel.html): The AWS Mobile SDK for .NET and Xamarin provides an Object Persistence model that enables you to map your client-side classes to a DynamoDB table.
- [Using the DynamoDB Service Level APIs](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/dynamodb-integration-lowlevelapi.html): The Dynamo Service Level APIs allow you to create, update and delete tables.


## [Amazon Simple Notification Service (SNS)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns.html)

- [Send Push Notifications (Xamarin Android)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns-send-push-notifications-android.html): This document explains how to send push notifications to a Xamarin Android application using Amazon Simple Notification Service (SNS) and the AWS Mobile SDK for .NET and Xamarin.
- [Send Push Notifications (Xamarin iOS)](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns-send-push-notifications-ios.html): This document explains how to send push notifications to a Xamarin iOS application using Amazon Simple Notification Service (SNS) and the AWS Mobile SDK for .NET and Xamarin.
- [Send and Receive SMS Notifications](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns-send-sms.html): You can use Amazon Simple Notification Service (Amazon SNS) to send and receive Short Message Service (SMS) notifications to SMS-enabled mobile phones and smart phones.
- [Send Messages to HTTP/HTTPS Endpoints](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns-send-http.html): You can use Amazon SNS to send notification messages to one or more HTTP or HTTPS endpoints.
- [Troubleshooting SNS](https://docs.aws.amazon.com/mobile/sdkforxamarin/developerguide/sns-troubleshooting.html)
