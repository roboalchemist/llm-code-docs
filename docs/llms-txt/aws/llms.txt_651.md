# AWS Partner Central Sales Guide

> Describes how to manage qualified leads and co-sell opportunities.

- [Getting started](https://docs.aws.amazon.com/partner-central/latest/sales-guide/getting-started.html)
- [Receiving leads shared by AWS](https://docs.aws.amazon.com/partner-central/latest/sales-guide/leads.html)
- [Document history](https://docs.aws.amazon.com/partner-central/latest/sales-guide/doc-history.html)

## [Managing Opportunities](https://docs.aws.amazon.com/partner-central/latest/sales-guide/opportunities.html)

- [Creating an opportunity](https://docs.aws.amazon.com/partner-central/latest/sales-guide/creating-opportunity.html): Opportunities can be added individually, by bulk import or managed through a CRM integration.
- [Cloning opportunities](https://docs.aws.amazon.com/partner-central/latest/sales-guide/cloning-opportunities.html): You can start creating a new opportunity by cloning any existing opportunity that is not in draft status.
- [AWS opportunity review process](https://docs.aws.amazon.com/partner-central/latest/sales-guide/opportunity-review-process.html): AWS conducts an initial review of submitted opportunities, and may require additional information from the AWS Partner, according to the ACE Terms and Conditions.
- [Accepting opportunities](https://docs.aws.amazon.com/partner-central/latest/sales-guide/accepting-opportunities.html): ACE eligible (ACE Terms and Conditions defines ACE Eligibility) AWS Partners can receive AWS Opportunity Referrals.
- [Updating next steps and opportunity stage](https://docs.aws.amazon.com/partner-central/latest/sales-guide/updating-opportunities.html): We recommend that AWS Partners update opportunities with AWS Sales visibility with next steps at each stage change.
- [AWS stage visibility](https://docs.aws.amazon.com/partner-central/latest/sales-guide/stage-visibility.html): When an opportunity has reached its terminal stage (status Launched or Closed/Lost), the AWS Partner must complete the following:
- [Opportunity contacts](https://docs.aws.amazon.com/partner-central/latest/sales-guide/contact-roles.html): You can view the AWS contacts for any validated opportunity in the ACE Opportunity Page.
- [Creating a collaboration channel](https://docs.aws.amazon.com/partner-central/latest/sales-guide/slack-collaboration-channel.html): AWS Partners can request collaboration on select strategic, high-value opportunities that require AWS co-sell support.
- [Search and filter opportunities](https://docs.aws.amazon.com/partner-central/latest/sales-guide/search-filter-opportunities.html): Use the search bar on the Leads and Opportunities tabs to find records in the current list.
- [Tagging AWS partition for opportunities](https://docs.aws.amazon.com/partner-central/latest/sales-guide/esc-aws-partition.html): Partners can now specify AWS partition details when creating or managing opportunities in AWS Partner Central in the console.
- [Bulk actions](https://docs.aws.amazon.com/partner-central/latest/sales-guide/bulk-actions.html)


## [Discovering and managing partner connections](https://docs.aws.amazon.com/partner-central/latest/sales-guide/partner-connections.html)

- [Partner Discovery](https://docs.aws.amazon.com/partner-central/latest/sales-guide/partner-discovery.html): From Partner Discovery, you can find or search for other partners to view their details or send them connection requests.
- [Partner Connections](https://docs.aws.amazon.com/partner-central/latest/sales-guide/manage-partner-connections.html): From Partner Connections, you can manage your existing partner connections and active incoming/outgoing connection requests.
