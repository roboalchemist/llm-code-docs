## [Flows](https://docs.aws.amazon.com/appflow/latest/userguide/flows.html)

### [Creating flows](https://docs.aws.amazon.com/appflow/latest/userguide/create-flow.html)

There are several ways to create flows in Amazon AppFlow.

- [Create a flow using the AWS console](https://docs.aws.amazon.com/appflow/latest/userguide/create-flow-console.html): There are several ways to gets started with creating your first flow by using the AWS console user interface, AWS CLI commands, APIs, or by specifying CloudFormation resources.
- [Create a flow using the AWS CLI](https://docs.aws.amazon.com/appflow/latest/userguide/create-flow-cli.html): You may also use the CLI to create a connector profile and configure a flow using the AWS CLI commands for create-connector-profile and create-flow.
- [Create a flow using the Amazon AppFlow APIs](https://docs.aws.amazon.com/appflow/latest/userguide/create-flow-api.html): You may also use the APIs to create a connector profile and configure a flow using the CreateConnectorProfile and CreateFlow APIs.
- [Create a flow using CloudFormation resources](https://docs.aws.amazon.com/appflow/latest/userguide/create-flow-cfn.html): You may also use CloudFormation to create a connector profile and configure a flow using the AWS::AppFlow::ConnectorProfile and AWS::AppFlow::Flow resources.
- [Managing flows](https://docs.aws.amazon.com/appflow/latest/userguide/flows-manage.html): Use the Amazon AppFlow console to view, initiate, update, and copy flows.
- [Cataloging flow output](https://docs.aws.amazon.com/appflow/latest/userguide/flows-catalog.html): Catalog the data that you transfer so that you can discover and access the data from AWS analytics and machine learning services.
- [Partitioning and aggregating](https://docs.aws.amazon.com/appflow/latest/userguide/flows-partition.html): Organize the data that you transfer into partitions and files.
- [Flow triggers](https://docs.aws.amazon.com/appflow/latest/userguide/flow-triggers.html): A trigger determines how a flow runs.
- [Private flows](https://docs.aws.amazon.com/appflow/latest/userguide/private-flows.html): With Amazon AppFlow, you can create private flows between AWS services and supported software as a service (SaaS) applications.
- [Flow notifications](https://docs.aws.amazon.com/appflow/latest/userguide/flow-notifications.html): Amazon AppFlow is integrated with Amazon EventBridge to publish events related to the status of a flow.
- [General information](https://docs.aws.amazon.com/appflow/latest/userguide/general.html): The following general information applies to all Amazon AppFlow flows.
