## [Resources](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/resources.html)

- [Configuration Set](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/v1-sms-voice-configuration-sets-configurationsetname.html): A configuration set is a set of rules that you apply to the voice messages that you send.
- [Configuration Sets](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/v1-sms-voice-configuration-sets.html): A configuration set is a set of rules that you apply to the voice messages that you send.
- [Event Destination](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/v1-sms-voice-configuration-sets-configurationsetname-event-destinations-eventdestinationname.html): A configuration set is a set of rules that you apply to the voice messages that you send.
- [Event Destinations](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/v1-sms-voice-configuration-sets-configurationsetname-event-destinations.html): An event destination is a location where you send response information about the voice messages that you send.
- [Voice Message](https://docs.aws.amazon.com/pinpoint-sms-voice/latest/APIReference/v1-sms-voice-voice-message.html): The Amazon Pinpoint SMS and Voice API lets you create voice messages from a text script, and then deliver those messages to your customers over the phone.
