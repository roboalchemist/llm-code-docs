## [Components](https://docs.aws.amazon.com/greengrass/v2/developerguide/greengrass-components.html)

### [AWS-provided components](https://docs.aws.amazon.com/greengrass/v2/developerguide/public-components.html)

AWS IoT Greengrass V2 provides prebuilt components that you can deploy to your devices or use as dependencies in your custom components.

- [Greengrass nucleus](https://docs.aws.amazon.com/greengrass/v2/developerguide/greengrass-nucleus-component.html): The AWS IoT Greengrass nucleus is a component that you can configure to customize and remotely update your AWS IoT Greengrass Core software v2.0 installation.
- [Greengrass nucleus lite](https://docs.aws.amazon.com/greengrass/v2/developerguide/greengrass-nucleus-lite-component.html): The Greengrass nucleus lite (aws.greengrass.NucleusLite) is a device runtime for constrained edge devices optimized for minimal memory footprint (uses less than 5MB RAM).
- [Client device auth](https://docs.aws.amazon.com/greengrass/v2/developerguide/client-device-auth-component.html): The client device auth component enables local IoT devices to connect to the AWS IoT Greengrass V2 core device.
- [CloudWatch metrics](https://docs.aws.amazon.com/greengrass/v2/developerguide/cloudwatch-metrics-component.html): The Amazon CloudWatch metrics component publishes custom metrics from AWS IoT Greengrass V2 core devices to Amazon CloudWatch.
- [AWS IoT Device Defender](https://docs.aws.amazon.com/greengrass/v2/developerguide/device-defender-component.html): The AWS IoT Device Defender component notifies administrators of changes in the state of AWS IoT Greengrass V2 core devices.
