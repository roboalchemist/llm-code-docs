### [Changing your password](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/change-password.html)

If you are an account owner, you can change your AWS account password from the AWS Management Console.

- [Root users](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/root-password.html): If you are a Root user, you can change your AWS account password from the AWS Management Console.
- [IAM users](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/iam-password.html): If you are an IAM user, you can change your AWS account password from the AWS Management Console or an AWS access portal.
- [IAM Identity Center users](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/idc-password.html): If you are an AWS IAM Identity Center user, you can change your AWS account password from an AWS access portal.
- [Federated identities](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/fed-password.html): If you use a federated identity, you can change your AWS account password from an AWS access portal.
- [Changing the language of the AWS Management Console](https://docs.aws.amazon.com/awsconsolehelpdocs/latest/gsg/change-language.html): Learn how to change the language for the AWS Management Console.
