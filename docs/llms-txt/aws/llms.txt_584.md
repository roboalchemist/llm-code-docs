# AWS Migration Hub Home Region API Reference

> The AWS Migration Hub home region APIs are available specifically for working with your Migration Hub home region. You can use these APIs to determine a home region, as well as to create and work with controls that describe the home region.

- [Welcome](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/Welcome.html)
- [Common Parameters](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/CommonParameters.html)
- [Common Errors](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/CommonErrors.html)

## [Data Types](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/API_Types.html)

- [HomeRegionControl](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/API_HomeRegionControl.html): A home region control is an object that specifies the home region for an account, with some additional information.
- [Target](https://docs.aws.amazon.com/migrationhub-home-region/latest/APIReference/API_Target.html): The target parameter specifies the identifier to which the home region is applied, which is always an ACCOUNT.
