## [Command line interface reference](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/CLIReference.html)

- [mon-cmd](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-cmd.html): List all the other CloudWatch commands using the mon-cmd command.
- [mon-delete-alarms](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-delete-alarms.html): Delete alarms using the mon-delete-alarms command.
- [mon-describe-alarm-history](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-describe-alarm-history.html): Retrieve the history of an alarm using the mon-describe-alarm-history command.
- [mon-describe-alarms](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-describe-alarms.html): Get information about an alarm using the mon-describe-alarms command.
- [mon-describe-alarms-for-metric](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-describe-alarms-for-metric.html): Get information about alarms associated with a metric using the mon-describe-alarms-for-metric command.
- [mon-disable-alarm-actions](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-disable-alarm-actions.html): Disable all actions for an alarms using the mon-disable-alarm-actions command.
- [mon-enable-alarm-actions](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-enable-alarm-actions.html): Enable all actions for an alarm using the mon-enable-alarm-actions command.
- [mon-get-stats](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-get-stats.html): Get time-series data for one or more statistics using the mon-get-stats command.
- [mon-list-metrics](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-list-metrics.html): List the names, namespaces, and dimensions of the metrics associated with your AWS account using the mon-list-metrics command.
- [mon-put-data](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-put-data.html): Add metric data points to a metric using the mon-put-data command.
- [mon-put-metric-alarm](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-put-metric-alarm.html): Create or update an alarm and associate it with a CloudWatch metric using the mon-put-metric-alarm command.
- [mon-set-alarm-state](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-set-alarm-state.html): Change the state of a CloudWatch alarm temporarily using the mon-set-alarm-state command.
- [mon-version](https://docs.aws.amazon.com/AmazonCloudWatch/latest/cli/cli-mon-version.html): Print the version number of the CloudWatch CLI and the API using the mon-version command.
