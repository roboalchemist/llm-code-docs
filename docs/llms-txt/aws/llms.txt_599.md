## [Resources](https://docs.aws.amazon.com/msk/1.0/apireference/resources.html)

- [Apache Kafka Versions](https://docs.aws.amazon.com/msk/1.0/apireference/kafka-versions.html): Objects that represent Apache Kafka versions.
- [BootstrapBrokers](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-bootstrap-brokers.html): A list of brokers that a client application can use to bootstrap.
- [Broker Count](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-nodes-count.html): The number of broker nodes in a cluster.
- [Broker Storage](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-nodes-storage.html): Represents the EBS storage associated with the brokers.
- [Broker Type](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-nodes-type.html): The type of brokers in the cluster.
- [Brokers](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-nodes.html): The broker nodes in the cluster.
- [Cluster](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn.html): Represents an Amazon MSK cluster.
- [Cluster Apache Kafka Version](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-version.html): The Apache Kafka version that is on the cluster.
- [Cluster Configuration](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-configuration.html): Represents the configuration of a specific cluster.
- [Cluster Connectivity](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-connectivity.html): Use this resource to update the connectivity setting for an MSK cluster.
- [Cluster Operation](https://docs.aws.amazon.com/msk/1.0/apireference/operations-clusteroperationarn.html): Represents an operation that was performed on an MSK cluster.
- [Cluster Operations](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-operations.html): Represents the operations that have been performed on an MSK cluster.
- [Cluster Security](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-security.html): Use this resource to update the security settings of a cluster.
- [Clusters](https://docs.aws.amazon.com/msk/1.0/apireference/clusters.html): Use this resource to create an Amazon MSK cluster and to get a list of existing clusters.
- [Clusters clusterArn Client-vpc-connection](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-client-vpc-connection.html)
- [Clusters clusterArn Client-vpc-connections](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-client-vpc-connections.html)
- [Clusters clusterArn Policy](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-policy.html)
- [Clusters clusterArn Rebalancing](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-rebalancing.html)
- [Compatible Apache Kafka Versions](https://docs.aws.amazon.com/msk/1.0/apireference/compatible-kafka-versions.html): The Apache Kafka versions to which you can update the cluster.
- [Configuration](https://docs.aws.amazon.com/msk/1.0/apireference/configurations-arn.html): Represents an MSK configuration.
- [Configuration Revision](https://docs.aws.amazon.com/msk/1.0/apireference/configurations-arn-revisions-revision.html): Represents a specific revision of an MSK configuration.
- [Configuration Revisions](https://docs.aws.amazon.com/msk/1.0/apireference/configurations-arn-revisions.html): Represents the revisions of an MSK configuration.
- [Configurations](https://docs.aws.amazon.com/msk/1.0/apireference/configurations.html): A collection of MSK configurations.
- [Monitoring Properties](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-monitoring.html): The options for monitoring an Amazon MSK cluster.
- [Reboot Broker](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-reboot-broker.html)
- [Scram Secrets](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-scram-secrets.html): Represents a secret stored in the Amazon Secrets Manager that can be used to authenticate with a cluster using your sign-in credentials.
- [Tags](https://docs.aws.amazon.com/msk/1.0/apireference/tags-resourcearn.html): Represents the set of tags for an MSK resource.
- [Topic](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-topics-topicname.html)
- [Topic Partitions](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-topics-topicname-partitions.html)
- [Topics](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-topics.html)
- [UpdateStorage](https://docs.aws.amazon.com/msk/1.0/apireference/clusters-clusterarn-storage.html): Updates cluster broker volume size (or) sets cluster storage mode to TIERED.
- [Vpc-connection](https://docs.aws.amazon.com/msk/1.0/apireference/vpc-connection.html)
- [Vpc-connection arn](https://docs.aws.amazon.com/msk/1.0/apireference/vpc-connection-arn.html)
- [Vpc-connections](https://docs.aws.amazon.com/msk/1.0/apireference/vpc-connections.html)
