## [Get started](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/getting-started.html)

- [Get started as a portal administrator](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/portal-admin-getting-started.html): Sign in to the a AWS IoT SiteWise Monitor portal, explore assets, create projects, and assign owners and assets to projects.
- [Get started as a project owner](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/project-owner-getting-started.html): Sign in to a project in the AWS IoT SiteWise Monitor Project portal, explore assets, create and customize dashboards, and invite viewers.
- [Get started as a project viewer](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/project-viewer-getting-started.html): Log in to a project in the AWS IoT SiteWise Monitor portal, and view and explore dashboards.


## [Explore assets](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/explore-assets.html)

- [View asset data](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/view-asset-data.html): View asset data using the Assets page in the AWS IoT SiteWise portal.
- [Add assets to projects](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/add-assets-to-projects-ea.html): Learn how to add assets to projects in AWS IoT SiteWise Monitor.
- [Add properties and alarms](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/add-assets-to-dashboards.html): Learn how to add asset properties and alarms to dashboards for your project viewers in AWS IoT SiteWise Monitor.

Learn how to customize visualizations in a dashboard in AWS IoT SiteWise Monitor.

- [Choose visualization types](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/choose-visualization-types.html): Learn about the types of visualizations you can choose in AWS IoT SiteWise Monitor.
- [Configure thresholds](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/configure-thresholds.html): Configure thresholds that indicate when asset properties are outside their normal operating ranges in SiteWise Monitor.

## [Monitor with alarms](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/monitor-alarms.html)

- [Create alarm definitions](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/create-alarms.html): Create alarms in AWS IoT SiteWise Monitor to monitor asset properties.
- [View alarm details](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/view-alarm-details.html): View alarm data and details on the Assets page in an AWS IoT SiteWise Monitor portal.
- [Respond to alarms](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/respond-to-alarms.html): Acknowledge or snooze alarms in AWS IoT SiteWise Monitor.
- [Configure alarms](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/configure-alarms.html): Configure alarms in AWS IoT SiteWise Monitor to update their thresholds and notification settings for your operation.
- [Visualize alarms in dashboards](https://docs.aws.amazon.com/iot-sitewise/latest/appguide/visualize-alarms.html): Visualize alarms in dashboards to monitor your industrial operation in AWS IoT SiteWise Monitor.
