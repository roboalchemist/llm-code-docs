## [Actions](https://docs.aws.amazon.com/transfer/latest/APIReference/API_Operations.html)

- [CreateAccess](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateAccess.html): Used by administrators to choose which groups in the directory should have access to upload and download files over the enabled protocols using AWS Transfer Family.
- [CreateAgreement](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateAgreement.html): Creates an agreement.
- [CreateConnector](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateConnector.html): Creates the connector, which captures the parameters for a connection for the AS2 or SFTP protocol.
- [CreateProfile](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateProfile.html): Creates the local or partner profile to use for AS2 transfers.
- [CreateServer](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateServer.html): Instantiates an auto-scaling virtual server based on the selected file transfer protocol in AWS.
- [CreateUser](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateUser.html): Creates a user and associates them with an existing file transfer protocol-enabled server.
- [CreateWebApp](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateWebApp.html): Creates a web app based on specified parameters, and returns the ID for the new web app.
- [CreateWorkflow](https://docs.aws.amazon.com/transfer/latest/APIReference/API_CreateWorkflow.html): Allows you to create a workflow with specified steps and step details the workflow invokes after file transfer completes.
- [DeleteAccess](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteAccess.html): Allows you to delete the access specified in the ServerID and ExternalID parameters.
- [DeleteAgreement](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteAgreement.html): Delete the agreement that's specified in the provided AgreementId.
- [DeleteCertificate](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteCertificate.html): Deletes the certificate that's specified in the CertificateId parameter.
- [DeleteConnector](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteConnector.html): Deletes the connector that's specified in the provided ConnectorId.
- [DeleteHostKey](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteHostKey.html): Deletes the host key that's specified in the HostKeyId parameter.

- [DeleteWorkflow](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DeleteWorkflow.html): Deletes the specified workflow.
- [DescribeAccess](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeAccess.html): Describes the access that is assigned to the specific file transfer protocol-enabled server, as identified by its ServerId property and its ExternalId.
- [DescribeAgreement](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeAgreement.html): Describes the agreement that's identified by the AgreementId.
- [DescribeCertificate](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeCertificate.html): Describes the certificate that's identified by the CertificateId.
- [DescribeConnector](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeConnector.html): Describes the connector that's identified by the ConnectorId.
- [DescribeExecution](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeExecution.html): You can use DescribeExecution to check the details of the execution of the specified workflow.
- [DescribeHostKey](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeHostKey.html): Returns the details of the host key that's specified by the HostKeyId and ServerId.
- [DescribeProfile](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeProfile.html): Returns the details of the profile that's specified by the ProfileId.
- [DescribeSecurityPolicy](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeSecurityPolicy.html): Describes the security policy that is attached to your server or SFTP connector.
- [DescribeServer](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeServer.html): Describes a file transfer protocol-enabled server that you specify by passing the ServerId parameter.
- [DescribeUser](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeUser.html): Describes the user assigned to the specific file transfer protocol-enabled server, as identified by its ServerId property.
- [DescribeWebApp](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeWebApp.html): Describes the web app that's identified by WebAppId.
- [DescribeWebAppCustomization](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeWebAppCustomization.html): Describes the web app customization object that's identified by WebAppId.
- [DescribeWorkflow](https://docs.aws.amazon.com/transfer/latest/APIReference/API_DescribeWorkflow.html): Describes the specified workflow.
- [ImportCertificate](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ImportCertificate.html): Imports the signing and encryption certificates that you need to create local (AS2) profiles and partner profiles.
- [ImportHostKey](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ImportHostKey.html): Adds a host key to the server that's specified by the ServerId parameter.
- [ImportSshPublicKey](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ImportSshPublicKey.html): Adds a Secure Shell (SSH) public key to a Transfer Family user identified by a UserName value assigned to the specific file transfer protocol-enabled server, identified by ServerId.
- [ListAccesses](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListAccesses.html): Lists the details for all the accesses you have on your server.

- [ListFileTransferResults](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListFileTransferResults.html): Returns real-time updates and detailed information on the status of each individual file being transferred in a specific file transfer operation.
- [ListHostKeys](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListHostKeys.html): Returns a list of host keys for the server that's specified by the ServerId parameter.
- [ListProfiles](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListProfiles.html): Returns a list of the profiles for your system.
- [ListSecurityPolicies](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListSecurityPolicies.html): Lists the security policies that are attached to your servers and SFTP connectors.
- [ListServers](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListServers.html): Lists the file transfer protocol-enabled servers that are associated with your AWS account.
- [ListTagsForResource](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListTagsForResource.html): Lists all of the tags associated with the Amazon Resource Name (ARN) that you specify.
- [ListUsers](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListUsers.html): Lists the users for a file transfer protocol-enabled server that you specify by passing the ServerId parameter.
- [ListWebApps](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListWebApps.html): Lists all web apps associated with your AWS account for your current region.
- [ListWorkflows](https://docs.aws.amazon.com/transfer/latest/APIReference/API_ListWorkflows.html): Lists all workflows associated with your AWS account for your current region.
- [SendWorkflowStepState](https://docs.aws.amazon.com/transfer/latest/APIReference/API_SendWorkflowStepState.html): Sends a callback for asynchronous custom steps.
- [StartDirectoryListing](https://docs.aws.amazon.com/transfer/latest/APIReference/API_StartDirectoryListing.html): Retrieves a list of the contents of a directory from a remote SFTP server.
- [StartFileTransfer](https://docs.aws.amazon.com/transfer/latest/APIReference/API_StartFileTransfer.html): Begins a file transfer between local AWS storage and a remote AS2 or SFTP server.

- [StartServer](https://docs.aws.amazon.com/transfer/latest/APIReference/API_StartServer.html): Changes the state of a file transfer protocol-enabled server from OFFLINE to ONLINE.
- [StopServer](https://docs.aws.amazon.com/transfer/latest/APIReference/API_StopServer.html): Changes the state of a file transfer protocol-enabled server from ONLINE to OFFLINE.
- [TagResource](https://docs.aws.amazon.com/transfer/latest/APIReference/API_TagResource.html): Attaches a key-value pair to a resource, as identified by its Amazon Resource Name (ARN).
- [TestConnection](https://docs.aws.amazon.com/transfer/latest/APIReference/API_TestConnection.html): Tests whether your SFTP connector is set up successfully.
- [TestIdentityProvider](https://docs.aws.amazon.com/transfer/latest/APIReference/API_TestIdentityProvider.html): If the IdentityProviderType of a file transfer protocol-enabled server is AWS_DIRECTORY_SERVICE or API_Gateway, tests whether your identity provider is set up successfully.
- [UntagResource](https://docs.aws.amazon.com/transfer/latest/APIReference/API_UntagResource.html): Detaches a key-value pair from a resource, as identified by its Amazon Resource Name (ARN).
- [UpdateAccess](https://docs.aws.amazon.com/transfer/latest/APIReference/API_UpdateAccess.html): Allows you to update parameters for the access specified in the ServerID and ExternalID parameters.
- [UpdateAgreement](https://docs.aws.amazon.com/transfer/latest/APIReference/API_UpdateAgreement.html): Updates some of the parameters for an existing agreement.
