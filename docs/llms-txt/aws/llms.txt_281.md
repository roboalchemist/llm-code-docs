## [Getting Started](https://docs.aws.amazon.com/dcv/latest/userguide/getting-started.html)

- [Requirements](https://docs.aws.amazon.com/dcv/latest/userguide/requirements.html): To use Amazon DCV, ensure that the client computers meet the following minimum requirements.
- [Step 1: Get the session information](https://docs.aws.amazon.com/dcv/latest/userguide/getting-started-session.html): After the Amazon DCV session is running on the Amazon DCV server, you must have specific information to connect to it.
- [Step 2: Choose a client](https://docs.aws.amazon.com/dcv/latest/userguide/getting-started-choose.html): Next, choose the Amazon DCV client that best meets your needs.

### [Step 3: Connect to a session](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting.html)

Information about using the Amazon DCV clients to connect to a session.

- [Connecting using the Windows client](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting-win.html): Connecting to a Amazon DCV session using the Windows client.
- [Connecting using the web browser client](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting-browser-connect.html): Connecting to a Amazon DCV session using the web browser client.
- [Connecting using the Linux client](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting-linux.html): Connecting to a Amazon DCV session using the Linux client.
- [Connecting using the macOS client](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting-mac.html): Connecting to a Amazon DCV session using the macOS client.
- [Connecting using URI](https://docs.aws.amazon.com/dcv/latest/userguide/using-connecting-uri.html): Connecting to a Amazon DCV session using URI.
- [Step 4: Create a connection file (optional)](https://docs.aws.amazon.com/dcv/latest/userguide/using-connection-file.html): Using the Windows, Linux, or macOS native client, you can create a connection file that you can use to instantly connect to a Amazon DCV session.
