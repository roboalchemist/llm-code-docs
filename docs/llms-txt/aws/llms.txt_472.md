# ExpressLink Getting Started Guide

> Provides information about getting started with ExpressLink modules that are connectivity modules connected via serial interface (initially UART based) that use an abstracted Application Programming Interface (API) to connect any host application to AWS IoT Core and its services.

## [Getting started with AWS IoT ExpressLink](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg.html)

- [AWS account set up and console login](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-set-up.html): Before you use AWS IoT Core for the first time, complete the tasks in this section.
- [Register an AWS IoT ExpressLink module](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-register-account.html): Register an ExpressLink module to your development account.
- [Connect and interact with AWS Cloud.](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-interact-with-cloud.html): In this section, you use the MQTT client in the AWS IoT console to monitor the communication between your evaluation kit and the AWS Cloud.

### [Perform a firmware OTA update](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-setup-ota-update.html)

Establish setting up the prerequisites.

- [Prerequisites](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-setup-ota-update-prereqs.html): Establish setting up the prerequisites.
- [Create a firmware update job in AWS IoT](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-setup-ota-update-create.html): After setting up the prerequisites, create a firmware update job.
- [Monitor and apply a new firmware update for AWS IoT ExpressLink](https://docs.aws.amazon.com/iot-expresslink/latest/gettingstartedguide/elgsg-setup-ota-update-monitor.html): After you create a firmware update job as described in the previous section, the ExpressLink module polls for firmware update jobs, receives and validates a job, and enters a state waiting for the update to be accepted.
