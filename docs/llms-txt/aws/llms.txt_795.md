# Snowball Edge Getting Started Guide 

- [What are Snow devices?](https://docs.aws.amazon.com/snow/latest/swsbe-pack/whatitis.html)
- [Snow Family device hardware description](https://docs.aws.amazon.com/snow/latest/swsbe-pack/hardware-information.html)

## [Getting started](https://docs.aws.amazon.com/snow/latest/swsbe-pack/getting-started.html)

- [Cancelling a job](https://docs.aws.amazon.com/snow/latest/swsbe-pack/cancel-job-order.html): Cancel the job to not receive a Snow Family device.
- [Receiving the Snow Family device](https://docs.aws.amazon.com/snow/latest/swsbe-pack/receive-device-sw.html): Receive your Snow Family device, inspect for any obvious damage or tampering, and set it up for use.
- [Getting your credentials and tools](https://docs.aws.amazon.com/snow/latest/swsbe-pack/get-credentials.html): Get credentials to authenticate your access to your Snow device.
- [Downloading and installing the Snowball Edge client](https://docs.aws.amazon.com/snow/latest/swsbe-pack/download-the-client.html): Download and install the Snowball Edge client from AWS Snowball Edge resources.
- [Bind the hardware security module to the Snow device](https://docs.aws.amazon.com/snow/latest/swsbe-pack/bind-hsm.html): Bind the Snow device to a hardware security module.
- [Unlocking the device](https://docs.aws.amazon.com/snow/latest/swsbe-pack/unlockdevice.html): Unlock the device using the Snowball Edge client.
- [Access Snow device and AWS service guides online](https://docs.aws.amazon.com/snow/latest/swsbe-pack/docs-online.html): Now that your device is set up, you can access Snow device and AWS service guides online to learn about service features.
- [Powering off the device](https://docs.aws.amazon.com/snow/latest/swsbe-pack/turnitoff.html): Turn off the device and prepare it to return to AWS.
- [Emergency shutdown](https://docs.aws.amazon.com/snow/latest/swsbe-pack/emergency-power.html): Use this procedure to turn the equipment off during an emergency, such as fire, water, smoke, or hazard to personnel.
- [Returning the device](https://docs.aws.amazon.com/snow/latest/swsbe-pack/return-device.html): The prepaid shipping label on the E Ink display contains the correct address to return the Snowball Edge.
