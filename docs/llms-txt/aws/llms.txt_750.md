# AWS SDK for Swift Developer Guide

> Describes the features of AWS SDK for Swift, as well as how to use it from your Swift code.

- [AWS SDK for Swift](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/home.html)
- [Document history](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/doc-history.html)

## [Getting started](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/getting-started.html)

- [Setting up](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/setting-up.html): Set up projects using the AWS SDK for Swift, configure your computer for use with the SDK, and learn how to create your first project using the SDK for Swift.
- [Authenticating with AWS](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/authenticating.html): Learn how to authenticate your code with AWS in the SDK for Swift.
- [Creating an application](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/simple-app.html): Follow an example to get started with the AWS SDK for Swift.


## [Configuring service clients](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/configuring.html)

- [Client configuration externally](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/config-external.html): Learn how to set environment variables or use shared configuration in the SDK for Swift to configure service client functionality in all applications.
- [Client configuration in code](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/config-code.html): Learn how to create and use service clients in the SDK for Swift.
- [AWS Region](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/region.html): Learn how to set AWS Region in the SDK for Swift.
- [Credential providers](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/credential-providers.html): Learn how to configure credential providers in the SDK for Swift.
- [Retry](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-retry.html): Configure the retry behavior of the AWS SDK for Swift when accessing AWS services.
- [HTTP](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/http.html): Learn how to configure HTTP-level settings in the SDK for Swift.


## [Using the SDK](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using.html)

- [Making AWS service requests](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/making-requests.html): Learn how to use service clients to make requests to AWS services using the SDK for Swift.
- [Handling errors](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-errors.html): This guide covers how to handle errors thrown by the AWS SDK for Swift when using AWS services.
- [Pagination](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-paginators.html): Learn how to handle pagination of data in the SDK for Swift.
- [Testing and debugging](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-testing-debugging.html): This guide covers testing and debugging software projects using the AWS SDK for Swift.
- [Waiters](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-waiters.html): This guide covers how to wait for AWS SDK for Swift operations to complete using waiters.
- [Lambda functions](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/lambda.html): Learn how to create AWS Lambda functions that use the AWS SDK for Swift.
- [Event streaming](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-event-streaming.html): How to use event streaming with a supported AWS service, with an example showing how to stream Amazon Transcribe streaming events.
- [Presigning requests](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-presigning.html): Learn how to presign URLs and requests to be authorized to access AWS services for a limited time.
- [Integrating with Apple](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/apple-integration.html): Learn how to use the AWS SDK for Swift to create applications that work seamlessly with Apple's ecosystem.


## [Working with AWS services](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-services.html)

### [Amazon S3](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-services-s3.html)

Learn how to work with Amazon S3 by using the AWS SDK for Swift.

- [Multipart uploads](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-multipart-uploads.html): This guide covers how to use the AWS SDK for Swift to upload large files to Amazon S3 in multiple parts.
- [Binary streaming](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/using-binary-streaming.html): How to use binary streaming when sending data to an AWS service.
- [Data integrity protection with checksums](https://docs.aws.amazon.com/sdk-for-swift/latest/developer-guide/s3-checksums.html): Learn how to use checksums with Amazon S3 in AWS SDK for Swift to ensure data integrity during object uploads and downloads.
