# AWS Elemental Conductor Live Installation Guide

> This guide describes how to install AWS Elemental Conductor Live software for the first time.

- [About this guide](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/about-cl3-ig.html)
- [Reference: Downloading software](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/detailed-dl-cl3-ig.html)
- [Reference: System requirements for VMs](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/vm-req.html)
- [Reference: Sample install](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/sample-cl3-ig.html)
- [Document History](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/doc-history.html)

## [Installing on qualified hardware](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-ig.html)

- [Step A: Prepare the hardware and download files](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-ig-prep.html)
- [Step B: Install (Kickstart) the operating system software](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-ig-install-ks.html)
- [Step C: Install the Conductor Live software](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-ig-install-sw.html): Perform these on the Conductor Live appliance, either directly at the appliance or from your workstation via SSH.
- [Step D: Set up licenses](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-license.html): Learn how to set up licenses on each Conductor Live node.
- [Step E: Complete cluster configuration](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-cl3-ig-complete.html): You have now installed and performed basic configuration of Conductor Live.


## [Installing on a VM](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig.html)

- [Step A: Prepare the hardware and download files](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig-prep.html)
- [Step B: Deploy the VM](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig-install-vm.html): Perform these steps from your workstation.
- [Step C: Install the AWS Elemental Software](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig-install-sw.html)
- [Step D: Set up licenses](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig-licensing.html): Learn how to set up licenses on each Conductor Live VM in a virtual machine deployment.
- [Step E: Complete cluster configuration](https://docs.aws.amazon.com/elemental-cl3/latest/installguide/install-vm-cl3-ig-complete.html): You have now installed and performed basic configuration of Conductor Live.
