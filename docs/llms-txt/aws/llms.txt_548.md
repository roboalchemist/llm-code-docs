# AMB Access Bitcoin Developer Guide

> Amazon Managed Blockchain (AMB) Access Bitcoin allows customers serverless access to the Bitcoin blockchain without provisioning and managing any bitcoin infrastructure including nodes. This managed service allows you to access the Bitcoin networks quickly, on demand reducing overall cost of ownership.
