# Amazon Monitron Getting Started Guide

> Describes how IT Managers create projects, how admins create assets (machines) and set up sensors, and how technicians start monitoring equipment.

- [Detailed documentation](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/step-7.html)
- [Document history](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/doc-history.html)

## [Getting started with Amazon Monitron](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/gsg-getting-started.html)

- [Setting up a project](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/step-1.html): Your organization's IT manager will use these steps to learn how to set up a project for monitoring factory floor equipment with Amazon Monitron.
- [Adding assets and installing devices](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/step-2.html): Install and set up gateways, sensors, and assets in Amazon Monitron.
- [Understanding warnings and alerts](https://docs.aws.amazon.com/Monitron/latest/getting-started-guide/step-3.html): Learn how factory technicians monitor machine conditions with Amazon Monitron.
