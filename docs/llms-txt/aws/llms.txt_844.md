## [Network Access Scopes](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/working-with-network-access-scopes.html)

- [Resource statements](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/resource-statement.html): Learn how to use resource statements in Network Access Analyzer to define network components for match and exclude conditions.
- [Packet header statements](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/packet-header-statement.html): Learn how to use packet header statements in Network Access Analyzer to define traffic types for match or exclude conditions.
- [Match conditions](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/match-paths.html): Learn how to use match conditions in Network Access Analyzer to define types of network paths for producing findings.
- [Exclusion conditions](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/exclude-paths.html): Learn how to use exclusion conditions in Network Access Analyzer to define types of network paths for producing findings.
- [Example Network Access Scopes](https://docs.aws.amazon.com/vpc/latest/network-access-analyzer/example-scopes.html): View example Network Access Scopes in Network Access Analyzer.
