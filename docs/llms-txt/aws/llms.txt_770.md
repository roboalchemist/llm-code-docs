# AWS Service Catalog User Guide

> Service Catalog enables organizations to create and manage catalogs of IT services that are approved for use on AWS. These IT services can include everything from virtual machine images, servers, software, and databases to complete multi-tier application architectures.

## [Using the end user console view](https://docs.aws.amazon.com/servicecatalog/latest/userguide/end-user-console.html)

- [Using the Service Catalog console](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-dashboard.html): Use the Service Catalog console to see lists of products and provisioned products.
- [Using the Products page](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-productlist.html): View the Products page to see the resources that your administrator has made available to you in the Service Catalog console.
- [Using the Provisioned products page](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-stacklist.html): View the Provisioned products list page to see all the provisioned products that you have created.
- [Viewing product details](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-viewproduct.html): View product information on the Product details page.

### [Launching a product](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-launch.html)

How to launch a AWS Service Catalog product in the AWS Management Console.

- [Creating a launch plan](https://docs.aws.amazon.com/servicecatalog/latest/userguide/launch-plan.html): Learn how to create a launch plan.
- [Launching products as stack sets in multiple accounts and Regions](https://docs.aws.amazon.com/servicecatalog/latest/userguide/launch-stacksets.html): TBD
- [Viewing provisioned product information](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-viewstack.html): Go to the Provisioned product details page to view information about provisioned products that you have deployed in Service Catalog.
- [Updating provisioned products](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-update.html): Update provisioned products to deploy a new version of an Service Catalog product or to change parameters.
- [Updating provisioned product properties](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-update-pp-properties.html): Update provisioned products properties in Service Catalog to change the owner.
- [Deleting provisioned products](https://docs.aws.amazon.com/servicecatalog/latest/userguide/enduser-delete.html): Delete provisioned products in the Service Catalog console.
