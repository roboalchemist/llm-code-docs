# AWS Quick Start Guide: Back Up Your Files to Amazon Simple Storage Service Quick Start

> You can use Amazon S3 to store and retrieve any amount of data at any time, from anywhere on the web. By completing the steps in this quick start guide, you will successfully create a new S3 bucket, add a file to it, retrieve this file, and finally delete it, all within the AWS Free Tier.

## [AWS Quick Start Guide: Back Up Your Files to Amazon Simple Storage Service](https://docs.aws.amazon.com/quickstarts/latest/s3backup/welcome.html)

- [Step 1: Create an Amazon S3 Bucket](https://docs.aws.amazon.com/quickstarts/latest/s3backup/step-1-create-bucket.html)
- [Step 2: Upload a File to Your Amazon S3 Bucket](https://docs.aws.amazon.com/quickstarts/latest/s3backup/step-2-upload-file.html): Now that you've created a bucket, you're ready to add an object to it.
- [Step 3: Retrieve a File from Your Amazon S3 Bucket](https://docs.aws.amazon.com/quickstarts/latest/s3backup/step-3-retrieve-file.html): Now that you've added an object to a bucket, you can open and view it in a browser.
- [Step 4: Delete a File From Your Amazon S3 Bucket](https://docs.aws.amazon.com/quickstarts/latest/s3backup/step-4-delete-file.html): If you no longer need to store the file you've uploaded to your Amazon S3 bucket, you can delete it.
