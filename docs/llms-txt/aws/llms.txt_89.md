## [Configuring your application](https://docs.aws.amazon.com/app2container/latest/UserGuide/configuration.html)

- [Manage secrets](https://docs.aws.amazon.com/app2container/latest/UserGuide/manage-secrets.html): Describes how to set up the remote access secrets for App2Container using AWS Secrets Manager.
- [Configure containers](https://docs.aws.amazon.com/app2container/latest/UserGuide/config-containers.html): Describes configuration files and fields that App2Container uses for creating application containers.
- [Configure deployment](https://docs.aws.amazon.com/app2container/latest/UserGuide/config-deployment.html): Learn how App2Container uses configuration files and fields to deploy application containers to their target container management services (Amazon ECR with Amazon ECS, Amazon EKS, or AWS App Runner).
- [Configure pipelines](https://docs.aws.amazon.com/app2container/latest/UserGuide/config-pipeline.html): Describes configuration files and fields that App2Container uses for creating and deploying container management pipelines using AWS CodeStar services.


## [Product and service integrations](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations.html)

- [Deploy to Amazon ECS](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-ecs.html): This section describes how AWS App2Container integrates with Amazon ECS, to deploy your application containers to Amazon ECS environments running on EC2 instances or in Fargate.
- [Deploy to Amazon EKS](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-eks.html): Learn how AWS App2Container integrates with Amazon EKS and how to deploy your application containers to Amazon EKS environments.
- [Deploy to App Runner](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-apprunner.html): This section describes how you can use AWS App2Container to configure your application containers to run as an AWS App Runner web service.
- [Set up CodePipeline pipelines](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integration_codepipeline.html): Integrate AWS App2Container with AWS CodePipeline to deploy your application containers with CI/CD pipelines.
- [Set up Jenkins pipelines](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-jenkins.html): This section describes how AWS App2Container integrates with Jenkins, to deploy your application containers using Jenkins CI/CD pipelines.
- [Set up Azure DevOps pipelines](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-azure-devops.html): This section describes how to integrate AWS App2Container with Microsoft Azure so that you can deploy your application containers with Microsoft Azure DevOps pipelines.

### [Route logs using FireLens](https://docs.aws.amazon.com/app2container/latest/UserGuide/a2c-integrations-firelens.html)

This section describes configuration for FireLens log file routing for Amazon ECS.

- [FireLens log routing for Linux](https://docs.aws.amazon.com/app2container/latest/UserGuide/firelens-setup-linux.html): This topic walks you through setting up log file routing with FireLens for Amazon ECS for your Linux application containers that were generated by App2Container.
