# AWS CloudTrail Lake Partner Onboarding Guide

> This guide for Partner Network partners describes how to configure CloudTrail Lake access to your application's events. Information in this guide helps you prepare for your customers to integrate their event logs from your applications with CloudTrail Lake. with AWS CloudTrail.

## [Onboarding partner events to AWS CloudTrail Lake](https://docs.aws.amazon.com/awscloudtrail/latest/partner-onboarding/cloudtrail-lake-partner-onboarding.html)

- [Onboard to AWS CloudTrail Lake](https://docs.aws.amazon.com/awscloudtrail/latest/partner-onboarding/lake-partner-onboarding-tasks.html): This topic for AWS Partner Network partners describes the steps to onboard their applications to AWS CloudTrail Lake.
- [CloudTrail Lake event schema](https://docs.aws.amazon.com/awscloudtrail/latest/partner-onboarding/lake-onboarding-cloudtrail-event-schema.html): This topic describes the required and optional schema elements for a CloudTrail Lake integration.
- [Learn more about CloudTrail Lake](https://docs.aws.amazon.com/awscloudtrail/latest/partner-onboarding/lake-partner-onboarding-learning.html): This topic list resources that AWS partners can access to learn more about what CloudTrail Lake is and how it works.
