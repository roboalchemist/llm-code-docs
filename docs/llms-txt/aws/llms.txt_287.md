# AWS DeepRacer Student User Guide

> AWS DeepRacer Student is a free portal for students currently enrolled in high school or an accredited university/community college. Using AWS AWS DeepRacer you can train a reinforcement learning model that will attempt to drive around a racetrack. The Student League is a place where you can join up with your friends and complete for prizes. The AWS AI & ML Scholarship program is available in the AWS DeepRacer Student portal.
