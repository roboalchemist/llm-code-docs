## [What is AWS Cloud Control API?](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/what-is-cloudcontrolapi.html)

- [How it works](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/how-it-works.html): Describes how Cloud Control API provides a centralized control over resources in an AWS account, which allows users to access and provision those resources.


## [Cloud Control API resource operations](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations.html)

- [Creating a resource](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-create.html): Describes how to use the create-resource command to create a resource.
- [Updating a resource](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-update.html): Describes how to use the update-resource command to make updates to an existing resource.
- [Deleting a resource](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-delete.html): Describes how to use the delete-resource command to delete an existing resource.
- [Discovering resources](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-list.html): Describes how to Use the list-resources command to discover the resources currently provisioned in your AWS account and AWS Region.
- [Reading a resource](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-read.html): Describes how to use the the get-resource command to retrieve detailed information about the resource.
- [Managing resource requests](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-operations-manage-requests.html): Describes how to manage resource operations using information returned from a ProgressEvent object, listing active resources, and tracking operation progress.
- [Identifying resources with AWS Cloud Control API](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/resource-identifier.html): Describes how to identify your resources using a primary identifier or a secondary identifier.


## [Security](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/security.html)

- [VPC endpoints (AWS PrivateLink)](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/vpc-interface-endpoints.html): Describes how to use an interface VPC endpoint to create a private connection between your VPC and AWS Cloud Control API.
- [CloudFormation Hooks](https://docs.aws.amazon.com/cloudcontrolapi/latest/userguide/security-hooks.html): Describes the security features available to you when using AWS CloudFormation Hooks with AWS Cloud Control API.
