## [Data Types](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Types.html)

- [Association](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Association.html): An object that represents the custom domain name association.
- [ConfigParameter](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ConfigParameter.html): An array of key-value pairs to set for advanced control over Amazon Redshift Serverless.
- [CreateSnapshotScheduleActionParameters](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_CreateSnapshotScheduleActionParameters.html): The parameters that you can use to configure a scheduled action to create a snapshot.
- [Endpoint](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Endpoint.html): The VPC endpoint object.
- [EndpointAccess](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_EndpointAccess.html): Information about an Amazon Redshift Serverless VPC endpoint.
- [ManagedWorkgroupListItem](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ManagedWorkgroupListItem.html): A collection of Amazon Redshift compute resources managed by AWS Glue.
- [Namespace](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Namespace.html): A collection of database objects and users.
- [NetworkInterface](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_NetworkInterface.html): Contains information about a network interface in an Amazon Redshift Serverless managed VPC endpoint.
- [PerformanceTarget](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_PerformanceTarget.html): An object that represents the price performance target settings for the workgroup.
- [RecoveryPoint](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_RecoveryPoint.html): The automatically created recovery point of a namespace.
- [Reservation](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Reservation.html): Represents an Amazon Redshift Serverless reservation, which gives you the option to commit to a specified number of Redshift Processing Units (RPUs) for a year at a discount from Serverless on-demand (OD) rates.
- [ReservationOffering](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ReservationOffering.html): The class of offering for the reservation.
- [ResourcePolicy](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ResourcePolicy.html): The resource policy object.
- [Schedule](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Schedule.html): The schedule of when Amazon Redshift Serverless should run the scheduled action.
- [ScheduledActionAssociation](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ScheduledActionAssociation.html): Contains names of objects associated with a scheduled action.
- [ScheduledActionResponse](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ScheduledActionResponse.html): The returned scheduled action object.
- [ServerlessTrack](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_ServerlessTrack.html): Defines a track that determines which Amazon Redshift version to apply after a new version is released.
- [Snapshot](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Snapshot.html): A snapshot object that contains databases.
- [SnapshotCopyConfiguration](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_SnapshotCopyConfiguration.html): The object that you configure to copy snapshots from one namespace to a namespace in another AWS Region.
- [TableRestoreStatus](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_TableRestoreStatus.html): Contains information about a table restore request.
- [Tag](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Tag.html): A map of key-value pairs.
- [TargetAction](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_TargetAction.html): A JSON format string of the Amazon Redshift Serverless API operation with input parameters.
- [UpdateTarget](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_UpdateTarget.html): A track that you can switch the current track to.
- [UsageLimit](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_UsageLimit.html): The usage limit object.
- [VpcEndpoint](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_VpcEndpoint.html): The connection endpoint for connecting to Amazon Redshift Serverless through the proxy.
- [VpcSecurityGroupMembership](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_VpcSecurityGroupMembership.html): Describes the members of a VPC security group.
- [Workgroup](https://docs.aws.amazon.com/redshift-serverless/latest/APIReference/API_Workgroup.html): The collection of computing resources from which an endpoint is created.
