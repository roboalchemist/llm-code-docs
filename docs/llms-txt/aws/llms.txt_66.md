## [Resources](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/resources.html)

- [Broker](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers-broker-id.html): A broker is a message broker environment running on Amazon MQ.
- [Broker Engine Types](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/broker-engine-types.html): Retrieve information about available broker engines.
- [Broker Instance Options](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/broker-instance-options.html): You can retrieve information about broker instances.
- [Broker Reboot](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers-broker-id-reboot.html): To apply a new configuration to a broker or to initiate a failover in a multi-AZ broker, you can reboot the broker.
- [Brokers](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers.html): This is a collection of brokers.
- [Brokers broker-id Promote](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers-broker-id-promote.html)
- [Configuration](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/configurations-configuration-id.html): A configuration contains all of the settings for your broker.
- [Configuration Revision](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/configurations-configuration-id-revisions-configuration-revision.html): To keep track of the changes you make to your configuration, you can create configuration revisions.
- [Configuration Revisions](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/configurations-configuration-id-revisions.html): This is a collection of configuration revisions.
- [Configurations](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/configurations.html): This is a collection of configurations.
- [Tag](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/tags-resource-arn.html): A tag is a key-value pair associated with a resource.
- [User](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers-broker-id-users-username.html)
- [Users](https://docs.aws.amazon.com/amazon-mq/latest/api-reference/brokers-broker-id-users.html)
