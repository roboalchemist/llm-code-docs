# User Guide

This chapter is to guide a developer to use Crafting sandbox in their day-to-day development. We will walk through a list of common tasks and best practices to improve your productivity for day-to-day coding, testing, debugging, etc.

For how to configure the Crafting platform for your team to effectively manage development environments, please see the [admin guide](https://docs.sandboxes.cloud/docs/admin-overview), which includes how to step templates, snapshots, testing data, PR integration, etc.
