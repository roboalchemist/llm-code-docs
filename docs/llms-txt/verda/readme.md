# Overview

**Verda** (formerly known as DataCrunch) is a cloud provider specializing in AI and machine learning services. We offer dedicated GPU instances, GPU clusters, containers and serverless inference.

All services are fully-managed and optimized for ML workflows by our expert team of infrastructure and ML engineers.
