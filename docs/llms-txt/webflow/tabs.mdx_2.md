## Class Naming

Each tab button uses a base class and can be styled to reflect different backgrounds or interface variants.

The base class is: <span class="sg-selector sg-class">Tab Menu Link</span>

## Surface Modifiers

Use these to adapt the tab appearance based on the background context.

Naming convention for combo classes is <span class="sg-selector sg-class">Tab Menu Link On `Surface`</span>,
where `Surface` can be `Inverse`, `Accent Primary`, `Accent Tertiary`, or `Accent Secondary`.

Example:
<span class="sg-selector sg-class">Tab Menu Link</span> <span class="sg-selector sg-class">Tab Menu Link On Inverse</span>

## Style Modifiers

Additional style options for tabs is <span class="sg-selector sg-class">Tab Menu Link Transparent</span>  It removes tab background but keeps underline and label styling.

Example:
<span class="sg-selector sg-class">Tab Menu Link Transparent</span>
