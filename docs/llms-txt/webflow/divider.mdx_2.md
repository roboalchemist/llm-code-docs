## Class Naming

Dividers are thin visual separators used to distinguish content sections either horizontally or vertically. There are two main base classes depending on the direction of the divider.

Base Classes:

| Class                                                      | Description         |
| ---------------------------------------------------------- | ------------------- |
| <span class="sg-selector sg-class">Divider</span>          | horizontal dividers |
| <span class="sg-selector sg-class">Divider Vertical</span> | vertical dividers   |

## Color Modifiers

These can be used in combination with either horizontal or vertical dividers to adjust color styling.

* <span class="sg-selector sg-class">
    Secondary Divider
  </span>
* <span class="sg-selector sg-class">
    Accent Divider
  </span>

Example:

* <span class="sg-selector sg-class">
    Divider
  </span>
  <span class="sg-selector sg-class">
    Secondary Divider
  </span>

* <span class="sg-selector sg-class">
    Divider Vertical
  </span>
  <span class="sg-selector sg-class">
    Accent Divider
  </span>
