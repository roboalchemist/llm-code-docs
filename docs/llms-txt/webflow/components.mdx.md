# Components

Flowkit provides a comprehensive set of component classes (button, card, etc) that help you build consistent and beautiful user interfaces. Each component is designed with accessibility, customization, and ease of use in mind.

## Available Components

### Interactive Elements

* [Buttons](/flowkit/components/buttons) - Primary, secondary, and tertiary button styles
* [Tag](/flowkit/components/tag) - Label and category indicators
* [Dropdown](/flowkit/components/dropdown) - Expandable selection menus
* [Accordion](/flowkit/components/accordion) - Collapsible content sections

### Content Display

* [Images](/flowkit/components/images) - Image display and responsive behavior
* [Cards](/flowkit/components/cards) - Content containers and layouts
* [Rich Text](/flowkit/components/richtext) - Text formatting and styles
* [Divider](/flowkit/components/divider) - Section separators

### Form Elements

* [Forms](/flowkit/components/forms) - Input fields, labels, and form layouts

### Navigation

* [Tabs](/flowkit/components/tabs) - Content organization and navigation
* [Slider](/flowkit/components/slider) - Carousel and content sliders
