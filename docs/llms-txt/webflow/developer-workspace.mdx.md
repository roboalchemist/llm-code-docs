The **Developer workspace plan** gives App developers access to advanced Webflow features often used by larger teams and enterprises for the purposes of testing during Webflow App development.

This plan allows you to explore premium capabilities and see how they can elevate your Apps, offering a dedicated environment for building, testing, and integrating with Webflow’s powerful premium features.

## What's included

As Webflow continues to expand its offerings, you may see additional premium capabilities added to the App Developer Workspace Plan.

| Feature                                                                                         | Details                                     |
| ----------------------------------------------------------------------------------------------- | ------------------------------------------- |
| **Workspace Users**                                                                             | Up to 3 users for collaboration and testing |
| **API Rate Limit**                                                                              | 120 requests per minute (RPM)               |
| **Unhosted Sites**                                                                              | Unlimited                                   |
| [**Custom Code**](https://developers.webflow.com/data/docs/custom-code)                         | Enabled                                     |
| **Site Activity Log**                                                                           | Enabled                                     |
| [**Page Branching**](https://university.webflow.com/lesson/page-branching?topics=collaboration) | Enabled                                     |

## Limitations

This workspace allows App Developers an environment to test and build their Apps, but has some key restrictions in place.

| Feature                      | Details                       |
| ---------------------------- | ----------------------------- |
| **Publishing to Production** | Disabled                      |
| **Exporting Code**           | Disabled                      |
| **Site Transfer**            | Disabled                      |
| **Monthly Site Visitors**    | 1,000 site visitors per month |

## Managing your developer workspace plan

<Accordion title="Is There a Cost for This Plan?">
  <p>
    The App Developer Workspace Plan is free of charge. However, any site plans purchased within this workspace will be billed at standard rates without discounts.
  </p>
</Accordion>

<Accordion title="Switching Plans">
  <p>
    If you choose to move to a different Workspace plan, you will lose access to features unique to the App Developer Workspace Plan, such as the Site Activity Log and Page Branching. Contact us if you need assistance with transitioning to another plan.
  </p>
</Accordion>
