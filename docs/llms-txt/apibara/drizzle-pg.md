<a href="#adding-the-plugin-to-your-indexer" class="

          "><h2 id="adding-the-plugin-to-your-indexer" class="hover:after:content-[&#x27;#&#x27;] after:relative after:pl-2">Adding the plugin to your indexer</h2></a>
<p class="leading-7 [&amp;:not(:first-child)]:mt-6">Add the <code class="text-sm">drizzleStorage</code> plugin to your indexer&#x27;s <code class="text-sm">plugins</code>. Notice the following:</p>
<ul class="my-6 ml-6 list-disc">
<li class="mt-2">Use the <code class="text-sm">drizzle</code> helper exported by <code class="text-sm">@apibara/plugin-drizzle</code> to create a
drizzle instance. This method supports creating an in-memory database (powered by PgLite)
by specifying the <code class="text-sm">memory:</code> connection string.</li>
<li class="mt-2">Always specify the database schema. This schema is used by the indexer to know which tables
it needs to protect against chain reorganizations.</li>
<li class="mt-2">By default, the connection string is read from the <code class="text-sm">POSTGRES_CONNECTION_STRING</code> environment variable. If left empty, a local PGLite database will be created. This is great because it means you don&#x27;t need to start Postgres on your machine to develop locally!</li>
</ul>

<a href="#specifying-the-id-column" class="

          "><h2 id="specifying-the-id-column" class="hover:after:content-[&#x27;#&#x27;] after:relative after:pl-2">Specifying the id column</h2></a>
<p class="leading-7 [&amp;:not(:first-child)]:mt-6">As mentioned in the previous section, the id column is required by the plugin to handle chain reorganizations.</p>
<p class="leading-7 [&amp;:not(:first-child)]:mt-6">The plugin allows you to specify the id column name for each table in the schema. You can do this by passing the <code class="text-sm">idColumn</code> option to the <code class="text-sm">drizzleStorage</code> plugin. This option accepts either a string value or a record mapping table names to column names. You can use the special <code class="text-sm">&quot;*&quot;</code> table name to define the default id column name for all tables.</p>
