<p class="leading-7 [&amp;:not(:first-child)]:mt-6">Indexers are extensible through hooks and plugins. Hooks are functions that are called at specific points in the indexer&#x27;s lifecycle. Plugins are components that contain reusable hooks callbacks.</p>
<a href="#hooks" class="
                block font-heading mt-12 border-b pb-2 text-2xl font-semibold tracking-tight first:mt-0
          "><h2 id="hooks" class="hover:after:content-[&#x27;#&#x27;] after:relative after:pl-2">Hooks</h2></a>
<p class="leading-7 [&amp;:not(:first-child)]:mt-6">The following hooks are available in all indexers.</p>
<ul class="my-6 ml-6 list-disc">
<li class="mt-2"><strong><code class="text-sm">run:before</code></strong><span class="arg-type"><code class="text-sm">() =&gt; void</code></span><br/>
<span class="arg-description">Called before the indexer starts running.</span></li>
<li class="mt-2"><strong><code class="text-sm">run:after</code></strong><span class="arg-type"><code class="text-sm">() =&gt; void</code></span><br/>
<span class="arg-description">Called after the indexer has finished running.</span></li>
<li class="mt-2"><strong><code class="text-sm">connect:before</code></strong><span class="arg-type"><code class="text-sm">({ request: StreamDataRequest&lt;TFilter&gt;, options: StreamDataOptions }) =&gt; void</code></span><br/>
<span class="arg-description">Called before the indexer connects to the DNA stream. Can be used to change the request or stream options.</span></li>
<li class="mt-2"><strong><code class="text-sm">connect:after</code></strong><span class="arg-type"><code class="text-sm">({ request: StreamDataRequest&lt;TFilter&gt; }) =&gt; void</code></span><br/>
<span class="arg-description">Called after the indexer has connected to the DNA stream.</span></li>
<li class="mt-2"><strong><code class="text-sm">connect:factory</code></strong><span class="arg-type"><code class="text-sm">({ request: StreamDataRequest&lt;TFilter&gt;, endCursor: { orderKey: bigint, uniqueKey?: string } }) =&gt; void</code></span><br/>
<span class="arg-description">Called before the indexer reconnects to the DNA stream with a new filter (in factory mode).</span></li>
<li class="mt-2"><strong><code class="text-sm">message</code></strong><span class="arg-type"><code class="text-sm">({ message: StreamDataResponse&lt;TBlock&gt; }) =&gt; void</code></span><br/>
<span class="arg-description">Called for each message received from the DNA stream. Additionally, message-specific hooks are available: <code class="text-sm">message:invalidate</code>, <code class="text-sm">message:finalize</code>, <code class="text-sm">message:heartbeat</code>, <code class="text-sm">message:systemMessage</code>.</span></li>
<li class="mt-2"><strong><code class="text-sm">handler:middleware</code></strong><span class="arg-type"><code class="text-sm">({ use: MiddlewareFunction) =&gt; void }) =&gt; void</code></span><br/>
<span class="arg-description">Called to register indexer&#x27;s middlewares.</span></li>
</ul>
