Confluent platform provides two options to integration with ClickHouse

- [ClickHouse Connect Sink on Confluent Cloud](/integrations/kafka/cloud/confluent/sink-connector)
- [ClickHouse Connect Sink on Confluent Platform](/integrations/kafka/cloud/confluent/custom-connector) using the custom connectors feature
- [HTTP Sink Connector for Confluent Platform](/integrations/kafka/cloud/confluent/http) that integrates Apache Kafka with an API via HTTP or HTTPS
