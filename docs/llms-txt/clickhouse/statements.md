Users interact with ClickHouse using SQL statements. ClickHouse supports common SQL statements like [SELECT](/reference/sql-reference/statements/select) and [CREATE](/reference/sql-reference/statements/create), but it also provides specialized statements like [KILL](/reference/sql-reference/statements/kill) and [OPTIMIZE](/reference/sql-reference/statements/optimize).
