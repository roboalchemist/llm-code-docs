# Postman Collection

> Test Plivo APIs using our official Postman collection

Plivo provides an official Postman collection with pre-configured endpoints for Voice, SMS, Phone Numbers, Account, Lookup, and PHLO APIs.

## What's Included

| Product           | APIs                                      |
| ----------------- | ----------------------------------------- |
| **Voice**         | Calls, Conferences, Recordings, Endpoints |
| **SMS**           | Messages, Media (MMS)                     |
| **Phone Numbers** | Search, Buy, Manage                       |
| **Account**       | Details, Subaccounts                      |
| **Lookup**        | Number validation                         |
| **PHLO**          | Workflow triggers                         |

***

## Quick Setup

1. **Fork the collection** using the button above
2. **Add credentials** in the collection's Authorization tab:
   * Username: Your Auth ID
   * Password: Your Auth Token
3. **Set environment variable** `auth_id` to your Auth ID
4. **Start testing** - all endpoints are pre-configured
