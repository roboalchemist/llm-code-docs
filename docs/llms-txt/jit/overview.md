# Reporting Overview

The Overview page displays a summary of key Jit metrics and functions and provides security champions with the updated information they need about their organization's security status so that they can take [Action](https://docs.jit.io/docs/actions). It has the following sections:

### Insights

This section displays your organization's total Detected Findings, total Pull Requests with Findings, and total Resolved Findings over the selected period. Additionally, you can view your Developers' Efficiency (see [Security Impact](https://docs.jit.io/docs/security-impact)).

![](https://files.readme.io/1200050-insights.png)

### Actions

This section displays any Actions that are available for you to remediate. For further information on Actions, see [Actions](https://docs.jit.io/docs/actions).

![](https://files.readme.io/e0e6661-actions.png)

### Coverage

This section displays your current utilization of Jit's security controls, as well as the resources that are covered by these controls. Use this information to identify and resolve gaps in security coverage.  If you want to add additional security controls, or add resources, select the [Activate Security Controls](https://docs.jit.io/docs/my-plan-tab) or [Add Resources](https://docs.jit.io/docs/manage-resources) links.

### Security posture

This section displays the number of security findings that are currently open for each of your active security controls.
