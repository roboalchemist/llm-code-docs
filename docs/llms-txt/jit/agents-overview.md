# Agents Overview

## What Are Agents in Jit?

Agents are Jit’s AI-powered automation units. Each agent performs a specific security task, such as monitoring cloud posture, analyzing GitHub activity, or running a full risk assessment, and can be executed manually or on a recurring schedule.

Agents combine:

* LLM reasoning
* Jit’s knowledge graph
* Your connected integrations

to analyze your environment and produce actionable, context-rich insights.

Agents can be used in two ways:

* **Agentic Chat:** Ask questions in natural language and receive dashboards, breakdowns, explanations, and widgets.
* **Automated Agents:** Save a task as an agent, configure its schedule, and let it run automatically.

Agents help you operationalize security workflows that typically require manual investigation, correlation, or repeated effort.

**Note:** Some agents can also use built-in tools (like Jira and Slack) to fetch data or create tickets.\
For details, see **[Tools](https://docs.jit.io/docs/agentic-tools)**.

***

## Where Agents Live in the Platform

You can access all agent-related features from the left navigation:

* **Agentic Chat** – Interactive chat with Sera
* **My Agents** – List of saved and scheduled agents
* **Agent Gallery** – Prebuilt agent templates curated by Jit

These three areas work together: **Ask → Create → Automate**.
