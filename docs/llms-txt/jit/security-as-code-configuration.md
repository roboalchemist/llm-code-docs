# Security as Code Configuration

Jit enables you to perform certain administrative tasks by modifying Jit configuration files stored in your repositories. You can add these *as code* features to your Jit solution as your organization's needs evolve.

The following features are currently available *as code*—

* [Monorepo Support](https://docs.jit.io/docs/monorepo-support).
* [Folder Exclusion](https://docs.jit.io/docs/folder-exclusion).
