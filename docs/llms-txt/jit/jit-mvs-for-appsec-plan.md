# Jit MVS for AppSec Plan

## Description

The Jit MVS (maximum viable security) for AppSec Plan leverages several open-source security tools to achieve a baseline security standard for your development environment. The Jit MVS for AppSec Plan enables R\&D to own application security from day zero with an automatically and continuously implemented solution that is hassle-free, transparent, and quiet.

## Plan Items

These are the items (product security requirements) included in Jit MVS plan:

| Item Name                                                                                          | Coverage                 |
| :------------------------------------------------------------------------------------------------- | :----------------------- |
| [Scan Code For Vulnerabilities (SAST)](https://docs.jit.io/docs/scan-code-for-vulnerabilities)                          | Application Security     |
| [Scan Code Dependencies For Vulnerabilities (SCA)](https://docs.jit.io/docs/scan-code-dependencies-for-vulnerabilities) | Application Security     |
| [Scan Code For Hard-Coded Secrets](https://docs.jit.io/docs/scan-code-for-hard-coded-secrets)                           | Application Security     |
| [Scan Dockerfiles](https://docs.jit.io/docs/scan-dockerfiles)                                                           | Cloud Security           |
| [Scan Kubernetes IaC Files](https://docs.jit.io/docs/scan-kubernetes-iac-for-misconfigurations)                         | Cloud Security           |
| [Scan IaC For Static Misconfigurations](https://docs.jit.io/docs/scan-iac-for-static-misconfigurations)                 | Cloud Security           |
| [Scan for Infrastructure Runtime Misconfigurations](https://docs.jit.io/docs/scan-runtime-infra)                        | Cloud Security           |
| [Require MFA For Cloud Providers](https://docs.jit.io/docs/require-mfa-for-cloud-providers)                             | Cloud Security           |
| [Ensure Your API is Secure](https://docs.jit.io/docs/ensure-your-api-is-secure)                                         | Web Application Security |
