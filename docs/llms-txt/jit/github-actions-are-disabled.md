# GitHub Actions are disabled

Configure GitHub Actions to be enabled

## Overview

Jit uses GitHub Actions to run its scans, both daily/weekly and PR scans. There are cases where, for organizatinal decisions, GitHub Actions are disabled. In this situation, Jit will not be able to run and GitHub Actions will need to be enabled.

## Solution

Follow [GitHub's documentation](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/enabling-features-for-your-repository/managing-github-actions-settings-for-a-repository) to enable GitHub Actions in the repository/organization.
