# Risk Mitigation Overview

## Overview

Jit provides tools and workflows designed to streamline vulnerability management and risk mitigation, helping you focus on the most critical issues and resolve them efficiently.

## Jit Findings: Centralized Oversight

The Findings page consolidates all security findings into a single place where you can search and filter them, giving you complete visibility across your environment. With filtering, custom views, and export options, teams can organize findings effectively and continuously monitor their security posture-ensuring no risks are overlooked and enabling quick action when new ones are detected.

Learn more about the Jit Findings [here](https://docs.jit.io/v5.0/docs/jit-findings).

## Contextual Prioritization: Focus on the Security Issues That Really Matter

Jit’s Context Engine prioritizes risks based on their specific environmental and potential business impact rather than using basic severity scores. Mapping resource connections and propagating priority factors like "Production Environment" or "Externally Accessible" helps you focus on the vulnerabilities that introduce real security risk. The Context Graph provides a clear visualization of these relationships, guiding your decisions and helping you explain prioritization to stakeholders.

Learn more about Jit's contextual prioritization [here](https://docs.jit.io/v5.0/docs/contextual-prioritization-context-engine).

## Managing Resources: Discovery, Organize, and Protect Critical Assets

The Resources feature helps you classify and prioritize repositories, cloud services, and other assets. The Resource Dashboard highlights key risk factors, while manual tagging allows you to group resources into  categories, such as applications or teams. These tools ensure that critical assets are always in focus and vulnerabilities are addressed with precision.

Learn more about managing resources with Jit [here](https://docs.jit.io/v5.0/docs/managing-resources-apps-and-repositories).

## Jit Actions: Streamlined Remediation

Actions simplify remediation by grouping related issues and automating fixes, such as creating pull requests or running scripts. This reduces the manual effort of risk mitigation and accelerates resolution, ensuring vulnerabilities are addressed quickly. Integrations with tools like Slack and Jira make collaboration seamless, while built-in tracking ensures progress is transparent and actionable.

Learn more about Jit Actions [here](https://docs.jit.io/v5.0/docs/jit-actions).
