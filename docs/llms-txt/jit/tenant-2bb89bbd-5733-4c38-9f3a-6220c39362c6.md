# Get integration file

Returns the integration file content.

The integration file content is a valid YAML string, representing a dictionary object.
More information on integration file content can be found in the [integration documentation](https://docs.jit.io/docs/integrating-with-third-party-products-and-services).


**Requires the following permission:**
`jit.integrations.read`
