# Exports an artifact

Exports an artifact.

Jit supports the following artifact types: 'SBOM', 'WIZ_ISSUES', 'ZAP_SCANNED_URLS_web' and 'ZAP_SCANNED_URLS_api'.

**Important**: Use the following `/artifacts/<artifact_id>` endpoint to check you can export the artifact. A Forbidden response indicates you don’t have permission.

**Requires the following permission:**
`jit.artifacts.read`
