# Web App and API Scanners

Jit's Dynamic Application Security Testing (DAST) solution scans web apps and APIs for code security vulnerabilities like code injections, path traversals, and many more.

* [Dynamic Application Security Testing (DAST) for web apps](https://docs.jit.io/docs/run-a-web-application-scanner): Scan your web apps periodically or after every deployment.
* [Dynamic Application Security Testing (DAST) for APIs](https://docs.jit.io/docs/ensure-your-api-is-secure): Scan your APIs periodically or after every deployment.
* Jit DAST automates the deployment and scanning process for ZAP, the leading open source DAST tool. See the list of Jit-supported ZAP rules [here](https://docs.jit.io/docs/vulnerabilities-detected-using-zap).
