# What information does Portainer collect?

Prior to Portainer version 2.38.0, we collected anonymous usage statistics using Matomo Analytics when users explicitly opted in. Starting with Portainer version 2.38.0, the Matomo integration has been removed. As a result, Portainer no longer collects anonymous usage statistics, and the option to opt in to this data collection has also been removed.
