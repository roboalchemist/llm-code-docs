# December 26, 2024

Introducing the **Observability system**, which helps you monitor the availability and performance of your services in Novita AI.

The Observability system is currently in beta and only available for the LLM API service, more details can be found in the [LLM API Metrics Documentation](/guides/observability-llm-api-metrics).
