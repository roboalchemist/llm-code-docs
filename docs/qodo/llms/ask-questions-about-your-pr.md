# Ask Questions About Your PR

#### Ask questions. Get answers. Stay in your PR.

Qodo lets you ask questions about your pull request and get answers grounded in the actual code changes. Whether you’re reviewing logic, clarifying intent, or digging into edge cases, Qodo uses the current PR context to give precise, relevant responses.

You can interact with Qodo in two ways:

* [**Command-based**](https://docs.qodo.ai/qodo-documentation/code-review/get-started/use-qodo-in-prs/code-review/ask-questions-with-ask) — use the `/ask` command in pull request comments
* [**Auto-detected**](https://docs.qodo.ai/qodo-documentation/code-review/get-started/use-qodo-in-prs/code-review/auto-detected-questions) — ask questions naturally in comments and let Qodo recognize them (requires configuration)

Both approaches work directly inside your PR, keeping discussions fast, focused, and close to the code.
