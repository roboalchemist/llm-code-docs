# Introduction

### IDE plugin — formerly Qodo Gen — local code review workflows in JetBrains and VS Code

{% endhint %}

### Shift code reviews left and validate code as you write.&#x20;

Qodo IDE Plugin brings automated, context-aware code review directly into your IDE. It helps developers detect issues, enforce standards, and validate fixes as they write — turning the IDE into a continuous review environment.

Qodo’s review agents analyze diffs, test logic, and apply organization-specific rules in real time, ensuring every change is correct, compliant, and ready for merge.

#### With Qodo, you can:

* Run review workflows on local changes to surface high-signal issues like breaking changes, security vulnerabilities and more.&#x20;
* Apply AI code suggestions and resolve issues in 1-click&#x20;
* Identify and run missing tests on code changes&#x20;

#### Customizable review agents&#x20;

Qodo’s IDE plugin adapts to your workflows and standards. Add your organization’s rules, extend review logic, and build custom review agents that reflect how your team writes and reviews code.

{% hint style="success" %}

* Visual Studio Code
* JetBrains IDEs
* Visual Studio Professional
