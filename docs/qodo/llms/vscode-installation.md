# VSCode Installation

### Install Qodo IDE Plugin <a href="#install-qodo-gen" id="install-qodo-gen"></a>

Visit [Qodo's page on Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Codium.codium) to install Qodo.

Alternatively, follow these steps to install the plugin from within VSCode:

1. In VSCode, open the Extensions menu by clicking the Extensions icon in the Activity Bar on the left side.

<figure><img src="https://782320861-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhllduvO2nHKZ2vKzfcEn%2Fuploads%2FgJEHzyeUcotztQxDLW9y%2FScreenshot%202025-03-05%20at%2011.21.19.png?alt=media&#x26;token=d88d4ea4-ed47-4c9a-bb1e-5c74c9bd85c5" alt="" width="202"><figcaption></figcaption></figure>

2. Type **Qodo** in the search bar.
3. Click the Install button.

<figure><img src="https://782320861-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhllduvO2nHKZ2vKzfcEn%2Fuploads%2FVE3rNKSpBEe8QK2Qonjp%2FScreenshot%202025-03-05%20at%2011.22.54.png?alt=media&#x26;token=eda24e16-c502-40b0-aa40-2e0d559d1250" alt="" width="449"><figcaption></figcaption></figure>

### Activate Qodo

1. Following installation, restart all extensions to activate Qodo IDE Plugin.

<figure><img src="https://782320861-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhllduvO2nHKZ2vKzfcEn%2Fuploads%2Fk8r6jMzyYhndIvZFlG4B%2FScreenshot%202025-03-05%20at%2011.24.27.png?alt=media&#x26;token=a4df8036-5cec-4cb3-89c4-343bf02f9ae2" alt="" width="434"><figcaption></figcaption></figure>

2. After restarting extensions, the Qodo icon should appear in the Activity Bar on the left side.

### Log in to Enterprise Plan <a href="#login-to-enterprise-plan" id="login-to-enterprise-plan"></a>

For Enterprise users, please log in with your work email with the registered domain. You will be automatically connect to your organization workspace. Choose one of the login methods to complete your login.

If your organization has Single Sign-On (SSO) configured, you may also use this method for an even smoother login process.
