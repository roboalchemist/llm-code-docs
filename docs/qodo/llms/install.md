# Install Qodo in Your Git Provider

Welcome to the first step in enhancing your code review workflow with Qodo in your Git provider. This guide walks you through installing and setting up Qodo so you can start using automated code review directly in your pull requests.

#### Why Qodo connects to your Git provider

Qodo integrates directly with your Git provider to analyze and enhance pull requests where your team already works.

This connection enables Qodo to:

* Pull the content of your pull request
* Analyze changes using context-aware review agents and your organization’s standards
* Publish insights, summaries, and review feedback back to the same pull request

By working directly inside your Git provider, Qodo keeps all review feedback, explanations, and updates tightly coupled to your source code and existing development workflow—without requiring developers to switch tools or contexts.
