Class
Abstract Look Around
====================

An abstract class that provides a common interface for Look Around views.

MapKit JS 5.79+

`abstract class AbstractLookAround extends EventTarget`

[Discussion](https://developer.apple.com/documentation/mapkitjs/abstractlookaround#Discussion)
----------------------------------------------------------------------------------------------

You can’t initialize an abstract class; use [`LookAround`](https://developer.apple.com/documentation/mapkitjs/lookaround) or [`LookAroundPreview`](https://developer.apple.com/documentation/mapkitjs/lookaroundpreview) to create a Look Around view.

[Topics](https://developer.apple.com/documentation/mapkitjs/abstractlookaround#topics)
--------------------------------------------------------------------------------------

[`ReadyStates`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/readystates)

A property that represents the Look Around view’s containing Document Object Model (DOM) element.

A Boolean value that indicates whether someone can navigate inside the Look Around view.

[`isScrollEnabled`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/isscrollenabled)

A Boolean value that indicates whether someone can scroll the Look Around view.

[`isZoomEnabled`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/iszoomenabled)

A Boolean value that indicates whether someone can zoom the Look Around view.

[`openDialog`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/opendialog)

[`padding`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/padding)

The padding options for the Look Around view.

[`readyState`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/readystate)

A value that represents the loading state of the Look Around object.

[`scene`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/scene)

The Look Around scene the framework is displaying.

[`showsPointsOfInterest`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/showspointsofinterest)

A Boolean value that indicates whether the Look Around view shows points of interest (POIs).

[`showsRoadLabels`](https://developer.apple.com/documentation/mapkitjs/abstractlookaround/showsroadlabels)

A Boolean value that indicates whether the Look Around view shows road labels.
