Class
Map Kit Event
=============

A generic MapKit JS event object.

MapKit JS 5.0+

`class MapKitEvent`

[Discussion](https://developer.apple.com/documentation/mapkitjs/mapkitevent#Discussion)
---------------------------------------------------------------------------------------

The event object implements similar interfaces as the DOM `Event`, but you shouldn’t mix them.

[Topics](https://developer.apple.com/documentation/mapkitjs/mapkitevent#topics)
-------------------------------------------------------------------------------

### [Instance Properties](https://developer.apple.com/documentation/mapkitjs/mapkitevent#Instance-Properties)

[`defaultPrevented`](https://developer.apple.com/documentation/mapkitjs/mapkitevent/defaultprevented)

A Boolean value that indicates whether the app canceled the event.

[`target`](https://developer.apple.com/documentation/mapkitjs/mapkitevent/target)

The object that dispatched the event.

[`type`](https://developer.apple.com/documentation/mapkitjs/mapkitevent/type)

A string that represents the type of the event.

### [Instance Methods](https://developer.apple.com/documentation/mapkitjs/mapkitevent#Instance-Methods)

[`preventDefault()`](https://developer.apple.com/documentation/mapkitjs/mapkitevent/preventdefault)

Cancels the event if it’s cancelable, without stopping further propagation of the event.

[`stopPropagation()`](https://developer.apple.com/documentation/mapkitjs/mapkitevent/stoppropagation)

Stops further propagation of the event.
