Generators
----------

[How to add your generator »](https://yeoman.io/generators/)

Your generator will automatically show up in the list if it meets the following criteria. The list is populated every hour.

* Be published on the npmjs.org repository
* Have the following in the module's `package.json` file:
  * The name should be `generator-your-unique-name`. This means it should begin with `generator-` and be followed by a unique (to npm) name. Scoped name support is coming in the future.
  * `yeoman-generator` in the keyword property.
  * A good description. If you forked the generator, it should describe what is different and why it was forked.

* It'd be nice if it also had the following information:
  * Author information including name and link to your homepage.
  * Generator documention link as its website.
  * GitHub link in its repository field.
