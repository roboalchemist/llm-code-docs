# AWS IQ User Guide for Customers

> Provides a conceptual overview of AWS IQ and includes detailed instructions for customers to use the service.

- [What is AWS IQ?](https://docs.aws.amazon.com/aws-iq/latest/user-guide/what-is-aws-iq.html)
- [Getting started](https://docs.aws.amazon.com/aws-iq/latest/user-guide/getting-started-customer-create-aws-iq-request.html)
- [Document history](https://docs.aws.amazon.com/aws-iq/latest/user-guide/doc-history.html)
- [AWS Glossary](https://docs.aws.amazon.com/aws-iq/latest/user-guide/glossary.html)

## [How it works](https://docs.aws.amazon.com/aws-iq/latest/user-guide/aws-iq-how-it-works.html)

- [Requests](https://docs.aws.amazon.com/aws-iq/latest/user-guide/create-request.html): Create a request in AWS IQ.
- [Responses](https://docs.aws.amazon.com/aws-iq/latest/user-guide/request-responses.html): See responses from experts in AWS IQ and review details and reviews of each expert.
- [Chat and screen sharing](https://docs.aws.amazon.com/aws-iq/latest/user-guide/chat-video-calls-screen-sharing.html): Chat with experts on different browsers in AWS IQ.
- [Direct messaging](https://docs.aws.amazon.com/aws-iq/latest/user-guide/customer-direct-messaging.html): Learn more about direct messaging in AWS IQ.
- [Proposals](https://docs.aws.amazon.com/aws-iq/latest/user-guide/working-with-proposals.html): Review proposals from experts in AWS IQ.
- [Permissions requests](https://docs.aws.amazon.com/aws-iq/latest/user-guide/permissions-requests.html): Learn about permissions requests in AWS IQ and the different AWS Identity and Access Management (IAM) access roles and policies that can be granted to experts.
- [Costs and payments](https://docs.aws.amazon.com/aws-iq/latest/user-guide/costs-payments-pricing.html): Learn about costs and payments in AWS IQ proposals.
- [Refunds](https://docs.aws.amazon.com/aws-iq/latest/user-guide/refunds.html): Learn about requesting refunds for work you have paid for.
- [Ratings and reviews](https://docs.aws.amazon.com/aws-iq/latest/user-guide/ratings-and-reviews.html): Rate experts in AWS IQ and leave private feedback or public reviews.


## [Security](https://docs.aws.amazon.com/aws-iq/latest/user-guide/security.html)

- [Account permissions](https://docs.aws.amazon.com/aws-iq/latest/user-guide/set-up-account-permissions-to-use-aws-iq.html): Learn how to create an AWS IQ account with IAMFullAccess and AWSIQFullAccess permissions to grant permissions to an AWS Certified expert.
- [Using service-linked roles](https://docs.aws.amazon.com/aws-iq/latest/user-guide/using-service-linked-roles.html): Learn how to call other AWS services on your behalf using serivce-linked roles in AWS IQ.
- [Monitoring activity](https://docs.aws.amazon.com/aws-iq/latest/user-guide/activity-log-monitoring-experts.html): Learn how to monitor expert activity in AWS IQ proposals with CloudTrail.
