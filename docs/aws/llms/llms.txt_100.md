- [DeleteVpcIngressConnection](https://docs.aws.amazon.com/apprunner/latest/api/API_DeleteVpcIngressConnection.html): Delete an App Runner VPC Ingress Connection resource that's associated with an App Runner service.
- [DescribeAutoScalingConfiguration](https://docs.aws.amazon.com/apprunner/latest/api/API_DescribeAutoScalingConfiguration.html): Return a full description of an AWS App Runner automatic scaling configuration resource.
- [DescribeCustomDomains](https://docs.aws.amazon.com/apprunner/latest/api/API_DescribeCustomDomains.html): Return a description of custom domain names that are associated with an AWS App Runner service.

- [ListAutoScalingConfigurations](https://docs.aws.amazon.com/apprunner/latest/api/API_ListAutoScalingConfigurations.html): Returns a list of active AWS App Runner automatic scaling configurations in your AWS account.
- [ListConnections](https://docs.aws.amazon.com/apprunner/latest/api/API_ListConnections.html): Returns a list of AWS App Runner connections that are associated with your AWS account.

- [ListServicesForAutoScalingConfiguration](https://docs.aws.amazon.com/apprunner/latest/api/API_ListServicesForAutoScalingConfiguration.html): Returns a list of the associated App Runner services using an auto scaling configuration.
- [ListTagsForResource](https://docs.aws.amazon.com/apprunner/latest/api/API_ListTagsForResource.html): List tags that are associated with for an AWS App Runner resource.
- [ListVpcConnectors](https://docs.aws.amazon.com/apprunner/latest/api/API_ListVpcConnectors.html): Returns a list of AWS App Runner VPC connectors in your AWS account.
- [ListVpcIngressConnections](https://docs.aws.amazon.com/apprunner/latest/api/API_ListVpcIngressConnections.html): Return a list of App Runner VPC Ingress Connections in your AWS account.
- [PauseService](https://docs.aws.amazon.com/apprunner/latest/api/API_PauseService.html): Pause an active AWS App Runner service.
- [ResumeService](https://docs.aws.amazon.com/apprunner/latest/api/API_ResumeService.html): Resume an active AWS App Runner service.
- [StartDeployment](https://docs.aws.amazon.com/apprunner/latest/api/API_StartDeployment.html): Initiate a manual deployment of the latest commit in a source code repository or the latest image in a source image repository to an AWS App Runner service.
- [TagResource](https://docs.aws.amazon.com/apprunner/latest/api/API_TagResource.html): Add tags to, or update the tag values of, an App Runner resource.
- [UntagResource](https://docs.aws.amazon.com/apprunner/latest/api/API_UntagResource.html): Remove tags from an App Runner resource.
- [UpdateDefaultAutoScalingConfiguration](https://docs.aws.amazon.com/apprunner/latest/api/API_UpdateDefaultAutoScalingConfiguration.html): Update an auto scaling configuration to be the default.
- [UpdateService](https://docs.aws.amazon.com/apprunner/latest/api/API_UpdateService.html): Update an AWS App Runner service.
- [UpdateVpcIngressConnection](https://docs.aws.amazon.com/apprunner/latest/api/API_UpdateVpcIngressConnection.html): Update an existing App Runner VPC Ingress Connection resource.
