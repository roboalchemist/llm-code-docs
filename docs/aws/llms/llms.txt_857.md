# AWS Wavelength Developer Guide

> AWS Wavelength expands the AWS infrastructure to communications service providers' (CSP) data centers at the edge of the 5G network. Wavelength Zones deploy standard AWS infrastructure racks, including both hardware and software, in CSP locations.
