# AWS Elemental Live Installation Guide

> AWS Elemental Live is a real-time video service that lets you create live outputs for broadcast and streaming delivery. This Installation Guide describes how to install software on hardware units and virtual machines (VMs).

- [About this guide](https://docs.aws.amazon.com/elemental-live/latest/installguide/about-lv-ig.html)
- [Reference: Downloading software](https://docs.aws.amazon.com/elemental-live/latest/installguide/detailed-dl-lv-ig.html)
- [Reference: System requirements for VMs](https://docs.aws.amazon.com/elemental-live/latest/installguide/vm-req.html)
- [Reference: Sample install](https://docs.aws.amazon.com/elemental-live/latest/installguide/sample-lv-ig.html)
- [Document History](https://docs.aws.amazon.com/elemental-live/latest/installguide/doc-history.html)

## [Installing on qualified hardware](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig.html)

- [Step A: Prepare hardware and download files](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig-prep.html)
- [Step B: Kickstart the OS](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig-install-ks.html): Install the operating system on the node.
- [Step C: Install the Elemental Live software](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig-install-sw.html): Perform these steps on each AWS Elemental Live hardware unit, either directly at the hardware unit or from your workstation via SSH.
- [Step D: Set up licenses](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig-licensing.html): Learn how to set up licenses on each AWS Elemental Live node.
- [Step E: Complete node configuration](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-lv-ig-complete.html): You have now installed and performed the basic initial configuration of Elemental Live.


## [Installing on a VM](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig.html)

- [Step A: Prepare hardware and download files](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig-prep.html)
- [Step B: Deploy the VM](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig-install-vm.html): Perform these steps from your workstation.
- [Step C: Install the Elemental Live software](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig-install-sw.html): Topics
- [Step D: Set up licenses](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig-licensing.html): Learn how to set up licenses on each Elemental Live VM in a virtual deployment.
- [Step E: Complete node configuration](https://docs.aws.amazon.com/elemental-live/latest/installguide/install-vm-lv-ig-complete.html): You have now installed and performed the basic initial configuration of Elemental Live.
