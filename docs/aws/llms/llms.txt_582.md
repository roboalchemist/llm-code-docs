# AWS Migration Hub Journeys Migration Hub Journeys User Guide

> AWS Migration Hub Journeys (Migration Hub Journeys) is a service that you can use to plan, perform, and track migrations into AWS. Migration Hub Journeys combines the tools that are available in AWS Migration Hub with the latest guidance from AWS Prescriptive Guidance.

## [Migration journeys](https://docs.aws.amazon.com/mhj/latest/userguide/migration-journeys.html)

- [Creating a journey](https://docs.aws.amazon.com/mhj/latest/userguide/journey-creation.html): Migration Hub Journeys provides templates that you can use to create your migration journey.
- [Journey status](https://docs.aws.amazon.com/mhj/latest/userguide/journey-status.html): A migration journey can have any of the status values that appear in the following table.
- [Copying a journey](https://docs.aws.amazon.com/mhj/latest/userguide/journey-copy.html): The following procedure describes how to create a copy of a migration journey.
- [Updating a journey](https://docs.aws.amazon.com/mhj/latest/userguide/journey-updates.html): The following procedure describes how to update the status, description, and completion date of a migration journey.
- [Transferring a journey](https://docs.aws.amazon.com/mhj/latest/userguide/journey-transfers.html): After you create a migration journey, you can send a request to transfer the journey to another individual.
- [Individuals and teams](https://docs.aws.amazon.com/mhj/latest/userguide/individuals-and-teams.html): A migration journey can have two types of members: individuals and teams.
- [Attached files](https://docs.aws.amazon.com/mhj/latest/userguide/attached-files.html): You can attach files to tasks and subtasks.
- [Customizing your journey](https://docs.aws.amazon.com/mhj/latest/userguide/customization.html): When you create a journey from a template, you can customize the journey to make it match your particular migration scenario more closely.
