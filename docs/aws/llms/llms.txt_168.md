## [Getting Started](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-templates-getting-started.html)

- [Set Up Prerequisites](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-template-getting-started-prerequisites.html): The AWS Blockchain Template for Ethereum configuration that you specify in this tutorial requires that you do the following:
- [Create the Ethereum Network](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-templates-create-stack.html): The Ethereum network that you specify using the template in this topic launches an CloudFormation stack that creates an Amazon ECS cluster of EC2 instances for the Ethereum network.
- [Connect to EthStats and EthExplorer Using the Bastion Host](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-bastion-host-connect.html): To connect to Ethereum resources in this tutorial, you set up SSH port forwarding (SSH tunneling) through the bastion host.
- [Clean Up Resources](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-templates-cleanup.html): CloudFormation makes it easy to clean up resources that the stack created.


## [AWS Blockchain Templates and Features](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-template-features.html)

- [AWS Blockchain Template for Ethereum](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-templates-ethereum.html): Ethereum is a blockchain framework that runs smart contracts using Solidity, an Ethereum-specific language.
- [AWS Blockchain Template for Hyperledger Fabric](https://docs.aws.amazon.com/blockchain-templates/latest/developerguide/blockchain-templates-hyperledger.html): Hyperledger Fabric is a blockchain framework that runs smart contracts called chaincode, which are written in Go.
