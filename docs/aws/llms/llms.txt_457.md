# AWS Systems Manager Incident Manager API Reference

> AWS Systems Manager Incident Manager is an incident management console designed to help users mitigate and recover from incidents affecting their applications hosted on AWS. An incident is any unplanned interruption or reduction in quality of services.

- [RelatedItemsUpdate](https://docs.aws.amazon.com/incident-manager/latest/APIReference/API_RelatedItemsUpdate.html): Details about the related item you're adding.
