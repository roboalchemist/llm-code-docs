## [Configuring Kinesis Agent for Windows](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/configuring-kinesis-agent-windows.html)

- [Basic Configuration Structure](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/basic-configuration-structure.html): Overview of the basic structure of the configuration file in Amazon Kinesis Agent for Microsoft Windows.
- [Source Declarations](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/source-object-declarations.html): Configure source declarations for Amazon Kinesis Agent for Microsoft Windows.
- [Sink Declarations](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/sink-object-declarations.html): Configure sink declarations for Amazon Kinesis Agent for Microsoft Windows.
- [Pipe Declarations](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/pipe-object-declarations.html): Configure pipe declarations for Amazon Kinesis Agent for Microsoft Windows.
- [Configuring Automatic Updates](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/update-configuration-options.html): Use the appsettings.json configuration file to automatically update the configuration file and Kinesis Agent for Windows.
- [Kinesis Agent for Windows Configuration Examples](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/configuring-kaw-examples.html): Configuration examples for Kinesis Agent for Windows.
- [Configuring Telemetrics](https://docs.aws.amazon.com/kinesis-agent-windows/latest/userguide/telemetrics-configuration-option.html): Use the telemetrics configuration option to control whether telemetry is gathered and transmitted to AWS using Kinesis Agent for Windows.
