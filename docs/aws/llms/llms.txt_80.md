# AWS Amplify Admin UI API Reference

> This guide is provided for reference purposes, and we strongly recommend that you use the Amplify Admin UI to manage the backend your Amplify app. For more information, see Introduction in the Amplify Admin UI Docs.

## [Resources](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/resources.html)

- [Backend](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend.html): All Amplify apps have a backend.
- [Backend appId Api](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api.html): This resource is an API that's used to provision AWS resources.
- [Backend appId Api backendEnvironmentName](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api-backendenvironmentname.html): The data model for your Amplify project.
- [Backend appId Api backendEnvironmentName Details](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api-backendenvironmentname-details.html): This contains resource-level configuration details for your configured data model.
- [Backend appId Api backendEnvironmentName GenerateModels](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api-backendenvironmentname-generatemodels.html): The generated datastore models for your Amplify app.
- [Backend appId Api backendEnvironmentName GetModels](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api-backendenvironmentname-getmodels.html): The generated datastore models for your Amplify app.
- [Backend appId Api backendEnvironmentName Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-api-backendenvironmentname-remove.html): The data model from your Amplify project.
- [Backend appId Auth](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-auth.html): The authentication and authorization settings from your Amplify project.
- [Backend appId Auth backendEnvironmentName](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-auth-backendenvironmentname.html): The authentication and authorization settings from your Amplify project.
- [Backend appId Auth backendEnvironmentName Details](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-auth-backendenvironmentname-details.html): The authentication and authorization settings from your Amplify project.
- [Backend appId Auth backendEnvironmentName Import](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-auth-backendenvironmentname-import.html): Imports an existing backend authentication resource.
- [Backend appId Auth backendEnvironmentName Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-auth-backendenvironmentname-remove.html): The authentication and authorization settings from your Amplify project.
- [Backend appId Challenge](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-challenge.html): The one-time challenge code used to authenticate a user into your Amplify Admin UI.
- [Backend appId Challenge sessionId](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-challenge-sessionid.html): The one-time challenge code used to authenticate a user into your Amplify Admin UI.
- [Backend appId Challenge sessionId Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-challenge-sessionid-remove.html): The one-time challenge code used to authenticate a user into your Amplify Admin UI.
- [Backend appId Config](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-config.html): A configuration object that contains the authentication resources required for a user to access the Amplify Admin UI.
- [Backend appId Config Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-config-remove.html): A configuration object that contains the authentication resources required for a user to access the Amplify Admin UI.
- [Backend appId Config Update](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-config-update.html): A configuration object that contains the authentication resources required for a user to access the Amplify Admin UI.
- [Backend appId Details](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-details.html): Project-level details for your Amplify project.
- [Backend appId Environments backendEnvironmentName Clone](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-environments-backendenvironmentname-clone.html): A clone of an existing environment in your Amplify project.
- [Backend appId Environments backendEnvironmentName Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-environments-backendenvironmentname-remove.html): An existing environment in your Amplify project.
- [Backend appId Job backendEnvironmentName](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-job-backendenvironmentname.html): Jobs perform backend API actions on your behalf based on your choices in the Amplify Admin UI.
- [Backend appId Job backendEnvironmentName jobId](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-job-backendenvironmentname-jobid.html): Jobs perform backend API actions on your behalf based on your choices in the Amplify Admin UI.
- [Backend appId Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-remove.html): All Amplify apps have a backend.
- [Backend appId Storage](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-storage.html): Create a storage resource in your Amplify app backend.
- [Backend appId Storage backendEnvironmentName](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-storage-backendenvironmentname.html): Update a storage resource in your Amplify app backend.
- [Backend appId Storage backendEnvironmentName Details](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-storage-backendenvironmentname-details.html): Get the details for a storage resource in your Amplify app backend.
- [Backend appId Storage backendEnvironmentName Import](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-storage-backendenvironmentname-import.html): Import a storage resource into your Amplify app backend.
- [Backend appId Storage backendEnvironmentName Remove](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/backend-appid-storage-backendenvironmentname-remove.html): Remove a storage resource from your Amplify app backend.
- [S3Buckets](https://docs.aws.amazon.com/amplify-admin-ui/latest/APIReference/s3buckets.html): List the Amazon S3 buckets in your AWS account.
