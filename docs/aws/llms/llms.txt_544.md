# Amazon Macie REST API Reference

## [Resources](https://docs.aws.amazon.com/macie/latest/APIReference/resources.html)

- [Account Administration](https://docs.aws.amazon.com/macie/latest/APIReference/macie.html): The Account Administration resource provides access to the status of your Amazon Macie account, and certain configuration settings for the account.
- [Administrator](https://docs.aws.amazon.com/macie/latest/APIReference/administrator.html): If your account is an Amazon Macie member account in an organization, the Administrator resource provides information about the Macie administrator account for your account.
- [Administrator Disassociation](https://docs.aws.amazon.com/macie/latest/APIReference/administrator-disassociate.html): The Administrator Disassociation resource provides access to the association between your Amazon Macie account and its Macie administrator account.
- [Allow List](https://docs.aws.amazon.com/macie/latest/APIReference/allow-lists-id.html): The Allow List resource provides access to existing allow lists for your Amazon Macie account.
- [Allow Lists](https://docs.aws.amazon.com/macie/latest/APIReference/allow-lists.html): In Amazon Macie, an allow list defines specific text or a text pattern that you want Macie to ignore when it inspects a data source for sensitive data.
- [AWS Organizations - Macie Administrator](https://docs.aws.amazon.com/macie/latest/APIReference/admin.html): The Macie Administrator resource for AWS Organizations provides settings for designating the delegated Amazon Macie administrator account for an organization in AWS Organizations.
- [AWS Organizations - Macie Configuration](https://docs.aws.amazon.com/macie/latest/APIReference/admin-configuration.html): The Macie Configuration resource for AWS Organizations provides access to certain Amazon Macie configuration settings for an organization in AWS Organizations.
- [Automated Sensitive Data Discovery - Accounts](https://docs.aws.amazon.com/macie/latest/APIReference/automated-discovery-accounts.html): The Accounts resource for automated sensitive data discovery provides access to the status of automated sensitive data discovery for accounts that are centrally managed as an organization in Amazon Macie.
- [Automated Sensitive Data Discovery - Configuration](https://docs.aws.amazon.com/macie/latest/APIReference/automated-discovery-configuration.html): The Configuration resource for automated sensitive data discovery provides access to configuration settings for performing automated sensitive data discovery, and the status of the configuration.
- [Classification Job](https://docs.aws.amazon.com/macie/latest/APIReference/jobs-jobid.html): The Classification Job resource provides detailed information about individual classification jobs for your Amazon Macie account.
- [Classification Job Creation](https://docs.aws.amazon.com/macie/latest/APIReference/jobs.html): The Classification Job Creation resource represents the collection of settings that define the scope and schedule for a classification job.
- [Classification Job List](https://docs.aws.amazon.com/macie/latest/APIReference/jobs-list.html): The Classification Job List resource provides a subset of information about the classification jobs for your Amazon Macie account.
- [Classification Results - Export Configuration](https://docs.aws.amazon.com/macie/latest/APIReference/classification-export-configuration.html): The Export Configuration resource for classification results provides access to settings for storing data classification results in an Amazon Simple Storage Service (Amazon S3) bucket.
- [Classification Scope](https://docs.aws.amazon.com/macie/latest/APIReference/classification-scopes-id.html): The Classification Scope resource provides access to the classification scope settings for your Amazon Macie account.
- [Classification Scopes](https://docs.aws.amazon.com/macie/latest/APIReference/classification-scopes.html): The Classification Scopes resource provides a subset of information about the classification scope for your Amazon Macie account.
- [Custom Data Identifier](https://docs.aws.amazon.com/macie/latest/APIReference/custom-data-identifiers-id.html): The Custom Data Identifier resource provides access to the repository of custom data identifiers for your Amazon Macie account.
- [Custom Data Identifier Creation](https://docs.aws.amazon.com/macie/latest/APIReference/custom-data-identifiers.html): A custom data identifier is a set of criteria that you define to detect sensitive data in a data source.
- [Custom Data Identifier Descriptions](https://docs.aws.amazon.com/macie/latest/APIReference/custom-data-identifiers-get.html): The Custom Data Identifier Descriptions resource provides information about the custom data identifiers for your Amazon Macie account.
- [Custom Data Identifier List](https://docs.aws.amazon.com/macie/latest/APIReference/custom-data-identifiers-list.html): The Custom Data Identifier List resource represents the repository of custom data identifiers for your Amazon Macie account.
- [Custom Data Identifier Testing](https://docs.aws.amazon.com/macie/latest/APIReference/custom-data-identifiers-test.html): The Custom Data Identifier Testing resource provides an environment for developing, testing, and refining detection criteria for a custom data identifier.
- [Data Sources - Amazon S3](https://docs.aws.amazon.com/macie/latest/APIReference/datasources-s3.html): The Amazon S3 Data Sources resource provides statistical data and other information about the Amazon Simple Storage Service (Amazon S3) buckets that Amazon Macie monitors and analyzes for your account.
- [Data Sources - Amazon S3 Statistics](https://docs.aws.amazon.com/macie/latest/APIReference/datasources-s3-statistics.html): The Amazon S3 Data Source Statistics resource provides aggregated statistical data for all the Amazon Simple Storage Service (Amazon S3) buckets that Amazon Macie monitors and analyzes for your account.
- [Data Sources - Search](https://docs.aws.amazon.com/macie/latest/APIReference/datasources-search-resources.html): The Search Data Sources resource provides statistical data and other information about AWS resources that Amazon Macie monitors and analyzes for your account.
- [Finding List](https://docs.aws.amazon.com/macie/latest/APIReference/findings.html): The Finding List resource provides a subset of information about the findings for your Amazon Macie account.
- [Finding Samples](https://docs.aws.amazon.com/macie/latest/APIReference/findings-sample.html): The Finding Samples resource provides a set of findings that use example data and placeholder values to help you understand and analyze the types of findings that Amazon Macie can generate.
- [Finding Statistics](https://docs.aws.amazon.com/macie/latest/APIReference/findings-statistics.html): The Finding Statistics resource provides aggregated statistical data about the findings for your Amazon Macie account.
- [Findings](https://docs.aws.amazon.com/macie/latest/APIReference/findings-describe.html): The Findings resource represents the repository of findings for your Amazon Macie account.
- [Findings - Publication Configuration](https://docs.aws.amazon.com/macie/latest/APIReference/findings-publication-configuration.html): The Publication Configuration resource for findings provides settings for publishing findings to AWS Security Hub CSPM.
- [Findings - Reveal Sensitive Data Occurrences](https://docs.aws.amazon.com/macie/latest/APIReference/findings-findingid-reveal.html): The Reveal Sensitive Data Occurrences resource provides options for retrieving sample occurrences of sensitive data that Amazon Macie reported in a finding.
- [Findings - Reveal Sensitive Data Occurrences Availability](https://docs.aws.amazon.com/macie/latest/APIReference/findings-findingid-reveal-availability.html): The Reveal Sensitive Data Occurrences Availability resource provides an environment for determining whether you can retrieve sample occurrences of sensitive data that Amazon Macie reported in a finding.
- [Findings - Reveal Sensitive Data Occurrences Configuration](https://docs.aws.amazon.com/macie/latest/APIReference/reveal-configuration.html): The Reveal Sensitive Data Occurrences Configuration resource provides access to settings for retrieving sample occurrences of sensitive data that Amazon Macie reports in findings.
- [Findings Filter](https://docs.aws.amazon.com/macie/latest/APIReference/findingsfilters-id.html): The Findings Filter resource represents an individual filter that you created and saved to review, analyze, and manage findings.
- [Findings Filters](https://docs.aws.amazon.com/macie/latest/APIReference/findingsfilters.html): The Findings Filters resource represents the repository of filters that you create and save to review, analyze, and manage findings.
- [Invitation Acceptance](https://docs.aws.amazon.com/macie/latest/APIReference/invitations-accept.html): In Amazon Macie, an invitation, also referred to as a membership invitation, is a request to become a member of an organization in Macie.
- [Invitation Count](https://docs.aws.amazon.com/macie/latest/APIReference/invitations-count.html): In Amazon Macie, an invitation, also referred to as a membership invitation, is a request to become a member of an organization in Macie.
- [Invitation Decline](https://docs.aws.amazon.com/macie/latest/APIReference/invitations-decline.html): In Amazon Macie, an invitation, also referred to as a membership invitation, is a request to become a member of an organization in Macie.
- [Invitation Deletion](https://docs.aws.amazon.com/macie/latest/APIReference/invitations-delete.html): In Amazon Macie, an invitation, also referred to as a membership invitation, is a request to become a member of an organization in Macie.
- [Invitation List](https://docs.aws.amazon.com/macie/latest/APIReference/invitations.html): In Amazon Macie, an invitation, also referred to as a membership invitation, is a request to become a member of an organization in Macie.
- [Managed Data Identifiers](https://docs.aws.amazon.com/macie/latest/APIReference/managed-data-identifiers-list.html): The Managed Data Identifiers resource represents the repository of managed data identifiers that Amazon Macie currently provides.
- [Master Account](https://docs.aws.amazon.com/macie/latest/APIReference/master.html): This resource has been deprecated and is retained only for backward compatibility.
- [Master Disassociation](https://docs.aws.amazon.com/macie/latest/APIReference/master-disassociate.html): This resource has been deprecated and is retained only for backward compatibility.
- [Member](https://docs.aws.amazon.com/macie/latest/APIReference/members-id.html): The Member resource provides information about an individual account that's currently associated with your Amazon Macie account, typically a Macie administrator account.
- [Member Disassociation](https://docs.aws.amazon.com/macie/latest/APIReference/members-disassociate-id.html): The Member Disassociation resource provides access to associations between an Amazon Macie administrator account and its member accounts.
- [Member Status](https://docs.aws.amazon.com/macie/latest/APIReference/macie-members-id.html): The Member Status resource provides access to the status of Amazon Macie for a member account in an organization.
- [Members](https://docs.aws.amazon.com/macie/latest/APIReference/members.html): The Members resource provides information about all the accounts that are currently associated with your Amazon Macie account, typically a Macie administrator account.
- [Resource Sensitivity Profile](https://docs.aws.amazon.com/macie/latest/APIReference/resource-profiles.html): The Resource Sensitivity Profile resource provides statistical data for sensitive data discovery metrics that apply to individual Amazon Simple Storage Service (Amazon S3) buckets for your account.
- [Resource Sensitivity Profile - Artifacts](https://docs.aws.amazon.com/macie/latest/APIReference/resource-profiles-artifacts.html): The Resource Sensitivity Profile Artifacts resource provides information about Amazon Simple Storage Service (Amazon S3) objects that Amazon Macie selected for analysis from individual S3 buckets for your account.
- [Resource Sensitivity Profile - Detections](https://docs.aws.amazon.com/macie/latest/APIReference/resource-profiles-detections.html): The Resource Sensitivity Profile Detections resource provides information about the types and amount of sensitive data that Amazon Macie has found in individual Amazon Simple Storage Service (Amazon S3) buckets for your account.
- [Sensitivity Inspection Template](https://docs.aws.amazon.com/macie/latest/APIReference/templates-sensitivity-inspections-id.html): The Sensitivity Inspection Template resource provides access to the sensitivity inspection template for your Amazon Macie account.
- [Sensitivity Inspection Templates](https://docs.aws.amazon.com/macie/latest/APIReference/templates-sensitivity-inspections.html): The Sensitivity Inspection Templates resource provides a subset of information about the sensitivity inspection template for your Amazon Macie account.
- [Tags](https://docs.aws.amazon.com/macie/latest/APIReference/tags-resourcearn.html): A tag is a label that you can define and associate with AWS resources, including certain types of Amazon Macie resources.
- [Usage Statistics](https://docs.aws.amazon.com/macie/latest/APIReference/usage-statistics.html): The Usage Statistics resource provides information about current quotas and usage statistics for your Amazon Macie account.
- [Usage Totals](https://docs.aws.amazon.com/macie/latest/APIReference/usage.html): The Usage Totals resource provides aggregated usage data for your Amazon Macie account.
