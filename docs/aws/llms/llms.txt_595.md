## [Migrating to Amazon Pinpoint](https://docs.aws.amazon.com/mobileanalytics/latest/ug/migrate.html)

- [Migrating to the Mobile SDKs or JavaScript Library](https://docs.aws.amazon.com/mobileanalytics/latest/ug/migrate-sdk.html): Switch from using the Amazon Mobile Analytics SDK client to the Amazon Pinpoint SDK client.
- [Migrating to the Amazon Pinpoint API](https://docs.aws.amazon.com/mobileanalytics/latest/ug/migrate-api.html): The deprecated Mobile Analytics API automatically redirects requests to the Amazon Pinpoint API.
- [Migrating to the Amazon Pinpoint Console](https://docs.aws.amazon.com/mobileanalytics/latest/ug/migrate-console.html): The Mobile Analytics console redirects to the Amazon Pinpoint console.
