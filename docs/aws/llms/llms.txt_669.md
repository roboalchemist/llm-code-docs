# Porting Assistant for .NET User Guide

> Porting Assistant for .NET is a new compatibility analyzer that reduces the manual effort required to port Microsoft .NET Framework applications to open source .NET Core. Porting Assistant for .NET assesses the .NET application source code, identifies incompatible APIs and third-party packages, and, where applicable, provides replacement suggestions that are compatible with .NET Core.
