# Amazon Virtual Private Cloud Traffic Mirroring

> Traffic Mirroring is a feature of Amazon Virtual Private Cloud (Amazon VPC) that you use to gain insight and access to the network traffic across your VPC infrastructure. Traffic Mirroring is a feature of Amazon Virtual Private Cloud (Amazon VPC) that you use to gain insight and access to the network traffic across your VPC infrastructure. You use Traffic Mirroring to inspect content and monitor threats. You can copy network traffic at any elastic network interface in your VPC, and send it to security and monitoring tools for analysis.
