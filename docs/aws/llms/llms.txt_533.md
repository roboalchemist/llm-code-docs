# AWS Local Zones User Guide

> AWS Local Zones are a type of infrastructure deployment that places compute, storage, database, and other select AWS services close to large population and industry centers.

## [Connectivity options](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity.html)

- [Internet gateway](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-igw.html): Learn how you can use the internet-gateway to connect to resources in your Local Zone.
- [NAT gateway](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-nat.html): Learn how you can use a NAT gateway to connect to resources in your Local Zone.
- [VPN](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-ec2-vpn.html): Learn how you can use a VPN to connect to resources in your Local Zone.
- [Direct Connect](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-direct-connect.html): Learn how you can use Direct Connect to manage the connection to your Local Zone.
- [Transit gateway between Local Zones](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-transit-gateway-lzs.html): Learn how you can use a transit gateway to connect between Local Zones.
- [Transit gateway to data center](https://docs.aws.amazon.com/local-zones/latest/ug/local-zones-connectivity-transit-gateway-dc.html): Learn how you can use transit gateway to connect data centers to a Local Zone.
