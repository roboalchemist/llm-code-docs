# AWS User Notifications Contacts API Reference

> AWS User Notifications Contacts is a service that allows you to create and manage email contacts for AWS User Notifications. The AWS User Notifications Contacts API Reference provides descriptions, API request parameters, and the JSON response for all email contact related API actions.

- [Welcome](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/Welcome.html)
- [Common Parameters](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/CommonParameters.html)
- [Common Errors](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/CommonErrors.html)

## [Actions](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_Operations.html)

- [ActivateEmailContact](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_ActivateEmailContact.html): Activates an email contact using an activation code.
- [CreateEmailContact](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_CreateEmailContact.html): Creates an email contact for the provided email address.
- [DeleteEmailContact](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_DeleteEmailContact.html): Deletes an email contact.
- [GetEmailContact](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_GetEmailContact.html): Returns an email contact.
- [ListEmailContacts](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_ListEmailContacts.html): Lists all email contacts created under the Account.
- [ListTagsForResource](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_ListTagsForResource.html): Lists all of the tags associated with the Amazon Resource Name (ARN) that you specify.
- [SendActivationCode](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_SendActivationCode.html): Sends an activation email to the email address associated with the specified email contact.
- [TagResource](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_TagResource.html): Attaches a key-value pair to a resource, as identified by its Amazon Resource Name (ARN).
- [UntagResource](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_UntagResource.html): Detaches a key-value pair from a resource, as identified by its Amazon Resource Name (ARN).


## [Data Types](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_Types.html)

- [EmailContact](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_EmailContact.html): An email contact.
- [ValidationExceptionField](https://docs.aws.amazon.com/notificationscontacts/latest/APIReference/API_ValidationExceptionField.html): Stores information about a field passed inside a request that resulted in an exception.
