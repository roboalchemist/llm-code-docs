# AWS IoT FleetWise Welcome

> AWS IoT FleetWise is a fully managed service that you can use to collect, model, and transfer vehicle data to the AWS cloud at scale. With AWS IoT FleetWise, you can standardize all of your vehicle data models, independent of the in-vehicle communication architecture, and define data collection rules to transfer only high-value data to the cloud.
