- [Architecture](https://docs.aws.amazon.com/rosa/latest/userguide/rosa-architecture-models.html): Learn about the differences between the ROSA with hosted control planes and ROSA classic architecture.


## [Get started with ROSA](https://docs.aws.amazon.com/rosa/latest/userguide/getting-started.html)

- [Set up](https://docs.aws.amazon.com/rosa/latest/userguide/set-up.html): Set up the AWS resources that are required to create a ROSA cluster.
- [Create a ROSA HCP cluster - CLI](https://docs.aws.amazon.com/rosa/latest/userguide/getting-started-hcp.html): Learn how to create your first ROSA with HCP cluster using the ROSA CLI, grant user access, deploy your first application, revoke access, and delete your cluster.
- [Create a ROSA classic cluster - CLI](https://docs.aws.amazon.com/rosa/latest/userguide/getting-started-classic-cli.html): Learn how to create your first ROSA classic cluster using the ROSA CLI, grant user access, deploy your first application, revoke access, and delete your cluster.
- [Create a ROSA classic cluster - AWS PrivateLink](https://docs.aws.amazon.com/rosa/latest/userguide/getting-started-classic-private-link.html): Learn how to create your first ROSA with HCP cluster using the ROSA CLI, grant user access, deploy your first application, revoke access, and delete your cluster.


## [Security](https://docs.aws.amazon.com/rosa/latest/userguide/security.html)

### [Data protection](https://docs.aws.amazon.com/rosa/latest/userguide/data-protection.html)

Learn how the AWS shared responsibility model applies to data protection in ROSA.

### [Data encryption](https://docs.aws.amazon.com/rosa/latest/userguide/data-protection-encryption.html)

Learn how ROSA protects your data in transit and at rest.

- [Key management](https://docs.aws.amazon.com/rosa/latest/userguide/data-protection-key-management.html): Learn how ROSA uses AWS KMS to securely manage keys for encrypted data.

### [Identity and access management](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam.html)

How to authenticate requests and manage access to your ROSA resources.

### [ROSA identity-based policy examples](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam-id-based-policy-examples.html)

Learn about example identity-based policies for ROSA.

- [ROSA classic account policies](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam-rosa-classic-account-policies.html): Learn about the account policies that ROSA classic uses to manage cluster resources and enable Red Hat site reliability engineer support for clusters.
- [ROSA classic operator policies](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam-rosa-classic-operator-policies.html): Learn about the operator policies that ROSA classic uses to allow the OpenShift operators to manage ROSA classic cluster nodes.
- [AWS managed policies](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam-awsmanpol.html): Learn about AWS managed policies for ROSA and recent changes to those policies.
- [Troubleshooting](https://docs.aws.amazon.com/rosa/latest/userguide/security-iam-troubleshoot.html): Troubleshoot common errors while using ROSA with IAM.
- [Resilience](https://docs.aws.amazon.com/rosa/latest/userguide/disaster-recovery-resiliency.html): Learn how AWS architecture supports data redundancy, and learn about specific ROSA features for data resiliency.
- [Infrastructure security](https://docs.aws.amazon.com/rosa/latest/userguide/infrastructure-security.html): Learn how network security is handled for ROSA cloud infrastructure, clusters, and pods.
