- [Creating and configuring an estimate](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/create-configure-estimate.html): Learn about creating and configuring estimates on AWS Pricing Calculator, including adding services, groups, and customizing specifications for accurate cost projections.

## [Managing estimates](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/generate-your-estimate.html)

- [Grouping your estimates](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/estimate-groups.html): You can organize your AWS estimates by defining groups.
- [Sharing your estimate](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/save-share-estimate.html): You can save the unique link for each estimate to share or revisit directly through your browser.
- [Exporting your estimates](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/export-estimate.html): Learn about exporting your AWS Pricing Calculator estimates as PDF or CSV files, allowing you to revisit the parameters used for cost estimation and service setup in the AWS console.
- [Migrating from Simple Monthly Calculator](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/migrate-SMC.html): Learn about migrating your Simple Monthly Calculator (SMC) estimates to the AWS Pricing Calculator

- [Estimate tutorial for Windows and SQL Servers on EC2](https://docs.aws.amazon.com/pricing-calculator/latest/userguide/estimate-workload-tutorial.html): Use this tutorial to generate a pricing estimate using the Windows Server and SQL Server on Amazon EC2 in AWS Pricing Calculator.
