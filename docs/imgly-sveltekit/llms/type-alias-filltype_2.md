# Type Alias: FillType

```
type FillType =  | FillTypeShorthand  | FillTypeLonghand;
```

The block type IDs for the fill blocks. These are the IDs used to create new fills using `cesdk.engine.block.createFill(id)`. Refer to [FillTypeShorthand](https://img.ly/docs/cesdk/sveltekit/api/engine/type-aliases/filltypeshorthand/) and [FillTypeLonghand](https://img.ly/docs/cesdk/sveltekit/api/engine/type-aliases/filltypelonghand/) for more details.
