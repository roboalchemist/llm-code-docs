# Type Alias: DesignBlockType

```
type DesignBlockType =  | DesignBlockTypeShorthand  | DesignBlockTypeLonghand;
```

The block type IDs for the top-level design blocks. These are the IDs used to create new blocks using `cesdk.engine.block.create(id)`. Refer to [DesignBlockTypeShorthand](https://img.ly/docs/cesdk/sveltekit/api/engine/type-aliases/designblocktypeshorthand/) and [DesignBlockTypeLonghand](https://img.ly/docs/cesdk/sveltekit/api/engine/type-aliases/designblocktypelonghand/) for more details.
