# Interface: RGBAColor

Represents an RGBA color value.

Defines an RGBA color value with components between 0 and 1.

*   ‘r’: The red component.
*   ‘g’: The green component.
*   ‘b’: The blue component.
*   ‘a’: The alpha component.

## Properties[#](#properties)

| Property | Type | Description |
| --- | --- | --- |
| `r` | `number` | Red |
| `g` | `number` | Green |
| `b` | `number` | Blue |
| `a` | `number` | Alpha |
