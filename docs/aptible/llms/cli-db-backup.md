# aptible db:backup

This command is used to create [Database Backups](/core-concepts/managed-databases/managing-databases/database-backups).

## Synopsis

```
Usage:
  aptible db:backup HANDLE

Options:
  --env, [--environment=ENVIRONMENT]
```

# Examples

```shell  theme={null}
aptible db:backup "$DB_HANDLE"
```
