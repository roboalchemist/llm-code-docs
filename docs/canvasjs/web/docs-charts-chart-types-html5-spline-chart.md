HTML5 & JS Spline Charts
========================

Spline Charts are much like line charts except that the dataPoints are connected using smooth curves.

**Cannot be combined with:** Doughnut, Pie, Bar, Stacked Bar charts.

* * *

Important Properties
--------------------

| Applies To | Attribute | Type | Default | Options/Examples |
| --- | --- | --- | --- | --- |
| dataSeries | [lineThickness](https://canvasjs.com/docs/charts/chart-options/data/linethickness/) | String | 2 | 4,6.. |
| dataSeries/ dataPoint | [markerSize](https://canvasjs.com/docs/charts/chart-options/data/markersize/) | Number | auto. | 5,10.. |
| dataSeries/ dataPoint | [markerType](https://canvasjs.com/docs/charts/chart-options/data/markertype/) | String | “circle” | “none”, “circle”, “square”, “cross”, “triangle” |
