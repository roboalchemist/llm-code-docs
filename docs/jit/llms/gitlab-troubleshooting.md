# Gitlab Troubleshooting

# Overview

This guide is here to help you address common issues with the integration with GitLab.

## Jit is not running properly

Jit works by creating a centralized configuration repository in your GitLab Organization. There are a few reasons Jit is not running on your GitLab Organization:

* [GitLab Runners being disabled](https://docs.jit.io/docs/gitlab-runners-are-disabled)
* [Gitlab Organization is using self-hosted runners](https://docs.jit.io/docs/self-hosted-runners-requirements)
