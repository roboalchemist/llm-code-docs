# GitHub Troubleshooting

# Overview

This guide is here to help you address common issues with the integration with GitHub.

## Jit is not running properly

Jit works by creating a centralized configuration repository in your GitHub Organization. There are a few reasons Jit is not running on your GitHub Organization:

* [Branch Protection](https://docs.jit.io/docs/a)
* [GitHub Actions being disabled](https://docs.jit.io/docs/github-actions-are-disabled)
* [GitHub Organization has ran out of Action minutes](https://docs.jit.io/docs/add-minutes-to-github-actions)
* [GitHub Organization is using self-hosted runners](https://docs.jit.io/docs/jit-on-github-self-hosted-runners)
* [GitHub has an outage](https://docs.jit.io/docs/github-outage)
