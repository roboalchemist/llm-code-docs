# Jit Docs Home

Welcome to Jit Docs! Here, you can learn how to onboard and use Jit's Product Security Platform to secure your applications in the cloud.

> 📘 Recent updates
>
> We’ve upgraded Jit’s scanning architecture to improve performance and developer experience.\
> This change has already been rolled out to most customers.
>
> * **Unified PR/MR check**\
>   Jit now reports a single “**Jit Security**” check in pull/merge requests, replacing the multiple Jit checks used previously.
> * **Improved AI-powered code reviews**\
>   Clearer insights with a single, unified PR/MR comment and enhanced security and code-quality suggestions.
> * **Scans run in Jit’s environment**\
>   Reduces CI load and infrastructure costs on your side.
> * **Configuration is now managed in the Jit platform**\
>   Configuration previously defined via a **centralized repository** or **as-code** has been automatically synced to Jit.
>   Going forward, configuration should be managed directly in the Jit platform. <br />
>   :information\_source: If anything is unclear or behaves differently than expected, please contact us.
