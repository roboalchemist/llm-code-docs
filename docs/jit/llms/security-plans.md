# Product Security Plans

Different product security plans are available, depending on your organization's unique security needs.

Jit currently supports the following security plans:

* [Jit MVS for AppSec Plan](https://docs.jit.io/docs/jit-mvs-for-appsec-plan)
* [AWS Foundational Technical Review (FTR)](https://docs.jit.io/docs/aws-foundational-technical-review-ftr)
* [GitHub Security Plan](https://docs.jit.io/docs/github-security-plan) - Coming soon! :rocket:
