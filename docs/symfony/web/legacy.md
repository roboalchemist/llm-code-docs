symfony 1.x legacy website
==========================

The symfony 1.x branch of the framework started in 2005 and the latest stable version and long term support release is symfony 1.4. As of November 2012, **symfony 1.x is not maintained anymore** and you are encouraged to have a look at [newer Symfony versions](https://symfony.com/releases) for any new projects.

[Download symfony 1.x legacy versions](https://symfony.com/legacy#download-symfony-1-x-legacy-versions "Permanent link to this headline")

### [Practical symfony](https://symfony.com/legacy#practical-symfony "Permanent link to this headline")

**Learn symfony:** 24 tutorials of 1 hour each, that's all it takes to build up a complete and effective application from scratch. Definitely the best way to become a good symfony developer!

[Read](https://symfony.com/legacy/doc/jobeet?orm=Propel)

![Image 12: Cover of the A gentle Introduction to symfony book](https://symfony.com/images/covers/9782918390305-3d.jpg)

### [A gentle Introduction to symfony](https://symfony.com/legacy#a-gentle-introduction-to-symfony "Permanent link to this headline")

Discover symfony: Read this book to get an overview of symfony. This book introduces you to symfony, showing you how to wield its many features to develop web applications faster and more efficiently, even if you only know a bit of PHP.

[Read](https://symfony.com/legacy/doc/gentle-introduction)

![Image 13: Cover of the The symfony Reference Book book](https://symfony.com/images/covers/9782918390145-3d.jpg)

### [The symfony Reference Book](https://symfony.com/legacy#the-symfony-reference-book "Permanent link to this headline")

**The reference guide:** The Symfony Reference Guide is a book where you can easily find answers to your questions at your fingertips. This is a book you will keep with you whenever you develop with symfony.

[Read](https://symfony.com/legacy/doc/reference)

![Image 14: Cover of the More with symfony book](https://symfony.com/images/covers/9782918390176-3d.jpg)

### [More with symfony](https://symfony.com/legacy#more-with-symfony "Permanent link to this headline")

**Do more with symfony:** This book is the symfony 2009 advent calendar, a set of 24 tutorials about advanced symfony topics, published day-by-day between December 1st and Christmas 2009. All tutorials are available in five languages: English, French, Spanish, Italian, and Japanese.

[Read](https://symfony.com/legacy/doc/more-with-symfony)
