[Overview](https://doc.qt.io/coco/overview.html)

Coco is a complete [code coverage](https://doc.qt.io/coco/code-coverage-analysis.html#cha-code-coverage-analysis) tool chain for Tcl, QML, C# and C/C++ programs (including [SystemC](http://www.systemc.org/) programs). It runs on macOS, Linux™ and Microsoft® Windows. For a full list of supported compilers, see [the company website](https://www.qt.io/product/quality-assurance/coco/feature-platforms-compilers).
