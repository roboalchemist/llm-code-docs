# Supported File Types

### Upload Formats (Knowledge Base)

| **Format** | **Extension** | **Support Level** | **Max Size** | **Notes** | 
| ---- | ---- | ---- | ---- | ---- | 
| PDF | .pdf | Full | 25MB | Max 500 pages | 
| Word | .doc, .docx | Full | 25MB | Max 200 pages | 
| Text | .txt | Full | 25MB | Plain text | 
| Markdown | .md | Full | 25MB | Formatted text | 
| Excel | .xls, .xlsx | Full | 25MB | Text extraction only | 
| PowerPoint | .pptx | Partial | 25MB | Text only, no images | 
| JSON | .json | Experimental | 10MB | Structured data | 
| XML | .xml | Experimental | 10MB | Structured data | 


### Export Formats

| **Resource** | **Format** | **Contains** | 
| ---- | ---- | ---- | 
| Conversation | PDF | Full conversation, formatting, timestamps | 
| Notebook Results | PDF, CSV, JSON | Query results and metadata | 
| Workflow Results | JSON | Execution details and outputs | 
| Audit Logs | CSV, JSON | User activity logs |
