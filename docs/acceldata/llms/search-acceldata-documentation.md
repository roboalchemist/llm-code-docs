# Search Acceldata Documentation

Acceldata documentation can be searched and accessed using **ChatGPT** or **Cursor** by connecting to the ADOC, Pulse, and ODP MCP servers. This allows you to query the docs directly using natural language and get contextual, accurate answers.

- ChatGPT acts as an AI interface, with Acceldata docs available as an app via an MCP connector. For configuration details, see [Search Docs in ChatGPT Using MCP Server](https://docs.acceldata.io/pulse/user-guide/search-docs-in-chatgpt-using-mcp-server).
- **Cursor** is an AI-powered code editor that integrates Acceldata docs via MCP servers. For configuration details, see [Search Docs in Cursor Using MCP Server](https://docs.acceldata.io/pulse/user-guide/search-docs-in-cursor-using-mcp-server).
