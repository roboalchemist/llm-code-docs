### [Bug Reporting](https://developer.apple.com/documentation/ios-ipados-release-notes#Bug-Reporting)

For issues not mentioned in release notes, send feedback through [Feedback Assistant](applefeedback://).

When writing your report, please include the full version number in the title and in the description. This looks like “13 (17A_xxxx_).” You can find the full version number in Settings > General > About > Version. For more information, visit [Bug Reporting](https://developer.apple.com/bug-reporting/).
