# Show Prompt Options

This API is used to show the Luciq invocation menu manually and does not take any arguments.

{% tabs fullWidth="true" %}
{% tab title="iOS - Swift" %}

```swift
//Invoke Luciq
Luciq.show()
```

{% endtab %}

{% tab title="iOS - ObjC" %}

```objectivec
//Invoke Luciq
[Luciq show];
```

{% endtab %}

{% tab title="And - Java" %}

```java
Luciq.show();
```

{% endtab %}

{% tab title="And - Kotlin" %}

```kotlin
Luciq.show()
```

{% endtab %}

{% tab title="RN" %}

```javascript
Luciq.show();
```

{% endtab %}

{% tab title="Flutter" %}

```csharp
Luciq.show();
```

{% endtab %}
