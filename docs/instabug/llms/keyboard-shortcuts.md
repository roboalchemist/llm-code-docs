# Keyboard Shortcuts

### Bug Reports Shortcuts

You can use the following shortcuts while on the **Bugs** page.

| Shortcut     | Action                                                  |
| ------------ | ------------------------------------------------------- |
| down         | Go to next bug                                          |
| up           | Go to previous bug                                      |
| 1            | Change status of the bug to "Open"                      |
| 2            | Change status of the bug to "In-progress"               |
| 3            | Change status of the bug to "Closed"                    |
| d            | Mark bug as a duplicate                                 |
| esc          | Close drop-downs or chat boxes and go back to bugs list |
| alt + r      | Open the reply to user chat                             |
| space        | Assign issue to yourself                                |
| ctrl + enter | Send message                                            |
| r            | Change focus to chat text box                           |

### Crash Reports Shortcuts

#### **Viewing a Specific Crash Report**

You can use the following shortcuts while viewing a particular crash.

| Shortcut | Action                                                     |
| -------- | ---------------------------------------------------------- |
| 1        | Change status of the crash to "Open"                       |
| 2        | Change status of the crash to "In-progress"                |
| 3        | Change status of the crash to "Closed"                     |
| d        | Mark crash as a duplicate                                  |
| f        | Forward issue to 3rd party integration                     |
| esc      | Close drop-downs or chat boxes and go back to crash report |
| down     | Go to next crash                                           |
| up       | Go to previous crash                                       |

#### **Viewing Occurrences**

You can use the following shortcuts while viewing occurrences of a particular crash.

| Shortcut | Action                        |
| -------- | ----------------------------- |
| left     | Go to the previous occurrence |
| right    | Go to the next occurrence     |
