# End App Launch

You can use the following APIs to define a specific point in time where the app launch can be considered complete.

{% tabs fullWidth="true" %}
{% tab title="iOS - Swift" %}

```swift
APM.endAppLaunch()
```

{% endtab %}

{% tab title="iOS - ObjC" %}

```objectivec
[LCQAPM endAppLaunch]
```

{% endtab %}

{% tab title="And - Java" %}

```java
APM.endAppLaunch();
```

{% endtab %}

{% tab title="And - Kotlin" %}

```kotlin
APM.endAppLaunch()
```

{% endtab %}

{% tab title="RN" %}

```javascript
APM.endAppLaunch()
```

{% endtab %}

{% tab title="Flutter" %}

```javascript
APM.endAppLaunch()
```

{% endtab %}
