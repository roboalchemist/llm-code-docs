# Can I edit HTML, CSS, JS, or other custom code on GitBook?

GitBook does not directly support editing HTML, CSS, JavaScript, or any other custom code within its platform.

You may want to explore integrations to enhance the functionality of your GitBook setup.

You can learn more about the available options on our [integrations page.](https://gitbook.com/docs/integrations/overview)
