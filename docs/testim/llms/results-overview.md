# Results Overview

After running the test or an execution, you can view the results through the following screens:

* [Test results on the Editor](https://help.testim.io/docs/test-results) -  Overall test run information is shown as well as detailed information about each step that was run.
* [Execution runs screen](https://help.testim.io/docs/execution-runs-screen) -  displays information about previous execution runs. These runs are grouped by Suite, Label, or Test plan.
* [Test runs screen](https://help.testim.io/docs/test-runs) - displays aggregated statistics on all the selected tests as well as statistics on specific tests.
