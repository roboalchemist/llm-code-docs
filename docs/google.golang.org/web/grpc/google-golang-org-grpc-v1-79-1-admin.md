### Overview [¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/admin#pkg-overview "Go to Overview")

*   [Experimental](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/admin#hdr-Experimental)

Package admin provides a convenient method for registering a collection of administration services to a gRPC server. The services registered are:

- Channelz: [https://github.com/grpc/proposal/blob/master/A14-channelz.md](https://github.com/grpc/proposal/blob/master/A14-channelz.md)

- CSDS: [https://github.com/grpc/proposal/blob/master/A40-csds-support.md](https://github.com/grpc/proposal/blob/master/A40-csds-support.md)

#### Experimental [¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/admin#hdr-Experimental "Go to Experimental")

Notice: All APIs in this package are experimental and may be removed in a later release.

#### func [Register](https://github.com/grpc/grpc-go/blob/v1.79.1/admin/admin.go#L56)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/admin#Register "Go to Register")

func Register(s [grpc](https://pkg.go.dev/google.golang.org/grpc@v1.79.1).[ServiceRegistrar](https://pkg.go.dev/google.golang.org/grpc@v1.79.1#ServiceRegistrar)) (cleanup func(), _ [error](https://pkg.go.dev/builtin#error))

Register registers the set of admin services to the given server.

The returned cleanup function should be called to clean up the resources allocated for the service handlers after the server is stopped.

Note that if `s` is not a *grpc.Server or a *xds.GRPCServer, CSDS will not be registered because CSDS generated code is old and doesn't support interface `grpc.ServiceRegistrar`. [https://github.com/envoyproxy/go-control-plane/issues/403](https://github.com/envoyproxy/go-control-plane/issues/403)
