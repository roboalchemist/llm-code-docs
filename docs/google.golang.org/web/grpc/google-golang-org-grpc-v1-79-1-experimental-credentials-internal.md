### Overview [¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/experimental/credentials/internal#pkg-overview "Go to Overview")

Package internal defines APIs for parsing SPIFFE ID.

All APIs in this package are experimental.

#### func [SPIFFEIDFromCert](https://github.com/grpc/grpc-go/blob/v1.79.1/experimental/credentials/internal/spiffe.go#L45)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/experimental/credentials/internal#SPIFFEIDFromCert "Go to SPIFFEIDFromCert")

func SPIFFEIDFromCert(cert *[x509](https://pkg.go.dev/crypto/x509).[Certificate](https://pkg.go.dev/crypto/x509#Certificate)) *[url](https://pkg.go.dev/net/url).[URL](https://pkg.go.dev/net/url#URL)

SPIFFEIDFromCert parses the SPIFFE ID from x509.Certificate. If the SPIFFE ID format is invalid, return nil with warning.

#### func [SPIFFEIDFromState](https://github.com/grpc/grpc-go/blob/v1.79.1/experimental/credentials/internal/spiffe.go#L36)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/experimental/credentials/internal#SPIFFEIDFromState "Go to SPIFFEIDFromState")

func SPIFFEIDFromState(state [tls](https://pkg.go.dev/crypto/tls).[ConnectionState](https://pkg.go.dev/crypto/tls#ConnectionState)) *[url](https://pkg.go.dev/net/url).[URL](https://pkg.go.dev/net/url#URL)

SPIFFEIDFromState parses the SPIFFE ID from State. If the SPIFFE ID format is invalid, return nil with warning.

#### func [WrapSyscallConn](https://github.com/grpc/grpc-go/blob/v1.79.1/experimental/credentials/internal/syscallconn.go#L49)[¶](https://pkg.go.dev/google.golang.org/grpc@v1.79.1/experimental/credentials/internal#WrapSyscallConn "Go to WrapSyscallConn")

func WrapSyscallConn(rawConn, newConn [net](https://pkg.go.dev/net).[Conn](https://pkg.go.dev/net#Conn)) [net](https://pkg.go.dev/net).[Conn](https://pkg.go.dev/net#Conn)

WrapSyscallConn tries to wrap rawConn and newConn into a net.Conn that implements syscall.Conn. rawConn will be used to support syscall, and newConn will be used for read/write.

This function returns newConn if rawConn doesn't implement syscall.Conn.
