# Guardrails Quick Start

## Follow these steps to begin using Fiddler Guardrails:

1. **Sign up for the Free Guardrails**: Go to[ fiddler.ai/guardrails-trial](https://fiddler.ai/free-guardrails) to register for access.
2. **Create your account**: Check your email for an invitation from Fiddler. Click the "Activate" button to create your account, then sign in. After activation, you can log in using the provided link.

   <figure><img src="https://3170638587-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F82RHcnYWV62fvrxMeeBB%2Fuploads%2Fgit-blob-e89a43fd6cdf1b1763070575bce0c88307b2ba68%2Fguardrail-freetrial-activation-email%20(1).png?alt=media" alt=""><figcaption><p>Fiddler Free Guardrails email confirmation</p></figcaption></figure>
3. **Generate your API key**: After signing in, you'll see the Getting Started page. Select Generate your API key to create the authentication key for Guardrails.

   <figure><img src="https://3170638587-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F82RHcnYWV62fvrxMeeBB%2Fuploads%2Fgit-blob-78ad147da650bb7ae46055ff42afd3b3419be2c7%2Fguardrail-freetrial-landing-page%20(1).png?alt=media" alt=""><figcaption><p>Generate your API key for access to Fiddler Guardrails</p></figcaption></figure>
4. **Explore implementation examples**: Choose any language or framework from the sidebar to view sample code. You can use these examples or any HTTP client of your choice.

   <figure><img src="https://3170638587-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F82RHcnYWV62fvrxMeeBB%2Fuploads%2Fgit-blob-e86cdedc660897510cd3c973d926c0059ab37ae9%2Fguardrail-freetrial-curl-example%20(1).png?alt=media" alt=""><figcaption><p>Code examples in multiple languages</p></figcaption></figure>

Review complete documentation: For API specifications and model details, see the [**Guardrails API Reference**](https://app.gitbook.com/s/rsvU8AIQ2ZL9arerribd/rest-api/guardrails-api-reference).
