# Selecting Enrichments

{% hint style="info" %}
For a complete reference of all LLM enrichments with output columns and sub-metrics, see the [LLM Observability Metrics Reference](https://docs.fiddler.ai/reference/llm-observability-metrics).
{% endhint %}

Fiddler offers enrichments out of the box to monitor different aspects of LLM applications. Use the below table to select the right enrichment for your specific use case.

This table provides high level information on the metric, the enrichment to use to measure the metric, if the metric uses LLMs, and if so, what LLM it uses.

If you have a use case not covered by the below enrichments out of the box, please contact your administrator.

| Metric                                                                                       | Metric Category | Description                                                                                                                                                                                                   | Enrichment                  | LLM Used? | LLM Type                 |
| -------------------------------------------------------------------------------------------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------- | --------- | ------------------------ |
| [Faithfulness](https://docs.fiddler.ai/observability/enrichments#faithfulness)               | Hallucination   | This enrichment identifies the accuracy and reliability of facts presented in AI-generated texts                                                                                                              | `faithfulness`              | Yes       | OpenAI                   |
| [Fast Faithfulness](https://docs.fiddler.ai/observability/enrichments#fast-faithfulness)     | Hallucination   | This enrichment identifies the accuracy and reliability of facts presented in AI-generated texts. It is generated by Fiddler's Fast Trust Models                                                              | `ftl_response_faithfulness` | Yes       | Fiddler Fast Trust Model |
| [Answer Relevance](https://docs.fiddler.ai/observability/enrichments#answer-relevance)       | Hallucination   | This enrichment measures the pertinence of AI-generated responses to their inputs                                                                                                                             | `answer_relevance`          | Yes       | OpenAI                   |
| [Conciseness](https://docs.fiddler.ai/observability/enrichments#conciseness)                 | Hallucination   | This enrichment evaluates the brevity and clarity of AI-generated responses                                                                                                                                   | `conciseness`               | Yes       | OpenAI                   |
| [Coherence](https://docs.fiddler.ai/observability/enrichments#coherence)                     | Hallucination   | This enrichment assesses the logical flow and clarity of AI-generated responses                                                                                                                               | `coherence`                 | Yes       | OpenAI                   |
| [Fast Safety](https://docs.fiddler.ai/observability/enrichments#fast-safety)                 | Safety          | This enrichment generates 11 different safety metrics to measure texts upon. These metrics are: `illegal, hateful, harassing, racist, sexist, violent, sexual, harmful, unethical, jailbreaking, roleplaying` | `ftl_prompt_safety`         | Yes       | Fiddler Fast Trust Model |
| [PII](https://docs.fiddler.ai/observability/enrichments#personally-identifiable-information) | Safety          | This enrichment flags the presence of sensitive information within textual data                                                                                                                               | `pii`                       | No        |                          |
| [Regex Match](https://docs.fiddler.ai/observability/enrichments#regex-match)                 | Safety          | This enrichment compares the text with a regular expression string                                                                                                                                            | `regex_match`               | No        |                          |
| [Topic](https://docs.fiddler.ai/observability/enrichments#topic)                             | Safety          | This enrichment classifies the text into several preset dimensions using a zero-shot classifier                                                                                                               | `topic_model`               | No        |                          |
| [Banned Keywords](https://docs.fiddler.ai/observability/enrichments#banned-keyword-detector) | Safety          | This enrichment detects the presence of banned keywords configured by the user                                                                                                                                | `banned_keywords`           | No        |                          |
| [Profanity](https://docs.fiddler.ai/observability/enrichments#profanity)                     | Safety          | This enrichment flags the use of offensive or inappropriate language                                                                                                                                          | `profanity`                 | No        |                          |
| [Language Detection](https://docs.fiddler.ai/observability/enrichments#language-detector)    | Safety          | This enrichment identifies the language of the source text                                                                                                                                                    | `language_detection`        | No        |                          |
| [Evaluate](https://docs.fiddler.ai/observability/enrichments#evaluate)                       | Text Statistics | This enrichment provides classic text evaluation methods such as BLEU, ROUGE, and Meteor                                                                                                                      | `evaluate`                  | No        |                          |
| [Sentiment](https://docs.fiddler.ai/observability/enrichments#sentiment)                     | Text Statistics | This enrichment provides sentiment analysis of the target text                                                                                                                                                | `sentiment`                 | No        |                          |
| [TextStat](https://docs.fiddler.ai/observability/enrichments#textstat)                       | Text Statistics | This enrichment provides various text statistics such as character/letter count, Flesch-Kincaid, and others                                                                                                   | `textstat`                  | No        |                          |
| [Token Count](https://docs.fiddler.ai/observability/enrichments#token-count)                 | Text Statistics | The Token Count enrichment is designed to count the number of tokens in a string.                                                                                                                             | `token_count`               | No        |                          |
| [SQLValidation](https://docs.fiddler.ai/observability/enrichments#sql-validation)            | Text Validation | Evaluates different query dialects for syntax correctness.                                                                                                                                                    | `sql_validation`            | No        |                          |
| [JSONValidation](https://docs.fiddler.ai/observability/enrichments#json-validation)          | Text Validation | Validates JSON for correctness and optionally against a user-defined schema.                                                                                                                                  | `json_validation`           | No        |                          |
