# Unsloth Installation

Unsloth works on Linux, Windows, NVIDIA, AMD, Google Colab and more. See our [system requirements](https://unsloth.ai/docs/get-started/fine-tuning-for-beginners/unsloth-requirements).

{% columns %}
{% column width="50%" %}
**Recommended install methods:**

```bash
pip install unsloth
```

```bash
uv pip install unsloth
```

{% endcolumn %}

{% column width="50%" %}
**To update Unsloth**

{% code overflow="wrap" %}

```bash
pip install --upgrade unsloth
```

{% endcode %}
