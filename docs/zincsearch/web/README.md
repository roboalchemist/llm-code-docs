# ZincSearch Documentation Reference

ZincSearch is a lightweight search engine that provides full-text indexing as an alternative to Elasticsearch.

**Official Documentation**: https://zincsearch-docs.zinc.dev/

## Features

- Full-text indexing capability
- Single binary installation
- Web UI for querying (Vue-based)
- Elasticsearch API compatibility
- Built-in authentication
- Schema-less indexing
- Disk-based storage
- Aggregation support

## Key Resources

- **Quickstart**: https://zincsearch-docs.zinc.dev/quickstart/
- **GitHub Repository**: https://github.com/zinclabs/zincsearch
- **Contributing Guide**: https://github.com/zinclabs/zincsearch/blob/main/CONTRIBUTING.md

## Use Cases

ZincSearch is ideal for:
- Application search features
- Full-text indexing on smaller datasets (100s of GB)
- Elasticsearch API compatibility requirements
- Resource-constrained environments

For log search, observability, and large-scale deployments, consider OpenObserve instead.

## Getting Started

1. Download binary from GitHub releases for your platform
2. Run ZincSearch (binary runs in ~2 minutes setup)
3. Access web UI at http://localhost:4080
4. Use Elasticsearch-compatible APIs for data ingestion

For detailed setup and API documentation, visit https://zincsearch-docs.zinc.dev/
