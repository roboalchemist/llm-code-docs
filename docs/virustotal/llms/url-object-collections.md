# 🔀 collections

Collections containing this URL.

The *collections* relationship lists all VirusTotal's collections where this URL is contained.

This relationship can be retrieved using the [relationships API endpoint](https://virustotal.readme.io/reference/urls-relationships). The response contains a list of [Collections](https://virustotal.readme.io/reference/collections) objects.

```json /urls/{url}/collections 
{
  "data": [
    <COLLECTION_OBJECT>,
    <COLLECTION_OBJECT>,
    ...
  ],
  "links": {
    "next": "<string>",
    "self": "<string>"
  },
  "meta": {
    "count": <int>,
    "cursor": "<string>"
  }
}
```
