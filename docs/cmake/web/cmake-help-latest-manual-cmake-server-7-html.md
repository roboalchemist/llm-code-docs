cmake-server(7)[¶](https://cmake.org/cmake/help/latest/manual/cmake-server.7.html#cmake-server-7 "Link to this heading")
========================================================================================================================

The [`cmake(1)`](https://cmake.org/cmake/help/latest/manual/cmake.1.html#manual:cmake(1) "cmake(1)") server mode has been removed since CMake 3.20. Clients should use the [`cmake-file-api(7)`](https://cmake.org/cmake/help/latest/manual/cmake-file-api.7.html#manual:cmake-file-api(7) "cmake-file-api(7)") instead.
