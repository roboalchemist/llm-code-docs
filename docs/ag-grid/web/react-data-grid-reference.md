React Data Grid AG Grid: Reference
==================================

![Image 2: react logo](https://ag-grid.com/_astro/react.CtDRhtxt.svg)React

The reference section of our documentation covers all the public interfaces for interacting with AG Grid across Grid, Columns and Rows.

Grid [Copy Link](https://ag-grid.com/react-data-grid/reference/#grid)
---------------------------------------------------------------------

The grid section starts with an overview explaining key information for configuring and interacting with the grid such as: how to configure the grid, update properties and interact with the grid via its api.

*   [Overview](https://ag-grid.com/react-data-grid/grid-interface/): Key information for working with the grid.

Reference pages list all configuration options, events and API methods:

*   [Options Reference](https://ag-grid.com/react-data-grid/grid-options/): Properties and callbacks used to configure the grid.
*   [Events Reference](https://ag-grid.com/react-data-grid/grid-events/): Events that can be used to trigger application logic.
*   [API Reference](https://ag-grid.com/react-data-grid/grid-api/): Methods used to interact with a grid instance.

Columns [Copy Link](https://ag-grid.com/react-data-grid/reference/#columns)
---------------------------------------------------------------------------

The columns reference section starts with an overview of Columns.

*   [Overview](https://ag-grid.com/react-data-grid/column-interface/): Key information for working with columns.

Reference pages list all column configurations and Column attributes, methods and events:

*   [Options Reference](https://ag-grid.com/react-data-grid/column-properties/): Properties and callbacks used to configure a column.
*   [Column Reference](https://ag-grid.com/react-data-grid/column-object/): Attributes and methods available on Columns.
*   [Column Group Reference](https://ag-grid.com/react-data-grid/column-object-group/): Attributes and methods available on Columns Groups.
*   [Events Reference](https://ag-grid.com/react-data-grid/column-events/): Events fired by Columns.

Rows [Copy Link](https://ag-grid.com/react-data-grid/reference/#rows)
---------------------------------------------------------------------

The rows reference section covers the ways to interact with instantiated rows displayed within the grid.

*   [Overview](https://ag-grid.com/react-data-grid/row-interface/): Key information for working with rows.

Reference pages list all Row object attributes, methods and events:

*   [Row Reference](https://ag-grid.com/react-data-grid/row-object/): Row node attributes and methods for rows.
*   [Events Reference](https://ag-grid.com/react-data-grid/row-events/): Events fired by Rows.
