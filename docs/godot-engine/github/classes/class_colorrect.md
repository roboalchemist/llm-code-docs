# ColorRect

**Inherits:** [Control<class_Control>] **<** [CanvasItem<class_CanvasItem>] **<** [Node<class_Node>] **<** [Object<class_Object>]

A control that displays a solid color rectangle.


## Description

Displays a rectangle filled with a solid [color<class_ColorRect_property_color>]. If you need to display the border alone, consider using a [Panel<class_Panel>] instead.
