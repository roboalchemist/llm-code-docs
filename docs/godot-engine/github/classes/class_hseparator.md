# HSeparator

**Inherits:** [Separator<class_Separator>] **<** [Control<class_Control>] **<** [CanvasItem<class_CanvasItem>] **<** [Node<class_Node>] **<** [Object<class_Object>]

A horizontal line used for separating other controls.


## Description

A horizontal separator used for separating other controls that are arranged **vertically**. **HSeparator** is purely visual and normally drawn as a [StyleBoxLine<class_StyleBoxLine>].
