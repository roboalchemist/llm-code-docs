**Inherits:** [BoxContainer<class_BoxContainer>] **<** [Container<class_Container>] **<** [Control<class_Control>] **<** [CanvasItem<class_CanvasItem>] **<** [Node<class_Node>] **<** [Object<class_Object>]

**Inherited By:** [ColorPicker<class_ColorPicker>], [ScriptEditorBase<class_ScriptEditorBase>]

A container that arranges its child controls vertically.


## Description

A variant of [BoxContainer<class_BoxContainer>] that can only arrange its child controls vertically. Child controls are rearranged automatically when their minimum size changes.
