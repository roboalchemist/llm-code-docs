# AnimationNodeTimeScale

**Inherits:** [AnimationNode<class_AnimationNode>] **<** [Resource<class_Resource>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

A time-scaling animation node used in [AnimationTree<class_AnimationTree>].


## Description

Allows to scale the speed of the animation (or reverse it) in any child [AnimationNode<class_AnimationNode>]\ s. Setting it to `0.0` will pause the animation.
