# TextureCubemapRD

**Inherits:** [TextureLayeredRD<class_TextureLayeredRD>] **<** [TextureLayered<class_TextureLayered>] **<** [Texture<class_Texture>] **<** [Resource<class_Resource>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

Texture for Cubemap that is bound to a texture created on the [RenderingDevice<class_RenderingDevice>].


## Description

This texture class allows you to use a cubemap texture created directly on the [RenderingDevice<class_RenderingDevice>] as a texture for materials, meshes, etc.

\ **Note:** **TextureCubemapRD** is intended for low-level usage with [RenderingDevice<class_RenderingDevice>]. For most use cases, use [Cubemap<class_Cubemap>] instead.
