# PopupPanel

**Inherits:** [Popup<class_Popup>] **<** [Window<class_Window>] **<** [Viewport<class_Viewport>] **<** [Node<class_Node>] **<** [Object<class_Object>]

A popup with a panel background.


## Description

A popup with a configurable panel background. Any child controls added to this node will be stretched to fit the panel's size (similar to how [PanelContainer<class_PanelContainer>] works). If you are making windows, see [Window<class_Window>].
