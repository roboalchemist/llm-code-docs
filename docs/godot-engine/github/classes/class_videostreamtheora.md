[VideoStream<class_VideoStream>] resource for Ogg Theora videos.


## Description

[VideoStream<class_VideoStream>] resource handling the [Ogg Theora ](https://www.theora.org/)_ video format with `.ogv` extension. The Theora codec is decoded on the CPU.

\ **Note:** While Ogg Theora videos can also have a `.ogg` extension, you will have to rename the extension to `.ogv` to use those videos within Godot.
