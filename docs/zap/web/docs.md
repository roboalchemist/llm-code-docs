Documentation
=============

##### Guides

**[Getting Started Guide](https://www.zaproxy.org/getting-started/)** - a good place to start if you are new to ZAP

**[Getting Further Guides](https://www.zaproxy.org/docs/getting-further/)** - in depth guides covering different ways you can use ZAP

**[Desktop User Guide](https://www.zaproxy.org/docs/desktop/)** - the help included with the ZAP desktop application

**[ZAP Developer Guide](https://www.zaproxy.org/docs/developer/)** - ZAP documentation for developers

**[Contributing Guide](https://www.zaproxy.org/docs/contribute/)** - lots more details on how you can get involved

**[FAQ](https://www.zaproxy.org/faq/)** - Frequently Asked Questions

**[ZAPping the OWASP Top 10 (2021)](https://www.zaproxy.org/docs/guides/zapping-the-top-10-2021)** - a guide mapping Top 10 items to ZAP functionality that can assist IT security personnel

##### In Depth Features

**[Automate](https://www.zaproxy.org/docs/automate/)** - the various options for automating ZAP

**[Authentication Decision Tree](https://www.zaproxy.org/docs/authentication/)** - the best place to start if you need ZAP to authenticate to your app

**[Docker](https://www.zaproxy.org/docs/docker/)** - detailed information on ZAP's Docker images

**[Burp to ZAP Feature Map](https://www.zaproxy.org/docs/burp-to-zap-feature-map/)** - a mapping from Burp Suite features to their ZAP equivalents

**[Statistics](https://www.zaproxy.org/docs/statistics/)** - public ZAP usage statistics

**[Software Bill of Materials](https://www.zaproxy.org/docs/sbom/)** - ZAP add-on SBOMs

**[Alert Details](https://www.zaproxy.org/docs/alerts/)** - detailed information on the alerts ZAP can raise

**[ZAP Vs Test Apps](https://www.zaproxy.org/docs/testapps/)** - how to configure ZAP to scan commonly used vulnerable applications

##### Internal Details

**[Constants](https://www.zaproxy.org/docs/constants/)** - defined in the ZAP codebase that are exposed via the API and/or scripts

**[Internal Events](https://www.zaproxy.org/docs/internal-events/)** - detailed information on the internal events ZAP publishes

**[Internal Statistics](https://www.zaproxy.org/docs/internal-statistics/)** - detailed information on the internal statistics ZAP maintains

**[Test Scans](https://www.zaproxy.org/docs/scans/)** - the results of running ZAP vs well known deliberately vulnerable apps
