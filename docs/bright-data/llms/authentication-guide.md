# Authentication Guide

Bright Data APIs use an “API key” to authenticate requests. You can view and manage your API keys under “Account settings” in the Control Panel.

## What is Bright Data's API key?

Bright Data's API key is a secure method to replace your credentials (i.e. email address and password) when sending requests to Bright Data via API.

<Tip>
  [Guide: How to generate an API key?](https://docs.brightdata.com/api-reference/authentication#how-do-i-generate-a-new-api-key%3F)
</Tip>

<Note>
  Only the account admin can generate a new API key for an account user
</Note>
