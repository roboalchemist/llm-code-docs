# Clarifying rate limit changes for non-Marketplace apps

June 3, 2025

We'd like to add additional clarity to the recent rate limits change for the `conversations.history` and `conversations.replies` Web API methods. Any internal customer-built apps will maintain their existing rate limits and will not be subject to the new posted limits. [Read the updated FAQ for more info](/changelog/2025/05/29/rate-limit-changes-for-non-marketplace-apps), and please continue to reach out with questions!
