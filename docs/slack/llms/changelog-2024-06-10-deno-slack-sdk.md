# Release: Deno Slack SDK v2.12.0

June 10, 2024

In the Deno Slack SDK `2.12.0` release, we've added the [`add_bookmark`](/tools/deno-slack-sdk/reference/slack-functions/add_bookmark) Slack function and the [`team_id`](/tools/deno-slack-sdk/guides/utilizing-slack-and-custom-data-types#teamid) type. In addition, you can now pass the optional `team_id` parameter to the [`create_channel`](/tools/deno-slack-sdk/reference/slack-functions/create_channel) and [`create_usergroup`](/tools/deno-slack-sdk/reference/slack-functions/create_usergroup) Slack functions.
