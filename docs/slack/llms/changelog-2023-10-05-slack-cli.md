# Release: Slack CLI v2.10.0

October 5, 2023

Version `2.10.0` of the developer tools for the Slack automations platform has entered the chat! We've updated the `deploy` command to prompt you to select an organization workspace, similar to the `run` command. We also added a little more polish to the [Admin-Approved Apps](/tools/deno-slack-sdk/guides/controlling-permissions-for-admins) flow, added support for the `apps.certified.install` method (coming soon to an API near you!), and implemented a few bug fixes behind the scenes to keep things running smoothly.
