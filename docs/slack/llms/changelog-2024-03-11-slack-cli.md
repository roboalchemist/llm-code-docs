# Release: Slack CLI v2.19.0

March 11, 2024

Version 2.19.0 of the developer tools for the Slack automations platform has joined the stage!

* We've added a new Time To Live (TTL) feature to datastores! Refer to [Delete items automatically](/tools/deno-slack-sdk/guides/deleting-items-from-a-datastore#delete_automatically) for more details.
* We fixed an issue so that any `GetHooks` errors display properly.
