# Release: Slack CLI v3.6.1

August 28, 2025

Version `3.6.1` of the developer tools for the Slack platform is here!

* The `deno` runtime is no longer installed with the Slack CLI installation script. We recommend installing `deno` separately and configuring CI systems as needed for app development (such as for [ROSI](https://docs.slack.dev/workflows/run-on-slack-infrastructure) apps).
