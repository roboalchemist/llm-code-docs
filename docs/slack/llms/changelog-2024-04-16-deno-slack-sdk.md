# Release: Deno Slack SDK v2.10.0 & Deno Slack API v2.4.0

April 16, 2024

In the Deno Slack SDK `2.10.0` and Deno Slack API `2.4.0` releases, we added support for the `all_resources` event for event triggers. Refer to [scoping channel-based event triggers](/tools/deno-slack-sdk/guides/creating-event-triggers#scoping) for more details.
