# openid scope

View information about a user's identity

## Facts

## Supported token types

[`User`](/authentication/tokens#user)

## Compatible API methods

[`openid.connect.userInfo`](/reference/methods/openid.connect.userinfo)

## Usage info {#usage-info}

This special scope allows your app to implement [Sign in with Slack](/authentication/sign-in-with-slack/).

As part of [Sign in with Slack](/authentication/sign-in-with-slack/), this scope allows your app to receive information about a user who signs into your service with their Slack profile.
