# Release: Slack CLI v3.0.5

April 24, 2025

Version `3.0.5` of the developer tools for the Slack platform has bloomed!

* The Slack CLI and installation scripts now point to our new Slack platform documentation at [docs.slack.dev](https://docs.slack.dev/) and our developer tools documentation at [tools.slack.dev](https://tools.slack.dev/).
* As part of the [Slack CLI going open source](https://docs.slack.dev/changelog/2025/04/10/slack-cli-goes-open-source), the Slack CLI documentation has gone open source as well!
* We fixed a bug: When running the `slack upgrade` command inside a project directory, the output now notes whether the latest SDK versions were checked.
