# Coded event trigger improvements are now live

April 25, 2024

You can now set an event as `all_resources: true` instead of listing individual `channel_ids`. This allows your event trigger to listen to any channel your app is part of. Check out [scoping channel-based event triggers](/tools/deno-slack-sdk/guides/creating-event-triggers#scoping) for more details.
