# Legacy APIs, features, and frameworks

While we’d prefer you to use our newest features, we still maintain documentation for a number of so-called legacy features still made available to developers. Almost everything you find marked legacy will one day be deprecated. Everything deprecated will one day retire and cease functioning.

Enjoy this bit of the past inside the present and **stay tuned to our [platform changelog](/changelog)** for the latest information on features like these.

_What happened to "workspace apps"?_ Workspace apps are both deprecated and retired and effectively, no longer exist. [Learn more about the retirement of workspace apps](/changelog/2021-01-workspace-apps-retiring-the-platform-graveyard-in-aug-2021).
