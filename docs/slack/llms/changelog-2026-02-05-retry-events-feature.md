# Retry delayed events feature now available

February 5, 2026

You can now replay events that your app might have missed! In case of an outage where your app may have missed events, we've released a **Delayed Events** feature that lets you replay past events. Read all about it in the [Events API docs](/apis/events-api/#delayed-events-retry).
