## Release: Deno Slack API v2.8.0

August 2, 2024

In the Deno Slack API `2.8.0` release, we've added an `event_timestamp` property to all event trigger data objects. Refer to [the event response object](/tools/deno-slack-sdk/guides/creating-event-triggers#response-object) for more details.
