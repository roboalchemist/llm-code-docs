# Release: Slack CLI v3.0.2

March 27, 2025

Version `3.0.2` of the developer tools for the Slack platform is here!

* We've updated the `slack version` command to include additional descriptive information and examples when using the `--help` flag.
* We've updated the feedback URL when using the `slack feedback` command to point to the home of our new and improved documentation on [docs.slack.dev](https://docs.slack.dev/)!
