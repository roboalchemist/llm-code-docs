# Munka az összekapcsolt munkatételekkel

Az Enate számos funkciója lehetővé teszi az Ön számára a különböző csapatok által végrehajtott műveletek kezelését, így Ön akkor is könnyedén létrehozhatja és felügyelheti a munkát, ha nincs teljes jogosultsága mások munkájához.
