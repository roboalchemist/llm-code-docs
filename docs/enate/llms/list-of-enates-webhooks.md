# List of Enate's Webhooks

The following Webhooks are currently available in Enate:

* [PacketCreated](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#packet-created) - fires when a Ticket, Case or Action is created.&#x20;
* [PacketUpdated](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#packet-updated) - fires when a Ticket, Case or Action is updated.
* [NewCommunication](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#newcommunication) - fires when a communication (email or note) is added to a specific work item
* [BusinessObjectCreated](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#businessobjectcreated) - fires when a COMPANY or a SCHEDULE is created.
* [BusinessObjectUpdated](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#businessobjectupdated) - fires when a COMPANY or a SCHEDULE is updated.
* [SpecificBusinessObjectUpdated](https://docs.enate.net/enate-help/integrations/enate-integrations/enates-webhooks#businessobjectupdated-1) - fires when a COMPANY or a SCHEDULE, specified within the subscription, is updated.

The sections below give details for the webhooks you can currently subscribe to, a short description of the events which will trigger them, the data you'll get returned from the webhook (with example values for each) and the parameters which are to be passed in.

{% hint style="info" %}
Note that the term 'Packet' here refers to Enate's Work Items - which can be of type Case, Action or Ticket.
{% endhint %}
