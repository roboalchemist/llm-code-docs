# Mit bereits bestehenden Arbeitsaufträgen verknüpfen

## Überblick

Neben der Möglichkeit, [neue verknüpfte Arbeitsaufgaben aus einem Fall oder Ticket heraus zu erstellen](https://docs.enate.net/enate-help/deutsch/mit-verknuepften-arbeitsauftraegen-arbeiten/erstellen-neuer-verknupfter-arbeitsauftrage), können Sie auch Verknüpfungen zwischen bestehenden Fällen und Tickets erstellen, um die von verschiedenen Teams durchgeführten Aktivitäten besser zu verwalten.

Sie können Links zwischen verknüpften Arbeitsaufträgen auf zwei verschiedene Arten erstellen:

1. [Von der Homepage aus](#arbeitsauftraege-von-der-homepage-aus-verknuepfen)
2. [Von einem Arbeitsauftrag aus](#verknuepfung-vorhandener-arbeitsauftraegen-aus-einem-arbeitsauftrag-heraus)

{% hint style="info" %}
Beachten Sie, dass Sie nur Verknüpfungen zwischen Vorgängen und Tickets erstellen können. Sie können keine Kommunikation, Dateien, Fehler, Verknüpfungen oder benutzerdefinierte Daten kopieren, wenn Sie eine Verknüpfung zwischen bestehenden Arbeitsaufträgen erstellen.
{% endhint %}

## Arbeitsaufträge von der Homepage aus verknüpfen

Sie können vorhandene Arbeitsaufträge von Ihrer Startseite aus verknüpfen, indem Sie die Fälle und Tickets aus dem Raster der Startseite (das Ihren Arbeitseingang, den Arbeitseingang des Teams, die eigene Arbeit oder die Arbeit des Teams anzeigen kann) auswählen, die Sie miteinander verknüpfen möchten, und die Option "Verknüpfen" wählen, die oben im Raster erscheint.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FL4sIEGKi8jn1doqk7oPs%2Fimage.png?alt=media&#x26;token=63f1cdb1-7e77-4ef7-ad08-468a89e54069" alt=""><figcaption></figcaption></figure>

Der erste Vorgang oder das erste Ticket, das Sie auswählen, ist der Arbeitsauftrag, mit dem alle weiteren ausgewählten Arbeitsaufträge verknüpft werden (oder der erste in Ihrer Liste, wenn Sie die Option "Alle auswählen" verwendet haben).

Das daraufhin angezeigte Popup-Fenster bestätigt Ihnen den Fall bzw. das Ticket, mit dem die Elemente verknüpft werden sollen, sowie die spezifischen Elemente selbst, bevor Sie die Verknüpfung bestätigen können:

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FSecdMCl6HgSVLeq02SfO%2Fimage.png?alt=media&#x26;token=60399343-321c-4f0a-8458-07b833823f98" alt=""><figcaption></figcaption></figure>

Sie können bis zu 50 Arbeitsaufträge auf einmal verknüpfen, aber es gibt keine Begrenzung für die Anzahl der Arbeitsaufträge, die jemals verknüpft werden können.&#x20;

Wenn Sie anschließend zu diesem Hauptticket / Fall gehen, mit dem alles verknüpft wurde, sehen Sie auf der [Registerkarte Verknüpfte Arbeit](https://docs.enate.net/enate-help/deutsch/mit-verknuepften-arbeitsauftraegen-arbeiten/verknufte-arbeitsauftrage-ansehen-der-verknupft-tab) ALLE Elemente, die Sie für die Verknüpfung ausgewählt haben.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FK7p7qZt9vHTSciOaC4l9%2Fimage.png?alt=media&#x26;token=bcfa6544-1be5-4263-a860-ad645e72b1f9" alt=""><figcaption></figcaption></figure>

Wenn Sie sich jedoch in einem der Arbeitsaufträge befinden, die Sie mit diesem Hauptantrag / -fall verknüpft haben, sehen Sie auf der Registerkarte Verknüpfte Arbeit nur einen Datensatz dieses Hauptantrags / -falls (und nicht alle anderen, die verknüpft wurden).

## Verknüpfung vorhandener Arbeitsaufträgen aus einem Arbeitsauftrag heraus

Sie können bestehende Arbeitsaufträge aus einem Arbeitsauftrag selbst heraus verknüpfen, indem Sie auf dem Bildschirm des Arbeitsauftrags die Option "+ Arbeitsauftrag" auswählen und dann "Bestehende Aufträge" hinzufügen.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FukL1OVqek6EpU3nVJ5zH%2Fimage.png?alt=media&#x26;token=5c2c05b0-d809-4ff1-8d80-4f1ce086fbfc" alt=""><figcaption></figcaption></figure>

In dem daraufhin angezeigten Pop-up-Fenster können Sie nach den Fällen und/oder Tickets suchen, mit denen Sie dieses Workitem verknüpfen möchten.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FuNjAZ9RjJHqO1jXMtXHS%2Fimage.png?alt=media&#x26;token=01c23de5-e23f-4294-b174-c37f59ed5a88" alt=""><figcaption></figcaption></figure>

Sie können bis zu 50 Arbeitsaufträge auf einmal verknüpfen, aber es gibt keine Begrenzung für die Anzahl der Arbeitsaufträge, die jemals verknüpft werden können.

Ein Link zu dem neuen verknüpften Fall oder Ticket wird auf der [Registerkarte Verknüpfte Arbeit](https://docs.enate.net/enate-help/deutsch/mit-verknuepften-arbeitsauftraegen-arbeiten/verknufte-arbeitsauftrage-ansehen-der-verknupft-tab) im Bildschirm der verknüpften Arbeitsaufgaben angezeigt.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FLsdfhO9ev3iSYCjxnuyV%2Fimage.png?alt=media&#x26;token=5c712c23-afae-49bc-b725-ffb9ba14cdc9" alt=""><figcaption></figcaption></figure>

## Verknüpfung der Arbeitsaufträge aufheben

Sie können die Verknüpfung von Arbeitsaufträgen jederzeit aufheben, indem Sie auf die Registerkarte "Verknüpfte Arbeit" im Bildschirm des Arbeitsauftrags gehen und das Symbol "Verknüpfung aufheben" auswählen.

<figure><img src="https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FSA2mqPU4Q0IABOxcJQOt%2F13%20Unlink-Work-Items.gif?alt=media&#x26;token=bed59851-993c-4c4d-81f9-9dfa956e1f2c" alt=""><figcaption></figcaption></figure>
