# Eine Notiz hinzufügen

{% embed url="<https://media.enate.net/WorkManager/11WorkItemScreens/AddingANoteDE.mp4>" %}

{% embed url="<https://enate.cdn.spotlightr.com/watch/MTM2MTYxMA==>" %}

Klicken Sie auf die Registerkarte Notiz, um eine Notiz hinzuzufügen. Jeder kann einem Ticket, einem Fall oder einer Aktion eine interne Notiz hinzufügen - Sie müssen den Auftrag nicht zugewiesen bekommen, und das Hinzufügen einer Notiz weist Ihnen die Arbeit nicht automatisch zu. Diese Notizen sind für den internen Gebrauch bestimmt und externe Kontakte werden diese nie einsehen können.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2Fxei9YuCE7kvX2b0CKeUJ%2Fnote.PNG?alt=media\&token=ae6f59ba-734d-4108-960b-894cfd4eefb9)

### Angeben des Notiztyps <a href="#angeben-des-notiztyps" id="angeben-des-notiztyps"></a>

Falls gewünscht, können Sie angeben, welche Art von Interaktion zu der Notiz geführt hat, z.B. telefonisch eingehend/ausgehend usw.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2F7uvsQqo2IBwx6MmKFSnD%2Fimage.png?alt=media\&token=059d6b1b-8f16-485b-872a-81235397a666)

### Eine Notiz hinzufügen und ein Ticket auflösen

Wenn wir eine Notiz zu einem Ticket hinzufügen , das uns selbst zugewiesen ist, haben wir die zusätzliche Möglichkeit, eine Notiz hinzuzufügen und den Arbeitsauftrag als erledigt zu markieren. Dadurch wir der Arbeitsauftrag als erledigt angezeigt.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FtxaPGKZF9XfzpqENV9U9%2Fimage.png?alt=media\&token=bb4bfcd3-62a2-4b7e-bfe0-884779a21898)

Nach dem Hinzufügen der Notiz wird der Bildschirm mit der auf der Zeitachse angezeigten neuen Notiz aktualisiert.&#x20;

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2FyIca807T3jsrZqNRIwzX%2Fimage.png?alt=media\&token=61d3ca73-ac2c-410d-b0bb-9059347fe29a)

### Arbeitsaufgabe mit unvollständigen Bemerkungen schließen <a href="#closing-an-item-with-incomplete-notes" id="closing-an-item-with-incomplete-notes"></a>

Wenn Sie mit der Eingabe einer Notiz beginnen, diese aber nicht gespeichert haben, wird auf der Registerkarte eine "\*"- Markierung angezeigt, die Sie darauf hinweist.

![](https://809551593-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F-MWYnDNwe3Cuo4zlGbs5-1666082760%2Fuploads%2Fc2ETH4KepDpeXALxrKGN%2Fimage.png?alt=media\&token=dcbb02df-08a0-4a02-b803-8e9bbc1629ee)

Wenn Sie anschließend versuchen, die Registerkarte „Arbeitsaufgaben" zu schließen, während diese „in Arbeit" befindlichen Elemente noch nicht gespeichert sind, werden Sie vom System gefragt, ob Sie fortfahren möchten.
