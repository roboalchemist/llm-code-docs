# Changing password

If your SonarQube Server instance is not using a 3rd party authentication mechanism such as LDAP or an OAuth provider (GitHub, Google Account, etc.), you can change your SonarQube Server password.

To change your password:

1. Select your account menu in the top right corner of the SonarQube Server interface.
2. In the menu, select **My Account**.
3. Select the **Security** tab.
4. In the **Enter a new password** section, enter your old and new password.
5. Select **Update**.
