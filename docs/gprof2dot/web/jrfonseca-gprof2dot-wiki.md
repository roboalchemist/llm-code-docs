Docker image
------------

[](https://github.com/jrfonseca/gprof2dot/wiki#docker-image)

Do

```
docker run -d -p 5000:5000 --name dotimage xieyanbo/dotimage
```

and use a browser to access `http://your.host.ip:5000/`

See also:

* [https://hub.docker.com/r/xieyanbo/dotimage/](https://hub.docker.com/r/xieyanbo/dotimage/)
* [https://github.com/xyb/dotimage](https://github.com/xyb/dotimage)
