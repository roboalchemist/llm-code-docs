# Export Test Scenarios

Test scenarios can be exported in `Apidog CLI`, `Postman`, and `JMeter` data files, allowing them to run in local environments or external continuous integration pipelines. This integration facilitates seamless alignment with existing testing workflows within your team.

## Exporting Configuration Files

Click the `...` button on the right side of the test scenario and select the desired export format.
