# Creating a Sprint Branch

When starting a new sprint with defined requirements, create a sprint branch to begin API design and development work in isolation from the main branch.

## Creating a Sprint Branch

### Method 1: From the Sprint Branch Switch

1. Navigate to your project in Apidog
2. Locate the sprint branch switch next to **APIs**
3. Click **New Sprint Branch**

<Background>
![Creating a new sprint branch from switch](https://assets.apidog.com/help/assets/images/create-branch-01-dc945e095806ebd5713c6dc640419be8.png)
</Background>

### Method 2: From Sprint Branch Management

1. Click **Manage Sprint Branches** in the sprint branch switch
2. Or navigate to **Settings** → **Manage Sprint Branches**
3. View the list of existing sprint branches
4. Click **New Sprint Branch** to create a new branch

<Background>
![Creating a new sprint branch from management page](https://assets.apidog.com/help/assets/images/create-branch-02-5193480c8838bbd2eb73484c43635646.png)
