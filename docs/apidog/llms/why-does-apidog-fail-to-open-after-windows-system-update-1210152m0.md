# Why does Apidog fail to open after Windows system update

#### Q: Why does the software stopped working after a Windows system update?
A: If Apidog software fails to open after a windows update, it may be due to custom installation directory permissions. Some users select custom directory during installation, and Windows updates can sometimes modify system permissions, causing the software to loose access.

#### Q: How I can get rid of this issue?
A: In order to avoid this issue, you can uninstall the software and reinstall it.Download the software installer (it is recommended to download it from the [official website](https://apidog.com)). During the installation process, select the directory you want, and select the Install for everyone option.

If you still face issue regarding installation, please feel free to reach us via [Apidog Support Center](https://docs.apidog.com/apidog-support-center-748035m0.md).
