# Redis

:::info
Connecting to Redis is a paid feature. Please check [Pricing](https://apidog.com/pricing/) for details.
:::

Redis is a high-performance key-value store. Apidog supports connecting to Redis to perform operations or execute commands directly.

## Connecting to Redis

<Steps>
  <Step>
    **Create Connection**
    
    Go to **Settings** > **Database Connections** > **New**. Select **Redis**.
  </Step>
  <Step>
    **Configure Details**
    
    Enter Host, Port, Password, and Database Index.
    
    <Background>
      ![Redis Connection Settings](https://assets.apidog.com/uploads/help/2023/12/01/239b6032dd1450b6dccc6b9360d94de6.png)
    </Background>
  </Step>
</Steps>

## Performing Operations

### Visual Operations
For standard actions (Get, Set, Delete), simply select the **Operation Type** and fill in the parameters.

**Example: Get Value**
*   **Operation**: GET
*   **Key**: `user:session:123`

<Background>
  ![Redis Visual Operation](https://assets.apidog.com/uploads/help/2023/12/01/7c483c982c92177cf56b3beb979a5006.png)
</Background>

### Running Commands
For advanced usage, select the **Run Redis Command** tab. You can execute any valid Redis command.

**Example: List Keys**
```bash
KEYS user:*
```

<Background>
  ![Run Redis Command](https://assets.apidog.com/uploads/help/2023/12/01/cc31e36bae13fbd2b22b01305a1c762f.png)
