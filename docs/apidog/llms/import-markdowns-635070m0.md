# Import Markdowns

Apidog supports the **batch import** of Markdown (`.md`) files. Each file is imported as a standalone documentation page.

:::warning[Important]
Imported Markdown files remain as **text documents**. They are *not* parsed or converted into executable API endpoints.
:::

## How to Import

<Steps>
  <Step>
    **Navigate to Import Settings**

    Open **Settings** > **Import Data** in your project.
  </Step>

  <Step>
    **Upload Files**
    
    Select **Markdown** as the data format and upload your `.md` files.
  </Step>
</Steps>
