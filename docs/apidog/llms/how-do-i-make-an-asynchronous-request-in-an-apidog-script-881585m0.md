# How do I make an asynchronous request in an Apidog script?

Apidog supports using the `pm.sendRequest` function in custom scripts to initiate asynchronous requests. For details, see Apidog [Postman scripts reference](https://docs.apidog.com/postman-scripts-reference-593586m0.md).
