# Can I duplicate a published Apidog document to use in my own project?

If you're working with an external interface document or a public project in Apidog, and you’d like to copy it into your own workspace, you can do so by exporting the document in Apidog format or by **cloning it**—depending on the permissions set by the document owner.


Here’s how:

1. **Check export or clone permission**

Visit the document link shared with you and look for “**Export**” or “**Clone**” buttons at the bottom of the interface page.
    - If you see the Export button, click “**Export Apidog File**”, then import the file into your team project.
    - If you see the Clone button, you can directly clone the document into your own team or workspace.


2. **Contact the document owner if options are unavilable**

If neither option is visible, it's likely that the owner has not enabled export or cloning. In this case, we recommend reaching out to the document owner and asking them to enable the export or clone feature in the document’s **publish settings.**
