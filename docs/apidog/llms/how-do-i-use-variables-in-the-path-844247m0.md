# How do I use variables in the path?

In Apidog, the variable part of the path is usually defined as a path parameter. For example, if you enter `{petId}`, you can see the path parameter below. You can then define the parameter value reference a global/environment variable, such as `{{variablesName}}`.When you make a request, Apidog will automatically replace the variable with its current value.
