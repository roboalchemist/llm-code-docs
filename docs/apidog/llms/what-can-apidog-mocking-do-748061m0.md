# What can Apidog mocking do?

Apidog's Mock functionality can fulfill the following requirements:

1. [Return automatically generated data based on the API Spec](https://docs.apidog.com/smart-mock-618190m0.md).
2. [Return the response example specified in the API Spec](https://docs.apidog.com/mock-priority-sequence-618208m0.md).
3. [Return a specified response](https://docs.apidog.com/custom-mock-618204m0.md).
4. [Return different custom responses based on different request parameters](https://docs.apidog.com/custom-mock-618204m0.md).
5. [Return a response whose parameters are related to the request parameters](https://docs.apidog.com/mock-scripts-618209m0.md).
