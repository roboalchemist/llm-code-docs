# Test Suite

> Apidog version 2.8.0 or later is required.

A Test Suite lets you group multiple [test cases](https://docs.apidog.com/test-cases-1605608m0.md) or [test scenarios](https://docs.apidog.com/create-a-test-scenario-599311m0.md) into a single unit, making it easy to run, schedule, and manage them together.

### Core Value
With Test Suite, you can:

1. **Batch Execution**: Run multiple endpoint tests or business workflows in a single execution.
2. **Flexible Setup**: Combine endpoint test cases and test scenarios in one suite, with support for both static selections and dynamic rule-based matching.
3. **Automated Workflow**: Integrate scheduled tasks with CI/CD pipelines to enable fully automated, unattended testing.
4. **Unified Reports**: View all test results in a single report, making review and analysis faster and easier.

### Use Cases
- **Smoke Testing**: Automatically run core business scenarios on every code commit.
- **Regression Testing**: Group large numbers of test cases by module or feature and run full regression tests before each release.
- **Regular Monitoring**: Schedule tasks to continuously monitor the availability of critical endpoints in production.
