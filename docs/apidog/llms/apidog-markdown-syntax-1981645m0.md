# Apidog Markdown Syntax

Apidog Markdown refers to the Markdown format used on the Apidog platform. It combines the simplicity and readability of Markdown with the powerful features of the Apidog platform, providing API developers with a more efficient documentation writing and collaboration experience. For more details, see [Apidog Markdown Syntax Reference](https://markdown.apidog.io/).
