Quarkus Benefits
----------------

![Image 2](https://quarkus.io/assets/images/about/icon-developerjoy.svg)![Image 3](https://quarkus.io/assets/images/about/icon-developerjoy-dark.svg)
### [Developer Joy](https://quarkus.io/developer-joy)

A cohesive platform for optimized developer joy with unified configuration and no hassle native executable generation. Zero config, live reload in the blink of an eye and streamlined code for the 80% common usages, flexible for the remainder 20%.

![Image 4](https://quarkus.io/assets/images/performance/icon-performance.svg)![Image 5](https://quarkus.io/assets/images/performance/icon-performance-dark.svg)
### [Performance](https://quarkus.io/performance)

Quarkus streamlines framework optimizations in the build phase to reduce runtime dependencies and improve efficiency. By precomputing metadata and optimizing class loading, it ensures fast startup times for JVM and native binary deployments, cutting down on memory usage.

![Image 6](https://quarkus.io/assets/images/about/icon-kube-native.svg)![Image 7](https://quarkus.io/assets/images/about/icon-kube-native-dark.svg)
### [Kube-Native](https://quarkus.io/kubernetes-native)

The combination of Quarkus and Kubernetes provides an ideal environment for creating scalable, fast, and lightweight applications. Quarkus significantly increases developer productivity with tooling, pre-built integrations, application services, and more.

![Image 8](https://quarkus.io/assets/images/about/icon-standards.svg)![Image 9](https://quarkus.io/assets/images/about/icon-standards-dark.svg)
### [Community and Standards](https://quarkus.io/standards)

Quarkus provides a cohesive, fun to use, full-stack framework by leveraging a growing list of over fifty best-of-breed libraries that you love and use. All wired on a standard backbone.

![Image 10](https://quarkus.io/assets/images/about/icon-versatility.svg)![Image 11](https://quarkus.io/assets/images/about/icon-versatility-dark.svg)
### [Reactive Core](https://quarkus.io/continuum)

Built on a robust reactive core, Quarkus ensures fast and efficient performance, supporting the development of a wide variety of modern applications.

![Image 12](https://quarkus.io/assets/images/about/icon-containerfirst.svg)![Image 13](https://quarkus.io/assets/images/about/icon-containerfirst-dark.svg)
### [Container First](https://quarkus.io/container-first)

Quarkus tailors your application for GraalVM and HotSpot. Amazingly fast boot time, incredibly low RSS memory (not just heap size!) offering near instant scale up and high density memory utilization in container orchestration platforms like Kubernetes. We use a technique we call compile time boot.

[![Image 14](https://quarkus.io/assets/images/home/userstories_lufthuansa.png)![Image 15](https://quarkus.io/assets/images/home/userstories_lufthuansa-dark.png)](https://quarkus.io/blog/aviatar-experiences-significant-savings/)

[![Image 16](https://quarkus.io/assets/images/home/userstories_logicdrop.png)![Image 17](https://quarkus.io/assets/images/home/userstories_logicdrop-dark.png)](https://quarkus.io/blog/logicdrop-customer-story/)

[![Image 18](https://quarkus.io/assets/images/home/userstories_bancodobrasil.png)![Image 19](https://quarkus.io/assets/images/home/userstories_bancodobrasil-dark.png)](https://quarkus.io/blog/banco-do-brasil-open-banking-user-story/)

[![Image 20](https://quarkus.io/assets/images/home/userstories_adoptium.png)![Image 21](https://quarkus.io/assets/images/home/userstories_adoptium-dark.png)](https://quarkus.io/blog/adoptium-customer-story/)

Quarkus is part of the Commonhaus Foundation
--------------------------------------------

In order to fulfill our goal of being a more inclusive and fostering a more collaborative environment, we have moved to the Commonhaus Foundation. This is a step to further solidify our commitment to open-source development and enterprise adoption, addressing the perception that it was overly dependent on a single vendor (Red Hat). This move aims to provide a neutral ground where other organizations and contributors can feel equally valued and involved, ensuring Quarkus continues to thrive, supported by a broad base of contributors from multiple organizations.

[Learn more about Commonhaus Foundation.](https://www.commonhaus.org/)
