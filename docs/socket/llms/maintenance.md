# Maintenance

| Severity | Alert Type                                             | Description                                                                                                                                                                                                     |
| :------- | :----------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Medium   | [Deprecated](https://socket.dev/alerts/deprecated)     | The maintainer of the package marked it as deprecated. This could indicate that a single version should not be used, or that the package is no longer maintained and any new vulnerabilities will not be fixed. |
| Low      | [Unmaintained](https://socket.dev/alerts/unmaintained) | Package has not been updated in more than 5 years and may be unmaintained. Problems with the package may go unaddressed.                                                                                        |
