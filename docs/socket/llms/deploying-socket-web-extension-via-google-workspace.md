# Deploying via Google Workspace

For organizations that are interested in deploying the [Socket Web Extension](https://chromewebstore.google.com/detail/socket-security/jbcobpbfgkhmjfpjjepkcocalmpkiaop) for Chrome via Google Workspace this document will walk you through the steps to do so.

<br />

1. Log into the [Google Admin Console](https://admin.google.com/) as an Administrator
2. Go to Devices → Chrome → Apps & Extensions
3. Select `Users and browsers` tab
4. On the left click on your top level domain

   <Image align="center" src="https://files.readme.io/ccd9c93437432811ccdc11911ac16a9d0adada4363135db84cae0dc5f9140c0e-Screenshot_2024-09-19_at_11.09.56_AM.png" />
5. Click the Yellow + sign
6. Click the Chrome symbol for `Add from Chrome Web Store`

   <Image align="center" src="https://files.readme.io/fd2a11c1d96b989403ce448eb6f9f9df0de14a730c2722fb4b65f1ee4239813d-Screenshot_2024-09-19_at_11.10.09_AM.png" />
7. Search for Socket Security

   <Image align="center" src="https://files.readme.io/0098fc13917520ceca04bfa28d8d4a302bf94efa853c72eb9269cc62c9188d50-Screenshot_2024-09-19_at_11.10.23_AM.png" />
8. Click on the Socket Security Tile
9. Click Select

   <Image align="center" src="https://files.readme.io/c6bef6b9e3ee1138e7f4b6699a3bfe294630f5f6655280477b28421cef808834-Screenshot_2024-09-19_at_11.10.28_AM.png" />
10. Click on Socket Security in the List of Apps

    <Image align="center" src="https://files.readme.io/5d5c86b4fe345746a6ec906cc65084d9a6ae4f2c2bb3f1633281886c428ae071-Screenshot_2024-09-19_at_11.11.16_AM.png" />
11. On the Right change the `installation policy` to `Force install + pin to browser toolbar`
12. At the top right click Save

    <Image align="center" src="https://files.readme.io/4e8a7d94b7f0cabfce6ad5f8d5285afd620a464faff068e43f50dde4a0f266e7-Screenshot_2024-09-19_at_11.11.34_AM.png" />
13. Have Users relaunch Chrome
