# Supported Node.js Versions

## Supported Environments

Socket CLI is designed for use with **[LTS versions of Node.js](https://nodejs.org/en/about/previous-releases)**.\
If you're using a different version of Node.js, we recommend upgrading for the best experience with Socket CLI.

If you need assistance or have questions about the CLI, please [reach out to our support team](https://docs.socket.dev/docs/contact-support).\
If scheduling a meeting is needed, please [pick from our calendar](https://socket.dev/demo).
