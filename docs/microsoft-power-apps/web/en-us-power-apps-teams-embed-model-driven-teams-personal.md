Embed a model-driven app as personal app in Teams (preview)
===========================================================

[This topic is pre-release documentation and is subject to change.]

You can share an app you've created by embedding it directly into Microsoft Teams. When completed, users can select **+** to add your app to any of **your** team channels or conversations in the team you are in. The app appears as a tile under **Tabs for your team**.

Important

*   This is a preview feature.
*   Preview features aren’t meant for production use and may have restricted functionality. These features are available before an official release so that customers can get early access and provide feedback.

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#prerequisites)
Prerequisites
-------------

*   You need a valid [Power Apps license](https://learn.microsoft.com/en-us/power-platform/admin/pricing-billing-skus).
*   To embed an app into Teams, you need an existing app [created using Power Apps](https://learn.microsoft.com/en-us/power-apps/maker/model-driven-apps/build-first-model-driven-app).

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#add-the-app-as-a-personal-app)
Add the app as a personal app
-----------------------------

1.   Sign into [Power Apps](https://make.powerapps.com/), and then select **Apps** in the menu.

![Image 10: Show list of apps.](https://learn.microsoft.com/en-us/power-apps/teams/media/embed-teams-app/file-apps2.png)

2.   Select **More actions** (...) for the app you want to share in Teams, and then select **Add to Teams**.

![Image 11: Add to Teams.](https://learn.microsoft.com/en-us/power-apps/teams/media/embed-teams-app/add-to-teams.png)

3.   **Add to Teams** panel opens on the right-side of the screen.

![Image 12: Add to Teams panel](https://learn.microsoft.com/en-us/power-apps/teams/media/embed-teams-app/add-to-teams-model.png)

4.   (Optional) If the app doesn't have any description, select **Edit details** to open the app in add designer to add.

5.   (Optional) Select **Advanced settings** to add additional details such as _Name_, _Website_, _Terms of Use_, _Privacy Policy_, _MPN ID_ (Microsoft Partner Network ID).

![Image 13: Add additional details.](https://learn.microsoft.com/en-us/power-apps/teams/media/embed-teams-app/additional-settings-embed.png)

6.   Select **Add to Teams** to add the app as a personal app or select **Add to a team** or **Add to a chat** to add the app as a tab within an existing channel or conversation.

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#download-app)
Download app
------------

You can also **Download app** in Power Apps. Power Apps will then generate your Teams manifest file using the app description and logo you've already set in your app.

1.   To add the app as a personal app or as a tab to any channel or conversation, select **Apps** in the left navigation and then select **Upload a custom app**.

Note

The **Upload a custom app** only appears if your Teams administrator has created a [custom app policy](https://learn.microsoft.com/en-us/microsoftteams/teams-app-setup-policies) and turned on **Allow uploading of custom apps**. 
![Image 14: Add app as tab.](https://learn.microsoft.com/en-us/power-apps/teams/media/embed-teams-app/upload-custom-app.png)

2.   Select **Add** to add the app as a personal app or select **Add to team** to add the app as a tab within an existing channel or conversation.

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#publish-the-app-to-the-teams-catalog)
Publish the app to the Teams catalog
------------------------------------

If you're an admin, you can use the **Download app** option to also [publish the app](https://learn.microsoft.com/en-us/microsoftteams/tenant-apps-catalog-teams#publish-a-custom-app-to-your-organizations-app-store) to the Microsoft Teams catalog.

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#features-currently-unavailable)
Features currently unavailable
------------------------------

*   Running **Model-driven apps** on the Microsoft Teams mobile app is currently not supported.
*   Embedded canvas apps are not supported.
*   Custom pages are not supported.

[](https://learn.microsoft.com/en-us/power-apps/teams/embed-model-driven-teams-personal#see-also)
