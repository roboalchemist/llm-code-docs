### Duration

A duration of time, represented as a unit and a value.

#### Attributes

- **unit** (string) *required* - The unit of time.
- **value** (integer) *required* - The value of the duration.

#### Example

```json
{
  "unit": "minutes",
  "value": 10
}
```
