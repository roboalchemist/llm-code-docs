# Initialize a new project

Initializes a new Knock project by creating a `knock.json` file in the current working directory.

### Flags

- **--knock-dir** (directory): Optional target directory path to point the Knock CLI to. When not provided, the CLI will prompt you for a directory path.

```bash title="Initialize a new Knock project"
knock init
```
