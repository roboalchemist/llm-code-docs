### Delete user

Permanently delete a user and all associated data.

#### Endpoint

`DELETE /v1/users/{user_id}`

**Rate limit tier:** 2

#### Path parameters

- **user_id** (string) *required* - The ID of the user to delete.

#### Responses

##### 204

No Content
