# PostHog badges - Docs

## Add a badge to your website's footer

Want to jump on the PostHog *badge*wagon?

Now you can garnish your website footer with a tasteful PostHog logo - it's like being in a super secret club, except if everything was open source. And if you add the badge and link back to us, [let us know](mailto:merch@posthog.com?subject=PostHog%20logo%20spotted%20on%20%7Byoursitehere.com%7D) and we might send some exclusive [merch](/merch.md) your way...
