# gm

**web3.py** is a Python library for interacting with Ethereum.

It’s commonly found in decentralized apps (dapps) [https://ethereum.org/en/dapps/] to help with
sending transactions, interacting with smart contracts, reading
block data, and a variety of other use cases.

For project updates, follow @EthereumPython [https://twitter.com/EthereumPython] and sign up
for new post notifications on the blog [https://snakecharmers.ethereum.org/].
