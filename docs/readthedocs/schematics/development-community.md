# Community

Schematics was created in Brooklyn, NY by James Dennis. Since then, the code has
been worked on by folks from around the world.  If you have ideas, we encourage
you to share them!

Special thanks to Hacker School [http://hackerschool.com], Plain Vanilla [http://www.plainvanilla.is/], Quantopian [http://quantopian.com], Apple [http://apple.com], Johns Hopkins University [http://jhu.edu], and
everyone who has contributed to Schematics.

## Bugs & Features

We track bugs, feature requests, and documentation requests with Github Issues [https://github.com/schematics/schematics/issues].
