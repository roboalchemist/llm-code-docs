# Installation guide

## Supported Python versions

Scrapy requires Python 3.10+, either the CPython implementation (default) or
the PyPy implementation (see Alternate Implementations [https://docs.python.org/3/reference/introduction.html#implementations]).

## Installing Scrapy

If you’re using Anaconda [https://docs.anaconda.com/anaconda/] or Miniconda [https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html], you can install the package from
the conda-forge [https://conda-forge.org/] channel, which has up-to-date packages for Linux, Windows
and macOS.

To install Scrapy using `conda`, run:

```
conda install -c conda-forge scrapy

```
