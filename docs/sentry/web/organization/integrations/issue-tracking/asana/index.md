Asana integration is deprecated and is not available to be installed on new projects.

This integration needs to be set up in each project for which you wish to use it. It is maintained and supported by the [Sentry community](https://open.sentry.io/).

## Install and Configure

Sentry owner, manager, or admin permissions are required to install this integration.

1. Navigate to **Settings > Integrations > Asana**

   ![Install Asana integration](./img/asana.png)

2. Link your Asana account and select an organization or workspace.
3. Click "Enable Plugin".
4. The option to create or link Asana issues will be displayed from your Sentry issue pages.
