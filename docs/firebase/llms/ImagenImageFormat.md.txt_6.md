An image format for images generated by Imagen.

To specify an image format for generated images, set `https://firebase.google.com/docs/reference/swift/firebasevertexai/api/reference/Structs/ImagenGenerationConfig.html#/s:16FirebaseVertexAI22ImagenGenerationConfigV11imageFormatAA0d5ImageH0VSgvp` in

for more details.

  ` PNG image format.

  Portable Network Graphic (PNG) is a lossless image format, meaning no image data is lost
  during compression. Images in PNG format are *typically* larger than JPEG images, though this
  depends on the image content and JPEG compression quality.

  ### [jpeg(compressionQuality:)](https://firebase.google.com/docs/reference/swift/firebasevertexai/api/reference/Structs/ImagenImageFormat#/s:16FirebaseVertexAI17ImagenImageFormatV4jpeg18compressionQualityACSiSg_tFZ)


  ` JPEG image format.

  Joint Photographic Experts Group (JPEG) is a lossy compression format, meaning some image data
  is discarded during compression. Images in JPEG format are *typically* larger than PNG images,
  though this depends on the image content and JPEG compression quality.

  #### Declaration

  Swift

      public static func jpeg(compressionQuality: Int? = nil) -> ImagenImageFormat

  #### Parameters

  |---|---|
  | ` compressionQuality ` | The JPEG quality setting from 0 to 100, where `0` is highest level of compression (lowest image quality, smallest file size) and `100` is the lowest level of compression (highest image quality, largest file size); defaults to `75`. |
