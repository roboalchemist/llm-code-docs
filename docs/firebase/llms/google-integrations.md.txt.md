# Integrate with Google and Firebase services

In Firebase Studio, you can directly add Google API keys and services
from the Firebase Studio panel in your workspace.

> [!TIP]
> **Tip:** The App Prototyping agent can also help you set up Firebase services. When you prompt it to connect a Firebase project or ask for assistance setting up Firebase services, it provisions and connects a Firebase project on your behalf to help you start the integration. For troubleshooting tips, visit [Firebase Studio troubleshooting \& FAQ](https://firebase.google.com/docs/studio/troubleshooting#integrations).

## Use the Firebase Studio panel

With fully-integrated workflows from Firebase Studio, you can add
Google and Firebase APIs and services to your app with the click of a button.

1. Open [Firebase Studio](https://firebase.google.com/docs/studio/https://studio.firebase.google.com/).

2. In Code view, click the Firebase Studio
   icon on the right side of the workspace.

3. Expand the **Google Integrations** section to see the list of available
   workflows.

4. Click the action you want to take for the API or service you want to add to
   your app.

5. Follow the prompts to authenticate your account, select a billing project
   (if required), and add the selected service to your app.

   The Firebase Studio panel provides the following integrations:
   - [Publish your app with Firebase Hosting or
     Cloud Run](https://firebase.google.com/docs/studio/deploy-app).
   - Enable Gemini Developer API and get a Gemini API key.
   - Set up the [Google Maps API](https://mapsplatform.google.com/) and get a Maps API key.
   - Use [Google Cloud Secret Manager](https://cloud.google.com/security/products/secret-manager) to securely store API keys and credentials.
