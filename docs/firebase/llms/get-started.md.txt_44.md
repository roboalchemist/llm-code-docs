# Get started with Firebase Studio

{ # disableFinding("traditional") #}

Firebase Studio helps you build and ship full-stack apps infused with AI.
It supports a variety of frameworks, and provides
an agentic, cloud-based development environment that contains
everything you need to develop your application. This includes powerful
tools like emulators, testing frameworks, and an always-available AI
assistant using Gemini in Firebase.

You can get started with Firebase Studio in the following ways:

- [**Import or upload an existing
  project**:](https://firebase.google.com/docs/studio/get-started-import) Import your project from
  source control (GitHub, GitLab, or Bitbucket) or a compressed archive.

  > [!TIP]
  > **Tip:** You can also [export your work to GitHub](https://firebase.google.com/docs/studio/github) directly from Firebase Studio.

<!-- -->

- [**Prototype with AI:**](https://firebase.google.com/docs/studio/get-started-ai) Use the App Prototyping agent with Gemini in Firebase to prototype, develop, test, iterate, and publish a web app using natural language.

<!-- -->

- [**Get started with a template from Firebase Studio's template
  gallery:**](https://firebase.google.com/docs/studio/get-started-template) Firebase Studio
  provides a wide array of
  templates and sample apps that support a
  variety of languages, frameworks, and databases.

- [**Try a guided tour of
  the App Prototyping agent**:](https://firebase.google.com/docs/studio/solution-build-with-ai)
  Develop, publish, and monitor a full-stack web app with
  the App Prototyping agent.

> [!IMPORTANT]
> **Important:** You may need to enable third-party cookies to use Firebase Studio. For instructions on enabling third-party cookies for your browser, see [Third-party cookies aren't
> enabled](https://firebase.google.com/docs/studio/troubleshooting#third-party-cookies).
