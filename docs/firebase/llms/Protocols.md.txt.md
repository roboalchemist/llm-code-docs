# FirebaseAuth Framework Reference

# Protocols

The following protocols are available globally.
- `


  ### [FIRAuthUIDelegate](https://firebase.google.com/docs/reference/ios/firebaseauth/api/reference/Protocols/FIRAuthUIDelegate)


  ` @protocol FIRAuthUIDelegate
  A protocol to handle user interface interactions for Firebase Auth.
  This protocol is available on iOS, macOS Catalyst, and tvOS only.

  #### Declaration

  Objective-C

      @protocol FIRAuthUIDelegate <NSObject>

- `


  ### [FIRFederatedAuthProvider](https://firebase.google.com/docs/reference/ios/firebaseauth/api/reference/Protocols/FIRFederatedAuthProvider)


  ` Utility type for constructing federated auth provider credentials.

  #### Declaration

  Objective-C

      @protocol FIRFederatedAuthProvider <NSObject>

- `


  ### [FIRUserInfo](https://firebase.google.com/docs/reference/ios/firebaseauth/api/reference/Protocols/FIRUserInfo)


  ` Represents user data returned from an identity provider.

  #### Declaration

  Objective-C

      @protocol FIRUserInfo <NSObject>
