# FirebaseInAppMessaging Framework Reference

# Protocols

The following protocols are available globally.
- `


  ### [InAppMessagingDisplayDelegate](https://firebase.google.com/docs/reference/swift/firebaseinappmessaging/api/reference/Protocols/InAppMessagingDisplayDelegate)


  ` A protocol defining those callbacks to be triggered by the message display component
  under appropriate conditions.
  This protocol is unavailable on macOS, macOS Catalyst, and watchOS.

  #### Declaration

  Swift

      protocol InAppMessagingDisplayDelegate : NSObjectProtocol

- `


  ### [InAppMessagingDisplay](https://firebase.google.com/docs/reference/swift/firebaseinappmessaging/api/reference/Protocols/InAppMessagingDisplay)


  ` The protocol that a FIAM display component must implement.
  This protocol is unavailable on macOS, macOS Catalyst, and watchOS.

  #### Declaration

  Swift

      protocol InAppMessagingDisplay
