# FirebaseAppCheck Framework Reference

# Protocols

The following protocols are available globally.
- `


  ### [FIRAppCheckProvider](https://firebase.google.com/docs/reference/ios/firebaseappcheck/api/reference/Protocols/FIRAppCheckProvider)


  ` Defines the methods required to be implemented by a specific Firebase App Check
  provider.

  #### Declaration

  Objective-C

      @protocol FIRAppCheckProvider <NSObject>

- `


  ### [FIRAppCheckProviderFactory](https://firebase.google.com/docs/reference/ios/firebaseappcheck/api/reference/Protocols/FIRAppCheckProviderFactory)


  ` This protocol defines the interface for classes that can create Firebase App Check providers.

  #### Declaration

  Objective-C

      @protocol FIRAppCheckProviderFactory <NSObject>
