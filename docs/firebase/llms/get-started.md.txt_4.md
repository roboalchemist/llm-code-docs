Follow our step-by-step get started guides or try out a sample app.

## Follow our get started guides

These guides provide step-by-step instructions to set up Firebase Cloud Messaging
in your mobile and web client apps so that you can reliably send messages.
For server environments, see
[Your server environment and FCM](https://firebase.google.com/docs/cloud-messaging/server-environment).

- [**Web**
