[What is the KendoReact Notification?](https://www.telerik.com/kendo-react-ui/components/notification#what-is-the-kendoreact-notification)
------------------------------------------------------------------------------------------------------------------------------------------

The Notification is a native KendoReact component built specifically for the React ecosystem and distributed via npm.

> [@progress/kendo-react-notification](https://www.npmjs.com/package/@progress/kendo-react-notification)

### [Key Features](https://www.telerik.com/kendo-react-ui/components/notification#key-features)

The KendoReact Notification provides the following key features. Note that this is not an exhaustive list.

*   Highly customizable and easy to style **appearance**
    *   [Built-in themes](https://www.telerik.com/kendo-react-ui/components/styling)
    *   [Theme customization](https://www.telerik.com/kendo-react-ui/components/styling/customizing)
    *   [Self-created themes](https://www.telerik.com/kendo-react-ui/components/styling/theme-builder)

*   Different built-in [Animations](https://www.telerik.com/kendo-react-ui/components/notification/animations)
*   Built-in [Types](https://www.telerik.com/kendo-react-ui/components/notification/types)
*   Full control over the [Aligning and Positioning](https://www.telerik.com/kendo-react-ui/components/notification/positioning)
*   Out-of-the-box **accessibility** and **keyboard navigation**

[How Does the KendoReact Notification Work?](https://www.telerik.com/kendo-react-ui/components/notification#how-does-the-kendoreact-notification-work)
------------------------------------------------------------------------------------------------------------------------------------------------------

> The KendoReact Notification is a [free React Notification](https://www.telerik.com/kendo-react-ui/components/notification#package-contents) that doesn't require a commercial license key or an active trial license key to install or use.

The KendoReact Notification is built natively for React, with no dependencies. They leverage key React concepts such as components-based architecture, props and state management, conditional rendering, and event handling.

You have complete control over your React notification, and you can extend or override its default behavior and appearance via props and event handlers.

### [React Notification with Other KendoReact Components](https://www.telerik.com/kendo-react-ui/components/notification#react-notification-with-other-kendoreact-components)

You can combine multiple KendoReact components with the React notification in the same application:

*   Display different notification types after clicking a [Button](https://www.telerik.com/kendo-react-ui/components/grid)
*   Show a message after a [Form](https://www.telerik.com/kendo-react-ui/components/form) has been submitted
*   Display a notification once the [Upload](https://www.telerik.com/kendo-react-ui/components/upload) successfully completes a file upload

> The Notification package is a dependency for multiple other KendoReact components.
