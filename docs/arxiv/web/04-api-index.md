# arXiv API Documentation Index

## Overview

arXiv provides public API access to "maximize its openness and interoperability." Key distinctions are made between commercial and noncommercial projects.

- **API Basics** - Foundational API information and quick start
- **API User Manual** - Comprehensive usage guide with detailed specifications
- **Terms of Use** - Legal requirements for API usage

## Usage Requirements

All API users must:

- Review Terms of Use and API Basics
- Acknowledge arXiv with: "Thank you to arXiv for use of its open access interoperability"
- Avoid branding conflicts with arXiv names/logos
- Engage with the API user community

## Project Classification

**Noncommercial projects** meeting specific criteria are considered independent, while **commercial projects** require additional review of brand guidelines and potential affiliation.
