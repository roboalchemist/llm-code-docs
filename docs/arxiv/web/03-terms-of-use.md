# arXiv API Terms of Use

## Core Agreements

By using arXiv APIs, users consent to several fundamental principles:

### Copyright Awareness

"E-prints from arXiv are subject to all applicable copyright protections. The redistribution of e-prints requires permission from the copyright holder."

### Metadata Usage

Users may freely use descriptive metadata under the "Creative Commons Universal (CC0 1.0) Public Domain Declaration."

### Code of Conduct Compliance

Users must abide by arXiv's Code of Conduct in all interactions.

### Rate Limit Respect

Users cannot attempt to circumvent usage limitations or authorization mechanisms.

### API Changes

Users are responsible for monitoring updates to APIs via arXiv News and the API group.

### Data Collection

arXiv collects private information (name, email) for support and improvement purposes, governed by their Privacy Policy.

## Rate Limits

For legacy APIs (OAI-PMH, RSS, arXiv API): "Make no more than one request every three seconds, and limit requests to a single connection at a time." Those needing higher rates should contact support.

## Permitted Uses

- Retrieve, store, and transform descriptive metadata
- Use e-print content for personal or research purposes
- Build discovery tools, mobile apps, and visualizations
- Direct users to arXiv.org for content access

## Prohibited Uses

- Serving e-prints from external servers without copyright holder permission
- Claiming endorsement without authorization
- Circumventing rate limits
- Using others' credentials
