### Overview [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#pkg-overview "Go to Overview")

Package vptree implements a vantage point tree. Vantage point trees provide an efficient search for nearest neighbors in a metric space.

See [http://pnylab.com/papers/vptree/vptree.pdf](http://pnylab.com/papers/vptree/vptree.pdf) for details of vp-trees.

Example (AccessiblePublicTransport) [¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#example-package-AccessiblePublicTransport "Go to Example (AccessiblePublicTransport)")

Output:

 Stations within 750 m of 51.501476N 0.140634W. St. James's Park: 0.545 km Green Park: 0.600 km Victoria: 0.621 km 5 closest stations to 51.501476N 0.140634W. St. James's Park: 0.545 km Green Park: 0.600 km Victoria: 0.621 km Hyde Park Corner: 0.846 km Picadilly Circus: 1.027 km

#### type [Comparable](https://github.com/gonum/gonum/blob/v0.17.0/spatial/vptree/vptree.go#L18)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#Comparable "Go to Comparable")

type Comparable interface {
// Distance returns the distance between the receiver and the	// parameter. The returned distance must satisfy the properties
	// of distances in a metric space.
	//
	// - a.Distance(a) == 0
	// - a.Distance(b) >= 0
	// - a.Distance(b) == b.Distance(a)
	// - a.Distance(b) <= a.Distance(c)+c.Distance(b)
	//
	Distance([Comparable](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#Comparable)) [float64](https://pkg.go.dev/builtin#float64)
}

Comparable is the element interface for values stored in a vp-tree.

#### type [ComparableDist](https://github.com/gonum/gonum/blob/v0.17.0/spatial/vptree/vptree.go#L260)[¶](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#ComparableDist "Go to ComparableDist")

type ComparableDist struct {
 Comparable [Comparable](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#Comparable) Dist [float64](https://pkg.go.dev/builtin#float64)}

func New(p [][Comparable](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#Comparable), effort [int](https://pkg.go.dev/builtin#int), src [rand](https://pkg.go.dev/math/rand/v2).[Source](https://pkg.go.dev/math/rand/v2#Source)) (t *[Tree](https://pkg.go.dev/gonum.org/v1/gonum@v0.17.0/spatial/vptree#Tree), err [error](https://pkg.go.dev/builtin#error))

New returns a vantage point tree constructed from the values in p. The effort parameter specifies how much work should be put into optimizing the choice of vantage point. If effort is one or less, random vantage points are chosen. The order of elements in p will be altered after New returns. The src parameter provides the source of randomness for vantage point selection. If src is nil global rand package functions are used. Points in p must not be infinitely distant.
