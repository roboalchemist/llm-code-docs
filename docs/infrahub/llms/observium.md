# Observium adapter

## What is Observium?[​](#what-is-observium "Direct link to What is Observium?")

Observium is a network monitoring platform that delivers auto-discovery, detailed performance metrics, and rich visualizations to help you effectively manage and optimize your network infrastructure.

## Sync directions supported[​](#sync-directions-supported "Direct link to Sync directions supported")

* Observium → Infrahub

info

Currently, the Observium adapter supports only **one-way synchronization** from Observium to Infrahub. Syncing data back into Observium is not yet supported.
