# environment set-active

Sets an environment as the active environment

```
kreyac environment set-active [EnvironmentName]
```

## Example[​](#example "Direct link to Example")

```
kreyac environment set-active -p /projects/my-project.krproj Production
```

## Arguments[​](#arguments "Direct link to Arguments")

### EnvironmentName `required`[​](#environmentname-required "Direct link to environmentname-required")

The name of the environment to activate
