## Pro & Enterprise feature highlights

Explore the powerful features that are not in included in the Free version

### Scripting & testing

Define tests with scripting and automatically test your API with each call. Straightforward without any complicated setup.

![Screenshot displaying Kreya Scripting and tests](/assets/ideal-img/kreya-scripting.032fb6a.400.png)

### Snapshot tests

Snapshot tests automatically capture and compare API responses, alerting you to any unexpected changes.

![Screenshot displaying Kreya Snapshot tests](/assets/ideal-img/kreya-snapshots.3b9b882.400.png)

### Collections

Run your operations in collections. All operations will be executed in the defined order.

![Screenshot of a Kreya collection with tests](/assets/ideal-img/kreya-collection.8a56f31.400.png)

### History of operations and responses

View all past invoked operations including request and response details.

![Screenshot of the Kreya operation history with multiple entries](/assets/ideal-img/kreya-history.34a0378.400.png)

### Operation and server timing

View the timing details of your requests.

![Screenshot of the timing chart](/assets/ideal-img/kreya-timing.6a1face.400.png)

### Customer portal

Manage the users of your organization.

![Screenshot of the Kreya customer portal](/assets/ideal-img/kreya-membreco.38bf960.400.png)
