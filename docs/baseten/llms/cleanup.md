# truss cleanup

> Clean up Truss data.

```sh  theme={"system"}
truss cleanup [OPTIONS]
```

Clears temporary directories created by Truss for operations like building Docker images. Use this to free up disk space.

**Example:**

To clean up temporary Truss data, use the following:

```sh  theme={"system"}
truss cleanup
```
