# [SlideToggle](https://bryntum.com/docs/gantt/api/Core/widget/SlideToggle)

SlideToggle field is a variation of [Checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) with a sliding toggle instead of box with check mark. It wraps `<input type="checkbox">`. Color can be specified, and you can optionally configure [text](https://bryntum.com/docs/gantt/api/#Core/widget/SlideToggle#config-text) to display in a label to the right of the toggle in addition to a standard field [label](https://bryntum.com/docs/gantt/api/#Core/widget/SlideToggle#config-label).

This field can be used as an [editor](https://bryntum.com/docs/gantt/api/#Grid/column/Column#config-editor) for the [Column](https://bryntum.com/docs/gantt/api/#Grid/column/Column).
