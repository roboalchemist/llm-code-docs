# [FieldSet](https://bryntum.com/docs/gantt/api/Core/widget/FieldSet)

The `FieldSet` widget wraps an `<fieldset>` element. A fieldset is a specially styled [Panel](https://bryntum.com/docs/gantt/api/#Core/widget/Panel) intended to hold form fields.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[inline](https://bryntum.com/docs/gantt/api/Core/widget/FieldSet#config-inline)
Setting this config to `true` assigns a horizontal box layout (`flex-flow: row`) to the items in this container, while `false` assigns a vertical box layout (`flex-flow: column`).

By default, this value is automatically determined based on the [label](https://bryntum.com/docs/gantt/api/#Core/widget/FieldSet#config-label) and [labelPosition](https://bryntum.com/docs/gantt/api/#Core/widget/FieldSet#config-labelPosition) configs.
