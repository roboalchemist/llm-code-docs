# [IgnoreResourceCalendarColumn](https://bryntum.com/docs/gantt/api/Gantt/column/IgnoreResourceCalendarColumn)

A column that displays (and allows user to change) whether the task ignores its assigned resource calendars when scheduling or not ([ignoreResourceCalendar](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-ignoreResourceCalendar) field).

This column uses a [checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) as its editor, and it is not intended to be changed.
