# [DependencyTab](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/taskeditor/DependencyTab)

Abstract base class for SuccessorsTab and PredecessorsTab.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[dependencyIdField](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/taskeditor/DependencyTab#config-dependencyIdField)
A task field (`id`, `wbsCode`, `sequenceNumber` etc) that will be used when displaying and editing linked tasks. Defaults to Gantt `dependencyIdField`.

[taskComboSortField](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/taskeditor/DependencyTab#config-taskComboSortField)
A task field (`id`, `wbsCode`, `sequenceNumber` etc) to sort tasks in the task combo by
