# [EffortColumn](https://bryntum.com/docs/gantt/api/Gantt/column/EffortColumn)

A column showing the task [effort](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-effort) and [units](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-effortUnit). The editor of this column understands the time units, so user can enter "4d" indicating 4 days effort, or "4h" indicating 4 hours, etc. The numeric magnitude can be either an integer or a float value. Both "," and "." are valid decimal separators. For example, you can enter "4.5d" indicating 4.5 days duration, or "4,5h" indicating 4.5 hours.

Default editor is a [DurationField](https://bryntum.com/docs/gantt/api/#Core/widget/DurationField).
