# [TimeRangeModel](https://bryntum.com/docs/gantt/api/Scheduler/model/TimeRangeModel)

This class represents a named time range. It is used by the [TimeRanges](https://bryntum.com/docs/gantt/api/#Scheduler/feature/TimeRanges) feature.

This class inherits most of its fields from [TimeSpan](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeSpan). The most important of these fields are the following:

* [name](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeRangeModel#field-name)
* [startDate](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeRangeModel#field-startDate)
* [endDate](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeRangeModel#field-endDate)
* [iconCls](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeRangeModel#field-iconCls)

It's an indirect subclass of [Model](https://bryntum.com/docs/gantt/api/#Core/data/Model). Please refer to documentation of those classes to become familiar with the base interface of this class. The data source of any field can be customized in the subclass. Please refer to [Model](https://bryntum.com/docs/gantt/api/#Core/data/Model) for details.

Time ranges may recur. See the [recurrenceRule](https://bryntum.com/docs/gantt/api/#Scheduler/model/TimeRangeModel#field-recurrenceRule) field.
