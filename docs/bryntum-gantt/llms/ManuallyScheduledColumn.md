# [ManuallyScheduledColumn](https://bryntum.com/docs/gantt/api/Gantt/column/ManuallyScheduledColumn)

A column that displays (and allows user to update) the task's [manuallyScheduled](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-manuallyScheduled) field.

This column uses a [Checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) as its editor, and it is not intended to be changed.
