# [SchedulingModeColumn](https://bryntum.com/docs/gantt/api/Gantt/column/SchedulingModeColumn)

A column which displays a task's scheduling [mode](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-schedulingMode) field.

Default editor is a [SchedulingModePicker](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/SchedulingModePicker).
