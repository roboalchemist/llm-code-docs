# [DemoHeader](https://bryntum.com/docs/gantt/api/Core/widget/DemoHeader)

This is an internal class used to create a demo header for Bryntum framework apps. Not required for end-user application.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[insertFirst](https://bryntum.com/docs/gantt/api/Core/widget/DemoHeader#config-insertFirst)
The ID of the owner div element, which will have a header as first child

[appFolder](https://bryntum.com/docs/gantt/api/Core/widget/DemoHeader#config-appFolder)
The path to the application folder to show code viewer
