# [CalendarEditorWeekTab](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/calendareditor/CalendarEditorWeekTab)

The [calendar editor](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/CalendarEditor) "Weeks" tab. The tab contains a [grid](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/calendareditor/CalendarEditorWeekGrid) and a [panel](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/calendareditor/CalendarEditorWeekPanel) partnered to display and edit week intervals respectively.
