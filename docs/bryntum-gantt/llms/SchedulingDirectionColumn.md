# [SchedulingDirectionColumn](https://bryntum.com/docs/gantt/api/Gantt/column/SchedulingDirectionColumn)

This is a column class for the [scheduling direction](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-direction) field of the task model. Please refer to the documentation of that field for more details.

Default editor is a [SchedulingDirectionPicker](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/SchedulingDirectionPicker).

The direction can be one of:

* Forward
* Backward
