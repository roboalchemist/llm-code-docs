- [Installation](#Gantt/guides/npm/repository/installation.md) - Installing trial and licensed packages, migrating

- [Components and wrappers](#Gantt/guides/npm/repository/components.md) - Available components, framework wrappers,
  and demo resources
- [Automation](#Gantt/guides/npm/repository/automation.md) - CI/CD integration, Artifactory, and offline packages
- [Troubleshooting](#Gantt/guides/npm/repository/troubleshooting.md) - Common issues and online references
