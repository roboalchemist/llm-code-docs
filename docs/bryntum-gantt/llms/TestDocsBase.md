# [TestDocsBase](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase)

Base class for subclassing. This class is required to check how docs are generated.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[_testGrandChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testGrandChildDescription)
TestDocsBase node. This description is NOT used in the docs of GrandChild because description is specified in GrandChild.

[_testChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testChildDescription)
TestDocsBase node. This description is NOT used in the docs of GrandChild because description is specified in Child and there is no description in GrandChild and ChildMixin.

[_testBaseDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testBaseDescription)
TestDocsBase node. This description is USED in the docs of GrandChild because there is no description in Child, GrandChild and ChildMixin.

[_testMixinGrandChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testMixinGrandChildDescription)
TestDocsBase node. This description is NOT used in the docs of GrandChild because description is specified in GrandChild.

[_testMixinChildDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testMixinChildDescription)
TestDocsBase node. This description is NOT used in the docs of GrandChild because description is specified in ChildMixin and there is no description in GrandChild.

[_testMixinBaseDescription](https://bryntum.com/docs/gantt/api/Core/test/docs/TestDocsBase#config-_testMixinBaseDescription)
TestDocsBase node. This description is NOT used in the docs of GrandChild because description is specified in ChildMixin and there is no description in Child and GrandChild.
