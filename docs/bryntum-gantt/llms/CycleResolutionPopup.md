# [CycleResolutionPopup](https://bryntum.com/docs/gantt/api/SchedulerPro/widget/CycleResolutionPopup)

Class implementing a dialog informing user of an infinite cycle in the data. The dialog displays tasks and dependencies causing the cycle and allows to pick one of the dependencies and either deactivate or remove it.
