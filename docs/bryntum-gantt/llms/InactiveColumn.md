# [InactiveColumn](https://bryntum.com/docs/gantt/api/Gantt/column/InactiveColumn)

A column that displays (and allows user to update) the task's [inactive](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-inactive) field.

This column uses a [Checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) as its editor, and it is not intended to be changed.
