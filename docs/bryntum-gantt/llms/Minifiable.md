# [Minifiable](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Minifiable)

Mixin for widgets that can present in a full and minified form. This behavior is used in [overflow](https://bryntum.com/docs/gantt/api/#Core/widget/Toolbar#config-overflow) handling.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[minifiable](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Minifiable#config-minifiable)
Set to `false` to prevent this widget from assuming its [minified](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Minifiable#config-minified) form automatically (for example, due to [overflow](https://bryntum.com/docs/gantt/api/#Core/widget/Toolbar#config-overflow) handling.

When this value is `true` (the default), the minifiable widget's [minified](https://bryntum.com/docs/gantt/api/#Core/widget/mixin/Minifiable#config-minified) config may be set to `true` to reduce toolbar overflow.

[minified](https://bryntum.com/docs/gantt/api/Core/widget/mixin/Minifiable#config-minified)
Set to `true` to present this widget in its minimal form.
