# [AddAction](https://bryntum.com/docs/gantt/api/Core/data/stm/action/AddAction)

Action to record the fact of models adding to a store.

## Configs

Configs are options you supply in a configuration object when creating an instance of this class

[store](https://bryntum.com/docs/gantt/api/Core/data/stm/action/AddAction#config-store)
Reference to a store that models have been added into.

[modelList](https://bryntum.com/docs/gantt/api/Core/data/stm/action/AddAction#config-modelList)
List of models added into the store.

[silent](https://bryntum.com/docs/gantt/api/Core/data/stm/action/AddAction#config-silent)
Flag showing if undo/redo should be done silently i.e. with events suppressed
