# [ShowInTimelineColumn](https://bryntum.com/docs/gantt/api/Gantt/column/ShowInTimelineColumn)

Column that shows if a task should be shown in the [Timeline](https://bryntum.com/docs/gantt/api/#SchedulerPro/widget/Timeline) widget.

This column uses a [checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) as its editor, and it is not intended to be changed.
