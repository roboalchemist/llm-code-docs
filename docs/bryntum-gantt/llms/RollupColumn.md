# [RollupColumn](https://bryntum.com/docs/gantt/api/Gantt/column/RollupColumn)

A column that displays a checkbox to edit the [rollup](https://bryntum.com/docs/gantt/api/#Gantt/model/TaskModel#field-rollup) data field. This field indicates if a task should rollup to its closest parent or not. Requires the [Rollups](https://bryntum.com/docs/gantt/api/#Gantt/feature/Rollups) feature to be enabled.

This column uses a [Checkbox](https://bryntum.com/docs/gantt/api/#Core/widget/Checkbox) as its editor, and it is not intended to be changed.
