Advanced Features for Reference to Variables
============================================

This section describes some advanced features you can use to reference variables in more flexible ways.

• [Substitution Refs](https://devdocs.io/gnu_make/substitution-refs)Referencing a variable with substitutions on the value.
• [Computed Names](https://devdocs.io/gnu_make/computed-names)Computing the name of the variable to refer to.
