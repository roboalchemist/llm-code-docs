### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.10.0#pkg-overview "Go to Overview")

Package semconv implements OpenTelemetry semantic conventions.

OpenTelemetry semantic conventions are agreed standardized naming patterns for OpenTelemetry things. This package represents the conventions as of the v1.10.0 version of the OpenTelemetry specification.
