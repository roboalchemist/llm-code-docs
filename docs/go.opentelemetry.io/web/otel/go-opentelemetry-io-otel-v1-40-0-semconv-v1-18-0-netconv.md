### Functions [¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.18.0/netconv#pkg-functions "Go to Functions")

#### func [Client](https://github.com/open-telemetry/opentelemetry-go/blob/v1.40.0/semconv/v1.18.0/netconv/net.go#L44)[¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.18.0/netconv#Client "Go to Client")

func Client(address [string](https://pkg.go.dev/builtin#string), conn [net](https://pkg.go.dev/net).[Conn](https://pkg.go.dev/net#Conn)) [][attribute](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute).[KeyValue](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute#KeyValue)

Client returns trace attributes for a client network connection to address. See net.Dial for information about acceptable address values, address should be the same as the one used to create conn. If conn is nil, only network peer attributes will be returned that describe address. Otherwise, the socket level information about conn will also be included.

#### func [Server](https://github.com/open-telemetry/opentelemetry-go/blob/v1.40.0/semconv/v1.18.0/netconv/net.go#L53)[¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.18.0/netconv#Server "Go to Server")

func Server(address [string](https://pkg.go.dev/builtin#string), ln [net](https://pkg.go.dev/net).[Listener](https://pkg.go.dev/net#Listener)) [][attribute](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute).[KeyValue](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute#KeyValue)

Server returns trace attributes for a network listener listening at address. See net.Listen for information about acceptable address values, address should be the same as the one used to create ln. If ln is nil, only network host attributes will be returned that describe address. Otherwise, the socket level information about ln will also be included.

#### func [Transport](https://github.com/open-telemetry/opentelemetry-go/blob/v1.40.0/semconv/v1.18.0/netconv/net.go#L35)[¶](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/semconv/v1.18.0/netconv#Transport "Go to Transport")

func Transport(network [string](https://pkg.go.dev/builtin#string)) [attribute](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute).[KeyValue](https://pkg.go.dev/go.opentelemetry.io/otel@v1.40.0/attribute#KeyValue)

Transport returns a trace attribute describing the transport protocol of the passed network. See the net.Dial for information about acceptable network values.
