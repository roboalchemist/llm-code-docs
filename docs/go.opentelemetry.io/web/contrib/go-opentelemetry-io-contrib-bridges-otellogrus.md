### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otellogrus#pkg-overview "Go to Overview")

*   [Record Conversion](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otellogrus#hdr-Record_Conversion)

Package otellogrus provides a [Hook](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otellogrus#Hook), a [logrus.Hook](https://pkg.go.dev/github.com/sirupsen/logrus#Hook) implementation that can be used to bridge between the [github.com/sirupsen/logrus](https://pkg.go.dev/github.com/sirupsen/logrus) API and [OpenTelemetry](https://opentelemetry.io/docs/concepts/signals/logs/).

#### Record Conversion [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otellogrus#hdr-Record_Conversion "Go to Record Conversion")

The [logrus.Entry](https://pkg.go.dev/github.com/sirupsen/logrus#Entry) records are converted to OpenTelemetry [log.Record](https://pkg.go.dev/go.opentelemetry.io/otel/log#Record) in the following way:

*   Time is set as the Timestamp.
*   Message is set as the Body using a [log.StringValue](https://pkg.go.dev/go.opentelemetry.io/otel/log#StringValue).
*   Level is transformed and set as the Severity. The SeverityText is also set.
*   Fields are transformed and set as the attributes.

The Level is transformed to the OpenTelemetry Severity types. For example:

*   [logrus.DebugLevel](https://pkg.go.dev/github.com/sirupsen/logrus#DebugLevel) is transformed to [log.SeverityDebug](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityDebug)
*   [logrus.InfoLevel](https://pkg.go.dev/github.com/sirupsen/logrus#InfoLevel) is transformed to [log.SeverityInfo](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityInfo)
*   [logrus.WarnLevel](https://pkg.go.dev/github.com/sirupsen/logrus#WarnLevel) is transformed to [log.SeverityWarn](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityWarn)
*   [logrus.ErrorLevel](https://pkg.go.dev/github.com/sirupsen/logrus#ErrorLevel) is transformed to [log.SeverityError](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityError)
*   [logrus.FatalLevel](https://pkg.go.dev/github.com/sirupsen/logrus#FatalLevel) is transformed to [log.SeverityFatal](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal)
*   [logrus.PanicLevel](https://pkg.go.dev/github.com/sirupsen/logrus#PanicLevel) is transformed to [log.SeverityFatal4](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal4)

Field values are transformed based on their type into log attributes, or into a string value encoded using [fmt.Sprintf](https://pkg.go.dev/fmt#Sprintf) if there is no matching type.

Example [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otellogrus#example-package "Go to Example")

package main

import (
	"github.com/sirupsen/logrus"
	"go.opentelemetry.io/otel/log/noop"

	"go.opentelemetry.io/contrib/bridges/otellogrus"
)

func main() {
	// Use a working LoggerProvider implementation instead e.g. using go.opentelemetry.io/otel/sdk/log.
	provider := noop.NewLoggerProvider()

	// Create an *otellogrus.Hook and use it in your application.
	hook := otellogrus.NewHook("my/pkg/name", otellogrus.WithLoggerProvider(provider))

	// Set the newly created hook as a global logrus hook
	logrus.AddHook(hook)
}
