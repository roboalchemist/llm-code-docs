### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/instrumentation/gopkg.in/macaron.v1/otelmacaron#pkg-overview "Go to Overview")

Package otelmacaron instruments gopkg.in/macaron.v1.

Currently only the routing of a received message can be instrumented. To do it, use the Middleware function.

Deprecated: otelmacaron has no Code Owner. After August 21, 2024, it may no longer be supported and may stop receiving new releases unless a new Code Owner is found. See [this issue](https://github.com/open-telemetry/opentelemetry-go-contrib/issues/5552) if you would like to become the Code Owner of this module.
