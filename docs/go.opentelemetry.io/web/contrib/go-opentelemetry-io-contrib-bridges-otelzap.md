### Overview [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#pkg-overview "Go to Overview")

*   [Record Conversion](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#hdr-Record_Conversion)

Package otelzap provides a bridge between the [go.uber.org/zap](https://pkg.go.dev/go.uber.org/zap) and [OpenTelemetry](https://opentelemetry.io/docs/concepts/signals/logs/).

#### Record Conversion [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#hdr-Record_Conversion "Go to Record Conversion")

The [zapcore.Entry](https://pkg.go.dev/go.uber.org/zap/zapcore#Entry) and [zapcore.Field](https://pkg.go.dev/go.uber.org/zap/zapcore#Field) are converted to OpenTelemetry [log.Record](https://pkg.go.dev/go.opentelemetry.io/otel/log#Record) in the following way:

*   Time is set as the Timestamp.
*   Message is set as the Body using a [log.StringValue](https://pkg.go.dev/go.opentelemetry.io/otel/log#StringValue).
*   Level is transformed and set as the Severity. The SeverityText is also set.
*   Fields are transformed and set as the Attributes.
*   Field value of type [context.Context](https://pkg.go.dev/context#Context) is used as context when emitting log records.
*   For named loggers, LoggerName is used to access [log.Logger](https://pkg.go.dev/go.opentelemetry.io/otel/log#Logger) from [log.LoggerProvider](https://pkg.go.dev/go.opentelemetry.io/otel/log#LoggerProvider)

The Level is transformed to the OpenTelemetry Severity types in the following way.

*   [zapcore.DebugLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#DebugLevel) is transformed to [log.SeverityDebug](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityDebug)
*   [zapcore.InfoLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#InfoLevel) is transformed to [log.SeverityInfo](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityInfo)
*   [zapcore.WarnLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#WarnLevel) is transformed to [log.SeverityWarn](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityWarn)
*   [zapcore.ErrorLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#ErrorLevel) is transformed to [log.SeverityError](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityError)
*   [zapcore.DPanicLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#DPanicLevel) is transformed to [log.SeverityFatal1](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal1)
*   [zapcore.PanicLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#PanicLevel) is transformed to [log.SeverityFatal2](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal2)
*   [zapcore.FatalLevel](https://pkg.go.dev/go.uber.org/zap/zapcore#FatalLevel) is transformed to [log.SeverityFatal3](https://pkg.go.dev/go.opentelemetry.io/otel/log#SeverityFatal3)

Fields are transformed based on their type into log attributes, or into a string value encoded using [fmt.Sprintf](https://pkg.go.dev/fmt#Sprintf) if there is no matching type.

Example [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#example-package "Go to Example")

package main

import (
	"context"

	"go.opentelemetry.io/otel/log/noop"
	"go.uber.org/zap"

	"go.opentelemetry.io/contrib/bridges/otelzap"
)

func main() {
	// Use a working LoggerProvider implementation instead e.g. use go.opentelemetry.io/otel/sdk/log.
	provider := noop.NewLoggerProvider()

	// Initialize a zap logger with the otelzap bridge core.
	// This method actually doesn't log anything on your STDOUT, as everything
	// is shipped to a configured otel endpoint.
	logger := zap.New(otelzap.NewCore("my/pkg/name", otelzap.WithLoggerProvider(provider)))

	// You can now use your logger in your code.
	logger.Info("something really cool")

	// You can set context for trace correlation using zap.Any or zap.Reflect
	ctx := context.Background()
	logger.Info("setting context", zap.Any("context", ctx))
}

Example (Multiple) [¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#example-package-Multiple "Go to Example (Multiple)")

package main

import (
	"os"

	"go.opentelemetry.io/otel/log/noop"
	"go.uber.org/zap"
	"go.uber.org/zap/zapcore"

	"go.opentelemetry.io/contrib/bridges/otelzap"
)

func main() {
	// Use a working LoggerProvider implementation instead e.g. use go.opentelemetry.io/otel/sdk/log.
	provider := noop.NewLoggerProvider()

	// If you want to log also on stdout, you can initialize a new zap.Core
	// that has multiple outputs using the method zap.NewTee(). With the following code,
	// logs will be written to stdout and also exported to the OTEL endpoint through the bridge.
	core := zapcore.NewTee(
		zapcore.NewCore(zapcore.NewJSONEncoder(zap.NewProductionEncoderConfig()), zapcore.AddSync(os.Stdout), zapcore.InfoLevel),
		otelzap.NewCore("my/pkg/name", otelzap.WithLoggerProvider(provider)),
	)
	logger := zap.New(core)

	// You can now use your logger in your code.
	logger.Info("something really cool")
}

Core is a [zapcore.Core](https://pkg.go.dev/go.uber.org/zap/zapcore#Core) that sends logging records to OpenTelemetry.

#### func [NewCore](https://github.com/open-telemetry/opentelemetry-go-contrib/blob/bridges/otelzap/v0.15.0/bridges/otelzap/core.go#L132)[¶](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#NewCore "Go to NewCore")

func NewCore(name [string](https://pkg.go.dev/builtin#string), opts ...[Option](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#Option)) *[Core](https://pkg.go.dev/go.opentelemetry.io/contrib/bridges/otelzap#Core)

NewCore creates a new [zapcore.Core](https://pkg.go.dev/go.uber.org/zap/zapcore#Core) that can be used with [go.uber.org/zap.New](https://pkg.go.dev/go.uber.org/zap#New).
