AppendMiddlewares attaches OTel middlewares to the AWS Go SDK V2 for instrumentation. OTel middlewares can be appended to either all aws clients or a specific operation. Please see more details in [https://aws.github.io/aws-sdk-go-v2/docs/middleware/](https://aws.github.io/aws-sdk-go-v2/docs/middleware/)
