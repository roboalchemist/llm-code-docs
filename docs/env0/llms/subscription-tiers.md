# Subscription Tiers

env zero offers three packages designed to meet organizations at different stages of their cloud infrastructure journey. All packages include unlimited concurrent runs and unlimited users. To get a quote or discuss which package is right for your organization, [contact us](https://www.env0.com/contact-us).

## env zero Cloud Compass

Chart your course with our rapid assessment tool for IaC coverage and drift risk across unlimited cloud resources.

**\$1,500/month**

Ideal for teams getting visibility into their cloud infrastructure without yet managing deployments through env zero.

**Includes:**

* IaC Coverage Tracking
* Risk Assessment
* IaC Code Generator
* Basic Drift Cause Analysis
* Drift Risk visibility
* Up to 100 Active Environments
* Unlimited Cloud Resources, Concurrent Runs, Users, Monthly Deployments, and Run Minutes
* Email Support

## env zero Cloud Navigator

Command control with our cloud provisioning toolbox. Built for maturing enterprises that need self-service, governance, and automation.

**Pricing per Apply or Environment - [get a free quote](https://www.env0.com/pricing)**

Everything in Cloud Compass, plus:

* IaC Automation & GitOps (GitOps Workflows, Deploy from PR Comments, Custom Flows, Workflows)
* Basic Drift Management (detection, cause analysis, code-to-cloud redeployment)
* Managed Self-Service (Environment Discovery, Slack/Teams & Email Notifications, Environment Scheduling, TTL)
* Basic Visibility & Dashboards
* Private Registry (Module & Provider Registry, Module Testing)
* Access Controls (RBAC, OpenID Connect, SAML Integration, Team Sync, Environment-Level Access, Custom RBAC Roles)
* Governance & Compliance (Custom Approval Policies, SOC 2 Report access)
* Basic Integrations (VCS, IaC frameworks)
* Unlimited Active Environments
* Email Support + Dedicated POC Support + Shared Slack/Teams Channel

## env zero Cloud Pilot

Run at scale with our cloud governance platform, powered by AI, delivering end-to-end control for sophisticated enterprises.

**Pricing per Apply or Environment - [get a free quote](https://www.env0.com/pricing)**

Everything in Cloud Navigator, plus:

* **Cloud Analyst** -AI-powered insights, agent, and dashboards
* Advanced Drift Cause Analysis, Remediation, and Monitoring
* Cost Management (Cost Estimation via Infracost, Actual Cost Visibility, Centralized Cost Tagging, Cost Alerts, Budget Thresholds)
* Self-Hosted VCS (Bitbucket, GitLab & GitHub)
* Proof-of-Value Guarantee
* Advanced Integrations (IDP, FinOps, Observability, PAC/SAST)
* Self-Hosted Agent support
* Enterprise SLA + env zero POC Guarantee

## Self-Hosted Agent

env zero can be delivered as a full SaaS offering or as a Hybrid SaaS + Self-Hosted Agent. The Self-Hosted Agent is available on Cloud Navigator and Cloud Pilot. [Contact us](https://www.env0.com/contact-us) for details.
