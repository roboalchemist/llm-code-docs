# 🧠 env zero MCP Server: Fast and Safe Infrastructure Actions from IDEs and Agents

> We're excited to introduce the env zero MCP Server – a major step forward for teams working in GitOps and IDE-based workflows.

We're excited to introduce the env zero MCP Server – a major step forward for teams working in GitOps and IDE-based workflows.

The MCP Server brings faster, more interactive infrastructure deployment straight into your IDE. With support for all major IaC frameworks, you can now:

* **Generate and test infrastructure** code right from your editor
* **Pull plan logs and AI-powered suggestions** to debug failed runs quickly
* **Approve, cancel, or rerun deployments** without switching tools
* **Codify cloud resources** into Infrastructure as Code snippets

The MCP Server keeps all your existing governance and policy enforcement intact – now just with faster iterations and tighter feedback loops.

* Same guardrails
* Much faster inner loop
* Works with your AI agent or editor

The env zero MCP Server is available now as an open source project. Install it from your IDE's catalog or run it locally with Docker, then follow our installation instructions to connect it with your env zero API token and start interacting with your environments directly from your editor.
