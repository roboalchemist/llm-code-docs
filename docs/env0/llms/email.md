# Email Notifications

> Set up email notifications in env zero for deployment events sent to specified addresses

To receive notifications via email address, enter the name of the Notification target and select Email in the Type dropdown menu:

<img src="https://mintcdn.com/envzero-b61043c8/f50k_bxcw7fbjToJ/images/guides/integrations/notifications/39a47be-image.png?fit=max&auto=format&n=f50k_bxcw7fbjToJ&q=85&s=bef7fbca31ec6021fd29767059692e3d" alt="" width="530" height="500" data-path="images/guides/integrations/notifications/39a47be-image.png" />

Next, enter a list of comma-separated email addresses where you want to receive the notifications.

You can also check the `Environment creator` checkbox to send the notifications to the user who created the environment, and the `Deployer` checkbox to send it to the user who started deployment.
