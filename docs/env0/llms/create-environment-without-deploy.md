# 📍 Create Environment Without Deploy

> Streamline your CI/CD workflow in env0 with `preventAutoDeploy`. This new [Create Environment API](https://docs.env0.com/reference/environments-create) capability allows you to create environments ready for continuous deployment (CD) while preventing immediate deployments. This gives you more control over your pipeline execution.

The default behavior of env0 immediately deploying environments upon creation might not suit all users' workflows. Some pipelines require additional configuration before deployment.

# ✨ Create Environment Without Deploy

Streamline your CI/CD workflow in env0 with `preventAutoDeploy`. This new [Create Environment API](https://docs.env0.com/reference/environments-create) capability enables you to create environments prepared for continuous deployment (CD) while preventing immediate execution, giving you more control over your pipeline.

This new capability is available to both API users and those using [env0 Terraform Provider](https://registry.terraform.io/providers/env0/env0/latest/docs/resources/environment).
