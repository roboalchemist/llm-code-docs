# 🔓 Official OpenTofu Support

> We're excited to announce that OpenTofu is now natively supported by env zero!
Previously named OpenTF, OpenTofu is a fork of Terraform that is open-source, community-driven, and managed by the Linux Foundation.
OpenTofu is backward compatible with Terraform. This means you can start using OpenTofu today with your existing code - just head over to the templates page and create a new OpenTofu template!
