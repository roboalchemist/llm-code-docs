# 🔗New Backstage Plugin: Manage and Deploy IaC from Your Internal Developer Portal

> The new env zero Backstage Plugin brings IaC management directly into the developer portal, enabling developers to create, deploy, and monitor cloud environments directly from Backstage.

The new env zero Backstage Plugin brings IaC management directly into the developer portal, enabling developers to create, deploy, and monitor cloud environments directly from Backstage.

# **Key Capabilities:**

**For teams responsible for platform operations**

* Configure Backstage templates aligned with organizational policies and workflows
* Enable developers to create and manage environments through env zero

**For engineers deploying in a self-service model**

* Select templates and deploy environments directly within Backstage
