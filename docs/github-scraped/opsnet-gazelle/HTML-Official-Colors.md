# HTML Official Colors

**Orpheus Development Papers #6 - HTML Official Colors**
Version 1 | From: Spine | Date: 2020-03-30

## Overview

List of valid colors to be used with the `[color]` BBCode tag. This should
match the list of colors defined in the `Text::$ColorName` constant.
