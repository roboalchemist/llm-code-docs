:::{rubric} About
:::

[MQTT] is an OASIS standard messaging protocol for the Internet of Things (IoT).
[Mosquitto] is an open-source (EPL/EDL licensed) message broker that
implements the MQTT protocol versions 5.0, 3.1.1 and 3.1. Mosquitto is
lightweight and is suitable for use on all devices from low-power single-board
computers to full servers.
