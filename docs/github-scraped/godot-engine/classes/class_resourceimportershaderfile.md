# ResourceImporterShaderFile

**Inherits:** [ResourceImporter<class_ResourceImporter>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

Imports native GLSL shaders (not Godot shaders) as an [RDShaderFile<class_RDShaderFile>].


## Description

This imports native GLSL shaders as [RDShaderFile<class_RDShaderFile>] resources, for use with low-level [RenderingDevice<class_RenderingDevice>] operations. This importer does *not* handle `.gdshader` files.
