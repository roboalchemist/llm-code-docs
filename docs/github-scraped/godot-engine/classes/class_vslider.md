# VSlider

**Inherits:** [Slider<class_Slider>] **<** [Range<class_Range>] **<** [Control<class_Control>] **<** [CanvasItem<class_CanvasItem>] **<** [Node<class_Node>] **<** [Object<class_Object>]

A vertical slider that goes from bottom (min) to top (max).


## Description

A vertical slider, used to adjust a value by moving a grabber along a vertical axis. It is a [Range<class_Range>]-based control and goes from bottom (min) to top (max). Note that this direction is the opposite of [VScrollBar<class_VScrollBar>]'s.
