# VSeparator

**Inherits:** [Separator<class_Separator>] **<** [Control<class_Control>] **<** [CanvasItem<class_CanvasItem>] **<** [Node<class_Node>] **<** [Object<class_Object>]

A vertical line used for separating other controls.


## Description

A vertical separator used for separating other controls that are arranged **horizontally**. **VSeparator** is purely visual and normally drawn as a [StyleBoxLine<class_StyleBoxLine>].
