# AudioEffectBandPassFilter

**Inherits:** [AudioEffectFilter<class_AudioEffectFilter>] **<** [AudioEffect<class_AudioEffect>] **<** [Resource<class_Resource>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

Adds a band pass filter to the audio bus.


## Description

Attenuates the frequencies inside of a range around the [AudioEffectFilter.cutoff_hz<class_AudioEffectFilter_property_cutoff_hz>] and cuts frequencies outside of this band.
