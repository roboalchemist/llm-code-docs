# VisualShaderNodeTexture2DArrayParameter

**Inherits:** [VisualShaderNodeTextureParameter<class_VisualShaderNodeTextureParameter>] **<** [VisualShaderNodeParameter<class_VisualShaderNodeParameter>] **<** [VisualShaderNode<class_VisualShaderNode>] **<** [Resource<class_Resource>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

A visual shader node for shader parameter (uniform) of type [Texture2DArray<class_Texture2DArray>].


## Description

This parameter allows to provide a collection of textures for the shader. You can use [VisualShaderNodeTexture2DArray<class_VisualShaderNodeTexture2DArray>] to extract the textures from array.
