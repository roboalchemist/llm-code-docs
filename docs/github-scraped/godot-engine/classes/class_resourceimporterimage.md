# ResourceImporterImage

**Inherits:** [ResourceImporter<class_ResourceImporter>] **<** [RefCounted<class_RefCounted>] **<** [Object<class_Object>]

Imports an image for use in scripting, with no rendering capabilities.


## Description

This importer imports [Image<class_Image>] resources, as opposed to [CompressedTexture2D<class_CompressedTexture2D>]. If you need to render the image in 2D or 3D, use [ResourceImporterTexture<class_ResourceImporterTexture>] instead.
