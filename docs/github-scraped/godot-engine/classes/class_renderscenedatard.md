# RenderSceneDataRD

**Inherits:** [RenderSceneData<class_RenderSceneData>] **<** [Object<class_Object>]

Render scene data implementation for the RenderingDevice based renderers.


## Description

Object holds scene data related to rendering a single frame of a viewport. See also [RenderSceneData<class_RenderSceneData>], [RenderData<class_RenderData>], and [RenderDataRD<class_RenderDataRD>].

\ **Note:** This is an internal rendering server object. Do not instantiate this class from a script.
