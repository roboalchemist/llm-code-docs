# Product limits

This page describes some of the product limits that apply to Mattermost,
including hard and recommended limits.

  Feature                                                                                                                                                      Hard Limit                             Recommended Limit
  ------------------------------------------------------------------------------------------------------------------------------------------------------------ -------------------------------------- ----------------------------------------------
  `Group messages </end-user-guide/collaborate/create-channels>`{.interpreted-text role="doc"}                                                                 8 participants                         If 8 isn\'t enough, create a private channel
  `Call participants </end-user-guide/collaborate/make-calls>`{.interpreted-text role="doc"}                                                                   Unlimited                              50
  `Custom brand text <administration-guide/configure/custom-branding-tools:custom brand text>`{.interpreted-text role="ref"}                                   500
  `Custom site description <administration-guide/configure/custom-branding-tools:site description>`{.interpreted-text role="ref"}                              1024
  `Advanced log output <administration-guide/manage/logging:advanced logging>`{.interpreted-text role="ref"}                                                   500 MB file size, 1000 audit records
  `Custom status <end-user-guide/preferences/set-your-status-availability:set a custom status>`{.interpreted-text role="ref"}                                  100 characters
  `Channel name <end-user-guide/collaborate/channel-header-purpose:channel name>`{.interpreted-text role="ref"}                                                64 characters
  `Channel purpose <end-user-guide/collaborate/channel-header-purpose:channel purpose>`{.interpreted-text role="ref"}                                          250 characters
  `Channel header <end-user-guide/collaborate/channel-header-purpose:channel header>`{.interpreted-text role="ref"}                                            1024 characters
  `Bookmarked links & files </end-user-guide/collaborate/manage-channel-bookmarks>`{.interpreted-text role="doc"}                                              50
  `Message acknowledgement <end-user-guide/collaborate/message-priority:request acknowledgements>`{.interpreted-text role="ref"} response time                 5 minutes
