# Compliance with Mattermost

Mattermost is purpose-built to help enterprises keep sensitive data safe
and compliant in the strictest, most complex environments. Mattermost
Enterprise Edition includes features designed to make compliance with
all relevant regulations and internal policies easy to achieve and
maintain.

- `Compliance exports </administration-guide/comply/compliance-export>`{.interpreted-text
  role="doc"} - Export compliance reports to third-party systems to
  archive history.
- `Compliance monitoring </administration-guide/comply/compliance-monitoring>`{.interpreted-text
  role="doc"} - Enable oversight and prevent unauthorized queries with
  compliance exports.
- `Electronic discovery </administration-guide/comply/electronic-discovery>`{.interpreted-text
  role="doc"} - Extract data from Mattermost for eDiscovery.
- `Data retention </administration-guide/comply/data-retention-policy>`{.interpreted-text
  role="doc"} - Control how long data is stored in Mattermost with
  global and custom retention policies to meet data retention compliance
  requirements.
- `Export channel data </administration-guide/comply/export-mattermost-channel-data>`{.interpreted-text
  role="doc"} - Migrate data between systems and back data up for
  operational continuity.
- `Legal Hold </administration-guide/comply/legal-hold>`{.interpreted-text
  role="doc"} - Preserve relevant Mattermost information when litigation
  is anticipated.
- `JSON audit log schema </administration-guide/comply/embedded-json-audit-log-schema>`{.interpreted-text
  role="doc"} - Learn how to configure Mattermost audit logging using a
  JSON object.
