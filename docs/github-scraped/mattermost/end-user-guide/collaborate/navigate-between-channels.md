between channels by selecting the **Find channel** option in the channel
sidebar, or by pressing `Ctrl`{.interpreted-text role="kbd"}

## Return to recently viewed channels

Using a web browser or the desktop app, use the **History** arrows at
the top of the sidebar to move back and forth through your channel
history.

- Select the left arrow to go back one page.
- Select the right arrow to go forward one page.
