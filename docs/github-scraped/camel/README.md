# CAMEL Documentation

CAMEL (Communicative Agents for "Mind" Exploration of Large-Scale Language Model Society) is a multi-agent framework for building intelligent systems with LLMs.

## Overview

CAMEL-AI is an open-source, modular framework for building intelligent multi-agent systems. It provides primitives for:

- Creating agents that reason, plan, and act
- Composing societies of agents with defined roles
- Integrating interpreters for code execution
- Managing memory for context and learning
- Building RAG (Retrieval-Augmented Generation) pipelines
- Generating synthetic data at scale
- Simulating worlds and agent interactions
