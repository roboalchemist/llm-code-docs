## About CAMEL

CAMEL (Communicative Agents for "Mind" Exploration of Large-Scale Language Model Society) is a multi-agent framework for exploring the scaling laws of agents.

- **Repository**: https://github.com/camel-ai/camel
- **Website**: https://www.camel-ai.org
- **Documentation**: https://docs.camel-ai.org
- **License**: Apache 2.0
