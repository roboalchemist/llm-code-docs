| Dictionary key | Purpose                                                                                                                                                                                                                                                                                                      |
| --- |--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `"unexpected_index_column_names"` | Takes a list to define the column(s) that will be used to identify unexpected results returned. For example, primary key (PK) column(s) or other columns with unique identifiers.                                                                                                           |
|`"return_unexpected_index_query"` | When running validations, a query (or a set of indices) is returned that allows you to retrieve the full set of unexpected results as well as the values of the identifying columns specified in `"unexpected_index_column_names"`.  Setting this value to `False` suppresses the output (default is `True`). |
| `"partial_unexpected_count"` | Sets the number of results to include in `"partial_missing_list"`, `"partial_unexpected_counts"`, `"partial_unexpected_list"`, `"partial_unexpected_index_list"`, and `"unexpected_rows"`. Set the value to zero to suppress the `partial_*` output.                                                                                                                           |
| `"exclude_unexpected_values"` | When running validations, a set of unexpected results' indices and values is returned.  Setting this value to `True` suppresses values from the output to only have indices (default is `False`).                                                                                                            |
| `"include_unexpected_rows"` | When `True`, this returns up to 200 entire rows that violate the Expectation (default is `False`). Applies to Column Map, Column Pair Map, Multicolumn Map, and Unexpected Rows Expectations only. Note that [`ExpectColumnValuesToBeOfType`](https://greatexpectations.io/expectations/expect_column_values_to_be_of_type/) and [`ExpectColumnValuesToBeInTypeList`](https://greatexpectations.io/expectations/expect_column_values_to_be_in_type_list/) will return unexpected rows for only Pandas Data Sources.|
