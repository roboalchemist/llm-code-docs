## 什么是 Home Assistant？

Home Assistant 是一款免费的**开源**软件，作为家庭自动化的中央控制系统，充当智能家居中枢。它提供了**无需依赖远程服务器或互联网连接的家庭自动化便利，使其高度安全可靠**。它可以通过 Home Assistant Cloud 进行扩展，您可以在任何地方通过语音助手控制您的 Home Assistant，完全加密。

- 使用 [**Home Assistant**](https://www.home-assistant.io/)，您可以本地控制智能家居设备，确保更好的安全性并消除对持续互联网连接的需求。这种本地控制还保证了系统的可靠性，使其能够 24/7 无缝运行。

- 使用 [**Nabu Casa**](https://www.nabucasa.com/)，在保持设备和 Home Assistant 实例之间数据完全加密的同时，您可以从手机、最喜欢的咖啡店或工作场所快速访问您的 Home Assistant 实例。更重要的是，您可以使用 Assist 通过语音控制家庭，它还支持 Amazon Echo 或 Google Home。

Home Assistant 已经超越了家庭自动化，将其影响力扩展到**智能办公室和建筑**，将单个节点转变为一个有凝聚力的**智能城市基础设施**。这种转变通过 Nabu Casa 项目扩展到云端，为在城市范围内管理各种任务提供了可扩展的解决方案。这种架构中互联设备的融合反映了向高效和先进智能生态系统的范式转变。

## 热门插件
 ### Frigate 

<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/wiki/ReTerminal/frigate/frigate2.png" alt="pir" width="200" height="auto"/></p>

 [**Frigate NVR**](https://frigate.video/) 是一款先进的开源网络视频录像机 (NVR)，专为实时 AI 物体检测而设计。它优先在 Raspberry Pi 5 上进行本地处理，确保摄像头数据流保留在您的家中，以增强隐私保护。主要功能包括：

1. **本地物体检测**：启用本地实时 AI 处理，通过将摄像头数据保留在家庭环境中来保护隐私。

2. **减少误报**：采用先进算法，最大限度地减少由阴影或环境条件等因素引起的误报。

3. **可自定义区域**：通过在摄像头视野内定义精确区域，根据您的特定需求定制事件和警报。

4. **集成**：通过与 Home Assistant 和其他自动化平台的无缝集成，实现连接家居体验。

5. **实时监控**：体验动态视频流进行主动监控，让用户能够实时了解其场所内的活动情况。

### ESPHome 

<p style={{textAlign: 'center'}}><img src="https://files.seeedstudio.com/wiki/HomeAsssitant_collection/esphomeicon.png" alt="pir" width="200" height="auto"/></p>

[**ESPHome**](https://esphome.io/) 是一个用户友好的系统，旨在通过家庭自动化系统中的简单配置文件远程控制微控制器。该工具通过读取 YAML 配置文件并生成自定义固件安装到您的 ESP 设备上，简化了 ESP 板的管理。任何集成到 ESPHome 配置中的设备或传感器都会无缝出现在 Home Assistant UI 中，确保无忧的集成体验。

### SenseCraft HA 

[**SenseCraft-HomeAssistant 插件**](https://github.com/Seeed-Solution/SenseCraft-HomeAssistant) 可通过 HACS 商店访问，通过 SenseCraft 数据平台将您的 Grove Vision AI V2 连接到 Home Assistant。这个过程简化且用户友好，只需登录 SenseCraft 账户即可将您的设备和传感器数据引入 Home Assistant 生态系统，准备好供您自定义和控制。
