您好，感谢您访问 Seeed Studio 技术主题知识库。本资源旨在帮助您掌握我们的智能硬件和计算设备系列。

- **[TinyML](/cn/tinyml_topic)** 使即使是低功耗微控制器也能在本地运行机器学习。
  - 我们推出了 [**TinyML 工作坊**](/cn/tinyml_workshop_course_new)（免费课程），让您全面了解嵌入式机器学习，只需要一块 XIAO ESP32S3 Sense（[wiki页面](/cn/xiao_esp32s3_getting_started)），这款强大的MCU开发板。
  - 我们荣幸地提供了一个嵌入式AI工具 - [**SenseCraft Model Assistant**](/cn/ModelAssistant_Introduce_Overview)，用于在嵌入式设备上实现更快、更准确的推理。
- [**Home Assistant**](/cn/home_assistant_topic) 旨在实现家庭自动化的高级隐私、多功能性和可持续性。
  - 我们为 Home Assistant 开发了多个即用型集成，如 [**SenseCAP**](/cn/home_assistant_sensecap)、[**LoRaWAN**](/cn/ha_xiao_esp32)、[**SenseCraft**](/cn/sensecraft_homeassistant_userguide)
- [**开源**](/cn/open_source_topic) 运动推动了硬件和软件的草根创新。
- [**边缘AI**](/cn/edge_ai_topic) 通过将智能推向边缘，更接近数据源，实现实时自动化决策，进一步推进了这一发展。

总的来说，这些技术将控制权交还给个人，实现根据本地需求定制的自动化和智能系统，同时保护隐私和自主权。
