# Seeed Studio 的开源项目

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/open_source_topic/open_source_topic.png" style={{width:1000, height:'auto'}}/></div>

<div>开源是推动技术世界创新和协作的强大力量。它促进了想法、知识和代码的共享，使开发者和爱好者能够共同合作，在现有解决方案的基础上进行构建。</div> 
<div>作为一家创新公司，Seeed Studio 在硬件和软件项目中拥抱开源原则，我们已经开发和制造了广泛的电子产品，从开发板和传感器到机器人和物联网（IoT）设备。</div>
<div>在本页面中，Seeed Studio 将持续介绍其开源硬件和开源软件。</div>

## 开源硬件

### Wio Terminal - 开放制造硬件

Seeed Studio 致力于开源硬件运动，本着这种精神，我们将其进一步发展为开放制造。我们希望让任何人都能够复制和制造这个设备。我们相信开放制造过程将释放创新和创造力的巨大潜力。

## 开源软件

### CodeCraft - 开源软件

Codecraft 是一款图形化编程软件，采用拖拽界面设计，旨在帮助初学者编程，无需使用传统基于文本的编程语言编写代码。该软件于 2018 年推出，在 2022 年被认定为 Bett Awards 决赛入围者和 The Edtech Awards Cool Tool 决赛入围者，现已完全开源。
