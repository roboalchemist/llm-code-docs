**Grove IoT 入门套件由 Amazon Web Service (AWS) 提供支持** 是四个不同的套件，专为 _AWS_ 设计以简化物联网开发。每个套件包含一个可连接到云端的开发平台（主板）和一套 Grove 模块。这些 Grove 模块为任何对 AWS IoT 感兴趣的人提供了必要的传感器和执行器。

_**注意：**_ AWS IoT 和套件中的开发板具有许多功能。下面的教程涵盖了将开发平台连接到 [AWS IoT](https://aws.amazon.com/) 的基本步骤。根据您选择的 _开发平台_ 和熟悉程度选择四个套件中的一个。

**特性**

* 开箱即用，即插即用的 Grove 模块，无需面包板或焊接工作。

* 完全符合 AWS 的服务和 AWS 的实践。

* 为开发者和创客提供逐步教程，实现更快的原型制作。

## 熟悉 AWS IoT 和套件

1. 阅读、学习并理解 [AWS IoT 如何工作](https://aws.amazon.com/iot/how-it-works/) 和 [入门指南](https://aws.amazon.com/iot/getting-started/)

2. 注册 [AWS 账户](https://portal.aws.amazon.com/gp/aws/developer/registration/index.html)（_如果尚未完成_）以试验 AWS IoT。

3. 阅读、学习并理解套件中的开发平台。主板文档的链接在套件用户手册中提供。请参见下面的各个套件部分。

4. 套件文档提供了设置开发板连接到 AWS 的操作指南。使用此说明下载板子所需的软件和固件。

5. 探索将 Grove 模块与主板结合使用的可能性。套件文档中提供了每个 Grove 模块文档的链接。

## Seeeduino Cloud 和 Grove IoT 入门套件由 AWS 提供支持

![enter image description here](https://files.seeedstudio.com/wiki/Grove_IoT_Starter_Kits_Powered_by_AWS/img/Aws_kit_seeeduino_cloud.JPG)

**[Seeeduino Cloud](/cn/Sensor/Seeeduino Series/Seeeduino_Cloud "Seeeduino Cloud")** 是基于 Dragino WiFi IoT 模块 [HE](http://www.dragino.com/products/linux-module/item/87-he.html) 和 ATmega32u4 的微控制器板。**HE** 是一个高性能、低成本的 150M、2.4G WiFi 模块，内置开源 OpenWrt 系统。Seeeduino Cloud 也是一个 [Arduino Yun](https://www.arduino.cc/en/Main/ArduinoBoardYun) 兼容板。Grove IoT 入门套件包含最常用的传感器和执行器，可将您的想法转化为与 AWS 云计算相结合的有形应用。

**[LinkIt ONE](https://wiki.seeedstudio.com/cn/LinkIt_ONE/ "LinkIt ONE")** 开发板是一个开源的高性能开发板，用于可穿戴设备和物联网设备的原型开发。它基于 MT2502（ARM7EJ-S™ MCU），在单个开发板上集成了 GSM、GPRS、Wi-Fi、蓝牙 BR/EDR/BLE、GPS、音频编解码器和 SD 卡连接器。该套件使您能够构建涉及智慧城市、交通运输和物流通信的云计算连接项目。
