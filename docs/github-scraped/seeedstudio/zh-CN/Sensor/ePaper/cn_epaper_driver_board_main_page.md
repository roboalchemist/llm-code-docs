# ePaper 驱动板概述

## 简介

ePaper 驱动板系列专为低功耗 ePaper 应用而设计，通用支持 24 针和 50 针显示屏，同时具备共享的核心功能：带电源开关的 JST 2.0 mm 电池连接器、集成充电 IC、一个复位按钮和三个用户可配置按钮；提供多种微控制器选项（如 XIAO ESP32-S3 Plus、XIAO nRF52840 Plus）——某些型号还增加了 NFC 等额外功能——使其成为数字标牌、电子标签、便携式信息板和其他低功耗显示项目的理想选择。

由 **XIAO ESP32-S3** Plus 驱动，显示板 EE04 支持 **24 针**和 **50 针** ePaper 显示屏。它具有带电源开关的 JST 2.0 mm 电池连接器、内置充电 IC，并配有一个复位按钮和三个用户按钮。非常适合低功耗 ePaper 项目，如数字标牌、电子标签和便携式信息板。

### 特性

- **由 XIAO ESP32-S3 Plus 驱动：** 连接到兼容的 ePaper 显示屏后即可立即工作。
- **多功能显示支持：** 兼容各种 ePaper 屏幕，支持 24 针和 50 针接口，通过跳线帽轻松切换。
- **带开关的电池连接器：** 提供简单的电池连接并集成开关，实现高效的电源管理和节能。
- **用户友好的按钮：** 包括 1 个复位按钮和 3 个用户可编程按钮，为项目加速和自定义功能提供灵活性。

## EE02 驱动板

**EE02** 是一款专为 **13.3 英寸**电子纸显示屏设计的高性能 E-ink 屏幕驱动板。由 ESP32-S3 芯片驱动，支持 Wi-Fi 和蓝牙连接，是无线显示应用的理想解决方案。

### 特性

- **强大的控制器**：基于 ESP32-S3 构建，提供强大的处理能力和无线功能。
- **专为大屏设计**：专门优化以驱动 13.3 英寸 E-ink 显示屏，具有高稳定性和清晰度。
- **无线连接**：支持 Wi-Fi 和蓝牙 5.0 (LE)，实现远程数据更新和云集成。
- **低功耗**：专为能效设计，适用于电池供电应用。
- **易于集成**：兼容 SenseCraft HMI，支持拖放界面设计和远程管理。
- **多功能接口**：包括用于供电/编程的 Type-C 和用于 E-ink 屏幕的标准连接器。
