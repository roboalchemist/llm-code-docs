[**SenseCAP T2000 追踪器**](https://www.seeedstudio.com/SenseCAP-Asset-Tracker-T2000-A-p-6580.html)是一款工业级 LoRaWAN® 资产追踪器，支持 GNSS、蓝牙和 Wi-Fi 定位，可在室内外环境中提供可靠的追踪功能。它具有 IP67 防护等级，内置 3 轴加速度计可检测运动状态，以及防拆按钮，当设备被移除时会触发最高优先级警报。T2000-A 和 T2000-B 支持长效电池供电，而太阳能供电的 T2000-C 配备可充电电池，确保持续的户外使用，使该系列产品成为长期免维护资产追踪的理想选择。
