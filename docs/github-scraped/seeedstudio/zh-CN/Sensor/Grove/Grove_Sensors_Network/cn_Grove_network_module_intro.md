# Grove 网络模块指南

Grove 网络模块是由 Seeed Studio 推出的模块化电子组件，是其 Grove 系列的一部分。Grove 系列模块旨在简化原型制作过程，使电子项目的构建更加直观和快速。网络模块主要提供网络连接功能，允许设备通过有线或无线方式连接到网络。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/Grove_Intro/Grove_Network_Module.png" style={{width:1000, height:'auto'}}/></div>

总的来说，Grove 网络模块通过简化连接和配置过程，显著降低了电子项目开发的门槛，让您能够专注于功能实现和应用创新。

## 特性

- 模块化设计：Grove 模块通过标准化接口（通常是 4 针连接器）连接，便于与各种主控板（如 Arduino、Raspberry Pi 等）集成。

- 即插即用：大多数 Grove 模块都配有详细的文档和示例代码，用户可以快速开始开发。

- 多样性：Grove 系列包括各种传感器、执行器和通信模块，适用于多种应用场景。


## 不同类型的 Grove 网络模块

我们有各种类型的 Grove 网络模块，可以应用于不同的场景，如物联网项目、无线传感器网络和智能家居。
