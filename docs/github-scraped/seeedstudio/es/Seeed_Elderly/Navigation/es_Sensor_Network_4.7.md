Este tema presenta la documentación de productos de Seeed Studio sobre el primer paso de los datos digitalizados del mundo físico, es decir, el despliegue de infraestructuras de información en un marco de detección dinámico, para sentar las bases del monitoreo social y la comprensión de los entornos físicos y naturales.

Puedes descubrir esta página con dos grandes porciones.

## IoT Industrial - Especializado

<strong><font color={'8DC215'} size={"4"}>Lanzamos sensores de grado industrial con conectividad de largo alcance e inteligencia, generando datos digitalizados del mundo físico con precisión a nivel instrumental. Puedes encontrar en esta porción:</font></strong>

- Sensores y Sondas para detección de datos ambientales de varios grados industriales
- Sensores basados en IA para escenarios inteligentes listos para desplegar
- Registrador de Datos para monitoreo remoto de datos ambientales
- Gateway para transmisión de datos de largo alcance, alta seguridad, interior y exterior
- Otras infraestructuras de red para transmisión de datos, como routers.
