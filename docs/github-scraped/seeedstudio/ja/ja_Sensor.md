センシングは、物理世界からデジタル化されたデータを取得するための最初であり、おそらく最も重要なステップです。動的なセンシングフレームワーク内でセンサーと情報インフラを活用することで、社会活動の監視や物理的および自然環境の理解の基盤を築きます。このセンシングプロセスにより、周囲の世界からデータを収集し理解することが可能になります。

このページでは、以下の2つの主要な製品カテゴリを紹介します。

 - **産業用センサーおよびネットワーキングデバイス**

 - **エントリーレベルのセンサーおよびネットワーキングモジュール**
