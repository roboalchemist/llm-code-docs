これらはSDカードシールドです。Arduinoにストレージスペースを提供します。ユーザーはArduinoの内蔵SDライブラリを使用してSDカードの読み書きが可能です。SD、SDHC、Micro SDカードをサポートしています。

|![](https://files.seeedstudio.com/wiki/SD_Card_Shield/img/Sdcardshield_01.jpg) |![](https://files.seeedstudio.com/wiki/SD_Card_Shield/img/Sdcardshield_01.jpg)|
|---|---|
|[SDカードシールド V3.0](https://seeeddoc.github.io/SD_Card_Shield_V3.0/) |[SDカードシールド V4.0](https://seeeddoc.github.io/SD_Card_shield_V4.0/) |

以下は、さまざまなバージョンのSDカードシールドの比較を示しています：

 |パラメータ|SDカードシールド V3.0|SDカードシールド V4.0|
 |---|---|---|
 |**動作電圧**|+2.7V~+3.6V|+5V|
 |**通信プロトコル**|SPI|SPI|
 |**SDカードソケット**|2つのソケット（SDカードソケットとMicro SDカードソケット）|標準SDソケット（アダプターを含み、Micro SDカードの使用を可能にします）|
|**標準シールド**|いいえ|はい|
 |**対応カードタイプ**|SDカード（&lt;=2G）；Micro SDカード（&lt;=2G）；SDHCカード（&lt;=16G）|SDカード（&lt;=16G）；Micro SDカード（&lt;=2G）；SDHCカード（&lt;=16G）|
 |**互換性**|Megaと互換性なし|Megaと互換性あり|
