## はじめに

BeagleBone®はマイクロコントローラーの世界では比較的新しい存在ですが、高性能と低消費電力により、オープンハードウェアの世界で人気のある3つのプラットフォームの1つとなっています。BeagleBone®は小型ながら大きな機能を持つPCであり、さまざまな用途に使用できるだけでなく、デスクトップPCと同じタスクを多く処理することができます。ArduinoやRaspberry Piと同様に、BeagleBone®ボードはアーティスト、デザイナー、ホビイストによってプロジェクトを実現するために使用されています。

BeagleBone®ボードは素晴らしいものですが、Seeedのメンバーは現在のBeagleBone®に満足せず、何か新しい変化を求めています。Seeeduinoで行ったように、私たちはBeagleBone.orgと話し合い、自分たちの考えを伝えた結果、彼らと一緒に全く新しいボードを作る機会を得ました。こうしてSeeed Studio独自のBeagleBone®、BeagleBone® Green（略称：BBG）が誕生しました。

<div style={{textAlign:'center'}}><img src="https://files.seeedstudio.com/wiki/BeagleBone_Green/images/cover.jpg" style={{width:800, height:'auto'}}/></div>

BeagleBone® GreenはBeagleBone® Black（略称：BBB）を基にしています。もしBBBについて知っているなら、BBBがBeagleBoardファミリーのスター的な存在であることをご存じでしょう。BBBはBeagleBone®の小型サイズを受け継ぎ、多くの優れた機能を追加しながら価格を大幅に抑えています。BBGを設計する際、BBBの小型サイズ、豊富なインターフェース、強力な拡張性といった優れた特徴をほぼすべて維持しつつ、HDMIインターフェースを2つのSeeed Groveコネクタに置き換え、Groveセンサーの大家族と互換性を持たせることにしました。

その後間もなく、BeagleBone® Green Wireless（略称：BBGW）という新しいBeagleBone®をリリースしました。BBGWはGroveコネクタに加え、高性能で柔軟なWiFi/Bluetoothインターフェースを搭載しており、BeagleBone®ファミリー初のWiFi/Bluetoothボードとなりました。

## オープンソースの理念

BeagleBone®はオープンソースの哲学を採用しており、ユーザーがハードウェアとソフトウェアの設計にアクセス、変更、共有することを可能にしています。このオープン性は、イノベーション、コラボレーション、透明性を促進し、知識とリソースが自由に交換されるコミュニティを育成します。オープンソースの原則を活用することで、BeagleBone®は誰もがその開発と改善に貢献できるようにし、プロジェクト、チュートリアル、アプリケーションの豊かなエコシステムを構築しています。

## シリーズの仕様の違い

BeagleBone®シリーズには、異なるニーズに対応するためのユニークな仕様を持つ複数のモデルがあります：

- **BeagleBone Black**: 最も人気のあるモデルで、1GHz ARM Cortex-A8プロセッサ、512MBのDDR3 RAM、4GBのeMMCフラッシュストレージを搭載しています。Ethernet、USB、HDMIなどの広範な接続オプションを提供します。

- **BeagleBone Blue**: ロボティクス専用に設計されており、オンボードIMUセンサー、サーボモーター出力、バッテリー管理を備えており、自律型車両プロジェクトに最適です。

- **BeagleBone Green**: Seeed Studioとのコラボレーションによるモデルで、BeagleBone Blackに似ていますが、HDMIポートを2つのGroveコネクタに置き換え、センサーの統合を容易にしています。
