このエンクロージャは軍用グレードのアルミニウムと耐久性のあるアクリルで構成されており、物理的な衝撃や腐食から究極の保護を提供します！スケルトンボックスは標準的なArduinoに対応するクリアランスを確保しており、エンクロージャは3次元すべてで積み重ね可能で、無限の拡張性を実現します。スケルトンは使いやすく、Arduinoを最高の保護で守ることができます！これはBeagleBone®専用に設計されたシンプルなケースです。

<p style={{textAlign: 'center'}}><a href="https://www.seeedstudio.com/Skeleton-Box-For-Beaglebone-p-1952.html" target="_blank"><img src="https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png"/></a></p>

## 特徴

---

* 整然として汎用性が高い

* 組み立てが簡単

* 頑丈で安定した素材
