![](https://files.seeedstudio.com/wiki/Wio_Link_Event_Kit/img/Wio_Link_Event_Kit_product_view_1200_s.jpg)

Wio Link Event Kitは、典型的なイベントトリガー型のGrove機能モジュールを含むイベント監視キットで、スマートIoTデバイスを構築するためのものです。このキットの最も驚くべき部分は、Wio Link開発ボードです。このボードは、プログラミングスキルが全く必要なく、さまざまなデバイスに利用できます。モジュールに実行させたいすべてのアクションや、Wio Linkへのコードのフラッシュ操作は、モバイルアプリでタップするだけで完了します。また、インターネットを介してどこからでもデバイスに指示を送ることができます。そのため、Wio Linkは周囲をよりスマートにするための完璧なツールとなります。では、このキットに含まれるいくつかのイベントトリガー型モジュールについて詳しく見ていきましょう。

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/depot/Wio-Link-Event-Kit-p-2611.html)


:::caution
     Wio IFTTT機能はEOL（End of Life）となっています。ただし、Wio APPは利用可能です。Wio APPのAPIを使用してセンサーの状態を読み取り、アクチュエータを制御することができます。
:::
特徴
--------

-   一般的で頻繁に使用されるイベント監視機能モジュールを含む。
-   周囲を完全にスマートにするのが簡単。
-   Groveポートインターフェース、より実用的な機能と少ない作業。
-   重量：145 g
