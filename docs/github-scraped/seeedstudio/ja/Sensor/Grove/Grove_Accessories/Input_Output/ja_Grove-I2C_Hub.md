![](https://files.seeedstudio.com/wiki/Grove-I2C_Hub/img/I2C_hub.jpg)

I2C Hub Groveは、複数のI2CデバイスをGrove Base Shieldに接続するための拡張Groveモジュールです。[Universal 4 Pin to X2 4 Pin cable](https://www.seeedstudio.com/depot/universal-4-pin-to-x2-4-pin-cable-5-pcs-pack-p-847.html?cPath=178_179)と組み合わせて使用することで、最大3つのI2Cデバイスを接続でき、ほとんどの開発目的をカバーすることができます。

[![](https://files.seeedstudio.com/wiki/common/Get_One_Now_Banner.png)](https://www.seeedstudio.com/Grove-I2C-Hub-p-851.html)

特徴
--------

- チェーン接続可能

## 仕様

| パラメータ             | 値/範囲       |

| 動作電圧             | 3.3/5V       |
| 動作温度             | -25℃ ～ +85℃ |
| サイズ               | 40mm x 20mm  |

アプリケーションアイデア
-----------------

- Groveプラットフォームで使用可能なI2Cデバイスの数を増やす。

使用方法
-----

同じI2Cソケットを同時に使用しても、各I2Cデバイスには固有のアドレスがあるため、競合することはありません。ハードウェアの設置方法は以下の通りです。
