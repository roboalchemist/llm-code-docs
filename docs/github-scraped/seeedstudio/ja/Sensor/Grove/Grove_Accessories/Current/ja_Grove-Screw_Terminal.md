Grove - スクリュー端子は、4つの3.5mmピッチピンを備え、最大125V @ 6Aに対応しています。この端子は30から20AWGのワイヤーを受け入れることができます。これは最も一般的なピッチのスクリュー端子です。このGroveには4つのポートがあり、GND、VCC、およびその他2つの未定義ポートが含まれます。スクリュー端子は、レコードプレーヤーやサージプロテクターなどのシャーシグラウンドを接続するためによく使用されます。また、建物内の公共アドレスシステムではスピーカーやその他の出力・入力に使用されることもあります。
