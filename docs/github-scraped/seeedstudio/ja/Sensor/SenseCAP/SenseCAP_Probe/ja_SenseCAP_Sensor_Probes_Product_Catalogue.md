## SenseCAP について

Seeed IIoT 製品シリーズの最初のローンチの中で、SenseCAP は無線環境センシングアプリケーションに焦点を当てています：スマート農業、精密農業、スマートシティなどです。ハードウェア製品（センサー、データロガー、ゲートウェイなど）、ソフトウェアサービス（SenseCAP Mate アプリ、SenseCAP ウェブポータル、オープンダッシュボード）、およびデバイスとデータ管理のための API で構成されています。

## センサープローブについて

当社の SenseCAP 産業用センサープローブシリーズは、産業用アプリケーションで一般的な振動や化学物質・ガスへの暴露などの過酷な環境に耐えるよう設計されており、当社の産業用センサーシリーズは IP66 から IP68 までの本格的な IP 等級を備えており、それぞれの過酷な環境で製品が確実に動作することを保証しています。
SenseCAP センサープローブは、測定対象媒体への確実な取り付けを保証するプロセス接続と、産業用計装システムに適合するよう信号を変換するセンサーデータトランスミッターも特徴としています。
一部の MODBUS RS485 産業用センサーには 2 つのバージョンがあります。主な違いはケーブルコネクターです。お客様のニーズに最適なバージョンをお選びください：
（1）フックアップワイヤー付きのものは、[SenseCAP LoRaWAN® データロガー](https://www.seeedstudio.com/SenseCAP-S2100-LoRaWAN-Data-Logger-p-5361.html?queryID=82d7de6ca64b8316686bdfe2c05664d2&objectID=5361&indexName=bazaar_retailer_products)と組み合わせて使用し、LoRaWAN® ネットワークに接続して、低消費電力・長距離通信の利点を享受することをお勧めします。
（2）航空コネクター付きのものは、すでに事前設定された [SenseCAP センサーハブ 4G データロガー](https://www.seeedstudio.com/SenseCAP-Sensor-Hub-4G-Data-Logger-with-builtin-battery-p-4852.html?queryID=b8ef1fa7e141765c30033fc3d8eddf10&objectID=4852&indexName=bazaar_retailer_products)でプラグアンドプレイ体験により、3G/4G LTE 経由で様々なセンサーデータを収集できます。
