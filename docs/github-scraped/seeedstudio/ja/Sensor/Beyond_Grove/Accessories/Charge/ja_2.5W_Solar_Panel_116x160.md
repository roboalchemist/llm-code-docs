![](https://files.seeedstudio.com/wiki/2.5W_Solar_Panel_116x160/img/2.5W.jpg)

これはカスタムソーラーパネルで、多くの開発ボードに直接接続でき、効率は15%と高いです。ユニットは透明エポキシコーティングと硬質基板を備えています。屋外用途に適した堅牢なシーリングが施されています！

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/2.5W-Solar-Panel-116X160-p-953.html)

##   仕様
---
*   PCBサイズ : 116*160*1.5 mm

*   モノリシック : 9.57*156 mm

*   効率 : 15%

*   電圧: 5.5V

*   電流: 450mA

*   出力: 2.5W

*   コネクタ: 2.0mm JST

*   タイプ: 156 二線式

*   PCB素材 : 全ガラス繊維

*   PCB要件 : 正極

*   品質要件 :

    *   平坦度は0.1mm以下
    *   清潔で、傷がないこと
    *   塗装剥がれや穴のずれなどの現象がないこと

*   **注意** : 許容誤差は正負0.1mm

*   名称: PETラミネート (116*160)

*   配列: 1*11（直列）間隔なし、黒色の貼り付け記事
