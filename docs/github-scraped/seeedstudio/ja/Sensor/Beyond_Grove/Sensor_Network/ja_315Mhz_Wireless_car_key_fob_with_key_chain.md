![](https://files.seeedstudio.com/wiki/315Mhz_Wireless_car_key_fob_with_key_chain/img/carkey_LRG.jpg)

これは4つのボタン（ロック、アンロック、ミュート、アラームオン）を備えたワイヤレス車用キーFOBです。いずれかのボタンを押すと青色のLEDが点灯します。このFOBを使用して、リモートコントロールプロジェクトのために[Codec-Adaptive Wireless](https://www.seeedstudio.com/depot/codecadaptive-wireless-relay-p-550.html)リレーと通信することができます。

[![](https://files.seeedstudio.com/wiki/Seeed-WiKi/docs/images/300px-Get_One_Now_Banner-ragular.png)](https://www.seeedstudio.com/depot/315mhz-wireless-car-key-fob-with-key-chain-battery-included-p-557.html?cPath=139_140)

## 特徴
---
*   周波数範囲: 高: 315.15MHz 平均: 315.025MHz 低: 314.85MHz

*   変調方式: 振幅変調 (AM)

*   データ転送速度: 625Hz

*   出力電力: ≤ 12mW

*   バッテリー電圧: DC12V

*   動作温度: -30℃ ~ +85℃
