# Plugin Seeding Time

## Description

The plugin adds the columns 'Finished' and 'Added' to the torrents list. This columns contains the time when download of the torrent was completed and, accordingly, the time when torrent was added.
As example:

![](images/PluginSeedingtime/seedingtime.png)

Of course the column contains values only for torrents completed/added after the plugin installed.

## Requirements

- rtorrent version >= 0.8.6
