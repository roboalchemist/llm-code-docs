# Plugin Feeds

## Description

This plugin is intended for making RSS feeds with information of torrents.

- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=all` - All torrents
- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=downloading` - Downloading torrents
- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=completed` - Completed torrents
- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=active` - Active torrents
- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=inactive` - Inactive torrents
- `http://your_server/ruTorrent/plugins/feeds/action.php?mode=error` - Error torrents
