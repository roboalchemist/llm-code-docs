## 19.6.0 / 2016/05/21

### Core & Logging

-   improvement of the binary upgrade behaviour using USR2: remove file
    locking (`1270`{.interpreted-text role="issue"})
-   add the `--capture-output` setting to capture stdout/stderr tot the
    log file (`1271`{.interpreted-text role="issue"})
-   Allow disabling `sendfile()` via the `SENDFILE` environment variable
    (`1252`{.interpreted-text role="issue"})
-   fix reload under pycharm (`1129`{.interpreted-text role="issue"})

### Workers

-   fix: make sure to remove the signal from the worker pipe
    (`1269`{.interpreted-text role="issue"})
-   fix: **gthread** worker, handle removed socket in the select loop
    (`1258`{.interpreted-text role="issue"})
