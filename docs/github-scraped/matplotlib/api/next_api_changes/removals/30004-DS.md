# `apply_theta_transforms` option in `PolarTransform`

Applying theta transforms in
[\~matplotlib.projections.polar.PolarTransform]{.title-ref} and
[\~matplotlib.projections.polar.InvertedPolarTransform]{.title-ref} has
been removed, and the `apply_theta_transforms` keyword argument removed
from both classes.

If you need to retain the behaviour where theta values are transformed,
chain the `PolarTransform` with a
[\~matplotlib.transforms.Affine2D]{.title-ref} transform that performs
the theta shift and/or sign shift.
