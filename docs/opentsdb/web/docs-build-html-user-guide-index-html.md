These pages serve as a user and administration guide. We highly recommend that you start with the writing and [Querying or Reading Data](https://opentsdb.net/docs/build/html/user_guide/query/index.html) sections to understand how OpenTSDB handles its core purpose of storing and serving time series information. Then follow up with the [Quick Start](https://opentsdb.net/docs/build/html/user_guide/quickstart.html) section to play around with getting some data into your OpenTSDB instance. Finally follow up with the other pages for details on the other features of OpenTSDB.
