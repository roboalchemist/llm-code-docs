COMPREHENSIVE LIST OF PROFILING VISUALIZATION AND PERFORMANCE ANALYSIS TOOLS

FLAME GRAPH GENERATORS AND VISUALIZERS
======================================

1. flamegraph.pl
   - Type: Generator
   - Language: Perl
   - Description: Original flame graph generator by Brendan Gregg. Converts folded stack traces into interactive SVG.
   - Workflow: perf/DTrace/Xperf → stackcollapse → flamegraph.pl → SVG
   - Status: Active, battle-tested

2. speedscope
   - Type: Visualizer
   - Language: JavaScript/Web
   - Description: Fast, interactive web-based flame graph viewer optimized for large datasets
   - Features: Zooming, filtering, multiple input formats
   - Status: Active, widely used

3. FlameScope
   - Type: Generator
   - Language: Open-source
   - Description: Subsecond heatmap visualization tool for CPU/GPU profiles
   - Features: Time range selection, temporal hotspot detection
   - Status: Active (Netflix)

4. AI Flame Graphs
   - Type: Generator
   - Language: Open-source
   - Description: Full-stack CPU+GPU flame graph generator with Intel GPU support
   - Features: Low overhead (<5%), iaprof integration
   - Status: Active, 2025+ development

5. d3-flame-graph
   - Type: Visualizer
   - Language: JavaScript/D3.js
   - Description: Interactive D3-based flame graph library suitable for web apps
   - Features: Zooming, transitions, bottom-up/top-down toggles
   - Status: Active

6. Grafana Pyroscope UI
   - Type: Visualizer
   - Language: Go/Web
   - Description: Cloud and open-source flame graph panels integrated into Grafana
   - Features: AI-assisted interpretation, table views, drilldowns, LLM analysis
   - Status: Active, 2025 enhancements
