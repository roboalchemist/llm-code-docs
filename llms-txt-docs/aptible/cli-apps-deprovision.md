# aptible apps:deprovision

This command deprovisions an [App](/core-concepts/apps/overview).

# Synopsis

```
Usage:
  aptible apps:deprovision

Options:
      [--app=APP]
  --env, [--environment=ENVIRONMENT]
  -r, [--remote=REMOTE]
```
