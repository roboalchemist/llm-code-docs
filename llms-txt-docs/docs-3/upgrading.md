# Upgrading

> Update the Turso CLI to the latest version to benefit from all the latest features, and fixes.

## macOS

```bash  theme={null}
brew update
brew upgrade
```

## Linux

```bash  theme={null}
turso update
```

## Windows

```bash  theme={null}
turso update
```
