# Mixpanel

> Track product analytics and user behavior with Mixpanel.

Add the following to your `docs.json` file to send analytics to Mixpanel.

```json Analytics options in docs.json theme={null}
"integrations": {
    "mixpanel": {
        "projectToken": "required"
    }
}
```
